## Introducción

Bienvenido a la guía completa para Azure Developer CLI (azd). Este repositorio está diseñado para ayudar a desarrolladores de todos los niveles, desde estudiantes hasta profesionales, a aprender y dominar Azure Developer CLI para implementaciones en la nube de manera eficiente. Este recurso estructurado proporciona experiencia práctica con implementaciones en la nube de Azure, resolución de problemas comunes e implementación de mejores prácticas para el despliegue exitoso de plantillas AZD.

## Objetivos de Aprendizaje

Al trabajar con este repositorio, podrás:  
- Dominar los fundamentos y conceptos principales de Azure Developer CLI  
- Aprender a implementar y aprovisionar recursos de Azure utilizando Infraestructura como Código  
- Desarrollar habilidades para solucionar problemas comunes en implementaciones de AZD  
- Comprender la validación previa al despliegue y la planificación de capacidad  
- Implementar mejores prácticas de seguridad y estrategias de optimización de costos  
- Ganar confianza en el despliegue de aplicaciones listas para producción en Azure  

## Resultados de Aprendizaje

Al completar este curso, serás capaz de:  
- Instalar, configurar y usar Azure Developer CLI con éxito  
- Crear e implementar aplicaciones utilizando plantillas AZD  
- Solucionar problemas de autenticación, infraestructura y despliegue  
- Realizar verificaciones previas al despliegue, incluida la planificación de capacidad y selección de SKU  
- Implementar mejores prácticas de monitoreo, seguridad y gestión de costos  
- Integrar flujos de trabajo de AZD en pipelines de CI/CD

## ¿Qué es Azure Developer CLI?

Azure Developer CLI (azd) es una interfaz de línea de comandos centrada en desarrolladores que acelera el proceso de creación e implementación de aplicaciones en Azure. Proporciona:  

- **Despliegues basados en plantillas** - Usa plantillas preconstruidas para patrones de aplicaciones comunes  
- **Infraestructura como Código** - Gestiona recursos de Azure utilizando Bicep o Terraform  
- **Flujos de trabajo integrados** - Aprovisiona, implementa y monitorea aplicaciones de manera fluida  
- **Amigable para desarrolladores** - Optimizado para la productividad y experiencia del desarrollador  

## Inicio Rápido

### Requisitos Previos
- Suscripción a Azure  
- Azure CLI instalado  
- Git (para clonar plantillas)  

### Instalación
```bash
# Windows (PowerShell)
powershell -ex AllSigned -c "Invoke-RestMethod 'https://aka.ms/install-azd.ps1' | Invoke-Expression"

# macOS/Linux
curl -fsSL https://aka.ms/install-azd.sh | bash
```  

### Tu Primer Despliegue
```bash
# Initialize a new project
azd init --template todo-nodejs-mongo

# Provision Azure resources and deploy
azd up
```
