# AZD para desarrolladores de IA

> Validado contra `azd 1.23.12` en marzo de 2026.

> [!IMPORTANT]  
> **Este taller está instrumentado con una guía que puedes previsualizar en tu navegador local. Para comenzar, abre el repositorio en GitHub Codespaces, en un contenedor de desarrollo o en una clonación local con MkDocs instalado—luego espera hasta que veas un terminal activo de VS Code y escribe:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Deberías ver un cuadro de diálogo emergente para abrir la página de vista previa en un navegador.**

Bienvenido al taller práctico para aprender Azure Developer CLI (AZD) con enfoque en el despliegue de aplicaciones de IA. Este taller te ayuda a obtener una comprensión aplicada de las plantillas AZD en 3 pasos:

1. **Descubrimiento** - encuentra la plantilla que es adecuada para ti.
1. **Despliegue** - despliega y valida que funciona
1. **Personalización** - modifica e itera para hacerla tuya!

A lo largo de este taller, también se te presentarán herramientas y flujos de trabajo fundamentales para desarrolladores, para ayudarte a optimizar tu recorrido de desarrollo de extremo a extremo.

## Descripción general del taller

**Duración:** 3-4 horas  
**Nivel:** Principiante a Intermedio  
**Requisitos previos:** Familiaridad con Azure, conceptos de IA, VS Code y herramientas de línea de comandos.

Este es un taller práctico donde aprendes haciendo. Una vez que hayas completado los ejercicios, recomendamos revisar el plan de estudios AZD para principiantes para continuar tu aprendizaje en prácticas recomendadas de Seguridad y Productividad.

| Tiempo| Módulo  | Objetivo |
|:---|:---|:---|
| 15 minutos | Introducción | Preparar el contexto, entender los objetivos |
| 30 minutos | Seleccionar plantilla de IA | Explorar opciones y elegir una plantilla inicial | 
| 30 minutos | Validar plantilla de IA | Desplegar la solución por defecto en Azure |
| 30 minutos | Desconstruir plantilla de IA | Explorar la estructura y la configuración |
| 30 minutos | Configurar plantilla de IA | Activar y probar las funciones disponibles |
| 30 minutos | Personalizar plantilla de IA | Adaptar la plantilla a tus necesidades |
| 30 minutos | Desmantelar infraestructura | Limpiar y liberar recursos |
| 15 minutos | Conclusión y próximos pasos | Recursos de aprendizaje, reto del taller |
| | |

## Qué aprenderás

Piensa en la plantilla AZD como un entorno de aprendizaje para explorar varias capacidades y herramientas para el desarrollo de extremo a extremo en Microsoft Foundry. Al final de este taller, deberías tener una sensación intuitiva de varias herramientas y conceptos en este contexto.

| Concepto  | Objetivo |
|:---|:---|
| **Azure Developer CLI** | Comprender los comandos de la herramienta y los flujos de trabajo|
| **AZD Templates**| Comprender la estructura del proyecto y la configuración|
| **Azure AI Agent**| Aprovisionar y desplegar un proyecto de Microsoft Foundry  |
| **Azure AI Search**| Habilitar ingeniería de contexto con agentes |
| **Observability**| Explorar trazas, monitoreo y evaluaciones |
| **Red Teaming**| Explorar pruebas adversariales y mitigaciones |
