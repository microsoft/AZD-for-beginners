# Taller AZD para Desarrolladores de IA

> Validado con `azd 1.27.1` en julio de 2026.

> [!IMPORTANT]  
> **Este taller está equipado con una guía del taller que puedes previsualizar en tu navegador local. Para comenzar, abre el repositorio en GitHub Codespaces, un contenedor de desarrollo o un clon local con MkDocs instalado — luego espera hasta que veas un terminal activo de VS Code y escribe:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Deberías ver un diálogo emergente para abrir la página de previsualización en un navegador.**

Bienvenido al taller práctico para aprender Azure Developer CLI (AZD) con un enfoque en el despliegue de aplicaciones de IA. Este taller te ayuda a obtener una comprensión aplicada de las plantillas AZD en 3 pasos:

1. **Descubrimiento** - encuentra la plantilla que es adecuada para ti.
1. **Despliegue** - despliega y valida que funcione
1. **Personalización** - modifica e itera para hacerlo tuyo!

A lo largo de este taller, también se te presentarán herramientas y flujos de trabajo esenciales para desarrolladores, para ayudarte a optimizar tu proceso de desarrollo de principio a fin.

## Resumen del Taller

**Duración:** 3-4 horas  
**Nivel:** Principiante a Intermedio  
**Requisitos:** Familiaridad con Azure, conceptos de IA, VS Code y herramientas de línea de comandos.

Este es un taller práctico donde aprendes haciendo. Una vez que completes los ejercicios, recomendamos revisar el plan de estudios AZD para Principiantes para continuar tu aprendizaje sobre mejores prácticas de Seguridad y Productividad.

| Tiempo| Módulo  | Objetivo |
|:---|:---|:---|
| 15 mins | Introducción | Establecer el contexto, entender los objetivos |
| 30 mins | Seleccionar Plantilla de IA | Explorar opciones y elegir un inicio | 
| 30 mins | Validar Plantilla de IA | Desplegar la solución predeterminada en Azure |
| 30 mins | Descomponer Plantilla de IA | Explorar estructura y configuración |
| 30 mins | Configurar Plantilla de IA | Activar y probar funciones disponibles |
| 30 mins | Personalizar Plantilla de IA | Adaptar la plantilla a tus necesidades |
| 30 mins | Desmantelar Infraestructura | Limpiar y liberar recursos |
| 15 mins | Conclusión y Próximos Pasos | Recursos de aprendizaje, desafío del taller |
| | |

## Lo Que Aprenderás

Piensa en la Plantilla AZD como un entorno de aprendizaje para explorar varias capacidades y herramientas para el desarrollo de extremo a extremo en Microsoft Foundry. Al final de este taller, deberías tener una intuición práctica sobre varias herramientas y conceptos en este contexto.

| Concepto  | Objetivo |
|:---|:---|
| **Azure Developer CLI** | Entender los comandos y flujos de trabajo de la herramienta|
| **Plantillas AZD**| Entender la estructura y configuración del proyecto|
| **Agente de Azure AI**| Provisionar y desplegar proyecto Microsoft Foundry  |
| **Búsqueda Azure AI**| Habilitar ingeniería de contexto con agentes |
| **Observabilidad**| Explorar trazabilidad, monitoreo y evaluaciones |
| **Pruebas Red Team**| Explorar pruebas adversariales y mitigaciones |
