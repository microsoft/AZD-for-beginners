# AZD 初學者課程：課程大綱及學習框架

## 課程概覽

透過結構化章節學習 Azure Developer CLI (azd)，**特別著重於結合 Microsoft Foundry 的 AI 應用程式部署**。

### 為什麼這課程對現代開發者至關重要

根據 Microsoft Foundry Discord 社群的洞察，**45% 的開發者希望使用 AZD 處理 AI 工作負載**，但面臨以下挑戰：
- 複雜的多服務 AI 架構
- 生產環境 AI 部署的最佳實踐  
- Azure AI 服務的整合與配置
- AI 工作負載的成本優化
- 解決 AI 部署特定問題的疑難排解

### 核心學習目標

完成這個結構化課程後，您將能夠：
- **掌握 AZD 基礎**：核心概念、安裝與配置
- **部署 AI 應用程式**：結合 Microsoft Foundry 服務使用 AZD
- **實現基礎設施即代碼**：使用 Bicep 模板管理 Azure 資源
- **解決部署問題**：處理常見問題並進行除錯
- **優化生產環境**：安全性、擴展性、監控及成本管理
- **構建多代理解決方案**：部署複雜的 AI 架構

**學習結構：**
- **靈活時間安排**：按自己的節奏完成章節，無論是幾天還是幾週
