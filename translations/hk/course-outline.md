# AZD 初學者課程：課程大綱及學習框架

## 課程概覽

透過結構化章節逐步學習 Azure Developer CLI (azd)，**特別著重於整合 Microsoft Foundry 部署 AI 應用程式**。

### 為什麼這課程對現代開發者至關重要

根據 Microsoft Foundry Discord 社群的洞察，**45% 的開發者希望使用 AZD 來處理 AI 工作負載**，但面臨以下挑戰：
- 複雜的多服務 AI 架構
- AI 部署至生產環境的最佳實踐  
- Azure AI 服務的整合及配置
- AI 工作負載的成本優化
- 解決 AI 部署特定問題的疑難排解

### 核心學習目標

完成這個結構化課程後，你將能：
- **掌握 AZD 基礎**：核心概念、安裝及配置
- **部署 AI 應用程式**：使用 AZD 與 Microsoft Foundry 服務
- **實現基礎設施即代碼**：使用 Bicep 模板管理 Azure 資源
- **排解部署問題**：解決常見問題及進行故障排除
- **優化生產環境**：安全性、擴展性、監控及成本管理
- **構建多代理解決方案**：部署複雜的 AI 架構

## 🎓 工作坊學習體驗

### 靈活的學習模式
此課程支持 **自學模式** 和 **工作坊模式**，讓學員能夠透過互動式練習獲得 AZD 的實際操作技能。

#### 🚀 自學模式
**適合個人開發者及持續學習**

**特色：**
- **瀏覽器介面**：透過任何網頁瀏覽器完成基於 MkDocs 的工作坊
- **GitHub Codespaces 整合**：一鍵啟動開發環境，內置工具已預配置
- **互動式 DevContainer 環境**：無需本地設置，立即開始編碼
- **進度追蹤**：內置檢查點及驗證練習
- **社群支持**：可加入 Azure Discord 頻道提問及合作

**學習結構：**
- **靈活時間安排**：按自己的節奏完成章節，可能需要幾天或幾週
- **檢查點系統**：在進入更複雜的主題前驗證學習成果
- **資源庫**：全面的文檔、範例及疑難排解指南
- **作品集開發**：構建可部署的專案以充實專業作品集

**自學模式入門：**
```bash
# Option 1: GitHub Codespaces (Recommended)
# Navigate to the repository and click "Code" → "Create codespace on main"

# Option 2: Local Development
git clone https://github.com/microsoft/azd-for-beginners.git
cd azd-for-beginners/workshop
# Follow setup instructions in workshop/README.md
```
