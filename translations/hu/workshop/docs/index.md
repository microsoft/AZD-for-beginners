# AZD az AI fejlesztők számára szánt workshophoz

> [!IMPORTANT]  
> **Ezt a workshopot egy workshop útmutatóval láttuk el, amelyet a helyi böngészőben megtekinthet. Az induláshoz el kell indítania a GitHub Codespaces-t a repóban — majd várjon, amíg aktív VS Code terminált nem lát, és írja be:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Ekkor egy felugró ablak jelenik meg, amely a böngészőben megnyitja az előnézeti oldalt.**

Üdvözlünk az Azure Developer CLI (AZD) gyakorlati workshopján, amely az AI alkalmazások telepítésére fókuszál. Ez a workshop segít, hogy gyakorlati ismereteket szerezz az AZD sablonokról 3 lépésben:

1. **Felfedezés** – találd meg a számodra megfelelő sablont.
1. **Telepítés** – telepítsd és igazold, hogy működik
1. **Testreszabás** – módosítsd és ismételd, hogy az a tiéd legyen!

A workshop során megismerkedsz a fő fejlesztői eszközökkel és munkafolyamatokkal, hogy egyszerűsítsd az end-to-end fejlesztési utadat.

## Workshop áttekintés

**Időtartam:** 3-4 óra  
**Szint:** Kezdő - középhaladó  
**Előfeltételek:** Azure, AI fogalmak, VS Code és parancssori eszközök ismerete.

Ez egy gyakorlati workshop, ahol tanulni csak csinálva lehet. A feladatok elvégzése után javasoljuk, hogy tekintsd át az AZD Kezdőknek tananyagot, hogy tovább folytathasd tanulásod a biztonság és termelékenység legjobb gyakorlatai felé.

| Idő| Modul  | Cél |
|:---|:---|:---|
| 15 perc | Bevezetés | A színpad beállítása, célok megértése |
| 30 perc | AI sablon kiválasztása | Lehetőségek feltérképezése és indító sablon kiválasztása | 
| 30 perc | AI sablon validálása | Alapértelmezett megoldás telepítése Azure-ra |
| 30 perc | AI sablon dekonstruálása | Szerkezet és konfiguráció feltérképezése |
| 30 perc | AI sablon konfigurálása | Elérhető funkciók aktiválása és kipróbálása |
| 30 perc | AI sablon testreszabása | Sablon átalakítása a saját igények szerint |
| 30 perc | Infrastruktúra lebontása | Takarítás és erőforrások felszabadítása |
| 15 perc | Összefoglalás és következő lépések | Tanulási források, workshop kihívás |
| | |

## Amit megtanulsz

Az AZD sablont úgy tekintheted, mint egy tanulós sandboxot, ahol felfedezheted a Microsoft Foundry end-to-end fejlesztési eszközeit és képességeit. A workshop végére intuitívan érteni fogod a legfontosabb eszközöket és fogalmakat.

| Fogalom  | Cél |
|:---|:---|
| **Azure Developer CLI** | Értsd meg az eszköz parancsait és munkafolyamatait|
| **AZD sablonok**| Értsd meg a projekt szerkezetét és konfigurációját |
| **Azure AI Agent**| Microsoft Foundry projekt előkészítése és telepítése  |
| **Azure AI Search**| Kontextus-kezelés engedélyezése ügynökökkel |
| **Megfigyelhetőség**| Tracing, monitoring és értékelések kipróbálása |
| **Red Teaming**| Ellenséges tesztelés és kockázatcsökkentés felfedezése |
| | |
