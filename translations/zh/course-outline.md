# AZD 初学者指南：课程大纲与学习框架

## 课程概述

通过结构化章节逐步学习 Azure Developer CLI (azd)。**特别关注通过 Microsoft Foundry 集成部署 AI 应用程序。**

### 为什么这门课程对现代开发者至关重要

根据 Microsoft Foundry Discord 社区的反馈，**45% 的开发者希望使用 AZD 处理 AI 工作负载**，但面临以下挑战：
- 复杂的多服务 AI 架构
- 生产环境中 AI 部署的最佳实践  
- Azure AI 服务的集成与配置
- AI 工作负载的成本优化
- AI 部署特定问题的排查

### 核心学习目标

完成本课程后，您将能够：
- **掌握 AZD 基础知识**：核心概念、安装与配置
- **部署 AI 应用程序**：结合 Microsoft Foundry 服务使用 AZD
- **实现基础设施即代码**：使用 Bicep 模板管理 Azure 资源
- **排查部署问题**：解决常见问题并调试故障
- **优化生产环境**：安全性、扩展性、监控与成本管理
- **构建多代理解决方案**：部署复杂的 AI 架构
