**此版本引入了使用 Azure Developer CLI (AZD) 的全面、可用于生产的容器应用部署示例，附带完整文档并集成到学习路径中。**

- **代码示例**：完整的 `azure.yaml`、Bicep 模板，以及多语言服务实现（Python、Node.js、C#、Go）
- **测试与故障排除**：端到端测试场景、监控命令、故障排除指南

#### 更改

- **examples/README.md**：更新以强调容器应用示例，添加比较矩阵条目，并更新技术/架构参考
- **课程大纲与学习指南**：更新以在相关章节中引用新的容器应用示例和部署模式
