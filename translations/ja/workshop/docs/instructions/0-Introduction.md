# AZD for AI開発者

!!! info "このワークショップの対象者"
   
    このワークショップは、Azure AI Foundryを活用してAIエージェントアプリケーションを構築したい方を対象としています。Azure、AIの基本概念、Pythonコードにある程度の知識がある初心者から経験豊富な開発者まで幅広く対応しています。開始する前に必要条件を確認してください。

    **所要時間:** 3～4時間  
    **レベル:** 初級～中級  

---

## 必要条件

- 個人のGitHubアカウント → [無料で作成](https://github.com/signup)
- 有効なAzureアカウント → [無料で作成](https://aka.ms/free)
- AIの基本概念、Python開発、コマンドラインツールに関する知識
- VS CodeやGitHubツール、ワークフローの使用経験

---

## 学習目標

このワークショップを終えると、以下のことができるようになります:

- [X] **Azure Developer CLIの使用** → Azureリソースのプロビジョニング、デプロイ、管理を行う
- [X] **AIアプリテンプレートの利用** → Azure AI FoundryでAIエージェントプロジェクトを迅速に開始する
- [X] **テンプレートの設定** → Red Teaming、評価、トレーシング＆モニタリングなどの機能を設定する
- [X] **AZDテンプレートとワークフローの分解と探索** → GitHub Copilotを使用して理解を深める
- [X] **カスタマイズと再デプロイ** → AIエージェントのスターターテンプレートを独自の要件に合わせてカスタマイズする

---

## ワークショップ概要

このワークショップはモジュール形式で構成されており、各モジュールには実践的なラボ演習が含まれています:

1. **イントロダクション** - AZDテンプレートとは何か、その仕組みを理解する
1. **選択** - 次のAIプロジェクトに最適な「スターター」テンプレートを見つける
1. **検証** - テンプレートをデプロイして、期待通りに動作するか確認する
1. **分解** - テンプレートを分解して、その構成要素と設定を理解する
1. **設定** - Azure AI Foundryのさまざまな機能を有効化して使用する方法を学ぶ
1. **カスタマイズ** - 自分のデータと要件を考慮してアプリケーションを再構築する
1. **解体** - インフラを解除し、ソフト削除されたリソースを回復する方法を学ぶ
