# Changelog - AZD For Beginners

## Introduction

या changelog मध्ये AZD For Beginners रेपॉझिटरीतील सर्व उल्लेखनीय बदल, अद्यतने आणि सुधारणा दस्तऐवजीकरण केले आहे. आम्ही सेमान्टिक व्हर्जनिंग तत्त्वांचे पालन करतो आणि वापरकर्त्यांना आवृत्त्यांमधील फरक समजण्यासाठी हे लॉग राखतो.

## Learning Goals

हा changelog पाहून, आपण:
- नवीन वैशिष्ट्ये आणि सामग्री वाढीबद्दल माहिती मिळवाल
- विद्यमान डॉक्युमेंटेशनमध्ये केलेल्या सुधारणा समजू शकता
- अचूकता सुनिश्चित करण्यासाठी बग फिक्स आणि सुधारणा ट्रॅक करू शकता
- वेळोवेळी शिक्षण साहित्याचा विकास पाठपुरावा करू शकता

## Learning Outcomes

changelog नोंदी पाहिल्यानंतर, आपण सक्षम असाल:
- शिकण्यासाठी उपलब्ध नवीन सामग्री आणि संसाधने ओळखण्यास
- कोणत्या विभागांचे अद्यतन किंवा सुधारणा झाली आहेत हे समजण्यास
- सर्वात वर्तमान साहित्यावर आधारित आपली शिकण्याची योजना बनविण्यास
- भविष्यातील सुधारणा साठी अभिप्राय आणि सूचना देण्यास

### [v3.10.0] - 2026-02-05

#### Azure Developer CLI कमांड अचूकता अद्यतन
**या आवृत्तीत संपूर्ण दस्तऐवजीकरणात अस्तित्वात नसलेल्या AZD कमांड्स दुरुस्त केल्या गेल्या आहेत, जेणेकरून सर्व कोड उदाहरणे वैध Azure Developer CLI सिंटॅक्स वापरतील.**

#### दुरुस्ती
- **🔧 अस्तित्वात नसलेले AZD कमांड्स काढले**: अवैध कमांड्सची सर्वसमावेशक तपासणी आणि दुरुस्ती:
  - `azd logs` (अस्तित्वात नाही) → `azd monitor --logs` किंवा Azure CLI पर्यायांनी बदलले
  - `azd service` उप-कमांड्स (अस्तित्वात नाहीत) → `azd show` आणि Azure CLI ने बदलले
  - `azd infra import/export/validate` (अस्तित्वात नाहीत) → काढले किंवा वैध पर्यायांनी बदलले
  - `azd deploy --rollback/--incremental/--parallel/--detect-changes` फ्लॅग्ज (अस्तित्वात नाहीत) → काढले
  - `azd provision --what-if/--rollback` फ्लॅग्ज (अस्तित्वात नाहीत) → `--preview` वापरून अद्यतनित
  - `azd config validate` (अस्तित्वात नाही) → `azd config list` ने बदलले
  - `azd info`, `azd history`, `azd metrics` (अस्तित्वात नाहीत) → काढले

- **ऑफलाइन प्रवेश**: डाउनलोड करण्यायोग्य दस्तऐवजीकरण पॅकेजेस
- **सुधारित IDE एकत्रीकरण**: AZD + AI कार्यप्रवाहांसाठी VS Code विस्तार
- **समुदाय डॅशबोर्ड**: रिअल-टाइम समुदाय मेट्रिक्स आणि योगदान ट्रॅकिंग

## चेंजलॉगमध्ये योगदान

### बदल नोंदवणे
या रेपॉजिटरीमध्ये योगदान करताना, कृपया सुनिश्चित करा की चेंजलॉग एन्ट्रीमध्ये समाविष्ट आहे:

1. **आवृत्ती क्रमांक**: सेमॅंटिक आवृत्ती (major.minor.patch) प्रमाणे
2. **तारीख**: प्रकाशन किंवा अद्यतन तारीख YYYY-MM-DD स्वरूपात
3. **श्रेणी**: Added, Changed, Deprecated, Removed, Fixed, Security
4. **स्पष्ट वर्णन**: काय बदलले याचे संक्षिप्त वर्णन
5. **प्रभाव मूल्यांकन**: विद्यमान वापरकर्त्यांवर बदलांच्या परिणामांचे वर्णन

### बदल श्रेण्या

#### समाविष्ट केले
- नवीन वैशिष्ट्ये, दस्तऐवजीकरण विभाग किंवा क्षमता
- नवीन उदाहरणे, टेम्पलेट्स किंवा शिकण्याचे संसाधने
- अतिरिक्त साधने, स्क्रिप्ट्स, किंवा युटिलिटीज

#### बदलले
- विद्यमान कार्यक्षमता किंवा दस्तऐवजीकरणात बदल
- स्पष्टता किंवा अचूकता सुधारण्यासाठी अद्यतने
- सामग्री किंवा संघटनेचे पुनर्रचना

#### निषेधित
- वापरातून काढले जाणारे वैशिष्ट्ये किंवा पद्धती
- काढून टाकण्याच्या शेड्यूलवरील दस्तऐवजीकरण विभाग
- ज्यांचे चांगले पर्याय उपलब्ध आहेत अशा पध्दती

#### काढले
- संबंधित नसलेली वैशिष्ट्ये, दस्तऐवजीकरण किंवा उदाहरणे
- जुनी किंवा निषेधित माहिती किंवा पद्धती
- पुनरावृत्ती किंवा एकत्रित सामग्री

#### दुरुस्त केले
- दस्तऐवजीकरण किंवा कोडमधील त्रुटी दुरुस्ती
- अहवालातल्या समस्यांचा निराकरण
- अचूकता किंवा कार्यक्षमता सुधारणा

#### सुरक्षा
- सुरक्षा-संबंधित सुधारणा किंवा दुरुस्त्या
- सुरक्षा सर्वोत्तम प्रथांचे अद्यतन
- सुरक्षा असुरक्षिततेचे निराकरण

### सेमॅंटिक आवृत्ती मार्गदर्शक

#### मुख्य आवृत्ती (X.0.0)
- वापरकर्त्यांच्या कृतीची गरज असलेले मोडणारे बदल
- कंटेंट किंवा संघटनेतील महत्त्वाचा पुनर्रचना
- मूलभूत दृष्टीकोन किंवा पद्धतीत बदल

#### लहान आवृत्ती (X.Y.0)
- नवीन वैशिष्ट्ये किंवा सामग्रीची भर
- मागील सुसंगतता राखणाऱ्या सुधारणां
- अतिरिक्त उदाहरणे, साधने किंवा संसाधने

#### पॅच आवृत्ती (X.Y.Z)
- बग दुरुस्त्या आणि सुधारणा
- विद्यमान सामग्रीतील लहान सुधारणा
- स्पष्टता आणि सूक्ष्म सुधारणा

## समुदाय अभिप्राय आणि सूचना

आम्ही या शिकवणी संसाधनात सुधारणा करण्यासाठी समुदाय अभिप्राय प्रोत्साहित करतो:

### अभिप्राय कसा द्यावा
- **GitHub Issues**: समस्या नोंदवा किंवा सुधारणा सुचवा (AI-विशिष्ट समस्या स्वागतार्ह)
- **Discord चर्चासत्रे**: कल्पना शेअर करा आणि Microsoft Foundry समुदायात सहभागी व्हा
- **पुल रिक्वेस्ट्स**: सामग्रीमध्ये थेट सुधारणा करा, विशेषतः AI टेम्पलेट्स आणि मार्गदर्शक
- **Microsoft Foundry Discord**: AZD + AI चर्चांसाठी #Azure चॅनेलमध्ये सहभागी व्हा
- **समुदाय फोरम्स**: विस्तृत Azure डिव्हेलपर चर्चांत सहभागी व्हा

### अभिप्राय श्रेण्या
- **AI सामग्रीचे अचूकता**: AI सेवा एकत्रीकरण आणि परिनियोजन माहितीतील दुरुस्त्या
- **शिकण्याचा अनुभव**: AI डिव्हेलपर शिकण्याच्या प्रवाहासाठी सूचना
- **गहाळ AI सामग्री**: अतिरिक्त AI टेम्पलेट्स, नमुने किंवा उदाहरणांसाठी विनंत्या
- **प्रवेशयोग्यता**: विविध शिकणाऱ्या गरजांसाठी सुधारणा
- **AI साधन एकत्रीकरण**: AI विकास कार्यप्रवाह एकत्रीकरणासाठी सूचना
- **उत्पादन AI नमुने**: एंटरप्राइज AI परिनियोजन नमुन्यांच्या विनंत्या

### प्रतिसाद वचनबद्धता
- **समस्या प्रतिसाद**: नोंदवलेल्या समस्या ४८ तासांत उत्तर
- **वैशिष्ट्य विनंत्या**: एका आठवड्यात मूल्यमापन
- **समुदाय योगदान**: एका आठवड्यात पुनरावलोकन
- **सुरक्षा समस्या**: तत्काळ प्राधान्याने प्रतिसाद

## देखभाल वेळापत्रक

### नियमित अद्यतने
- **मासिक पुनरावलोकने**: सामग्रीची अचूकता आणि दुवे तपासणी
- **त्रैमासिक अद्यतने**: महत्त्वपूर्ण सामग्री वाढ आणि सुधारणा
- **सहाअरधी पुनरावलोकने**: व्यापक पुनर्रचना आणि सुधारणा
- **वार्षिक प्रकाशने**: महत्त्वपूर्ण आवृत्ती अद्यतने

### निरीक्षण आणि गुणवत्ता आश्वासन
- **स्वयंचलित चाचण्या**: कोड उदाहरणे आणि दुव्यांचे नियमित सत्यापन
- **समुदाय अभिप्राय समाकलन**: वापरकर्ता सूचना नियमित समाविष्ट करणे
- **तंत्रज्ञान अद्यतने**: नवीनतम Azure सेवा आणि azd आवृत्त्यांशी संरेखन
- **प्रवेशयोग्यता ऑडिट्स**: समावेशक डिझाइन तत्त्वांसाठी नियमित पुनरावलोकन

## आवृत्ती समर्थन धोरण

### चालू आवृत्ती समर्थन
- **नवीनतम मुख्य आवृत्ती**: नियमित अद्यतने आणि पूर्ण समर्थन
- **मागील मुख्य आवृत्ती**: सुरक्षा अद्यतने आणि महत्त्वपूर्ण दुरुस्ती १२ महिने
- **जुन्या आवृत्त्या**: फक्त समुदाय समर्थन, अधिकृत अद्यतने नाहीत

### स्थलांतर मार्गदर्शन
जेव्हा मुख्य आवृत्त्या जारी केल्या जातात, तेव्हा आम्ही प्रदान करतो:
- **मायग्रेशन मार्गदर्शक**: टप्प्याटप्प्याने संक्रमण सूचना
- **सुसंगतता नोंदी**: मोडणाऱ्या बदलांबद्दल तपशील
- **टूल समर्थन**: मायग्रेशनसाठी मदत करणारे स्क्रिप्ट्स किंवा सुविधादायक साधने
- **समुदाय समर्थन**: मायग्रेशन प्रश्नांसाठी समर्पित फोरम
