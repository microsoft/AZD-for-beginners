# AI विकसकांसाठी AZD कार्यशाळा

> `azd 1.25.6` च्या विरुद्ध जून 2026 मध्ये पडताळलेले.

> [!IMPORTANT]  
> **ही कार्यशाळा एका कार्यशाळा मार्गदर्शकाशी सुसज्ज आहे जी तुम्ही तुमच्या स्थानिक ब्राउझरमध्ये पूर्वावलोकन करू शकता. सुरू करण्यासाठी, GitHub Codespaces, एक dev container, किंवा MkDocs स्थापित केलेल्या स्थानिक क्लोनमध्ये रेपो उघडा—नंतर सक्रिय VS Code टर्मिनल दिसेपर्यंत थांबा आणि टाइप करा:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **तुम्हाला ब्राउझरमध्ये पूर्वावलोकन पृष्ठ उघडण्यासाठी एक पॉप-अप डायलॉग दिसायला हवे.**

Azure Developer CLI (AZD) शिकण्यासाठी AI अनुप्रयोग तैनातीवर लक्ष केंद्रित केलेल्या या हाताळणी कार्यशाळेत आपले स्वागत आहे. ही कार्यशाळा तुम्हाला AZD टेम्पलेट्स चा 3 टप्प्यांमध्ये व्यावहारिक समज प्राप्त करण्यास मदत करते:

1. **शोध** - तुमच्यासाठी योग्य टेम्पलेट शोधा.
1. **तैनाती** - तैनात करा आणि ते कार्य करीत आहे की नाही हे पडताळा
1. **स्वतंत्रीकरण** - तुमचे स्वतःचे बनवण्यासाठी सुधारणा करा आणि वारंवार करा!

या कार्यशाळेच्या काळात, तुम्हाला मुख्य विकसक साधने आणि कार्यप्रवाहांशी देखील परिचय करून दिला जाईल, ज्यामुळे तुमच्या संपूर्ण विकास प्रवासास सुलभ करण्यास मदत होईल.

## कार्यशाळेचे आढावा

**कालावधी:** 3-4 तास  
**स्तर:** प्रारंभिक ते मध्यम  
**पूर्वआवारकता:** Azure, AI संकल्पना, VS Code & कमांड-लाइन साधनांशी परिचय.

ही एक हाताळणी कार्यशाळा आहे जिथे तुम्ही करत शिकता. एकदा तुम्ही सराव पूर्ण केल्यानंतर, आम्ही तुम्हाला AZD For Beginners अभ्यासक्रम पुनरावलोकन करण्याची शिफारस करतो जेणेकरून तुम्ही सुरक्षा आणि उत्पादकता सर्वोत्तम प्रथांचे शिक्षण पुढे चालू ठेवू शकाल.

| वेळ| विभाग  | उद्दिष्ट |
|:---|:---|:---|
| 15 मिनिटे | परिचय | संदर्भ सेट करा, उद्दिष्ट समजून घ्या |
| 30 मिनिटे | AI टेम्पलेट निवडा | पर्याय शोधा आणि स्टार्टर निवडा | 
| 30 मिनिटे | AI टेम्पलेट पडताळा | Azure वर डिफॉल्ट सोल्यूशन तैनात करा |
| 30 मिनिटे | AI टेम्पलेट विघटन करा | रचना आणि कॉन्फिगरेशन एक्सप्लोर करा |
| 30 मिनिटे | AI टेम्पलेट कॉन्फिगर करा | उपलब्ध फिचर्स सक्रिय करा आणि वापरून पहा |
| 30 मिनिटे | AI टेम्पलेट सानुकूल करा | टेम्पलेट तुमच्या गरजेनुसार रूपांतर करा |
| 30 मिनिटे | इन्फ्रास्ट्रक्चर हटवा | साफसफाई करा आणि संसाधने मुक्त करा |
| 15 मिनिटे | समारोप आणि पुढील टप्पे | शिक्षण संसाधने, कार्यशाळा आव्हान |
| | |

## तुम्ही काय शिकाल

AZD टेम्पलेटला Microsoft Foundry वरील संपूर्ण विकासासाठी विविध क्षमता आणि साधने शोधण्यासाठी शिकण्याचा sandbox म्हणून विचार करा. या कार्यशाळेच्या शेवटी, तुम्हाला या संदर्भातील विविध साधने आणि संकल्पनांसाठी अंतर्निहीत जाणीव असावी.

| संकल्पना  | उद्दिष्ट |
|:---|:---|
| **Azure Developer CLI** | साधन आदेश आणि कार्यप्रवाह समजून घ्या|
| **AZD टेम्पलेट्स**| प्रकल्प रचना आणि कॉन्फिगरेशन समजून घ्या|
| **Azure AI Agent**| Microsoft Foundry प्रकल्प तैनात करा आणि पुरवठा करा |
| **Azure AI सर्च**| एजंटसह संदर्भ अभियांत्रिकी सक्षम करा |
| **ऑब्झर्व्हेबिलिटी**| ट्रेसिंग, मॉनिटरिंग आणि मूल्यमापन अन्वेषण करा |
| **रेड टीमिंग**| शत्रुत्वपूर्वक चाचणी आणि प्रतिबंधक उपाय अन्वेषण करा |
| | |
