# AI डेव्हलपर्ससाठी AZD कार्यशाळा

> [!IMPORTANT]  
> **ही कार्यशाळा एका कार्यशाळा मार्गदर्शकाशी सुसज्ज आहे ज्याचा तुम्ही तुमच्या स्थानिक ब्राउझरमध्ये पूर्वावलोकन करू शकता. सुरूवात करण्यासाठी, तुम्हाला रेपोवर GitHub Codespaces लॉन्च करावे लागेल—नंतर सक्रिय VS Code टर्मिनल दिसेपर्यंत थांबा आणि टाइप करा:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **तुम्हाला पूर्वावलोकन पृष्ठ ब्राउझरमध्ये उघडण्यासाठी एक पॉप-अप संवाद दिसेल.**

Azure Developer CLI (AZD) शिकण्यासाठी आणि AI अनुप्रयोग तैनात करण्यावर लक्ष केंद्रीत करण्यात आलेल्या या हाताळणी कार्यशाळेत आपले स्वागत आहे. ही कार्यशाळा तुम्हाला AZD टेम्पलेट्सचे तीन टप्प्यांत व्यावहारिक समज मिळवून देते:

1. **शोध** - तुमच्यासाठी योग्य असलेला टेम्पलेट शोधा.
1. **तैनाती** - तैनात करा आणि ते चालते का ते पडताळा
1. **सानुकूलन** - सुधारणा करा आणि ते तुमचं बनवा!

या कार्यशाळे दरम्यान, तुम्हाला कोअर डेव्हलपर साधने आणि वर्कफ्लोची ओळख देखील करून दिली जाईल, जी तुमच्या संपूर्ण विकास प्रवासाला सुलभ बनवेल.

## कार्यशाळेचे सारांश

**कालावधी:** 3-4 तास  
**पातळी:** प्रारंभिक ते मध्यम  
**पूर्वअटी:** Azure, AI संकल्पना, VS Code आणि कमांड-लाइन टूल्सची ओळख असणे.

ही एक हाताळणी कार्यशाळा आहे जिथे तुम्ही करून शिकता. एकदा तुम्ही सराव पूर्ण केल्यानंतर, सिक्युरिटी आणि उत्पादनक्षमतेच्या सर्वोत्तम पद्धतींबद्दल तुमचा अभ्यास चालू ठेवण्यासाठी AZD For Beginners अभ्यासक्रम पुनरावलोकन करण्याचा सल्ला देण्यात येतो.

| वेळ| मॉड्यूल  | उद्दिष्ट |
|:---|:---|:---|
| 15 मिनिटे | परिचय | व्यासपीठ तयार करा, उद्दिष्टे समजून घ्या |
| 30 मिनिटे | AI टेम्पलेट निवडा | पर्याय पाहा आणि प्रारंभ करा | 
| 30 मिनिटे | AI टेम्पलेट पडताळा | डिफॉल्ट सोल्युशन Azure वर तैनात करा |
| 30 मिनिटे | AI टेम्पलेटची रचना समजून घ्या | संरचना आणि कॉन्फिगरेशन शोधा |
| 30 मिनिटे | AI टेम्पलेट कॉन्फिगर करा | उपलब्ध वैशिष्ट्ये सक्रिय करा आणि वापरून पहा |
| 30 मिनिटे | AI टेम्पलेट सानुकूलित करा | टेम्पलेट तुमच्या गरजेनुसार बदला |
| 30 मिनिटे | इन्फ्रास्ट्रक्चर हटवा | साफसफाई करा आणि संसाधने मुक्त करा |
| 15 मिनिटे | समारोप आणि पुढील पावले | शिक्षण संसाधने, कार्यशाळेची आव्हानं |
| | |

## तुम्हाला काय शिकायला मिळेल

AZD टेम्पलेटला Microsoft Foundry वर संपूर्ण विकासासाठी विविध क्षमता आणि साधने एक्सप्लोर करण्यासाठी शिकण्याचा एक सैंडबॉक्स म्हणून विचार करा. या कार्यशाळेच्या शेवटी, तुम्हाला या संदर्भातील विविध साधने आणि संकल्पनांसाठी अंतर्ज्ञान मिळालेले पाहिजे.

| संकल्पना  | उद्दिष्ट |
|:---|:---|
| **Azure Developer CLI** | साधनांच्या कमांड आणि वर्कफ्लो समजून घ्या|
| **AZD टेम्पलेट्स**| प्रकल्पाची रचना आणि कॉन्फिगरेशन समजून घ्या|
| **Azure AI Agent**| Microsoft Foundry प्रकल्पाची तैनाती आणि व्यवस्था करा  |
| **Azure AI Search**| एजंट्ससह संदर्भ अभियांत्रिकी सक्षम करा |
| **Observability**| ट्रेसिंग, मॉनिटरिंग आणि मूल्यांकन पाहा |
| **Red Teaming**| विरोधी चाचणी आणि प्रतिबंधांचे अभ्यास करा |
| | |
