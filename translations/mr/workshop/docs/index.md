# AI विकसकांसाठी AZD कार्यशाळा

> `azd 1.27.1` च्या विरुद्ध जुलै 2026 मध्ये सत्यापित.

> [!IMPORTANT]  
> **ही कार्यशाळा कार्यशाळा मार्गदर्शकाशिवाय सुसज्ज आहे ज्याची आपण आपल्या स्थानिक ब्राउझरमध्ये पूर्वदर्शन करू शकता. सुरू करण्यासाठी, GitHub Codespaces मध्ये, एक dev कंटेनर मध्ये, किंवा MkDocs स्थापित केलेल्या स्थानिक क्लोनमध्ये रेपो उघडा—नंतर एक सक्रिय VS Code टर्मिनल दिसेपर्यंत थांबा आणि टाइप करा:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **आपल्याला ब्राउझरमध्ये पूर्वदर्शन पृष्ठ उघडण्याचा एक पॉप-अप संवाद बॉक्स दिसायला हवा.**

Azure Developer CLI (AZD) शिकण्यासाठी आणि AI अनुप्रयोग तैनातीवर लक्ष केंद्रित करणारी ही प्रत्यक्ष कार्यशाळा आपल्याला AZD टेम्पलेट्सची व्यावहारिक समज तीन टप्प्यांत मिळवून देते:

1. **शोध** - आपल्यासाठी योग्य टेम्पलेट शोधा.
1. **तैनात** - तैनात करा आणि कार्यरत आहे का याची पडताळणी करा
1. **सानुकूलन** - सुधारणा करा आणि आपल्या गरजेनुसार परिष्कृत करा!

या कार्यशाळेदरम्यान, आपण मुख्य विकसक साधने व कार्यप्रवाहांशी देखील परिचित व्हाल, जे आपली एंड-टू-एंड विकास प्रवास सुलभ करतील.

## कार्यशाळा आढावा

**कालावधी:** ३-४ तास  
**पातळी:** प्रारंभिक ते मध्यम  
**पूर्वआधार:** Azure, AI संकल्पना, VS Code आणि कमांड-लाइन साधनांची ओळख.

ही एक प्रत्यक्ष-करण्याची कार्यशाळा आहे जिथे आपण करत-करत शिकलात. एकदा आपण व्यायाम पूर्ण केल्यावर, सुरक्षा आणि उत्पादकता सर्वोत्तम पद्धतींमध्ये आपले शिक्षण सुरू ठेवण्यासाठी AZD For Beginners अभ्यासक्रम पुनरावलोकन करण्याची शिफारस केली जाते.

| वेळ| मोड्यूल  | उद्दिष्ट |
|:---|:---|:---|
| १५ मिनिटे | परिचय | मंच तयार करा, उद्दिष्टे समजा |
| ३० मिनिटे | AI टेम्पलेट निवडा | पर्यायांचा शोध घ्या आणि प्रारंभी निवडा | 
| ३० मिनिटे | AI टेम्पलेटची पडताळणी करा | Azure वर डीफॉल्ट सोल्यूशन तैनात करा |
| ३० मिनिटे | AI टेम्पलेटचा तपशील समजून घ्या | संरचना आणि कॉन्फिगरेशन तपासा |
| ३० मिनिटे | AI टेम्पलेट कॉन्फिगर करा | वैशिष्ट्ये सक्रिय करा आणि वापरून पहा |
| ३० मिनिटे | AI टेम्पलेट सानुकूल करा | आपल्या गरजेनुसार टेम्पलेट अनुकूल करा |
| ३० मिनिटे | पायाभूत सुविधा खाली उत्रा | संसाधने साफ करा आणि मुक्त करा |
| १५ मिनिटे | समारोप आणि पुढील पाऊले | शिक्षण स्रोत, कार्यशाळा चॅलेंज |
| | |

## आपण काय शिकाल

AZD टेम्पलेटला Microsoft Foundry वरील एंड-टू-एंड विकासासाठी विविध क्षमता आणि साधने शोधण्याच्या शिकण्याच्या सँडबॉक्स म्हणून विचार करा. या कार्यशाळेच्या शेवटी, आपल्याला या संदर्भातील विविध साधने आणि संकल्पनांची समज सहजपणे येईल.

| संकल्पना  | उद्दिष्ट |
|:---|:---|
| **Azure Developer CLI** | साधन आदेश आणि कार्यप्रवाह समजून घ्या|
| **AZD टेम्पलेट्स**| प्रकल्पाची रचना आणि कॉन्फिगरेशन समजून घ्या|
| **Azure AI एजंट**| Microsoft Foundry प्रकल्प प्राव्हिजन व तैनात करा  |
| **Azure AI शोध**| एजंटसह संदर्भ अभियांत्रिकी सक्षम करा |
| **निरीक्षणक्षमता**| ट्रेसिंग, निरीक्षण आणि मूल्यांकनांचा अभ्यास करा |
| **रेड टीमिंग**| विरोधी चाचणी व प्रतिबंधांचा अभ्यास करा |
| | |
