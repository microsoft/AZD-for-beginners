# AZD AI विकासकांसाठी

!!! info "या कार्यशाळेचा प्रेक्षक वर्ग"
   
    ही कार्यशाळा कोणत्याही व्यक्तीसाठी आहे जे Microsoft Foundry वर उत्कृष्ट साधने आणि कार्यप्रवाहांचा उपयोग करून AI एजंटिक अनुप्रयोग तयार करू इच्छितात. ही कार्यशाळा सुरुवातीपासून ते अनुभवी विकासकांपर्यंत सर्वांसाठी उपयुक्त आहे ज्यांना Azure, AI संकल्पना आणि Python कोडची काही माहिती आहे. कृपया प्रारंभ करण्यापूर्वी पूर्वअट तपासा.

    **कालावधी:** ३-४ तास  
    **स्तर:** सुरुवातीपासून मध्यवर्ती

## पूर्वअटी

- AI संकल्पना, Python विकास, कमांड-लाइन साधनांची ओळख
- VS Code आणि GitHub साधने व कार्यप्रवाहांचा अनुभव.

## शिकण्याची उद्दिष्टे

या कार्यशाळेच्या शेवटी, तुम्हाला कसे करायचे हे माहिती असेल:

- [X] **Azure Developer CLI सह कार्य करा** → तुमची Azure संसाधने तयार करा, तैनात करा, आणि व्यवस्थापित करा.
- [X] **AI अ‍ॅप टेंप्लेट वापरा** → Microsoft Foundry वर AI एजंट प्रकल्प पटकन सुरू करण्यासाठी.
- [X] **टेंप्लेट कॉन्फिगर करा** → Red Teaming, मूल्यांकन, ट्रेसिंग आणि मॉनिटरिंग सारख्या वैशिष्ट्यांसाठी.
- [X] **AZD टेंप्लेट आणि कार्यप्रवाह GitHub Copilot चा वापर करून विघटित व शोधा**
- [X] **तुमच्या आवश्यकतानुसार AI एजंट स्टार्टर टेंप्लेट सानुकूलित करा आणि पुन्हा तैनात करा**

## कार्यशाळा आराखडा

कार्यशाळा विभागांमध्ये आयोजित केली आहे, प्रत्येक विभागात एक हाताने करण्यायोग्य प्रयोग असतो:

1. **परिचय** - AZD टेंप्लेट काय आहेत आणि ते कसे काम करतात याची समज मिळवा
1. **निवड** - तुमच्या पुढील AI प्रकल्पासाठी योग्य "स्टार्टर" टेंप्लेट शोधा
1. **वापराचे परीक्षण** - टेंप्लेट तैनात करा आणि तपासा की ते अपेक्षेप्रमाणे कार्य करते का
1. **विघटन** - टेंप्लेटचे घटक आणि कॉन्फिगरेशन समजून घेण्यासाठी ते विभाजित करा
1. **कॉन्फिगरेशन** - Microsoft Foundry च्या विविध वैशिष्ट्यांना सक्रिय करायला आणि वापरण्यास शिका
1. **सानुकूलन** - तुमचा डेटा आणि आवश्यकतांचा विचार करून अनुप्रयोग पुन्हा तयार करा
1. **संबंधित संसाधने हटवणे** - इन्फ्रास्ट्रक्चर डी-प्रोव्हिजन करण्याचे, आणि सॉफ्ट-डिलीटेड संसाधने पुनर्प्राप्त करण्याचे शिका
