## 2. मुख्य संकल्पना अवगाही

### Azure Developer CLI (azd)
- `azd init` - टेम्पलेट्स मधून प्रोजेक्ट सुरू करा
- `azd up` - एकाच आज्ञेत प्राव्हिजन आणि तैनात करा
- `azd env` - पर्यावरणीय चल व्यवस्थापित करा
- `azd down` - संसाधने सुरक्षितपणे स्वच्छ करा

### AZD टेम्पलेट संरचना
- `azure.yaml` - कॉन्फिगरेशन आणि तैनाती हुक्स
- `infra/` - Bicep इन्फ्रास्ट्रक्चर-अस-कोड
- `src/` - अनुप्रयोग स्रोत कोड
- `scripts/` - स्वयंचलिती आणि सेटअप स्क्रिप्ट्स

### Microsoft Foundry क्षमता
- बुद्धिमान सहाय्यकांसाठी AI एजंट सेवा
- ज्ञान पुनर्प्राप्तीसाठी फाइल सर्च आणि Azure AI सर्च
- अंगभूत ट्रेसिंग, निरीक्षण आणि मूल्यांकन
- AI सुरक्षा प्रमाणीकरणासाठी रेड टीमिंग

## 4. कार्यशाळेचा आव्हान

आपल्या कौशल्यांची परीक्षा देण्यासाठी तयार आहात का? या आव्हानांचा प्रयत्न करा:

| आव्हान | वर्णन | अवघडपणा |
|:---|:---|:---|
| **1. मल्टी-एजंट परिस्थिती** | वेगळ्या उद्योगासाठी (उदा., आरोग्यसेवा, वित्त) तैनात करा आणि सानुकूलित करा | ⭐⭐ |
| **2. सानुकूल ज्ञान आधार** | तुमची स्वतःची डेटा फाइल्स जोडा आणि सानुकूल शोध निर्देशांक तयार करा | ⭐⭐⭐ |
| **3. उत्पादन तैनाती** | `azd pipeline config` वापरून GitHub Actions सह CI/CD सेट करा | ⭐⭐⭐⭐ |
