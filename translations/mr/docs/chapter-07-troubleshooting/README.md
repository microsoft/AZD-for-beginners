# अध्याय 7: समस्या निवारण आणि डीबगिंग

**📚 कोर्स**: [AZD For Beginners](../../README.md) | **⏱️ कालावधी**: 1-1.5 तास | **⭐ कठीणपणा**: मध्यम

---

## आढावा

हा अध्याय Azure Developer CLI वापरताना उद्भवणाऱ्या सामान्य समस्या निदान आणि निराकरण करण्यात तुम्हाला मदत करतो. डिप्लॉयमेंट अयशस्वित्वांपासून AI-विशिष्ट समस्या पर्यंत.

## शिकण्याची उद्दिष्टे

या अध्यायाचे पूर्ण केल्यावर तुम्ही:
- सामान्य AZD डिप्लॉयमेंट अयशस्वितांचे निदान करू शकता
- प्रमाणीकरण आणि परवानगीच्या समस्या डीबग करू शकता
- AI सेवा कनेक्टिव्हिटी समस्यांचे निराकरण करू शकता
- समस्या निवारणासाठी Azure पोर्टल आणि CLI वापरू शकता

## 🚨 त्वरित उपाय

### प्रमाणीकरण समस्या
```bash
azd auth login
az login
azd auth whoami
```

### प्रोव्हिजनिंग अयशस्वी
```bash
azd show
azd monitor --logs
az deployment sub list --query "[?properties.provisioningState!='Succeeded']"
```

### संसाधन संघर्ष
```bash
azd down --force --purge
azd env new different-name
azd up
```

### कोटा ओलांडला
```bash
az vm list-usage --location eastus --output table
azd env set AZURE_LOCATION westus2
azd up
```

## 📋 त्रुटी कोड संदर्भ

| त्रुटी | कारण | उपाय |
|-------|-------|----------|
| `AuthenticationError` | लॉग इन केलेले नाही | `azd auth login` |
| `ResourceNotFound` | संसाधन सापडले नाही | संसाधन नावे तपासा |
| `QuotaExceeded` | सदस्यता मर्यादा | कोटा वाढीसाठी विनंती करा |
| `InvalidTemplate` | Bicep सिंटॅक्स त्रुटी | `az bicep build` |
| `Conflict` | संसाधन अस्तित्वात आहे | नवीन नाव वापरा किंवा हटवा |
| `Forbidden` | पर्याप्त परवानग्या नाहीत | RBAC भूमिका तपासा |

## 🔄 रीसेट आणि पुनर्प्राप्ती

```bash
# सॉफ्ट रीसेट (संसाधने ठेवा, कोड पुन्हा तैनात करा)
azd deploy --force

# हार्ड रीसेट (सर्व काही हटवा, नव्याने सुरू करा)
azd down --force --purge
azd up
```
