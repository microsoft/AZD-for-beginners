# Chapter 5: बहु-एजंट एआय समाधान

**📚 Course**: [AZD प्रारंभिकांसाठी](../../README.md) | **⏱️ Duration**: 2-3 तास | **⭐ Complexity**: उन्नत

---

## आढावा

हा अध्याय प्रगत बहु-एजंट आर्किटेक्चर पॅटर्न, एजंट समन्वय आणि क्लिष्ट प्रकरणांसाठी उत्पादन-तयार एआय तैनातींचे आच्छादन करतो.

## शिकण्याची उद्दिष्टे

या अध्यायाचे पूर्ण केल्यावर, आपण:
- बहु-एजंट आर्किटेक्चर पॅटर्न समजून घेणार
- समन्वयित एआय एजंट प्रणाली तैनात करणार
- एजंट-टू-एजंट संवाद अंमलात आणणार
- उत्पादन-तयार बहु-एजंट उपाय तयार करणार

## 🎯 वैशिष्ट्यीकृत समाधान: रिटेल बहु-एजंट

The [रिटेल बहु-एजंट समाधान](../../examples/retail-scenario.md) दाखवते:

- **ग्राहक एजंट**: वापरकर्त्याशी संवाद आणि प्राधान्ये हाताळतो
- **इन्व्हेंटरी एजंट**: स्टॉक आणि ऑर्डर प्रक्रिया व्यवस्थापित करतो
- **ऑर्केस्ट्रेटर**: एजंट्समधील समन्वय साधतो
- **सामायिक स्मृती**: एजंट्समधील संदर्भ व्यवस्थापन

### वापरलेल्या सेवा

| सेवा | उद्देश |
|---------|---------|
| Azure OpenAI | भाषा समज |
| Azure AI Search | उत्पादन निर्देशिका |
| Cosmos DB | एजंट स्थिती आणि स्मृती |
| Container Apps | एजंट होस्टिंग |
| Application Insights | निगराणी |
