# अध्याय 1: पायाभूत सुविधा आणि त्वरित प्रारंभ

**📚 कोर्स**: [AZD For Beginners](../../README.md) | **⏱️ कालावधी**: 30-45 मिनिटे | **⭐ गुंतागुंत**: प्रारंभिक

---

## आढावा

हा अध्याय Azure Developer CLI (azd) मूलभूत गोष्टी परिचित करुन देतो. तुम्ही मुख्य संकल्पना शिकाल, साधने इंस्टॉल कराल, आणि तुमचे पहिले एप्लिकेशन Azure वर तैनात कराल.

> `azd 1.27.1` च्या तुलनेत जुलै 2026 मध्ये प्रमाणीकरण केले गेले.

## शिक्षण उद्दिष्टे

हा अध्याय पूर्ण करून, तुम्ही:
- Azure Developer CLI काय आहे आणि तो Azure CLI पेक्षा कसा वेगळा आहे हे समजून घ्या
- तुमच्या प्लॅटफॉर्मवर AZD इंस्टॉल आणि कॉन्फिगर करा
- `azd up` वापरून तुमचे पहिले एप्लिकेशन Azure वर तैनात करा
- `azd down` वापरून संसाधने साफ करा

## ✅ येथे प्रारंभ करा: तुमची सेटअप पडताळणी करा

सुरू करण्यापूर्वी, तुमची स्थानिक संगणक अध्याय 1 टेम्प्लेटसाठी तयार आहे का ते सुनिश्चित करा:

**विंडोज:**
```powershell
.\validate-setup.ps1
```

**macOS / लिनक्स:**
```bash
bash ./validate-setup.sh
```

जर स्क्रिप्टमुळे साधने गायब असल्याचा अहवाल आला, तर आधी त्या सुधारा आणि नंतर अध्यायासह पुढे जा.

---

## 🚀 त्वरित प्रारंभ

```bash
# स्थापना तपासा
azd version

# AZD साठी प्रमाणीकरण करा
# ऐच्छिक: Azure CLI कमांड थेट चालवण्याची योजना असल्यास az login करा
azd auth login

# आपला पहिला अॅप डिप्लॉय करा
azd init --template todo-nodejs-mongo
azd up

# पूर्ण झाल्यावर साफसफाई करा
azd down --force --purge
```

---

## ✅ यशस्वी निकष

हा अध्याय पूर्ण केल्यानंतर, तुम्ही हे करू शकता:

```bash
azd version              # स्थापित आवृत्ती दर्शविते
azd init --template todo-nodejs-mongo  # प्रकल्प प्रारंभ करते
azd up                   # Azure वर नियुक्त करते
azd show                 # चालत असलेल्या अ‍ॅपचा URL दर्शविते
azd down --force --purge # संसाधने स्वच्छ करते
```
