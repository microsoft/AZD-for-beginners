# अध्याय 1: पायाभूत आणि जलद प्रारंभ

**📚 कोर्स**: [AZD नवशिक्यांसाठी](../../README.md) | **⏱️ कालावधी**: 30-45 मिनिटे | **⭐ क्लिष्टता**: प्राथमिक

---

## विहंगावलोकन

हा अध्याय Azure Developer CLI (azd) चे मूलभूत तत्त्वे परिचित करून देतो. आपण मुख्य संकल्पना शिकाल, टूल्स इन्स्टॉल आणि कॉन्फिगर कराल, आणि आपले पहिले अनुप्रयोग Azure वर तैनात कराल.

> मार्च 2026 मध्ये `azd 1.23.12` सोबत पडताळले.

## शिकण्याचे उद्दिष्टे

हा अध्याय पूर्ण केल्यानंतर, आपण:
- Azure Developer CLI काय आहे आणि तो Azure CLI पासून कसा वेगळा आहे हे समजून घ्या
- आपल्या प्लॅटफॉर्मवर AZD इन्स्टॉल आणि कॉन्फिगर करा
- `azd up` वापरून आपले पहिले आवेदन Azure वर तैनात करा
- `azd down` वापरून संसाधने हटवा

## ✅ येथे प्रारंभ करा: आपले सेटअप पडताळा

सुरू करण्यापूर्वी, आपल्या स्थानिक मशीनवर अध्याय 1 टेम्पलेटसाठी तयारी आहे का हे पुष्टी करा:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

If the script reports missing tools, fix those first and then continue with the chapter.

---

## 🚀 जलद प्रारंभ

```bash
# स्थापना तपासा
azd version

# AZD साठी प्रमाणीकरण करा
# वैकल्पिक: जर आपण Azure CLI आदेश थेट चालवण्याची योजना आखत असाल तर az login करा
azd auth login

# आपले पहिले अॅप तैनात करा
azd init --template todo-nodejs-mongo
azd up

# काम संपल्यावर साफ करा
azd down --force --purge
```

---

## ✅ यशाची निकष

हा अध्याय पूर्ण केल्यावर, आपण सक्षम असाल:

```bash
azd version              # स्थापित आवृत्ती दाखवते
azd init --template todo-nodejs-mongo  # प्रकल्प प्रारंभ करते
azd up                   # Azure वर तैनात करते
azd show                 # चालू अॅपचा URL दाखवते
azd down --force --purge # संसाधने साफ करते
```
