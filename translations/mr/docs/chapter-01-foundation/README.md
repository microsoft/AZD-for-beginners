# अध्याय 1: पाया आणि जलद प्रारंभ

**📚 कोर्स**: [AZD प्रारंभिकांसाठी](../../README.md) | **⏱️ कालावधी**: 30-45 मिनिटे | **⭐ जटिलता**: प्रारंभिक

---

## आढावा

हा अध्याय Azure Developer CLI (azd) मुलतत्त्वे परिचय करतो. तुम्हाला मुख्य संकल्पना समजून घेता येतील, साधने स्थापित करू शकता, आणि तुमचे पहिले अनुप्रयोग Azure वर तैनात करू शकाल.

> `azd 1.25.6` जून 2026 मध्ये मान्यताप्राप्त.

## शिक्षण उद्दिष्टे

हा अध्याय पूर्ण केल्यानंतर, तुम्ही:
- Azure Developer CLI काय आहे आणि ते Azure CLI पासून कसे वेगळे आहे हे समजून घेणार आहात
- तुमच्या प्लॅटफॉर्मवर AZD कसे स्थापित आणि कॉन्फिगर करायचे ते शिकाल
- `azd up` वापरून तुमचे पहिले अनुप्रयोग Azure वर तैनात कराल
- `azd down` वापरून साधने साफ केली जातील

## ✅ येथे प्रारंभ करा: तुमचे सेटअप पडताळा

सुरू करण्यापूर्वी, तुमची स्थानिक मशीन अध्याय 1 साचा तयार आहे का ते खात्री करा:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

जर स्क्रिप्ट साधने हरवलेली दर्शवित असेल, तर ती प्रथम दुरुस्त करा आणि नंतर अध्याय सुरू करा.

---

## 🚀 जलद प्रारंभ

```bash
# इंस्टॉलेशन तपासा
azd version

# AZD साठी प्रमाणीकरण करा
# पर्यायी: जर तुम्ही Azure CLI कमांड थेट चालवण्याचा विचार करत असाल तर az login करा
azd auth login

# तुमचे पहिले अॅप तैनात करा
azd init --template todo-nodejs-mongo
azd up

# पूर्ण झाल्यावर स्वच्छ करा
azd down --force --purge
```

---

## ✅ यशाचे निकष

हा अध्याय पूर्ण केल्यानंतर, तुम्हाला खालील गोष्टी करता याव्यात:

```bash
azd version              # स्थापित केलेली आवृत्ती दर्शविते
azd init --template todo-nodejs-mongo  # प्रकल्प प्रारंभ करते
azd up                   # Azure वर तैनात करते
azd show                 # चालू असलेल्या अॅपचा URL दर्शविते
azd down --force --purge # संसाधने साफ करते
```
