# अध्याय 1: पायाभूत आणि जलद प्रारंभ

**📚 अभ्यासक्रम**: [AZD नवशिक्यांसाठी](../../README.md) | **⏱️ कालावधी**: 30-45 मिनिटे | **⭐ कठीणपणा**: प्रारंभिक

---

## आढावा

हा अध्याय Azure Developer CLI (azd) च्या पायाभूत तत्त्वांची ओळख करुन देतो. आपण मूलभूत संकल्पना शिकणार आहात, साधने स्थापित कराल, आणि आपले पहिले अनुप्रयोग Azure वर डिप्लॉय कराल.

## शिकण्याची उद्दिष्टे

हा अध्याय पूर्ण केल्यावर आपण:
- Azure Developer CLI काय आहे आणि ते Azure CLI पासून कसे वेगळे आहे हे समजून घ्या
- आपल्या प्लॅटफॉर्मवर AZD इन्स्टॉल आणि कॉन्फिगर करा
- `azd up` वापरून आपले पहिले अनुप्रयोग Azure वर डिप्लॉय करा
- `azd down` वापरून संसाधने स्वच्छ करा

## 🚀 जलद प्रारंभ

```bash
# स्थापना तपासा
azd version

# Azure मध्ये लॉगिन करा
azd auth login

# आपले पहिले अॅप तैनात करा
azd init --template todo-nodejs-mongo
azd up

# काम पूर्ण झाल्यावर साफ करा
azd down --force --purge
```

---

## ✅ यश निकष

हा अध्याय पूर्ण केल्यावर, आपण हे करण्यास सक्षम असले पाहिजे:

```bash
azd version              # स्थापित आवृत्ती दाखवते
azd init --template todo-nodejs-mongo  # प्रकल्प प्रारंभ करतो
azd up                   # Azure वर तैनात करते
azd show                 # चालू अॅपचा URL दाखवते
azd down --force --purge # संसाधने साफ करते
```
