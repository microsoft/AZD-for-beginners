# Chapter 3: कॉन्फिगरेशन आणि प्रमाणीकरण

**📚 कोर्स**: [AZD For Beginners](../../README.md) | **⏱️ कालावधी**: 45-60 मिनिटे | **⭐ क्लिष्टता**: मध्यवर्ती

---

## आढावा

हा प्रकरण पर्यावरण कॉन्फिगरेशन, प्रमाणीकरण नमुने आणि Azure Developer CLI डिप्लॉयमेंटसाठी सुरक्षा सर्वोत्तम सराव यावर आधारित आहे.

> जून 2026 मध्ये `azd 1.25.6` विरुद्ध तपासलेले.

## शिक्षण उद्दिष्टे

हा प्रकरण पूर्ण केल्यावर, तुम्ही:
- AZD कॉन्फिगरेशन हायार्कीमध्ये प्रभुत्व मिळवाल
- एकाधिक पर्यावरणे (डेव्हलपमेंट, स्टेजिंग, प्रॉडक्शन) व्यवस्थापित कराल
- व्यवस्थापित ओळखींसह सुरक्षित प्रमाणीकरण अंमलात आणाल
- पर्यावरण-विशिष्ट सेटिंग्ज कॉन्फिगर कराल

## 🚀 जलद प्रारंभ

```bash
# अनेक वातावरण तयार करा
azd env new dev
azd env new staging
azd env new prod

# वातावरणे बदला
azd env select prod

# वातावरण बदल्याचे सेट करा
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# संरचना पहा
azd env get-values
```

---

## 🔧 कॉन्फिगरेशन हायार्की

AZD या क्रमाने सेटिंग्ज लागू करतो (नंतरचे आधीचे अधिलेखन करते):

1. **पूर्वनिर्धारित मूल्ये** (टेम्पलेटमध्ये अंगभूत)
2. **azure.yaml** (प्रकल्प कॉन्फिगरेशन)
3. **पर्यावरण चल** (`azd env set`)
4. **कमान्ड-लाइन फ्लॅग्ज** (`--location eastus`)

## 🔐 सुरक्षा सर्वोत्तम सराव

```bash
# व्यवस्थापित ओळख वापरा (शिफारस केली आहे)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD प्रमाणीकरण स्थिती तपासा
azd auth status

# ऐच्छिक: az कमांड चालविण्याचा विचार करीत असल्यास Azure CLI संदर्भ सत्यापित करा
az account show

# आवश्यक असल्यास पुन्हा प्रमाणीकरण करा
azd auth login

# ऐच्छिक: az कमांडसाठी Azure CLI प्रमाणीकरण पुन्हा ताजेतवाने करा
az login
```
