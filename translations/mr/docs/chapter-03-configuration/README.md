# अध्याय 3: कॉन्फिगरेशन आणि प्रमाणीकरण

**📚 कोर्स**: [AZD फॉर बिगिनर्स](../../README.md) | **⏱️ कालावधी**: ४५-६० मिनिटे | **⭐ गुंतागुंत**: मध्यम

---

## आढावा

हा अध्याय Azure Developer CLI च्या तैनातींसाठी वातावरण कॉन्फिगरेशन, प्रमाणीकरणाचे नमुने आणि सुरक्षा सर्वोत्तम प्रथांशी संबंधित आहे.

> जुलै २०२६ मध्ये `azd 1.27.1` विरुद्ध पडताळणी केली आहे.

## अध्ययन उद्दिष्टे

हा अध्याय पूर्ण केल्यावर, आपण:
- AZD कॉन्फिगरेशन पदानुक्रम mastery करा
- एकाधिक वातावरणे (dev, staging, prod) व्यवस्थापित करा
- व्यवस्थापित ओळखांसह सुरक्षित प्रमाणीकरण प्रतिपादित करा
- वातावरण-विशिष्ट सेटिंग्ज कॉन्फिगर करा

## 🚀 जलद प्रारंभ

```bash
# अनेक वातावरण तयार करा
azd env new dev
azd env new staging
azd env new prod

# वातावरण बदलून टाका
azd env select prod

# पर्यावरणीय चल सेट करा
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# संरचना पहा
azd env get-values
```

---

## 🔧 कॉन्फिगरेशन पदानुक्रम

AZD खालील अनुक्रमे सेटिंग्ज लागू करते (नंतरचे आधीचे ओव्हरराईड करते):

१. **डीफॉल्ट मूल्ये** (टेम्पलेट्स मध्ये अंगभूत)
२. **azure.yaml** (प्रकल्प कॉन्फिगरेशन)
३. **पर्यावरण चल (Environment variables)** (`azd env set`)
४. **कमांड-लाइन फ्लॅग्ज** (`--location eastus`)

---

## 🔐 सुरक्षा सर्वोत्तम प्रथा

```bash
# व्यवस्थापित ओळख वापरा (शिफारस केलेली)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD प्रमाणीकरण स्थिती तपासा
azd auth status

# पर्यायी: az आदेश चालविण्याची योजना असल्यास Azure CLI संदर्भ सत्यापित करा
az account show

# आवश्यक असल्यास पुन्हा प्रमाणीकरण करा
azd auth login

# पर्यायी: az आदेशांसाठी Azure CLI प्रमाणीकरण ताजे करा
az login
```
