# अध्याय ६: प्री-डिप्लॉयमेंट नियोजन आणि प्रमाणीकरण

**📚 कोर्स**: [AZD For Beginners](../../README.md) | **⏱️ कालावधी**: 1 तास | **⭐ गुंतागुंत**: मध्यम

---

## आराखडा

हा अध्याय तुमच्या अर्जाच्या डिप्लॉयमेंटपूर्वी आवश्यक नियोजन आणि प्रमाणीकरण टप्पे या विषयी आहे. योग्य क्षमता नियोजन, SKU निवड, आणि प्रीफ्लाइट तपासणीसह महागड्या चुका टाळणे शिका.

> `azd 1.27.1` ला जुलै 2026 मध्ये प्रमाणीकरण केलेले.

## शिकण्याचे उद्दिष्टे

या अध्याय पूर्ण केल्यावर तुम्ही:
- डिप्लॉयमेंटपूर्वी प्रीफ्लाइट तपासण्या चालवू शकाल
- क्षमता नियोजन कराल आणि संसाधन गरजा अंदाज कराल
- खर्च कमी करण्यासाठी योग्य SKU निवडाल
- Application Insights चे निरीक्षणासाठी सानुकूलन कराल
- टीम समन्वयाचे नमुने समजून घ्याल

## 🚀 जलद प्रारंभ

```bash
# सदस्यता कोटा तपासा
az vm list-usage --location eastus --output table

# तैनाती पूर्वावलोकन (कोणतेही संसाधने तयार केलेले नाहीत)
azd provision --preview

# Bicep सिंटॅक्स मान्य करा
az bicep build --file infra/main.bicep

# वातावरण कॉन्फिगरेशन तपासा
azd env get-values
```

## ☑️ प्री-डिप्लॉयमेंट तपासणी यादी

### `azd provision` आधी

- [ ] प्रदेशासाठी कोटा तपासलेला आहे
- [ ] SKU योग्यरित्या निवडलेले आहे
- [ ] खर्चाचा अंदाज पाहिला आहे
- [ ] नावकरण पद्धत सुसंगत आहे
- [ ] सुरक्षा/RBAC सानुकूलित केलेले आहे

### `azd deploy` आधी

- [ ] पर्यावरण व्हेरिएबल्स सेट केलेले आहेत
- [ ] की व्हॉल्टमधील रहस्ये आहेत
- [ ] कनेक्शन स्ट्रिंग्ज तपासलेले आहेत
- [ ] हेल्थ तपासण्या सानुकूलित केल्या आहेत
