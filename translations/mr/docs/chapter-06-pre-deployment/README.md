# अध्याय 6: पूर्व-आव्हान योजना आणि प्रमाणीकरण

**📚 कोर्स**: [AZD For Beginners](../../README.md) | **⏱️ कालावधी**: 1 तास | **⭐ अवघडपण**: मध्यम

## अवलोकन

हा अध्याय तुमच्या अनुप्रयोगाच्या तैनातीपूर्वी आवश्यक योजना आणि प्रमाणीकरण पावले समाविष्ट करतो. योग्य क्षमतानियोजन, SKU निवड आणि पूर्वचाचण्या करून खर्चिक चुका टाळायला शिका.

## शिक्षण उद्दिष्टे

हा अध्याय पूर्ण केल्यावर, तुम्ही:
- तैनातीपूर्वी पूर्वचाचण्या चालवाल
- क्षमता नियोजन कराल आणि संसाधन गरजा अंदाजित कराल
- खर्च कमी करण्यासाठी योग्य SKU निवडाल
- देखरेखीकरिता Application Insights कॉन्फिगर कराल
- टीम समन्वय नमुने समजून घ्याल

## 🚀 जलद प्रारंभ

```bash
# सदस्यता कोटा तपासा
az vm list-usage --location eastus --output table

# तैनातीचे पूर्वावलोकन करा (कोणतीही संसाधने तयार होत नाहीत)
azd provision --preview

# Bicep सिंटॅक्सची सत्यता तपासा
az bicep build --file infra/main.bicep

# पर्यावरण कॉन्फिगरेशन तपासा
azd env get-values
```

## ☑️ पूर्व-तैनाती तपासणी यादी

### `azd provision` करण्यापूर्वी

- [ ] प्रदेशासाठी कोटा तपासली आहे
- [ ] SKU योग्य प्रकारे निवडले आहे
- [ ] खर्चाचा अंदाज तपासला आहे
- [ ] नामनिर्देशन पद्धत सुसंगत आहे
- [ ] सुरक्षा/RBAC कॉन्फिगर केले आहे

### `azd deploy` करण्यापूर्वी

- [ ] पर्यावरणीय चल (environment variables) सेट केले आहेत
- [ ] रहस्ये Key Vault मध्ये आहेत
- [ ] कनेक्शन स्ट्रिंग्ज तपासली आहेत
- [ ] आरोग्य तपासण्या कॉन्फिगर केले आहेत
