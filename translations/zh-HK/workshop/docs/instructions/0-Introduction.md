# AZD 給 AI 開發者

!!! info "本工作坊的對象"
   
    本工作坊適合任何想利用頂尖工具和工作流程，在 Microsoft Foundry 上構建 AI 代理應用程式的人士。無論是初學者還是對 Azure、AI 概念及 Python 程式碼有一定認識的經驗開發者，都適合參加。請先查看先決條件。

    **時長：** 3-4 小時  
    **水平：** 初學至中級  

---


## 先決條件

- 一個個人 GitHub 帳戶 → [免費建立](https://github.com/signup)
- 一個有效的 Azure 帳戶 → [免費建立](https://aka.ms/free)
- 熟悉 AI 概念、Python 開發、命令行工具
- 有使用 VS Code 和 GitHub 工具與工作流程的經驗。

---

## 學習目標

完成本工作坊後，您將能：

- [X] **使用 Azure 開發者 CLI** → 來佈署、管理您的 Azure 資源。
- [X] **使用 AI 應用程式範本** → 快速啟動 Microsoft Foundry 上的 AI 代理專案。
- [X] **設定範本** → 開啟紅隊作業、評估、追蹤及監控等功能
- [X] **拆解及探索** → 使用 GitHub Copilot 解讀 AZD 範本與工作流程
- [X] **自訂及重新部署** → AI 代理起始範本，符合您的自訂需求

---

## 工作坊大綱

本工作坊以模組結構呈現，每個模組均有實作練習：

1. **介紹** - 了解什麼是 AZD 範本，以及其運作方式
1. **範本選擇** - 發掘適合您下一個 AI 專案的「起步」範本
1. **驗證** - 部署並使用範本，確認其功能與宣稱一致
1. **拆解** - 拆解範本，理解其組件與設定
1. **設定** - 學習啟用及使用 Microsoft Foundry 各項功能
1. **自訂** - 按照您的資料與需求，重新建構應用程式
1. **清理** - 學習解除基礎架構佈署及回復軟刪除資源
