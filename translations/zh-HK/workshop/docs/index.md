## 工作坊概覽

**時長：** 3-4 小時  
**程度：** 初學者至中階  
**先備知識：** 熟悉 Azure、AI 概念、VS Code 與指令列工具。

這是個動手實作的工作坊，透過實際操作學習。完成練習後，我們建議您回顧 AZD For Beginners 課程，持續深入學習安全性與生產力最佳實務。

| 時間| 模組  | 目標 |
|:---|:---|:---|
| 15 分鐘 | 介紹 | 了解背景與目標 |
| 30 分鐘 | 選擇 AI 範本 | 探索選項並挑選起始範本 | 
| 30 分鐘 | 驗證 AI 範本 | 將預設解決方案部署到 Azure |
| 30 分鐘 | 拆解 AI 範本 | 探索結構與設定 |
| 30 分鐘 | 配置 AI 範本 | 啟用並試用可用功能 |
| 30 分鐘 | 自訂 AI 範本 | 根據需求調整範本 |
| 30 分鐘 | 拆除基礎設施 | 清理並釋放資源 |
| 15 分鐘 | 總結與下一步 | 學習資源、工作坊挑戰 |

## 您將學到什麼

將 AZD 範本想像成一個學習沙盒，可探索 Microsoft Foundry 端對端開發的各項能力與工具。在本工作坊結束時，您應該對此情境下的各種工具與概念有直覺性的理解。

| 概念  | 目標 |
|:---|:---|
| **Azure Developer CLI** | 理解工具指令與工作流程|
| **AZD 範本**| 理解專案結構與設定|
| **Azure AI Agent**| 預備及部署 Microsoft Foundry 專案  |
| **Azure AI Search**| 以代理啟用情境工程 |
| **可觀察性**| 探索追蹤、監控與評估 |
| **紅隊測試**| 探索對抗測試與緩解措施 |
