## परिचय

यस व्यापक शब्दावलीले Azure Developer CLI र Azure क्लाउड विकासमा प्रयोग गरिने शब्दहरू, अवधारणाहरू, र संक्षेपाक्षरहरूको परिभाषा प्रदान गर्छ। प्राविधिक दस्तावेजहरू बुझ्न, समस्या समाधान गर्न, र azd परियोजना तथा Azure सेवाहरूका बारेमा प्रभावकारी रूपमा सञ्चार गर्नका लागि आवश्यक सन्दर्भ हो।

## सिकाइका लक्ष्यहरू

यस शब्दावली प्रयोग गरेर, तपाईंले:
- Azure Developer CLI का आवश्यक शब्दावली र अवधारणाहरू बुझ्नुहुनेछ
- Azure क्लाउड विकास सम्बन्धी शब्दावली र प्राविधिक शर्तहरूमा निपूर्ण हुन सक्नुहुनेछ
- Infrastructure as Code र तैनाथीकरण सम्बन्धी शब्दावली प्रभावकारी रूपमा सन्दर्भ गर्न सक्नुहुनेछ
- Azure सेवा नामहरू, संक्षेपाक्षरहरू, र तिनीहरूको उद्देश्यहरू बुझ्न सक्नुहुनेछ
- समस्याहरू समाधान र डिबगिङ सम्बन्धी शब्दहरूको परिभाषाहरू पहुँच गर्न सक्नुहुनेछ
- उन्नत Azure वास्तुकला र विकास अवधारणाहरू सिक्न सक्नुहुनेछ

## सिकाइका नतिजाहरू

नियमित रूपमा यस शब्दावलۍलाई सन्दर्भ गर्दा, तपाईंले सक्षम हुनुहुनेछ:
- उपयुक्त Azure Developer CLI शब्दावली प्रयोग गरी प्रभावकारी रूपमा सञ्चार गर्न
- प्राविधिक दस्तावेज र त्रुटि सन्देशहरू स्पष्ट रूपमा बुझ्न
- Azure सेवाहरू र अवधारणाहरू आत्मविश्वासका साथ नेभिगेट गर्न
- उपयुक्त प्राविधिक शब्दावली प्रयोग गरी समस्याहरू समाधान गर्न
- टोली छलफलहरूमा सही प्राविधिक भाषामा योगदान पुर्‍याउन
- व्यवस्थित रूपमा आफ्नो Azure क्लाउड विकास ज्ञान विस्तार गर्न

## A

**ARM Template**  
ARM टेम्पलेट। JSON-आधारित Infrastructure as Code ढाँचा जुन Azure स्रोतहरूलाई घोषणात्मक रूपमा परिभाषित र तैनाथ गर्न प्रयोग गरिन्छ।

**App Service**  
वेब एप्लिकेसनहरू, REST API हरू, र मोबाइल ब्याकएन्डहरू बिना पूर्वाधार व्यवस्थापन गर्दा होस्ट गर्नको लागि Azure को platform-as-a-service (PaaS) सेवा।

**Application Insights**  
एप्लिकेसन प्रदर्शन, उपलब्धता, र प्रयोगमा गहिरो अन्तर्दृष्टि प्रदान गर्ने Azure को application performance monitoring (APM) सेवा।

**Azure CLI**  
Azure स्रोतहरू व्यवस्थापन गर्नको लागि कमाण्ड-लाइन इन्टरफेस। प्रमाणिकरण र केही अपरेशन्सका लागि azd द्वारा प्रयोग गरिन्छ।

**Azure Developer CLI (azd)**  
डेभलपर-केन्द्रित कमाण्ड-लाइन उपकरण जसले टेम्पलेट र Infrastructure as Code प्रयोग गरी Azure मा एप्लिकेसन निर्माण र तैनाथीकरणको प्रक्रियालाई तेज बनाउँछ।

**azure.yaml**  
azd परियोजनाको मुख्य कन्फिगरेसन फाइल जसले सर्भिसहरू, पूर्वाधार, र तैनाथीकरण हुकहरू परिभाषित गर्छ।

**Azure Resource Manager (ARM)**  
स्रोतहरू सिर्जना, अपडेट, र मेटाउनको लागि व्यवस्थापन तह प्रदान गर्ने Azure को तैनाथीकरण र व्यवस्थापन सेवा।

## B

**Bicep**  
Azure स्रोतहरू तैनाथ गर्न Microsoft द्वारा विकास गरिएको domain-specific language (DSL)। ARM टेम्पलेटहरू भन्दा सधैं-readable सिन्ट्याक्स प्रदान गर्छ र ARM मा कम्पाइल हुन्छ।

**Build**  
स्रोत कोड कम्पाइल गर्ने, निर्भरता इन्स्टल गर्ने, र एप्लिकेसनहरू तैनाथका लागि तयार पार्ने प्रक्रिया।

**Blue-Green Deployment**  
दुई एकसमान उत्पादन वातावरण (blue र green) प्रयोग गर्ने तैनाथीकरण रणनीति जसले डाउनटाइम र जोखिम घटाउँछ।

## C

**Container Apps**  
जटिल पूर्वाधार व्यवस्थापन नगरी कन्टेनराइज्ड एप्लिकेसनहरू चलाउन सक्षम गराउने Azure को serverless container सेवा।

**CI/CD**  
Continuous Integration/Continuous Deployment। कोड परिवर्तनहरू समावेश गर्ने र एप्लिकेसन तैनाथ गर्ने स्वचालित अभ्यासहरू।

**Cosmos DB**  
थ्रुपुट, लेटेंसी, उपलब्धता, र निरन्तरताका लागि व्यापक SLA हरू प्रदान गर्ने Azure को ग्लोबली वितरण गरिएको, मल्टि-मोडल डेटाबेस सेवा।

**Configuration**  
एप्लिकेसन व्यवहार र तैनाथीकरण विकल्पहरू नियन्त्रण गर्ने सेटिङहरू र प्यारामिटरहरू।

## D

**Deployment**  
एप्लिकेसनहरू र तिनीहरूका निर्भरतालाई लक्षित पूर्वाधारमा इन्स्टल र कन्फिगर गर्ने प्रक्रिया।

**Docker**  
कन्टेनराइजेसन प्रविधि प्रयोग गरी एप्लिकेसनहरू विकास, शिप, र चलाउनको लागि प्लेटफर्म।

**Dockerfile**  
Docker कन्टेनर इमेज बनाउने निर्देशनहरू समावेश गर्ने टेक्स्ट फाइल।

## E

**Environment**  
तैनाथीकरण लक्ष्य जसले तपाईंको एप्लिकेसनको विशेष उदाहरण प्रतिनिधित्व गर्छ (जस्तै, विकास, स्टेजिङ, उत्पादन)।

**Environment Variables**  
रनटाइममा एप्लिकेसनहरूले पहुँच गर्न सक्ने कुञ्जी-मूल्य जोडीको रूपमा भण्डारित कन्फिगरेसन मानहरू।

**Endpoint**  
एप्लिकेसन वा सेवामा पहुँच गरिने URL वा नेटवर्क ठेगाना।

## F

**Function App**  
इभेन्ट-ड्रिभन कोड चलाउन सक्षम गराउने Azure को serverless compute सेवा जसले पूर्वाधार व्यवस्थापन गर्न आवश्यक पर्दैन।

## G

**GitHub Actions**  
GitHub रिपोजिटरीहरूसँग संयोजन गरी वर्कफ्लोहरू स्वचालित गर्ने CI/CD प्लेटफर्म।

**Git**  
स्रोत कोडमा भएका परिवर्तनहरू ट्र्याक गर्न प्रयोग गरिने वितरित भर्सन कन्ट्रोल सिस्टम।

## H

**Hooks**  
तैनाथीकरण जीवनचक्रका विशेष बिन्दुहरूमा चल्ने कस्टम स्क्रिप्टहरू वा कमाण्डहरू (preprovision, postprovision, predeploy, postdeploy)।

**Host**  
एप्लिकेसन कहाँ तैनाथ गरिने Azure सेवा प्रकार (जस्तै, appservice, containerapp, function)।

## I

**Infrastructure as Code (IaC)**  
पूर्वाधारलाई म्यानुअल प्रक्रियाको सट्टा कोड मार्फत परिभाषित र व्यवस्थापन गर्ने अभ्यास।

**Init**  
सामान्यतः टेम्पलेटबाट नयाँ azd परियोजना आरम्भ गर्ने प्रक्रिया।

## J

**JSON**  
JavaScript Object Notation। कन्फिगरेसन फाइलहरू र API प्रतिक्रियाहरूका लागि सामान्यतया प्रयोग गरिने डाटा अन्तरबिमो ढाँचा।

**JWT**  
JSON Web Token। पक्षहरू बीच सुरक्षापूर्वक जानकारी पठाउने मानक जुन JSON वस्तुको रूपमा हुन्छ।

## K

**Key Vault**  
सिक्रेट, कुञ्जीहरू, र प्रमाणपत्रहरू सुरक्षित रूपमा भण्डारण र व्यवस्थापन गर्नको लागि Azure सेवा।

**Kusto Query Language (KQL)**  
Azure Monitor, Application Insights, र अन्य Azure सेवाहरूमा डेटा विश्लेषण गर्न प्रयोग गरिने क्वेरी भाषा।

## L

**Load Balancer**  
आउने नेटवर्क ट्राफिकलाई बहु सर्भर वा इन्स्ट्यान्सहरूमा वितरण गर्ने सेवा।

**Log Analytics**  
क्लाउड र अन-प्रिमाइस वातावरणबाट प्राप्त टेलेमेट्री डेटा सङ्कलन, विश्लेषण, र कार्य गर्ने Azure सेवा।

## M

**Managed Identity**  
अन्य Azure सेवाहरूमा प्रमाणिकरणका लागि Azure सेवाहरूलाई स्वचालित रूपमा व्यवस्थापन गरिने पहिचान प्रदान गर्ने Azure सुविधा।

**Microservices**  
एप्लिकेसनहरू साना, स्वतन्त्र सेवाहरूको संग्रहको रूपमा निर्माण गर्ने वास्तुकलागत दृष्टिकोण।

**Monitor**  
एप्लिकेसन र पूर्वाधारभरि पूर्ण-स्ट्याक अवलोकनक्षमता प्रदान गर्ने Azure को एकीकृत निगरानी समाधान।

## N

**Node.js**  
Chrome को V8 JavaScript इन्जिनमा आधारित सर्भर-साइड एप्लिकेसन निर्माणका लागि बनाइएको JavaScript रनटाइम।

**npm**  
Node.js का लागि प्याकेज म्यानेजर जसले निर्भरता र प्याकेजहरू व्यवस्थापन गर्छ।

## O

**Output**  
पूर्वाधार तैनाथीकरणबाट फर्काइएका मानहरू जुन एप्लिकेसन वा अन्य स्रोतहरूले प्रयोग गर्न सक्छन्।

## P

**Package**  
एप्लिकेसन कोड र निर्भरतालाई तैनाथीकरणका लागि तयार पार्ने प्रक्रिया।

**Parameters**  
तैनाथीकरण अनुकूल बनाउन पूर्वाधार टेम्पलेटहरूलाई पास गरिएका इनपुट मानहरू।

**PostgreSQL**  
ओपन-सोर्स रिलेसनल डेटाबेस प्रणाली जुन Azure मा व्यवस्थापित सेवा रूपमा समर्थन गरिन्छ।

**Provisioning**  
पूर्वाधार टेम्पलेटहरूमा परिभाषित Azure स्रोतहरू सिर्जना र कन्फिगर गर्ने प्रक्रिया।

## Q

**Quota**  
Azure सदस्यता वा क्षेत्रभित्र सिर्जना गर्न सकिने स्रोतहरूको मात्रा सम्बन्धी सीमा।

## R

**Resource Group**  
त्यस्तै जीवनचक्र, अनुमतिहरू, र नीतिहरू साझेदार गर्ने Azure स्रोतहरूको तार्किक कन्टेनर।

**Resource Token**  
तैनाथीकरणहरूमा स्रोत नामहरू अनन्य बनाउन azd द्वारा उत्पन्न विशिष्ट स्ट्रिङ।

**REST API**  
HTTP विधिहरू प्रयोग गरी नेटवर्केड एप्लिकेसनहरू डिजाइन गर्ने वास्तुकलागत शैली।

**Rollback**  
एप्लिकेशन वा पूर्वाधार कन्फिगरेसनको पहिला संस्करणमा फर्कने प्रक्रिया।

## S

**Service**  
azure.yaml मा परिभाषित तपाईंको एप्लिकेसनको एक कम्पोनेन्ट (जस्तै, वेब फ्रन्टएन्ड, API ब्याकएन्ड, डेटाबेस)।

**SKU**  
Stock Keeping Unit। Azure स्रोतहरूको फरक सेवा तह वा प्रदर्शन स्तर प्रतिनिधित्व गर्ने इकाई।

**SQL Database**  
Microsoft SQL Server मा आधारित Azure को व्यवस्थापित रिलेसनल डेटाबेस सेवा।

**Static Web Apps**  
स्रोत कोड रिपोजिटरीहरूबाट पूर्ण-स्ट्याक वेब एप्लिकेसनहरू निर्माण र तैनाथ गर्नको लागि Azure सेवा।

**Storage Account**  
ब्लबहरू, फाइलहरू, क्युहरू, र तालिकाहरू सहित डाटा वस्तुहरूको लागि क्लाउड स्टोरेज प्रदान गर्ने Azure सेवा।

**Subscription**  
संसाधन समूहहरू र स्रोतहरू राख्ने Azure खाता कन्टेनर, जससँग बिलिङ र पहुँच व्यवस्थापन सम्बन्धित हुन्छ।

## T

**Template**  
समय-पूर्वनिर्मित परियोजना संरचना जसमा एप्लिकेसन कोड, पूर्वाधार परिभाषाहरू, र सामान्य परिदृश्यहरूको लागि कन्फिगरेसन हुन्छ।

**Terraform**  
कई क्लाउड प्रदायकहरू सहित Azure समर्थन गर्ने open-source Infrastructure as Code उपकरण।

**Traffic Manager**  
ग्लोबल Azure क्षेत्रहरूभरि ट्राफिक वितरण गर्न DNS-आधारित ट्राफिक लोड ब्यालेन्सर।

## U

**URI**  
Uniform Resource Identifier। कुनै विशेष स्रोतलाई पहिचान गर्ने स्ट्रिङ।

**URL**  
Uniform Resource Locator। स्रोत कहाँ अवस्थित छ र कसरी प्राप्त गर्ने भन्ने निर्दिष्ट गर्ने URI प्रकार।

## V

**Virtual Network (VNet)**  
Azure मा निजी नेटवर्कहरूको आधारभूत निर्माण ब्लक, जसले अलगाव र विभाजन प्रदान गर्छ।

**VS Code**  
Visual Studio Code। उत्कृष्ट Azure र azd इंटीग्रेशन भएको लोकप्रिय कोड सम्पादक।

## W

**Webhook**  
विशेष घटनाहरूले ट्रिगर गर्ने HTTP कलब्याक, सामान्यतः CI/CD पाइपलाइनहरूमा प्रयोग गरिन्छ।

**What-if**  
तैनाथीकरणले वास्तविक रूपमा कार्यान्वयन नगरी के-कस्ता परिवर्तनहरू हुने थिए भनेर देखाउने Azure सुविधा।

## Y

**YAML**  
YAML Ain't Markup Language। azure.yaml जस्ता कन्फिगरेसन फाइलहरूका लागि प्रयोग गरिने मानवीय रिडेबल डेटा सिरियालाइजेसन मानक।

## Z

**Zone**  
Azure क्षेत्रभित्रका भौतिक रूपमा अलग स्थानहरू जुन redundancy र उच्च उपलब्धता प्रदान गर्छन्।

## सन्दर्भ-विशेष शब्दहरू

### विकास सम्बन्धी शब्दहरू
- **Hot Reload**: विकास गर्दा एप्लिकेसनहरूलाई पुनःसुरु नगरी स्वतः अपडेट गर्ने क्षमता
- **Build Pipeline**: कोड बनाउने र परीक्षण गर्ने स्वचालित प्रक्रिया
- **Deployment Slot**: App Service भित्रको स्टेजिङ वातावरण
- **Environment Parity**: विकास, स्टेजिङ, र उत्पादन वातावरणहरूलाई समान राख्ने अभ्यास

### सुरक्षा सम्बन्धी शब्दहरू
- **Managed Identity**: स्वचालित प्रमाणपत्र व्यवस्थापन प्रदान गर्ने Azure सुविधा
- **Key Vault**: सिक्रेट, कुञ्जी, र प्रमाणपत्रहरूको लागि सुरक्षित भण्डारण
- **RBAC**: Azure स्रोतहरूका लागि भूमिका-आधारित पहुँच नियन्त्रण
- **Network Security Group**: नेटवर्क ट्राफिक नियन्त्रण गर्नको लागि भर्चुअल फायरवाल

### निगरानी सम्बन्धी शब्दहरू
- **Telemetry**: मापन र डेटा स्वचालित रूपमा सङ्कलन गर्ने प्रक्रिया
- **Application Performance Monitoring (APM)**: सफ्टवेयर प्रदर्शन निगरानी
- **Log Analytics**: लग डेटा सङ्कलन र विश्लेषण गर्ने सेवा
- **Alert Rules**: मेट्रिक्स वा शर्तहरूमा आधारित स्वचालित सूचनाहरू

### तैनाथीकरण सम्बन्धी शब्दहरू
- **Blue-Green Deployment**: शून्य-डाउनटाइम तैनाथीकरण रणनीति
- **Canary Deployment**: प्रयोगकर्ताहरूको उपसमूहमा क्रमिक रुपमा रोलआउट गर्ने तरीका
- **Rolling Update**: एप्लिकेशन इन्स्ट्यान्सहरूको क्रमिक प्रतिस्थापन
- **Rollback**: एप्लिकेशनको पुरानो संस्करणमा फर्किने प्रक्रिया
