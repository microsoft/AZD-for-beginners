# AZD for AI Developers कार्यशाला

> जून 2026 मा `azd 1.25.6` विरुद्ध मान्य गरिएको।

> [!IMPORTANT]  
> **यो कार्यशाला एउटा कार्यशाला मार्गदर्शकसँग सुसज्जित गरिएको छ जुन तपाईंले आफ्नो स्थानीय ब्राउजरमा पूर्वावलोकन गर्न सक्नुहुन्छ। सुरु गर्नका लागि, रिपो GitHub Codespaces, dev container, वा MkDocs इन्स्टल गरिएको स्थानीय क्लोनमा खोल्नुहोस्—त्यसपछि सक्रिय VS Code टर्मिनल देखिनसक्ने गरी पर्खनुहोस् र टाइप गर्नुहोस्:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **तपाईंले पूर्वावलोकन पृष्ठ खोल्न ब्राउजरमा पप-अप संवाद देख्नुपर्नेछ।**

Azure Developer CLI (AZD) सिक्न र AI अनुप्रयोग डिप्लोयमेन्टमा केन्द्रित यो हात-मा-हात सिकाइ कार्यशालामा स्वागत छ। यस कार्यशालाले तपाईंलाई AZD टेम्पलेटहरूको प्रयोग गरेर व्यावहारिक बुझाइ दिन मद्दत गर्छ 3 चरणमा:

1. **Discovery** - तपाईंका लागि उपयुक्त टेम्पलेट फेला पार्नुहोस्।
1. **Deployment** - डिप्लोय गर्नुहोस् र काम गर्छ कि भनेर प्रमाणित गर्नुहोस्
1. **Customization** - परिमार्जन गर्नुहोस् र आफ्नै बनाउनुहोस्!

यस कार्यशालाको क्रममा, तपाईंलाई मुख्य विकास उपकरणहरू र कार्यप्रवाहहरूसँग पनि परिचय गराइनेछ, जसले तपाईंको अन्त्य-देखि-अन्त्य विकास यात्रा सरल बनाउन मद्दत गर्दछ।

## कार्यशाला अवलोकन

**Duration:** 3-4 hours  
**Level:** Beginner to Intermediate  
**Prerequisites:** Azure, AI अवधारणा, VS Code र कमाण्ड-लाइन उपकरणहरूसँग परिचित हुनुपर्ने।

यो एक हात-मा-हात सिकाइ कार्यशाला हो जहाँ तपाईंले गरेर सिक्नुहुनेछ। अभ्यासहरू पूरा गरेपछि, सिकाइ यात्रा जारी राख्न सुरक्षा र उत्पादकत्वका उत्तम अभ्यासहरूमा जानका लागि AZD For Beginners पाठ्यक्रम पुनरावलोकन गर्न सिफारिस गरिन्छ।

| Time| Module  | Objective |
|:---|:---|:---|
| 15 mins | Introduction | सेट द स्टेज, लक्ष्यहरू बुझ्नुहोस् |
| 30 mins | Select AI Template | विकल्पहरू अन्वेषण गर्नुहोस् र सुरुवाती चयन गर्नुहोस् | 
| 30 mins | Validate AI Template | डिफल्ट समाधान Azure मा डिप्लोय गर्नुहोस् |
| 30 mins | Deconstruct AI Template | संरचना र कन्फिगरेसन अन्वेषण गर्नुहोस् |
| 30 mins | Configure AI Template | उपलब्ध सुविधाहरू सक्रिय गरी प्रयास गर्नुहोस् |
| 30 mins | Customize AI Template | टेम्पलेटलाई आफ्नो आवश्यकताअनुसार अनुकूल बनाउनुहोस् |
| 30 mins | Teardown Infrastructure | सफा गर्नुहोस् र स्रोतहरू रिलिज गर्नुहोस् |
| 15 mins | Wrap-Up & Next Steps | सिकाइ स्रोतहरू, कार्यशाला चुनौती |
| | |

## तपाईंले के सिक्नुहुनेछ

AZD टेम्पलेटलाई Microsoft Foundry मा अन्त्य-देखि-अन्त्य विकासका विभिन्न क्षमताहरू र उपकरणहरू अन्वेषण गर्नको लागि सिकाइ स्यान्डबक्सको रूपमा सोच्नुहोस्। यस कार्यशाला समाप्त हुँदा, तपाईंले यस सन्दर्भमा विभिन्न उपकरण र अवधारणाहरूको सहज बुझाइ पाउनु पर्नेछ।

| Concept  | Objective |
|:---|:---|
| **Azure Developer CLI** | उपकरणका आदेश र कार्यप्रवाहहरू बुझ्नुहोस्|
| **AZD Templates**| प्रोजेक्ट संरचना र कन्फिग बुझ्नुहोस्|
| **Azure AI Agent**| Microsoft Foundry प्रोजेक्ट provision र डिप्लोय गर्नुहोस्  |
| **Azure AI Search**| एजेण्टहरूसँग सन्दर्भ इन्जिनियरिङ सक्षम गर्नुहोस् |
| **Observability**| ट्रेसिङ, मोनिटरिङ र मूल्यांकनहरू अन्वेषण गर्नुहोस् |
| **Red Teaming**| प्रतिकूल परीक्षण र निवारणहरू अन्वेषण गर्नुहोस् |
| | |
