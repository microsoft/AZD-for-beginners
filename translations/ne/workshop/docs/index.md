# AZD for AI Developers कार्यशाला

> [!IMPORTANT]  
> **यो कार्यशाला एउटा कार्यशाला मार्गदर्शकले सुसज्जित छ जुन तपाईंले आफ्नो स्थानीय ब्राउजरमा पूर्वावलोकन गर्न सक्नुहुन्छ। सुरू गर्नका लागि, तपाईंले यस रिपोमा GitHub Codespaces सुरु गर्नुपर्छ—पछि सक्रिय VS Code टर्मिनल देखिने बित्तिकै टाइप गर्नुहोस्:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **ब्राउजरमा पूर्वावलोकन पृष्ठ खोल्न एउटा पप-अप संवाद देखिनुपर्छ।**

Azure Developer CLI (AZD) सँग एआई एप्लिकेसन परिनियोजनमा केन्द्रित हातेमालो कार्यशालामा स्वागत छ। यो कार्यशालाले तपाईंलाई AZD टेम्पलेटहरूको व्यावहारिक समझ 3 चरणमा प्राप्त गर्न मद्दत गर्छ:

1. **खोज** - तपाईंका लागि उपयुक्त टेम्पलेट फेला पार्नुहोस्।
1. **परिनियोजन** - परिनियोजन गर्नुहोस् र काम गर्छ कि गर्दैन प्रमाणित गर्नुहोस्
1. **अनुकूलन** - परिमार्जन गर्नुहोस् र पुनरावृत्ति गरेर यसलाई आफ्नो बनाउनुहोस्!

यस कार्यशालाको क्रममा, तपाईंलाई कोर डेभलपर उपकरण र कार्यप्रवाहहरूसँग पनि परिचय गराइनेछ, जसले तपाईंको अन्त्य-देखि-अन्त्य विकास यात्रा सरल बनाउन मद्दत पुर्‍याउनेछ।

## कार्यशाला अवलोकन

**अवधि:** 3-4 घन्टा  
**स्तर:** सुरुबाट मध्यम स्तरसम्म  
**पूर्वआवश्यकता:** Azure, AI अवधारणाहरू, VS Code र कमाण्ड-लाइन उपकरणहरूको परिचिती।

यो एक हातेमालो कार्यशाला हो जहाँ तपाईंले गर्दै गर्दा सिक्नुहुनेछ। अभ्यासहरू पूरा भएपछि, हामी तपाईंलाई सिकाइ यात्रा जारी राख्न AZD For Beginners पाठ्यक्रम समीक्षा गर्न सिफारिस गर्छौं जसले सुरक्षा र उत्पादकताका उत्तम अभ्यासहरूमा मार्गदर्शन दिनेछ।

| Time| Module  | Objective |
|:---|:---|:---|
| 15 mins | परिचय | अवस्था सेट गर्नुहोस्, लक्ष्यहरू बुझ्नुहोस् |
| 30 mins | एआई टेम्पलेट चयन गर्नुहोस् | विकल्पहरू अन्वेषण गर्नुहोस् र सुरुवाती टेम्पलेट छान्नुहोस् | 
| 30 mins | एआई टेम्पलेट प्रमाणित गर्नुहोस् | डिफल्ट समाधान Azure मा परिनियोजन गर्नुहोस् |
| 30 mins | एआई टेम्पलेट विखण्डन गर्नुहोस् | संरचना र कन्फिगरेसन अन्वेषण गर्नुहोस् |
| 30 mins | एआई टेम्पलेट कन्फिगर गर्नुहोस् | उपलब्ध सुविधाहरू सक्रिय गरेर प्रयास गर्नुहोस् |
| 30 mins | एआई टेम्पलेट अनुकूलन गर्नुहोस् | टेम्पलेटलाई आफ्नो आवश्यकताअनुसार अनुकूल गर्नुहोस् |
| 30 mins | पूर्वाधार टुटाउन | सफाई गर्नुहोस् र स्रोतहरू रिलिज गर्नुहोस् |
| 15 mins | समापन र अर्को चरणहरू | सिकाइ स्रोतहरू, कार्यशाला चुनौती |
| | |

## तपाईंले के सिक्नुहुनेछ

AZD टेम्पलेटलाई Microsoft Foundry मा अन्त्य-देखि-अन्त्य विकासका लागि विभिन्न क्षमताहरू र उपकरणहरू अन्वेषण गर्ने एउटा सिकाइ स्यान्डबक्सको रूपमा सोच्नुहोस्। कार्यशालाको अन्त्यसम्ममा, तपाईंले यस सन्दर्भमा विभिन्न उपकरण र अवधारणाहरूको सहज अनुभव प्राप्त गर्नुहुनेछ।

| Concept  | Objective |
|:---|:---|
| **Azure Developer CLI** | उपकरणका कमाण्ड र कार्यप्रवाहहरू बुझ्नुहोस् |
| **AZD Templates**| परियोजना संरचना र कन्फिगरेसन बुझ्नुहोस् |
| **Azure AI Agent**| Microsoft Foundry परियोजनालाई सेटअप र परिनियोजन गर्नुहोस् |
| **Azure AI Search**| एजेन्टहरूसँग सन्दर्भ इन्जिनियरिङ सक्षम गर्नुहोस् |
| **Observability**| ट्रेसिङ, अनुगमन र मूल्याङ्कनहरू अन्वेषण गर्नुहोस् |
| **Red Teaming**| प्रतिकूल परीक्षण र रोकथाम उपायहरू अन्वेषण गर्नुहोस् |
| | |
