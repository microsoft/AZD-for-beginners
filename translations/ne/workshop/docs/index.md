# AI विकासकर्ताहरūkको लागि AZD कार्यशाला

> `azd 1.23.12` विरुद्ध मार्च 2026 मा प्रमाणित गरिएको।

> [!IMPORTANT]  
> **यो कार्यशाला एक कार्यशाला मार्गदर्शक सहित सज्जित छ जुन तपाईंले आफ्नो स्थानीय ब्राउजरमा पूर्वावलोकन गर्न सक्नुहुन्छ। सुरु गर्नका लागि, GitHub Codespaces, dev container, वा MkDocs स्थापना गरिएको स्थानीय क्लोनमा रिपो खोल्नुहोस्—त्यसपछि सक्रिय VS Code टर्मिनल देखिने सम्म पर्खनुहोस् र टाइप गर्नुहोस्:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **तपाईंले ब्राउजरमा पूर्वावलोकन पृष्ठ खोल्नको लागि एउटा पप-अप संवाद देख्नुपर्नेछ।**

Azure Developer CLI (AZD) सिक्नको लागि एआई अनुप्रयोग परिनियोजनमा केन्द्रित हातमा कार्यशालामा तपाईंलाई स्वागत छ। यो कार्यशालाले तपाईंलाई AZD टेम्प्लेटहरूको व्यावहारिक समझ उपलब्ध गराउँछ ३ चरणमा:

1. **पत्ता लगाउने** - तपाईंको लागि उपयुक्त टेम्प्लेट खोज्नुहोस्।
1. **परिनियोजन** - तैनाथ गर्नुहोस् र काम गर्छ कि भनी प्रमाणित गर्नुहोस्।
1. **अनुकूलन** - संशोधन गर्नुहोस् र तपाईंको बनाउन दोहोर्याउनुहोस्!

यस कार्यशालाको क्रममा, तपाईंलाई मुख्य विकास उपकरणहरू र कार्यप्रवाहहरू पनि परिचित गराइनेछ, जसले तपाईंको सम्पूर्ण विकास यात्रालाई सहज बनाउँछ।

## कार्यशाला अवलोकन

**अवधि:** ३-४ घण्टा  
**स्तर:** प्रारम्भ देखि मध्यम  
**पूर्वआवश्यकताहरू:** Azure, AI अवधारणाहरू, VS Code र कमाण्ड लाइन उपकरणहरूमा परिचय।

यो एउटा हातमा सिकाइ कार्यशाला हो जहाँ तपाईं अभ्यास गरेर सिक्नुहुन्छ। अभ्यासहरू पूरा गरेपछि, हामी तपाईंलाई AZD आरम्भकर्ताहरूका लागि पाठ्यक्रम समीक्षा गर्न सिफारिस गर्छौं ताकि सुरक्षा र उत्पादकत्वका उत्कृष्ट अभ्यासहरूमा सिकाइ यात्रालाई निरन्तरता दिन सकियोस्।

| समय| मोड्युल  | उद्देश्य |
|:---|:---|:---|
| १५ मिनेट | परिचय | स्थापन गर्नुहोस्, लक्ष्यहरू बुझ्नुहोस् |
| ३० मिनेट | AI टेम्प्लेट चयन गर्नुहोस् | विकल्पहरू अन्वेषण गर्नुहोस् र सुरु गर्ने छनौट गर्नुहोस् | 
| ३० मिनेट | AI टेम्प्लेट प्रमाणित गर्नुहोस् | Azure मा पूर्वनिर्धारित समाधान परिनियोजन गर्नुहोस् |
| ३० मिनेट | AI टेम्प्लेट विघटन गर्नुहोस् | संरचना र कन्फिगरेसन अन्वेषण गर्नुहोस् |
| ३० मिनेट | AI टेम्प्लेट कन्फिगर गर्नुहोस् | उपलब्ध सुविधाहरू सक्रिय गर्नुहोस् र प्रयास गर्नुहोस् |
| ३० मिनेट | AI टेम्प्लेट अनुकूलन गर्नुहोस् | टेम्प्लेटलाई तपाईंको आवश्यकताहरू अनुसार समायोजन गर्नुहोस् |
| ३० मिनेट | पूर्वाधार टुटाउनुहोस् | सफाई गर्नुहोस् र स्रोतहरू मुक्त गर्नुहोस् |
| १५ मिनेट | समापन र अर्को कदमहरू | सिकाइ स्रोतहरू, कार्यशाला चुनौती |
| | |

## तपाईंले के सिक्नुहुनेछ

AZD टेम्प्लेटलाई एउटा सिकाइ सँडबक्सको रुपमा सोच्नुस् जहाँ विभिन्न कार्यक्षमताहरू र उपकरणहरू Microsoft Foundry मा सम्पूर्ण विकासका लागि अन्वेषण गर्न सकिन्छ। यस कार्यशालाको अन्त्य सम्म, तपाईंले यस सन्दर्भमा विभिन्न उपकरण र अवधारणाहरूको सहज अनुभूति पाउनुहुनेछ।

| अवधारणा  | उद्देश्य |
|:---|:---|
| **Azure Developer CLI** | उपकरण आदेशहरू र कार्यप्रवाहहरू बुझ्नुहोस् |
| **AZD टेम्प्लेटहरू**| परियोजना संरचना र कन्फिगरेसन बुझ्नुहोस् |
| **Azure AI Agent**| Microsoft Foundry परियोजना प्रावधान एवं परिनियोजन गर्नुहोस् |
| **Azure AI Search**| एजेन्टहरूसँग सन्दर्भ अभियांत्रण सक्षम गर्नुहोस् |
| **Observability**| ट्रेसिङ, अनुगमन र मूल्याङ्कन अन्वेषण गर्नुहोस् |
| **Red Teaming**| प्रतिद्वन्द्वी परीक्षण र न्यूनीकरणहरू अन्वेषण गर्नुहोस् |
| | |
