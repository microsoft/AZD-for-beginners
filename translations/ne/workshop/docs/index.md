# AZD for AI Developers Workshop

> Validated against `azd 1.27.1` in July 2026.

> [!IMPORTANT]  
> **यस कार्यशालामा एक कार्यशाला मार्गदर्शक समावेश छ जुन तपाईंले आफ्नो स्थानीय ब्राउजरमा पूर्वावलोकन गर्न सक्नुहुन्छ। सुरू गर्नका लागि, GitHub Codespaces, dev container, वा MkDocs स्थापना गरिएको स्थानीय क्लोनमा रिपो खोल्नुहोस्—पछि एक सक्रिय VS Code टर्मिनल देखिने सम्म पर्खनुहोस् र टाइप गर्नुहोस्:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **तपाईंले ब्राउजरमा पूर्वावलोकन पृष्ठ खोल्नको लागि पप-अप संवाद देख्नुपर्दछ।**

Azure Developer CLI (AZD) सिक्नको लागि प्रस्तुति रत कार्यशालामा स्वागत छ जसको फोकस AI एप्लिकेसन डिप्लोइमेन्टमा छ। यो कार्यशालाले तपाईलाई 3 चरणहरूमा AZD टेम्प्लेटहरूको व्यावहारिक बुझाइ प्राप्त गर्न मद्दत गर्दछ:

1. **अवलोकन** - तपाईंको लागि उपयुक्त टेम्प्लेट पत्ता लगाउनुहोस्।
1. **डिप्लोइमेन्ट** - डिप्लोइ गर्नुहोस् र काम गर्छ कि भनेर प्रमाणित गर्नुहोस्।
1. **अनुकूलन** - परिमार्जन गर्नुहोस् र तपाईको बनाउन पुनरावृत्ति गर्नुहोस्!

यस कार्यशालाको क्रममा, तपाईंलाई मुख्य विकास उपकरणहरू र कार्यप्रवाहहरू पनि परिचित गराइनेछ, जसले तपाईंको अन्त-देखि-अन्त विकास यात्रालाई सरल बनाउन सहयोग पुर्‍याउँछ।

## कार्यशाला अवलोकन

**अवधि:** ३-४ घण्टा  
**स्तर:** शुरुवातीदेखि माध्यमिक  
**पूर्वापेक्षाएँ:** Azure, AI अवधारणाहरू, VS Code र कमाण्ड-लाइन उपकरणहरूमा परिचित हुनु।

यो एक हात-ले-देखाउँदै कार्यशाला हो जहाँ तपाईंले गर्ने क्रममा सिक्नुहुन्छ। एकपटक तपाईंले अभ्यासहरू पूरा गरेपछि, हामी AZD For Beginners पाठ्यक्रम समीक्षा गर्न सिफारिस गर्छौं ताकि सुरक्षाका र उत्पादकता उत्कृष्ट अभ्यासहरूमा तपाईंको सिकाई जारी राख्न सकियोस्।

| समय| मोड्युल  | उद्देश्य |
|:---|:---|:---|
| १५ मिनेट | परिचय | वातावरण तयार पार्नुहोस्, लक्ष्यहरू बुझ्नुहोस् |
| ३० मिनेट | AI टेम्प्लेट चयन गर्नुहोस् | विकल्पहरू अन्वेषण गर्नुहोस् र सुरु गर्नुहोस् | 
| ३० मिनेट | AI टेम्प्लेट प्रमाणित गर्नुहोस् | डिफल्ट समाधान Azure मा डिप्लोइ गर्नुहोस् |
| ३० मिनेट | AI टेम्प्लेट विखण्डन गर्नुहोस् | संरचना र कन्फिगरेसन अन्वेषण गर्नुहोस् |
| ३० मिनेट | AI टेम्प्लेट कन्फिगर गर्नुहोस् | सक्रिय पार्नुहोस् र उपलब्ध सुविधाहरू प्रयास गर्नुहोस् |
| ३० मिनेट | AI टेम्प्लेट अनुकूलन गर्नुहोस् | टेम्प्लेटलाई तपाईंको आवश्यकताअनुसार अनुकूलित गर्नुहोस् |
| ३० मिनेट | पूर्वाधार हटाउनुहोस् | सफा गर्नुहोस् र स्रोतहरू रिलीज गर्नुहोस् |
| १५ मिनेट | समापन र अर्को कदमहरू | सिकाइ सामग्रीहरू, कार्यशाला चुनौती |
| | |

## तपाईंले के सिक्नुभनेछ

AZD टेम्प्लेटलाई सिक्ने र अभ्यास गर्ने स्यान्डबक्सको रूपमा सोच्नुहोस्, जसले Microsoft Foundry को अन्त-देखि-अन्त विकासका लागि विभिन्न क्षमता र उपकरणहरू अन्वेषण गर्न मद्दत गर्छ। यस कार्यशालाको अन्त्यमा तपाईंलाई यस प्रसंगमा विभिन्न उपकरण र अवधारणाहरूको सहज अनुभूति हुनु पर्छ।

| अवधारणा  | उद्देश्य |
|:---|:---|
| **Azure Developer CLI** | उपकरणका आदेशहरू र कार्यप्रवाहहरू बुझ्नुहोस्|
| **AZD टेम्प्लेटहरू**| परियोजना संरचना र कन्फिगरेसन बुझ्नुहोस्|
| **Azure AI Agent**| Microsoft Foundry परियोजना प्रबन्धन र डिप्लोइ गर्नुहोस्  |
| **Azure AI Search**| एजेन्टहरूसँग सन्दर्भ इन्जिनियरिङ सक्षम गर्नुहोस् |
| **अवलोकनयोग्यता**| ट्रेसिङ, निरिक्षण र मूल्यांकनहरू अन्वेषण गर्नुहोस् |
| **रेड टीमिङ**| विरोधी परीक्षण र न्यूनीकरणहरू अन्वेषण गर्नुहोस् |
| | |
