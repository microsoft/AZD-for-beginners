# AZD AI विकासकर्ताहरूका लागि

!!! info "यस कार्यशालाका लागि सहभागी"
   
    यो कार्यशाला Microsoft Foundry मा उत्कृष्ट उपकरणहरू र कार्यप्रवाहहरूको उपयोग गरेर AI Agentic अनुप्रयोगहरू निर्माण गर्न चाहने कुनै पनि व्यक्तिका लागि हो। कार्यशाला Azure, AI अवधारणाहरू र Python कोडसँग केही परिचितता भएका शुरुवातकर्ताहरू र अनुभवी विकासकर्ताहरू दुवैका लागि उपयुक्त छ। कृपया सुरु गर्नु अघि पूर्व-आवश्यकताहरू जाँच गर्नुहोस्।

    **अवधि:** 3-4 घण्टा  
    **स्तर:** शुरुवाती देखि मध्यम  

---


## पूर्व आवश्यकताहरू

- व्यक्तिगत GitHub खाता → [निःशुल्क एक बनाउनुहोस्](https://github.com/signup)
- सक्रिय Azure खाता → [निःशुल्क एक बनाउनुहोस्](https://aka.ms/free)
- AI अवधारणाहरू, Python विकास, कमाण्ड-लाइन उपकरणहरूसँग परिचित
- VS Code र GitHub उपकरण र कार्यप्रवाहहरूसँग अनुभव।

---

## सिकाइ उद्देश्यहरू

यस कार्यशाला समाप्त हुँदासम्म, तपाईं जान्न सक्नुहुनेछ कि:

- [X] **Work with Azure Developer CLI** → तपाईंका Azure स्रोतहरू प्राविजन, परिनियोजन, र व्यवस्थापन गर्न।
- [X] **Use AI App Templates** → Microsoft Foundry मा AI Agent परियोजनाहरू छिटो सुरु गर्न।
- [X] **Configure Templates** → Red Teaming, मूल्याङ्कनहरू, ट्रेसिङ र निगरानी जस्ता सुविधाहरूका लागि
- [X] **Deconstruct and Explore** → GitHub Copilot प्रयोग गरेर AZD टेम्पलेटहरू र कार्यप्रवाहहरू विश्लेषण र अन्वेषण गर्न।
- [X] **Customize and Redeploy** → AI Agent स्टार्टर टेम्पलेटलाई तपाईंका अनुकूल आवश्यकताहरू अनुरूप अनुकूलन र पुनःतैनाथ गर्न।

---

## कार्यशाला रूपरेखा

यो कार्यशाला मोड्युलहरूमा संरचित छ, प्रत्येक हात-ओन ल्याब अभ्यासद्वारा आधारित:

1. **Introduction** - AZD टेम्पलेटहरू के हुन् र तिनीहरू कसरी काम गर्छन् भन्ने बुझाइ प्राप्त गर्नुहोस्
1. **Selection** - तपाईंको अर्को AI परियोजनाका लागि उपयुक्त "starter" टेम्पलेट पत्ता लगाउनुहोस्
1. **Validation** - टेम्पलेटलाई तैनाथ र अन्वेषण गरी यो विज्ञापनअनुसार काम गर्छ कि गर्दैन जाँच गर्नुहोस्
1. **Deconstruction** - टेम्पलेटलाई अलग गरी यसको कम्पोनेन्टहरू र कन्फिगरेसन बुझ्नुहोस्
1. **Configuration** - विभिन्न Microsoft Foundry सुविधाहरू सक्रिय गर्न र प्रयोग गर्न सिक्नुहोस्
1. **Customization** - तपाईंको डेटा र आवश्यकताहरू ध्यानमा राखेर एप्लिकेशन पुनर्निर्माण गर्नुहोस्
1. **Teardown** - पूर्वाधार डिप्रोभिजन गर्न र सफ्ट-डिलेट गरिएका स्रोतहरू पुनःप्राप्त गर्न सिक्नुहोस्

---

## प्रतिक्रिया र समस्याहरू

हामी पाठ्यक्रमहरूलाई सक्रिय रूपमा अपडेट गर्छौं ताकि शुरुवातकर्ताहरूका लागि उत्कृष्ट सिकाइ अनुभव प्रदान होस्। तपाईंको प्रतिक्रिया स्वागतयोग्य छ।

1. यस रिपोमा समस्या पोस्ट गर्नुहोस् - सहजताको लागि यसलाई `Workshop` ट्याग गर्नुहोस्।
1. Microsoft Foundry Discord मा सहभागी हुनुहोस् - सहपाठी सिक्नेहरूसँग जडान हुनुहोस्!
