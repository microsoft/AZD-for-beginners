# अध्याय ३: कन्फिगरेसन र प्रमाणीकरण

**📚 कोर्स**: [AZD For Beginners](../../README.md) | **⏱️ अवधि**: ४५-६० मिनेट | **⭐ जटिलता**: मध्यम

---

## अवलोकन

यो अध्यायले Azure Developer CLI डिप्लोयमेन्टहरूका लागि वातावरण कन्फिगरेसन, प्रमाणीकरण ढाँचाहरू, र सुरक्षा उत्तम अभ्यासहरू समेट्छ।

> मार्च २०२६ मा `azd 1.23.12` विरुद्ध मान्य गरिएको।

## सिकाइ उद्देश्यहरू

यस अध्याय पूरा गरेपछि तपाईंले:
- AZD कन्फिगरेसन पदानुक्रममा निपूर्णता हासिल गर्नुहुनेछ
- धेरै वातावरणहरू (डेभ, स्टेजिंग, प्रोड) व्यवस्थापन गर्नुहोस्
- व्यवस्थापन गरिएको पहिचानहरूसँग सुरक्षित प्रमाणीकरण कार्यान्वयन गर्नुहोस्
- वातावरण-विशिष्ट सेटिङहरू कन्फिगर गर्नुहोस्

## 🚀 द्रुत सुरु

```bash
# धेरै वातावरणहरू सिर्जना गर्नुहोस्
azd env new dev
azd env new staging
azd env new prod

# वातावरणहरू परिवर्तन गर्नुहोस्
azd env select prod

# वातावरण भेरिएबलहरू सेट गर्नुहोस्
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# कन्फिगरेसन हेर्नुहोस्
azd env get-values
```

---

## 🔧 कन्फिगरेसन पदानुक्रम

AZD ले सेटिङहरू यस क्रममा लागू गर्छ (पछिल्लाले पहिलेको ओभरराइड गर्छ):

१. **पूर्वनिर्धारित मानहरू** (टेम्पलेटमा निर्मित)
२. **azure.yaml** (प्रोजेक्ट कन्फिगरेसन)
३. **वातावरण चल बिच्छेदनहरू** (`azd env set`)
४. **कमाण्ड-लाइन फ्ल्यागहरू** (`--location eastus`)

---

## 🔐 सुरक्षा उत्तम अभ्यासहरू

```bash
# व्यवस्थापन गरिएको परिचय प्रयोग गर्नुहोस् (सिफारिस गरिएको)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD प्रमाणीकरण स्थिति जाँच गर्नुहोस्
azd auth status

# वैकल्पिक: यदि तपाईं az आदेशहरू चलाउनुहुने योजना छ भने Azure CLI सन्दर्भ प्रमाणीकरण गर्नुहोस्
az account show

# आवश्यक पर्दा पुनः प्रमाणीकरण गर्नुहोस्
azd auth login

# वैकल्पिक: az आदेशहरूको लागि Azure CLI प्रमाणीकरण नयाँ गर्नुहोस्
az login
```
