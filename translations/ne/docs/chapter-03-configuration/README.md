# अध्याय ३: कन्फिगरेसन र प्रमाणीकरण

**📚 पाठ्यक्रम**: [AZD आरम्भिकहरूको लागि](../../README.md) | **⏱️ अवधि**: 45-60 मिनेट | **⭐ जटिलता**: मध्यम

---

## अवलोकन

यो अध्यायले Azure Developer CLI तैनातीहरूका लागि वातावरण कन्फिगरेसन, प्रमाणीकरण ढाँचाहरू, र सुरक्षा सर्वोत्तम अभ्यासहरू समेट्छ।

## शिक्षण उद्देश्यहरू

पूरा गरेपछि, तपाईंले:
- AZD कन्फिगरेसन पदानुक्रममा विशेषज्ञता हासिल गर्नुहुनेछ
- एकाधिक वातावरणहरू (dev, staging, prod) व्यवस्थापन गर्नुहोस्
- प्रबन्धित पहिचानहरू प्रयोग गरेर सुरक्षित प्रमाणीकरण लागू गर्नुहोस्
- वातावरण-विशिष्ट सेटिङहरू कन्फिगर गर्नुहोस्

## 🚀 छिटो सुरु

```bash
# एकाधिक वातावरणहरू सिर्जना गर्नुहोस्
azd env new dev
azd env new staging
azd env new prod

# वातावरणहरू परिवर्तन गर्नुहोस्
azd env select prod

# वातावरण चरहरू सेट गर्नुहोस्
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# विन्यास हेर्नुहोस्
azd env get-values
```

---

## 🔧 कन्फिगरेसन पदानुक्रम

AZD ले सेटिङहरू यो क्रममा लागू गर्छ (पछिकोले अघिल्लोलाई ओभरराइड गर्छ):

1. **डिफल्ट मानहरू** (टेम्प्लेटहरूमा निर्मित)
2. **azure.yaml** (प्रोजेक्ट कन्फिगरेसन)
3. **वातावरण भेरिएबलहरू** (`azd env set`)
4. **कमाण्ड-लाइन झण्डाहरू** (`--location eastus`)

---

## 🔐 सुरक्षा सर्वोत्तम अभ्यासहरू

```bash
# प्रबन्धित आइडेन्टिटी प्रयोग गर्नुहोस् (सिफारिस गरिएको)
azd env set AZURE_USE_MANAGED_IDENTITY true

# प्रमाणीकरण स्थिति जाँच्नुहोस्
azd auth whoami
az account show

# आवश्यक भएमा पुन: प्रमाणीकरण गर्नुहोस्
azd auth login
az login
```
