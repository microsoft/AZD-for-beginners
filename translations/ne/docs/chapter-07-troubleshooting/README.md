# अध्याय ७: समस्या समाधान र डिबगिङ

**📚 कोर्स**: [AZD प्रारम्भकर्ताहरूका लागि](../../README.md) | **⏱️ अवधि**: १-१.५ घण्टा | **⭐ जटिलता**: मध्यवर्ती

---

## अवलोकन

यस अध्यायले तपाईंलाई Azure Developer CLI सँग काम गर्दा सामान्य समस्या पत्ता लगाउन र समाधान गर्न मद्दत गर्छ। डिप्लोइमेन्ट असफलताबाट AI-विशिष्ट समस्यासम्म।

> मार्च २०२६ मा `azd 1.23.12` विरुद्ध मान्य गरिएको।

## सिकाइ लक्ष्यहरू

यस अध्याय पूरा गरेर, तपाईंले:
- साधारण AZD डिप्लोइमेन्ट असफलता पत्ता लगाउने
- प्रमाणीकरण र अनुमति समस्याहरू डिबग गर्ने
- AI सेवा कनेक्टिविटी समस्याहरू समाधान गर्ने
- समस्या समाधानका लागि Azure Portal र CLI प्रयोग गर्ने

## 🚨 छिटो समाधानहरू

### प्रमाणीकरण समस्या
```bash
# AZD कार्यप्रवाहहरूको लागि आवश्यक
azd auth login

# यदि तपाईं Azure CLI आदेशहरू सिधै प्रयोग गर्दै हुनुहुन्छ भने वैकल्पिक
az login

azd auth status
```

### प्रोभिजनिङ असफलता
```bash
azd show
azd monitor --logs
az deployment sub list --query "[?properties.provisioningState!='Succeeded']"
```

### स्रोत द्वन्द्वहरू
```bash
azd down --force --purge
azd env new different-name
azd up
```

### कोटा भन्दा बढी
```bash
az vm list-usage --location eastus --output table
azd env set AZURE_LOCATION westus2
azd up
```

## 📋 त्रुटि कोड संदर्भ

| त्रुटि | कारण | समाधान |
|-------|-------|----------|
| `AuthenticationError` | लगइन भएन | `azd auth login` |
| `ResourceNotFound` | स्रोत हराइरहेको छ | स्रोत नामहरू जाँच गर्नुहोस् |
| `QuotaExceeded` | सदस्यता सीमा | कोटा वृद्धि अनुरोध गर्नुहोस् |
| `InvalidTemplate` | बाइसेप वाक्य संरचना त्रुटि | `az bicep build` |
| `Conflict` | स्रोत पहिले नै छ | नयाँ नाम प्रयोग गर्नुहोस् वा मेटाउनुहोस् |
| `Forbidden` | अनुमति अभाव | RBAC भूमिका जाँच गर्नुहोस् |

## 🔄 रिसेट र पुन: प्राप्ति

```bash
# सफ्ट रिसेट (स्रोतहरू राख्नुहोस्, कोड फेरि लागू गर्नुहोस्)
azd deploy --force

# हार्ड रिसेट (सबै कुरा मेट्नुहोस्, नयाँ सुरु गर्नुहोस्)
azd down --force --purge
azd up
```
