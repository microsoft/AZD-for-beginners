# अध्याय ६: पूर्व-वितरण योजना र प्रमाणीकरण

**📚 कोर्स**: [ AZD सुरुकर्ताहरूका लागि](../../README.md) | **⏱️ अवधि**: १ घण्टा | **⭐ जटिलता**: मध्यम

---

## अवलोकन

यस अध्यायले तपाईंको एप्लिकेशन वितरण गर्नुअघि आवश्यक योजना बनाउने र प्रमाणीकरण गर्ने कदमहरू कभर गर्दछ। उचित क्षमता योजना, SKU चयन, र पूर्व-जाँचहरू मार्फत महँगो गल्तीहरूबाट बच्न सिक्नुहोस्।

> `azd 1.27.1` मा जुलाई २०२६ मा प्रमाणित।

## सिकाइ लक्ष्यहरू

यस अध्याय पूरा गरेर, तपाईंले:
- वितरण अघि पूर्व-जाँचहरू चलाउने
- क्षमता योजना बनाउने र स्रोत आवश्यकताहरू अनुमान गर्ने
- लागत अनुकूलनका लागि उपयुक्त SKU चयन गर्ने
- अनुगमनका लागि एप्लिकेशन इनसाइट्स कन्फिगर गर्ने
- टोली समन्वय नमूनाहरू बुझ्ने

## 🚀 छिटो सुरु

```bash
# सदस्यता कोटा जाँच गर्नुहोस्
az vm list-usage --location eastus --output table

# पूर्वावलोकन वितरण (कुनै स्रोतहरू सिर्जना गरिएका छैनन्)
azd provision --preview

# बाइसप वाक्य रचना प्रमाणित गर्नुहोस्
az bicep build --file infra/main.bicep

# वातावरण कन्फिगरेसन जाँच गर्नुहोस्
azd env get-values
```

## ☑️ पूर्व-वितरण जाँचसूची

### `azd provision` अघि

- [ ] क्षेत्रको लागि कोटा प्रमाणित
- [ ] SKU हरू उपयुक्त रूपमा चयन गरिएको
- [ ] लागत अनुमान समीक्षा गरिएको
- [ ] नामकरण सम्मेलन सुसंगत
- [ ] सुरक्षा/RBAC कन्फिगर गरिएको

### `azd deploy` अघि

- [ ] वातावरण भेरियेबल सेट गरिएको
- [ ] की भल्टमा गोप्य सूचना
- [ ] जडान स्ट्रिङहरू प्रमाणीकरण गरिएको
- [ ] स्वास्थ्य जाँचहरू कन्फिगर गरिएको
