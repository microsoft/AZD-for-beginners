# अध्याय १: आधार र छिटो सुरुवात

**📚 कोर्स**: [शुरुआतीहरूको लागि AZD](../../README.md) | **⏱️ अवधि**: ३०-४५ मिनेट | **⭐ जटिलता**: शुरुवाती

---

## अवलोकन

यस अध्यायले Azure Developer CLI (azd) का आधारभूत कुरा परिचय गराउँछ। तपाईंले मुख्य अवधारणाहरू सिक्नुहुनेछ, उपकरणहरू इन्स्टल गर्नुहुनेछ, र Azure मा आफ्नो पहिलो एप्लिकेशन डिप्लोय गर्नुहुनेछ।

> `azd 1.27.1` सँग जुलाई २०२६ मा प्रमाणित गरिएको।

## सिकाई उद्देश्यहरू

यस अध्याय पूरा गरेर, तपाईं:
- Azure Developer CLI के हो र Azure CLI सँग कसरी फरक छ बुझ्नुहुनेछ
- आफ्नो प्लेटफर्ममा AZD इन्स्टल र कन्फिगर गर्नुहुनेछ
- `azd up` कमाण्ड प्रयोग गरी Azure मा आफ्नो पहिलो एप्लिकेशन डिप्लोय गर्नुहुनेछ
- `azd down` कमाण्ड प्रयोग गरी स्रोतहरू सफा गर्नुहुनेछ

## ✅ यहाँबाट सुरु गर्नुहोस्: आफ्नो सेटअप प्रमाणित गर्नुहोस्

सुरु गर्नु अघि, सुनिश्चित गर्नुहोस् कि तपाईंको स्थानीय मेसिन अध्याय १ टेम्प्लेटका लागि तयार छ:

**विन्डोज:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

यदि स्क्रिप्टले उपकरणहरू हराएका छन् भन्छ भने, पहिले ती समस्या समाधान गर्नुहोस् र त्यसपछि अध्यायसँग जारी राख्नुहोस्।

---

## 🚀 छिटो सुरुवात

```bash
# स्थापना जाँच गर्नुहोस्
azd version

# AZD को लागि प्रमाणीकरण गर्नुहोस्
# वैकल्पिक: तपाईंले Azure CLI आदेशहरू सिधै चलाउने योजना बनाउनु भएको छ भने az login गर्नुहोस्
azd auth login

# आफ्नो पहिलो अनुप्रयोग तैनाथ गर्नुहोस्
azd init --template todo-nodejs-mongo
azd up

# समाप्त भएपछि सफा गर्नुहोस्
azd down --force --purge
```

---

## ✅ सफलता मापदण्ड

यो अध्याय पूरा गरेपछि, तपाईंले सक्नुहुनेछ:

```bash
azd version              # स्थापना गरिएको संस्करण देखाउँछ
azd init --template todo-nodejs-mongo  # परियोजना सुरु गर्दछ
azd up                   # Azure मा प्रक्षेपण गर्दछ
azd show                 # चलिरहेको अनुप्रयोग URL देखाउँछ
azd down --force --purge # स्रोतहरू सफा गर्दछ
```
