# अध्याय १: आधार र छिटो सुरु

**📚 कोर्स**: [शुरुआतीका लागि AZD](../../README.md) | **⏱️ अवधि**: ३०-४५ मिनेट | **⭐ जटिलता**: शुरुवाती

---

## अवलोकन

यस अध्यायले Azure Developer CLI (azd) का आधारभूत कुराहरू परिचय गराउँछ। तपाईंले मुख्य अवधारणाहरू सिक्नुहुनेछ, उपकरणहरू इन्स्टल गर्नुहुनेछ, र Azure मा आफ्नो पहिलो अनुप्रयोग डिप्लोय गर्नुहुनेछ।

> मार्च २०२६ मा `azd 1.23.12` सँग प्रमाणित गरिएको।

## सिकाइका उद्देश्यहरू

यस अध्याय पूरा गरेर, तपाईंले:
- Azure Developer CLI के हो र Azure CLI बाट कसरी फरक हुन्छ बुझ्नुहुनेछ
- तपाईंको प्लेटफर्ममा AZD इन्स्टल र कन्फिगर गर्न सक्नुहुनेछ
- `azd up` सँग Azure मा आफ्नो पहिलो अनुप्रयोग डिप्लोय गर्न सक्नुहुनेछ
- `azd down` सँग स्रोतहरू सफा गर्न सक्नुहुनेछ

## ✅ यहाँबाट सुरु गर्नुहोस्: तपाईंको सेटअप प्रमाणित गर्नुहोस्

सुरु गर्नु अघि, सुनिश्चित गर्नुहोस् कि तपाईंको स्थानीय मेसिन अध्याय १ टेम्प्लेटका लागि तयार छ:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

यदि स्क्रिप्टले हराएका उपकरणहरू रिपोर्ट गर्छ भने, पहिले ती समाधान गर्नुहोस् र त्यसपछि अध्याय जारी राख्नुहोस्।

---

## 🚀 छिटो सुरु

```bash
# स्थापना जाँच गर्नुहोस्
azd version

# AZD को लागि प्रमाणीकरण गर्नुहोस्
# वैकल्पिक: यदि तपाईंले Azure CLI आदेशहरू सिधै चलाउने योजना गर्नुभएको छ भने az लगइन गर्नुहोस्
azd auth login

# तपाईंको पहिलो एप डिप्लोय गर्नुहोस्
azd init --template todo-nodejs-mongo
azd up

# सकेपछि सफा गर्नुहोस्
azd down --force --purge
```

---

## ✅ सफलताको मापदण्ड

यस अध्याय पूरा गरेपछि, तपाईंले निम्न कार्यहरू गर्न सक्नुपर्छ:

```bash
azd version              # स्थापना गरिएको संस्करण देखाउँछ
azd init --template todo-nodejs-mongo  # परियोजना आरम्भ गर्दछ
azd up                   # Azure मा परिनियोजन गर्दछ
azd show                 # चलिरहेको अनुप्रयोग URL प्रदर्शन गर्दछ
azd down --force --purge # स्रोतहरू सफा गर्दछ
```
