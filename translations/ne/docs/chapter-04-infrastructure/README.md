# अध्याय ४: इन्फ्रास्ट्रक्चर एज कोड र डिप्लोइमेन्ट

**📚 कोर्स**: [AZD For Beginners](../../README.md) | **⏱️ अवधि**: १-१.५ घण्टा | **⭐ जटिलता**: मध्यम

---

## अवलोकन

यस अध्यायमा Bicep टेम्प्लेट, स्रोत संसाधन उपलब्ध गराउने, र Azure Developer CLI प्रयोग गरेर डिप्लोइमेन्ट रणनीतिहरू सहित Infrastructure as Code (IaC) ढाँचाहरू समेटिएका छन्।

> मार्च २०२६ मा `azd 1.23.12` विरुद्ध प्रमाणित।

## अध्ययन उद्देश्यहरू

यस अध्याय पूरा गरेपछि, तपाईँले:
- Bicep टेम्प्लेटको संरचना र वाक्य रचना बुझ्नु हुनेछ
- `azd provision` प्रयोग गरेर Azure स्रोतहरू उपलब्ध गराउन सक्नु हुनेछ
- `azd deploy` प्रयोग गरेर अनुप्रयोगहरू डिप्लोइ गर्न सक्नु हुनेछ
- ब्लु-ग्रिन र रोलिंग डिप्लोइमेन्ट रणनीतिहरू कार्यान्वयन गर्न सक्नु हुनेछ

## 🚀 छिटो सुरु

```bash
# टेम्प्लेटबाट सुरु गर्नुहोस्
azd init --template azure-functions-python-v2-http

# के सिर्जना गरिनेछ preview गर्नुहोस्
azd provision --preview

# केवल पूर्वाधार प्रावधान गर्नुहोस्
azd provision

# केवल कोड तैनाथ गर्नुहोस्
azd deploy

# वा दुवै सँगै
azd up
```

## 📁 AZD प्रोजेक्ट संरचना

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

## 🔧 आवश्यक आदेशहरू

| आदेश | विवरण |
|---------|-------------|
| `azd init` | प्रोजेक्ट सुरु गर्ने |
| `azd provision` | Azure स्रोतहरू सिर्जना गर्ने |
| `azd deploy` | अनुप्रयोग कोड डिप्लोइ गर्ने |
| `azd up` | उपलब्ध गराउने + डिप्लोइ गर्ने |
| `azd down` | सबै स्रोतहरू मेटाउने |
