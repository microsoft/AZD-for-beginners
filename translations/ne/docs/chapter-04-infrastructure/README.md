# अध्याय ४: इन्फ्रास्ट्रक्चर एज कोड र डिप्लोयमेन्ट

**📚 पाठ्यक्रम**: [AZD लागि प्रारम्भिक](../../README.md) | **⏱️ अवधि**: १-१.५ घण्टा | **⭐ जटिलता**: मध्यम

---

## अवलोकन

यस अध्यायमा Bicep टेम्प्लेटहरू सहित Infrastructure as Code (IaC) ढाँचाहरू, स्रोत साधनहरू उपलब्ध गराउने, र Azure Developer CLI प्रयोग गरेर डिप्लोयमेन्ट रणनीतिहरू समेटिएका छन्।

> `azd 1.27.1` मा जुलाई २०२६ मा प्रमाणीकरण गरिएको।

## सिक्ने उद्देश्यहरू

यस अध्याय पूरा गरेपछि, तपाईं:
- Bicep टेम्प्लेट संरचना र सिन्ट्याक्स बुझ्ने
- `azd provision` संग Azure स्रोत साधनहरू उपलब्ध गराउने
- `azd deploy` प्रयोग गरेर अनुप्रयोगहरू डिप्लोय गर्ने
- निलो-हरियो र रोलिङ डिप्लोयमेन्ट रणनीतिहरू लागू गर्ने

## 🚀 छिटो सुरु

```bash
# ढाँचा बाट आरम्भ गर्नुहोस्
azd init --template azure-functions-python-v2-http

# के सिर्जना हुनेछ पूर्वावलोकन गर्नुहोस्
azd provision --preview

# केवल पूर्वधारण संरचना व्यवस्था गर्नुहोस्
azd provision

# केवल कोड तैनाथ गर्नुहोस्
azd deploy

# वा दुबै सँगै
azd up
```

## 📁 AZD प्रोजेक्ट संरचना

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

## 🔧 आवश्यक कमाण्डहरू

| कमाण्ड | विवरण |
|---------|-------------|
| `azd init` | प्रोजेक्ट सुरु गर्ने |
| `azd provision` | Azure स्रोतहरू सिर्जना गर्ने |
| `azd deploy` | अनुप्रयोग कोड डिप्लोय गर्ने |
| `azd up` | स्रोतहरू उपलब्ध गराउने + डिप्लोय गर्ने |
| `azd down` | सबै स्रोतहरू मेटाउने |
