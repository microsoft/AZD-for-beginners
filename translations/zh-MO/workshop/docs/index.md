歡迎參加專注於 AI 應用程式部署的 Azure Developer CLI (AZD) 實作工作坊。此工作坊助您通過 3 個步驟對 AZD 範本獲得應用理解：

1. <strong>探索</strong> - 找出適合您的範本。
1. <strong>部署</strong> - 部署並驗證其功能
1. <strong>自訂</strong> - 修改並持續迭代以符合您的需求！

在工作坊過程中，您還將接觸到核心開發工具和工作流程，幫助您優化端對端的開發歷程。

## 工作坊概覽

**時長：** 3-4 小時  
**程度：** 初學者至中階  
**先決條件：** 熟悉 Azure、AI 概念、VS Code 及命令列工具。

這是一個讓您透過實作學習的工作坊。完成練習後，我們建議回顧 AZD For Beginners 課程內容，繼續學習安全與生產力的最佳實務。

| 時間| 模組  | 目標 |
|:---|:---|:---|
| 15 分鐘 | 介紹 | 建立基調，了解目標 |
| 30 分鐘 | 選擇 AI 範本 | 探索選項並選擇入門範本 | 
| 30 分鐘 | 驗證 AI 範本 | 部署預設解決方案到 Azure |
| 30 分鐘 | 解構 AI 範本 | 探索結構和配置 |
| 30 分鐘 | 配置 AI 範本 | 啟用並嘗試可用功能 |
| 30 分鐘 | 自訂 AI 範本 | 根據需求調整範本 |
| 30 分鐘 | 拆除基礎建設 | 清理並釋放資源 |
| 15 分鐘 | 總結與後續步驟 | 學習資源，工作坊挑戰 |
| | |

## 您將學習到什麼

請將 AZD 範本視為探索 Microsoft Foundry 端對端開發各種功能和工具的學習沙盒。完成本工作坊後，您應該對相關的多種工具和概念有直觀的認識。

| 概念  | 目標 |
|:---|:---|
| **Azure Developer CLI** | 了解工具指令和工作流程|
| **AZD 範本**| 了解專案結構和配置|
| **Azure AI Agent**| 預配和部署 Microsoft Foundry 專案  |
| **Azure AI Search**| 啟用代理的上下文工程 |
| <strong>可觀察性</strong>| 探索追蹤、監控和評估 |
| <strong>紅隊測試</strong>| 探索對抗性測試與緩解措施 |
| | |
