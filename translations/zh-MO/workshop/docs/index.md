# AZD for AI Developers Workshop

> [!IMPORTANT]  
> **本工作坊附有一個可以在本機瀏覽器預覽的工作坊指南。要開始，您必須在此儲存庫上啟動 GitHub Codespaces──然後等到看到一個活動的 VS Code 終端機並輸入：**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **您應該會看到一個彈出對話方塊，提示在瀏覽器中開啟預覽頁面。**

歡迎參加以 AI 應用部署為重點的 Azure Developer CLI (AZD) 實作工作坊。本工作坊幫助您透過 3 個步驟實際瞭解 AZD 範本：

1. **Discovery** - 找到最適合您的範本。
1. **Deployment** - 部署並驗證其運作
1. **Customization** - 修改並反覆調整，使之成為您的專案！

在本工作坊過程中，您也會接觸到核心開發工具與工作流程，幫助您精簡端對端的開發旅程。

## Workshop Overview

**時長:** 3-4 小時  
**等級:** 初級到中級  
**先決條件:** 熟悉 Azure、AI 概念、VS Code 與命令列工具。

這是一個透過實作學習的工作坊。完成練習後，我們建議您回顧 AZD 新手指南 課程，以繼續學習安全性與提升生產力的最佳實務。

| Time| Module  | Objective |
|:---|:---|:---|
| 15 mins | Introduction | 建立背景，了解目標 |
| 30 mins | Select AI Template | 瀏覽選項並選擇入門範本 | 
| 30 mins | Validate AI Template | 將預設解決方案部署到 Azure |
| 30 mins | Deconstruct AI Template | 探索結構與設定 |
| 30 mins | Configure AI Template | 啟用並嘗試可用功能 |
| 30 mins | Customize AI Template | 根據需求調整範本 |
| 30 mins | Teardown Infrastructure | 清理並釋放資源 |
| 15 mins | Wrap-Up & Next Steps | 學習資源、工作坊挑戰 |
| | |

## What You'll Learn

把 AZD 範本當作一個學習沙箱，用來探索在 Microsoft Foundry 上進行端對端開發的各種功能與工具。完成本工作坊後，您應該能對此情境中的各種工具與概念有直觀的理解。

| Concept  | Objective |
|:---|:---|
| **Azure Developer CLI** | 了解工具指令與工作流程|
| **AZD Templates**| 了解專案結構與設定|
| **Azure AI Agent**| 佈建並部署 Microsoft Foundry 專案  |
| **Azure AI Search**| 使用代理啟用情境工程 |
| **Observability**| 探索追蹤、監控與評估 |
| **Red Teaming**| 探索對抗性測試與緩解措施 |
