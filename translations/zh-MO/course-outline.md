# AZD 初學者課程：課程大綱及學習框架

## 課程概述

透過結構化章節掌握 Azure Developer CLI (azd)，**特別著重於整合 Microsoft Foundry 部署 AI 應用程式**。

### 為什麼這課程對現代開發者至關重要

根據 Microsoft Foundry Discord 社群的洞察，**45% 的開發者希望使用 AZD 來處理 AI 工作負載**，但面臨以下挑戰：
- 複雜的多服務 AI 架構
- 生產環境 AI 部署的最佳實踐  
- Azure AI 服務的整合與配置
- AI 工作負載的成本優化
- AI 特定部署問題的故障排除

### 核心學習目標

完成這門結構化課程後，您將能：
- **掌握 AZD 基礎知識**：核心概念、安裝及配置
- **部署 AI 應用程式**：使用 AZD 與 Microsoft Foundry 服務
- **實現基礎架構即代碼**：使用 Bicep 模板管理 Azure 資源
- **故障排除部署問題**：解決常見問題並進行調試
- **優化生產環境**：安全性、擴展性、監控及成本管理
- **構建多代理解決方案**：部署複雜的 AI 架構

#### 🚀 自學模式
**適合個人開發者及持續學習**

**特色：**
- **基於瀏覽器的介面**：透過任何網頁瀏覽器完成 MkDocs 驅動的工作坊

**學習結構：**
