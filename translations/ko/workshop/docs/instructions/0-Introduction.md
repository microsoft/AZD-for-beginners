# AI 개발자를 위한 AZD

!!! info "이 워크숍 대상"
   
    이 워크숍은 최첨단 도구와 워크플로를 활용하여 Microsoft Foundry에서 AI 에이전트 애플리케이션을 구축하려는 모든 사람을 위한 것입니다. 이 워크숍은 Azure, AI 개념 및 Python 코드에 대해 어느 정도 알고 있는 초보자와 경험 있는 개발자 모두에게 적합합니다. 시작하기 전에 사전 필수 조건을 확인하세요.

    **소요 시간:** 3-4 시간  
    **수준:** 초급에서 중급  

---


## 사전 필수 조건

- 개인 GitHub 계정 → [무료로 만들기](https://github.com/signup)
- 활성 Azure 계정 → [무료로 만들기](https://aka.ms/free)
- AI 개념, Python 개발, 명령줄 도구에 대한 친숙함
- VS Code 및 GitHub 도구와 워크플로 경험

---

## 학습 목표

이 워크숍이 끝날 때쯤 다음을 알게 됩니다:

- [X] **Azure Developer CLI 사용** → Azure 리소스를 프로비저닝, 배포 및 관리하기
- [X] **AI 앱 템플릿 사용** → Microsoft Foundry에서 AI 에이전트 프로젝트를 빠르게 시작하기
- [X] **템플릿 구성** → 레드 팀, 평가, 추적 및 모니터링 같은 기능 활성화하기
- [X] **템플릿 분해 및 탐색** → GitHub Copilot을 사용하여 AZD 템플릿과 워크플로 살펴보기
- [X] **맞춤 설정 및 재배포** → 맞춤 요구사항으로 AI 에이전트 시작 템플릿 수정하기

---

## 워크숍 구성

워크숍은 각 모듈이 실습 랩 연습으로 구성되어 있습니다:

1. **소개** - AZD 템플릿이 무엇인지, 어떻게 작동하는지 이해하기
1. **선택** - 다음 AI 프로젝트에 적합한 "스타터" 템플릿 찾기
1. **검증** - 템플릿 배포 및 탐색하여 정상 작동 여부 확인하기
1. **분해** - 템플릿을 분해하여 구성 요소 및 설정 이해하기
1. **구성** - 다양한 Microsoft Foundry 기능 활성화 및 사용법 배우기
1. **맞춤화** - 자신의 데이터와 요구사항을 반영하여 애플리케이션 재구성하기
1. **해체** - 인프라 프로비저닝 해제 및 소프트 삭제된 리소스 복구 배우기
