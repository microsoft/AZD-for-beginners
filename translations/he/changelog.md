## מטרות הלמידה

על ידי סקירת יומן השינויים הזה, תוכלו:
- להישאר מעודכנים בתכונות חדשות והוספות תוכן
- להבין שיפורים שבוצעו בתיעוד הקיים
- לעקוב אחר תיקוני באגים ותיקונים להבטחת דיוק
- לעקוב אחר התפתחות חומרי הלמידה לאורך זמן

## תוצאות הלמידה

לאחר סקירת הערכים ביומן השינויים, תוכלו:
- לזהות תוכן ומשאבים חדשים הזמינים ללמידה
- להבין אילו חלקים עודכנו או שופרו
- לתכנן את מסלול הלמידה שלכם בהתבסס על החומרים העדכניים ביותר
- להציע משוב והצעות לשיפורים עתידיים

### [v3.10.0] - 05-02-2026

#### עדכון דיוק פקודות Azure Developer CLI
**גרסה זו מתקנת פקודות AZD שאינן קיימות בתיעוד, ומוודאת שכל דוגמאות הקוד משתמשות בתחביר תקין של Azure Developer CLI.**

#### תוקן
- **🔧 הסרת פקודות AZD לא קיימות**: ביקורת נרחבת ותיקון פקודות שגויות:  
  - `azd logs` (לא קיים) → הוחלף ב-`azd monitor --logs` או חלופות Azure CLI  
  - פקודות משנה של `azd service` (לא קיימות) → הוחלפו ב-`azd show` וב-Azure CLI  
  - `azd infra import/export/validate` (לא קיימות) → הוסרו או הוחלפו באלטרנטיבות תקינות  
  - דגלי `azd deploy --rollback/--incremental/--parallel/--detect-changes` (לא קיימים) → הוסרו  
  - דגלי `azd provision --what-if/--rollback` (לא קיימים) → עודכנו לשימוש ב-`--preview`  
  - `azd config validate` (לא קיימת) → הוחלף ב-`azd config list`  
  - `azd info`, `azd history`, `azd metrics` (לא קיימות) → הוסרו  

- **📚 קבצים עודכנו עם תיקוני פקודות**:

  - `docs/getting-started/azd-basics.md`: תיקוני פקודות ניטור

- **גרסה ראשית קודמת**: עדכוני אבטחה ותיקוני שגיאות למשך 12 חודשים  
- **גרסאות ישנות**: תמיכה קהילתית בלבד, ללא עדכונים רשמיים  

### הנחיות הגירה  
כששוחררו גרסאות ראשיות, אנו מספקים:
- **מדריכי הגירה**: הוראות מעבר שלב אחר שלב  
- **הערות תאימות**: פרטים על שינויים מחייבים  
- **תמיכה בכלים**: סקריפטים או כלים לסיוע בהגירה  
- **תמיכה קהילתית**: פורומים ייעודיים לשאלות על הגירה
