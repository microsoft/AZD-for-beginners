## 🚀 התחלה מהירה

```bash
# פרוס את הפתרון הרב-סוכני לקמעונאות
cd examples/retail-multiagent-arm-template
./deploy.sh

# או השתמש בתבנית ישירות
azd init --template agent-openai-python-prompty
azd up
```

└────────────────────┬─────────────────┬───────────────────────┘
                     │                 │
         ┌───────────▼───────┐ ┌───────▼───────────┐
         │  Customer Agent   │ │  Inventory Agent  │

         │   preferences)    │ │   orders)         │
         └───────────────────┘ └───────────────────┘

## 🎯 פתרון מוצג: פתרון רב-סוכני לקמעונאות

ה[פתרון רב-סוכני לקמעונאות](../../examples/retail-scenario.md) מדגים:

- **סוכן לקוח**: מטפל באינטראקציות עם המשתמש ובהעדפותיו
- **סוכן מלאי**: מנהל מלאי ועיבוד הזמנות
- **אורקסטרטור**: מתאם בין הסוכנים
- **זיכרון משותף**: ניהול הקשר בין הסוכנים

### שירותים בשימוש

| Service | Purpose |
|---------|---------|
| Azure OpenAI | הבנת שפה |
| Azure AI Search | קטלוג מוצרים |
| Cosmos DB | מצב וזיכרון סוכן |
| Container Apps | אירוח סוכנים |
| Application Insights | ניטור |
