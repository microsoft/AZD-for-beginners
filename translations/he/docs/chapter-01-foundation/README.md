# פרק 1: יסודות והתחלה מהירה

**📚 קורס**: [AZD For Beginners](../../README.md) | **⏱️ משך**: 30-45 דקות | **⭐ מורכבות**: למתחילים

---

## סקירה כללית

בפרק זה תכירו את יסודות Azure Developer CLI (azd). תלמדו מושגים מרכזיים, תתקינו את הכלים ותפרסמו את היישום הראשון שלכם ל-Azure.

## מטרות הלמידה

בסיום פרק זה, תוכלו:
- להבין מהי Azure Developer CLI וכיצד היא שונה מ-Azure CLI
- להתקין ולהגדיר את AZD בפלטפורמה שלכם
- לפרוס את היישום הראשון שלכם ל-Azure עם `azd up`
- לנקות משאבים עם `azd down`

## 🚀 התחלה מהירה

```bash
# בדוק את ההתקנה
azd version

# התחבר ל-Azure
azd auth login

# פרוס את האפליקציה הראשונה שלך
azd init --template todo-nodejs-mongo
azd up

# נקה כאשר תסיים
azd down --force --purge
```

---

## ✅ קריטריונים להצלחה

בסיום פרק זה, אתם אמורים להיות מסוגלים:

```bash
azd version              # מציג את הגרסה המותקנת
azd init --template todo-nodejs-mongo  # מאתחל את הפרויקט
azd up                   # מפרסם ל-Azure
azd show                 # מציג את כתובת ה-URL של האפליקציה הפועלת
azd down --force --purge # מנקה את המשאבים
```
