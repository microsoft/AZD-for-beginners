# פרק 1: יסודות והתחלה מהירה

**📚 קורס**: [AZD למתחילים](../../README.md) | **⏱️ משך זמן**: 30-45 דקות | **⭐ רמת קושי**: מתחיל

---

## סקירה כללית

פרק זה מציג את יסודות Azure Developer CLI (azd). תלמדו מושגי יסוד, תתקינו את הכלים ותפרסמו את היישום הראשון שלכם ב-Azure.

> אומת נגד `azd 1.23.12` במרץ 2026.

## מטרות הלמידה

לאחר השלמת פרק זה, תוכלו:
- להבין מהו Azure Developer CLI וכיצד הוא שונה מ-Azure CLI
- להתקין ולהגדיר את AZD בפלטפורמה שלכם
- לפרוס את היישום הראשון שלכם ב-Azure באמצעות `azd up`
- לנקות משאבים באמצעות `azd down`

## ✅ התחל כאן: אמת את ההגדרות שלך

לפני שתתחיל, וודא שמחשבך המקומי מוכן לשימוש בתבנית פרק 1:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

אם הסקריפט מדווח על כלים חסרים, תקן אותם תחילה ואז המשך בפרק.

---

## 🚀 התחלה מהירה

```bash
# בדוק התקנה
azd version

# אמת ל-AZD
# אופציונלי: az login אם אתה מתכנן להריץ פקודות Azure CLI ישירות
azd auth login

# פרוס את האפליקציה הראשונה שלך
azd init --template todo-nodejs-mongo
azd up

# נקה כאשר סיימת
azd down --force --purge
```

---

## ✅ קריטריוני הצלחה

לאחר השלמת פרק זה, עליך להיות מסוגל:

```bash
azd version              # מציג את הגרסה המותקנת
azd init --template todo-nodejs-mongo  # מאתחל את הפרויקט
azd up                   # מפרסם ל-Azure
azd show                 # מציג את כתובת האתר של האפליקציה הרצה
azd down --force --purge # מנקה משאבים
```
