# פרק 1: יסודות והתחלה מהירה

**📚 קורס**: [AZD למתחילים](../../README.md) | **⏱️ משך זמן**: 30-45 דקות | **⭐ רמת קושי**: מתחילים

---

## מבוא

פרק זה מציג את יסודות Azure Developer CLI (azd). תלמדו את המושגים המרכזיים, תתקינו את הכלים ותפרסמו את היישום הראשון שלכם ב-Azure.

> אומת נגד `azd 1.25.6` ביוני 2026.

## מטרות למידה

בסיום פרק זה, תוכלו:
- להבין מהו Azure Developer CLI וכיצד הוא שונה מ-Azure CLI
- להתקין ולהגדיר את AZD בפלטפורמה שלכם
- לפרוס את היישום הראשון שלכם ב-Azure באמצעות `azd up`
- לנקות משאבים עם `azd down`

## ✅ התחל כאן: אמת את ההתקנה שלך

לפני שתתחיל, ודא שהמחשב המקומי שלך מוכן לתבנית פרק 1:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

אם התסריט מדווח על כלים חסרים, תקן אותם תחילה ואז המשך בפרק.

---

## 🚀 התחלה מהירה

```bash
# בדוק התקנה
azd version

# אמת עבור AZD
# אופציונלי: היכנס באמצעות az אם אתם מתכננים להריץ פקודות Azure CLI ישירות
azd auth login

# פרוס את האפליקציה הראשונה שלך
azd init --template todo-nodejs-mongo
azd up

# נקה כאשר סיימת
azd down --force --purge
```

---

## ✅ קריטריוני הצלחה

לאחר סיום פרק זה, עליך להיות מסוגל:

```bash
azd version              # מציג את הגרסה המותקנת
azd init --template todo-nodejs-mongo  # מאתחל את הפרויקט
azd up                   # מפעיל פריסה ל-Azure
azd show                 # מציג את כתובת ה-URL של האפליקציה הפועלת
azd down --force --purge # מנקה את המשאבים
```
