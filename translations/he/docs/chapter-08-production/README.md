# פרק 8: תבניות ייצור וארגוניות

**📚 קורס**: [AZD למתחילים](../../README.md) | **⏱️ משך**: 2-3 שעות | **⭐ מורכבות**: מתקדם

---

## סקירה כללית

פרק זה עוסק בתבניות פריסה מוכנות לארגון, החמרת אבטחה, ניטור ואופטימיזציה של עלויות לעומסי עבודה של AI בסביבת ייצור.

## יעדי הלמידה

בסיום פרק זה, תוכל/י:
- לפרוס יישומים חסינים במספר אזורים
- ליישם דפוסי אבטחה ארגוניים
- להגדיר ניטור מקיף
- לאופטימיזציה של עלויות בקנה מידה
- להגדיר צינורות CI/CD עם AZD

## 🔐 שיטות אבטחה מומלצות

```bicep
// Use managed identity
identity: {
  type: 'SystemAssigned'
}

// Private endpoints for AI services
properties: {
  publicNetworkAccess: 'Disabled'
  networkAcls: {
    defaultAction: 'Deny'
  }
}
```

## 💰 אופטימיזציה של עלויות

| אסטרטגיה | חסכון |
|----------|---------|
| קנה מידה לאפס (Container Apps) | 60-80% |
| שימוש בשכבות צריכה לפיתוח | 50-70% |
| קנה מידה מתוזמן | 30-50% |
| קיבולת שמורה | 20-40% |

```bash
# הגדר התראות על התקציב
az consumption budget create \
  --budget-name "AI-Budget" \
  --amount 500 \
  --category Cost \
  --time-grain Monthly
```
