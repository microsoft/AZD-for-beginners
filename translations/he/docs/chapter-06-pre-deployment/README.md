# פרק 6: תכנון ואימות לפני פריסה

**📚 קורס**: [AZD למתחילים](../../README.md) | **⏱️ משך**: שעה אחת | **⭐ מורכבות**: בינונית

## סקירה כללית

פרק זה מכסה שלבים חיוניים של תכנון ואימות לפני פריסת האפליקציה שלך. למד כיצד להימנע משגיאות יקרות באמצעות תכנון קיבולת נכון, בחירת SKU ובדיקות מקדימות.

## מטרות למידה

בסיום פרק זה, תוכל:
- להריץ בדיקות מקדימות לפני פריסה
- לתכנן קיבולת ולהעריך דרישות משאבים
- לבחור SKU מתאימים לאופטימיזציית עלויות
- לקנפג Application Insights לניטור
- להבין דפוסי תיאום צוותים

## 🚀 התחלה מהירה

```bash
# בדוק מכסות מינוי
az vm list-usage --location eastus --output table

# תצוגה מקדימה של פריסה (ללא יצירת משאבים)
azd provision --preview

# אמת את תחביר ה-Bicep
az bicep build --file infra/main.bicep

# בדוק הגדרות סביבה
azd env get-values
```

## ☑️ רשימת בדיקות לפני פריסה

### לפני `azd provision`

- [ ] אימות מכסת משאבים לאזור
- [ ] בחירת SKU מתאימות
- [ ] סקירת הערכת עלויות
- [ ] עקביות בשמות
- [ ] קונפיגורציית אבטחה/RBAC

### לפני `azd deploy`

- [ ] קביעת משתני סביבה
- [ ] סודות ב-Key Vault
- [ ] אימות מחרוזות חיבור
- [ ] קונפיגורציית בדיקות בריאות
