# פרק 6: תכנון ואימות לפני פריסה

**📚 קורס**: [AZD למתחילים](../../README.md) | **⏱️ משך**: שעה אחת | **⭐ מורכבות**: בינוני

---

## סקירה כללית

פרק זה מכסה שלבים חיוניים בתכנון ואימות לפני פריסת האפליקציה שלך. למד איך להימנע מטעויות יקרות עם תכנון קיבולת נכון, בחירת SKU ובדיקות מקדימות.

> אומת נגד `azd 1.25.6` ביוני 2026.

## מטרות הלמידה

בסיום פרק זה, תוכל:
- להריץ בדיקות מקדימות לפני הפריסה
- לתכנן קיבולת ולהעריך דרישות משאבים
- לבחור SKU מותאם לאופטימיזציה של עלויות
- לקנפג Application Insights למעקב
- להבין דפוסי תיאום צוותים

## 🚀 התחלה מהירה

```bash
# בדוק מכסות מנוי
az vm list-usage --location eastus --output table

# תצוגה מקדימה של פריסה (ללא יצירת משאבים)
azd provision --preview

# אמת תחביר של Bicep
az bicep build --file infra/main.bicep

# בדוק תצורת סביבה
azd env get-values
```

## ☑️ רשימת בדיקות לפני פריסה

### לפני `azd provision`

- [ ] אחוזה מאומתת לאזור
- [ ] SKU נבחרו בהתאם
- [ ] הערכת עלות נבדקה
- [ ] קונבנציה לשמות עקבית
- [ ] אבטחה/RBAC מוקנפגים

### לפני `azd deploy`

- [ ] משתני סביבה מוגדרים
- [ ] סודות ב-Key Vault
- [ ] מחרוזות קישור מאומתות
- [ ] בדיקות בריאות מוקנפגות
