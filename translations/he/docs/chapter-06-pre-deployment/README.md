# פרק 6: תכנון ואימות לפני פריסה

**📚 קורס**: [AZD למתחילים](../../README.md) | **⏱️ משך**: שעה | **⭐ מורכבות**: בינוני

---

## סקירה כללית

פרק זה מכסה את שלבי התכנון והאימות החיוניים לפני פריסת האפליקציה שלך. למד כיצד להימנע מטעויות יקרות עם תכנון קיבולת נאות, בחירת SKU ובדיקות מקדימות.

> אומת ב־`azd 1.27.1` ביולי 2026.

## מטרות הלמידה

בסיום פרק זה תוכל:
- לבצע בדיקות מקדימות לפני הפריסה
- לתכנן קיבולת ולהעריך דרישות משאבים
- לבחור SKU מתאימים לאופטימיזציה של עלויות
- להגדיר Application Insights למעקב
- להבין דפוסי שיתוף פעולה בצוות

## 🚀 התחלה מהירה

```bash
# בדוק מכסות מנוי
az vm list-usage --location eastus --output table

# תצוגה מקדימה של פריסה (אין יצירת משאבים)
azd provision --preview

# אמת תחביר Bicep
az bicep build --file infra/main.bicep

# בדוק תצורת סביבה
azd env get-values
```

## ☑️ רשימת בדיקה לפני פריסה

### לפני `azd provision`

- [ ] אימות מכסה לאזור
- [ ] בחירת SKU מתאימה
- [ ] סקירת הערכת עלות
- [ ] עקביות בהסכמת שמות
- [ ] הגדרת אבטחה/RBAC

### לפני `azd deploy`

- [ ] הגדרת משתני סביבה
- [ ] סודות ב-Key Vault
- [ ] אימות מחרוזות חיבור
- [ ] הגדרת בדיקות בריאות
