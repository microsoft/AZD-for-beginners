# פרק 6: תכנון ואימות לפני פריסה

**📚 קורס**: [AZD למתחילים](../../README.md) | **⏱️ משך**: שעה אחת | **⭐ רמת קושי**: בינוני

## סקירה כללית

פרק זה מכסה שלבים חיוניים של תכנון ואימות לפני פריסת האפליקציה שלך. למד כיצד להימנע מטעויות יקרות באמצעות תכנון קיבולת נכון, בחירת SKU ובדיקות טרום טיסה.

> אומת נגד `azd 1.23.12` במרץ 2026.

## מטרות הלמידה

בסיום פרק זה, תוכל:
- לבצע בדיקות טרום טיסה לפני פריסה
- לתכנן קיבולת ולהעריך דרישות משאבים
- לבחור SKU מתאימים לאופטימיזציית עלויות
- להגדיר Application Insights למעקב
- להבין דפוסי תיאום צוותים

## 🚀 התחלה מהירה

```bash
# בדוק מכסי מנוי
az vm list-usage --location eastus --output table

# תצוגה מקדימה של פריסה (אין יצירת משאבים)
azd provision --preview

# אמת תחביר Bicep
az bicep build --file infra/main.bicep

# בדוק תצורת סביבה
azd env get-values
```

## ☑️ רשימת בדיקה לפני פריסה

### לפני `azd provision`

- [ ] אימות הקצאה לאזור
- [ ] בחירת SKU מתאימה
- [ ] סקירת הערכת עלויות
- [ ] עקביות בשיטת ההקראה
- [ ] הגדרות אבטחה/RBAC

### לפני `azd deploy`

- [ ] הגדרת משתני סביבה
- [ ] סיסמאות במפתח Key Vault
- [ ] אימות מיתרי חיבור
- [ ] הגדרת בדיקות בריאות
