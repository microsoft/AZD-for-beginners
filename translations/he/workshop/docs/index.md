# סדנת AZD למפתחים של בינה מלאכותית

> אומת עם `azd 1.25.6` ביוני 2026.

> [!IMPORTANT]  
> **סדנה זו מצוידת במדריך סדנה שניתן לצפות בו בדפדפן המקומי שלך. כדי להתחיל, פתח את המאגר ב-GitHub Codespaces, במכלול פיתוח, או בשכפול מקומי עם MkDocs מותקן—ואז המתן עד שתראה טרמינל פעיל ב-VS Code והקלד:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **עליך לראות חלון קופץ לפתיחת דף התצוגה המקדימה בדפדפן.**

ברוכים הבאים לסדנת המעבדה המעשית ללמידת Azure Developer CLI (AZD) עם דגש על פריסת יישומי בינה מלאכותית. סדנה זו תעזור לכם להשיג הבנה מעשית של תבניות AZD בשלושה שלבים:

1. **גילוי** - מצא את התבנית שמתאימה לך.  
1. **פריסה** - פרוס ואמת שהיא פועלת  
1. **התאמה אישית** - ערוך וחתור להפוך אותה לשלך!

במהלך הסדנה תיחשפו גם לכלים ולזרימות עבודה מרכזיים למפתחים, שיעזרו לכם לייעל את מסלול הפיתוח מקצה לקצה.

## סקירת הסדנה

**משך:** 3-4 שעות  
**רמה:** מתחילים עד בינוניים  
**דרישות מוקדמות:** היכרות עם Azure, מושגי בינה מלאכותית, VS Code וכלי שורת פקודה.

זוהי סדנת מעבדה מעשית שבה לומדים בעשייה. לאחר שתשלים את התרגילים, אנו ממליצים לעיין בתכנית הלימודים של AZD למתחילים כדי להמשיך במסע הלמידה שלך לעבר פרקטיקות אבטחה ופרודוקטיביות מיטביות.

| זמן | מודול | מטרה |
|:---|:---|:---|
| 15 דקות | מבוא | להציב את המסגרת, להבין את המטרות |
| 30 דקות | בחירת תבנית AI | לבחון אפשרויות ולבחור התחלה |
| 30 דקות | אימות תבנית AI | לפרוס פתרון ברירת מחדל ב-Azure |
| 30 דקות | ניתוח תבנית AI | לבחון מבנה וקונפיגורציה |
| 30 דקות | קביעת תצורה לתבנית AI | להפעיל ולנסות תכונות זמינות |
| 30 דקות | התאמת תבנית AI | להתאים את התבנית לצרכיך |
| 30 דקות | פירוק תשתית | לנקות ולשחרר משאבים |
| 15 דקות | סיכום והשלבים הבאים | משאבי למידה, אתגר הסדנה |
| | |

## מה תלמדו

חשבו על תבנית AZD כסנדבוקס ללמידה לחקור יכולות וכלים שונים לפיתוח מקצה לקצה ב-Microsoft Foundry. בסוף הסדנה, אמור להיות לכם תחושה אינטואיטיבית לגבי כלי ומושגים שונים בהקשר זה.

| מונח | מטרה |
|:---|:---|
| **Azure Developer CLI** | להבין פקודות וזרימות עבודה של הכלי |
| **תבניות AZD**| להבין מבנה פרויקט וקונפיגורציה |
| **Azure AI Agent**| להקצות ולפרוס פרויקט Microsoft Foundry |
| **Azure AI Search**| לאפשר הנדסת הקשר עם סוכנים |
| **תצפית**| לחקור מעקב, ניטור והערכות |
| **Red Teaming**| לחקור בדיקות עוינות וצעדי הקלה |
| | |
