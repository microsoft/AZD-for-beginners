# סדנת AZD למפתחי AI

> מאומת מול `azd 1.27.1` ביולי 2026.

> [!IMPORTANT]  
> **סדנה זו מצוידת במדריך סדנה שאפשר לצפות בו בדפדפן המקומי שלך. כדי להתחיל, פתח את הריפו ב-GitHub Codespaces, במיכל פיתוח, או בעותק מקומי עם התקנת MkDocs—ואז המתן עד שתראה טרמינל פעיל ב-VS Code והקלד:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **עליך לראות חלון קופץ לפתיחת דף התצוגה המקדימה בדפדפן.**

ברוך הבא לסדנה מעשית ללימוד Azure Developer CLI (AZD) עם התמקדות בפריסת יישומים של AI. הסדנה הזו עוזרת לך לקבל הבנה מעשית של תבניות AZD ב-3 שלבים:

1. **גילוי** - מצא את התבנית המתאימה לך.
1. **פריסה** - פרוס ולאמת שהיא פועלת
1. **התאמה אישית** - שנה וחזור כדי להפוך אותה לשלך!

במהלך הסדנה תכיר גם כלים וזרמי עבודה מרכזיים למפתחים, שיעזרו לך לייעל את מסלול הפיתוח מקצה לקצה.

## סקירת הסדנה

**משך:** 3-4 שעות  
**רמה:** מתחיל עד בינוני  
**דרישות מוקדמות:** היכרות עם Azure, מושגי AI, VS Code וכלי שורת פקודה.

זוהי סדנה מעשית שבה לומדים באמצעות עשייה. לאחר השלמת התרגילים, מומלץ לעיין בתכנית AZD למתחילים כדי להמשיך את מסע הלמידה שלך לעבר אבטחה ושיטות עבודה מיטביות לפרודוקטיביות.

| זמן| מודול  | מטרה |
|:---|:---|:---|
| 15 דקות | מבוא | להציב את הבמה, להבין את היעדים |
| 30 דקות | בחירת תבנית AI | לחקור אפשרויות ולבחור התחלה | 
| 30 דקות | אימות תבנית AI | לפרוס פתרון ברירת מחדל ל-Azure |
| 30 דקות | פירוק תבנית AI | לחקור מבנה וקונפיגורציה |
| 30 דקות | קונפיגורציית תבנית AI | להפעיל ולנסות תכונות זמינות |
| 30 דקות | התאמת תבנית AI | להתאים את התבנית לצרכים שלך |
| 30 דקות | פירוק תשתית | ניקוי ושחרור משאבים |
| 15 דקות | סיכום וצעדים הבאים | משאבי למידה, אתגר הסדנה |
| | |

## מה תלמד

חשוב על תבנית AZD כסביבת למידה לחקור יכולות וכלים שונים לפיתוח מקצה לקצה ב-Microsoft Foundry. בסוף הסדנה, אמור להיות לך תחושה אינטואיטיבית לכלים ולמושגים שונים בהקשר זה.

| מושג  | מטרה |
|:---|:---|
| **Azure Developer CLI** | להבין פקודות וזרמי עבודה של הכלי |
| **תבניות AZD**| להבין מבנה הפרויקט וקונפיגורציה |
| **סוכן Azure AI**| פריסה והקמה של פרויקט Microsoft Foundry  |
| **Azure AI Search**| לאפשר הנדסת הקשר עם סוכנים |
| **ניטור**| לחקור מעקב, ניטור והערכות |
| **בדיקות אדברסריות**| לחקור בדיקות וטיפולים תוקפניים |
| | |
