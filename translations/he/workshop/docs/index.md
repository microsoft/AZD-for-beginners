# סדנת AZD למפתחי AI

> אומת מול `azd 1.23.12` במרץ 2026.

> [!IMPORTANT]  
> **סדנה זו מצוידת במדריך סדנה שניתן לצפות בו בדפדפן המקומי שלך. כדי להתחיל, פתח את הרפוזיטורי ב-GitHub Codespaces, בקונטיינר פיתוח, או בשיבוט מקומי עם MkDocs מותקן — ואז המתן עד שתראה טרמינל VS Code פעיל והקלד:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **עליך לראות דו-שיח קופץ לפתיחת דף התצוגה בדפדפן.**

ברוכים הבאים לסדנה מעשית ללמידת Azure Developer CLI (AZD) עם דגש על פריסת יישומי AI. סדנה זו עוזרת לך לקבל הבנה מעשית של תבניות AZD ב-3 שלבים:

1. **גילוי** - מצא את התבנית שמתאימה לך.  
1. **פריסה** - פרוס ואמת שהכל עובד  
1. **התאמה אישית** - שייפה וחזור כדי להפוך את זה לשלך!

במהלך הסדנה תתוודע גם לכלי פיתוח מרכזיים ולזרימות עבודה, שיעזרו לך לייעל את מסע הפיתוח מאפס ועד סיום.

## סקירת הסדנה

**משך:** 3-4 שעות  
**רמה:** מתחילים עד בינוני  
**דרישות מוקדמות:** היכרות עם Azure, מושגי AI, VS Code וכלי שורת פקודה.

זו סדנה מעשית שבה לומדים מתוך ביצוע. לאחר שתסיים את התרגילים, מומלץ לעבור על התכנית "AZD למתחילים" כדי להמשיך ללמוד על אבטחה ושיטות עבודה מיטביות בתחום הפרודוקטיביות.

| זמן| מודול  | מטרה |
|:---|:---|:---|
| 15 דקות | מבוא | הצבת המסגרת, הבנת היעדים |
| 30 דקות | בחירת תבנית AI | חקור אפשרויות ובחר תבנית התחלה | 
| 30 דקות | אימות תבנית AI | פרוס פתרון ברירת מחדל ב-Azure |
| 30 דקות | ניתוח תבנית AI | חקור מבנה וקונפיגורציה |
| 30 דקות | קונפיגורציה של תבנית AI | הפעל ונסה תכונות זמינות |
| 30 דקות | התאמת תבנית AI | התאם את התבנית לצרכיך |
| 30 דקות | פירוק תשתית | נקה ושחרר משאבים |
| 15 דקות | סיכום ושלבים הבאים | משאבי למידה, אתגר הסדנה |
| | |

## מה תלמד

חשוב על תבנית AZD כמגרש למידה לחקור יכולות שונות וכלים לפיתוח מקצה לקצה על Microsoft Foundry. בסיום הסדנה, יהיה לך מושג אינטואיטיבי על כלים ומושגים שונים בהקשר זה.

| מושג  | מטרה |
|:---|:---|
| **Azure Developer CLI** | הבן פקודות וזרימות עבודה של הכלי|
| **תבניות AZD**| הבן מבנה הפרויקט והקונפיגורציה|
| **סוכן Azure AI**| הטמע ופרוס פרויקט Microsoft Foundry  |
| **חיפוש Azure AI**| אפשר הנדסת הקשר עם סוכנים |
| **אפשרויות תצפית**| חקור מעקב, ניטור והערכות |
| **בדיקות Red Teaming**| חקור בדיקות עוינות ופתרונות הפחתה |
| | |
