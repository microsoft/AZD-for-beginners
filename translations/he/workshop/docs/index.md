# סדנת AZD למפתחי בינה מלאכותית

> [!IMPORTANT]  
> **סדנה זו מצוידת במדריך סדנה שניתן לצפות בו בדפדפן המקומי שלך. כדי להתחיל, עליך להפעיל GitHub Codespaces על הרפו—ולאחר מכן להמתין עד שתראה טרמינל VS Code פעיל ולהקליד:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **עליך לראות חלון קופץ לפתיחת דף התצוגה המקדימה בדפדפן.**

ברוכים הבאים לסדנה מעשית ללימוד Azure Developer CLI (AZD) עם דגש על פריסה של יישומי בינה מלאכותית. סדנה זו תעזור לכם לרכוש הבנה יישומית של תבניות AZD ב-3 שלבים:

1. **גילוי** - מצא את התבנית המתאימה לך.
1. **פריסה** - הפץ ואמת שהיא פועלת
1. **התאמה אישית** - ערוך וחזור על התהליך כדי להפוך אותה לשלך!

במהלך הסדנה תכירו גם את כלי המפתח והזרימות עבודה הליבה, שיעזרו לכם לייעל את מסע הפיתוח מקצה לקצה.

## סקירת הסדנה

**משך:** 3-4 שעות  
**רמה:** מתחילים עד בינוניים  
**דרישות מוקדמות:** היכרות עם Azure, מושגי בינה מלאכותית, VS Code וכלי שורת פקודה.

זוהי סדנה מעשית שבה לומדים באמצעות ביצוע. לאחר שתסיים את התרגילים, מומלץ לעיין בתכנית הלימודים AZD למתחילים כדי להמשיך בלימוד על אבטחה ושיטות עבודה מיטביות לייצוריות.

| זמן| מודול  | מטרה |
|:---|:---|:---|
| 15 דקות | מבוא | להציב את הבמה, להבין את המטרות |
| 30 דקות | בחירת תבנית AI | לחקור אפשרויות ולבחור התחלתית | 
| 30 דקות | אימות תבנית AI | לפרוס פתרון ברירת מחדל ל-Azure |
| 30 דקות | פירוק תבנית AI | לחקור מבנה וקונפיגורציה |
| 30 דקות | הגדרת תבנית AI | להפעיל ולנסות תכונות זמינות |
| 30 דקות | התאמת תבנית AI | להתאים את התבנית לצרכיך |
| 30 דקות | פירוק תשתית | לנקות ולשחרר משאבים |
| 15 דקות | סיכום ושלבים הבאים | משאבים ללמידה, אתגר הסדנה |
| | |

## מה תלמדו

חשוב על תבנית AZD כסנדבוקס ללמידה לחקור יכולות שונות וכלים לפיתוח מקצה לקצה על Microsoft Foundry. בסיום הסדנה, אמורה להיות לכם הבנה אינטואיטיבית של כלים ומושגים שונים בהקשר זה.

| מושג  | מטרה |
|:---|:---|
| **Azure Developer CLI** | להבין פקודות וכלי עבודה |
| **תבניות AZD**| להבין מבנה פרויקט והגדרות |
| **Azure AI Agent**| פריסת פרויקט Microsoft Foundry |
| **Azure AI Search**| לאפשר הנדסת הקשר עם סוכנים |
| **ניתור**| לחקור מעקב, ניטור והערכות |
| **Red Teaming**| לחקור בדיקות אויבים ודרכי הגנה |
| | |
