# AZD למפתחים של בינה מלאכותית

!!! info "קהל היעד לסדנה זו"
   
    סדנה זו מיועדת לכל מי שמעוניין לבנות יישומי AI Agentic על Microsoft Foundry באמצעות ניצול כלים וזרימות עבודה מהשורה הראשונה. הסדנה מתאימה למתחילים ולמפתחים מנוסים שיש להם היכרות מסוימת עם Azure, מושגי AI וקוד Python. אנא בדקו את הדרישות המקדימות לפני שתתחילו.

    **משך:** 3-4 שעות  
    **רמה:** מתחילים עד ביניים  

---


## דרישות מקדימות

- חשבון GitHub פרטי → [צרו חשבון בחינם](https://github.com/signup)
- חשבון Azure פעיל → [צרו חשבון בחינם](https://aka.ms/free)
- היכרות עם מושגי AI, פיתוח ב-Python, כלי שורת פקודה
- ניסיון עם VS Code וכלי וזרימות עבודה של GitHub.

---

## יעדי למידה

בסיום סדנה זו תדעו כיצד:

- [X] **לעבוד עם Azure Developer CLI** → לפרוס, להפעיל ולנהל את משאבי Azure שלכם.
- [X] **להשתמש בתבניות אפליקציה של AI** → כדי להתחיל פרויקטים של AI Agent ב-Microsoft Foundry.
- [X] **להגדיר תבניות** → עבור תכונות כמו Red Teaming, הערכות, מעקב וניטור
- [X] **לפענח ולחקור** → תבניות וזרימות עבודה של AZD באמצעות GitHub Copilot
- [X] **להתאים אישית ולפרוס מחדש** → את תבנית ההתחלה של AI Agent, עם הדרישות המותאמות לכם

---

## מתווה הסדנה

הסדנה בנויה כמודולים, כל אחד מהם נתמך על ידי תרגיל מעשי במעבדה:

1. **מבוא** - לקבל הבנה מהן תבניות AZD, וכיצד הן פועלות
1. **בחירה** - לגלות את תבנית ה"מתחילים" הנכונה לפרויקט ה-AI הבא שלך
1. **אימות** - לפרוס ולחקור את התבנית כדי לוודא שהיא פועלת כפי שפורסם
1. **פירוק** - לפרק את התבנית כדי להבין את הרכיבים וההגדרות שלה
1. **הגדרה** - ללמוד להפעיל ולהשתמש בתכונות שונות של Microsoft Foundry
1. **התאמה אישית** - לבנות מחדש את האפליקציה עם הנתונים והדרישות שלכם
1. **פירוק** - ללמוד להפטר מתשתיות ולשחזר משאבים שנמחקו ברכות
