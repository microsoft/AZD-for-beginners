## 2. מושגי מפתח שרכשתם

### Azure Developer CLI (azd)
- `azd init` - אתחול פרויקטים מתבניות
- `azd up` - הקצאה ופריסה בפקודה אחת
- `azd env` - ניהול משתני סביבה
- `azd down` - ניקוי משאבים בצורה בטוחה

### מבנה תבניות AZD
- `azure.yaml` - תצורה וקריאות פריסה
- `infra/` - תשתית ב-Bicep כקוד
- `src/` - קוד מקור של האפליקציה
- `scripts/` - סקריפטים לאוטומציה והגדרה

### יכולות Microsoft Foundry
- שירות סוכני AI לעוזרים חכמים
- חיפוש קבצים ו-Azure AI Search לשליפת ידע
- מעקב, ניטור והערכה מובנים
- בדיקות Red teaming לאבטחת AI

## 3. המשיכו את מסע הלמידה שלכם

### שלבים מומלצים להמשך

| מסלול | משאב | זמן |
|:---|:---|:---|
| **קורס מלא** | [תוכנית AZD למתחילים](../../README.md) | 8-12 שעות |
| **נסו תבנית נוספת** | [תבניות Microsoft Foundry](https://ai.azure.com/templates) | 1-2 שעות |
| **העמקה** | [תיעוד Azure Developer CLI](https://learn.microsoft.com/azure/developer/azure-developer-cli/) | לאורך זמן |

### תבניות מומלצות לנסות בהמשך

1. **[התחילו עם צ׳אט AI](https://github.com/Azure-Samples/get-started-with-ai-chat)** - צ׳אט בסיסי עם הנתונים שלכם  
2. **[דמו חיפוש Azure OpenAI](https://github.com/Azure-Samples/azure-search-openai-demo)** - תבניות RAG מתקדמות  
3. **[צ׳אט Contoso](https://github.com/Azure-Samples/contoso-chat)** - תרחישים עסקיים לקמעונאות
