## סקירת הקורס

למדו את Azure Developer CLI (azd) דרך פרקים מובנים המיועדים ללמידה מתקדמת. **התמקדות מיוחדת בפריסת יישומי AI עם אינטגרציה של Azure AI Foundry.**

### למה הקורס הזה חיוני למפתחים מודרניים

בהתבסס על תובנות מקהילת Azure AI Foundry ב-Discord, **45% מהמפתחים רוצים להשתמש ב-AZD לעומסי עבודה של AI** אך נתקלים באתגרים עם:
- ארכיטקטורות AI מרובות שירותים ומורכבות
- שיטות עבודה מומלצות לפריסת AI בסביבת ייצור  
- אינטגרציה וקונפיגורציה של שירותי Azure AI
- אופטימיזציה של עלויות לעומסי עבודה של AI
- פתרון בעיות בפריסות AI

### מטרות למידה

בסיום הקורס המובנה הזה, תלמדו:
- **שליטה ביסודות AZD**: מושגים מרכזיים, התקנה וקונפיגורציה
- **פריסת יישומי AI**: שימוש ב-AZD עם שירותי Azure AI Foundry
- **יישום Infrastructure as Code**: ניהול משאבי Azure עם תבניות Bicep
- **פתרון בעיות בפריסות**: זיהוי ופתרון בעיות נפוצות
- **אופטימיזציה לסביבת ייצור**: אבטחה, סקיילינג, ניטור וניהול עלויות
- **בניית פתרונות מרובי סוכנים**: פריסת ארכיטקטורות AI מורכבות

## 📖 מהו Azure Developer CLI?

Azure Developer CLI (azd) הוא ממשק שורת פקודה ממוקד מפתחים שמאיץ את תהליך בניית ופריסת אפליקציות ל-Azure. הוא מספק:

- **פריסות מבוססות תבניות** - שימוש בתבניות מוכנות מראש לדפוסי אפליקציות נפוצים
- **תשתית כקוד** - ניהול משאבי Azure באמצעות Bicep או Terraform  
- **זרימות עבודה משולבות** - פריסה, ניטור והקצאת משאבים בצורה חלקה
- **ידידותי למפתחים** - מותאם לפרודוקטיביות וחוויית מפתחים

### **AZD + Azure AI Foundry: מושלם לפריסות AI**

**מדוע AZD לפתרונות AI?** AZD מתמודד עם האתגרים המרכזיים של מפתחי AI:

- **תבניות מוכנות ל-AI** - תבניות מוגדרות מראש ל-Azure OpenAI, Cognitive Services ועומסי עבודה של ML
- **פריסות AI מאובטחות** - דפוסי אבטחה מובנים לשירותי AI, מפתחות API ונקודות קצה של מודלים  
- **דפוסי AI בייצור** - שיטות עבודה מומלצות לפריסות אפליקציות AI בקנה מידה גדול ובעלות נמוכה
- **זרימות עבודה מקצה לקצה** - מפיתוח מודלים ועד פריסת ייצור עם ניטור מתאים
- **אופטימיזציה של עלויות** - אסטרטגיות הקצאת משאבים חכמות ועיצוב בקנה מידה
- **שילוב עם Azure AI Foundry** - חיבור חלק לקטלוג מודלים ונקודות קצה של AI Foundry
