- **📚 דף הבית של הקורס**: [AZD למתחילים](../README.md)

## מבוא

מדריך למידה מקיף זה מספק יעדים לימודיים מובנים, מושגים מרכזיים, תרגילים מעשיים וחומרי הערכה שיעזרו לך לשלוט ב-Azure Developer CLI (azd). השתמש במדריך זה למעקב ההתקדמות שלך ולהבטיח כי כיסית את כל הנושאים החיוניים.

## יעדי הלמידה

בסיום מדריך למידה זה, תוכל:
- לשלוט בכל המושגים הבסיסיים והמתקדמים של Azure Developer CLI
- לפתח מיומנויות מעשיות בפריסה וניהול יישומי Azure
- לבנות ביטחון בפתרון תקלות ואופטימיזציית פריסות
- להבין שיטות פריסה מוכנות לייצור ושיקולי אבטחה

## תוצאות הלמידה

לאחר השלמת כל חלקי מדריך הלמידה, תוכל:
- לתכנן, לפרוס ולנהל אדריכלות יישומים שלמה באמצעות azd
- ליישם אסטרטגיות מעקב, אבטחה ואופטימיזציה מקיפות של עלויות
- לפתור בעיות פריסה מורכבות באופן עצמאי
- ליצור תבניות מותאמות אישית ולתרום לקהילת azd

#### תרגילים מעשיים

**תרגיל 1.1: התקנה והגדרה**
```bash
# השלם את המשימות הללו:
1. Install azd using your preferred method
2. Install Azure CLI and authenticate
3. Verify installation with: azd version
4. Test connectivity with: azd auth login
5. Explore available templates: azd template list
```

**תרגיל 1.2: פריסה ראשונה**
```bash
# לפרוס יישום ווב פשוט:
1. Initialize project: azd init --template todo-nodejs-mongo
2. Review project structure and configuration files
3. Deploy to Azure: azd up
4. Test the deployed application
5. Clean up resources: azd down
```

**תרגיל 1.3: ניתוח מבנה פרויקט**
```
Analyze the following components:
1. azure.yaml - service definitions and hooks
2. infra/ directory - Bicep templates and modules
3. src/ directory - application source code
4. .azure/ directory - environment configurations
```

## מעקב התקדמות ומסגרת הערכה
```bash
# צור וקבע הגדרות לסביבות מרובות:
1. Create development environment: azd env new development
2. Create staging environment: azd env new staging
3. Create production environment: azd env new production
4. Configure different settings for each environment
```
