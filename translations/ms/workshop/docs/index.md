# Bengkel AZD untuk Pembangun AI

> Disahkan menggunakan `azd 1.25.6` pada Jun 2026.

> [!IMPORTANT]  
> **Bengkel ini dilengkapi dengan panduan bengkel yang boleh anda pratonton dalam pelayar tempatan anda. Untuk bermula, buka repo dalam GitHub Codespaces, bekas pembangunan, atau klon tempatan dengan MkDocs dipasang—kemudian tunggu sehingga anda melihat terminal VS Code aktif dan taip:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Anda sepatutnya melihat dialog pop-up untuk membuka halaman pratonton dalam pelayar.**

Selamat datang ke bengkel amali untuk mempelajari Azure Developer CLI (AZD) dengan fokus pada penyebaran aplikasi AI. Bengkel ini membantu anda memperoleh pemahaman praktikal mengenai templat AZD dalam 3 langkah:

1. **Penemuan** - cari templat yang sesuai untuk anda.  
1. **Penyebaran** - sebarkan dan sahkan ia berfungsi  
1. **Penyesuaian** - ubah suai dan ulang untuk menjadikannya milik anda!

Sepanjang bengkel ini, anda juga akan diperkenalkan kepada alat dan aliran kerja pembangun teras, untuk membantu anda melicinkan perjalanan pembangunan hujung ke hujung anda.

## Gambaran Keseluruhan Bengkel

**Tempoh:** 3-4 jam  
**Tahap:** Pemula ke Pertengahan  
**Prasyarat:** Familiar dengan Azure, konsep AI, VS Code & alat baris arahan.

Ini adalah bengkel amali di mana anda belajar dengan melakukan. Setelah anda menyelesaikan latihan, kami mengesyorkan untuk mengkaji kurikulum AZD Untuk Pemula bagi meneruskan perjalanan pembelajaran anda ke dalam amalan terbaik Keselamatan dan Produktiviti.

| Masa| Modul  | Objektif |
|:---|:---|:---|
| 15 minit | Pengenalan | Tetapkan pentas, fahami matlamat |
| 30 minit | Pilih Templat AI | Terokai pilihan dan pilih permulaan | 
| 30 minit | Sahkan Templat AI | Sebarkan penyelesaian lalai ke Azure |
| 30 minit | Buka Templat AI | Terokai struktur dan konfigurasi |
| 30 minit | Konfigurasikan Templat AI | Aktifkan dan cuba ciri yang tersedia |
| 30 minit | Sesuaikan Templat AI | Sesuaikan templat mengikut keperluan anda |
| 30 minit | Bongkar Infrastruktur | Bersihkan dan lepaskan sumber |
| 15 minit | Penutup & Langkah Seterusnya | Sumber pembelajaran, cabaran bengkel |
| | |

## Apa yang Akan Anda Pelajari

Fikirkan Templat AZD sebagai kawasan latihan untuk meneroka pelbagai keupayaan dan alat untuk pembangunan hujung-ke-hujung pada Microsoft Foundry. Pada akhir bengkel ini, anda sepatutnya mempunyai rasa intuitif tentang pelbagai alat dan konsep dalam konteks ini.

| Konsep  | Objektif |
|:---|:---|
| **Azure Developer CLI** | Fahami arahan alat dan aliran kerja|
| **Templat AZD**| Fahami struktur projek dan konfigurasi|
| **Ejen AI Azure**| Sediakan & sebarkan projek Microsoft Foundry  |
| **Carian AI Azure**| Aktifkan kejuruteraan konteks dengan ejen |
| **Kebolehamatan**| Terokai penjejakan, pemantauan dan penilaian |
| **Pasukan Merah**| Terokai ujian adversari dan mitigasi |
