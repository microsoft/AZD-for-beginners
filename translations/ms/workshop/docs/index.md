# Bengkel AZD untuk Pembangun AI

> Disahkan terhadap `azd 1.23.12` pada Mac 2026.

> [!IMPORTANT]  
> **Bengkel ini dilengkapi dengan panduan bengkel yang boleh anda pratonton dalam pelayar tempatan anda. Untuk memulakan, buka repo di GitHub Codespaces, bekas dev, atau klon tempatan dengan MkDocs dipasang—kemudian tunggu sehingga anda melihat terminal VS Code aktif dan taip:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Anda sepatutnya melihat dialog popup untuk membuka halaman pratonton dalam pelayar.**

Selamat datang ke bengkel praktikal untuk mempelajari Azure Developer CLI (AZD) dengan fokus pada penyebaran aplikasi AI. Bengkel ini membantu anda mendapat pemahaman aplikasi tentang templat AZD dalam 3 langkah:

1. **Penemuan** - cari templat yang sesuai untuk anda.
1. **Penyebaran** - sebarkan dan sahkan ia berfungsi
1. **Penyesuaian** - ubah dan ulang untuk menjadikannya milik anda!

Sepanjang bengkel ini, anda juga akan diperkenalkan kepada alat dan aliran kerja pembangun teras, bagi membantu anda memperkemaskan perjalanan pembangunan hujung ke hujung anda.

## Gambaran Keseluruhan Bengkel

**Tempoh:** 3-4 jam  
**Tahap:** Pemula hingga Pertengahan  
**Prasyarat:** Kefahaman mengenai Azure, konsep AI, VS Code & alat baris perintah.

Ini adalah bengkel praktikal di mana anda belajar dengan melakukan. Setelah melengkapkan latihan, kami mengesyorkan meninjau kurikulum AZD Untuk Pemula untuk meneruskan perjalanan pembelajaran anda ke arah amalan terbaik Keselamatan dan Produktiviti.

| Masa| Modul  | Objektif |
|:---|:---|:---|
| 15 minit | Pengenalan | Menetapkan pentas, memahami matlamat |
| 30 minit | Pilih Templat AI | Terokai pilihan dan pilih permulaan | 
| 30 minit | Sahkan Templat AI | Sebarkan penyelesaian lalai ke Azure |
| 30 minit | Bina Semula Templat AI | Terokai struktur dan konfigurasi |
| 30 minit | Konfigurasikan Templat AI | Aktifkan dan cuba ciri tersedia |
| 30 minit | Sesuaikan Templat AI | Sesuaikan templat mengikut keperluan anda |
| 30 minit | Bongkar Infrastruktur | Bersihkan dan lepaskan sumber |
| 15 minit | Penutup & Langkah Seterusnya | Sumber pembelajaran, cabaran bengkel |
| | |

## Apa Yang Anda Akan Pelajari

Fikirkan Templat AZD sebagai ruang pembelajaran untuk meneroka pelbagai keupayaan dan alat untuk pembangunan hujung ke hujung pada Microsoft Foundry. Pada akhir bengkel ini, anda seharusnya mempunyai rasa intuitif terhadap pelbagai alat dan konsep dalam konteks ini.

| Konsep  | Objektif |
|:---|:---|
| **Azure Developer CLI** | Fahami arahan alat dan aliran kerja|
| **Templat AZD**| Fahami struktur projek dan konfigurasi|
| **Ejen Azure AI**| Sediakan & sebarkan projek Microsoft Foundry  |
| **Carian Azure AI**| Aktifkan kejuruteraan konteks dengan ejen |
| **Pengamatan**| Terokai penjejakan, pemantauan dan penilaian |
| **Red Teaming**| Terokai ujian adversarial dan mitigasi |
