# Bengkel AZD untuk Pembangun AI

> [!IMPORTANT]  
> **Bengkel ini dilengkapi dengan panduan bengkel yang boleh anda pratonton dalam pelayar tempatan anda. Untuk memulakan, anda mesti melancarkan GitHub Codespaces pada repo—kemudian tunggu sehingga anda melihat terminal VS Code yang aktif dan taip:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Anda sepatutnya melihat dialog pop-up untuk membuka halaman pratonton dalam pelayar.**

Selamat datang ke bengkel praktikal untuk mempelajari Azure Developer CLI (AZD) dengan fokus pada penyebaran aplikasi AI. Bengkel ini membantu anda memperoleh pemahaman praktikal mengenai templat AZD dalam 3 langkah:

1. **Penemuan** - cari templat yang sesuai untuk anda.
1. **Penyebaran** - sebarkan dan sahkan bahawa ia berfungsi
1. **Pengubahsuaian** - ubah dan iterasi untuk menjadikannya milik anda!

Sepanjang bengkel ini, anda juga akan diperkenalkan kepada alat dan aliran kerja pembangun teras, untuk membantu anda mempermudah perjalanan pembangunan hujung-ke-hujung anda.

## Gambaran Keseluruhan Bengkel

**Tempoh:** 3-4 jam  
**Tahap:** Pemula ke Pertengahan  
**Prasyarat:** Kebiasaan dengan Azure, konsep AI, VS Code & alat baris perintah.

Ini adalah bengkel praktikal di mana anda belajar dengan melakukan. Setelah anda menyelesaikan latihan, kami mengesyorkan untuk meninjau kurikulum AZD Untuk Pemula untuk meneruskan perjalanan pembelajaran anda ke amalan terbaik Keselamatan dan Produktiviti.

| Masa| Modul  | Objektif |
|:---|:---|:---|
| 15 minit | Pengenalan | Sediakan pentas, fahami matlamat |
| 30 minit | Pilih Templat AI | Terokai pilihan dan pilih permulaan | 
| 30 minit | Sahkan Templat AI | Sebarkan penyelesaian lalai ke Azure |
| 30 minit | Bina Semula Templat AI | Terokai struktur dan konfigurasi |
| 30 minit | Konfigurasi Templat AI | Aktifkan dan cuba ciri yang tersedia |
| 30 minit | Sesuaikan Templat AI | Sesuaikan templat mengikut keperluan anda |
| 30 minit | Bongkar Infrastruktur | Bersihkan dan lepaskan sumber |
| 15 minit | Penutup & Langkah Seterusnya | Sumber pembelajaran, Cabaran bengkel |
| | |

## Apa yang Akan Anda Pelajari

Fikirkan Templat AZD sebagai kotak pasir pembelajaran untuk meneroka pelbagai keupayaan dan alat bagi pembangunan hujung-ke-hujung pada Microsoft Foundry. Pada akhir bengkel ini, anda sepatutnya mempunyai rasa intuitif terhadap pelbagai alat dan konsep dalam konteks ini.

| Konsep  | Objektif |
|:---|:---|
| **Azure Developer CLI** | Fahami arahan alat dan aliran kerja|
| **Templat AZD**| Fahami struktur projek dan konfigurasi|
| **Ejen Azure AI**| Sediakan & sebarkan projek Microsoft Foundry  |
| **Carian Azure AI**| Aktifkan kejuruteraan konteks dengan ejen |
| **Kebolehlihatan**| Terokai penjejakan, pemantauan dan penilaian |
| **Red Teaming**| Terokai ujian adversarial dan mitigasi |
