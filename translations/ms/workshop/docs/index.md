# Bengkel AZD untuk Pembangun AI

> Disahkan terhadap `azd 1.27.1` pada Julai 2026.

> [!IMPORTANT]  
> **Bengkel ini dilengkapi dengan panduan bengkel yang anda boleh pratonton dalam pelayar tempatan anda. Untuk memulakan, buka repositori dalam GitHub Codespaces, bekas dev, atau klon tempatan dengan MkDocs dipasang—kemudian tunggu sehingga anda melihat terminal VS Code aktif dan taip:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Anda sepatutnya melihat dialog timbul untuk membuka halaman pratonton dalam pelayar.**

Selamat datang ke bengkel praktik untuk mempelajari Azure Developer CLI (AZD) dengan fokus pada penyebaran aplikasi AI. Bengkel ini membantu anda memperoleh pemahaman terapan tentang templat AZD dalam 3 langkah:

1. **Penemuan** - cari templat yang sesuai untuk anda.
1. **Penyebaran** - sebarkan dan sahkan bahawa ia berfungsi
1. **Penyesuaian** - ubah suai dan iterasi untuk menjadikannya milik anda!

Sepanjang bengkel ini, anda juga akan diperkenalkan dengan alat pembangun utama dan aliran kerja, untuk membantu mempermudahkan perjalanan pembangunan hujung-ke-hujung anda.

## Gambaran Keseluruhan Bengkel

**Tempoh:** 3-4 jam  
**Tahap:** Pemula ke Pertengahan  
**Prasyarat:** Kefahaman tentang Azure, konsep AI, VS Code & alat baris perintah.

Ini adalah bengkel praktik di mana anda belajar dengan melakukan. Setelah anda menyelesaikan latihan, kami mengesyorkan untuk meninjau kurikulum AZD Untuk Pemula bagi meneruskan perjalanan pembelajaran anda ke dalam amalan terbaik Keselamatan dan Produktiviti.

| Masa| Modul  | Objektif |
|:---|:---|:---|
| 15 minit | Pengenalan | Menetapkan pentas, memahami matlamat |
| 30 minit | Pilih Templat AI | Terokai pilihan dan pilih permulaan | 
| 30 minit | Sahkan Templat AI | Sebarkan penyelesaian lalai ke Azure |
| 30 minit | Kupas Templat AI | Terokai struktur dan konfigurasi |
| 30 minit | Konfigurasikan Templat AI | Aktifkan dan cuba ciri sedia ada |
| 30 minit | Sesuaikan Templat AI | Sesuaikan templat mengikut keperluan anda |
| 30 minit | Bongkar Infrastruktur | Bersihkan dan lepaskan sumber |
| 15 minit | Penutup & Langkah Seterusnya | Sumber pembelajaran, cabaran bengkel |
| | |

## Apa yang Akan Anda Pelajari

Fikirkan Templat AZD sebagai kotak pasir pembelajaran untuk meneroka pelbagai keupayaan dan alat untuk pembangunan hujung-ke-hujung pada Microsoft Foundry. Pada akhir bengkel ini, anda seharusnya mempunyai rasa intuitif terhadap pelbagai alat dan konsep dalam konteks ini.

| Konsep  | Objektif |
|:---|:---|
| **Azure Developer CLI** | Memahami arahan alat dan aliran kerja|
| **Templat AZD**| Memahami struktur projek dan konfigurasi|
| **Ejen Azure AI**| Menyediakan & menyebarkan projek Microsoft Foundry  |
| **Carian Azure AI**| Mengaktifkan kejuruteraan konteks dengan ejen |
| **Keterlihatan**| Terokai penjejakan, pemantauan dan penilaian |
| **Pasukan Merah**| Terokai ujian adversarial dan mitigasi |
