## 2. Konsep Utama Dikuasai

### Azure Developer CLI (azd)
- `azd init` - Memulakan projek dari templat
- `azd up` - Menyediakan dan melaksanakan dalam satu arahan
- `azd env` - Menguruskan pembolehubah persekitaran
- `azd down` - Membersihkan sumber dengan selamat

### Struktur Templat AZD
- `azure.yaml` - Konfigurasi dan kaitan pelaksanaan
- `infra/` - Infrastruktur sebagai kod Bicep
- `src/` - Kod sumber aplikasi
- `scripts/` - Skrip automasi dan persediaan

### Kebolehan Microsoft Foundry
- Perkhidmatan Agen AI untuk pembantu pintar
- Carian Fail dan Azure AI Search untuk pengambilan pengetahuan
- Jejak, pemantauan, dan penilaian terbina dalam
- Red teaming untuk pengesahan keselamatan AI
