# บทที่ 8: รูปแบบการผลิต & องค์กร

**📚 หลักสูตร**: [AZD สำหรับผู้เริ่มต้น](../../README.md) | **⏱️ ระยะเวลา**: 2-3 ชั่วโมง | **⭐ ความซับซ้อน**: ขั้นสูง

---

## ภาพรวม

บทนี้ครอบคลุมรูปแบบการปรับใช้ที่พร้อมใช้งานในองค์กร การเสริมความแข็งแกร่งด้านความปลอดภัย การตรวจสอบ และการเพิ่มประสิทธิภาพค่าใช้จ่ายสำหรับงาน AI ในสภาพแวดล้อมการผลิต

> ผ่านการตรวจสอบกับ `azd 1.25.6` ในเดือนมิถุนายน 2026

## วัตถุประสงค์การเรียนรู้

เมื่อคุณเรียนจบบทนี้ คุณจะสามารถ:
- ปรับใช้แอปพลิเคชันที่มีความยืดหยุ่นหลายภูมิภาค
- นำรูปแบบความปลอดภัยสำหรับองค์กรไปใช้
- ตั้งค่าการตรวจสอบอย่างครบถ้วน
- ปรับค่าใช้จ่ายให้เหมาะสมในระดับใหญ่
- ตั้งค่า pipeline CI/CD ด้วย AZD

## 🚀 รายการตรวจสอบสำหรับการผลิต

- [ ] การปรับใช้หลายภูมิภาคเพื่อความยืดหยุ่น
- [ ] ตัวตนที่จัดการสำหรับการยืนยันตัวตน (ไม่มีคีย์)
- [ ] Application Insights สำหรับการตรวจสอบ
- [ ] ตั้งงบประมาณและการแจ้งเตือนค่าใช้จ่าย
- [ ] เปิดใช้งานการสแกนความปลอดภัย
- [ ] การรวม pipeline CI/CD
- [ ] แผนการกู้คืนระบบเมื่อเกิดภัยพิบัติ

## 💰 การเพิ่มประสิทธิภาพค่าใช้จ่าย

| กลยุทธ์ | การประหยัด |
|---------|------------|
| ปรับสเกลเป็นศูนย์ (Container Apps) | 60-80% |
| ใช้ระดับการบริโภคสำหรับการพัฒนา | 50-70% |
| การปรับสเกลตามกำหนดเวลา | 30-50% |
| ความจุที่จองไว้ | 20-40% |

```bash
# ตั้งค่าการแจ้งเตืองบประมาณ
az consumption budget create \
  --budget-name "AI-Budget" \
  --amount 500 \
  --category Cost \
  --time-grain Monthly
```

## 📊 การตั้งค่าการตรวจสอบ

```bash
# สตรีมบันทึก
azd monitor --logs

# ตรวจสอบข้อมูลเชิงลึกของแอปพลิเคชัน
azd monitor --overview

# ดูเมตริกส์
az monitor metrics list --resource <resource-id>
```
