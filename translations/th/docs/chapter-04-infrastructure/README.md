# บทที่ 4: โครงสร้างพื้นฐานในรูปแบบโค้ด & การปรับใช้งาน

**📚 คอร์ส**: [AZD สำหรับผู้เริ่มต้น](../../README.md) | **⏱️ ระยะเวลา**: 1-1.5 ชั่วโมง | **⭐ ความซับซ้อน**: ระดับกลาง

---

## ภาพรวม

บทนี้ครอบคลุมรูปแบบ Infrastructure as Code (IaC) กับเทมเพลต Bicep, การจัดเตรียมทรัพยากร และกลยุทธ์การปรับใช้งานโดยใช้ Azure Developer CLI

> ได้รับการตรวจสอบกับ `azd 1.27.1` ในเดือนกรกฎาคม 2026

## วัตถุประสงค์การเรียนรู้

เมื่อจบบทนี้ คุณจะสามารถ:
- เข้าใจโครงสร้างและไวยากรณ์ของเทมเพลต Bicep
- จัดเตรียมทรัพยากร Azure ด้วย `azd provision`
- ปรับใช้งานแอปพลิเคชันด้วย `azd deploy`
- ใช้กลยุทธ์การปรับใช้แบบ blue-green และ rolling deployment

## 🚀 เริ่มต้นอย่างรวดเร็ว

```bash
# เริ่มต้นจากแม่แบบ
azd init --template azure-functions-python-v2-http

# ดูตัวอย่างสิ่งที่จะถูกสร้าง
azd provision --preview

# จัดเตรียมโครงสร้างพื้นฐานเท่านั้น
azd provision

# นำส่งโค้ดเท่านั้น
azd deploy

# หรือทั้งสองอย่างพร้อมกัน
azd up
```

## 📁 โครงสร้างโปรเจกต์ AZD

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

---

## 🔧 คำสั่งพื้นฐาน

| คำสั่ง | คำอธิบาย |
|---------|-------------|
| `azd init` | เริ่มโปรเจกต์ |
| `azd provision` | สร้างทรัพยากร Azure |
| `azd deploy` | ปรับใช้งานโค้ดแอปพลิเคชัน |
| `azd up` | จัดเตรียม + ปรับใช้ |
| `azd down` | ลบทรัพยากรทั้งหมด |
