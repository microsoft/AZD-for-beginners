# บทที่ 3: การกำหนดค่า & การรับรองความถูกต้อง

**📚 หลักสูตร**: [AZD สำหรับผู้เริ่มต้น](../../README.md) | **⏱️ ระยะเวลา**: 45-60 นาที | **⭐ ความซับซ้อน**: ระดับปานกลาง

---

## ภาพรวม

บทนี้ครอบคลุมการกำหนดค่าสภาพแวดล้อม รูปแบบการรับรองความถูกต้อง และแนวปฏิบัติด้านความปลอดภัยที่ดีที่สุดสำหรับการปรับใช้ด้วย Azure Developer CLI

## วัตถุประสงค์การเรียนรู้

By completing this chapter, you will:
- เชี่ยวชาญลำดับชั้นการกำหนดค่า AZD
- จัดการหลายสภาพแวดล้อม (dev, staging, prod)
- ใช้การรับรองความถูกต้องที่ปลอดภัยด้วย managed identities
- กำหนดค่าการตั้งค่าที่เฉพาะเจาะจงสำหรับแต่ละสภาพแวดล้อม

## 🚀 เริ่มต้นด่วน

```bash
# สร้างหลายสภาพแวดล้อม
azd env new dev
azd env new staging
azd env new prod

# สลับสภาพแวดล้อม
azd env select prod

# ตั้งค่าตัวแปรสภาพแวดล้อม
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# ดูการกำหนดค่า
azd env get-values
```

---

## 🔧 ลำดับชั้นการกำหนดค่า

AZD ใช้การตั้งค่าในลำดับนี้ (ค่าที่มากหลังกว่าจะเขียนทับค่าก่อนหน้า):

1. **ค่าเริ่มต้น** (ฝังอยู่ในเทมเพลต)
2. **azure.yaml** (การกำหนดค่าโปรเจกต์)
3. **ตัวแปรสภาพแวดล้อม** (`azd env set`)
4. **แฟลกบรรทัดคำสั่ง** (`--location eastus`)

---

## 🔐 แนวปฏิบัติด้านความปลอดภัยที่ดีที่สุด

```bash
# ใช้ Managed Identity (แนะนำ)
azd env set AZURE_USE_MANAGED_IDENTITY true

# ตรวจสอบสถานะการยืนยันตัวตน
azd auth whoami
az account show

# ยืนยันตัวตนอีกครั้งหากจำเป็น
azd auth login
az login
```
