# Chapter 1: Foundation & Quick Start

**📚 คอร์ส**: [AZD สำหรับผู้เริ่มต้น](../../README.md) | **⏱️ ระยะเวลา**: 30-45 นาที | **⭐ ความซับซ้อน**: ระดับเริ่มต้น

## ภาพรวม

บทนี้แนะนำพื้นฐานของ Azure Developer CLI (azd) คุณจะได้เรียนรู้แนวคิดหลัก ติดตั้งเครื่องมือ และปรับใช้แอปพลิเคชันแรกของคุณไปยัง Azure

> ได้รับการตรวจสอบกับ `azd 1.23.12` ในเดือนมีนาคม 2026

## วัตถุประสงค์การเรียนรู้

เมื่อจบบทนี้แล้ว คุณจะสามารถ:
- เข้าใจว่า Azure Developer CLI คืออะไรและแตกต่างจาก Azure CLI อย่างไร
- ติดตั้งและกำหนดค่า AZD บนแพลตฟอร์มของคุณ
- ปรับใช้แอปพลิเคชันแรกของคุณไปยัง Azure ด้วย `azd up`
- ทำความสะอาดทรัพยากรด้วย `azd down`

## ✅ เริ่มที่นี่: ตรวจสอบการตั้งค่าของคุณ

ก่อนเริ่มต้น ให้ยืนยันว่าเครื่องของคุณพร้อมสำหรับเทมเพลตบทที่ 1:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

หากสคริปต์แจ้งว่าขาดเครื่องมือ ให้แก้ไขส่วนที่ขาดก่อนแล้วจึงดำเนินการบทนี้ต่อ

---

## 🚀 เริ่มต้นอย่างรวดเร็ว

```bash
# ตรวจสอบการติดตั้ง
azd version

# ตรวจสอบสิทธิ์สำหรับ AZD
# ตัวเลือก: az login ถ้าคุณวางแผนที่จะรันคำสั่ง Azure CLI โดยตรง
azd auth login

# ติดตั้งแอปแรกของคุณ
azd init --template todo-nodejs-mongo
azd up

# ทำความสะอาดเมื่อเสร็จสิ้น
azd down --force --purge
```

---

## ✅ เกณฑ์ความสำเร็จ

หลังจากจบบทนี้แล้ว คุณควรจะสามารถ:

```bash
azd version              # แสดงเวอร์ชันที่ติดตั้ง
azd init --template todo-nodejs-mongo  # เริ่มต้นโปรเจกต์
azd up                   # โอนไปยัง Azure
azd show                 # แสดง URL แอปที่กำลังทำงาน
azd down --force --purge # ทำความสะอาดทรัพยากร
```
