# บทที่ 1: พื้นฐานและเริ่มต้นอย่างรวดเร็ว

**📚 หลักสูตร**: [AZD สำหรับผู้เริ่มต้น](../../README.md) | **⏱️ Duration**: 30-45 นาที | **⭐ ความซับซ้อน**: ระดับเริ่มต้น

---

## ภาพรวม

บทนี้จะแนะนำพื้นฐานของ Azure Developer CLI (azd). คุณจะได้เรียนรู้แนวคิดหลัก ติดตั้งเครื่องมือ และปรับใช้แอปพลิเคชันแรกของคุณไปยัง Azure.

## วัตถุประสงค์การเรียนรู้

เมื่อจบบทนี้ คุณจะสามารถ:
- เข้าใจว่า Azure Developer CLI คืออะไร และมันแตกต่างจาก Azure CLI อย่างไร
- ติดตั้งและกำหนดค่า AZD บนแพลตฟอร์มของคุณ
- ปรับใช้แอปพลิเคชันแรกของคุณไปยัง Azure ด้วย `azd up`
- ทำความสะอาดทรัพยากรด้วย `azd down`

## 🚀 เริ่มต้นอย่างรวดเร็ว

```bash
# ตรวจสอบการติดตั้ง
azd version

# ลงชื่อเข้าใช้ Azure
azd auth login

# ปรับใช้แอปแรกของคุณ
azd init --template todo-nodejs-mongo
azd up

# ทำความสะอาดเมื่อเสร็จแล้ว
azd down --force --purge
```

---

## ✅ เกณฑ์ความสำเร็จ

หลังจากจบบทนี้ คุณควรจะสามารถ:

```bash
azd version              # แสดงเวอร์ชันที่ติดตั้ง
azd init --template todo-nodejs-mongo  # เริ่มต้นโครงการ
azd up                   # ปรับใช้ไปยัง Azure
azd show                 # แสดง URL ของแอปที่กำลังทำงาน
azd down --force --purge # ทำความสะอาดทรัพยากร
```
