# AZD สำหรับเวิร์กช็อปนักพัฒนา AI

> ตรวจสอบกับ `azd 1.25.6` ในเดือนมิถุนายน 2026

> [!IMPORTANT]  
> **เวิร์กช็อปนี้มีคำแนะนำเวิร์กช็อปที่คุณสามารถดูตัวอย่างในเบราว์เซอร์ในเครื่องของคุณเองได้ เริ่มต้นโดยการเปิด repo ใน GitHub Codespaces, dev container หรือโคลนในเครื่องที่ติดตั้ง MkDocs แล้วรอจนกว่าจะเห็นเทอร์มินัล VS Code ที่ทำงานอยู่ จากนั้นพิมพ์:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **คุณจะเห็นกล่องโต้ตอบป๊อปอัพเพื่อเปิดหน้าแสดงตัวอย่างในเบราว์เซอร์**

ยินดีต้อนรับสู่เวิร์กช็อปแบบลงมือทำเพื่อเรียนรู้ Azure Developer CLI (AZD) โดยมุ่งเน้นที่การปรับใช้แอป AI เวิร์กช็อปนี้จะช่วยให้คุณเข้าใจการใช้งาน AZD เทมเพลตใน 3 ขั้นตอน:

1. **ค้นพบ** - หาตัวเทมเพลตที่เหมาะกับคุณ  
1. **ปรับใช้** - ปรับใช้และตรวจสอบว่าใช้งานได้  
1. **ปรับแต่ง** - แก้ไขและทำซ้ำเพื่อให้เป็นของคุณ!

ในช่วงเวลาของเวิร์กช็อป คุณจะได้รู้จักเครื่องมือและเวิร์กโฟลว์หลักสำหรับนักพัฒนา เพื่อช่วยให้กระบวนการพัฒนาครบวงจรของคุณราบรื่นยิ่งขึ้น

## ภาพรวมเวิร์กช็อป

**ระยะเวลา:** 3-4 ชั่วโมง  
**ระดับ:** เริ่มต้นถึงระดับกลาง  
**ความรู้พื้นฐาน:** คุ้นเคยกับ Azure, แนวคิด AI, VS Code & เครื่องมือบรรทัดคำสั่ง

นี่คือเวิร์กช็อปลงมือทำที่คุณเรียนรู้ผ่านการปฏิบัติ เมื่อคุณทำแบบฝึกหัดเสร็จ เราแนะนำให้ทบทวนหลักสูตร AZD สำหรับผู้เริ่มต้น เพื่อเรียนรู้ต่อเนื่องในด้านการรักษาความปลอดภัยและแนวปฏิบัติที่ดีที่สุดด้านประสิทธิภาพ

| เวลา | โมดูล | วัตถุประสงค์ |
|:---|:---|:---|
| 15 นาที | แนะนำ | ตั้งเวที เข้าใจเป้าหมาย |
| 30 นาที | เลือกเทมเพลต AI | สำรวจตัวเลือกและเลือกตัวเริ่มต้น | 
| 30 นาที | ตรวจสอบเทมเพลต AI | ปรับใช้โซลูชันค่าเริ่มต้นลงใน Azure |
| 30 นาที | แยกวิเคราะห์เทมเพลต AI | สำรวจโครงสร้างและการกำหนดค่า |
| 30 นาที | กำหนดค่าเทมเพลต AI | เปิดใช้งานและลองใช้ฟีเจอร์ที่มี |
| 30 นาที | ปรับแต่งเทมเพลต AI | ปรับเทมเพลตให้เหมาะกับความต้องการของคุณ |
| 30 นาที | รื้อถอนโครงสร้างพื้นฐาน | ทำความสะอาดและปล่อยทรัพยากร |
| 15 นาที | สรุป & ขั้นตอนถัดไป | แหล่งเรียนรู้ ความท้าทายในเวิร์กช็อป |
| | |

## สิ่งที่คุณจะได้เรียนรู้

คิดว่า AZD Template เป็นสนามทดลองให้สำรวจความสามารถและเครื่องมือต่างๆ สำหรับการพัฒนาครบวงจรบน Microsoft Foundry เมื่อจบเวิร์กช็อปนี้ คุณควรมีความเข้าใจในเครื่องมือและแนวคิดต่างๆ ในบริบทนี้อย่างเป็นธรรมชาติ

| แนวคิด | วัตถุประสงค์ |
|:---|:---|
| **Azure Developer CLI** | เข้าใจคำสั่งและเวิร์กโฟลว์ของเครื่องมือ |
| **AZD Templates** | เข้าใจโครงสร้างโปรเจกต์และการกำหนดค่า |
| **Azure AI Agent** | จัดเตรียมและปรับใช้โปรเจกต์ Microsoft Foundry |
| **Azure AI Search** | เปิดใช้งานการสร้างบริบทด้วยเอเย่นต์ |
| **Observability** | สำรวจการติดตาม ตรวจสอบและประเมินผล |
| **Red Teaming** | สำรวจการทดสอบปฏิปักษ์และวิธีบรรเทา |
| | |
