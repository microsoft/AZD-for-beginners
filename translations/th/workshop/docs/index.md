# AZD สำหรับนักพัฒนา AI การประชุมเชิงปฏิบัติการ

> ตรวจสอบกับ `azd 1.27.1` ในเดือนกรกฎาคม 2026

> [!IMPORTANT]  
> **การประชุมเชิงปฏิบัติการนี้มีคู่มือการประชุมเชิงปฏิบัติการที่คุณสามารถดูตัวอย่างในเบราว์เซอร์ท้องถิ่นของคุณได้ เพื่อเริ่มต้น ให้เปิดรีโปใน GitHub Codespaces, dev container หรือโคลนท้องถิ่นที่ติดตั้ง MkDocs แล้วรอจนกว่าคุณจะเห็นเทอร์มินัล VS Code ที่ใช้งานอยู่และพิมพ์:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **คุณควรเห็นกล่องโต้ตอบป๊อปอัปเพื่อเปิดหน้าดูตัวอย่างในเบราว์เซอร์**

ยินดีต้อนรับสู่การประชุมเชิงปฏิบัติการแบบลงมือทำสำหรับการเรียนรู้ Azure Developer CLI (AZD) โดยมุ่งเน้นไปที่การปรับใช้แอปพลิเคชัน AI การประชุมเชิงปฏิบัติการนี้ช่วยให้คุณเข้าใจการใช้งาน AZD templates ใน 3 ขั้นตอน:

1. **การค้นพบ** - ค้นหาเทมเพลตที่เหมาะกับคุณ
1. **การปรับใช้** - ปรับใช้และตรวจสอบว่ามันทำงานได้
1. **การปรับแต่ง** - แก้ไขและทำซ้ำจนเป็นของคุณ!

ตลอดระยะเวลาการประชุมเชิงปฏิบัติการนี้ คุณจะได้แนะนำเครื่องมือและเวิร์กโฟลว์สำหรับนักพัฒนาหลัก ๆ เพื่อช่วยให้คุณปรับปรุงเส้นทางการพัฒนาจากต้นจนจบอย่างมีประสิทธิภาพยิ่งขึ้น

## ภาพรวมการประชุมเชิงปฏิบัติการ

**ระยะเวลา:** 3-4 ชั่วโมง  
**ระดับ:** ผู้เริ่มต้นถึงระดับกลาง  
**ข้อกำหนดเบื้องต้น:** คุ้นเคยกับ Azure, แนวคิด AI, VS Code และเครื่องมือบรรทัดคำสั่ง

นี่คือการประชุมเชิงปฏิบัติการแบบลงมือทำที่คุณเรียนรู้จากการทำ เมื่อคุณทำแบบฝึกหัดเสร็จแล้ว เราแนะนำให้ทบทวนหลักสูตร AZD For Beginners เพื่อเดินหน้าต่อในการเรียนรู้เกี่ยวกับแนวทางปฏิบัติด้านความปลอดภัยและประสิทธิภาพ

| เวลา| โมดูล | วัตถุประสงค์ |
|:---|:---|:---|
| 15 นาที | บทนำ | กำหนดเป้าหมาย เข้าใจเป้าหมาย |
| 30 นาที | เลือก AI Template | สำรวจตัวเลือกและเลือกตัวเริ่มต้น | 
| 30 นาที | ตรวจสอบ AI Template | ปรับใช้โซลูชันเริ่มต้นไปยัง Azure |
| 30 นาที | วิเคราะห์ AI Template | สำรวจโครงสร้างและการตั้งค่า |
| 30 นาที | กำหนดค่า AI Template | เปิดใช้งานและลองใช้ฟีเจอร์ที่มี |
| 30 นาที | ปรับแต่ง AI Template | ปรับเทมเพลตให้เหมาะกับความต้องการของคุณ |
| 30 นาที | รื้อโครงสร้างพื้นฐาน | ทำความสะอาดและปล่อยทรัพยากร |
| 15 นาที | สรุป & ขั้นตอนต่อไป | แหล่งเรียนรู้ ความท้าทายของการประชุมเชิงปฏิบัติการ |
| | |

## สิ่งที่คุณจะได้เรียนรู้

คิดว่า AZD Template เป็นพื้นที่ทดลองเรียนรู้เพื่อสำรวจความสามารถและเครื่องมือต่าง ๆ สำหรับการพัฒนาจากต้นจนจบบน Microsoft Foundry เมื่อจบการประชุมเชิงปฏิบัติการนี้ คุณควรมีความเข้าใจสัญชาตญาณเกี่ยวกับเครื่องมือและแนวคิดต่าง ๆ ในบริบทนี้

| แนวคิด | วัตถุประสงค์ |
|:---|:---|
| **Azure Developer CLI** | เข้าใจคำสั่งเครื่องมือและเวิร์กโฟลว์ |
| **AZD Templates** | เข้าใจโครงสร้างโปรเจ็กต์และการตั้งค่า |
| **Azure AI Agent** | Provision & ปรับใช้โปรเจ็กต์ Microsoft Foundry |
| **Azure AI Search** | เปิดใช้งานการออกแบบบริบทด้วยเอเจนต์ |
| **Observability** | สำรวจการติดตาม ตรวจสอบ และการประเมินผล |
| **Red Teaming** | สำรวจการทดสอบแบบโจมตีและมาตรการป้องกัน |
| | |
