# AZD สำหรับเวิร์กช็อปนักพัฒนา AI

> [!IMPORTANT]  
> **เวิร์กช็อปนี้มาพร้อมกับคู่มือเวิร์กช็อปที่คุณสามารถดูตัวอย่างได้ในเบราว์เซอร์ของคุณเอง ในการเริ่มต้น คุณต้องเปิด GitHub Codespaces บนรีโป—แล้วรอจนกว่าคุณจะเห็นเทอร์มินัล VS Code ที่ใช้งานอยู่และพิมพ์:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **คุณควรเห็นกล่องโต้ตอบเด้งขึ้นเพื่อเปิดหน้าพรีวิวในเบราว์เซอร์**

ยินดีต้อนรับสู่เวิร์กช็อปแบบลงมือปฏิบัติสำหรับการเรียนรู้ Azure Developer CLI (AZD) โดยเน้นที่การปรับใช้แอปพลิเคชัน AI เวิร์กช็อปนี้จะช่วยให้คุณเข้าใจการใช้งาน AZD เทมเพลตภายใน 3 ขั้นตอน:

1. **ค้นพบ** - หาชุดเทมเพลตที่เหมาะสมกับคุณ
1. **ปรับใช้** - ปรับใช้และยืนยันว่าใช้งานได้
1. **ปรับแต่ง** - แก้ไขและปรับเปลี่ยนให้เป็นของคุณ!

ในระหว่างเวิร์กช็อปนี้ คุณจะได้แนะนำเครื่องมือและเวิร์กโฟลว์สำหรับนักพัฒนาหลัก เพื่อช่วยให้คุณทำงานพัฒนาแบบครบวงจรได้อย่างมีประสิทธิภาพ

## ภาพรวมของเวิร์กช็อป

**ระยะเวลา:** 3-4 ชั่วโมง  
**ระดับ:** ผู้เริ่มต้นถึงระดับกลาง  
**ข้อกำหนดเบื้องต้น:** คุ้นเคยกับ Azure, แนวคิด AI, VS Code และเครื่องมือบรรทัดคำสั่ง

นี่คือเวิร์กช็อปแบบลงมือทำที่คุณจะเรียนรู้โดยการทำ เมื่อคุณทำแบบฝึกหัดเสร็จสิ้น เราแนะนำให้ทบทวนหลักสูตร AZD For Beginners เพื่อเรียนรู้ต่อในแนวทางปฏิบัติที่ดีที่สุดด้านความปลอดภัยและประสิทธิภาพ

| เวลา| โมดูล  | วัตถุประสงค์ |
|:---|:---|:---|
| 15 นาที | บทนำ | กำหนดบริบทและเข้าใจเป้าหมาย |
| 30 นาที | เลือกเทมเพลต AI | สำรวจตัวเลือกและเลือกตัวเริ่มต้น | 
| 30 นาที | ตรวจสอบเทมเพลต AI | ปรับใช้โซลูชันเริ่มต้นใน Azure |
| 30 นาที | วิเคราะห์เทมเพลต AI | สำรวจโครงสร้างและการตั้งค่า |
| 30 นาที | กำหนดค่าเทมเพลต AI | เปิดใช้งานและลองฟีเจอร์ที่มี |
| 30 นาที | ปรับแต่งเทมเพลต AI | ปรับเทมเพลตให้เหมาะกับความต้องการ |
| 30 นาที | รื้อถอนโครงสร้างพื้นฐาน | ทำความสะอาดและปล่อยทรัพยากร |
| 15 นาที | สรุปและขั้นตอนต่อไป | แหล่งเรียนรู้, ความท้าทายในเวิร์กช็อป |
| | |

## สิ่งที่คุณจะได้เรียนรู้

คิดว่า AZD Template คือพื้นที่เรียนรู้เพื่อสำรวจความสามารถและเครื่องมือต่าง ๆ สำหรับการพัฒนาแบบครบวงจรบน Microsoft Foundry เมื่อจบเวิร์กช็อปนี้ คุณควรจะเข้าใจเครื่องมือและแนวคิดต่าง ๆ ภายในบริบทนี้อย่างเป็นธรรมชาติ

| แนวคิด  | วัตถุประสงค์ |
|:---|:---|
| **Azure Developer CLI** | เข้าใจคำสั่งเครื่องมือและเวิร์กโฟลว์ |
| **AZD Templates**| เข้าใจโครงสร้างโปรเจกต์และการตั้งค่า |
| **Azure AI Agent**| ดำเนินการจัดเตรียมและปรับใช้โปรเจกต์ Microsoft Foundry  |
| **Azure AI Search**| เปิดใช้งานวิศวกรรมบริบทด้วยเอเย่นต์ |
| **Observability**| สำรวจการติดตาม การตรวจสอบ และการประเมินผล |
| **Red Teaming**| สำรวจการทดสอบคู่แข่งและการป้องกัน |
| | |
