# เวิร์กช็อป AZD สำหรับนักพัฒนา AI

> ตรวจสอบความถูกต้องกับ `azd 1.23.12` ในเดือนมีนาคม 2026

> [!IMPORTANT]  
> **เวิร์กช็อปนี้ติดตั้งคู่มือเวิร์กช็อปที่คุณสามารถดูตัวอย่างได้ในเบราว์เซอร์ของคุณเอง เพื่อเริ่มต้น ให้เปิด repo ใน GitHub Codespaces, dev container, หรือโคลนท้องถิ่นที่ติดตั้ง MkDocs ไว้—แล้วรอจนกว่าคุณจะเห็นเทอร์มินัล VS Code ที่ใช้งานอยู่ และพิมพ์:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **คุณควรเห็นกล่องโต้ตอบเด้งขึ้นมาเพื่อเปิดหน้าแสดงตัวอย่างในเบราว์เซอร์**

ยินดีต้อนรับสู่เวิร์กช็อปเชิงปฏิบัติสำหรับการเรียนรู้ Azure Developer CLI (AZD) โดยเน้นที่การปรับใช้แอป AI เวิร์กช็อปนี้จะช่วยให้คุณเข้าใจการใช้เทมเพลต AZD ผ่าน 3 ขั้นตอน:

1. **ค้นพบ** - ค้นหาเทมเพลตที่เหมาะกับคุณ  
1. **ปรับใช้** - ปรับใช้และตรวจสอบว่าใช้งานได้  
1. **ปรับแต่ง** - แก้ไขและปรับปรุงให้เป็นของคุณ!

ตลอดเวิร์กช็อปนี้ คุณจะได้รู้จักกับเครื่องมือและเวิร์กโฟลว์สำหรับนักพัฒนาพื้นฐาน เพื่อช่วยให้คุณปรับปรุงการเดินทางในการพัฒนาอย่างครบวงจร

## ภาพรวมเวิร์กช็อป

**ระยะเวลา:** 3-4 ชั่วโมง  
**ระดับ:** ผู้เริ่มต้นถึงระดับกลาง  
**ความรู้เบื้องต้น:** คุ้นเคยกับ Azure, แนวคิด AI, VS Code และเครื่องมือบรรทัดคำสั่ง

นี่คือเวิร์กช็อปเชิงปฏิบัติที่คุณเรียนรู้จากการปฏิบัติ เมื่อทำแบบฝึกหัดครบถ้วน เราแนะนำให้ทบทวนหลักสูตร AZD สำหรับผู้เริ่มต้นเพื่อเรียนรู้ต่อเนื่องในเรื่องแนวปฏิบัติที่ดีที่สุดเกี่ยวกับความปลอดภัยและประสิทธิภาพ

| เวลา| โมดูล  | วัตถุประสงค์ |
|:---|:---|:---|
| 15 นาที | แนะนำ | วางแผน ทำความเข้าใจเป้าหมาย |
| 30 นาที | เลือกเทมเพลต AI | เรียนรู้ตัวเลือกและเลือกตัวเริ่มต้น |
| 30 นาที | ตรวจสอบเทมเพลต AI | ปรับใช้โซลูชันเริ่มต้นสู่ Azure |
| 30 นาที | วิเคราะห์เทมเพลต AI | ศึกษาโครงสร้างและการตั้งค่า |
| 30 นาที | กำหนดค่าเทมเพลต AI | เปิดใช้งานและทดลองฟีเจอร์ที่มี |
| 30 นาที | ปรับแต่งเทมเพลต AI | ปรับเทมเพลตให้ตรงกับความต้องการ |
| 30 นาที | รื้อถอนโครงสร้างพื้นฐาน | ทำความสะอาดและปล่อยทรัพยากร |
| 15 นาที | สรุปและขั้นตอนถัดไป | แหล่งเรียนรู้ ความท้าทายเวิร์กช็อป |
| | |

## สิ่งที่คุณจะได้เรียนรู้

คิดว่าเทมเพลต AZD เป็นสนามทดลองเรียนรู้เพื่อสำรวจความสามารถและเครื่องมือต่างๆ สำหรับการพัฒนาแบบครบวงจรบน Microsoft Foundry หลังจากเวิร์กช็อปนี้ คุณจะเข้าใจความรู้สึกเชิงสัญชาตญาณเกี่ยวกับเครื่องมือและแนวคิดต่างๆ ในบริบทนี้

| แนวคิด  | วัตถุประสงค์ |
|:---|:---|
| **Azure Developer CLI** | เข้าใจคำสั่งของเครื่องมือและเวิร์กโฟลว์ |
| **เทมเพลต AZD**| เข้าใจโครงสร้างโปรเจกต์และการตั้งค่า |
| **Azure AI Agent**| การจัดเตรียมและปรับใช้โปรเจกต์ Microsoft Foundry |
| **Azure AI Search**| เปิดใช้งานวิศวกรรมบริบทด้วยเอเย่นต์ |
| **การสังเกตการณ์**| ศึกษาการติดตาม การตรวจสอบ และการประเมินผล |
| **การทดสอบ Red Team**| ศึกษาการทดสอบและการป้องกันการโจมตี |
| | |
