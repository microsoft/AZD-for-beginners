# AZD สำหรับนักพัฒนา AI

!!! info "ผู้เข้าร่วมสำหรับเวิร์กชอปนี้"
   
    เวิร์กชอปนี้เหมาะสำหรับใครก็ตามที่ต้องการสร้างแอปพลิเคชัน AI Agentic บน Microsoft Foundry โดยใช้เครื่องมือและเวิร์กโฟลว์ที่ดีที่สุดในระดับเดียวกัน เวิร์กชอปนี้เหมาะสำหรับทั้งผู้เริ่มต้นและนักพัฒนาที่มีประสบการณ์ซึ่งมีความคุ้นเคยกับ Azure, แนวคิด AI และโค้ด Python โปรดตรวจสอบข้อกำหนดเบื้องต้นก่อนเริ่มต้น

    **ระยะเวลา:** 3-4 ชั่วโมง  
    **ระดับ:** เริ่มต้นถึงกลาง  

---


## ข้อกำหนดเบื้องต้น

- บัญชี GitHub ส่วนตัว → [สร้างบัญชีได้ฟรี](https://github.com/signup)
- บัญชี Azure ที่ใช้งานอยู่ → [สร้างบัญชีได้ฟรี](https://aka.ms/free)
- ความคุ้นเคยกับแนวคิด AI, การพัฒนา Python, เครื่องมือบรรทัดคำสั่ง
- มีประสบการณ์กับเครื่องมือและเวิร์กโฟลว์ของ VS Code และ GitHub

---

## เป้าหมายการเรียนรู้

หลังจากเวิร์กชอปนี้ คุณจะรู้วิธีที่จะ:

- [X] **ใช้งาน Azure Developer CLI** → เพื่อจัดเตรียม, ติดตั้ง, และจัดการทรัพยากร Azure ของคุณ
- [X] **ใช้เทมเพลตแอป AI** → เพื่อเริ่มต้นโครงการ AI Agent บน Microsoft Foundry อย่างรวดเร็ว
- [X] **ตั้งค่าเทมเพลต** → สำหรับคุณสมบัติต่างๆ เช่น Red Teaming, การประเมินผล, การติดตาม และการตรวจสอบ
- [X] **แยกวิเคราะห์และสำรวจ** → เทมเพลต AZD และเวิร์กโฟลว์โดยใช้ GitHub Copilot
- [X] **ปรับแต่งและติดตั้งใหม่** → เทมเพลตตัวอย่าง AI Agent ด้วยความต้องการของคุณเอง

---

## โครงร่างเวิร์กชอป

เวิร์กชอปถูกจัดเป็นโมดูลแต่ละโมดูลมีกิจกรรมแล็บแบบปฏิบัติ ดังนี้:

1. **แนะนำ** - เข้าใจว่าเทมเพลต AZD คืออะไร และมันทำงานอย่างไร
1. **การเลือก** - ค้นหาเทมเพลต "เริ่มต้น" ที่เหมาะสมสำหรับโครงการ AI ครั้งต่อไปของคุณ
1. **การตรวจสอบ** - ติดตั้งและสำรวจเทมเพลตเพื่อยืนยันว่าใช้งานได้ตามที่โฆษณาไว้
1. **การแยกวิเคราะห์** - แยกเทมเพลตออกเพื่อทำความเข้าใจส่วนประกอบและการตั้งค่าของมัน
1. **การตั้งค่า** - เรียนรู้การเปิดใช้งานและใช้ฟีเจอร์ต่างๆ ของ Microsoft Foundry
1. **การปรับแต่ง** - สร้างแอปพลิเคชันใหม่ด้วยข้อมูลและความต้องการของคุณเอง
1. **การรื้อถอน** - เรียนรู้วิธีการลบโครงสร้างพื้นฐานและกู้คืนทรัพยากรที่ถูกลบนุ่มนวล
