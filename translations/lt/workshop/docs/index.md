# AZD praktinis seminaras AI kūrėjams

> [!IMPORTANT]  
> **Šis seminaras turi seminaro vadovą, kurį galite peržiūrėti savo vietiniame naršyklės lange. Norėdami pradėti, turite paleisti GitHub Codespaces šiame repo—tada palaukite, kol pamatysite aktyvų VS Code terminalą ir įveskite:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Turėtumėte pamatyti iškylančią dialogo dėžutę, leidžiančią atidaryti peržiūros puslapį naršyklėje.**

Sveiki atvykę į praktinį seminarą, skirtą susipažinti su Azure Developer CLI (AZD), orientuojantis į AI programų diegimą. Šis seminaras padės jums praktiškai suprasti AZD šablonus per 3 žingsnius:

1. **Atranka** - raskite jums tinkamą šabloną.
1. **Diegimas** - įdiekite ir patikrinkite, kad tai veikia
1. **Pritaikymas** - modifikuokite ir tobulinkite, kad tai taptų jūsų!

Per šį seminarą taip pat susipažinsite su pagrindiniais kūrėjų įrankiais ir darbo srautais, kurie padės supaprastinti visą jūsų kūrimo procesą.

## Seminario apžvalga

**Trukmė:** 3–4 valandos  
**Lygis:** nuo pradedančiojo iki vidutinio  
**Reikalavimai:** susipažinimas su Azure, AI sąvokomis, VS Code ir komandų eilutės įrankiais.

Tai praktinis seminaras, kuriame mokotės darydami. Baigę užduotis, rekomenduojame peržiūrėti kursą AZD pradedantiesiems, kad tęstumėte mokymąsi apie saugumą ir produktyvumą bei gerąsias praktikas.

| Laikas| Modulis  | Tikslas |
|:---|:---|:---|
| 15 mins | Įvadas | Nustatyti kontekstą, suprasti tikslus |
| 30 mins | Pasirinkite AI šabloną | Išnagrinėkite galimybes ir pasirinkite pradinį šabloną | 
| 30 mins | Patvirtinkite AI šabloną | Įdiekite numatytą sprendimą į Azure |
| 30 mins | Išanalizuokite AI šabloną | Tyrinėkite struktūrą ir konfigūraciją |
| 30 mins | Konfigūruokite AI šabloną | Aktyvuokite ir išbandykite prieinamas funkcijas |
| 30 mins | Pritaikykite AI šabloną | Priderinkite šabloną pagal savo poreikius |
| 30 mins | Išardyti infrastruktūrą | Išvalyti ir atlaisvinti išteklius |
| 15 mins | Apibendrinimas ir tolesni žingsniai | Mokymosi ištekliai, seminaro iššūkis |
| | |

## Ką sužinosite

Laikykite AZD šabloną kaip mokymosi smėlio dėžę, skirtą nagrinėti įvairias galimybes ir įrankius viso kūrimo proceso Microsoft Foundry aplinkoje. Baigę šį seminarą turėtumėte intuiciškai suprasti įvairius įrankius ir koncepcijas šiame kontekste.

| Koncepcija  | Tikslas |
|:---|:---|
| **Azure Developer CLI** | Suprasti įrankio komandas ir darbo eigas|
| **AZD šablonai**| Suprasti projekto struktūrą ir konfigūraciją|
| **Azure AI Agent**| Paruošti ir įdiegti Microsoft Foundry projektą  |
| **Azure AI Search**| Įgalinti kontekstinę inžineriją su agentais |
| **Observabilumas**| Tyrinėkite sekimą, stebėjimą ir vertinimus |
| **Red Teaming**| Tyrinėkite priešišką testavimą ir švelninimo priemones |
| | |
