# AI ডেভেলপারদের জন্য AZD কর্মশালা

Azure Developer CLI (AZD) ব্যবহার করে AI অ্যাপ্লিকেশন ডিপ্লয়মেন্টে ফোকাস করে শেখার জন্য এই হ্যান্ডস-অন কর্মশালায় আপনাকে স্বাগতম। এই কর্মশালা আপনাকে 3 ধাপে AZD টেমপ্লেটগুলোর প্রয়োগমুখী বোঝাপড়া অর্জন করতে সাহায্য করে:

1. **Discovery** - আপনার জন্য সঠিক টেমপ্লেট খুঁজুন।
1. **Deployment** - ডিপ্লয় করুন এবং যাচাই করুন যে এটি কাজ করছে
1. **Customization** - পরিবর্তন করুন এবং পুনরায় চেষ্টা করে এটিকে আপনার মত করুন!

এই কর্মশালার সময়কালে, আপনাকে মূল ডেভেলপার টুল ও ওয়ার্কফ্লোগুলোর সাথে পরিচয় করানো হবে, যাতে আপনার পূর্ণাঙ্গ ডেভেলপমেন্ট যাত্রাকে সহজতর করা যায়।

<br/>

## ব্রাউজার-ভিত্তিক নির্দেশিকা

কর্মশালা পাঠগুলো Markdown এ রয়েছে। আপনি GitHub-এ সরাসরি এগুলো ব্রাউজ করতে পারেন - অথবা নিচের স্ক্রিনশটের মতো ব্রাউজার-ভিত্তিক প্রিভিউ চালু করতে পারেন।

![কর্মশালা](../../../translated_images/bn/workshop.75906f133e6f8ba0.webp)

এই অপশনটি ব্যবহার করতে - রিপোজিটরিটিকে আপনার প্রোফাইলে ফরক করুন, এবং GitHub Codespaces চালু করুন। একবার VS Code টার্মিনাল সক্রিয় হলে, এই কমান্ড টাইপ করুন:

```bash title="" linenums="0"
mkdocs serve > /dev/null 2>&1 &
```

কয়েক সেকেন্ডের মধ্যে একটি পপ-আপ ডায়ালগ দেখাবে। `Open in browser` অপশনটি নির্বাচন করুন। ওয়েব-ভিত্তিক গাইডটি এখন একটি নতুন ব্রাউজার ট্যাবে খুলে যাবে। এই প্রিভিউয়ের কিছু সুবিধা:

1. **Built-in search** - কীওয়ার্ড বা পাঠ দ্রুত খুঁজে পেতে পারবেন।
1. **Copy icon** - কোডব্লকের উপর হোভার করলে এই অপশনটি দেখা যাবে
1. **Theme toggle** - ডার্ক এবং লাইট থিমের মধ্যে স্যুইচ করুন
1. **Get help** - ফুটারে Discord আইকনে ক্লিক করে যোগদান করুন!

## আপনি কী শিখবেন

AZD টেমপ্লেটকে একটি শেখার স্যান্ডবক্স হিসেবে ভাবুন যাতে আপনি Microsoft Foundry-এ পূর্ণাঙ্গ ডেভেলপমেন্টের বিভিন্ন সক্ষমতা ও টুল অন্বেষণ করতে পারেন। এই কর্মশালার শেষে, আপনাকে এই প্রসঙ্গে বিভিন্ন টুল ও ধারণাগুলোর একটি প্রাথমিক ও বোধগম্য ধারণা থাকা উচিত।

| Concept  | Objective |
|:---|:---|
| **Azure Developer CLI** | টুল কমান্ড ও ওয়ার্কফ্লোগুলো বুঝুন |
| **AZD Templates**| প্রজেক্ট স্ট্রাকচার ও কনফিগারেশন বোঝা |
| **Azure AI Agent**| Microsoft Foundry প্রজেক্ট প্রোভিশন ও ডিপ্লয় করা |
| **Azure AI Search**| এজেন্টদের সাথে কনটেক্সট ইঞ্জিনিয়ারিং সক্ষম করা |
| **Observability**| ট্রেসিং, মনিটরিং এবং মূল্যায়ন অন্বেষণ করা |
| **Red Teaming**| বিরোধী (রেড টিমিং) পরীক্ষা ও প্রতিরোধ অন্বেষণ করা |

## কর্মশালার কাঠামো

কর্মশালাটি আপনাকে টেমপ্লেট সনাক্তকরণ থেকে শুরু করে ডিপ্লয়মেন্ট, বিশ্লেষণ এবং কাস্টমাইজেশনের দিকে নিয়ে যাওয়ার জন্য গঠন করা হয়েছে - এবং এর ভিত্তি হিসেবে অফিসিয়াল [Getting Started with AI Agents](https://github.com/Azure-Samples/get-started-with-ai-agents) স্টার্টার টেমপ্লেট ব্যবহৃত হয়েছে।

### [মডিউল 1: AI টেমপ্লেট নির্বাচন](docs/instructions/1-Select-AI-Template.md) (30 মিনিট)

- AI টেমপ্লেট কী?
- AI টেমপ্লেট কোথায় পাব?
- কীভাবে AI এজেন্ট তৈরি শুরু করব?
- **ল্যাব**: GitHub Codespaces দিয়ে দ্রুত শুরু

### [মডিউল 2: AI টেমপ্লেট যাচাই](docs/instructions/2-Validate-AI-Template.md) (30 মিনিট)

- AI টেমপ্লেট আর্কিটেকচার কী?
- AZD ডেভেলপমেন্ট ওয়ার্কফ্লো কী?
- AZD ডেভেলপমেন্টে কীভাবে সাহায্য পাব?
- **ল্যাব**: AI এজেন্ট টেমপ্লেট ডিপ্লয় ও ভ্যালিডেট

### [মডিউল 3: AI টেমপ্লেট বিশ্লেষণ](docs/instructions/3-Deconstruct-AI-Template.md) (30 মিনিট)

- আপনার পরিবেশ অন্বেষণ করুন `.azure/` 
- আপনার রিসোর্স সেটআপ অন্বেষণ করুন `infra/` 
- আপনার AZD কনফিগারেশন অন্বেষণ করুন `azure.yaml`s
- **ল্যাব**: পরিবেশ ভ্যারিয়েবল পরিবর্তন করুন ও পুনরায় ডিপ্লয় করুন

### [মডিউল 4: AI টেমপ্লেট কনফিগার](docs/instructions/4-Configure-AI-Template.md) (30 মিনিট)
- অন্বেষণ: রিট্রিভাল অগমেন্টেড জেনারেশন
- অন্বেষণ: এজেন্ট মূল্যায়ন ও রেড টিমিং
- অন্বেষণ: ট্রেসিং ও মনিটরিং
- **ল্যাব**: AI এজেন্ট + Observability অন্বেষণ

### [মডিউল 5: AI টেমপ্লেট কাস্টমাইজ](docs/instructions/5-Customize-AI-Template.md) (30 মিনিট)
- নির্ধারণ: সিনারিও প্রয়োজনীয়তা সহ PRD
- কনফিগার: AZD-এর জন্য পরিবেশ ভ্যারিয়েবলসমূহ
- বাস্তবায়ন: অতিরিক্ত কাজের জন্য লাইফসাইকেল হুক
- **ল্যাব**: আমার সিনারিওর জন্য টেমপ্লেট কাস্টমাইজ করুন

### [মডিউল 6: ইনফ্রাস্ট্রাকচার টিয়ারডাউন](docs/instructions/6-Teardown-Infrastructure.md) (30 মিনিট)
- পুনরায় স্মরণ: AZD টেমপ্লেট কী?
- পুনরায় স্মরণ: কেন Azure Developer CLI ব্যবহার করবেন?
- পরবর্তী ধাপ: একটি ভিন্ন টেমপ্লেট চেষ্টা করুন!
- **ল্যাব**: ইনফ্রাস্ট্রাকচার ডিপ্রোভিশন ও ক্লিনআপ

## কর্মশালা চ্যালেঞ্জ

আরও চ্যালেঞ্জ নিতে চান? এখানে কিছু প্রকল্প পরামর্শ দেয়া হল - অথবা আপনার আইডিয়া আমাদের সাথে শেয়ার করুন!!

| Project | Description |
|:---|:---|
|1. **একটি জটিল AI টেমপ্লেট বিশ্লেষণ করুন** | আমরা যে ওয়ার্কফ্লো এবং টুলগুলো দিয়েছি তা ব্যবহার করুন এবং দেখবেন কি আপনি একটি ভিন্ন AI সলিউশন টেমপ্লেট ডিপ্লয়, ভ্যালিডেট এবং কাস্টমাইজ করতে পারেন। _আপনি কি শিখলেন?_|
|2. **আপনার সিনারিও অনুযায়ী কাস্টমাইজ করুন**  | একটি ভিন্ন সিনারিওর জন্য একটি PRD (Product Requirements Document) লিখে দেখুন। এরপর Agent Model-এ আপনার টেমপ্লেট রিপোতে GitHub Copilot ব্যবহার করে এটিতে একটি কাস্টমাইজেশন ওয়ার্কফ্লো জেনারেট করতে বলুন। _আপনি কি শিখলেন? এগুলো কীভাবে উন্নত করা যায়?_|
