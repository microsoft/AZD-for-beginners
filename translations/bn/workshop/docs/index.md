# AI ডেভেলপারদের জন্য AZD কর্মশালা

> মার্চ ২০২৬-এ `azd 1.23.12` এর বিরুদ্ধে যাচাই করা হয়েছে।

> [!IMPORTANT]  
> **এই কর্মশালাটি একটি কর্মশালা গাইড দ্বারা সজ্জিত যা আপনি আপনার স্থানীয় ব্রাউজারে দেখতে পারেন। শুরু করতে, GitHub Codespaces, একটি ডেভ কন্টেইনার, অথবা MkDocs ইনস্টলকৃত একটি লোকাল ক্লোনে রিপোটি খুলুন—তারপর একটি সক্রিয় VS Code টার্মিনাল দেখানো পর্যন্ত অপেক্ষা করুন এবং টাইপ করুন:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **আপনি একটি পপ-আপ ডায়ালগ দেখতে পাবেন যা আপনাকে ব্রাউজারে প্রিভিউ পৃষ্ঠা খোলার অনুমতি দেবে।**

Azure Developer CLI (AZD) শেখার হাতে কলম কর্মশালায় আপনাকে স্বাগতম, যা AI অ্যাপ্লিকেশন ডিপ্লয়মেন্টে মনোযোগ দেয়। এই কর্মশালা আপনাকে ৩ ধাপে AZD টেমপ্লেট সম্পর্কে প্রয়োগমুখী জ্ঞান অর্জনে সাহায্য করবে:

1. **আবিষ্কার** - আপনার জন্য সঠিক টেমপ্লেট খুঁজে বের করুন।
1. **ডিপ্লয়মেন্ট** - ডিপ্লয় করে যাচাই করুন যে এটি কাজ করছে
1. **কাস্টমাইজেশন** - পরিবর্তন করুন এবং নিজের মতো করে নিন!

এই কর্মশালা চলাকালে, আপনাকে মূল ডেভেলপার টুল এবং ওয়ার্কফ্লো-র সঙ্গে পরিচয় করানো হবে, যা আপনার এন্ড-টু-এন্ড ডেভেলপমেন্ট যাত্রাকে সুশৃঙ্খল করতে সাহায্য করবে।

## কর্মশালার সারসংক্ষেপ

**সময়কাল:** ৩-৪ ঘণ্টা  
**স্তর:** শিক্ষানবীশ থেকে মধ্যবর্তী  
**প্রারম্ভিক জ্ঞান:** Azure, AI ধারণা, VS Code ও কমান্ড-লাইন টুল সম্পর্কে পরিচিতি।

এটি একটি হাতে কলম কর্মশালা যেখানে আপনি কার্যক্রম করে শিখবেন। একবার আপনি অনুশীলনগুলি সম্পন্ন করলে, আমরা সুপারিশ করি AZD For Beginners কারিকুলাম পর্যালোচনা করতে যাতে আপনি নিরাপত্তা এবং উৎপাদনশীলতা সেরা অনুশীলনে আপনার শেখার যাত্রা অব্যাহত রাখতে পারেন।

| সময় | মডিউল  | উদ্দেশ্য |
|:---|:---|:---|
| ১৫ মিনিট | পরিচিতি | মঞ্চ নির্ধারণ, লক্ষ্য বুঝুন |
| ৩০ মিনিট | AI টেমপ্লেট নির্বাচন | বিকল্পগুলি অন্বেষণ করুন এবং শুরু করুন | 
| ৩০ মিনিট | AI টেমপ্লেট যাচাই | Azure-এ ডিফল্ট সমাধান ডিপ্লয় করুন |
| ৩০ মিনিট | AI টেমপ্লেট বিশ্লেষণ | কাঠামো এবং কনফিগারেশন অন্বেষণ করুন |
| ৩০ মিনিট | AI টেমপ্লেট কনফিগার করা | সক্রিয় করুন এবং উপলব্ধ বৈশিষ্ট্য চেষ্টা করুন |
| ৩০ মিনিট | AI টেমপ্লেট কাস্টমাইজ করা | আপনার প্রয়োজন অনুযায়ী টেমপ্লেট অভিযোজিত করুন |
| ৩০ মিনিট | অবকাঠামো ধ্বংস | পরিষ্কার করুন এবং সম্পদ মুক্ত করুন |
| ১৫ মিনিট | সমাপ্তি ও পরবর্তী ধাপ | শেখার সম্পদ, কর্মশালা চ্যালেঞ্জ |

## আপনি যা শিখবেন

AZD টেমপ্লেটকে একটি শেখার স্যান্ডবক্স হিসেবে ভাবুন যা Microsoft Foundry-এ এন্ড-টু-এন্ড ডেভেলপমেন্টের বিভিন্ন সক্ষমতা এবং টুলস অন্বেষণ করতে দেয়। এই কর্মশালা শেষ হওয়ার আগেই, আপনি এই প্রসঙ্গে বিভিন্ন টুল এবং ধারণার জন্য একটি স্বজ্ঞাত ধারণা পাবেন।

| ধারণা  | উদ্দেশ্য |
|:---|:---|
| **Azure Developer CLI** | টুল কমান্ড এবং ওয়ার্কফ্লো বোঝা |
| **AZD টেমপ্লেট**| প্রকল্প কাঠামো এবং কনফিগারেশন বোঝা |
| **Azure AI Agent**| Microsoft Foundry প্রকল্প প্রদান ও ডিপ্লয়মেন্ট  |
| **Azure AI Search**| এজেন্টের মাধ্যমে প্রসঙ্গ প্রকৌশল সক্ষম করা |
| **পর্যবেক্ষণযোগ্যতা**| ট্রেসিং, মনিটরিং এবং মূল্যায়ন অন্বেষণ |
| **রেড টিমিং**| প্রতিকূল পরীক্ষা এবং প্রতিকার অন্বেষণ |
| | |
