# AZD ফর AI ডেভেলপারদের কর্মশালা

> যাচাই করা হয়েছে `azd 1.25.6` (জুন 2026)-এ।

> [!IMPORTANT]  
> **এই কর্মশালায় একটি ওয়ার্কশপ গাইড অন্তর্ভুক্ত আছে যা আপনি আপনার লোকাল ব্রাউজারে প্রিভিউ করতে পারবেন। শুরু করতে, রিপোটি GitHub Codespaces, একটি dev container, অথবা MkDocs ইনস্টল করা একটি লোকাল ক্লোনে খুলুন—এরপর একটি সক্রিয় VS Code টার্মিনাল দেখা পর্যন্ত অপেক্ষা করুন এবং টাইপ করুন:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **আপনি ব্রাউজারে প্রিভিউ পৃষ্ঠা খুলতে একটি পপ-আপ ডায়ালগ দেখতে পাবেন।**

স্বাগত — Azure Developer CLI (AZD) শেখার এই হাতে-কলমে কর্মশালায়, যেখানে ফোকাস করা হয়েছে AI অ্যাপ্লিকেশন মোতায়েনের ওপর। এই কর্মশালা আপনাকে AZD টেমপ্লেটগুলো ব্যবহারিকভাবে বুঝতে সাহায্য করবে 3 ধাপে:

1. **অন্বেষণ** - আপনার জন্য উপযুক্ত টেমপ্লেট খুঁজুন।
1. **ডিপ্লয়মেন্ট** - ডিপ্লয় করুন এবং কাজ করছে কিনা যাচাই করুন
1. **কাস্টমাইজেশন** - এটি আপনার মতো করে পরিবর্তন ও পুনরাবৃত্তি করুন!

এই কর্মশালার সময়কালে, আপনাকে কোর ডেভেলপার টুল এবং ওয়ার্কফ্লো সম্পর্কে পরিচয় করিয়ে দেওয়া হবে, যাতে আপনার এন্ড-টু-এন্ড ডেভেলপমেন্ট যাত্রা সহজ হয়।

## কর্মশালা ওভারভিউ

**সময়কাল:** 3-4 ঘন্টা  
**স্তর:** নবীন থেকে মধ্যবর্তী  
**প্রয়োজনীয়তা:** Azure, AI ধারণা, VS Code ও কমান্ড-লাইন টুলগুলোর সঙ্গে পরিচিতি।

এটি একটি হাতে-কলমে কর্মশালা যেখানে আপনি কাজ করে শিখবেন। একবার আপনি এক্সারসাইজসমূহ সম্পন্ন করার পরে, আমরা সুপারিশ করি যে আপনি AZD নবাগতদের জন্য পাঠ্যক্রমটি পর্যালোচনা করুন যাতে Security এবং Productivity শ্রেষ্ঠ অনুশীলনসমূহ সম্পর্কে আপনার শেখার যাত্রা অব্যাহত থাকে।

| সময়| মডিউল  | উদ্দেশ্য |
|:---|:---|:---|
| 15 মিনিট | পরিচিতি | পরিপ্রেক্ষিত নির্ধারণ, লক্ষ্যগুলো বুঝুন |
| 30 মিনিট | AI টেমপ্লেট নির্বাচন | বিকল্পগুলো অন্বেষণ করে স্টার্টার নির্বাচন করুন | 
| 30 মিনিট | AI টেমপ্লেট যাচাই | ডিফল্ট সলিউশন Azure-এ ডিপ্লয় করুন |
| 30 মিনিট | AI টেমপ্লেট বিশ্লেষণ | স্ট্রাকচার এবং কনফিগারেশন অন্বেষণ করুন |
| 30 মিনিট | AI টেমপ্লেট কনফিগার করুন | উপলব্ধ ফিচারগুলো সক্রিয় করে চেষ্টা করুন |
| 30 মিনিট | AI টেমপ্লেট কাস্টমাইজ করুন | টেমপ্লেটটি আপনার চাহিদা অনুযায়ী অভিযোজিত করুন |
| 30 মিনিট | ইনফ্রাস্ট্রাকচার অপসারণ | পরিচ্ছন্ন করুন এবং রিসোর্স মুক্ত করুন |
| 15 মিনিট | সমাপ্তি ও পরবর্তী ধাপ | শেখার রিসোর্স, ওয়ার্কশপ চ্যালেঞ্জ |

## আপনি কী শিখবেন

AZD টেমপ্লেটকে একটি শিক্ষণীয় স্যান্ডবক্স হিসেবে ভাবুন, যা Microsoft Foundry-এ এন্ড-টু-এন্ড ডেভেলপমেন্টের বিভিন্ন সক্ষমতা ও টুল অন্বেষণে সাহায্য করে। কর্মশালার শেষে, আপনি এই প্রেক্ষাপটে বিভিন্ন টুল ও ধারণা সম্পর্কে একটি স্বজ্ঞাত ধারণা পাবেন।

| Concept  | Objective |
|:---|:---|
| **Azure Developer CLI** | টুলের কমান্ড এবং ওয়ার্কফ্লো বুঝুন|
| **AZD Templates**| প্রকল্পের স্ট্রাকচার ও কনফিগারেশন বুঝুন|
| **Azure AI Agent**| Microsoft Foundry প্রকল্প প্রোভিশন ও ডিপ্লয় করুন  |
| **Azure AI Search**| এজেন্টদের সঙ্গে কনটেক্সট ইঞ্জিনিয়ারিং সক্ষম করুন |
| **Observability**| ট্রেসিং, মনিটরিং এবং মূল্যায়ন অন্বেষণ করুন |
| **Red Teaming**| বৈরী টেস্টিং এবং প্রশমন কৌশল অন্বেষণ করুন |
| | |
