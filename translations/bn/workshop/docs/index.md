# AI ডেভেলপারদের জন্য AZD কর্মশালা  

> জুলাই ২০২৬-এ `azd 1.27.1` দিয়ে যাচাইকৃত।  

> [!IMPORTANT]  
> **এই কর্মশালা একটি ওয়ার্কশপ গাইড দিয়েই পরিচালিত, যেটি আপনি আপনার লোকাল ব্রাউজারে পূর্বদর্শন করতে পারেন। শুরু করতে, GitHub Codespaces, একটি dev container, বা MkDocs ইনস্টলকৃত একটি লোকাল ক্লোনে রিপোটি খুলুন—তারপর সক্রিয় VS Code টার্মিনাল দেখালে টাইপ করুন:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **আপনাকে একটি পপ-আপ ডায়ালগ দেখা উচিত যা ব্রাউজারে পূর্বদর্শন পৃষ্ঠা খুলতে বলবে।**  

Azure Developer CLI (AZD) শেখার জন্য AI অ্যাপ্লিকেশন ডিপ্লয়মেন্টের উপর কেন্দ্র করে হাতে-কলমে এই কর্মশালায় স্বাগতম। এই কর্মশালাটি আপনাকে ৩ ধাপে AZD টেমপ্লেটগুলি ব্যবহারিকভাবে বুঝতে সাহায্য করবে:  

১. **আবিষ্কার** - আপনার জন্য সঠিক টেমপ্লেট খুঁজুন।  
১. **ডিপ্লয়মেন্ট** - ডিপ্লয় করুন এবং কাজ করছে কিনা যাচাই করুন।  
১. **কাস্টমাইজেশন** - আপনার মতো করে পরিবর্তন করুন এবং পুনরাবৃত্তি করুন!  

এই কর্মশালার সময়, আপনি মূল ডেভেলপার টুল এবং ওয়ার্কফ্লো-র সঙ্গে পরিচিত হবেন, যা আপনার পূর্ণাঙ্গ উন্নয়ন যাত্রাকে সহজতর করবে।

## কর্মশালার সারসংক্ষেপ  

**সময়কাল:** ৩-৪ ঘণ্টা  
**স্তর:** নবীন থেকে মধ্যবর্তী  
**প্রয়োজনীয়তা:** Azure, AI ধারণা, VS Code ও কমান্ড-লাইন টুলগুলোর পরিচিতি।  

এটি একটি হাতে-কলমে কর্মশালা যেখানে আপনি কাজ করে শিখবেন। একবার ব্যায়ামগুলি সম্পন্ন হলে, আমরা সুপারিশ করি AZD For Beginners কারিকুলাম পর্যালোচনা করতে যাতে আপনি নিরাপত্তা এবং উৎপাদনশীলতার সেরা অভ্যাসগুলি সম্পর্কে আপনার শেখার যাত্রা চালিয়ে যেতে পারেন।  

| সময়| মডিউল  | উদ্দেশ্য |  
|:---|:---|:---|

| ৩০ মিনিট | AI টেমপ্লেট নির্বাচন | বিকল্পগুলি অন্বেষণ করে শুরু করুন |   
| ৩০ মিনিট | AI টেমপ্লেট যাচাই | Azure-এ ডিফল্ট সমাধান ডিপ্লয় করুন |  
| ৩০ মিনিট | AI টেমপ্লেট বিশ্লেষণ | কাঠামো ও কনফিগারেশন অন্বেষণ |  
| ৩০ মিনিট | AI টেমপ্লেট কনফিগার | সক্রিয় করুন এবং উপলব্ধ বৈশিষ্ট্যগুলি পরীক্ষা করুন |  
| ৩০ মিনিট | AI টেমপ্লেট কাস্টমাইজ | টেমপ্লেট আপনার প্রয়োজন অনুযায়ী মানিয়ে নিন |  
| ৩০ মিনিট | অবকাঠামো অপসারণ | ক্লিনআপ করুন এবং রিসোর্স মুক্ত করুন |  
| ১৫ মিনিট | উপসংহার ও পরবর্তী ধাপ | শেখার রিসোর্স, কর্মশালা চ্যালেঞ্জ |

## আপনি যা শিখবেন  

AZD টেমপ্লেটকে একটি শেখার স্যান্ডবক্স হিসেবে ভাবুন যা মাইক্রোসফট ফাউন্ড্রির জন্য পূর্ণাঙ্গ উন্নয়নের বিভিন্ন সক্ষমতা ও টুলগুলি অন্বেষণ করার সুযোগ দেয়। কর্মশালা শেষে, আপনাকে এই প্রসঙ্গে বিভিন্ন টুল এবং ধারণাগুলির একটি অন্তর্দৃষ্টিপূর্ণ ধারণা থাকা উচিত।  

| ধারণা  | উদ্দেশ্য |  
|:---|:---|  
| **Azure Developer CLI** | টুল কমান্ড ও ওয়ার্কফ্লো বোঝা |  
| **AZD টেমপ্লেট**| প্রজেক্ট কাঠামো ও কনফিগার বোঝা |  
| **Azure AI Agent**| Microsoft Foundry প্রজেক্ট প্রদান ও ডিপ্লয় |  
| **Azure AI Search**| এজেন্টের মাধ্যমে কনটেক্সট ইঞ্জিনিয়ারিং সক্রিয় করা |  
| **Observability**| ট্রেসিং, মনিটরিং এবং মূল্যায়ন অন্বেষণ |  
| **Red Teaming**| প্রতিদ্বন্দ্বিতামূলক টেস্টিং ও প্রতিকার অন্বেষণ |
