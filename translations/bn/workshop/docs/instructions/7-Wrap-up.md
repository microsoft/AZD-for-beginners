## 2. আপনি যে মূল ধারণাগুলি আয়ত্ত করেছেন

### Azure Developer CLI (azd)
- `azd init` - টেমপ্লেট থেকে প্রজেক্ট ইনিশিয়ালাইজ করুন
- `azd up` - এক কমান্ডে প্রোভিশন এবং ডিপ্লয় করুন
- `azd env` - এনভায়রনমেন্ট ভেরিয়েবল পরিচালনা করুন
- `azd down` - রিসোর্সগুলো নিরাপদে পরিষ্কার করুন

### AZD টেমপ্লেটের কাঠামো
- `azure.yaml` - কনফিগারেশন এবং ডিপ্লয়মেন্ট হুক
- `infra/` - Bicep ইনফ্রাস্ট্রাকচার-অ্যাস-কোড
- `src/` - অ্যাপ্লিকেশন সোর্স কোড
- `scripts/` - অটোমেশন এবং সেটআপ স্ক্রিপ্ট

### Microsoft Foundry ক্ষমতাসমূহ
- বুদ্ধিমান সহকারীদের জন্য AI Agent সার্ভিস
- জ্ঞান আহরণের জন্য File Search এবং Azure AI Search
- অন্তর্নির্মিত ট্রেসিং, মনিটরিং, এবং মূল্যায়ন
- AI নিরাপত্তা যাচাইয়ের জন্য রেড টিমিং

## 4. কর্মশালা চ্যালেঞ্জ

আপনার দক্ষতা পরীক্ষা করতে প্রস্তুত? এগুলো চ্যালেঞ্জ চেষ্টা করুন:

| Challenge | Description | Difficulty |
|:---|:---|:---|
| **1. মাল্টি-এজেন্ট সিনারিও** | ভিন্ন একটি শিল্পের জন্য ডিপ্লয় এবং কাস্টমাইজ করুন (e.g., স্বাস্থ্যসেবা, ফাইন্যান্স) | ⭐⭐ |
| **2. কাস্টম নোলেজ বেস** | আপনার নিজস্ব ডেটা ফাইল যোগ করুন এবং একটি কাস্টম সার্চ ইনডেক্স তৈরি করুন | ⭐⭐⭐ |
| **3. প্রোডাকশন ডিপ্লয়মেন্ট** | `azd pipeline config` ব্যবহার করে GitHub Actions দিয়ে CI/CD সেট আপ করুন | ⭐⭐⭐⭐ |
