# অধ্যায় 3: কনফিগারেশন ও প্রমাণীকরণ

**📚 কোর্স**: [AZD For Beginners](../../README.md) | **⏱️ সময়**: 45-60 মিনিট | **⭐ জটিলতা**: মধ্যবর্তী

---

## সংক্ষিপ্ত বিবরণ

এই অধ্যায়ে Azure Developer CLI ডিপ্লয়মেন্টের জন্য পরিবেশ কনফিগারেশন, প্রমাণীকরণ প্যাটার্ন এবং নিরাপত্তা সেরা অনুশীলনগুলোর আলোচনা করা হয়েছে।

## শিখন লক্ষ্য

এই অধ্যায় সম্পন্ন করে, আপনি:
- AZD কনফিগারেশন হায়ারার্কি আয়ত্ত করবেন
- একাধিক পরিবেশ (dev, staging, prod) পরিচালনা করবেন
- ম্যানেজ্ড আইডেন্টিটিগুলোর মাধ্যমে সুরক্ষিত প্রমাণীকরণ বাস্তবায়ন করবেন
- পরিবেশ-নির্দিষ্ট সেটিং কনফিগার করবেন

## 🚀 দ্রুত শুরু

```bash
# একাধিক পরিবেশ তৈরি করুন
azd env new dev
azd env new staging
azd env new prod

# পরিবেশ পরিবর্তন করুন
azd env select prod

# পরিবেশ ভেরিয়েবলসমূহ সেট করুন
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# কনফিগারেশন দেখুন
azd env get-values
```

---

## 🔧 কনফিগারেশন শ্রেণিবিন্যাস

AZD নিম্নলিখিত ক্রমে সেটিংস প্রয়োগ করে (পরে থাকাগুলো আগেরগুলোকে ওভাররাইড করে):

1. **ডিফল্ট মান** (টেমপ্লেটে বিল্ট-ইন)
2. **azure.yaml** (প্রজেক্ট কনফিগারেশন)
3. **পরিবেশ পরিবর্তনশীল** (`azd env set`)
4. **কমান্ড-লাইন ফ্ল্যাগ** (`--location eastus`)

---

## 🔐 নিরাপত্তা সেরা অনুশীলন

```bash
# ম্যানেজড আইডেন্টিটি ব্যবহার করুন (প্রস্তাবিত)
azd env set AZURE_USE_MANAGED_IDENTITY true

# প্রমাণীকরণের অবস্থা পরীক্ষা করুন
azd auth whoami
az account show

# প্রয়োজন হলে পুনরায় প্রমাণীকরণ করুন
azd auth login
az login
```
