# অধ্যায় 3: কনফিগারেশন ও প্রমাণীকরণ

**📚 কোর্স**: [শুরুদের জন্য AZD](../../README.md) | **⏱️ সময়কাল**: 45-60 মিনিট | **⭐ জটিলতা**: মধ্যবর্তী

---

## সারসংক্ষেপ

এই অধ্যায়টি Azure Developer CLI ডিপ্লয়মেন্টের জন্য পরিবেশ কনফিগারেশন, প্রমাণীকরণ প্যাটার্ন এবং নিরাপত্তার সেরা অনুশীলনসমূহ কভার করে।

> জুন 2026-এ `azd 1.25.6` অনুযায়ী যাচাই করা হয়েছে।

## শেখার উদ্দেশ্য

এই অধ্যায়টি সম্পন্ন করার মাধ্যমে, আপনি:
- AZD কনফিগারেশন শ্রেণিবিন্যাসে দক্ষতা অর্জন করা
- একাধিক পরিবেশ (dev, staging, prod) পরিচালনা করা
- ম্যানেজড আইডেন্টিটিগুলির মাধ্যমে নিরাপদ প্রমাণীকরণ বাস্তবায়ন করা
- পরিবেশ-নির্দিষ্ট সেটিংস কনফিগার করা

## 🚀 দ্রুত শুরু

```bash
# একাধিক পরিবেশ তৈরি করুন
azd env new dev
azd env new staging
azd env new prod

# পরিবেশ পরিবর্তন করুন
azd env select prod

# পরিবেশ ভেরিয়েবল সেট করুন
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# কনফিগারেশন দেখুন
azd env get-values
```

## 🔧 কনফিগারেশন শ্রেণিবিন্যাস

AZD সেটিংস এই ক্রমে প্রয়োগ করে (পরে থাকাটি পূর্বতিকে ওভাররাইড করে):

1. **ডিফল্ট মান** (টেমপ্লেটগুলিতে অন্তর্নির্মিত)
2. **azure.yaml** (প্রকল্প কনফিগারেশন)
3. **পরিবেশ ভেরিয়েবল** (`azd env set`)
4. **কমান্ড-লাইন ফ্ল্যাগস** (`--location eastus`)

## 🔐 নিরাপত্তার সেরা অনুশীলন

```bash
# ম্যানেজড আইডেন্টিটি ব্যবহার করুন (প্রস্তাবিত)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD প্রমাণীকরণের অবস্থা পরীক্ষা করুন
azd auth status

# ঐচ্ছিক: আপনি az কমান্ড চালানোর পরিকল্পনা করলে Azure CLI কনটেক্সট যাচাই করুন
az account show

# প্রয়োজন হলে পুনরায় প্রমাণীকরণ করুন
azd auth login

# ঐচ্ছিক: az কমান্ডগুলোর জন্য Azure CLI প্রমাণীকরণ রিফ্রেশ করুন
az login
```
