# অধ্যায় ৩: কনফিগারেশন ও প্রমাণীকরণ

**📚 কোর্স**: [AZD ফর বিগিনার্স](../../README.md) | **⏱️ সময়কাল**: ৪৫-৬০ মিনিট | **⭐ জটিলতা**: মধ্যম

---

## ওভারভিউ

এই অধ্যায়ে Azure Developer CLI ডিপ্লয়মেন্টের জন্য পরিবেশ কনফিগারেশন, প্রমাণীকরণ প্যাটার্ন এবং সিকিউরিটি সেরা অনুশীলন আলোচনা করা হয়েছে।

> `azd 1.23.12` এর বিরুদ্ধে বৈধকরণ মার্চ ২০২৬-এ।

## শেখার উদ্দেশ্য

এই অধ্যায় সম্পূর্ণ করার মাধ্যমে আপনি:
- AZD কনফিগারেশন স্তরজ্ঞান আয়ত্ত করবেন
- একাধিক পরিবেশ (ডেভ, স্টেজিং, প্রোড) পরিচালনা করবেন
- পরিচালিত পরিচয়ের মাধ্যমে সুরক্ষিত প্রমাণীকরণ বাস্তবায়ন করবেন
- পরিবেশ-নির্দিষ্ট সেটিংস কনফিগার করবেন

## 🚀 দ্রুত শুরু করুন

```bash
# একাধিক পরিবেশ তৈরি করুন
azd env new dev
azd env new staging
azd env new prod

# পরিবেশ পরিবর্তন করুন
azd env select prod

# পরিবেশ পরিবর্তনশীল সেট করুন
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# কনফিগারেশন দেখুন
azd env get-values
```

## 🔧 কনফিগারেশন স্তরনির্দেশিকা

AZD নিম্নোক্ত ক্রমে সেটিংস প্রয়োগ করে (পরে আসা আপাতত আগে আসা ওভাররাইড করে):

১. **ডিফল্ট মান** (টেমপ্লেটের মধ্যে নির্মিত)
২. **azure.yaml** (প্রকল্প কনফিগারেশন)
৩. **পরিবেশ ভেরিয়েবলসমূহ** (`azd env set`)
৪. **কমান্ড-লাইন ফ্ল্যাগস** (`--location eastus`)

## 🔐 সিকিউরিটি সেরা অনুশীলন

```bash
# ম্যানেজড আইডেন্টিটি ব্যবহার করুন (প্রস্তাবিত)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD প্রমাণীকরণ স্থিতি পরীক্ষা করুন
azd auth status

# ঐচ্ছিক: az কমান্ড চালানোর পরিকল্পনা থাকলে Azure CLI প্রসঙ্গ যাচাই করুন
az account show

# প্রয়োজনে পুনরায় প্রমাণীকরণ করুন
azd auth login

# ঐচ্ছিক: az কমান্ডগুলোর জন্য Azure CLI প্রমাণীকরণ রিফ্রেশ করুন
az login
```
