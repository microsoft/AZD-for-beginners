# অধ্যায় ৩: কনফিগারেশন এবং প্রমাণীকরণ

**📚 কোর্স**: [AZD ফর বিগিনারস](../../README.md) | **⏱️ সময়কাল**: ৪৫-৬০ মিনিট | **⭐ জটিলতা**: মাঝারি

---

## ওভারভিউ

এই অধ্যায়ে Azure Developer CLI ডিপ্লয়মেন্টের জন্য পরিবেশ কনফিগারেশন, প্রমাণীকরণ প্যাটার্ন, এবং সুরক্ষা সেরা অনুশীলনগুলি আলোচনা করা হবে।

> `azd 1.27.1` এর বিরুদ্ধে জুলাই ২০২৬ সালে যাচাই করা হয়েছে।

## শেখার লক্ষ্যসমূহ

এই অধ্যায়টি সম্পন্ন করে আপনি:
- AZD কনফিগারেশন শ্রেণিবিন্যাসে দক্ষ হবেন
- একাধিক পরিবেশ (ডেভ, স্টেজিং, প্রোড) পরিচালনা করতে পারবেন
- ব্যবস্থাপিত পরিচয় দিয়ে নিরাপদ প্রমাণীকরণ বাস্তবায়ন করতে পারবেন
- পরিবেশ-নির্দিষ্ট সেটিংস কনফিগার করতে পারবেন

## 🚀 দ্রুত শুরু

```bash
# একাধিক পরিবেশ তৈরি করুন
azd env new dev
azd env new staging
azd env new prod

# পরিবেশ পরিবর্তন করুন
azd env select prod

# পরিবেশ পরিবর্তনশীল সেট করুন
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# কনফিগারেশন দেখুন
azd env get-values
```

## 🔧 কনফিগারেশন শ্রেণিবিন্যাস

AZD নিম্নলিখিত ক্রমে সেটিংস প্রয়োগ করে (পরে যেগুলো থাকে আগে যেগুলো ছিল তা ওভাররাইড করে):

১. **ডিফল্ট মানগুলি** (টেমপ্লেটগুলোর মধ্যে নেভে উপস্থিত)
২. **azure.yaml** (প্রজেক্ট কনফিগারেশন)
৩. **পরিবেশ ভেরিয়েবল** (`azd env set`)
৪. **কমান্ড-লাইন ফ্ল্যাগস** (`--location eastus`)

## 🔐 সুরক্ষার সেরা অনুশীলনসমূহ

```bash
# ব্যবস্থাপিত পরিচয় ব্যবহার করুন (প্রস্তাবিত)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD প্রমাণীকরণ অবস্থা পরীক্ষা করুন
azd auth status

# ঐচ্ছিক: যদি আপনি az কমান্ড চালানোর পরিকল্পনা করেন তবে Azure CLI প্রসঙ্গ যাচাই করুন
az account show

# প্রয়োজনে পুনরায় প্রমাণীকরণ করুন
azd auth login

# ঐচ্ছিক: az কমান্ডের জন্য Azure CLI প্রমাণীকরণ হালনাগাদ করুন
az login
```
