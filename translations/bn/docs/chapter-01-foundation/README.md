# অধ্যায় ১: ভিত্তি এবং দ্রুত শুরু

**📚 কোর্স**: [শিক্ষানবিশদের জন্য AZD](../../README.md) | **⏱️ সময়কাল**: ৩০-৪৫ মিনিট | **⭐ জটিলতা**: শিক্ষানবিশ

---

## ওভারভিউ

এই অধ্যায়টি Azure Developer CLI (azd) এর মূল বিষয়গুলি পরিচয় করিয়ে দেয়। আপনি মূল ধারণাগুলি শিখবেন, টুলগুলি ইনস্টল করবেন এবং Azure এ আপনার প্রথম অ্যাপ্লিকেশন ডিপ্লয় করবেন।

> मार्च ২০২৬ এ `azd 1.23.12` এর বিরুদ্ধে যাচাই করা হয়েছে।

## শেখার উদ্দেশ্য

এই অধ্যায় সম্পন্ন করার মাধ্যমে আপনি:
- বুঝতে পারবেন Azure Developer CLI কী এবং এটি Azure CLI থেকে কীভাবে আলাদা
- আপনার প্ল্যাটফর্মে AZD ইনস্টল ও কনফিগার করতে পারবেন
- `azd up` দিয়ে Azure এ আপনার প্রথম অ্যাপ্লিকেশন ডিপ্লয় করতে পারবেন
- `azd down` দিয়ে রিসোর্স পরিষ্কার করতে সক্ষম হবেন

## ✅ এখান থেকে শুরু করুন: আপনার সেটআপ যাচাই করুন

শুরু করার আগে নিশ্চিত করুন আপনার লোকাল মেশিন অধ্যায় ১ এর টেমপ্লেটের জন্য প্রস্তুত:

**উইন্ডোজ:**
```powershell
.\validate-setup.ps1
```

**ম্যাকওএস / লিনাক্স:**
```bash
bash ./validate-setup.sh
```

যদি স্ক্রিপ্ট কোনও টুল মিসিং রিপোর্ট করে, তাহলে আগে সেগুলো মেরামত করুন এবং তারপর অধ্যায় চালিয়ে যান।

## 🚀 দ্রুত শুরু

```bash
# ইনস্টলেশন পরীক্ষা করুন
azd version

# AZD এর জন্য প্রমাণীকরণ করুন
# ঐচ্ছিক: সরাসরি Azure CLI কমান্ড চালানোর পরিকল্পনা করলে az login করুন
azd auth login

# আপনার প্রথম অ্যাপ ডেপ্লয় করুন
azd init --template todo-nodejs-mongo
azd up

# সম্পন্ন হলে পরিষ্কার করুন
azd down --force --purge
```

---

## ✅ সাফল্যের মানদণ্ড

এই অধ্যায় সম্পন্ন করার পরে, আপনি সক্ষম হবেন:

```bash
azd version              # ইনস্টল করা সংস্করণ দেখায়
azd init --template todo-nodejs-mongo  # প্রকল্প আরম্ভ করে
azd up                   # আজুরে ডিপ্লয় করে
azd show                 # চলমান অ্যাপ্লিকেশনের ইউআরএল প্রদর্শন করে
azd down --force --purge # সম্পদগুলি পরিষ্কার করে
```
