# অধ্যায় 1: ভিত্তি ও দ্রুত শুরু

**📚 কোর্স**: [নবীনদের জন্য AZD](../../README.md) | **⏱️ Duration**: 30-45 minutes | **⭐ Complexity**: নবীন

---

## সারসংক্ষেপ

এই অধ্যায়টি Azure Developer CLI (azd) এর মৌলিক ধারণাগুলো উপস্থাপন করে। আপনি মূল ধারণাগুলি শিখবেন, টুলগুলি ইনস্টল করবেন, এবং আপনার প্রথম অ্যাপ্লিকেশনটি Azure-এ ডিপ্লয় করবেন।

> `azd 1.25.6` অনুযায়ী জুন 2026-এ যাচাইকৃত।

## শেখার উদ্দেশ্য

এই অধ্যায়টি সম্পন্ন করলে, আপনি:
- বুঝতে পারবেন Azure Developer CLI কী এবং এটি Azure CLI থেকে কীভাবে ভিন্ন
- আপনার প্ল্যাটফর্মে AZD ইনস্টল ও কনফিগার করবেন
- `azd up` দিয়ে আপনার প্রথম অ্যাপ্লিকেশন Azure-এ ডিপ্লয় করবেন
- `azd down` দিয়ে রিসোর্সগুলি পরিষ্কার করবেন

## ✅ এখান থেকে শুরু করুন: আপনার সেটআপ যাচাই করুন

শুরু করার আগে নিশ্চিত করুন যে আপনার লোকাল মেশিন Chapter 1 টেমপ্লেটের জন্য প্রস্তুত:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

If the script reports missing tools, fix those first and then continue with the chapter.

## 🚀 দ্রুত শুরু

```bash
# ইনস্টলেশন পরীক্ষা করুন
azd version

# AZD-এর জন্য প্রমাণীকরণ করুন
# ঐচ্ছিক: সরাসরি Azure CLI কমান্ড চালাতে চাইলে az login ব্যবহার করুন
azd auth login

# আপনার প্রথম অ্যাপ ডিপ্লয় করুন
azd init --template todo-nodejs-mongo
azd up

# কাজ শেষ হলে পরিষ্কার করুন
azd down --force --purge
```

## ✅ সফলতার মানদণ্ড

এই অধ্যায় শেষ করার পরে, আপনি সক্ষম হবেন:

```bash
azd version              # ইনস্টল করা সংস্করণ দেখায়
azd init --template todo-nodejs-mongo  # প্রকল্প আরম্ভ করে
azd up                   # Azure-এ মোতায়েন করে
azd show                 # চলমান অ্যাপের URL দেখায়
azd down --force --purge # রিসোর্সগুলো পরিষ্কার করে
```
