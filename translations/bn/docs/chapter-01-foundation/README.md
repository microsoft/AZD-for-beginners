# অধ্যায় ১: ভিত্তি ও দ্রুত শুরু

**📚 কোর্স**: [AZD ফর বিগিনার্স](../../README.md) | **⏱️ সময়কাল**: ৩০-৪৫ মিনিট | **⭐ জটিলতা**: শুরুয়ার জন্য

## ওভারভিউ

এই অধ্যায়ে Azure Developer CLI (azd) এর মৌলিক বিষয়াবলী পরিচিত করা হয়েছে। আপনি মূল ধারণাগুলো শিখবেন, সরঞ্জাম ইনস্টল করবেন, এবং Azure-তে আপনার প্রথম অ্যাপ্লিকেশন ডিপ্লয় করবেন।

> `azd 1.27.1` দিয়ে জুলাই ২০২৬ এ যাচাই করা হয়েছে।

## শেখার উদ্দেশ্যসমূহ

এই অধ্যায়টি সম্পন্ন করার পর আপনি পারবেন:
- Azure Developer CLI কী এবং Azure CLI থেকে এটি কীভাবে আলাদা তা বুঝতে
- আপনার প্ল্যাটফর্মে AZD ইনস্টল ও কনফিগার করতে
- `azd up` কমান্ড ব্যবহার করে Azure-তে আপনার প্রথম অ্যাপ্লিকেশন ডিপ্লয় করতে
- `azd down` কমান্ড ব্যবহার করে রিসোর্সগুলো পরিষ্কার করতে

## ✅ এখানে শুরু করুন: আপনার সেটআপ যাচাই করুন

শুরু করার আগে নিশ্চিত করুন যে আপনার লোকাল মেশিন অধ্যায় ১ টেমপ্লেটের জন্য প্রস্তুত:

**উইন্ডোজ:**
```powershell
.\validate-setup.ps1
```

**ম্যাকওএস / লিনাক্স:**
```bash
bash ./validate-setup.sh
```

যদি স্ক্রিপ্টে কোনো টুল অনুপস্থিত দেখানো হয়, সেগুলো প্রথমে ঠিক করুন এবং তারপর অধ্যায়টি চালিয়ে যান।

## 🚀 দ্রুত শুরু

```bash
# ইনস্টলেশন পরীক্ষা করুন
azd version

# AZD জন্য প্রমাণীকরণ করুন
# ঐচ্ছিক: আপনি যদি সরাসরি Azure CLI কমান্ড চালানোর পরিকল্পনা করেন তবে az login করুন
azd auth login

# আপনার প্রথম অ্যাপ স্থাপন করুন
azd init --template todo-nodejs-mongo
azd up

# কাজ শেষ হলে পরিষ্কার করুন
azd down --force --purge
```

## ✅ সফলতার মানদণ্ড

এই অধ্যায় সম্পন্ন করার পর, আপনি সক্ষম হবেন:

```bash
azd version              # ইনস্টল করা সংস্করণ দেখায়
azd init --template todo-nodejs-mongo  # প্রকল্প শুরু করে
azd up                   # Azure-এ মোতায়েন করে
azd show                 # চলমান অ্যাপ্লিকেশন URL প্রদর্শন করে
azd down --force --purge # সম্পদ পরিষ্কার করে
```
