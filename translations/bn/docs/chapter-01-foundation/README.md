**📚 কোর্স**: [শুরুদের জন্য AZD](../../README.md) | **⏱️ সময়**: 30-45 মিনিট | **⭐ জটিলতা**: প্রারম্ভিক

## সারসংক্ষেপ

এই অধ্যায়টি Azure Developer CLI (azd) এর মৌলিক বিষয়গুলি পরিচয় করিয়ে দেয়। আপনি মূল ধারণাগুলো শিখবেন, টুলগুলো ইনস্টল করবেন, এবং আপনার প্রথম অ্যাপ্লিকেশনটি Azure-এ ডিপ্লয় করবেন।

## শেখার লক্ষ্য

এই অধ্যায়টি সম্পন্ন করার পর, আপনি:
- Azure Developer CLI কী এবং এটি Azure CLI থেকে কীভাবে আলাদা তা বোঝা
- আপনার প্ল্যাটফর্মে AZD ইনস্টল ও কনফিগার করা
- `azd up` ব্যবহার করে আপনার প্রথম অ্যাপ্লিকেশনটি Azure-এ ডিপ্লয় করা
- `azd down` ব্যবহার করে রিসোর্সগুলো পরিষ্কার করা

## 🚀 দ্রুত শুরু

```bash
# ইনস্টলেশন পরীক্ষা করুন
azd version

# Azure-এ লগইন করুন
azd auth login

# আপনার প্রথম অ্যাপ ডিপ্লয় করুন
azd init --template todo-nodejs-mongo
azd up

# কাজ শেষ হলে পরিষ্কার করুন
azd down --force --purge
```

---

## ✅ সফলতার মানদণ্ড

এই অধ্যায়টি সম্পন্ন করার পর, আপনি সক্ষম হবেন:

```bash
azd version              # ইনস্টল করা সংস্করণ দেখায়
azd init --template todo-nodejs-mongo  # প্রকল্প শুরু করে
azd up                   # Azure-এ স্থাপন করে
azd show                 # চলমান অ্যাপের URL দেখায়
azd down --force --purge # রিসোর্সগুলো পরিষ্কার করে
```
