# Chapter 4: ইনফ্রাস্ট্রাকচার অ্যাজ কোড ও ডিপ্লয়মেন্ট

**📚 Course**: [AZD নবাগতদের জন্য](../../README.md) | **⏱️ Duration**: 1-1.5 ঘণ্টা | **⭐ Complexity**: মধ্যম

---

## ওভারভিউ

এই অধ্যায়টি Bicep টেমপ্লেটের মাধ্যমে ইনফ্রাস্ট্রাকচার অ্যাজ কোড (IaC) প্যাটার্ন, রিসোর্স প্রোভিশনিং, এবং Azure Developer CLI ব্যবহার করে ডিপ্লয়মেন্ট কৌশলগুলো ব্যাখ্যা করে।

## শেখার লক্ষ্য

By completing this chapter, you will:
- Bicep টেমপ্লেটের কাঠামো এবং সিনট্যাক্স বোঝা
- `azd provision` দিয়ে Azure রিসোর্স প্রোভিশন করা
- `azd deploy` ব্যবহার করে অ্যাপ্লিকেশন ডিপ্লয় করা
- ব্লু-গ্রীন এবং রোলিং ডিপ্লয়মেন্ট কৌশলগুলি বাস্তবায়ন করা

## 🚀 দ্রুত শুরু

```bash
# টেমপ্লেট থেকে আরম্ভ করুন
azd init --template azure-functions-python-v2-http

# কি তৈরি হবে তা পূর্বদর্শন করুন
azd provision --preview

# শুধুমাত্র অবকাঠামো স্থাপন করুন
azd provision

# শুধুমাত্র কোড স্থাপন করুন
azd deploy

# অথবা উভয়ই একসাথে
azd up
```

## 🔧 প্রয়োজনীয় কমান্ডসমূহ

| কমান্ড | বর্ণনা |
|---------|-------------|
| `azd init` | প্রজেক্ট শুরু করুন |
| `azd provision` | Azure রিসোর্স তৈরি করুন |
| `azd deploy` | অ্যাপ্লিকেশন কোড ডিপ্লয় করুন |
| `azd up` | প্রোভিশন + ডিপ্লয় |
| `azd down` | সব রিসোর্স মুছে ফেলুন |
