# অধ্যায় ৪: ইনফ্রাস্ট্রাকচার অ্যাজ কোড & ডিপ্লয়মেন্ট

**📚 কোর্স**: [AZD ফর বিগিনার্স](../../README.md) | **⏱️ সময়কাল**: ১-১.৫ ঘন্টা | **⭐ জটিলতা**: মধ্যম

---

## ওভারভিউ

এই অধ্যায়ে Bicep টেমপ্লেট, রিসোর্স প্রোভিশনিং এবং Azure Developer CLI ব্যবহার করে ডিপ্লয়মেন্ট স্ট্র্যাটেজি নিয়ে আলোচনা করা হয়েছে।

> `azd 1.27.1` এর সঙ্গে জুলাই ২০২৬-এ যাচাই করা হয়েছে।

## শেখার লক্ষ্যসমূহ

এই অধ্যায় সম্পন্ন করার মাধ্যমে আপনি:
- Bicep টেমপ্লেটের গঠন এবং সিনট্যাক্স বুঝতে পারবেন
- `azd provision` কমান্ড দিয়ে Azure রিসোর্স প্রোভিশন করতে পারবেন
- `azd deploy` ব্যবহার করে অ্যাপ্লিকেশন ডিপ্লয় করতে পারবেন
- ব্লু-গ্রিন এবং রোলিং ডিপ্লয়মেন্ট স্ট্র্যাটেজি কার্যকর করতে পারবেন

## 🚀 দ্রুত শুরু

```bash
# টেমপ্লেট থেকে আরম্ভ করুন
azd init --template azure-functions-python-v2-http

# কী তৈরি হবে তা প্রিভিউ করুন
azd provision --preview

# শুধুমাত্র অবকাঠামো প্রদান করুন
azd provision

# শুধুমাত্র কোড ডিপ্লয় করুন
azd deploy

# অথবা উভয় একসাথে
azd up
```

## 🔧 প্রয়োজনীয় কমান্ডসমূহ

| কমান্ড | বিবরণ |
|---------|-------------|
| `azd init` | প্রকল্প শুরু করা |
| `azd provision` | Azure রিসোর্স তৈরি করা |
| `azd deploy` | অ্যাপ্লিকেশন কোড ডিপ্লয় করা |
| `azd up` | প্রোভিশন + ডিপ্লয় |
| `azd down` | সমস্ত রিসোর্স মুছে ফেলা |
