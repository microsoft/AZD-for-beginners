# অধ্যায় ৪: কোড হিসেবে অবকাঠামো এবং ডিপ্লয়মেন্ট

**📚 কোর্স**: [AZD নবাগতদের জন্য](../../README.md) | **⏱️ সময়কাল**: ১-১.৫ ঘন্টা | **⭐ জটিলতা**: মধ্যম

---

## ওভারভিউ

এই অধ্যায়ে Bicep টেমপ্লেট ব্যবহার করে অবকাঠামো কোড (IaC) প্যাটার্ন, রিসোর্স প্রোভিশনিং, এবং Azure Developer CLI ব্যবহার করে ডিপ্লয়মেন্ট কৌশলগুলি আলোচনা করা হয়েছে।

> `azd 1.23.12` এ মার্চ ২০২৬-এ যাচাই করা হয়েছে।

## শিখন লক্ষ্যসমূহ

এই অধ্যায় সফলভাবে সম্পন্ন করার মাধ্যমে, আপনি:
- Bicep টেমপ্লেটের কাঠামো এবং সিনট্যাক্স বুঝতে পারবেন
- `azd provision` দিয়ে Azure রিসোর্স প্রোভিশন করতে পারবেন
- `azd deploy` দিয়ে অ্যাপ্লিকেশন ডিপ্লয় করতে পারবেন
- ব্লু-গ্রিন এবং রোলিং ডিপ্লয়মেন্ট কৌশল প্রয়োগ করতে পারবেন

## 🚀 দ্রুত শুরু

```bash
# টেমপ্লেট থেকে শুরু করুন
azd init --template azure-functions-python-v2-http

# কি তৈরি হবে তা প্রাকদর্শণ করুন
azd provision --preview

# শুধুমাত্র অবকাঠামো প্রদান করুন
azd provision

# শুধুমাত্র কোড স্থাপন করুন
azd deploy

# অথবা উভয় একসাথে
azd up
```

## 🔧 প্রয়োজনীয় কমান্ডসমূহ

| কমান্ড | বর্ণনা |
|---------|-------------|
| `azd init` | প্রকল্প আরম্ভ করুন |
| `azd provision` | Azure রিসোর্স তৈরি করুন |
| `azd deploy` | অ্যাপ্লিকেশন কোড ডিপ্লয় করুন |
| `azd up` | প্রোভিশন + ডিপ্লয় |
| `azd down` | সমস্ত রিসোর্স মুছে ফেলুন |
