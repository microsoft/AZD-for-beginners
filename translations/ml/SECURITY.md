## സുരക്ഷ

മൈക്രോസോഫ്റ്റ് ഞങ്ങളുടെ സോഫ്റ്റ്വെയർ ഉൽപ്പന്നങ്ങളും സേവനങ്ങളും, അതിൽ ഉൾപ്പെടെ GitHub ഓർഗനൈസേഷനുകളിലെ എല്ലാ സോഴ്‌സ് കോഡ് റീപോസിറ്ററികളും, സുരക്ഷിതമാക്കുന്നതിൽ ഗൗരവമായി സമീപിക്കുന്നു.

**ദയവായി പൊതുവായ GitHub പ്രശ്നങ്ങളിലൂടെ സുരക്ഷാ ദൗർബല്യങ്ങൾ റിപ്പോർട്ട് ചെയ്യരുത്.**

സുരക്ഷാ റിപ്പോർട്ടിംഗ് വിവരങ്ങൾ, സ്ഥലങ്ങൾ, ബന്ധപ്പെടാനുള്ള വിവരങ്ങൾ, നയങ്ങൾ എന്നിവയ്ക്കായി, മൈക്രോസോഫ്റ്റ് റീപോസിറ്ററികൾക്കായുള്ള ഏറ്റവും പുതിയ മാർഗ്ഗനിർദ്ദേശങ്ങൾ [https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md) എന്ന ലിങ്കിൽ പരിശോധിക്കുക.
