# അധ്യായം 3: കോൺഫിഗറേഷൻ & ഓതന്റിക്കേഷൻ

**📚 കോഴ്സ്**: [AZD For Beginners](../../README.md) | **⏱️ ദൈർഘ്യം**: 45-60 മിനിറ്റ് | **⭐ സങ്കീർണ്ണത**: മധ്യമന്

## അവലോകനം

ഈ അധ്യായം Azure Developer CLI വിന്യാസങ്ങൾക്കായി പരിസ്ഥിതി കോൺഫിഗറേഷൻ, ഓതന്റിക്കേഷൻ മാതൃകകൾ, സുരക്ഷാ മികച്ച പ്രവർത്തനങ്ങൾ എന്നിവ ഉൾക്കൊള്ളിക്കുന്നു.

> `azd 1.27.1`-ൽ ജൂലൈ 2026-ൽ സ്ഥിരീകരിച്ചിരിക്കുന്നു.

## പഠനലക്ഷ്യങ്ങൾ

ഈ അധ്യായം പൂര്‍ത്തിയാക്കിയാൽ, നിങ്ങൾക്ക്:
- AZD കോൺഫിഗറേഷൻ ഹയർകർക്കി ആഴത്തിൽ അറിയാം
- പല പരിസ്ഥിതികളെയും (dev, staging, prod) കൈകാര്യം ചെയ്യാം
- മാനേജുചെയ്ത ഐഡന്റിറ്റികളുള്ള സുരക്ഷിത ഓതന്റിക്കേഷൻ നടപ്പിലാക്കാം
- പരിസ്ഥിതി-നിര്ദ്ദിഷ്ട ക്രമീകരണങ്ങൾ നിർവ്വചിക്കാം

## 🚀 ത്വരിത ആരംഭം

```bash
# ഒട്ടനവധി പരിതസ്ഥിതികള്‍ സൃഷ്ടിക്കുക
azd env new dev
azd env new staging
azd env new prod

# പരിതസ്ഥിതികള്‍ മാറുക
azd env select prod

# പരിതസ്ഥിതി വേരിയബിളുകള്‍ സജ്ജമാക്കുക
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# കോൺഫിഗറേഷൻ കാണുക
azd env get-values
```

## 🔧 കോൺഫിഗറേഷൻ ഹയർകർക്കി

AZD പരാമർശങ്ങൾ ഈ ക്രമത്തിൽ പ്രയോഗിക്കുന്നു (പശ്ചാത്തലം നേരത്തെ ഉള്ളതു മറികടക്കും):

1. **ഡീഫോൾട്ട് മൂല്യങ്ങൾ** (ടെംപ്ലേറ്റുകളിൽ ഉൾക്കൊള്ളിക്കുന്നു)
2. **azure.yaml** (പ്രോജക്ട് കോൺഫിഗറേഷൻ)
3. **പരിസ്ഥിതി മാറ്റങ്ങൾ** (`azd env set`)
4. **കമാൻഡ്-ലൈൻ ഫ്ലാഗുകൾ** (`--location eastus`)

## 🔐 സുരക്ഷ മികവിലുള്ള ശീലങ്ങൾ

```bash
# മാനേജ് ചെയ്ത തിരിച്ചറിയൽ ഉപയോഗിക്കുക (ശുപാർശചെയ്യപ്പെടുന്നു)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD പ്രാമാണീകരണ അവസ്ഥ പരിശോധിക്കുക
azd auth status

# ഐച്ഛികം: az കമാൻഡുകൾ ഓടിക്കാൻ ആസൂത്രണം ചെയ്താൽ Azure CLI സന്ദർഭം സ്ഥിരീകരിക്കുക
az account show

# ആവശ്യമുള്ളപ്പോൾ ಮರುಪ್ರാമാണೀಕರಣമಾಚെയ്യുക
azd auth login

# ഐച്ഛികം: az കമാൻഡുകൾക്കായുള്ള Azure CLI പ്രാമാണീകരണം പുതുക്കുക
az login
```
