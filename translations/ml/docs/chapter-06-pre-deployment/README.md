# Chapter 6: പ്രീ-ഡിപ്ലോയ്മെന്റ് പ്ലാനിങ് & പരിശോധന

**📚 കോഴ്സ്**: [AZD For Beginners](../../README.md) | **⏱️ ദൈർഘ്യം**: 1 മണിക്കൂര്‍ | **⭐ സങ്കീർണ്ണത**: ഇടത്തരം

---

## അവലോകനം

നിങ്ങളുടെ ആപ്ലിക്കേഷൻ ഡിപ്ലോയ്മെന്റ് നടത്തുന്നതിന് മുന്നോടിയായി ആവശ്യമായ പദ്ധതിയിടലുകളും പരിശോധനകളും ഈ അധ്യായം ഉൾക്കൊള്ളുന്നു. യഥാർത്ഥ ശേഷി പ്ലാനിങ്, SKU തെരഞ്ഞെടുപ്പ്, പ്രിഫ്ലൈറ്റ് പരിശോധനകൾ എന്നിവയിലൂടെ ചെലവനഷ്ടങ്ങൾ ഒഴിവാക്കാൻ പഠിക്കുക.

> `azd 1.23.12` മാർച്ച് 2026-ന് പരിശോധന നടത്തിയതാണ്.

## പഠനലക്ഷ്യങ്ങൾ

ഈ അധ്യായം സമ്പൂർത്തിയാക്കിയാൽ, നിങ്ങൾക്ക് താഴെപ്പറയുന്നവയാകും സാധിക്കുക:
- ഡിപ്ലോയ്മെന്റിന് മുൻപ് പ്രിഫ്ലൈറ്റ് പരിശോധനകൾ നടത്തുക
- ശേഷി പദ്ധതി തയ്യാറാക്കുകയും സ്രോതസ്സ് ആവശ്യകത കണക്കുകൂട്ടുക
- ചെലവ് പ്ര оптимീസേഷനിക്ക് അനുയോജ്യമായ SKU-കൾ തിരഞ്ഞെടുക്കുക
- Application Insights മോണിറ്ററിംഗിനായി ക്രമീകരിക്കുക
- ടീം സഹകരണ പാറ്റേൺസുകൾ മനസ്സിലാക്കുക

## 🚀 ക്വിക്ക്ഡ്സ്റ്റാർട്ട്

```bash
# സബ്സ്ക്രിപ്ഷൻ ക്വോട്ടകൾ പരിശോധിക്കുക
az vm list-usage --location eastus --output table

# വിന്യാസം മുൻകൂട്ടി കാണുക (ഏതും സ്രഷ്ടിക്കപ്പെട്ടിട്ടില്ല)
azd provision --preview

# Bicep സംവൃതം സാധുത പരിശോധന
az bicep build --file infra/main.bicep

# പരിസര ക്രമീകരണം പരിശോധിക്കുക
azd env get-values
```

## ☑️ പ്രീ-ഡിപ്ലോയ്മെന്റ് ചെക്ക്‌ലിസ്റ്റ്

### `azd provision` മുമ്പ്

- [ ] മേഖലക്ക് ക്വോട്ട സ്ഥിരീകരിച്ചത്
- [ ] SKU-കൾ അനുയോജ്യമായി തിരഞ്ഞെടുക്കപ്പെട്ടത്
- [ ] ചെലവ് കണക്കുകൂട്ടൽ അവലോകനം ചെയ്തു
- [ ] നാമവ്യവസ്ഥ സുസ്ഥിരം
- [ ] സുരക്ഷ / RBAC ക്രമീകരിച്ചത്

### `azd deploy` മുമ്പ്

- [ ] പരിതസ്ഥിതി ചാരങ്ങൾ ക്രമീകരിച്ചതായി
- [ ] കീ വാള്റ്റിൽ രഹസ്യങ്ങൾ
- [ ] കണക്ഷൻ സ്ട്രിംഗുകൾ സ്ഥിരീകരിച്ചു
- [ ] ആരോഗ്യ പരിശോധനകൾ ക്രമീകരിച്ചു
