# അദ്ധ്യായം 4: ഇൻഫ്രാസ്ട്രക്ചർ അസ്കോഡ് & ഡിപ്ലോയ്മെന്റ്

**📚 കോഴ്‌സ്**: [AZD ഫോർ ബിഗിന്നേഴ്സ്](../../README.md) | **⏱️ ദൈര്‍ഘ്യം**: 1-1.5 മണിക്കൂർ | **⭐ സങ്കീർണത**: മദ്ധ്യമ

## അവലോകനം

ഈ അദ്ധ്യായത്തിൽ Bicep ടെംപ്ലേറ്റുകൾ ഉപയോഗിച്ചുള്ള ഇൻഫ്രാസ്ട്രക്ചർ അസ്കോഡ് (IaC) പാറ്റേണുകളും, റിസോഴ്‌സ് പ്രൊവിഷനിങ്ങും, Azure Developer CLI ഉപയോഗിച്ച് ഡിപ്ലോയ്മെന്റ് തന്ത്രങ്ങളും ഉൾക്കൊള്ളാം.

> 2026 മാർച്ചിൽ `azd 1.23.12` ഉപയോഗിച്ച് സ്ഥിരീകരിച്ചിരിക്കുന്നത്.

## പഠന ലക്ഷ്യങ്ങൾ

ഈ അദ്ധ്യായം പൂർത്തിയാക്കിയാൽ, നിങ്ങൾ:
- Bicep ടെംപ്ലേറ്റ് ഘടനയും സിന്റാക്സും മനസിലാക്കും
- `azd provision` ഉപയോഗിച്ച് Azure റിസോഴ്‌സുകൾ പ്രൊവിഷൻ ചെയ്യാൻ കഴിയും
- `azd deploy` ഉപയോഗിച്ച് ആപ്ലിക്കേഷനുകൾ ഡിപ്ലോയ് ചെയ്യാൻ കഴിയും
- ബ്ലൂ-ഗ്രീൻ, റോള്ളിംഗ് ഡിപ്ലോയ്മെന്റ് തന്ത്രങ്ങൾ നടപ്പിലാക്കും

## 🚀 ക്വിക്ക് സ്റ്റാർട്ട്

```bash
# ടെംപ്ലേറ്റിൽ നിന്ന് ആരംഭിക്കുക
azd init --template azure-functions-python-v2-http

# സൃഷ്ടിക്കാനുള്ളതിനെ മുൻകൂട്ടിയറിയുക
azd provision --preview

# ഇൻഫ്രാസ്ട്രക്ചർ മാത്രം ഒരുക്കുക
azd provision

# കോഡ് മാത്രം വിന്യസിക്കുക
azd deploy

# അല്ലെങ്കിൽ ഇരുവരും തമ്മിൽ ചേർത്തു
azd up
```

## 🔧 അനിവാര്യമായ കമാൻഡുകൾ

| കമാൻഡ് | വിവരണം |
|---------|-------------|
| `azd init` | പ്രോജക്ട് ആരംഭിക്കുക |
| `azd provision` | Azure റിസോഴ്‌സുകൾ സൃഷ്‌ടിക്കുക |
| `azd deploy` | ആപ്ലിക്കേഷൻ കോഡ് ഡിപ്ലോയ് ചെയ്യുക |
| `azd up` | പ്രൊവിഷൻ + ഡിപ്ലോയ്മെന്റ് |
| `azd down` | എല്ലാ റിസോഴ്‌സുകളും ഇല്ലാതാക്കുക |
