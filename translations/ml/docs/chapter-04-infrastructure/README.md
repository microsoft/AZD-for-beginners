# അദ്ധ്യായം 4: ഇൻഫ്രസ്ട്രക്ചർ അസ്സ് കോഡ് & ഡിപ്ലോയ്മെന്റ്

**📚 കോഴ്സ്**: [AZD For Beginners](../../README.md) | **⏱️ ദൈർഘ്യം**: 1-1.5 മണിക്കൂർ | **⭐ സങ്കീർണത**: ഇടമധ്യസ്ഥം

## അവലോകനം

ഈ അദ്ധ്യായം Bicep ടെംപ്ലേറ്റുകളോടെ ഇൻഫ്രസ്ട്രക്ചർ അസ്സ് കോഡ് (IaC) മാതൃകകൾ, റിസോഴ്‌സ് പ്രൊവിഷണിംഗ്, അൽപ്പം Azure Developer CLI ഉപയോഗിച്ചുള്ള ഡിപ്ലോയ്മെന്റ് തന്ത്രങ്ങൾ ഉൾക്കൊള്ളുന്നു.

> `azd 1.27.1` ല്‍ 2026 ജൂലൈ മാസത്തിൽ പരിശോധിച്ചു.

## പഠന ലക്ഷ്യങ്ങൾ

ഈ അദ്ധ്യായം പൂര്‍ത്തീകരിച്ചതിനുശേഷം, നിങ്ങൾക്ക്:
- Bicep ടെംപ്ലേറ്റ് ഘടനയും സൂത്രവാക്യവും മനസിലാക്കാൻ കഴിയും
- `azd provision` ഉപയോഗിച്ച് Azure റിസോഴ്‌സുകൾ പ്രൊവിഷൻ ചെയ്യാൻ കഴിയും
- `azd deploy` ഉപയോഗിച്ച് ആപ്ലിക്കേഷനുകൾ ഡിപ്ലോയ്മെന്റ് ചെയ്യാൻ കഴിയും
- ബ്ലൂ-ഗ്രീൻ, റോളിംഗ് ഡിപ്ലോയ്മെന്റ് തന്ത്രങ്ങൾ നടപ്പിലാക്കാൻ കഴിയും

## 🚀 ക്വിക്ക് സ്റ്റാർട്ട്

```bash
# ടെംപ്ലേറ്റിൽ നിന്ന് ആരംഭിക്കുക
azd init --template azure-functions-python-v2-http

# സൃഷ്ടിക്കപ്പെടുന്നതിന്റെ പ്രിവ്യൂ കാണുക
azd provision --preview

# ആധാര സൗകര്യം മാത്രമേ ഒരുക്കൂ
azd provision

# കോഡ് മാത്രം വിന്യസിക്കുക
azd deploy

# അല്ലെങ്കിൽ ഇരുപേരും ഒന്നിച്ച്
azd up
```

## 🔧 അത്യാവശ്യം കമാൻഡുകൾ

| കമാൻഡ് | വിവരണം |
|---------|-------------|
| `azd init` | പ്രോജക്ട് ആരംഭിക്കുക |
| `azd provision` | Azure റിസോഴ്‌സുകൾ സൃഷ്ടിക്കുക |
| `azd deploy` | ആപ്ലിക്കേഷൻ കോഡ് ഡിപ്ലോയ് ചെയ്യുക |
| `azd up` | പ്രൊവിഷൻ + ഡിപ്ലോയ്മെന്റ് |
| `azd down` | എല്ലാ റിസോഴ്‍സുകളും നീക്കം ചെയ്യുക |
