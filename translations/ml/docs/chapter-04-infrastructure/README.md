# Chapter 4: ഇൻഫ്രാസ്ട്രക്ചർ അസ്സ് കോഡ് & ഡിപ്ലോയ്മെന്റ്

## അവലോകനം

ഈ അധ്യായം Bicep ടെമ്പ്‌ലേറ്റുകൾ ഉപയോഗിച്ചുള്ള ഇൻഫ്രാസ്ട്രക്ചർ അസ്സ് കോഡ് (IaC) മാതൃകകൾ, റിസോഴ്‌സ് പ്രൊവിഷനിംഗ്, Azure Developer CLI ഉപയോഗിച്ചുള്ള ഡിപ്ലോയ്മെന്റ് നയങ്ങൾ എന്നിവ ഉൾക്കൊള്ളുന്നു.

> `azd 1.25.6` ഉപയോഗിച്ച് ജൂൺ 2026-ൽ സാധുതയുള്ളതായി സ്ഥിരീകരിച്ചവ.

## പഠന ലക്ഷ്യങ്ങൾ

ഈ അധ്യായം പൂർത്തിയാക്കിയാൽ, നിങ്ങൾക്ക് താഴെപ്പറയുന്നവ മനസ്സിലാകും:
- Bicep ടെമ്പ്‌ലേറ്റ് ഘടനയും സിന്റാക്സും
- `azd provision` ഉപയോഗിച്ച് Azure റിസോഴ്‌സുകൾ പ്രൊവിഷൻ ചെയ്യുക
- `azd deploy` ഉപയോഗിച്ച് അപ്ലിക്കേഷനുകൾ ഡിപ്ലോയ് ചെയ്യുക
- ബ്ലൂ-ഗ്രീൻ, റോളിങ് ഡിപ്ലോയ്മെന്റ് നയങ്ങൾ നടപ്പാക്കിയേക്കാം

## 🚀 വേഗത്തിലുള്ള ആരംഭം

```bash
# ടെംപ്ലേറ്റിൽ നിന്ന് ആരംഭിക്കുക
azd init --template azure-functions-python-v2-http

# സൃഷ്ടിക്കപ്പെടുന്നതിന്റെ പ്രിവ്യൂ
azd provision --preview

# മാത്രമേ അടിസ്ഥാന സൗകര്യം ഒരുക്കൂ
azd provision

# മാത്രമേ കോഡ് വിന്യസിക്കൂ
azd deploy

# അല്ലെങ്കിൽ ഇരുവരും ചേർന്ന്
azd up
```

## 🔧 പ്രധാന കമാൻഡുകൾ

| കമാൻഡ് | വിവരണം |
|---------|-------------|
| `azd init` | പ്രോജക്ട് ആരംഭിക്കുക |
| `azd provision` | Azure റിസോഴ്‌സുകൾ സൃഷ്ടിക്കുക |
| `azd deploy` | അപ്ലിക്കേഷൻ കോഡ് ഡിപ്ലോയ് ചെയ്യുക |
| `azd up` | പ്രൊവിഷൻ + ഡിപ്ലോയ് |
| `azd down` | എല്ലാ റിസോഴ്‌സുകളും നീക്കം ചെയ്യുക |
