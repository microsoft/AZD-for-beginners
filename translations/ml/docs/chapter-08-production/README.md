## അവലോകനം

ഈ അധ്യായം എന്റർപ്രൈസ്-സജ്ജമായ വിന്യാസ മാതൃകകൾ, സുരക്ഷാ ശക്തീകരണം, നിരീക്ഷണം, പ്രൊഡക്ഷൻ AI വര്‍ക്ക്ലോഡുകൾക്കുള്ള ചെലവ് ആപ്റ്റിമൈസേഷൻ എന്നിവയെക്കുറിച്ച് വിശദീകരിക്കുന്നു.

## പഠന ലക്ഷ്യങ്ങൾ

ഈ അധ്യായം പൂര്‍ത്തീകരിച്ചാൽ, നിങ്ങൾ:
- മൾട്ടി-റീജനിൽ പ്രതിരോധശേഷിയുള്ള ആപ്ലിക്കേഷനുകൾ വിന്യസിക്കുക
- എന്റർപ്രൈസ് സുരക്ഷാ മാതൃകകൾ നടപ്പിലാക്കുക
- സമഗ്ര നിരീക്ഷണം കോൺഫിഗർ ചെയ്യുക
- വ്യാപകമായി ചെലവുകൾ ആപ്റ്റിമൈസ് ചെയ്യുക
- AZD ഉപയോഗിച്ച് CI/CD പൈപ്പ്ലൈനുകൾ സജ്ജമാക്കുക

## 💰 ചെലവ് ആപ്ടിമൈസേഷൻ

| തന്ത്രം | ലാഭം |
|----------|---------|
| ശൂന്യത്തിലേക്ക് സ്കെയിൽ ചെയ്യുക (Container Apps) | 60-80% |
| ഡെവലപ്പ്മെന്റിന് consumption ടിയേഴ്സ് ഉപയോഗിക്കുക | 50-70% |
| ഷെഡ്യൂൾ ചെയ്ത സ്കെയ്ലിംഗ് | 30-50% |
| നിശ്ചിത ശേഷി (Reserved capacity) | 20-40% |

```bash
# ബജറ്റ് അലർട്ടുകൾ സജ്ജീകരിക്കുക

  --budget-name "AI-Budget" \
  --amount 500 \
  --category Cost \
  --time-grain Monthly
```
