# അധ്യായം 7: പ്രശ്നപരിഹാരം & ഡീബഗ്ഗിംഗ്

**📚 കോഴ്‌സ്**: [AZD തുടക്കക്കാർക്ക്](../../README.md) | **⏱️ ദൈർഘ്യം**: 1-1.5 മണിക്കൂർ | **⭐ സങ്കീർണത**: മധ്യമത്ഘാത

---

## അവലോകനം

Azure Developer CLI ഉപയോഗിക്കുമ്പോൾ സാധാരണയായി നേരിടുന്ന പ്രശ്നങ്ങൾ തിരിച്ചറിയാനും പരിഹരിക്കാനും ഈ അധ്യായം സഹായിക്കും. ഡിപ്പ്ലോയ്മെന്റ് പരാജയങ്ങളിൽനിന്ന് AI-നിർദ്ദേശ പ്രശ്നങ്ങൾ വരെ.

> `azd 1.27.1` എന്ന പതിപ്പിൽ 2026 ജൂലൈയിൽ പരിശോധന നടത്തി.

## പഠന ലക്ഷ്യങ്ങൾ

ഈ അധ്യായം പൂര്‍ത്തിയാക്കിയാൽ, നിങ്ങൾക്ക്:
- സാധാരണ AZD ഡിപ്പ്ലോയ്മെന്റ് പരാജയങ്ങൾ തിരിച്ചറിയാൻ
- പ്രാമాణീകരണവും അനുമതി പ്രശ്നങ്ങളും ഡീബഗ് ചെയ്യാൻ
- AI സേവന കണക്‌ടിവിറ്റി പ്രശ്നങ്ങൾ പരിഹരിക്കാൻ
- പ്രശ്നപരിഹാരത്തിന് Azure Portal, CLI ഉപയോഗിക്കാനെന്നും കഴിയും

## 🚨 വേഗത്തിലുള്ള പരിഹാരങ്ങൾ

### പ്രാമാണീകരണ പ്രശ്നങ്ങൾ
```bash
# AZD പ്രവർത്തനപ്രവാഹങ്ങൾക്ക് ആവശ്യമാണ്
azd auth login

# നിങ്ങൾ Azure CLI കമാൻഡുകളും നേരിട്ടു ഉപയോഗിക്കുന്നുവെങ്കിൽ ഐച്ഛികമാണ്
az login

azd auth status
```

### പ്രൊവിഷനിംഗ് പരാജയങ്ങൾ
```bash
azd show
azd monitor --logs
az deployment sub list --query "[?properties.provisioningState!='Succeeded']"
```

### റിസോകഴ്‌സ് കാനിഫ്ലിക്ട്സ്
```bash
azd down --force --purge
azd env new different-name
azd up
```

### ക്വോട്ടാ പരാര്ജ്ജിതം
```bash
az vm list-usage --location eastus --output table
azd env set AZURE_LOCATION westus2
azd up
```

## 📋 പിശക് കോഡ് റഫറൻസ്

| പിശക് | കാരണം | പരിഹാരം |
|-------|-------|----------|
| `AuthenticationError` | ലോഗിൻ ചെയ്തിട്ടില്ല | `azd auth login` |
| `ResourceNotFound` | റിസോഴ്സ് കാണപ്പെടുന്നില്ല | റിസോഴ്സ് നാമങ്ങൾ പരിശോധിക്കുക |
| `QuotaExceeded` | സബ്സ്‌ക്രിപ്ഷൻ പരിധികൾ | ക്വോട്ടാ വർധിപ്പിക്കാൻ അപേക്ഷിക്കുക |
| `InvalidTemplate` | ബൈസിപ് സിന്റാക്‌സ് പിഴവ് | `az bicep build` |
| `Conflict` | റിസോഴ്സ് നിലവിലുണ്ട് | പുതിയ പേർ ഉപയോഗിക്കുക അല്ലെങ്കിൽ മായ്ച്ചിടുക |
| `Forbidden` | അനുമതികൾ പോരായ്മ | RBAC റോളുകൾ പരിശോധിക്കുക |

## 🔄 റീസെറ്റ് ആൻഡ് റിക്കവറി

```bash
# സൗമ്യമായി റിസെറ്റ് ചെയ്യുക (സ്രോതസ്സ് നിലനിർത്തുക, കോഡ് പുനഃപ്രയോഗിക്കുക)
azd deploy --force

# കടുത്ത റിസെറ്റ് (എല്ലാം മായ്ച്ചെഴുക്ക്, പുതുക്കി ആരംഭിക്കുക)
azd down --force --purge
azd up
```
