# അദ്ധ്യായം 1: അടിസ്ഥാനവും ദ്രുതാരംഭം

**📚 കോഴ്‌സ്**: [AZD തുടങ്ങുന്നവർക്കുള്ള](../../README.md) | **⏱️ ദൈർഘ്യം**: 30-45 മിനിറ്റ് | **⭐ സങ്കീർണ്ണത**: പ്രാരംഭ

---

## അവലോകനം

ഈ അദ്ധ്യായം Azure Developer CLI (azd) ന്റെ അടിസ്ഥാനങ്ങളുമായി പരിചയപ്പെടുത്തുന്നു. നിങ്ങൾ പ്രധാന ആശയങ്ങൾ പഠിക്കുകയും, ടൂളുകൾ ഇൻസ്റ്റാൾ ചെയ്യുകയും, നിങ്ങളുടെ ആദ്യ ആപ്ലിക്കേഷൻ Azure-ലേക്ക് വിന്യസിക്കുകയും ചെയ്യുക.

## പഠന ലക്ഷ്യങ്ങൾ

ഈ അദ്ധ്യായം പൂർത്തിയാക്കിയതോടെ, നിങ്ങൾക്ക് താഴെപ്പറയുന്ന കാര്യങ്ങൾ ചെയ്യാൻ കഴിയണം:
- Azure Developer CLI എന്താണെന്ന്, അതെങ്ങനെ Azure CLI-ലൂടെ വ്യത്യസ്തമാണെന്ന് മനസ്സിലാക്കുക
- നിങ്ങളുടെ പ്ലാറ്റ്ഫോമിൽ AZD ഇൻസ്റ്റാൾ ചെയ്ത് കോൺഫിഗർ ചെയ്യുക
- `azd up` ഉപയോഗിച്ച് നിങ്ങളുടെ ആദ്യ ആപ്ലിക്കേഷൻ Azure-ലേക്ക് വിന്യസിക്കുക
- `azd down` ഉപയോഗിച്ച് റിസോഴ്‌സുകൾ നീക്കം ചെയ്യുക

## 🚀 ദ്രുതാരംഭം

```bash
# സ്ഥാപനം പരിശോധിക്കുക
azd version

# Azure-ലേക്ക് ലോഗിൻ ചെയ്യുക
azd auth login

# നിങ്ങളുടെ ആദ്യ ആപ്പ് വിന്യസിക്കുക
azd init --template todo-nodejs-mongo
azd up

# ചെയ്തതിനു ശേഷം ശുചീകരിക്കുക
azd down --force --purge
```

## ✅ വിജയ മാനദണ്ഡങ്ങൾ

ഈ അദ്ധ്യായം പൂർത്തിയാക്കിയശേഷം, നിങ്ങൾക്ക് താഴെപ്പറയുന്ന കാര്യങ്ങൾ ചെയ്യാൻ കഴിയണം:

```bash
azd version              # സ്ഥാപിച്ചിരിക്കുന്ന പതിപ്പ് കാണിക്കുന്നു
azd init --template todo-nodejs-mongo  # പ്രോജക്ട് പ്രാരംഭമാക്കുന്നു
azd up                   # Azure-ലേക്ക് വിന്യസിക്കുന്നു
azd show                 # ഓടുന്ന ആപ്പിന്റെ URL കാണിക്കുന്നു
azd down --force --purge # റിസോഴ്‌സുകൾ ശുചീകരിക്കുന്നു
```
