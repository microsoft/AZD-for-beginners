# Chapter 1: അടിസ്ഥാനവും ക്വിക് സ്റ്റാർട്ടും

**📚 കോഴ്‌സ്**: [AZD For Beginners](../../README.md) | **⏱️ ദൈർഖ്യം**: 30-45 മിനുട്ട് | **⭐ സമര്‍ത്ഥത**: ആരംഭക്കാർക്ക്

## അവലോകനം

ഈ അധ്യായം Azure Developer CLI (azd) അടിസ്ഥാനങ്ങൾ പരിചയപ്പെടുത്തുന്നു. നിങ്ങൾ മുഖ്യ ആശയങ്ങൾ മനസിലാക്കി, ടൂളുകൾ ഇൻസ്റ്റാൾ ചെയ്ത്, നിങ്ങളുടെ ആദ്യ ആപ്ലിക്കേഷൻ Azure-യിലേക്ക് ഡിപ്ലോയ് ചെയ്യാനാണ് പഠിക്കുന്നത്.

> മാർച്ച് 2026-ൽ `azd 1.23.12` ഉപയോഗിച്ച് സാധൂകരിച്ചത്.

## പഠന ലക്ഷ്യങ്ങൾ

ഈ അധ്യായം പൂർത്തിയാക്കിയാൽ, നിങ്ങൾക്ക് ചേരാം:
- Azure Developer CLI എന്താണെന്നും അത് Azure CLI-യിൽ നിന്ന് എങ്ങനെ വ്യത്യസ്തമാണെന്നും മനസ്സിലാക്കുക
- നിങ്ങളുടെ പ്ലാറ്റ്ഫോമിൽ AZD ഇൻസ്റ്റാൾ ചെയ്യുകയും കോൺഫിഗർ ചെയ്യുകയും ചെയ്യുക
- `azd up` ഉപയോഗിച്ച് നിങ്ങളുടെ ആദ്യ ആപ്ലിക്കേഷൻ Azure-യിലേക്ക് ഡിപ്ലോയ് ചെയ്യുക
- `azd down` ഉപയോഗിച്ച് റിസോഴ്‌സുകൾ ക്ലീൻ ചെയ്യുക

## ✅ ഇവിടെ തുടങ്ങുക: നിങ്ങളുടെ സെറ്റപ് സാധൂകരിക്കുക

നിങ്ങൾ ആരംഭിക്കുന്നതിന് മുമ്പ്, നിങ്ങളുടെ ലൊക്കൽ മെഷീൻ ചാപ്റ്റർ 1 ടെംപ്ലേറ്റിനായി തയ്യാറാണെന്ന് സ്ഥിരീകരിക്കുക:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

സ്ക്രിപ്റ്റ് മിസ്സിംഗ് ടൂളുകൾ റിപ്പോർട്ട് ചെയ്താൽ, ആദ്യം അവ പരിഹരിച്ച് പിന്നീട് അധ്യായം തുടരു.

## 🚀 ക്വിക്ക് സ്റ്റാർട്ട്

```bash
# ഇൻസ്റ്റാളേഷൻ പരിശോധിക്കുക
azd version

# AZDയ്ക്ക് പ്രാമാണീകരണം നടത്തുക
# ഐച്ഛികം: നേരിട്ട് Azure CLI കമാൻഡുകൾ നടത്താൻ പദ്ധതിയുണ്ടെങ്കിൽ az ൽ ലോഗിൻ ചെയ്യുക
azd auth login

# നിങ്ങളുടെ ആദ്യ ആപ്പ് ഡിപ്ലോയ് ചെയ്യുക
azd init --template todo-nodejs-mongo
azd up

# തീർച്ചയായാൽ ക്ലീൻ അപ്പ് നടത്തുക
azd down --force --purge
```

## ✅ വിജയം നിശ്ചയിക്കുന്ന മാനദണ്ഡങ്ങൾ

ഈ അധ്യായം പൂർത്തിയാക്കിയതിൽ ശേഷം, നിങ്ങൾക്കുള്ള കഴിവുകൾ:

```bash
azd version              # ഇൻസ്റ്റാൾ ചെയ്ത പതിപ്പ് കാണിക്കുന്നു
azd init --template todo-nodejs-mongo  # പ്രോജക്റ്റ് ആരംഭിക്കുന്നു
azd up                   # അസ്യൂറിലേക്ക് വിന്യസിക്കുന്നു
azd show                 # இயங்கുന്ന ആപ്പ് URL പ്രദർശിപ്പിക്കുന്നു
azd down --force --purge # വിഭവങ്ങൾ ശുചിയാക്കുന്നു
```
