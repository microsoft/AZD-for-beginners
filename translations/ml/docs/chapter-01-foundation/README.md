# Chapter 1: അടിസ്ഥാനവും ദ്രുതാരംഭവും

**📚 കോഴ്‌സ്**: [AZD For Beginners](../../README.md) | **⏱️ ദൈർഘ്യം**: 30-45 മിനിറ്റ് | **⭐ സങ്കീർത്തനത**: തുടക്കക്കാരൻ

---

## അവലോകനം

ഈ അധ്യായം Azure Developer CLI (azd) അടിസ്ഥാനങ്ങൾ പരിചയപ്പെടുത്തുന്നു. നിങ്ങൾ പ്രധാന ആശയങ്ങൾ പഠിക്കും, ഉപകരണങ്ങൾ ഇൻസ്റ്റാൾ ചെയ്തു, നിങ്ങളുടെ ആദ്യ അപ്ലിക്കേഷൻ Azure-ൽ വിന്യസിക്കും.

> ജൂൺ 2026-ൽ `azd 1.25.6` മേൽ സ്ഥിരീകരിച്ചിരിക്കുന്നു.

## പഠന ലക്ഷ്യങ്ങൾ

ഈ അധ്യായം പൂർത്തിയാക്കുന്നത് വഴി, നിങ്ങൾ:
- Azure Developer CLI എന്താണെന്ന്, Azure CLI-യുമായി എങ്ങനെ വ്യത്യാസമുണ്ടെന്ന് മനസ്സിലാക്കും
- നിങ്ങളുടെ പ്ലാറ്റ്ഫോമിൽ AZD ഇൻസ്റ്റാൾ ചെയ്ത് കോൺഫിഗർ ചെയ്യുക
- `azd up` ഉപയോഗിച്ച് നിങ്ങളുടെ ആദ്യ അപ്ലിക്കേഷൻ Azure-ൽ വിന്യസിക്കുക
- `azd down` ഉപയോഗിച്ച് വിഭവങ്ങൾ ശുദ്ധമാക്കുക

## ✅ ഇവിടെ നിന്ന് തുടങ്ങൂ: നിങ്ങളുടെ സെറ്റപ്പ് സ്ഥിരീകരിക്കുക

നിങ്ങൾ തുടങ്ങുന്നതിന് മുമ്പ്, നിങ്ങളുടെ പ്രാദേശിക യന്ത്രം Chapter 1 ടെംപ്ലേറ്റിനായി തയ്യാറാണെന്ന് സ്ഥിരീകരിക്കുക:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

സ്ക്രിപ്റ്റ് നഷ്ടപ്പെട്ട ഉപകരണങ്ങൾ റിപ്പോർട്ട് ചെയ്താൽ, ആദ്യം അവ ശരിയാക്കി അതിനു ശേഷം അധ്യായത്തിൽ മുന്നോട്ട് പോകുക.

## 🚀 ദ്രുതാരംഭം

```bash
# ഇൻസ്റ്റാളേഷൻ പരിശോധിക്കുക
azd version

# AZD-യ്ക്ക് അംഗീകാരം നൽകുക
# ഐച്ഛികം: നിങ്ങൾ നേരിട്ട് Azure CLI കമാൻഡുകൾ പ്രവർത്തിപ്പിക്കാൻ ഉദ്ദേശിക്കുന്നുവെങ്കിൽ az ലോഗിൻ ചെയ്യുക
azd auth login

# നിങ്ങളുടെ ആദ്യ എപ്പ്ലിക്കേഷൻ പ്രക്ഷേപണം ചെയ്യുക
azd init --template todo-nodejs-mongo
azd up

# പൂർത്തിയായാൽ ശുചീകരണം നടത്തുക
azd down --force --purge
```

## ✅ വിജയ മാനദണ്ഡങ്ങൾ

ഈ അധ്യായം പൂർത്തിയാക്കിയ ശേഷം, നിങ്ങൾ കഴിയും:

```bash
azd version              # ഇൻസ്റ്റാൾ ചെയ്ത പതിപ്പ് കാണിക്കുന്നു
azd init --template todo-nodejs-mongo  # പ്രോജക്ട് ആരംഭിക്കുന്നു
azd up                   # അസ്യൂറിൽ വിന്യസിക്കുന്നു
azd show                 # പ്രവർത്തനത്തിലുള്ള ആപ്പ് URL കാണിക്കുന്നു
azd down --force --purge # വിഭവങ്ങൾ ശുദ്ധമാക്കുന്നു
```
