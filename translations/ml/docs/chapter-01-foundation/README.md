# അദ്ധ്യായം 1: അടിസ്ഥാനവും ദ്രുതാരംഭവും

**📚 കോഴ്സ്**: [AZD For Beginners](../../README.md) | **⏱️ ദൈർഘ്യം**: 30-45 മിനിറ്റുകൾ | **⭐ സങ്കീർണ്ണത**: തുടക്കക്കാർക്ക്

---

## അവലോകനം

ഈ അദ്ധ്യായം Azure Developer CLI (azd) അടിസ്ഥാനങ്ങൾ പരിചയപ്പെടുത്തുന്നു. നിങ്ങൾക്ക് പ്രധാന ആശയങ്ങൾ പഠിക്കാനും, ഉപകരണങ്ങൾ ഇൻസ്റ്റാൾ ചെയ്യാനും, ആദ്യത്തെ അപ്ലിക്കേഷൻ Azure-ൽ ഡിപ്ലോയ് ചെയ്യാനും കഴിയുകയും ചെയ്യും.

> `azd 1.27.1` ൽ 2026 ജൂലൈയിൽ സ്ഥിരീകരിച്ചത്.

## പഠന ലക്ഷ്യങ്ങൾ

ഈ അദ്ധ്യായം പൂർത്തിയാക്കിയാൽ, നിങ്ങൾക്ക് കഴിയും:
- Azure Developer CLI എന്നത് എന്താണെന്ന്, Azure CLI-ൽ നിന്ന് അത് എങ്ങനെ വ്യത്യസ്തമാണെന്ന് മനസ്സിലാക്കുക
- നിങ്ങളുടെ പ്ലാറ്റ്‌ഫോമിൽ AZD ഇൻസ്റ്റാൾ ചെയ്യുകയും കോൺഫിഗർ ചെയ്യുകയും ചെയ്യുക
- `azd up` ഉപയോഗിച്ച് Azure-ടയിൽ നിങ്ങളുടെ ആദ്യ അപ്ലിക്കേഷൻ ഡിപ്ലോയ് ചെയ്യുക
- `azd down` ഉപയോഗിച്ച് റിസോഴ്‌സുകൾ ക്ലീൻ അപ്പ് ചെയ്യുക

## ✅ ഇവിടത്തുനിന്ന് തുടങ്ങുക: നിങ്ങളുടെ സജ്ജീകരണം പരിശോധിക്കുക

തുടങ്ങുന്നതു മുമ്പ്, ചാപ്റ്റർ 1 ടെംപ്ലേറ്റ് പ്രവർത്തിക്കാനായി നിങ്ങളുടെ ലൊക്കൽ മെഷീൻ തയ്യാറാണോ എന്ന് സ്ഥിരീകരിക്കാം:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

സ്ക്രിപ്റ്റ് ഉപകരണങ്ങൾ കാണമില്ലെന്ന് റിപ്പോർട്ട് ചെയിതാൽ, ആദ്യം അവ പരിഹരിച്ച് ശേഷം അദ്ധ്യായം തുടരണം.

## 🚀 ദ്രുതാരംഭം

```bash
# ഇൻസ്റ്റലേഷൻ പരിശോധിക്കുക
azd version

# AZD ന് അവകാശം പരിശോധിക്കുക
# സ്വന്തമായും Azure CLI കമാൻഡുകൾ പ്രവർത്തിപ്പിക്കാൻ ആഗ്രഹിച്ചാൽ: az login
azd auth login

# നിങ്ങളുടെ ആദ്യ ആപ്പ് പ്രയോഗിപ്പിക്കുക
azd init --template todo-nodejs-mongo
azd up

# പൂർത്തിയായാൽ നീക്കം ചെയ്യുക
azd down --force --purge
```

## ✅ വിജയ മാനദണ്ഡങ്ങൾ

ഈ അദ്ധ്യായം പൂർത്തീകരിച്ചശേഷം, നിങ്ങൾക്ക് കഴിയും:

```bash
azd version              # ഇൻസ്റ്റാൾ ചെയ്ത വെർഷൻ കാണിക്കുന്നു
azd init --template todo-nodejs-mongo  # പദ്ധതി ആരംഭിക്കുന്നു
azd up                   # ആസ്യൂറിൽ വിന്യസിക്കുന്നു
azd show                 # പ്രവർത്തിക്കുന്ന ആപ്പ് URL പ്രദർശിപ്പിക്കുന്നു
azd down --force --purge # വിഭവങ്ങൾ ശുചീകരിക്കുന്നു
```
