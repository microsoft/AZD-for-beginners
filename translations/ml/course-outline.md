# AZD For Beginners: Course Outline & Learning Framework

## Course Overview

Master Azure Developer CLI (azd) through structured chapters designed for progressive learning. **പ്രധാനമായി Microsoft Foundry സംയോജനം ഉപയോഗിച്ച് AI അപ്ലിക്കേഷൻ വിന്യാസത്തിനാണ് പ്രത്യേക ശ്രദ്ധ.**

### Why This Course is Essential for Modern Developers

Microsoft Foundry Discord സമുദായത്തിൽ നിന്നുള്ള洞അവലോകനങ്ങളുടെ അടിസ്ഥാനത്തിൽ, **45% വികസകർ AZD AI ജോലിഭാരങ്ങൾക്കായി ഉപയോഗിക്കാൻ ആഗ്രഹിക്കുന്നു** എന്നാൽ ഇവരുടെ മുന്നിൽ വരുന്ന വെല്ലുവിളികൾ:
- സങ്കീർണ മൾട്ടി-സർവീസ് AI ആർക്കിടെക്ചറുകൾ
- പ്രൊഡക്ഷൻ AI വിന്യാസത്തിന്റെ മികച്ച പ്രവർത്തനരീതികൾ  
- Azure AI സേവനങ്ങളുടെ സംയോജനം và കോൺഫിഗറേഷൻ
- AI ജോലിഭാരങ്ങൾക്കുള്ള ചെലവ്_OPTIMO_ (cost optimization)
- AI-വിശേഷ വിന്യാസ തകരാറുകൾ നിവർത്തൽ

### Core Learning Objectives

By completing this structured course, you will:
- **AZD അടിസ്ഥാനത്തിൽ മെഡറ്ല് ചെയ്യുക**: പ്രധാന ആശയങ്ങൾ, ഇൻസ്റ്റളേഷൻ, và കോൺഫിഗറേഷൻ
- **AI അപ്ലിക്കേഷനുകൾ വിന്യസിക്കുക**: Microsoft Foundry സേവനങ്ങളോടുകൂടിയ AZD ഉപയോഗിക്കുക
- **Infrastructure as Code നടപ്പാക്കുക**: Bicep ടെംപ്ലേറ്റുകൾ ഉപയോഗിച്ച് Azure റിസോഴ്‌സുകൾ മാനേജ് ചെയ്യുക
- **വിന്യാസങ്ങൾ പ്രശ്നപരിഹാരം ചെയ്യുക**: സാധാരണ പ്രശ്നങ്ങൾ പരിഹരിക്കുക và ഡെബഗ് ചെയ്യുക
- **പ്രൊഡക്ഷനിൽ ഓപ്റ്റിമൈസ് ചെയ്യുക**: സുരക്ഷ, സ്കേലിയിംഗ്, മോണിറ്ററിംഗ്, và ചെലവ് മാനേജ്‌മെന്റ്
- **മൾട്ടി-ഏജന്റ് പരിഹാരങ്ങൾ നിർമ്മിക്കുക**: സങ്കീർണ AI ആർക്കിടെക്ചറുകൾ വിന്യസിക്കുക

## 🎓 Workshop Learning Experience

### Flexible Learning Delivery Options
This course is designed to support both **self-paced individual learning** and **facilitated workshop sessions**, enabling learners to get hands-on experience with AZD while developing practical skills through interactive exercises.

#### 🚀 Self-Paced Learning Mode
**സ്വയം പഠിക്കാനുള്ളവർക്കും തുടർച്ചയായ അക്കാദമിക് വളർച്ചയ്ക്കും അനുയോജ്യം**

**സ്വഭാവങ്ങൾ:**
- **ബ്രൗസർ-ആധാരിത ഇന്റർഫേസ്**: ഏത് വെബ് ബ്രൗസറിലൂടെയും പ്രവേശിക്കാവുന്ന MkDocs-ചാലിത വർക്ഷോപ്
- **GitHub Codespaces സംയോജനം**: മുൻകൂട്ടി ക്രമീകരിച്ച ടൂളുകളോടെ ഒരു ക്ലിക്ക് ഡെവലപ്‌മെന്റ് അന്തരീക്ഷം
- **ഇന്ററാക്ടീവ് DevContainer അന്തരീക്ഷം**: യാതൊരു ലോക്കൽ സെറ്റപ്പ് ആവശ്യമായില്ല - ഉടൻ കോഡിംഗ് ആരംഭിക്കുക
- **പ്രോഗ്രസ്സ് ട്രാക്കിംഗ്**: ഉൾക്കൊള്ളിച്ച ചെക്ക്പോയിന്റുകളും വാലിഡേഷൻ അഭ്യാസങ്ങളും
- **സമുദായ പിന്തുണ**: ചോദ്യങ്ങളുടെയും സഹകരണത്തിനുമുള്ള Azure Discord ചാനലുകളിൽ പ്രവേശനം

**പഠന ഘടന:**
- **മികവാർന്ന സമയ ക്രമീകരണം**: ദിവസങ്ങളിലോ həവഴികളിലോ സ്വന്തം താളത്തിൽ അധ്യാപനങ്ങൾ പൂർത്തിയാക്കുക
- **Checkpoint സംവിധാനം**: സങ്കീർണ വിഷയങ്ങളിലേക്ക് കടക്കുന്നതിന് മുമ്പ് പഠനം സാധുവെന്ന് സ്ഥിരീകരിക്കുക
- **റിസോഴ്‌സ് ലൈബ്രറി**: സമഗ്ര ഡോക്യുമെന്റേഷൻ, ഉദാഹരണങ്ങൾ, và ട്രബ്ലഷൂട്ടിംഗ് മാർഗ്ഗനിർദേശങ്ങൾ
- **പോർട്ട്ഫോളിയോ വികസനം**: പ്രൊഫഷണൽ പോർട്ട്ഫോളിയോകയ്ക്ക് വിന്യസിക്കാവുന്ന പ്രോജക്ടുകൾ നിർമ്മിക്കുക

**Getting Started (Self-Paced):**
```bash
# ഓപ്ഷൻ 1: GitHub Codespaces (ശുപാർശ ചെയ്യപ്പെടുന്നു)
# റിപോസിറ്ററിയിലേക്ക് നാവിഗേറ്റ് ചെയ്ത് "Code" → "Create codespace on main" ക്ലിക്ക് ചെയ്യുക

# ഓപ്ഷൻ 2: ലോക്കൽ ഡവലപ്പ്മെന്റ്
git clone https://github.com/microsoft/azd-for-beginners.git
cd azd-for-beginners/workshop
# workshop/README.md-ൽ ഉള്ള സെറ്റപ്പ് നിർദ്ദേശങ്ങൾ പിന്തുടരുക
```

#### 🏛️ Facilitated Workshop Sessions
**കോർപ്പറേറ്റ് ട്രയിനിംഗ്, ബൂറ്റ്‌കമ്പുകൾ, വിദ്യാഭ്യാസ സ്ഥാപനങ്ങൾ എന്നിവയ്ക്ക് അനുയോജ്യം**

**Workshop Format Options:**

**📚 Academic Course Integration (8-12 weeks)**
- **സെർവകലാശാല പ്രോഗ്രാമുകൾ**: സെമസ്റ്റർ ദൈർഘ്യമുള്ള കോഴ്‌സ്, ആഴ്ചയിൽ 2-മണിക്കൂർ സെഷനുകൾ
- **ബൂട്ട്‌കാംപ് ഫോർമാറ്റ്**: ദിനംപ്രതി 6-8 മണിക്കൂർ സെഷനുകളോടുള്ള concentrated 3-5 ദിവസ പ്രോഗ്രാം
- **കോർപ്പറേറ്റ് ട്രെയിനിംഗ്**: പ്രതിമാസ ടീം സെഷനുകൾ പ്രായോഗിക പ്രോജക്ട് നടപ്പാക്കലോടുകൂടി
- **മൂല്യനിർണയ ഘടന**: ഗ്രേഡ് ചെയ്‌ത അസൈൻമെന്റുകൾ, സഹപാഠി അവലോകനങ്ങൾ, അന്തിമ പ്രോജക്ടുകൾ

**🚀 Intensive Workshop (1-3 days)**
- **Day 1**: Foundation + AI Development (Chapters 1-2) - 6 hours
- **Day 2**: Configuration + Infrastructure (Chapters 3-4) - 6 hours  
- **Day 3**: Advanced Patterns + Production (Chapters 5-8) - 8 hours
- **Follow-up**: Optional 2-week mentorship for project completion

**⚡ Executive Briefing (4-6 hours)**
- **Strategic Overview**: AZD മൂല്യ നിർദ്ദേശവും ബിസിനസ് ബാധ്യതയും (1 hour)
- **Hands-On Demo**: end-to-end AI അപ്ലിക്കേഷൻ വിന്യാസ ഡെമോ (2 hours)
- **Architecture Review**: എൻറെപ്രൈസ് മാതൃകകളും ഗവണൻസും (1 hour)
- **Implementation Planning**: സംഘടനാ ആഡപ്‌ഷൻ തന്ത്രം (1-2 hours)

#### 🛠️ Workshop Learning Methodology
**പരിശോധന → വിന്യാസം → ഇഷ്‌ടാനുസരണം എന്ന സമീപനം കൈകൊണ്ട് പഠന നൈപുണ്യ വികസനത്തിനുള്ളത്**

**Phase 1: Discovery (45 minutes)**
- **ടെംപ്ലേറ്റ് എക്ക്സ്പ്ലോറേഷൻ**: Microsoft Foundry ടെംപ്ലേറ്റുകളും സേവനങ്ങളും വിലയിരുത്തുക
- **ആർക്കിടെക്ചർ വിശകലനം**: മൾട്ടി-ഏജന്റ് മാതൃകകളും വിന്യാസ തന്ത്രങ്ങളും മനസിലാക്കുക
- **ആവശ്യകതാ மதനിർണ്ണയം**: സംഘടനാ ആവശ്യങ്ങളും നിയന്ത്രണങ്ങളും തിരിച്ചറിഞ്ഞുക
- **പരിസ്ഥിതി ക്രമീകരണം**: ഡെവലപ്പ്മെന്റ് പരിസ്ഥിതിയും Azure റിസോഴ്‌സുകളും ക്രമീകരിക്കുക

**Phase 2: Deployment (2 hours)**
- **ഗൈഡ് ചെയ്ത നടപ്പിലാക്കൽ**: AZD ഉപയോഗിച്ച് AI അപ്ലിക്കേഷനുകളുടെ ഘട്ടം കൂടിയ വിന്യാസം
- **സർവീസ് കോൺഫിഗറേഷൻ**: Azure AI സേവനങ്ങൾ, എൻഡ്പോയിന്റുകൾ, പ്രാമാണീകരണം ക്രമീകരിക്കുക
- **സുരക്ഷ നടപ്പിലാക്കൽ**: എൻറപ്രൈസ് സുരക്ഷാ മാതൃകകളും ആക്‌സസ് നിയന്ത്രണങ്ങളും പ്രയോഗിക്കുക
- **വാലിഡേഷൻ ടെസ്റ്റിംഗ്**: വിന്യാസങ്ങൾ സ്ഥിരീകരിക്കുകയും സാധാരണ പ്രശ്നങ്ങൾ പരിഹരിക്കുകയും ചെയ്യുക

**Phase 3: Customization (45 minutes)**
- **അപ്ലിക്കേഷൻ മാറ്റംവരുത്തൽ**: പ്രത്യേക ഉപയോഗകേസുകൾക്കും ആവശ്യങ്ങൾക്കും ടെംപ്ലേറ്റുകൾ അനുയോജ്യമായി മാറ്റുക
- **പ്രൊഡക്ഷൻ ഓപ്റ്റിമൈസേഷൻ**: മോണിറ്ററിംഗ്, ചെലവ് മാനേജ്‌മെന്റ്, സ്കേലിങ് തന്ത്രങ്ങൾ നടപ്പാക്കുക
- **അഡ്വാൻസ്ഡ് മാതൃകകൾ**: മൾട്ടി-ഏജന്റ് കോഓർഡിനേഷൻ và സങ്കീർണ ആർക്കിടെക്ചറികൾ പരിശോധിക്കുക
- **അടുത്ത നടപടികൾ പദ്ധതികൽ**: തുടർച്ചയായ നൈപുണ്യ വികസനത്തിന് പഠന മാർഗരേഖ നിർവചിക്കുക

- **Container App പാത**: അദ്ധ്യായങ്ങൾ 4 (കണ്ടെയ്നറൈസ്ഡ് ഡിപ്പ്ലോയ്മെന്റ്), 5 (മൈക്രോസർവിസ്സ് ഇന്റഗ്രേഷൻ), 8 (പ്രൊഡക്ഷൻ മികച്ച പ്രവൃത്തികൾ)

### സങ്കീർണത സൂചികകൾ

- **⭐ അടിസ്ഥാന**: ഏക ആശയങ്ങൾ, ഗൈഡുചെയ്ത ടൂട്ടോറിയലുകൾ, 30-60 മിനിറ്റ്
- **⭐⭐ ഇടത്തരം**: ഒന്നിലധികം ആശയങ്ങൾ, പ്രായോഗിക പരിശീലനം, 1-2 മണിക്കൂർ  
- **⭐⭐⭐ ഉയർന്നതരം**: സങ്കീർണ ആർക്കിടെക്ചറുകൾ, ഇഷ്ടാനുസൃത പരിഹാരങ്ങൾ, 1-3 മണിക്കൂർ
- **⭐⭐⭐⭐ വിദഗ്ധ**: പ്രൊഡക്ഷൻ സിസ്റ്റങ്ങൾ, എന്റർപ്രൈസ് മാതൃകകൾ, 2-4 മണിക്കൂർ

### സൗകര്യപ്രദമായ പഠന പാതകൾ

#### 🎯 AI ഡെവലപ്പർ ഫാസ്റ്റ് ട്രാക്ക് (4-6 മണിക്കൂർ)
1. **അദ്ധ്യായം 1**: അടിസ്ഥാനം & ക്വിക്ക് സ്റ്റാർട്ട് (45 മിനിറ്റ്)
2. **അദ്ധ്യായം 2**: AI-ഫസ്റ്റ് ഡെവലപ്മെന്റ് (2 മണിക്കൂർ)  
3. **അദ്ധ്യായം 5**: മൾട്ടി-ഏജന്റ് AI പരിഹാരങ്ങൾ (3 മണിക്കൂർ)
4. **അദ്ധ്യായം 8**: പ്രൊഡക്ഷൻ AI മികച്ച രീതികൾ (1 മണിക്കൂർ)

#### 🛠️ ഇന്‍ഫ്രാസ്ട്രക്ചർ സ്‌പെഷ്യലിസ്റ്റ് പാത (5-7 മണിക്കൂർ)
1. **അദ്ധ്യായം 1**: അടിസ്ഥാനം & ക്വിക്ക് സ്റ്റാർട്ട് (45 മിനിറ്റ്)
2. **അദ്ധ്യായം 3**: കോൺഫിഗറേഷൻ & ഓതന്റിക്കേഷൻ (1 മണിക്കൂർ)
3. **അദ്ധ്യായം 4**: ഇൻഫ്രാസ്ട്രക്ചർ എസ് കോഡ് & ഡിപ്പ്ലോയ്മെന്റ് (1.5 മണിക്കൂർ)
4. **അദ്ധ്യായം 6**: പ്രീ-ഡിപ്പ്ലോയ്മെന്റ് വാലിഡേഷൻ & പദ്ധതീകരണം (1 മണിക്കൂർ)
5. **അദ്ധ്യായം 7**: പ്രശ്നപരിഹാരം & ഡീബഗിംഗ് (1.5 മണിക്കൂർ)
6. **അദ്ധ്യായം 8**: പ്രൊഡക്ഷൻ & എന്റർപ്രൈസ് മാതൃകകൾ (2 മണിക്കൂർ)

#### 🎓 പൂർണ്ണ പഠന യാത്ര (8-12 മണിക്കൂർ)
എല്ലാ 8 അദ്ധ്യായങ്ങളും ക്രമത്തിൽ പൂർത്തിയാക്കുക, പ്രായോഗിക പരിശീലനത്തോടുകൂടി പരിശോധനകൾ നടത്തുക

- **തൊഴിൽ പുരോഗതി**: ക്ലൗഡ് ആർക്കിടെക്റ്റ് 및 AI ഡിപ്പ്ലോയ്മെന്റ് സ്പെഷ്യലിസ്റ്റ് സ്ഥാനങ്ങൾക്കുള്ള യോഗ്യത
- **കമ്മ്യൂണിറ്റി നയതന്ത്രം**: Azure ഡെവലപ്പർ 및 AI കമ്മ്യൂണിറ്റികളിൽ സജീവ അംഗത്വം
- **തുടർച്ചയായ പഠനം**: Microsoft Foundry-യുടെ ആഴത്തിലുള്ള സ്പെഷ്യലൈസേഷനുകൾക്കുള്ള അടിസ്ഥാനമാക്കുക

**പോർട്ട്ഫോളിയോ ആസ്തികൾ:**

- **പ്രൊഫഷണൽ നെറ്റ്‌വർക്ക്**: Azure വിദഗ്ധരുമായും AI ഡിപ്പ്ലോയ്മെന്റ് പ്രായോഗികരുമായുള്ള ബന്ധങ്ങൾ

- **കോർപ്പറേറ്റ് പരിശീലനം**: സംഘടനകളിൽ AZD 및 AI ഡിപ്പ്ലോയ്മെന്റ് കഴിവുകൾ അഭ്യസിപ്പിക്കുക
