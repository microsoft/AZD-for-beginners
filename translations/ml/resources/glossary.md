- **📖 അടിസ്ഥാനങ്ങൾ പഠിക്കുക**: [അധ്യായം 1: AZD അടിസ്ഥാനങ്ങൾ](../docs/getting-started/azd-basics.md)  
- **🤖 AI ടെർംസുകൾ**: [അധ്യായം 2: AI-പ്രഥമ വികസനം](../docs/microsoft-foundry/microsoft-foundry-integration.md)  

## പരിചയം

ഈ സമഗ്ര ഗ്ലോസറി അസ്യൂർ ഡവലപ്പർ CLIയും അസ്യൂർ ക്ലൗഡ് ഡവലപ്പ്മെന്റും ഉപയോഗിക്കുന്ന പദങ്ങൾ, ആശയങ്ങൾ, സംക്ഷേപങ്ങൾ എന്നിവയ്ക്ക് വ്യാഖ്യാനങ്ങൾ നൽകുന്നു. സാങ്കേതിക രേഖകൾ മനസിലാക്കുന്നതിനും പ്രശ്ന പരിഹാരത്തിനും azd പ്രോജക്റ്റുകളും അസ്യൂർ സേവനങ്ങളും സംബന്ധിച്ച് ഫലപ്രദമായി ആശയവിനിമയം നടത്തുന്നതിനും ഇതൊരു അനിവാര്യ റഫറൻസാണ്.

## പഠന ലക്ഷ്യങ്ങൾ

ഈ ഗ്ലോസറി ഉപയോഗിച്ച്, നിങ്ങൾക്ക്:  
- അസ്യൂർ ഡവലപ്പർ CLI യുടെ മുഖ്യ പദങ്ങൾയും ആശയങ്ങളും മനസിലാക്കാം  
- അസ്യൂർ ക്ലൗഡ് വികസനത്തിന്റെ പദസമൂഹവും സാങ്കേതിക പദങ്ങൾ ആസ്പദമാക്കാം  
- Infrastructure as Code (IaC)ക്കും പ്രക്ഷേപണ ടെർമിനോളജിക്കും ഫലപ്രദമായി റഫർൻസ് ചെയ്യാം  
- അസ്യൂർ സർവീസ് നാമങ്ങളും സംക്ഷേപങ്ങളും അവയുടെ ഉപയോഗങ്ങളും മനസിലാക്കാം  
- പ്രശ്നപരിഹാരത്തിനും ഡീബഗിംഗിനുമായുള്ള പദങ്ങൾ ലഭ്യമാക്കാം  
- അസ്യൂർ ആർക്കിടെക്ചർ avancികൾക്കും വികസന ആശയങ്ങൾക്കും പഠനം നടത്താം  

## പഠന ഫലങ്ങൾ

ഈ ഗ്ലോസറി സ്ഥിരതയോടെ പരാമർശിക്കുന്നതിലൂടെയെങ്കിൽ, നിങ്ങൾക്ക്:  
- ശരിയായ അസ്യൂർ ഡവലപ്പർ CLI ടെർമിനോളജിയിലൂടെ ഫലപ്രദമായി ആശയവിനിമയം നടത്താൻ സാധിക്കും  
- സാങ്കേതിക രേഖകൾ ഉത്തരം വരുന്ന പിശകു സന്ദേശങ്ങൾ കൂടുതൽ വ്യക്തമായി മനസിലാക്കാനാകും  
- അസ്യൂർ സേവനങ്ങളും ആശയങ്ങളും ആത്മവിശ്വാസത്തോടെ വഴിതെളിയാൻ കഴിയും  
- പ്രശ്നപരിഹാരത്തിന് അനുയോജ്യമായ സാങ്കേതിക പദസമൂഹം ഉപയോഗിച്ച് സ്റ്റാർട്ട് ചെയ്യും  
- കൃത്യമായ സാങ്കേതിക ഭാഷയുമായി ടീമിൽ സംഭാവന നൽകാൻ കഴിയും  
- ക്രമബദ്ധമായി അസ്യൂർ ക്ലൗഡ് വികസന വിജ്ഞാനം വിപുലീകരിക്കും  

## A

**ARM Template**  
Azure റിസോഴ്‌സ് മാനേജർ ടെംപ്ലേറ്റ്. Azure റിസോഴ്‌സുകൾ പരാമർശിച്ച് നിർവചിക്കുന്ന JSON-ആധാരിത Infrastructure as Code ഫോർമാറ്റ്.

**App Service**  
വെബ് അപ്ലിക്കേഷനുകൾ, REST APIകൾ, ಮತ್ತು മൊബൈൽ ബാക്ക്‌എൻഡുകൾ ഹോസ്റ്റ് ചെയ്യുന്നതിനുള്ള അസ്യൂറിന്റെ പ്ലാറ്റ്ഫോം-ആസ്-എ-സർവീസ് (PaaS) ഓഫറിംഗ്. ഇൻഫ്രാസ്ട്രക്ചർ മാനേജ് ചെയ്യേണ്ടതില്ല.

**Application Insights**  
അപ്ലിക്കേഷൻ പ്രകടന നിരീക്ഷണം (APM) സേവനം, അപ്ലിക്കേഷൻ പ്രകടനം, ലഭ്യത, ഉപയോഗം എന്നിവയിൽ വിദഗ്ധ ധാരണകൾ നൽകുന്നു.

**Azure CLI**  
അസ്യൂർ റിസോഴ്‌സുകൾ നിയന്ത്രിക്കുന്ന കമാൻഡ്-ലൈൻ ഇന്റർഫേസ്. azd പ്രമാണിച്ചു കൊണ്ട് ഓതന്റിക്കേഷൻക്കും കുറച്ച് പ്രവർത്തനങ്ങൾക്കുമായി ഉപയോഗിക്കുന്നു.

**Azure Developer CLI (azd)**  
ഡവലപ്പർമാർക്ക് കേന്ദ്രീകരിച്ച കമാൻഡ്-ലൈൻ ടൂൾ, ടിംപ്ലേറ്റുകളും Infrastructure as Code ഉപയോഗിച്ച് അസ്യൂറിലേക്ക് അപ്ലിക്കേഷനുകൾ നിർമിക്കാനും പ്രക്ഷേപിക്കാനും സഹായിക്കുന്നു.

**azure.yaml**  
azd പ്രോജക്റ്റിനുള്ള പ്രധാനം കോൺഫിഗറേഷൻ ഫയൽ, സേവനങ്ങൾ, ഇൻഫ്രാസ്ട്രക്ചർ, പ്രക്ഷേപണ ഹൂക്കുകൾ നിർവചിക്കുന്നു.

**Azure Resource Manager (ARM)**  
അസ്യൂറിന്റെ വികസനവും മാനേജ്മെൻറും ചെയ്യാൻ ഉള്ള സേവനം, റിസോഴ്‌സുകൾ സൃഷ്ടിക്കുക, അപ്‌ഡേറ്റ് ചെയ്യുക, ഡിലീറ്റ് ചെയ്യുക എന്നിവയ്ക്ക് മാനേജ്മെന്റ് ലെയർ നൽകുന്നു.

## B

**Bicep**  
മൈക്രോസോഫ്റ്റ് വികസിപ്പിച്ച ഡിസ്പ്ലോയ്‌മെന്റിനുള്ള ഡൊമൈൻ സ്പെസിഫിക് ലാംഗ്വേജ് (DSL). ARM ടെംപ്ലേറ്റുകളുമായി താരതമ്യപ്പെടുത്തുമ്പോൾ ലളിതമായി സിന്താക്സ് നൽകുകയും ARM യിലേക്ക് കമ്പൈൽ ചെയ്യുകയും ചെയ്യുന്നു.

**Build**  
സോഴ്സ് കോഡ് കമ്പൈൽ ചെയ്യുക, ആശ്രിതത്വങ്ങൾ ഇൻസ്റ്റാൾ ചെയ്യുക, അപ്ലിക്കേഷൻ പ്രക്ഷേപണത്തിന് തയ്യാറാക്കുക എന്ന പ്രക്രിയ.

**Blue-Green Deployment**  
ഇടക്കാല ഡൗൺടൈം കുറക്കുകയും റിസ്‌ക്ക് നിയന്ത്രിക്കുകയും ചെയ്യാൻ രണ്ട് സമാന പ്രൊഡക്ഷൻ എന്വയോൺമെന്റുകളിൽ (ബ്ലൂ, ഗ്രീൻ) ഒരു ഡിപ്ലോയ്മെന്റ് തന്ത്രം.

## C

**Container Apps**  
ഹോർഡുചെയ്യാൻ സങ്കീർണമായ ഇൻഫ്രാസ്ട്രക്ചർ ഇല്ലാതെ കന്റെയ്‌നർ ആപ്ലിക്കേഷനുകൾ ഓടിക്കാൻ അനുവദിക്കുന്ന അസ്യൂറിന്റെ സർവർലെസ് കന്റെയ്‌നർ сервис്.

**CI/CD**  
കണ്‍ടീന്യുവസ് ഇന്റഗ്രേഷൻ / കണ്‍ടീന്യുവസ് ഡിപ്ലോയ്‌മെന്റ്. കോഡ് മാറ്റങ്ങൾ സംയോജിപ്പിക്കുന്നതും അപ്ലിക്കേഷനുകൾ ഡിപ്ലോയിംഗ് ചെയ്യുന്ന തന്ത്രപരവും ഓട്ടോമേറ്റഡ് രീതികൾ.

**Cosmos DB**  
അസ്യൂറിന്റെ അതുല്യമായി ലോകമാകമാനമായി വിതരണം ചെയ്ത, മൾട്ടി-മോഡൽ ഡാറ്റാബേസ് സേവനം. ത്രൂപുട്ട്, ലാറ്റൻസി, ലഭ്യത, ഏകീകൃതത എന്നിവയ്ക്കുള്ള സമഗ്ര SLAകൾ നൽകുന്നു.

**Configuration**  
അപ്ലിക്കേഷൻ പ്രവൃത്തി രീതിയും ഡിപ്ലോയ്‌മെന്റ് ഓപ്ഷനുകളും നിയന്ത്രിക്കുന്ന ക്രമീകരണങ്ങളും പരാമർശനങ്ങളും.

## D

**Deployment**  
അപ്ലിക്കേഷനുകളും അവയുടെ ആശ്രിതത്വങ്ങളും ലക്ഷ്യ ഇൻഫ്രാസ്ട്രക്ചറിൽ ഇൻസ്റ്റാൾ ചെയ്ത് കോൺഫിഗർ ചെയ്യുന്നതിനുള്ള പ്രക്രിയ.

**Docker**  
കന്റെയ്‌നർ സാങ്കേതികവിദ്യ ഉപയോഗിച്ച് അപ്ലിക്കേഷനുകൾ വികസിപ്പിക്കുന്നതും ഷിപ്പുചെയ്യുന്നതും ഓടിക്കുന്നതും സാധ്യമാക്കുന്ന പ്ലാറ്റ്ഫോം.

**Dockerfile**  
Docker കന്റെയ്‌നർ ഇമേജ് നിർമ്മിക്കാൻ വേണ്ട നിർദ്ദേശങ്ങൾ ഉൾക്കുന്ന ടെക്സ്റ്റ് ഫയൽ.

## E

**Environment**  
നിങ്ങളുടെ ആപ്ലിക്കേഷന്റെ ഒരു പ്രത്യേക പ്രതിലിപി (ഉദാ: വികസനം, സ്റ്റേജിംഗ്, പ്രൊഡക്ഷൻ) പ്രതിനിധീകരിക്കുന്ന പ്രക്ഷേപണ ലക്ഷ്യം.

**Environment Variables**  
കീ-വ്യാലു ജോഡികളായി സൂക്ഷിക്കപ്പെട്ട, സമയംപ്രവർത്തനത്തു അപ്ലിക്കേഷനുകൾ ആക്‌സസ് ചെയ്തേക്കാവുന്ന ക്രമീകരണ മൂല്യങ്ങൾ.

**Endpoint**  
ഒരു ആപ്ലിക്കേഷൻ അല്ലെങ്കിൽ സർവീസ് ആക്‌സസ് ചെയ്യാവുന്ന URL അല്ലെങ്കിൽ നെറ്റ്‌വർക്ക് വിലാസം.

## F

**Function App**  
ഇന്ഫ്രാസ്ട്രക്ചർ മാനേജ് ചെയ്യാതെ ഇവന്റ് ഡ്രീവൻ കോഡ് ഓടിക്കാൻ അനുവദിക്കുന്ന അസ്യൂറിന്റെ സർവർലെസ് കംപ്യൂട്ട് സേവനം.

## G

**GitHub Actions**  
GitHub റവോസിറ്ററികളുമായി ലയിച്ചിരിക്കുന്ന CI/CD പ്ലാറ്റ്ഫോം, പ്രവർത്തനപ്രവാഹങ്ങൾ സ്വയം ഓട്ടോമേറ്റുചെയ്യുന്നു.

**Git**  
സോഴ്സ് കോഡിലെ മാറ്റങ്ങൾ ട്രാക്ക് ചെയ്യാനുള്ള വിതരിച്ച പതിപ്പുമാറ്റം നിയന്ത്രണ സംവിധാനം.

## H

**Hooks**  
പ്രക്ഷേപണ ജീവിതചക്രത്തിൽ നിശ്ചിത നിമിഷങ്ങളിൽ (preprovision, postprovision, predeploy, postdeploy) പ്രവർത്തിക്കുന്ന കസ്റ്റം സ്ക്രിപ്റ്റുകളും കമാൻഡുകളും.

**Host**  
ഒരു ആപ്ലിക്കേഷൻ പ്രക്ഷേപണം ചെയ്യപ്പെടുന്ന അസ്യൂർ സർവീസ് തരം (ഉദാ: appservice, containerapp, function).

## I

**Infrastructure as Code (IaC)**  
ഇൻഫ്രാസ്ട്രക്ചർ മാനേജ്മെന്റ് കൈമുറിവൻ പ്രക്രിയകളേക്കാൾ കോഡിന്റെ മുഖാന്തിരം നിർവഹിക്കുന്ന രീതി.

**Init**  
പുതിയ azd പ്രോജക്റ്റ് തുടക്കം നടത്താനുള്ള പ്രക്രിയ, സാധാരണയായി ഒരു ടിംപ്ലേറ്റ് ഉപയോഗിച്ചാണ്.

## J

**JSON**  
JavaScript ഒബ്‌ജക്റ്റ് നോട്ടേഷൻ. കോൺഫിഗറേഷൻ ഫയലുകൾക്കും API പ്രതികരണങ്ങൾക്കും സാധാരണയായി ഉപയോഗിക്കുന്ന ഡേറ്റാ ഇന്റർചേഞ്ച് ഫോർമാറ്റ്.

**JWT**  
JSON വെബ് ടോക്കൺ. JSON ഒബ്‌ജക്റ്റായി പാര്‍ട്ടികൾക്കിടയിലെ വിവരങ്ങൾ സുരക്ഷിതമായി പ്രേഷണം ചെയ്യാനുള്ള സ്റ്റാൻഡേര്‍ഡ്.

## K

**Key Vault**  
രഹസ്യങ്ങൾ, കീകൾ, സർട്ടിഫിക്കറ്റുകൾ എന്നിവ സുരക്ഷിതമായി സൂക്ഷിക്കാനും മാനേജുമെന്റ് ചെയ്യാനുമുള്ള അസ്യൂർ സേവനം.

**Kusto Query Language (KQL)**  
അസ്യൂർ മോണിറ്റർ, ആപ്ലിക്കേഷൻ ഇൻസൈറ്റ്സുകൾ, മറ്റ് അസ്യൂർ സേവനങ്ങളിൽ ഡാറ്റ അനാലിസിസ് ചെയ്യുന്നതിനുള്ള ക്വറി ഭാഷ.

## L

**Load Balancer**  
വേണ്ടുന്ന സേര്‍വേഴ്‌സിലോ ഇൻസ്റ്റൻസിലോ ഇൻകമിംഗ് നെറ്റ്‌വർക്ക് ട്രാഫിക് വിഭജിക്കുന്ന സേവനം.

**Log Analytics**  
ക്ലൗഡ്, ഓൺ-പ്രെമിസസ് പരിസരങ്ങളിൽ നിന്നുള്ള ടെലിമെട്രി ഡാറ്റ ശേഖരിക്കുകയും വിശകലനം ചെയ്‌തും നടപടികൾ സ്വീകരിക്കുന്ന അസ്യൂർ സേവനം.

## M

**Managed Identity**  
അസ്യൂർ മറ്റ് അസ്യൂർ സേവനങ്ങളിലേക്ക് ഓതന്റിക്കേഷൻ നടത്താൻ സ്വയമേവ മാനേജുചെയ്യുന്ന ഐഡന്റിറ്റി നൽകുന്ന സവിശേഷത.

**Microservices**  
സ്രഷ്ടാവിൽ ചെറിയ, സ്വതന്ത്ര സേവനങ്ങളുടെ കൂട്ടമായി ആപ്ലിക്കേഷനുകൾ നിർമ്മിക്കുന്ന ആർക്കിടെക്ചറൽ സമീപനം.

**Monitor**  
അപ്ലിക്കേഷനുകളിലും ഇൻഫ്രാസ്ട്രക്ചറിലും പൂർണ്ണ-സ്റ്റാക്ക് നിരീക്ഷണം നൽകുന്ന അസ്യൂറിന്റെ ഏകീകൃത നിരീക്ഷണ പരിഹാരം.

## N

**Node.js**  
ക്രോം V8 ജാവാസ്ക്രിപ്റ്റ് എന്‍ജിനിൽ നിർമ്മിച്ച സർവർ-പുറം აპ്ലിക്കേഷൻ നിർമ്മാണത്തിനുള്ള ജാവാസ്ക്രിപ്റ്റ് റൺടൈം.

**npm**  
Node.js നു വേണ്ടി ആശ്രിതത്വങ്ങളും പാക്കേജുകളും നിയന്ത്രിക്കുന്ന പാക്കേജ് മാനേജർ.

## O

**Output**  
തയാറാക്കിയ ഇൻഫ്രാസ്ട്രക്ചർ പ്രക്ഷേപണത്തിൽ നിന്നുള്ള മൂല്യങ്ങൾ, അപ്ലിക്കേഷനുകൾക്കും മറ്റ് റിസോഴ്‌സുകൾക്കും ഉപയോഗിക്കാൻ പറ്റുക.

## P

**Package**  
അപ്ലിക്കേഷൻ കോഡ്, ആശ്രിതത്വ നിർവചനം എന്നിവ പ്രക്ഷേപണത്തിന് തയ്യാറാക്കുന്നതിന്റെ പ്രക്രിയ.

**Parameters**  
പ്രക്ഷേപണങ്ങൾ അനുയോജ്യമായി ഇനംമാറ്റാൻ ഇൻഫ്രാസ്ട്രക്ചർ ടെംപ്ലേറ്റുകൾക്ക് നൽകുന്ന ഇൻപുട്ട് മൂല്യങ്ങൾ.

**PostgreSQL**  
വളരെയധികം പിന്തുണ ലഭിച്ച ഏകസ്രോതസ്സുകൂടിയ റിലേഷണൽ ഡാറ്റാബേസ് സംവിധാനം, അസ്യൂറിൽ മാനേജ് ചെയ്ത സേവനമായി.

**Provisioning**  
ഇൻഫ്രാസ്ട്രക്ചർ ടെംപ്ലേറ്റുകളിൽ നിർവചിച്ച അസ്യൂർ റിസോഴ്‌സുകൾ സൃഷ്ടിക്കുകയും ക്രമീകരിക്കുകയും ചെയ്യുന്നത്.

## Q

**Quota**  
അസ്യൂർ സബ്‌സ്‌ക്രിപ്ഷനിലും പ്രദേശത്തും സൃഷ്ടിക്കാനാകുന്ന വിഭവങ്ങളുടെ പരിധികൾ.

## R

**Resource Group**  
പോലുള്ള ലൈഫ് സൈക്കിൾ, അനുമതികൾ, നയം എന്നിവ പങ്കിടുന്ന അസ്യൂർ റിസോഴ്‌സുകളുടെ ലൊജിക്കൽ കൺടെയ്നർ.

**Resource Token**  
azd ജനിപ്പിക്കുന്ന പ്രത്യേക സ്‌ട്രിംഗ്, പ്രക്ഷേപണങ്ങൾക്കിടയിൽ റിസോഴ്‌സ് നാമങ്ങൾ വ്യക്തിമാക്കാൻ.

**REST API**  
HTTP രീതികൾ ഉപയോഗിച്ച് നെറ്റ്‌വർക്കിലായുള്ള ആപ്ലിക്കേഷനുകളെ രൂപകൽപ്പന ചെയ്യുന്ന ആർക്കിടെക്ചറൽ ശൈലി.

**Rollback**  
ഒരു ആപ്ലിക്കശ്യന്റെ അല്ലെങ്കിൽ ഇൻഫ്രാസ്ട്രക്ചർ കോൺഫിഗറേഷൻ മുമ്പത്തെ പതിപ്പിലേക്കു മടങ്ങുക.

## S

**Service**  
azure.yaml ൽ നിർവചിച്ച ആപ്ലിക്കേഷൻ ഘടകം (ഉദാ: വെബ് ഫ്രണ്ട്‌എൻഡ്, API ബാക്ക്‌എൻഡ്, ഡാറ്റാബേസ്).

**SKU**  
സ്റ്റോക്ക് കീപ്പിംഗ് യൂണിറ്റ്. അസ്യൂർ റിസോഴ്‌സുകൾക്കുള്ള വിവിധ സേവന സ്റ്റെർ/പ്രകടനനിലകൾ.

**SQL Database**  
മൈക്രോസോഫ്റ്റ് SQL സർവറിനെ അടിസ്ഥാനമാക്കി അസ്യൂറിന്റെ മാനേജ് ചെയ്ത റിലേഷണൽ ഡാറ്റാബേസ് സേവനം.

**Static Web Apps**  
സോഴ്സ് കോഡ് റവോസിറ്റ്‌റികളിൽ നിന്ന് പൂർണ്ണ-സ്റ്റാക് വെബ് ആപ്ലിക്കേഷനുകൾ നിർമ്മിക്കുന്നതിന് ഒരു അസ്യൂർ സേവനം.

**Storage Account**  
ഡേറ്റാ വസ്തുക്കൾ (ബ്ലോബുകൾ, ഫയലുകൾ, ക്യൂകൾ, പട്ടികകൾ) സൂക്ഷിക്കുന്ന ക്ലൗഡ് സ്റ്റോറേജ് അസ്യൂർ സേവനം.

**Subscription**  
റിസോഴ്‌സ് ഗ്രൂപ്പുകളും റിസോഴ്‌സുകളും, ബില്ലിംഗ്, പ്രവേശന നിയന്ത്രണവും ഉൾപ്പെടുന്ന അസ്യൂർ അക്കൗണ്ട് കൺടെയ്നർ.

## T

**Template**  
പ്രീ-നിർമ്മിച്ച പ്രോജക്റ്റ് ഘടന, ആപ്ലിക്കേഷൻ കോഡ്, ഇൻഫ്രാസ്ട്രക്ചർ നിർവചനങ്ങൾ, കോൺഫിഗറേഷൻ എന്നിവ ഉൾപ്പെടുത്തുന്നു.

**Terraform**  
അസ്യൂർ അടക്കം പല ക്ലൗഡ് പ്രൊവൈഡറുകളെയും പിന്തുണയ്ക്കുന്ന ഒപൺ സോഴ്‌സ് Infrastructure as Code ഉപകരണം.

**Traffic Manager**  
ഗ്ലോബൽ അസ്യൂർ മേഖലകളെക്കിടയിൽ ട്രാഫിക് വിതരണത്തിനുള്ള DNS-അടിസ്ഥാനമുള്ള അസ്യൂർ ട്രാഫിക് ലോഡ് ബാലൻസർ.

## U

**URI**  
യൂണിഫോം റിസോഴ്‌സ് ഐഡന്റിഫയർ. ഒരു പ്രത്യേക റിസോഴ്‌സ് തിരിച്ചറിയുന്ന സ്റ്റ്രിംഗ്.

**URL**  
യൂണിഫോം റിസോഴ്‌സ് ലോക്കേറ്റർ. ഒരു റിസോഴ്‌സ് എവിടെ ഉണ്ട്, എങ്ങനെ ലഭ്യമാക്കാം എന്ന് വ്യക്തമാക്കുന്ന ഒരു URI തരം.

## V

**Virtual Network (VNet)**  
അസ്യൂറിൽ പ്രൈവറ്റ് നെറ്റ്‌വർക്കുകൾക്കുള്ള അടിസ്ഥാന ഘടകം, ഐസൊലേഷൻ കൂടാതെ സെഗ്മെന്റേഷൻ നൽകുന്നു.

**VS Code**  
വിഷ്വൽ സ്റ്റുക്കിയോ കോഡ്. മികച്ച അസ്യൂർ, azd ഏകീകരണത്തോടെയുള്ള പ്രശസ്തമായ കോഡ് എഡിറ്റർ.

## W

**Webhook**  
നിശ്ചിത ഇവൻറുകൾ പിറകെ പ്രേഷിപ്പിക്കുന്ന HTTP കോलबാക്ക്, സാധാരണയായി CI/CD പൈപ്പ് ലൈൻസ്.

**What-if**  
പ്രക്ഷേപണം നടപ്പിലാക്കാതെ മാറ്റങ്ങൾ എന്തെല്ലാമാകുമെന്ന് കാണിക്കുന്ന അസ്യൂർ സവിശേഷത.

## Y

**YAML**  
YAML Ain't Markup Language. azure.yaml പോലുള്ള കോൺഫിഗറേഷൻ ഫയലുകൾക്കായി ഉപയോഗിക്കുന്ന മനുഷ്യൻ വായിക്കാവുന്ന ഡാറ്റാ സീരിയലൈസേഷൻ സ്റ്റാൻഡേര്‍ഡ്.

## Z

**Zone**  
ഒരു അസ്യൂർ മേഖലയിലെ ഭൗതിക വേർതിരിച്ചിട്ടുള്ള സ്ഥിതികൾ, പുനരവതരണം ഉയർന്ന ലഭ്യത എന്നിവ നൽകുന്നു.

## സാധാരണ സംക്ഷേപങ്ങൾ

| സംക്ഷേപം | പൂർണ്ണ രൂപം | വിവരണം |
|---------|-------------|--------|
| AAD | Azure Active Directory (ഇപ്പോൾ Microsoft Entra ID) | ഐഡന്റിറ്റി, പ്രവേശന നിയന്ത്രണ സേവനം |
| ACR | Azure Container Registry | കന്റെയ്‌നർ ഇമേജ് രജിസ്റ്റി സേവനം |
| AKS | Azure Kubernetes Service | മാനേജ് చేయപ്പെട്ട കുബർനെറ്റിസ് സേവനം |
| API | Application Programming Interface | സോഫ്റ്റ്‌വെയർ നിർമ്മാണത്തിനുള്ള പ്രോട്ടോക്കോളുകൾ |
| ARM | Azure Resource Manager | അസ്യൂർ ഡിപ്ലോയ്‌മെന്റ്, മാനേജ്മെന്റ് സേവനം |
| CDN | Content Delivery Network | വിതരിച്ചിട്ടുള്ള സർവേഴ്സ് നെറ്റ്‌വർക്ക് |
| CI/CD | Continuous Integration/Continuous Deployment | ഓട്ടോമേറ്റഡ് വികസന രീതികൾ |
| CLI | Command Line Interface | ടെക്സ്റ്റ് അടിസ്ഥാനമുള്ള ഉപയോക്തൃ ഇന്റർഫേസ് |
| DNS | Domain Name System | ഡൊമെയ്ൻ നാമങ്ങൾ IP വിലാസങ്ങളിലേക്കു മാറ്റുന്ന സംവിധാനം |
| HTTPS | Hypertext Transfer Protocol Secure | HTTP ന്റെ സുരക്ഷിത പതിപ്പ് |
| IaC | Infrastructure as Code | കോഡിലൂടെ ഇൻഫ്രാസ്ട്രക്ചർ മാനേജ്‌മെന്റ് |
| JSON | JavaScript Object Notation | ഡാറ്റാ ഇന്റർചേഞ്ച് ഫോർമാറ്റ് |
| JWT | JSON Web Token | സുരക്ഷിത വിവര പ്രേഷണത്തിന് ടോക്കൺ ഫോർമാറ്റ് |
| KQL | Kusto Query Language | അസ്യൂർ ഡാറ്റാ സേവനങ്ങളിലെ ക്വറി ഭാഷ |
| RBAC | Role-Based Access Control | ഉപയോക്താവിന്റെ ഭുമികയുടെ അടിസ്ഥാനത്തിൽ പ്രവേശന നിയന്ത്രണം |
| REST | Representational State Transfer | വെബ് സേവനങ്ങൾക്ക് ഉള്ള ആർക്കിടെക്ചറൽ ശൈലി |
| SDK | Software Development Kit | വികസന ഉപകരണങ്ങളുടെ കൂട്ടം |
| SLA | Service Level Agreement | സേവന ലഭ്യത/പ്രകടന പ്രതിജ്ഞ |
| SQL | Structured Query Language | റിലേഷണൽ ഡാറ്റാബേസ് മാനേജ്മെന്റ് ഭാഷ |
| SSL/TLS | Secure Sockets Layer/Transport Layer Security | ക്രിപ്റ്റോഗ്രാഫിക് പ്രോട്ടോകോളുകൾ |

## സാന്ദർഭിക പദങ്ങൾ

### വികസന പദങ്ങൾ  
- **ഹോട്ട് റീലോഡ്**: പുനരാരംഭം നടത്താതെ വികസന സമയത്ത് ആപ്ലിക്കേഷൻ സ്വയം അപ്ഡേറ്റ് ചെയ്യുന്നത്  
- **ബിൽഡ് പൈപ്പ്‌ലൈൻ**: കോഡ് ബിൽഡ്, ടെസ്റ്റ് ഓട്ടോമേറ്റുചെയ്യുന്ന പ്രക്രിയ  
- **ഡിപ്ലോയ്മെന്റ് സ്ലോട്ട്**: App Service ൽ ഉള്ള സ്റ്റേജിംഗ് ഇന്വയോൺമെന്റ്  
- **എൻവയോൺമെന്റ് പോരിത്യ**: വികസനം, സ്റ്റേജിംഗ്, പ്രൊഡക്ഷൻ പരിസരങ്ങൾ സമാനമാക്കൽ  

### സുരക്ഷാ പദങ്ങൾ  
- **Managed Identity**: അശ്രിതവർഗ്ഗ അത്തരം സവിശേഷത  
- **Key Vault**: രഹസ്യങ്ങൾ, കീകൾ, സർട്ടിഫിക്കറ്റുകൾ സുരക്ഷിതമായ സൂക്ഷിപ്പ്  
- **RBAC**: അസ്യൂർ റിസോഴ്‌സുകൾക്കായി ഭുമികാ അധിഷ്ഠിത പ്രവേശന നിയന്ത്രണം  
- **Network Security Group**: നെറ്റ്‌വർക്ക് ട്രാഫിക് നിയന്ത്രിക്കുന്ന വെർച്ച്വൽ ഫയർവാൾ  

### നിരീക്ഷണ പദങ്ങൾ  
- **Telemetry**: കഴിവോടെ ശേഖരിക്കുന്ന അളവുകളും ഡാറ്റയും  
- **APM**: അപ്ലിക്കേഷൻ പ്രകടന നിരീക്ഷണം  
- **Log Analytics**: ലോഗ് ഡാറ്റ ശേഖരണവും വിശകലനവും  
- **Alert Rules**: മെട്രിക് അടിസ്ഥാനമാക്കുന്ന ഓട്ടോമേറ്റഡ് അറിയിപ്പുകൾ  

### പ്രക്ഷേപണ പദങ്ങൾ  
- **Blue-Green Deployment**: ഡൗൺടൈം ഇല്ലാത്ത പ്രക്ഷേപണ തന്ത്രം  
- **Canary Deployment**: ഉപയോക്താക്കളിൽ കുറച്ച് മാത്രം ശ്രദ്ധാപൂർവം പ്രകാശനം  
- **Rolling Update**: അപ്ലിക്കേഷൻ ഇൻസ്റ്റൻസുകൾ പരമ്പരാഗതമായി മാറ്റിക്കൽ  
- **Rollback**: മുമ്പത്തെ അപ്ലിക്കേഷൻ പതിപ്പിലേക്ക് മടങ്ങൽ
