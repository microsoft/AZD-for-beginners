**എല്ലാ അധ്യായങ്ങൾക്കും റഫറൻസ്**

- **📖 അടിസ്ഥാനങ്ങൾ പഠിക്കുക**: [Chapter 1: AZD Basics](../docs/getting-started/azd-basics.md)  
- **🤖 AI പദങ്ങൾ**: [Chapter 2: AI-First Development](../docs/microsoft-foundry/microsoft-foundry-integration.md)  

## പരിചയം

ഈ സമഗ്ര പദവിഭവം ആസ്യൂർ ഡെവലപ്പർ CLIയും ആസ്യൂർ ക്ലൗഡ് വികസനവും ഉപയോഗിക്കുന്ന പദങ്ങൾ, ആശയങ്ങൾ, ചുരുക്കപ്പേരുകൾ എന്നവയുടെ വ്യാഖ്യാനങ്ങൾ നൽകുന്നു. സാങ്കേതിക രേഖകൾ മനസിലാക്കാനും പ്രശ്നപരിഹാരത്തിനും azd പ്രോജക്റ്റുകളിലും ആസ്യൂർ സേവനങ്ങളിലും കാര്യക്ഷമമായി ആശയവിനിമയം നടത്തുന്നതിനും മൂല്യവത്തായ റഫറൻസാണ്.

## പഠന ലക്ഷ്യങ്ങൾ

ഈ പദവിഭവം ഉപയോഗിച്ച്, നിങ്ങൾക്ക്:  
- നിര്‍ണായകമായ ആസ്യൂർ ഡെവലപ്പർ CLI പദങ്ങൾക്കും ആശയങ്ങൾക്കും മനസ്സിലാക്കൽ  
- ആസ്യൂർ ക്ലൗഡ് വികസന ഉപദേശങ്ങൾക്കും സാങ്കേതിക പദങ്ങൾക്കുമുള്ള വിദഗ്ധത  
- Infrastructure as Code ൽ ഉപയോഗിക്കുന്ന deployment പദങ്ങൾ കാര്യക്ഷമമായി അനുസ്മരണം  
- ആസ്യൂർ സേവനങ്ങളുടെ പേര്, ചുരുക്കപ്പേരുകൾ, അവയുടെ ഉദ്ദേശ്യങ്ങൾ ബോധ്യമായിരിക്കുക  
- പ്രശ്നപരിഹാരത്തിനും ഡീബഗ്ഗിംഗിനുമുള്ള പദങ്ങളുടെ വ്യാഖ്യാനം ലഭിക്കുക  
- വികസിത ആസ്യൂർ ആർക്കിടെക്ചർ, വികസന ആശയങ്ങൾ പഠിക്കുക  

## പഠന ഫലങ്ങൾ

ഈ പദവിഭവം പതിവായി റഫറൻസായി ഉപയോഗിക്കുമ്പോൾ, നിങ്ങൾക്ക്:  
- യോജിച്ച ആസ്യൂർ ഡെവലപ്പർ CLI പദങ്ങൾ ഉപയോഗിച്ച് കാര്യക്ഷമമായ ആശയവിനിമയം നടത്താൻ  
- സാങ്കേതിക രേഖകളും പിശക് സന്ദേശങ്ങളും കൂടുതൽ വ്യക്തമായി മനസ്സിലാക്കാൻ  
- ആസ്യൂർ സേവനങ്ങളും ആശയങ്ങളും ആത്മവിശ്വാസത്തോടെ നാവിഗേറ്റ് ചെയ്യാൻ  
- അനുയോജ്യമായ സാങ്കേതിക പദങ്ങൾ ഉപയോഗിച്ച് പ്രശ്നങ്ങൾ പരിഹരിക്കാൻ  
- കൃത്യമായ സാങ്കേതിക ഭാഷ ഉപയോഗിച്ച് ടീമുമായി സംവേദനം നടത്താൻ  
- ആസ്യൂർ ക്ലൗഡ് വികസന അറിവ് സമ്പ്രസാരിപ്പിക്കാൻ  

## A

**ARM Template**  
ആസ്യൂർ റിസോഴ്സ് മാനേജർ ടെംപ്ലേറ്റ്. JSON അടിസ്ഥാനത്തിൽ പുറുപ്പാട് നിർവചിക്കുന്ന Infrastructure as Code ഫോർമാറ്റ്.

**App Service**  
ആസ്യൂറിന്റെ പി.ഏ.എസ്. പ്ലാറ്റ്ഫോം വെബ്ബ് ആപ്ലിക്കേഷനുകൾ, REST APIs, മൊബൈൽ ബാക്ക്എൻഡുകൾ infrastructure മാനേജ്മെന്റ് ഇല്ലാതെ ഹോസ്റ്റുചെയ്യാൻ.

**Application Insights**  
ആസ്യൂർ ആപ്ലിക്കേഷൻ പ്രകടനം നിരീക്ഷിക്കുന്ന (APM) സേവനം, ആപ്ലിക്കേഷൻ പ്രവർത്തനം, ലഭ്യത, ഉപയോഗം എന്നിവയുടെ ആഴമുള്ള ദൃശ്യങ്ങൾ നൽകും.

**Azure CLI**  
ആസ്യൂർ റിസോഴ്സുകൾ നിയന്ത്രിക്കാൻ കമാൻഡ്-ലൈൻ ഇന്റർഫേസ്. azd 인증മേഖലക്കും ചില പ്രവർത്തനങ്ങൾക്കും ഉപയോഗിക്കുന്നു.

**Azure Developer CLI (azd)**  
ഡെവലപർമാർക്ക് ഉദ്ദേശിച്ച കമാൻഡ്-ലൈൻ ടൂൾ, templates ഉം Infrastructure as Code ഉം ഉപയോഗിച്ച് ആപ്പുകൾ ആസ്യൂറിലേക്ക് നിർമ്മിക്കുകയും വിന്യസിക്കുകയും ചെയ്യുന്നതിനായി പ്രക്രിയ വേഗത്തിലാക്കുന്നു.

**azure.yaml**  
azd പ്രോജക്റ്റിന് പ്രധാന കൺഫിഗറേഷൻ ഫയൽ, സერვീസുകൾ, ഇൻഫ്രാസ്ട്രക്ചർ, deployment ഹുക്കുകൾ നിർവചിക്കുന്നു.

**Azure Resource Manager (ARM)**  
ആസ്യൂറിന്റെ deployment മാർഗ്ഗവും മാനേജ്മെന്റും ചെയ്യാൻ ഒരു മേൽനോട്ട പാളി.

## B

**Bicep**  
മൈക്രോസോഫ്റ്റ് വികസിപ്പിച്ച പ്രത്യേക ഡൊമെയ്ൻ-സ്പെസിഫിക് ഭാഷ (DSL) ആസ്യൂർ റിസോഴ്സ് ഡിപ്ലോയ്മെന്റിന്. ARM ടെംപ്ലേറ്റുകൾക്കു പകരം ലളിതമായ syntax നൽകുന്നു.

**Build**  
സോഴ്‌സ് കോഡ് കമ്പൈൽ ചെയ്യലും, ഡിപ്പെൻഡൻസികൾ ഇൻസ്റ്റാൾ ചെയ്യലും, അപ്ലിക്കേഷൻ deployment-നായി തയാറാക്കലും.

**Blue-Green Deployment**  
ഉല്പാദന രണ്ടു സമാന പരിസ്ഥിതികൾ (നീല-പച്ച) ഉപയോഗിച്ച് ഡൗൺടൈമും അപകടവും കുറയ്ക്കുന്ന deployment രീതി.

## C

**Container Apps**  
സർവർ ലെസ് കൺടെയ്‌നർ സേവനം, സങ്കീർണ്ണമായ ഇൻഫ്രാസ്ട്രക്ചർ മാനേജ്മെന്റ് ഇല്ലാതെ കൺടെയ്‌നറൈസ് ചെയ്ത ആപ്ലിക്കേഷൻ നടത്താൻ.

**CI/CD**  
Continuous Integration/Continuous Deployment. കോഡ് മാറ്റങ്ങൾ ഏകോപിപ്പിക്കുകയും ആപ്ലിക്കേഷൻ സ്വയമേറ്റു വിന്യസിക്കുകയും ചെയ്യുന്നതിന്റെ ഓട്ടോമേറ്റഡ് പ്രാക്ടിസുകൾ.

**Cosmos DB**  
ആസ്യൂറിന്റെ ആഗോളമായി വിതരണം ചെയ്ത മൾട്ടി മോഡൽ ഡാറ്റാബേസ് സേവനം, throughput, latency, ലഭ്യത, പ്രത്യേകത എന്നിവയ്ക്കായി സമഗ്ര SLA പ്രകാരം.

**Configuration**  
ആപ്ലിക്കേഷന്റെ പ്രവർത്തനം, deployment ഓപ്ഷനുകൾ നിയന്ത്രിക്കുന്ന സെറ്റിംഗുകളും പരാമീറ്ററുകളും.

## D

**Deployment**  
ആപ്ലിക്കേഷനുകളും അവയുടെ ഡിപ്പെൻഡൻസികളും ടാർഗെറ്റ് ഇൻഫ്രാസ്ട്രക്ചറില് ഇൻസ്റ്റാൾ ചെയ്ത് ക്രമീകരിക്കുന്ന പ്രക്രിയ.

**Docker**  
കൺടെയ്‌നറേഷൻ സാങ്കേതികത ഉപയോഗിച്ച് ആപ്ലിക്കേഷനുകൾ വികസിപ്പിക്കുകയും ഷിപ്പും ചെയ്യുകയും റൺ ചെയ്യുകയും ചെയ്യുന്ന പ്ലാറ്റ്ഫോം.

**Dockerfile**  
ഡോക്കർ കൺടെയ്‌നർ ഇമേജ് നിർമ്മിക്കാൻ നിർദ്ദേശങ്ങൾ അടങ്ങിയ ടെക്സ്റ്റ് ഫയൽ.

## E

**Environment**  
കുറഞ്ഞും ഒരു പ്രത്യേക ആപ്ലിക്കേഷൻ ഇൻസ്റ്റൻസ് (ഉദാഹരണത്തിന്, വികസനം, സ്റ്റേജിംഗ്, ഉല്പാദനം) പ്രതിനിധീകരിക്കുന്ന deployment ലക്ഷ്യം.

**Environment Variables**  
ആപ്ലിക്കേഷനുകൾ റൺടൈമിൽ ആക്‌സസ് ചെയ്യാവുന്ന കീ-വാല്യു ജോഡികൾ രൂപത്തിലുള്ള ക്രമീകരണ മൂല്യങ്ങൾ.

**Endpoint**  
ആപ്ലിക്കേഷൻ അല്ലെങ്കിൽ സേവനം എത്തിക്കുന്ന URL അല്ലെങ്കിൽ നെറ്റ്‌വർക്ക് വിലാസം.

## F

**Function App**  
ഇന്ഫ്രാസ്ട്രക്ചർ മാനേജ്മെന്റ് ഇല്ലാതെ ഇവന്റ്-ഡ്രിവൻ കോഡ് പ്രവർത്തിപ്പിക്കുന്ന ആസ്യൂർ സർവർ ലെസ് കംപ്യൂട്ട് സേവനം.

## G

**GitHub Actions**  
GitHub റിപോസിറ്ററികളുമായി സംയോജിപ്പിച്ച CI/CD പ്ലാറ്റ്ഫോം, വർക്ക് ഫ്‌ളോകൾ ഓട്ടോമേറ്റുചെയ്യാൻ.

**Git**  
സോഴ്‌സ് കോഡിന്റെ മാറ്റങ്ങൾ ട്രാക്കുചെയ്യാൻ ഉപയോഗിക്കുന്ന വിതരണതന്ത്ര നിയന്ത്രണ സംവിധാനം.

## H

**Hooks**  
പ്രിവിഷൻ, പോസ്റ്റ് പ്രിവിഷൻ, പ്രിഡിപ്ലോയ്, പോസ്റ്റ് ഡിപ്ലോയ് തുടങ്ങിയ ഡിപ്ലോയ്മെന്റ് ലൈഫ്‌സൈക്കിൾ中特െ കോൺട്ൺ സ്‌ക്രിപ്റ്റുകളോ കമാൻഡുകളോ.

**Host**  
ആപ്ലിക്കേഷൻ ഒരുപിക്കുന്ന ആസ്യൂർ സേവന തരം (ഉദാ: appservice, containerapp, function).

## I

**Infrastructure as Code (IaC)**  
മാനുവൽ പ്രക്രിയകളുടെ പകരം കോഡ് വഴി ഇൻഫ്രാസ്ട്രക്ചർ നിർവചിക്കുകയും നിയന്ത്രിക്കുകയും ചെയ്യുന്നത്.

**Init**  
പുതിയ azd പ്രോജക്റ്റ് template-ൽ നിന്ന് ഇൻഷ്യലൈസ് ചെയ്യാനുള്ള പ്രക്രിയ.

## J

**JSON**  
JavaScript Object Notation. കോൺഫിഗറേഷൻ ഫയലുകൾക്കും API പ്രതികരണങ്ങൾക്കും സാധാരണ ഉപയോഗിക്കുന്ന ഡാറ്റാ ഇന്റർചേഞ്ച് ഫോർമാറ്റ്.

**JWT**  
JSON Web Token. JSON объക്തമായ സുരക്ഷിതമായി വിവരങ്ങൾ കൈമാറുന്നതിന് സ്റ്റാൻഡേർഡ്.

## K

**Key Vault**  
രഹസ്യങ്ങൾ, കീകൾ, സർട്ടിഫിക്കറ്റുകൾ സുരക്ഷിതമായി സംഭരിക്കുകയും നിയന്ത്രിക്കുകയും ചെയ്യുന്ന ആസ്യൂർ സേവനം.

**Kusto Query Language (KQL)**  
ആസ്യൂർ മോണിറ്റർ, അപ്ലിക്കേഷൻ ഇൻസൈറ്റ്സ്, മറ്റ് ആസ്യൂർ സേവനങ്ങളിൽ ഡാറ്റാ വിശകലനത്തിനായി ഉപയോഗിക്കുന്ന ക്വറി ഭാഷ.

## L

**Load Balancer**  
ഒരു നെറ്റ്‌വർക്ക് ട്രാഫിക് ബഹിര്ഭാഗവും നിരവധി സെർവറുകൾക്കോ ഇൻസ്റ്റൻസുകൾക്കോ വിതരണം ചെയ്യുന്നു.

**Log Analytics**  
ക്ലൗഡ്, ഓൺ-പ്രമൈസസ് പരിസ്ഥിതികളിൽ നിന്നും ടെലിമെട്രി ഡാറ്റ ശേഖരിക്കുകയും വിശകലനം ചെയ്യുകയും ചെയ്യുന്ന ആസ്യൂർ സേവനം.

## M

**Managed Identity**  
മറ്റ് ആസ്യൂർ സേവനങ്ങളിൽ സ്ഥിരീകരണത്തിന് സ്വയം മാനേജ്ഡ് ഐഡന്റിറ്റി നൽകുന്ന ആസ്യൂർ ഫീച്ചർ.

**Microservices**  
ഒറ്റപ്പെട്ട, ചെറിയ സ്വതന്ത്ര സേവനങ്ങളുടെ സമാഹാരമായി ആപ്ലിക്കേഷനുകൾ നിർമ്മിക്കുന്ന ആർക്കിടെക്ചറൽ സമീപനം.

**Monitor**  
ആസ്യൂർയുടെ ഐക്യമായ നിരീക്ഷണ പരിഹാരമായ, ആപ്ലിക്കേഷനും ഇൻഫ്രാസ്ട്രക്ചർ യും മുഴുവൻ കാണികൾ നൽകുന്നു.

## N

**Node.js**  
Chromeയിലെ V8 ജാവാസ്ക്രിപ്റ്റ് എഞ്ചിനിൽ നിർമ്മിച്ച സർവർ-സൈഡ് ആപ്ലിക്കേഷനുകൾക്കുള്ള ജാവാസ്ക്രിപ്റ്റ് റൺടൈം.

**npm**  
Node.js ക്കായി ഡിപ്പെൻഡൻസികളും പാക്കേജുകളും മാനേജുചെയ്യുന്ന പാക്കേജ് മാനേജർ.

## O

**Output**  
ഇൻഫ്രാസ്ട്രക്ചർ ഡിപ്ലോയ്മെൻറിൽ നിന്ന് ലഭിക്കുന്ന മൂല്യങ്ങൾ, ആപ്ലിക്കേഷനുകളിൽ അല്ലെങ്കിൽ മറ്റ് റിസോഴ്‌സുകളിൽ ഉപയോഗിക്കാം.

## P

**Package**  
ആപ്ലിക്കേഷൻ കോഡും ഡിപ്പെൻഡൻസികളും deployment-നായി തയ്യാറാക്കുന്ന പ്രക്രിയ.

**Parameters**  
Infrastructure ടെംപ്ലേറ്റുകളിൽ deployment കസ്റ്റമൈസ് ചെയ്യാൻ നൽകുന്ന ഇൻപുട്ട് മൂല്യങ്ങൾ.

**PostgreSQL**  
ആസ്യൂറിൽ മാനേജ്ഡ് സേവനമായി ലഭ്യമാകുന്ന ഓപ്പൺ സോഴ്‌സ് റിലേഷണൽ ഡാറ്റാബേസ് സിസ്റ്റം.

**Provisioning**  
Infrastructure ടെംപ്ലേറ്റുകളിൽ നിർവചിച്ച Azure റിസോഴ്‌സുകൾ സൃഷ്ടിക്കുകയും ക്രമീകരിക്കുകയും ചെയ്യുന്നത്.

## Q

**Quota**  
ആസ്യൂർ സബ്സ്ക്രിപ്ഷനിൽ അല്ലെങ്കിൽ പ്രദേശത്ത് സൃഷ്ടിക്കാവുന്ന റിസോഴ്‌സ് പരിമിതികൾ.

## R

**Resource Group**  
ഇടവേള, അനുമതികൾ, നയങ്ങൾ ഒന്നായി പങ്കിടുന്ന ആസ്യൂർ റിസോഴ്‌സുകൾക്ക് ലൊജിക്കൽ കണ്ടെയ്‌നർ.

**Resource Token**  
azd ഡിപ്ലോയ്മെന്റുകളിൽ റിസോഴ്‌സ് പേരുകൾ വ്യത്യസ്തമായിരിക്കാൻ സൃഷ്ടിക്കുന്ന വ്യത്യസ്തമായ സ്ട്രിംഗ്.

**REST API**  
HTTP രീതികൾ ഉപയോഗിച്ച് നെറ്റ്‌വർക് ആപ്ലിക്കേഷനുകൾ ഡിസൈൻ ചെയ്യുന്നതിനുള്ള ആർക്കിടെക്ചറൽ ശൈലി.

**Rollback**  
ആപ്ലിക്കേഷൻ അല്ലെങ്കിൽ ഇൻഫ്രാസ്ട്രക്ചർ കോൺഫിഗറേഷൻ മുൻകാല പതിപ്പിലേക്ക് തിരിച്ചുള്ള പ്രക്രിയ.

## S

**Service**  
azure.yaml ൽ നിർവചിച്ച നിങ്ങളുടെ ആപ്ലിക്കേഷന്റെ ഘടകം (ഉദാ: വെബ് ഫ്രണ്ട്എൻഡ്, API ബാക്ക്എൻഡ്, ഡാറ്റാബേസ്).

**SKU**  
Stock Keeping Unit. ആസ്യൂർ റിസോഴ്‌സുകൾക്കുള്ള വ്യത്യസ്ത സേവന നിരകളും പ്രവർത്തന നിലയും.

**SQL Database**  
Microsoft SQL Server ആസ്പദമാക്കി നടത്തപ്പെടുന്ന ആസ്യൂർ മാനേജ്ഡ് റിലേഷണൽ ഡാറ്റാബേസ് സേവനം.

**Static Web Apps**  
സോഴ്‌സ് കോഡ് റിപോസിറ്ററികളിൽ നിന്ന് പൂർണ്ണ സ്ടാക്ക് വെബ് ആപ്ലിക്കേഷനുകൾ നിർമ്മിച്ച് ഡിപ്ലോയ് ചെയ്യാനുള്ള ആസ്യൂർ സേവനം.

**Storage Account**  
ബ്ലോബുകൾ, ഫയലുകൾ, ക്യൂകൾ, ടേബിളുകൾ ഉൾപ്പെടെയുള്ള ഡാറ്റാ ഒബ്ജക്റ്റുകൾക്കായുള്ള ആസ്യൂർ ക്ലൗഡ് സ്റ്റോറെജ്സ് സേവനം.

**Subscription**  
അസ്സോസിയേറ്റഡ് ബില്ലിങ്, ആക്‌സസ് മാനേജുമെന്റ് എന്നിവയുള്ള റിസോഴ്‌സ് ഗ്രൂപ്പുകളും റിസോഴ്‌സുകളും ഉൾപ്പെടുന്ന ആസ്യൂർ അക്കൗണ്ട് കണ്ടെയ്‌നർ.

## T

**Template**  
ആപ്ലിക്കേഷൻ കോഡ്, ഇൻഫ്രാസ്ട്രക്ചർ നിർവചനങ്ങൾ, സാധാരണ വിവരം സജ്ജീകരണങ്ങൾ അടങ്ങിയ മുൻനിർമിച്ച പ്രോജക്റ്റ് ഘടന.

**Terraform**  
Azure ഉൾപ്പെടെയുള്ള വിവിധ ക്ലൗഡ് പ്രൊവൈഡറുകൾ പിന്തുണയ്ക്കുന്ന ഓപ്പൺ സോഴ്‌സ് Infrastructure as Code ടൂൾ.

**Traffic Manager**  
ആഗോള Azure പ്രദേശങ്ങളിലായി ട്രാഫിക് വിതരണം ചെയ്യാൻ DNS അടിസ്ഥാനമാക്കിയ ലോഡ് ബാലൻസർ.

## U

**URI**  
Uniform Resource Identifier. പ്രത്യേക റിസോഴ്‌സിനെ തിരിച്ചറിയുന്ന ഒരു സ്ട്രിംഗ്.

**URL**  
Uniform Resource Locator. റിസോഴ്‌സ് എവിടെയാണെന്ന്, എങ്ങനെ കിട്ടുമെന്ന് നിർ‍വചിക്കുന്ന URI തരം.

## V

**Virtual Network (VNet)**  
ആസ്യൂറിലെ സ്വകാര്യ നെറ്റ്‌വർക്ക് സ്ഥാപിക്കുന്ന അടിസ്ഥാന ഘടകം, ഐസൊലേഷൻ പരിതിവം നൽകുന്നു.

**VS Code**  
Visual Studio Code. ആസ്യൂർ, azd സംയോജനം മികച്ചുള്ള ജനപ്രിയ കോഡ് എഡിറ്റർ.

## W

**Webhook**  
നിർദ്ദിഷ്ട ഇവന്റുകൾ വഴി താൽപ്പര്യമുള്ള HTTP കോള്ബാക്ക്, സാധാരണമാക CI/CD പൈപ്പ്ലൈനുകളിൽ ഉപയോഗിച്ചു വരുന്നു.

**What-if**  
deployment യാഥാർത്ഥ്യമായി നടത്താതെ എന്ത് മാറ്റങ്ങൾ വരുമെന്ന് കാണിക്കുന്ന ആസ്യൂർ ഫീച്ചർ.

## Y

**YAML**  
YAML Ain't Markup Language. azure.yaml പോലുള്ള കോൺഫിഗറേഷൻ ഫയലുകൾക്ക് ഉപയോഗിക്കുന്ന മനുഷ്യനക്കാരായ ഡാറ്റാ സീരിയലൈസേഷൻ സ്റ്റാൻഡേർഡ്.

## Z

**Zone**  
ആസ്യൂർ റീജ്യന്റെടങ്ങിലുള്ള അത്ര തന്നെ ഷാരിയായ സ്ഥലങ്ങൾ, റെണ്ടൻസിയും ഉയർന്ന ലഭ്യതയും നൽകുന്നു.

## സാന്ദർഭിക പദങ്ങൾ

### വികസന പദങ്ങൾ  
- **Hot Reload**: പതുവായി അപ്ലിക്കേഷനുകൾ സ്വയം അപ്ഡേറ്റ് ചെയ്യുക, റീസ്റ്റാർട്ട് ആവാതെ  
- **Build Pipeline**: കോഡ് നിർമ്മാണവും പരിശോധനയും ഓട്ടോമേറ്റഡ് പ്രക്രിയ  
- **Deployment Slot**: App Service-ലെ സ്റ്റേജിംഗ് പരിസ്ഥിതി  
- **Environment Parity**: വികസനം, സ്റ്റേജിംഗ്, ഉല്പാദനം പരിസ്ഥിതികൾ സമാനമായി നിലനിർത്തൽ  

### സുരക്ഷാ പദങ്ങൾ  
- **Managed Identity**: ഓട്ടോമാറ്റിക് ക്രെഡൻഷ്യൽ മാനേജ്മെന്റ് നൽകുന്ന ആസ്യൂർ ഫീച്ചർ  
- **Key Vault**: രഹസ്യങ്ങൾ, കീകൾ, സർട്ടിഫിക്കറ്റുകൾ സੁਰക്ഷിതമായി സംഭരിക്കൽ  
- **RBAC**: ആസ്യൂർ റിസോഴ്‌സുകളുടെ റോളുകളടിസ്ഥാനമായ ആക്‌സസ് നിയന്ത്രണം  
- **Network Security Group**: നെറ്റ്‌വർക്ക് ട്രാഫിക് നിയന്ത്രിക്കുന്ന വെർച്വൽ ഫയർവാൾ  

### മോണിറ്ററിംഗ് പദങ്ങൾ  
- **Telemetry**: അളവുകൾ, ഡാറ്റ ശ്രേണികൾ സ്വയം ശേഖരിക്കൽ  
- **Application Performance Monitoring (APM)**: സോഫ്റ്റ്‌വെയർ പ്രകടനം നിരീക്ഷണം  
- **Log Analytics**: ലോഗ് ഡാറ്റ ശേഖരിക്കുകയും വിശകലനം ചെയ്യുകയും ചെയുന്ന സേവനം  
- **Alert Rules**: മെട്രിക് അല്ലെങ്കിൽ നിലകളും അടിസ്ഥാനമാക്കി ഓട്ടോമേറ്റഡ് നോട്ടിഫിക്കേഷനുകൾ  

### ഡിപ്ലോയ്മെന്റ് പദങ്ങൾ  
- **Blue-Green Deployment**: ഡൗൺടൈം ഇല്ലാത്ത ഡിപ്ലോയ്മെന്റ് തന്ത്രം  
- **Canary Deployment**: ഉപഭോക്താക്കളുടെ ഒരു ചെറിയ വിഭാഗത്തിൽ തികഞ്ഞ രീതിയിൽ വിന്യസം  
- **Rolling Update**: ആപ്ലിക്കേഷൻ ഇൻസ്റ്റൻസുകൾ അനുക്രമം മാറ്റുന്നതും നവീകരണവും  
- **Rollback**: മുൻകാല ആപ്ലിക്കേഷൻ പതിപ്പിലേക്ക് തിരിച്ചുള്ള നടപടികൾ  

---

**ഉപയോഗ ടിപ്പ്**: ഈ പദവിഭവത്തിൽ പ്രത്യേക പദങ്ങൾ തെളിച്ചറിയാൻ `Ctrl+F` ഉപയോഗിക്കുക. പദങ്ങൾ ആവശ്യാനുസരിച്ചു പരസ്പരം ലിങ്ക് ചെയ്തിട്ടുണ്ട്.
