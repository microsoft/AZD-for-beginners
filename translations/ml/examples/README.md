## പരിചയം

ഈ ഡയറക്ടറി പ്രായോഗിക ഉദാഹരണങ്ങളും റഫറൻസുകളും നൽകുന്നു, അതിലൂടെ നിങ്ങൾ Azure Developer CLI കൈകാര്യം പ്രായോഗികമായി അഭ്യസിക്കാനാകും. Retail Multi-Agent ദൃശ്യം ഈ റിപ്പോസിറ്ററിയിലുള്ള പൂര്‍ണ, ഉത്പാദന-തയാറാക്കിയ നടപ്പിലാക്കലാണ്. പരമാവധി ഉദാഹരണങ്ങൾ ഔദ്യോഗിക Azure Samples റിപോസിറ്ററികൾക്ക് റഫറൻസാണ്, വിവിധ AZD മാതൃകകൾ കാണിക്കുക.

### സങ്കീർണ്ണത റേറ്റിങ് ലെജൻഡ്

- ⭐ **ആരంభകർ** - അടിസ്ഥാന ആശയങ്ങൾ, ഏകദേശം ഒരു സർവീസ്, 15-30 മിനിട്ട്  
- ⭐⭐ **ഇന്റർമീഡിയറ്റ്** - പല സേവനങ്ങൾ, ഡാറ്റാബേസ് ഇൻറഗ്രേഷൻ, 30-60 മിനിട്ട്  
- ⭐⭐⭐ **അഡ്വാൻസ്ഡ്** - സങ്കീർണ്ണ ആർക്കിടെക്‌ചർ, AI സംയോജനം, 1-2 മണിക്കൂർ  
- ⭐⭐⭐⭐ **എക്‌സ്പർട്ട്** - ഉത്പാദന തയാറായ, എന്റർപ്രൈസ് മാതൃകകൾ, 2+ മണിക്കൂർ  

## 🎯 ഈ റിപ്പോസിറ്ററിയിൽ യാഥാർഥ്യം എന്താണ്?

### ✅ ലോക്കൽ നടപ്പിലാക്കൽ (ഉപയോഗത്തിന് തയ്യാറായവ)

#### [Azure OpenAI Chat ആപ്ലിക്കേഷൻ](azure-openai-chat/README.md) 🆕  
**ഈ റിപ്പോസിറ്ററിയിൽ പൂര്‍ണമായ GPT-4 വിന്യസവും ചാറ്റ് ഇന്റർഫേസും ഉൾക്കൊള്ളുന്നു**  

- **സ്ഥലം:** `examples/azure-openai-chat/`  
- **സങ്കീർണ്ണത:** ⭐⭐ (ഇന്റർമീഡിയറ്റ്)  
- **ഉൾക്കൊള്ളുന്നവ:**  
  - പൂര്‍ണ Azure OpenAI വിന്യാസം (GPT-4)  
  - പൈഥൺ കമാൻഡ്-ലൈൻ ചാറ്റ് ഇന്റർഫേസ്  
  - സുരക്ഷിത API കീകൾക്കായി കി വോൾട്ട് സംയോജനം  
  - ബൈസിപ് ഇൻഫ്രാസ്ട്രക്ചർ ടെംപ്ലേറ്റുകൾ  
  - ടോക്കൺ ഉപയോഗവും ചെലവും ട്രാക്കിംഗ്  
  - റേറ്റ് ലിമിറ്റിംഗ്, പിശകുകൾ കൈകാര്യം ചെയ്യൽ  

**വേഗത്തിലുള്ള തുടക്കം:**  
```bash
# ഉദാഹരണത്തിലേക്ക്_NR
cd examples/azure-openai-chat

# എല്ലാം വിന്യസിക്കുക
azd up

# ആശ്രിതങ്ങൾ ഇൻസ്റ്റാൾ ചെയ്‌തിരിച്ച് ചാറ്റ് ആരംഭിക്കുക
pip install -r src/requirements.txt
python src/chat.py
```
  
**സാങ്കേതിക വിദ്യകൾ:** Azure OpenAI, GPT-4, കി വോൾട്ട്, പൈഥൺ, ബൈസിപ്

- **സങ്കീർണ്ണത:** ⭐-⭐⭐⭐⭐ (ആരഭക മുതൽ അഡ്വാൻസ്ഡ് വരെ)

**[പൂർണ നടപ്പിലാക്കൽ ഗൈഡ്](./retail-scenario.md)**  

AZD ഉപയോഗിച്ച് ഏന്റർപ്രൈസ്-ഗ്രേഡ് AI ആപ്ലിക്കേഷൻ വിന്യാസം സാദ്ധ്യമാക്കുന്ന സമഗ്രവും ഉത്പാദന-പ്രവൃത്തി തയ്യാറായ മൾട്ടി-ഏജന്റ് ഉപഭോക്തൃ സമർപ്പണ പരിഹാരമാണ് ഇത്. ഈ ദൃശ്യം നൽകുന്നത്:

- **പൂർണ ആർക്കിടെക്ചർ**: പ്രത്യേക ഉത്പഭോക്തൃ സേവനവും ഇൻവെൻററി കൈവരവ് ഏജന്റുമാരുള്ള മൾട്ടി-ഏജന്റ് സിസ്റ്റം
- **ഉത്പാദന അടിസ്ഥാനസൗകര്യം**: ബഹുസ്ഥാന ആസ്യൂർ ഓപ്പൺഎഐ പ്രയോഗങ്ങൾ, എഐ സെർച്ച്, കണ്ടെയ്‌നർ ആപ്പുകൾ, സമഗ്രമായ നിരീക്ഷണം  
- **പ്രവർത്തനത്തിലാക്കാനൊരുങ്ങിയ ARM ടെംപ്ലേറ്റ്**: ബഹുമുഖ-ക്രമീകരണ രീതികളോടുള്ള ഒറ്റ ക്ലിക്ക് വിന്യസനം (കുറഞ്ഞത്/സ്റ്റാൻഡേർഡ്/പ്രീമിയം)  
- **അസ്‌തിത്വം മികവു**: റെഡ് ടീം സുരക്ഷാ പരിശോധന, ഏജന്റ് വിലയിരുത്തൽ ഘടന, ചെലവ് തൊളവ്, പ്രശ്‌ന പരിഹാര മാർഗ്ഗനിർദ്ദേശങ്ങൾ  
- **സാക്ഷാത് ബിസിനസ് സാഹചര്യമുൾപ്പെടെ**: റീടെയ്‌ലർ കസ്റ്റമർ സപ്പോർട്ട് ഉപയോഗകേസ്, ഫയൽ അപ്‌ലോഡുകൾ, സെർച്ച് ഇന്റഗ്രേഷൻ, സജീവ സ്കേലിംഗ്  

**ടെക്നോളജികൾ**: ആസ്യൂർ ഓപ്പൺഎഐ (GPT-4o, GPT-4o-mini), ആസ്യൂർ എഐ സെർച്ച്, കണ്ടെയ്‌നർ ആപ്പുകൾ, കോസ്മോസ് ഡിബി, അപ്ലിക്കേഷൻ ഇൻസൈറ്റ്സ്, ഡോക്യുമെന്റ് ഇന്റലിജൻസ്, ബിൻഗ് സെർച്ച് API  

**കമ്പ്ളക്സിറ്റി**: ⭐⭐⭐⭐ (മികച്ച - എന്റർപ്രൈസ് ഉത്പാദനത്തിനൊ‌രുങ്ങിയ)

ഏതെങ്കിലും ഉദാഹരണം പ്രവർത്തിപ്പിക്കുന്നതിന് മുമ്പ്:  
- ✅ സ്വാമി അല്ലെങ്കിൽ സംഭാഗിത്താവിന്റെ ആസ്യൂർ സബ്സ്ക്രിപ്ഷൻ  
- ✅ ആസ്യൂർ ഡവലപ്പർ CLI ഇൻസ്റ്റാൾ ചെയ്തിരിക്കുക ([ഇൻസ്റ്റലേഷൻ ഗൈഡ്](../docs/chapter-01-foundation/installation.md))  
- ✅ ഡോക്കർ ഡെസ്ക്‌ടോപ്പ് പ്രവർത്തന സ്ഥിതിയിൽ (കണ്ടെയ്‌നർ ഉദാഹരണങ്ങൾക്കായി)  
- ✅ അനുയോജ്യമായ ആസ്യൂർ ക്വോട്ടകൾ (ഉദാഹരണത്തിനുസൃത ആവശ്യങ്ങൾ പരിശോധിക്കുക)  

> **💰 ചെലവ് മുന്നറിയിപ്പ്:** എല്ലാ ഉദാഹരണങ്ങളും യഥാർത്ഥ ആസ്യൂർ വിഭവങ്ങൾ സൃഷ്ടിച്ച് ചാർജ്ജുകൾ നൽകും. ചെലവു കണക്കുകൾക്കായി ഓരോ README ഫയലും കാണുക. തുടര്‍ന്നുള്ള ചെലവുകൾ ഒഴിവാക്കാൻ പൂർത്തിയായി `azd down` ഓടിക്കാൻ മറക്കരുത്.  

### ഉദാഹരണങ്ങൾ ലൊക്കലായും പ്രയോജനപ്പെടുത്തൽ  

1. **ഉദാഹരണം ക്ലോൺ ചെയ്യുക അല്ലെങ്കിൽ പകർത്തുക**  
   ```bash
   # ആഗ്രഹിച്ച ഉദാഹരണത്തിലേക്ക് നാവിഗേറ്റ് ചെയ്യുക
   cd examples/simple-web-app
   ```
  
2. **AZD പരിസ്ഥിതി ആരംഭിക്കുക**  
   ```bash
   # നിലവിലുള്ള ടെംപ്ലേറ്റ് ഉപയോഗിച്ച് ആരംഭിക്കുക
   azd init
   
   # അല്ലെങ്കിൽ പുതിയ പരിസരം സൃഷ്ടിക്കുക
   azd env new my-environment
   ```
  
3. **പരിസ്ഥിതി ക്രമീകരിക്കുക**  
   ```bash
   # ആവശ്യമായ ചാരങ്ങൾ സജ്ജമാക്കുക
   azd env set AZURE_LOCATION eastus
   azd env set AZURE_SUBSCRIPTION_ID your-subscription-id
   ```
  
4. **വിന്യസിക്കുക**  
   ```bash
   # ഇൻ‌ഫ്രാസ്ട്രക്ചർ և അപ്ലിക്കേഷൻ വിന്യസിക്കുക
   azd up
   ```
  
5. **വിന്യാസം സ്ഥിരീകരിക്കുക**  
   ```bash
   # സേവന എണ്ട്പോയിന്റുകൾ നേടുക
   azd env get-values
   
   # എണ്ട്പോയിന്റ് പരിശോധന (ഉദാഹരണം)
   curl https://your-app-url.azurecontainer.io/health
   ```
   
   **പ്രതീക്ഷിക്കുന്ന വിജയ സൂചികകൾ:**  
   - ✅ `azd up` പിശകുകൾ കൂടാതെ പൂർത്തീകരിക്കുക  
   - ✅ സർവീസ് എൻഡ്‌പോയിന്റ് HTTP 200 തിരിച്ചറിയുക  
   - ✅ ആസ്യൂർ പോർട്ടൽ "Running" നില കാണിക്കുക  
   - ✅ അപ്ലിക്കേഷൻ ഇൻസൈറ്റ്സ് ടെലിമെട്രി സ്വീകരിക്കുന്നു  

> **⚠️ പ്രശ്‌നങ്ങളുണ്ടോ?** വിന്യാസ പ്രശ്‌നപരിഹാരത്തിനായി [സാധാരണ പ്രശ്‌നങ്ങൾ](../docs/chapter-07-troubleshooting/common-issues.md) കാണുക  

### ഉദാഹരണങ്ങളെ ഇച്ഛാനുസൃതമാക്കൽ  

ഓരോ ഉദാഹരണത്തിലും ഉൾക്കൊള്ളുന്നു:  
- **README.md** - വിശദമായ സജ്ജീകരണവും കസ്റ്റമൈസേഷനും  
- **azure.yaml** - കമന്റുകളുള്ള AZD ക്രമീകരണം  
- **infra/** - പാരാമീറ്റർ വിശദീകരണങ്ങളോടുള്ള ബൈസപ്പ് ടെംപ്ലേറ്റുകൾ  
- **src/** - സാമ്പിൾ അപ്ലിക്കേഷൻ കോഡ്  
- **scripts/** - പൊതുവായ പ്രവർത്തനങ്ങൾക്ക് സഹായക്കാരായ സ്ക്രിപ്റ്റുകൾ

## 🛠 ക്രമീകരണ ഉദാഹരണങ്ങൾ  

`configurations/` ഡയറക്ടറി പുനരുപയോഗിക്കാവുന്ന ഘടകങ്ങൾ ഉൾക്കൊള്ളുന്നു:  

### പരിസ്ഥിതി ക്രമീകരണങ്ങൾ  
- ഡെവലപ്പ്മെന്റ് പരിസ്ഥിതി ക്രമീകരണങ്ങൾ  
- സ്റ്റേജിംഗ് പരിസ്ഥിതി ക്രമീകരണങ്ങൾ  
- ഉത്പാദനത്തിനൊരുങ്ങിയ ക്രമീകരണങ്ങൾ  
- ബഹുസ്ഥാന വിന്യാസ ക്രമീകരണങ്ങൾ  

### ബൈസപ്പ് മോഡ്യൂളുകൾ  
- പുനരുപയോഗിക്കാവുന്ന അടിസ്ഥാനസൗകര്യ ഘടകങ്ങൾ  
- പൊതുവായ വിഭവ മാതൃകകൾ  
- സുരക്ഷ ഉറപ്പാക്കുന്ന ടെംപ്ലേറ്റുകൾ  
- ചെലവ് മെച്ചപ്പെടുത്തുന്ന ക്രമീകരണങ്ങൾ  

### സഹായി സ്ക്രിപ്റ്റുകൾ  
- പരിസ്ഥിതി സജ്ജീകരണ ഓട്ടോമേഷൻ  
- ഡാറ്റാബേസ് മാറ്റുചെയ്യൽ സ്ക്രിപ്റ്റുകൾ  
- വിന്യാസ സ്ഥിരീകരണ ഉപകരണങ്ങൾ  
- ചെലവ് നിരീക്ഷണ സഹായികൾ  

## 🔧 കസ്റ്റമൈസേഷൻ ഗൈഡ്  

### നിങ്ങളുടെ ഉപയോഗത്തിനായി ഉദാഹരണങ്ങൾ ഇച്ഛാനുസൃതമാക്കൽ  

1. **അവശ്യക്കുറിപ്പുകൾ അവലോകനം ചെയ്യുക**  
   - ആസ്യൂർ സേവന ആവശ്യങ്ങൾ പരിശോധിക്കുക

   - ചെലവ് ഫലപ്രദത മനസിലാക്കുക  

2. **ക്രമീകരണം പരിഷ്‌കരിക്കുക**  
   - `azure.yaml` സർവീസ് നിർവചനങ്ങൾ പുതുക്കുക  
   - ബൈസപ്പ് ടെംപ്ലേറ്റുകൾ അനുകൂലമാക്കുക  
   - പരിസ്ഥിതി ചാരങ്ങൾ ക്രമീകരിക്കുക  

3. **പരീക്ഷണം വ്യാപകമായി നടത്തുക**  
   - ആദ്യം ഡെവലപ്പ്മെന്റ് പരിസ്ഥിതിയിലേക്ക് വിന്യാസിക്കുക  
   - പ്രവർത്തനക്ഷമത സ്ഥിരീകരിക്കുക  
   - സ്കെയ്ലിംഗ്, പ്രകടനം പരിശ്രമിക്കുക  

4. **സുരക്ഷാ അവലോകനം**

   - വ്യത്യസ്ത ഹോസ്റ്റിംഗ് ഓപ്ഷനുകൾ പഠിക്കുക  
   - CDN ഇന്റഗ്രേഷൻ ഔണ്ടറൂരു   
   - DNS ക്രമീകരണം മനസ്സിലാക്കുക  

3. **കണ്ടെയ്‌നർ ആപ്പിലേക്ക് മടക്കി വരുക**  
   - കണ്ടെയ്‌നറൈസേഷൻ അടിസ്ഥാനങ്ങൾ പഠിക്കുക  
   - സ്കേലിംഗ് ആശയങ്ങൾ മനസിലാക്കുക  
   - ഡോക്കറുമായി അഭ്യാസം നടത്തുക  

4. **ഡാറ്റാബേസ് ഇന്റഗ്രേഷൻ ചേർക്കുക**  
   - ഡാറ്റാബേസ് പ്രൊവിഷനിംഗ് പഠിക്കുക  
   - കണക്ഷൻ സ്ട്രിംഗുകൾ മനസ്സിലാക്കുക  
   - രഹസ്യ മാനേജ്‌മെന്റ് അഭ്യസിക്കുക  

5. **സെർവർലെസ് അന്വേഷിക്കുക**  
   - ഇവന്റ്-ഓറിയന്റഡ് വാസ്തുവിദ്യ മനസ്സിലാക്കുക  
   - ട്രിഗർസ്, ബൈൻഡിങ്ങുകൾ പഠിക്കുക  
   - API-കളുമായി പരിശീലനം നടത്തുക  

6. **മൈക്രോസെർവിസസ് നിർമ്മിക്കുക**  
   - സർവീസ് ആശയവിനിമയം പഠിക്കുക  
   - വിതരിത സിസ്റ്റങ്ങൾ മനസ്സിലാക്കുക
