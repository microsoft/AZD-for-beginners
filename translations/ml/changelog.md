# മാറ്റങ്ങൾ - AZD For Beginners

## ಪರಿಚയം

ഇ മൂല്യനിരീക്ഷണ രേഖ AZD For Beginners റെപ്പോസിറ്ററിയിലുള്ള എല്ലാ പ്രധാന മാറ്റങ്ങളും അപ്‌ഡേറ്റുകളും ഇളവുകളും രേഖപ്പെടുത്തുന്നു. നാം സിമാന്റിക് വേർഷനിംഗ് സിദ്ധാന്തങ്ങൾ പിന്തുടരുന്നു, വേർഷനുകൾക്കിടയിൽ എന്താണ് മാറിയിട്ടുള്ളത് ഉപയോക്താക്കൾക്ക് മനസ്സിലാകാൻ ഈ ലോഗ് നിലനിർത്തുന്നു.

## പഠന ലക്ഷ്യങ്ങൾ

ഈ മാറ്റങ്ങൾ പരിശോധിക്കുന്നതിലൂടെ, നിങ്ങൾ:
- പുതിയ ഫീച്ചറുകളും ഉള്ളടക്ക ചേർക്കലുകളും അറിയുന്നു
- നിലവിലുള്ള ഡോക്യുമെന്റേഷനിൽ നടത്തിയ മെച്ചപ്പെടുത്തലുകൾ മനസ്സിലാക്കുന്നു
- ശരിതീരുമാനം ഉറപ്പാക്കാൻ ബഗ് ഫിക്സുകളും പിഴവുകൾ പഠിക്കുന്നു
- പഠന സാമഗ്രികളുടെ പുരോഗതി പിന്തുടരുന്നു

## പഠന ഫലങ്ങൾ

മാറ്റങ്ങളുടെയും രേഖകളുടെ പരിശോധിച്ച ശേഷം, നിങ്ങൾക്ക് കഴിയും:
- പുതിയ ഉള്ളടക്കവും പഠന വിഭവങ്ങളും തിരിച്ചറിയുക
- എവിടെ പുതുക്കലുകൾ മാറ്റങ്ങൾ വന്നുവെന്ന് മനസ്സിലാക്കുക
- ഏറ്റവും പുതിയത് ഉള്ള പഠന സാമഗ്രികളുടെ അടിസ്ഥാനത്തിൽ നിങ്ങളുടെ പഠന മാർഗ്‌ഗം ആസൂത്രണം ചെയ്യുക
- ഭാവിയിലെ മെച്ചപ്പെടുത്തലുകൾക്കായി അഭിപ്രായം നൽകുകയും നിർദേശങ്ങൾ സമർപ്പിക്കുകയും ചെയ്യുക

**ഈ പതിപ്പ് എല്ലാ AI ബന്ധപ്പെട്ട ചാപ്റ്ററുകളിലും `azd ai`, `azd extension`, `azd mcp` കമാൻഡ് സമ്പർക്കം ചേർക്കുന്നു, agents.md-യിലെ തകർന്ന ലിങ്കുകളും കാലഹരണപ്പെട്ട കോഡും തിരുത്തുന്നു, ചീറ്റ്ഷീറ്റ് അപ്ഡേറ്റ് ചെയ്യുന്നു, ഉദാഹരണ ടെംപ്ലേറ്റുകളുടെ വിഭാഗം പരിശോധിച്ച വിവരണങ്ങളും പുതിയ Azure AI AZD ടെംപ്ലേറ്റുകളും ഉൾപ്പെടുത്തി പുനഃസംഘടിപ്പിക്കുന്നു.**

  - `docs/chapter-02-ai-development/microsoft-foundry-integration.md` — "AZD Extensions for Foundry" & "Agent-First Deployment" ഉപവിഭാഗങ്ങൾ

  - `docs/chapter-06-pre-deployment/coordination-patterns.md` — ഡിപ്ലോയ്മെന്റ് സെക്ഷനിൽ `azd ai agent init` ഓപ്‌ഷൻ ഉൾപ്പെടുത്തി
  - `docs/chapter-07-troubleshooting/ai-troubleshooting.md` — "AZD AI എക്സ്റ്റെൻഷൻ കമാൻഡുകൾ ഡയഗ്നോസ്റ്റിക് വേണ്ടി" ഉപവിഭാഗം
  - `resources/cheat-sheet.md` — പുതിയ "AI & എക്സ്റ്റെൻഷൻ കമാൻഡുകൾ" വിഭാഗം, `azd extension`, `azd ai agent init`, `azd mcp`, `azd infra generate` ഉൾപ്പെടുത്തി
- **📦 പുതിയ AZD AI ഉദാഹരണ ടെംപ്ലേറ്റുകൾ** `microsoft-foundry-integration.md` ൽ:
  - **azure-search-openai-demo-csharp** — Blazor WebAssembly, Semantic Kernel, വോയ്സ് ചാറ്റ് പിന്തുണയുള്ള .NET RAG ചാറ്റ്
  - **azure-search-openai-demo-java** — Langchain4J ഉപയോഗിച്ച് ജാവയിലുള്ള RAG ചാറ്റ്, ACA/AKS ഡിപ്ലോയ്മെന്റ് ഓപ്ഷനുകൾപ്പോലുള്ളത്
  - **contoso-creative-writer** — Azure AI Agent Service, Bing Grounding, Prompty ഉപയോഗിക്കുന്ന മൾട്ടി-ഏജന്റ് ക്രിയേറ്റീവ് റൈറ്റിംഗ് അപ്ലിക്കേഷൻ
  - **serverless-chat-langchainjs** — Azure Functions + LangChain.js + Cosmos DB ഉപയോഗിച്ചുള്ള സർവർലെസ്സ് RAG, ഒല്ലാമ ലൊക്കൽ ഡെവ് പിന്തുണയോടെ
  - **chat-with-your-data-solution-accelerator** — അഡ്മിൻ പോർട്ടൽ, ടീംസ് ഇന്റഗ്രേഷൻ, PostgreSQL/Cosmos DB ഓപ്ഷനുകളുള്ള എന്റർപ്രൈസ് RAG ആക്സലറേറ്റർ
  - **azure-ai-travel-agents** — .NET, പൈതൺ, ജാവ, ടൈപ്പ്‌സ്‌ക്രിപ്റ്റ് സർവറുകൾ ഉപയോഗിക്കുന്ന മൾട്ടി-ഏജന്റ് MCP ഓർക്ക്സ്ട്രേഷൻ റഫറൻസ് അപ്ലിക്കേഷൻ
  - **azd-ai-starter** — മിനിമൽ Azure AI ഇൻഫ്രാസ്ട്രക്ചർ ബൈസപ്പ് സ്റ്റാർട്ടർ ടെംപ്ലേറ്റ്
  - **🔗 ആധികാരിക AZD AI ഗാലറി ലിങ്ക്** — [awesome-azd AI gallery](https://azure.github.io/awesome-azd/?tags=ai) (80+ ടെംപ്ലേറ്റുകൾ)റഫറൻസ്

### [v3.10.0] - 2026-02-05

#### Azure Developer CLI കമാൻഡ് കൃത്യത അപ്‌ഡേറ്റ്  
**ഡോക്യുമെന്റേഷനിലെ നിലനിൽക്കാത്ത AZD കമാൻഡുകൾ ശരിയാക്കി, എല്ലാ കോഡ് ഉദാഹരണങ്ങളിലും സാധുവായ Azure Developer CLI സിന്തക്സ് ഉറപ്പാക്കി.**

#### പരിഹരിച്ചു
- **🔧 നിലനിൽക്കാത്ത AZD കമാൻഡുകൾ നീക്കം ചെയ്തു**:  
  - `azd logs` വറുതെ → `azd monitor --logs` അല്ലെങ്കിൽ Azure CLI സാമീപ്യം  
  - `azd service` സബ്‌കമാൻഡുകൾ ഇല്ല → `azd show` , Azure CLI മാറ്റം  
  - `azd infra import/export/validate` ഇല്ല → നീക്കം ചെയ്തു/സാധുവായവയാക്കി  
  - `azd deploy --rollback/--incremental/--parallel/--detect-changes` എന്ന ഫ്ലാഗുകൾ ഇല്ല → നീക്കം  
  - `azd provision --what-if/--rollback` ഫ്ലാഗുകൾ ഇല്ല → `--preview` ആയി മാറ്റം  
  - `azd config validate` ഇല്ല → `azd config list` ആയി മാറ്റം  
  - `azd info`, `azd history`, `azd metrics` ഇല്ല → നീക്കം

- **കോൺഫിഗറേഷൻ ഉദാഹരണങ്ങൾ**: പുതുക്കിയ നാമകരണ പദങ്ങളോടെ ആധുനിക ഏജന്റ് കോൺഫിഗറേഷൻ മാതൃകകൾ
- **ഡോക്യുമെന്റേഷൻ സാന്ദ്രത**: പ്രൊഫഷണൽ, വിവരണാത്മക ഏജന്റ് നാമങ്ങൾ എല്ലാ റഫറൻസുകളിലും ഉറപ്പാക്കി

#### മെച്ചപ്പെടുത്തിയതും
- **ARM ടെംപ്ലേറ്റ് പാക്കേജ്**: retail-multiagent-arm-template അപ്ഡേറ്റ് ചെയ്തു Customer agent റഫറൻസുകളോടൊപ്പം

- **മെച്ചപ്പെട്ട IDE ഇന്റഗ്രേഷൻ**: AZD + AI വർക്ക്‌ഫ്ലോകൾക്കായുള്ള VS കോഡ് എക്സ്റ്റൻഷൻ  
- **കമ്മ്യൂണിറ്റി ഡാഷ്ബോർഡ്**: യഥാർത്ഥ സമയം കമ്മ്യൂണിറ്റി മെട്രിക്‌സ്, സംഭാവന ട്രാക്കിംഗ്  

## ചേഞ്ച്ലോഗിൽ സംഭാവന ചെയ്യൽ

### മാറ്റങ്ങൾ റിപ്പോർട്ട് ചെയ്യുക  
ഈ റീപോസിറ്ററിയിൽ സംഭാവന ചെയ്യുമ്പോൾ ചേഞ്ച്ലോഗ് എൻട്രികളിൽ താഴെ ഉള്ളത് ഉൾക്കൊള്ളണം:

1. **പതിപ്പ് നമ്പർ**: സിമാന്റിക് വേർഷനിങ്ങ് പാലിച്ച് (major.minor.patch)  
2. **തിയതി**: YYYY-MM-DD ഫോർമാറ്റിലുള്ള റിലീസ് അല്ലെങ്കിൽ അപ്ഡേറ്റ്  
3. **വിഭാഗം**: Added, Changed, Deprecated, Removed, Fixed, Security  
4. **വിവരണം**: എന്താണ് മാറിയതെന്ന് ദൃഢമായ വിശദീകരണം  
5. **പ്രഭാവം**: നിലവിലെ ഉപഭോക്താക്കളെ ബാധിക്കുന്ന വിധം  

### മാറ്റം വിഭാഗങ്ങൾ  

#### Added  
- പുതിയ ഫീച്ചറുകൾ, ഡോക്യുമെന്റേഷൻ വിഭാഗങ്ങൾ, കഴിവുകൾ  
- പുതിയ ഉദാഹരണങ്ങൾ, ടെംപ്ലേറ്റുകൾ, പഠന സ്രോതസ്സുകൾ  
- അധിക ഉപകരണങ്ങൾ, സ്‌ക്രിപ്റ്റുകൾ, യൂട്ടിലിറ്റികൾ  

#### Changed  
- നിലവിലുള്ള പ്രവർത്തനം അല്ലെങ്കിൽ ഡോക്യുമെന്റേഷൻ മാറ്റങ്ങൾ  
- കൃത്യതയും വ്യക്തതയും മെച്ചപ്പെടുത്തൽ  
- ഉള്ളടക്കവുമായി ബന്ധപ്പെട്ട പുനസംഘടിതമാകല്‍  

#### Deprecated  
- പിന്‍വലിക്കാനിടയായ ഫീച്ചറുകൾ അല്ലെങ്കിൽ സമീപനങ്ങൾ  
- ഇല്ലാതാക്കാനുള്ള ഡോക്യുമെന്റേഷൻ വിഭാഗങ്ങൾ  
- മികച്ച ബദലുകൾ ഉള്ള രീതികൾ  

#### Removed  
- പ്രസക്തമല്ലാത്ത ഫീച്ചറുകൾ, ഡോക്യുമെന്റേഷൻ, ഉദാഹരണങ്ങൾ  
- പഴയ വിവരങ്ങൾ അല്ലെങ്കിൽ പിന്‍വലிக்கப்பட்ட സമീപനങ്ങൾ  
- ആവർത്തിച്ച അല്ലെങ്കിൽ ഏകീകൃത ഉള്ളടക്കം  

#### Fixed  
- ഡോക്യുമെന്റേഷൻ അല്ലെങ്കിൽ കോഡിലെ പിശകുകളുടെ തിരുത്തലുകൾ  
- റിപ്പോർട്ട് ചെയ്ത പ്രശ്നങ്ങൾക്കും പ്രശ്നപരിഹാരങ്ങൾക്കും  
- കൃത്യതയും പ്രവർത്തനക്ഷമതയും മെച്ചപ്പെടുത്തൽ  

#### Security  
- സുരക്ഷാ സംബന്ധിയായ മെച്ചപ്പെടുത്തലുകളും തിരുത്തലുകളും  
- മികച്ച സുരക്ഷാ നടപടികളും അനുബന്ധ അപ്ഡേറ്റുകളും  
- സുരക്ഷാ ദൗർബല്യങ്ങളുടെ പരിഹാരങ്ങൾ  

### സിമാന്റിക് വേർഷനിങ്ങ് മാർഗനിർദ്ദേശം  

#### പ്രധാന പതിപ്പ് (X.0.0)  
- ഉപയോക്തൃ നടപടികൾ ആവശ്യമായ പൊതു മാറ്റങ്ങൾ  
- ഉള്ളടക്കത്തിന്റെ വലിയ പുനസംഘടന  
- അടിസ്ഥാന സമീപനത്തിലോ മെഥഡോളജി മാറലുകൾ  

#### ചെറിയ പതിപ്പ് (X.Y.0)  
- പുതിയ ഫീച്ചറുകളും ഉള്ളടക്ക കൂട്ടിച്ചേർക്കലുകളും  
- പിന്നിലേക്കുള്ള പൊരുത്തം നിലനിർത്തുന്ന മെച്ചപ്പെടുത്തലുകൾ  
- അധിക ഉദാഹരണങ്ങൾ, ടൂൾസ്, സ്രോതസ്സുകൾ  

#### പാച്ച് പതിപ്പ് (X.Y.Z)  
- ബഗ് ഫിക്സുകളും തിരുത്തലുകളും  
- നിലവിലുള്ള ഉള്ളടക്കത്തിന്റെ ചെറിയ മെച്ചപ്പെടുത്തലുകൾ  
- വ്യക്തതയും കൂടുതൽ ചെറു മെച്ചപ്പെടുത്തലുകളും  

## കമ്മ്യൂണിറ്റി പ്രതികരണവും നിർദ്ദേശങ്ങളും  

നമ്മുടെ പഠന സ്രോതസ്സ് മെച്ചപ്പെടുത്താൻ കമ്മ്യൂണിറ്റി പ്രതികരണം പ്രോത്സാഹിപ്പിക്കുന്നു:  

### പ്രതികരണം നൽകുന്ന വിധം  
- **GitHub Issues**: പ്രശ്നങ്ങൾ റിപ്പോർട്ട് ചെയ്യുക അല്ലെങ്കിൽ മെച്ചപ്പെടുത്തലുകൾ ശുപാർശ ചെയ്യുക (AI-നിർദ്ദിഷ്ട പ്രശ്നങ്ങൾ എപ്പോഴും സ്വാഗതം)

- **പുള്‍ റിക്വസ്റ്റുകൾ**: ഉള്ളടക്കത്തിനുള്ള നേരിട്ടുള്ള മെച്ചപ്പെടുത്തലുകൾ, പ്രത്യേകിച്ച് AI ടെംപ്ലേറ്റുകളും ഗൈഡുകളും  
- **Microsoft Foundry Discord**: AZD + AI ചർച്ചയ്ക്ക് #Azure ചാനലിൽ പങ്കെടുക്കുക  
- **കമ്മ്യൂണിറ്റി ഫോറങ്ങൾ**: ആഴ്റി Azure ഡെവലപ്പർ ചർച്ചകളിൽ പങ്കാളിത്വം  

### പ്രതികരണ വിഭാഗങ്ങൾ  
- **AI ഉള്ളടക്ക കൃത്യത**: AI സേവന ഇന്റഗ്രേഷൻ, ഡിപ്ലോയ്മെന്റ് വിവരങ്ങൾക്ക് തിരുത്തലുകൾ  
- **പഠന അനുഭവം**: മെച്ചപ്പെട്ട AI ഡെവലപ്പർ പഠന പ്രവാഹത്തിന് നിർദ്ദേശങ്ങൾ  
- **കുറഞ്ഞ AI ഉള്ളടക്കം**: അധിക AI ടെംപ്ലേറ്റുകൾ, മാതൃകകൾ, ഉദാഹരണങ്ങൾ അഭ്യർത്ഥനകൾ  
- **ആക്‌സസിബിലിറ്റി**: വ്യത്യസ്ത പഠന ആവശ്യങ്ങൾക്കുള്ള മെച്ചപ്പെടുത്തലുകൾ  
- **AI ടൂൾ ഇന്റഗ്രേഷൻ**: മെച്ചപ്പെട്ട AI ഡെവലപ്പ്മെന്റ് പ്രവാഹത്തിനുള്ള നിർദ്ദേശങ്ങൾ  
- **പ്രൊഡക്ഷൻ AI മാതൃകകൾ**: എന്റർപ്രൈസ് AI ഡിപ്ലോയ്മെന്റ് മാതൃകകൾക്കുള്ള അഭ്യർത്ഥനകൾ  

### പ്രതികരണ പ്രതിജ്ഞ  
- **ഇഷ്യൂ പ്രതികരണം**: റിപ്പോർട്ട് ചെയ്ത പ്രശ്നങ്ങൾക്ക് 48 മണിക്കൂറിനുള്ളിൽ  
- **ഫീച്ചർ അഭ്യർത്ഥനകൾ**: ഒരു ആഴ്ചയ്ക്കുള്ളിൽ വിലയിരുത്തല്‍  
- **കമ്മ്യൂണിറ്റി സംഭാവനകൾ**: ഒരു ആഴ്ചക്കുള്ളിൽ അവലോകനം  
- **സുരക്ഷാ പ്രശ്നങ്ങൾ**: തടസ്സരഹിതമായി പ്രാഥമിക്യമോടെയും തിരയൽ  

## പരിപാലന സമയക്രമം

### പതിവ് അപ്ഡേറ്റുകള്‍  
- **മാസാന്ത പരിശോധനകൾ**: ഉള്ളടക്ക കൃത്യതയും ലിങ്ക് പരിശോധനവും  
- **ത്രൈമാസ അപ്ഡേറ്റുകൾ**: പ്രധാന ഉള്ളടക്ക കൂട്ടിച്ചേർക്കലുകളും മെച്ചപ്പെടുത്തലുകളും  
- **അർധവാർഷിക അവലോകനങ്ങൾ**: സമഗ്ര പുനസംഘടനയും മെച്ചപ്പെടുത്തലും  
- **വാർഷിക റിലീസുകൾ**: പ്രധാന പതിപ്പുകളുടെ അപ്ഡേറ്റുകൾ സാംഹിതാഭ്യന്തര മെച്ച പൂര്‍ത്തിയോടെ  

### നിരീക്ഷണവും ഗുണനിലവാര ഉറപ്പും  
- **സ്വയംപരിശോധന**: കോഡ് ഉദാഹരണങ്ങളും ലിങ്കുകളും സ്ഥിരമായി പരിശോധന  
- **കമ്മ്യൂണിറ്റി പ്രതികരണ സംയോജനം**: ഉപയോക്തൃ നിർദ്ദേശങ്ങൾ സ്ഥിരമായി ഉൾപ്പെടുത്തല്‍  
- **സാങ്കേതിക അപ്ഡേറ്റ്**: ഏറ്റവും പുതിയ Azure സേവനങ്ങളും azd റിലീസുകളുടെ അനുയോജ്യമായ പൊരുത്തം  
- **ആക്‌സസിബിലിറ്റി ഓഡിട്**: ഉൾക്കൊള്ളുന്ന രൂപകല്പനാ സങ്കേതങ്ങൾക്കുള്ള നിരന്തരം അവലോകനം  

## പതിപ്പ് പിന്തുണ നയം

### നിലവിലുള്ള പതിപ്പ് പിന്തുണ  
- **സമീപകാല പ്രധാന പതിപ്പ്**: പൂർണ്ണ പിന്തുണയോടെ സ്ഥിരമായ അപ്ഡേറ്റുകൾ  
- **മുമ്പ് പ്രധാന പതിപ്പുകൾ**: 12 മാസം സുരക്ഷാ അപ്ഡേറ്റുകൾ, ഗുണഗ്രാഹക പരിഹാരങ്ങൾ  
- **പഴയ പതിപ്പുകൾ**: കമ്മ്യൂണിറ്റി പിന്തുണ മാത്രം, ഔദ്യോഗിക അപ്ഡേറ്റുകൾ ഇല്ല  

### മൈഗ്രേഷൻ മാർഗ്ഗനിർദ്ദേശം  
പ്രധാന പതിപ്പുകൾ പുറത്തു വന്നപ്പോൾ നാം നൽകുന്നത്:  
- **മൈഗ്രേഷൻ ഗൈഡുകൾ**: പടിവട്ടം മാറ്റത്തിന്റെ വിശദമായ മാർഗ്ഗനിർദ്ദേശങ്ങൾ  
- **പൊരുത്ത നോട്ടുകൾ**: ജടിലമാകുന്ന മാറ്റങ്ങൾക്കായുള്ള വിശദാംശങ്ങൾ  
- **ടൂൾ പിന്തുണ**: മൈഗ്രേഷനു സഹായകമായ സ്‌ക്രിപ്റ്റുകളും യൂട്ടിലിറ്റികളും  
- **കമ്മ്യൂണിറ്റി പിന്തുണ**: മൈഗ്രേഷൻ ചോദ്യങ്ങൾക്ക് പ്രത്യേകം ഫോറങ്ങൾ
