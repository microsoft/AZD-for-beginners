# AI ഡെവലപ്പർമാർക്കുള്ള AZD വർക്ക്‌ഷോപ്പ്

AI ആപ്പളിക്കേഷൻ വിന്യസനത്തിൽ ശ്രദ്ധ കേന്ദ്രീകരിച്ച് Azure Developer CLI (AZD) പഠിക്കാൻ ഒരു ഹാൻഡ്‌സ്-ഓൺ വർക്ക്‌ഷോപ്പിലേക്ക് സ്വാഗതം. ഈ വർക്ക്‌ഷോപ്പ് നിങ്ങൾക്കു AZD ടെംപ്ലേറ്റുകളുടെ പ്രായോഗിക മനസ്സിലാക്കൽ 3 ഘട്ടങ്ങളിൽ സഹായിക്കും:

1. **അന്വേഷണം** - നിങ്ങൾക്കു യോജിക്കുന്ന ടെംപ്ലേറ്റ് കണ്ടെത്തുക.
1. **വിന്യസനം** - വിന്യസിക്കുകയും അത് പ്രവർത്തിക്കുന്നതായി സ്ഥിരീകരിക്കുകയും ചെയ്യുക
1. **അനുകൂലനം** - മാറ്റങ്ങൾ വരുത്തി നിങ്ങളുടെ അനുസൃതമാക്കുക!

ഈ വർക്ക്‌ഷോപ്പിൽ, നിങ്ങളുടെ അവസാനം-തലക്കെട്ട് ഡെവലപ്പ്മെന്റ് യാത്ര എളുപ്പമാക്കാൻ പ്രധാന ഡെവലപ്പർ ഉപകരണങ്ങളും വർക്ക്‌ഫ്ലോകളും പരിചയപ്പെടുത്തും.

<br/>

## ബ്രൗസർ-അധാരിത ഗൈഡ്

വർക്ക്‌ഷോപ്പ് പാഠങ്ങൾ Markdown-ൽ ആണ്. നിങ്ങൾക്ക് അവയെ നേരിട്ട് GitHub-ൽ നോക്കാനാകും - അല്ലെങ്കിൽ താഴെ കാണിക്കുന്ന സ്ക്രീൻഷോട്ടുപോലെ ബ്രൗസർ-അധാരിത പ്രിവ്യൂ ലോഞ്ച് ചെയ്യാം.

![Workshop](../../../translated_images/ml/workshop.75906f133e6f8ba0.webp)

ഈ ഓപ്ഷൻ ഉപയോഗിക്കാൻ - റിപ്പോസിറ്ററി നിങ്ങളുടെ പ്രൊഫൈലിലേക്ക് ഫോർക് ചെയ്യുക, പിന്നെ GitHub Codespaces ആരംഭിക്കുക. VS Code ടെർമിനൽ സജീവമായപ്പോൾ, ഈ കമാൻഡ് ടൈപ്പ് ചെയ്യുക:

```bash title="" linenums="0"
mkdocs serve > /dev/null 2>&1 &
```

കുറച്ച് സെക്കൻഡുകളിലേയ്ക്ക് പോപ്-അപ്പ് ഡയലോഗ് കാണിക്കും. `Open in browser` എന്ന ഓപ്ഷൻ തിരഞ്ഞെടുക്കുക. വെബ്-അധാരിത ഗൈഡ് ഇപ്പോൾ പുതിയ ബ്രൗസർ ടാബിൽ തുറക്കും. ഈ പ്രിവ്യൂവിന്റെ ചില ഗുണങ്ങൾ:

1. **ഉൾപ്പെട്ട സംവേദനം** - കീ വർഡുകളും പാഠങ്ങളും വേഗത്തിൽ കണ്ടെത്താം.
1. **കോപി ഐക്കൺ** - കോഡ് ബ്ലോക്കുകൾക്ക് മുകളിലേയ്ക്ക് ഹോവർ ചെയ്താൽ ഈ ഓപ്ഷൻ കാണാം
1. **തീം ടോഗിള്‍** - ഇരുണ്ടതും വെളുത്തതുമായ തീമുകൾക്കിടയിൽ മാറാം
1. **സഹായം ലഭിക്കുക** - ഫൂട്ടറിലുള്ള Discord ഐക്കൺ ക്ലിക്ക് ചെയ്ത് ചേർക്കുക!

## നിങ്ങൾ എന്ത് പഠിക്കും

AZD ടെംപ്ലേറ്റ് Microsoft Foundry-യിലെ അവസാനം-തലക്കെട്ട് ഡെവലപ്പ്മെന്റിനുള്ള വിവിധ ശേഷികളെയും ഉപകരണങ്ങളെയും പരീക്ഷിക്കാൻ ഒരു പഠന സാന്റ്ബോക്സ് എന്ന രീതിയിൽ ചിന്തിക്കുക. ഈ വർക്ക്‌ഷോപ്പ് അവസാനം, ഈ രംഗത്തെ വിവിധ ഉപകരണങ്ങളും ആശയങ്ങളും നിങ്ങൾക്ക് ഒരു സ്വാഭാവിക ബോധം ലഭിച്ചിരിക്കണം.

| ആശയം  | ലക്ഷ്യം |
|:---|:---|
| **Azure Developer CLI** | ഉപകരണം കമാൻഡുകളും വർക്ക്‌ഫ്ലോകളും മനസ്സിലാക്കുക|
| **AZD ടെംപ്ലേറ്റുകൾ**| പ്രോജക്ട് ഘടനയും കോൺഫിഗറേഷനും മനസ്സിലാക്കുക|
| **Azure AI ഏജന്റ്**| Microsoft Foundry പ്രോജക്ട് പ്രൊവിഷൻ & വിന്യസനം |
| **Azure AI സെർച്ച്**| ഏജന്റ്‌സുമായി കോൺടെക്‌സ്‌റ്റ് എൻജിനീയറിംഗ് സജ്ജമാക്കുക |
| **ഓബ്സർവബിലിറ്റി**| ട്രെയിസിംഗ്, മോണിറ്ററിങ്, മൂല്യനിർണയം പരീക്ഷിക്കുക |
| **റെഡ് ടീമിംഗ്**| എതിര്‍പക്ഷ പരിശോധനയും അതിനെതിരെ വിമോചനങ്ങളും പരീക്ഷിക്കുക |
