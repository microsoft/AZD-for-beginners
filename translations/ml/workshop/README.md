# AI വികസപകർക്ക് предназнач workshop

AI അപ്ലിക്കേഷൻ വിന്യാസത്തിൽ കേന്ദ്രീകരിച്ച് Azure Developer CLI (AZD) പഠിക്കുന്നതിനായുള്ള പ്രായോഗിക വർക്‌ഷോപ്പിലേക്ക് സ്വാഗതം. ഈ വർക്‌ഷോപ്പ് AZD ടെംപ്ലേറ്റുകൾ 3 പടികളിൽ പ്രയോഗപരമായി മനസ്സിലാക്കാൻ സഹായിക്കുന്നു:

1. **അന്വേഷണം** - നിങ്ങളെത്തക്കുള്ള അനുയോജ്യമായ ടെംപ്ലേറ്റ് കണ്ടെത്തുക.
1. **വിന്യാസം** - വിന്യസിച്ച് അത് പ്രവർത്തിക്കുന്നതായി സ്ഥിരീകരിക്കുക
1. **ഇരുപക്ഷീയമാക്കൽ** - ഫറിർത്തും മാറ്റിയും നിങ്ങളുടെ സ്വഭാവമാക്കുക!

ഈ വർക്‌ഷോപ്പിന്റെ ընթացքում നിങ്ങൾ പ്രധാന ഡെവലപ്പറിന്റെ ഉപകരണങ്ങളെയും പ്രവൃത്തിപദ്ധതികളെയും പരിചയപ്പെടും, നിങ്ങളുടെ ആനുബന്ധ വികസനയാത്ര സുഗമമാക്കാൻ.

<br/>

## ബ്രushaർ അടിസ്ഥാനമായ മാർഗ്ഗനിർദേശം

വർക്‌ഷോപ്പ് പാഠങ്ങൾ മാർക്‌ഡൗണിൽ ആണ്. നിങ്ങൾക്ക് അവ നേരിട്ട് GitHub-ല്‍ കാണാം - അല്ലെങ്കിൽ താഴെ കാണിക്കുന്ന സ്ക്രീൻഷോട്ട് പോലെ ബ്രൗസർ അടിസ്ഥാന പ്രിവ്യൂ തുടങ്ങാം.

![Workshop](../../../translated_images/ml/workshop.75906f133e6f8ba0.webp)

ഈ ഓപ്ഷൻ ഉപയോഗിക്കാൻ - റിപ്പോസിറ്ററി നിങ്ങളുടെ പ്രൊഫൈലിലേക്ക് ഫോർക്ക് ചെയ്ത് GitHub Codespaces ആരംഭിക്കൂ. VS Code ടെർമിനൽ പ്രവർത്തനക്ഷമമാകുമ്പോൾ, ഈ കമാൻഡ് ടൈപ്പുചെയ്യുക:

ഈ ബ്രൗസർ പ്രിവ്യൂ GitHub Codespaces, ഡെവ് കണ്ടെയിനറുകൾ, അല്ലെങ്കിൽ Python, MkDocs ഇൻസ്റ്റാൾ ചെയ്ത ഒരു ലോകൽ ക്ലോൺ എന്നിവയിൽ പ്രവർത്തിക്കുന്നു.

```bash title="" linenums="0"
mkdocs serve > /dev/null 2>&1 &
```

കുറച്ചു സെക്കൻഡുകളിൽ ഒരു പాప్-അപ്പ് ഡയലോഗ് ദൃശ്യമാകും. `Open in browser` ഓപ്ഷൻ തിരഞ്ഞെടുക്കുക. വെബ് അടിസ്ഥാനമുള്ള മാർഗ്ഗനിർദേശം ഇനി പുതിയ ബ്രൗസർ ടാബിൽ തുറക്കും. ഈ പ്രിവ്യുവിന്റെ ചില ഗുണങ്ങൾ:

1. **നിർമിതമായ തിരയൽ** - കീ-വാക്കുകളും പാഠങ്ങളും എളുപ്പത്തിൽ കണ്ടെത്തുക.
1. **കോപി ഐക്കൺ** - കോഡ് ബ്ലോക്കുകളുടെ മുകളിൽ ഹോവർ ചെയ്താൽ കാണാം
1. **തീം ഫോർക്കൽ** - ഇരുണ്ടതും വെളുത്തതുമായ തീമുകൾ арасында മാറുക
1. **സഹായം നേടുക** - അടിക്കുറിപ്പിൽ ഡിസ്‌കോർഡ് ഐക്കണിൽ ക്ലിക്ക് ചെയ്ത് ചേരുക!

## നിങ്ങൾ പഠിക്കുന്നതെന്താണ്

AZD ടെംപ്ലേറ്റ് എന്നത് Microsoft Foundry-ൽ end-to-end വികസനത്തിന് വിവിധ കഴിവുകളും ഉപകരണങ്ങളും പരീക്ഷിക്കാൻ പഠന സാങ്‌ബോക്സായി ചിന്തിക്കാം. ഈ വർക്‌ഷോപ്പ് അവസാനം, ഈ പ്രസ്താവനയിൽ ഉള്ള სხვადასხვა ടൂളുകളുടെയും ആശയങ്ങളുടെയും കൂടുതല്‍ അവബോധമുണ്ടാകും.

| ആശയം  | ലക്ഷ്യം |
|:---|:---|
| **Azure Developer CLI** | ഉപകരണം കമാൻഡുകളും പ്രവൃത്തിപദ്ധതികളെയും മനസ്സിലാക്കുക|
| **AZD ടെംപ്ലേറ്റുകൾ**| പ്രോജക്ട് ഘടനയും കോൺഫിഗറേഷനും മനസ്സിലാക്കുക|
| **Azure AI ഏജന്റ്**| Microsoft Foundry പ്രോജക്ട് അനുവദിക്കുകയും വിന്യാസം ചെയ്യുക  |
| **Azure AI സേർച്ച്**| ഏജന്റുമാരോടൊപ്പം ബന്ധമായ എഞ്ചിനീയറിങ് സജ്ജമാക്കുക |

| **റെഡ് ടീം**| എതിരാളിത്ത പരിശോധനയും പരിഹാരങ്ങളും പഠിക്കുക |

- **ലാബ്**: എന്റെ സീനാരിയോക്ക് അനുയോജ്യമായി ടെംപ്ലേറ്റ് ഇഷ്‌ടാനുസൃതമാക്കുക
