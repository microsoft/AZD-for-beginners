# AI ഡെവലപ്പർമാർക്കുള്ള AZD

!!! info "ഈ വർക്ക്‌ഷോപ്പിന്റെ പ്രേക്ഷകർ"
   
    Azure AI Foundry-ൽ മികച്ച ഉപകരണങ്ങളും പ്രവൃത്തി രീതികളും ഉപയോഗിച്ച് AI ഏജന്റിക് ആപ്ലിക്കേഷനുകൾ നിർമ്മിക്കാൻ ആഗ്രഹിക്കുന്നവർക്കുള്ള വർക്ക്‌ഷോപ്പാണ് ഇത്. Azure, AI ആശയങ്ങൾ, Python കോഡ് എന്നിവയിൽ പരിചയമുള്ള തുടക്കക്കാരും പരിചയസമ്പന്നരായ ഡെവലപ്പർമാർക്കും ഈ വർക്ക്‌ഷോപ്പ് അനുയോജ്യമാണ്. ആരംഭിക്കുന്നതിന് മുമ്പ് മുൻ‌വ്യവസ്ഥകൾ പരിശോധിക്കുക.

    **ദൈർഘ്യം:** 3-4 മണിക്കൂർ  
    **നില:** തുടക്കക്കാരൻ മുതൽ ഇടത്തരം വരെ  

---

## മുൻ‌വ്യവസ്ഥകൾ

- വ്യക്തിഗത GitHub അക്കൗണ്ട് → [ഇതൊരു സൗജന്യമായി സൃഷ്ടിക്കുക](https://github.com/signup)
- സജീവ Azure അക്കൗണ്ട് → [ഇതൊരു സൗജന്യമായി സൃഷ്ടിക്കുക](https://aka.ms/free)
- AI ആശയങ്ങൾ, Python ഡെവലപ്മെന്റ്, കമാൻഡ്-ലൈൻ ഉപകരണങ്ങൾ എന്നിവയിൽ പരിചയം
- VS Code, GitHub ഉപകരണങ്ങൾ, പ്രവൃത്തി രീതികൾ എന്നിവയിൽ പരിചയം.

---

## പഠന ലക്ഷ്യങ്ങൾ

ഈ വർക്ക്‌ഷോപ്പ് അവസാനിപ്പിക്കുന്നതിന് മുമ്പ്, നിങ്ങൾക്ക് താഴെ പറയുന്നവ ചെയ്യാൻ കഴിയും:

- [X] **Azure Developer CLI ഉപയോഗിക്കുക** → നിങ്ങളുടെ Azure റിസോഴ്സുകൾ പ്രൊവിഷൻ ചെയ്യാനും, ഡിപ്ലോയ് ചെയ്യാനും, മാനേജ് ചെയ്യാനും.
- [X] **AI ആപ്പ് ടെംപ്ലേറ്റുകൾ ഉപയോഗിക്കുക** → Azure AI Foundry-ൽ AI ഏജന്റ് പ്രോജക്റ്റുകൾ ആരംഭിക്കാൻ.
- [X] **ടെംപ്ലേറ്റുകൾ കോൺഫിഗർ ചെയ്യുക** → Red Teaming, Evaluations, Tracing & Monitoring പോലുള്ള സവിശേഷതകൾക്കായി.
- [X] **AZD ടെംപ്ലേറ്റുകളും പ്രവൃത്തി രീതികളും** → GitHub Copilot ഉപയോഗിച്ച് ഡീകൺസ്ട്രക്റ്റ് ചെയ്യുകയും എക്സ്പ്ലോർ ചെയ്യുകയും ചെയ്യുക.
- [X] **AI ഏജന്റ് സ്റ്റാർട്ടർ ടെംപ്ലേറ്റ്** → നിങ്ങളുടെ കസ്റ്റം ആവശ്യങ്ങൾക്കനുസരിച്ച് കസ്റ്റമൈസ് ചെയ്യുകയും വീണ്ടും ഡിപ്ലോയ് ചെയ്യുകയും ചെയ്യുക.

## വർക്ക്‌ഷോപ്പ് രൂപരേഖ

വർക്ക്‌ഷോപ്പ് മോഡ്യൂളുകളായി ഘടനയുള്ളതാണ്, ഓരോന്നും പ്രായോഗിക ലാബ് വ്യായാമം കൊണ്ട് ആAnchored:

1. **പരിചയം** - AZD ടെംപ്ലേറ്റുകൾ എന്താണെന്ന്, അവ എങ്ങനെ പ്രവർത്തിക്കുന്നു എന്ന് മനസിലാക്കുക
1. **തിരഞ്ഞെടുപ്പ്** - നിങ്ങളുടെ അടുത്ത AI പ്രോജക്റ്റിനുള്ള ശരിയായ "സ്റ്റാർട്ടർ" ടെംപ്ലേറ്റ് കണ്ടെത്തുക
1. **സാധൂകരണം** - ടെംപ്ലേറ്റ് ഡിപ്ലോയ് ചെയ്യുകയും, അത് പരസ്യപ്പെടുത്തിയതുപോലെ പ്രവർത്തിക്കുന്നുണ്ടെന്ന് പരിശോധിക്കുകയും ചെയ്യുക
1. **ഡീകൺസ്ട്രക്ഷൻ** - ടെംപ്ലേറ്റ് അതിന്റെ ഘടകങ്ങളും കോൺഫിഗറേഷനും മനസിലാക്കാൻ പകർത്തുക
1. **കോൺഫിഗറേഷൻ** - വിവിധ Azure AI Foundry സവിശേഷതകൾ സജീവമാക്കാനും ഉപയോഗിക്കാനും പഠിക്കുക
1. **കസ്റ്റമൈസേഷൻ** - നിങ്ങളുടെ ഡാറ്റയും ആവശ്യങ്ങളും മനസ്സിൽ വെച്ച് ആപ്ലിക്കേഷൻ പുനർനിർമിക്കുക
1. **ടിയർഡൗൺ** - ഇൻഫ്രാസ്ട്രക്ചർ ഡീപ്രൊവിഷൻ ചെയ്യാനും സോഫ്റ്റ്-ഡിലീറ്റ് ചെയ്ത റിസോഴ്സുകൾ വീണ്ടെടുക്കാനും പഠിക്കുക

## ഫീഡ്ബാക്ക് & പ്രശ്നങ്ങൾ

തുടക്കക്കാർക്കുള്ള മികച്ച പഠനാനുഭവം നൽകാൻ കറിക്കുലം സജീവമായി അപ്ഡേറ്റ് ചെയ്യുന്നു. നിങ്ങളുടെ ഫീഡ്ബാക്ക് സ്വാഗതം ചെയ്യുന്നു.

1. ഈ റീപ്പോയിൽ ഒരു പ്രശ്നം പോസ്റ്റ് ചെയ്യുക - അത് `Workshop` എന്ന് ടാഗ് ചെയ്യുക.
1. Azure AI Foundry Discord-ൽ ചേരുക - സഹപാഠികളുമായി ബന്ധപ്പെടുക!
