# AZD For AI Developers

!!! info "ഈ വർക്ക്‌ഷോപ്പിനുള്ള പ്രേക്ഷകർ"
   
    മികച്ച ടൂളുകളും പ്രവർത്തനപ്രവഹങ്ങളും ഉപയോഗിച്ച് Microsoft Foundry-ൽ AI Agentic ആപ്ലിക്കേഷനുകൾ നിർമ്മിക്കാൻ ആഗ്രഹിക്കുന്ന ഏതൊരു വ്യക്തിക്കും ഈ വര്‍ക്ക്‌ഷോപ്പ് ആണ്. Azure, AI ആശയങ്ങൾ, Python കോഡ് എന്നിവയിൽ ചില പരിചയമുള്ള പുതിയവർക്കും പരിചയസമ്പന്നർക്കും ഈ വർക്ക്‌ഷോപ്പ് അനുയോജ്യമാണ്. തുടങ്ങുന്നതിന് മുമ്പ് ആവശ്യമായ മുൻകൂർ ശിപാർശകൾ പരിശോധിക്കുക.

    **അവധി:** 3-4 മണിക്കൂർ  
    **നില:** പ്രാരംഭം മുതൽ മധ്യനില വരെ  

---


## മുൻകൂർ ആവശ്യങ്ങൾ

- വ്യക്തിഗത GitHub അക്കൗണ്ട് → [മुफ്തമായി സൃഷ്ടിക്കുക](https://github.com/signup)
- സജീവമായ Azure അക്കൗണ്ട് → [മुफ്തമായി സൃഷ്ടിക്കുക](https://aka.ms/free)
- AI ആശയങ്ങൾ, Python ഡെവലപ്പ്മെന്റ്, കമാൻഡ്-ലൈൻ ടൂളുകൾ എന്നിവയോടെ പരിചയം
- VS Code, GitHub ടൂളുകളും പ്രവർത്തനപ്രവഹങ്ങളും ഉപയോഗിക്കാനുള്ള പരിചയം.

---

## പഠന ലക്ഷ്യങ്ങൾ

ഈ വർക്ക്‌ഷോപ്പ് അവസാനിക്കുമ്പോൾ, നിങ്ങൾക്ക് അറിയാം:

- [X] **Azure Developer CLI** ഉപയോഗിച്ച് → നിങ്ങളുടെ Azure വിഭവങ്ങൾ ഒരുക്കാനും, വിന്യസിക്കാനും, നിയന്ത്രിക്കാനും.
- [X] **AI ആപ്പ് ടെംപ്ലേറ്റുകൾ ഉപയോഗിക്കുക** → Microsoft Foundry-ൽ AI Agent പ്രോജക്ടുകൾ ത്വരിതപ്പെടുത്താൻ.
- [X] **ടെംപ്ലേറ്റുകൾ ക്രമീകരിക്കുക** → Red Teaming, Evaluations, Tracing & Monitoring പോലുള്ള ഫീച്ചറുകൾക്ക്.
- [X] **AZD ടെംപ്ലേറ്റുകളും പ്രവർത്തനപ്രവാഹങ്ങളും വിച്ഛേദിച്ച് പരിശോധിക്കുക** → GitHub Copilot ഉപയോഗിച്ച്.
- [X] **ഇഷ്ടാനുസൃതമാക്കി വീണ്ടും വിന്യസിക്കുക** → നിങ്ങളുടെ കസ്റ്റം ആവശ്യങ്ങളുമായി AI Agent സ്റ്റാർട്ടർ ടെംപ്ലേറ്റ്.

## വർക്ക്‌ഷോപ്പ് രൂപരേഖ

വർക്ക്ഷോപ്പ് ലക്ഷ്യമിടുന്ന ലൈവ് ലാബ് എക്സरसൈസുകളാൽ അടിസ്ഥാനമാക്കിയുള്ള മോഡ്യൂളുകളായി ക്രമീകരിച്ചിരിക്കുന്നു:

1. **പരിചയം** - AZD ടെംപ്ലേറ്റുകൾ എന്താണെന്ന്, എങ്ങനെ പ്രവർത്തിക്കുന്നു എന്ന് മനസ്സിലാക്കുക
1. **തിരഞ്ഞെടുപ്പ്** - നിങ്ങളുടെ അടുത്ത AI പ്രോജക്ടിനുള്ള ശരിയായ "സ്റ്റാർട്ടർ" ടെംപ്ലേറ്റ് കണ്ടെത്തുക
1. **സത്യമാക്കൽ** - ടെംപ്ലേറ്റ് വിന്യാസം ചെയ്യുകയും അത് ശരിയായി പ്രവർത്തിക്കുന്നുവെന്ന് പരിശോധിക്കുക
1. **വ്യാഖ്യാനം** - ടെംപ്ലേറ്റ് വിച്ഛേദിച്ച് അവയവങ്ങളും ക്രമീകരണങ്ങളും മനസ്സിലാക്കുക
1. **ക്രമീകരണം** - വ്യത്യസ്ത Microsoft Foundry ഫീച്ചറുകൾ സജീവമാക്കാനും ഉപയോഗിക്കാനും പഠിക്കുക
1. **ഇഷ്ടാനുസൃതി** - നിങ്ങളുടെ ഡാറ്റയും ആവശ്യങ്ങളും പരിഗണിച്ച് ആപ്ലിക്കേഷൻ പുനരോദ്യോഗം ചെയ്യുക
1. **അവസാനീകരണം** - ഇൻഫ്രാസ്ട്രക്ചർ ഉപതിപ്പിച്ച് സോഫ്റ്റ്-ഡിലീറ്റഡ് വിഭവങ്ങൾ പുനഃസ്ഥാപിക്കാനും പഠിക്കുക

## പ്രതികരണങ്ങളും പ്രശ്‌നങ്ങളും

പ്രാരംഭ വിദ്യാർത്ഥികൾക്ക് ഏറ്റവും നല്ല പഠന അനുഭവം നൽകുന്നതിനായി നാം പാഠ്യപദ്ധതികൾ സജീവമായി പുതുക്കുന്നു. നിങ്ങളുടെ പ്രതികരണം നിർണായകമാണ്.

1. ഈ റീപ്പോയിൽ പ്രശ്‌നം പോസ്റ്റ് ചെയ്യുക - സൌകര്യത്തിനായി `Workshop` എന്ന് ടാഗ് ചെയ്യുക.
1. Microsoft Foundry Discord- ൽ ചേരുക - സഹപാഠികളുമായി ബന്ധപ്പെടുക!
