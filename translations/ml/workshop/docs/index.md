# എഐ ഡെവലപ്പർമാർക്കുള്ള AZD വര്‍ക്ക്ഷോപ്പ്

> `azd 1.27.1` യിൽ 2026 ജൂലൈയിൽ സ്ഥിരീകരിച്ചിട്ടുണ്ട്.

> [!IMPORTANT]  
> **ഈ വര്‍ക്ക്ഷോപ്പുകൊപ്പിയിലേക്ക് ഒരു വർക്ക്ഷോപ്പ് ഗൈഡ് ഉൾപ്പെടുത്തിയ വിവർത്തനമാണ്, അത് നിങ്ങൾക്ക് നിങ്ങളുടെ ലോക്കൽ ബ്രൗസറിൽPreview ചെയ്യാം. ആരംഭിക്കാൻ, GitHub Codespaces, ഒരു ഡെവ് കണ്ടൈനർ, അല്ലെങ്കിൽ MkDocs ഇൻസ്റ്റാൾ ചെയ്ത ലോക്കൽ ക്ലോൺ ഉപയോഗിച്ച് റിപ്പോ തുറക്കുക—അപ്പോൾ സജീവമായ VS കോഡ് ടെർമിനൽ കാണുന്നതു വരെയും കാത്തിരിക്കുക, പിന്നെ ടൈപ്പ് ചെയ്യുക:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **നിങ്ങൾ ബ്രൗസറിൽ പ്രിവ്യൂ പേജ് തുറക്കാൻ ഒരു പോപ്പ്-അപ്പ് ഡയലോഗ് കാണണം.**

എഐ ആപ്ലിക്കേഷൻ ഡിപ്ലോയ്മെന്റിന്റെ മേൽനോട്ടത്തോടെ ആസ്യൂർ ഡെവലപ്പർ CLI (AZD) പഠിക്കുന്നതിനുള്ള ഹാൻസ്ഒൺ വര്‍ക്ക്ഷോപ്പിലേക്ക് സ്വാഗതം. ഈ വര്‍ക്ക്ഷോപ്പ് AZD ടെംപ്ലേറ്റുകൾക്കുള്ള പ്രായോഗിക ബോധം നേടാൻ 3 ഘട്ടങ്ങളിൽ സഹായിക്കുന്നു:

1. **അന്വേഷണം** - നിങ്ങളുടെ ആവശ്യത്തിന് അനുയോജ്യമായ ടെംപ്ലേറ്റ് കണ്ടെത്തുക.
1. **ഡിപ്ലോയ്മെന്റ്** - ഡിപ്പ്ലോയ് ചെയ്ത് അത് ശരിയാണെന്ന് സ്ഥിരീകരിക്കുക
1. **അനുകൂലന** - നിങ്ങൾക്കായി മാറ്റങ്ങൾ വരുത്തി വരുത്തുക!

ഈ വര്‍ക്ക്ഷോപ്പിന്റെ COURSE എന്നിങ്ങനെയാണ്, നിങ്ങൾക്ക് കോർ ഡെവലപ്പർ ടൂളുകളും വർക്ക്‌ഫ്ലോകളും പരിചയപ്പെടുത്തും, നിങ്ങളുടെ ആൻഡ്-ടു-ആൻഡ് വികസനയാത്ര സുഗമമാക്കാൻ സഹായിക്കാൻ.

## വർക്ക്ഷോപ്പ് അവലോകനം

**ദൈർഘ്യം:** 3-4 മണിക്കൂർ  
**നിലവാരം:** ആരംഭികന്മാരിൽ നിന്നും ഇടത്തരം വരെ  
**മേൽവിലാസം:** ആസ്യൂർ, എഐ ആശയങ്ങൾ, VS കോഡ് & കമാൻഡ്-ലൈൻ ടൂളുകളിൽ പരിചയം.

ഇത് ചെയ്യുന്നതിലൂടെ പഠിക്കുന്ന ഹാൻസ്ഒൺ വര്‍ക്ക്ഷോപ്പാണ്. വ്യായാമങ്ങൾ പൂർത്തിയാക്കിയശേഷം, സിക്യൂരിറ്റി, ഉൽപാദകക്ഷമത മികച്ച രീതികൾ എന്നിവയിൽ നിങ്ങളുടെ പഠനയാത്ര തുടരാന് AZD For Beginners പാഠ്യക്രമം പരിശോധിക്കാൻ നിർദ്ദേശിക്കുന്നു.

| സമയം| മലയാളം | ലക്ഷ്യം |
|:---|:---|:---|
| 15 മിനിറ്റ് | പരിചയം | വേദി ഒരുക്കുക, ലക്ഷ്യങ്ങൾ മനസിലാക്കുക |
| 30 മിനിറ്റ് | എഐ ടെംപ്ലേറ്റ് തിരഞ്ഞെടുക്കുക | ഓപ്ഷനുകൾ പര്യവേക്ഷണം ചെയ്ത് ആരംഭിക്കുക | 
| 30 മിനിറ്റ് | എഐ ടെംപ്ലേറ്റ് സ്ഥിരീകരിക്കുക | അസ്യൂറിലേക്ക് ഡിഫോൾട്ട് പരിഹാരം ഡിപ്ലോയ് ചെയ്യുക |
| 30 മിനിറ്റ് | എഐ ടെംപ്ലേറ്റ് വിശദീകരിക്കുക | ഘടനയും കോൺഫിഗറേഷനും പരിശോധിക്കുക |
| 30 മിനിറ്റ് | എഐ ടെംപ്ലേറ്റ് കോൺഫിഗർ ചെയ്യുക | സജീവമാക്കി ലഭ്യമായ ഫീച്ചറുകൾ പരീക്ഷിക്കുക |
| 30 മിനിറ്റ് | എഐ ടെംപ്ലേറ്റ് ഇഷ്ടാനുസൃതമാക്കുക | നിങ്ങളുടെ ആവശ്യങ്ങൾക്കനുസരിച്ച് അറ്റംപ്ലേറ്റ് മാറ്റുക |

## നിങ്ങൾ പഠിക്കacak

AZD ടെംപ്ലേറ്റ് ഒരു പഠന സാൻബോക്സായി കരുതുക, അത് മൈക്രോസോഫ്റ്റ് ഫൗണ്ടറിയിൽ ആൻഡ്-ടു-ആൻഡ് വികസനത്തിന് വിവിധ കഴിവുകളും ടൂളുകളും പരിശോധിക്കാൻ സഹായിക്കുന്നു. ഈ വര്‍ക്ക്ഷോപ്പ് അവസാനിക്കുന്നത് വരെ നിങ്ങൾക്ക് വിവിധ ടൂളുകളും ആശയങ്ങളും ന്യായമായ ബോധമുണ്ടാകും.

| ആശയം | ലക്ഷ്യം |
|:---|:---|
| **ആസ്യൂർ ഡെവലപ്പർ CLI** | ടൂൾ കമാൻഡുകളും വർക്ക്‌ഫ്ലോകളും മനസിലാക്കുക|
| **AZD ടെംപ്ലേറ്റുകൾ**| പദ്ധതി ഘടനയും കോൺഫിഗറേഷനും മനസിലാക്കുക|
| **ആസ്യൂർ എഐ ഏജന്റ്**| മൈക്രോസോഫ്റ്റ് ഫൗണ്ടറി പദ്ധതി പ്രൊവിഴൻ ചെയ്ത് ഡിപ്ലോയ് ചെയ്യുക  |
| **ആസ്യൂർ എഐ സെർച്ചു**| ഏജന്റുകളുമായി സന്ധർഭ എഞ്ചിനീയറിങ്ങ് സജീവമാക്കുക |
| **ഒബ്സർവബിലിറ്റി**| ട്രേസിംഗും മോണിറ്ററിംഗും വിലയിരുത്തലുകളുമായി പരിചയം നേടുക |
| **റെഡ് ടീം**| എതിരാളി പരിശോധനകളും തടയലുകളും കണ്ടെത്തുക |
| | |
