# AZD for AI Developers Workshop

> [!IMPORTANT]  
> **ഈ വർക്ക്‌ഷോപ്പ് ഒരു വർക്ക്‌ഷോപ്പ് ഗൈഡോടെയാണ് ഇൻസ്ട്രുമെന്റഡ് ചെയ്യപ്പെട്ടത്, അത് നിങ്ങളുടെ ലോക്കൽ ബ്രൗസറിൽ പ്രിവ്യൂ ചെയ്യാം. തുടങ്ങാൻ, നിങ്ങൾ റെപ്പോയിൽ GitHub Codespaces പ്രവർത്തിപ്പിക്കണം—അതിനു ശേഷം സജീവമായ VS Code ടർമിനൽ കാണുമ്പോൾ ടൈപ്പ് ചെയ്യുക:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **പ്രിവ്യൂ പേജ് ബ്രൗസറിൽ തുറക്കാൻ ഒരു പോപ്പ്-അപ്പ് ഡയലോഗ് കാണണം.**

Azure Developer CLI (AZD) ക്കു കൈകൊണ്ട് ചെയ്യാവുന്ന വേർക്ക്ഷോപ്പിലേക്ക് സ്വാഗതം, വളരെ പ്രത്യേകമായി AI അപ്ലിക്കേഷൻ ഡെപ്ലോയ്മെന്റിന് കേന്ദ്രീകരിച്ച്. ഈ വേർക്ക്ഷോപ്പ് AZD ടെംപ്ലേറ്റുകൾ 3 പടികളിൽ പ്രയോഗപരമായ فهمം നേടാൻ സഹായിക്കുന്നു:

1. **അന്വേഷണം** - നിങ്ങളുടെ ആവശ്യത്തിനു യോജിക്കുന്ന ടെംപ്ലേറ്റ് കണ്ടെത്തുക.
1. **ഡെപ്ലോയ്മെന്റ്** - ഡെപ്ലോയ് ചെയ്തു പ്രവർത്തിക്കുന്നു എന്ന് ശരിവച്ച് നോക്കുക
1. **അനുവാദനങ്ങൾ** - മാറ്റം വരുത്തി നിങ്ങളുടെ ആകൃതി തയ്യാറാക്കുക!

ഈ വേർക്ക്ഷോപ്പിന്റെ കാലയളവിൽ, കോർ ഡെവലപ്പർ ടൂൾസും വർക്ഫ്ലോകളും പരിചയപ്പെടാം, ഇത് നിങ്ങളുടെ എൻഡ്-ടു-എൻഡ് വികസന ജേർണിയെ എളുപ്പവത്കരിക്കാൻ സഹായിക്കും.

## വേർക്ക്ഷോപ്പ് അവലോകനം

**കാലാവധി:** 3-4 മണിക്കൂർ  
**തരം:** അറിവില്ലാത്തവരിൽ നിന്ന് ഇടത്തരം വരെ  
**അവസാനപരിചയം:** Azure, AI ആശയങ്ങൾ, VS Code & കമാൻഡ്-ലൈൻ ടൂൾസിൽ പരിചയം.

ഇത് ഒരു കൈകൊണ്ട് ചെയ്യാനുള്ള വേർക്ക്ഷോപ്പാണ്, നിങ്ങളെത്തുടർന്ന് ചെയ്യുമ്പോൾ പഠിക്കുന്നതാണ്. വ്യായാമങ്ങൾ പൂർത്തിയാക്കിയ ശേഷം, AZD For Beginners പാഠ്യപദ്ധതി പുനരവലോകനം ചെയുക, സുരക്ഷയും ഉൽപാദനക്ഷമതയും ഉൾപ്പെടെയുള്ള മികച്ച പ്രവർത്തനങ്ങൾക്കായി പഠനം തുടർക്ക.

## നിങ്ങൾ പഠിക്കാനുള്ളത്

AZD ടെംപ്ലേറ്റ് എന്നാണ് Microsoft Foundry-യിൽ എന്റു മുതൽ എന്റു വരെ വികസനത്തിനായി വിവിധ കഴിവുകളും ടൂളുകളും പരീക്ഷിക്കാൻ പഠന സാൻഡ്ബോക്സ് എന്നുപ്രത്യേകം കരുതുക. ഈ വേർക്ക്ഷോപ്പ് അവസാനിച്ചപ്പോൾ, വിവിധ ടൂളുകളും ആശയങ്ങളും മനസ്സിലാക്കുന്നതിൽ നിങ്ങൾക്ക് ഒരു സ്വാഭാവിക ബോധമുണ്ടാകണം.

| ആശയം  | ലക്ഷ്യം |
|:---|:---|
| **Azure Developer CLI** | ടൂൾ കമാൻഡ്‌കളും വർക്ക്‌ഫ്ലോകളും മനസ്സിലാക്കുക |
| **AZD ടെംപ്ലേറ്റുകൾ**| പ്രൊജക്ടിന്റെ ഘടനയും കോൺഫിഗറേഷനും മനസിലാക്കുക |
| **Azure AI Agent**| Microsoft Foundry പ്രൊജക്ട് പ്രൊവിഷൻ ചെയ്ത് ഡെപ്ലോയ് ചെയ്യുക |
| **Azure AI Search**| ഏജന്റുകളിലൂടെ കോൺടക്സ് എഞ്ചിനീയറിംഗ് സജീവമാക്കുക |
| **Observability**| ട്രെയ്‌സിംഗ്, മോണിറ്ററിംഗ്, വിലയിരുത്തൽ പഠിക്കുക |
| **Red Teaming**| എതിര് വെല്ലുവിളി പരിശോധനയും ഉപശമനങ്ങളും പഠിക്കുക |
| | |
