# AZD for AI ഡെവലപ്പർമാർക്കുള്ള വർക്‌ഷോപ്പ്

> മാർച്ച് 2026-ൽ `azd 1.23.12` ഇൽ സ്ഥിരീകരിച്ചിരിക്കുന്നു.

> [!IMPORTANT]  
> **ഈ വർക്‌ഷോപ്പിന് ഒരു വർക്‌ഷോപ്പ് ഗൈഡ് ഇൻസ്റ്റ്രുമെന്റഡ് ചെയ്യപ്പെട്ടിട്ടുണ്ട്, അത് നിങ്ങളുടെ ലോക്കൽ ബ്രൗസറിൽ പ്രിവ്യൂ ചെയ്യാൻ കഴിയും. ആരംഭിക്കാൻ, GitHub Codespaces, ഒരു ഡെവ് കണ്ടെയ്നർ, അല്ലെങ്കിൽ MkDocs ഇൻസ്റ്റാൾ ചെയ്ത ഒരു ലോക്കൽ ക്ലോൺ ഉപയോഗിച്ച് റീപൊ തുറക്കുക—അതിനുശേഷം സജീവമായ ഒരു VS കോഡ് ടെർമിനൽ കാണാതെ വരെ കാത്തിരിക്കുക, തുടർന്ന് ടൈപ്പ് ചെയ്യുക:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **പ്രിവ്യൂ പേജ് ബ്രൗസറിൽ തുറക്കുന്നതിനുള്ള പോപ്പ്-അപ്പ് ഡയലോഗ് നിങ്ങൾക്ക് കാണപ്പെടണം.**

Azure Developer CLI (AZD) പഠിക്കുന്നതിനുള്ള ഹാൻഡ്‌സ്-ഓൺ വർക്‌ഷോപ്പിലേക്ക് സ്വാഗതം. ഇത് AI അപ്ലിക്കേഷൻ ഡെപ്രോയ്മെന്റിനായി കേന്ദ്രീകരിച്ചിരിക്കുന്നു. ഈ വർക്‌ഷോപ്പ് AZD ടെംപ്ലേറ്റുകളുടെ പ്രായോഗിക മനസ്‌കാഴ്ച നേടാൻ 3 ഘട്ടങ്ങളിൽ സഹായിക്കുന്നു:

1. **ആവിഷ്കാരം** - നിങ്ങളുടെ ആവശ്യത്തിന് അനുയോജ്യമായ ടെംപ്ലേറ്റ് കണ്ടെത്തുക.  
1. **ഡെപ്രോയ്മെന്റ്** - ഡെപ്രോയ് ചെയ്ത് അത് സജ്ജമായതായി പരിശോധിക്കുക  
1. **കസ്റ്റമൈസേഷൻ** - ഭേദഗതി ചെയ്ത് ഇത് നിങ്ങളുടെത് ആക്കുക!

ഈ വർക്‌ഷോപ്പിന്റെ ദൈർഘ്യത്തിൽ, പ്രാഥമിക ഡെവലപ്പർ ടൂൾസിനും വർക്ക്‌ഫ്ഹ്ലോസിനും പരിചയം തേടുകയും ചെയ്യും, ഇത് നിങ്ങളുടെ ആഖ്യാന്ത ഡെവലപ്‌മെന്റ് യാത്രയെ ലഘൂകരിക്കും.

## വർക്‌ഷോപ്പ് അവലോകനം

**ദൈർഘ്യം:** 3-4 മണിക്കൂർ  
**നില:** തുടക്കക്കാരൻ മുതൽ ഇടത്തരം വരെ  
**ആവശ്യങ്ങൾ:** Azure, AI ആശയങ്ങൾ, VS Code & കമാൻഡ്-ലൈൻ ടൂൾസുകൾ പരിചയം.

ഇത് ചെയ്‌തുകൊണ്ട് മനസ്സിലാക്കുന്ന ഒരു ഹാൻഡ്‌സ്-ഓൺ വർക്‌ഷോപ്പാണ്. വ്യായാമങ്ങൾ പൂർത്തിയാക്കിയതിന് ശേഷം, AZD For Beginners പാഠക്കുറി പരിശോധിച്ച് സുരക്ഷയും ഉൽപാദനക്ഷമതയും സംബന്ധിച്ച മികച്ച പ്രവർത്തനങ്ങൾക്കായി പഠന യാത്ര തുടരും.

## നിങ്ങൾക്ക് പഠിക്കാനുള്ളത്

AZD ടെംപ്ലേറ്റിനെ Microsoft Foundry-യിൽ end-to-end ഡെവലപ്പ്മെന്റിനുള്ള വിവിധ കഴിവുകളും ടൂൾസുകളും പരിശോധിക്കാനായി പഠന സാൻഡ്ബോക്ഷായി കാണുക. ഈ വർക്‌ഷോപ്പ് അവസാനം, നമുക്ക് ഇവിടെയുള്ള വിവിധ ടൂളുകളും ആശയങ്ങളും സ്വാഭാവികമായി മനസ്സിലാകും.

| ആശയം  | ലക്ഷ്യം |
|:---|:---|
| **Azure Developer CLI** | ടൂൾ കമാൻഡുകളും വർക്ക്‌ഫ്ലോകളും മനസ്സിലാക്കുക |
| **AZD ടെംപ്ലേറ്റുകൾ** | പ്രോജക്ട് ഘടനയും കോൺഫിഗറേഷനും മനസ്സിലാക്കുക |
| **Azure AI ഏജന്റ്** | Microsoft Foundry പ്രോജക്ട് പ്രൊവിഷൻ ചെയ്യാനും ഡെപ്രോയ് ചെയ്യാനും |
| **Azure AI Search** | ഏജന്റുകളുമായി കോൺടെക്‌സ്റ്റ് എഞ്ചിനീയറിംഗ് സജ്ജമാക്കുക |
| **ഒബ്‌സർവബിലിറ്റി** | ട്രേസിംങ്ങ്, മോണിറ്ററിംഗ്, മൂല്യനിർണയങ്ങൾ പരിശോധിക്കുക |
| **റെഡ് ടീംിംഗ്** | എഡ്വേഴ്സറിയൽ ടെസ്റ്റിംഗും തീർപ്പുകളും അന്വേഷിക്കുക |
| | |
