# AI ഡെവലപ്പർമാർക്കുള്ള AZD വർക്‌ഷോപ്പ്

> ജൂൺ 2026-ൽ `azd 1.25.6` ഉപയോഗിച്ച് സ്ഥിരീകരിച്ചിട്ടുണ്ട്.

> [!IMPORTANT]  
> **ഈ വർക്‌ഷോപ്പ് ഒരു വർക്‌ഷോപ്പ് ഗൈഡ് ഉപയോഗിച്ച് സജ്ജമാക്കിയിരിക്കുന്നു, അത് നിങ്ങളുടെ പ്രാദേശിക ബ്രൗസറിൽ നിങ്ങൾ പ്രിവ്യൂ ചെയ്യാം. തുടങ്ങാൻ GitHub Codespaces, ഒരു ഡെവ് കണ്ടൈനർ, അല്ലെങ്കിൽ MkDocs ഇൻസ്റ്റാൾ ചെയ്തിട്ടുള്ള ഒരു ലൊക്കൽ ക്ലോൺ തുറക്കുക—അതിന്റെ ശേഷം സജീവമായ VS Code ടെർമിനൽ കാണാൻ വരെ കാത്തിരിക്കുക, പിന്നെ ടൈപ്പ് ചെയ്യുക:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **ബ്രൗസറിൽ പ്രിവ്യൂ പേജ് തുറക്കാൻ ഒരു പാപ്പ്-അപ്പ് ഡയലോഗ് കാണണം.**

Azure Developer CLI (AZD) പഠിപ്പിക്കുന്ന AI അപ്ലിക്കേഷൻ ഡിപ്ലോയ്മെന്റിൽ തരത്തിൽ ശ്രദ്ധ കേന്ദ്രീകരിച്ചുള്ള പ്രായോഗിക വർക്‌ഷോപ്പിലേക്ക് സ്വാഗതം. ഈ വർക്‌ഷോപ്പ് 3 ഘട്ടങ്ങളിൽ AZD ടെംപ്ലേറ്റുകൾ പ്രായോഗികമായി മനസ്സിലാക്കാൻ സഹായിക്കും:

1. **അന്വേഷണം** - നിങ്ങളുടെ ആവശ്യത്തിനായി യുക്തമായ ടെംപ്ലേറ്റ് കണ്ടെത്തുക.
1. **ഡിപ്ലോയ്മെന്റ്** - ഡിപ്ലോയ് ചെയ്ത് പ്രവർത്തനക്ഷമമാണെന്ന് സ്ഥിരീകരിക്കുക.
1. **ഇടപെടൽ** - മാറ്റം വരുത്തുകയും താങ്കളുടെ ആവശ്യങ്ങൾക്കനുസരിച്ച് ഉൾപ്പെടുത്തുകയും ചെയ്യുക!

ഈ വർക്‌ഷോപ്പ് മുഴുവൻ, നിങ്ങളുടെ സമ്പൂർണ്ണ വികസനയാത്ര സുഗമമാക്കാൻ പ്രധാന ഡെവലപ്പർ ടൂളുകളും പ്രവൃത്തിബധതകളും പരിചയപ്പെടുത്തും.

## വർക്‌ഷോപ്പ് അവലോകനം

**കാലാവധി:** 3-4 മണിക്കൂർ  
**നിലവാരം:** ആരംഭക്കാരനിൽ നിന്നും ഇടത്തരം വരെ  
**മുന്‍‌പരിചയം:** Azure, AI ആശയങ്ങൾ, VS Code & കമാൻഡ്-ലൈൻ ടൂളുകളിൽ പരിചയം.

ഇത് പ്രായോഗികമായ ഒരു വർക്‌ഷോപ്പ് ആണെന്ന് നിങ്ങൾ ചെയ്യുന്നതിലൂടെ പഠിക്കും. വ്യായാമങ്ങൾ പൂർത്തിയാക്കിയതിന് ശേഷം, സുരക്ഷയും ഉത്പാദനക്ഷമതാ നല്ല അഭ്യസന മാർഗങ്ങളും ഉൾപ്പെടെ പഠന യാത്ര തുടരാൻ AZD For Beginners പാഠ്യപദ്ധതി വീണ്ടും അവലോകനം ചെയ്യുക.

| 30 മിനിറ്റ് | AI ടെംപ്ലേറ്റ് തിരഞ്ഞെടുക്കുക | ഓപ്ഷനുകൾ വിലയിരുത്തി തുടങ്ങിയതെടുക്കുക |

## നിങ്ങൾ എന്ത് പഠിക്കും

AZD ടെംപ്ലേറ്റിനെ Microsoft Foundryൽ സമഗ്ര വികസനത്തിനായുള്ള വിവിധ ശേഷികളും ടൂളുകളും പരിശോധിക്കാൻ ഉപയോഗിക്കുന്ന ഒരു പഠന രം​ഗഭൂമിയെന്നു കരുതുക. ഈ വർക്‌ഷോപ്പ് അവസാനിപ്പിക്കുന്നത് വരെ, നിങ്ങൾക്ക് ഈ പശ്ചാത്തലത്തിലെ വിവിധ ഉപകരണങ്ങളുടെയും ആശയങ്ങളുടെയും സ്വാഭാവികബോധം ഉണ്ടാകണം.

| ആശയം | ലക്ഷ്യം |
|:---|:---|
| **Azure Developer CLI** | ടൂൾ കമാൻഡുകളും പ്രവൃത്തിബധതകളും മനസ്സിലാക്കുക |
| **AZD ടെംപ്ലേറ്റുകൾ**| പ്രോജക്റ്റ് ഘടനയും കോൺഫിഗറേഷനും മനസ്സിലാക്കുക |
| **Azure AI ഏജന്റ്**| Microsoft Foundry പ്രോജക്റ്റ് പ്രൊവിഷൻ & ഡിപ്ലോയ് ചെയ്യുക  |
| **Azure AI സര്‍ച്ച്**| ഏജന്റുകളുമായി കോൺടെക്സ്റ്റ് എഞ്ചിനീയറിംഗ് സജ്ജമാക്കുക |
| **ഓബ്സർവബിലിറ്റി**| ട്രേസിങ്, നിരീക്ഷണം, വിലയിരുത്തലുകൾ അന്വേഷിക്കുക |
| **റെഡ് ടീമിംഗ്**| എതിരാളി സെക്യൂരിറ്റി പരിശോധനകളും പ്രതിരോധവും പരിശോധിക്കുക |
| | |
