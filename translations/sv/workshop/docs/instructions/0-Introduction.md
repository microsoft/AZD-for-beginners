# AZD For AI Developers

!!! info "MÅLGRUPP FÖR DENNA WORKSHOP"
   
    Denna workshop är för alla som vill bygga agentiska AI-applikationer på Microsoft Foundry genom att utnyttja förstklassiga verktyg och arbetsflöden. Workshopen passar både nybörjare och erfarna utvecklare som har viss kännedom om Azure, AI-koncept och Python-kod. Vänligen kontrollera förutsättningarna innan du börjar.

    **Duration:** 3-4 timmar  
    **Level:** Nybörjare till medelnivå  

---


## Förutsättningar

- Ett personligt GitHub-konto → [Create one for free](https://github.com/signup)
- Ett aktivt Azure-konto → [Create one for free](https://aka.ms/free)
- Bekantskap med AI-koncept, Python-utveckling, kommandoradsverktyg
- Erfarenhet av VS Code samt GitHub-verktyg och arbetsflöden.

---

## Lärandemål

När du har avslutat denna workshop kommer du att kunna:

- [X] **Arbeta med Azure Developer CLI** → för att provisionera, distribuera och hantera, dina Azure-resurser.
- [X] **Använda AI App Templates** →  för att snabbt komma igång med AI Agent-projekt på Microsoft Foundry.
- [X] **Konfigurera mallar** →  för funktioner som Red Teaming, utvärderingar, spårning och övervakning
- [X] **Dekomponera och utforska** →  AZD-mallar och arbetsflöden med GitHub Copilot
- [X] **Anpassa och distribuera om** →  AI Agent-startmallen, med dina egna krav

---

## Workshopöversikt

Workshopen är uppbyggd som moduler, var och en förankrad i en praktisk labbövning:

1. **Introduktion** - Få en förståelse för vad AZD-mallar är, och hur de fungerar
1. **Val** - Upptäck rätt "starter"-mall för ditt nästa AI-projekt
1. **Validering** - Distribuera och utforska mallen för att verifiera att den fungerar som utlovat
1. **Dekonstruktion** - Ta isär mallen för att förstå dess komponenter och konfiguration
1. **Konfiguration** - Lär dig aktivera och använda olika Microsoft Foundry-funktioner
1. **Anpassning** - Återskapa applikationen med dina data och krav i åtanke
1. **Nedmontering** - Lär dig att avprovisionera infrastruktur och återställa mjukt raderade resurser
