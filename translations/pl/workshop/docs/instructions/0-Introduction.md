# AZD dla deweloperów AI

!!! info "Odbiorcy tego warsztatu"
   
    Ten warsztat jest przeznaczony dla wszystkich, którzy chcą tworzyć aplikacje AI Agentic na platformie Microsoft Foundry, wykorzystując najlepsze narzędzia i procesy robocze. Warsztat jest odpowiedni zarówno dla początkujących, jak i doświadczonych deweloperów, którzy mają pewną wiedzę na temat Azure, koncepcji AI i kodu w Pythonie. Przed rozpoczęciem prosimy o sprawdzenie wymagań wstępnych.

    **Czas trwania:** 3-4 godziny  
    **Poziom:** Podstawowy do średniozaawansowanego  

---


## Wymagania wstępne

- Osobiste konto GitHub → [Załóż bezpłatnie](https://github.com/signup)
- Aktywne konto Azure → [Załóż bezpłatnie](https://aka.ms/free)
- Znajomość koncepcji AI, programowania w Pythonie, narzędzi wiersza poleceń
- Doświadczenie w pracy z VS Code oraz narzędziami i procesami GitHub.

---

## Cele nauki

Na koniec tego warsztatu będziesz wiedzieć, jak:

- [X] **Pracować z Azure Developer CLI** → aby tworzyć, wdrażać i zarządzać zasobami Azure.
- [X] **Korzystać z szablonów aplikacji AI** → aby szybko rozpocząć projekty AI Agent na Microsoft Foundry.
- [X] **Konfigurować szablony** → dla funkcji takich jak Red Teaming, Oceny, Śledzenie i Monitorowanie
- [X] **Dezkonstruować i badać** → szablony i procesy AZD za pomocą GitHub Copilot
- [X] **Dostosowywać i ponownie wdrażać** → szablon startowy AI Agent, zgodnie z własnymi wymaganiami

---

## Plan warsztatu

Warsztat jest podzielony na moduły, z których każdy opiera się na praktycznym ćwiczeniu laboratoryjnym:

1. **Wprowadzenie** - Poznaj, czym są szablony AZD i jak działają
1. **Wybór** - Odkryj odpowiedni „startowy” szablon dla swojego następnego projektu AI
1. **Walidacja** - Wdróż i zbadaj szablon, aby potwierdzić, że działa zgodnie z opisem
1. **Dezkonstruowanie** - Rozłóż szablon na części, aby zrozumieć jego komponenty i konfigurację
1. **Konfiguracja** - Naucz się aktywować i korzystać z różnych funkcji Microsoft Foundry
1. **Dostosowywanie** - Przebuduj aplikację zgodnie z własnymi danymi i wymaganiami
1. **Zakończenie** - Naucz się usuwać infrastrukturę i odzyskiwać miękko usunięte zasoby
