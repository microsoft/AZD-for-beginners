# AZD για Προγραμματιστές AI - Εργαστήριο

> Επικυρώθηκε με `azd 1.25.6` τον Ιούνιο 2026.

> [!IMPORTANT]  
> **Αυτό το εργαστήριο είναι εξοπλισμένο με έναν οδηγό εργαστηρίου που μπορείτε να προεπισκοπήσετε στο τοπικό σας πρόγραμμα περιήγησης. Για να ξεκινήσετε, ανοίξτε το repo σε GitHub Codespaces, ένα dev container, ή ένα τοπικό αντίγραφο με εγκατεστημένο το MkDocs—στη συνέχεια περιμένετε μέχρι να δείτε ένα ενεργό τερματικό VS Code και πληκτρολογήστε:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Θα πρέπει να δείτε ένα αναδυόμενο παράθυρο διαλόγου για να ανοίξετε τη σελίδα προεπισκόπησης σε ένα πρόγραμμα περιήγησης.**

Καλώς ήλθατε στο πρακτικό εργαστήριο για να μάθετε το Azure Developer CLI (AZD) με έμφαση στην ανάπτυξη εφαρμογών AI. Αυτό το εργαστήριο σας βοηθά να αποκτήσετε μια εφαρμοσμένη κατανόηση των προτύπων AZD σε 3 βήματα:

1. **Ανακάλυψη** - βρείτε το πρότυπο που σας ταιριάζει.
1. **Ανάπτυξη** - αναπτύξτε και επικυρώστε ότι λειτουργεί
1. **Προσαρμογή** - τροποποιήστε και επαναλάβετε για να το κάνετε δικό σας!

Κατά τη διάρκεια αυτού του εργαστηρίου, θα εισαχθείτε επίσης σε βασικά εργαλεία και ροές εργασίας ανάπτυξης, για να σας βοηθήσουν να απλοποιήσετε το ταξίδι ανάπτυξης από άκρο σε άκρο.

## Επισκόπηση Εργαστηρίου

**Διάρκεια:** 3-4 ώρες  
**Επίπεδο:** Αρχάριοι έως Ενδιάμεσοι  
**Προαπαιτούμενα:** Εξοικείωση με το Azure, έννοιες AI, VS Code & εργαλεία γραμμής εντολών.

Πρόκειται για ένα πρακτικό εργαστήριο όπου μαθαίνετε κάνοντας. Μόλις ολοκληρώσετε τις ασκήσεις, προτείνουμε να αναθεωρήσετε το πρόγραμμα σπουδών AZD για Αρχάριους για να συνεχίσετε το ταξίδι μάθησής σας προς τις βέλτιστες πρακτικές Ασφαλείας και Παραγωγικότητας.

| Χρόνος| Ενότητα  | Στόχος |
|:---|:---|:---|
| 15 λεπτά | Εισαγωγή | Θέτουμε το πλαίσιο, κατανοούμε τους στόχους |
| 30 λεπτά | Επιλογή Προτύπου AI | Εξερευνήστε επιλογές και επιλέξτε ένα αρχικό πρότυπο | 
| 30 λεπτά | Επικύρωση Προτύπου AI | Αναπτύξτε την προεπιλεγμένη λύση στο Azure |
| 30 λεπτά | Αποσύνθεση Προτύπου AI | Εξερευνήστε τη δομή και τη διαμόρφωση |
| 30 λεπτά | Διαμόρφωση Προτύπου AI | Ενεργοποιήστε και δοκιμάστε τις διαθέσιμες λειτουργίες |
| 30 λεπτά | Προσαρμογή Προτύπου AI | Προσαρμόστε το πρότυπο στις ανάγκες σας |
| 30 λεπτά | Απομάκρυνση Υποδομής | Καθαρισμός και απελευθέρωση πόρων |
| 15 λεπτά | Σύνοψη & Επόμενα Βήματα | Πόροι μάθησης, Πρόκληση εργαστηρίου |
| | |

## Τι θα μάθετε

Σκεφτείτε το Πρότυπο AZD ως ένα περιβάλλον μάθησης για να εξερευνήσετε διάφορες δυνατότητες και εργαλεία για ολοκληρωμένη ανάπτυξη στο Microsoft Foundry. Στο τέλος αυτού του εργαστηρίου, θα πρέπει να έχετε μια διαισθητική αίσθηση για διάφορα εργαλεία και έννοιες σε αυτό το πλαίσιο.

| Έννοια  | Στόχος |
|:---|:---|
| **Azure Developer CLI** | Κατανόηση των εντολών του εργαλείου και των ροών εργασίας |
| **Πρότυπα AZD**| Κατανόηση της δομής του έργου και των ρυθμίσεων |
| **Azure AI Agent**| Δημιουργία πόρων και ανάπτυξη έργου Microsoft Foundry  |
| **Azure AI Search**| Ενεργοποίηση μηχανικής συμφραζομένων με πράκτορες |
| **Παρατηρησιμότητα**| Εξερεύνηση ιχνηλάτησης, παρακολούθησης και αξιολογήσεων |
| **Red Teaming**| Εξερεύνηση επιθετικών δοκιμών και μετριασμών |
