# AZD για Προγραμματιστές AI - Εργαστήριο

> Επικυρώθηκε με `azd 1.23.12` τον Μάρτιο του 2026.

> [!IMPORTANT]  
> **Αυτό το εργαστήριο έχει ενσωματωμένο έναν οδηγό εργαστηρίου που μπορείτε να προεπισκεφθείτε στον τοπικό σας περιηγητή. Για να ξεκινήσετε, ανοίξτε το αποθετήριο στο GitHub Codespaces, σε ένα dev container ή σε ένα τοπικό αντίγραφο με εγκατεστημένο το MkDocs—έπειτα περιμένετε μέχρι να δείτε ένα ενεργό τερματικό VS Code και πληκτρολογήστε:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Θα πρέπει να δείτε ένα αναδυόμενο παράθυρο για να ανοίξετε τη σελίδα προεπισκόπησης σε έναν περιηγητή.**

Καλώς ήρθατε στο πρακτικό εργαστήριο για την εκμάθηση του Azure Developer CLI (AZD) με έμφαση στην ανάπτυξη εφαρμογών AI. Αυτό το εργαστήριο σας βοηθά να αποκτήσετε μια εφαρμοσμένη κατανόηση των προτύπων AZD σε 3 βήματα:

1. **Ανακάλυψη** - βρείτε το πρότυπο που είναι κατάλληλο για εσάς.
1. **Ανάπτυξη** - αναπτύξτε και επαληθεύστε ότι λειτουργεί
1. **Προσαρμογή** - τροποποιήστε και επαναλάβετε για να το κάνετε δικό σας!

Κατά τη διάρκεια αυτού του εργαστηρίου, θα εισαχθείτε επίσης σε βασικά εργαλεία και ροές εργασίας για προγραμματιστές, για να σας βοηθήσουν να απλοποιήσετε το ταξίδι ανάπτυξης από την αρχή μέχρι το τέλος.

## Επισκόπηση Εργαστηρίου

**Διάρκεια:** 3-4 ώρες  
**Επίπεδο:** Από αρχάριο έως μεσαίο  
**Προαπαιτούμενα:** Εξοικείωση με το Azure, έννοιες AI, VS Code & εργαλεία γραμμής εντολών.

Αυτό είναι ένα πρακτικό εργαστήριο όπου μαθαίνετε κάνοντας. Μόλις ολοκληρώσετε τις ασκήσεις, προτείνουμε να αναθεωρήσετε το πρόγραμμα AZD For Beginners για να συνεχίσετε το ταξίδι μάθησής σας προς βέλτιστες πρακτικές Ασφάλειας και Παραγωγικότητας.

| Time| Module  | Objective |
|:---|:---|:---|
| 15 mins | Introduction | Set the stage, understand the goals |
| 30 mins | Select AI Template | Explore options and pick starter | 
| 30 mins | Validate AI Template | Deploy default solution to Azure |
| 30 mins | Deconstruct AI Template | Explore structure and configuration |
| 30 mins | Configure AI Template | Activate and try available features |
| 30 mins | Customize AI Template | Adapt the template to your needs |
| 30 mins | Teardown Infrastructure | Cleanup and release resources |
| 15 mins | Wrap-Up & Next Steps | Learning resources, Workshop challenge |
| | |

## Τι θα μάθετε

Σκεφτείτε το Πρότυπο AZD ως έναν χώρο άσκησης για να εξερευνήσετε διάφορες δυνατότητες και εργαλεία για ανάπτυξη από άκρο σε άκρο στο Microsoft Foundry. Στο τέλος αυτού του εργαστηρίου, θα πρέπει να έχετε μια διαισθητική αίσθηση για διάφορα εργαλεία και έννοιες σε αυτό το πλαίσιο.

| Concept  | Objective |
|:---|:---|
| **Azure Developer CLI** | Κατανόηση εντολών εργαλείου και ροών εργασίας|
| **AZD Templates**| Κατανόηση δομής έργου και ρυθμίσεων|
| **Azure AI Agent**| Διάθεση και ανάπτυξη έργου Microsoft Foundry  |
| **Azure AI Search**| Ενεργοποίηση μηχανικής συμφραζομένων με agents |
| **Observability**| Εξερεύνηση ιχνηλάτησης, παρακολούθησης και αξιολογήσεων |
| **Red Teaming**| Εξέταση αντιπαραθετικών δοκιμών και μετριασμών |
| | |
