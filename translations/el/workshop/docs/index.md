# AZD για Προγραμματιστές AI Εργαστήριο

> [!IMPORTANT]  
> **Αυτό το εργαστήριο διαθέτει έναν οδηγό εργαστηρίου που μπορείτε να προεπισκόπησετε στον τοπικό σας περιηγητή. Για να ξεκινήσετε, πρέπει να ξεκινήσετε το GitHub Codespaces στο αποθετήριο — στη συνέχεια περιμένετε μέχρι να δείτε ένα ενεργό τερματικό VS Code και πληκτρολογήστε:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Θα πρέπει να δείτε ένα αναδυόμενο παράθυρο για να ανοίξετε τη σελίδα προεπισκόπησης σε έναν περιηγητή.**

Καλωσορίσατε στο πρακτικό εργαστήριο για την εκμάθηση του Azure Developer CLI (AZD) με έμφαση στην ανάπτυξη εφαρμογών AI. Αυτό το εργαστήριο σας βοηθά να αποκτήσετε πρακτική κατανόηση των προτύπων AZD σε 3 βήματα:

1. **Ανακάλυψη** - βρείτε το πρότυπο που σας ταιριάζει.
1. **Ανάπτυξη** - αναπτύξτε και επαληθεύστε ότι λειτουργεί
1. **Προσαρμογή** - τροποποιήστε και επαναλάβετε για να το κάνετε δικό σας!

Κατά τη διάρκεια αυτού του εργαστηρίου, θα γνωρίσετε επίσης βασικά εργαλεία και ροές εργασίας για προγραμματιστές, ώστε να απλοποιήσετε το ταξίδι ανάπτυξης από άκρο σε άκρο.

## Επισκόπηση Εργαστηρίου

**Διάρκεια:** 3-4 ώρες  
**Επίπεδο:** Αρχάριοι έως Ενδιάμεσοι  
**Προαπαιτούμενα:** Εξοικείωση με το Azure, έννοιες AI, VS Code & εργαλεία γραμμής εντολών.

Πρόκειται για ένα πρακτικό εργαστήριο όπου μαθαίνετε κάνοντας. Μόλις ολοκληρώσετε τις ασκήσεις, προτείνουμε να αναθεωρήσετε το πρόγραμμα AZD για Αρχάριους για να συνεχίσετε το ταξίδι μάθησής σας σε βέλτιστες πρακτικές Ασφαλείας και Παραγωγικότητας.

| Χρόνος| Ενότητα  | Στόχος |
|:---|:---|:---|
| 15 mins | Εισαγωγή | Θέστε το πλαίσιο, κατανοήστε τους στόχους |
| 30 mins | Επιλογή Προτύπου AI | Εξερευνήστε επιλογές και επιλέξτε αρχικό πρότυπο | 
| 30 mins | Επαλήθευση Προτύπου AI | Αναπτύξτε την προεπιλεγμένη λύση στο Azure |
| 30 mins | Αποδόμηση Προτύπου AI | Εξερευνήστε δομή και ρυθμίσεις |
| 30 mins | Διαμόρφωση Προτύπου AI | Ενεργοποιήστε και δοκιμάστε διαθέσιμες λειτουργίες |
| 30 mins | Προσαρμογή Προτύπου AI | Προσαρμόστε το πρότυπο στις ανάγκες σας |
| 30 mins | Αποσυναρμολόγηση Υποδομής | Καθαρισμός και απελευθέρωση πόρων |
| 15 mins | Σύνοψη & Επόμενα Βήματα | Πόροι μάθησης, Πρόκληση εργαστηρίου |
| | |

## Τι Θα Μάθετε

Σκεφτείτε το Πρότυπο AZD ως έναν εκπαιδευτικό χώρο (sandbox) για να εξερευνήσετε διάφορες δυνατότητες και εργαλεία για ανάπτυξη από άκρο σε άκρο στο Microsoft Foundry. Στο τέλος αυτού του εργαστηρίου, θα πρέπει να έχετε μια διαισθητική αίσθηση για διάφορα εργαλεία και έννοιες σε αυτό το πλαίσιο.

| Έννοια  | Στόχος |
|:---|:---|
| **Azure Developer CLI** | Κατανοήστε τις εντολές του εργαλείου και τις ροές εργασίας|
| **Πρότυπα AZD**| Κατανοήστε τη δομή του έργου και τη διαμόρφωση|
| **Azure AI Agent**| Παροχή και ανάπτυξη έργου Microsoft Foundry  |
| **Azure AI Search**| Ενεργοποιήστε μηχανική συμφραζομένων με πράκτορες |
| **Παρατηρησιμότητα**| Εξερευνήστε ιχνηλάτηση, παρακολούθηση και αξιολογήσεις |
| **Red Teaming**| Εξερευνήστε επιθετικές δοκιμές και αντίμετρα |
| | |
