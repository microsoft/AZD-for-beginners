# Εργαστήριο AZD για Προγραμματιστές Τεχνητής Νοημοσύνης

> Επικυρωμένο με την έκδοση `azd 1.27.1` τον Ιούλιο του 2026.

> [!IMPORTANT]  
> **Αυτό το εργαστήριο είναι εξοπλισμένο με έναν οδηγό εργαστηρίου που μπορείτε να προβάλετε στον τοπικό σας περιηγητή. Για να ξεκινήσετε, ανοίξτε το αποθετήριο στο GitHub Codespaces, ένα dev container, ή ένα τοπικό αντίγραφο με εγκατεστημένο το MkDocs—και στη συνέχεια περιμένετε μέχρι να δείτε ένα ενεργό τερματικό VS Code και πληκτρολογήστε:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Θα πρέπει να δείτε ένα αναδυόμενο παράθυρο για να ανοίξετε τη σελίδα προεπισκόπησης σε περιηγητή.**

Καλώς ήρθατε στο πρακτικό εργαστήριο εκμάθησης του Azure Developer CLI (AZD) με έμφαση στην ανάπτυξη εφαρμογών Τεχνητής Νοημοσύνης. Αυτό το εργαστήριο σας βοηθά να αποκτήσετε πρακτική κατανόηση των υποδειγμάτων AZD σε 3 βήματα:

1. **Ανακάλυψη** - βρείτε το υπόδειγμα που σας ταιριάζει.
1. **Ανάπτυξη** - αναπτύξτε και επικυρώστε ότι λειτουργεί
1. **Προσαρμογή** - τροποποιήστε και επαναλάβετε για να το κάνετε δικό σας!

Κατά τη διάρκεια αυτού του εργαστηρίου, θα σας παρουσιαστούν επίσης βασικά εργαλεία και ροές εργασίας προγραμματιστών, για να βοηθήσουν στο να απλοποιήσετε το πλήρες ταξίδι ανάπτυξης.

## Επισκόπηση Εργαστηρίου

**Διάρκεια:** 3-4 ώρες  
**Επίπεδο:** Αρχάριος έως Μεσαίο  
**Προαπαιτούμενα:** Εξοικείωση με Azure, έννοιες Τεχνητής Νοημοσύνης, VS Code & εργαλεία γραμμής εντολών.

Αυτό είναι ένα πρακτικό εργαστήριο όπου μαθαίνετε κάνοντας. Μόλις ολοκληρώσετε τις ασκήσεις, προτείνουμε να αναθεωρήσετε το πρόγραμμα σπουδών AZD για Αρχάριους για να συνεχίσετε το ταξίδι μάθησής σας σε βέλτιστες πρακτικές Ασφαλείας και Παραγωγικότητας.

| Χρόνος| Ενότητα  | Στόχος |
|:---|:---|:---|
| 15 λεπτά | Εισαγωγή | Θέτουμε το πλαίσιο, κατανοούμε τους στόχους |
| 30 λεπτά | Επιλογή Υποδείγματος AI | Εξερευνήστε επιλογές και διαλέξτε ένα αρχικό | 
| 30 λεπτά | Επικύρωση Υποδείγματος AI | Αναπτύξτε την προεπιλεγμένη λύση στο Azure |
| 30 λεπτά | Αποδόμηση Υποδείγματος AI | Εξερευνήστε τη δομή και τις ρυθμίσεις |
| 30 λεπτά | Διαμόρφωση Υποδείγματος AI | Ενεργοποιήστε και δοκιμάστε τις διαθέσιμες λειτουργίες |
| 30 λεπτά | Προσαρμογή Υποδείγματος AI | Προσαρμόστε το υπόδειγμα στις ανάγκες σας |
| 30 λεπτά | Αποσυναρμολόγηση Υποδομής | Καθαρισμός και απελευθέρωση πόρων |
| 15 λεπτά | Ολοκλήρωση & Επόμενα Βήματα | Πόροι μάθησης, πρόκληση εργαστηρίου |
| | |

## Τι Θα Μάθετε

Σκεφτείτε το Υπόδειγμα AZD ως ένα sandbox μάθησης για να διερευνήσετε διάφορες δυνατότητες και εργαλεία για συνολική ανάπτυξη στο Microsoft Foundry. Μέχρι το τέλος αυτού του εργαστηρίου, θα πρέπει να έχετε μια διαισθητική αίσθηση για τα διάφορα εργαλεία και έννοιες σε αυτό το πλαίσιο.

| Έννοια  | Στόχος |
|:---|:---|
| **Azure Developer CLI** | Κατανόηση εντολών εργαλείων και ροών εργασίας |
| **Υποδείγματα AZD**| Κατανόηση δομής έργου και ρύθμισης |
| **Azure AI Agent**| Παροχή & ανάπτυξη έργου Microsoft Foundry  |
| **Azure AI Search**| Ενεργοποίηση μηχανικής συμφραζομένων με agents |
| **Παρατηρησιμότητα**| Εξερεύνηση ιχνηλάτησης, παρακολούθησης και αξιολογήσεων |
| **Red Teaming**| Εξερεύνηση δοκιμών αντιπαράθεσης και αντιμετώπισης |
