## 2. Κύριες Έννοιες που Κατακτήσατε

### Azure Developer CLI (azd)
- `azd init` - Αρχικοποίηση έργων από πρότυπα
- `azd up` - Προμήθεια και ανάπτυξη με μία εντολή
- `azd env` - Διαχείριση μεταβλητών περιβάλλοντος
- `azd down` - Καθαρισμός πόρων με ασφάλεια

### AZD Template Structure
- `azure.yaml` - Διαμόρφωση και deployment hooks
- `infra/` - Bicep υποδομή-ως-κώδικα
- `src/` - Πηγαίος κώδικας εφαρμογής
- `scripts/` - Σενάρια αυτοματοποίησης και εγκατάστασης

### Microsoft Foundry Capabilities
- Υπηρεσία AI Agent για έξυπνους βοηθούς
- Αναζήτηση αρχείων και Azure AI Search για ανάκτηση γνώσης
- Ενσωματωμένη ιχνηλασία, παρακολούθηση και αξιολόγηση
- Red teaming για επικύρωση ασφάλειας AI

## 4. Πρόκληση Εργαστηρίου

Έτοιμοι να δοκιμάσετε τις ικανότητές σας; Δοκιμάστε αυτές τις προκλήσεις:

| Challenge | Description | Difficulty |
|:---|:---|:---|
| **1. Σενάριο Πολλαπλών Πρακτόρων** | Αναπτύξτε και προσαρμόστε για άλλο κλάδο (π.χ., υγεία, οικονομικά) | ⭐⭐ |
| **2. Προσαρμοσμένη Βάση Γνώσης** | Προσθέστε τα δικά σας αρχεία δεδομένων και δημιουργήστε ένα προσαρμοσμένο ευρετήριο αναζήτησης | ⭐⭐⭐ |
| **3. Παραγωγική Ανάπτυξη** | Διαμορφώστε CI/CD με GitHub Actions χρησιμοποιώντας `azd pipeline config` | ⭐⭐⭐⭐ |
