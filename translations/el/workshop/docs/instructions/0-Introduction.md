# AZD Για Προγραμματιστές AI

!!! info "ΚΟΙΝΟ ΓΙΑ ΑΥΤΟ ΤΟ ΕΡΓΑΣΤΗΡΙΟ"
   
    Αυτό το εργαστήριο απευθύνεται σε οποιονδήποτε επιθυμεί να δημιουργήσει εφαρμογές AI Agentic στο Microsoft Foundry αξιοποιώντας κορυφαία εργαλεία και ροές εργασίας. Το εργαστήριο είναι κατάλληλο τόσο για αρχάριους όσο και για έμπειρους προγραμματιστές που έχουν κάποια εξοικείωση με το Azure, τις έννοιες AI και τον κώδικα Python. Ελέγξτε τις προαπαιτήσεις πριν ξεκινήσετε.

    **Διάρκεια:** 3-4 ώρες  
    **Επίπεδο:** Από αρχάριο έως ενδιάμεσο  

---


## Προαπαιτούμενα

- Ένας προσωπικός λογαριασμός GitHub → [Δημιουργήστε έναν δωρεάν λογαριασμό](https://github.com/signup)
- Ένας ενεργός λογαριασμός Azure → [Δημιουργήστε έναν δωρεάν λογαριασμό](https://aka.ms/free)
- Εξοικείωση με έννοιες AI, ανάπτυξη σε Python, εργαλεία γραμμής εντολών
- Εμπειρία με το VS Code και τα εργαλεία και τις ροές εργασίας του GitHub.

---

## Στόχοι Εκμάθησης

Στο τέλος αυτού του εργαστηρίου, θα γνωρίζετε πώς να:

- [X] **Work with Azure Developer CLI** → για την παροχή, ανάπτυξη και διαχείριση των πόρων Azure σας.
- [X] **Use AI App Templates** →  για να ξεκινήσετε γρήγορα έργα AI Agent στο Microsoft Foundry.
- [X] **Configure Templates** →  για λειτουργίες όπως Red Teaming, Evaluations, Tracing & Monitoring
- [X] **Deconstruct and Explore** →  τα πρότυπα και τις ροές εργασίας AZD χρησιμοποιώντας το GitHub Copilot
- [X] **Customize and Redeploy** →  το αρχικό πρότυπο AI Agent, με τις προσαρμοσμένες απαιτήσεις σας

---

## Δομή Εργαστηρίου

Το εργαστήριο δομείται σε ενότητες, καθεμία με μια πρακτική άσκηση εργαστηρίου:

1. **Introduction** - Κατανοήστε τι είναι τα πρότυπα AZD και πώς λειτουργούν
1. **Selection** - Ανακαλύψτε το κατάλληλο "starter" πρότυπο για το επόμενο έργο AI σας
1. **Validation** - Αναπτύξτε και εξερευνήστε το πρότυπο για να επαληθεύσετε ότι λειτουργεί όπως διαφημίζεται
1. **Deconstruction** - Αποδομήστε το πρότυπο για να κατανοήσετε τα συστατικά & τη διαμόρφωσή του
1. **Configuration** - Μάθετε να ενεργοποιείτε και να χρησιμοποιείτε διάφορες δυνατότητες του Microsoft Foundry
1. **Customization** - Ανακατασκευάστε την εφαρμογή με τα δεδομένα και τις απαιτήσεις σας κατά νου
1. **Teardown** - Μάθετε πώς να απεγκαθιστάτε την υποδομή και να ανακτάτε πόρους που έχουν διαγραφεί προσωρινά

---

## Ανατροφοδότηση & Ζητήματα

Ενημερώνουμε ενεργά το πρόγραμμα σπουδών για να διασφαλίσουμε ότι παρέχει την καλύτερη εμπειρία μάθησης για αρχάριους. Η ανατροφοδότησή σας εκτιμάται.

1. Δημιουργήστε ένα issue σε αυτό το αποθετήριο - επισημάνετέ το με `Workshop` για ευκολία.
