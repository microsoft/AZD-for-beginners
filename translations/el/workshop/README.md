# Εργαστήριο AZD για Προγραμματιστές AI

Καλώς ήρθατε στο πρακτικό εργαστήριο για την εκμάθηση του Azure Developer CLI (AZD) με έμφαση στην ανάπτυξη εφαρμογών AI. Αυτό το εργαστήριο θα σας βοηθήσει να αποκτήσετε μια εφαρμοσμένη κατανόηση των προτύπων AZD σε 3 βήματα:

1. **Ανακάλυψη** - βρείτε το πρότυπο που σας ταιριάζει.
1. **Ανάπτυξη** - αναπτύξτε και επαληθεύστε ότι λειτουργεί
1. **Προσαρμογή** - τροποποιήστε και επαναλάβετε για να το κάνετε δικό σας!

Κατά τη διάρκεια αυτού του εργαστηρίου, θα σας παρουσιαστούν επίσης βασικά εργαλεία και ροές εργασίας για προγραμματιστές, για να βοηθήσουν στην απλοποίηση του end-to-end ταξιδιού ανάπτυξής σας.

## Τι θα Μάθετε

Σκεφτείτε το Πρότυπο AZD ως έναν χώρο εκμάθησης για να εξερευνήσετε διάφορες δυνατότητες και εργαλεία για end-to-end ανάπτυξη στο Microsoft Foundry. Μέχρι το τέλος αυτού του εργαστηρίου, θα πρέπει να έχετε μια διαισθητική αίσθηση για διάφορα εργαλεία και έννοιες σε αυτό το πλαίσιο.

| Concept  | Objective |
|:---|:---|
| **Azure Developer CLI** | Κατανόηση των εντολών του εργαλείου και των ροών εργασίας |
| **AZD Templates**| Κατανόηση της δομής του έργου και της διαμόρφωσης |
| **Azure AI Agent**| Παροχή & ανάπτυξη έργου Microsoft Foundry  |
| **Azure AI Search**| Ενεργοποίηση context engineering με agents |
| **Observability**| Εξερευνήστε tracing, monitoring και αξιολογήσεις |
| **Red Teaming**| Εξερευνήστε αντιπαραβατικές δοκιμές και μετριασμούς |

## Δομή Εργαστηρίου

Το εργαστήριο δομείται ώστε να σας οδηγήσει σε ένα ταξίδι από την ανακάλυψη προτύπου, στην ανάπτυξη, στην αποδόμηση και την προσαρμογή - χρησιμοποιώντας ως βάση το επίσημο [Getting Started with AI Agents](https://github.com/Azure-Samples/get-started-with-ai-agents) starter template.

### [Ενότητα 1: Επιλογή Προτύπου AI](docs/instructions/1-Select-AI-Template.md) (30 mins)

- Τι είναι τα Πρότυπα AI;
- Πού μπορώ να βρω Πρότυπα AI;
- Πώς μπορώ να ξεκινήσω να χτίζω AI Agents;
- **Εργαστήριο**: Γρήγορο ξεκίνημα σε Codespaces ή σε dev container

### [Ενότητα 2: Επαλήθευση Προτύπου AI](docs/instructions/2-Validate-AI-Template.md) (30 mins)

- Ποια είναι η Αρχιτεκτονική του Προτύπου AI;
- Ποια είναι η Ροή Εργασίας Ανάπτυξης AZD;
- Πώς μπορώ να λάβω βοήθεια με την ανάπτυξη AZD;
- **Εργαστήριο**: Ανάπτυξη & Επαλήθευση του προτύπου AI Agents

### [Ενότητα 3: Αποδόμηση Προτύπου AI](docs/instructions/3-Deconstruct-AI-Template.md) (30 mins)

- Εξερευνήστε το περιβάλλον σας στο `.azure/` 
- Εξερευνήστε τη ρύθμιση πόρων σας στο `infra/` 
- Εξερευνήστε τη διαμόρφωση AZD σας στα `azure.yaml`s
- **Εργαστήριο**: Τροποποιήστε Μεταβλητές Περιβάλλοντος & Επανααναπτύξτε

### [Ενότητα 4: Διαμόρφωση Προτύπου AI](docs/instructions/4-Configure-AI-Template.md) (30 mins)
- Εξερευνήστε: Retrieval Augmented Generation
- Εξερευνήστε: Αξιολόγηση Agent & Red Teaming
- Εξερευνήστε: Tracing & Monitoring
- **Εργαστήριο**: Εξερευνήστε τον AI Agent + Observability 

### [Ενότητα 5: Προσαρμογή Προτύπου AI](docs/instructions/5-Customize-AI-Template.md) (30 mins)
- Ορίστε: PRD (Έγγραφο Απαιτήσεων Προϊόντος) με Απαιτήσεις Σεναρίου
- Διαμορφώστε: Μεταβλητές Περιβάλλοντος για AZD
- Εφαρμόστε: Lifecycle Hooks για επιπλέον εργασίες
- **Εργαστήριο**: Προσαρμόστε το πρότυπο για το σενάριό μου

### [Ενότητα 6: Αποσυναρμολόγηση Υποδομής](docs/instructions/6-Teardown-Infrastructure.md) (30 mins)
- Επανάληψη: Τι είναι τα Πρότυπα AZD;
- Επανάληψη: Γιατί να χρησιμοποιήσετε το Azure Developer CLI;
- Επόμενα Βήματα: Δοκιμάστε ένα διαφορετικό πρότυπο!
- **Εργαστήριο**: Αφαίρεση πόρων υποδομής & καθαρισμός

## Πρόκληση Εργαστηρίου

Θέλετε να προκαλέσετε τον εαυτό σας για περισσότερα; Εδώ είναι μερικές προτάσεις έργων - ή μοιραστείτε τις ιδέες σας μαζί μας!!

|:---|:---|

|2. **Προσαρμόστε για το δικό σας σενάριο**  | Δοκιμάστε να γράψετε ένα PRD (Έγγραφο Απαιτήσεων Προϊόντος) για ένα διαφορετικό σενάριο.
