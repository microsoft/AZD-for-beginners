# Εργαστήριο AZD για Προγραμματιστές AI

Καλώς ήρθατε στο πρακτικό εργαστήριο για να μάθετε το Azure Developer CLI (AZD) με έμφαση στην ανάπτυξη εφαρμογών AI. Αυτό το εργαστήριο θα σας βοηθήσει να αποκτήσετε μια εφαρμοσμένη κατανόηση των προτύπων AZD σε 3 βήματα:

1. **Ανακάλυψη** - βρείτε το πρότυπο που σας ταιριάζει.
1. **Ανάπτυξη** - αναπτύξτε και επαληθεύστε ότι λειτουργεί
1. **Προσαρμογή** - τροποποιήστε και επαναλάβετε για να το κάνετε δικό σας!

Κατά τη διάρκεια αυτού του εργαστηρίου, θα παρουσιαστούν επίσης βασικά εργαλεία και ροές εργασίας για προγραμματιστές, για να βοηθήσουν στη βελτιστοποίηση του end-to-end ταξιδιού ανάπτυξής σας.

<br/>

## Τι Θα Μάθετε

Σκεφτείτε το Πρότυπο AZD ως μια πλατφόρμα εκμάθησης για να εξερευνήσετε διάφορες δυνατότητες και εργαλεία για end-to-end ανάπτυξη στο Microsoft Foundry. Μέχρι το τέλος αυτού του εργαστηρίου, θα πρέπει να έχετε μια διαισθητική αντίληψη για διάφορα εργαλεία και έννοιες σε αυτό το πλαίσιο.

| Έννοια  | Στόχος |
|:---|:---|
| **Azure Developer CLI** | Κατανόηση εντολών εργαλείων και ροών εργασίας|
| **AZD Templates**| Κατανόηση δομής έργου και ρυθμίσεων|
| **Azure AI Agent**| Παροχή & ανάπτυξη έργου Microsoft Foundry  |
| **Azure AI Search**| Ενεργοποίηση μηχανικής συμφραζομένων με πράκτορες |
| **Observability**| Εξερευνήστε ιχνηλάτηση, παρακολούθηση και αξιολογήσεις |
| **Red Teaming**| Εξερευνήστε επιθετικές δοκιμές και μετριασμούς |

## Δομή Εργαστηρίου

Το εργαστήριο έχει δομηθεί για να σας μεταφέρει από την ανακάλυψη προτύπων, στην ανάπτυξη, την αποδόμηση και την προσαρμογή - χρησιμοποιώντας το επίσημο [Ξεκινώντας με AI Agents](https://github.com/Azure-Samples/get-started-with-ai-agents) starter template ως βάση.

### [Ενότητα 1: Επιλογή Προτύπου AI](docs/instructions/1-Select-AI-Template.md) (30 λεπτά)

- Τι είναι τα Πρότυπα AI;
- Πού μπορώ να βρω Πρότυπα AI;
- Πώς μπορώ να ξεκινήσω την κατασκευή AI Agents;
- **Εργαστήριο**: Γρήγορο ξεκίνημα με GitHub Codespaces

### [Ενότητα 2: Επαλήθευση Προτύπου AI](docs/instructions/2-Validate-AI-Template.md) (30 λεπτά)

- Ποια είναι η Αρχιτεκτονική του Προτύπου AI;
- Ποια είναι η Ροή Εργασίας Ανάπτυξης με AZD;
- Πώς μπορώ να λάβω βοήθεια για την ανάπτυξη με AZD;
- **Εργαστήριο**: Αναπτύξτε & Επαληθεύστε το πρότυπο AI Agents

### [Ενότητα 3: Αποδόμηση Προτύπου AI](docs/instructions/3-Deconstruct-AI-Template.md) (30 λεπτά)

- Εξερευνήστε το περιβάλλον σας στο `.azure/` 
- Εξερευνήστε τη ρύθμιση πόρων σας σε `infra/` 
- Εξερευνήστε τη διαμόρφωση AZD σας στα `azure.yaml`
- **Εργαστήριο**: Τροποποίηση μεταβλητών περιβάλλοντος & Επαναανάπτυξη

### [Ενότητα 4: Διαμόρφωση Προτύπου AI](docs/instructions/4-Configure-AI-Template.md) (30 λεπτά)
- Εξερευνήστε: Retrieval Augmented Generation
- Εξερευνήστε: Agent Evaluation & Red Teaming
- Εξερευνήστε: Tracing & Monitoring
- **Εργαστήριο**: Εξερευνήστε AI Agent + Observability 

### [Ενότητα 5: Προσαρμογή Προτύπου AI](docs/instructions/5-Customize-AI-Template.md) (30 λεπτά)
- Ορίστε: PRD με Απαιτήσεις Σεναρίου
- Διαμορφώστε: Μεταβλητές περιβάλλοντος για το AZD
- Υλοποιήστε: Lifecycle Hooks για επιπλέον εργασίες
- **Εργαστήριο**: Προσαρμόστε το πρότυπο για το σενάριό μου

### [Ενότητα 6: Κατάργηση Υποδομής](docs/instructions/6-Teardown-Infrastructure.md) (30 λεπτά)
- Επαναληπτικά: Τι είναι τα πρότυπα AZD;
- Επαναληπτικά: Γιατί να χρησιμοποιήσετε το Azure Developer CLI;
- Επόμενα Βήματα: Δοκιμάστε ένα διαφορετικό πρότυπο!
- **Εργαστήριο**: Αποπρομήθεια υποδομής & καθαρισμός

## Πρόκληση Εργαστηρίου

Θέλετε να δοκιμάσετε κάτι περισσότερο; Εδώ είναι μερικές προτάσεις έργων — ή μοιραστείτε τις ιδέες σας μαζί μας!!

| Έργο | Περιγραφή |
|:---|:---|
|1. **Αποδόμηση ενός Σύνθετου Προτύπου AI** | Χρησιμοποιήστε τη ροή εργασίας και τα εργαλεία που περιγράψαμε και δείτε αν μπορείτε να αναπτύξετε, να επαληθεύσετε και να προσαρμόσετε ένα διαφορετικό πρότυπο λύσης AI. _Τι μάθατε;_|
|2. **Προσαρμόστε στο Σενάριό σας**  | Δοκιμάστε να γράψετε ένα PRD (Έγγραφο Απαιτήσεων Προϊόντος) για ένα διαφορετικό σενάριο. Στη συνέχεια, χρησιμοποιήστε το GitHub Copilot στο αποθετήριο του προτύπου σας στο Agent Model - και ζητήστε του να δημιουργήσει μια ροή εργασίας προσαρμογής για εσάς. _Τι μάθατε; Πώς θα μπορούσατε να βελτιώσετε αυτές τις προτάσεις;_|
