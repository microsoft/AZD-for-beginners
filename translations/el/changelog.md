# Αρχείο αλλαγών - AZD Για Αρχάριους

## Εισαγωγή

Αυτό το αρχείο αλλαγών καταγράφει όλες τις σημαντικές αλλαγές, ενημερώσεις και βελτιώσεις στο αποθετήριο AZD Για Αρχάριους. Ακολουθούμε αρχές semantic versioning και διατηρούμε αυτό το αρχείο για να βοηθάμε τους χρήστες να κατανοήσουν τι έχει αλλάξει μεταξύ εκδόσεων.

## Στόχοι Μάθησης

Μελετώντας αυτό το αρχείο αλλαγών, θα:
- Παραμένετε ενημερωμένοι για νέες λειτουργίες και προσθήκες περιεχομένου
- Κατανοείτε τις βελτιώσεις που έγιναν στην υπάρχουσα τεκμηρίωση
- Παρακολουθείτε διορθώσεις σφαλμάτων και διορθώσεις για διασφάλιση ακρίβειας
- Ακολουθείτε την εξέλιξη του διδακτικού υλικού με την πάροδο του χρόνου

## Μαθησιακά Αποτελέσματα

Μετά την ανασκόπηση των εγγραφών του αρχείου αλλαγών, θα μπορείτε να:
- Αναγνωρίζετε νέο περιεχόμενο και πόρους διαθέσιμους για μάθηση
- Κατανοείτε ποιες ενότητες έχουν ενημερωθεί ή βελτιωθεί
- Σχεδιάζετε το μονοπάτι μάθησης σας βάσει του πιο ενημερωμένου υλικού
- Συνεισφέρετε σχόλια και προτάσεις για μελλοντικές βελτιώσεις

### Αναφορά αλλαγών
Κατά τη συνεισφορά σε αυτό το αποθετήριο, παρακαλώ βεβαιωθείτε ότι οι εγγραφές του changelog περιλαμβάνουν:

1. **Αριθμός Έκδοσης**: Σύμφωνα με semantic versioning (major.minor.patch)
2. **Ημερομηνία**: Ημερομηνία κυκλοφορίας ή ενημέρωσης σε μορφή YYYY-MM-DD
3. **Κατηγορία**: Added, Changed, Deprecated, Removed, Fixed, Security
4. **Σαφής Περιγραφή**: Συνοπτική περιγραφή του τι άλλαξε
5. **Αξιολόγηση Επιπτώσεων**: Πώς οι αλλαγές επηρεάζουν τους υπάρχοντες χρήστες

### Κατηγορίες αλλαγών

#### Προστέθηκε
- Νέες δυνατότητες, ενότητες τεκμηρίωσης ή ικανότητες
- Νέα παραδείγματα, πρότυπα ή εκπαιδευτικοί πόροι
- Πρόσθετα εργαλεία, scripts ή βοηθητικά προγράμματα

- Τροποποιήσεις σε υπάρχουσα λειτουργικότητα ή τεκμηρίωση
- Ενημερώσεις για βελτίωση σαφήνειας ή ακρίβειας
- Αναδιάρθρωση περιεχομένου ή οργάνωσης

#### Deprecated
- Δυνατότητες ή προσεγγίσεις που αποσύρονται σταδιακά
- Ενότητες τεκμηρίωσης προγραμματισμένες για αφαίρεση
- Μέθοδοι που έχουν καλύτερες εναλλακτικές

- Δυνατότητες, τεκμηρίωση ή παραδείγματα που δεν είναι πλέον σχετικά
- Παρωχημένες πληροφορίες ή προσεγγίσεις σε απόσυρση

- Επίλυση αναφερόμενων προβλημάτων

- Βελτιώσεις ή διορθώσεις σχετικές με την ασφάλεια
- Ενημερώσεις βέλτιστων πρακτικών ασφάλειας
