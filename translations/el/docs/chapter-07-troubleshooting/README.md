## Επισκόπηση

Αυτό το κεφάλαιο σας βοηθά να διαγνώσετε και να επιλύσετε κοινά προβλήματα όταν εργάζεστε με το Azure Developer CLI. Από αποτυχίες ανάπτυξης μέχρι προβλήματα ειδικά για AI.

> Επικυρώθηκε με `azd 1.25.6` τον Ιούνιο 2026.

## Στόχοι Μάθησης

Με την ολοκλήρωση αυτού του κεφαλαίου, θα:
- Διαγνώσετε κοινές αποτυχίες ανάπτυξης AZD
- Εντοπίσετε σφάλματα αυθεντικοποίησης και δικαιωμάτων
- Επιλύσετε προβλήματα συνδεσιμότητας υπηρεσιών AI
- Χρησιμοποιήσετε το Azure Portal και το CLI για αντιμετώπιση προβλημάτων

## 🚨 Γρήγορες Επιλύσεις

### Προβλήματα Αυθεντικοποίησης
```bash
# Απαραίτητο για τις ροές εργασίας του AZD
azd auth login

# Προαιρετικό εάν χρησιμοποιείτε επίσης απευθείας εντολές του Azure CLI
az login

azd auth status
```

### Σφάλματα Παροχής
```bash
azd show
azd monitor --logs
az deployment sub list --query "[?properties.provisioningState!='Succeeded']"
```

### Συγκρούσεις Πόρων
```bash
azd down --force --purge
azd env new different-name
azd up
```

### Υπέρβαση Ορίου
```bash
az vm list-usage --location eastus --output table
azd env set AZURE_LOCATION westus2
azd up
```

## 📋 Αναφορά Κωδικών Σφάλματος

| Error | Cause | Solution |
|-------|-------|----------|
| `AuthenticationError` | Δεν έχετε συνδεθεί | `azd auth login` |
| `ResourceNotFound` | Λείπει πόρος | Ελέγξτε τα ονόματα πόρων |
| `QuotaExceeded` | Όρια συνδρομής | Ζητήστε αύξηση ορίου |
| `InvalidTemplate` | Σφάλμα σύνταξης Bicep | `az bicep build` |
| `Conflict` | Ο πόρος υπάρχει | Χρησιμοποιήστε νέο όνομα ή διαγράψτε |
| `Forbidden` | Ανεπαρκή δικαιώματα | Ελέγξτε ρόλους RBAC |

## 🔄 Επαναφορά και Ανάκτηση

```bash
# Ήπια επαναφορά (διατήρηση πόρων, επανανάπτυξη του κώδικα)
azd deploy --force

# Σκληρή επαναφορά (διαγραφή όλων, ξεκινήστε από την αρχή)
azd down --force --purge
azd up
```
