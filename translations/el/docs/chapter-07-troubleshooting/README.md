## Επισκόπηση

Αυτό το κεφάλαιο σας βοηθά να διαγνώσετε και να επιλύσετε κοινά ζητήματα κατά τη χρήση του Azure Developer CLI. Από αποτυχίες ανάπτυξης μέχρι προβλήματα ειδικά για AI.

> Επικυρωμένο με `azd 1.27.1` τον Ιούλιο του 2026.

## Στόχοι Μάθησης

Με την ολοκλήρωση αυτού του κεφαλαίου, θα:
- Διαγνώσετε κοινές αποτυχίες ανάπτυξης AZD
- Αποσφαλματώσετε ζητήματα αυθεντικοποίησης και δικαιωμάτων
- Επίλυση προβλημάτων σύνδεσης υπηρεσιών AI
- Χρησιμοποιήσετε το Azure Portal και την CLI για επίλυση προβλημάτων

## 🚨 Γρήγορες Διορθώσεις

### Ζητήματα Αυθεντικοποίησης
```bash
# Απαιτείται για τις ροές εργασίας AZD
azd auth login

# Προαιρετικό αν χρησιμοποιείτε επίσης απευθείας εντολές Azure CLI
az login

azd auth status
```

### Αποτυχίες Παροχής Πόρων
```bash
azd show
azd monitor --logs
az deployment sub list --query "[?properties.provisioningState!='Succeeded']"
```

### Συγκρούσεις Πόρων
```bash
azd down --force --purge
azd env new different-name
azd up
```

### Υπέρβαση Ποσόστωσης
```bash
az vm list-usage --location eastus --output table
azd env set AZURE_LOCATION westus2
azd up
```

## 📋 Αναφορά Κωδικών Σφάλματος

| Σφάλμα | Αιτία | Λύση |
|-------|-------|----------|
| `AuthenticationError` | Δεν έχετε συνδεθεί | `azd auth login` |
| `ResourceNotFound` | Ελλείπει πόρος | Ελέγξτε τα ονόματα πόρων |
| `QuotaExceeded` | Όρια συνδρομής | Ζητήστε αύξηση ποσόστωσης |
| `InvalidTemplate` | Σφάλμα σύνταξης Bicep | `az bicep build` |
| `Conflict` | Ο πόρος υπάρχει | Χρησιμοποιήστε νέο όνομα ή διαγράψτε |
| `Forbidden` | Ανεπαρκή δικαιώματα | Ελέγξτε τους ρόλους RBAC |

## 🔄 Επαναφορά και Ανάκτηση

```bash
# Μαλακή επαναφορά (διατήρηση πόρων, επανατοποθέτηση κώδικα)
azd deploy --force

# Σκληρή επαναφορά (διαγραφή όλων, εκκίνηση από την αρχή)
azd down --force --purge
azd up
```
