# Κεφάλαιο 1: Θεμέλιο & Γρήγορη Εκκίνηση

**📚 Μαθήματα**: [AZD Για Αρχάριους](../../README.md) | **⏱️ Διάρκεια**: 30-45 λεπτά | **⭐ Πολυπλοκότητα**: Αρχάριος

## Επισκόπηση

Αυτό το κεφάλαιο εισάγει τις βασικές αρχές του Azure Developer CLI (azd). Θα μάθετε βασικές έννοιες, θα εγκαταστήσετε τα εργαλεία και θα αναπτύξετε την πρώτη σας εφαρμογή στο Azure.

> Επικυρώθηκε με `azd 1.27.1` τον Ιούλιο του 2026.

## Στόχοι Μάθησης

Με την ολοκλήρωση αυτού του κεφαλαίου, θα:
- Κατανοήσετε τι είναι το Azure Developer CLI και πώς διαφέρει από το Azure CLI
- Εγκαταστήσετε και διαμορφώσετε το AZD στην πλατφόρμα σας
- Αναπτύξετε την πρώτη σας εφαρμογή στο Azure με `azd up`
- Καθαρίσετε πόρους με `azd down`

## ✅ Ξεκινήστε Εδώ: Επικυρώστε τη Ρύθμισή Σας

Πριν ξεκινήσετε, επιβεβαιώστε ότι ο τοπικός σας υπολογιστής είναι έτοιμος για το πρότυπο του Κεφαλαίου 1:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

Αν το script αναφέρει ότι λείπουν εργαλεία, διορθώστε αυτά πρώτα και μετά συνεχίστε με το κεφάλαιο.

---

## 🚀 Γρήγορη Εκκίνηση

```bash
# Ελέγξτε την εγκατάσταση
azd version

# Πιστοποίηση για AZD
# Προαιρετικό: az login εάν σκοπεύετε να εκτελέσετε εντολές Azure CLI απευθείας
azd auth login

# Ανάπτυξη της πρώτης σας εφαρμογής
azd init --template todo-nodejs-mongo
azd up

# Καθαρίστε όταν τελειώσετε
azd down --force --purge
```

---

## ✅ Κριτήρια Επιτυχίας

Μετά την ολοκλήρωση αυτού του κεφαλαίου, θα πρέπει να μπορείτε να:

```bash
azd version              # Εμφανίζει την εγκατεστημένη έκδοση
azd init --template todo-nodejs-mongo  # Αρχικοποιεί το έργο
azd up                   # Αναπτύσσει στο Azure
azd show                 # Εμφανίζει το URL της τρέχουσας εφαρμογής
azd down --force --purge # Καθαρίζει τους πόρους
```
