# Κεφάλαιο 1: Θεμέλια & Γρήγορη Εκκίνηση

**📚 Course**: [AZD Για Αρχάριους](../../README.md) | **⏱️ Διάρκεια**: 30-45 λεπτά | **⭐ Πολυπλοκότητα**: Αρχάριος

## Επισκόπηση

Αυτό το κεφάλαιο εισάγει τα βασικά του Azure Developer CLI (azd). Θα μάθετε βασικές έννοιες, θα εγκαταστήσετε τα εργαλεία και θα αναπτύξετε την πρώτη σας εφαρμογή στο Azure.

> Επαληθεύτηκε με `azd 1.23.12` τον Μάρτιο 2026.

## Στόχοι Μάθησης

Με την ολοκλήρωση αυτού του κεφαλαίου, θα:
- Κατανοήσετε τι είναι το Azure Developer CLI και πώς διαφέρει από το Azure CLI
- Εγκαταστήσετε και ρυθμίσετε το AZD στην πλατφόρμα σας
- Αναπτύξετε την πρώτη σας εφαρμογή στο Azure με `azd up`
- Καθαρίσετε τους πόρους με `azd down`

## ✅ Ξεκινήστε Εδώ: Επαληθεύστε τη Ρύθμισή σας

Πριν ξεκινήσετε, επιβεβαιώστε ότι ο τοπικός σας υπολογιστής είναι έτοιμος για το πρότυπο του Κεφαλαίου 1:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

Εάν το script αναφέρει ότι λείπουν εργαλεία, διορθώστε πρώτα αυτά και μετά συνεχίστε με το κεφάλαιο.

---

## 🚀 Γρήγορη Εκκίνηση

```bash
# Ελέγξτε την εγκατάσταση
azd version

# Αυθεντικοποιηθείτε για το AZD
# Προαιρετικό: az login εάν σκοπεύετε να εκτελέσετε εντολές του Azure CLI απευθείας
azd auth login

# Αναπτύξτε την πρώτη σας εφαρμογή
azd init --template todo-nodejs-mongo
azd up

# Καθαρίστε όταν τελειώσετε
azd down --force --purge
```

---

## ✅ Κριτήρια Επιτυχίας

Μετά την ολοκλήρωση αυτού του κεφαλαίου, θα πρέπει να μπορείτε να:

```bash
azd version              # Εμφανίζει την εγκατεστημένη έκδοση
azd init --template todo-nodejs-mongo  # Αρχικοποιεί το έργο
azd up                   # Αναπτύσσει στο Azure
azd show                 # Εμφανίζει το URL της τρέχουσας εφαρμογής
azd down --force --purge # Καθαρίζει τους πόρους
```
