# Κεφάλαιο 1: Θεμέλια & Γρήγορη Εκκίνηση

**📚 Μάθημα**: [AZD For Beginners](../../README.md) | **⏱️ Διάρκεια**: 30-45 λεπτά | **⭐ Δυσκολία**: Αρχάριος

## Επισκόπηση

Αυτό το κεφάλαιο εισάγει τα θεμέλια του Azure Developer CLI (azd). Θα μάθετε τις βασικές έννοιες, θα εγκαταστήσετε τα εργαλεία και θα αναπτύξετε την πρώτη σας εφαρμογή στο Azure.

> Ελεγμένο με `azd 1.25.6` τον Ιούνιο του 2026.

## Στόχοι Μάθησης

Με την ολοκλήρωση αυτού του κεφαλαίου, θα:
- Κατανοείτε τι είναι το Azure Developer CLI και πώς διαφέρει από το Azure CLI
- Εγκαταστήσετε και ρυθμίσετε το AZD στην πλατφόρμα σας
- Αναπτύξετε την πρώτη σας εφαρμογή στο Azure με `azd up`
- Καθαρίσετε πόρους με `azd down`

Πριν ξεκινήσετε, επιβεβαιώστε ότι ο τοπικός σας υπολογιστής είναι έτοιμος για το πρότυπο του Κεφαλαίου 1:

**Windows:**
```powershell

```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

Εάν το σενάριο αναφέρει ότι λείπουν εργαλεία, διορθώστε πρώτα αυτά και στη συνέχεια συνεχίστε με το κεφάλαιο.

## 🚀 Γρήγορη Εκκίνηση

```bash
# Ελέγξτε την εγκατάσταση
azd version

# Πιστοποιηθείτε για το AZD
# Προαιρετικό: az login αν σκοπεύετε να εκτελέσετε εντολές του Azure CLI απευθείας
azd auth login

# Αναπτύξτε την πρώτη σας εφαρμογή
azd init --template todo-nodejs-mongo
azd up

# Καθαρίστε όταν τελειώσετε
azd down --force --purge
```

## ✅ Κριτήρια Επιτυχίας

Μετά την ολοκλήρωση αυτού του κεφαλαίου, θα πρέπει να μπορείτε να:

```bash
azd version              # Εμφανίζει την εγκατεστημένη έκδοση
azd init --template todo-nodejs-mongo  # Αρχικοποιεί το έργο
azd up                   # Αναπτύσσει στο Azure
azd show                 # Εμφανίζει το URL της εφαρμογής που εκτελείται
azd down --force --purge # Καθαρίζει τους πόρους
```
