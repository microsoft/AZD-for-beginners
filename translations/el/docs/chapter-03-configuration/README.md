# Κεφάλαιο 3: Διαμόρφωση & Αυθεντικοποίηση

**📚 Μάθημα**: [AZD για Αρχάριους](../../README.md) | **⏱️ Διάρκεια**: 45-60 λεπτά | **⭐ Πολυπλοκότητα**: Ενδιάμεσο

## Επισκόπηση

Αυτό το κεφάλαιο καλύπτει τη διαμόρφωση περιβάλλοντος, πρότυπα αυθεντικοποίησης και βέλτιστες πρακτικές ασφάλειας για αναπτύξεις με το Azure Developer CLI.

> Επαληθεύθηκε έναντι `azd 1.23.12` τον Μάρτιο του 2026.

## Στόχοι Μάθησης

Με την ολοκλήρωση αυτού του κεφαλαίου, θα:
- Κατανοήσετε την ιεραρχία διαμόρφωσης του AZD
- Διαχειρίζεστε πολλαπλά περιβάλλοντα (dev, staging, prod)
- Εφαρμόζετε ασφαλή αυθεντικοποίηση με διαχειριζόμενες ταυτότητες
- Διαμορφώνετε ρυθμίσεις ανά περιβάλλον

## 🚀 Γρήγορη Εκκίνηση

```bash
# Δημιουργία πολλαπλών περιβαλλόντων
azd env new dev
azd env new staging
azd env new prod

# Εναλλαγή περιβαλλόντων
azd env select prod

# Ρύθμιση μεταβλητών περιβάλλοντος
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# Προβολή ρυθμίσεων
azd env get-values
```

---

## 🔧 Ιεραρχία Ρυθμίσεων

AZD εφαρμόζει ρυθμίσεις με αυτή τη σειρά (τα μεταγενέστερα αντικαθιστούν τα προηγούμενα):

1. **Default values** (built into templates)
2. **azure.yaml** (διαμόρφωση έργου)
3. **Μεταβλητές περιβάλλοντος** (`azd env set`)
4. **Σημαίες γραμμής εντολών** (`--location eastus`)

---

## 🔐 Βέλτιστες Πρακτικές Ασφαλείας

```bash
# Χρησιμοποιήστε διαχειριζόμενη ταυτότητα (συνιστάται)
azd env set AZURE_USE_MANAGED_IDENTITY true

# Ελέγξτε την κατάσταση ελέγχου ταυτότητας του AZD
azd auth status

# Προαιρετικό: επαληθεύστε το περιβάλλον του Azure CLI εάν σκοπεύετε να εκτελέσετε εντολές az
az account show

# Επαναλάβετε τον έλεγχο ταυτότητας αν χρειάζεται
azd auth login

# Προαιρετικό: ανανεώστε τον έλεγχο ταυτότητας του Azure CLI για τις εντολές az
az login
```
