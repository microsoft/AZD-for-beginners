# Κεφάλαιο 3: Διαμόρφωση & Αυθεντικοποίηση

**📚 Μάθημα**: [AZD Για Αρχάριους](../../README.md) | **⏱️ Διάρκεια**: 45-60 λεπτά | **⭐ Πολυπλοκότητα**: Ενδιάμεσο

## Επισκόπηση

Αυτό το κεφάλαιο καλύπτει τη διαμόρφωση περιβάλλοντος, πρότυπα αυθεντικοποίησης και βέλτιστες πρακτικές ασφάλειας για αναπτύξεις Azure Developer CLI.

> Επαληθευμένο με `azd 1.27.1` τον Ιούλιο του 2026.

## Στόχοι Μάθησης

Ολοκληρώνοντας αυτό το κεφάλαιο, θα:
- Κατακτήσετε την ιεραρχία διαμόρφωσης AZD
- Διαχειριστείτε πολλαπλά περιβάλλοντα (dev, staging, prod)
- Υλοποιήσετε ασφαλή αυθεντικοποίηση με διαχειριζόμενες ταυτότητες
- Διαμορφώσετε ρυθμίσεις ειδικές για το περιβάλλον

## 🚀 Γρήγορη Εκκίνηση

```bash
# Δημιουργήστε πολλαπλά περιβάλλοντα
azd env new dev
azd env new staging
azd env new prod

# Εναλλαγή περιβαλλόντων
azd env select prod

# Ορίστε μεταβλητές περιβάλλοντος
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# Προβολή ρυθμίσεων
azd env get-values
```

---

## 🔧 Ιεραρχία Διαμόρφωσης

Το AZD εφαρμόζει ρυθμίσεις με αυτή τη σειρά (τα μετά υπερισχύουν των πριν):

1. **Προεπιλεγμένες τιμές** (ενσωματωμένες στα πρότυπα)
2. **azure.yaml** (διαμόρφωση έργου)
3. **Μεταβλητές περιβάλλοντος** (`azd env set`)
4. **Σημαίες γραμμής εντολών** (`--location eastus`)

---

## 🔐 Βέλτιστες Πρακτικές Ασφαλείας

```bash
# Χρησιμοποιήστε διαχειριζόμενη ταυτότητα (συνιστάται)
azd env set AZURE_USE_MANAGED_IDENTITY true

# Ελέγξτε την κατάσταση πιστοποίησης AZD
azd auth status

# Προαιρετικό: επαληθεύστε το πλαίσιο Azure CLI αν σκοπεύετε να εκτελέσετε εντολές az
az account show

# Επαναπιστοποιηθείτε αν χρειάζεται
azd auth login

# Προαιρετικό: ανανεώστε την πιστοποίηση Azure CLI για τις εντολές az
az login
```
