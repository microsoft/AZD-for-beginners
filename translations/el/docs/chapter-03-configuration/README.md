# Κεφάλαιο 3: Διαμόρφωση & Πιστοποίηση

**📚 Μάθημα**: [AZD για Αρχάριους](../../README.md) | **⏱️ Διάρκεια**: 45-60 λεπτά | **⭐ Πολυπλοκότητα**: Ενδιάμεσο

---

## Επισκόπηση

Αυτό το κεφάλαιο καλύπτει τη διαμόρφωση του περιβάλλοντος, πρότυπα πιστοποίησης και βέλτιστες πρακτικές ασφάλειας για αναπτύξεις με το Azure Developer CLI.

## Στόχοι Μάθησης

Με την ολοκλήρωση αυτού του κεφαλαίου, θα:
- Κατακτήσετε την ιεραρχία διαμόρφωσης του AZD
- Διαχειριστείτε πολλαπλά περιβάλλοντα (dev, staging, prod)
- Υλοποιήσετε ασφαλή πιστοποίηση με managed identities
- Διαμορφώσετε ρυθμίσεις ειδικές για κάθε περιβάλλον

## 🚀 Γρήγορη Εκκίνηση

```bash
# Δημιουργία πολλαπλών περιβαλλόντων
azd env new dev
azd env new staging
azd env new prod

# Εναλλαγή περιβαλλόντων
azd env select prod

# Ορισμός μεταβλητών περιβάλλοντος
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# Προβολή διαμόρφωσης
azd env get-values
```

---

## 🔧 Ιεραρχία Διαμόρφωσης

AZD εφαρμόζει ρυθμίσεις με αυτήν τη σειρά (οι τελευταίες αντικαθιστούν τις προηγούμενες):

1. **Προεπιλεγμένες τιμές** (ενσωματωμένες στα πρότυπα)
2. **azure.yaml** (διαμόρφωση έργου)
3. **Μεταβλητές περιβάλλοντος** (`azd env set`)
4. **Σημαίες γραμμής εντολών** (`--location eastus`)

---

## 🔐 Βέλτιστες Πρακτικές Ασφάλειας

```bash
# Χρησιμοποιήστε διαχειριζόμενη ταυτότητα (συνιστάται)
azd env set AZURE_USE_MANAGED_IDENTITY true

# Ελέγξτε την κατάσταση αυθεντικοποίησης
azd auth whoami
az account show

# Αυθεντικοποιηθείτε ξανά αν χρειάζεται
azd auth login
az login
```
