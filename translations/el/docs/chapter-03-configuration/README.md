# Κεφάλαιο 3: Διαμόρφωση & Αυθεντικοποίηση

**📚 Course**: [AZD για Αρχάριους](../../README.md) | **⏱️ Duration**: 45-60 λεπτά | **⭐ Complexity**: Ενδιάμεσο

## Επισκόπηση

Αυτό το κεφάλαιο καλύπτει τη διαμόρφωση περιβάλλοντος, προτύπα αυθεντικοποίησης και βέλτιστες πρακτικές ασφάλειας για αναπτύξεις με το Azure Developer CLI.

> Επικυρώθηκε με `azd 1.25.6` τον Ιούνιο 2026.

## Στόχοι Μάθησης

Με την ολοκλήρωση αυτού του κεφαλαίου, θα:
- Κατανοήσετε την ιεραρχία διαμόρφωσης του AZD
- Διαχειριστείτε πολλαπλά περιβάλλοντα (dev, staging, prod)
- Υλοποιήσετε ασφαλή αυθεντικοποίηση με διαχειριζόμενες ταυτότητες
- Διαμορφώσετε ρυθμίσεις ανά περιβάλλον

## 🚀 Γρήγορη Εκκίνηση

```bash
# Δημιουργία πολλαπλών περιβαλλόντων
azd env new dev
azd env new staging
azd env new prod

# Εναλλαγή περιβαλλόντων
azd env select prod

# Ορισμός μεταβλητών περιβάλλοντος
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# Προβολή ρυθμίσεων
azd env get-values
```

---

## 🔧 Ιεραρχία Διαμόρφωσης

Το AZD εφαρμόζει ρυθμίσεις με αυτή τη σειρά (οι μεταγενέστερες αντικαθιστούν τις προηγούμενες):

1. **Default values** (built into templates)
2. **azure.yaml** (διαμόρφωση έργου)
3. **Environment variables** (`azd env set`)
4. **Command-line flags** (`--location eastus`)

---

## 🔐 Βέλτιστες Πρακτικές Ασφάλειας

```bash
# Χρησιμοποιήστε διαχειριζόμενη ταυτότητα (συνιστάται)
azd env set AZURE_USE_MANAGED_IDENTITY true

# Ελέγξτε την κατάσταση πιστοποίησης του AZD
azd auth status

# Προαιρετικό: επαληθεύστε το πλαίσιο του Azure CLI αν σκοπεύετε να εκτελέσετε εντολές az
az account show

# Επαναπιστοποιηθείτε αν χρειάζεται
azd auth login

# Προαιρετικό: ανανεώστε την πιστοποίηση του Azure CLI για εντολές az
az login
```
