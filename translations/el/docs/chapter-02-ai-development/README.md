# Κεφάλαιο 2: Ανάπτυξη με προτεραιότητα στην Τεχνητή Νοημοσύνη

**📚 Μάθημα**: [AZD για Αρχάριους](../../README.md) | **⏱️ Διάρκεια**: 1-2 ώρες | **⭐ Πολυπλοκότητα**: Ενδιάμεση

---

## Επισκόπηση

Αυτό το κεφάλαιο επικεντρώνεται στην ανάπτυξη εφαρμογών με δυνατότητες AI χρησιμοποιώντας το Azure Developer CLI και τις υπηρεσίες Microsoft Foundry. Από απλές εφαρμογές συνομιλίας AI έως ευφυείς πράκτορες με εργαλεία.

> **Σημείωση επικύρωσης (2026-03-25):** Η ροή εντολών και οι οδηγίες για τις επεκτάσεις σε αυτό το κεφάλαιο ελέγχθηκαν σε σύγκριση με `azd` `1.23.12` και την τρέχουσα προεπισκόπηση της επέκτασης πρακτόρων AI `azure.ai.agents` `0.1.18-preview`. Εάν χρησιμοποιείτε παλαιότερη έκδοση του AZD, ενημερώστε πρώτα και συνεχίστε με τις ασκήσεις.

## Στόχοι Μάθησης

Με την ολοκλήρωση αυτού του κεφαλαίου, θα:
- Αναπτύξετε εφαρμογές AI χρησιμοποιώντας προ-κατασκευασμένα πρότυπα AZD
- Κατανοήσετε την ενσωμάτωση του Microsoft Foundry με το AZD
- Διαμορφώσετε και προσαρμόσετε πράκτορες AI με εργαλεία
- Αναπτύξετε εφαρμογές RAG (Retrieval-Augmented Generation)

## 🚀 Γρήγορη Εκκίνηση

```bash
# Επιλογή 1: Εφαρμογή συνομιλίας RAG
azd init --template azure-search-openai-demo
azd up

# Επιλογή 2: Πράκτορες τεχνητής νοημοσύνης
azd init --template get-started-with-ai-agents
azd up

# Επιλογή 3: Γρήγορη εφαρμογή συνομιλίας
azd init --template openai-chat-app-quickstart
azd up
```
