# Κεφάλαιο 4: Υποδομή ως Κώδικας & Ανάπτυξη

**📚 Μάθημα**: [AZD για Αρχάριους](../../README.md) | **⏱️ Διάρκεια**: 1-1.5 ώρες | **⭐ Πολυπλοκότητα**: Ενδιάμεσο

---

## Επισκόπηση

Αυτό το κεφάλαιο καλύπτει πρότυπα Υποδομή ως Κώδικας (IaC) με πρότυπα Bicep, προμήθεια πόρων και στρατηγικές ανάπτυξης χρησιμοποιώντας το Azure Developer CLI.

## Στόχοι Μάθησης

Με την ολοκλήρωση αυτού του κεφαλαίου, θα:
- Θα κατανοήσετε τη δομή και τη σύνταξη των προτύπων Bicep
- Θα δημιουργήσετε πόρους Azure με `azd provision`
- Θα αναπτύξετε εφαρμογές με `azd deploy`
- Θα εφαρμόσετε στρατηγικές ανάπτυξης blue-green και κυλιόμενες

## 🚀 Γρήγορη Εκκίνηση

```bash
# Αρχικοποίηση από πρότυπο
azd init --template azure-functions-python-v2-http

# Προεπισκόπηση του τι θα δημιουργηθεί
azd provision --preview

# Μόνο εγκατάσταση υποδομής
azd provision

# Μόνο ανάπτυξη κώδικα
azd deploy

# Ή και τα δύο μαζί
azd up
```

## 📁 Δομή Έργου AZD

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

## 🔧 Βασικές Εντολές

| Εντολή | Περιγραφή |
|---------|-------------|
| `azd init` | Αρχικοποίηση έργου |
| `azd provision` | Δημιουργία πόρων Azure |
| `azd deploy` | Ανάπτυξη κώδικα εφαρμογής |
| `azd up` | προμήθεια + ανάπτυξη |
| `azd down` | Διαγραφή όλων των πόρων |
