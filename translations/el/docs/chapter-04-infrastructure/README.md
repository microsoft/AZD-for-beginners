# Κεφάλαιο 4: Υποδομή ως Κώδικας & Ανάπτυξη

**📚 Μάθημα**: [AZD Για Αρχάριους](../../README.md) | **⏱️ Διάρκεια**: 1-1.5 ώρες | **⭐ Πολυπλοκότητα**: Μεσαίο Επίπεδο

---

## Επισκόπηση

Αυτό το κεφάλαιο καλύπτει τα πρότυπα Υποδομής ως Κώδικα (IaC) με πρότυπα Bicep, προμήθεια πόρων, και στρατηγικές ανάπτυξης χρησιμοποιώντας το Azure Developer CLI.

> Επαληθευμένο με `azd 1.27.1` τον Ιούλιο του 2026.

## Μαθησιακοί Στόχοι

Με την ολοκλήρωση αυτού του κεφαλαίου, θα:
- Κατανοήσετε τη δομή και τη σύνταξη των προτύπων Bicep
- Προμηθεύετε πόρους Azure με `azd provision`
- Αναπτύσσετε εφαρμογές με `azd deploy`
- Υλοποιείτε στρατηγικές ανάπτυξης blue-green και rolling

## 🚀 Γρήγορη Εκκίνηση

```bash
# Αρχικοποίηση από πρότυπο
azd init --template azure-functions-python-v2-http

# Προεπισκόπηση του τι θα δημιουργηθεί
azd provision --preview

# Παροχή μόνο υποδομής
azd provision

# Ανάπτυξη μόνο κώδικα
azd deploy

# Ή και τα δύο μαζί
azd up
```

## 📁 Δομή Έργου AZD

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

## 🔧 Βασικές Εντολές

| Εντολή | Περιγραφή |
|---------|-------------|
| `azd init` | Εκκίνηση έργου |
| `azd provision` | Δημιουργία πόρων Azure |
| `azd deploy` | Ανάπτυξη κώδικα εφαρμογής |
| `azd up` | προμήθεια + ανάπτυξη |
| `azd down` | Διαγραφή όλων των πόρων |
