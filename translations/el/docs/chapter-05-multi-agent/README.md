# Κεφάλαιο 5: Λύσεις AI με Πολλούς Πράκτορες

**📚 Μάθημα**: [AZD για Αρχάριους](../../README.md) | **⏱️ Διάρκεια**: 2-3 hours | **⭐ Πολυπλοκότητα**: Προχωρημένο

---

## Επισκόπηση

Αυτό το κεφάλαιο καλύπτει προχωρημένα πρότυπα αρχιτεκτονικής πολλαπλών πρακτόρων, ορχήστρωση πρακτόρων και έτοιμες για παραγωγή αναπτύξεις AI για σύνθετα σενάρια.

## Στόχοι Μάθησης

Ολοκληρώνοντας αυτό το κεφάλαιο, θα:
- Κατανοείτε τα πρότυπα αρχιτεκτονικής πολλαπλών πρακτόρων
- Αναπτύσσετε συντονισμένα συστήματα πρακτόρων AI
- Υλοποιείτε επικοινωνία πράκτορα-πράκτορα
- Δημιουργείτε λύσεις πολλαπλών πρακτόρων έτοιμες για παραγωγή

## 🚀 Γρήγορη Εκκίνηση

```bash
# Αναπτύξτε τη λύση πολλαπλών πρακτόρων για τη λιανική
cd examples/retail-multiagent-arm-template
./deploy.sh

# Ή χρησιμοποιήστε το πρότυπο απευθείας
azd init --template agent-openai-python-prompty
azd up
```

                     │                 │
         ┌───────────▼───────┐ ┌───────▼───────────┐

         └───────────────────┘ └───────────────────┘

## 🎯 Προτεινόμενη Λύση: Λιανική με Πολλούς Πράκτορες

Η [Λύση Λιανικής με Πολλούς Πράκτορες](../../examples/retail-scenario.md) επιδεικνύει:

- **Πράκτορας Πελάτη**: Διαχειρίζεται τις αλληλεπιδράσεις χρηστών και τις προτιμήσεις
- **Πράκτορας Αποθέματος**: Διαχειρίζεται τα αποθέματα και την επεξεργασία παραγγελιών
- **Ορχηστρωτής**: Συντονίζει τους πράκτορες
- **Κοινή Μνήμη**: Διαχείριση κοινού πλαισίου μεταξύ πρακτόρων

### Χρησιμοποιούμενες Υπηρεσίες

| Υπηρεσία | Σκοπός |
|---------|---------|
| Azure OpenAI | Κατανόηση γλώσσας |
| Azure AI Search | Κατάλογος προϊόντων |
| Cosmos DB | Κατάσταση και μνήμη πράκτορα |
| Container Apps | Φιλοξενία πρακτόρων |
| Application Insights | Παρακολούθηση |
