## Επισκόπηση Μαθήματος

Κατακτήστε το Azure Developer CLI (azd) μέσω δομημένων κεφαλαίων σχεδιασμένων για προοδευτική μάθηση. **Ειδική έμφαση στην ανάπτυξη εφαρμογών τεχνητής νοημοσύνης με ενσωμάτωση του Microsoft Foundry.**

### Γιατί αυτό το μάθημα είναι απαραίτητο για τους σύγχρονους προγραμματιστές

Βασισμένο σε πληροφορίες από την κοινότητα Microsoft Foundry στο Discord, **το 45% των προγραμματιστών θέλει να χρησιμοποιήσει το AZD για εργασίες τεχνητής νοημοσύνης** αλλά συναντούν προκλήσεις με:
- Σύνθετες αρχιτεκτονικές ΤΝ με πολλαπλές υπηρεσίες
- Βέλτιστες πρακτικές ανάπτυξης ΤΝ σε παραγωγή  
- Ενσωμάτωση και ρύθμιση των υπηρεσιών Azure AI
- Βελτιστοποίηση κόστους για φόρτους εργασίας ΤΝ
- Επίλυση προβλημάτων ανάπτυξης ειδικών για ΤΝ

### Στόχοι Μάθησης

Ολοκληρώνοντας αυτό το δομημένο μάθημα, θα:
- **Κατακτήστε τα βασικά του AZD**: Βασικές έννοιες, εγκατάσταση και ρύθμιση
- **Αναπτύξτε εφαρμογές ΤΝ**: Χρησιμοποιήστε το AZD με τις υπηρεσίες του Microsoft Foundry
- **Εφαρμόστε Υποδομή ως Κώδικας**: Διαχειριστείτε πόρους Azure με πρότυπα Bicep
- **Επιλύστε προβλήματα αναπτύξεων**: Εντοπίστε και διορθώστε κοινά ζητήματα
- **Βελτιστοποιήστε για παραγωγή**: Ασφάλεια, κλιμάκωση, παρακολούθηση και διαχείριση κόστους
- **Δημιουργήστε λύσεις πολλαπλών πρακτόρων**: Αναπτύξτε σύνθετες αρχιτεκτονικές ΤΝ

#### Πηγές Μάθησης
- **🎯 Ξεκινήστε Εδώ**: [Ενσωμάτωση Microsoft Foundry](docs/microsoft-foundry/microsoft-foundry-integration.md)
- **📖 Προτύπα**: [Ανάπτυξη Μοντέλου ΤΝ](docs/microsoft-foundry/ai-model-deployment.md) - Αναπτύξτε και διαχειριστείτε μοντέλα ΤΝ
- **🛠️ Εργαστήριο**: [Εργαστήριο ΤΝ](docs/microsoft-foundry/ai-workshop-lab.md) - Κάντε τις λύσεις ΤΝ έτοιμες για AZD
- **🎥 Διαδραστικός Οδηγός**: [Υλικά Εργαστηρίου](workshop/README.md) - Μάθηση μέσω προγράμματος περιήγησης με MkDocs * DevContainer Environment

#### Πρακτικές Ασκήσεις
```bash
# Αναπτύξτε την πρώτη σας εφαρμογή τεχνητής νοημοσύνης
azd init --template azure-search-openai-demo
azd up

# Δοκιμάστε επιπλέον πρότυπα τεχνητής νοημοσύνης
azd init --template openai-chat-app-quickstart
azd init --template agent-openai-python-prompty
```

**💡 Αποτέλεσμα Κεφαλαίου**: Αναπτύξτε και ρυθμίστε μια εφαρμογή chat με τεχνητή νοημοσύνη και δυνατότητες RAG

**✅ Επιβεβαίωση Επιτυχίας:**
```bash
# Μετά το Κεφάλαιο 2, θα πρέπει να μπορείτε να:
azd init --template azure-search-openai-demo
azd up
# Να δοκιμάσετε τη διεπαφή συνομιλίας τεχνητής νοημοσύνης
# Να κάνετε ερωτήσεις και να λαμβάνετε απαντήσεις από τεχνητή νοημοσύνη με πηγές
# Να επαληθεύσετε ότι η ενσωμάτωση αναζήτησης λειτουργεί
azd monitor  # Να ελέγξετε ότι το Application Insights εμφανίζει τηλεμετρία
azd down --force --purge
```

**📊 Χρόνος Επένδυσης:** 1-2 ώρες  
**📈 Επίπεδο Δεξιοτήτων Μετά:** Μπορείτε να αναπτύξετε και να ρυθμίσετε εφαρμογές ΤΝ έτοιμες για παραγωγή  
**💰 Ενημέρωση Κόστους:** Κατανοήστε τα έξοδα ανάπτυξης $80-150/μήνα, κόστη παραγωγής $300-3500/μήνα

#### 💰 Εκτιμήσεις Κόστους για Αναπτύξεις ΤΝ

**Περιβάλλον Ανάπτυξης (Εκτιμώμενο $80-150/μήνα):**
- Azure OpenAI (Πληρωμή κατά χρήση): $0-50/μήνα (με βάση τη χρήση token)
- AI Search (Βασικό tier): $75/μήνα
- Container Apps (Κατανάλωση): $0-20/μήνα
- Storage (Standard): $1-5/μήνα

**Περιβάλλον Παραγωγής (Εκτιμώμενο $300-3,500+/μήνα):**
- Azure OpenAI (PTU για σταθερή απόδοση): $3,000+/μήνα OR Pay-as-go with high volume
- AI Search (Standard tier): $250/μήνα
- Container Apps (Dedicated): $50-100/μήνα
- Application Insights: $5-50/μήνα
- Storage (Premium): $10-50/μήνα

**💡 Συμβουλές Βελτιστοποίησης Κόστους:**
- Χρησιμοποιήστε το **Free Tier** του Azure OpenAI για μάθηση (50,000 tokens/μήνα συμπεριλαμβάνονται)
- Εκτελέστε `azd down` για να απελευθερώσετε πόρους όταν δεν αναπτύσσετε ενεργά
- Ξεκινήστε με τιμολόγηση βάσει κατανάλωσης, αναβαθμίστε σε PTU μόνο για παραγωγή
- Χρησιμοποιήστε `azd provision --preview` για να εκτιμήσετε τα κόστη πριν την ανάπτυξη
- Ενεργοποιήστε αυτόματη κλιμάκωση: πληρώνετε μόνο για την πραγματική χρήση

**Παρακολούθηση Κόστους:**
```bash
# Ελέγξτε τα εκτιμώμενα μηνιαία κόστη
azd provision --preview

# Παρακολουθήστε τα πραγματικά κόστη στο Azure Portal
az consumption budget list --resource-group <your-rg>
```

- **Αναπτύξεις με βάση πρότυπα** - Χρησιμοποιήστε προ-κατασκευασμένα πρότυπα για κοινά μοτίβα εφαρμογών

- **Βελτιστοποίηση Κόστους** - Έξυπνη κατανομή πόρων και στρατηγικές κλιμάκωσης για φόρτους AI

```bash
az account show
```

```bash
# Δοκιμάστε διαφορετική περιοχή του Azure

# Ή χρησιμοποιήστε μικρότερα SKUs κατά την ανάπτυξη

  "sku": "B1"  // Instead of "P1V2"
}
```

```bash
# Επιλογή 1: Καθαρίστε και προσπαθήστε ξανά
azd down --force --purge
azd up

# Επιλογή 2: Απλώς διορθώστε την υποδομή
azd provision

# Επιλογή 3: Ελέγξτε τα λεπτομερή αρχεία καταγραφής
azd show
azd logs
```

```bash
# Επαλήθευση αυθεντικοποίησης
az account show
```

```bash

azd env new dev-v2
azd up
```

### ❌ Template deployment taking too long

**Κανονικοί χρόνοι αναμονής:**
- Απλή web εφαρμογή: 5-10 λεπτά
- Εφαρμογή με βάση δεδομένων: 10-15 λεπτά
- Εφαρμογές AI: 15-25 λεπτά (η παροχή πόρων OpenAI είναι αργή)

```bash
# Ελέγξτε την πρόοδο
azd show

# Αν έχει κολλήσει για περισσότερο από 30 λεπτά, ελέγξτε το Azure Portal:
azd monitor
# Αναζητήστε αποτυχημένες αναπτύξεις
```

```bash
# Ελέγξτε τον ρόλο σας στο Azure

# Χρειάζεστε τουλάχιστον τον ρόλο «Συνεισφέρων»
# Ζητήστε από τον διαχειριστή του Azure να χορηγήσει:
# - Συνεισφέρων (για πόρους)
# - Διαχειριστής Πρόσβασης Χρηστών (για τις αναθέσεις ρόλων)
```

### ❌ Can't find deployed application URL

```bash
# Εμφάνιση όλων των τελικών σημείων υπηρεσιών
azd show

# Ή ανοίξτε την Πύλη του Azure
azd monitor

# Ελέγξτε συγκεκριμένη υπηρεσία
azd env get-values
# Αναζητήστε μεταβλητές *_URL
```
