## Επισκόπηση Μαθήματος

Εξασκήστε τον Azure Developer CLI (azd) μέσα από δομημένα κεφάλαια σχεδιασμένα για προοδευτική μάθηση. **Ειδική έμφαση στην ανάπτυξη εφαρμογών AI με ενσωμάτωση Microsoft Foundry.**

### Γιατί Αυτό το Μάθημα Είναι Απαραίτητο για Σύγχρονους Προγραμματιστές

Βασισμένο στις πληροφορίες από την κοινότητα Discord του Microsoft Foundry, **το 45% των προγραμματιστών θέλουν να χρησιμοποιήσουν το AZD για εργασίες AI** αλλά αντιμετωπίζουν προκλήσεις με:
- Πολύπλοκες αρχιτεκτονικές AI που αφορούν πολλαπλές υπηρεσίες
- Βέλτιστες πρακτικές ανάπτυξης AI για παραγωγή  
- Ενσωμάτωση και ρύθμιση υπηρεσιών Azure AI
- Βελτιστοποίηση κόστους για εργασίες AI
- Επίλυση προβλημάτων ειδικών στην ανάπτυξη AI

### Στόχοι Μάθησης

Ολοκληρώνοντας αυτό το δομημένο μάθημα, θα:
- **Κατακτήσετε τα Βασικά του AZD**: Κεντρικές έννοιες, εγκατάσταση και ρύθμιση
- **Αναπτύξετε Εφαρμογές AI**: Χρησιμοποιώντας το AZD με υπηρεσίες Microsoft Foundry
- **Υλοποιήσετε Υποδομές ως Κώδικα**: Διαχείριση πόρων Azure με πρότυπα Bicep
- **Αντιμετωπίσετε Προβλήματα Ανάπτυξης**: Επίλυση συχνών θεμάτων και αποσφαλμάτωση
- **Βελτιστοποιήσετε για Παραγωγή**: Ασφάλεια, κλιμάκωση, παρακολούθηση και διαχείριση κόστους
- **Δημιουργήσετε Πολυ-Πρακτορικές Λύσεις**: Ανάπτυξη πολύπλοκων αρχιτεκτονικών AI

### 🤖 Κεφάλαιο 2: Ανάπτυξη με Προτεραιότητα στην AI (Συνιστάται για Προγραμματιστές AI)

- Ανάπτυξη εφαρμογών με τεχνητή νοημοσύνη
- Κατανόηση ρυθμίσεων υπηρεσιών AI

#### Πόροι Μάθησης
- **🎯 Ξεκινήστε Εδώ**: [Ενσωμάτωση Microsoft Foundry](docs/microsoft-foundry/microsoft-foundry-integration.md)
- **📖 Πρότυπα**: [Ανάπτυξη Μοντέλων AI](docs/microsoft-foundry/ai-model-deployment.md) - Ανάπτυξη και διαχείριση μοντέλων AI
- **🛠️ Εργαστήριο**: [AI Workshop Lab](docs/microsoft-foundry/ai-workshop-lab.md) - Κάντε τις λύσεις AI έτοιμες για AZD
- **🎥 Διαδραστικός Οδηγός**: [Υλικά Εργαστηρίου](workshop/README.md) - Μάθηση μέσω browser με MkDocs * Περιβάλλον DevContainer
- **📋 Πρότυπα**: [Πρότυπα Microsoft Foundry](../..)
- **📝 Παραδείγματα**: [Παραδείγματα Ανάπτυξης AZD](examples/README.md)

#### Πρακτικές Ασκήσεις
```bash
# Αναπτύξτε την πρώτη σας εφαρμογή τεχνητής νοημοσύνης
azd init --template azure-search-openai-demo
azd up

# Δοκιμάστε επιπλέον πρότυπα τεχνητής νοημοσύνης
azd init --template openai-chat-app-quickstart
azd init --template agent-openai-python-prompty
```

**💡 Αποτέλεσμα Κεφαλαίου**: Ανάπτυξη και ρύθμιση μιας εφαρμογής συνομιλίας με τεχνητή νοημοσύνη και δυνατότητες RAG

**✅ Επικύρωση Επιτυχίας:**
```bash
# Μετά το Κεφάλαιο 2, θα πρέπει να μπορείτε να:
azd init --template azure-search-openai-demo
azd up
# Δοκιμάσετε τη διεπαφή συνομιλίας AI
# Κάνετε ερωτήσεις και λαμβάνετε απαντήσεις με τεχνητή νοημοσύνη και πηγές
# Επιβεβαιώσετε ότι η ενσωμάτωση αναζήτησης λειτουργεί
azd monitor  # Ελέγξετε αν το Application Insights εμφανίζει τηλεμετρία
azd down --force --purge
```

**📊 Επένδυση Χρόνου:** 1-2 ώρες  
**📈 Επίπεδο Δεξιότητας Μετά:** Μπορεί να αναπτύσσει και να ρυθμίζει έτοιμες για παραγωγή AI εφαρμογές  
**💰 Ενημέρωση Κόστους:** Κατανόηση κόστους ανάπτυξης $80-150/μήνα, κόστους παραγωγής $300-3500/μήνα

#### 💰 Εξετάσεις Κόστους για Ανάπτυξη AI

**Περιβάλλον Ανάπτυξης (Εκτιμώμενο $80-150/μήνα):**
- Azure OpenAI (Πληρωμή κατά Χρήση): $0-50/μήνα (ανάλογα με τη χρήση token)
- AI Search (Βασικό επίπεδο): $75/μήνα
- Container Apps (Κατανάλωση): $0-20/μήνα
- Αποθήκευση (Κανονική): $1-5/μήνα

**Περιβάλλον Παραγωγής (Εκτιμώμενο $300-3,500+/μήνα):**
- Azure OpenAI (PTU για σταθερή απόδοση): $3,000+/μήνα ή Πληρωμή κατά Χρήση με μεγάλο όγκο
- AI Search (Κανονικό επίπεδο): $250/μήνα
- Container Apps (Αφιερωμένα): $50-100/μήνα
- Application Insights: $5-50/μήνα
- Αποθήκευση (Premium): $10-50/μήνα

**💡 Συμβουλές Βελτιστοποίησης Κόστους:**
- Χρησιμοποιήστε **Δωρεάν Επίπεδο** Azure OpenAI για μάθηση (περιλαμβάνει 50,000 tokens/μήνα)
- Εκτελέστε `azd down` για αποδέσμευση πόρων όταν δεν αναπτύσσετε ενεργά
- Ξεκινήστε με χρέωση κατανάλωσης και αναβαθμίστε σε PTU μόνο για παραγωγή
- Χρησιμοποιήστε `azd provision --preview` για εκτίμηση κόστους πριν την ανάπτυξη
- Ενεργοποιήστε την αυτόματη κλιμάκωση: πληρώνετε μόνο για την πραγματική χρήση

**Παρακολούθηση Κόστους:**
```bash
# Ελέγξτε τα εκτιμώμενα μηνιαία έξοδα
azd provision --preview

# Παρακολουθήστε τα πραγματικά έξοδα στο Azure Portal
az consumption budget list --resource-group <your-rg>
```

## 📖 Τι είναι το Azure Developer CLI;  

Το Azure Developer CLI (azd) είναι ένα γραμμή εντολών προσανατολισμένη στον προγραμματιστή που επιταχύνει τη διαδικασία δημιουργίας και ανάπτυξης εφαρμογών στο Azure. Προσφέρει:  

- **Αναπτύξεις βάσει προτύπων** - Χρήση προκατασκευασμένων προτύπων για κοινά μοτίβα εφαρμογών  
- **Υποδομή ως Κώδικας** - Διαχείριση πόρων Azure με Bicep ή Terraform  
- **Ενσωματωμένες ροές εργασίας** - Ομαλή παροχή, ανάπτυξη και παρακολούθηση εφαρμογών  
- **Φιλικότητα προς τον προγραμματιστή** - Βελτιστοποίηση για παραγωγικότητα και εμπειρία προγραμματιστή  

### **AZD + Microsoft Foundry: Ιδανικό για Αναπτύξεις AI**  

**Γιατί AZD για λύσεις AI;** Το AZD αντιμετωπίζει τις κορυφαίες προκλήσεις που έχουν οι προγραμματιστές AI:  

- **Πρότυπα έτοιμα για AI** - Προδιαμορφωμένα πρότυπα για Azure OpenAI, Cognitive Services, και εργασίες ML  
- **Ασφαλείς αναπτύξεις AI** - Ενσωματωμένα πρότυπα ασφάλειας για υπηρεσίες AI, κλειδιά API και τελικά σημεία μοντέλων  
- **Πρότυπα παραγωγής AI** - Βέλτιστες πρακτικές για επεκτάσιμες και οικονομικές αναπτύξεις AI  
- **Ροές εργασίας AI από άκρη σε άκρη** - Από την ανάπτυξη μοντέλου έως την παραγωγική ανάπτυξη με κατάλληλη παρακολούθηση  
- **Βελτιστοποίηση κόστους** - Έξυπνες στρατηγικές κατανομής πόρων και κλιμάκωσης για φορτία AI  
- **Ενσωμάτωση Microsoft Foundry** - Ομαλή σύνδεση με κατάλογο μοντέλων Microsoft Foundry και τελικά σημεία

```bash
winget install microsoft.azd

# macOS:
brew tap azure/azd && brew install azd

# Linux:
curl -fsSL https://aka.ms/install-azd.sh | bash

# Επαληθεύστε την εγκατάσταση
azd version
```

### ❌ "Δεν βρέθηκε συνδρομή" ή "Η συνδρομή δεν έχει οριστεί"

```bash
# Λίστα διαθέσιμων συνδρομών
az account list --output table

# Ορισμός προεπιλεγμένης συνδρομής
az account set --subscription "<subscription-id-or-name>"

# Ορισμός για περιβάλλον AZD
azd env set AZURE_SUBSCRIPTION_ID "<subscription-id>"

# Επιβεβαίωση
az account show
```

### ❌ "InsufficientQuota" ή "Υπέρβαση Κουότας"

```bash
# Δοκιμάστε διαφορετική περιοχή Azure
azd env set AZURE_LOCATION "westus2"
azd up

# Ή χρησιμοποιήστε μικρότερες SKU στην ανάπτυξη
# Επεξεργαστείτε το infra/main.parameters.json:
{
  "sku": "B1"  // Instead of "P1V2"
}
```

### ❌ Το "azd up" αποτυγχάνει στη μέση της διαδικασίας

```bash
# Επιλογή 1: Καθαρίστε και δοκιμάστε ξανά
azd down --force --purge
azd up

# Επιλογή 2: Απλά διορθώστε την υποδομή
azd provision

# Επιλογή 3: Ελέγξτε λεπτομερή αρχεία καταγραφής
azd show
azd logs
```

### ❌ "Αποτυχία πιστοποίησης" ή "Η λήξη του διακριτικού έληξε"

```bash
# Επαναναγνώριση
az logout
az login

azd auth logout
azd auth login

# Επαλήθευση ταυτοποίησης
az account show
```

### ❌ "Ο πόρος ήδη υπάρχει" ή συγκρούσεις ονομασίας

```bash
# Το AZD δημιουργεί μοναδικά ονόματα, αλλά αν υπάρχει σύγκρουση:
azd down --force --purge

# Στη συνέχεια, δοκιμάστε ξανά με νέο περιβάλλον
azd env new dev-v2
azd up
```

### ❌ Η ανάπτυξη προτύπου διαρκεί πολύ

**Κανονικές χρόνοι αναμονής:**
- Απλή web εφαρμογή: 5-10 λεπτά
- Εφαρμογή με βάση δεδομένων: 10-15 λεπτά
- Εφαρμογές AI: 15-25 λεπτά (Η παροχή υπηρεσιών OpenAI είναι αργή)

```bash
# Ελέγξτε την πρόοδο
azd show

# Εάν κολλήσετε >30 λεπτά, ελέγξτε το Azure Portal:
azd monitor
# Αναζητήστε αποτυχημένες αναπτύξεις
```

### ❌ "Απαγόρευση πρόσβασης" ή "Απαγορευμένο"

```bash
# Ελέγξτε το ρόλο σας στο Azure
az role assignment list --assignee $(az account show --query user.name -o tsv)

# Χρειάζεστε τουλάχιστον ρόλο "Συνεισφέρων"
# Ζητήστε από τον διαχειριστή Azure να παραχωρήσει:
# - Συνεισφέρων (για πόρους)
# - Διαχειριστής Πρόσβασης Χρήστη (για εκχωρήσεις ρόλων)
```

### ❌ Δεν εντοπίζεται το URL της αναπτυγμένης εφαρμογής

```bash
# Εμφάνιση όλων των τελικών σημείων υπηρεσίας
azd show

# Ή άνοιγμα του Azure Portal
azd monitor

# Έλεγχος συγκεκριμένης υπηρεσίας
azd env get-values
# Αναζήτηση μεταβλητών *_URL
```
