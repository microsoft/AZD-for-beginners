## Εισαγωγή

Αυτό το πλήρες γλωσσάρι παρέχει ορισμούς για όρους, έννοιες και ακρωνύμια που χρησιμοποιούνται στο Azure Developer CLI και στην ανάπτυξη στο cloud της Azure. Απαραίτητη αναφορά για την κατανόηση τεχνικής τεκμηρίωσης, την αντιμετώπιση προβλημάτων και την αποτελεσματική επικοινωνία σχετικά με έργα azd και υπηρεσίες Azure.

## Στόχοι Μάθησης

Με τη χρήση αυτού του γλωσσαρίου, θα:
- Κατανοήσετε βασική ορολογία και έννοιες του Azure Developer CLI
- Εξοικειωθείτε με το λεξιλόγιο και τους τεχνικούς όρους της ανάπτυξης στο Azure cloud
- Ανατρέξετε αποτελεσματικά σε όρους Infrastructure as Code και ανάπτυξης
- Κατανοήσετε ονόματα υπηρεσιών Azure, ακρωνύμια και τους σκοπούς τους
- Έχετε πρόσβαση σε ορισμούς για όρους αντιμετώπισης σφαλμάτων και debugging
- Μάθετε προχωρημένες έννοιες αρχιτεκτονικής και ανάπτυξης στο Azure

## Αποτελέσματα Μάθησης

Με τακτική αναφορά σε αυτό το γλωσσάρι, θα μπορείτε να:
- Επικοινωνείτε αποτελεσματικά χρησιμοποιώντας τη σωστή ορολογία του Azure Developer CLI
- Κατανοείτε την τεχνική τεκμηρίωση και τα μηνύματα σφαλμάτων πιο σαφώς
- Πλοηγείστε στις υπηρεσίες και τις έννοιες του Azure με αυτοπεποίθηση
- Αντιμετωπίζετε προβλήματα χρησιμοποιώντας κατάλληλο τεχνικό λεξιλόγιο
- Συμβάλλετε σε ομαδικές συζητήσεις με ακριβή τεχνική γλώσσα
- Επεκτείνετε συστηματικά τις γνώσεις σας στην ανάπτυξη cloud του Azure

## A

**ARM Template**  
Πρότυπο Azure Resource Manager. Μορφότυπος Infrastructure as Code βασισμένος σε JSON που χρησιμοποιείται για τον δηλωτικό ορισμό και την ανάπτυξη πόρων Azure.

**App Service**  
Προσφορά πλατφόρμας ως υπηρεσία (PaaS) της Azure για τη φιλοξενία web εφαρμογών, REST API και mobile backends χωρίς τη διαχείριση της υποδομής.

**Application Insights**  
Υπηρεσία Application Performance Monitoring (APM) της Azure που παρέχει βαθιές πληροφορίες για την απόδοση, τη διαθεσιμότητα και τη χρήση των εφαρμογών.

**Azure CLI**  
Διεπαφή γραμμής εντολών για τη διαχείριση πόρων Azure. Χρησιμοποιείται από το azd για αυθεντικοποίηση και κάποιες λειτουργίες.

**Azure Developer CLI (azd)**  
Εργαλείο γραμμής εντολών επικεντρωμένο στον προγραμματιστή που επιταχύνει τη διαδικασία δημιουργίας και ανάπτυξης εφαρμογών στο Azure χρησιμοποιώντας πρότυπα και Infrastructure as Code.

**azure.yaml**  
Το κύριο αρχείο ρυθμίσεων για ένα έργο azd που ορίζει υπηρεσίες, υποδομή και hooks ανάπτυξης.

**Azure Resource Manager (ARM)**  
Η υπηρεσία ανάπτυξης και διαχείρισης της Azure που παρέχει ένα επίπεδο διαχείρισης για τη δημιουργία, την ενημέρωση και τη διαγραφή πόρων.

## B

**Bicep**  
Γλώσσα ειδικού τομέα (DSL) που αναπτύχθηκε από τη Microsoft για την ανάπτυξη πόρων Azure. Παρέχει απλούστερη σύνταξη από τα ARM templates ενώ μεταγλωττίζεται σε ARM.

**Build**  
Η διαδικασία μεταγλώττισης του πηγαίου κώδικα, εγκατάστασης εξαρτήσεων και προετοιμασίας εφαρμογών για ανάπτυξη.

**Blue-Green Deployment**  
Στρατηγική ανάπτυξης που χρησιμοποιεί δύο ταυτόσημα περιβάλλοντα παραγωγής (blue και green) για τη μείωση του χρόνου διακοπής και του κινδύνου.

## C

**Container Apps**  
Serverless υπηρεσία container της Azure που επιτρέπει τη λειτουργία containerized εφαρμογών χωρίς τη διαχείριση πολύπλοκης υποδομής.

**CI/CD**  
Continuous Integration/Continuous Deployment. Αυτοματοποιημένες πρακτικές για την ενσωμάτωση αλλαγών κώδικα και την ανάπτυξη εφαρμογών.

**Cosmos DB**  
Παγκοσμίως διανεμημένη, multi-model υπηρεσία βάσης δεδομένων της Azure που παρέχει ολοκληρωμένα SLAs για throughput, latency, διαθεσιμότητα και συνοχή.

**Configuration**  
Ρυθμίσεις και παράμετροι που ελέγχουν τη συμπεριφορά της εφαρμογής και τις επιλογές ανάπτυξης.

## D

**Deployment**  
Η διαδικασία εγκατάστασης και ρύθμισης εφαρμογών και των εξαρτήσεών τους στην προοριζόμενη υποδομή.

**Docker**  
Πλατφόρμα για ανάπτυξη, διανομή και εκτέλεση εφαρμογών χρησιμοποιώντας τεχνολογία containerization.

**Dockerfile**  
Αρχείο κειμένου που περιέχει οδηγίες για τη δημιουργία ενός Docker container image.

## E

**Environment**  
Στόχος ανάπτυξης που αντιπροσωπεύει μια συγκεκριμένη έκδοση της εφαρμογής σας (π.χ., development, staging, production).

**Environment Variables**  
Τιμές ρύθμισης αποθηκευμένες ως ζεύγη κλειδιού-τιμής που οι εφαρμογές μπορούν να προσπελάσουν κατά το runtime.

**Endpoint**  
URL ή δικτυακή διεύθυνση όπου μια εφαρμογή ή υπηρεσία είναι προσβάσιμη.

## F

**Function App**  
Serverless υπολογιστική υπηρεσία της Azure που επιτρέπει την εκτέλεση event-driven κώδικα χωρίς διαχείριση υποδομής.

## G

**GitHub Actions**  
Πλατφόρμα CI/CD ενσωματωμένη σε αποθετήρια GitHub για την αυτοματοποίηση workflows.

**Git**  
Κατανεμημένο σύστημα ελέγχου εκδόσεων που χρησιμοποιείται για την παρακολούθηση αλλαγών στον πηγαίο κώδικα.

## H

**Hooks**  
Προσαρμοσμένα σενάρια ή εντολές που εκτελούνται σε συγκεκριμένα σημεία του κύκλου ζωής ανάπτυξης (preprovision, postprovision, predeploy, postdeploy).

**Host**  
Ο τύπος υπηρεσίας Azure όπου μια εφαρμογή θα αναπτυχθεί (π.χ., appservice, containerapp, function).

## I

**Infrastructure as Code (IaC)**  
Πρακτική ορισμού και διαχείρισης υποδομής μέσω κώδικα αντί για χειροκίνητες διαδικασίες.

**Init**  
Η διαδικασία αρχικοποίησης ενός νέου έργου azd, συνήθως από ένα πρότυπο.

## J

**JSON**  
JavaScript Object Notation. Μορφότυπος ανταλλαγής δεδομένων που χρησιμοποιείται συνήθως για αρχεία ρυθμίσεων και απαντήσεις API.

**JWT**  
JSON Web Token. Πρότυπο για την ασφαλή μετάδοση πληροφοριών μεταξύ μερών ως αντικείμενο JSON.

## K

**Key Vault**  
Υπηρεσία της Azure για ασφαλή αποθήκευση και διαχείριση μυστικών, κλειδιών και πιστοποιητικών.

**Kusto Query Language (KQL)**  
Γλώσσα ερωτημάτων που χρησιμοποιείται για την ανάλυση δεδομένων στο Azure Monitor, το Application Insights και άλλες υπηρεσίες Azure.

## L

**Load Balancer**  
Υπηρεσία που διανέμει την εισερχόμενη δικτυακή κίνηση σε πολλαπλούς servers ή instances.

**Log Analytics**  
Υπηρεσία της Azure για τη συλλογή, ανάλυση και ενέργεια πάνω σε telemetry δεδομένα από cloud και on-premises περιβάλλοντα.

## M

**Managed Identity**  
Δυνατότητα της Azure που παρέχει στις υπηρεσίες Azure ένα αυτόματα διαχειριζόμενο αναγνωριστικό για αυθεντικοποίηση σε άλλες υπηρεσίες Azure.

**Microservices**  
Αρχιτεκτονική προσέγγιση όπου οι εφαρμογές κατασκευάζονται ως σύνολο μικρών, ανεξάρτητων υπηρεσιών.

**Monitor**  
Ενοποιημένη λύση παρακολούθησης της Azure που παρέχει observability σε όλο το στοίχο εφαρμογών και υποδομής.

## N

**Node.js**  
Runtime JavaScript χτισμένο πάνω στη μηχανή V8 του Chrome για την κατασκευή server-side εφαρμογών.

**npm**  
Διαχειριστής πακέτων για Node.js που διαχειρίζεται εξαρτήσεις και πακέτα.

## O

**Output**  
Τιμές που επιστρέφονται από την ανάπτυξη υποδομής και μπορούν να χρησιμοποιηθούν από εφαρμογές ή άλλους πόρους.

## P

**Package**  
Η διαδικασία προετοιμασίας του κώδικα εφαρμογής και των εξαρτήσεων για ανάπτυξη.

**Parameters**  
Εισαγόμενες τιμές που δίνονται σε πρότυπα υποδομής για την προσαρμογή των αναπτύξεων.

**PostgreSQL**  
Ανοικτού κώδικα σύστημα σχεσιακής βάσης δεδομένων που υποστηρίζεται ως managed υπηρεσία στο Azure.

**Provisioning**  
Η διαδικασία δημιουργίας και ρύθμισης πόρων Azure που ορίζονται σε πρότυπα υποδομής.

## Q

**Quota**  
Όρια στον αριθμό πόρων που μπορούν να δημιουργηθούν σε μια συνδρομή ή περιοχή Azure.

## R

**Resource Group**  
Λογικός κοντέινερ για πόρους Azure που μοιράζονται τον ίδιο κύκλο ζωής, δικαιώματα και πολιτικές.

**Resource Token**  
Μοναδική συμβολοσειρά που δημιουργείται από το azd για να διασφαλίσει ότι τα ονόματα πόρων είναι μοναδικά μεταξύ των αναπτύξεων.

**REST API**  
Αρχιτεκτονικό στυλ για το σχεδιασμό δικτυακών εφαρμογών χρησιμοποιώντας μεθόδους HTTP.

**Rollback**  
Διαδικασία επαναφοράς σε προηγούμενη έκδοση εφαρμογής ή ρύθμισης υποδομής.

## S

**Service**  
Συστατικό της εφαρμογής σας που ορίζεται στο azure.yaml (π.χ., web frontend, API backend, database).

**SKU**  
Stock Keeping Unit. Αντιπροσωπεύει διαφορετικά επίπεδα υπηρεσίας ή επιδόσεων για πόρους Azure.

**SQL Database**  
Managed σχεσιακή υπηρεσία βάσης δεδομένων της Azure βασισμένη σε Microsoft SQL Server.

**Static Web Apps**  
Υπηρεσία της Azure για την κατασκευή και ανάπτυξη full-stack web εφαρμογών από αποθετήρια πηγαίου κώδικα.

**Storage Account**  
Υπηρεσία της Azure που παρέχει cloud storage για δεδομένα όπως blobs, files, queues και tables.

**Subscription**  
Κοντέινερ λογαριασμού Azure που περιέχει resource groups και πόρους, με σχετική χρέωση και διαχείριση πρόσβασης.

## T

**Template**  
Προ-χτισμένη δομή έργου που περιέχει κώδικα εφαρμογής, ορισμούς υποδομής και ρυθμίσεις για κοινά σενάρια.

**Terraform**  
Open-source εργαλείο Infrastructure as Code που υποστηρίζει πολλούς παρόχους cloud, συμπεριλαμβανομένου του Azure.

**Traffic Manager**  
DNS-based traffic load balancer της Azure για την κατανομή κίνησης σε παγκόσμιες περιοχές Azure.

## U

**URI**  
Uniform Resource Identifier. Συμβολοσειρά που προσδιορίζει έναν συγκεκριμένο πόρο.

**URL**  
Uniform Resource Locator. Τύπος URI που καθορίζει πού βρίσκεται ένας πόρος και πώς να τον ανακτήσετε.

## V

**Virtual Network (VNet)**  
Θεμελιώδες δομικό στοιχείο για ιδιωτικά δίκτυα στο Azure, παρέχοντας απομόνωση και τμηματοποίηση.

**VS Code**  
Visual Studio Code. Δημοφιλής επεξεργαστής κώδικα με εξαιρετική ενσωμάτωση για Azure και azd.

## W

**Webhook**  
HTTP callback που ενεργοποιείται από συγκεκριμένα συμβάντα, συνήθως χρησιμοποιείται σε CI/CD pipelines.

**What-if**  
Λειτουργία της Azure που δείχνει ποιες αλλαγές θα γίνουν από μια ανάπτυξη χωρίς να τις εκτελέσει πραγματικά.

## Y

**YAML**  
YAML Ain't Markup Language. Ανθρώπινα αναγνώσιμο πρότυπο σειριοποίησης δεδομένων που χρησιμοποιείται για αρχεία ρυθμίσεων όπως το azure.yaml.

## Z

**Zone**  
Φυσικά ξεχωριστές τοποθεσίες μέσα σε μια περιοχή Azure που παρέχουν πλεονασμό και υψηλή διαθεσιμότητα.

## Όροι Ειδικού Πλαισίου

### Όροι Ανάπτυξης
- **Hot Reload**: Αυτόματη ενημέρωση εφαρμογών κατά την ανάπτυξη χωρίς επανεκκίνηση
- **Build Pipeline**: Αυτοματοποιημένη διαδικασία για την κατασκευή και τον έλεγχο του κώδικα
- **Deployment Slot**: Περιβάλλον staging εντός ενός App Service
- **Environment Parity**: Διατήρηση ομοιότητας μεταξύ περιβαλλόντων development, staging και production

### Όροι Ασφάλειας
- **Managed Identity**: Δυνατότητα της Azure που παρέχει αυτόματη διαχείριση διαπιστευτηρίων
- **Key Vault**: Ασφαλής αποθήκευση για μυστικά, κλειδιά και πιστοποιητικά
- **RBAC**: Έλεγχος πρόσβασης βάσει ρόλων για πόρους Azure
- **Network Security Group**: Εικονικό firewall για έλεγχο της δικτυακής κίνησης

### Όροι Παρακολούθησης
- **Telemetry**: Αυτόματη συλλογή μετρήσεων και δεδομένων
- **Application Performance Monitoring (APM)**: Παρακολούθηση της απόδοσης του λογισμικού
- **Log Analytics**: Υπηρεσία για τη συλλογή και ανάλυση δεδομένων καταγραφής
- **Alert Rules**: Αυτόματες ειδοποιήσεις με βάση μετρήσεις ή συνθήκες

### Όροι Ανάπτυξης
- **Blue-Green Deployment**: Στρατηγική ανάπτυξης χωρίς διακοπή υπηρεσίας
- **Canary Deployment**: Σταδιακή κυκλοφορία σε υποσύνολο χρηστών
- **Rolling Update**: Διαδοχική αντικατάσταση των instances της εφαρμογής
- **Rollback**: Επαναφορά σε προηγούμενη έκδοση εφαρμογής
