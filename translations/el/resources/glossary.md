## Εισαγωγή

Αυτό το ολοκληρωμένο γλωσσάρι παρέχει ορισμούς για όρους, έννοιες και ακρωνύμια που χρησιμοποιούνται στο Azure Developer CLI και στην ανάπτυξη στο cloud της Azure. Απαραίτητη αναφορά για την κατανόηση τεχνικής τεκμηρίωσης, την αντιμετώπιση προβλημάτων και την αποτελεσματική επικοινωνία σχετικά με έργα azd και υπηρεσίες Azure.

## Στόχοι Μάθησης

Χρησιμοποιώντας αυτό το γλωσσάρι, θα:
- Κατανοήσετε βασική ορολογία και έννοιες του Azure Developer CLI
- Εξοικειωθείτε με το λεξιλόγιο και τους τεχνικούς όρους της ανάπτυξης στο Azure
- Ανατρέχετε αποτελεσματικά σε όρους υποδομής ως κώδικα και όρους ανάπτυξης
- Κατανοήσετε τα ονόματα υπηρεσιών της Azure, τα ακρωνύμια και τις λειτουργίες τους
- Έχετε πρόσβαση σε ορισμούς για όρους αντιμετώπισης προβλημάτων και αποσφαλμάτωσης
- Μάθετε προχωρημένες έννοιες αρχιτεκτονικής και ανάπτυξης στο Azure

## Αποτελέσματα Μάθησης

Με τακτική αναφορά σε αυτό το γλωσσάρι, θα μπορείτε να:
- Επικοινωνείτε αποτελεσματικά χρησιμοποιώντας τη σωστή ορολογία του Azure Developer CLI
- Κατανοείτε την τεχνική τεκμηρίωση και τα μηνύματα σφάλματος με μεγαλύτερη σαφήνεια
- Πλοηγείστε στις υπηρεσίες και τις έννοιες της Azure με αυτοπεποίθηση
- Αντιμετωπίζετε προβλήματα χρησιμοποιώντας κατάλληλο τεχνικό λεξιλόγιο
- Συμβάλλετε σε συζητήσεις ομάδας με ακριβή τεχνική ορολογία
- Επεκτείνετε συστηματικά τις γνώσεις σας στην ανάπτυξη στο cloud της Azure

## A

**Πρότυπο ARM**  
Πρότυπο Azure Resource Manager. Μορφή Infrastructure as Code βασισμένη σε JSON που χρησιμοποιείται για τον ορισμό και την ανάπτυξη πόρων Azure με δηλωτικό τρόπο.

**App Service**  
Η πλατφόρμα ως υπηρεσία (PaaS) της Azure για τη φιλοξενία web εφαρμογών, REST API και mobile backends χωρίς διαχείριση υποδομής.

**Application Insights**  
Η υπηρεσία παρακολούθησης απόδοσης εφαρμογών (APM) της Azure που παρέχει βαθιές πληροφορίες για την απόδοση, τη διαθεσιμότητα και τη χρήση των εφαρμογών.

**Azure CLI**  
Διεπαφή γραμμής εντολών για τη διαχείριση πόρων Azure. Χρησιμοποιείται από το azd για πιστοποίηση και ορισμένες λειτουργίες.

**Azure Developer CLI (azd)**  
Εργαλείο γραμμής εντολών προσανατολισμένο σε προγραμματιστές που επιταχύνει τη διαδικασία δημιουργίας και ανάπτυξης εφαρμογών στην Azure χρησιμοποιώντας πρότυπα και Infrastructure as Code.

**azure.yaml**  
Το κύριο αρχείο ρυθμίσεων για ένα έργο azd που ορίζει υπηρεσίες, υποδομή και hooks ανάπτυξης.

**Azure Resource Manager (ARM)**  
Η υπηρεσία ανάπτυξης και διαχείρισης της Azure που παρέχει ένα επίπεδο διαχείρισης για τη δημιουργία, ενημέρωση και διαγραφή πόρων.

## B

**Bicep**  
Εξειδικευμένη γλώσσα τομέα (DSL) που αναπτύχθηκε από τη Microsoft για την ανάπτυξη πόρων Azure. Παρέχει απλούστερη σύνταξη σε σχέση με τα πρότυπα ARM ενώ μεταγλωττίζεται σε ARM.

**Build**  
Η διαδικασία μεταγλώττισης του πηγαίου κώδικα, εγκατάστασης εξαρτήσεων και προετοιμασίας εφαρμογών για ανάπτυξη.

**Blue-Green Deployment**  
Στρατηγική ανάπτυξης που χρησιμοποιεί δύο όμοια παραγωγικά περιβάλλοντα (blue και green) για να ελαχιστοποιήσει τον χρόνο διακοπής και τον κίνδυνο.

## C

**Container Apps**  
Η serverless υπηρεσία κοντέινερ της Azure που επιτρέπει την εκτέλεση containerized εφαρμογών χωρίς διαχείριση σύνθετης υποδομής.

**CI/CD**  
Συνεχής Ενσωμάτωση/Συνεχής Ανάπτυξη. Αυτοματοποιημένες πρακτικές για την ενσωμάτωση αλλαγών κώδικα και την ανάπτυξη εφαρμογών.

**Cosmos DB**  
Η παγκόσμια κατανεμημένη, πολυμοντέλου υπηρεσία βάσης δεδομένων της Azure που παρέχει ολοκληρωμένα SLA για διαμέριση, καθυστέρηση, διαθεσιμότητα και συνέπεια.

**Configuration**  
Ρυθμίσεις και παράμετροι που ελέγχουν τη συμπεριφορά της εφαρμογής και τις επιλογές ανάπτυξης.

## D

**Deployment**  
Η διαδικασία εγκατάστασης και ρύθμισης εφαρμογών και των εξαρτήσεών τους στην στοχευόμενη υποδομή.

**Docker**  
Πλατφόρμα για την ανάπτυξη, αποστολή και εκτέλεση εφαρμογών χρησιμοποιώντας τεχνολογία containerization.

**Dockerfile**  
Αρχείο κειμένου που περιέχει εντολές για την κατασκευή ενός Docker container image.

## E

**Environment**  
Στόχος ανάπτυξης που αντιπροσωπεύει μια συγκεκριμένη εμφάνιση της εφαρμογής σας (π.χ., ανάπτυξη, staging, παραγωγή).

**Environment Variables**  
Τιμές ρυθμίσεων αποθηκευμένες ως ζεύγη κλειδιού-τιμής που οι εφαρμογές μπορούν να προσπελάσουν κατά την εκτέλεση.

**Endpoint**  
URL ή δικτυακή διεύθυνση όπου μπορεί να προσπελαστεί μια εφαρμογή ή υπηρεσία.

## F

**Function App**  
Η serverless υπηρεσία υπολογισμού της Azure που επιτρέπει την εκτέλεση κώδικα με βάση γεγονότα χωρίς διαχείριση υποδομής.

## G

**GitHub Actions**  
Πλατφόρμα CI/CD ενσωματωμένη με αποθετήρια GitHub για την αυτοματοποίηση ροών εργασίας.

**Git**  
Κατανεμημένο σύστημα ελέγχου εκδόσεων που χρησιμοποιείται για την παρακολούθηση αλλαγών στον πηγαίο κώδικα.

## H

**Hooks**  
Προσαρμοσμένα σενάρια ή εντολές που εκτελούνται σε συγκεκριμένα σημεία του κύκλου ζωής ανάπτυξης (preprovision, postprovision, predeploy, postdeploy).

**Host**  
Ο τύπος υπηρεσίας Azure όπου θα αναπτυχθεί μια εφαρμογή (π.χ., appservice, containerapp, function).

## I

**Infrastructure as Code (IaC)**  
Πρακτική ορισμού και διαχείρισης υποδομής μέσω κώδικα αντί για χειροκίνητες διαδικασίες.

**Init**  
Η διαδικασία αρχικοποίησης ενός νέου έργου azd, συνήθως από ένα πρότυπο.

## J

**JSON**  
JavaScript Object Notation. Μορφή ανταλλαγής δεδομένων που χρησιμοποιείται συνήθως για αρχεία ρυθμίσεων και απαντήσεις API.

**JWT**  
JSON Web Token. Πρότυπο για την ασφαλή μετάδοση πληροφοριών μεταξύ μερών ως αντικείμενο JSON.

## K

**Key Vault**  
Η υπηρεσία της Azure για την ασφαλή αποθήκευση και διαχείριση μυστικών, κλειδιών και πιστοποιητικών.

**Kusto Query Language (KQL)**  
Γλώσσα ερωτημάτων που χρησιμοποιείται για την ανάλυση δεδομένων στο Azure Monitor, στο Application Insights και σε άλλες υπηρεσίες Azure.

## L

**Load Balancer**  
Υπηρεσία που κατανέμει την εισερχόμενη δικτυακή κίνηση σε πολλούς διακομιστές ή instances.

**Log Analytics**  
Υπηρεσία της Azure για τη συλλογή, ανάλυση και αντίδραση σε τηλεμετρία από cloud και on-premises περιβάλλοντα.

## M

**Managed Identity**  
Δυνατότητα της Azure που παρέχει στις υπηρεσίες Azure μια αυτόματα διαχειριζόμενη ταυτότητα για πιστοποίηση σε άλλες υπηρεσίες Azure.

**Microservices**  
Αρχιτεκτονική προσέγγιση όπου οι εφαρμογές δημιουργούνται ως σύνολο μικρών, ανεξάρτητων υπηρεσιών.

**Monitor**  
Ενοποιημένη λύση παρακολούθησης της Azure που παρέχει παρατηρησιμότητα πλήρους στοίβας σε εφαρμογές και υποδομή.

## N

**Node.js**  
Χρόνος εκτέλεσης JavaScript βασισμένος στη μηχανή V8 της Chrome για την κατασκευή server-side εφαρμογών.

**npm**  
Διαχειριστής πακέτων για το Node.js που διαχειρίζεται εξαρτήσεις και πακέτα.

## O

**Output**  
Τιμές που επιστρέφονται από ανάπτυξη υποδομής και μπορούν να χρησιμοποιηθούν από εφαρμογές ή άλλους πόρους.

## P

**Package**  
Η διαδικασία προετοιμασίας του κώδικα εφαρμογής και των εξαρτήσεων για ανάπτυξη.

**Parameters**  
Τιμές εισόδου που περνιούνται σε πρότυπα υποδομής για προσαρμογή της ανάπτυξης.

**PostgreSQL**  
Ανοικτού κώδικα σχεσιακό σύστημα βάσης δεδομένων που υποστηρίζεται ως διαχειριζόμενη υπηρεσία στην Azure.

**Provisioning**  
Η διαδικασία δημιουργίας και ρύθμισης πόρων Azure που ορίζονται σε πρότυπα υποδομής.

## Q

**Quota**  
Όρια ως προς το πλήθος πόρων που μπορούν να δημιουργηθούν σε μια συνδρομή ή περιοχή Azure.

## R

**Resource Group**  
Λογικό δοχείο για πόρους Azure που μοιράζονται τον ίδιο κύκλο ζωής, δικαιώματα και πολιτικές.

**Resource Token**  
Μοναδική συμβολοσειρά που δημιουργείται από το azd για να διασφαλίσει ότι τα ονόματα πόρων είναι μοναδικά σε αναπτύξεις.

**REST API**  
Αρχιτεκτονικό στυλ για το σχεδιασμό δικτυακών εφαρμογών χρησιμοποιώντας μεθόδους HTTP.

**Rollback**  
Διαδικασία επαναφοράς σε προηγούμενη έκδοση εφαρμογής ή διαμόρφωσης υποδομής.

## S

**Service**  
Συστατικό της εφαρμογής σας που ορίζεται στο azure.yaml (π.χ., web frontend, API backend, database).

**SKU**  
Stock Keeping Unit. Αντιπροσωπεύει διαφορετικά επίπεδα υπηρεσίας ή επιδόσεων για πόρους Azure.

**SQL Database**  
Η διαχειριζόμενη σχεσιακή υπηρεσία βάσης δεδομένων της Azure βασισμένη σε Microsoft SQL Server.

**Static Web Apps**  
Υπηρεσία της Azure για την κατασκευή και ανάπτυξη πλήρους στοίβας web εφαρμογών από αποθετήρια πηγαίου κώδικα.

**Storage Account**  
Υπηρεσία της Azure που παρέχει αποθήκευση στο cloud για αντικείμενα δεδομένων όπως blobs, files, queues και tables.

**Subscription**  
Δοχείο λογαριασμού Azure που περιέχει resource groups και πόρους, με σχετική χρέωση και διαχείριση πρόσβασης.

## T

**Template**  
Προ-διαμορφωμένη δομή έργου που περιέχει κώδικα εφαρμογής, ορισμούς υποδομής και ρυθμίσεις για κοινά σενάρια.

**Terraform**  
Ανοιχτού κώδικα εργαλείο Infrastructure as Code που υποστηρίζει πολλούς παρόχους cloud συμπεριλαμβανομένης της Azure.

**Traffic Manager**  
DNS-based load balancer της Azure για την κατανομή κίνησης σε παγκόσμιες περιοχές Azure.

## U

**URI**  
Uniform Resource Identifier. Συμβολοσειρά που αναγνωρίζει έναν συγκεκριμένο πόρο.

**URL**  
Uniform Resource Locator. Τύπος URI που δηλώνει πού βρίσκεται ένας πόρος και πώς να τον ανακτήσετε.

## V

**Virtual Network (VNet)**  
Βασικό συστατικό για ιδιωτικά δίκτυα στην Azure, παρέχοντας απομόνωση και τμηματοποίηση.

**VS Code**  
Visual Studio Code. Δημοφιλής κειμενογράφος κώδικα με εξαιρετική ενσωμάτωση Azure και azd.

## W

**Webhook**  
HTTP callback που ενεργοποιείται από συγκεκριμένα γεγονότα, συνήθως χρησιμοποιείται σε CI/CD pipelines.

**What-if**  
Δυνατότητα της Azure που δείχνει ποιες αλλαγές θα γίνουν από μια ανάπτυξη χωρίς να τις εκτελέσει πραγματικά.

## Y

**YAML**  
YAML Ain't Markup Language. Πρότυπο σειριοποίησης δεδομένων κατανοητό από ανθρώπους που χρησιμοποιείται για αρχεία ρυθμίσεων όπως το azure.yaml.

## Z

**Zone**  
Φυσικά ξεχωριστές τοποθεσίες εντός μιας περιοχής Azure που παρέχουν πλεονασμό και υψηλή διαθεσιμότητα.

## Όροι με Συγκεκριμένο Πλαίσιο

### Όροι Ανάπτυξης
- **Hot Reload**: Αυτόματη ενημέρωση των εφαρμογών κατά την ανάπτυξη χωρίς επανεκκίνηση
- **Build Pipeline**: Αυτοματοποιημένη διαδικασία για τη δημιουργία και το testing του κώδικα
- **Deployment Slot**: Σταδιακό περιβάλλον εντός μιας App Service
- **Environment Parity**: Διατήρηση παρόμοιων περιβαλλόντων ανάπτυξης, staging και παραγωγής

### Όροι Ασφαλείας
- **Managed Identity**: Δυνατότητα της Azure που παρέχει αυτόματη διαχείριση διαπιστευτηρίων
- **Key Vault**: Ασφαλής αποθήκευση για μυστικά, κλειδιά και πιστοποιητικά
- **RBAC**: Έλεγχος πρόσβασης βάσει ρόλων για πόρους Azure
- **Network Security Group**: Εικονικό firewall για τον έλεγχο της δικτυακής κίνησης

### Όροι Παρακολούθησης
- **Telemetry**: Αυτόματη συλλογή μετρήσεων και δεδομένων
- **Application Performance Monitoring (APM)**: Παρακολούθηση της απόδοσης λογισμικού
- **Log Analytics**: Υπηρεσία για τη συλλογή και ανάλυση δεδομένων καταγραφής
- **Alert Rules**: Αυτόματες ειδοποιήσεις βάσει μετρικών ή συνθηκών

### Όροι Ανάπτυξης
- **Blue-Green Deployment**: Στρατηγική ανάπτυξης χωρίς διακοπή
- **Canary Deployment**: Σταδιακή κυκλοφορία σε υποσύνολο χρηστών
- **Rolling Update**: Διαδοχική αντικατάσταση instances της εφαρμογής
- **Rollback**: Επαναφορά σε προηγούμενη έκδοση της εφαρμογής
