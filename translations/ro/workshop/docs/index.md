# Atelier AZD pentru Dezvoltatori AI

> Validat cu `azd 1.27.1` în iulie 2026.

> [!IMPORTANT]  
> **Acest atelier este dotat cu un ghid de atelier pe care îl puteți previzualiza în browserul local. Pentru a începe, deschideți repo-ul în GitHub Codespaces, un container de dezvoltare sau o clonă locală cu MkDocs instalat—apoi așteptați până când vedeți un terminal activ VS Code și tastați:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Ar trebui să vedeți un dialog pop-up pentru a deschide pagina de previzualizare în browser.**

Bine ați venit la atelierul practic pentru învățarea Azure Developer CLI (AZD) cu accent pe implementarea aplicațiilor AI. Acest atelier vă ajută să dobândiți o înțelegere aplicată a șabloanelor AZD în 3 pași:

1. **Descoperire** - găsiți șablonul potrivit pentru dvs.
1. **Implementare** - implementați și validați că funcționează
1. **Personalizare** - modificați și iterați pentru a-l face al vostru!

Pe parcursul acestui atelier, veți fi introdus și în instrumentele și fluxurile de lucru de bază pentru dezvoltatori, pentru a vă ajuta să vă simplificați călătoria de dezvoltare de la început până la sfârșit.

## Prezentare Atelier

**Durată:** 3-4 ore  
**Nivel:** Începător până la Intermediar  
**Precondiții:** Familiaritate cu Azure, concepte AI, VS Code și unelte de linie de comandă.

Acesta este un atelier practic în care învățați făcând. După ce ați finalizat exercițiile, recomandăm să parcurgeți curriculumul AZD Pentru Începători pentru a continua călătoria dvs. de învățare către bune practici de Securitate și Productivitate.

| Timp| Modul  | Obiectiv |
|:---|:---|:---|
| 15 minute | Introducere | Stabiliți cadrul, înțelegeți obiectivele |
| 30 minute | Selectați șablonul AI | Explorați opțiunile și alegeți unul de început | 
| 30 minute | Validați șablonul AI | Implementați soluția implicită în Azure |
| 30 minute | Deconstruiți șablonul AI | Explorați structura și configurarea |
| 30 minute | Configurați șablonul AI | Activați și încercați funcțiile disponibile |
| 30 minute | Personalizați șablonul AI | Adaptați șablonul la nevoile dvs. |
| 30 minute | Demontați infrastructura | Curățare și eliberare resurse |
| 15 minute | Încheiere și pași următori | Resurse de învățare, provocarea atelierului |
| | |

## Ce veți învăța

Gândiți-vă la șablonul AZD ca la un spațiu de joacă pentru învățare, unde puteți explora diverse capabilități și unelte pentru dezvoltarea end-to-end pe Microsoft Foundry. Până la finalul acestui atelier, ar trebui să aveți o înțelegere intuitivă a diferitelor unelte și concepte în acest context.

| Concept  | Obiectiv |
|:---|:---|
| **Azure Developer CLI** | Înțelegerea comenzilor și fluxurilor de lucru ale uneltei |
| **Șabloane AZD** | Înțelegerea structurii proiectului și configurarea |
| **Agent Azure AI** | Provisionare și implementare proiect Microsoft Foundry |
| **Căutare Azure AI** | Activarea ingineriei contextuale cu agenți |
| **Observabilitate** | Explorarea tragerii, monitorizării și evaluărilor |
| **Red Teaming** | Explorarea testării adversariale și a măsurilor |
| | |
