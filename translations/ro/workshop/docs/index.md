# Atelier AZD pentru Dezvoltatori AI

> Validat cu `azd 1.25.6` în iunie 2026.

> [!IMPORTANT]  
> **Acest atelier este dotat cu un ghid de atelier pe care îl puteți previzualiza în browserul local. Pentru a începe, deschideți repo-ul în GitHub Codespaces, un container dev sau un clon local cu MkDocs instalat—apoi așteptați până când vedeți un terminal activ VS Code și tastați:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Ar trebui să vedeți un dialog pop-up pentru a deschide pagina de previzualizare în browser.**

Bine ați venit la atelierul practic pentru a învăța Azure Developer CLI (AZD) cu accent pe implementarea aplicațiilor AI. Acest atelier vă ajută să dobândiți o înțelegere aplicată a șabloanelor AZD în 3 pași:

1. **Descoperire** - găsiți șablonul potrivit pentru dvs.
1. **Implementare** - implementați și validați că funcționează
1. **Personalizare** - modificați și iterați pentru a-l face al dvs.!

Pe parcursul acestui atelier, veți fi introdus și la instrumentele și fluxurile de lucru esențiale pentru dezvoltatori, pentru a vă ajuta să eficientizați întregul parcurs de dezvoltare.

## Prezentarea Atelierului

**Durată:** 3-4 ore  
**Nivel:** Începător până la Intermediar  
**Prerechizite:** Familiaritate cu Azure, concepte AI, VS Code și instrumente de linie de comandă.

Acesta este un atelier practic în care învățați făcând. După ce ați terminat exercițiile, vă recomandăm să parcurgeți curricula AZD Pentru Începători pentru a continua călătoria de învățare către bune practici de Securitate și Productivitate.

| Timp | Modul  | Obiectiv |
|:---|:---|:---|
| 15 min | Introducere | Stabiliți contextul, înțelegeți obiectivele |
| 30 min | Selectare Șablon AI | Explorați opțiunile și alegeți unul de pornire | 
| 30 min | Validare Șablon AI | Implementați soluția implicită în Azure |
| 30 min | Deconstrucție Șablon AI | Explorați structura și configurația |
| 30 min | Configurare Șablon AI | Activați și testați funcționalitățile disponibile |
| 30 min | Personalizare Șablon AI | Adaptați șablonul nevoilor dvs. |
| 30 min | Demontare Infrastructură | Curățare și eliberare resurse |
| 15 min | Încheiere & Pașii următori | Resurse de învățare, provocarea atelierului |
| | |

## Ce Veți Învăța

Gândiți-vă la Șablonul AZD ca la un sandbox de învățare pentru a explora diverse capabilități și instrumente pentru dezvoltarea end-to-end pe Microsoft Foundry. La finalul acestui atelier, ar trebui să aveți o senzație intuitivă despre diferite instrumente și concepte în acest context.

| Concept  | Obiectiv |
|:---|:---|
| **Azure Developer CLI** | Înțelegeți comenzile și fluxurile de lucru ale instrumentului |
| **Șabloane AZD**| Înțelegeți structura proiectului și configurația |
| **Agent Azure AI**| Provisionați & implementați proiectul Microsoft Foundry  |
| **Azure AI Search**| Activați ingineria contextului cu agenți |
| **Observabilitate**| Explorați trasabilitatea, monitorizarea și evaluările |
| **Red Teaming**| Explorați testarea adversarială și măsurile de atenuare |
| | |
