# Atelier AZD pentru Dezvoltatori AI

> Validat cu `azd 1.23.12` în martie 2026.

> [!IMPORTANT]  
> **Acest atelier include un ghid interactiv pe care îl puteți previzualiza în browserul local. Pentru a începe, deschideți repo-ul în GitHub Codespaces, un dev container sau o clonă locală cu MkDocs instalat—apoi așteptați până când vedeți un terminal VS Code activ și tastați:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Ar trebui să vedeți un dialog pop-up pentru a deschide pagina de previzualizare într-un browser.**

Bine ați venit la atelierul practic pentru învățarea Azure Developer CLI (AZD) cu accent pe implementarea aplicațiilor AI. Acest atelier vă ajută să dobândiți o înțelegere aplicată a șabloanelor AZD în 3 pași:

1. **Descoperire** - găsiți șablonul care vi se potrivește.
1. **Implementare** - implementați și validați că funcționează
1. **Personalizare** - modificați și iterați pentru a-l face al dumneavoastră!

Pe parcursul acestui atelier, veți fi introdus în instrumentele și fluxurile de lucru principale pentru dezvoltatori, pentru a vă ajuta să eficientizați parcursul de dezvoltare end-to-end.

## Prezentare generală a atelierului

**Durată:** 3-4 ore  
**Nivel:** Începător până la Intermediar  
**Precondiții:** Familiaritate cu Azure, conceptele AI, VS Code și instrumentele de linie de comandă.

Acesta este un atelier practic unde învățați prin practică. După ce ați finalizat exercițiile, vă recomandăm să parcurgeți curriculumul AZD pentru începători pentru a continua parcursul de învățare către bunele practici în Securitate și Productivitate.

| Timp| Modul  | Obiectiv |
|:---|:---|:---|
| 15 minute | Introducere | Stabilește contextul, înțelege obiectivele |
| 30 minute | Selectați șablonul AI | Explorați opțiunile și alegeți un șablon inițial | 
| 30 minute | Validați șablonul AI | Implementați soluția implicită în Azure |
| 30 minute | Deconstruiți șablonul AI | Explorați structura și configurația |
| 30 minute | Configurați șablonul AI | Activați și încercați funcționalitățile disponibile |
| 30 minute | Personalizați șablonul AI | Adaptați șablonul la nevoile dumneavoastră |
| 30 minute | Demontarea infrastructurii | Curățați și eliberați resursele |
| 15 minute | Încheiere și pașii următori | Resurse de învățare, provocarea atelierului |
| | |

## Ce veți învăța

Gândiți-vă la Șablonul AZD ca la un mediu de învățare (sandbox) pentru a explora diverse capabilități și instrumente pentru dezvoltare end-to-end pe Microsoft Foundry. La sfârșitul acestui atelier, ar trebui să aveți o înțelegere intuitivă a diferitelor instrumente și concepte în acest context.

| Concept  | Obiectiv |
|:---|:---|
| **Azure Developer CLI** | Înțelegeți comenzile instrumentului și fluxurile de lucru |
| **Șabloane AZD**| Înțelegeți structura proiectului și configurarea |
| **Azure AI Agent**| Provisionați și implementați proiectul Microsoft Foundry  |
| **Azure AI Search**| Activați ingineria contextului cu agenți |
| **Observabilitate**| Explorați urmărirea, monitorizarea și evaluările |
| **Red Teaming**| Explorați testarea adversarială și măsurile de atenuare |
| | |
