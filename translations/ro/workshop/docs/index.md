# Atelier AZD pentru Dezvoltatori AI

> [!IMPORTANT]  
> **Acest atelier este dotat cu un ghid de atelier pe care îl puteți previzualiza în browserul local. Pentru a începe, trebuie să lansați GitHub Codespaces pe repository—apoi așteptați până când vedeți un terminal VS Code activ și tastați:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Ar trebui să vedeți o fereastră pop-up pentru a deschide pagina de previzualizare într-un browser.**

Bine ați venit la atelierul practic pentru a învăța Azure Developer CLI (AZD) cu accent pe implementarea aplicațiilor AI. Acest atelier vă ajută să dobândiți o înțelegere aplicată a șabloanelor AZD în 3 pași:

1. **Descoperire** - găsiți șablonul potrivit pentru dvs.
1. **Implementare** - implementați și validați că funcționează
1. **Personalizare** - modificați și iterați pentru a-l face al dvs.!

Pe parcursul acestui atelier, veți fi introdus și în unelte și fluxuri de lucru esențiale pentru dezvoltatori, pentru a vă ajuta să vă optimizați călătoria completă de dezvoltare.

## Prezentare generală a atelierului

**Durată:** 3-4 ore  
**Nivel:** Începător până la Intermediar  
**Prerechizite:** Familiaritate cu Azure, concepte AI, VS Code și unelte de linie de comandă.

Acesta este un atelier practic în care învățați prin aplicare. După ce ați terminat exercițiile, recomandăm să parcurgeți curriculum-ul AZD pentru începători pentru a continua călătoria de învățare despre bunele practici în Securitate și Productivitate.

| Timp| Modul  | Obiectiv |
|:---|:---|:---|
| 15 min | Introducere | Stabiliți cadrul, înțelegeți obiectivele |
| 30 min | Selectați șablon AI | Explorați opțiunile și alegeți un șablon de start | 
| 30 min | Validați șablon AI | Implementați soluția implicită în Azure |
| 30 min | Deconstruiți șablon AI | Explorați structura și configurația |
| 30 min | Configurați șablon AI | Activați și testați funcționalitățile disponibile |
| 30 min | Personalizați șablon AI | Adaptați șablonul nevoilor dvs. |
| 30 min | Demontați infrastructura | Curățare și eliberarea resurselor |
| 15 min | Încheiere & Pașii următori | Resurse de învățare, provocarea workshop-ului |
| | |

## Ce veți învăța

Gândiți-vă la Șablonul AZD ca la un spațiu de învățare pentru a explora diverse capabilități și unelte pentru dezvoltarea completă pe Microsoft Foundry. La finalul acestui atelier, ar trebui să aveți o înțelegere intuitivă a diverselor unelte și concepte din acest context.

| Concept  | Obiectiv |
|:---|:---|
| **Azure Developer CLI** | Înțelegeți comenzile și fluxurile de lucru ale uneltei|
| **Șabloane AZD**| Înțelegeți structura proiectului și configurația|
| **Agent Azure AI**| Provisionați și implementați un proiect Microsoft Foundry  |
| **Căutare Azure AI**| Activați ingineria contextuală cu agenți |
| **Observabilitate**| Explorați urmărirea, monitorizarea și evaluările |
| **Red Teaming**| Explorați testarea adversarială și măsurile de atenuare |
| | |
