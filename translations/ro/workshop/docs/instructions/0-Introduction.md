# AZD pentru Dezvoltatori AI

!!! info "PUBLICUL ȚINTĂ PENTRU ACEST WORKSHOP"
   
    Acest workshop este destinat oricui dorește să construiască aplicații AI Agentic pe Microsoft Foundry prin utilizarea celor mai bune instrumente și fluxuri de lucru. Workshop-ul este potrivit atât pentru începători, cât și pentru dezvoltatori experimentați care au unele cunoștințe despre Azure, concepte AI și cod Python. Vă rugăm să verificați cerințele prealabile înainte de a începe.

    **Durata:** 3-4 ore  
    **Nivel:** Începător până la Intermediar  

---


## Cerințe Prealabile

- Un cont personal GitHub → [Creează unul gratuit](https://github.com/signup)
- Un cont Azure activ → [Creează unul gratuit](https://aka.ms/free)
- Familiaritate cu conceptele AI, dezvoltare Python, instrumente de linie de comandă
- Experiență cu instrumentele și fluxurile de lucru VS Code și GitHub.

---

## Obiective de Învățare

La finalul acestui workshop, veți ști cum să:

- [X] **Lucrați cu Azure Developer CLI** → pentru a provisiona, implementa și gestiona resursele Azure.
- [X] **Utilizați Șabloane pentru Aplicații AI** → pentru a începe rapid proiecte AI Agent pe Microsoft Foundry.
- [X] **Configurați Șabloane** → pentru caracteristici precum Red Teaming, Evaluări, Urmărire și Monitorizare
- [X] **Deconstruiți și Explorați** → șabloanele și fluxurile AZD folosind GitHub Copilot
- [X] **Personalizați și Redeploați** → șablonul de start AI Agent, cu cerințele dvs. personalizate

---

## Structura Workshop-ului

Workshop-ul este structurat în module, fiecare ancorat de un exercițiu practic:

1. **Introducere** - Obțineți o înțelegere a ceea ce sunt șabloanele AZD și cum funcționează
1. **Selecție** - Descoperiți șablonul „starter” potrivit pentru următorul vostru proiect AI
1. **Validare** - Implementați și explorați șablonul pentru a verifica dacă funcționează conform descrierii
1. **Deconstrucție** - Demontați șablonul pentru a înțelege componentele și configurația sa
1. **Configurare** - Învațați să activați și să utilizați diferite caracteristici Microsoft Foundry
1. **Personalizare** - Reconstruiți aplicația având în vedere datele și cerințele dvs.
1. **Demontare** - Învațați să dezprovisionați infrastructura și să recuperați resursele șterse moale
