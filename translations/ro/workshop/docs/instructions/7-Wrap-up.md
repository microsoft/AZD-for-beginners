## 2. Concepte Cheie Stăpânite

### Azure Developer CLI (azd)
- `azd init` - Inițializează proiecte din șabloane
- `azd up` - Provisionare și implementare cu o singură comandă
- `azd env` - Gestionarea variabilelor de mediu
- `azd down` - Curățarea resurselor în siguranță

### Structura Șabloanelor AZD
- `azure.yaml` - Configurare și hook-uri de implementare
- `infra/` - Infrastructură ca cod Bicep
- `src/` - Cod sursă al aplicației
- `scripts/` - Scripturi de automatizare și configurare

### Capacități Microsoft Foundry
- Serviciu AI Agent pentru asistenți inteligenți
- File Search și Azure AI Search pentru recuperarea cunoștințelor
- Trasare, monitorizare și evaluare integrate
- Red teaming pentru validarea siguranței AI
