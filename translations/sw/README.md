## Muhtasari wa Kozi

Shinda Azure Developer CLI (azd) kupitia sura zilizopangwa kwa kujifunza hatua kwa hatua. **Kuzingatia maalum kwa uenezaji wa programu za AI kwa ujumuishaji wa Microsoft Foundry.**

### Kwa Nini Kozi Hii ni Muhimu kwa Waendelezaji wa Kisasa

Kulingana na maarifa ya jamii ya Discord ya Microsoft Foundry, **45% ya waendelezaji wanataka kutumia AZD kwa mizigo ya kazi ya AI** lakini wanakutana na changamoto na:
- Mimarisha tata ya huduma nyingi za AI
- Mbinu bora za uenezaji wa AI kwa uzalishaji  
- Ujumuishaji na usanidi wa huduma za Azure AI
- Uboreshaji wa gharama kwa mizigo ya kazi ya AI
- Utatuzi wa matatizo maalum ya uenezaji wa AI

### Malengo ya Kujifunza

Kwa kumaliza kozi hii iliyopangwa, ut:
- **Tumia Msingi wa AZD:** Misingi, usakinishaji, na usanidi
- **Endesha Programu za AI:** Tumia AZD na huduma za Microsoft Foundry
- **Tekeleza Miundombinu kama Msimbo:** Simamia rasilimali za Azure kwa templeti za Bicep
- **Tatua Matatizo ya Ueneaji:** Rekebisha masuala ya kawaida na debug
- **Boresha kwa Uzalishaji:** Usalama, upanuzi, ufuatiliaji, na usimamizi wa gharama
- **Tengeneza Suluhisho za Wakala Wengi:** Endesha miundo tata ya AI

**üí∞ Uelewa wa Gharama:** Elewa gharama za maendeleo $80-150/mwezi, gharama za uzalishaji $300-3500/mwezi

#### üí∞ Mambo ya Kuzingatia Kuhusu Gharama kwa Ueneaji wa AI

**Mazingira ya Maendeleo (Kadiria $80-150/mwezi):**
- Azure OpenAI (Pay-as-you-go): $0-50/mwezi (kulingana na matumizi ya tokeni)
- AI Search (Basic tier): $75/mwezi
- Container Apps (Consumption): $0-20/mwezi
- Storage (Standard): $1-5/mwezi

**Mazingira ya Uzalishaji (Kadiria $300-3,500+/mwezi):**
- Azure OpenAI (PTU for consistent performance): $3,000+/mwezi AU Pay-as-go kwa kiwango kikubwa
- AI Search (Standard tier): $250/mwezi
- Container Apps (Dedicated): $50-100/mwezi
- Application Insights: $5-50/mwezi
- Storage (Premium): $10-50/mwezi

**üí° Vidokezo vya Kuboresha Gharama:**
- Tumia **Free Tier** Azure OpenAI kwa kujifunza (50,000 tokens/mwezi included)
- Endesha `azd down` kuondoa rasilimali zilizotengwa wakati hauendelei kazi
- Anza na bili zinazotegemea matumizi, boresha kwenda PTU tu kwa uzalishaji
- Tumia `azd provision --preview` kukadiria gharama kabla ya ueneaji
- Weka auto-scaling: lipa tu kwa matumizi halisi

**Ufuatiliaji wa Gharama:**
```bash
# Angalia makadirio ya gharama za kila mwezi
azd provision --preview

# Fuatilia gharama halisi katika Azure Portal
az consumption budget list --resource-group <your-rg>
```

## üìñ Azure Developer CLI ni nini?

Azure Developer CLI (azd) ni kiolesura cha amri kwa waendelezaji kinacholenga kuharakisha mchakato wa kujenga na kutekeleza programu kwenye Azure. Kinatoa:

- **Utekelezaji unaotegemea violezo** - Tumia violezo vilivyotengenezwa awali kwa mifumo ya kawaida ya programu
- **Miundombinu kama Msimbo** - Simamia rasilimali za Azure kwa kutumia Bicep au Terraform  
- **Mtiririko uliounganishwa** - Sanidi, tekeleza, na fuatililia programu bila mshono
- **Rafiki kwa Waendelezaji** - Imeboreshwa kwa ufanisi na uzoefu wa waendelezaji

## üîß Mwongozo wa Haraka wa Utatuzi wa Matatizo

**Masuala ya kawaida yanayokumbana nayo waanzilishi na suluhisho za haraka:**

### ‚ùå "azd: command not found"

```bash
# Sakinisha AZD kwanza
# Windows (PowerShell):
winget install microsoft.azd

# macOS:
brew tap azure/azd && brew install azd

# Linux:
curl -fsSL https://aka.ms/install-azd.sh | bash

# Thibitisha usakinishaji
azd version
```

### ‚ùå "No subscription found" or "Subscription not set"

```bash
# Orodhesha usajili zinazopatikana
az account list --output table

# Weka usajili wa chaguo-msingi
az account set --subscription "<subscription-id-or-name>"

# Weka kwa mazingira ya AZD
azd env set AZURE_SUBSCRIPTION_ID "<subscription-id>"

# Thibitisha
az account show
```

### ‚ùå "InsufficientQuota" or "Quota exceeded"

```bash
# Jaribu eneo tofauti la Azure
azd env set AZURE_LOCATION "westus2"
azd up

# Au tumia SKU ndogo zaidi katika maendeleo
# Hariri infra/main.parameters.json:
{
  "sku": "B1"  // Instead of "P1V2"
}
```

### ‚ùå "azd up" fails halfway through

```bash
# Chaguo 1: Safisha na jaribu tena
azd down --force --purge
azd up

# Chaguo 2: Rekebisha tu miundombinu
azd provision

# Chaguo 3: Angalia logi za kina
azd show
azd logs
```

### ‚ùå "Authentication failed" or "Token expired"

```bash
# Thibitisha tena
az logout
az login

azd auth logout
azd auth login

# Thibitisha uthibitisho
az account show
```

### ‚ùå "Resource already exists" or naming conflicts

```bash
# AZD hutengeneza majina ya kipekee, lakini ikiwa kuna mgongano:
azd down --force --purge

# Kisha jaribu tena kwa mazingira mapya
azd env new dev-v2
azd up
```

### ‚ùå Template deployment taking too long

**Muda wa kawaida wa kusubiri:**
- Simple web app: 5-10 minutes
- App with database: 10-15 minutes
- AI applications: 15-25 minutes (OpenAI provisioning is slow)

```bash
# Angalia maendeleo
azd show

# Ikiwa imekwama kwa zaidi ya dakika 30, angalia Azure Portal:
azd monitor
# Tafuta deployments zilizoshindwa
```

### ‚ùå "Permission denied" or "Forbidden"

```bash
# Angalia jukumu lako la Azure
az role assignment list --assignee $(az account show --query user.name -o tsv)

# Unahitaji angalau jukumu la "Contributor"
# Omba msimamizi wako wa Azure akupe:
# - Contributor (kwa rasilimali)
# - User Access Administrator (kwa ugawaji wa majukumu)
```

### ‚ùå Can't find deployed application URL

```bash
# Onyesha miisho yote ya huduma
azd show

# Au fungua Portal ya Azure
azd monitor

# Kagua huduma maalum
azd env get-values
# Tafuta vigezo vya *_URL
```
