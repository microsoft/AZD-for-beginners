# AZD 為 AI 開發者工作坊

> 於 2026 年 3 月以 `azd 1.23.12` 驗證通過。

> [!IMPORTANT]  
> **此工作坊搭配可在本機瀏覽器預覽的工作坊指南。開始之前，請在 GitHub Codespaces、開發容器或安裝了 MkDocs 的本機複本中打開此儲存庫 — 然後等待看到 VS Code 活動終端機並輸入：**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **您應該會看到跳出視窗提示，讓您在瀏覽器中打開預覽頁面。**

歡迎參加針對 Azure Developer CLI (AZD) 並以 AI 應用程式部署為焦點的實作工作坊。本工作坊以三個步驟幫助您獲得 AZD 模板的應用理解：

1. <strong>探索</strong> - 找出最適合您的模板。  
1. <strong>部署</strong> - 部署並驗證其運作。  
1. <strong>客製化</strong> - 修改並反覆調整，使其符合您的需求！

在工作坊的過程中，您也會被引介核心開發工具與工作流程，以幫助您簡化端對端的開發旅程。

## 工作坊概覽

**時間:** 3-4 小時  
**程度:** 新手到中階  
**先備知識:** 熟悉 Azure、AI 概念、VS Code 與命令列工具。

這是一個實作工作坊，您透過動手做來學習。完成練習後，我們建議您回顧 AZD 新手入門課程，以繼續您的學習旅程，深入安全性與生產力最佳實務。

| 時間| 模組  | 目標 |
|:---|:---|:---|
| 15 分鐘 | 介紹 | 鋪陳背景，了解目標 |
| 30 分鐘 | 選擇 AI 模板 | 探索選項並挑選起始模板 | 
| 30 分鐘 | 驗證 AI 模板 | 部署預設解決方案至 Azure |
| 30 分鐘 | 解構 AI 模板 | 探索架構與配置 |
| 30 分鐘 | 配置 AI 模板 | 啟用並嘗試現有功能 |
| 30 分鐘 | 客製化 AI 模板 | 調整模板以符合您的需求 |
| 30 分鐘 | 拆除基礎架構 | 清理並釋放資源 |
| 15 分鐘 | 總結與下一步 | 學習資源，工作坊挑戰 |
| | |

## 您將學習到

將 AZD 模板視為學習沙盒，探索 Microsoft Foundry 端對端開發的多種能力與工具。完成此工作坊後，您將對該情境中的各種工具與概念具備直覺理解。

| 概念  | 目標 |
|:---|:---|
| **Azure Developer CLI** | 了解工具指令與工作流程 |
| **AZD 模板**| 了解專案架構與配置 |
| **Azure AI Agent**| 佈建與部署 Microsoft Foundry 專案 |
| **Azure AI Search**| 啟用代理人上下文工程 |
| <strong>可觀察性</strong>| 探索追蹤、監控與評估 |
| <strong>紅隊演練</strong>| 探討對抗性測試與防禦措施 |
