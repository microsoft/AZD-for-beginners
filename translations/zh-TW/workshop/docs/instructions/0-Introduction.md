# AZD 適用於 AI 開發人員

!!! info "AUDIENCE FOR THIS WORKSHOP"
   
    本工作坊適合任何希望在 Microsoft Foundry 上，透過一流的工具與工作流程打造 AI Agentic 應用的人士。此工作坊適合既是初學者也有經驗的開發者，且對 Azure、AI 概念及 Python 程式有一定的熟悉度。請在開始前檢查先決條件。

    **持續時間：** 3-4 小時  
    **等級：** 初級到中級  

---


## 先決條件

- 個人 GitHub 帳戶 → [Create one for free](https://github.com/signup)
- 一個有效的 Azure 帳戶 → [Create one for free](https://aka.ms/free)
- 熟悉 AI 概念、Python 開發、命令列工具
- 有使用 VS Code 以及 GitHub 工具與工作流程的經驗。

---

## 學習目標

完成本工作坊後，您將學會如何：

- [X] **使用 Azure Developer CLI** → 佈建、部署並管理您的 Azure 資源。
- [X] **使用 AI 應用範本** →  讓您在 Microsoft Foundry 上快速啟動 AI 代理專案。
- [X] **設定範本** →  啟用像是紅隊測試、評估、追蹤與監控等功能
- [X] **拆解與探索** →  使用 GitHub Copilot 探索 AZD 範本與工作流程
- [X] **自訂並重新部署** →  AI 代理啟動範本，並加入您自訂的需求

---

## 工作坊大綱

本工作坊以模組化方式構成，每個模組都以實作實驗為核心：

1. **介紹** - 了解 AZD 範本是什麼，以及它們如何運作
1. **選擇** - 發現適合您下一個 AI 專案的 "starter" 範本
1. **驗證** - 部署並探索範本，以確認其如宣稱般運作
1. **拆解** - 將範本拆解以了解其組件與設定
1. **設定** - 學習如何啟用與使用不同的 Microsoft Foundry 功能
1. **自訂** - 依據您的資料與需求重構應用程式
1. **拆除** - 學習如何取消佈建基礎架構並回復軟刪除的資源
