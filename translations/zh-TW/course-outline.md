# AZD 初學者指南：課程大綱與學習框架

## 課程概述

透過結構化章節學習 Azure Developer CLI (azd)，**特別著重於整合 Microsoft Foundry 的 AI 應用部署**。

### 為什麼這門課程對現代開發者至關重要

根據 Microsoft Foundry Discord 社群的洞察，**45% 的開發者希望使用 AZD 處理 AI 工作負載**，但面臨以下挑戰：
- 複雜的多服務 AI 架構
- 生產環境 AI 部署的最佳實踐  
- Azure AI 服務的整合與配置
- AI 工作負載的成本優化
- AI 部署特定問題的排查

### 核心學習目標

完成這門結構化課程後，您將能夠：
- **掌握 AZD 基礎**：核心概念、安裝與配置
- **部署 AI 應用**：使用 AZD 與 Microsoft Foundry 服務
- **實現基礎設施即代碼**：使用 Bicep 模板管理 Azure 資源
- **排查部署問題**：解決常見問題並進行調試
- **優化生產環境**：安全性、擴展性、監控與成本管理
- **構建多代理解決方案**：部署複雜的 AI 架構

**適合個人開發者與持續學習者**

**特色：**
- **基於瀏覽器的介面**：通過任何瀏覽器訪問基於 MkDocs 的工作坊

- **社群支持**：可在 Azure Discord 頻道提問與合作

**學習結構：**

**開始學習（自學模式）：**
