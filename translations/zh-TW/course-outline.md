# AZD 新手入門：課程大綱與學習框架

## 課程概述

透過結構化章節掌握 Azure Developer CLI (azd) ，設計循序漸進學習。**特別聚焦於結合 Microsoft Foundry 的 AI 應用部署。**

### 為什麼本課程對現代開發者至關重要

根據 Microsoft Foundry Discord 社群見解，**45% 開發者希望使用 AZD 處理 AI 工作負載**但面臨挑戰包括：
- 複雜的多服務 AI 架構
- 生產環境 AI 部署最佳實踐  
- Azure AI 服務整合與配置
- AI 工作負載的成本優化
- AI 特定部署問題的故障排除

### 核心學習目標

完成本結構化課程後，您將能：
- **掌握 AZD 基礎**：核心概念、安裝與設定
- **部署 AI 應用程式**：結合 AZD 與 Microsoft Foundry 服務
- **實施基礎架構即程式碼**：使用 Bicep 模板管理 Azure 資源
- **排除部署問題**：解決常見錯誤與除錯
- **優化生產環境**：安全性、擴展性、監控與成本管理
- **建構多代理人解決方案**：部署複雜 AI 架構
