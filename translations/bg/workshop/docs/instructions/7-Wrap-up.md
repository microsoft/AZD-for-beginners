## 2. Ключови концепции

### Azure Developer CLI (azd)
- `azd init` - Инициализира проекти от шаблони
- `azd up` - Създава ресурси и разгръща с една команда
- `azd env` - Управлява променливите на средата
- `azd down` - Премахва ресурсите безопасно

### Структура на AZD шаблона
- `azure.yaml` - Конфигурация и hook-ове за разгръщане
- `infra/` - Bicep инфраструктура като код
- `src/` - Изходен код на приложението
- `scripts/` - Скриптове за автоматизация и настройка

### Възможности на Microsoft Foundry
- Услуга AI Agent за интелигентни асистенти
- File Search и Azure AI Search за извличане на знания
- Вградени възможности за трасировка, мониторинг и оценка
- Red teaming за проверка на безопасността на AI
