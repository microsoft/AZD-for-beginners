# Глава 8: Производствени и корпоративни модели

**📚 Курс**: [AZD за начинаещи](../../README.md) | **⏱️ Продължителност**: 2-3 часа | **⭐ Сложност**: Напреднало

## Преглед

Тази глава обхваща модели за внедряване, подходящи за предприятия, засилване на сигурността, мониторинг и оптимизация на разходите за производствени AI натоварвания.

## Цели на обучението

След завършване на тази глава ще:
- Внедрите устойчиви приложения в няколко региона
- Реализирате корпоративни модели за сигурност
- Конфигурирате цялостен мониторинг
- Оптимизирате разходите в мащаб
- Настроите CI/CD конвейери с AZD

## 🔐 Най-добри практики за сигурност

```bicep
// Use managed identity
identity: {
  type: 'SystemAssigned'
}

// Private endpoints for AI services
properties: {
  publicNetworkAccess: 'Disabled'
  networkAcls: {
    defaultAction: 'Deny'
  }
}
```

## 💰 Оптимизация на разходите

| Стратегия | Спестяване |
|----------|---------|
| Мащабиране до нула (Container Apps) | 60-80% |
| Използвайте нива на потребление за разработка | 50-70% |
| Планирано мащабиране | 30-50% |
| Резервиран капацитет | 20-40% |

```bash
# Задайте бюджетни предупреждения
az consumption budget create \
  --budget-name "AI-Budget" \
  --amount 500 \
  --category Cost \
  --time-grain Monthly
```

## 📊 Настройка на мониторинга

```bash
# Поточно показване на регистрационни файлове
azd monitor --logs

# Проверете Application Insights
azd monitor

# Преглед на показатели
az monitor metrics list --resource <resource-id>
```
