## 🎯 Представено решение: Многоагентно решение за търговия на дребно

The [Многоагентно решение за търговия на дребно](../../examples/retail-scenario.md) демонстрира:

- **Агент за клиенти**: Обработва взаимодействията с потребителя и предпочитанията
- **Агент за наличности**: Управлява запасите и обработката на поръчки
- **Оркестратор**: Координира работата на агентите
- **Споделена памет**: Управление на контекста между агентите
