## Преглед

Тази глава представя основите на Azure Developer CLI (azd). Ще научите основните концепции, ще инсталирате инструментите и ще публикувате първото си приложение в Azure.

## Цели на обучението

By completing this chapter, you will:
- Разберете какво е Azure Developer CLI и как се различава от Azure CLI
- Инсталирайте и конфигурирайте AZD на вашата платформа
- Публикувайте първото си приложение в Azure с `azd up`
- Премахнете ресурсите с `azd down`

## 🚀 Бърз старт

```bash
# Проверете инсталацията
azd version

# Влезте в Azure
azd auth login

# Разположете първото си приложение
azd init --template todo-nodejs-mongo
azd up

# Почистете след като приключите
azd down --force --purge
```

---

## ✅ Критерии за успех

След като завършите тази глава, трябва да можете:

```bash
azd version              # Показва инсталираната версия
azd init --template todo-nodejs-mongo  # Инициализира проекта
azd up                   # Разгръща в Azure
azd show                 # Показва URL адреса на работещото приложение
azd down --force --purge # Почиства ресурсите
```
