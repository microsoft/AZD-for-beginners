## Общ Преглед

Тази глава представя основите на Azure Developer CLI (azd). Ще научите основни концепции, ще инсталирате инструментите и ще публикувате първото си приложение в Azure.

> Валидирано с `azd 1.23.12` през март 2026.

## Цели на Обучението

След завършване на тази глава, вие ще:
- Разберете какво е Azure Developer CLI и как се различава от Azure CLI
- Инсталирате и конфигурирате AZD на вашата платформа
- Пуснете първото си приложение в Azure с `azd up`
- Почистите ресурсите с `azd down`

## ✅ Започнете тук: Потвърдете Вашата Настройка

Преди да започнете, уверете се, че локалната ви машина е готова за шаблона на Глава 1:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

Ако скриптът съобщава за липсващи инструменти, първо ги инсталирайте и след това продължете с главата.

---

## 🚀 Бърз Старт

```bash
# Проверете инсталацията
azd version

# Удостоверяване за AZD
# По избор: az login ако планирате да изпълнявате Azure CLI команди директно
azd auth login

# Разположете първото си приложение
azd init --template todo-nodejs-mongo
azd up

# Почистете след като приключите
azd down --force --purge
```

---

## ✅ Критерии за Успех

След завършване на тази глава, трябва да можете:

```bash
azd version              # Показва инсталираната версия
azd init --template todo-nodejs-mongo  # Инициализира проекта
azd up                   # Разгръща в Azure
azd show                 # Показва URL адреса на работещото приложение
azd down --force --purge # Изчиства ресурсите
```
