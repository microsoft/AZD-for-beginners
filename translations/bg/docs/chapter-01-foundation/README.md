## Преглед

Тази глава представя основите на Azure Developer CLI (azd). Ще научите основните концепции, как да инсталирате инструментите и как да разположите първото си приложение в Azure.

> Валидирано с `azd 1.25.6` през юни 2026 г.

## Учебни цели

След като завършите тази глава, ще можете:
- Разберете какво представлява Azure Developer CLI и как се различава от Azure CLI
- Инсталирате и конфигурирате AZD на вашата платформа
- Разположите първото си приложение в Azure с `azd up`
- Премахнете ресурсите с `azd down`

Преди да започнете, потвърдете, че локалната ви машина е готова за шаблона на Глава 1:

**Windows:**
```powershell
.\validate-setup.ps1
```

```bash
bash ./validate-setup.sh
```

If the script reports missing tools, fix those first and then continue with the chapter.

---

## 🚀 Бърз старт

```bash
# Проверете инсталацията
azd version

# Аутентифицирайте се за AZD
# По избор: az login, ако планирате да изпълнявате команди на Azure CLI директно
azd auth login

# Разположете първото си приложение
azd init --template todo-nodejs-mongo
azd up

# Почистете след като приключите
azd down --force --purge
```

---

## ✅ Критерии за успех

След като завършите тази глава, трябва да можете да:

```bash
azd version              # Показва инсталираната версия
azd init --template todo-nodejs-mongo  # Инициализира проекта
azd up                   # Разгръща в Azure
azd show                 # Показва URL адреса на работещото приложение
azd down --force --purge # Почиства ресурсите
```
