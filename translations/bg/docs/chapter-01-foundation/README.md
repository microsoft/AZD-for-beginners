# Глава 1: Основи и бърз старт

**📚 Курс**: [AZD за начинаещи](../../README.md) | **⏱️ Продължителност**: 30-45 минути | **⭐ Сложност**: Начинаещ

## Обзор

Тази глава въвежда основите на Azure Developer CLI (azd). Ще научите основни концепции, ще инсталирате инструментите и ще публикувате първото си приложение в Azure.

> Валидирано спрямо `azd 1.27.1` през юли 2026.

## Цели на обучението

След завършване на тази глава ще:
- Разберете какво е Azure Developer CLI и как се различава от Azure CLI
- Инсталирате и конфигурирате AZD на вашата платформа
- Публикувате първото си приложение в Azure с `azd up`
- Почистите ресурсите с `azd down`

## ✅ Започнете тук: Проверете настройката си

Преди да започнете, уверете се, че вашата локална машина е готова за шаблона на Глава 1:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

Ако скриптът съобщи за липсващи инструменти, първо ги поправете и след това продължете с главата.

---

## 🚀 Бърз старт

```bash
# Проверете инсталацията
azd version

# Удостоверяване за AZD
# По избор: az login, ако планирате да изпълнявате команди на Azure CLI директно
azd auth login

# Разположете първото си приложение
azd init --template todo-nodejs-mongo
azd up

# Почистете след като приключите
azd down --force --purge
```

## ✅ Критерии за успех

След завършване на тази глава ще можете да:

```bash
azd version              # Показва инсталираната версия
azd init --template todo-nodejs-mongo  # Инициализира проекта
azd up                   # Разгръща в Azure
azd show                 # Показва URL на работещото приложение
azd down --force --purge # Почиства ресурсите
```
