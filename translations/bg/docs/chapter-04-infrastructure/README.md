# Глава 4: Инфраструктура като Код и Разгръщане

**📚 Курс**: [AZD За Начинаещи](../../README.md) | **⏱️ Продължителност**: 1-1.5 часа | **⭐ Сложност**: Средно ниво

---

## Преглед

Тази глава разглежда модели за Инфраструктура като Код (IaC) с шаблони Bicep, осигуряване на ресурси и стратегии за разгръщане с помощта на Azure Developer CLI.

> Валидирано с `azd 1.27.1` през юли 2026 г.

## Учебни цели

След завършване на тази глава, ще можете:
- Да разбирате структурата и синтаксиса на шаблоните Bicep
- Да осигурявате Azure ресурси с `azd provision`
- Да разгръщате приложения с `azd deploy`
- Да прилагате стратегии за синьо-зелено и постепенно разгръщане

## 🚀 Бърз старт

```bash
# Инициализиране от шаблон
azd init --template azure-functions-python-v2-http

# Преглед на това, което ще бъде създадено
azd provision --preview

# Осигуряване само на инфраструктура
azd provision

# Изпълнение само на код
azd deploy

# Или и двете заедно
azd up
```

## 📁 Структура на AZD проект

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

---

## 🔧 Основни команди

| Команда | Описание |
|---------|-------------|
| `azd init` | Инициализиране на проект |
| `azd provision` | Създаване на Azure ресурси |
| `azd deploy` | Разгръщане на код на приложението |
| `azd up` | осигуряване + разгръщане |
| `azd down` | Изтриване на всички ресурси |
