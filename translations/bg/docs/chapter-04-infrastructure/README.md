# Глава 4: Инфраструктура като код и внедряване

**📚 Курс**: [AZD за начинаещи](../../README.md) | **⏱️ Продължителност**: 1-1.5 часа | **⭐ Ниво на сложност**: Средно ниво

## Преглед

Тази глава разглежда модели на Инфраструктура като код (IaC) с шаблони Bicep, осигуряване на ресурси и стратегии за внедряване чрез Azure Developer CLI.

## Учебни цели

След като завършите тази глава, ще:
- Ще разберете структурата и синтаксиса на шаблоните Bicep
- Ще създавате Azure ресурси с `azd provision`
- Ще внедрявате приложения с `azd deploy`
- Ще приложите стратегии за внедряване blue-green и rolling

## 🚀 Бърз старт

```bash
# Инициализиране от шаблон
azd init --template azure-functions-python-v2-http

# Преглед на това, което ще бъде създадено
azd provision --preview

# Само разполагане на инфраструктурата
azd provision

# Само разгръщане на кода
azd deploy

# Или и двете заедно
azd up
```

## 📁 Структура на проекта AZD

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

---

## 🔧 Основни команди

| Команда | Описание |
|---------|-------------|
| `azd init` | Инициализиране на проекта |
| `azd provision` | Създаване на Azure ресурси |
| `azd deploy` | Внедряване на код на приложението |
| `azd up` | провизиране + внедряване |
| `azd down` | Изтриване на всички ресурси |
