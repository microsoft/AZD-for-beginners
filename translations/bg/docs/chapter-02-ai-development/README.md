# Глава 2: Разработка, ориентирана към ИИ

**📚 Course**: [AZD за начинаещи](../../README.md) | **⏱️ Duration**: 1-2 часа | **⭐ Complexity**: Средно

---

## Преглед

Тази глава се фокусира върху внедряването на приложения, захранвани от ИИ, използвайки Azure Developer CLI и услугите на Microsoft Foundry. От прости чат-приложения с ИИ до интелигентни агенти с инструменти.

> **Бележка за валидация (2026-06-15):** Потокът на командите и указанията за разширението в тази глава бяха прегледани спрямо `azd` `1.25.6` и текущото предварително издание на AI агента `azure.ai.agents` `0.1.40-preview`. Ако използвате по-стар AZD билд, първо го обновете и след това продължете с упражненията.

## Учебни цели

След като завършите тази глава, вие ще:
- Ще разположите приложения с ИИ, използвайки предварително създадени AZD шаблони
- Ще разберете интеграцията на Microsoft Foundry с AZD
- Ще конфигурирате и персонализирате ИИ агенти с инструменти
- Ще разположите RAG (Retrieval-Augmented Generation) приложения

## 🚀 Бърз старт

```bash
# Опция 1: RAG чат приложение
azd init --template azure-search-openai-demo
azd up

# Опция 2: ИИ агенти
azd init --template get-started-with-ai-agents
azd up

# Опция 3: Бързо чат приложение
azd init --template openai-chat-app-quickstart
azd up
```
