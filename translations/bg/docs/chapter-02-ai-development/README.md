# Глава 2: Разработка с приоритет на изкуствения интелект

**📚 Курс**: [AZD за начинаещи](../../README.md) | **⏱️ Продължителност**: 1-2 часа | **⭐ Сложност**: Средно ниво

---

## Обзор

Тази глава се фокусира върху внедряването на приложения с изкуствен интелект, използвайки Azure Developer CLI и услугите на Microsoft Foundry. От прости AI чат приложения до интелигентни агенти с инструменти.

> **Бележка за валидация (2026-03-25):** Потокът от команди и указанията за разширения в тази глава бяха прегледани спрямо `azd` `1.23.12` и текущото предварително издание на AI агент разширението `azure.ai.agents` `0.1.18-preview`. Ако използвате по-стара версия на AZD, първо я обновете и после продължете с упражненията.

## Учебни цели

След завършване на тази глава, ще можете да:
- Внедрявате AI приложения с използване на предварително изготвени AZD шаблони
- Разберете интеграцията на Microsoft Foundry с AZD
- Конфигурирате и персонализирате AI агенти с инструменти
- Разгръщате RAG (генериране с допълване чрез извличане) приложения

## 🚀 Бърз старт

```bash
# Опция 1: RAG чат приложение
azd init --template azure-search-openai-demo
azd up

# Опция 2: AI агенти
azd init --template get-started-with-ai-agents
azd up

# Опция 3: Бързо чат приложение
azd init --template openai-chat-app-quickstart
azd up
```
