# Глава 2: Разработка с приоритет на ИИ

**📚 Курс**: [AZD За начинаещи](../../README.md) | **⏱️ Продължителност**: 1-2 часа | **⭐ Сложност**: Средно ниво

---

## Преглед

Тази глава се фокусира върху внедряването на приложения с изкуствен интелект чрез Azure Developer CLI и услугите на Microsoft Foundry. От прости чат приложения с ИИ до интелигентни агенти с инструменти.

> **Бележка за валидиране (2026-07-13):** Потокът от команди и насоките за разширения в тази глава бяха прегледани спрямо `azd` `1.27.1` и настоящото Preview издание на AI агент разширението `azure.ai.agents` `1.0.0-beta.5`. Ако използвате по-стара версия на AZD, първо актуализирайте и след това продължете с упражненията.

## Цели на обучението

След завършване на тази глава ще можете:
- Да внедрявате AI приложения с готови AZD шаблони
- Да разбирате интеграцията на Microsoft Foundry с AZD
- Да конфигурирате и персонализирате AI агенти с инструменти
- Да внедрявате RAG (Retrieval-Augmented Generation) приложения

## 🚀 Бърз старт

```bash
# Опция 1: RAG чат приложение
azd init --template azure-search-openai-demo
azd up

# Опция 2: AI агенти
azd init --template get-started-with-ai-agents
azd up

# Опция 3: Бързо чат приложение
azd init --template openai-chat-app-quickstart
azd up
```
