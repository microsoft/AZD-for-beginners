# AZD for AI-utviklere Workshop

> [!IMPORTANT]  
> **Denne workshopen har en guide som du kan forhåndsvise i din lokale nettleser. For å komme i gang må du starte GitHub Codespaces på repoet—og deretter vente til du ser en aktiv VS Code-terminal og skrive:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Du skal se en popup-dialog for å åpne forhåndsvisningssiden i en nettleser.**

Velkommen til den praktiske workshopen for å lære Azure Developer CLI (AZD) med fokus på utrulling av AI-applikasjoner. Denne workshopen hjelper deg til en anvendt forståelse av AZD-maler i 3 steg:

1. **Oppdagelse** - finn malen som passer for deg.
1. **Utrulling** - rull ut og valider at det fungerer
1. **Tilpasning** - modifiser og iterer for å gjøre den din!

I løpet av denne workshopen vil du også bli introdusert for kjerneverktøy og arbeidsflyter for utviklere, for å hjelpe deg å effektivisere hele utviklingsreisen.

## Workshop Oversikt

**Varighet:** 3-4 timer  
**Nivå:** Nybegynner til middels  
**Forutsetninger:** Kjennskap til Azure, AI-konsepter, VS Code & kommandolinjeverktøy.

Dette er en hands-on workshop hvor du lærer ved å gjøre. Når du har fullført øvelsene anbefaler vi å gjennomgå AZD For Beginners-kurset for å fortsette læringsreisen inn i sikkerhet og produktivitets beste praksis.

| Tid| Modul  | Mål |
|:---|:---|:---|
| 15 min | Introduksjon | Sett scenen, forstå målene |
| 30 min | Velg AI-mal | Utforsk alternativer og velg startmal | 
| 30 min | Valider AI-mal | Rull ut standards løsning til Azure |
| 30 min | Bryt ned AI-mal | Undersøk struktur og konfigurasjon |
| 30 min | Konfigurer AI-mal | Aktiver og prøv tilgjengelige funksjoner |
| 30 min | Tilpass AI-mal | Tilpass malen til dine behov |
| 30 min | Riv ned infrastruktur | Rydd opp og frigjør ressurser |
| 15 min | Oppsummering & neste steg | Læringsressurser, Workshop-utfordring |
| | |

## Hva du vil lære

Tenk på AZD-malen som en læringssandbox for å utforske ulike muligheter og verktøy for ende-til-ende utvikling på Microsoft Foundry. Ved slutten av denne workshopen skal du ha en intuitiv forståelse for forskjellige verktøy og konsepter i denne sammenhengen.

| Konsept  | Mål |
|:---|:---|
| **Azure Developer CLI** | Forstå verktøykommandoer og arbeidsflyter |
| **AZD-maler**| Forstå prosjektstruktur og konfigurasjon |
| **Azure AI Agent**| Opprett og rull ut Microsoft Foundry-prosjekt  |
| **Azure AI Search**| Aktiver kontekstutforming med agenter |
| **Observability**| Utforsk sporing, overvåking og evalueringer |
| **Red Teaming**| Utforsk fiendtlig testing og mottiltak |
