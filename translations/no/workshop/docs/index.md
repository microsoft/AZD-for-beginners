# AZD for AI-utviklere Workshop

> Validert mot `azd 1.25.6` i juni 2026.

> [!IMPORTANT]  
> **Denne workshoppen er utstyrt med en workshopguide som du kan forhåndsvise i din lokale nettleser. For å komme i gang, åpne repoet i GitHub Codespaces, en utviklercontainer eller en lokal klone med MkDocs installert—vent deretter til du ser en aktiv VS Code-terminal og skriv:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Du skal se en popup-dialog for å åpne forhåndsvisningssiden i en nettleser.**

Velkommen til den praksisorienterte workshopen for å lære Azure Developer CLI (AZD) med fokus på utrulling av AI-applikasjoner. Denne workshoppen hjelper deg med å få en anvendt forståelse av AZD-maler i 3 trinn:

1. **Oppdagelse** - finn malen som passer for deg.  
1. **Utrulling** - deploy og valider at det fungerer  
1. **Tilpasning** - endre og iterer for å gjøre den til din egen!

I løpet av workshopen vil du også bli introdusert for kjerneverktøy og arbeidsflyter for utviklere, for å hjelpe deg å effektivisere din ende-til-ende utviklingsreise.

## Workshopoversikt

**Varighet:** 3-4 timer  
**Nivå:** Nybegynner til middels  
**Forutsetninger:** Kjennskap til Azure, AI-konsepter, VS Code og kommandolinjeverktøy.

Dette er en praksisworkshop der du lærer ved å gjøre. Når du har fullført øvelsene anbefaler vi at du går gjennom AZD For Beginners-lærekurset for å fortsette læringsreisen inn i sikkerhet og produktivitets beste praksis.

| Tid| Modul  | Mål |
|:---|:---|:---|
| 15 min | Introduksjon | Legg grunnlaget, forstå målene |
| 30 min | Velg AI-mal | Utforsk alternativer og velg startmal | 
| 30 min | Valider AI-mal | Utrull standardløsning til Azure |
| 30 min | Dekonstruer AI-mal | Utforsk struktur og konfigurasjon |
| 30 min | Konfigurer AI-mal | Aktiver og prøv tilgjengelige funksjoner |
| 30 min | Tilpass AI-mal | Tilpass malen til dine behov |
| 30 min | Rydd opp infrastruktur | Rengjør og frigjør ressurser |
| 15 min | Oppsummering & Neste steg | Læringsressurser, Workshop-utfordring |
| | |

## Hva du vil lære

Tenk på AZD-malen som en læringslekeplass for å utforske ulike funksjoner og verktøy for ende-til-ende utvikling på Microsoft Foundry. Når du er ferdig med denne workshoppen, skal du ha en intuitiv forståelse for forskjellige verktøy og konsepter i denne sammenhengen.

| Konsept  | Mål |
|:---|:---|
| **Azure Developer CLI** | Forstå kommandoer og arbeidsflyter |
| **AZD-maler**| Forstå prosjektstruktur og konfigurasjon |
| **Azure AI Agent**| Opprett og deploy Microsoft Foundry-prosjekt |
| **Azure AI Search**| Aktiver kontekstuell ingeniørkunst med agenter |
| **Observabilitet**| Utforsk sporing, overvåking og evalueringer |
| **Red Teaming**| Utforsk motstandstest og mottiltak |
