# AZD For AI-utviklere

!!! info "MÅLGRUPPE FOR DETTE VERKSTEDET"
   
    Dette verkstedet er for alle som ønsker å bygge AI-agentiske applikasjoner på Microsoft Foundry ved å bruke de beste verktøyene og arbeidsflytene. Verkstedet passer både for nybegynnere og erfarne utviklere som har noe kjennskap til Azure, AI-konsepter og Python-kode. Vennligst sjekk forutsetningene før du begynner.

    **Varighet:** 3-4 timer  
    **Nivå:** Nybegynner til middels

---


## Forutsetninger

- En personlig GitHub-konto → [Opprett en gratis](https://github.com/signup)
- En aktiv Azure-konto → [Opprett en gratis](https://aka.ms/free)
- Kjennskap til AI-konsepter, Python-utvikling, kommandolinjeverktøy
- Erfaring med VS Code og GitHub-verktøy og arbeidsflyter.

---

## Læringsmål

Ved slutten av dette verkstedet vil du kunne:

- [X] **Arbeide med Azure Developer CLI** → for å opprette, distribuere og administrere Azure-ressursene dine.
- [X] **Bruke AI App-maler** → for å komme i gang med AI-agentprosjekter på Microsoft Foundry.
- [X] **Konfigurere maler** → for funksjoner som Red Teaming, Evalueringer, Sporing og Overvåking
- [X] **De-konstruere og utforske** → AZD-maler og arbeidsflyter ved hjelp av GitHub Copilot
- [X] **Tilpasse og distribuere på nytt** → AI Agent- startmalen med dine egne krav

---

## Verkstedsoversikt

Verkstedet er strukturert som moduler, hver forankret i en praktisk lab-øvelse:

1. **Introduksjon** - Få en forståelse av hva AZD-maler er, og hvordan de fungerer
1. **Valg** - Oppdag riktig "start" mal for ditt neste AI-prosjekt
1. **Validering** - Distribuer og utforsk malen for å verifisere at den fungerer som annonsert
1. **De-konstruksjon** - Ta malen fra hverandre for å forstå komponentene og konfigurasjonen
1. **Konfigurasjon** - Lær å aktivere og bruke ulike Microsoft Foundry-funksjoner
1. **Tilpasning** - Gjenoppbygg applikasjonen med dine data og krav i tankene
1. **Nedpakking** - Lær å fjerne infrastruktur og gjenopprette mykt slettede ressurser
