# AZD untuk Pengembang AI Workshop

> Divalidasi dengan `azd 1.27.1` pada Juli 2026.

> [!IMPORTANT]  
> **Workshop ini dilengkapi dengan panduan workshop yang dapat Anda pratinjau di browser lokal Anda. Untuk memulai, buka repo di GitHub Codespaces, dev container, atau klon lokal dengan MkDocs terpasang—kemudian tunggu hingga Anda melihat terminal VS Code aktif dan ketik:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Anda akan melihat dialog pop-up untuk membuka halaman pratinjau di browser.**

Selamat datang di workshop praktis untuk mempelajari Azure Developer CLI (AZD) dengan fokus pada penerapan aplikasi AI. Workshop ini membantu Anda mendapatkan pemahaman terapan tentang template AZD dalam 3 langkah:

1. **Penemuan** - temukan template yang tepat untuk Anda.
1. **Penerapan** - terapkan dan validasi bahwa itu berfungsi
1. **Kustomisasi** - modifikasi dan iterasi agar menjadi milik Anda!

Sepanjang workshop ini, Anda juga akan diperkenalkan dengan alat dan alur kerja inti pengembang, untuk membantu Anda mempermudah perjalanan pengembangan end-to-end Anda.

## Ikhtisar Workshop

**Durasi:** 3-4 jam  
**Tingkat:** Pemula sampai Menengah  
**Prasyarat:** Familiar dengan Azure, konsep AI, VS Code & alat baris perintah.

Ini adalah workshop praktis di mana Anda belajar dengan melakukan. Setelah menyelesaikan latihan, kami sarankan meninjau kurikulum AZD Untuk Pemula untuk melanjutkan perjalanan pembelajaran Anda ke praktik terbaik Keamanan dan Produktivitas.

| Waktu| Modul  | Tujuan |
|:---|:---|:---|
| 15 menit | Pengantar | Menetapkan panggung, memahami tujuan |
| 30 menit | Pilih Template AI | Jelajahi opsi dan pilih pemula | 
| 30 menit | Validasi Template AI | Terapkan solusi default ke Azure |
| 30 menit | Bongkar Template AI | Jelajahi struktur dan konfigurasi |
| 30 menit | Konfigurasi Template AI | Aktifkan dan coba fitur yang tersedia |
| 30 menit | Kustomisasi Template AI | Sesuaikan template sesuai kebutuhan Anda |
| 30 menit | Bongkar Infrastruktur | Bersihkan dan lepaskan sumber daya |
| 15 menit | Penutup & Langkah Selanjutnya | Sumber belajar, tantangan Workshop |
| | |

## Apa yang Akan Anda Pelajari

Pikirkan Template AZD sebagai sandbox pembelajaran untuk mengeksplorasi berbagai kemampuan dan alat untuk pengembangan end-to-end di Microsoft Foundry. Pada akhir workshop ini, Anda harus memiliki pemahaman intuitif tentang berbagai alat dan konsep dalam konteks ini.

| Konsep  | Tujuan |
|:---|:---|
| **Azure Developer CLI** | Memahami perintah alat dan alur kerja|
| **Template AZD**| Memahami struktur proyek dan konfigurasi|
| **Azure AI Agent**| Menyediakan & menerapkan proyek Microsoft Foundry  |
| **Azure AI Search**| Mengaktifkan rekayasa konteks dengan agen |
| **Observabilitas**| Jelajahi pelacakan, pemantauan dan evaluasi |
| **Red Teaming**| Jelajahi pengujian adversarial dan mitigasi |
