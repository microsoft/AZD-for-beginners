# Workshop AZD untuk Pengembang AI

> Tervalidasi terhadap `azd 1.25.6` pada Juni 2026.

> [!IMPORTANT]  
> **Workshop ini dilengkapi dengan panduan workshop yang dapat Anda pratinjau di browser lokal Anda. Untuk memulai, buka repo di GitHub Codespaces, dev container, atau klon lokal dengan MkDocs terpasang—kemudian tunggu sampai Anda melihat terminal VS Code aktif dan ketik:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Anda akan melihat dialog pop-up untuk membuka halaman pratinjau di browser.**

Selamat datang di workshop praktik untuk mempelajari Azure Developer CLI (AZD) dengan fokus pada penerapan aplikasi AI. Workshop ini membantu Anda mendapatkan pemahaman terapan tentang template AZD dalam 3 langkah:

1. **Penemuan** - temukan template yang tepat untuk Anda.
1. **Penerapan** - terapkan dan validasi bahwa itu berfungsi
1. **Kustomisasi** - modifikasi dan iterasi untuk menjadikannya milik Anda!

Selama workshop ini, Anda juga akan diperkenalkan ke alat dan alur kerja inti developer, untuk membantu menyederhanakan perjalanan pengembangan ujung-ke-ujung Anda.

## Ikhtisar Workshop

**Durasi:** 3-4 jam  
**Tingkat:** Pemula hingga Menengah  
**Prasyarat:** Familiar dengan Azure, konsep AI, VS Code & alat baris perintah.

Ini adalah workshop praktik di mana Anda belajar dengan melakukan. Setelah Anda menyelesaikan latihan, kami menyarankan meninjau kurikulum AZD Untuk Pemula untuk melanjutkan perjalanan pembelajaran Anda ke praktik terbaik Keamanan dan Produktivitas.

| Time| Module  | Objective |
|:---|:---|:---|
| 15 mins | Introduction | Mempersiapkan konteks, memahami tujuan |
| 30 mins | Select AI Template | Menjelajahi opsi dan memilih starter | 
| 30 mins | Validate AI Template | Terapkan solusi default ke Azure |
| 30 mins | Deconstruct AI Template | Menjelajahi struktur dan konfigurasi |
| 30 mins | Configure AI Template | Aktifkan dan coba fitur yang tersedia |
| 30 mins | Customize AI Template | Sesuaikan template dengan kebutuhan Anda |
| 30 mins | Teardown Infrastructure | Hapus dan lepaskan sumber daya |
| 15 mins | Wrap-Up & Next Steps | Sumber pembelajaran, tantangan workshop |
| | |

## Apa yang Akan Anda Pelajari

Pikirkan Template AZD sebagai sandbox pembelajaran untuk mengeksplorasi berbagai kemampuan dan alat untuk pengembangan ujung-ke-ujung di Microsoft Foundry. Di akhir workshop ini, Anda seharusnya memiliki pemahaman intuitif tentang berbagai alat dan konsep dalam konteks ini.

| Concept  | Objective |
|:---|:---|
| **Azure Developer CLI** | Memahami perintah alat dan alur kerja|
| **AZD Templates**| Memahami struktur proyek dan konfigurasi|
| **Azure AI Agent**| Menyediakan & menerapkan proyek Microsoft Foundry  |
| **Azure AI Search**| Mengaktifkan rekayasa konteks dengan agen |
| **Observability**| Menjelajahi pelacakan, pemantauan dan evaluasi |
| **Red Teaming**| Menjelajahi pengujian adversarial dan mitigasi |
