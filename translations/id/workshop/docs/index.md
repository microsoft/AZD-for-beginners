# AZD untuk Pengembang AI Lokakarya

> Divalidasi terhadap `azd 1.23.12` pada Maret 2026.

> [!IMPORTANT]  
> **Lokakarya ini dilengkapi panduan lokakarya yang dapat Anda pratinjau di browser lokal Anda. Untuk memulai, buka repo di GitHub Codespaces, sebuah dev container, atau klon lokal dengan MkDocs terpasang—kemudian tunggu sampai Anda melihat terminal VS Code aktif dan ketik:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Anda akan melihat dialog pop-up untuk membuka halaman pratinjau di browser.**

Selamat datang di lokakarya praktik langsung untuk mempelajari Azure Developer CLI (AZD) dengan fokus pada penyebaran aplikasi AI. Lokakarya ini membantu Anda memperoleh pemahaman terapan tentang template AZD dalam 3 langkah:

1. **Penemuan** - menemukan template yang tepat untuk Anda.
1. **Penerapan** - menerapkan dan memvalidasi bahwa itu berfungsi
1. **Kustomisasi** - memodifikasi dan mengiterasi agar sesuai kebutuhan Anda!

Selama lokakarya ini, Anda juga akan diperkenalkan pada alat dan alur kerja inti untuk pengembang, untuk membantu menyederhanakan perjalanan pengembangan end-to-end Anda.

## Ikhtisar Lokakarya

**Durasi:** 3-4 jam  
**Tingkat:** Pemula hingga Menengah  
**Prasyarat:** Familiaritas dengan Azure, konsep AI, VS Code & alat baris perintah.

Ini adalah lokakarya praktik langsung di mana Anda belajar dengan melakukan. Setelah Anda menyelesaikan latihan, kami menyarankan meninjau kurikulum AZD For Beginners untuk melanjutkan perjalanan pembelajaran Anda ke praktik terbaik Keamanan dan Produktivitas.

| Waktu| Modul  | Tujuan |
|:---|:---|:---|
| 15 mins | Pendahuluan | Menetapkan konteks, memahami tujuan |
| 30 mins | Pilih Template AI | Jelajahi opsi dan pilih template awal | 
| 30 mins | Validasi Template AI | Terapkan solusi default ke Azure |
| 30 mins | Menguraikan Template AI | Jelajahi struktur dan konfigurasi |
| 30 mins | Konfigurasi Template AI | Aktifkan dan coba fitur yang tersedia |
| 30 mins | Kustomisasi Template AI | Sesuaikan template dengan kebutuhan Anda |
| 30 mins | Hentikan Infrastruktur | Bersihkan dan lepaskan sumber daya |
| 15 mins | Penutupan & Langkah Selanjutnya | Sumber belajar, Tantangan lokakarya |
| | |

## Apa yang Akan Anda Pelajari

Anggap Template AZD sebagai ruang belajar untuk mengeksplorasi berbagai kemampuan dan alat untuk pengembangan end-to-end di Microsoft Foundry. Pada akhir lokakarya ini, Anda seharusnya memiliki pemahaman intuitif tentang berbagai alat dan konsep dalam konteks ini.

| Konsep  | Tujuan |
|:---|:---|
| **Azure Developer CLI** | Memahami perintah alat dan alur kerja|
| **AZD Templates**| Memahami struktur proyek dan konfigurasi|
| **Azure AI Agent**| Menyediakan & menerapkan proyek Microsoft Foundry  |
| **Azure AI Search**| Mengaktifkan rekayasa konteks dengan agen |
| **Observability**| Jelajahi tracing, monitoring, dan evaluasi |
| **Red Teaming**| Jelajahi pengujian adversarial dan mitigasi |
