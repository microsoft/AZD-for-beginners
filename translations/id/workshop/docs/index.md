# AZD untuk Pengembang AI - Lokakarya

> [!IMPORTANT]  
> **Lokakarya ini dilengkapi dengan panduan workshop yang dapat Anda pratinjau di browser lokal Anda. Untuk memulai, Anda harus meluncurkan GitHub Codespaces pada repo—kemudian tunggu hingga Anda melihat terminal VS Code yang aktif dan ketik:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Anda akan melihat dialog pop-up untuk membuka halaman pratinjau di browser.**

Selamat datang di lokakarya praktik langsung untuk mempelajari Azure Developer CLI (AZD) dengan fokus pada penerapan aplikasi AI. Lokakarya ini membantu Anda memperoleh pemahaman terapan tentang template AZD dalam 3 langkah:

1. **Discovery** - temukan template yang tepat untuk Anda.
1. **Deployment** - terapkan dan validasi bahwa itu berfungsi
1. **Customization** - modifikasi dan iterasi agar menjadi milik Anda!

Selama lokakarya ini, Anda juga akan diperkenalkan pada alat dan alur kerja inti pengembang, untuk membantu menyederhanakan perjalanan pengembangan end-to-end Anda.

## Ikhtisar Lokakarya

**Durasi:** 3-4 jam  
**Tingkat:** Pemula hingga Menengah  
**Prasyarat:** Familiaritas dengan Azure, konsep AI, VS Code & alat baris perintah.

Ini adalah lokakarya praktik langsung di mana Anda belajar melalui praktik. Setelah menyelesaikan latihan, kami menyarankan meninjau kurikulum AZD Untuk Pemula untuk melanjutkan perjalanan pembelajaran Anda ke praktik terbaik Keamanan dan Produktivitas.

| Time| Module  | Objective |
|:---|:---|:---|
| 15 mins | Introduction | Menetapkan konteks, memahami tujuan |
| 30 mins | Select AI Template | Jelajahi opsi dan pilih starter | 
| 30 mins | Validate AI Template | Terapkan solusi default ke Azure |
| 30 mins | Deconstruct AI Template | Jelajahi struktur dan konfigurasi |
| 30 mins | Configure AI Template | Aktifkan dan coba fitur yang tersedia |
| 30 mins | Customize AI Template | Sesuaikan template dengan kebutuhan Anda |
| 30 mins | Teardown Infrastructure | Bersihkan dan lepaskan sumber daya |
| 15 mins | Wrap-Up & Next Steps | Sumber belajar, tantangan lokakarya |
| | |

## Apa yang Akan Anda Pelajari

Anggap Template AZD sebagai kotak pasir pembelajaran untuk menjelajahi berbagai kapabilitas dan alat untuk pengembangan end-to-end di Microsoft Foundry. Pada akhir lokakarya ini, Anda seharusnya memiliki pemahaman intuitif tentang berbagai alat dan konsep dalam konteks ini.

| Concept  | Objective |
|:---|:---|
| **Azure Developer CLI** | Memahami perintah alat dan alur kerja|
| **AZD Templates**| Memahami struktur proyek dan konfigurasi|
| **Azure AI Agent**| Menyediakan & menerapkan proyek Microsoft Foundry  |
| **Azure AI Search**| Mengaktifkan rekayasa konteks dengan agen |
| **Observability**| Menjelajahi penelusuran, pemantauan dan evaluasi |
| **Red Teaming**| Menjelajahi pengujian adversarial dan mitigasi |
