# AZD Untuk Pengembang AI

!!! info "AUDIENCE FOR THIS WORKSHOP"
   
    Workshop ini ditujukan bagi siapa saja yang ingin membangun aplikasi Agen AI di Microsoft Foundry dengan memanfaatkan alat dan alur kerja terbaik. Workshop ini cocok untuk pemula maupun pengembang berpengalaman yang memiliki sedikit pemahaman tentang Azure, konsep AI, dan kode Python. Harap periksa prasyarat sebelum Anda memulai.

    **Durasi:** 3-4 jam  
    **Level:** Pemula hingga Menengah  

---


## Prasyarat

- Akun GitHub pribadi → [Create one for free](https://github.com/signup)
- Akun Azure aktif → [Create one for free](https://aka.ms/free)
- Familiar dengan konsep AI, pengembangan Python, alat baris perintah
- Pengalaman dengan VS Code serta alat dan alur kerja GitHub.

---

## Tujuan Pembelajaran

Pada akhir workshop ini, Anda akan mengetahui cara:

- [X] **Bekerja dengan Azure Developer CLI** → untuk menyediakan, menyebarkan, dan mengelola sumber daya Azure Anda.
- [X] **Menggunakan Template Aplikasi AI** → untuk memulai cepat proyek Agen AI di Microsoft Foundry.
- [X] **Mengonfigurasi Template** → untuk fitur seperti Red Teaming, Evaluasi, Penelusuran & Pemantauan
- [X] **Membongkar dan Menjelajahi** → template dan alur kerja AZD menggunakan GitHub Copilot
- [X] **Mengustomisasi dan Menerapkan Ulang** → template awal Agen AI, dengan kebutuhan kustom Anda

---

## Garis Besar Workshop

Workshop disusun sebagai modul-modul, masing-masing dipusatkan pada latihan praktis:

1. **Pendahuluan** - Dapatkan pemahaman tentang apa itu template AZD, dan bagaimana cara kerjanya
1. **Pemilihan** - Temukan template "starter" yang tepat untuk proyek AI Anda berikutnya
1. **Validasi** - Terapkan dan jelajahi template untuk memverifikasi bahwa ia bekerja sesuai yang dijanjikan
1. **Dekonstruksi** - Bongkar template untuk memahami komponen & konfigurasinya
1. **Konfigurasi** - Pelajari cara mengaktifkan dan menggunakan berbagai fitur Microsoft Foundry
1. **Kustomisasi** - Rekonstruksi aplikasi dengan mempertimbangkan data dan kebutuhan Anda
1. **Pembongkaran** - Pelajari cara menghapus penyediaan infrastruktur dan memulihkan sumber daya yang dihapus secara lunak
