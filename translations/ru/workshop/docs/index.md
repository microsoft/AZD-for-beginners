# AZD для разработчиков ИИ: Практический семинар

> Проверено с `azd 1.25.6` в июне 2026 года.

> [!IMPORTANT]  
> **Этот семинар оснащён руководством, которое вы можете просмотреть в локальном браузере. Чтобы начать, откройте репозиторий в GitHub Codespaces, контейнере для разработки или локальном клоне с установленным MkDocs — затем дождитесь появления активного терминала VS Code и введите:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Вы должны увидеть всплывающее окно с предложением открыть страницу просмотра в браузере.**

Добро пожаловать на практический семинар по изучению Azure Developer CLI (AZD) с акцентом на развертывание приложений ИИ. Этот семинар поможет вам получить практическое понимание шаблонов AZD в 3 шага:

1. **Открытие** — найдите подходящий шаблон.
1. **Развертывание** — разверните и убедитесь, что всё работает.
1. **Настройка** — модифицируйте и адаптируйте под себя!

В ходе семинара вы также познакомитесь с основными инструментами и рабочими процессами для разработчиков, чтобы упростить ваш полный цикл разработки.

## Обзор семинара

**Продолжительность:** 3-4 часа  
**Уровень:** Начинающий — Средний  
**Требования:** Знание Azure, концепций ИИ, VS Code и командных инструментов.

Это практический семинар, где вы учитесь на практике. После выполнения упражнений рекомендуем ознакомиться с учебной программой AZD для начинающих, чтобы продолжить обучение по лучшим практикам безопасности и продуктивности.

| Время | Модуль | Цель |
|:---|:---|:---|
| 15 мин | Введение | Определить задачи и цели |
| 30 мин | Выбор шаблона ИИ | Изучить варианты и выбрать стартовый |
| 30 мин | Проверка шаблона ИИ | Развернуть стандартное решение в Azure |
| 30 мин | Анализ шаблона ИИ | Изучить структуру и конфигурацию |
| 30 мин | Настройка шаблона ИИ | Активировать и попробовать функции |
| 30 мин | Кастомизация шаблона ИИ | Адаптировать шаблон под себя |
| 30 мин | Удаление инфраструктуры | Очистка и освобождение ресурсов |
| 15 мин | Итоги и дальнейшие шаги | Ресурсы для обучения, вызов семинара |

## Чему вы научитесь

Думайте о шаблоне AZD как о обучающей песочнице для изучения различных возможностей и инструментов полного цикла разработки в Microsoft Foundry. К концу семинара у вас будет интуитивное представление о разнообразных инструментах и концепциях в этом контексте.

| Концепция | Цель |
|:---|:---|
| **Azure Developer CLI** | Понять команды инструментов и рабочие процессы |
| **Шаблоны AZD** | Понять структуру проекта и конфигурацию |
| **Azure AI Agent** | Развернуть и запустить проект Microsoft Foundry |
| **Azure AI Search** | Активировать контекстное проектирование с агентами |
| **Наблюдаемость** | Изучить трассировку, мониторинг и оценку |
| **Red Teaming** | Изучить методы контрмер и тестирования безопасности |
| | |
