# AZD для разработчиков ИИ Workshop

Добро пожаловать на практический семинар по изучению Azure Developer CLI (AZD) с акцентом на развертывание AI-приложений. Этот семинар поможет вам получить прикладное понимание шаблонов AZD в 3 шага:

1. **Поиск** - найдите подходящий для вас шаблон.
1. **Развертывание** - разместите и подтвердите, что он работает
1. **Настройка** - изменяйте и итеративно улучшайте под себя!

В ходе этого семинара вы также познакомитесь с основными инструментами разработчика и рабочими процессами, чтобы оптимизировать ваш полный цикл разработки.

## Чему вы научитесь

Подумайте о шаблоне AZD как о учебной песочнице для исследования различных возможностей и инструментов для полного цикла разработки на Microsoft Foundry. К концу семинара у вас должно сложиться интуитивное понимание различных инструментов и концепций в этом контексте.

| Концепция  | Цель |
|:---|:---|
| **Azure Developer CLI** | Понять команды инструмента и рабочие процессы |
| **Шаблоны AZD** | Понять структуру проекта и конфигурацию |
| **Azure AI Agent** | Развертывание проекта Microsoft Foundry  |
| **Azure AI Search** | Включить контекстуслу инженерии с агентами |
| **Наблюдаемость** | Изучить трассировку, мониторинг и оценки |
| **Red Teaming** | Изучить методы противостояния атакующим тестам и mitigations |

## Структура семинара

Семинар построен так, чтобы провести вас от поиска шаблона к развертыванию, разбору и кастомизации — используя официальный стартовый шаблон [Getting Started with AI Agents](https://github.com/Azure-Samples/get-started-with-ai-agents) в качестве основы.

### [Модуль 1: Выбор шаблона AI](docs/instructions/1-Select-AI-Template.md) (30 мин)

- Что такое AI шаблоны?
- Где найти AI шаблоны?
- Как начать создавать AI агентов?
- **Лабораторная работа**: Быстрый старт в Codespaces или dev контейнере

### [Модуль 2: Проверка шаблона AI](docs/instructions/2-Validate-AI-Template.md) (30 мин)

- Что такое архитектура AI шаблона?
- Какова разработческая рабочая процедура AZD?
- Где получить помощь по AZD разработке?
- **Лабораторная работа**: Развертывание и проверка шаблона AI агентов

### [Модуль 3: Разбор шаблона AI](docs/instructions/3-Deconstruct-AI-Template.md) (30 мин)

- Изучение вашего окружения в `.azure/` 
- Изучение настроек ресурсов в `infra/` 
- Изучение конфигураций AZD в файлах `azure.yaml`
- **Лабораторная работа**: Изменение переменных окружения и повторное развертывание

### [Модуль 4: Настройка шаблона AI](docs/instructions/4-Configure-AI-Template.md) (30 мин)
- Изучение: Retrieval Augmented Generation
- Изучение: Оценка агентов и Red Teaming
- Изучение: Трассировка и мониторинг
- **Лабораторная работа**: Исследование AI агента и наблюдаемости

### [Модуль 5: Кастомизация шаблона AI](docs/instructions/5-Customize-AI-Template.md) (30 мин)
- Определение: PRD с требованиями сценария
- Конфигурация: Переменные окружения для AZD
- Реализация: Lifecycle Hooks для дополнительных задач
- **Лабораторная работа**: Кастомизация шаблона под мой сценарий

### [Модуль 6: Удаление инфраструктуры](docs/instructions/6-Teardown-Infrastructure.md) (30 мин)
- Итог: Что такое AZD шаблоны?
- Итог: Зачем использовать Azure Developer CLI?
- Дальнейшие шаги: Попробовать другой шаблон!
- **Лабораторная работа**: Деактивация инфраструктуры и очистка
