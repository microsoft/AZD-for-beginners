# Учебное пособие - AZD для начинающих

## Введение

Это подробное учебное пособие предлагает структурированные цели обучения, ключевые концепции, практические упражнения и материалы для оценки, чтобы помочь вам освоить Azure Developer CLI (azd). Используйте это руководство для отслеживания своего прогресса и уверенности в том, что вы охватили все важные темы.

## Цели обучения

Пройдя это учебное пособие, вы:
- Освоите все базовые и продвинутые концепции Azure Developer CLI
- Разовьете практические навыки развертывания и управления приложениями Azure
- Укрепите уверенность в устранении неполадок и оптимизации развертываний
- Поймете практики развертывания для продакшена и аспекты безопасности

## Результаты обучения

После завершения всех разделов этого учебного пособия вы сможете:
- Проектировать, развертывать и управлять архитектурами приложений с использованием azd
- Реализовывать комплексные стратегии мониторинга, безопасности и оптимизации затрат
- Самостоятельно устранять сложные проблемы с развертыванием
- Создавать собственные шаблоны и вносить вклад в сообщество azd

## Структура курса

### Модуль 1: Начало работы (1-2 недели)

#### Цели обучения
- Понять основы и ключевые концепции Azure Developer CLI
- Успешно установить и настроить azd в вашей среде разработки
- Выполнить первое развертывание с использованием существующего шаблона
- Ориентироваться в структуре проекта azd и понимать ключевые компоненты

#### Ключевые концепции для освоения
- Шаблоны, окружения и сервисы
- Структура конфигурации azure.yaml
- Основные команды azd (init, up, down, deploy)
- Принципы Infrastructure as Code
- Аутентификация и авторизация в Azure

#### Практические упражнения

**Упражнение 1.1: Установка и настройка**
```bash
# Complete these tasks:
1. Install azd using your preferred method
2. Install Azure CLI and authenticate
3. Verify installation with: azd version
4. Test connectivity with: azd auth login
5. Explore available templates: azd template list
```

**Упражнение 1.2: Первое развертывание**
```bash
# Deploy a simple web application:
1. Initialize project: azd init --template todo-nodejs-mongo
2. Review project structure and configuration files
3. Deploy to Azure: azd up
4. Test the deployed application
5. Clean up resources: azd down
```

**Упражнение 1.3: Анализ структуры проекта**
```
Analyze the following components:
1. azure.yaml - service definitions and hooks
2. infra/ directory - Bicep templates and modules
3. src/ directory - application source code
4. .azure/ directory - environment configurations
```
