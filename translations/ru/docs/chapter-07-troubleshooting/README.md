# Глава 7: Устранение неполадок и отладка

**📚 Курс**: [AZD для начинающих](../../README.md) | **⏱️ Время**: 1-1.5 часа | **⭐ Сложность**: Средний уровень

## Обзор

Эта глава поможет вам диагностировать и решать распространённые проблемы при работе с Azure Developer CLI. От сбоев развертывания до специфичных проблем с ИИ.

> Проверено на `azd 1.23.12` в марте 2026 года.

## Цели обучения

Пройдя эту главу, вы сможете:
- Диагностировать распространённые ошибки развертывания AZD
- Отлаживать проблемы аутентификации и разрешений
- Решать проблемы с подключением к сервисам ИИ
- Использовать Azure Portal и CLI для устранения неполадок

## 🚨 Быстрые решения

### Проблемы аутентификации
```bash
# Необходимо для рабочих процессов AZD
azd auth login

# Необязательно, если вы также используете команды Azure CLI напрямую
az login

azd auth status
```

### Ошибки при развертывании
```bash
azd show
azd monitor --logs
az deployment sub list --query "[?properties.provisioningState!='Succeeded']"
```

### Конфликты ресурсов
```bash
azd down --force --purge
azd env new different-name
azd up
```

### Превышение квоты
```bash
az vm list-usage --location eastus --output table
azd env set AZURE_LOCATION westus2
azd up
```

---

## 📋 Справочник кодов ошибок

| Ошибка | Причина | Решение |
|-------|-------|----------|
| `AuthenticationError` | Не выполнен вход | `azd auth login` |
| `ResourceNotFound` | Ресурс отсутствует | Проверьте имена ресурсов |
| `QuotaExceeded` | Лимиты по подписке | Запросите увеличение квоты |
| `InvalidTemplate` | Ошибка синтаксиса Bicep | `az bicep build` |
| `Conflict` | Ресурс уже существует | Используйте новое имя или удалите |
| `Forbidden` | Недостаточно разрешений | Проверьте роли RBAC |

## 🔄 Сброс и восстановление

```bash
# Мягкая перезагрузка (сохранить ресурсы, заново развернуть код)
azd deploy --force

# Жесткая перезагрузка (удалить всё, начать заново)
azd down --force --purge
azd up
```
