# Глава 6: Планирование и проверка перед развертыванием

**📚 Курс**: [AZD для начинающих](../../README.md) | **⏱️ Длительность**: 1 час | **⭐ Сложность**: Средний уровень

---

## Обзор

В этой главе рассматриваются основные шаги по планированию и проверке перед развертыванием вашего приложения. Узнайте, как избежать дорогостоящих ошибок с помощью правильного планирования емкости, выбора SKU и предварительных проверок.

## Цели обучения

Выполнив эту главу, вы сможете:
- Выполнять предварительные проверки перед развертыванием
- Планировать емкость и оценивать требования к ресурсам
- Выбирать подходящие SKU для оптимизации затрат
- Настраивать Application Insights для мониторинга
- Понимать схемы координации команд

## 🚀 Быстрый старт

```bash
# Проверить квоты подписки
az vm list-usage --location eastus --output table

# Предварительный просмотр развертывания (ресурсы не создаются)
azd provision --preview

# Проверить синтаксис Bicep
az bicep build --file infra/main.bicep

# Проверить конфигурацию окружения
azd env get-values
```

## ☑️ Контрольный список перед развертыванием

### Перед `azd provision`

- [ ] Квота проверена для региона
- [ ] SKU выбран корректно
- [ ] Оценка затрат рассмотрена
- [ ] Соблюдены правила именования
- [ ] Настроена безопасность / RBAC

### Перед `azd deploy`

- [ ] Установлены переменные окружения
- [ ] Секреты в Key Vault
- [ ] Проверены строки подключения
- [ ] Настроены проверки работоспособности
