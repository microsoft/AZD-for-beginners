# Глава 6: Планирование и проверка перед развертыванием

**📚 Курс**: [AZD For Beginners](../../README.md) | **⏱️ Продолжительность**: 1 час | **⭐ Сложность**: Средний

---

## Обзор

В этой главе рассматриваются важные этапы планирования и проверки перед развертыванием вашего приложения. Научитесь избегать дорогостоящих ошибок с помощью правильного планирования емкости, выбора SKU и предварительной проверки.

> Проверено на `azd 1.23.12` в марте 2026 года.

## Цели обучения

Выполнив эту главу, вы сможете:
- Запускать предварительные проверки перед развертыванием
- Планировать емкость и оценивать требования к ресурсам
- Выбирать подходящие SKU для оптимизации затрат
- Настраивать Application Insights для мониторинга
- Понимать паттерны координации команды

## 🚀 Быстрый старт

```bash
# Проверить квоты подписки
az vm list-usage --location eastus --output table

# Предварительный просмотр развертывания (ресурсы не создаются)
azd provision --preview

# Проверить синтаксис Bicep
az bicep build --file infra/main.bicep

# Проверить конфигурацию среды
azd env get-values
```

## ☑️ Контрольный список перед развертыванием

### До `azd provision`

- [ ] Квота проверена для региона
- [ ] SKU выбраны правильно
- [ ] Пересмотрена оценка затрат
- [ ] Последовательное именование
- [ ] Настроена безопасность/RBAC

### До `azd deploy`

- [ ] Установлены переменные окружения
- [ ] Секреты в Key Vault
- [ ] Проверены строки подключения
- [ ] Настроены проверки работоспособности
