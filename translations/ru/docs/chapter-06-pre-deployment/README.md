# Глава 6: Планирование и проверка перед развертыванием

**📚 Курс**: [AZD для начинающих](../../README.md) | **⏱️ Длительность**: 1 час | **⭐ Сложность**: Средний уровень

---

## Обзор

В этой главе рассматриваются важные шаги планирования и проверки перед развертыванием вашего приложения. Узнайте, как избежать дорогостоящих ошибок с помощью правильного планирования емкости, выбора SKU и предварительных проверок.

> Проверено на `azd 1.27.1` в июле 2026 года.

## Цели обучения

После изучения этой главы вы сможете:
- Выполнять предварительные проверки перед развертыванием
- Планировать емкость и оценивать потребности в ресурсах
- Выбирать подходящие SKU для оптимизации затрат
- Настраивать Application Insights для мониторинга
- Понимать схемы координации команды

## 🚀 Быстрый старт

```bash
# Проверить квоты подписки
az vm list-usage --location eastus --output table

# Предпросмотр развертывания (ресурсы не создаются)
azd provision --preview

# Проверить синтаксис Bicep
az bicep build --file infra/main.bicep

# Проверить конфигурацию окружения
azd env get-values
```

## ☑️ Контрольный список перед развертыванием

### Перед `azd provision`

- [ ] Проверена квота для региона
- [ ] Выбраны подходящие SKU
- [ ] Проверена смета затрат
- [ ] Соблюдена единообразная схема именования
- [ ] Настроена безопасность/RBAC

### Перед `azd deploy`

- [ ] Установлены переменные окружения
- [ ] Секреты сохранены в Key Vault
- [ ] Проверены строки подключения
- [ ] Настроены проверки состояния
