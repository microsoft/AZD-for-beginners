# Глава 4: Инфраструктура как Код и Развертывание

**📚 Курс**: [AZD для начинающих](../../README.md) | **⏱️ Продолжительность**: 1-1.5 часа | **⭐ Сложность**: Средний уровень

## Обзор

Эта глава охватывает шаблоны Infrastructure as Code (IaC) с Bicep, создание ресурсов и стратегии развертывания с помощью Azure Developer CLI.

> Проверено с `azd 1.23.12` в марте 2026 года.

## Цели обучения

После прохождения этой главы вы сможете:
- Понимать структуру и синтаксис шаблонов Bicep
- Создавать ресурсы Azure с помощью `azd provision`
- Развертывать приложения с помощью `azd deploy`
- Реализовывать стратегии Blue-Green и Rolling развертывания

## 🚀 Быстрый старт

```bash
# Инициализация из шаблона
azd init --template azure-functions-python-v2-http

# Предварительный просмотр того, что будет создано
azd provision --preview

# Только обеспечение инфраструктуры
azd provision

# Только развертывание кода
azd deploy

# Или оба вместе
azd up
```

## 📁 Структура проекта AZD

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

---

## 🔧 Основные команды

| Команда | Описание |
|---------|-------------|
| `azd init` | Инициализация проекта |
| `azd provision` | Создание ресурсов Azure |
| `azd deploy` | Развертывание кода приложения |
| `azd up` | создание ресурсов + развертывание |
| `azd down` | Удаление всех ресурсов |
