# Глава 1: Основы и быстрое начало

**📚 Курс**: [AZD для начинающих](../../README.md) | **⏱️ Продолжительность**: 30-45 минут | **⭐ Сложность**: Для начинающих

## Обзор

В этой главе рассмотрены основы Azure Developer CLI (azd). Вы изучите основные концепции, установите инструменты и развернете своё первое приложение в Azure.

> Проверено с `azd 1.25.6` в июне 2026 года.

## Цели обучения

По завершении этой главы вы:
- Поймёте, что такое Azure Developer CLI и чем он отличается от Azure CLI
- Установите и настроите AZD на своей платформе
- Развернёте своё первое приложение в Azure с помощью `azd up`
- Очистите ресурсы с помощью `azd down`

## ✅ Начните здесь: проверьте вашу установку

Перед началом убедитесь, что ваш локальный компьютер готов к шаблону главы 1:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

Если скрипт сообщает об отсутствии инструментов, сначала устраните эти проблемы, затем продолжайте работу с главой.

---

## 🚀 Быстрый старт

```bash
# Проверить установку
azd version

# Аутентификация для AZD
# Необязательно: az login, если вы планируете запускать команды Azure CLI напрямую
azd auth login

# Развернуть ваше первое приложение
azd init --template todo-nodejs-mongo
azd up

# Очистить после завершения работы
azd down --force --purge
```

---

## ✅ Критерии успеха

После завершения этой главы вы должны:

```bash
azd version              # Показывает установленную версию
azd init --template todo-nodejs-mongo  # Инициализирует проект
azd up                   # Развёртывает в Azure
azd show                 # Отображает URL запущенного приложения
azd down --force --purge # Очищает ресурсы
```
