**📚 Курс**: [AZD для начинающих](../../README.md) | **⏱️ Длительность**: 30-45 минут | **⭐ Сложность**: Начинающий

## Обзор

В этой главе представлены основы Azure Developer CLI (azd). Вы узнаете основные концепции, установите инструменты и развернёте своё первое приложение в Azure.

> Проверено на версии `azd 1.23.12` в марте 2026 года.

## Цели обучения

Пройдя эту главу, вы:
- Поймёте, что такое Azure Developer CLI и чем он отличается от Azure CLI
- Установите и настроите AZD на своей платформе
- Развернёте первое приложение в Azure с помощью `azd up`
- Очистите ресурсы с помощью `azd down`

## 🚀 Быстрый старт

```bash
# Проверить установку
azd version

# Аутентификация для AZD
# Необязательно: az login, если вы планируете запускать команды Azure CLI напрямую
azd auth login

# Разверните ваше первое приложение
azd init --template todo-nodejs-mongo
azd up

# Очистить после завершения работы
azd down --force --purge
```

---

## ✅ Критерии успешного выполнения

По завершении этой главы вы должны уметь:

```bash
azd version              # Показывает установленную версию
azd init --template todo-nodejs-mongo  # Инициализирует проект
azd up                   # Разворачивает в Azure
azd show                 # Отображает URL запущенного приложения
azd down --force --purge # Очищает ресурсы
```
