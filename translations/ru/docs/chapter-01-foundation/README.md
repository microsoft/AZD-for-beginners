# Глава 1: Основы и Быстрый старт

**📚 Курс**: [AZD для начинающих](../../README.md) | **⏱️ Длительность**: 30-45 минут | **⭐ Сложность**: Для начинающих

## Обзор

В этой главе рассматриваются основы Azure Developer CLI (azd). Вы узнаете ключевые концепции, установите инструменты и развернете ваше первое приложение в Azure.

> Проверено на `azd 1.27.1` в июле 2026.

## Цели обучения

По завершении этой главы вы:
- Поймёте, что такое Azure Developer CLI и чем он отличается от Azure CLI
- Установите и настроите AZD на своей платформе
- Развернёте ваше первое приложение в Azure с помощью `azd up`
- Очистите ресурсы с помощью `azd down`

## ✅ Начните здесь: Проверьте вашу среду

Перед началом убедитесь, что ваша локальная машина готова для шаблона главы 1:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

Если скрипт сообщает о недостающих инструментах, сначала установите их, а затем продолжайте главу.

---

## 🚀 Быстрый старт

```bash
# Проверить установку
azd version

# Аутентификация для AZD
# Опционально: az login, если вы планируете запускать команды Azure CLI напрямую
azd auth login

# Разверните ваше первое приложение
azd init --template todo-nodejs-mongo
azd up

# Очистить после завершения
azd down --force --purge
```

---

## ✅ Критерии успешного завершения

По окончании этой главы вы должны уметь:

```bash
azd version              # Показывает установленную версию
azd init --template todo-nodejs-mongo  # Инициализирует проект
azd up                   # Разворачивает на Azure
azd show                 # Отображает URL работающего приложения
azd down --force --purge # Очищает ресурсы
```
