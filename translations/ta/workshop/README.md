# AZD for AI Developers Workshop

Azure Developer CLI (AZD) ஐப் பற்றி கற்றுக்கொள்ளும் இந்த செயலில் ஈடுபடும் பயிற்சிக்கு வருகை. இந்த பயிற்சி AZD டெம்ப்ளேட்களை 3 படிகள் மூலம் நடைமுறை அறிவாக கற்றுக் கொடுக்க உதவுகிறது:

1. **Discovery** - உங்களுக்கு பொருத்தமான டெம்ப்ளேட்டை கண்டுபிடிக்கவும்.
1. **Deployment** - வெளியிடி அதை செயற்படுகிறதா என்று உறுதிசெய்க
1. **Customization** - மாற்றம் செய்து உங்கள் தேவைக்கு ஏற்ப திருத்தி மீண்டும் முயற்சி செய்யுங்கள்!

இந்த பயிற்சி முழுவதும், முக்கிய டெவலப்பர் கருவிகள் மற்றும் பணியெழுத்துக்களையும் அறிமுகப்படுத்துவோம், உங்கள் முன் முதல் முடிந்த வரை உள்ள மேம்பாட்டு பயணத்தை எளிதாக்க உதவும்.

## Browser-Based Guide

பயிற்சி பாடங்கள் Markdown இல் உள்ளன. நீங்கள் அவற்றை நேரடியாக GitHub-இல் வழிசெலுத்தலாம் - அல்லது கீழே உள்ள ஸ்க்ரீன்ஷாட்டில் காட்டப்படுள்ள போல ஒரு உலாவி அடிப்படையிலான முன்னோட்டத்தை துவங்கலாம்.

![வொர்க்ஷாப்](../../../translated_images/ta/workshop.75906f133e6f8ba0.webp)

இந்த விருப்பத்தை பயன்படுத்த - கோத் உருப்படியை உங்கள் ப்ரொஃபைலுக்கு fork செய்து, GitHub Codespaces-ஐ துவங்கவும். VS Code டெர்மினல் செயல்படும்போது, இந்த கட்டளையை تایப் செய்யவும்:

```bash title="" linenums="0"
mkdocs serve > /dev/null 2>&1 &
```

ஒரு சில விநாடிகளுக்குள், ஒரு pop-up உரையாடல் தோன்றும். `Open in browser` என்ற விருப்பத்தை தேர்ந்தெடுக்கவும். இணைய அடிப்படையிலான கையேடு ஒரு புதிய உலாவி தாவலில் திறக்கப்படும். இந்த முன்னோட்டத்தின் சில நன்மைகள்:

1. **உள்ளகத் தேடல்** - விரைவில் சொல்லற்தொடர்கள் அல்லது பாடங்களை கண்டுபிடிக்கவும்.
1. **நகல் ஐகான்** - குறியீடு தொகுதிகளில் மீது ஊடுருவி இந்த விருப்பத்தை காணலாம்
1. **தீம் மாறுதல்** - இருண்ட மற்றும் ஒளிர் தீம்களுக்கு இடையே மாறுங்கள்
1. **உதவி பெறவும்** - காலடி பகுதியில் உள்ள Discord ஐகானை கிளிக் செய்து இணையுங்கள்!

## What You'll Learn

AZD டெம்ப்ளேட்டை Microsoft Foundry-இல் முதலிலிருந்து இறுதி வரை மேம்பாட்டு திறன்களை ஆராய ஒரு கற்றல் sandbox-ஆக நினைத்துக் கொள்ளுங்கள். இந்த பயிற்சி முடிவிற்குள், நீங்கள் இந்த சூழலில் பல கருவிகள் மற்றும் கருத்துக்களை உணர்வூட்டத்தன்மையோடு புரிந்துகொள்வீர்கள்.

| கருத்து  | நோக்கம் |
|:---|:---|
| **Azure Developer CLI** | கருவியின் கட்டளைகள் மற்றும் பணியெழுத்து நடைமுறைகளை புரிந்துகொள்ளுதல் |
| **AZD Templates**| திட்டத்தின் கட்டமைப்பு மற்றும் உள்ளமைவை புரிந்துகொள்ளுதல் |
| **Azure AI Agent**| Microsoft Foundry திட்டத்தை உருவாக்கி வெளியிடுதல்  |
| **Azure AI Search**| எஜன்ட்களுடன் சூழல் பொறியியல் செயல்படுத்துதல் |
| **Observability**| வழித்தட கண்காணிப்பு, மானிடரிங் மற்றும் மதிப்பீடுகளை ஆராய்தல் |
| **Red Teaming**| எதிர்த்தரப்பு சோதனைகள் மற்றும் தடுப்பு நடைமுறைகளை ஆராய்தல் |

## Workshop Structure

பயிற்சி டெம்ப்ளேட் கண்டறிதலை துவங்கி, வெளியீடு, பிரித்தறிதல் மற்றும் தனிப்பயனாக்கம் வரை ஒரு பயணமாக அமைக்கப்பட்டுள்ளது - அதிகாரப்பூர்வமான [AI ஏஜன்ட்களுடன் துவங்குதல்](https://github.com/Azure-Samples/get-started-with-ai-agents) ஸ்டார்டர் டெம்ப்ளேட்டை அடிப்படையாகக் கொண்டு.

## Workshop Challenge

| Project | Description |
|:---|:---|
