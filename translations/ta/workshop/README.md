# AI டெவலப்பர்களுக்கான AZD பட்டறை

Azure Developer CLI (AZD) கற்றுக்கொள்ளும் மற்றும் AI பயன்பாட்டு இணைப்பை (deployment) கவனிக்கும்துறைமையில் அமைக்கப்பட்ட கைகொடுத்த பட்டறைக்கு வரவேற்கிறோம். இந்த பட்டறை உங்களுக்கு AZD மாதிரிகளை பயன்படுத்தி செயல்பாட்டைத் தெளிவாக புரிந்துகொள்ள 3 படிகளாக உதவுகிறது:

1. **ஆராய்வு** - உங்களுக்கு பொருத்தமான மாதிரியை கண்டுபிடிக்க.
1. **நிறுவல்** - நிறுவி அது பணிபுரிகிறதா என்று சரிபார்க்க.
1. **தனிப்பயனாக்கம்** - மாற்றி மீண்டும் சுழற்றி அதை உங்கள் தேவைக்கதாக மாற்றுதல்!

இந்த பட்டறையின் போது, உங்களுக்கு கருவிகள் மற்றும் வேலைமுறைகள் பற்றிய முக்கியமான டெவலப்பர் கருவிகள் அறிமுகம் செய்யப்படும், இதனால் உங்கள் முடிவுக்கான டெவலப்மென்ட் பயணத்தை எளிதாக்கலாம்.

## உலாவி அடிப்படையிலான வழிகாட்டி

பட்டறை பாடங்கள் Markdown இல் உள்ளன. நீங்கள் அவற்றை நேரடியாக GitHub இல் வழிசெய்து செல்ல முடியும் - அல்லது கீழே திரைபடத்தில் காட்டப்படுவது போல ஒரு உலாவி அடிப்படையிலான முன்னோட்டத்தை (preview) துவங்கலாம்.

![பயிற்சி](../../../translated_images/ta/workshop.75906f133e6f8ba0.webp)

இந்த விருப்பத்தை பயன்படுத்த - கோப்பகத்தை உங்கள் சுயவிவரத்திற்கு fork செய்து GitHub Codespaces ஐ துவங்கி விடுங்கள். VS Code டெர்மினல் செயல்படும்போது, கீழ்க்கண்ட கட்டளையை உள்ளீடு செய்யுங்கள்:

This browser preview works in GitHub Codespaces, dev containers, or a local clone with Python and MkDocs installed.

```bash title="" linenums="0"
mkdocs serve > /dev/null 2>&1 &
```

In a few seconds, you will see a pop-up dialog. Select the option to `Open in browser`. The web-based guide will now open in a new browser tab. Some benefits of this preview:

1. **இறுக்க உள்ள தேடல்** - விசையொற்றுக்கள் அல்லது பாடங்களை விரைவாக கண்டுபிடிக்க.
1. **ஒளிப்பகிர்வு ஐகான்** - கோட்ப்ளாக்களின் மீது மவுஸ் வைத்தால் இதன் விருப்பம் தெரியும்
1. **தீம் மாற்றி** - இருண்ட மற்றும் ஒளির தீம்களுக்கு மாறுக
1. **உதவி பெறுங்கள்** - காலடியில் உள்ள Discord ஐகானை கிளிக் செய்து சேருங்கள்!

## நீங்கள் என்ன கற்பீர்கள்

AZD மாதிரியை Microsoft Foundry இல் முடிவுத் தொடக்கம் வரை செயல்படுத்தும் பல திறன்கள் மற்றும் கருவிகளை ஆராய ஒரு கற்றல் சந்தையாக நினைத்துக்கொள்ளுங்கள். இந்த பட்டறையின் முடிவில், இந்த சூழ்நிலையில் பயன்படும் பல கருவுகள் மற்றும் கருத்துக்களுக்கான ஒரு உணர்ச்சிகரமான புரிதல் கிடைக்கும்.

| Concept  | Objective |
|:---|:---|
| **Azure Developer CLI** | கருவி கட்டளைகள் மற்றும் வேலைநடப்புகளை புரிந்துகொள்ள|
| **AZD Templates**| திட்ட கட்டமைப்பும் குறிப்புகளும் பற்றிய புரிதல்|
| **Azure AI Agent**| Microsoft Foundry திட்டத்தினைப் Provision செய்து நிறுவலு |
| **Azure AI Search**| ஏஜெண்டுகளுடன் சூழல் பொறியியலை செயல்படுத்துதல் |
| **Observability**| இடுகாட்டுதல், கண்காணிப்பு மற்றும் மதிப்பீடுகளை ஆராய்வு |
| **Red Teaming**| எதிரி சோதனைகள் மற்றும் தடுப்பு நடவடிக்கைகள் பற்றி ஆராய்வு |
