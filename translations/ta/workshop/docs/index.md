# AI டெவலப்பர்களுக்கான AZD பயிற்சி பட்டறை

> Validated against `azd 1.25.6` in June 2026.
> [!IMPORTANT]  
> **இந்த பயிற்சி பட்டறையில் உள்ளூர் உலாவியில் முன்னோட்டம் பார்க்கக்கூடிய ஒரு பட்டறை வழிகாட்டி இணைக்கப்பட்டிருக்கிறது. தொடங்க, repo-ஐ GitHub Codespaces-இல், ஒரு dev container-இல், அல்லது MkDocs நிறுவப்பட்ட உள்ளூர் கிளோனில் திறந்தவுடன் — செயல்பாட்டில் உள்ள VS Code டெர்மினலை காணும் வரை காத்திருங்கள் மற்றும் பின்னர் টাইப் செய்யவும்:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **உலாவியில் முன்னோட்டப் பக்கத்தைத் திறக்க ஒரு பாப்அப் உரையாடல் தோன்றும்.**

Azure Developer CLI (AZD) கற்றுக்கொள்வதற்கான, AI பயன்பாடு டெப்பிளாய்மென்டில் கவனம் செலுத்திய கைபடி (hands-on) பட்டறைக்கு வரவேற்பு. இந்த பட்டறை AZD வார்ப்புருக்களை 3 படிகளில் நடைமுறையில் புரிந்துகொள்ள உதவுகிறது:

1. **கண்டுபிடிப்பு** - உங்கள் தேவைக்கு ஏற்ற வார்ப்புருவை கண்டறிதல்.
1. **பயன்படுத்தல்** - அதை டெப்பிளாய் செய்து அது வேலை செய்கிறதா என்பதை சரிபார்த்தல்.
1. **தனிப்பயன் அமைத்தல்** - அதை மாற்றி, சீரமைத்து உங்கள் தேவைக்கு ஏற்பியடுக்கும் வரை செயல்முறை செய்தல்!

இந்த பட்டறையின் காலப்பகுதியில், நீங்கள் முக்கியமான டெவலப்பர் கருவிகள் மற்றும் பணிச்சார்ந்துதல் (workflows) பற்றி அறிமுகமடைவீர்கள், இது உங்கள் தொடக்கம் முதல் முடிவு வரை உள்ள முழுமையான டெவலப்மெண்ட் பயணத்தை எளிதாக்க உதவும்.

## பட்டறை கண்ணோட்டம்

**காலம்:** 3-4 மணி நேரம்  
**நிலை:** ஆரம்பம் முதல் இடைநிலை வரை  
**முன் தேவைகள்:** Azure, AI கருத்துகள், VS Code மற்றும் கட்டளை வரிசை கருவிகளுடன் பரிச்சயம்.

இது செய்முறை-கையடி பட்டறை; செயல்படுவதன் மூலம் நீங்கள் கற்றுக்கொள்ளirsiniz. பயிற்சிகளை முடித்தவுடன், பாதுகாப்பு மற்றும் செயல்திறன் சிறந்த பயிற்சிகள் ஆகியவைக்கு தொடர்வதற்காக AZD தொடக்கத்திற்கான பாடத்திட்டத்தை (AZD For Beginners curriculum) மீண்டும் பரிசீலிக்க நாம் பரிந்துரைக்கிறோம்.

| Time| Module  | Objective |
|:---|:---|:---|
| 15 mins | Introduction | துவக்க நிலையை அமைத்து, நோக்கங்களைப் புரிந்துகொள்ளுதல் |
| 30 mins | Select AI Template | விருப்பங்களை ஆராய்ந்து தொடக்கத்தை தேர்வு செய்தல் | 
| 30 mins | Validate AI Template | இயல்புநிலை தீர்வை Azure-க்கு அமைத்து சரிபார்த்தல் |
| 30 mins | Deconstruct AI Template | கட்டமைப்பு மற்றும் உள்ளமைவுகளை ஆராய்தல் |
| 30 mins | Configure AI Template | கிடைக்கும் அம்சங்களை இயக்கி முயற்சித்து பார்க்குதல் |
| 30 mins | Customize AI Template | வார்ப்புருவை உங்கள் தேவைகளுக்கு ஏற்ப தனிப்பயனாக்கல் |
| 30 mins | Teardown Infrastructure | வளங்களை சுத்தம் செய்து மற்றும் விடுவித்தல் |
| 15 mins | Wrap-Up & Next Steps | கற்பனை வளங்கள் மற்றும் பட்டறை சவால் |
| | |

## நீங்கள் என்ன கற்றுக்கொள்வீர்கள்

AZD வார்ப்புருவை Microsoft Foundry-இல் முடிந்தவரை கற்றல் சாங்க்ஃபாக்ஸ்பாக கருதுங்கள் — இது துவக்கத்திலிருந்து இறுதி வரை டெவலப்மென்ட் திறன்களை மற்றும் கருவிகளை ஆராய ஒரு sandbox ஆகும். இந்த பட்டறையை முடித்தவுடன், இச்சூழலில் உள்ள பல கருவிகள் மற்றும் கருத்துக்களை இயல்பாக உணர்வீர்கள்.

| Concept  | Objective |
|:---|:---|
| **Azure Developer CLI** | கருவி கட்டளைகள் மற்றும் வேலைநடைகளைப் புரிந்துகொள்ளுதல் |
| **AZD Templates**| திட்ட அமைப்பு மற்றும் கட்டமைப்புகளைப் புரிந்துகொள்ளுதல் |
| **Azure AI Agent**| Microsoft Foundry திட்டத்தை உருவாக்கி மற்றும் செயல்படுத்துதல் |
| **Azure AI Search**| ஏஜென்ட்களுடன் உள்ளடக்க பொறியியலுக்கு ஆதரவு வழங்குதல் |
| **Observability**| டிரேசிங், கண்காணிப்பு மற்றும் மதிப்பீடுகளை ஆராய்தல் |
| **Red Teaming**| எதிரணித் சோதனை மற்றும் குறைபாடுகளை குறைக்கும் நடவடிக்கைகளை ஆராய்தல் |
| | |
