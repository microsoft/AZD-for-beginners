# AI டெவலப்பர்களுக்கான AZD வேலைநிறுவல்

> `azd 1.27.1` இன் கீழ் ஜூலை 2026 இல் செயல்முறைசெய்யப்பட்டது.

> [!IMPORTANT]  
> **இந்த வேலைநிறுவல் உங்கள் உள்ளூர் உலாவியில் முன்னோட்டமாக பார்க்கக்கூடிய வேலைநிறுவல் கையேட்டுடன் உபகரணப்படுத்தப்பட்டுள்ளது. தொடங்க, GitHub Codespaces இல், ஒரு dev container இல், அல்லது MkDocs நிறுவப்பட்ட உள்ளூர் கிளோனில் repo நுகர்ந்து, செயல்படுத்தப்பட்ட VS Code டெர்மினல் வரை காத்திருந்து பின்னர் கீழ்காணும் கட்டளையை தவிர்க்கவும்:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **நீங்கள் உலாவியில் முன்னோட்ட பக்கத்தை திறக்கும் ஒரு பாப்-அப் உரையாடல் காண்பீர்கள்.**

AI பயன்பாட்டை விரிவுபடுத்துவதில் கவனம் வைத்து Azure Developer CLI(AZD) கற்றுக்கொள்ள இந்த கைலான வேலைநிறுவலில் வரவேற்கின்றோம். இந்த வேலைநிறுவல் AZD தளங்களை 3 படிகள் மூலம் பயன்பாட்டு புரிதலை வழங்குகிறது:

1. **கண்டுபிடித்தல்** - உங்களுக்கு சரியான தளத்தை கண்டுபிடிக்கவும்.
1. **விரிவுபடுத்தல்** - விரிவுபடுத்தி அது செயல்படுவதை உறுதிப்படுத்தவும்
1. **தனிப்பயனாக்கல்** - திருத்தி அதை உங்கள் மார்க்கமாக மாற்றவும்!

இந்த வேலைநிறுவல் காலத்தில், நீங்கள் முக்கிய தொடக்க கருவிகள் மற்றும் வேலைநிறுவல் பாணிகளையும் அறிந்துகொண்டு, உங்கள் முழு வளர்ச்சி பயணத்தை எளிதாக்க உதவுகிறோம்.

## வேலைநிறுவல் முன்னோட்டம்

**காலவரைவு:** 3-4 மணிகள்  
**நிலை:** தொடக்கம் முதல் மைய நிலை வரை  
**முன் தேவைகள்:** Azure, AI கருத்துகள், VS Code மற்றும் கட்டளை வரிசை கருவிகளுடன் அறிமுகம்.

இது செய்து கற்றுக்கொள்ளும் வேலைநிறுவல் ஆகும். பயிற்சிகள் முடிந்ததும், பாதுகாப்பு மற்றும் உற்பத்தித்திறன் சிறந்த நடைமுறைகள் பற்றிய உங்கள் கற்றலை தொடர AZD தொடக்கர் பாடத்திட்டத்தை பரிந்துரைக்கிறோம்.

| நேரம்| தொகுப்பு  | நோக்கம் |
|:---|:---|:---|
| 15 நிமிடங்கள் | அறிமுகம் | துவக்க நிலை அமைத்து நோக்கங்களை புரிந்து கொள் |
| 30 நிமிடங்கள் | AI தளத்தை தேர்ந்தெடு | விருப்பங்களை ஆராய்ந்து தொடக்கத்தை தேர்ந்தெடு | 
| 30 நிமிடங்கள் | AI தளத்தை சரிபார் | Azure இல் இயங்குமாறு விரிவுபடுத்துக |
| 30 நிமிடங்கள் | AI தளத்தின் அமைப்பைக் கண்டு புரிந்து கொள் |
| 30 நிமிடங்கள் | AI தளத்தை கட்டமைக்கவும் | அமிலமான மற்றும் பயன்பாட்டைக் கற்று பாரு |
| 30 நிமிடங்கள் | AI தளத்தை தனிப்பயனாக்கு | உங்கள் தேவைக்கேற்ப மாற்றவும் |
| 30 நிமிடங்கள் | பண்படுத்திய கட்டமைப்பை அழி | சுத்தம் செய்யவும் வளங்களை விடுவிக்கவும் |
| 15 நிமிடங்கள் | முடிவு & அடுத்த படிகள் | கற்றல் வளங்கள், வேலைநிறுவல் சவால் |
| | |

## நீங்கள் கற்றுக்கொள்ளப்போகும் விஷயங்கள்

AZD தளத்தை Microsoft Foundry இல் முடிவுசார் வளர்ச்சிக்கு பல்வேறு திறன்கள் மற்றும் கருவிகளை ஆராயும் கற்றல் மைதானமாக காணுங்கள். இந்த வேலைநிறுவல் முடிவில், இந்த சூழலில் பல கருவிகள் மற்றும் கருத்துக்களை புலப்படுத்தும் உணர்வு கிடைக்கும்.

| கருத்து  | நோக்கம் |
|:---|:---|
| **Azure Developer CLI** | கருவி கட்டளைகள் மற்றும் வேலைநிறுவல் பாணிகளை அறிந்து கொள்ளவும்|
| **AZD தளங்கள்**| திட்ட அமைப்பு மற்றும் கட்டமைப்பை புரிந்துகொள்ளவும்|
| **Azure AI முகவர்**| Microsoft Foundry திட்டத்தை ஏற்படுத்து மற்றும் விரிவுபடுத்து |
| **Azure AI தேடல்**| முகவர்களுடன் சூழல் பொறியியல் செயல்படுத்துக |
| **காணொளித்தன்மை**| தடயப்படுத்தல், கண்காணிப்பு மற்றும் மதிப்பீடுகளை ஆராயவும் |
| **சிவப்பு அணிக் குழு**| எதிர்ப்புமிக்க சோதனை மற்றும் தடுப்பு முறைகளை ஆராயவும் |
| | |
