# AZD AI டெவலப்பர்களுக்கான

!!! info "இந்த பயிற்சி நிகழ்ச்சிக்கான பங்கேற்பாளர்கள்"
   
    இந்த பயிற்சி Microsoft Foundry-இல் சிறந்த கருவிகள் மற்றும் பணிமுறைகளை பயன்படுத்தி AI ஏஜென்ட் செயலிகளை உருவாக்க விரும்பும் எவருக்கும் பொருந்தும். இந்த பயிற்சி Azure, AI கருத்துகள் மற்றும் Python குறியீட்டுடன் சில அளவிலான பரிச்சயமுண்டான தொடக்க நிலை மற்றும் அனுபவமுள்ள டெவலப்பர்களுக்குப் பொருத்தமாகும். தொடங்குவதற்கு முன் முன்-தேவைகளை சரிபார்க்கவும்.

    **காலம்:** 3-4 மணி நேரம்  
    **நிலை:** தொடக்க நிலை முதல் நடுத்தர வரை  

---


## முன்-தேவைகள்

- தனிப்பட்ட GitHub கணக்கு → [இதை இலவசமாக உருவாக்குக](https://github.com/signup)
- செயலில் உள்ள Azure கணக்கு → [இதை இலவசமாக உருவாக்குக](https://aka.ms/free)
- AI கருத்துகள், Python அபிவிருத்தி மற்றும் கட்டளை வரி கருவிகளில் அடிப்படை அறிமுகம்
- VS Code மற்றும் GitHub கருவிகள் மற்றும் பணிமுரைகளில் அனுபவம்.

---

## கற்றல் நோக்கங்கள்

இந்த பயிற்சி முடிவில், நீங்கள் இதைச் செய்ய தெரிந்திருப்பீர்கள்:

- [X] **Azure Developer CLI உடன் பணியாற்றுதல்** → உங்கள் Azure வளங்களை ஒதுக்க, நிறுவ மற்றும் நிர்வகிக்க.
- [X] **AI App Templates பயன்படுத்துதல்** → Microsoft Foundry இல் AI ஏஜென்ட் திட்டங்களை துவக்க உதவ.
- [X] **சाँचுகளை கட்டமைத்தல்** → Red Teaming, மதிப்பீடுகள், பின்தொடர்தல் & கண்காணிப்பு போன்ற அம்சங்களுக்கு
- [X] **பகுத்தறிதல் மற்றும் ஆராய்ச்சி** → GitHub Copilot-ஐப் பயன்படுத்தி AZD சாஞ்சுகள் மற்றும் பணிமுறைகளை ஆராய்ந்து புரிதல்
- [X] **தனிப்பயனாக்கி மீண்டும் வெளியிடுதல்** → உங்கள் தனிப்பட்ட தேவைகளுடன் AI Agent தொடக்க சாங்கை அமைத்து மீண்டும் வெளியிடுதல்

## பயிற்சி அமைப்பு

பயிற்சி பல மாட்யூல்களாக அமைக்கப்பட்டுள்ளது, ஒவ்வொன்றும் ஒரு கைமுறை ஆய்வு பயிற்சியால் ஆதாரம் கொண்டவை:

1. **அறிமுகம்** - AZD சாஞ்சுகள் என்ன என்பதையும் அவை எப்படி செயல்படுகின்றன என்பதையும் புரிந்துகொள்ளுதல்
1. **தேர்வு** - உங்கள் அடுத்த AI திட்டத்திற்கு சரியான "தொடக்க" சாஞ்சை கண்டுபிடித்தல்
1. **சரிபார்ப்பு** - சாஞ்சை орналுப்படுத்து, அது விளம்பரிக்கப்பட்டபடி செயல்படுகிறதா என்று உறுதிப்படுத்தல்
1. **பிரித்தெடுத்தல்** - அதன் கூறுகள் மற்றும் கட்டமைப்புகளை புரிந்துகொள்ள சாஞ்சை பிரித்து பகுப்பாய்வு செய்தல்
1. **விளக்கம்** - Microsoft Foundry இன் விதவிதமான அம்சங்களை இயக்கு மற்றும் பயன்படுத்த கற்றுக் கொள்வது
1. **தனிப்பயனாக்கல்** - உங்கள் தரவு மற்றும் தேவைகளை கருத்தில் கொண்டு பயன்பாட்டை மீள்தயார் செய்தல்
1. **இறுதிப் பணி** - இன்ஃப்ராஸ்ட்ரக்சரை அகற்று மற்றும் மென்மையான-நீக்கப்பட்ட வளங்களை மீட்டெடுக்க கற்று கொள்வது

## கருத்துக்கள் மற்றும் பிரச்சினைகள்

நாங்கள் பாடத்திட்டத்தை தொடர்ச்சியாக புதுப்பித்து, தொடக்க நிலையில் உள்ளவர்களுக்கு சிறந்த கற்றல் அனுபவம் வழங்க உறுதிசெய்கிறோம். உங்கள் கருத்துக்களை எதிர்நோக்குகிறோம்.

1. இந்த ரெப்போவில் ஒரு issue பதிவிடுங்கள் - வசதிக்காக அதற்கு `Workshop` என்ற குறிச்சொல்லை சேர்க்கவும்.
1. Microsoft Foundry Discord-இல் சேருங்கள் - மற்ற பயில்வோருடன் இணைந்துகொள்ளுங்கள்!
