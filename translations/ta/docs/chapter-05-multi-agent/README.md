**📚 பாடநெறி**: [AZD தொடக்கக்காரர்கள்](../../README.md) | **⏱️ காலம்**: 2-3 மணிநேரம் | **⭐ சிரமம்**: மேம்பட்ட

## கண்ணோட்டம்

இந்த அதிகாரம் மேம்பட்ட பல-ஏஜென்ட் கட்டமைப்பு மாதிரிகள், ஏஜென்ட் ஒர்க்கெஸ்ட்ரேஷன் மற்றும் சிக்கலான சூழல்களுக்கு உற்பத்தி-தயார் AI ஒருங்கட்டமைப்புகள் பற்றி பேசுகிறது.

## கற்றல் நோக்கங்கள்

இந்த அதிகாரத்தை முடித்தவுடன், நீங்கள்:
- பல-ஏஜென்ட் கட்டமைப்பு மாதிரிகளைப் புரிந்துகொள்வீர்கள்
- இணைந்த AI ஏஜென்ட் அமைப்புகளை நிறுவுவது
- ஏஜென்ட்-மீது-ஏஜென்ட் தொடர்பை நடைமுறைப்படுத்துவது
- உற்பத்தி-தயார் பல-ஏஜென்ட் தீர்வுகளை உருவாக்குவது

## 🚀 விரைவான தொடக்கம்

```bash
# சில்லறை பல முகவர்கள் கொண்ட தீர்வை செயல்படுத்தவும்
cd examples/retail-multiagent-arm-template
./deploy.sh

# அல்லது வார்ப்புருவை நேரடியாகப் பயன்படுத்தவும்
azd init --template agent-openai-python-prompty
azd up
```

## 🎯 சிறப்பு தீர்வு: சில்லறை பல-ஏஜென்ட்

இந்த [சில்லறை பல-ஏஜென்ட் தீர்வு](../../examples/retail-scenario.md) வெளிப்படுத்துகிறது:

- **Customer Agent**: பயனர் தொடர்புகள் மற்றும் விருப்பங்களை கையாளுகிறது
- **Inventory Agent**: பங்கு மற்றும் ஆர்டர் செயலாக்கத்தை நிர்வகிக்கிறது
- **Orchestrator**: ஏஜென்டுகளுக்கு இடையில் ஒருங்கிணைக்கிறது
- **Shared Memory**: ஏஜென்டுகளுக்கு இடையிலான சூழ்நிலை நிர்வாகம்

| Container Apps | ஏஜென்ட் ஹோஸ்டிங் |
| Application Insights | கண்காணிப்பு |
