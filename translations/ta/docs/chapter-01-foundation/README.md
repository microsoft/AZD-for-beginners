# Chapter 1: அடித்தளம் மற்றும் விரைவு தொடக்கம்

**📚 Course**: [AZD For Beginners](../../README.md) | **⏱️ Duration**: 30-45 நிமிடம் | **⭐ Complexity**: தொடக்க நிலை

---

## கண்ணோட்டம்

இந்த அத்தியாயம் Azure Developer CLI (azd) அடிப்படைகளை அறிமுகப்படுத்துகிறது. நீங்கள் முக்கிய கருத்துகளை, கருவிகளை நிறுவுவது மற்றும் உங்கள் முதல் பயன்பாட்டை Azure-க்கு பதிவேற்றுவது (deploy) பற்றி கற்றுக்கொள்வீர்கள்.

> 2026 மார்ச்சில் `azd 1.23.12` உடன் சரிபார்க்கப்பட்டது.

## கற்றல் நோக்கங்கள்

இந்த அத்தியாயத்தை முடித்தவுடன், நீங்கள்:
- Azure Developer CLI என்ன என்பது மற்றும் அது Azure CLI-இருந்து எப்படி வேறுபடுகிறது என்பதைப் புரிந்துகொள்வீர்கள்
- உங்கள் பிளாட்ஃபாரத்தில் AZD-ஐ நிறுவி கட்டமைப்பதை கற்றுக்கொள்வீர்கள்
- `azd up` பயன்படுத்தி Azure-க்கு உங்கள் முதல் பயன்பாட்டை பதிவேற்றுவது
- `azd down` மூலம் வளங்களை அகற்றுவது

## ✅ இங்கே தொடங்குங்கள்: உங்கள் அமைப்பை சரிபார்க்கவும்

Before you begin, confirm that your local machine is ready for the Chapter 1 template:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

If the script reports missing tools, fix those first and then continue with the chapter.

## 🚀 விரைவு தொடக்கம்

```bash
# நிறுவலைச் சரிபார்க்கவும்
azd version

# AZD-க்கு அங்கீகாரம் செய்யவும்
# விருப்பமாக: Azure CLI கட்டளைகளை நேரடியாக இயக்க திட்டமிட்டால் az login செய்யவும்
azd auth login

# உங்கள் முதல் செயலியை வெளியிடவும்
azd init --template todo-nodejs-mongo
azd up

# முடிந்ததும் சுத்தம் செய்யவும்
azd down --force --purge
```

## ✅ வெற்றியின் அளவுகோல்கள்

இந்த அத்தியாயத்தை முடித்தவுடன், நீங்கள் கீழ்க்கண்டவற்றை செய்ய முடியும்:

```bash
azd version              # நிறுவப்பட்ட பதிப்பை காட்டுகிறது
azd init --template todo-nodejs-mongo  # திட்டத்தை துவக்குகிறது
azd up                   # Azureக்கு வெளியிடுகிறது
azd show                 # இயங்கும் செயலியின் URL-ஐ காட்டுகிறது
azd down --force --purge # வளங்களை சுத்தம் செய்கிறது
```
