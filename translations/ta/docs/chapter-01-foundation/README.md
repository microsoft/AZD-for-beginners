## கண்ணோட்டம்

இந்த அத்தியாயம் Azure Developer CLI (azd)ன் அடிப்படைகளை அறிமுகப்படுத்துகிறது. நீங்கள் முக்கியக் கருத்துக்களைப் புரிந்து கொள்வீர்கள், கருவிகளை நிறுவி அமைக்கவும், மற்றும் உங்கள் முதல் பயன்பாட்டை Azure-க்கு பதிந்திடவும் கற்பீர்கள்.

> 2026 ஜூன் மாதத்தில் `azd 1.25.6` உடன் சரிபார்க்கப்பட்டது.

## Learning Objectives

இந்த அத்தியாயத்தை முடித்தவுடன், நீங்கள்:
- Azure Developer CLI என்ன என்பது மற்றும் அது Azure CLI-ஐவிட எப்படி வேறுபடுகிறது என்பதை புரிந்துகொள்ளலாம்
- உங்கள் பிளாட்ஃபாரத்தில் AZD-ஐ நிறுவி கட்டமைக்கலாம்
- `azd up` மூலம் உங்கள் முதல் பயன்பாட்டை Azure-க்கு பதிந்திடலாம்
- `azd down` மூலம் வளங்களை சுத்தம் செய்யலாம்

## ✅ இங்கு தொடங்குங்கள்: உங்கள் அமைப்பை சரிபார்க்கவும்

தொடங்குவதற்கு முன், உங்கள் உள்ளூர் கணினி அத்தியாயம் 1 டெம்ப்ளேட்டிற்காக தயாராக உள்ளதா என்பதை உறுதிப்படுத்தவும்:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

ஸ்கிரிப்ட் குறைந்த கருவிகளை அறிவித்தால், முதலில் அவற்றை திருத்தி பின்னர் அத்தியாயத்துடன் தொடருங்கள்.

---

## 🚀 விரைவான துவக்கம்

```bash
# நிறுவலைச் சரிபார்க்கவும்
azd version

# AZD-க்கு அங்கீகாரம் செய்யவும்
# விருப்பத்தேர்வு: Azure CLI கட்டளைகளை நேரடியாக இயக்க திட்டமிட்டால் az login செய்யவும்
azd auth login

# உங்கள் முதல் பயன்பாட்டை நிறுவவும்
azd init --template todo-nodejs-mongo
azd up

# முடிந்தவுடன் சுத்தம் செய்யவும்
azd down --force --purge
```

## ✅ வெற்றிக்கான அளவுகோல்கள்

இந்த அத்தியாயத்தை முடித்த பிறகு, நீங்கள் பின்வரும் திறன்களை பெறுவீர்கள்:

```bash
azd version              # நிறுவப்பட்ட பதிப்பை காட்டுகிறது
azd init --template todo-nodejs-mongo  # திட்டத்தை ஆரம்பிக்கிறது
azd up                   # Azure-க்கு வெளியிடுகிறது
azd show                 # ஓடிக்கொண்டிருக்கும் செயலியின் URL-ஐ காட்டுகிறது
azd down --force --purge # வளங்களை சுத்திகரிக்கிறது
```
