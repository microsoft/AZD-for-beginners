# அத்தியாயம் 1: அடித்தளம் மற்றும் விரைவு தொடக்கம்

**📚 பாடநெறி**: [AZD ஆரம்பத்திற்கான](../../README.md) | **⏱️ காலம்**: 30-45 நிமிடங்கள் | **⭐ சிக்கல்தனம்**: தொடக்கநிலை

---

## மேலோட்டம்

இந்த அத்தியாயம் Azure Developer CLI (azd) அடித்தளங்களை அறிமுகப்படுத்துகிறது. நீங்கள் முக்கிய கருத்துக்களை கற்றுக்கொள்வீர்கள், கருவிகளை நிறுவுவீர்கள், மற்றும் உங்கள் முதற் பயன்பாட்டை Azure இல் நிறுவுவீர்கள்.

## கற்றல் நோக்கங்கள்

இந்த அத்தியாயத்தை முடித்தவுடன், நீங்கள்:
- Azure Developer CLI என்ன மற்றும் அது Azure CLI இருந்து எப்படி வேறுபடுகிறது என்பதை புரிந்துகொள்வீர்கள்
- உங்கள் தளத்தில் AZD ஐ நிறுவி கட்டமைப்பீர்கள்
- `azd up` பயன்படுத்தி உங்கள் முதற் பயன்பாட்டை Azure இல் நிறுவுவீர்கள்
- `azd down` மூலம் வளங்களை அகற்றுவீர்கள்

## 🚀 விரைவான தொடக்கம்

```bash
# நிறுவலை சரிபார்க்கவும்
azd version

# Azure இல் உள்நுழையவும்
azd auth login

# உங்கள் முதல் செயலியை வெளியிடவும்
azd init --template todo-nodejs-mongo
azd up

# முடிந்ததும் சுத்தம் செய்யவும்
azd down --force --purge
```

## ✅ வெற்றி அளவுகோல்கள்

இந்த அத்தியாயத்தை முடித்த பிறகு, நீங்கள் அதைப் படைப்பதற்கும் நடத்தியதும்:

```bash
azd version              # நிறுவப்பட்ட பதிப்பை காட்டுகிறது
azd init --template todo-nodejs-mongo  # திட்டத்தை ஆரம்பிக்கிறது
azd up                   # Azure-க்கு வெளியிடுகிறது
azd show                 # இயங்கும் செயலியின் URL-ஐ காட்டுகிறது
azd down --force --purge # வளங்களை சுத்தம் செய்கிறது
```
