# அத்தியாயம் 1: அடித்தளம் மற்றும் விரைவில் துவக்கம்

**📚 பாடம்**: [AZD தொடக்கங்களுக்காக](../../README.md) | **⏱️ கால அளவு**: 30-45 நிமிடங்கள் | **⭐ சிக்கல்**: துவக்க நிலை

---

## அவலோக்கனம்

இந்த அத்தியாயத்தில் Azure Developer CLI (azd) அடிப்படைகளைக் கற்றுக்கொள்ளவும். நீங்கள் முக்கிய கருத்துக்களைப் புரிந்து கொள்ளுவீர்கள், கருவிகளை நிறுவுவீர்கள் மற்றும் உங்கள் முதல் பயன்பாட்டை Azure இல் பிரயோகம் செய்வீர்கள்.

> `azd 1.27.1` இன் கீழ் 2026 ஜூலை மாதத்தில் சரிபார்க்கப்பட்டது.

## கற்றல்திறன்கள்

இந்த அத்தியாயத்தை முடித்த பிறகு, நீங்கள்:
- Azure Developer CLI என்பது என்ன மற்றும் அது Azure CLI-இன் வேறுபாடுகள் பற்றி புரிந்து கொள்வீர்கள்
- உங்கள் தளத்தில் AZD-ஐ நிறுவி அமைப்பதைக் கற்றுக்கொள்வீர்கள்
- `azd up` கொண்டு உங்கள் முதல் பயன்பாட்டை Azure இல் பிரயோகம் செய்வீர்கள்
- `azd down` மூலம் வளங்களை சுத்தம் செய்வீர்கள்

## ✅ இங்கே துவக்கவும்: உங்கள் அமைப்பை சரிபார்க்கவும்

துவங்குவதற்கு முன்பு, உங்கள் உள்ளூர்முக்கிய கணினி அத்தியாயம் 1 திட்டத்திற்கு தயார் என உறுதி செய்யவும்:

**விண்டோஸ்:**
```powershell
.\validate-setup.ps1
```

**macOS / லினக்ஸ்:**
```bash
bash ./validate-setup.sh
```

ஸ்கிரிப்ட் குறித்த கருவிகள் இல்லையெனில், முதலில் அவற்றை சரிசெய்து அத்தியாயத்துடன் தொடரவும்.

---

## 🚀 விரைவில் துவக்கம்

```bash
# நிறுவலைச் சரிபார்க்கவும்
azd version

# AZD க்கான அங்கீகாரம் செய்க
# விருப்பம்: நீங்கள் நேரடியாக Azure CLI கட்டளைகளை இயக்க திட்டமிட்டால் az உள்நுழையவும்
azd auth login

# உங்கள் முதல் பயன்பாட்டை அமல்படுத்தவும்
azd init --template todo-nodejs-mongo
azd up

# முடிந்தபின் சுத்தம் செய்யவும்
azd down --force --purge
```

## ✅ வெற்றி அளவுகோல்கள்

இந்த அத்தியாயத்தை முடித்த பிறகு, நீங்கள் செய்யக்கூடியவை:

```bash
azd version              # நிறுவப்பட்ட பதிப்பை காட்டுகிறது
azd init --template todo-nodejs-mongo  # திட்டத்தை துவக்குகிறது
azd up                   # Azure-க்கு பிரயாணம் செய்கிறது
azd show                 # இயங்கும் செயலியின் URL ஐ காட்டு
azd down --force --purge # வளங்களை சுத்தம் செய்கிறது
```
