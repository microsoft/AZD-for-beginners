# அத்தியாயம் 8: உற்பத்தி & நிறுவன மாதிரிகள்

**📚 பாடநெறி**: [AZD For Beginners](../../README.md) | **⏱️ காலம்**: 2-3 மணிநேரங்கள் | **⭐ சிக்கல்தன்மை**: உயர்நிலை

---

## மேலோட்டம்

இந்த அத்தியாயம் நிறுவன-தகுதியான பிரசுரத் திட்டங்கள், பாதுகாப்பு உறுதிப்படுத்தல், கண்காணிப்பு மற்றும் உற்பத்தி AI பணிகளுக்கான செலவு தொகுப்புப் பலன்களை கையாள்வது பற்றி சுருக்கமாக கூறுகிறது.

## கற்றல் இலக்குகள்

இந்த அத்தியாயத்தை முடித்துவிட்டால், நீங்கள்:
- பல பிராந்தியங்களில் நிலைத்தன்மையுடன் செயல்படும் பயன்பாடுகளை 배포 செய்யலாம்
- நிறுவன பாதுகாப்பு மாதிரிகளை நடைமுறைபடுத்தலாம்
- விரிவான கண்காணிப்பை कॉConfigure செய்யலாம்
- பரிமாண அளவில் செலவுகளை சிறப்பாக 최적화 செய்யலாம்
- AZD உடன் CI/CD குழாய்களை அமைக்கலாம்

- [ ] அங்கீகாரத்திற்கு மேலாண்மை அடையாளம் (கீகள் இல்லை)

- [ ] செலவுப் பட்ஜெட்டுகள் மற்றும் எச்சரிக்கைகள் அமைத்தல்
- [ ] பாதுகாப்பு ஸ்கானிங் செயல்படுத்தப்பட்டது
- [ ] CI/CD குழாய் ஒருங்கிணைப்பு
- [ ] விபத்து மீட்பு திட்டம்
