# அத்தியாயம் 4: அடிக்கமைப்பு குறியீடாக & நிலைத்தன்மை

**📚 பாடநெறி**: [AZD தொடக்கத்திற்கான](../../README.md) | **⏱️ நேரம்**: 1-1.5 மணி | **⭐ சிக்கல்**: நடுத்தர

---

## வழிகாட்டு பார்வை

இந்த அத்தியாயம் Bicep வடிவங்கள் கொண்ட அடிக்கமைப்பு குறியீடாக (IaC) மாதிரிகள், வளங்கள் ஏற்படுத்தல் மற்றும் Azure Developer CLI பயன்படுத்தி நிலைத்தன்மை நெறிமுறைகளை உள்ளடக்கியது.

> `azd 1.27.1`க்கு எதிராக 2026 ஜூலை மாதம் சரிபார்க்கப்பட்டது.

## கற்கும் குறிக்கோள்கள்

இந்த அத்தியாயத்தை முடித்தபின், நீங்கள்:
- Bicep வடிவமைப்பின் அமைப்பு மற்றும் சொற்வடிவத்தை புரிந்து கொள்வீர்கள்
- `azd provision` மூலம் Azure வளங்களை ஏற்படுத்துவீர்கள்
- `azd deploy` மூலம் பயன்பாடுகளை நிலைநிறுத்துவீர்கள்
- நீல-பச்சை மற்றும் எழுமுறை நிலைத்தன்மை நெறிமுறைகளை செயல்படுத்துவீர்கள்

## 🚀 விரைவான தொடக்கம்

```bash
# மாதிரியில் இருந்து துவக்கவும்
azd init --template azure-functions-python-v2-http

# உருவாக்கப்படும் ஒன்றை முன்னோட்டமாக பார்க்கவும்
azd provision --preview

# கட்டமைப்பை மட்டுமே ஏற்படுத்தவும்
azd provision

# கோடை மட்டும் அனுப்பவும்
azd deploy

# அல்லது இரண்டையும் ஒன்றாக செய்து விடவும்
azd up
```

## 🔧 அவசியமான கட்டளைகள்

| கட்டளை | விளக்கம் |
|---------|-------------|
| `azd init` | திட்டத்தை துவக்குகிறது |
| `azd provision` | Azure வளங்களை உருவாக்குகிறது |
| `azd deploy` | பயன்பாடு குறியீட்டினை நிலைநிறுத்துகிறது |
| `azd up` | வளங்கள் ஏற்படுத்தல் + நிலைநிறுத்தல் |
| `azd down` | அனைத்து வளங்களையும் அகற்றுகிறது |
