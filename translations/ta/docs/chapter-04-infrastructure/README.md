# அத்தியாயம் 4: கட்டமைப்பை குறியாக்கமாக (Infrastructure as Code) மற்றும் வெளியீடு

**📚 பாடநெறி**: [AZD தொடக்கர்களுக்காக](../../README.md) | **⏱️ காலம்**: 1-1.5 மணிநேரம் | **⭐ கடினத்தன்மை**: இடைநிலை

---

## கண்ணோட்டம்

இந்த அத்தியாயம் Bicep டெம்ப்ளேட்களுடன் Infrastructure as Code (IaC) மாதிரிகள், வளங்கள் ஒதுக்குதல் மற்றும் Azure Developer CLI ஐப் பயன்படுத்தி வெளியீட்டு உத்திகள் குறித்து உள்ளடக்குகிறது.

## கற்றல் இலக்குகள்

இந்த அத்தியாயத்தை முடித்தவுடன், நீங்கள்:
- Bicep டெம்ப்ளேட் கட்டமைப்பும் இலக்கணமும் பற்றி புரிந்துகொள்வீர்கள்
- `azd provision` பயன்படுத்தி Azure வளங்களை ஒதுக்குதல்
- `azd deploy` மூலம் பயன்பாடுகளை வெளியீடு செய்தல்
- blue-green மற்றும் rolling வெளியீட்டு உத்திகளை அமல்படுத்துதல்

## 🚀 விரைவான தொடக்கம்

```bash
# வார்ப்புருவிலிருந்து ஆரம்பிக்கவும்
azd init --template azure-functions-python-v2-http

# உருவாக்கப்படவுள்ளதை முன்னோட்டமாகப் பார்க்கவும்
azd provision --preview

# அடிப்படை கட்டமைப்புகளை மட்டும் ஏற்பாடு செய்க
azd provision

# குறியீட்டை மட்டுமே நிறுவவும்
azd deploy

# அல்லது இரண்டும் ஒன்றாகவும்
azd up
```

## 📁 AZD திட்ட அமைப்பு

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

## 🔧 அவசியமான கட்டளைகள்

| கட்டளை | விளக்கம் |
|---------|-------------|
| `azd init` | திட்டத்தை ஆரம்பிக்க |
| `azd provision` | Azure வளங்களை உருவாக்குதல் |
| `azd deploy` | பயன்பாட்டு குறியீட்டை வெளியீடு செய்வது |
| `azd up` | ஒதுக்குதல் + வெளியீடு |
| `azd down` | அனைத்து வளங்களையும் நீக்குதல் |
