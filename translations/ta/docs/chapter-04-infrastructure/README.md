**📚 பாடநெறி**: [அறிமுகத்திற்கான AZD](../../README.md) | **⏱️ கால அளவு**: 1-1.5 மணி நேரம் | **⭐ சிரமம்**: மத்திய நிலை

## கண்ணோட்டம்

இந்த அத்தியாயம் Bicep டெம்ப்ளேட் மாதிரிகள், வள அமைத்தல், மற்றும் Azure Developer CLI பயன்படுத்தி வெளியீட்டு நுட்பங்களை (deployment strategies) கையாள்தல் ஆகியவற்றை குறிப்பிடுகிறது.

> ஜூன் 2026 இல் `azd 1.25.6` உடன் சரிபார்க்கப்பட்டது.

## கற்றல் இலக்குகள்

இந்த அத்தியாயத்தை முடித்தவுடன், நீங்கள்:
- Bicep டெம்ப்ளேட் கட்டமைப்பு மற்றும் syntax பற்றி புரிந்துகொள்வீர்கள்
- `azd provision` மூலம் Azure வளங்களை அமைக்கத் தெரிந்து கொள்வீர்கள்
- `azd deploy` மூலம் பயன்பாடுகளை வெளியிடத் தெரிந்து கொள்வீர்கள்
- ப்ளூ-கிரீன் மற்றும் ரோலிங் வெளியீட்டு நுட்பங்களை செயல்படுத்தத் தெரிந்து கொள்வீர்கள்

## 🚀 விரைவு துவக்கம்

```bash
# வார்ப்புருவிலிருந்து ஆரம்பிக்கவும்
azd init --template azure-functions-python-v2-http

# என்ன உருவாக்கப்படப்போகிறது என்பதைக் முன்னோட்டமாகப் பார்க்கவும்
azd provision --preview

# கட்டமைப்புகளை மட்டுமே உருவாக்கவும்
azd provision

# குறியீட்டை மட்டுமே வெளியிடவும்
azd deploy

# அல்லது இரண்டையும் ஒன்றாகவும்
azd up
```

## 📁 AZD திட்ட அமைப்பு

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

## 🔧 அவசியமான கட்டளைகள்

| கட்டளை | விளக்கம் |
|---------|-------------|
| `azd init` | திட்டத்தை ஆரம்பிக்கவும் |
| `azd provision` | Azure வளங்களை உருவாக்கவும் |
| `azd deploy` | பயன்பாட்டுக் குறியீட்டை வெளியிடவும் |
| `azd up` | வள அமைத்தல் + வெளியீடு |
| `azd down` | அனைத்து வளங்களையும் நீக்கு |
