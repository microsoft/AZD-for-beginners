# அத்தியாயம் 3: அமைப்பும் அங்கீகாரமும்

**📚 பாடநெறி**: [AZD துவக்கத்திற்கான](../../README.md) | **⏱️ காலம்**: 45-60 நிமிடங்கள் | **⭐ சிக்கல்தன்மை**: நடுத்தரம்

---

## கண்ணோட்டம்

இந்த அத்தியாயம் Azure Developer CLI ஏவலுக்கு சுற்றுச்சூழல் கட்டமைப்பு, அங்கீகார முறைமைகள் மற்றும் பாதுகாப்பு சிறந்த நடைமுறைகள் குறித்து கवर் செய்கிறது.

> `azd 1.25.6` இன் மீது ஜூன் 2026 இல் சரிபார்க்கப்பட்டது.

## கற்றலின் நோக்கங்கள்

இந்த அத்தியாயத்தை முடித்தவுடன், நீங்கள்:
- AZD கட்டமைப்பு தரவரிசையை நன்கு கையாளுதல்
- பல சூழல்களை (dev, staging, prod) நிர்வகித்தல்
- மேனேஜ்டு அடையாளங்களைப் பயன்படுத்தி பாதுகாப்பான அங்கீகாரத்தை அமல்படுத்துதல்
- சூழல்-சார்ந்த அமைப்புகளை கட்டமைத்தல்

## 🚀 விரைவு துவக்கம்

```bash
# பல சூழல்களை உருவாக்கவும்
azd env new dev
azd env new staging
azd env new prod

# சூழல்களை மாற்றவும்
azd env select prod

# சூழல் மாறிலிகளை அமைக்கவும்
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# கட்டமைப்பைக் காண்க
azd env get-values
```

---

## 🔧 கட்டமைப்பு தரவரிசை

AZD அமைப்புகளை இந்த வரிசையிலே அமைக்கும் (பின்னர் உள்ளவை முன் உள்ளவற்றை மீறும்):

1. **இயல்பு மதிப்புகள்** (டெம்ப்ளேட்டுகளில் உட்பட்டவை)
2. **azure.yaml** (திட்டக் கட்டமைப்பு)
3. **சூழல் மாறிகள்** (`azd env set`)
4. **கமாண்ட்-லைன் கொடிகள்** (`--location eastus`)

---

## 🔐 பாதுகாப்பு சிறந்த நடைமுறைகள்

```bash
# மேலாண்மை அடையாளத்தை பயன்படுத்தவும் (பரிந்துரைக்கப்படுகிறது)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD அங்கீகார நிலையை சரிபார்க்கவும்
azd auth status

# விருப்பம்: நீங்கள் az கட்டளைகள் இயக்க திட்டமிட்டால் Azure CLI சூழலைச் சரிபார்க்கவும்
az account show

# தேவையானால் மீண்டும் அங்கீகரிக்கவும்
azd auth login

# விருப்பம்: az கட்டளைகளுக்காக Azure CLI அங்கீகாரத்தை புதுப்பிக்கவும்
az login
```
