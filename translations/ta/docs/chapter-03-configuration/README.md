# அத்தியாயம் 3: அமைப்பு மற்றும் சான்றளிப்பு

**📚 பாடத்திட்டம்**: [AZD For Beginners](../../README.md) | **⏱️ காலம்**: 45-60 நிமிடங்கள் | **⭐ சிக்கல்தன்மை**: நடுத்தர

---

## கண்ணோட்டம்

இந்த அத்தியாயம் Azure Developer CLI அமைப்புகள், சான்றளிப்பு மாதிரிகள் மற்றும் பாதுகாப்பு சிறந்த நடைமுறைகளைப் பற்றி குறித்துள்ளது.

> `azd 1.27.1` இல் 2026 ஜூலை மாதம் சரிபார்க்கப்பட்டது.

## கற்றல் நோக்கங்கள்

இந்த அத்தியாயத்தை முடித்தவுடன், நீங்கள்:
- AZD அமைப்புக் கட்டமைப்பை வல்லுநர் ஆகுவீர்கள்
- பல சூழல்களை (dev, staging, prod) நிர்வகிப்பதைக் கற்றுக்கொள்வீர்கள்
- நிர்வகிக்கப்பட்ட அடையாளங்களுடன் பாதுகாப்பான சான்றளிப்பை செயல்படுத்துவீர்கள்
- சூழல்-சார்ந்த அமைப்புகளை கட்டமைப்பீர்கள்

## 🚀 விரைவு தொடக்கம்

```bash
# பல சுற்றுச்சூழல்களை உருவாக்கவும்
azd env new dev
azd env new staging
azd env new prod

# சுற்றுச்சூழல்களை மாற்றவும்
azd env select prod

# சுற்றுச்சூழல் மாறிகளை அமைக்கவும்
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# தொடக்கவடிவமைப்பினைக் காண்க
azd env get-values
```

---

## 🔧 அமைப்பு கட்டமைப்பு

AZD அமைப்புகளை இந்த வரிசையில் பயன்படுத்துகிறது (பிறகு முன் அமைப்புகளை மாற்றும்):

1. **பொதுவான மதிப்புகள்** (தானாக மாறிகள் உள்ளடக்கப்பட்டவை)
2. **azure.yaml** (திட்ட அமைப்பு)
3. **சூழல் மாறிகள்** (`azd env set`)
4. **கட்டளை வரி கொடிகள்** (`--location eastus`)

---

## 🔐 பாதுகாப்பு சிறந்த நடைமுறைகள்

```bash
# முகாமையிடப்பட்ட அடையாளத்தை பயன்படுத்தவும் (추천됨)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD அங்கீகார நிலையை சரிபார்க்கவும்
azd auth status

# விருப்பமாக: நீங்கள் az கட்டளைகள் இயக்க திட்டமிட்டால் Azure CLI சூழலை உறுதிப்படுத்தவும்
az account show

# தேவையெனில் மீண்டும் அங்கீகரிக்கவும்
azd auth login

# விருப்பமாக: az கட்டளைகள் için Azure CLI அங்கீகாரத்தை புதுப்பிக்கவும்
az login
```
