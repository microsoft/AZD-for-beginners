# அத்தியாயம் 3: கட்டமைப்பு மற்றும் அங்கீகாரம்

**📚 பாடநெறி**: [AZD - தொடக்கர்களுக்கு](../../README.md) | **⏱️ நேரம்**: 45-60 நிமிடங்கள் | **⭐ கடினத்தன்மை**: இடைநிலை

---

## மேலோட்டம்

இந்த அத்தியாயம் Azure Developer CLI உடன் செய்யப்படும் டெப்பிளாய்மென்டுகளுக்கான சூழல் கட்டமைப்பு, அங்கீகார மாதிரிகள் மற்றும் பாதுகாப்பு சிறந்த நடைமுறைகளை உள்ளடக்கியது.

## கற்றல் நோக்கங்கள்

இந்த அத்தியாயத்தை முடித்த பிறகு, நீங்கள்:
- AZD கட்டமைப்பு வரிசையில் நிபுணத்தன்மை பெறுதல்
- பல்வேறு சூழல்களை (dev, staging, prod) நிர்வகிக்கவும்
- மேனேஜ் செய்யப்பட்ட அடையாளங்களுடன் பாதுகாப்பான அங்கீகாரத்தை செயல்படுத்துதல்
- சூழல்-குறிப்பிட்ட அமைப்புகளை கட்டமைக்கவும்

## 🚀 விரைவில் துவக்கம்

```bash
# பல சூழல்களை உருவாக்கவும்
azd env new dev
azd env new staging
azd env new prod

# சூழல்களை மாற்றவும்
azd env select prod

# சூழல் மாறிகளை அமைக்கவும்
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# கட்டமைப்பைப் பார்க்கவும்
azd env get-values
```

---

## 🔧 கட்டமைப்பு தரவரிசை

AZD கீழ்காணும் வரிசையில் அமைப்புகளை போடும் (பிறகு வரும் அமைப்புகள் முன்னையவற்றை மீறும்):

1. **இயல்பான மதிப்புகள்** (வார்ப்புருக்களில் உள்ளடக்கப்பட்டவை)
2. **azure.yaml** (திட்டக் கட்டமைப்பு)
3. **சூழல் மாறிகள்** (`azd env set`)
4. **கமாண்ட்-லைன் கொடிகள்** (`--location eastus`)

---

## 🔐 பாதுகாப்பு சிறந்த நடைமுறைகள்

```bash
# மேலாண்மை அடையாளத்தைப் பயன்படுத்தவும் (பரிந்துரைக்கப்படுகிறது)
azd env set AZURE_USE_MANAGED_IDENTITY true

# அங்கீகார நிலையை சரிபார்க்கவும்
azd auth whoami
az account show

# தேவையானால் மீண்டும் அங்கீகரிக்கவும்
azd auth login
az login
```
