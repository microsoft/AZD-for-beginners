# அத்தியாயம் 7: பிரச்சனைகள் தீர்வு மற்றும் பிழைத்திருத்தம்

**📚 பாடநெறி**: [AZD ஆரம்பக்கருவிகளுக்கு](../../README.md) | **⏱️ கால அளவு**: 1-1.5 மணி நேரம் | **⭐ சிக்கல் நிலை**: இடைநிலை

---

## சுருக்கம்

இந்த அத்தியாயம் Azure Developer CLI-க்கான பொதுவான பிரச்சனைகளைக் கண்டறிந்து தீர்க்க உதவுகிறது. பராமரிப்பு தோல்விகள் முதல் AI-வுக்கான காரணங்களுக்கான பிரச்சனைகள் வரை.

> `azd 1.27.1` இல் 2026 ஜூலை மாதத்தில் சரிபார்க்கப்பட்டது.

## கற்றல் நோக்கங்கள்

இந்த அத்தியாயத்தை முடித்தல் மூலம், நீங்கள்:
- பொதுவான AZD பராமரிப்பு தோல்விகளை கண்டறிதல்
- அங்கீகாரம் மற்றும் அனுமதி பிரச்சனைகளை பிழைத்திருத்துதல்
- AI சேவை இணைப்பு பிரச்சனைகளை தீர்க்குதல்
- பிழைத்திருத்தத்திற்கு Azure போர்டல் மற்றும் CLI பயன்படுத்துதல்

## 🚨 விரைவான தீர்வுகள்

### அங்கீகார பிரச்சனைகள்
```bash
# AZD வேலைப்பாட்டுகளுக்கு அவசியம்
azd auth login

# நீங்கள் நேரடியாக Azure CLI கட்டளைகளையும் பயன்படுத்தினால் விருப்பமானது
az login

azd auth status
```

### பராமரிப்பு தோல்விகள்
```bash
azd show
azd monitor --logs
az deployment sub list --query "[?properties.provisioningState!='Succeeded']"
```

### வளம் மோதல்
```bash
azd down --force --purge
azd env new different-name
azd up
```

### அளவு அதிகம் கடந்தது
```bash
az vm list-usage --location eastus --output table
azd env set AZURE_LOCATION westus2
azd up
```

## 📋 பிழை குறியீட்டு குறிப்பு

| பிழை | காரணம் | தீர்வு |
|-------|-------|----------|
| `AuthenticationError` | உள்நுழையவில்லை | `azd auth login` |
| `ResourceNotFound` | வளம் காணவில்லை | வள பெயர்களை சரி பார்க்கவும் |
| `QuotaExceeded` | சந்தா வரம்புகள் | அளவு அதிகரிப்பு கோருங்கள் |
| `InvalidTemplate` | Bicep வரையறை பிழை | `az bicep build` |
| `Conflict` | வளம் ஏற்கனவே உள்ளது | புதிய பெயரை பயன்படுத்தவும் அல்லது அழிக்கவும் |
| `Forbidden` | போதுமான அனுமதி இல்லை | RBAC பங்குகளை சரிபார்க்கவும் |

## 🔄 மீட்டமைப்பு மற்றும் மீட்பு

```bash
# மிருதுவான மீட்டெடுக்கும் (வளங்களை வைத்திரு, கோடுகளை மீண்டும் разப்பிளாய் செய்)
azd deploy --force

# கடுமையான மீட்டெடுக்கும் (எல்லாவற்றையும் நீக்கு, புதியதாக தொடங்கு)
azd down --force --purge
azd up
```
