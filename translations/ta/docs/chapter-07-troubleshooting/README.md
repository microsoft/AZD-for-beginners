# அத்தியாயம் 7: பிரச்சினை தீர்வு & பிழைதிருத்தம்

**📚 பாடநெறி**: [AZD தொடக்கர்களுக்கான](../../README.md) | **⏱️ கால அளவு**: 1-1.5 மணி நேரம் | **⭐ சிக்கல்நிலை**: இடைநிலை

---

## கண்ணோட்டம்

இந்த அத்தியாயம் Azure Developer CLI உடன் பணியாற்றும்போது ஏற்படும் பொதுவான பிரச்சினைகளை கண்டறிந்து தீர்க்க உதவுகிறது. வழங்கல் தோல்விகளிலிருந்து AI-சார்ந்த பிரச்சினைகள் வரை.

> மார்ச் 2026 இல் `azd 1.23.12`-ஐ சரிபார்த்தது.

## கற்கும் நோக்கங்கள்

இந்த அத்தியாயத்தை முடித்தவுடன், நீங்கள்:
- AZD வழங்கல் தோல்விகளை கண்டறிய முடியும்
- அங்கீகாரம் மற்றும் அனுமதி முறைகள் தொடர்பான பிழைகளை பிழைதிருத்த முடியும்
- AI சேவை இணைப்பு பிரச்சினைகளை தீர்க்க முடியும்
- பிழைத் தேடலுக்கு Azure போர்டல் மற்றும் CLI-யைப் பயன்படுத்த முடியும்

## 🚨 விரைவு தீர்வுகள்

### அங்கீகாரப் பிரச்சினைகள்
```bash
# AZD வேலைநடவடிக்கைகளுக்குத் தேவையானது
azd auth login

# நீங்கள் Azure CLI கட்டளைகளை நேரடியாகப் பயன்படுத்தினால் இது விருப்பமானதாகும்
az login

azd auth status
```

### வழங்கல் தோல்விகள்
```bash
azd show
azd monitor --logs
az deployment sub list --query "[?properties.provisioningState!='Succeeded']"
```

### வள மோதல்கள்
```bash
azd down --force --purge
azd env new different-name
azd up
```

### கோட்டா மீறல்
```bash
az vm list-usage --location eastus --output table
azd env set AZURE_LOCATION westus2
azd up
```

## 📋 பிழை குறியீடு குறிப்பு

| பிழை | காரணம் | தீர்வு |
|-------|-------|----------|
| `AuthenticationError` | உள்நுழையப்படவில்லை | `azd auth login` |
| `ResourceNotFound` | வளம் காணப்படவில்லை | வள பெயர்களை சரிபார்க்கவும் |
| `QuotaExceeded` | சந்தா வரம்புகள் | கோட்டா உயர்வுக்கு கோருங்கள் |
| `InvalidTemplate` | Bicep இலக்கணம் பிழை | `az bicep build` |
| `Conflict` | வளம் ஏற்கனவே 있음 | புதிய பெயரை பயன்படுத்தவோ அல்லது நீக்கவோ செய்யவும் |
| `Forbidden` | போதுமான அனுமதிகள் இல்லை | RBAC பங்குகளை சரிபார்க்கவும் |

## 🔄 மீட்டமைப்பு மற்றும் மீட்பு

```bash
# மென்மையான மீட்டமைப்பு (வளங்களை வைத்திருத்தல், குறியீட்டை மீண்டும் வெளியிடுதல்)
azd deploy --force

# கடுமையான மீட்டமைப்பு (அனைத்தையும் நீக்குதல், புதியதாய் தொடங்குதல்)
azd down --force --purge
azd up
```
