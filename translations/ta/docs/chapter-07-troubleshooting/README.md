# அத்தியாயம் 7: பிரச்சனை தீர்ப்பு & பிழைதிருத்தம்

**📚 பாடநெறி**: [AZD For Beginners](../../README.md) | **⏱️ நேரம்**: 1-1.5 மணிநேரம் | **⭐ சிக்கலான நிலை**: இடைநிலை

---

## கண்ணோட்டம்

இந்த அத்தியாயம் Azure Developer CLI உடன் பணியாற்றும் போது பொதுவாக எதிர்கொள்ளப்படும் பிரச்சனைகளை கண்டறிந்து தீர்க்க உதவுகிறது. விநியோக் தோல்விகளிலிருந்து AI-சார்ந்த பிரச்சனைகள் வரை.

> `azd 1.25.6` உடன் ஜூன் 2026-இல் சரிபார்க்கப்பட்டது.

## கற்றல் குறிக்கோள்கள்

இந்த அத்தியாயத்தை முடித்தவுடன், நீங்கள்:
- பொதுவான AZD டெப்ளாய்மென்ட் தோல்விகளை கண்டறிதல்
- அங்கீகார மற்றும் அனுமதி பிரச்சனைகளை பிழைதிருத்துதல்
- AI சேவை இணைப்பு பிரச்சனைகளை தீர்த்தல்
- பிரச்சனை தீர்ப்பிற்காக Azure Portal மற்றும் CLI-ஐப் பயன்படுத்துதல்

## 🚨 விரைவு தீர்வுகள்

### அங்கீகார பிரச்சனைகள்
```bash
# AZD பணிநெறிகளுக்குத் தேவையானது
azd auth login

# நீங்கள் நேரடியாக Azure CLI கட்டளைகளையும் பயன்படுத்தினால் விருப்பமானது
az login

azd auth status
```

### வழங்கல் தோல்விகள்
```bash
azd show
azd monitor --logs
az deployment sub list --query "[?properties.provisioningState!='Succeeded']"
```

### வள மோதல்கள்
```bash
azd down --force --purge
azd env new different-name
azd up
```

### குவோட்டா மீறல்
```bash
az vm list-usage --location eastus --output table
azd env set AZURE_LOCATION westus2
azd up
```

## 📋 பிழை குறியீட்டு குறிப்புகள்

| பிழை | காரணம் | தீர்வு |
|-------|-------|----------|
| `AuthenticationError` | உள்நுழையவில்லை | `azd auth login` |
| `ResourceNotFound` | வளம் காணப்படவில்லை | வளப் பெயர்களை சரிபார்க்கவும் |
| `QuotaExceeded` | சப்ஸ்கிரிப்ஷன் வரம்புகள் | குவோட்டாவை உயர்த்த கோரவும் |
| `InvalidTemplate` | Bicep இலக்கண பிழை | `az bicep build` |
| `Conflict` | வளம் ஏற்கனவே உள்ளது | புதிய பெயரை பயன்படுத்தவும் அல்லது நீக்கவும் |
| `Forbidden` | போதுமான அனுமதிகள் இல்லை | RBAC பொறுப்புகளை சரிபார்க்கவும் |

## 🔄 மீட்டமைப்பு மற்றும் மீட்பு

```bash
# மென்மையான மீட்டமைப்பு (வளங்களை வைத்துக் கொண்டு, குறியீட்டை மீண்டும் பதியுதல்)
azd deploy --force

# கடுமையான மீட்டமைப்பு (எல்லாவற்றையும் நீக்கி, புதிதாக தொடங்குதல்)
azd down --force --purge
azd up
```
