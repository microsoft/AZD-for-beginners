# அத்தியாயம் 7: பிழைத் தீர்வு & பிழைதிருத்தம்

**📚 பாடநெறி**: [புதியவர்களுக்கு AZD](../../README.md) | **⏱️ நீளம்**: 1-1.5 மணிநேரம் | **⭐ சிக்கல்தவ்வம்**: இடைநிலை

---

## மேலோட்டம்

இந்த அத்தியாயம் Azure Developer CLI உடன் வேலை செய்யும்போது நிகழக்கூடிய பொதுவான சிக்கல்களை கண்டறிந்து தீர்க்க உதவுகிறது. பிரசார தோல்விகளிலிருந்து AI-சார்ந்த பிரச்சினைகள் வரை.

## கற்றல் குறிக்கோள்கள்

இந்த அத்தியாயத்தை முடித்தவுடன், நீங்கள்:
- AZD பிரசார தோல்விகளை கண்டறிவது
- அங்கீகாரம் மற்றும் அனுமதி சிக்கல்களை டீபக் செய்வது
- AI சேவை இணைப்பு பிரச்சினைகளை தீர்ப்பது
- பிரச்சினை தீர்க்க Azure போர்டல் மற்றும் CLI பயன்படுத்துவது

## 🚨 விரைவு சரிசெய்தல்கள்

### அங்கீகாரம் தொடர்பான பிரச்சினைகள்
```bash
azd auth login
az login
azd auth whoami
```

### வழங்குதல் தோல்விகள்
```bash
azd show
azd monitor --logs
az deployment sub list --query "[?properties.provisioningState!='Succeeded']"
```

### வள மோதல்கள்
```bash
azd down --force --purge
azd env new different-name
azd up
```

### கோட்டா மீறப்பட்டது
```bash
az vm list-usage --location eastus --output table
azd env set AZURE_LOCATION westus2
azd up
```

## 📋 பிழை குறியீடு குறிப்புகள்

| பிழை | காரணம் | தீர்வு |
|-------|-------|----------|
| `AuthenticationError` | உள் நுழைவில் இல்லை | `azd auth login` |
| `ResourceNotFound` | வளம் காணப்படவில்லை | வள பெயர்களை சரிபார்க்கவும் |
| `QuotaExceeded` | சந்தா வரம்புகள் | கோட்டாவை அதிகரிக்க கோரிக்கை விடுங்கள் |
| `InvalidTemplate` | Bicep இலக்கணம் பிழை | `az bicep build` |
| `Conflict` | வளம் ஏற்கனவே உள்ளது | புதிய பெயரை பயன்படுத்தவும் அல்லது அழிக்கவும் |
| `Forbidden` | போதுமான அனுமதிகள் இல்லை | RBAC பங்குகளை சரிபார்க்கவும் |

## 🔄 மறுசீரமைப்பு மற்றும் மீட்பு

```bash
# மென்மையான மீட்டமைப்பு (வளங்களை வைத்திருத்தல், குறியீட்டை மீண்டும் வெளியிடுதல்)
azd deploy --force

# முழுமையான மீட்டமைப்பு (எல்லாவற்றையும் நீக்குதல், புதியதாக தொடங்குதல்)
azd down --force --purge
azd up
```
