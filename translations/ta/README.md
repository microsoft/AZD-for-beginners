## 🆕 azd இல் இன்று என்ன புதியது

Azure Developer CLI தற்போது பாரம்பரிய வலை செயலிகள் மற்றும் API-களைக் கடந்து வளர்ந்துள்ளது. இன்று, azd என்பது Azure-க்கு எந்தவொரு பயன்பாடையும் (AI-சக்தி பெற்ற பயன்பாடுகள் மற்றும் அறிவுசார் முகவர்கள் உட்பட) орналுத்துவதற்கான ஒரே கருவியாக உள்ளது.

இது உங்கள் பொறுப்புக்கு என்ன அர்த்தம் என்பதைப் பார்க்கலாம்:

- **AI வினைமுறைகளும் இப்போது முதன்மை azd பணிகளாக இருக்கின்றன.** நீங்கள் அறிந்த `azd init` → `azd up` வேலைபகுப்பை பயன்படுத்தி AI முகவர்கள் திட்டங்களை தொடங்கவும், வெளியிடவும் மற்றும் நிர்வகிக்க முடியும்.
- **Microsoft Foundry ஒருங்கிணைவு** மாதிரி வெளியீடு, முகவர் ஹோஸ்டிங் மற்றும் AI சேவை அமைப்புகளை azd டெம்ப்ளேட் சுற்றுவட்டாரத்தில் நேரடியாக கொண்டு வருகிறது.
- **மூல வேலைபழக்கம் மாற்றமடைதவில்லை.** நீங்கள் todo செயலியை, மைக்ரோசெர்வீஸ் அல்லது பல முகவர் AI தீர்வை வெளியிடுகின்றீர்கள் என்பதைப் பொருட்படுத்தாமல், கட்டளைகள் ஒரே மாதிரியே இருக்கின்றன.

முன்பு azd பயன்படுத்தியிருந்தால், AI ஆதரவு ஒரு இயல்பான விரிவாகும் — தனி கருவியாகவோ அல்லது முன்னேறிய பாதையாகவோ அல்ல. புதியவராக ஆரம்பித்தால், எல்லாவற்றிற்கும் பொருந்தும் ஒரே வேலைபழக்கத்தை நீங்கள் கற்பீர்கள்.

## 🚀 Azure Developer CLI (azd) என்பது என்ன?

**Azure Developer CLI (azd)** என்பது Azure-க்கு பயன்பாடுகளை வெளியிட எளிதாக்கும் டெவலப்பர்-பிரியமான கட்டளைப் பிழை கருவி ஆகும். பத்துகளாக Azure வளங்களை கையால் உருவாக்கி இணைப்பதை விட, ஒரு கட்டளையால் முழு பயன்பாடுகளை வெளியிட முடியும்.

### `azd up` இன் மாயாஜாலம்

```bash
# இந்த ஒரே கட்டளை அனைத்தையும் செய்கிறது:
# ✅ அனைத்து அஸ்யூர் வளங்களையும் உருவாக்குகிறது
# ✅ நெட்வொர்க் மற்றும் பாதுகாப்பை அமைக்கிறது
# ✅ உங்கள் பயன்பாட்டு குறியீட்டை உருவாக்குகிறது
# ✅ அஸ்யூருக்கு பாய்ச்சுகிறது
# ✅ உங்களுக்கு வேலை செய்யும் URL ஐ வழங்குகிறது
azd up
```

**அதுவே போதுமானது!** Azure போர்டலைச் சொடுக்க hoef, ARM டெம்ப்ளேட்களை முதலில் கற்றுக்கொள்ள தேவையில்லை, கையேடு அமைப்பு இல்லை - வெறுமனே Azure-இல் செயல்படும் பயன்பாடுகள்.

## ❓ Azure Developer CLI மற்றும் Azure CLI: வேறுபாடு என்ன?

இது புதியவர்கள் அடிக்கடி கேட்கும் பொதுவான கேள்வி. இங்கே எளிய பதில் உள்ளது:

| அம்சம் | **Azure CLI (`az`)** | **Azure Developer CLI (`azd`)** |
|---------|---------------------|--------------------------------|
| **நோக்கம்** | தனிப்பட்ட Azure வளங்களை நிர்வகிக்க | முழுமையான பயன்பாடுகளை வெளியிடவும் |
| **மனப்பான்மை** | இன்ஃப்ராஸ்ட்ரக்சர்-மையமானது | பயன்பாடு-மையமானது |
| **உதாரணம்** | `az webapp create --name myapp...` | `azd up` |
| **கற்றல் வளைவு** | Azure சேவைகளை தெரிந்திருக்க வேண்டும் | உங்கள் பயன்பாட்டை மட்டும் தெரிந்திருக்க வேண்டும் |
| **சிறந்தது** | DevOps, இன்ஃப்ராஸ்ட்ரக்சர் | டெவலப்பர்கள், முன்மாதிரி உருவாக்கம் |

### எளிய ஒப்பீடு

- **Azure CLI** என்பது வீடு கட்ட கருவிகளின் முழு செட்டைப் போல — ஹாமர், செமி, நுகரில். நீங்கள் எதையும் கட்டலாம், ஆனால் கட்டடக் கலைகளைத் தெரிந்து கொள்ள வேண்டும்.
- **Azure Developer CLI** என்பது ஒரு கட்டுமான ஒப்பந்ததாரரை வேலைக்கு அமர்த்துவது போன்றது — நீங்கள் என்ன வேண்டும் என்று விவரிக்கிறீர்கள், அவர்கள் கட்டுப்படுத்துவார்.

### எப்போது எதை பயன்படுத்த வேண்டும்

| சூழ்நிலை | இதை பயன்படுத்தவும் |
|----------|----------|
| "I want to deploy my web app quickly" | `azd up` |
| "I need to create just a storage account" | `az storage account create` |
| "I'm building a full AI application" | `azd init --template azure-search-openai-demo` |
| "I need to debug a specific Azure resource" | `az resource show` |
| "I want production-ready deployment in minutes" | `azd up --environment production` |

### அவை சேர்ந்து வேலை செய்கின்றன!

AZD ன் உள்ளகத்தில் Azure CLI பயன்படுத்தப்படுகிறது. இரண்டையும் பயன்படுத்தலாம்:
```bash
# உங்கள் செயலியை AZD உடன் орналுத்தவும்
azd up

# பின்னர் Azure CLI உடன் குறிப்பிட்ட வளங்களை நுணுக்கமாக சரிசெய்க
az webapp config set --name myapp --always-on true
```

## 🎯 3 படிகளில் தொடங்குவது

### படி 1: AZD ஐ நிறுவவும் (2 நிமிடம்)

**Windows:**
```powershell
winget install microsoft.azd
```

**macOS:**
```bash
brew tap azure/azd && brew install azd
```

**Linux:**
```bash
curl -fsSL https://aka.ms/install-azd.sh | bash
```

### படி 2: Azure இல் உள்நுழைக

```bash
azd auth login
```

### படி 3: உங்கள் முதல் செயலியை வெளியிடவும்

```bash
# வடிவமைப்பிலிருந்து துவக்குங்கள்
azd init --template todo-nodejs-mongo

# அஜூருக்கு வெளியிடுங்கள் (எல்லாவற்றையும் உருவாக்குகிறது!)
azd up
```

**🎉 அது இதுவே!** உங்கள் செயலி இப்போது Azure இல் வடிவமைக்கப்பட்டு இயங்குகிறது.

### சுத்தம் செய்யவும் (மறக்க வேண்டாம்!)

```bash
# சோதனை முடிந்தவுடன் அனைத்து வளங்களையும் நீக்கவும்
azd down --force --purge
```

## பாடநெறி மேலோட்டம்

Azure Developer CLI (azd) ஐ கட்டமைக்கப்பட்ட அத்தியாயங்களின் மூலம் ஆழமாக கற்றுக்கொள்ளுங்கள். **Microsoft Foundry ஒருங்கிணைப்புடன் AI செயலி வெளியீடுகள் மீது சிறப்பு கவனம்.**

### இந்த பாடநெறி நவீன டெவலப்பர்களுக்கு ஏன் அவசியம்

Microsoft Foundry Discord சமுதாயத்தின் கருத்துக்களின் அடிப்படையில், **45% டெவலப்பர்கள் AZD ஐ AI பணிகளுக்கு பயன்படுத்த விரும்புகிறார்கள்** ஆனால் பலவிதமான சவால்களை எதிர்கொள்கிறார்கள்:
- சிக்கலான பல-சேவை AI கட்டமைப்புகள்
- உற்பத்தி AI வெளியீடுகளுக்கான சிறந்த நடைமுறைகள்  
- Azure AI சேவை ஒருங்கிணைப்பு மற்றும் கட்டமைப்பு
- AI பணிகளுக்கான செலவு tốiமயீக்கம்
- AI-சார்ந்த வெளியீடு குறித்த பிரச்சினைகளின் டிரபிள்ஷூட்டிங்

### கற்றல் குறிக்கோள்கள்

இந்த கட்டமைக்கப்பட்ட பாடநெறியை முடித்தவுடன், நீங்கள்:
- **AZD அடிப்படைத்தில் நிபுணத்துவம் பெறுவீர்கள்**: கோர் கருத்துகள், நிறுவல் மற்றும் கட்டமைப்பு
- **AI செயலிகளை வெளியிடுவீர்கள்**: AZD ஐ Microsoft Foundry சேவைகளுடன் பயன்படுத்துவது
- **Infrastructure as Code செயல்முறை நடைமுறை**: Bicep டெம்ப்ளேட்டுகளுடன் Azure வளங்களை நிர்வகிக்க
- **வெளியீடுகளை டிரபிள்ஷூட் செய்யவும்**: பொதுவான பிரச்சினைகளை தீர்க்கவும் மற்றும் பிழைகளை டீபக் செய்யவும்
- **உற்பத்திக்காக tốiமயீக்கவும்**: பாதுகாப்பு, ஸ்கேலிங், மானிட்டரிங் மற்றும் செலவு மேலாண்மை
- **பல முகவர் தீர்வுகளை கட்டமைக்கவும்**: சிக்கலான AI கட்டமைப்புகளை வெளியிடவும்

**தொலைபுரியும் மேம்பாட்டு சுற்றுப்புறம் (கணிக்கப்பட்டது $80-150/மாதம்):**
- Microsoft Foundry மாதிரிகள் (பயன்பாடு-படி கட்டணம்): $0-50/மாதம் (டைக்கன் பயன்பாட்டின் அடிப்படையில்)
- AI Search (அடிப்படை தட்டு): $75/மாதம்
- Container Apps (புகுநிலை): $0-20/மாதம்
- சேமிப்பு (ஸ்டாண்டர்ட்): $1-5/மாதம்

**உற்பத்தி சுற்றுப்புறம் (கணிக்கப்பட்டது $300-3,500+/மாதம்):**
- Microsoft Foundry மாதிரிகள் (இடைவெளி நிலையான செயல்திறனைக் கொண்ட PTU): $3,000+/மாதம் அல்லது உயர் பரிமாற்றத்துடன் பயன்பாடு-படி கட்டணம்
- AI Search (நிலையான தட்டு): $250/மாதம்
- Container Apps (ஒதுக்கப்பட்ட): $50-100/மாதம்
- Application Insights: $5-50/மாதம்
- சேமிப்பு (பிரீமியம்): $10-50/மாதம்

**💡 செலவு குறைத்தல் குறிப்பு:**
- கற்றலுக்காக **Free Tier** Microsoft Foundry மாதிரிகளை பயன்படுத்தவும் (Azure OpenAI 50,000 tokens/மாதம் உட்பட)
- செயல்படாமல் இருக்கும்போது வளங்களை விட `azd down` இயக்கவும்
- உற்பத்திக்கே மட்டும் PTU ஐ மேம்படுத்தவும்; முதலில் consumption-அடிப்படையிலான பில்லிங்குடன் தொடங்குங்கள்
- `azd provision --preview` மூலம் வெளியீட்டிற்கு முன் செலவுகளை மதிப்பிடவும்
- தானாக அளவீடு செய்யலை இயக்கு: நிஜப் பயன்பாட்டிற்கே மட்டுமே செலுத்துங்கள்

```bash
# Azure போர்டலில் உண்மையான செலவுகளை கண்காணிக்கவும்
```

- **முதல்-முடிவு AI வேலைநெறிகள்** - மாடல் மேம்பாட்டிலிருந்து தயாரிப்பு வெளியீவரை சரியான கண்காணிப்புடன்

### உதாரண வகைகளின் மூலம் கற்றல்

> **📌 உள்ளூர் vs. வெளிப்புற உதாரணங்கள்:**  
> **உள்ளூர் உதாரணங்கள்** (இந்த repo-வில்) = உடனடியாகப் பயன்பாட்டிற்கு தயாராக உள்ளன  
> **வெளிப்புற உதாரணங்கள்** (Azure உதாரணங்கள்) = இணைப்பில் கொடுக்கப்பட்ட ரெப்போக்களில் இருந்து கிளோன் செய்யவும்

#### உள்ளூர் உதாரணங்கள் (உடனடி பயன்பாட்டிற்கு தயாராக)
