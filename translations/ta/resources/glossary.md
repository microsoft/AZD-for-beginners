- **📖 அடிப்படைகளை கற்றுக்கொள்ளுங்கள்**: [Chapter 1: AZD Basics](../docs/getting-started/azd-basics.md)
- **🤖 AI தொடக்கத் தொடர்புடைய 용ர்கள்**: [Chapter 2: AI-First Development](../docs/microsoft-foundry/microsoft-foundry-integration.md)

## வரவேற்பு

இந்த விரிவான அகராதி Azure Developer CLI மற்றும் Azure மேக மேம்பாட்டில் பயன்படுத்தப்படும் 용ர்கள், கருத்துக்கள் மற்றும் குறுக்கெழுத்து சொற்களுக்கு விளக்கக் குறிப்புகளை வழங்குகிறது. தொழில்நுட்ப ஆவணங்களை புரிந்துகொள்ள, பிரச்னைகள் திருத்த மற்றும் azd திட்டங்கள் மற்றும் Azure சேவைகள் பற்றி தெளிவாக தொடர்பு கொள்ள இது அவசியமான குறிப்பு ஆகும்.

## கற்றலின் இலக்குகள்

இந்த அகராதியை பயன்படுத்துவதன் மூலம், நீங்கள்:
- Azure Developer CLI தொடர்புடைய அவசியமான 용ற்கள் மற்றும் கருத்துக்களை புரிந்துகொள்வீர்கள்
- Azure மேக மேம்பாட்டு சொற்றொடர்களையும் தொழில்நுட்ப 용ர்களையும் கையாள்வதில் தேர்ச்சி பெறுவீர்கள்
- Infrastructure as Code மற்றும் உள்ளீட்டு/பயன்படுத்தும் 용ர்களை சிறப்பாக மேற்கோள் இட முடியும்
- Azure சேவைப் பெயர்கள், குறுக்கெழுத்துகள் மற்றும் அவைகள் வழங்கும் பண்புகளை புரிந்துகொள்வீர்கள்
- பிழைத்திருத்தம் மற்றும் டிபக்கிங் 용ர்களுக்கு வரையறைகளை அணுகுவீர்கள்
- மேம்பட்ட Azure கட்டமைப்பு மற்றும் மேம்பாட்டு கருத்துக்களை கற்றுக்கொள்வீர்கள்

## கற்றலின் விளைவுகள்

இவையை மறுசீரமைப்பாக பயன்படுத்துவதன் மூலம், நீங்கள் மூலம்:
- சரியான Azure Developer CLI 용ற்கள் பயன்படுத்தி திறம்பட தொடர்பு கொள்ள முடியும்
- தொழில்நுட்ப ஆவணங்கள் மற்றும் பிழைச் செய்திகளை தெளிவாக புரிந்துகொள்ள முடியும்
- Azure சேவைகள் மற்றும் கருத்துக்களை தாழ்மையாக அணுக முடியும்
- சரியான தொழில்நுட்ப சொற்பொருளைப் பயன்படுத்தி பிரச்னைகளை தீர்க்க முடியும்
- துல்லியமான தொழில்நுட்ப மொழியுடன் குழு விவாதங்களில் பங்களிக்க முடியும்
- உங்கள் Azure மேக மேம்பாட்டு அறிவை திட்டமிட்ட முறையில் விரிவாக்கிக்கொள்ள முடியும்

## A

**ARM Template**  
Azure Resource Manager template. JSON அடிப்படையிலான Infrastructure as Code வடிவம், Azure வளங்களை அறிவுறுத்தல்பூர்வமாக வரையறுத்து 배포 செய்ய பயன்படுத்தப்படுகிறது.

**App Service**  
வலை செயலிகள், REST APIகள் மற்றும் மொபைல் பின்தளங்களை உள்கொண்டு, அடிக்கடி உட்பட கட்டமைப்பை நிர்வகிக்க வேண்டாமைக்கேற்றவாறு ஹோஸ்ட் செய்ய Azure வழங்கும் platform-as-a-service (PaaS) ப்ரொவிஷன்.

**Application Insights**  
 செயலியின் செயல்திறன், கிடைக்கும் தன்மை மற்றும் பயன்பாடு குறித்து ஆழமான洞察ங்களை வழங்கும் Azureன் application performance monitoring (APM) சேவை.

**Azure CLI**  
Azure வளங்களை நிர்வகிக்க பயன்படும் கட்டளை வரிசை இடைமுகம். azd இன் அங்கீகாரம் மற்றும் சில செயல்களில் பயன்படுத்தப்படுகிறது.

**Azure Developer CLI (azd)**  
டெவலபர் நெறிமுறைக்கு உகந்த கட்டளை வரிசை கருவி, டெம்ப்ளேடுகள் மற்றும் Infrastructure as Code பயன்படுத்தி Azureக்கு பயன்பாடுகளை கட்டமைக்கவும் 배포் செய்யவும் பணியை விரைவுபடுத்துகிறது.

**azure.yaml**  
ஒரு azd திட்டத்திற்கான முக்கிய கட்டமைப்பு கோப்பு; சேவைகள், கட்டமைப்பு மற்றும் 배포 hooks ஆகியவற்றை வரையறுக்கிறது.

**Azure Resource Manager (ARM)**  
வளங்களை உருவாக்க, புதுப்பிக்க மற்றும் நீக்குவதற்கான மேலாண்மை அடுக்கு வழங்கும் Azureன் 배포 மற்றும் மேலாண்மை சேவை.

## B

**Bicep**  
Azure வளங்களை 배포 செய்வதற்காக Microsoft உருவாக்கிய domain-specific மொழி (DSL). ARM டெம்ப்ளேட்டுகளுக்கு compile ஆகும்போது சுலபமானyntax ஐ வழங்குகிறது.

**Build**  
மூலக் குறியீட்டை தொகுப்பது, சார்புகளை நிறுவுவது மற்றும் பயன்பாடுகளை 배포க்கு தயார்ப்படுத்தும் செயல்முறை.

**Blue-Green Deployment**  
இரு ஒரேபோன்ற உற்பத்தித்தள சூழல்கள் (blue மற்றும் green) பயன்படுத்தி பதிவு நேரம் மற்றும் ஆபத்துகளை குறைப்பதற்கான 배포_STRATEGY.

## C

**Container Apps**  
சிக்கலான கட்டமைப்பை நிர்வகிக்காமலும் கன்டெய்னரை இயக்கி பயன்பாடுகளை ஓடவிடும் Azureன் serverless container சேவை.

**CI/CD**  
Continuous Integration/Continuous Deployment. குறியீடு மாற்றங்களை ஒருங்கிணைக்கவும் பயன்பாடுகளை 자동மாக 배포் செய்யவும் பயன்படும் செயல்முறைகள்.

**Cosmos DB**  
உலகளாவியம்படஒப்ப்முறை, பல-வடிவமைப்பு தரவுத்தள சேவை; throughput, latency, கிடைக்கும் தன்மை மற்றும் ஒத்திசைவை சார்வனைப் பூர்த்தி செய்கிறது.

**Configuration**  
பயன்பாட்டு நடத்தை மற்றும் 배포 விருப்பங்களை கட்டுப்படுத்தும் அமைப்புகள் மற்றும் பண்புகள்.

## D

**Deployment**  
பயன்பாடுகள் மற்றும் அவற்றின் சார்புகளை இலக்கு கட்டமைப்பில் நிறுவி கட்டமைக்கும்முறை.

**Docker**  
கன்டெய்னரைப் பயன்படுத்தி பயன்பாடுகளை உருவாக்க, கப்பல் செய்ய மற்றும் இயக்க பயன்படும் தளம்.

**Dockerfile**  
Docker கன்டெய்னர் இமேஜ் கட்டமைக்க வழிமுறைகள் கொண்ட உரை கோப்பு.

## E

**Environment**  
உங்கள் பயன்பாட்டின் ஒரு குறிப்பிட்ட இன்ஸ்டன்சைக் குறிக்கும் 배포 இலக்கு (உதா., development, staging, production).

**Environment Variables**  
இயல்புநிலை நேரத்தில் பயன்பாடு அணுகக்கூடிய தாள்பட்ட key-value ஜோடிகளாக சேமிக்கப்பட்ட கட்டமைப்பு மதிப்புகள்.

**Endpoint**  
ஒரு பயன்பாடு அல்லது சேவையை அணுகக்கூடிய URL அல்லது நெட்வொர்க் முகவரி.

## F

**Function App**  
நிகழ்வு சார்ந்த குறியீட்டை கட்டமைப்பு நிர்வகிக்காமலும் ஓட்டுவதற்கு Azure வழங்கும் serverless கணினி சேவை.

## G

**GitHub Actions**  
GitHub சேமிப்பகங்களுடன் ஒருங்கிணைக்கப்பட்ட CI/CD பிளாட்ஃபார்ம், வேலை جریانங்களை 자동화 செய்ய பயன்படுகிறது.

**Git**  
மூலக் குறியீட்டின் மாற்றங்களை கண்காணிக்க பயன்படும் பரவியுள்ள பதிப்பிதழ் கட்டுப்பாட்டு முறைமை.

## H

**Hooks**  
배포 வாழ்க்கைசுற்றில் குறிப்பிட்ட இடங்களில் (preprovision, postprovision, predeploy, postdeploy) ஓடும் தனிப்பயன் ஸ்கிரிப்டுகள் அல்லது கட்டளைகள்.

**Host**  
பயன்பாடு 배포 செய்யப்படவுள்ள Azure சேவை வகை (உதா., appservice, containerapp, function).

## I

**Infrastructure as Code (IaC)**  
கைமுறை செயல்முறைகளுக்குப் பதிலாக கட்டிடக்கூறு மூலம் கட்டமைப்பை வரையறுக்கும் மற்றும் நிர்வகிக்கும் நடைமுறை.

**Init**  
ஒரு புதிய azd திட்டத்தை ஆரம்பிக்க எனப்படும் செயல்முறை, வழக்கமாக ஒரு டெம்ப்ளேட்டில் இருந்து.

## J

**JSON**  
JavaScript Object Notation. கட்டமைப்பு கோப்புகள் மற்றும் API பதில்களுக்கு பொதுவாக பயன்படுத்தப்படும் தரவுச் பரிமாற்ற வடிவம்.

**JWT**  
JSON Web Token. பக்கங்களுக்குப் பதிலாக தகவல்களை ஒரு JSON பொருளாக பாதுகாப்பாக பரிமாறுவதற்கான தரநிலையம்.

## K

**Key Vault**  
மர்மங்கள், திறவுகோல்கள் மற்றும் சான்றிதழ்களை பாதுகாப்பாக தக்கவைத்துக்கொள்வதற்கான Azure சேவை.

**Kusto Query Language (KQL)**  
Azure Monitor, Application Insights மற்றும் மற்ற Azure சேவைகளில் தரவுகளை பகுப்பாய்வு செய்யப் பயன்படும் விசாரனை மொழி.

## L

**Load Balancer**  
பல சர்வர்கள் அல்லது இன்ஸ்டன்ஸ்களுக்கு இடையே வரும் நெட்வொர்க் போக்குவரத்தை dağıத்து செய்யும் சேவை.

**Log Analytics**  
மேகத்திலும் உள்ள-தளத்திலும் இருந்து telemetry தரவுகளை சேகரித்து, பகுப்பாய்வு செய்து, அதற்கேற்ற நடவடிக்கைகள் எடுக்க உதவும் Azure சேவை.

## M

**Managed Identity**  
மற்ற Azure சேவைகளுக்கு அங்கீகாரம் பெற Azure சேவைகள் தானாகவே நிர்வகிக்கப்படும் அடையாளத்தை வழங்கும் Azure செயல்முறை.

**Microservices**  
பயன்பாடுகளை சிறிய, தனித்துவமான சேவைகளாகக் கட்டமைக்கும் கட்டமைப்பு அணுகுமுறை.

**Monitor**  
பயன்பாடுகள் மற்றும் கட்டமைப்பின் முழு ஸ்டாக் கவனிப்புக்கான Azureன் ஒருங்கிணைந்த கண்காணிப்பு தீர்வு.

## N

**Node.js**  
Chrome இன் V8 JavaScript என்ஜினில் கட்டப்பட்ட server-side பயன்பாடுகளை உருவாக்கும் JavaScript ஓட்டவியல்.

**npm**  
Node.js க்கான தொகுப்பு மேலாண்மை கருவி; சார்புகள் மற்றும் தொகுப்புகளை நிர்வகிக்க பயன்படுகிறது.

## O

**Output**  
கட்டமைப்பு 배포 இலிருந்து திரும்பப்படும் மதிப்புகள்; அவை பயன்பாடுகள் அல்லது பிற வளங்கள் பயன்படுத்தக்கூடியவை.

## P

**Package**  
பயன்பாட்டு குறியீட்டு மற்றும் சார்புகளை 배포க்காக தயார்ப்படுத்தும் செயல்முறை.

**Parameters**  
배포களை தனிப்பயனாக்க இஸ்தாஉம் டெம்ப்ளேட்டுகளுக்குப் 전달ிக்கப்படும் உள்ளீடு மதிப்புகள்.

**PostgreSQL**  
திறந்தソース சார்ந்த தொடர்பு தரவுத்தள அமைப்பு; Azureஇல் நிர்வகிக்கப்படும் சேவையாக ஆதரிக்கப்படுகிறது.

**Provisioning**  
இன்ஃப்ராஸ்ட்ரக்சர் டெம்ப்ளேட்டுகளில் வரையறுக்கப்பட்ட Azure வளங்களை உருவாக்கி கட்டமைக்கும் செயல்முறை.

## Q

**Quota**  
ஒரு Azure சந்தாவிற்கு அல்லது பகுதியிற்கு உருவாக்கக்கூடிய வளங்களின் அளவைப் பற்றிய வரம்புகள்.

## R

**Resource Group**  
ஒரே வாழ்க்கைசுற்று, அனுமதிகள் மற்றும் கொள்கைகளை பகிரும் Azure வளங்களுக்கான தர்க்கதரிசி கான்டெய்னர்.

**Resource Token**  
배포களில் வளப் பெயர்கள் தனித்துவமாக இருக்க azd உருவாக்கும் தனித்துவமான நெற்றிச் சரம்.

**REST API**  
HTTP முறைமைகளைப் பயன்படுத்தி வலைவழி பயன்பாடுகளை வடிவமைக்கும் கட்டமைப்பு நடைமுறை.

**Rollback**  
பயன்பாடு அல்லது கட்டமைப்பின் முன் பதிப்புக்கு இடம்பெயர்த்தல் செயல்.

## S

**Service**  
azure.yaml இல் வரையறுக்கப்படும் உங்கள் பயன்பாட்டின் ஒரு கூறு (உதா., web frontend, API backend, database).

**SKU**  
Stock Keeping Unit. Azure வளங்களுக்கு வெவ்வேறு சேவை நிலைகள் அல்லது செயல்திறன் மட்டங்களை பிரதிநிதித்துவம் செய்யும் குறியீடு.

**SQL Database**  
Microsoft SQL Server அடிப்படையிலான Azureன் நிர்வகிக்கப்பட்ட தொடர்பு தரவுத்தள சேவை.

**Static Web Apps**  
மூலக் குறியீட்டு சேமிப்பகங்களிலிருந்து முழு ஸ்டாக் வலை பயன்பாடுகளை உருவாக்கி 배포 செய்வதற்கான Azure சேவை.

**Storage Account**  
blobs, files, queues மற்றும் tables உட்பட தரவுப் பொருட்களுக்கு மேகவழிப் பராமரிப்பை வழங்கும் Azure சேவை.

**Subscription**  
வளக்குழுக்களும் வளங்களும் அடங்கியமை, பில்லிங் மற்றும் அணுகல் மேலாண்மையுடன் கூடிய Azure கணக்கு கட்டமைப்பு.

## T

**Template**  
சாதாரண senarios க்கு பயன்பாட்டு குறியீடு, கட்டமைப்பு வரையறைகள் மற்றும் கட்டமைப்புகளை கொண்ட முன்கூட்டியே உருவாக்கப்பட்டத் திட்ட அமைப்பு.

**Terraform**  
பல மேக வழங்குநர்களை ஆதரிக்கும் திறந்தソース Infrastructure as Code கருவி; Azure ஐ உட்பட ஆதரிக்கிறது.

**Traffic Manager**  
உலகளாவிய Azure பிராந்தியங்களுக்கு போக்குவரத்தை விநியோகிக்க DNS அடிப்படையிலான traffic load balancer.

## U

**URI**  
Uniform Resource Identifier. ஒரு குறிப்பிட்ட வளத்தை அடையாளப்படுத்தும் எழுத்துச்சரம்.

**URL**  
Uniform Resource Locator. ஒரு வளம் எங்கு இருக்கிறது மற்றும் அதை எப்படி பெறுவது என்பதை குறிப்பிடும் URI வகை.

## V

**Virtual Network (VNet)**  
Azureஇல் தனியுரிமை மற்றும் பிரிக்கப்பட்ட நெட்வொர்க்கு அடிப்படை கட்டுமானம், தனிமைப்படுத்தல் மற்றும் பிரிவுகளை வழங்குகிறது.

**VS Code**  
Visual Studio Code. Azure மற்றும் azd உடன் சிறந்த ஒருங்கிணைப்பையும் கொண்ட பிரபலமான குறியீட்டு தொகுப்பி.

## W

**Webhook**  
குறிப்பிட்ட நிகழ்வுகளால் trigger செய்யப்படும் HTTP callback; பொதுவாக CI/CD pipelineகளில் பயன்படுத்தப்படுகிறது.

**What-if**  
ஒரு 배포 செயலில் நேரடியாக அதைச் செயலாக்காமலேயே எந்த மாற்றங்கள் நடக்கும் என்பதை காட்டும் Azure அம்சம்.

## Y

**YAML**  
YAML Ain't Markup Language. azure.yaml போன்ற கட்டமைப்பு கோப்புகளுக்கு பயன்படும் 사람이 படிக்கக்கூடிய தரவு serialization தரநிலை.

## Z

**Zone**  
ஒரு Azure பிராந்தியத்தின் உளைச்சப்படாத, பிரிக்கப்பட்ட இடங்கள்; redundancy மற்றும் உயர்ந்த கிடைக்கும் தன்மையை வழங்குகின்றன.

## சூழலார்த்த பயன்பாட்டு 용ர்கள்

### மேம்பாட்டு 용ர்கள்
- **Hot Reload**: மேம்பாட்டின் போது חוזமாக சேராமல் 자동மாக பயன்பாடுகளை புதுப்பித்தல்
- **Build Pipeline**: குறியீட்டை கட்டமைத்து சோதிக்கும் 자동 செயல்முறை
- **Deployment Slot**: App Service உள்ளே ஒரு ஸ்டேஜிங் சூழ்கூடம்
- **Environment Parity**: development, staging மற்றும் production சூழல்கள் இணையானவையாக வைத்திருத்தல்

### பாதுகாப்பு 용ர்கள்
- **Managed Identity**: தானாக விசைப்பத்திர மேலாண்மையை வழங்கும் Azure அம்சம்
- **Key Vault**: மர்மங்கள், திறவுகோல்கள் மற்றும் சான்றிதழ்களுக்கு பாதுகாப்பான சேமிப்பு
- **RBAC**: Azure வளங்களுக்கான வேடங்களின் அடிப்படையில் அணுகல் கட்டுப்பாடு
- **Network Security Group**: நெட்வொர்க் போக்குவரத்தை கட்டுப்படுத்தும் மெய்நிகர் ஃபயர்வால்

### கண்காணிப்பு 용ர்கள்
- **Telemetry**: அளவுகள் மற்றும் தரவின் 자동 சேகரிப்பு
- **Application Performance Monitoring (APM)**: மென்பொருள் செயல்திறனைக் கண்காணித்தல்
- **Log Analytics**: பதிவு தரவை சேகரித்து பகுப்பாய்வு செய்வதற்கான சேவை
- **Alert Rules**: அளவுகள் அல்லது நிபந்தனைகளின் அடிப்படையில் 자동 அறிவிப்புகள்

### 배포 용ர்கள்
- **Blue-Green Deployment**: நேரமில்லாத 배포 STRATEGY
- **Canary Deployment**: பயனர்களின் ஒரு துணுக்கிற்கு徐徐மாக வெளியீடு
- **Rolling Update**: பயன்பாட்டு இன்ஸ்டன்ஸுகளை தொடர்ச்சியாக மாற்றுதல்
- **Rollback**: முந்தைய பயன்பாடு பதிப்பிற்கு திரும்புதல்

---

**பயன்பாட்டு குறிப்பாக**: இந்த அகராதியில் குறிப்பிட்ட 용ர்களைக் விரைவு தேட `Ctrl+F` ஐ பயன்படுத்தவும். 용ர்கள் பொருத்தமான இடங்களில் குறுக்கே குறிப்பிட்டுள்ளன.
