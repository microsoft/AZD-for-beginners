## Introduction

இந்த விரிவான அகராதி Azure Developer CLI மற்றும் Azure கிளவுட் மேம்பாட்டில் பயன்படுத்தப்படும் 용ர்கள், கருத்துகள் மற்றும் சுருக்கப்பெயர்களுக்கு வரையறைகளை வழங்குகிறது. தொழில்நுட்ப ஆவணங்களை புரிந்துகொள்ள, பிரச்சனைகளை தீர்க்க மற்றும் azd திட்டங்கள் மற்றும் Azure சேவைகள் பற்றி விளக்கமாக தொடர்பு கொள்ள உதவும் அவசியமான குறிப்பாகும்.

## Learning Goals

இந்த அகராதியை பயன்படுத்துவதன் மூலம், நீங்கள்:
- Azure Developer CLI தொடர்பான அவசியமான 용ர்கள் மற்றும் கருத்துகளைப் புரிந்துகொள்வீர்கள்
- Azure கிளவுட் மேம்பாட்டு சொற்களையும் தொழில்நுட்ப 용ரையும் நவீனமாகக் கையாள்வதில் திறன்மிக்க நிலையில் இருப்பீர்கள்
- Infrastructure as Code மற்றும் அமைப்பு ஒதுக்கீடு 용ர்களை விளக்கமாகக் குறிப்பிட முடியும்
- Azure சேவைகள் பெயர்கள், சுருக்கப்பெயர்கள் மற்றும் அவற்றின் நோக்கங்களை புரிந்துகொள்வீர்கள்
- பிழை கண்டறிதல் மற்றும் டிபக்கிங் 용ர்களுக்கான வரையறைகளை அணுக முடியும்
- மேம்பட்ட Azure கட்டமைப்பு மற்றும் மேம்பாட்டு கருத்துக்களை கற்றுக்கொள்வீர்கள்

## Learning Outcomes

இந்த அகராதியை நிதானமாக பயன்படுத்தினால், நீங்கள் செய்ய முடியும்:
- சரியான Azure Developer CLI 용 நிபுணத்துவத்துடன் விளக்கமாக தொடர்புகொள்ள
- தொழில்நுட்ப ஆவணங்கள் மற்றும் பிழை செய்திகளை தெளிவாக புரிந்துகொள்ள
- Azure சேவைகள் மற்றும் கருத்துக்களில்/confidence-ஆக வழிசெலுத்த
- தொடர்புடைய தொழில்நுட்ப சொற்களைக் கொண்டு பிரச்சனைகளை தீர்க்க
- குழு விவாதங்களில் சரியான தொழில்நுட்ப மொழியுடன் பங்களிக்க
- உங்கள் Azure கிளவுட் மேம்பாட்டு அறிவை முறையாக விரிவுபடுத்த

## A

**ARM Template**  
JSON-ஆதாரமான Infrastructure as Code வடிவம். Azure வளங்களை அறிவிக்கப்படுத்தவும் அல்லது ஒதுக்குவதற்கும் பயன்படும்.

**App Service**  
இன்ஃப்ராஸ்ட்ரக்சர் மேலாண்மையை கவலைப்படாமல் வலை பயன்பாடுகள், REST APIs மற்றும் மொபைல் பின்தளங்களை ஹோஸ்ட் செய்ய Azure இன் platform-as-a-service (PaaS) சலுகை.

**Application Insights**  
பயன்பாட்டு செயல்திறன், கிடைக்கத்தன்மை மற்றும் பயன்பாட்டு உபயோகப்படுத்தலைப் பற்றி ஆழ்ந்த洞察ங்களை வழங்கும் Azure செயலி செயல்திறன் கண்காணிப்பு (APM) சேவை.

**Azure CLI**  
Azure வளங்களை நிர்வகிக்க பயன்படும் கட்டளை வரி இடைமுகம். அங்கீகாரம் மற்றும் சில செயல்பாடுகளுக்காக azd பயன்படுத்துகிறது.

**Azure Developer CLI (azd)**  
டெவலப்பர்-மையக்கூற olan கட்டளை வரி கருவி, டெம்ப்ளேட்கள் மற்றும் Infrastructure as Code பயன்படுத்தி Azureக்கு பயன்பாடுகளை கட்டியமைத்து ஒதுக்குவதற்கான செயல்முறையை வேகப்படுத்துகிறது.

**azure.yaml**  
azd திட்டத்திற்கான முக்கிய அமைப்பு கோப்பு, இது சேவைகள், உட்கட்டமைப்பு மற்றும் ஒதுக்கீட்டு ஹுக்‌களை வரையறுக்கிறது.

**Azure Resource Manager (ARM)**  
வளங்களை உருவாக்க, புதுப்பிக்க மற்றும் நீக்குவதற்கான மேலாண்மைக் தகடு வழங்கும் Azure இன் ஒதுக்கீடு மற்றும் மேலாண்மை சேவை.

## B

**Bicep**  
Microsoft உருவாக்கிய டொமைன்-சார்ந்த மொழி (DSL) ஆகும், இது Azure வளங்களை ஒதுக்கீடு செய்ய பயன்படுகிறது. ARM டெம்ப்ளேட்களுக்கு ஒப்பிடுகையில் எளிதான சொற்றொடரை வழங்கி, ARM க்கு compile ஆகுகிறது.

**Build**  
மூலக் குறியீட்டை தொகுத்தல், சார்புகளை நிறுவுதல் மற்றும் பயன்பாடுகளை ஒதுக்கீட்டிற்கு தயார்படுத்தும் செயல்முறை.

**Blue-Green Deployment**  
இரண்டு ஒரே மாதிரியான உற்பத்தி சுற்றுச்சூழல்களை (blue மற்றும் green) பயன்படுத்தி டவுன்டைம் மற்றும் அபாயத்தை குறைக்கும் ஒதுக்கீட்டு_STRATEGY.

## C

**Container Apps**  
சிக்கலான இன்ஃப்ராஸ்ட்ரக்சரை மேலாண்மையின்றி container செய்யப்பட்ட பயன்பாடுகளை இயக்கும் Azure இன் serverless container சேவை.

**CI/CD**  
Continuous Integration/Continuous Deployment. குறியீட்டு மாற்றங்களை ஒருங்கிணைக்கும் மற்றும் பயன்பாடுகளை தானாக ஒதுக்கீடு செய்யும் நடைமுறைகள்.

**Cosmos DB**  
உலகளாவியமாகப் பகிர்ந்துள்ள, பல-மாதிரி தரவுத்தள சேவை; throughput, latency, கிடைக்கத்தன்மை மற்றும் ஒருமைப்பாட்டிற்கான விரிவான SLA-களை வழங்குகிறது.

**Configuration**  
பயன்பாட்டு நடத்தை மற்றும் ஒதுக்கீட்டு விருப்பங்களை கட்டுப்படுத்தும் அமைப்புகள் மற்றும் அளவுருக்கள்.

## D

**Deployment**  
பயன்பாடுகள் மற்றும் அவற்றின் சார்புகளை இலக்கு இன்ஃப்ராஸ்ட்ரக்சரில் நிறுவி அமைக்கும் செயல்முறை.

**Docker**  
containerization தொழில்நுட்பத்தைப் பயன்படுத்தி பயன்பாடுகளை உருவாக்குதல், அனுப்புதல் மற்றும் இயக்குதல் ஆகியவற்றிற்கான பிளாட்ஃபார்ம்.

**Dockerfile**  
Docker container image உருவாக்கக் குரிய அறிவுறுத்தல்களை கொண்ட உரை கோப்பு.

## E

**Environment**  
உங்கள் பயன்பாட்டின் ஒரு குறிப்பிட்ட நகலை பிரதிநிதித்துவப்படுத்தும் ஒதுக்கீட்டு இலக்கு (உதாரணம்: development, staging, production).

**Environment Variables**  
பயன்பாடுகள் runtime-இல் அணுகக்கூடிய விசை-மதிப்பு ஜோடிகளாகக் சேமிக்கப்படும் அமைப்பு மதிப்புகள்.

**Endpoint**  
ஒரு பயன்பாடு அல்லது சேவையை அணுகக்கூடிய URL அல்லது நெட்வொர்ந்க் முகவரி.

## F

**Function App**  
இன்ஃப்ராஸ்ட்ரக்சரை மேலாண்மை செய்யாமல் ஈவெண்ட்-சார்ந்த குறியீட்டை இயக்க உதவும் Azure இன் serverless கணினி சேவை.

## G

**GitHub Actions**  
GitHub ரெப்போசிடரிகளுடன் ஒருங்கிணைக்கப்பட்ட CI/CD பிளாட்ஃபார்ம், வேலைய흐flows-ஐ தானாக இயக்க பயன்படும்.

**Git**  
மூலக் குறியீட்டில் மாற்றங்களை கண்காணிக்க பயன்படும் விநியோகிக்கப்பட்ட பதிப்புத்தான் கண்காணிப்பு அமைப்பு.

## H

**Hooks**  
ஒதுக்கீட்டு வாழ்க்கைச் சுழற்சியின் குறிப்பிட்ட புள்ளிகளில் இயக்கப்படும் கஸ்டம் ஸ்கிரிப்ட் அல்லது கட்டளைகள் (preprovision, postprovision, predeploy, postdeploy).

**Host**  
பயன்பாடு ஒதுக்கீடு செய்யப்படும் Azure சேவை வகை (e.g., appservice, containerapp, function).

## I

**Infrastructure as Code (IaC)**  
மனித கையால் செய்யும் செயல்முறைகளின் பதிலாக, கோடியால் இன்ஃப்ராஸ்ட்ரக்சரை வரையறுக்கும் மற்றும் நிர்வகிக்கும் நடைமுறை.

**Init**  
புதிய azd திட்டத்தை, பொதுவாக ஒரு டெம்ப்ளேட் இருந்து, ஆரம்பிப்பதற்கான செயல்முறை.

## J

**JSON**  
JavaScript Object Notation. கட்டமைப்பு கோப்புகள் மற்றும் API பதில்களை 위해 பொதுவாகப் பயன்படுத்தப்படும் தரவுப் பரிமாற்ற வடிவம்.

**JWT**  
JSON Web Token. தரப்புகள் இடையே பாதுகாப்பாக தகவலை JSON பொருளாக அனுப்புவதற்கான முறை.

## K

**Key Vault**  
ரகசியங்கள், விசைகள் மற்றும் சான்றிதழ்களை பாதுகாப்பாக சேமித்து நிர்வகிக்க Azure இன் சேவை.

**Kusto Query Language (KQL)**  
Azure Monitor, Application Insights மற்றும் பிற Azure சேவைகளிலுள்ள தரவை விசாரணை செய்ய பயன்படும் கேள்வி மொழி.

## L

**Load Balancer**  
வெளியே வரும் நெட்வொர்க் போக்குவரத்தை பல சேவையகம் அல்லது இன்ஸ்டன்ஸ்களுக்கு விநியோகிக்கும் சேவை.

**Log Analytics**  
கிளவுட் மற்றும் ஆன்-பிரேமிஸ்ஸு சூழல்களிலிருந்து டெலெமெட்ரி தரவை சேகரித்து, பகுப்பாய்வு செய்து, நடவடிக்கை எடுக்கும் Azure சேவை.

## M

**Managed Identity**  
இப்பகுதி Azure சேவைகளுக்கு மற்ற Azure சேவைகளுக்கு அங்கீகாரம் பெற தானாக நிர்வகிக்கப்படும் அடையாளத்தை வழங்கும் அம்சம்.

**Microservices**  
பயன்பாடுகள் சிறு, சுயாதீன சேவைகளின் தொகுப்பாக கட்டமைக்கப்படும் கட்டமைப்பியல் அணுகுமுறை.

**Monitor**  
பயன்பாடுகள் மற்றும் இன்ஃப்ராஸ்ட்ரக்சர் முழு ஸ்டாக்கில் காணொளி வழங்கும் ஒரேகைக் கண்காணிப்பு தீர்வு.

## N

**Node.js**  
Chrome இன் V8 JavaScript என்ஜினில் கட்டப்பட்ட, சர்வர் பக்க பயன்பாடுகளை உருவாக்க பயன்படும் JavaScript runtime.

**npm**  
Node.js க்கான தொகுப்பு மேலாளர்; சார்புகள் மற்றும் பேக்கேஜ்களை நிர்வகிக்கிறது.

## O

**Output**  
இன்ஃப்ராஸ்ட்ரகச்சர் ஒதுக்கீட்டிலிருந்து திருப்பி அளிக்கப்படும் மதிப்புகள்; அவை பயன்பாடுகள் அல்லது பிற வளங்கள் பயன்படுத்தக்கூடும்.

## P

**Package**  
பயன்பாட்டு குறியீடு மற்றும் சார்புகளை ஒதுக்கீட்டிற்கு தயாரிப்பது.

**Parameters**  
ஒதுக்கீடுகளை தனிப்பயனாக்கக் கட்டமைப்பு டெம்ப்ளேட்களுக்கு 전달ிக்கப்படும் உள்ளீட்டு மதிப்புகள்.

**PostgreSQL**  
Open-source தொடர்புடைய தரவுத்தள முறைமையை Azure இல் ஒரு நிர்வகிக்கப்பட்ட சேவையாக ஆதரிக்கிறது.

**Provisioning**  
இன்ஃப்ராஸ்ட்ரக்சர் டெம்ப்ளேட்களில் வரையறுக்கப்பட்ட Azure வளங்களை உருவாக்கி அமைக்கும் செயல்முறை.

## Q

**Quota**  
ஒரு Azure சந்தா அல்லது மண்டலத்தில் உருவாக்கக்கூடிய வளங்களின் அளவு மீதான வரம்புகள்.

## R

**Resource Group**  
ஒரே வாழ்க்கைநாள், அனுமதிகள் மற்றும் கொள்கைகளை பகிர்ந்துகொள்ளும் Azure வளங்களுக்கு தர்க்கீக அடைவு.

**Resource Token**  
azd மூலம் உருவாக்கப்படும் தனித்துவமான ஸ்ட்ரிங்; ஒதுக்கீடுகளில் வள பெயர்கள் தனித்துவமாக இருக்க இந்ததன் உதவியாகும்.

**REST API**  
HTTP முறைமைகளை பயன்படுத்தி நெட்வொர்க் செய்யப்பட்ட பயன்பாடுகளை வடிவமைக்கும் கட்டமைப்பியல் பாணி.

**Rollback**  
பயன்பாடு அல்லது இன்ஃப்ராஸ்ட்ரக்சர் கட்டமைப்பின் முந்தைய பதிப்புக்கு மறு-மாற்றி செல்வது.

## S

**Service**  
azure.yaml இல் வரையறுக்கப்பட்ட உங்கள் பயன்பாட்டின் ஒரு கூறு (e.g., web frontend, API backend, database).

**SKU**  
Stock Keeping Unit. Azure வளங்களுக்கு வெவ்வேறு சேவை அடுக்குகள் அல்லது செயல்திறன் நிலைகளை பிரதிநிதித்துவப்படுத்தும் குறியீடு.

**SQL Database**  
Microsoft SQL Server அடிப்படையிலான Azure இன் நிர்வகிக்கப்பட்ட தொடர்பு தரவுத்தள சேவை.

**Static Web Apps**  
மூலக் குறியீட்டு கோப்பகங்கள் இருந்து முழு ஸ்டாக் வலை பயன்பாடுகளை கட்டியமைத்து ஒதுக்கீடு செய்ய Azure சேவை.

**Storage Account**  
பிளாக்கள் (blobs), கோப்புகள், க்யூ-கள் மற்றும் அட்டவணைகள் உட்பட தரவுப் பொருட்களுக்கு கிளவுட் சேமிப்பை வழங்கும் Azure சேவை.

**Subscription**  
ஆதரிக்கப்படும் பில்லிங் மற்றும் அணுகல் மேலாண்மையுடன் வள குழுக்கள் மற்றும் வளங்களை வைத்திருக்கும் Azure கணக்கு தொட்டி.

## T

**Template**  
பயன்பாட்டு குறியீடு, இன்ஃப்ராஸ்ட்ரக்சர் வரையறைகள் மற்றும் பொதுவான senarios-க்கு உள்ளமைவு கொண்ட முன்னமைக்கப்பட்ட திட்ட அமைப்பு.

**Terraform**  
பல கிளவுட் வழங்குநர்களையும் ஆதரிக்கும் திறந்த மூல Infrastructure as Code கருவி; Azure-ஐ உட்பட ஆதரிக்கிறது.

**Traffic Manager**  
உலகெங்கும் Azure மண்டலங்களுக்கு போக்குவரத்தை விநியோகிக்க DNS-அடிப்படையிலான போக்குவரத்து சுமை சமநிலைப் பணியாளர்.

## U

**URI**  
Uniform Resource Identifier. ஒரு குறிப்பிட்ட வளத்தை அடையாளம் காட்டும் ஸ்ட்ரிங்.

**URL**  
Uniform Resource Locator. ஒரு வளம் எங்கு உள்ளது மற்றும் அதை எவ்வாறு பெறுவது என்பதை குறிப்பதற்கான URI இன் ஒரு வகை.

## V

**Virtual Network (VNet)**  
Azure இல் தனிப்பட்ட நெட்வொர்க்குகளுக்கான அடிப்படைக் கட்டமைப்புக் கூறு; தனிமைப்படுத்தலும் வெட்டுமொலை சக்தியையும் வழங்குகிறது.

**VS Code**  
Visual Studio Code. சிறந்த Azure மற்றும் azd ஒருங்கிணைப்புடன் பிரபலமான குறியீட்டு தொகுப்பி.

## W

**Webhook**  
சிறப்பான விழிப்புகளில் செயற்படுத்தப்படும் HTTP callback; பொதுவாக CI/CD குழுக்களில் பயன்படுத்தப்படுகிறது.

**What-if**  
ஒதுக்கீடு சாத்தியமாக செய்யும் மாற்றங்கள் என்னவென்று உண்மையில் செயல்படுத்தாமல் காட்டும் Azure அம்சம்.

## Y

**YAML**  
YAML Ain't Markup Language. azure.yaml போன்ற அமைப்பு கோப்புகளுக்காக பயன்படும் மனிதனைப் படிக்கக்கூடிய தரவமைப்பு தரநிலைக்கு.

## Z

**Zone**  
Azure மண்டலத்தினுள் பிரித்துக் கொடுக்கப்பட்ட உடல்-அளவிலான இடங்கள்; மீட்பு மற்றும் உயர் கிடைக்கத்தன்மையை வழங்குகின்றன.

## Context-Specific Terms

### Development Terms
- **Hot Reload**: ανάπτυப்பு நேரத்தில் மறுவிடாமலேயே பயன்பாடுகளை தானாகப் புதுப்பித்தல்
- **Build Pipeline**: குறியீட்டை கட்டி சோதிக்கும் தானியங்கி நடைமுறை
- **Deployment Slot**: App Service-ல் உள்ள ஸ்டேஜிங் சூழல்
- **Environment Parity**: development, staging மற்றும் production சூழல்களை ஒத்துப்போடும் படி வைத்திருத்தல்

### Security Terms
- **Managed Identity**: தானாகச் சரிபார்க்கும் அங்கீகாரம் வழங்கும் Azure அம்சம்
- **Key Vault**: ரகசியங்கள், விசைகள் மற்றும் சான்றிதழ்களுக்கு பாதுகாப்பான சேமிப்பு
- **RBAC**: Azure வளங்களுக்கு பாத்திர அடிப்படையிலான அணுகல் கட்டுப்பாடு
- **Network Security Group**: நெட்வொர்க் போக்குவரத்தை கட்டுப்படுத்த ஒரு மெய்நிகர் ஃபயர்வால்

### Monitoring Terms
- **Telemetry**: அளவீடுகளும் தரவுகளும் தானாக சேகரிப்பது
- **Application Performance Monitoring (APM)**: மென்பொருள் செயல்திறனை கண்காணித்தல்
- **Log Analytics**: பதிவு தரவுகளை சேகரித்து பகுப்பாய்வு செய்வது
- **Alert Rules**: அளவுகோல்களோ அல்லது நிபந்தனைகளோ அடிப்படையில் தானாக அறிவித்தல்கள் பெறும் விதிகள்

### Deployment Terms
- **Blue-Green Deployment**: பூட்டு நேரமின்றி ஒதுக்கீடு செய்யும் STRATEGY
- **Canary Deployment**: பயனாளர்களின் ஒரு சிறு தொகுதிக்குத் தாமதமாக வெளியிடல்
- **Rolling Update**: பயன்பாட்டு இன்ஸ்டன்ஸ்களை வரிசையாக மாற்றுதல்
- **Rollback**: முந்தைய பயன்பாட்டு பதிப்பிற்கு மறு-மாற்றுதல்
