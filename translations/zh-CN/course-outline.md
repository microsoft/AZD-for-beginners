# AZD 初学者课程大纲与学习框架

## 课程概述

通过结构化章节掌握 Azure Developer CLI（azd），实现渐进式学习。**特别聚焦于结合 Microsoft Foundry 进行 AI 应用部署。**

### 为什么这门课程对现代开发者至关重要

基于 Microsoft Foundry Discord 社区的见解，**45% 的开发者希望使用 AZD 处理 AI 工作负载**，但面临以下挑战：
- 复杂的多服务 AI 架构
- 生产级 AI 部署最佳实践  
- Azure AI 服务集成与配置
- AI 工作负载的成本优化
- AI 特定部署问题的故障排除

### 核心学习目标

完成本结构化课程后，您将能够：
- **掌握 AZD 基础知识**：核心概念、安装与配置
- **部署 AI 应用程序**：使用 AZD 结合 Microsoft Foundry 服务
- **实现基础设施即代码**：使用 Bicep 模板管理 Azure 资源
- **排查部署问题**：解决常见问题并调试
- **优化生产环境**：安全、扩展、监控和成本管理
- **构建多智能体解决方案**：部署复杂的 AI 架构

## 🎓 研讨会学习体验

### 灵活的学习方式选择
本课程支持 **自学模式** 和 **带指导的研讨会**，让学习者通过交互式练习亲自动手实践 AZD，培养实战技能。

#### 🚀 自学模式
**适合个人开发者和持续学习**

**特点：**
- **基于浏览器的界面**：完整的 MkDocs 驱动研讨会，可通过任意浏览器访问
- **GitHub Codespaces 集成**：一键启动预配置开发环境
- **交互式 DevContainer 环境**：无需本地设置，立即开始编码
- **进度跟踪**：内置检查点和验证练习
- **社区支持**：可访问 Azure Discord 频道，进行问答和协作

**学习结构：**
- **灵活时间安排**：按自己的节奏完成章节，时间跨度从几天到数周
- **检查点系统**：通过验证后再进入更复杂主题
- **资源库**：全面的文档、示例和故障排除指南
- **作品集开发**：构建可部署项目，提升专业作品集

**开始使用（自学）：**
```bash
# 选项 1：GitHub Codespaces（推荐）
# 进入仓库并点击 “代码” → “在 main 上创建代码空间”

# 选项 2：本地开发
git clone https://github.com/microsoft/azd-for-beginners.git
cd azd-for-beginners/workshop
# 请遵循 workshop/README.md 中的设置说明
```
