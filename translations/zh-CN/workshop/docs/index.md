# 面向 AI 开发者的 AZD 研讨会  

> 基于 `azd 1.27.1` 于 2026 年 7 月验证。  

> [!IMPORTANT]  
> **本研讨会配备了可在本地浏览器预览的研讨会指南。开始前，请在 GitHub Codespaces、开发容器中或安装有 MkDocs 的本地克隆库中打开仓库——然后等待 VS Code 终端激活，输入：**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **系统应弹出对话框，提示在浏览器中打开预览页面。**  

欢迎参加面向 AI 应用部署的 Azure Developer CLI (AZD) 实战研讨会。本研讨会通过三步帮助你深入理解 AZD 模板的应用：  

1. <strong>发现</strong> - 找到适合你的模板。  
1. <strong>部署</strong> - 部署并验证其可用性。  
1. <strong>自定义</strong> - 修改并反复迭代，使其专属于你！  

在本研讨会过程中，你还将了解核心开发工具和工作流，助力简化你的端到端开发流程。

## 研讨会概览  

**时长：** 3-4 小时  
**级别：** 初学者到中级  
**先决条件：** 熟悉 Azure、AI 概念、VS Code 及命令行工具。  

这是一场实操型研讨会，通过动手实践学习。完成所有练习后，建议继续研读 AZD 入门课程，深入探索安全性和生产力最佳实践。  

| 时间 | 模块 | 目标 |  
|:---|:---|:---|  
| 15 分钟 | 介绍 | 设定背景，理解目标 |  
| 30 分钟 | 选择 AI 模板 | 探索选项，选择起点 |  
| 30 分钟 | 验证 AI 模板 | 部署默认解决方案到 Azure |  
| 30 分钟 | 拆解 AI 模板 | 探索结构和配置 |  
| 30 分钟 | 配置 AI 模板 | 启用并尝试可用功能 |  
| 30 分钟 | 自定义 AI 模板 | 调整模板，满足你的需求 |  
| 30 分钟 | 清理基础设施 | 清理并释放资源 |  
| 15 分钟 | 总结及后续步骤 | 学习资源，研讨会挑战 |  
| | |  

## 你将学到什么  

将 AZD 模板视为一个学习沙盒，用来探索 Microsoft Foundry 端到端开发的多种能力及工具。研讨会结束时，你应对相关工具和概念有直观的理解。  

| 概念 | 目标 |  
|:---|:---|  
| **Azure Developer CLI** | 理解工具命令及工作流 |  
| **AZD 模板**| 理解项目结构及配置 |  
| **Azure AI Agent**| 配置并部署 Microsoft Foundry 项目 |  
| **Azure AI 搜索**| 通过代理启用上下文工程 |  
| <strong>可观测性</strong>| 探索追踪、监控及评估 |  
| <strong>红队演练</strong>| 探索对抗性测试及缓解措施 |
