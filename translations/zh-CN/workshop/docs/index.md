欢迎参与以 AI 应用部署为重点的 Azure Developer CLI (AZD) 实战研讨会。本研讨会将通过 3 个步骤帮助你掌握 AZD 模板的实际应用理解：

1. <strong>发现</strong> - 找到适合你的模板。
1. <strong>部署</strong> - 部署并验证其可用性
1. <strong>自定义</strong> - 修改并迭代以满足你的需求！

在本研讨会过程中，你还将接触到核心开发工具和工作流，帮助你简化端到端的开发流程。

## 研讨会概览

**时长：** 3-4 小时  
**级别：** 初学者至中级  
**先决条件：** 熟悉 Azure、AI 概念、VS Code 和命令行工具。

这是一个以实践为主的研讨会，通过动手学习。一旦完成练习，我们建议你回顾 AZD 入门课程，以继续在安全性和生产力最佳实践方面的学习旅程。

| 时间| 模块  | 目标 |
|:---|:---|:---|
| 15 mins | 介绍 | 设定背景，理解目标 |
| 30 mins | 选择 AI 模板 | 探索选项并选择入门模板 | 
| 30 mins | 验证 AI 模板 | 将默认解决方案部署到 Azure 并验证 |
| 30 mins | 解构 AI 模板 | 探索结构和配置 |
| 30 mins | 配置 AI 模板 | 激活并试用可用功能 |
| 30 mins | 自定义 AI 模板 | 将模板调整为你的需求 |
| 30 mins | 拆除基础设施 | 清理并释放资源 |
| 15 mins | 总结与下一步 | 学习资源，研讨会挑战 |
| | |

## 你将学到什么

将 AZD 模板视为一个学习沙盒，用于探索 Microsoft Foundry 上端到端开发的各种功能和工具。完成本研讨会后，你应对该环境中的各种工具和概念有直观的理解。

| 概念  | 目标 |
|:---|:---|
| **Azure Developer CLI** | 了解工具命令和工作流|
| **AZD 模板**| 了解项目结构和配置|
| **Azure AI Agent**| 预配并部署 Microsoft Foundry 项目  |
| **Azure AI Search**| 使用 agent 启用上下文工程 |
| <strong>可观察性</strong>| 探索追踪、监控与评估 |
| <strong>红队测试</strong>| 探索对抗性测试与缓解措施 |
