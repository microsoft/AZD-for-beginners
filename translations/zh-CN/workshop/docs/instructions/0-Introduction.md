# AZD For AI Developers

!!! info "AUDIENCE FOR THIS WORKSHOP"
   
    本次工作坊适合希望通过利用一流工具和工作流程在 Microsoft Foundry 上构建 AI Agentic 应用的任何人。该工作坊适合对 Azure、AI 概念和 Python 代码有一定了解的初学者和有经验的开发者。在开始之前请查看先决条件。

    **Duration:** 3-4 hours  
    **Level:** Beginner to Intermediate  

---


## Pre-Requisites

- A personal GitHub account → [Create one for free](https://github.com/signup)
- An active Azure account → [Create one for free](https://aka.ms/free)
- Familiarity with AI concepts, Python development, Command-line tools
- Experience with VS Code and GitHub tools and workflows.

---

## Learning Objectives

By the end of this workshop, you will know how to:

- [X] **Work with Azure Developer CLI** → 用于配置、部署和管理，您的 Azure 资源。
- [X] **Use AI App Templates** →  用来快速启动 Microsoft Foundry 上的 AI Agent 项目。
- [X] **Configure Templates** →  用于像 Red Teaming、Evaluations、Tracing & Monitoring 这样的功能
- [X] **Deconstruct and Explore** →  使用 GitHub Copilot 解构并探索 AZD 模板和工作流
- [X] **Customize and Redeploy** →  根据您的自定义需求自定义并重新部署 AI Agent 入门模板

---

## Workshop Outline

The workshop is structured as modules, each anchored by a hands-on lab exercise:

1. **Introduction** - 了解 AZD 模板是什么，以及它们如何工作
1. **Selection** - 发现适合您下一个 AI 项目的 "starter" 模板
1. **Validation** - 部署并探索模板以验证其是否如宣传所述运行
1. **Deconstruction** - 拆解模板以了解其组件 & 配置
1. **Configuration** - 学习启用并使用不同的 Microsoft Foundry 功能
1. **Customization** - 根据您的数据和需求重构应用
1. **Teardown** - 学习取消部署基础设施并恢复软删除的资源
