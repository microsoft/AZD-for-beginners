# AZD бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбҹ– бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶ бһ“бһ·бһ„бһҹбҹҠбһ»бһҳбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶

## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…бһ“бҹғбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶

бһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ AZD (Azure Developer CLI) бһҸбһ¶бһҳбһҡбһҷбҹҲбһҮбҹҶбһ–бһјбһҖбһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһ‘бҹ…бһҳбһ»бһҒбҹ” **бһ•бҹ’бһҸбҹ„бһҸбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһ–бһ·бһҹбҹҒбһҹбһӣбһҫбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһІбҹ’бһҷбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳ Microsoft Foundryбҹ”**

### бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһңбһӮбҹ’бһӮбһ“бҹҒбҹҮбһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ‘бҹҶбһ“бһҫбһ”

бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ‘бһҹбҹ’бһҹбһ“бҹҲбһҖбҹ’бһ“бһ»бһ„бһҹбһ бһӮбһҳбһ“бҹҚ Discord Microsoft Foundry, **45% бһ“бҹғбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ…бһ„бҹӢбһ”бҹ’бһҡбһҫ AZD бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ“бҹ’бһ‘бһ»бһҖ AI** бһҸбҹӮбһ”бҹ’бһҡбһҲбһҳбһҳбһ»бһҒбһ“бһ№бһ„бһ”бһүбҹ’бһ бһ¶бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбһјбһ…бһҮбһ¶бҹҲ
- бһҡбҹҖбһ”бһ…бҹҶбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ AI бһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ…бҹ’бһҡбһҫбһ“бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү
- бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡвҖӢбһҠбһ¶бһҖбҹӢбһҮбһ¶бһҳбһҪбһҷ AI бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ•бһӣбһ·бһҸ
- бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ AI бһҡбһ”бһҹбҹӢ Azure
- бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбһҳбҹ’бһҡбҹғбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ“бҹ’бһ‘бһ»бһҖ AI
- бһҶбҹӮбһҖбһӮбҹ’бһҡбҹ„бҹҮбһҗбҹ’бһ“бһ¶бһҖбҹӢ бһ“бһ·бһ„бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫ AI

### бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбһ·бһҖбҹ’бһҹбһ¶бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһ…бһҳбҹ’бһ”бһ„

бһҸбһ¶бһҳбһ–бҹҒбһӣбһ”бһүбҹ’бһ…бһ”бҹӢбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһ¶бһ“бҹҲ
- **бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһҮбҹҶбһ“бһ¶бһү AZD бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“**бҹ– бһӮбҹҶбһ“бһ·бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’
- **бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI**бҹ– бһ”бҹ’бһҡбһҫ AZD бһҮбһ¶бһҳбһҪбһҷбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Microsoft Foundry
- **бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ…бҹҶбһҺбһ»бһ…бһӮбҹ„бһӣ вҖңInfrastructure as CodeвҖқ**бҹ– бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ’бһ“бһ’бһ¶бһ“ Azure бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бҹҶбһ–бһ„бҹӢ Bicep
- **бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶вҖӢбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫ**бҹ– бһҮбҹҶбһ“бһҪбһҷбһ”бһүбҹ’бһ бһ¶бһ’бһҳбҹ’бһҳбһҸбһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһҖбһјбһҠ
- **бһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ**бҹ– бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖ бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҗбҹ’бһӣбҹғбһҠбһҫбһҳ
- **бһҖбһҹбһ¶бһ„бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷвҖӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бҹ’бһҡбһҫбһ“**бҹ– бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ AI бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү

## рҹҺ“ бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҖбҹ’бһ“бһ»бһ„бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶

### бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбһ№бһҖбһ“бһ¶бҹҶбһ…бҹ’бһҡбһҫбһ“бһҸбһ¶бһҳбһ”бҹӮбһ”бһҡбҹҖбһ“

бһңбһӮбҹ’бһӮбһ“бҹҒбҹҮбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҮбһјбһ“бһ‘бһ¶бҹҶбһ„ **бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҜбһҖбһҡбһ¶бһҮбҹ’бһҷ** бһ“бһ·бһ„ **бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһҠбһ№бһҖбһ“бһ¶бҹҶбһҠбҹ„бһҷбһӮбҹ’бһҡбһј** бһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһҹбһ·бһҖбҹ’бһҹбһ¶бһўбһ¶бһ…бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҮбһ¶бһҳбһҪбһҷ AZD бһҒбһҺбҹҲбһ–бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһҮбҹҶбһ“бһ¶бһүбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҸбһ¶бһҳбһӣбҹҶбһ бһ¶бһҸбҹӢбһҳбһ¶бһ“бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбҹ”

#### рҹҡҖ бһҡбһ”бҹҖбһ”бһҡбҹҖбһ“бһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„
**бһҹбһҳбһҹбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҜбһҖбһҡбһ¶бһҮбҹ’бһҷ бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҡбһҷбҹҲбһ–бҹҒбһӣбһҷбһјбһҡ**

**бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбҹ–**
- **бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбҹҒбһҡбҹҒбһ бҹ’бһң MkDocs**бҹ– бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбҹҒбһ”бһ·бһ‘бһ”бһҫбһҖбһҸбһ¶бһҳбһҖбһ“бҹ’бһӣбҹӮбһ„бһӣбҹҒбһ„ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбҹҒбһҡбҹҒбһ бҹ’бһң MkDocs бһҸбһ¶бһҳбһңбһ¶бҹҶбһ„бһҠбһјбһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢ
- **бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһ”бһ¶бһ“ GitHub Codespaces**бҹ– бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҳбһҪбһҷбһ…бһ»бһ…бһҠбҹ„бһҷбһҳбһ¶бһ“бһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһҪбһ…бһҡбһ¶бһӣбҹӢ
- **бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ DevContainer бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳ**бҹ– бһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҸбҹҶбһЎбһҫбһ„бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ - бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһ—бҹ’бһӣбһ¶бһҳбҹ—
- **бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“**бҹ– бһҳбһ¶бһ“бһ…бҹҶбһҺбһ»бһ…бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ“бһ·бһ„бһӣбҹҶбһ бһ¶бһҸбҹӢбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ
- **бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҹбһ бһӮбһҳбһ“бҹҚ**бҹ– бһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбҹҖбһңбһ—бҹ… Discord Azure бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҶбһҺбһҪбһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһ бһҖбһ¶бһҡбһ–бһёбһўбҹ’бһ“бһҖбһҮбҹҶбһ“бһ¶бһү

**бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбһ·бһҖбҹ’бһҹбһ¶бҹ–**
- **бһҖбһ¶бһҡбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһҷбҹ„бһӮбһҷбһӣбҹӢ**бҹ– бһ”бһүбҹ’бһ…бһ”бҹӢбһҮбҹҶбһ–бһјбһҖбһҸбһ¶бһҳбһӣбҹ’бһ”бһҝбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣбһҗбҹ’бһ„бҹғбһ¬бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚ
- **бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ…бҹҶбһҺбһ»бһ…**бҹ– бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҳбһ»бһ“бһ”бһ“бҹ’бһҸбһ‘бҹ…бһҳбһ»бһҒбһңбһ·бһүбҹ’бһүбһ¶бһҹбһ¶бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү
- **бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ’бһ“бһ’бһ¶бһ“**бҹ– бһҜбһҖбһҹбһ¶бһҡбһҮбһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢ бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһ“бһ·бһ„бһҳбҹҒбһҡбҹҖбһ“бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶

#### рҹҸӣпёҸ бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһҠбһ№бһҖбһ“бһ¶бҹҶбһҠбҹ„бһҷбһӮбҹ’бһҡбһј
**бһҹбһҳбһҹбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“ бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҒбҹ’бһӣбһё бһ“бһ·бһ„бһҹбҹ’бһҗбһ¶бһ”бҹҗбһ“бһўбһ”бҹӢбһҡбҹҶ**

**бһҮбһҳбҹ’бһҡбһҫбһҹбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бҹ–**

**рҹ“ҡ бһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһҹбһ¶бһҖбһӣбһңбһ·бһ‘бҹ’бһҷбһ¶бһӣбҹҗбһҷ (бҹЁ-бҹЎбҹў бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚ)**
- **бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбһ¶бһҖбһӣбһңбһ·бһ‘бҹ’бһҷбһ¶бһӣбҹҗбһҷ**бҹ– бһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһҡбһҷбҹҲбһ–бҹҒбһӣбһҹбһҳбһ¶бһҹбһҗбһ·бһҖбһ¶бһҳбһҪбһҷ бһҳбһ¶бһ“бһңбһӮбҹ’бһӮбһҡбҹҖбһ“бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚ бҹўбһҳбҹүбҹ„бһ„
- **бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ Bootcamp**бҹ– бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ№бһҸбһҸбҹ’бһ”бһ·бһҸ бҹЈ-бҹҘ бһҗбҹ’бһ„бҹғ бһҳбһ¶бһ“бһңбһӮбҹ’бһӮбһҡбҹҖбһ“бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҗбҹ’бһ„бҹғбһҡбһҷбҹҲбһ–бҹҒбһӣ бҹҰ-бҹЁ бһҳбҹүбҹ„бһ„бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҗбҹ’бһ„бҹғ
- **бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“**бҹ– бһңбһӮбҹ’бһӮбһҡбҹҖбһ“бһҖбҹ’бһҡбһ»бһҳбһ”бҹ’бһҡбһ…бһ¶бҹҶбһҒбҹӮ бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӮбһҳбҹ’бһҡбҹ„бһ„бһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„
- **бһ’бһ¶бһҸбһ»бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғ**бҹ– бһҖбһ¶бһҡбһ”бҹ’бһҡбһӮбһӣбҹӢбһҖбһ¶бһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ–бһёбһҳбһ·бһҸбҹ’бһҸбһ—бһҖбҹ’бһҸбһ· бһ“бһ·бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ

**рҹҡҖ бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһҮбһҳбҹ’бһҡбҹ… (бҹЎ-бҹЈ бһҗбҹ’бһ„бҹғ)**
- **бһҗбҹ’бһ„бҹғбһ‘бһё бҹЎ**бҹ– бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ + бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ AI (бһҮбҹҶбһ–бһјбһҖ бҹЎ-бҹў) - бҹҰбһҳбҹүбҹ„бһ„
- **бһҗбҹ’бһ„бҹғбһ‘бһё бҹў**бҹ– бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ + бһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ (бһҮбҹҶбһ–бһјбһҖ бҹЈ-бҹӨ) - бҹҰбһҳбҹүбҹ„бһ„  
- **бһҗбҹ’бһ„бҹғбһ‘бһё бҹЈ**бҹ– бһ”бҹҶбһҺбһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбҹҶбһҠбһ¶бһ”бҹӢбһҒбҹ’бһ–бһҹбҹӢ + бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ (бһҮбҹҶбһ–бһјбһҖ бҹҘ-бҹЁ) - бҹЁбһҳбҹүбҹ„бһ„
- **бһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“**бҹ– бһҮбҹҶбһ“бһҪбһҷбһҮбһҳбҹ’бһҡбһҫбһҹбһҡбһҷбҹҲбһ–бҹҒбһӣ бҹў бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһ”бҹӢбһӮбһҳбҹ’бһҡбҹ„бһ„

**вҡЎ бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҹбһ„бҹ’бһҒбҹҒбһ”бһҡбһҠбҹ’бһӢбһ”бһ¶бһӣ (бҹӨ-бҹҰ бһҳбҹүбҹ„бһ„)**
- **бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ**бҹ– бһҸбһҳбҹ’бһӣбҹғ AZD бһ“бһ·бһ„бһ•бһӣбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳ (бҹЎ бһҳбҹүбҹ„бһ„)
- **бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ**бҹ– бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһ–бһёбһҠбһҫбһҳбһҠбһӣбҹӢбһ…бһ»бһ„ (бҹў бһҳбҹүбҹ„бһ„)
- **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’**бҹ– бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ’бһ“бһ’бһ¶бһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ (бҹЎ бһҳбҹүбҹ„бһ„)
- **бһ•бҹӮбһ“бһҖбһ¶бһҡвҖӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚ**бҹ– бһўбһ“бһ»бһҳбҹҗбһҸбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҲбһ¶бһ“бһ‘бҹ…бһҖбҹ’бһҡбҹ„бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ„бҹ’бһӮбһҖбһ¶бһҡ (бҹЎ-бҹў бһҳбҹүбҹ„бһ„)

#### рҹӣ пёҸ бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҡбҹҖбһ“бһҖбҹ’бһ“бһ»бһ„бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶
**бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ вҶ’ бһҠбһ¶бһҖбҹӢбһҮбһ¶бһҳбһҪбһҷ вҶ’ бһҖбҹӮбһ”бҹ’бһҡбҹӮ бһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҮбҹҶбһ“бһ¶бһүбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„**

**бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣ бҹЎбҹ– бһҹбҹ’бһңбҹӮбһ„бһҡбһҖ (бҹӨбҹҘ бһ“бһ¶бһ‘бһё)**
- **бһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бҹҶбһ–бһ„бҹӢ**бҹ– бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ”бҹҶбһ–бһ„бҹӢ бһ“бһ·бһ„бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Microsoft Foundry
- **бһңбһ·бһ—бһ¶бһӮбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’**бҹ– бһҷбһӣбҹӢбһ–бһёбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бҹ’бһҡбһҫбһ“ бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫ
- **бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ**бҹ– бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһўбһ„бҹ’бһӮбһҖбһ¶бһҡбһ“бһ·бһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһҖбҹҶбһҺбһҸбҹӢ
- **бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ**бҹ– бһҖбҹҶбһҺбһҸбҹӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ бһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“ Azure

**бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣ бҹўбҹ– бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫ (бҹў бһҳбҹүбҹ„бһ„)**

- **бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ**бҹ– бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһүбҹ’бһ…бһјбһӣ бһ“бһ·бһ„бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ‘бһјбһ‘бҹ…

**бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣ бҹЈбҹ– бһҖбҹӮбһ”бҹ’бһҡбҹӮ (бҹӨбҹҘ бһ“бһ¶бһ‘бһё)**
- **бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё**бҹ– бһ”бҹ’бһҠбһјбһҡбһ”бҹҶбһ–бһ„бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ–бһ·бһҹбҹҒбһҹбһ“бһ·бһ„бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ
- **бһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ**бҹ– бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡвҖӢбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҗбҹ’бһӣбҹғ бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ–бһ„бҹ’бһҡбһёбһҖ
- **бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһҒбҹ’бһ–бһҹбҹӢ**бҹ– бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бҹ’бһҡбһҫбһ“ бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү
- **бһ•бҹӮбһ“бһҖбһ¶бһҡбһҖбҹ’бһҡбҹ„бһҷ**бҹ– бһҖбҹҶбһҺбһҸбҹӢбһ•бҹ’бһӣбһјбһңбһҡбҹҖбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҮбҹҶбһ“бһ¶бһүбһ”бһ“бҹ’бһҸ

#### рҹҺҜ бһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶
**бһҮбҹҶбһ“бһ¶бһүбһҠбҹӮбһӣбһңбһ¶бһҹбҹӢбһңбҹӮбһ„бһ”бһ¶бһ“вҖӢбһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„**

**бһҮбҹҶбһ“бһ¶бһүбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбҹ–**
- **бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ**бҹ– бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ AI бһҠбҹ„бһҷбһҮбҹ„бһӮбһҮбҹҗбһҷ
- **бһҮбҹҶбһ“бһ¶бһү Infrastructure as Code**бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸ бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бҹҶбһ–бһ„бҹӢ Bicep бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“
- **бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бҹ’бһҡбһҫбһ“**бҹ– бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһӮбҹ’бһ“бһ¶
- **бһ—бһ¶бһ–бһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ**бҹ– бһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ
- **бһҮбҹҶбһ“бһ¶бһүбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶**бҹ– бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҠбҹ„бһҷбһҜбһҖбһҡбһ¶бһҮбҹ’бһҷ

**бһҮбҹҶбһ“бһ¶бһүбһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲбҹ–**
- **бһ—бһ¶бһ–бһҠбһ№бһҖбһ“бһ¶бҹҶбһӮбһҳбҹ’бһҡбҹ„бһ„**бҹ– бһҠбһ№бһҖбһ“бһ¶бҹҶбһҖбҹ’бһҡбһ»бһҳбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһңбҹҒбһҡбһҖбһҳбҹ’бһҳбһ–бһ–бһҖ
- **бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ**бҹ– бһҡбһ…бһ“бһ¶бһҸбҹҶбһӣбҹғбһҗбҹ’бһӣбҹғбһҹбһҳбһҡбһҳбҹ’бһҷ бһ“бһ·бһ„бһӣбҹ’бһ”бһҝбһ“бһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ Azure
- **бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҒбһҡбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„**бҹ– бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ бһ“бһ·бһ„бһҮбһҪбһҷбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“ AZD
- **бһ•бҹӮбһ“бһҖбһ¶бһҡбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ**бҹ– бһҳбһ¶бһ“бһҘбһ‘бҹ’бһ’бһ·бһ–бһӣбһӣбһҫбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡвҖӢбһ—бһ¶бһ–бһ‘бһ‘бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһ“бҹӢбһ–бһ–бһҖбһҡбһ”бһҹбҹӢбһўбһ„бҹ’бһӮбһҖбһ¶бһҡ

#### рҹ“Ӣ бһҜбһҖбһҹбһ¶бһҡбһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶

**бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһҹбһ·бһҖбҹ’бһҹбһ¶бҹ–**
- **бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҹбһ·бһҖбҹ’бһҹбһ¶бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳ**бҹ– [бһҜбһҖбһҹбһ¶бһҡбһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶](workshop/README.md) -бһңбҹҒбһ‘бһ·бһҖбһ¶бһҹбһ·бһҖбҹ’бһҹбһ¶бһ‘бҹ…бһҸбһ¶бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбҹҒбһҡбҹҒбһ бҹ’бһң
- **бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҮбһ¶бһҮбҹҶбһ бһ¶бһ“**бҹ– [бһӣбҹҶбһ бһ¶бһҸбҹӢбһҠбһ№бһҖбһ“бһ¶бҹҶ](../../workshop/docs/instructions) - бһҮбҹҶбһ бһ¶бһ“бһ“бһ¶бһ“бһ¶
- **бһҜбһҖбһҹбһ¶бһҡбһҷбҹ„бһ„**бҹ– [бһҳбһ“бҹ’бһ‘бһёбһҡбһҹбһ·бһҖбҹ’бһҹбһ¶ AI](docs/microsoft-foundry/ai-workshop-lab.md) - бһҮбҹ’бһҡбһ¶бһ”бһҮбҹ’бһҡбһ¶бһҹбһ–бһ·бһҹбҹҒбһҹ AI
- **бһ’бһ“бһ’бһ¶бһ“бһҹбһ бһӮбһҳбһ“бҹҚ**бҹ– бһ”бһҺбҹ’бһҠбһ¶бһү Discord Azure, бһҖбһ¶бһҡбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶ GitHub бһ“бһ·бһ„бһҮбҹҶбһ“бһҪбһҷбһ‘бһёбһ•бҹ’бһҹбһ¶бһҡ

#### рҹҸў бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸвҖӢбһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶вҖӢбһўбһ„бҹ’бһӮбһҖбһ¶бһҡ
**бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ бһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһўбһ„бҹ’бһӮбһҖбһ¶бһҡ**

**бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бҹ–**
- **бһ…бһјбһӣбһҡбһҪбһҳбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҗбҹ’бһҳбһё**бҹ– бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ AZD (бҹў-бҹӨ бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚ)
- **бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҖбҹ’бһҡбһ»бһҳ**бҹ– бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҸбҹ’бһҡбһёбһҳбһ¶бһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹ’бһҡбһ»бһҳбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ (бҹЎ-бҹў бһҗбҹ’бһ„бҹғ)
- **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’**бҹ– бһҹбһҳбһ¶бһҹбһ—бһ¶бһӮбһ”бҹ’бһҡбһ…бһ¶бҹҶбһҒбҹӮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһ·бһҹбҹ’бһңбһҖбһҡ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһҡбһ…бһ“бһ¶ (бҹӨ бһҳбҹүбҹ„бһ„)
- **бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһўбҹҶбһ–бһёбһҠбһ№бһҖбһ“бһ¶бҹҶ**бҹ– бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһҡбһҠбҹ’бһӢбһ”бһ¶бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ (бһ–бһ¶бһҖбҹӢбһҖбһҺбҹ’бһҠбһ¶бһӣбһҗбҹ’бһ„бҹғ)

**бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ–**
- **бһҡбһ…бһ“бһ¶бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһҳбһ¶бһ“бһ•бҹ’бһ‘бһ¶бһӣбҹӢ**бҹ– бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҒбҹ’бһӣбһҪбһ“бһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһўбһ„бҹ’бһӮбһҖбһ¶бһҡ
- **бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбҹҶбһҡбһјбһӮбһ“бҹ’бһӣбһ„**бҹ– бһ”бһүбҹ’бһ…бҹҒбһүбһҠбҹ„бһҷбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҮбһ¶бһҳбһҪбһҷбһ…бҹҶбһҺбһ»бһ…бһҮбҹ„бһӮбһҮбҹҗбһҷ бһ“бһ·бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢ  
- **бһҖбһ¶бһҡбһҮбҹҶбһ“бһҪбһҷбһ”бһ“бҹ’бһҸ**бҹ– бһҮбҹҶбһ“бһҪбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӮбһҳбҹ’бһҡбҹ„бһ„
- **бһҖбһҹбһ¶бһ„бһҹбһ бһӮбһҳбһ“бҹҚ**бҹ– бһҹбһ бһӮбһҳбһ“бҹҚбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚ Azure AI бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„

**бһҳбһ¶бһҸбҹ’бһҡбһҠбҹ’бһӢбһ¶бһ“бһҮбҹ„бһӮбһҮбҹҗбһҷбҹ–**
- **бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҮбҹҶбһ“бһ¶бһү**бҹ– бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҳбһ»бһ“бһ“бһ·бһ„бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһ»бһҺбһ—бһ¶бһ–бһҮбҹҶбһ“бһ¶бһүбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ
- **бһҮбҹ„бһӮбһҮбҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫ**бҹ– бһ—бһ¶бһӮбһҡбһҷбһўбҹ’бһ“бһҖбһ…бһјбһӣбһҡбһҪбһҳбһҠбҹӮбһӣбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһ”бһ¶бһ“бһҮбҹ„бһӮбһҮбҹҗбһҷ
- **бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ‘бҹ…бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ**бҹ– бһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбһҷбһ–бҹҒбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһӮбһҳбҹ’бһҡбҹ„бһ„ Azure AI бһҗбҹ’бһҳбһё
- **бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„**бҹ– бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ бҹЈ-бҹҰ бһҒбҹӮбһҖбҹ’бһҡбҹ„бһҷбһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶

## бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбһ·бһҖбҹ’бһҹбһ¶ бҹЁ бһҮбҹҶбһ–бһјбһҖ

### бһҮбҹҶбһ–бһјбһҖ бҹЎбҹ– бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ“бһ·бһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҡбһ бҹҗбһҹ (бҹЈбҹ -бҹӨбҹҘ бһ“бһ¶бһ‘бһё) рҹҢұ
**бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҳбһ»бһ“**бҹ– бһҳбһ¶бһ“бһӮбһҺбһ“бһё Azure, бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бһүбҹ’бһҮбһ¶commandline  
**бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү**бҹ– вӯҗ

#### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҹбһ·бһҖбҹ’бһҹбһ¶
- бһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһё AZD бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“
- бһҸбҹҶбһЎбһҫбһ„ AZD бһ“бҹ…бһӣбһҫбһңбҹҒбһ‘бһ·бһҖбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ  
- бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһҮбһ¶бһӣбһҫбһҖбһҠбҹҶбһ”бһјбһ„бһҠбҹ„бһҷбһҮбҹ„бһӮбһҮбҹҗбһҷ
- бһӮбҹҶбһ“бһ·бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ“бһ·бһ„бһ–бһ¶бһҖбҹ’бһҷбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ

#### бһ’бһ“бһ’бһ¶бһ“бһҹбһ·бһҖбҹ’бһҹбһ¶
- [AZD бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“](docs/getting-started/azd-basics.md) - бһӮбҹҶбһ“бһ·бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“
- [бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’](docs/getting-started/installation.md) - бһҳбҹҒбһҡбҹҖбһ“бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҸбһ¶бһҳбһңбҹҒбһ‘бһ·бһҖбһ¶
- [бһӮбһҳбҹ’бһҡбҹ„бһ„бһҠбҹҶбһ”бһјбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ](docs/getting-started/first-project.md) - бһҳбҹҒбһҡбҹҖбһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбҹғ
- [бһҹбҹҖбһңбһ—бҹ… cheat sheet](resources/cheat-sheet.md) - бһҜбһҖбһҹбһ¶бһҡбһҡбһ бҹҗбһҹбһҷбһӣбҹӢ

#### бһӣбһ‘бҹ’бһ’бһ•бһӣбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„
бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҹбһ¶бһҳбһүбҹ’бһүбһ‘бҹ…бһӣбһҫ Azure бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ AZD бһ”бһ¶бһ“бһҮбҹ„бһӮбһҮбҹҗбһҷ

### бһҮбҹҶбһ–бһјбһҖ бҹўбҹ– бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ AI бһҮбһ¶бһӣбһҫбһҖбһҠбҹҶбһ”бһјбһ„ (бҹЎ-бҹў бһҳбҹүбҹ„бһ„) рҹӨ–

#### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҹбһ·бһҖбҹ’бһҹбһ¶
- бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Microsoft Foundry бһҮбһ¶бһҳбһҪбһҷ AZD

- бһҷбһӣбҹӢбһ–бһёбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбҹҒбһңбһ¶ AI
- бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ RAG (Retrieval-Augmented Generation)

#### бһ’бһ“бһ’бһ¶бһ“бһҹбһ·бһҖбҹ’бһҹбһ¶
- [бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Microsoft Foundry](docs/microsoft-foundry/microsoft-foundry-integration.md)
- [бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣ AI](docs/microsoft-foundry/ai-model-deployment.md)

- [бһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бҹҒбһҹбһҖбҹҚбһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳ](workshop/README.md) - **бһҗбҹ’бһҳбһё**бҹ– бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһҸбһ¶бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбҹҒбһҡбҹҒбһ бҹ’бһң MkDocs
- [бһ”бҹҶбһ–бһ„бҹӢ Microsoft Foundry бһҮбһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢ](README.md#featured-microsoft-foundry-templates)
- [бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҖбҹ’бһ“бһ»бһ„бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶](../../workshop/docs/instructions) - **бһҗбҹ’бһҳбһё**бҹ– бһӣбҹҶбһ бһ¶бһҸбҹӢбһҠбһ№бһҖбһ“бһ¶бҹҶбһҮбһ¶бһҮбҹҶбһ бһ¶бһ“

#### бһӣбһ‘бҹ’бһ’бһ•бһӣбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„
бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбһҮбҹӮбһҖ AI бһҮбһ¶бһҳбһҪбһҷбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– RAG

#### бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶ (бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбһ”бһ“бҹ’бһҗбҹӮбһҳ)
**бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҗбҹ’бһҳбһё**бҹ– [бһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бҹҒбһҹбһҖбҹҚбһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһ–бҹҒбһүбһӣбҹҒбһү](workshop/README.md)
1. **бһҹбҹ’бһңбҹӮбһ„бһҡбһҖ** (бҹЈбҹ  бһ“бһ¶бһ‘бһё): бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ бһ“бһ·бһ„бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ”бҹҶбһ–бһ„бҹӢ
2. **бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫ** (бҹӨбҹҘ бһ“бһ¶бһ‘бһё): бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡ
3. **бһ”бҹҶбһ”бҹӮбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ** (бҹЈбҹ  бһ“бһ¶бһ‘бһё): бһҷбһӣбҹӢбһ–бһёбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһ’бһ¶бһҸбһ»
4. **бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’** (бҹЈбҹ  бһ“бһ¶бһ‘бһё): бһ”бҹ’бһҸбһјбһҡбһҸбҹҶбһҡбһјбһңбһҖбһ¶бһҡбһ“бһ·бһ„бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ
5. **бһҖбҹӮбһ”бҹ’бһҡбҹӮ** (бҹӨбҹҘ бһ“бһ¶бһ‘бһё): бһҖбҹӮбһ”бҹ’бһҡбҹӮ бһ“бһ·бһ„бһ”бһ“бҹ’бһҸбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһ“бһҖ
6. **бһ”бһүбҹ’бһ…бһ”бҹӢ** (бҹЎбҹҘ бһ“бһ¶бһ‘бһё): бһҹбһҳбҹ’бһўбһ¶бһҸбһ’бһ“бһ’бһ¶бһ“ бһ“бһ·бһ„бһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҡбһҷбҹҲбһ–бҹҒбһӣбһҮбһёбһңбһ·бһҸ
7. **бһ”бһ·бһ‘бһ”бһ¶бҹҶбһ„** (бҹЎбҹҘ бһ“бһ¶бһ‘бһё): бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ бһ“бһ·бһ„бһ•бҹ’бһӣбһјбһңбһ“бҹғбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһЎбһҫбһ„

### бһҮбҹҶбһ–бһјбһҖ бҹЈбҹ– бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ (бҹӨбҹҘ-бҹҰбҹ  бһ“бһ¶бһ‘бһё) вҡҷпёҸ
**бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҳбһ»бһ“**бҹ– бһҮбҹҶбһ–бһјбһҖ бҹЎ бһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢ  
**бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү**бҹ– вӯҗвӯҗ

#### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҹбһ·бһҖбҹ’бһҹбһ¶
- бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ
- бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ бһ“бһ·бһ„бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ
- бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҲбҹ’бһҳбҹ„бҹҮбһ’бһ“бһ’бһ¶бһ“ бһ“бһ·бһ„бһҡбҹҖбһ”бһ…бҹҶ
- бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ…бҹ’бһҡбһҫбһ“

#### бһ’бһ“бһ’бһ¶бһ“бһҹбһ·бһҖбҹ’бһҹбһ¶
- [бһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бҹҒбһҹбһҖбҹҚбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’](docs/getting-started/configuration.md) - бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ
- [бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ бһ“бһ·бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–](docs/getting-started/authsecurity.md) - бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ managed identity бһ“бһ·бһ„ Key Vault
- бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ…бҹ’бһҡбһҫбһ“

#### бһӣбһ‘бҹ’бһ’бһ•бһӣбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„
бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ…бҹ’бһҡбһҫбһ“бһҠбҹ„бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ бһ“бһ·бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң

### бһҮбҹҶбһ–бһјбһҖ бҹӨбҹ– бһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҮбһ¶бһӮбһјбһҠ бһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫ (бҹЎ-бҹЎ.бҹҘ бһҳбҹүбҹ„бһ„) рҹҸ—пёҸ

**бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү**бҹ– вӯҗвӯҗвӯҗ

#### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҹбһ·бһҖбҹ’бһҹбһ¶
- бһ‘бҹҶбһ–бҹҗбһҡбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһӣбҹҶбһҠбһ¶бһ”бҹӢбһҒбҹ’бһ–бһҹбҹӢ
- бһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҮбһ¶ Code бһҮбһ¶бһҳбһҪбһҷ Bicep
- бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ•бҹ’бһҠбһӣбҹӢбһ’бһ“бһ’бһ¶бһ“
- бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹҶбһ–бһ„бҹӢбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“

- бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё container бһҮбһ¶бһҳбһҪбһҷ Azure Container Apps бһ“бһ·бһ„ AZD

#### бһ’бһ“бһ’бһ¶бһ“бһҹбһ·бһҖбҹ’бһҹбһ¶

бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё multi-service бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бҹҶбһ–бһ„бҹӢбһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“

### бһҮбҹҶбһ–бһјбһҖ бҹҘбҹ– бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ AI бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бҹ’бһҡбһҫбһ“ (бҹў-бҹЈ бһҳбҹүбҹ„бһ„) рҹӨ–рҹӨ–
**бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҳбһ»бһ“**бҹ– бһҮбҹҶбһ–бһјбһҖ бҹЎ-бҹў бһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢ  
**бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү**бҹ– вӯҗвӯҗвӯҗвӯҗ

#### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҹбһ·бһҖбҹ’бһҹбһ¶
- бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһ—бҹҒбһ‘бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бҹ’бһҡбһҫбһ“
- бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶ бһ“бһ·бһ„бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһ“бҹғбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“
- бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫ AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ
- бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһўбһҸбһ·бһҗбһ·бһҮбһ“ бһ“бһ·бһ„бһҹбһ¶бһҡбһ–бһҫбһ—бһҺбҹ’бһҢ

- бһ”бһүбҹ’бһ…бһјбһӣ microservices бһҮбһ¶бһҖбһ»бһ„бһҸбһәбһ“бҹҗбһҡбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһ“бҹғбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бҹ’бһҡбһҫбһ“

#### бһ’бһ“бһ’бһ¶бһ“бһҹбһ·бһҖбҹ’бһҹбһ¶
- [бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ Multi-Agent бһӣбһҖбҹӢбһҡбһ¶бһҷ](examples/retail-scenario.md) - бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ–бҹҒбһүбһӣбҹҒбһү
- [бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё ARM Template](../../examples/retail-multiagent-arm-template) - бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһҠбҹ„бһҷбһҲбҹ’бһҳбҹ„бҹҮбһ…бһ»бһ…бһҸбҹӮбһҳбһҪбһҷ
- бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бҹ’бһҡбһҫбһ“
- [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ Microservices Architecture](../../examples/container-app/microservices) - бһҖбһ¶бһҡбһ”бҹ’бһҡбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҹбҹҒбһңбһ¶ бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһўбһ¶бһҹбһёбһ„ бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ

#### бһӣбһ‘бҹ’бһ’бһ•бһӣбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„
бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ AI ready бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бҹ’бһҡбһҫбһ“

### бһҮбҹҶбһ–бһјбһҖ бҹҰбҹ– бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ бһ“бһ·бһ„бһ•бҹӮбһ“бһҖбһ¶бһҡбһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһ“бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫ (бҹЎ бһҳбҹүбҹ„бһ„) рҹ”Қ
**бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҳбһ»бһ“**бҹ– бһҮбҹҶбһ–бһјбһҖ бҹӨ бһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢ  
**бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү**бҹ– вӯҗвӯҗ

#### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҹбһ·бһҖбҹ’бһҹбһ¶
- бһ•бҹӮбһ“бһҖбһ¶бһҡбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– бһ“бһ·бһ„бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ’бһ“бһ’бһ¶бһ“
- бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ SKU
- бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһ»бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бһ·бһ„бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ
- бһ•бҹӮбһ“бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҗбҹ’бһӣбҹғбһҠбһҫбһҳ

#### бһ’бһ“бһ’бһ¶бһ“бһҹбһ·бһҖбҹ’бһҹбһ¶
- [бһ•бҹӮбһ“бһҖбһ¶бһҡбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–](docs/pre-deployment/capacity-planning.md) - бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ’бһ“бһ’бһ¶бһ“
- [бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ SKU](docs/pre-deployment/sku-selection.md) - бһҮбһҳбҹ’бһҡбһҫбһҹбһҠбҹӮбһӣбһҳбһ¶бһ“бһҗбҹ’бһӣбҹғбһҹбһҳбһҡбһҳбҹ’бһҷ
- [бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһ»бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ](docs/pre-deployment/preflight-checks.md) - script бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·
- [бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Application Insights](docs/pre-deployment/application-insights.md) - бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһҹбһ„бҹ’бһҖбҹҒбһҸ
- [бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бҹ’бһҡбһҫбһ“](docs/pre-deployment/coordination-patterns.md) - бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ

#### бһӣбһ‘бҹ’бһ’бһ•бһӣбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„
бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫ бһҳбһ»бһ“бһ–бҹҒбһӣбһўбһ“бһ»бһңбһҸбҹ’бһҸ

### бһҮбҹҶбһ–бһјбһҖ бҹ§бҹ– бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶ бһ“бһ·бһ„бһҠбһјбһҖбһҖбһјбһҠбһҖбҹҶбһ бһ»бһҹ (бҹЎ-бҹЎ.бҹҘ бһҳбҹүбҹ„бһ„) рҹ”§
**бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“**бҹ– бһҳбҹҒбһҡбҹҖбһ“бһ…бҹӮбһҖбһ…бһ¶бһҷбһҺбһ¶бһҳбһҪбһҷбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢ  
**бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү**бҹ– вӯҗвӯҗ

#### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“
- бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ”бһүбҹ’бһҮбҹ’бһҡбһ¶бһҖбҹӢбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ”бҹӮбһ”бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’
- бһ”бһүбҹ’бһ бһ¶бһ‘бһјбһ‘бҹ… бһ“бһ·бһ„бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ
- бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһҸбҹӮбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ AI
- бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһ“бһ‘бһ·бһ“бҹ’бһ“бһ•бһӣ

#### бһ”бҹ’бһҡбҹүбҹ„бһ—бһҳбҹҒбһҡбҹҖбһ“
- [бһ”бһүбҹ’бһ бһ¶бһ‘бһјбһ‘бҹ…](docs/troubleshooting/common-issues.md) - бһҹбҹҶбһ“бһҪбһҡбһүбһ№бһҖбһүбһ¶бһ”бҹӢ бһ“бһ·бһ„бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ
- [бһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бһҡбһҺбҹҚбһ”бһүбҹ’бһҮбҹ’бһҡбһ¶бһҖбҹӢбһ‘бһҳбҹ’бһҡбһ„бҹӢ](docs/troubleshooting/debugging.md) - бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҮбҹҶбһ бһ¶бһ“бҹӢбһҮбҹҶбһ бһ¶бһ“бҹӢ
- [бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ AI](docs/troubleshooting/ai-troubleshooting.md) - бһ”бһүбҹ’бһ бһ¶бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ AI

#### бһӣбһ‘бҹ’бһ’бһ•бһӣбһ•бҹ’бһ“бҹӮбһҖбһўбһ“бһ»бһңбһҸбҹ’бһҸ
бһўбһ¶бһ…бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғ бһ“бһ·бһ„бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ…бҹӮбһҖбһ…бһ¶бһҷбһ‘бһјбһ‘бҹ…бһҠбҹ„бһҷбһЎбҹӮбһҖ

### бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 8бҹ– бһӣбҹҶбһ“бһ¶бҹҶбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһңбһ·бһҹбҹҗбһҷбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ (2-3 бһҳбҹүбҹ„бһ„) рҹҸў

**бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү**бҹ– вӯҗвӯҗвӯҗвӯҗ

#### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“
- бһҷбһ»бһ‘бҹ’бһ’бһ“бһ¶бһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ
- бһӣбҹҶбһ“бһ¶бҹҶбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ
- бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҗбҹ’бһӣбҹғбһҠбһҫбһҳ
- бһҖбһ¶бһҡбһҸбҹҶбһҺбһҖбҹӢбһҸбҹҶбһҺбһ¶бһ„ бһ“бһ·бһ„бһҡбһҠбҹ’бһӢбһ”бһ¶бһӣ

- бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһӣбҹ’бһўбһҘбһҸбһҒбҹ’бһ…бҹ„бҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё container app (бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–, бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ, бһҗбҹ’бһӣбҹғбһҠбһҫбһҳ, CI/CD)

#### бһ”бҹ’бһҡбҹүбҹ„бһ—бһҳбҹҒбһҡбҹҖбһ“
- [бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ AI бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ](docs/microsoft-foundry/production-ai-practices.md) - бһӣбҹҶбһ“бһ¶бҹҶбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ
- бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ Microservices бһ“бһ·бһ„бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ
- бһҹбҹҠбһ»бһҳбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһҡбһҠбҹ’бһӢбһ”бһ¶бһӣ
- [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ Microservices](../../examples/container-app/microservices) - бһ…бҹӮбһҖбһ…бһ¶бһҷбһ”бҹ’бһӣбһјбһЎбһј/бһ бҹ’бһӮбҹ’бһҡбһёбһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“бһ…бҹӮбһҖбһ…бһ¶бһҷ, бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҗбҹ’бһӣбҹғбһҠбһҫбһҳ

#### бһӣбһ‘бҹ’бһ’бһ•бһӣбһ•бҹ’бһ“бҹӮбһҖбһўбһ“бһ»бһңбһҸбҹ’бһҸ
бһ…бҹӮбһҖбһ…бһ¶бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһ–бҹҒбһүбһӣбҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ

## бһҖбҹҶбһҺбһҸбҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҡбҹҖбһ“ бһ“бһ·бһ„бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү

### бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҮбҹҶбһ“бһ¶бһүбһҮбһ¶бһ”бһ“бҹ’бһ‘бһ»бҹҮбһ”бһ“бҹ’бһҠ

- **рҹҢұ бһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ**бҹ– бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 1 (бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“) вҶ’ бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 2 (бһўбһ—бһ·бһңбһҢбҹ’бһҚ AI)
- **рҹ”§ бһҳбһ’бҹ’бһҷбһҳ**бҹ– бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 3-4 (бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ & бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’) вҶ’ бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 6 (бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ)
- **рҹҡҖ бһҮбҹҶбһ“бһ¶бһүбһҒбҹ’бһ–бһҹбҹӢ**бҹ– бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 5 (бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ Multi-Agent) вҶ’ бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 7 (бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶)
- **рҹҸў бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ**бҹ– бһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ, бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 8 (бһӣбҹҶбһ“бһ¶бҹҶбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ)

- **бһ•бҹ’бһӣбһјбһңбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Container App**бҹ– бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 4 (бһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷ Containerized), 5 (бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Microservices), 8 (бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ)

### бһҹбһүбҹ’бһүбһ¶бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү

- **вӯҗ бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“**бҹ– бһӮбҹҶбһ“бһ·бһҸбһҸбҹӮбһҳбһҪбһҷ, бһҳбҹҒбһҡбҹҖбһ“бһҺбҹӮбһ“бһ¶бҹҶ, 30-60 бһҳбҹ„бһ„бһ‘бһ¶бһҸбҹӢ
- **вӯҗвӯҗ бһҳбһ’бҹ’бһҷбһҳ**бҹ– бһӮбҹҶбһ“бһ·бһҸбһ…бҹ’бһҡбһҫбһ“, бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹ„бһҷбһҠбҹғ, 1-2 бһҳбҹүбҹ„бһ„  
- **вӯҗвӯҗвӯҗ бһҒбҹ’бһ–бһҹбҹӢ**бҹ– бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү, бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бҹ’бһҠбһјбһҡ, 1-3 бһҳбҹүбҹ„бһ„
- **вӯҗвӯҗвӯҗвӯҗ бһ–бһ·бһҹбҹҒбһҹ**бҹ– бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ, бһӣбҹҶбһ“бһ¶бҹҶбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ, 2-4 бһҳбҹүбҹ„бһ„

### бһ•бҹ’бһӣбһјбһңбһҡбҹҖбһ“бһ”бһҸбҹӢбһ”бҹӮбһ“бһ”бһ¶бһ“

#### рҹҺҜ бһ•бҹ’бһӣбһјбһңбһҡбһ бҹҗбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚ AI (4-6 бһҳбҹүбҹ„бһ„)
1. **бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 1**бҹ– бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ“бһ·бһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҷбҹүбһ¶бһ„бһҡбһ бҹҗбһҹ (45 бһ“бһ¶бһ‘бһё)
2. **бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 2**бҹ– бһўбһ—бһ·бһңбһҢбҹ’бһҚ AI бһҮбһ¶бһ…бһҳбҹ’бһ”бһ„ (2 бһҳбҹүбҹ„бһ„)  
3. **бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 5**бҹ– бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ Multi-Agent AI (3 бһҳбҹүбҹ„бһ„)
4. **бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 8**бҹ– бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ AI бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ (1 бһҳбҹүбҹ„бһ„)

#### рҹӣ пёҸ бһ•бҹ’бһӣбһјбһңбһўбҹ’бһ“бһҖбһҮбҹҶбһ“бһ¶бһүбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ (5-7 бһҳбҹүбҹ„бһ„)
1. **бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 1**бҹ– бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ“бһ·бһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҷбҹүбһ¶бһ„бһҡбһ бҹҗбһҹ (45 бһ“бһ¶бһ‘бһё)
2. **бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 3**бҹ– бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺ (1 бһҳбҹүбҹ„бһ„)
3. **бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 4**бҹ– бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҮбһ¶бһўбһҖбҹ’бһҹбһҡ бһ“бһ·бһ„бһ…бҹӮбһҖбһ…бһ¶бһҷ (1.5 бһҳбҹүбҹ„бһ„)
4. **бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 6**бҹ– бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһ»бһ“бһ…бҹӮбһҖбһ…бһ¶бһҷ бһ“бһ·бһ„бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡ (1 бһҳбҹүбҹ„бһ„)
5. **бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 7**бҹ– бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶ бһ“бһ·бһ„бһ”бһүбҹ’бһҮбҹ’бһҡбһ¶бһҖбҹӢбһ‘бһҳбҹ’бһҡбһ„бҹӢ (1.5 бһҳбҹүбҹ„бһ„)
6. **бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 8**бҹ– бһӣбҹҶбһ“бһ¶бҹҶбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ (2 бһҳбҹүбҹ„бһ„)

#### рҹҺ“ бһ•бҹ’бһӣбһјбһңбһҡбҹҖбһ“бһ–бҹҒбһүбһӣбҹҒбһү (8-12 бһҳбҹүбҹ„бһ„)
бһ”бһүбҹ’бһ…бһ”бҹӢбһҮбһ¶бһҸбһ¶бһҡбһјбһ„бһ‘бһ¶бҹҶбһ„ 8 бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹ„бһҷбһҠбҹғ бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ

## бһҹбҹҠбһ»бһҳбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“

### бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„
- **бһ…бҹҶбһҺбһ»бһ…бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбҹҒбһҡбҹҖбһ“**бҹ– бһҖбҹҶбһ бһ·бһҸбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҳбһҪбһҷбһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһңбһ¶бһҹбҹӢбһңбҹӮбһ„бһ”бһ¶бһ“
- **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҠбҹ„бһҷбһҠбҹғ**бҹ– бһ…бҹӮбһҖбһ…бһ¶бһҷбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбһ¶бһҡбһ„бһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһҡбҹҖбһ„бҹ—бһҒбҹ’бһӣбһҪбһ“

### бһҹбһүбҹ’бһүбһ¶бһ”бһҸбҹ’бһҡбһ“бһ·бһ„бһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһҹбҹ’бһӮбһ¶бһӣбҹӢ

- **бһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһҹбҹ’бһӮбһ¶бһӣбҹӢбһ–бһёбһҹбһ бһӮбһҳбһ“бҹҚ**бҹ– бһ…бһјбһӣбһҡбһҪбһҳбһҹбһҖбһҳбҹ’бһҳбһҖбҹ’бһ“бһ»бһ„ Microsoft Foundry Discord
- **бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲ**бҹ– бһҮбҹҶбһ“бһ¶бһү AZD бһ“бһ·бһ„ AI бһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ§бһҹбҹ’бһҹбһ¶бһ бһҖбһҳбҹ’бһҳ
- **бһҖбһ¶бһҡбһҖбһҫбһ“бһЎбһҫбһ„бһҖбҹ’бһ“бһ»бһ„бһўбһ¶бһҮбһёбһ–**бҹ– бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ…бҹӮбһҖбһ…бһ¶бһҷбһ–бһ–бһҖбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ

### бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ (бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 1-2)  
бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһўбҹ’бһ“бһҖбһҡбҹҖбһ“бһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбҹ–

**бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбҹ–**
- бһ…бҹӮбһҖбһ…бһ¶бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҸбһ¶бһҳ AZD
- бһҖбҹҶбһҺбһҸбҹӢ бһ“бһ·бһ„бһ…бҹӮбһҖбһ…бһ¶бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбһҮбҹӮбһҖ AI бһҮбһ¶бһҳбһҪбһҷ RAG
- бһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһӮбҹҶбһ“бһ·бһҸ AZD бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“: бһӮбҹҶбһҡбһј, бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ, бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһ’бһ“бһ’бһ¶бһ“
- бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Microsoft Foundry бһҮбһ¶бһҳбһҪбһҷ AZD
- бһҡбҹҖбһ”бһ…бҹҶбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ AI Azure бһ“бһ·бһ„бһ…бҹҶбһҺбһ»бһ…бһ…бһјбһӣ API

**бһҮбҹҶбһ“бһ¶бһүбһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲбҹ–**
- бһҸбһ¶бһҳбһҠбһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһҡбһ”бҹҖбһ”
- бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ•бҹ’бһ“бҹӮбһҖбһ…бҹӮбһҖбһ…бһ¶бһҷбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҮбһ¶бһҳбһҪбһҷбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ» бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡ
- бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһўбҹҶбһ–бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷбһ–бһ–бһҖбһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹҶбһ“бһ¶бҹҶбһӣбҹ’бһўбһ…бҹҶбһ–бҹ„бҹҮбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ AI бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–

**бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһҡбҹҖбһ“бҹ–**
- вң… бһ…бҹӮбһҖбһ…бһ¶бһҷбһӮбҹҶбһҡбһј `todo-nodejs-mongo` бһ”бһ¶бһ“бһҮбҹ„бһӮбһҮбҹҗбһҷ
- вң… бһ…бҹӮбһҖбһ…бһ¶бһҷ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢ `azure-search-openai-demo` бһҮбһ¶бһҳбһҪбһҷ RAG
- вң… бһ”бһүбҹ’бһ…бһ”бҹӢбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһңбһӮбҹ’бһӮбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳ (бһҮбҹҶбһ бһ¶бһ“бһҹбҹ’бһңбҹӮбһ„бһҡбһҖ)
