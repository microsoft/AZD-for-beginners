# AZD бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ AI

бһҹбһјбһҳбһҹбҹ’бһңбһ¶бһӮбһҳбһ“бҹҚвҖӢбһҳбһҖбһҖбһ¶бһ“бҹӢвҖӢбһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҒбҹҮ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбҹҖбһ“вҖӢбһўбҹҶбһ–бһё Azure Developer CLI (AZD) бһҮбһ¶бһҳбһҪбһҷвҖӢбһҖбһ¶бһҡвҖӢбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫвҖӢбһҖбһ¶бһҡвҖӢбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AIбҹ” бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһ“бҹҒбҹҮвҖӢбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ…бҹҶбһ–бҹ„бҹҮбһ‘бһҳбҹ’бһҡбһ„бҹӢ AZD бһҸбһ¶бһҳ 3 бһҮбҹҶбһ бһ¶бһ“бҹ–

1. бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ - бһҡбһҖбһ‘бһҳбҹ’бһҡбһ„бҹӢбһҠбҹӮбһӣбһҹбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбҹ”
1. бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһЎбһҫбһ„ - бһҠбһ¶бһҖбҹӢбһЎбһҫбһ„ бһ“бһ·бһ„бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҗбһ¶бһңбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ”
1. бһҖбһ¶бһҡбһҖбҹӮбһҸбһҳбҹ’бһҡбһјбһң - бһҖбҹӮбһ”бҹ’бһҡбҹӮ бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһҹбҹ’бһңбҹӮбһ„бһҮбһјбһҸ бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһңбһ¶бһҮбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ!

Over the course of this workshop, you will also be introduced to core developer tools and workflows, to help you streamline your end-to-end development journey.
