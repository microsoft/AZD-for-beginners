# AZD សម្រាប់អ្នកអភិវឌ្ឍ AI សិក្ខាសាលា

> Validated against `azd 1.23.12` in March 2026.

> [!IMPORTANT]  
> **សិក្ខាសាលានេះមានណែនាំសិក្ខាសាលាមួយដែលអ្នកអាចមើលជាមុនក្នុងកម្មវិធីរុករកនៅក្នុងម៉ាស៊ីនរបស់អ្នក។ ដើម្បីចាប់ផ្តើម បើក repo ក្នុង GitHub Codespaces, dev container, ឬ local clone ដែលមាន MkDocs ដំឡើង—បន្ទាប់មករង់ចាំរហូតដល់អ្នកឃើញ terminal VS Code ដែលកំពុងដំណើរការ និងវាយ:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **អ្នកគួរតែឃើញប្រអប់លេចចេញសម្រាប់បើកទំព័រមើលជាមុននៅក្នុងកម្មវិធីរុករក។**

ស្វាគមន៍មកកាន់សិក្ខាសាលាដោយអនុវត្តសម្រាប់រៀន Azure Developer CLI (AZD) ដែលផ្តោតលើការដាក់បញ្ចូលកម្មវិធី AI។ សិក្ខាសាលានេះជួយអ្នកទទួលបានការយល់ដឹងអនុវត្ត​អំពីគំរូ AZD ក្នុង 3 ជំហាន៖

1. **ការស្វែងរក** - ស្វែងរកគំរូដែលសមស្របសម្រាប់អ្នក។
1. **ការដាក់បញ្ចូល** - ដាក់បញ្ចូល និងបញ្ជាក់ថាវាដំណើរការ។
1. **ការកែតម្រូវ** - កែប្រែ និងធ្វើចំណាប់ដើម្បីធ្វើឲ្យវាក្លាយជារបស់អ្នក!

នៅក្នុងអំឡុងសិក្ខាសាលានេះ អ្នកនឹងទទួលបានការណែនាំទាក់ទងនឹងឧបករណ៍អភិវឌ្ឍន៍មូលដ្ឋាន និងលំហូរការងារ ដើម្បីជួយលោកអ្នកធ្វើឲ្យដំណើរការអភិវឌ្ឍចាប់ពីដើមដល់ចុងរបស់អ្នករលូន។

## ទិដ្ឋភាពទូទៅនៃសិក្ខាសាលា

**រយៈពេល៖** 3-4 ម៉ោង  
**កម្រិត៖** ចាប់ផ្តើមទៅមធ្យម  
**លក្ខខណ្ឌមុន៖** មានសំនឹងអំពី Azure, គំនិត AI, VS Code និងឧបករណ៍ command-line។

នេះជាសិក្ខាសាលាដែលធ្វើដោយអនុវត្ត ដែលអ្នករៀនតាមការធ្វើ។ ពេលដែលអ្នកបានបញ្ចប់លំហាត់ទាំងអស់ យើងផ្តល់អនុសាសន៍ឲ្យពិនិត្យមើលមាតិកា AZD សម្រាប់អ្នកចាប់ផ្តើម ដើម្បីបន្តដំណើរសិក្សារបស់អ្នកទៅកាន់ការអនុវត្តប្រព័ន្ធសុវត្ថិភាព និងផលិតភាពដែលល្អបំផុត។

| Time| Module  | Objective |
|:---|:---|:---|
| 15 mins | Introduction | ដាក់បរិបទ និងយល់ពីគោលបំណង |
| 30 mins | Select AI Template | រុករកជម្រើស និងជ្រើសគំរូចាប់ផ្តើម | 
| 30 mins | Validate AI Template | ដាក់ដំណើរការដំណោះស្រាយលំនាំដើមទៅ Azure |
| 30 mins | Deconstruct AI Template | រុករករចនាសម្ព័ន្ធ និងការកំណត់ |
| 30 mins | Configure AI Template | បើកដំណើរការ និងសាកល្បងលក្ខណៈដែលមាន |
| 30 mins | Customize AI Template | កែតម្រូវគំរូឲ្យសមរម្យតាមតម្រូវការ |
| 30 mins | Teardown Infrastructure | សម្អាត និងដោះសោធនធាន |
| 15 mins | Wrap-Up & Next Steps | ធនធានសម្រាប់រៀន និងបញ្ហាសិក្ខាសាលា |
| | |

## អ្វីដែលអ្នកនឹងរៀន

គិតអំពីគំរូ AZD ជាផ្ទាំងលេងសម្រាប់រៀន ដើម្បីស្វែងរកសមត្ថភាព និងឧបករណ៍នានាសម្រាប់ការអភិវឌ្ឍពីដើមដល់ចុងលើ Microsoft Foundry។ នៅចុងសិក្ខាសាលានេះ អ្នកគួរតែមានការយល់ដឹងយ៉ាងយូណាអំពីឧបករណ៍ និងគំនិតនានាមាននៅក្នុងបរិបទនេះ។

| Concept  | Objective |
|:---|:---|
| **Azure Developer CLI** | យល់ដឹងអំពីពាក្យបញ្ជា និងលំហូរការងាររបស់ឧបករណ៍ |
| **AZD Templates**| យល់ដឹងអំពីរចនាសម្ព័ន្ធគម្រោង និងការកំណត់ |
| **Azure AI Agent**| ផ្តល់ និងដាក់បញ្ចូលគម្រោង Microsoft Foundry  |
| **Azure AI Search**| អនុញ្ញាតវិស្វកម្មបរិបទជាមួយភ្នាក់ងារ |
| **Observability**| រុករកការតាមដាន, ការត្រួតពិនិត្យ និងការវាយតម្លៃ |
| **Red Teaming**| ស្វែងរកការធ្វើតេស្តប្រឆាំង និងវិធានការការពារ |
| | |
