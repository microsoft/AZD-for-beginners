# AZD សម្រាប់ស développeurs AI សិក្ខាសាលា

> បានផ្ទៀងផ្ទាត់ជាមួយ `azd 1.27.1` នៅខែកក្កដា ២០២៦។

> [!IMPORTANT]  
> **សិក្ខាសាលានេះត្រូវបានរៀបចំជាមួយមគ្គុទេសក៍សិក្ខាសាលាដែលអ្នកអាចមើលមុនក្នុងកម្មវិធីរុករកក្នុងកន្លែងរបស់អ្នក។ ដើម្បីចាប់ផ្តើម បើក repo នៅក្នុង GitHub Codespaces, dev container, ឬ local clone ដែលបានដំឡើង MkDocs—បន្ទាប់មករង់ចាំរហូតដល់អ្នកឃើញប្រអប់ប៊ូបបង្ហាញ VS Code ដែលសកម្ម ហើយវាយ:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **អ្នកគួរតែឃើញបង្អួចបង្ហាញដើម្បីបើកទំព័រមើលមុននៅក្នុងកម្មវិធីរុករក។**

សូមស្វាគមន៍មកកាន់សិក្ខាសាលាផ្ទាល់ខ្លួនសម្រាប់រៀន Azure Developer CLI (AZD) ដែលផ្តោតលើការដាក់ពាក្យកម្មវិធី AI។ សិក្ខាសាលានេះជួយអ្នកទទួលបានការយល់ដឹងអំពីការប្រើប្រាស់ AZD តាងការងារជាមួយខ្នាត១:  

1. **ស្វែងរក** - រកតាងការងារដែលសាកសមសម្រាប់អ្នក។
1. **ដាក់ពាក្យកម្មវិធី** - ដាក់ពាក្យកម្មវិធី និងផ្ទៀងផ្ទាត់ថាវាដំណើរការ
1. **ប្ដូរតាងកា្លការងារ** - កែប្រែ និងបន្តប្រើប្រាស់ដើម្បីធ្វើឲ្យវាជារបស់អ្នក!

នៅក្នុងវគ្គសិក្ខាសាលានេះ អ្នកនឹងបានណែនាំអំពីឧបករណ៍ និងដំណើរការសម្រាប់អ្នកអភិវឌ្ឍន៍ ដើម្បីជួយធ្វើឲ្យដំណើរអភិវឌ្ឍមានភាពរលូនពីដើមដល់ចប់។

## សេចក្ដីសង្ខេបសិក្ខាសាលា

**រយៈពេល:** ៣-៤ ម៉ោង  
**កម្រិត:** ចាប់ផ្តើម ដល់ មធ្យម  
**លក្ខខ័ណ្ឌជាមុន:** មានផ្ទាល់ក្នុង Azure, គន្លឹះ AI, VS Code និងឧបករណ៍ command-line ។

នេះជាសិក្ខាសាលាផ្ទាល់ខ្លួនដែលអ្នករៀនតាមការធ្វើ។ បន្ទាប់ពីបញ្ចប់លំហាត់ សូមណែនាំអោយពិនិត្យមើលកម្មវិធី AZD For Beginners ដើម្បីបន្តដំណើររៀនទៅរកការពារជាផ្នែកសុវត្ថិភាព និងប្រសិទ្ធភាព។

| ពេលវេលា| ម៉ូឌុល  | គោលដៅ |
|:---|:---|:---|
| ១៥ នាទី | សម្រុក | កំណត់វេទិកា, យល់ពីគោលដៅ |
| ៣០ នាទី | ជ្រើសរើសតាងការងារ AI | ស្វែងរកជម្រើស និងជ្រើសរើសសម្រាប់ចាប់ផ្តើម | 
| ៣០ នាទី | ផ្ទៀងផ្ទាត់តាងការងារ AI | ដាក់ពាក្យកម្មវិធីលំនាំដើមទៅ Azure |
| ៣០ នាទី | បំបែកតាងការងារ AI | ស្វែងរករចនាសម្ព័ន្ធ និងការកំណត់ |
| ៣០ នាទី | កំណត់តាងការងារ AI | បើកអនុគមន៍ និងសាកល្បងមុខងារមានស្រាប់ |
| ៣០ នាទី | ប្ដូរតាងការងារ AI | ប្ដូរតាងការងារឲ្យសមស្របជាមួយតម្រូវការ |
| ៣០ នាទី | សម្អាតសេវាផ្ទុក | សម្អាត និងចេញធនធាន |
| ១៥ នាទី | បិទវគ្គ និង ជំហានបន្ទាប់ | ប្រភពរៀន, ការប្រកួតសិក្ខាសាលា |
| | |

## អ្វីដែលអ្នកនឹងរៀន

គិតពី AZD តាងការងារជាទីលំហររៀនសម្រាប់ស្វែងរកមុខងារ និងឧបករណ៍ផ្សេងៗសម្រាប់អភិវឌ្ឍពីដើមដល់ចប់លើ Microsoft Foundry។ នៅចុងបញ្ចប់សិក្ខាសាលានេះ អ្នកគួរតែមានអារម្មណ៍ច្បាស់លាស់អំពីឧបករណ៍ និងគំនិតក្នុងបរិបទនេះ។

| គំនិត  | គោលដៅ |
|:---|:---|
| **Azure Developer CLI** | យល់ពីការបញ្ជា និងដំណើរការឧបករណ៍|
| **តាងការងារ AZD**| យល់ពីរចនាសម្ព័ន្ធគម្រោង និងការកំណត់|
| **ភេជក AI Azure**| រៀបចំ និងដាក់ពាក្យកម្មវិធី Microsoft Foundry |
| **ស្វែងរក AI Azure**| បើកប្រើវិស្វកម្មបរិបទជាមួយភេជក |
| **ការវិភាគមើល**| ស្វែងរកការតាមដាន, ត្រួតពិនិត្យ និងការវាយតម្លៃ |
| **ក្រុមក្រហម**| ស្វែងរកការប្រឡងប្រឆាំង និងការការពារ |
| | |
