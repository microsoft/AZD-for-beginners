| Module | Accomplishment |
|:---|:---|
| **ម៉ូឌុល 1** | បានស្វែងរក Azure Developer CLI និងបានជ្រើសពុម្ព AI មួយ |
| **ម៉ូឌុល 2** | បានដាក់បញ្ចូល និងផ្ទៀងផ្ទាត់ដំណោះស្រាយភ្នាក់ងារ AI ទាំងមូល |

| **ម៉ូឌុល 5** | បានប្តូរពុម្ពឱ្យសមស្របសម្រាប់ករណីរបស់អ្នក |
| **ម៉ូឌុល 6** | បានបំបាក់ហេដ្ឋារចនាសម្ព័ន្ធយ៉ាងសុវត្ថិភាព និងគ្រប់គ្រងធនធាន |

## 2. គំនិតសំខាន់ៗដែលបានយល់

### Azure Developer CLI (azd)
- `azd init` - ចាប់ផ្តើមគម្រោងពីពុម្ព
- `azd up` - ផ្តល់ធនធាន និងដាក់បញ្ចូលក្នុងពាក្យបញ្ជាតែមួយ
- `azd env` - គ្រប់គ្រងអថេរបរិយាកាស
- `azd down` - សម្អាតធនធានយ៉ាងសុវត្ថិភាព

### AZD Template Structure
- `azure.yaml` - ការកំណត់ និង deployment hooks
- `infra/` - Bicep infrastructure-as-code
- `src/` - កូដប្រភពកម្មវិធី
- `scripts/` - ស្គ្រីបស្វ័យប្រវត្តិ និងសាច់ការដំឡើង

### Microsoft Foundry Capabilities
- សេវាភ្នាក់ងារ AI សម្រាប់ជំនួយឆ្លាត
- File Search និង Azure AI Search សម្រាប់ស្វែងរកចំណេះដឹង
- ការតាមដាន ថតសំគាល់ និងការវាយតម្លៃដែលបញ្ចូលក្នុងខ្លួន
- Red teaming សម្រាប់ផ្ទៀងផ្ទាត់សុវត្ថិភាព AI

## 4. លំហាត់សិក្ខាសាលា

រួចរាន់ដើម្បីសាកល្បងជំនាញរបស់អ្នកទេ? សាកល្បងលំហាត់ទាំងនេះ៖

| Challenge | Description | Difficulty |
|:---|:---|:---|
| **1. ស្ថានភាពភ្នាក់ងារច្រើន** | ដាក់បញ្ចូល និងប្តូរតាមតម្រូវការសម្រាប់ឧស្សាហកម្មផ្សេង (ឧ. សុខាភិបាល, ហិរញ្ញវត្ថុ) | ⭐⭐ |
| **2. មូលដ្ឋានចំណេះដឹងផ្ទាល់ខ្លួន** | បន្ថែមឯកសារទិន្នន័យរបស់អ្នក និងបង្កើត search index ផ្ទាល់ខ្លួន | ⭐⭐⭐ |
| **3. ការដាក់បញ្ចូលផលិតកម្ម** | កំណត់ CI/CD ជាមួយ GitHub Actions ដោយប្រើ `azd pipeline config` | ⭐⭐⭐⭐ |
