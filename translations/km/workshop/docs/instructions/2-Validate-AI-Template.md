## 1. Introduction

The [Azure Developer CLI](https://learn.microsoft.com/en-us/azure/developer/azure-developer-cli/) or `azd` бһӮбһәбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ”бһҫбһҖбһ…бҹҶбһ бһҠбҹӮбһӣбһҮбһҪбһҷбһҹбһҳбҹ’бһҡбһҪбһӣбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸ бһ“бһ·бһ„бһҸбһҳбҹ’бһӣбһҫбһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бҹ… Azure бҹ”

[AZD Templates](https://learn.microsoft.com/azure/developer/azure-developer-cli/azd-templates) бһӮбһәбһҮбһ¶бһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһғбҹ’бһӣбһ¶бҹҶбһ„бһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһӣбһ„бҹӢбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ бһҠбҹӮбһӣбһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһјбһҠбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбҹҶбһҡбһј бһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ· _infrastructure-as-code_ бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡвҖӢбһҖбҹҶбһҺбһҸбҹӢвҖӢбһҖбһ»бһ„бһ бҹ’бһңбһёбһӮ `azd` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҳбһҪбһҷбҹ” бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһҹбһ¶бһҳбһүбҹ’бһүбһҠбһјбһ…бһҮбһ¶ command `azd provision` - бһҒбһҺбҹҲбһ–бҹҒбһӣбһҠбҹӮбһӣ `azd up` бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖ provision бһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ **бһ“бһ·бһ„** бһҠбһ¶бһҖбҹӢбһ–бһ¶бһҖбҹ’бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷ!

бһҠбҹ„бһҷбһҹбһ¶бһҡбҹ’бһ“бҹҒбҹҮ бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҹбһ¶бһҳбһүбҹ’бһүбһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ _AZD Starter template_ бһҠбҹӮбһӣбһҳбһ¶бһ“бһҒбҹ’бһӣбҹҮбһҹбҹ’бһҡбһҠбҹҖбһ„бһ‘бҹ…бһ“бһ·бһ„бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһ“бһ·бһ„бһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ - бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһғбҹ’бһӣбһ¶бҹҶбһ„бһұбҹ’бһҷбһҹбһҳбһҡбһҳбҹ’бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҒбһҺбһ¶бһҡбһёбһҷбҹүбһјбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһҳбһ»бһ“бһҷбһҫбһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһҳбһҖбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„ Azure Developer CLI бһҠбҹ„бһҷбһҡбһҪбһ…бҹ”

1. бһ”бһҫбһҖ terminal бһ“бҹ… VS Code бһ бһҫбһҷбһңбһ¶бһҷбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһ“бҹҒбҹҮ:

      ```bash title="" linenums="0"

      ```

1. бһўбһҗбҹҒбһҡбһҹбҹҗбһҖбҹ’бһҸбһ·бһҹбһ·бһ‘бҹ’бһ’бһ· (Environment variables) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„ `.azure/<env-name>/.env` - бһ“бҹҒбҹҮбһ”бҹӮбһ„бһ…бҹӮбһҖбһ–бһҪбһҖбһңбһ¶бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ `env-name` бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҠбһ¶бһҖбҹӢбһ–бһ¶бһҖбҹ’бһҷ бһ бһҫбһҷбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһ”бҹҶбһ”бҹӮбһҖбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ…бҹҶбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ…бҹҶбһҺбһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҮбһ¶бһҳбһҪбһҷбһӮбҹ„бһӣбһҠбҹ…бһҠбһ¶бһҖбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ“бҹ„бҹҮбҹ”

1. бһўбһҗбҹҒбһҡбһҹбҹҗбһҖбҹ’бһҸбһ·бһҹбһ·бһ‘бҹ’бһ’бһ·бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһҠбҹ„бһҷбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ `azd` бһ“бҹ…бһ–бҹҒбһӣбһңбһ¶бһ”бһүбҹ’бһ…бһ”бҹӢ command бһҹбҹ’бһңбһёбһ…бһ…бһҖбҹ’бһҒбһ»бһ“ (бһ§. `azd up`)бҹ” бһҹбһјбһҳбһ…бҹҶбһҺбһ¶бҹҶбһҗбһ¶ `azd` бһҳбһ·бһ“бһўбһ¶бһ“бһўбһҗбҹҒбһҡбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸ OS бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· (бһ§. бһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҖбҹ’бһ“бһ»бһ„ shell) - бһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһ“бҹҒбҹҮ бһ”бҹ’бһҡбһҫ `azd set env` бһ“бһ·бһ„ `azd get env` бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бҹҒбһҡбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„ scriptбҹ”
