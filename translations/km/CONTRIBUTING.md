# ចូលរួមចំណែកក្នុង AZD សម្រាប់អ្នកចាប់ផ្ដើម

សូមអរគុណចំពោះចំណាប់អារម្មណ៍របស់អ្នកក្នុងការចូលរួមចំណែកក្នុងមុខវិជ្ជា AZD សម្រាប់អ្នកចាប់ផ្ដើម! ឯកសារនេះផ្តល់នូវណែនាំ និងព័ត៌មានសម្រាប់អ្នកចូលរួមចំណែក។

## របៀបចូលរួម

### រាយការណ៍បញ្ហា

បើអ្នករកឃើញកំហុស, ខុសឆ្គង ឬមានយោបល់សម្រាប់ការកែលម្អ៖

1. **ពិនិត្យបញ្ហាមានរួច**: ស្វែងរក [បញ្ហាមានរួច](https://github.com/microsoft/azd-for-beginners/issues) ដើម្បីជៀសវាងភាពស្ទួន
2. **បង្កើតបញ្ហាថ្មី**: ប្រើគំរូបញ្ហាត្រឹមត្រូវ
3. **ផ្ដល់ព័ត៌មានលម្អិត**: រួមបញ្ចូលជំហានក្នុងការបង្កើតឡើងវិញ, អ្វីដែលរំពឹងទុក និងរូបភាពបង្ហាញប្រសិនបើអាច

### ការផ្តល់យោបល់ប្ដូរថ្មី

1. **ផFork ឃ្លាំងទិន្នន័យ**: បង្កើតfork ផ្ទាល់ខ្លួននៃគម្រោង
2. **បង្កើតសាខា**: បង្កើតសាខាសម្រាប់លក្ខណៈពិសេសនៃការផ្លាស់ប្ដូរ
3. **ធ្វើប្ដូរ**: បើកបរតាមស្តង់ដារកូដ និងឯកសារ
4. **សាកល្បងប្ដូររបស់អ្នក**: ប្រកាន់ទុកថា​តំណភ្ជាប់ទាំងអស់ដំណើរការបាន និងមាតិកាបង្ហាញមានត្រឹមត្រូវ
5. **ដាក់សំណើ Pull Request**: ផ្ដល់ការពណ៌នា​ច្បាស់លាស់អំពីការផ្លាស់ប្ដូរបស់អ្នក

### ប្រភេទការចូលរួម

យើងស្វាគមន៍ប្រភេទចូលរួមដូចខាងក្រោម៖

#### ការកែលម្អមាតិកា
- កែសម្រួលកំហុសអក្ខរកម្ម និងរចនាសម្ព័ន្ធ
- បង្កើនភាពច្បាស់លាស់នៃឯកសារមានរួច
- ធ្វើបច្ចុប្បន្នភាពពត៌មានចាស់
- បន្ថែមការពន្យល់ឬបរិបទដែលខ្វះ

#### ឧទាហរណ៍ថ្មី
- បន្ថែមស្ថានการณ์ចាក់ផ្ដាច់ពិភពលោកពិតប្រាកដ
- បង្កើតគំរូ AZD ថ្មីសម្រាប់ករណីការប្រើប្រាស់ទូទៅ
- ផ្ដល់ឧទាហរណ៍កូដបន្ថែម និងលំហាត់

#### គាំទ្របកប្រែ
- ជួយថែរក្សាបកប្រែមានរួច
- បង្កើនគុណភាពបកប្រែ
- បន្ថែមការបកប្រែសម្រាប់មាតិកាថ្មី

#### របាយការណ៍កំហុស
- រាយការណ៍តំណខូច ឬឧទាហរណ៍កូដខុស
- បញ្ជាក់ភាពមិនស្របគ្នានៃឯកសារ
- បង្កើតសញ្ញាបញ្ហាជាច្របល់ក្នុងការចូលប្រើ

## ទស្សនៈការចូលរួម

### ស្តង់ដារឯកសារ

- ប្រើភាសាច្បាស់លាស់ងាយស្រួលសម្រាប់អ្នកចាប់ផ្ដើម
- រួមបញ្ចូលគោលបំណង និងលទ្ធផលការរៀន
- ផ្ដល់ជំហានមួយៗជាមួយលទ្ធផលដែលរំពឹងទុក
- បន្ថែមតំណភ្ជាប់ទៅមាតិកាសមតុល្យ
- ប្រើរចនាសម្ព័ន្ធនិងស្តាយ markdown ដែលមានក្អាត

### ឧទាហរណ៍កូដ

- ធ្វើឲ្យប្រាកដថា ឧទាហរណ៍កូដទាំងអស់ត្រូវបានសាកល្បង និងដំណើរការ
- រួមបញ្ចូលមតិយោបល់ចំពោះគំនិតស្នូល
- តាមដានអនុសាសន៍ល្អៗរបស់ Azure និង AZD
- ផ្ដល់លទ្ធផលដែលរំពឹងទុកនៅកន្លែងដែលអាចអនុវត្ត

### សារ Commit

ប្រើសារ commit ដែលច្បាស់លាស់ និងពិពណ៌នាពីការផ្លាស់ប្ដូរ៖
- `docs: កែសម្រួលកំហុសអក្សរនៅម គ្រឹះការដំឡើង`
- `feat: បន្ថែមឧទាហរណ៍កម្មវិធីផ្ទុកទ្វារ​ថ្មី`
- `fix: ស្ដារតំណខូចនៅជំពូក ២`
