## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…

бһҮбҹҶбһ–бһјбһҖбһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһӣбһҫбһӣбҹҶбһ“бһ¶бҹҶбһҹбҹҶбһҺбһ„бҹӢбһҳбҹүбһјбһӣбҹ’бһ‘бһё-бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбҹҒбһёбһ”бҹ’бһҡбһ¶бһҹбҹӢ AI бһҖбҹ’бһ“бһ»бһ„бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбҹ”

> бһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҸбһ¶бһҳ `azd 1.23.12` бһ“бҹ…бһҒбҹӮбһҳбһ·бһ“бһ¶ бһҶбҹ’бһ“бһ¶бҹҶ 2026бҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбҹҖбһ“

бһҠбҹ„бһҷбһ”бһүбҹ’бһ…бһ”бҹӢбһҮбҹҶбһ–бһјбһҖбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бҹ–
- бһҷбһӣбҹӢбһ–бһёбһӣбҹҶбһ“бһ¶бҹҶбһҹбҹҶбһҺбһ„бҹӢбһҳбҹүбһјбһӣбҹ’бһ‘бһё-бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ
- бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбҹҒбһёбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһҠбҹӮбһӣбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһўбһ“бҹ’бһҸбһҡбһҖбһ¶бһҡбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ
- бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҳбҹүбһјбһӣбҹ’бһ‘бһё-бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹӮбһӣбһҹбҹ’бһҡбһ¶бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ

## рҹҡҖ бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҡбһ бҹҗбһҹ

```bash
# бһҮбһҳбҹ’бһҡбһҫбһҹ 1: бһ…бһ¶бһҖбҹӢбһ•бҹ’бһҹбһ¶бһҷбһ–бһёбһ‘бҹҶбһ–бҹҗбһҡбһӮбҹҶбһҡбһј

azd up

# бһҮбһҳбҹ’бһҡбһҫбһҹ 2: бһ…бһ¶бһҖбҹӢбһ•бҹ’бһҹбһ¶бһҷбһ–бһёбһҜбһҖбһҹбһ¶бһҡбһҸбҹҶбһҺбһ¶бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ (бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡвҖӢбһ”бһ“бҹ’бһҗбҹӮбһҳ azure.ai.agents)
azd extension install azure.ai.agents

azd up
```

> **бһҡбһ”бҹҖбһ”бһҺбһ¶?** бһ”бҹ’бһҡбһҫ `azd init --template` бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһӮбҹҶбһҡбһјбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбҹ” бһ”бҹ’бһҡбһҫ `azd ai agent init` бһ“бҹ…бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҳбһ¶бһ“ manifest бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҳбһҫбһӣ [бһҜбһҖбһҹбһ¶бһҡбһҷбҹ„бһ„ AZD AI CLI](../chapter-08-production/production-ai-practices.md#azd-ai-cli-commands-and-extensions) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһ”бҹҗбһҸбҹ’бһҡбһ–бҹҒбһүбһӣбҹҒбһүбҹ”

## рҹҺҜ бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ–бһ·бһҹбҹҒбһҹбҹ– бһҳбҹүбһјбһӣбҹ’бһ‘бһё-бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһӣбһҖбҹӢбһҡбһ¶бһҷ

[бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҳбҹүбһјбһӣбҹ’бһ‘бһё-бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһӣбһҖбҹӢбһҡбһ¶бһҷ](../../examples/retail-scenario.md) бһ”бһ„бҹ’бһ бһ¶бһүбҹ–

- **бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһўбһҸбһ·бһҗбһ·бһҮбһ“**: бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ·бһ„бһ…бҹҶбһҺбһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ
- **бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бҹҶбһ“бһ·бһү**: бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӮбҹ’бһҡбһҝбһ„бһҹбһ„бҹ’бһ бһ¶бһҡбһ·бһҳбһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһ‘бһ·бһү
- **бһўбҹ’бһ“бһҖбһҡбҹҖбһ”бһ…бҹҶбһҡбһҝбһ„**: бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҡбһңбһ¶бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ
