# бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 5бҹ– бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ AI бһ…бҹ’бһҡбһҫбһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ

**рҹ“ҡ бһңбһӮбҹ’бһӮбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣ**: [AZD бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ](../../README.md) | **вҸұпёҸ бһҡбһҷбҹҲбһ–бҹҒбһӣ**: 2-3 бһҳбҹүбҹ„бһ„ | **вӯҗ бһҖбһҳбҹ’бһҡбһ·бһҸбһ—бһ¶бһ–бһ…бһҳбҹ’бһӣбһ„бһ…бҹ’бһҡбһЎбҹҶ**: бһҒбҹ’бһ–бһҹбҹӢ

---

## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһӣбһҫбһҡбҹҖбһ”бһ…бҹҶбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ AI бһ…бҹ’бһҡбһҫбһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҒбһҺбһ¶бһҡбһёбһҷбҹүбһјбһҹбҹ’бһҳбһ»бһӮбһҳбһҒбҹ’бһ–бһҹбҹӢбҹ”

> бһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҠбҹ„бһҷ `azd 1.27.1` бһҖбҹ’бһ“бһ»бһ„бһҒбҹӮбһҖбһҖбҹ’бһҖбһҠбһ¶ бһҶбҹ’бһ“бһ¶бҹҶ 2026бҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбҹҖбһ“

бһҠбҹ„бһҷбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„:
- бһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһӣбҹҶбһ“бһ¶бҹҶбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ…бҹ’бһҡбһҫбһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ
- бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһҠбҹӮбһӣбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһӮбҹ’бһ“бһ¶
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһ–бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ
- бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ…бҹ’бһҡбһҫбһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ

## рҹҺҜ бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбҹҒбһ…бһ’бҹ’бһӣбҹ„бҹ– бһ…бҹ’бһҡбһҫбһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһўбҹ„бһҷбһҡбһ”бһҹбҹӢбһӣбһҖбҹӢбһҡбһ¶бһҷ

[бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ…бҹ’бһҡбһҫбһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһўбҹ„бһҷбһҡбһ”бһҹбҹӢбһӣбһҖбҹӢбһҡбһ¶бһҷ](../../examples/retail-scenario.md) бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбҹ–

- **бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһўбһҸбһ·бһҗбһ·бһҮбһ“**: бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһ“бһ·бһ„бһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ
- **бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ–бһ·бһҹбҹҒбһҹбһҡбһ”бһҹбҹӢбһҹбҹ’бһҸбһ»бһҖ**: бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҸбһ»бһҖ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһ‘бһ·бһү
- **бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ**: бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһ–бһёбһҡбһ¶бһ„бһҖбһ¶бһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ
