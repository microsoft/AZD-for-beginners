# бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ Multi-Agent - бһҠбһ¶бһҖбҹӢбһұбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһӣбһҫбһҖбһҠбҹҶбһ”бһјбһ„бһҠбҹӮбһӣбһҹбһ бһҖбһ¶бһҡбһӮбҹ’бһ“бһ¶

**бһҖбһ¶бһҡбһҡбһ»бһҖбһҡбһҖбһҮбҹҶбһ–бһјбһҖ:**
- **рҹ“ҡ бһ‘бҹҶбһ–бҹҗбһҡбһҳбҹҒбһңбһӮбҹ’бһӮ**: [AZD For Beginners](../../README.md)

- **вһЎпёҸ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ**: [бһӣбҹҶбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ](../chapter-06-pre-deployment/coordination-patterns.md)

## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҮбҹҶбһ–бһјбһҖбһҳбһ»бһ“бҹ— бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҸбҹӮбһҳбһҪбһҷвҖ”and бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҮбҹҶбһ–бһјбһҖ 2 бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһҸбҹӮбһҳбһҪбһҷбҹ” бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ“бһ¶бҹҶбһўбҹ’бһ“бһҖбһ‘бҹ…бһҮбҹҶбһ бҹҠбһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ– бһҠбһ¶бһҖбҹӢбһұбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ **бһ…бҹ’бһҡбһҫбһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ**, бһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбҹҶбһ“бһ¶бһүбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһ бһӮбҹ’бһ“бһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбһҳбһ·бһ“бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһӣбҹ’бһўбһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„бҹ”

бһҠбҹҶбһҺбһ№бһ„бһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбҹ– **бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҡбһҗбҹ’бһҳбһёбһ‘бҹҒбҹ”** бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ…бҹ’бһҡбһҫбһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ“бҹ…бһҸбҹӮбһҮбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ azd бһҳбһҪбһҷбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„ `azd init`, `azd up`, бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„, бһ“бһ·бһ„ `azd down`вҖ”бһҮбһ¶бһ…бҹ’бһҡбһҖбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡвҖӢбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҹбҹ’бһӮбһ¶бһӣбҹӢбһҡбһҪбһ…бҹ” бһўбҹ’бһңбһёбһҠбҹӮбһӣбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһӮбһә *бһҡбһ¶бһ„* бһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһ“бҹӮбһҖбһҖбҹ’бһ“бһ»бһ„бҹ”

## бһӮбҹ„бһӣбһҠбҹ…бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶

бһ“бҹ…бһ…бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ¶бһ“:
- бһҷбһӣбҹӢбһҗбһ¶ "multi-agent" бһҳбһ¶бһ“бһ“бҹҗбһҷбһўбҹ’бһңбһё бһ“бһ·бһ„бһ–бҹҒбһӣбһҺбһ¶бһңбһ¶бһҳбһ¶бһ“бһҸбҹҶбһӣбҹғбһ‘бҹ’бһңбҹҒбһҠбһ„бһ“бһ№бһ„бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ”бһ“бҹ’бһҗбҹӮбһҳ
- бһҹбҹ’бһӮбһ¶бһӣбҹӢбһҸбһҪбһ“бһ¶бһ‘бһёбһ‘бһјбһ‘бҹ…бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ…бҹ’бһҡбһҫбһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ (бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ + бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбҹҶбһ“бһ¶бһү)
- бһҠбһ¶бһҖбҹӢбһұбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһҳбҹ’бһҡбһ„бҹӢбһҹбҹҶбһҺбһ»бҹҶбһҳбһҪбһҷбһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһҖбһ¶бһҡ Multi-Agent бһҸбһ¶бһҳ `azd up`
- бһҷбһӣбҹӢбһ–бһёбһ’бһ“бһ’бһ¶бһ“ Azure бһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ…бҹ’бһҡбһҫбһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ
- бһҠбһ№бһ„бһңбһ·бһ’бһёбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ, бһ”бҹ’бһҠбһјбһҡ, бһ“бһ·бһ„бһӣбһ»бһ”бһ…бҹ„бһӣбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҷбҹүбһ¶бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–

## бһӣбһ‘бҹ’бһ’бһ•бһӣбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһҹбһјбһҳбһўбһ¶бһ…бҹ–
- бһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бһёбһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹӮбһҳбһҪбһҷ бһ“бһ·бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ…бҹ’бһҡбһҫбһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ
- бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҡбһңбһ¶бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ…бҹ’бһҡбһҫбһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ’бһҳбҹ’бһҳбһҸбһ¶
- бһҠбһ¶бһҖбҹӢбһұбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ“бһ·бһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ‘бһҳбҹ’бһҡбһ„бҹӢ multi-agent бһ–бһёбһҠбһҫбһҳбһ…бһ»бһ„бһҮбһ¶бһҳбһҪбһҷ azd
- бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҗбһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһҪбһҷбҹ—бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡвҖӢбһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһҺбһ¶ бһ“бһ·бһ„бһ–бһҪбһҖбһӮбҹҒбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһҖбһ¶бһҡбһ”бҹ’бһҡбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ”бҹӮбһ”бһҺбһ¶
- бһҹбһҳбҹ’бһўбһ¶бһҸбһ’бһ“бһ’бһ¶бһ“бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһ…бҹҶбһҺбһ¶бһҷбһҮбһ¶бһ”бһ“бҹ’бһҸ

## бһҸбһҫбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ Multi-Agent бһҮбһ¶бһўбҹ’бһңбһё?

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһҸбҹӮбһҳбһҪбһҷбһӮбһәбһҮбһ¶бһҳбҹүбһјбһҠбҹӮбһӣбһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹҶбһҺбһ»бҹҶбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҳбһҪбһҷ бһ“бһ·бһ„ (бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ) бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһ“бҹғбһ§бһ”бһҖбһҡбһҺбҹҚбҹ” бһңбһ¶бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҠбҹӮбһӣбһҳбһ¶бһ“бһ…бҹҶбһҺбһ»бһ…бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ–бҹҒбһӣбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҖбһҫбһ“бһЎбһҫбһ„вҖ”бһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң, бһ”бһ“бҹ’бһҸбһҹбһҡбһҹбҹҒбһҡ, бһҖбҹӮбһ”бҹ’бһҡбҹӮ, бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ–бһ·бһҸвҖ”бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһҸбһўбҹ’бһңбһёбһӮбҹ’бһҡбһ”бҹӢбһҷбҹүбһ¶бһ„бһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ prompt бһҸбҹӮбһҳбһҪбһҷбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһ„бһ¶бһҷбһҷбһәбһҸ, бһҒбһ»бһҹбһ”бһ¶бһ“бһ„бһ¶бһҷ, бһ“бһ·бһ„бһўбһҹбһ·бһҖбҹ’бһҹбһ¶бһҡбһӣбҹҶбһ”бһ¶бһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһҶбҹ„бһҸбһ”бһүбҹ’бһ бһ¶бҹ”

бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ **бһ…бҹ’бһҡбһҫбһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ** бһ”бҹҶбһ”бҹӮбһҖбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡвҖӢбһҮбһ¶вҖӢбһўбҹ’бһ“бһҖбһҮбҹҶбһ“бһ¶бһүвҖӢбһҠбҹӮбһӣбһ“бһёбһҳбһҪбһҷбҹ—бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ„бһ¶бһҡбһҳбһҪбһҷбһІбҹ’бһҷбһ”бһ¶бһ“бһӣбҹ’бһў, бһ бһҫбһҷбһҳбһ¶бһ“бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбҹ–

### бһҸбһҫбһўбҹ’бһ“бһҖбһ–бһ·бһҸбһҮбһ¶вҖӢбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһ‘бҹҒ?

бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһ–бһёбһҡбһҝбһ„бһҹбһ¶бһҳбһүбҹ’бһүбҹ” бһ”бҹ’бһҡбһҫ multi-agent **бһҸбҹӮ** бһ“бҹ…бһ–бҹҒбһӣбһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбҹ„бһҳбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ бһҳбһ¶бһ“бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң:

- вң… бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҳбһ¶бһ“ **бһҮбҹҶбһ бҹҠбһ¶бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ** бһҠбҹӮбһӣбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ–бһёбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶ (бһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң бһ‘бһӣбҹӢбһ“бһ№бһ„ бһҹбһҡбһҹбҹҒбһҡ бһ‘бһӣбҹӢбһ“бһ№бһ„ бһҹбһ„бҹ’бһҖбҹҒбһҸ)
- вң… бһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһІбҹ’бһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбҹҶбһ“бһ¶бһүбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһЎбһҫбһ„ **бһҮбһ¶бһҹбһҳаЁ•а©ҮбһҖ (in parallel)** бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ“бҹ’бһҹбҹҶбһ–бҹҒбһӣ
- вң… бһҮбҹҶбһ бҹҠбһ¶бһ“бһ•бҹ’бһҹбҹҒбһ„бҹ—бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ¬бһ§бһ”бһҖбһҡбһҺбҹҚ **бһ•бҹ’бһҹбҹҒбһ„бһӮбҹ’бһ“бһ¶**
- вң… бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһІбҹ’бһҹбһҮбҹҶбһ бҹҠбһ¶бһ“бһ“бһёбһҳбһҪбһҷбҹ—бһўбһ¶бһ… **бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹбһҠбҹ„бһҷбһЎбҹӮбһҖ**

бһ”бһҫбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҮбһ¶бһҹбҹҶбһ“бһҪбһҡ-бһ…бһҳбҹ’бһӣбһҫбһҷбһҸбҹӮбһҳбһҪбһҷ бһ¬бһҖбһ¶бһҡбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһ¶бһҳбһүбҹ’бһү бһҳбһҪбһҷ **бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚ** (бһҮбҹҶбһ–бһјбһҖ 2) бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣ, бһҗбҹ„бһҖбһӣбһҫбһҖ, бһ“бһ·бһ„бһ„бһ¶бһҷбһҖбҹ’бһ“бһ»бһ„бһңбһ·бһ’бһёбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бҹ”

> **бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбҹ–** "бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“" бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶ "бһӣбҹ’бһўбһҮбһ¶бһ„" бһ‘бҹҒбҹ” бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһҪбһҷбһ“бһёбһҳбһҪбһҷбҹ—бһ”бһ“бҹ’бһҗбҹӮбһҳбһ–бҹҒбһӣбһҷбһјбһҡ, бһҖбһ¶бһҡбһ…бҹҶбһҺбһ¶бһҷ, бһ“бһ·бһ„бһҡбһ”бһҹбҹӢбһҗбҹ’бһҳбһёбҹ—бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҸбһ¶бһҳбһҠбһ¶бһ“бҹ” бһ”бһ“бҹ’бһҗбҹӮбһҳбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹӮбһ–бҹҒбһӣбһ”бһүбҹ’бһ бһ¶вҖӢбһ…бҹӮбһҖбһЎбһҫбһ„бһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбҹ”

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһңбһ·бһ’бһёбһ‘бһёбһ–бһёбһҡ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ **бһ‘бҹҶбһ–бҹҗбһҡбһҹбҹҶбһҺбһ»бҹҶбһҡбһҪбһ…бһҡбһ¶бһӣбҹӢ**, бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһүбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ multi-agent бһ–бһ·бһҸбһҮбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбһ»бһ“бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҹбһ¶бһ„бһҹбһ„бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҜбһ„бҹ”

## бһӣбҹҶбһ бһ¶бһҸбҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸ: бһҠбһ¶бһҖбҹӢбһұбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Multi-Agent бһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡвҖӢбһ”бһ¶бһ“

бһҷбһҫбһ„бһ“бһ№бһ„бһҠбһ¶бһҖбҹӢбһұбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ **Contoso Creative Writer**, бһӮбҹҶбһҡбһјбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢ Azure бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“ (бһўбҹ’бһ“бһҖбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң, бһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒ, бһўбҹ’бһ“бһҖбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣ) бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһӮбҹ’бһ“бһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһ•бһӣбһ·бһҸбһўбһҸбҹ’бһҗбһ”бһ‘бҹ” бһңбһ¶бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Multi-Agent бһҠбҹҶбһ”бһјбһ„бһҠбҹҸбһӣбҹ’бһў бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮбһҸбһҪбһ“бһ¶бһ‘бһёбһ„бһ¶бһҷбһҷбһӣбҹӢбҹ”

### бһҮбҹҶбһ бҹҠбһ¶бһ“ 1: бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһҹбҹҶбһҺбһ»бҹҶбһ‘бҹҶбһ–бҹҗбһҡ

```bash
# бһ”бһ„бҹ’бһҖбһҫбһҸбһҗбһҸбһ’бҹ’бһңбһҫбһҖбһ¶бһҡ
mkdir creative-writer && cd creative-writer

# бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһ–бһёбһӮбҹҶбһҡбһјбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“
azd init --template contoso-creative-writer
```

> бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһ»бһҖбһҡбһҖбһ‘бҹҶбһ–бҹҗбһҡбһҹбҹҶбһҺбһ»бҹҶбһ”бҹӮбһ”бһ”бһ‘ multi-agent бһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹ…бһ–бҹҒбһӣбһҺбһ¶бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„ [Awesome AZD AI gallery](https://azure.github.io/awesome-azd/?tags=ai)гҖӮ бһҮбһҳбҹ’бһҡбһҫбһҹбһҹбһҳбһҹбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҡбһҪбһҳбһҳбһ¶бһ“ `get-started-with-ai-agents` бһ“бһ·бһ„ `azure-ai-travel-agents`бҹ”

### бһҮбҹҶбһ бҹҠбһ¶бһ“ 2: бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺ

```bash
# бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҶбһҺбһ»бҹҶбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ azd
```

### бһҮбҹҶбһ бҹҠбһ¶бһ“ 4: бһҳбһҫбһӣбһҳбһ»бһ“ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҠбһ¶бһҖбҹӢбһұбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡ

```bash
# бһҳбһҫбһӣбһҗбһ¶бһўбҹ’бһңбһёбһҠбҹӮбһӣбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ бһҳбһ»бһ“бһ“бһ№бһ„бһ…бҹҶбһҺбһ¶бһҷбһўбҹ’бһңбһёбҹ— (бһҺбҹӮбһ“бһ¶бҹҶ)

# бһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһұбҹ’бһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһҖбҹ’бһ“бһ»бһ„бһҮбҹҶбһ бһ¶бһ“бһҸбҹӮбһҳбһҪбһҷ
azd up
```

`azd up` бһ“бһ№бһ„бһҹбҹ’бһ“бһҫбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһҮбҹ’бһҡбһҫбһҹ subscription бһ“бһ·бһ„ region бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҡбҹҖбһ”бһ…бҹҶбһ’бһ“бһ’бһ¶бһ“ Azure бһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһІбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбҹ” бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ AI бһўбһ¶бһ…бһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһҷбһјбһҡбһҮбһ¶бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҹбһ¶бһҷбһўбһҹбһ¶бһҳбһүбҹ’бһүвҖ”бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһҠбһ¶бһҖбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ’бҹҶбҹ— бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ–бһ„бҹ’бһҡбһёбһҖбһ–бҹҒбһӣбһҡбһ„бһ…бһ¶бҹҶбһҠбһ¶бһҖбҹӢбһҠбҹҶбһҺбһҫбһҡбҹ–

```bash
azd deploy --timeout 1800
```

> **бһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ…бҹҶбһҺбһ¶бһҷ бһ“бһ·бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–:** бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё multi-agent бһҠбһ¶бһҖбҹӢбһҳбҹүбһјбһҠбҹӮбһӣ AI бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҖбһјбһҸбһ¶бһ бҹ’бһҹбҹҠбһё бһ“бһ·бһ„бһ”бһҺбҹ’бһҸбһ¶бһӣбһІбҹ’бһҷбһҳбһ¶бһ“бһ…бҹҶбһҺбһ¶бһҷбҹ” бһ”бһҫ `azd up` бһ”бһҡбһ¶бһҮбҹҗбһҷбһҠбҹ„бһҷбһҹбһ¶бһҡбһҖбһјбһҸбһ¶бһ бҹ’бһҹбҹҠбһёбһҳбҹүбһјбһҠбҹӮбһӣ бһҹбһјбһҳбһҳбһҫбһӣ [AI Troubleshooting](../chapter-07-troubleshooting/ai-troubleshooting.md) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҸбҹҶбһ”бһ“бҹӢбһ“бһ·бһ„бһҖбһјбһҸбһ¶бһ бҹ’бһҹбҹҠбһё бһ бһҫбһҷбһҮбҹҶбһ–бһјбһҖ 6 [Capacity Planning](../chapter-06-pre-deployment/capacity-planning.md)бҹ”

### бһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһӮбҹ’бһ“бһ¶

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ multi-agent бһ…бҹ’бһҡбһҫбһ“бһҡбһ”бһҹбҹӢ azd, **бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ** (бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһ”бҹ’бһҡбһҫ framework бһҺбһ¶бһҳбһҪбһҷбһҠбһјбһ…бһҮбһ¶ Semantic Kernel бһ¬ Microsoft Agent Framework)бҹ” бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһ“бһ№бһ„бһ бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбҹҶбһ“бһ¶бһүбһ“бһёбһҳбһҪбһҷбҹ—бһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢ, бһ•бҹ’бһүбһҫбһӣбһ‘бҹ’бһ’бһ•бһӣ, бһ бһҫбһҷбһ”бҹ’бһҡбһҳбһјбһӣбһ…бһҳбҹ’бһӣбһҫбһҷбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбҹ” бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ”бһҡбһ·бһ”бһ‘бһҸбһ¶бһҳбһҡбһҷбҹҲбҹ–

- **бһҖбһ¶бһҡбһ бҹ… Function/бһ§бһ”бһҖбһҡбһҺбҹҚ** вҖ” бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһ бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡвҖӢбһҮбҹҶбһ“бһ¶бһү бһ бһҫбһҷбһ‘бһ‘бһҪбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһңбһ·бһү
- **бһ…бһ„бһ…бһ¶бҹҶвҖӢбһҠбҹӮбһӣбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖ** вҖ” бһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһҠбһҫбһҳ Cosmos DB) бһҖбһ¶бһ“бҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһўбһ¶бһ…бһўбһ¶бһ“
- **бһҹбһ¶бһҡ/бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ** вҖ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бһ¶бһ”бһҮбһ¶бһ„бһ“бҹҒбҹҮ бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҸбһ¶бһҳбһҡбһҷбҹҲ queue бһ¬ Service Bus

> **бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһңбһ¶бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷвҖӢбһҖбҹҶбһ бһ»бһҹ:** бһҠбҹ„бһҷбһҹбһ¶бһҡбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ“бһёбһҳбһҪбһҷбҹ—бһҮбһ¶бһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖ бһ•бҹ’бһҸбһӣбҹӢбһІбҹ’бһҷ Application Insights бһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶ *бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҺбһ¶* бһҠбҹӮбһӣбһҷбһәбһҸ бһ¬бһ”бһҡбһ¶бһҮбҹҗбһҷбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҳбһјбһӣбһ бҹҒбһҸбһ»бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҳбһҪбһҷбһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһӮбһҪбһ”бһҠбһ¶бһҖбҹӢбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбҹ”

## бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһҠбҹҶбһҺбһҫбһҡ

бһҹбһјбһҳбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҗбһ¶бһңбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһ·бһҸбһҳбһ»бһ“бһҹбһ·бһ“бҹ–

```bash
# бһ”бһ„бҹ’бһ бһ¶бһүбһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһІбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ
azd show

# бһ”бһҫбһҖбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһҸбһ¶бһҳбһҠбһ¶бһ“бһҡбһ”бһҹбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё
azd monitor

# бһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ» бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ¶бһ“бһўбҹ’бһңбһёбһҳбһҫбһӣбһ‘бҹ…бһҳбһ·бһ“бһ”бҹ’бһҡбһҖбҹ’бһҡбһҸбһё
azd monitor --logs
```

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ”бһҫбһҖ URL бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ–бһё `azd show` бһ бһҫбһҷбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ (бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Creative Writer бһҹбһјбһҳбһ”бһүбҹ’бһҮбһ¶бһҹбһҡбһҹбҹҒбһҡбһўбһҸбҹ’бһҗбһ”бһ‘бһҒбҹ’бһӣбһёбһ–бһёбһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҳбһҪбһҷ)бҹ” бһҖбҹ’бһ“бһ»бһ„ Application Insights **transaction search**, бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҳбһҫбһӣбһғбһҫбһүбһҹбҹҶбһҺбһҫбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбһҠбҹӮбһҖбһҲбһҡ бһҠбһӣбҹӢбһҮбҹҶбһ бҹҠбһ¶бһ“бһўбҹ’бһ“бһҖбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң, бһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбҹ”

бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҮбҹ„бһӮбһҮбҹҗбһҷ:
- вң… `azd show` бһ”бһүбҹ’бһҮбһё endpoint бһҠбҹӮбһӣбһўбһ¶бһ…бһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ
- вң… бһҹбҹҶбһҺбһҫбһҳбһҪбһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһ…бҹ’бһ”бһ¶бһҹбҹӢбһҗбһ¶бһ”бһ¶бһ“бһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһ бҹҠбһ¶бһ“
- вң… Application Insights бһ”бһ„бҹ’бһ бһ¶бһүбһҸбҹ’бһҡбҹҒбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһ бҹҠбһ¶бһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ

## бһҖбҹҶбһҺбһҸбҹӢбһ”бҹ’бһҠбһјбһҡ: бһ”бһ“бҹ’бһҗбҹӮбһҳ бһ¬ бһҸбһҳбҹ’бһҡбһјбһң бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ

бһҠбҹ„бһҷбһҹбһ¶бһҡбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡвҖӢбһҸбҹӮбһҳбһҪбһҷбһӮбһәбһҸбҹӮбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚ бһҖбһ¶бһҡбһҖбҹӮбһ…бҹ’бһ“бҹғбһӮбһәбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбҹ–

1. **бһҹбһјбһҳбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбһ¶бһҡбһ”бһҖбһҹбҹ’бһҡбһ¶бһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһ‘бҹҶбһ–бҹҗбһҡ (бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҮбһ¶ `prompts/`, `agents/`, бһ¬ `*.prompty` бһҮбһ¶бһҹбҹҶбһҺбһ»бҹҶбһҜбһҖбһҹбһ¶бһҡ)бҹ”
2. **бһҖбҹӮбһӣбһҳбҹ’бһўбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҡбһ”бһҹбҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ** вҖ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһ”бһүбҹ’бһҮбһ¶бһІбҹ’бһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбһёбһҸбһ·бһҹбҹ’бһҡбһҪбһӣбһ…бҹ’бһ”бһ¶бһҹбҹӢбһ¬бһ…бҹҶбһ“бһҪбһ“бһ–бһ¶бһҖбҹ’бһҷбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбҹ”
3. **бһҠбһ¶бһҖбҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҗбҹ’бһҳбһёбһҸбҹӮбһҖбһјбһҠбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ** (бһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбһ·бһ“бһ”бһ¶бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡ)бҹ–

   ```bash
   azd deploy
   ```

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ“бҹ’бһҸбһ“бһ·бһ„бһҹбһ„бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ–бһё manifest бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ *бһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„* бһҹбһјбһҳбһ”бҹ’бһҡбһҫбһ•бҹ’бһ“бҹӮбһҖбһҮбҹҶбһ“бһҪбһҷ agent extension бһ“бһ·бһ„бһҡбһ„бҹ’бһңбһ·бһӣбһҮбһёбһңбһ·бһҸбһ–бҹҒбһүбһӣбҹҒбһүбһҡбһ”бһҹбҹӢбһңбһ¶бҹ–

```bash
azd ai agent init -m agent-manifest.yaml
azd up
azd ai agent invoke      # бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„, бһҮбһ¶бһҳбһҪбһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҶбҹ’бһӣбһҫбһҷбһҸбһ”
```

бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё multi-agent бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбһ»бһҒбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһӮбһәбһ…бһ¶бһҷбһ…бҹҶбһҺбһјбһӣбҹ” бһҹбһјбһҳбһӣбһ»бһ”бһ…бҹҒбһүбһӮбҹ’бһҡбһ”бҹӢбһҷбҹүбһ¶бһ„бһ“бҹ…бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ”бһүбҹ’бһ…бһ”бҹӢбҹ–

```bash
azd down --force --purge
```

бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ `--purge` бһҖбҹҸбһҷбһҖбһ’бһ“бһ’бһ¶бһ“ AI бһҠбҹӮбһӣбһӣбһ»бһ”бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– soft-deleted бһ•бһ„ (бһҠбһјбһ…бһҮбһ¶ Foundry/Azure AI Services accounts) бһҠбҹ„бһҷбһҳбһ·бһ“бһІбҹ’бһҷбһ–бһҪбһҖбһңбһ¶бһ бһ¶бһҳбһғбһ¶бһҸбҹӢбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһҠбҹҶбһҺбһҫбһҡбһЎбһҫбһ„бһңбһ·бһүбһҖбҹ’бһ“бһ»бһ„бһўбһ“бһ¶бһӮбһҸ бһ¬бһ”бһ“бҹ’бһҸбһ”бһ„бҹ’бһҖбһҫбһ“бһ…бҹҶбһҺбһ¶бһҷбҹ”

[Retail Multi-Agent Solution](../../examples/retail-scenario.md) бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһӮбҹҶбһҡбһјбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҳбһҪбһҷ, бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶вҖӢбһ‘бҹҶбһ–бҹҗбһҡбһҹбҹҶбһҺбһ»бҹҶбһҠбҹӮбһӣбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбһ”бһүбҹ’бһҮбһ¶вҖ”бһңбһ¶бһҜбһҖбһҹбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһҗбһ¶ бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһ¶бһҷбһӣбһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ *бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһң* бһҹбһ¶бһ„бһҹбһ„бҹӢбһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ… (бһ бһҫбһҷбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢбһ–бҹҒбһүбһӣбҹҒбһүбһҸбһҳбҹ’бһҡбһјбһңбһІбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҒбһ·бһҸбһҒбҹҶбһҷбҹүбһ¶бһ„бһ…бҹ’бһҡбһҫбһ“)бҹ” бһ”бҹ’бһҡбһҫбһңбһ¶бһҮбһ¶бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҡбһ…бһ“бһ¶бһ”бһҗ *бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһё* бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһұбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӮбҹҶбһҡбһјбһҳбһҪбһҷбҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ бһ¶бһҖбҹ’бһ“бһ»бһ„бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ (бһ—бһ¶бһ–бһ’бһ“бҹӢбһ‘бҹ’бһҡбһ¶бҹҶ, бһҖбһ¶бһҡбһ…бҹҶбһҺбһ¶бһҷ, бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ, бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҠбһ№бһҖбһ“бһ¶бҹҶ), бһ”бһ“бҹ’бһҸбһ‘бҹ… [бһҮбҹҶбһ–бһјбһҖ 8: Production AI Practices](../chapter-08-production/production-ai-practices.md)бҹ”

## бһ…бҹҶбһҺбһ»бһ…бһҹбһ„бҹ’бһҒбҹҒбһ”

- бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ multi-agent бһ”бҹҶбһ”бҹӮбһҖбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡвҖӢбһҡбһңбһ¶бһ„бһўбҹ’бһ“бһҖбһҮбҹҶбһ“бһ¶бһү бһҠбҹӮбһӣбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҠбҹ„бһҷбһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбҹ”
- бһ”бҹ’бһҡбһҫбһңбһ¶бһҸбҹӮбһ–бҹҒбһӣбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҳбһ¶бһ“бһҮбҹҶбһ бҹҠбһ¶бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢ, бһҖбһ¶бһҡбһ„бһ¶бһҡбһҮбһ¶бһҹбһҳбһҒ, бһ¬бһ§бһ”бһҖбһҡбһҺбҹҚбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶вҖ”бһ•бҹ’бһҹбҹҒбһ„бһ–бһёбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһҮбһҳбҹ’бһҡбһҫбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбҹ”
- бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ azd бһҳбһ·бһ“бһ”бһ¶бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбҹ– `azd init` вҶ’ `azd up` вҶ’ бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ вҶ’ `azd down`бҹ”
- бһ‘бҹҶбһ–бҹҗбһҡбһҹбҹҶбһҺбһ»бҹҶбһ–бһ·бһҸбһҮбһ¶бһҠбһјбһ…бһҮбһ¶ `contoso-creative-writer` бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҳбһҫбһӣ бһ“бһ·бһ„бһҖбҹӮбһ…бҹ’бһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё multi-agent бһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһҗбҹ’бһ„бҹғбһ“бҹҒбҹҮбҹ”
- бһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“ Application Insights бһҠбҹӮбһӣбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ–бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһҪбһҷбһ‘бҹ…бһҳбһҪбһҷбһӮбһәбһҮбһ¶бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҠбҹҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҳбһҪбһҷбһ“бҹғбһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ multi-agentбҹ”
