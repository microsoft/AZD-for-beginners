# ជំពូក 4: ហេដ្ឋារចនាសម្ព័ន្ធជាកូដ និង ការចែកចាយ

**📚 វគ្គ**: [AZD For Beginners](../../README.md) | **⏱️ រយៈពេល**: 1-1.5 ម៉ោង | **⭐ កម្រិតភាពស្មុគស្មាញ**: មធ្យម

---

## ទិដ្ឋភាពទូទៅ

ជំពូកនេះគ្របដណ្តប់លើលំនាំ (patterns) របស់ ហេដ្ឋារចនាសម្ព័ន្ធជាកូដ (IaC) ដោយប្រើទម្រង់ Bicep, ការផ្គត់ផ្គង់ធនធាន និងយុទ្ធសាស្ត្រចែកចាយ ដោយប្រើ Azure Developer CLI។

> បានផ្ទៀងផ្ទាត់ជាមួយ `azd 1.25.6` នៅខែមិថុនា 2026។

## គោលដៅការរៀន

ដោយបញ្ចប់ជំពូកនេះ អ្នកនឹង:
- យល់ដឹងអំពីរចនាសម្ព័ន្ធ និងវេយ្យាករណ៍របស់ទម្រង់ Bicep
- ផ្គត់ផ្គង់ធនធាន Azure ជាមួយ `azd provision`
- ដាក់ចេញកម្មវិធី ជាមួយ `azd deploy`
- អនុវត្តយុទ្ធនាការចែកចាយប្រភេទ blue-green និង rolling

## 🚀 ចាប់ផ្តើមរហ័ស

```bash
# ចាប់ផ្តើមពីគំរូ
azd init --template azure-functions-python-v2-http

# មើលជាមុនអំពីអ្វីដែលនឹងត្រូវបានបង្កើត
azd provision --preview

# ផ្គត់ផ្គង់ហេដ្ឋារចនាសម្ព័ន្ធតែប៉ុណ្ណោះ
azd provision

# ដាក់ចេញកូដតែប៉ុណ្ណោះ
azd deploy

# ឬទាំងពីរជាមួយគ្នា
azd up
```

## 📁 រចនាសម្ព័ន្ធគម្រោង AZD

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

## 🔧 ពាក្យបញ្ជាសំខាន់

| ពាក្យបញ្ជា | ពណ៌នា |
|---------|-------------|
| `azd init` | ចាប់ផ្ដើមគម្រោង |
| `azd provision` | បង្កើតធនធាន Azure |
| `azd deploy` | ដាក់បញ្ចូនកូដកម្មវិធី |
| `azd up` | ផ្គត់ផ្គង់ + ចែកចាយ |
| `azd down` | លុបធនធានទាំងអស់ |
