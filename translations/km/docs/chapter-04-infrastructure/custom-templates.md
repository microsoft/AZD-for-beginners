## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһҠбһӣбҹӢбһ–бҹҒбһӣбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ”бһ¶бһ“ *бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ* бһ–бһ»бһҳбҹ’бһ–бһҠбҹ„бһҷ `azd init --template <name>`бҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӮбһҳбҹ’бһҡбҹ„бһ„бһҠбҹӮбһӣбһҖбҹ’бһҡбһ»бһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸвҖ”бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ Container App бһҮбһ¶бһҳбһҪбһҷ Cosmos DB бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҠбҹӮбһӣбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңвҖ”бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ…бһ„бҹӢбһ”бһҳбҹ’бһӣбҹӮбһ„бһңбһ¶бһ‘бҹ…бһҮбһ¶бһ–бһ»бһҳбҹ’бһ–бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбҹ” бһ–бһ»бһҳбҹ’бһ–бһӮбһәбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮ Git repository бһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣ azd бһўбһ¶бһ…бһўбһ¶бһ“бһ”бһ¶бһ“бҹ” бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһўбҹ’бһ“бһҖбһ–бһёбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһңбһ¶бһ–бһёбһ…бһ“бҹ’бһӣбҹ„бҹҮбһҹбһјбһ“бҹ’бһҷ бһҸбҹҒбһҹбҹ’бһҸбһңбһ¶ бһ“бһ·бһ„ (бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ) бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһңбһ¶бһ‘бҹ…бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһҹбһ бһӮбһҳбһ“бҹҚбҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбһ·бһҖбҹ’бһҹбһ¶

бһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…:
- бһҷбһӣбҹӢбһ–бһёбһўбҹ’бһңбһёбһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҗбһҸбһҳбһҪбһҷбһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһ–бһ»бһҳбҹ’бһ– "azd"
- бһҠбһ№бһ„бһ–бһёбһҜбһҖбһҹбһ¶бһҡ бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҗбһҸбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ
- бһ”бһ“бҹ’бһҗбҹӮбһҳ `azure.yaml` бһ“бһ·бһ„ `infra/` бһҠбһҫбһҳбҹ’бһ”бһёбһўбҹ„бһҷбһўбҹ’бһ“бһҖбһҠбһ‘бҹғбһўбһ¶бһ… reuse бһ”бһ¶бһ“
- бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ–бһ»бһҳбҹ’бһ–бһ“бҹ…бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ Ш§Щ„Щ…ШӯЩ„ЩҠШ© бһҳбһ»бһ“бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖ
- бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһңбһ¶ бһ“бһ·бһ„ (бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ) бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһ”бһүбҹ’бһ…бһјбһӣбһ‘бҹ… Awesome AZD

## бһӣбһ‘бҹ’бһ’бһ•бһӣбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…:

- бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡвҖӢбһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһңбһ¶бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ”бһ¶бһ“бһ“бҹ…бһҖбһ¶бһӣбһҺбһ¶бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„ subscription бһҳбһҪбһҷбһҺбһ¶

## бһ–бһ»бһҳбҹ’бһ–бһӮбһәбһҮбһ¶бһўбҹ’бһңбһёбһ–бһ·бһҸбҹ—?

бһ–бһ»бһҳбҹ’бһ– azd бһӮбһәбһҮбһ¶бһҡбҹҖбһ”бҹүбһёбһҹбҹҠбһёбһҸбһјбһҡбһё Git бһҳбһҪбһҷ бһҠбҹӮбһӣбһҳбһ¶бһ“бһҷбҹүбһ¶бһ„бһ бҹ„бһ…бһҺбһ¶бһҹбҹӢ:

| File / folder | Purpose | Required? |
|---------------|---------|-----------|
| `azure.yaml` | Describes services, hosts, and infra provider | вң… бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ |
| `infra/` | Bicep, Terraform, or Pulumi that provisions resources | вң… бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ |
| `src/` (or your code) | The application code azd deploys | вң… бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ |
| `README.md` | How to use the template | вң… бһ•бҹ’бһҠбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚбһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„ |
| `.azdo/` or `.github/` | CI/CD pipeline definitions | бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ |
| `.devcontainer/` | Reproducible dev environment | бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ |
| `azure.yaml` `metadata` | Gallery + telemetry info | бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ (бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡвҖӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢвҖӢбһҖбһ¶бһҡвҖӢбһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷ) |

бһҳбһ·бһ“бһҳбһ¶бһ“бһўбҹ’бһңбһёбһўбһ¶бһҗбҹҢбһҖбҹҶбһ”бһ¶бҹҶбһ„бһ‘бҹҒбҹ– бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҡбһҸбҹӢ `azd init --template you/your-repo` azd бһ“бһ№бһ„ clone repo бһ“бһ·бһ„бһўбһ¶бһ“ `azure.yaml`бҹ”

## бһҮбҹҶбһ бһ¶бһ“ 1: бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹҠбҹ’бһҡбһ»бһҳ Repository

бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҗбһҸбһҠбҹ„бһҷбһҠбҹғ бһ¬бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһ–бһ»бһҳбҹ’бһ–бһҸбһ·бһ…бһҸбһҪбһ… бһ бһҫбһҷбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһңбһ¶:

```bash
mkdir my-azd-template && cd my-azd-template
git init

# бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһӣбһ„бҹӢбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ
mkdir -p src infra
```

бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢбһ‘бһјбһ‘бҹ…бһҳбһ¶бһ“бһҡбһјбһ”бһҡбһ¶бһ„бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

```
my-azd-template/
в”ңв”Җв”Җ azure.yaml              # вҶҗ template definition
в”ңв”Җв”Җ README.md               # вҶҗ how to use it
в”ңв”Җв”Җ infra/
в”Ӯ   в”ңв”Җв”Җ main.bicep          # entry point
в”Ӯ   в”ңв”Җв”Җ main.parameters.json
в”Ӯ   в””в”Җв”Җ modules/
в”Ӯ       в””в”Җв”Җ web.bicep
в”ңв”Җв”Җ src/
в”Ӯ   в””в”Җв”Җ app/                # your application code
в””в”Җв”Җ .github/
    в””в”Җв”Җ workflows/
        в””в”Җв”Җ azure-dev.yml   # optional CI/CD
```

## бһҮбҹҶбһ бһ¶бһ“ 2: бһҹбһҡбһҹбҹҒбһҡ `azure.yaml`

бһ“бҹҒбҹҮбһҮбҹ’бһҡбһҳбһ»бҹҶбһ“бҹғбһ–бһ»бһҳбҹ’бһ–бҹ” бһңбһ¶бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһўбҹҶбһ–бһёбһўбҹ’бһңбһёбһҠбҹӮбһӣ azd бһ“бһ№бһ„бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһ“бһ·бһ„бһңбһ·бһ’бһёбһ’бҹ’бһңбһҫбһҖбһ¶бһҡ:

```yaml
# azure.yaml
# yaml-language-server: $schema=https://raw.githubusercontent.com/Azure/azure-dev/main/schemas/v1.0/azure.yaml.json
name: my-azd-template
metadata:
  template: my-azd-template@1.0.0   # name@version вҖ” used by the gallery and telemetry
infra:
  provider: bicep                   # or "terraform" / "pulumi"
  path: infra
services:
  web:
    project: ./src/app
    language: js                    # js | python | dotnet | java | go
    host: containerapp              # appservice | containerapp | function | aks | staticwebapp
```

> бһңбһ¶бһӣ `metadata.template` бһӮбһәбһҮбһ¶бһўбҹ’бһңбһёбһҠбҹӮбһӣ telemetry бһҡбһ”бһҹбҹӢбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ”бҹ’бһҡбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ” бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһҖбһ¶бһҡбһҡбһҪбһҳ `name@version`бҹ”

## бһҮбҹҶбһ бһ¶бһ“ 3: бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡвҖӢбһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’

бһ…бҹ’бһ”бһ¶бһ”бҹӢбһҠбҹӮбһӣбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ»бһҳбҹ’бһ–бһҠбҹӮбһӣбһўбһ¶бһ… reuse бһ”бһ¶бһ“: **бһҖбһ»бҹҶ ever hardcode бһҲбҹ’бһҳбҹ„бҹҮ бһҸбҹҶбһ”бһ“бҹӢ бһ¬бһҸбһҳбҹ’бһӣбҹғбһ–бһ·бһҹбҹҒбһҹбһҖбҹ’бһ“бһ»бһ„ subscriptionбҹ”** бһ”бҹ’бһҡбһҫбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ бһ“бһ·бһ„бһӮбҹҶбһҡбһј token бһҜбһҖбһҹбһ¶бһҡбһҡбһ”бһҹбҹӢбһ’бһ“бһ’бһ¶бһ“ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ–бһ»бһҳбҹ’бһ–бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ“бһ№бһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡвҖӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ subscription бһ“бҹғбһўбҹ’бһ“бһҖбһҺбһ¶бһҳбһҪбһҷбҹ”

```bicep
// infra/main.bicep
targetScope = 'resourceGroup'

@minLength(1)
@description('Primary location for all resources')
param location string = resourceGroup().location

@minLength(1)
@maxLength(64)
@description('Name of the environment (used to generate resource names)')
param environmentName string

// Generate globally-unique, stable names from the environment + subscription
var resourceToken = uniqueString(subscription().id, environmentName, location)
var tags = { 'azd-env-name': environmentName }

module web 'modules/web.bicep' = {
  name: 'web'
  params: {
    name: 'app-${resourceToken}'
    location: location
    tags: tags
  }
}

output SERVICE_WEB_ENDPOINT_URL string = web.outputs.uri
```

бһ–бһёбһҡбһҲбҹ’бһҳбҹ„бҹҮбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһңбһ¶бһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһҳбһ·бһҸбҹ’бһҸбһ—бһ¶бһ–бһ–бһ»бһҳбҹ’бһ–:

1. **`azd-env-name` tag** вҖ” azd бһ”бҹ’бһҡбһҫбһңбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһҸбһ¶бһҳбһҠбһ¶бһ“ бһ“бһ·бһ„бһҹбһҳбҹ’бһўбһ¶бһҸбһ’бһ“бһ’бһ¶бһ“бһҸбһ¶бһҳбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҳбһҪбһҷбҹ—бҹ”
2. **`uniqueString(...)` resource token** вҖ” бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ“бҹ’бһҹбҹ„бһҳбһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖбһҳбһ¶бһ“бһ—бһ¶бһ–бһҜбһҖбһҸбҹ’бһҸбһ· бһ“бһ·бһ„бһҳбһ·бһ“бһҮаёұаё§бһҡвҖӢбһҠбһјбһ…бҹ’бһ“бҹ„бҹҮбһҲбҹ’бһҳбҹ„бҹҮбһҳбһ·бһ“бһ”бҹүбҹҮбһ‘бһ„бҹ’бһӮбһ·бһ…бҹ”

бһ•бҹ’бһҸбһӣбҹӢбһҜбһҖбһҹбһ¶бһҡбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҠбҹӮбһӣбһўбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣ azd бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ–бһёбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбҹ–

```json
// infra/main.parameters.json
{
  "$schema": "https://schema.management.azure.com/schemas/2019-04-01/deploymentParameters.json#",
  "contentVersion": "1.0.0.0",
  "parameters": {
    "environmentName": { "value": "${AZURE_ENV_NAME}" },
    "location": { "value": "${AZURE_LOCATION}" }
  }
}
```

azd бһҮбҹҶбһ“бһҪбһҹ `${AZURE_ENV_NAME}` бһ“бһ·бһ„ `${AZURE_LOCATION}` бһ–бһёбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһҷбҹүбһ¶бһ„бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ”

---

## бһҮбҹҶбһ бһ¶бһ“ 4: бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ–бһ»бһҳбҹ’бһ–бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бҹ…бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҒбҹ’бһӣбһҪбһ“бһҜбһ„

бһҳбһ»бһ“бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖ бһ”бҹүбһ¶бһ“бҹӢбһ”бҹ’бһҡбһҳбһ¶бһҺбһҗбһ¶бһ–бһ»бһҳбҹ’бһ–бһ’бҹ’бһңбһҫбһҖбһ¶бһҡвҖӢбһ–бһёбһҹбһ—бһ¶бһ–бһҸбҹ’бһҡбһ„бҹӢбһҹбһ¶бһ…бҹӢбҹ”

**бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ–бһёбһҗбһҸбһҖбҹ’бһ“бһ»бһ„бһҜбһ„** (бһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ Git push):

```bash
# бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһ–бһёбһҗбһҸбһ‘бһ‘бҹҒ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ•бҹ’бһӣбһјбһңбһӮбҹҶбһҡбһјбһҠбҹӮбһӣбһҳбһ¶бһ“бһ“бҹ…бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖ
mkdir /tmp/test-run && cd /tmp/test-run
azd init --template /path/to/my-azd-template

# бһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢ бһ“бһ·бһ„ бһҠбһ¶бһҖбҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ–бһёбһҠбһҫбһҳбһҠбһӣбҹӢбһ…бһ”бҹӢ
azd auth login
azd up
```

**бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбһ¶бһҡбһҠбҹ„бҹҮбһӣбҹӮбһ„**вҖ”бһ–бһ»бһҳбҹ’бһ–бһӣбҹ’бһўбһӮбһҪбһҡбһҹбһҳбҹ’бһўбһ¶бһҸбһўбһҹбҹӢбһҖбһӣбҹӮбһ„бҹ–

```bash
azd down --force --purge
```

бһ”бһҫ `azd down` бһ‘бһ»бһҖбһ’бһ“бһ’бһ¶бһ“бһ“бҹ…бһҸбҹ’бһҡбһ„бҹӢбһҖбҹ’бһҡбҹ„бһҷ бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһ’бҹ’бһӣбһ¶бһ”бҹӢбһ—бҹ’бһӣбҹҒбһ…бһҠбһ¶бһҖбҹӢ tag `azd-env-name` бһӣбһҫбһ’бһ“бһ’бһ¶бһ“бһҳбһҪбһҷбһҺбһ¶бҹ”

> **бһӮбһ“бҹ’бһӣбһ№бҹҮ:** бһҡбһҸбҹӢ `azd provision --preview` бһҖбһ¶бһӣбһҠбҹҶбһ”бһјбһ„бҹ” бһңбһ¶бһ’бҹ’бһңбһҫ what-if бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһҖбҹҶбһ бһ»бһҹбһ–бһ»бһҳбҹ’бһ–бһҳбһ»бһ“бһ–бҹҒбһӣбһ’бһ“бһ’бһ¶бһ“бһҺбһ¶бһҳбһҪбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбҹ”

## бһҮбҹҶбһ бһ¶бһ“ 5: бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһ–бһ»бһҳбҹ’бһ–

бһҮбһҳбҹ’бһҡбһ¶бһ” repo бһ‘бҹ… GitHub (бһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҠбһ‘бҹғбһ”бҹ’бһҡбһҫбһңбһ¶):

```bash
gh repo create my-azd-template --public --source=. --push
```

бһҘбһЎбһјбһңбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһ“бһ¶бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһңбһ¶бҹ–

```bash
azd init --template your-github-username/my-azd-template
```

## бһҮбҹҶбһ бһ¶бһ“ 6 (бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ): бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһ…бһјбһӣбһ‘бҹ… Awesome AZD

[Awesome AZD gallery](https://azure.github.io/awesome-azd/) бһҡбһ¶бһҷбһ“бһ¶бһҳбһ–бһ»бһҳбҹ’бһ–бһҹбһ бһӮбһҳбһ“бҹҚбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ¶бһҷбһ“бһ¶бһҳ repo бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“:

- вң… README.md бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ бһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһҡбһҝбһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ (prerequisites), бһӮбҹҶбһ“бһјбһҡбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ, бһ“бһ·бһ„бһҖбһҸбҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҗбҹ’бһӣбҹғбһ…бҹҶбһҺбһ¶бһҷ
- вң… `azure.yaml` бһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ“бһ·бһ„бһҳбһ¶бһ“ `metadata.template`
- вң… бһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣ provision бһ”бһ¶бһ“бһҹбҹ’бһўбһ¶бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ subscription бһҗбҹ’бһҳбһё
- вң… бһҜбһҖбһҹбһ¶бһҡ `LICENSE`
- вң… (бһ•бҹ’бһҸбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚ) `.devcontainer/` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Codespaces бһҳбһҪбһҷбһ…бһ»бһ…

бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҠбҹ„бһҷбһ”бһҫбһҖ pull request бһҠбҹӮбһӣбһ”бһ“бҹ’бһҗбҹӮбһҳбһ–бһ»бһҳбҹ’бһ–бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһҜбһҖбһҹбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹғбһ•бҹ’бһ‘бһ¶бҹҶбһ„ бһҠбҹ„бһҷбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҖбһ¶бһҡбһҡбһҪбһҳбһ…бҹҶбһҺбҹӮбһҖбһ“бҹ… [Awesome AZD repository](https://github.com/Azure/awesome-azd)бҹ”

## бһ”бһүбҹ’бһ бһ¶бһҳбһ»бһ“бҹ—бһҠбҹӮбһӣбһ”бҹ’бһҡбһ‘бҹҮбһғбһҫбһүбһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢ

| Pitfall | Fix |
|---------|-----|
| Hardcoded resource names | Use the `uniqueString()` resource token |
| `azd down` leaves resources | Tag every resource (or the resource group) with `azd-env-name` |
| Template works for you, fails for others | Remove subscription IDs, tenant IDs, and region assumptions |
| Outputs not wired into the app | Export `SERVICE_<NAME>_ENDPOINT_URL` and other `AZURE_*` outputs |
| Gallery submission rejected | Add `README.md`, `LICENSE`, and `metadata.template` |

## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ„бҹ’бһҒбҹҒбһ”

- бһ–бһ»бһҳбҹ’бһ–бһӮбһәбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮ Git repo бһҠбҹӮбһӣбһҳбһ¶бһ“ `azure.yaml`, `infra/`, бһ“бһ·бһ„бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
- бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡвҖӢбһҡбһ¶бһӣбҹӢбһўбҹ’бһңбһёбҹ—вҖ”бһҲбҹ’бһҳбҹ„бҹҮ бһҸбҹҶбһ”бһ“бҹӢ бһ“бһ·бһ„ IDвҖ”бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһңбһ¶бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһ‘бһёбһҖбһ“бҹ’бһӣбҹӮбһ„бҹ”
- бһҸбҹӮбһ„бһҸбҹӮбһҠбһ¶бһҖбҹӢ tag бһӣбһҫбһ’бһ“бһ’бһ¶бһ“бһҮбһ¶бһҳбһҪбһҷ `azd-env-name` бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ `azd down` бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ”бһ¶бһ“бҹ”
- бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбҹ’бһ“бһ»бһ„бһҮбһҳбҹ’бһҡбһҫбһҹ `azd init --template <local-path>` бһҳбһ»бһ“бһ”бҹ„бҹҮбһ•бҹ’бһҹбһ¶бһҷбҹ”
- бһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбҹҒбһҸбһ¶бһҢбһ¶бһҸбһ¶ бһ“бһ·бһ„ README бһҠбһҫбһҳбҹ’бһ”бһёбһҠбһ¶бһҖбҹӢбһҹбҹ’бһ“бһҫбһ‘бҹ… Awesome AZDбҹ”
