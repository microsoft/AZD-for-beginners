бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [бһӮбһҳбҹ’бһҡбҹ„бһ„бһҠбҹҶбһ”бһјбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ](first-project.md) бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бҹ„бҹҮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹ„бһҷбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһӮбҹҶбһҡбһјбҹ” бһ”бҹүбһ»бһ“бҹ’бһҠбҹӮбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“ бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҳбһ¶бһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһҡбһ¶бһ”бҹӢвҖ”API Node.js, бһҹбҹҒбһңбһ¶ Python Flask, бһңбҹҒбһ” .NETвҖ”бһҠбҹӮбһӣбһҳбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһ”бһ“бҹ’бһҗбҹӮбһҳ azd бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҠбһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бҹ„бҹҮбһңбһ¶бһҮбһ¶бһҳбһҪбһҷ `azd up` бһҠбҹ„бһҷбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҳбһ¶бһ“бһӮбҹҶбһҡбһјбҹ”

## бһӮбҹ„бһӣбһҠбҹ…бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶

вҖӢбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бҹ–
- бһҷбһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһ…бҹҶбһ“бһҪбһ“бһ”бһёбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһӮбһҳбҹ’бһҡбҹ„бһ„ azd
- бһҡбһҸбҹӢ `azd init` бһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ
- бһҷбһӣбҹӢбһ–бһёбһҸбһҪбһ“бһ¶бһ‘бһёбһҡбһ”бһҹбҹӢ `azure.yaml` бһ“бһ·бһ„бһҗбһҸ `infra/` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
- бһҠбһ№бһ„бһ–бҹҒбһӣбһҺбһ¶бһІбҹ’бһҷ azd бһ”бһ„бҹ’бһҖбһҫбһҸбһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ¬бһ–бҹҒбһӣбһҺбһ¶бһІбҹ’бһҷбһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢ
- бһҠбһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бҹ„бҹҮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһ‘бҹ… Azure бһҮбһ¶бһҳбһҪбһҷ `azd up`

## бһӣбһ‘бҹ’бһ’бһ•бһӣбһҖбһ¶бһҡбһҡбҹҖбһ“

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…бҹ–
- бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ azd бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ
- бһўбһ¶бһ“ бһ“бһ·бһ„бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҜбһҖбһҹбһ¶бһҡ `azure.yaml` бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“
- бһ”бһ„бҹ’бһҖбһҫбһҸбһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷ `azd infra generate`
- бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ•бҹ’бһ‘бһ»бһҖ Azure бһҠбҹӮбһӣбһҹбһҳбһҹбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
- бһҠбһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бҹ„бҹҮ бһ“бһ·бһ„бһҹбһҳбҹ’бһўбһ¶бһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖа№ҖаёӯаёҮ

## бҹЈ бһңбһ·бһ’бһёбһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһӮбһҳбҹ’бһҡбҹ„бһ„ azd

| бһ…бҹҶбһҺбһ»бһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ | бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ | бһ–бҹҒбһӣбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫ |
|------------------|------------|--------------------|
| **бһ–бһёбһӮбҹҶбһҡбһј** | `azd init --template <name>` | бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбҹҖбһ“ бһ¬бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҗбҹ’бһҳбһёбһ–бһёбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ |
| **бһ–бһёбһҖбһјбһҠбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ** | `azd init` (бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһӮбһҳбҹ’бһҡбҹ„бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ) | бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһҡбһ¶бһ”бҹӢбһ бһҫбһҷбһ…бһ„бҹӢбһҠбһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бҹ„бҹҮбһңбһ¶ |
| **бһ–бһё Git repo** | `azd init --from-code` (бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ repo бһҠбҹӮбһӣбһ”бһ¶бһ“ clone) | бһҖбһ¶бһҡбһҷбһҖ azd бһҳбһҖбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ бҹҒбһҸбһ»бһҖбһ¶бһҡбһҺбҹҚ repository бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ |

бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһӣбҹҶбһ бһ¶бһҸбҹӢбһҮбһҳбҹ’бһҡбһҫбһҹбһ‘бһёбһҳбһҪбһҷбһҡбһҪбһ…бһ бһҫбһҷбҹ” бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһҮбһҳбҹ’бһҡбһҫбһҹбһ‘бһёбһ–бһёбһҡвҖ”бһҹаё–аёІаёҷаёҒаёІаёЈаё“а№Ңбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһ“бҹ…бһ–бҹҒбһӣбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бҹ”

## бһҮбҹҶбһ бһ¶бһ“ 1: бһҡбһҸбҹӢ `azd init` бһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

бһ”бһҫбһҖ terminal **бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһӮбһҳбҹ’бһҡбҹ„бһ„бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ** бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһҡбһҸбҹӢбҹ–

```bash
cd my-existing-app
azd init
```

azd бһ“бһ№бһ„бһҹбһҪбһҡбһҗбһ¶бһҸбһҫбһўбҹ’бһ“бһҖбһ…бһ„бҹӢINITIALIZE бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бҹ” бһҮбҹ’бһҡбһҫбһҹбҹ–

```
? How do you want to initialize your app?
> Use code in the current directory
  Select a template
```

бһҮбҹ’бһҡбһҫбһҹ **"бһ”бҹ’бһҡбһҫбһҖбһјбһҠбһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бҹ”"** azd бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҹбҹ’бһҖбҹҒбһ“бһҗбһҸбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҖбҹҶбһҺбһҸбҹӢбһ—бһ¶бһҹбһ¶ бһ“бһ·бһ„ framework бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһҹбҹ’бһ“бһҫбһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ•бҹ’бһ‘бһ»бһҖ (host) бһҠбҹӮбһӣбһҹбһҳбһҹбҹ’бһҡбһ”бҹ”

### бһўбҹ’бһңбһёбһҠбҹӮбһӣ azd бһҖбҹҶбһҺбһҸбҹӢ

azd бһҳбһҫбһӣбһҹбһүбҹ’бһүбһ¶бһҠбһјбһ…бһҮбһ¶ `package.json`, `requirements.txt`, `pom.xml`, `*.csproj`, бһ¬ `Dockerfile` бһ бһҫбһҷбһ•бҹ’бһҸбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ•бҹ’бһ‘бһ»бһҖ Azure бһҠбҹӮбһӣбһҹбһҳбһҹбҹ’бһҡбһ”бҹ–

| бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ | бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ•бҹ’бһ‘бһ»бһҖбһҠбҹӮбһӣбһ”бҹ’бһҡбһ бҹӮбһӣбһ”бһ¶бһ“бһҡбһҖбһғбһҫбһү |
|------------------|------------------------------------|
| Node.js / Python / .NET web app | Azure App Service or Container Apps |
| Containerized app (`Dockerfile`) | Azure Container Apps |

бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҹбҹҒбһңбһ¶(бҹ—) бһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбһҖбһғбһҫбһү бһ бһҫбһҷ azd бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбҹ”

## бһҮбҹҶбһ бһ¶бһ“ 2: бһҷбһӣбҹӢбһ–бһёбһўбҹ’бһңбһёбһҠбҹӮбһӣ azd бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһё init бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҳбһ¶бһ“бһҡбһҝбһ„бһҗбҹ’бһҳбһёбһ–бһёбһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–

```
my-existing-app/
в”ңв”Җв”Җ azure.yaml          # вҶҗ NEW: tells azd what to deploy and where
в”ңв”Җв”Җ infra/              # вҶҗ NEW: Infrastructure as Code (Bicep by default)
в”Ӯ   в”ңв”Җв”Җ main.bicep
в”Ӯ   в””в”Җв”Җ main.parameters.json
в”ңв”Җв”Җ src/ (your code)    # unchanged
в””в”Җв”Җ ...                 # your existing files, untouched
```

### `azure.yaml` вҖ” бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһӮбһҳбҹ’бһҡбҹ„бһ„

бһ“бҹҒбҹҮбһҮбһ¶вҖӢбһҠбһҪбһ„бһ…бһ·бһҸбҹ’бһҸбһ“бҹғбһӮбһҳбҹ’бһҡбҹ„бһ„ azd бһҳбһҪбһҷбҹ” бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһўбһ”бҹ’бһ”бһ—бһ¶бһ–бһҳбһ¶бһ“бһҡбһјбһ”бһҡбһ¶бһ„бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

```yaml
# azure.yaml
name: my-existing-app
services:
  web:
    project: ./src           # path to your app code
    language: js             # js | python | dotnet | java
    host: appservice         # appservice | containerapp | function | staticwebapp
```

бһҖбһ¶бһҡ `services` бһҮбһ¶бһ”бҹ’бһӣбһ»бһҖбһӮбһәбһҮбһ¶вҖӢбһ•бҹ’бһ“бҹӮбһҖбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ– бһҳбҹүбҹ„бһ„бһ…бҹҒбһүбһ“бһёбһҳбһҪбһҷбҹ—бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҗбһҸбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһІбҹ’бһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ•бҹ’бһ‘бһ»бһҖ Azureбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ‘бһ¶бҹҶбһ„ frontend бһ“бһ·бһ„ API бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҳбһ¶бһ“бһҹбҹҒбһңбһ¶бһ–бһёбһҡбһ–бһёбһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бҹ”

### `infra/` вҖ” бһ’бһ“бһ’бһ¶бһ“ Azure бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶вҖӢбһҖбһјбһҠ

бһҗбһҸ `infra/` бһҡбһҖбҹ’бһҹбһ¶вҖӢбһҜбһҖбһҹбһ¶бһҡ Bicep бһҠбҹӮбһӣбһҖбҹҶбһҺбһҸбҹӢбһ’бһ“бһ’бһ¶бһ“ Azure бһҡбһҠбһјбһңбһҠбҹӮбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ (App Service, database, бһӣбҹ”) бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҡбһҹбҹҒбһҡбһ–бһҪбһҖбһ“бҹҒбҹҮбһҠбҹ„бһҷбһҠбҹғбһ‘бҹҒвҖ”azd бһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹҶбһҺбһ»бһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹӮбһӣбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҖбҹӮбһңбһ¶бһҖбҹ’бһҡбҹ„бһҷбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ’бһ“бһ’бһ¶бһ“ бһ¬бһ”бһ„бҹ’бһҡбһҪбһҳбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– (бһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [бһҮбҹҶбһ–бһјбһҖ 4](../chapter-04-infrastructure/README.md))бҹ”

> **бһҠбҹҶбһ”бһјбһ“бҹ’бһҳбһ¶бһ“:** бһҸбһҫбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһҳбһҫбһӣ бһ¬бһ”бҹ’бһҸбһјбһҡбһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ бһҳбһ»бһ“бһ–бҹҒбһӣбһҠбһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бҹ„бҹҮбһ‘бҹҒ? бһҡбһҸбҹӢ `azd infra generate` (бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҮбһ¶бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡ `azd infra synth`) бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҡбһҹбҹҒбһҡ IaC бһ‘бҹ…бһӣбһҫбһҢбһёбһҹбһҖбҹҸбһҠбһјбһ…бһҮбһ¶бһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһҺбҹӮбһңбһ¶бҹ”

## бһҮбҹҶбһ бһ¶бһ“ 3: бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһҖбһ»бһ„бһ бҹ’бһӮбһёбһҖбһҡбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ

бһ”бһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ¬бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ (string бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ, API key), бһҖбһ»бҹҶбһ”бһүбҹ’бһ…бһјбһӣбһңбһ¶вҖӢбһҮбһ¶бһҖбһјбһҠбһҮбһ¶бһ”бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ” бһ”бҹ’бһҡбһҫбһҸбһҳбҹ’бһӣбҹғбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ (environment values):

```bash
# бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ
azd env new dev

# бһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ
azd env set API_VERSION 1.0.0
```

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ бһҹбһјбһҳбһҡбһҖбҹ’бһҹбһ¶бһ–бһҪбһҖбһңбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Key Vault бһ бһҫбһҷбһҷбҹ„бһ„бһ–бһҪбһҖбһңбһ¶бһ–бһёбһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ вҖ” бһҳбһҫбһӣ [бһҮбҹҶбһ–бһјбһҖ 3: бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ бһ“бһ·бһ„ бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ](../chapter-03-configuration/authsecurity.md)бҹ”

## бһҮбҹҶбһ бһ¶бһ“ 4: бһҠбһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бҹ„бҹҮ

бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһ”бҹ’бһҡбһҫбһӣбҹҶбһ бһ¶бһҸбҹӢбһҠбһҠбҹӮбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҹбҹ’бһӮбһ¶бһӣбҹӢбһҡбһҪбһ…бһ бһҫбһҷбҹ–

```bash
# бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ (бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ azd)
azd auth login

# бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһ’бһ“бһ’бһ¶бһ“бһҠбҹӮбһӣбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ
azd provision --preview

# бһҡбҹҖбһ”бһ…бҹҶбһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡвҖӢбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
azd up
```

бһ–бҹҒбһӣбһңбһ¶бһ”бһүбҹ’бһ…бһ”бҹӢ azd бһ“бһ№бһ„бһ”бһ·бһ‘бһ”бһ„бҹ’бһ бһ¶бһү URL бһҡбһ”бһҹбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбҹ’бһ“бһҖбҹ” бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһңбһ¶бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһ’бһёбһҠбһјбһ…бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё azd бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ–

```bash
azd show           # бһ”бһ„бҹ’бһ бһ¶бһүбһ…бҹҶбһҺбһ»бһ…бһ”бһүбҹ’бһ…бһ”бҹӢ
azd monitor --logs # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ
```

| бһҡбҹ„бһӮбһҹбһүбҹ’бһүбһ¶ | бһҳбһјбһӣбһ бҹҒбһҸбһ»бһҹбһҳбһ бҹҒбһҸбһ» | бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ |
|----------|------------------|------------|

## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ„бҹ’бһҒбҹҒбһ”

- `azd init` вҶ’ **"бһ”бҹ’бһҡбһҫбһҖбһјбһҠбһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“"** бһ”бһ“бҹ’бһҗбҹӮбһҳ azd бһ‘бҹ…бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбҹ”
- `azure.yaml` бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҗбһҸбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ•бҹ’бһ‘бһ»бһҖ Azure; `infra/` бһҖбҹҶбһҺбһҸбҹӢбһ’бһ“бһ’бһ¶бһ“бһҮбһ¶бһҡбҹҖбһ” Bicepбҹ”
- `azd infra generate` бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ¬бһ”бҹ’бһҸбһјбһҡбһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбҹ”
- бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҡбһҪбһ… бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһӣбҹҶбһ бһ¶бһҸбҹӢбһҠбһҠбҹӮбһӣбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ–бһёбһӮбҹҶбһҡбһј бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ `azd up` / `azd down`бҹ”
