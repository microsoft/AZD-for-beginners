# ជំពូក ១៖ មូលដ្ឋាន និងការចាប់ផ្តើមរហ័ស

**📚 វគ្គសិក្សា**: [AZD សម្រាប់អ្នកដំបូង](../../README.md) | **⏱️ រយៈពេល**: ៣០-៤៥ នាទី | **⭐ កម្រិតស្មុគស្មាញ**: អ្នកដំបូង

---

## ទិដ្ឋភាពទូទៅ

ជំពូកនេះណែនាំអំពីគ្រឹះសម្រាប់ Azure Developer CLI (azd)។ អ្នកនឹងរៀនពីកំណត់សំខាន់ៗ ដំឡើងឧបករណ៍ និងចាក់ផ្សាយកម្មវិធីដំបូងរបស់អ្នកទៅ Azure។

> បានអះអាងជាមួយ `azd 1.23.12` ក្នុង ខែមីនា ឆ្នាំ ២០២៦។

## គោលបំណងការសិក្សា

ដោយបញ្ចប់ជំពូកនេះ អ្នកនឹងអាច៖
- យល់ពី Azure Developer CLI គឺជាអ្វី និងវាខុសគ្នាយ៉ាងដូចម្តេចពី Azure CLI
- ដំឡើង និងកំណត់រចនាសម្ព័ន្ធ AZD នៅលើវេទិការបស់អ្នក
- ចាក់ផ្សាយកម្មវិធីដំបូងទៅ Azure ជាមួយ `azd up`
- សំអាតធនធានជាមួយ `azd down`

## ✅ ចាប់ផ្តើមនេះ៖ អះអាងការដំឡើងរបស់អ្នក

មុនពេលចាប់ផ្តើម សូមធានាថាម៉ាស៊ីនក្នុងស្រុករបស់អ្នកបានរៀបចំសម្រាប់គំរូជំពូក១៖

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

បើសិន script រាយការណ៍ថាឧបករណ៍ខ្វះ សូមជួសជុលមុន ហើយបន្តជាមួយជំពូកនេះ។

---

## 🚀 ចាប់ផ្តើមរហ័ស

```bash
# ពិនិត្យការដំឡើង
azd version

# សម迅្រីសម្រាប់ AZD
# ជាជម្រើសៈ az login ប្រសិនបើអ្នកគ្រោងចិត្តដំណើរការការបញ្ជា Azure CLI តាមផ្ទាល់
azd auth login

# ចេញផ្សាយកម្មវិធីដំបូងរបស់អ្នក
azd init --template todo-nodejs-mongo
azd up

# សំអាតបន្ទាប់ពីបញ្ចប់
azd down --force --purge
```

---

## ✅ សង្កត់លក្ខខណ្ឌជោគជ័យ

បន្ទាប់ពីបញ្ចប់ជំពូកនេះ អ្នកគួរតែអាច៖

```bash
azd version              # បង្ហាញកំណែបានដំឡើង
azd init --template todo-nodejs-mongo  # ស៊ីសងគម្រោង
azd up                   # ចែកចាយទៅកាន់ Azure
azd show                 # បង្ហាញ URL កម្មវិធីកំពុងដំណើរការ
azd down --force --purge # សម្អាតធនធាន
```
