# ជំពូក 1: មូលដ្ឋាន និង ចាប់ផ្ដើមរហ័ស

**📚 មុខវិជ្ជា**: [AZD For Beginners](../../README.md) | **⏱️ រយៈពេល**: 30-45 minutes | **⭐ ជំនាញ**: Beginner

---

## សេចក្តីសង្ខេប

ជំពូកនេះណែនាំអំពីមូលដ្ឋាននៃ Azure Developer CLI (azd)។ អ្នកនឹងរៀនពីគំនិតស្នូល ការដំឡើងឧបករណ៍ និងវិធីបង្ហោះកម្មវិធីដំបូងរបស់អ្នកទៅកាន់ Azure។

> បានផ្ទៀងផ្ទាត់ជាមួយ `azd 1.25.6` នៅខែមិថុនា 2026។

## គោលបំណងការរៀន

ដោយបញ្ចប់ជំពូកនេះ អ្នកនឹងអាច:
- យល់ពីអ្វីដែល Azure Developer CLI ជា និង វាផ្សេងពី Azure CLI យ៉ាងដូចម្តេច
- ដំឡើង និងកំណត់រចនាសម្ព័ន្ធ AZD នៅលើប្លាតពីមរបស់អ្នក
- បង្ហោះកម្មវិធីដំបូងរបស់អ្នកទៅកាន់ Azure ជាមួយ `azd up`
- សម្អាតធនធានជាមួយ `azd down`

## ✅ ចាប់ផ្ដើមនៅទីនេះ: សូមផ្ទៀងផ្ទាត់ការកំណត់របស់អ្នក

មុនពេលចាប់ផ្ដើម សូមបញ្ជាក់ថា ម៉ាស៊ីនមូលដ្ឋានរបស់អ្នករួចរាល់សម្រាប់ពុម្ពជំពូក 1៖

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

ប្រសិនបើស្គ្រីបបានរាយការណ៍ថាត្រូវខ្វះឧបករណ៍ណាមួយ សូមជួសជុលទាំងនោះសិន ហើយបន្ទាប់មកបន្តជំពូកនេះ។

---

## 🚀 ចាប់ផ្ដើមរហ័ស

```bash
# ពិនិត្យការដំឡើង
azd version

# ផ្ទៀងផ្ទាត់អត្តសញ្ញាណសម្រាប់ AZD
# ជាជម្រើស៖ ប្រើ az login ប្រសិនបើអ្នកចង់រត់ពាក្យបញ្ជា Azure CLI ដោយផ្ទាល់
azd auth login

# ដាក់ដំណើរការកម្មវិធីដំបូងរបស់អ្នក
azd init --template todo-nodejs-mongo
azd up

# សម្អាតបន្ទាប់ពីបញ្ចប់
azd down --force --purge
```

---

## ✅ មាតិកាដែលត្រូវបានចាប់យក

បន្ទាប់ពីបញ្ចប់ជំពូកនេះ អ្នកគួរតែអាច:

```bash
azd version              # បង្ហាញកំណែដែលបានដំឡើង
azd init --template todo-nodejs-mongo  # ចាប់ផ្ដើមបង្កើតគម្រោង
azd up                   # បង្ហោះទៅ Azure
azd show                 # បង្ហាញ URL របស់កម្មវិធីដែលកំពុងដំណើរការ
azd down --force --purge # សម្អាតធនធាន
```
