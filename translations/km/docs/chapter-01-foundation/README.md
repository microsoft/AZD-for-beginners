# ជំពូក ១៖ មូលដ្ឋាន និង ចាប់ផ្តើមលឿន

**📚 មេរៀន**: [AZD សម្រាប់អ្នកថ្មី](../../README.md) | **⏱️ រយៈពេល**: ៣០-៤៥ នាទី | **⭐ សំខាន់ភាព**: អ្នកចាប់ផ្តើម

---

## ទិដ្ឋភាពទូទៅ

ជំពូកនេះណែនាំអំពីមូលដ្ឋាន Azure Developer CLI (azd)។ អ្នកនឹងរៀនពីគំនិតសំខាន់ៗ តំឡើងឧបករណ៍ និងដាក់ពាក្យសុំដំបូងទៅ Azure។

> បានផ្ទៀងផ្ទាត់សម្រាប់ `azd 1.27.1` នៅខែកក្កដា ២០២៦។

## គោលបំណងការរៀន

ដោយបញ្ចប់ជំពូកនេះ អ្នកនឹង៖
- យល់ពីអ្វីទៅជា Azure Developer CLI ហើយវាផ្សេងពី Azure CLI ដោយយ៉ាងណា
- តំឡើង និងកំណត់រចនាសម្ព័ន្ធ AZD លើវេទិការរបស់អ្នក
- ដាក់ពាក្យសុំដំបូងរបស់អ្នកទៅ Azure ជាមួយ `azd up`
- សម្អាតធនធានជាមួយ `azd down`

## ✅ ចាប់ផ្តើមទីនេះ៖ ពិនិត្យមើលការតំឡើងរបស់អ្នក

មុនចាប់ផ្តើម សូមបញ្ជាក់ថា ម៉ាស៊ីនក្នុងតំបន់របស់អ្នកបានត្រៀមសម្រាប់ក្រុមរចនាសម្ព័ន្ធជំពូក ១៖

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

ប្រសិនបើស្ព្រីបដាក់របាយការណ៍ថាមិនមានឧបករណ៍ណាមួយ សូមជួសជុលវាមុន ហើយបន្តបន្តជំពូក។

---

## 🚀 ចាប់ផ្តើមលឿន

```bash
# ពិនិត្យការដំឡើង
azd version

# ធ្វើការផ្ទៀងផ្ទាត់សម្រាប់ AZD
# ជាជម្រើស: az login ប្រសិនបើអ្នកមានផែនការដើម្បីរត់ពាក្យបញ្ជា Azure CLI ដោយផ្ទាល់
azd auth login

# បញ្ចេញកម្មវិធីដំបូងរបស់អ្នក
azd init --template todo-nodejs-mongo
azd up

# សំអាតពេលបញ្ចប់
azd down --force --purge
```

---

## ✅ លក្ខណៈជោគជ័យ

បន្ទាប់ពីបញ្ចប់ជំពូកនេះ អ្នកគួរតែអាច៖

```bash
azd version              # បង្ហាញកំណែដែលបានដំឡើង
azd init --template todo-nodejs-mongo  # ចាប់ផ្តើមគម្រោង
azd up                   # ចាក់តម្រង់ទៅ Azure
azd show                 # បង្ហាញ URL នៃកម្មវិធីដំណើរការ
azd down --force --purge # សម្អាតធនធាន
```
