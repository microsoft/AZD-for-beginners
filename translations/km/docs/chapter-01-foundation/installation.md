## Prerequisites

бһҳбһ»бһ“вҖӢбһ–бҹҒбһӣвҖӢбһҠбҹҶбһЎбһҫбһ„ azd бһҹбһјбһҳвҖӢбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢвҖӢбһўбҹ’бһ“бһҖвҖӢбһҳбһ¶бһ“бҹ–  
- **Azure subscription** - [Create a free account](https://azure.microsoft.com/free/)  
- **Azure CLI** - бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢвҖӢбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺ бһ“бһ·бһ„вҖӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„вҖӢбһ’бһ“бһ’бһ¶бһ“  
- **Git** - бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢвҖӢбһ…бһҳбҹ’бһӣбһ„вҖӢбһ‘бҹҶбһ–бҹҗбһҡбһӮбҹҶбһҡбһј бһ“бһ·бһ„вҖӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„вҖӢбһҖбҹҶбһҺбҹӮ  
- **Docker** (бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ) - бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёвҖӢбһҡбһ№бһҸбһ”бһ“бҹ’бһҸбһ№бһ„бһҖбҹ’бһ“бһ»бһ„вҖӢбһҖбһ»бһ„бһҸбһәбһ“бҹҗбһҡ  

## Installation Methods

### Windows

#### Option 1: Windows Package Manager (Recommended)
```cmd
winget install microsoft.azd
```

#### Option 2: PowerShell Install Script
```powershell
# бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣ winget бһҳбһ·бһ“бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“
powershell -ex AllSigned -c "Invoke-RestMethod 'https://aka.ms/install-azd.ps1' | Invoke-Expression"
```

#### Option 3: Chocolatey
```cmd
choco install azd
```

#### Option 4: Manual Installation
1. бһ‘бһ¶бһүбһҷбһҖвҖӢбһҖбһ¶бһҡвҖӢбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷвҖӢбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷвҖӢбһ–бһё [GitHub](https://github.com/Azure/azure-dev/releases)  
2. бһҠбһҖвҖӢбһҜбһҖбһҹбһ¶бһҡвҖӢбһ‘бҹ… `C:\Program Files\azd\`  
3. бһ”бһүбҹ’бһҮбһјбһӣвҖӢбһңбһ¶вҖӢбһ…бһјбһӣвҖӢбһҖбҹ’бһ“бһ»бһ„вҖӢбһўбһҗбҹҒбһҡвҖӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ PATH



## рҹҗӣ Troubleshooting Installation

### Common Issues

#### Permission Denied (Windows)  
```powershell
# бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ· PowerShell бһҮбһ¶ бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

#### PATH Issues  
бһ”бһ“бҹ’бһҗбҹӮбһҳ azd бһҠбҹ„бһҷвҖӢбһҠбҹғбһ…бһјбһӣвҖӢбһҖбҹ’бһ“бһ»бһ„ PATHбҹ–

**Windows:**  
```cmd
setx PATH "%PATH%;C:\Program Files\azd\"
```

**macOS/Linux:**  
```bash
echo 'export PATH=$PATH:/usr/local/bin' >> ~/.bashrc
source ~/.bashrc
```

#### Network/Proxy Issues  
```bash
# бһҖбҹҶбһҺбһҸбҹӢ proxy
azd config set http.proxy http://proxy:8080
azd config set https.proxy https://proxy:8080

# бһҮбҹҶбһ“бһ»бҹҶбһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ SSL (бһҳбһ·бһ“бһ”бһ¶бһ“бһҺбҹӮбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ)
azd config set http.insecure true
```

#### Version Conflicts  
```bash
# бһҠбһҖбһ…бҹҒбһүбһҖбһ¶бһҡбһҸбһҳбҹ’бһӣбһҫбһ„бһ…бһ¶бһҹбҹӢбҹ—
# Windows: winget uninstall microsoft.azd
# macOS: brew uninstall azd
# Linux: бһӣбһ»бһ”бһ”бҹҠбһёбһҺбһ¶бһҡбһё azd бһҳбһ»бһ“вҖӢвҖӢбһ¬ symlink бһҳбһ»бһ“бһ–бҹҒбһӣбһҸбһҳбҹ’бһӣбһҫбһ„бһЎбһҫбһ„бһңбһ·бһү

# бһҹбһҳбҹ’бһўбһ¶бһҸбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡ
rm -rf ~/.azd
```
