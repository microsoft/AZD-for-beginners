# бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡвҖӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– - бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһ’бһ“бһ’бһ¶бһ“ Azure

**бһ…бһҡбһҺбҹҚбһҡбһ»бһҖбҹ’бһҒбһҮбһ¶бһҸбһ·вҖӢбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бҹҲ**
- **рҹ“ҡ бһ‘бҹҶбһ–бҹҗбһҡбһҠбһҫбһҳбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶**: [AZD бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳ](../../README.md)
- **рҹ“– бһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“**: бһңбһӮбҹ’бһӮ 6 - бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ бһ“бһ·бһ„бһ•бҹӮбһ“бһҖбһ¶бһҡ бһҳбһ»бһ“бһ”бһүбҹ’бһ….deploy
- **в¬…пёҸ бһңбһӮбҹ’бһӮбһҳбһ»бһ“**: [бһңбһӮбҹ’бһӮ 5: бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ Multi-Agent AI](../../examples/retail-scenario.md)
- **вһЎпёҸ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ**: [бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ SKU](sku-selection.md)
- **рҹҡҖ бһңбһӮбҹ’бһӮбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ**: [бһңбһӮбҹ’бһӮ 7: бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ„бһ”бһүбҹ’бһ бһ¶](../chapter-07-troubleshooting/common-issues.md)

## бһҺбҹӮбһ“бһ¶бҹҶ

бһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бҹҒбһҹбһҖбҹҚбһ–бҹҒбһүбһҳбһҪбһҷбһ“бҹҒбҹҮ бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҡбҹҖбһ”бһ…бҹҶ бһ“бһ·бһ„бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ’бһ“бһ’бһ¶бһ“ Azure бһҳбһ»бһ“бһ“бһ№бһ„бһ”бһүбҹ’бһ….deploy бһҮбһ¶бһҳбһҪбһҷ Azure Developer CLIбҹ” бһҹбһјбһҳбһҡбҹҖбһ“бһ–бһёбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„, бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҸбҹҶбһ”бһ“бҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҗбһ¶бһҖбһ¶бһҡбһ”бһүбҹ’бһ….deploy бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҮбҹ„бһӮбһҮбҹҗбһҷ бһҒбһҺбҹҲбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһ“бһ·бһ„бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ…бҹҶбһҺбһ¶бһҷбҹ” бһҮбҹҶбһ“бһ¶бһүбһ•бҹӮбһ“бһҖбһ¶бһҡбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ…бҹҶбһ–бҹ„бҹҮбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бҹ— бһ“бһ·бһ„бһҹбҹ’бһ‘бһ„бҹӢбһҳбһҸбһ·бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһ“бһ№бһ„бһҹбҹ’бһҖбҹҒбһӣбҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбҹҖбһ“

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бҹҒбһҹбһҖбҹҚбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…бҹҲ
- бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһўбҹҶбһ–бһёбһӮбҹ„бһӣбһҖбҹҶбһҺбһҸбҹӢ, бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҸбҹҶбһ”бһ“бҹӢ Azure
- бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҮбҹҶбһ“бһ¶бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ’бһ“бһ’бһ¶бһ“ бһҳбһ»бһ“бһҖбһ¶бһҡбһ”бһүбҹ’бһ….deploy
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– бһ“бһ·бһ„бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·
- бһҖбҹӮбһӣбҹ’бһ”бһҫбһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹ„бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһӮбһ·бһҸбһӮбһјбһҡбһўбҹҶбһ–бһёбһ‘бҹҶбһ бҹҶбһ’бһ“бһ’бһ¶бһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбҹ’бһҖбҹҒбһӣбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ…бҹҶбһҺбһ¶бһҷбһҸбһ¶бһҳбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҷбһӣбҹӢбһҠбһ№бһ„
- бһҖбҹҶбһҺбһҸбҹӢбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„ бһ“бһ·бһ„бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһӮбҹ„бһӣбһҖбҹҶбһҺбһҸбҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ’бһ“бһ’бһ¶бһ“

## бһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбһ·бһҖбҹ’бһҹбһ¶

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһ”бҹӢ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бҹҲ
- бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғ бһ“бһ·бһ„бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ’бһ“бһ’бһ¶бһ“ Azure бһҳбһ»бһ“бһҖбһ¶бһҡбһ”бһүбҹ’бһ….deploy
- бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– бһ“бһ·бһ„бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһӮбҹ„бһӣбһҖбҹҶбһҺбһҸбҹӢ
- бһҡбһ…бһ“бһ¶бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһўбһ¶бһ…бһҹбҹ’бһҖбҹҒбһӣбһ”бһ¶бһ“бһҠбҹ„бһҷбһӮбһ·бһҸбһӮбһјбһҡбһўбҹҶбһ–бһёбһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҸбҹҶбһ”бһ“бҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһҮбһ¶бһң
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҖбҹҶбһҺбһҸбҹӢбһ‘бҹҶбһ бҹҶбһ’бһ“бһ’бһ¶бһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҖбһ¶бһҡбһ„бһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—
- бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҮбһ¶бһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһ•бҹ’бһҸбһ»бҹҶбһҮбһ¶бһҳбһ»бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ бһ¶бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–
- бһӮбҹҶбһҡбҹ„бһ„бһ”бһүбҹ’бһ….deploy бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҸбҹҶбһ”бһ“бҹӢбһҠбҹ„бһҷбһ…бҹӮбһҖбһ…бһ¶бһҷбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң

## бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶ бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбҹҶбһҒбһ¶бһ“бҹӢ

бһҳбһ»бһ“бһ”бһүбҹ’бһ….deploy бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ’бһ¶бһ“бһ¶бҹҲ
- **бһӮбҹ„бһӣбһҖбҹҶбһҺбһҸбҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢ** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бһ“бһ’бһ¶бһ“бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ
- **бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ’бһ“бһ’бһ¶бһ“** бһ“бҹ…бһҸбҹҶбһ”бһ“бҹӢбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бһүбҹ’бһ….deploy
- **бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Tier** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҖбһ¶бһҡбһҮбһ¶бһңбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
- **бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ”бһҺбҹ’бһҠбһ¶бһү** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһҠбҹӮбһӣбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖ
- **бһҖбһ¶бһҡбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ…бҹҶбһҺбһ¶бһҷ** бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ‘бҹҶбһ бҹҶвҖӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң

## бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– бһҳбһ»бһ“бһҖбһ¶бһҡбһ”бһүбҹ’бһ….deploy

### бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·
```bash
#!/bin/bash
# capacity-check.sh - бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– Azure бһҳбһ»бһ“бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһІбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ

set -e

LOCATION=${1:-eastus2}
SUBSCRIPTION_ID=$(az account show --query id -o tsv)

echo "Checking Azure capacity for location: $LOCATION"
echo "Subscription: $SUBSCRIPTION_ID"
echo "======================================================"

# бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбҹҶбһҺбһҸбҹӢ
check_quota() {
    local resource_type=$1
    local required=$2
    
    echo "Checking $resource_type quota..."
    
    case $resource_type in
        "compute")
            usage=$(az vm list-usage --location "$LOCATION" --query "[?localName=='Total Regional vCPUs'].{current:currentValue,limit:limit}" -o json)
            current=$(echo "$usage" | jq -r '.[0].current')
            limit=$(echo "$usage" | jq -r '.[0].limit')
            ;;
        "storage")
            usage=$(az storage account show-usage --query "{current:value,limit:limit}" -o json)
            current=$(echo "$usage" | jq -r '.current')
            limit=$(echo "$usage" | jq -r '.limit')
            ;;
        "network")
            usage=$(az network list-usages --location "$LOCATION" --query "[?localName=='Virtual Networks'].{current:currentValue,limit:limit}" -o json)
            current=$(echo "$usage" | jq -r '.[0].current')
            limit=$(echo "$usage" | jq -r '.[0].limit')
            ;;
    esac
    
    available=$((limit - current))
    
    if [ "$available" -ge "$required" ]; then
        echo "вң… $resource_type: $available/$limit available (need $required)"
    else
        echo "вқҢ $resource_type: Only $available/$limit available (need $required)"
        return 1
    fi
}

# бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбҹҶбһҺбһҸбҹӢбһ’бһ“бһ’бһ¶бһ“бһ•бҹ’бһҹбҹҒбһ„бҹ—
check_quota "compute" 4      # бһ‘бһ¶бһҳбһ‘бһ¶бһҡ vCPU 4 бһҒбҹ’бһ‘бһ„бҹӢ
check_quota "storage" 2      # бһ‘бһ¶бһҳбһ‘бһ¶бһҡвҖӢбһӮбһҺбһ“бһёбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ 2 бһӮбһҺбһ“бһё
check_quota "network" 1      # бһ‘бһ¶бһҳбһ‘бһ¶бһҡвҖӢбһ”бһҺбҹ’бһҠбһ¶бһүбһңбҹҒбһҡбһ»бһҹбҹ’бһ‘бһҪбһҡ 1

echo "======================================================"
echo "вң… Capacity check completed successfully!"
```

### бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ

#### бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– App Service
```bash
# бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһӣбҹ’бһ”бһҝбһ“бһ•бҹӮбһ“бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё
check_app_service_capacity() {
    local location=$1
    local sku=$2
    
    echo "Checking App Service Plan capacity for $sku in $location"
    
    # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ SKU бһҗбҹ’бһҳбһёбһ“бҹ…бһҸбҹҶбһ”бһ“бҹӢ
    available_skus=$(az appservice list-locations --sku "$sku" --query "[?name=='$location']" -o tsv)
    
    if [ -n "$available_skus" ]; then
        echo "вң… $sku is available in $location"
    else
        echo "вқҢ $sku is not available in $location"
        
        # бһ•бҹ’бһҸбһӣбҹӢбһҹбҹҶбһҺбһҫбһҸбҹҶбһ”бһ“бҹӢбһҮбҹҶбһ“бһҪбһҹ
        echo "Available regions for $sku:"
        az appservice list-locations --sku "$sku" --query "[].name" -o table
        return 1
    fi
    
    # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“
    current_plans=$(az appservice plan list --query "length([?location=='$location' && sku.name=='$sku'])")
    echo "Current $sku plans in $location: $current_plans"
}

# бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ
check_app_service_capacity "eastus2" "P1v3"
```

#### бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– Database
```bash
# бһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– PostgreSQL
check_postgres_capacity() {
    local location=$1
    local sku=$2
    
    echo "Checking PostgreSQL capacity for $sku in $location"
    
    # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҗбһ¶ SKU бһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһ¬бһ‘бҹҒ
    available=$(az postgres flexible-server list-skus --location "$location" \
        --query "contains([].name, '$sku')" -o tsv)
    
    if [ "$available" = "true" ]; then
        echo "вң… PostgreSQL $sku is available in $location"
    else
        echo "вқҢ PostgreSQL $sku is not available in $location"
        
        # бһ”бһ„бҹ’бһ бһ¶бһү SKU бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“
        echo "Available PostgreSQL SKUs in $location:"
        az postgres flexible-server list-skus --location "$location" \
            --query "[].{name:name,tier:tier,vCores:vCores,memory:memorySizeInMb}" -o table
        return 1
    fi
}

# бһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– Cosmos DB
check_cosmos_capacity() {
    local location=$1
    local tier=$2
    
    echo "Checking Cosmos DB capacity in $location"
    
    # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҖбһ¶бһҡбһ…бһјбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҸбҹҶбһ”бһ“бҹӢ
    available_regions=$(az cosmosdb locations list --query "[?name=='$location']" -o tsv)
    
    if [ -n "$available_regions" ]; then
        echo "вң… Cosmos DB is available in $location"
        
        # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҗбһ¶ serverless бһ”бһ¶бһ“бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ¬бһ‘бҹҒ (бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ)
        if [ "$tier" = "serverless" ]; then
            serverless_regions=$(az cosmosdb locations list \
                --query "[?supportsAvailabilityZone==true && name=='$location']" -o tsv)
            
            if [ -n "$serverless_regions" ]; then
                echo "вң… Cosmos DB Serverless is supported in $location"
            else
                echo "вҡ пёҸ  Cosmos DB Serverless may not be supported in $location"
            fi
        fi
    else
        echo "вқҢ Cosmos DB is not available in $location"
        return 1
    fi
}
```

## рҹ“Қ бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҸбҹҶбһ”бһ“бҹӢ

### бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҸбһ¶бһҳбһҸбҹҶбһ”бһ“бҹӢ
```bash
# бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһ—бһ¶бһ–бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ“бҹ…бһҸбҹҶбһ”бһ“бҹӢбһ“бһ¶бһ“бһ¶
check_service_availability() {
    local service=$1
    
    echo "Checking $service availability across regions..."
    
    case $service in
        "appservice")
            az appservice list-locations --query "[].{region:name,displayName:displayName}" -o table
            ;;
        "containerapp")
            az provider show --namespace Microsoft.App \
                --query "resourceTypes[?resourceType=='containerApps'].locations" -o table
            ;;
        "postgres")
            az postgres flexible-server list-skus --location eastus2 >/dev/null 2>&1 && \
            echo "PostgreSQL Flexible Server regions:" && \
            az account list-locations --query "[?metadata.regionType=='Physical'].{name:name,displayName:displayName}" -o table
            ;;
        "cosmosdb")
            az cosmosdb locations list --query "[].{name:name,documentationUrl:documentationUrl}" -o table
            ;;
    esac
}

# бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ
for service in appservice containerapp postgres cosmosdb; do
    check_service_availability "$service"
    echo ""
done
```

### бһ“бһ№бһ„бһ•бҹ’бһҸбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚбһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҸбҹҶбһ”бһ“бҹӢ

```bash
# бһҺбҹӮбһ“бһ¶бҹҶбһҸбҹҶбһ”бһ“бҹӢбһҠбҹӮбһӣбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ
```

```bash
    az deployment group validate \

        --parameters @infra/main.parameters.json \

    az group delete --name "$temp_rg" --yes --no-wait

    echo "рҹ’Ў Use Azure Pricing Calculator for detailed cost estimates:"
    echo "   https://azure.microsoft.com/pricing/calculator/"
    echo ""
    echo "рҹ’Ў Consider using Azure Cost Management for ongoing monitoring:"
    echo "   https://portal.azure.com/#blade/Microsoft_Azure_CostManagement/Menu/overview"
```

```bash
        deployment_status=$(az deployment group list \
            --resource-group "$resource_group" \
            --query "[0].properties.provisioningState" -o tsv)
        
        if [ "$deployment_status" = "Succeeded" ]; then
            log_info "Deployment completed successfully"
            break
        elif [ "$deployment_status" = "Failed" ]; then
            log_error "Deployment failed"
            break
        fi
        
        # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ’бһ“бһ’бһ¶бһ“бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“
        current_resources=$(az resource list \
            --resource-group "$resource_group" \
            --query "length([])")
        
        echo "$(date): Deployment in progress, $current_resources resources created"
        sleep 30
    done
}
```
