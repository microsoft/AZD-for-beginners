# бһӣбҹҶбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“

вҸұпёҸ **бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҠбҹӮбһӣбһ”бҹүбһ¶бһ“бҹӢбһ”бҹ’бһҡбһҳбһ¶бһҺ**: бҹҰбҹ -бҹ§бҹҘ бһ“бһ¶бһ‘бһё | рҹ’° **бһҖбһ¶бһҡбһ…бҹҶбһҺбһ¶бһҷбһҠбҹӮбһӣбһ”бҹүбһ¶бһ“бҹӢбһ”бҹ’бһҡбһҳбһ¶бһҺ**: ~$бҹЎбҹ бҹ -бҹЈбҹ бҹ /бһҒбҹӮ | вӯҗ **бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү**: бһҮбһ¶бһ“бҹӢбһҒбҹ’бһ–бһҹбҹӢ

**рҹ“ҡ бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҡбҹҖбһ“:**
- вҶҗ бһҳбһ»бһ“бһ“бҹҒбҹҮ: [бһ•бҹӮбһ“бһҖбһ¶бһҡбһ…бҹҶбһ бһ¶бһҷ](capacity-planning.md) - бһңбһ·бһ’бһ¶бһ“бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ’бһ“бһ’бһ¶бһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖ
- рҹҺҜ **бһўбҹ’бһ“бһҖбһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ**: бһӣбҹҶбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“ (бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶ, бһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„, бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–)
- вҶ’ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ: [бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ SKU](sku-selection.md) - бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Azure бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
- рҹҸ  [бһ‘бҹҶбһ–бҹҗбһҡбһҳбҹҒбһңбһӮбҹ’бһӮ](../../README.md)

## бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“

бһҠбҹ„бһҷбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһ¶бһ“бҹ–
- бһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһӣбҹҶбһ“бһ¶бҹҶбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“ бһ“бһ·бһ„бһ–бҹҒбһӣбһҺбһ¶вҖӢбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫ
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹҶбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ (бһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣ, бһ…бҹӮбһҖбһ…бһ¶бһҷ, бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҸбҹ’бһҡбһ„бҹӢ)
- бһҡбһ…бһ“бһ¶бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ (бһҹбһҳбҹҗбһҷбһҹбһҳбҹҗбһҷ, бһҳбһ·бһ“бһҹбһҳбҹҗбһҷбһҹбһҳбҹҗбһҷ, бһ“бһ·бһҷбһҳбһ“бҹҗбһҷбһҠбҹ„бһҷбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ)
- бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһӣбһҫбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷ
- бһ”бһүбҹ’бһ…бһјбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһӣбһҫ Azure бһҮбһ¶бһҳбһҪбһҷ AZD
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹҶбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҒбһҺбһ¶бһҡбһёбһҷбҹүбһј AI бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ
- бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бһүбҹ’бһ бһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷ

## бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶ бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢ

### бһҖбһ¶бһҡбһңбһҢбҹ’бһҚбһ”бҹҶбһ•бһ»бһҸ: бһ–бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбһ‘бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“

**бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹӮбһҳбһҪбһҷ (бһҹбһ¶бһҳбһүбҹ’бһү):**
```
User вҶ’ Agent вҶ’ Response
```
- вң… бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҷбһӣбҹӢ бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸ
- вң… бһӣбһҝбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҹбһ¶бһҳбһүбҹ’бһү
- вқҢ бһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҳбҹүбһјбһҠбҹӮбһӣбһҸбҹӮбһҳбһҪбһҷ
- вқҢ бһҳбһ·бһ“бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһ бһ»бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ”бһ¶бһ“
- вқҢ бһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһҡбһ·бһ…бҹ’бһҶбҹҒбһ‘бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ

**бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“ (бһҒбҹ’бһ–бһҹбҹӢ):**
```mermaid

    Orchestrator[бһўбҹ’бһ“бһҖбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸ] --> Agent1[бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ1<br/>бһ•бҹӮбһ“бһҖбһ¶бһҡ]

    Orchestrator --> Agent3[бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ3<br/>бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһЎбһҫбһ„бһңбһ·бһү]
```- вң… бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһҡбһ·бһ…бҹ’бһҶбҹҒбһ‘бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ
- вң… бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһ бһ»бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹ„бһҷбһӣбһҝбһ“
- вң… бһҳбҹүбһјбһҢбһ»бһӣ бһ“бһ·бһ„бһҗбҹӮбһ‘бһ¶бҹҶбһ”бһ¶бһ“бһӣбҹ’бһў

- вҡ пёҸ бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҷбһӣбҹӢбһўбҹҶбһ–бһёбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ

**бһ”бҹҶбһҡбҹӮбһ”бҹҶбһҡбһҪбһӣ**: бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбһӮбһәбһҠбһјбһ…бһҳбһ“бһ»бһҹбҹ’бһҹбһҳбҹ’бһ“бһ¶бһҖбҹӢбһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһҸбҹӮбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһӮбһәбһҠбһјбһ…бһҖбҹ’бһҡбһ»бһҳбһҳбһ“бһ»бһҹбҹ’бһҹбһҠбҹӮбһӣбһҳбҹ’бһ“бһ¶бһҖбҹӢбҹ—бһҳбһ¶бһ“бһҮбҹҶбһ“бһ¶бһүбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ (бһўбҹ’бһ“бһҖбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң, бһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡ, бһўбҹ’бһ“бһҖбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ, бһўбҹ’бһ“бһҖбһ“бһ·бһ–бһ“бҹ’бһ’) бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶бҹ”

## бһӣбҹҶбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—

### бһӣбҹҶбһ“бһ¶бҹҶ бҹЎбҹ– бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢ (бһҒбҹ’бһҹбҹӮбһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһң)

**бһ–бҹҒбһӣбһҺбһ¶бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫ**: бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҸбҹ’бһҡбһјбһңбһ”бһүбҹ’бһ…бһ”бҹӢбһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ“бһёбһҳбһҪбһҷбҹ—бһ”бһ„бҹ’бһҖбһҫбһҸбһӣбһ‘бҹ’бһ’бһ•бһӣбһ–бһёбһҳбһ»бһ“бҹ”

- бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү (бһҡбҹҖбһ”бһ…бҹҶ вҶ’ бһ…бҹӮбһҖбһ…бһ¶бһҷбһ’бһ“бһ’бһ¶бһ“ вҶ’ бһўбһ“бһ»бһңбһҸбҹ’бһҸ вҶ’ бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ)

### бһӣбҹҶбһ“бһ¶бҹҶ бҹӨбҹ– бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҠбҹ„бһҷбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ (бһ•бҹ’бһҹбһ¶бһҷ-бһҮбһ¶бһң)

**бһ–бҹҒбһӣбһҺбһ¶бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫ**: бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһўбһ¶бһ…бһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ‘бҹ…бһ“бһ№бһ„бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ бһҳбһ¶бһ“бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһҲбҹ’бһҳбҹ„бҹҮбһҸбһ·бһ…бһҸбһҪбһ…бҹ”

- вң… бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбһ·бһ“бһҹбһҳбҹҗбһҷбһҹбһҳбҹҗбһҷ
- вң… бһҳбһ¶бһ“бһ—бһ¶бһ–бһҡбһ№бһ„бһҳбһ¶бҹҶ (бһ•бҹ’бһ‘бһ»бһҖбһҹбһ¶бһҡбһ”бһ¶бһ“)

**бһҖбҹҶбһҺбһҸбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸ:**
- вҡ пёҸ бһ—бһ¶бһ–бһҜбһҖбһ—бһ¶бһ–бһ“бҹ…бһ–бҹҒбһӣбһҖбҹ’бһҡбҹ„бһҷ
- вҡ пёҸ бһ–бһ·бһ”бһ¶бһҖбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбҹҶбһ бһ»бһҹ
- вҡ пёҸ бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһ”бһ„бҹ’бһ бһ¶бһүбһҹбһ¶бһҡбһӣбҹҶбһ”бһ¶бһҖ

**бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ:**
- бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ–бһ·бһҸ (бһҖбһ¶бһҡбһ–бҹ’бһҡбһҳбһ¶бһ“, бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ”бһ„бҹ’бһ бһ¶бһү, бһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»)
- бһҹбҹҒбһңбһ¶вҖӢбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһ…бҹ’бһҡбһҫбһ“бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ (бһўбҹҠбһёбһҳбҹӮбһӣ, SMS, бһҮбһјбһ“бһ•бҹ’бһҹбһ¶бһҷ, Slack)
- бһ”бһҺбҹ’бһҸбһ¶бһүбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбһјбһ…бһӮбҹ’бһ“бһ¶)

### бһӣбҹҶбһ“бһ¶бҹҶ бҹҘбҹ– бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҠбҹ„бһҷбһҖбһ¶бһҡбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“ (бһҶбҹ’бһ“бҹ„бһҸ / Quorum)

**бһ–бҹҒбһӣбһҺбһ¶бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫ**: бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳбһ–бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһҳбһ»бһ“бһ–бҹҒбһӣбһ”бһ“бҹ’бһҸбҹ”

```python
from azure.cosmos import CosmosClient, PartitionKey
from datetime import datetime
import os

class StateManager:
    """Manages shared state across agents using Cosmos DB"""
    
    def __init__(self):
        endpoint = os.environ['COSMOS_ENDPOINT']
        key = os.environ['COSMOS_KEY']
        
        self.client = CosmosClient(endpoint, key)
        self.database = self.client.get_database_client('agent-state')
        self.container = self.database.get_container_client('tasks')
    
    def create_task(self, task_id: str, task_type: str, input_data: dict):
        """Create a new task"""
        task = {
            'id': task_id,
            'type': task_type,
            'status': 'pending',
            'input': input_data,
            'created_at': datetime.utcnow().isoformat(),
            'steps': []
        }
        self.container.create_item(task)
        return task
    
    def update_task_step(self, task_id: str, step_name: str, result: dict):
        """Update task with completed step"""
        task = self.container.read_item(task_id, partition_key=task_id)
        
        task['steps'].append({
            'name': step_name,
            'completed_at': datetime.utcnow().isoformat(),
            'result': result
        })
        
        self.container.replace_item(task_id, task)
        return task
    
    def complete_task(self, task_id: str, final_result: dict):
        """Mark task as complete"""
        task = self.container.read_item(task_id, partition_key=task_id)
        task['status'] = 'completed'
        task['result'] = final_result
        task['completed_at'] = datetime.utcnow().isoformat()
        self.container.replace_item(task_id, task)
        return task
    
    def get_task(self, task_id: str):
        """Retrieve task state"""
        return self.container.read_item(task_id, partition_key=task_id)
```

    Sequential workflow: Research вҶ’ Write вҶ’ Edit вҶ’ Publish

    topic = data.get('topic')
    
    if not topic:
        return jsonify({'error': 'Topic required'}), 400

    sender = servicebus_client.get_queue_sender('research-tasks')

        body=json.dumps({

            'topic': topic,
            'next_queue': 'writer-tasks'  # бһ‘бһёбһҖбһ“бҹ’бһӣбҹӮбһ„бһ•бҹ’бһүбһҫбһӣбһ‘бҹ’бһ’бһ•бһӣ
        }),
        content_type='application/json'
    )
    
    with sender:
        sender.send_messages(message)
    
    return jsonify({
        'task_id': task_id,
        'status': 'started',
        'workflow': 'sequential',
        'steps': ['research', 'write', 'edit', 'publish'],
        'message': 'Content creation pipeline initiated'
    }), 202

@app.route('/task/<task_id>', methods=['GET'])

```python
from azure.servicebus import ServiceBusClient
import json
import os
from collections import defaultdict
from shared.state_manager import StateManager

state_manager = StateManager()
servicebus_client = ServiceBusClient.from_connection_string(
    os.environ['SERVICEBUS_CONNECTION_STRING']
)

# бһҸбһ¶бһҳбһҠбһ¶бһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбһ¶бһӣбҹӢбһ”бһ¶бһҸбһ»бһҖбһҳбҹ’бһҳ
task_results = defaultdict(list)
expected_agents = 4  # бһңбҹҒбһ”бһҹбһ¶бһҷ, бһҹбһ·бһҖбҹ’бһҹбһ¶бһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң, бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“, бһҹбһ„бҹ’бһӮбһҳ

def process_result(message_data):
    """Aggregate results from parallel agents"""
    task_id = message_data['task_id']
    agent_type = message_data['agent_type']
    result = message_data['result']
    
    # бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһӣбһ‘бҹ’бһ’бһ•бһӣ
    task_results[task_id].append({
        'agent': agent_type,
        'data': result
    })
    
    print(f"рҹ“Ҡ Received result from {agent_type} agent ({len(task_results[task_id])}/{expected_agents})")
    
    # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҗбһ¶бһҸбһҫбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢбһ бһҫбһҷбһ¬бһ“бҹ… (fan-in)
    if len(task_results[task_id]) == expected_agents:
        print(f"вң… All agents completed for task {task_id}. Aggregating...")
        
        # бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣ
        aggregated = {
            'query': message_data['query'],
            'sources': task_results[task_id],
            'summary': generate_summary(task_results[task_id])
        }
        
        # бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҗбһ¶бһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢ
        state_manager.complete_task(task_id, aggregated)
        
        # бһҠбһ¶бһҖбҹӢбһўбҹ„бһҷбһҹбҹ’бһўбһ¶бһҸбһЎбһҫбһ„бһңбһ·бһү
        del task_results[task_id]
        
        print(f"вң… Aggregation complete for task {task_id}")

def generate_summary(results):
    """Generate summary from all sources"""
    summaries = [r['data'].get('summary', '') for r in results]
    return '\n\n'.join(summaries)

def main():
    """Listen to aggregation queue"""
    receiver = servicebus_client.get_queue_receiver('aggregation-queue')
    
    print("рҹ“Ҡ Aggregator started, listening for results...")
    
    with receiver:
        while True:
            messages = receiver.receive_messages(max_wait_time=5)
            for message in messages:
                try:
                    message_data = json.loads(str(message))
                    process_result(message_data)
                    receiver.complete_message(message)
                except Exception as e:
                    print(f"вқҢ Error: {e}")
                    receiver.abandon_message(message)

if __name__ == '__main__':
    main()

### бһӣбҹҶбһ бһ¶бһҸбҹӢ бҹўбҹ– бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ…бҹ’бһ”бһ¶бһ”бҹӢ Retry вӯҗвӯҗвӯҗ (бһҮбһ¶бһ“бҹӢбһҒбҹ’бһ–бһҹбҹӢ)

**бһӮбҹ„бһӣбһҠбҹ…**: Retry бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһҡбһ¶бһҮбҹҗбһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҳбһ»бһ“бһҖбһ¶бһҡбһ”бҹ„бҹҮбһ”бһ„бҹӢбҹ”

**бһҮбҹҶбһ бһ¶бһ“**:

1. **бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“ retry бһ‘бҹ…бһўбҹ’бһ“бһҖбһҡбҹҖбһ”бһ…бҹҶбҹ–**

```python
from dataclasses import dataclass
from typing import Dict

@dataclass
class RetryConfig:
    max_retries: int = 3
    backoff_seconds: int = 5

retry_counts: Dict[str, int] = {}  # message_id -> бһ…бҹҶбһ“бһҪбһ“бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҳбҹ’бһҠбһ„бһҳбҹ’бһҖбһ¶бһӣ

def send_with_retry(queue_name: str, message_data: dict, retry_config: RetryConfig):
    """Send message with retry metadata"""
    message_id = message_data.get('message_id', str(uuid.uuid4()))
    message_data['message_id'] = message_id
    message_data['retry_count'] = retry_counts.get(message_id, 0)
    message_data['max_retries'] = retry_config.max_retries
    
    sender = servicebus_client.get_queue_sender(queue_name)
    message = ServiceBusMessage(
        body=json.dumps(message_data),
        content_type='application/json',
        message_id=message_id
    )
    
    with sender:
        sender.send_messages(message)
```

```python
def process_with_retry(message, receiver, process_func):
    """Process message with automatic retry on failure"""
    try:
        message_data = json.loads(str(message))
        
        # бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҹбһ¶бһҡбһ“бҹҒбҹҮ
        process_func(message_data)
        
        # бһҹбһҳбҹ’бһҡбҹҒбһ…бһ”бһ¶бһ“бһҮбҹ„бһӮбһҮбҹҗбһҷ - бһ”бһүбҹ’бһ…бһ”бҹӢ
        receiver.complete_message(message)
        
    except Exception as e:
        message_id = message.message_id
        retry_count = message_data.get('retry_count', 0)
        max_retries = message_data.get('max_retries', 3)
        
        if retry_count < max_retries:
            # бһ’бҹ’бһңбһҫбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбҹ– бһ”бҹ„бҹҮбһ”бһ„бҹӢбһ бһҫбһҷбһҠбһ¶бһҖбҹӢбһ…бһјбһӣбһҮбһ¶бһҡбһ„бҹ’бһҹбһёбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбһҮбһ¶бһҳбһҪбһҷбһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбһҫбһ„
            print(f"вҡ пёҸ Retry {retry_count + 1}/{max_retries} for message {message_id}")
            
            message_data['retry_count'] = retry_count + 1
            
            # бһ•бҹ’бһүбһҫбһҸбҹ’бһҡбһЎбһ”бҹӢбһ‘бҹ…бһҡбһ„бҹ’бһҹбһёбһҠбһҠбҹӮбһӣбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ–бһ“бҹ’бһҷбһ¶бһҡбһ–бҹҒбһӣ
            time.sleep(5 * (retry_count + 1))  # бһҖбһ¶бһҡбһӣбҹӮбһ„бһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҮбҹҶбһҡбһ»бһү
            send_with_retry(queue_name, message_data, RetryConfig())
            
            receiver.complete_message(message)  # бһӣбһ»бһ”бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҠбһҫбһҳ
        else:
            # бһӣбһҫбһҹбһ…бҹҶбһ“бһҪбһ“бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһўбһҸбһ·бһ”бһҡбһҳбһ¶ - бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бҹ…бһҡбһ„бҹ’бһҹбһёбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҲбһ”бҹӢбһҹбһҳбҹҗбһҷ
            print(f"вқҢ Max retries exceeded for message {message_id}")
            receiver.dead_letter_message(
                message,
                reason="MaxRetriesExceeded",
                error_description=str(e)
            )
```

        self.failure_threshold = failure_threshold
        self.timeout_seconds = timeout_seconds
        self.failure_count = 0
        self.last_failure_time = None
        self.state = CircuitState.CLOSED
    
    def call(self, func):
        """Execute function with circuit breaker protection"""
        if self.state == CircuitState.OPEN:
            # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҗбһ¶бһҸбһҫбһ–бҹҒбһӣбһҹбҹҶбһҡбһ¶бһҖбһ”бһ¶бһ“бһ•бһ»бһҸбһҠбҹӮбһҡ
            if datetime.utcnow() - self.last_failure_time > timedelta(seconds=self.timeout_seconds):
                self.state = CircuitState.HALF_OPEN
                print("рҹ”„ Circuit breaker: HALF_OPEN (testing)")
            else:
                raise Exception(f"Circuit breaker OPEN for agent. Try again in {self.timeout_seconds}s")
        
        try:
            result = func()
            
            # бһҮбҹ„бһӮбһҮбҹҗбһҷ
            if self.state == CircuitState.HALF_OPEN:
                self.state = CircuitState.CLOSED
                self.failure_count = 0
                print("вң… Circuit breaker: CLOSED (recovered)")
            
            return result
            
        except Exception as e:
            self.failure_count += 1
            self.last_failure_time = datetime.utcnow()
            
            if self.failure_count >= self.failure_threshold:
                self.state = CircuitState.OPEN
                print(f"рҹ”ҙ Circuit breaker: OPEN (too many failures)")
            
            raise e
```

```python
from opencensus.ext.azure.log_exporter import AzureLogHandler
from opencensus.ext.azure.trace_exporter import AzureExporter
from opencensus.trace import config_integration
from opencensus.trace.tracer import Tracer
from opencensus.trace.samplers import AlwaysOnSampler

import os

    sampler=AlwaysOnSampler()
)

# бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҖбһ¶бһҡбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»

logger.setLevel(logging.INFO)

def trace_agent_call(agent_name, task_id, operation):
    """Trace agent operations"""
    with tracer.span(name=f'{agent_name}.{operation}') as span:
        span.add_attribute('agent', agent_name)
        span.add_attribute('task_id', task_id)
        span.add_attribute('operation', operation)
        
        try:
            result = operation()
            span.add_attribute('status', 'success')
            return result
        except Exception as e:
            span.add_attribute('status', 'error')
            span.add_attribute('error', str(e))
            raise

   # вқҢ бһўбһ¶бһҖбҹ’бһҡбһҖбҹӢ: бһҮбһҳбҹ’бһҡбҹҮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһ”бһҫбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбҹ’бһ“бһ¶бһҖбҹӢбһ”бһҡбһ¶бһҮбҹҗбһҷ

### бһ”бһүбҹ’бһ бһ¶бҹҲ бһҹбһ¶бһҡбһўбһ„бҹ’бһӮбһ»бһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҮбһҪбһҡбһҠбҹҒбһҖ

**бһҡбҹ„бһӮбһҹбһүбҹ’бһүбһ¶бҹҲ**
- бһҹбһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ“бҹ’бһҹбһӣбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҮбһҪбһҡбһҠбҹҒбһҖ
- бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһ·бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ
- бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҲбһҡбһ“бҹ… "бһҖбҹҶбһ–бһ»бһ„бһҡбһ„бҹӢбһ…бһ¶бҹҶ"

**бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбҹҲ**
```bash
# бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҮбһҳбҹ’бһҡбҹ…бһҮбһҪбһҡбһҠбһ¶бһҖбҹӢ
az servicebus queue show \
  --namespace-name mybus \
  --name research-tasks \
  --query "countDetails"

# бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһҒ Р°РіРөРҪСӮ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Azure CLI
az containerapp logs show --name research-agent --resource-group $RG_NAME --tail 50
```

**бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹҲ**

1. **бһ”бһ„бҹ’бһҖбһҫбһ“бһ…бҹҶбһ“бһҪбһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ:**

     --min-replicas 3 \

   ```

- бһҳбһ·бһ“бһҳбһ¶бһ“бһҹбһ¶бһҡбһҖбҹҶбһ бһ»бһҹбһҺбһ¶бһҳбһҪбһҷ
