## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…

бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбһ¶бһҖбҹӢбһІбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ“бҹ’бһ‘бһ»бһҖбһҖбһ¶бһҡбһ„бһ¶бһҡ AI бһҠбҹӮбһӣбһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Azure Developer CLI (AZD)бҹ” бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһ–бһёбһҹбһ бһӮбһҳбһ“бҹҚ Microsoft Foundry Discord бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ…бҹҒбһүбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ–бһ·бһҸбһ“бҹ…бһ—бһ–бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһҹбһ¶бһҡбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбҹ”

## бһ”бһүбҹ’бһ бһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҠбҹӮбһӣбһ”бһ¶бһ“бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ

бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһӣбһ‘бҹ’бһ’бһ•бһӣбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбҹ’бһ‘бһ„бҹӢбһҳбһҸбһ·бһ–бһёбһҹбһ бһӮбһҳбһ“бҹҚ бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶вҖӢбһ”бһүбҹ’бһ бһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—вҖӢбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бҹ’бһҡбһҲбһҳбһҳбһ»бһҒбҹ–

- **45%** бһ”бҹ’бһҡбһҲбһҳбһҳбһ»бһҒбһ“бһ№бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ…бҹҒбһү AI бһҠбҹӮбһӣбһҳбһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ…бҹ’бһҡбһҫбһ“
- **38%** бһҳбһ¶бһ“бһ”бһүбҹ’бһ бһ¶бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҜбһҖбһҹбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҠбҹҒбһҸбһ¶бһ”бһ“бҹ’бһҗбҹӮбһҳ бһ“бһ·бһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ  
- **35%** бһҡбһҖбһғбһҫбһүбһҗбһ¶бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖвҖӢбһҳбһ¶бһ“бһ—бһ¶бһ–бһӣбҹҶбһ”бһ¶бһҖ
- **32%** бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ…бҹҶбһҺбһ¶бһҷбһӣбҹ’бһўбһҮбһ¶бһ„бһ“бҹҒбҹҮ
- **29%** бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһ»бһҖбһҡбһҖбһҖбҹҶбһ бһ»бһҹбһҖбһ¶бһ“бҹӢбһҸбҹӮбһ”бҹ’бһҡбһҹбһҫбһҡ

## бһӮбҹҶбһҡбһјбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ AI бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ

### бһӮбҹҶбһҡбһј 1: бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ AI бһҳбһёбһҖбҹ’бһҡбһјбһҹбҹҒбһңбһ¶

**бһ–бҹҒбһӣбһ”бҹ’бһҡбһҫ**: бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү бһҠбҹӮбһӣбһҳбһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡвҖӢбһ…бҹ’бһҡбһҫбһ“

### бһӮбҹҶбһҡбһј 2: бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ AI (Event-Driven)

**бһ–бҹҒбһӣбһ”бҹ’бһҡбһҫ**: бһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһҮбһ¶бһӮбһјбһ” (batch processing), бһңбһ·бһ—бһ¶бһӮбһҜбһҖбһҹбһ¶бһҡ, бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбһ·бһ“бһҹбһҳбҹҗбһҷ (asynchronous workflows)

## бһҖбһ¶бһҡбһӮбһ·бһҸбһўбҹҶбһ–бһёбһҹбһ»бһҒбһ—бһ¶бһ–бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI

бһ–бҹҒбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһ”бҹӮбһ”бһ”бһ»бһҡбһ¶бһҺбһҒбһјбһ… бһҖбһ¶бһҡбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһўбһ¶бһҖбһ¶бһҡбҹҲбһӮбһәбһҹбҹ’бһӮбһ¶бһӣбҹӢбһ”бһ¶бһ“: бһ‘бҹҶбһ–бҹҗбһҡбһҳбһ·бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ, API бһҸбҹ’бһҡбһЎбһ”бҹӢбһҖбҹҶбһ бһ»бһҹ, бһ¬бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ…бҹҒбһүбһ”бһҡбһ¶бһҮбҹҗбһҷбҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮбһҠбҹ„бһҷ AI бһўбһ¶бһ…бһҒбһјбһ…бһҠбҹ„бһҷбһҳбһ¶бһ“бһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮвҖ”бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ¶бһҖбҹҸбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҖбҹ’бһ“бһ»бһ„бһңбһ·бһ’бһёбһҠбҹӮбһӣбһҹбҹ’бһҸбһҫбһ„бһҹбҹ’бһ‘бһ¶бһҸбҹӢ бһ“бһ·бһ„бһҳбһ·бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһҹбһ¶бһҡбһҖбҹҶбһ бһ»бһҹбһ…бҹ’бһ”бһ¶бһҹбҹӢбҹ”

### бһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ–бҹҒбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҳбһ·бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһӣбһҖбҹ’бһҒбһҺбҹҲбһҠбһјбһ…бһҠбҹӮбһӣбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖ

бһ”бһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбһҠбҹӮбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҡбҹҶбһ–бһ№бһ„ бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҸбһ¶бһҳбһ”бһүбҹ’бһҮбһёбһӮбҹҶбһ“бһ·бһҸбһ“бҹҒбҹҮбҹ–

1. **бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бҹ”** бһҸбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бҹҒ? бһҸбһҫбһңбһ¶бһўбһ¶бһ…бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһңбһ¶бһ‘бҹҒ? бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ `azd monitor` бһ“бһ·бһ„бһҹбһ»бһҒбһ—бһ¶бһ–бһ’бһ“бһ’бһ¶бһ“ бһҠбһјбһ…бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ’бҹ’бһңбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҺбһ¶бһҳбһҪбһҷбҹ”
2. **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҳбҹүбһјбһҠбҹӮбһӣбҹ”** бһҸбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ бҹ…бһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹ„бһҷбһҮбҹ„бһӮбһҮбҹҗбһҷбһ‘бҹҒ? бһҖбһҡбһҺбһёбһ бҹ…бһҳбҹүбһјбһҠбҹӮбһӣбһ”бһҡбһ¶бһҮбҹҗбһҷ бһ¬аё«аёЎаё”бһ–бҹҒбһӣбһҮбһ¶бһ‘бһјбһ‘бҹ…бһҮбһ¶бһҳбһјбһӣбһ бҹҒбһҸбһ»бһ“бһ·бһ„бһ”бҹҶбһ•бҹ’бһӣбһ¶бһүбһ”бһүбҹ’бһ бһ¶ AI бһ бһҫбһҷбһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
3. **бһҳбһҫбһӣбһўбҹ’бһңбһёбһҠбҹӮбһӣбһҳбҹүбһјбһҠбҹӮбһӣбһ”бһ¶бһ“бһ‘бһ‘бһҪбһӣбҹ”** бһ…бһҳбҹ’бһӣбһҫбһҷ AI бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһ”бһүбҹ’бһ…бһјбһӣ (prompt бһ“бһ·бһ„ context бһҠбҹӮбһӣбһ”бһ¶бһ“бһҷбһҖбһҳбһҖ)бҹ” бһ”бһҫбһӣбһ‘бҹ’бһ’бһ•бһӣбһҒбһ»бһҹ бһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢбһҠбҹӮбһҡбҹ” бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҗбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ•бҹ’бһүбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ‘бҹ…бһҳбҹүбһјбһҠбҹӮбһӣбһ¬бһўбһҸбҹӢбҹ”
4. **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ–бҹҒбһӣбһҷбһәбһҸбһ“бҹғбһ…бһҳбҹ’бһӣбһҫбһҷбҹ”** бһҖбһ¶бһҡбһ бҹ…бһҳбҹүбһјбһҠбҹӮбһӣ AI бһҷбһәбһҸбһҮбһ¶бһ„бһҖбһ¶бһҡбһ бҹ… API бһ‘бһјбһ‘бҹ…бҹ” бһ”бһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһ¶бһ“бһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҷбһәбһҸ бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҗбһ¶бһ–бҹҒбһӣбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҳбҹүбһјбһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбһҫбһ“бһЎбһҫбһ„бһ¬бһўбһҸбҹӢвҖ”бһ“бҹҒбҹҮбһўбһ¶бһ…бһҹбһүбҹ’бһүбһ¶бһҖбһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһҖбһ¶бһҸбҹӢбһ…бһҡ (throttling), бһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– бһ¬бһҖбһ¶бһҡбһ§бһ”бһ‘бҹ’бһ”бһ¶бһүбһҖбһҳбҹ’бһҡбһ·бһҸбһҸбҹҶбһ”бһ“бҹӢбҹ”
5. **бһҸбһ¶бһҳбһҠбһ¶бһ“бһҹбһүбҹ’бһүбһ¶бһ…бҹҶбһҺбһ¶бһҷбҹ”** бһҖбһ¶бһҡбһҖбһҫбһ“бһЎбһҫбһ„бһҠбҹҸбһҳбһ·бһ“бһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ token бһ¬бһҖбһ¶бһҡбһ бҹ… API бһўбһ¶бһ…бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бһ„бҹ’бһңбһ·бһӣ, prompt бһҖбҹҶбһҺбһҸбҹӢбһҒбһ»бһҹ бһ¬бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһ“бһЎбһҫбһ„бһңбһ·бһүбһ…бҹ’бһҡбһҫбһ“бһ–бҹҒбһҖбҹ”

бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһұбҹ’бһҷбһҳбһ¶бһ“бһҮбҹҶбһ“бһ¶бһүбһҒбҹ’бһ–бһҹбҹӢбһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ observability бһ–бҹҒбһӣбһ—бҹ’бһӣбһ¶бһҳбһ‘бҹҒбҹ” бһ…бҹҶбһҺбһ»бһ…бһҹбҹҶбһҒбһ¶бһ“бҹӢбһӮбһәбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҳбһ¶бһ“бһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһ”бһ“бҹ’бһҗбҹӮбһҳбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҸбһ¶бһҳбһҠбһ¶бһ“ бһ бһҫбһҷбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҠбҹӮбһӣ built-in бһҡбһ”бһҹбҹӢ azd (`azd monitor`) бһ•бҹ’бһҸбһӣбҹӢбһҮбһ¶бһ…бҹҶбһҺбһ»бһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбҹ”

**бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ…бҹҶбһҺбһ¶бһҷ OpenAI**:

```typescript

class TokenOptimizer {
  private readonly maxTokens = 4000;
  private readonly reserveTokens = 500;
  
  optimizePrompt(userInput: string, context: string): string {
    const availableTokens = this.maxTokens - this.reserveTokens;
    const estimatedTokens = this.estimateTokens(userInput + context);
    
    if (estimatedTokens > availableTokens) {
      // бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ”бһҡбһ·бһ”бһ‘ бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ
      context = this.truncateContext(context, availableTokens - this.estimateTokens(userInput));
    }
    
    return `${context}\n\nUser: ${userInput}`;
  }
  
  private estimateTokens(text: string): number {
    // бһҖбһ¶бһҡбһ”бҹүбһ¶бһ“бҹӢбһҹбҹ’бһҳбһ¶бһ“бһ”бҹ’бһҡбһ бҹӮбһӣбҹ– 1 бһҸбһјбһҖбһ·бһ“ вүҲ 4 бһҸбһҪбһўбһҖбҹ’бһҹбһҡ
    return Math.ceil(text.length / 4);
  }
}
```

> бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ `azure.ai.agents` бһҖбҹҶбһ–бһ»бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһҷбҹүбһ¶бһ„бһӣбһҝбһ“бҹ” бһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ”бҹ’бһҡбһҶбһ¶бҹҶбһ„бһ“бһ№бһ„ `0.1.40-preview`бҹ” бһ”бһҫбһҖбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶ `azd extension upgrade --all` бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһҖбһҹбҹҶбһҺбһ»бһҖбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҗбҹ’бһҳбһёбҹ— бһ“бһ·бһ„ `azd extension show azure.ai.agents` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҖбҹҶбһҺбҹӮбһҠбҹӮбһӣбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бҹ”

**бһўбҹ’бһңбһёбһ‘бҹ…бһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҗбҹ’бһҳбһё `skills` бһ“бһ·бһ„ `connections`?**

бһҳбһ¶бһ“бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ preview бһ–бһёбһҡбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһҡбһҪбһ…бһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһ бһҫбһҷбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ‘бҹ„бҹҮбһўбҹ’бһ“бһҖбһҮбһ¶бһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһҖбҹҸбһҠбҹ„бһҷбҹ–

- **`azure.ai.skills`** вҖ” бһҳбһҪбһҷ **skill** бһӮбһәбһҮбһ¶бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһЎбһҫбһ„бһңбһ·бһү (бһ§бһ”бһҖбһҡбһҺбҹҚбһ¬бһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбҹӮбһӣбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ) бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ…бһјбһӣбһ‘бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһҪбһҷбһ¬бһ…бҹ’бһҡбһҫбһ“ бһҮбһ¶бһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһ“бҹғбһ”бҹ’бһӣбһ»бһҖбһҡбһ¶бһ„бһҖбҹ’бһҡбһ»бһҳбһҠбҹӮбһӣбһўбһ¶бһ…бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбҹ– бһ“бһ·бһҷбһ¶бһҷбһҗбһ¶ бһҖбҹҶбһҺбһҸбҹӢ skill "бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҜбһҖбһҹбһ¶бһҡ" бһ¬ "бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ”бһүбҹ’бһҮбһ¶бһ‘бһ·бһү" бһҳбһҪбһҷбһҠбһ„ бһ бһҫбһҷбһ”бҹ’бһҡбһҫбһЎбһҫбһ„бһңбһ·бһүбһҖбҹ’бһ“бһ»бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бҹ” бһ“бҹҒбҹҮбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“ (бһҮбҹҶбһ–бһјбһҖ 5) бһҳбһ¶бһ“бһҹбһҳбһҡбһҳбҹ’бһҷ бһ“бһ·бһ„бһҮбҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһ…бһҳбҹ’бһӣбһ„бһҖбһјбһҠбҹ”
- **`azure.ai.connections`** вҖ” бһҳбһҪбһҷ **connection** бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ–бһёбһӮбһҳбҹ’бһҡбҹ„бһ„ Foundry бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһ”бҹ’бһҡбһ—бһ–бһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢбһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҮбһ¶бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢвҖ”бһ”бҹ’бһҡбһ—бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (бһҠбһјбһ…бһҮбһ¶ Azure AI Search), бһ…бҹҶбһҺбһ»бһ…бһ§бһ”бһҖбһҡбһҺбҹҚ бһ¬бһҹбҹҒбһңбһ¶бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ” Connections бһ”бһҳбҹ’бһҡбһҫбһҮбһ¶бһҖбһ“бҹ’бһӣбҹӮбһ„бһҖбһҺбҹ’бһҸбһ¶бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ *бһҖбһ“бҹ’бһӣбҹӮбһ„* бһ“бһ·бһ„ *бһҡбһ”бҹҖбһ”* бһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡACCESSбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбҹҶбһ„бһ¶бһҸбҹӢ бһ“бһ·бһ„бһ…бҹҶбһҺбһ»бһ…бһ…бһјбһӣбһҡбһҹбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ¶бһ“ бһҳбһ·бһ“бһҡбһ¶бһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһҡбһёбһҖбһҡбһ¶бһҷбһЎбһҫбһҷбҹ”

бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ”бҹ’бһҡбһҫбһ–бһҪбһҖбһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһҠбһ¶бһҖбҹӢбһ…бҹҒбһүбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹҶбһ”бһјбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖвҖ”бһҹбһјбһҳбһ“бҹ…бһҮбһ¶бһҳбһҪбһҷ `azure.ai.agents` бһҒбһҺбҹҲбһўбҹ’бһ“бһҖбһҡбҹҖбһ“бҹ” бһ”бҹ’бһҡбһҫ `skills` бһ“бҹ…бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһғбһҫбһүбһҒбҹ’бһӣбһҪбһ“бһҜбһ„бһҖбҹҶбһ–бһ»бһ„бһ…бһҳбҹ’бһӣбһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ…бҹ’бһҡбһҫбһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһ“бһ·бһ„ `connections` бһ“бҹ…бһ–бҹҒбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ”бҹ’бһҡбһ—бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ”

### бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһӮбһҳбҹ’бһҡбҹ„бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ `azd ai agent init`

бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ `azd ai agent init` бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһ»бһ‘бһ·бһҠбҹ’бһӢбһ·бһ“бһ·бһҷбһҳбһӮбһҳбҹ’бһҡбҹ„бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһҠбҹӮбһӣбһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷ Microsoft Foundry Agent Service:

```bash
# бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһӮбһҳбҹ’бһҡбҹ„бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҗбҹ’бһҳбһёбһ–бһёбһҜбһҖбһҹбһ¶бһҡ manifest бһ“бҹғбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ
azd ai agent init -m <manifest-path-or-uri>

# бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһӮбҹ„бһӣбһҠбҹ…бһ‘бҹ…бһӮбһҳбҹ’бһҡбҹ„бһ„ Foundry бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ
azd ai agent init -m agent-manifest.yaml --project-id <foundry-project-id>

# бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҗбһҸбһ”бҹ’бһҡбһ—бһ–бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“
azd ai agent init -m agent-manifest.yaml --src ./agents/my-agent

# бһҖбҹҶбһҺбһҸбҹӢ Container Apps бһҮбһ¶ бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“ бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ (host)
azd ai agent init -m agent-manifest.yaml --host containerapp
```

**бһ‘бһ„бҹӢбһҹбҹҶбһҒбһ¶бһ“бҹӢ:**

| `-p, --project-id` | Microsoft Foundry Project ID бһҗбҹ’бһҳбһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ azd бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ |

- **Resource discovery**: бһ§бһ”бһҖбһҡбһҺбҹҚ AI бһўбһ¶бһ…бһҡбһҖбһғбһҫбһүбһҗбһ¶бһ’бһ“бһ’бһ¶бһ“ Azure бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҠбҹӮбһӣбһӮбһҳбҹ’бһҡбҹ„бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ

бһ“бҹҒбҹҮбһ“бһ№бһ„бһҹбһҡбһҹбҹҒбһҡ IaC бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҢбһёбһҹбһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ¶бһ…:
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһ’бҹ’бһңбһҫ audit бһ‘бҹ…бһӣбһҫбһўбһ¶бһ бһ¶бһҡбһҠбҹ’бһӢбһ¶бһ“бһҳбһ»бһ“бһ–бҹҒбһӣбһ…бҹӮбһҖбһ…бһ¶бһҷ
- бһ”бһ“бҹ’бһҗбҹӮбһҳбһӮбҹ„бһӣбһ“бһҷбҹ„бһ”бһ¶бһҷбһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖ (бһ…бҹ’бһ”бһ¶бһ”бҹӢбһ”бһҺбҹ’бһҠбһ¶бһү, private endpoints)
- бһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ IaC бһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ
- бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһҺбҹӮбһ”бһҳбҹ’бһӣбҹӮбһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбҹ„бһҷбһҠбһ¶бһ…бҹӢбһ–бһёбһҖбһјбһҠбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё

### Production Lifecycle Hooks

AZD hooks бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бһүбҹ’бһ…бһјбһӣбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ”бһ»бһӮбҹ’бһӮбһӣбһ“бҹ…бһӮбҹ’бһҡбһ”бҹӢбһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһ“бҹғбһҮбһёбһңбһ…бһҖбҹ’бһҡбһ¶бһҡбһ…бһ“бһ¶бһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷвҖ”бһҠбҹӮбһӣбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһ“бҹ…бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбҹ–

```yaml
# azure.yaml - Production hooks example
name: ai-production-app
hooks:
  preprovision:
    shell: sh
    run: scripts/validate-quotas.sh    # Check AI model quota before provisioning
  postprovision:
    shell: sh
    run: scripts/configure-networking.sh  # Set up private endpoints
  predeploy:
    shell: sh
    run: scripts/run-ai-safety-tests.sh  # Run prompt safety checks
  postdeploy:
    shell: sh
    run: scripts/smoke-test.sh           # Verify agent responses post-deploy
services:
  agent-api:
    project: ./src/agent
    host: containerapp
    hooks:
      predeploy:
        shell: sh
        run: scripts/validate-model-access.sh  # Per-service hook
```

```bash
# бһҡбһҸбҹӢбһ бҹҠбһ»бһ…бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҠбҹ„бһҷбһҠбҹғбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһЎбһ»бһ„бһ–бҹҒбһӣбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ
azd hooks run predeploy
```

**бһҷбҹ„бһ„ hook бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһ”бһ¶бһ“бһҺбҹӮбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ„бһ¶бһҡ AIбҹ–**

| Hook | бһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫ |
|------|----------|
| `preprovision` | бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһӮбһҺбҹҲбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ subscription бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҳбҹүбһјбһҢбҹӮбһӣ AI |
| `postprovision` | бһҖбҹҶбһҺбһҸбҹӢ private endpoints, бһ…бҹӮбһҖбһ…бһ¶бһҷбһ‘бҹҶбһ„бһ“бҹӢбһҳбҹүбһјбһҢбҹӮбһӣ |
| `predeploy` | бһҡбһҸбҹӢбһҸбҹҒбһҹбҹ’бһҸбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– AI, бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ‘бҹҶбһҡбһ„Prompt |
| `postdeploy` | бһҸбҹҒбһҹбҹ’бһҸбһ”бһ„бҹ’бһ бһ¶бһүбһ…бһҳбҹ’бһӣбһҫбһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ, бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ”бҹ’бһҡбһ¶бһҖбһҠбһ–бһёбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбҹүбһјбһҢбҹӮбһӣ |

### CI/CD Pipeline Configuration

бһ”бҹ’бһҡбһҫ `azd pipeline config` бһҠбһҫбһҳбҹ’бһ”бһёбһ—бҹ’бһҮбһ¶бһ”бҹӢбһӮбһҳбҹ’бһҡбҹ„бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ… GitHub Actions бһ¬ Azure Pipelines бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺ Azure бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбҹ–

```bash
# бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ CI/CD (бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳ)
azd pipeline config

# бһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһ“бһҖбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ
azd pipeline config --provider github
```

бһҖбҹҶбһҺбһҸбҹӢбһ”бһүбҹ’бһҮбһ¶бһ“бҹҒбҹҮ:
- бһ”бһ„бҹ’бһҖбһҫбһҸ service principal бһҮбһ¶бһҳбһҪбһҷбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбҹ’бһ”бһҪбһ“бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ (least-privilege)
- бһҖбҹҶбһҺбһҸбҹӢ federated credentials (бһӮбҹ’бһҳбһ¶бһ“бһўбһ¶бһҗбҹҢбһҖбҹҶбһ”бһ¶бҹҶбһ„бһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ)
- бһ”бһ„бҹ’бһҖбһҫбһҸбһ¬бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһҜбһҖбһҹбһ¶бһҡбһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷ pipeline бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
- бһҖбҹҶбһҺбһҸбҹӢбһўбһҗбҹҒбһҡбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ CI/CD бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

#### Step-by-step: your first GitHub Actions pipeline

бһ“бҹҒбҹҮбһҮбһ¶бһҳбһ¶бһҸбһ·бһҖбһ¶бһӮбҹ’бһҡбһ”бҹӢбһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһ…бһ¶бһ”бҹӢбһ–бһёбһӮбһҳбҹ’бһҡбҹ„бһ„ azd бһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“ бһҠбһӣбҹӢбһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҠбҹ„бһҷбһӣбһҫбһҖбһҠбһ¶бһҖбҹӢбһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһ•бһ»бһҹбҹ”

**1. Make sure your project is on GitHub**

```bash
git init
git add .
git commit -m "Initial azd project"
gh repo create my-ai-app --private --source=. --push
```

**2. Run pipeline config**

```bash
azd pipeline config --provider github
```

azd бһ“бһ№бһ„, бһҖбҹ’бһ“бһ»бһ„бһҡбһ”бҹҖбһ”бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳ:
- бһҹбһҪбһҡбһҗбһ¶бһҸбһҫ subscription бһ“бһ·бһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ Azure бһҺбһ¶бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҠбһ¶бһҖбҹӢбһҖбҹҶбһҺбһҸбҹӢ
- бһ”бһ„бҹ’бһҖбһҫбһҸ Entra **app registration + service principal** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ pipeline
- бһҖбҹҶбһҺбһҸбҹӢ **federated credentials (OIDC)**вҖ”бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ GitHub бһўбҹҮбһўбһ¶бһ„бһ‘бҹ… Azure бһҠбҹ„бһҷбһҳбһ¶бһ“ token бһҡбһҷбҹҲбһ–бҹҒбһӣбһҒбҹ’бһӣбһё бһ бһҫбһҷ **бһӮбҹ’бһҳбһ¶бһ“бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҺбһ¶бһҳбһҪбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ**
- бһ”бһүбҹ’бһ…бһјбһ“ **бһўбһҗбҹҒбһҡ** бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ‘бҹ…бһҖбһ¶бһ“бҹӢ repo GitHub бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ (`AZURE_CLIENT_ID`, `AZURE_TENANT_ID`, `AZURE_SUBSCRIPTION_ID`, `AZURE_ENV_NAME`, `AZURE_LOCATION`)

**3. Understand the generated workflow**

azd бһ”бһ“бҹ’бһҗбҹӮбһҳ `.github/workflows/azure-dev.yml`бҹ” бһ•бҹ’бһ“бҹӮбһҖбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҠбһјбһ…бһҮбһ¶бҹ–

```yaml
# .github/workflows/azure-dev.yml
on:
  push:
    branches: [ main ]
  workflow_dispatch:        # lets you run it manually too

permissions:
  id-token: write           # required for OIDC federated login
  contents: read

jobs:
  build:
    runs-on: ubuntu-latest
    env:
      AZURE_CLIENT_ID: ${{ vars.AZURE_CLIENT_ID }}
      AZURE_TENANT_ID: ${{ vars.AZURE_TENANT_ID }}
      AZURE_SUBSCRIPTION_ID: ${{ vars.AZURE_SUBSCRIPTION_ID }}
      AZURE_ENV_NAME: ${{ vars.AZURE_ENV_NAME }}
      AZURE_LOCATION: ${{ vars.AZURE_LOCATION }}
    steps:
      - uses: actions/checkout@v4
      - name: Install azd
        uses: Azure/setup-azd@v2
      - name: Log in with OIDC
        run: azd auth login --client-id "$AZURE_CLIENT_ID" --federated-credential-provider "github" --tenant-id "$AZURE_TENANT_ID"
      - name: Provision infrastructure
        run: azd provision --no-prompt
      - name: Deploy application
        run: azd deploy --no-prompt
```

**4. Verify it works**

```bash
# бһ”бһүбҹ’бһ…бһјбһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ pipeline
git commit -am "Trigger pipeline" --allow-empty
git push
```

бһ”бһҫбһҖбһ•бҹ’бһ‘бһ¶бҹҶбһ„ **Actions** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ repo GitHub бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһҳбһҫбһӣбһӣбҹҶбһ бһ¶бһҸбҹӢбһҖбҹҶбһҺбһҸбҹӢбһҠбҹӮбһӣбһҡбһҸбҹӢ `azd provision` бһ“бһ·бһ„ `azd deploy` бһ”бһ¶бһ“бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ”

> **Why federated credentials matter:** бһҹбһ“бҹ’бһӣбһ№бһҖ pipeline бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһ…бһ¶бһҹбҹӢбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶ client secret бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ GitHubбҹ” federated credentials OIDC бһҷбһҖбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһ“бҹ„бҹҮбһІбҹ’бһҷбһўбһҹбҹӢбһҹбһјбһ“бҹ’бһҷвҖ”GitHub бһ“бһ№бһ„бһҹбҹҶбһҺбһҫбһҹбһ»бҹҶ token бһҡбһҷбҹҲбһ–бҹҒбһӣбһҒбҹ’бһӣбһёбһ“бҹ…бһ–бҹҒбһӣбһҡбһҸбҹӢ бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҮбһ¶бһ„ бһ“бһ·бһ„бһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ”бһҳбҹ’бһӣбҹӮбһ„ бһ¬бһҡбһӣбһҖбһ…бҹҒбһүбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҠбһҫбһҳбһҠбҹӮбһӣ `azd pipeline config` бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бҹ”

> **Secrets vs. variables:** бһҲбҹ’бһҳбҹ„бҹҮбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҠбҹӮбһӣбһҳбһ·бһ“бһҳбһ¶бһ“бһҹбһ»бһ–бһӣбһ—бһ¶бһ–бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ (`AZURE_CLIENT_ID`, бһӣбҹ”) бһӮбһҪбһҡбһҸбҹ’бһҡбһјбһңбһҠбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„ **variables** бһҡбһ”бһҹбҹӢ repoбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ–бһ·бһҸбһҮбһ¶бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһ“бҹ…бһ–бҹҒбһӣбһҖбһҹбһ¶бһ„ бһҹбһјбһҳбһ”бһ“бҹ’бһҗбҹӮбһҳбһңбһ¶бһҮбһ¶ GitHub **secret** бһ бһҫбһҷбһҷбҹ„бһ„бһңбһ¶бһҮбһ¶бһҳбһҪбһҷ `${{ secrets.NAME }}`вҖ”бһ”бҹүбһ»бһ“бҹ’бһҠбҹӮбһўбһ’бһ·бһ”бҹ’бһ”бһ¶бһҷбһ”бҹ’бһҡбһҫ Key Vault + managed identity бһ“бҹ… runtime бһӮбһәбһӣбҹ’бһўбһҮбһ¶бһ„ (бһҳбһҫбһӣ [бһҮбҹҶбһ–бһјбһҖ 3](../chapter-03-configuration/authsecurity.md))бҹ”

**Production workflow with pipeline config:**

```bash
# 1. бһҡбҹҖбһ”бһ…бҹҶбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ•бһӣбһ·бһҸ
azd env new production
azd env set AZURE_OPENAI_CAPACITY 100

# 2. бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ”бһҺбҹ’бһҸбһ»бҹҶбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ
azd pipeline config --provider github

# 3. бһ”бһҺбҹ’бһҠбһ»бҹҶбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһҡбһҸбҹӢ azd deploy бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһ“ (push) бһ‘бҹ…бһҹбһ¶бһҒбһ¶ main
```

#### Step-by-step: Azure DevOps Pipelines

бһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸ Azure DevOps бһҳбһ»бһ“ GitHub Actions? azd бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһңбһ¶вҖӢбһҠбҹ„бһҷбһҹбҹ’бһҡбҹ’бһҸбһёбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҮбһ¶бһҳбһҪбһҷ provider `azdo`бҹ” бһҲбһ»бһҸбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһӮбһәбһҹбҹ’бһ‘бһҫбһҡбһҸбҹӮбһҠбһјбһ…бһӮбҹ’бһ“бһ¶вҖ”azd бһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡ pipeline, бһ”бһ„бҹ’бһҖбһҫбһҸ service connection, бһ“бһ·бһ„бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбҹ”

**1. Make sure you have an Azure DevOps project**

бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһўбһ„бҹ’бһӮбһ—бһ¶бһ– бһ“бһ·бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹ… `https://dev.azure.com/<your-org>`бҹ” бһ”бһ„бҹ’бһҖбһҫбһҸ Personal Access Token (PAT) бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ·бһ‘бҹ’бһ’бһ· **Build (Read & execute)**, **Code (Read & write)**, бһ“бһ·бһ„ **Service Connections (Read, query & manage)**вҖ”azd бһ“бһ№бһ„бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһ¶бҹ”

**2. Configure the pipeline**

```bash
azd pipeline config --provider azdo
```

azd бһ“бһ№бһ„:
- бһҹбһҪбһҡбһўбҹҶбһ–бһёбһўбһ„бҹ’бһӮбһ—бһ¶бһ– Azure DevOps бһ“бһ·бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
- бһ”бһ„бҹ’бһҖбһҫбһҸ (бһ¬ reuse) **service connection** бһ‘бҹ… Azure бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ service principal
- бһҖбҹҶбһҺбһҸбҹӢ **workload identity federation (OIDC)** бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһӮбҹ’бһҳбһ¶бһ“ client secret бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ
- commit бһҜбһҖбһҹбһ¶бһҡ `azure-dev.yml` pipeline definition бһ‘бҹ… repo бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

**3. Review the generated `azure-dev.yml`**

azd бһҹбһҡбһҹбҹҒбһҡ pipeline бһҠбҹӮбһӣбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢ бһ“бһ·бһ„бһ…бҹӮбһҖбһ…бһ¶бһҷбһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһҳбһ¶бһ“ push бһ‘бҹ… `main`бҹ–

```yaml
# azure-dev.yml
trigger:
  - main

pool:
  vmImage: ubuntu-latest

steps:
  - task: setup-azd@1
    displayName: Install azd

  - script: azd provision --no-prompt
    displayName: Provision Infrastructure
    env:
      AZURE_SUBSCRIPTION_ID: $(AZURE_SUBSCRIPTION_ID)
      AZURE_ENV_NAME: $(AZURE_ENV_NAME)
      AZURE_LOCATION: $(AZURE_LOCATION)

  - script: azd deploy --no-prompt
    displayName: Deploy Application
    env:
      AZURE_SUBSCRIPTION_ID: $(AZURE_SUBSCRIPTION_ID)
      AZURE_ENV_NAME: $(AZURE_ENV_NAME)
      AZURE_LOCATION: $(AZURE_LOCATION)
```

**4. Where the variables come from**

azd бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҸбһҳбҹ’бһӣбҹғбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ (`AZURE_ENV_NAME`, `AZURE_LOCATION`, `AZURE_SUBSCRIPTION_ID`) бһҮбһ¶ **variable group** бһҖбҹ’бһ“бһ»бһ„ Azure DevOps бһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷ pipeline бһўбһ¶бһ…бһўбһ¶бһ“бһ”бһ¶бһ“бҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҫбһӣ бһ“бһ·бһ„бһҖбҹӮбһ”бҹ’бһҡбҹӮбһңбһ¶бһ‘бҹ…бһҖбҹ’бһҡбҹ„бһҳ **Pipelines вҶ’ Library**бҹ”

> **Same OIDC benefit as GitHub:** provider `azdo` бһҖбҹҸбһҖбҹҶбһҺбһҸбҹӢ workload identity federation бһҮбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҠбҹӮбһҡ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһӮбҹ’бһҳбһ¶бһ“ client secret бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ service connectionвҖ”Azure DevOps бһ“бһ№бһ„бһ”бҹ’бһҸбһјбһҡбһ‘бҹ… token бһҡбһҷбҹҲбһ–бҹҒбһӣбһҒбҹ’бһӣбһёбһ“бҹ…бһ–бҹҒбһӣ runtimeбҹ” бһ”бҹ’бһҡбһҫ `--auth-type client-credentials` бһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбһ„бҹ’бһӮбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһўбһ¶бһ…бһ”бҹ’бһҡбһҫ OIDC бһ‘бҹҒбҹ”

**5. Run it**

```bash
git commit -am "Add Azure DevOps pipeline" --allow-empty
git push
```

бһ”бһҫбһҖ **Pipelines** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Azure DevOps бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣ `azd provision` бһ“бһ·бһ„ `azd deploy` бһҡбһҸбҹӢбҹ”

### Adding Components with `azd add`

бһ”бһ“бҹ’бһҗбҹӮбһҳбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Azure бһҮбһ¶бһ…бҹҶбһҺбһ»бһ…бҹ— бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбҹ–

```bash
# бһ”бһ“бҹ’бһҗбҹӮбһҳбһ•бҹ’бһ“бҹӮбһҖбһҹбҹҒбһңбһ¶бһҗбҹ’бһҳбһёбһҠбҹ„бһҷбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳ
azd add
```

бһ“бҹҒбҹҮбһҳбһ¶бһ“бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҮбһ¶бһ–бһ·бһҹбҹҒбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ„бҹ’бһҡбһёбһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳвҖ”бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһ”бһ“бҹ’бһҗбҹӮбһҳбһҹбҹҒбһңбһ¶бһҖбҹ’бһҹбҹӮбһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһңбҹүбһ·бһ…бһ‘бҹҗбһҡ (vector search service), бһ…бҹҶбһҺбһ»бһ…бһ”бһүбҹ’бһ…бһ”бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҗбҹ’бһҳбһё, бһ¬бһ§бһ”бһҖбһҡбһҺбҹҚбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҪбһҷбһ…бҹҶбһҺбһ»бһ… бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбҹ”
