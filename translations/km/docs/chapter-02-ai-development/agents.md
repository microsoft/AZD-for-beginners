## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸбһҮбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҜбһҖбһҡбһ¶бһҮбҹ’бһҷбһҠбҹӮбһӣбһўбһ¶бһ…бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ–бһёбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒ, бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸ бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҳбҹ’бһҡбҹҒбһ…бһ”бһ¶бһ“бһӮбҹ„бһӣбһҠбҹ…бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбҹ” бһ•бҹ’бһ‘бһ»бһҷбһ–бһё chatbot бһ„бһ¶бһҷбҹ—бһҠбҹӮбһӣбһ…бһҳбҹ’бһӣбһҫбһҷбһҸбһ¶бһҳбһҹбҹҶбһҺбһҪбһҡ, бһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһўбһ¶бһ…бҹ–

- **бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ** - бһ бҹ… API, бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ, бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡвҖӢбһҖбһјбһҠ
- **бһҡбҹҖбһ”бһ…бҹҶ бһ“бһ·бһ„бһҷбһӣбҹӢбһҠбһ№бһ„** - бһ”бҹҶбһ”бҹӮбһҖбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҮбһ¶бһҖбҹҶбһ бһ»бһҹбҹ—
- **бһҡбҹҖбһ“бһ–бһёбһ”бһҡбһ·бһ”бһ‘** - бһҡбһҖбҹ’бһҹбһ¶бһҡбһҳбҹўбҹ бһӣ бһ“бһ·бһ„бһ”бһҸбҹӢбһ”бҹӮбһ“бһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶
- **бһҹбһ бһҖбһ¶бһҡбһӮбҹ’бһ“бһ¶** - бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ (бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ–бһ бһ»бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ)

бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһ”бһүбҹ’бһ…бһјбһӣбһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһ‘бҹ… Azure бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Azure Developer CLI (azd)бҹ”

> **бһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ (2026-07-13):** бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ“бҹҒбҹҮбһ”бһ¶бһ“бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҮбһ¶бһҳбһҪбһҷ `azd` `1.27.1` бһ“бһ·бһ„ `azure.ai.agents` `1.0.0-beta.5`бҹ” бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚ `azd ai` бһ“бҹ…бһҸбҹӮбһ„бһҮбһ¶бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҳбһҫбһӣбһҳбһ»бһ“, бһҠбһјбһ…бҹ’бһ“бҹ„бҹҮбһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҮбҹҶбһ“бһҪбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһ…бһјбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбҹҖбһ“

бһҠбҹ„бһҷбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһ¶бһ“бҹ–
- бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһёбһўбҹ’бһңбһёбһ‘бҹ…бһҮбһ¶бһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһ“бһ·бһ„бһ…бҹҶбһҺбһ»бҹҮбһҡбһңбһ¶бһ„бһңбһ¶бһ“бһ·бһ„ chatbot
- бһ”бһүбҹ’бһ…бһјбһӣбһ‘бһҳбҹ’бһҡбһ„бҹӢбһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ AZD
- бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ Foundry Agents бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹҶбһ“бһ¶бҹҶбһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ (бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ, RAG, бһ–бһ бһ»бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ)
- бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ

## бһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбҹҖбһ“

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһ”бҹӢ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…бҹ–
- бһ”бһүбҹ’бһ…бһјбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһ‘бҹ… Azure бһҠбҹ„бһҷбһҳбһҪбһҷбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶
- бһҖбҹҶбһҺбһҸбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ“бҹ’бһҗбҹӮбһҳ (RAG) бһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ
- бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ–бһ бһ»бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бһҡбһ“бҹ’бһҸбһҖбһ¶бһҡбһ„бһ¶бһҡбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү
- бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ–бҹҒбһүбһ“бһ·бһҷбһҳбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһҮбһјбһ“бһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ

- **бһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI** = бһҮбҹҶбһ“бһҪбһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҠбҹӮбһӣбһўбһ¶бһ…бһ бҹ…бһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘, бһҖбһҖбҹӢбһ–бҹҒбһӣбһҮбһҪбһ”, бһ“бһ·бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһ¶бһҡбһ„бһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖ

### бһҮбһҳбҹ’бһҡбһҫбһҹбһ‘бһё 4: AZD AI Agent Init (бһҖбһ¶бһҡбһҳбһҫбһӣбһҳбһ»бһ“бһҠбҹ„бһҷбһўбҹҶбһ–бһё Manifest бһ¬ Template)

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡ manifest бһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ `azd ai` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһҳбҹ’бһҡбҹ„бһ„ Foundry Agent Service бһҸбһ¶бһҳбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ” бһҖбһ¶бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһҳбһҫбһӣбһҳбһ»бһ“бһҗбҹ’бһҳбһёбҹ—бһҖбҹҸбһ”бһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ•бһ„бһҠбҹӮбһҡ, бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһӣбҹҶбһҠбһ¶бһ”бҹӢ prompt бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһўбһ¶бһ…бһҳбһ¶бһ“бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ”бһ“бҹ’бһҸбһ·бһ…бһҸбһ¶бһҳбһ…бҹ’бһ”бһ¶бһ”бҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бҹ”

```bash
# бһҸбҹҶбһЎбһҫбһ„бһҹбһ¶бһҒбһ¶ AI agents

# бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбҹҲ бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһҖбҹҶбһҺбҹӮбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„

# бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ‘бҹ… Azure

```

**бһ–бҹҒбһӣбһҠбҹӮбһӣбһҗбҹ…бһҖбҹӮ `azd ai agent init` бһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ” `azd init --template`:**

| бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ | бһҹбһҳбҹ’бһҡбһҪбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ | бһңбһ·бһ’бһёбһҠбҹӮбһӣбһңбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ |
|----------|----------|------|
| `azd init --template` | бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбҹҶбһҡбһјбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ | бһ…бһҳбҹ’бһӣбһ„бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ–бҹҒбһүбһӣбҹҒбһүбһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһјбһҠ + бһӮбһҳбҹ’бһҡбһ” |
| `azd ai agent init -m` | бһҖбһ¶бһҡбһҹбһ„бҹӢбһ–бһёбһҜбһҖбһҹбһ¶бһҡ manifest бһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ | бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӮбһҳбҹ’бһҡбҹ„бһ„бһ–бһёбһҖбһ¶бһҡбһ”бһҖбһҹбҹ’бһҡбһ¶бһҷбһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ |

> **бһӮбһ“бҹ’бһӣбһ№бҹҮ:** бһ”бҹ’бһҡбһҫ `azd init --template` бһ–бҹҒбһӣбһҡбҹҖбһ“ (бһҮбһҳбҹ’бһҡбһҫбһҹ 1-3 бһҒбһ¶бһ„бһӣбһҫ)бҹ” бһ”бҹ’бһҡбһҫ `azd ai agent init` бһ–бҹҒбһӣбһҹбһ¶бһ„бһҹбһ„бҹӢбһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһҪбһҷ manifest бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһё `azd up`, бһ”бһ“бҹ’бһҠбһ”бһ“бҹ’бһҠ бһҒбҹ’бһҹбҹӮбһҖбһ¶бһҡбһўбһәбһҖбһҹбһ„бҹӢбһ“бҹҒбҹҮбһ“бһ¶бҹҶбһўбҹ’бһ“бһҖбһҶбҹ’бһ–бҹ„бҹҮбһ‘бҹ…бһҡбһҖбһҮбһёбһңбһ…бһӣбһ“бҹҚбһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбҹ– `azd ai agent invoke` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹҒбһҹбҹ’бһҸ, `azd ai agent eval generate` бһ“бһ·бһ„ `azd ai agent optimize` бһҠбһҫбһҳбҹ’бһ”бһёбһңбһ¶бһҹбҹӢбһңбҹӮбһ„ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһ»бһҺбһ—бһ¶бһ–, бһ“бһ·бһ„ `azd ai agent delete` бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҳбҹ’бһўбһ¶бһҸбҹ” бһҳбһҫбһӣ [AZD AI CLI Commands](../chapter-08-production/production-ai-practices.md#azd-ai-cli-commands-and-extensions) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹҶбһҺбһҸбҹӢбһҷбҹ„бһ„бһ–бҹҒбһүбһӣбҹҒбһүбҹ”

## вҡҷпёҸ бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ

бһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһҳбҹ’бһӣбһ·бҹҮбһ–бһ–бһҪбһҖбһ–бҹҒбһүбһ…бһ·бһҸбҹ’бһҸбһ“бҹ…бһ–бҹҒбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһ¶бһ“бҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҡбһ”бҹҖбһ”бһҖбҹҶбһҺбһҸбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһјбһ‘бҹ…бҹ–

### бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹ… Foundry Agents

```python
# agent_config.py
from azure.ai.projects import AIProjectClient
from azure.ai.projects.models import FunctionTool, CodeInterpreterTool

# бһҖбҹҶбһҺбһҸбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“
search_tool = FunctionTool(
    name="search_knowledge_base",
    description="Search the company knowledge base for relevant documents",
    parameters={
        "type": "object",
        "properties": {
            "query": {
                "type": "string",
                "description": "The search query"
            }
        },
        "required": ["query"]
    }
)

# бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚ
agent = project_client.agents.create_agent(
    model="gpt-4.1",
    name="Support Agent",
    instructions="You are a helpful support agent. Use the search tool to find relevant information.",
    tools=[search_tool, CodeInterpreterTool()]
)
```

### бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ”бһҡбһ·бһ”бһјбһҺбҹҢ

```bash

from azure.monitor.opentelemetry import configure_azure_monitor
from opentelemetry import trace

# бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ Azure Monitor бһҮбһ¶бһҳбһҪбһҷ OpenTelemetry

    connection_string=os.environ["APPLICATIONINSIGHTS_CONNECTION_STRING"]
)

tracer = trace.get_tracer(__name__)

def log_agent_interaction(user_query, agent_response, tools_used, latency_ms):
    with tracer.start_as_current_span("agent_interaction") as span:
        span.set_attributes({
            "user_query": user_query,
            "response_length": len(agent_response),
            "tools_used": tools_used,
            "latency_ms": latency_ms
        })
```

```python
# бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ„бҹ’бһўбһҪбһ…бһ”бһҡбһ·бһ”бһ‘
def truncate_context(messages, max_tokens=8000, model="gpt-4.1"):
    """Keep only recent messages within token limit."""
    import tiktoken
    encoding = tiktoken.encoding_for_model(model)
    total_tokens = 0
    truncated = []
    
    for msg in reversed(messages):
        msg_tokens = len(encoding.encode(msg.content))
        if total_tokens + msg_tokens > max_tokens:
            break
        truncated.insert(0, msg)
        total_tokens += msg_tokens
    
    return truncated
```

### бһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҺбһ‘бһё 2: бһ”бһ“бҹ’бһҗбҹӮбһҳбһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ (бҹЈбҹ  бһ“бһ¶бһ‘бһё)

**бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бҹ–** бһ–бһ„бҹ’бһҡбһ№бһ„бһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“

1. бһ”бһүбҹ’бһ…бһјбһӣбһ‘бһҳбҹ’бһҡбһ„бҹӢбһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбҹ–
   ```bash
   azd init --template get-started-with-ai-agents
   azd up
   ```
2. бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡбһ§бһ”бһҖбһҡбһҺбҹҚбһҗбҹ’бһҳбһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбҹ–
   ```python
   def get_weather(location: str) -> str:
       """Get current weather for a location."""
       # бһҖбһ¶бһҡбһ бҹ… API бһ‘бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһўбһ¶бһҖбһ¶бһҹбһ’бһ¶бһҸбһ»
       return f"Weather in {location}: Sunny, 72В°F"
   ```
3. бһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһ§бһ”бһҖбһҡбһҺбҹҚбһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбҹ–
   ```python
   from azure.ai.projects.models import FunctionTool

   weather_tool = FunctionTool(
       name="get_weather",
       description="Get current weather for a location",
       parameters={
           "type": "object",
           "properties": {
               "location": {"type": "string", "description": "City name"}
           },
           "required": ["location"]
       }
   )

   agent = project_client.agents.create_agent(
       model="gpt-4.1",
       name="Weather Agent",
       tools=[weather_tool]
   )
   ```

   # бһҷбҹ„бһ„бһҸбһ¶бһҳбһҖбһ¶бһҡбҹ– бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ бҹ… get_weather("Seattle") бһ бһҫбһҷбһҸбҹ’бһҡбһЎбһ”бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбһ¶бһҖбһ¶бһҹбһ’бһ¶бһҸбһ»бһҸбһ¶бһҳбһҳбһҖ
