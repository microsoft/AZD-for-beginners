# មុខទី 2៖ ការអភិវឌ្ឍ AI-ជាដើម

**📚 មុខវិជ្ជា**: [AZD សម្រាប់អ្នកចាប់ផ្តើម](../../README.md) | **⏱️ រយៈពេល**: 1-2 ម៉ោង | **⭐ កម្រិតចំណេះដឹង**: មធ្យម

---

## ទិដ្ឋភាពទូទៅ

មុខទីនេះផ្តោតលើការដាក់បញ្ចូលកម្មវិធីដែលបង្កើតដោយ AI ដោយប្រើ Azure Developer CLI និងសេវាកម្ម Microsoft Foundry។ ចាប់ពីកម្មវិធីជជែក AI សាមញ្ញទល់ទៅនឹងភ្នាក់ងារឆ្លាតវៃជាមួយឧបករណ៍។

> **កំណត់ចំណាំផ្ទៀងផ្ទាត់ (2026-03-25):** លំហូរការបញ្ជារនិងការណែនាំបន្ថែមក្នុងមុខនេះត្រូវបានពិនិត្យប្រឆាំងនឹង `azd` `1.23.12` និងការចេញផ្សាយបន្ថែមភ្នាក់ងារមុនពេលបញ្ចេញ `azure.ai.agents` `0.1.18-preview` ។ ប្រសិនបើលោកអ្នកកំពុងប្រើកំណែ AZD ចាស់ សូមធ្វើការអាប់ដេតជាមុនសិនហើយបន្តធ្វើយុទ្ធនាការ។

## គោលបំណងការសិក្សា

ដោយបញ្ចប់មុខនេះ អ្នកនឹងមានសមត្ថភាព៖
- ដាក់បញ្ចូលកម្មវិធី AI ដោយប្រើទំព័រគំរូ AZD ដែលបានកំណត់រួច
- យល់ដឹងអំពីការភ្ជាប់ Microsoft Foundry ជាមួយ AZD
- កំណត់រចនាសម្ព័ន្ធ និងប្តូរតាមខ្លួនភ្នាក់ងារជាមួយឧបករណ៍
- ដាក់បញ្ចូលកម្មវិធី RAG (Retrieval-Augmented Generation)

## 🚀 ចាប់ផ្តើមរហ័ស

```bash
# ជម្រើសទី 1: កម្មវិធីជជែក RAG
azd init --template azure-search-openai-demo
azd up

# ជម្រើសទី 2: អ្នកប្រើប្រាស់ AI
azd init --template get-started-with-ai-agents
azd up

# ជម្រើសទី 3: កម្មវិធីជជែករហ័ស
azd init --template openai-chat-app-quickstart
azd up
```
