## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһҗбһҸбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚ бһ“бһ·бһ„бһҷбҹ„бһ„бһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҡбҹҖбһ“ Azure Developer CLI бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹ„бһҷбһҠбҹғбҹ” бһҹбҹҒбһҺбһ¶бһҡбһёбһҷбҹүбһј Retail Multi-Agent бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ–бҹҒбһүбһӣбҹҒбһү бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ”бһ“бҹ’бһҗбҹӮбһҳбһҮбһ¶бһ’бһ¶бһҸбһ»бһҷбҹ„бһ„бһ‘бҹ…бһғбҹ’бһӣбһ¶бҹҶбһ„ Azure Samples бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһӮбҹҶбһҡбһј AZD бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

### бһҖбһ¶бһҡбһ”бһҡбһ·бһҷбһ¶бһҷбһ…бҹҶбһҺбһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү

- вӯҗ **бһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳ** - бһӮбҹҶбһ“бһ·бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“, бһҹбҹҒбһңбһ¶бһҜбһҖбһҸбһ¶, 15-30 бһ“бһ¶бһ‘бһё  
- вӯҗвӯҗ **бһҳбһ’бҹ’бһҷбһҳ** - бһҹбҹҒбһңбһ¶бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“, бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ, 30-60 бһ“бһ¶бһ‘бһё  
- вӯҗвӯҗвӯҗ **бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ** - бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү, бһ”бһүбҹ’бһ…бһјбһӣ AI, 1-2 бһҳбҹүбҹ„бһ„  
- вӯҗвӯҗвӯҗвӯҗ **бһўбҹ’бһ“бһҖбһҮбҹҶбһ“бһ¶бһү** - бһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ, бһӮбҹҶбһҡбһјбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ, 2+ бһҳбҹүбҹ„бһ„

- бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҠбҹӮбһӣбһҹбһҳбһҹбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҗбҹ’бһӣбҹғ

### бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһҮбҹҶбһ“бһҪбһҷ
- бһҹбҹ’бһңбҹҗбһҷбһҖбҹ’бһҡбһ·бһҸбһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“
- бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһҡбҹҶбһҖбһ·бһӣбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ
- бһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ

## рҹ”§ бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһҖбһ¶бһҡбһҖбҹӮбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡ

### бһҖбҹӮбһ…бҹ’бһ“бҹғбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

1. **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҳбһ»бһ“**
   - бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Azure бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ
   - бһ”бҹ’бһҡбһ¶бһҖбһҠбһ–бһёбһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ subscription
   - бһҷбһӣбҹӢбһ–бһёбһ•бһӣбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһ‘бҹ…бһӣбһҫбһҗбҹ’бһӣбҹғ

2. **бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ**
   - бһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢбһҹбһҳбҹҗбһҷ `azure.yaml` бһҖбҹҶбһҺбһҸбҹӢбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ
   - бһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһ‘бҹҶбһ–бҹҗбһҡ Bicep
   - бһҖбҹҶбһҺбһҸбҹӢбһўбһҗбҹҒбһҡбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“

3. **бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҷбҹүбһ¶бһ„бһҳбҹүбһ¶бһҸбҹӢ**
   - бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ‘бҹ…бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҮбһ¶бһҳбһҪбһҷбһҮбҹҶбһ бһ¶бһ“бһҠбҹҶбһ”бһјбһ„
   - бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡ
   - бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖ бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–

4. **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒ**
   - бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҹбһ·бһ‘бҹ’бһ’бһ· бһ“бһ·бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ…бһјбһӣ
   - бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ
   - бһ”бһҫбһҖбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„

## рҹҺ“ бһ•бҹ’бһӣбһјбһңбһҡбҹҖбһ“

   - бһҷбһӣбҹӢбһ–бһёбһ•бҹ’бһӣбһјбһңбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ
   - бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“

   - бһҡбҹҖбһ“бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ CDN

бһҸбһҫбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ…бһ„бҹӢбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ‘бҹҒ? бһҷбһҫбһ„бһҹбһјбһҳбһҹбҹ’бһңбһ¶бһӮбһҳбһ“бҹҚбһҖбһ¶бһҡбһҡбһҪбһҳбһ…бҹҶбһҺбҹӮбһҖ!

### бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһЎбһҫбһ„бһҹбҹ’бһ“бһҫ
