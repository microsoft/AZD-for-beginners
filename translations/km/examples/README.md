## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһҗбһҸбһҜбһҖбһҹбһ¶бһҡбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚ бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡбһҷбҹ„бһ„бһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҡбҹҖбһ“ Azure Developer CLI бһҸбһ¶бһҳбһҡбһ”бҹҖбһ”бһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ” бһҹбҹ’бһҖбҹҒбһ“бһ¶бһҡбһёбһҷбҹүбһј Retail Multi-Agent бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ–бҹҒбһүбһӣбҹҒбһүбһҠбҹӮбһӣбһҹбҹ’бһҡбҹҒбһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ AZD бһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲбһ–бһё Azure Samplesбҹ”

### бһ‘бҹҶбһ“бҹ„бһӣбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҖбһҳбҹ’бһҡбһ·бһҸвҖӢбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү

- вӯҗ **бһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳ** - бһӮбҹҶбһ“бһ·бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“, бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҸбҹӮбһҳбһҪбһҷ, 15-30 бһ“бһ¶бһ‘бһё
- вӯҗвӯҗ **бһҳбһ’бҹ’бһҷбһҳ** - бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ…бҹ’бһҡбһҫбһ“, бһҖбһ¶бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ, 30-60 бһ“бһ¶бһ‘бһё
- вӯҗвӯҗвӯҗ **бһҒбҹ’бһ–бһҹбҹӢ** - бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү, бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ AI, 1-2 бһҳбҹүбҹ„бһ„
- вӯҗвӯҗвӯҗвӯҗ **бһҮбҹҶбһ“бһ¶бһүбһҒбҹ’бһ–бһҹбҹӢ** - бһҡбһҪбһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ, бһӮбҹҶбһҡбһјбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ, 2+ бһҳбҹүбҹ„бһ„

бһҠбҹ„бһҷбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһ¶бһ“бҹ–

- бһҷбһӣбҹӢбһ–бһёбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бҹ— бһ“бһ·бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ azd бһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһңбһ¶

- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ•бҹ’бһ“бҹӮбһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбһ¶бһҖбҹӢбһ•бҹ’бһҹбҹҒбһ„вҖӢбһ‘бһёбһҸбһ¶бҹҶбһ„вҖӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ

## рҹҡҖ бһ‘бһҫбһ”бһҸбҹ’бһҡбҹҖбһҳбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ?

### бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

1. **бһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„?** вҶ’ бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷ [Flask API](../../../examples/container-app/simple-flask-api) (вӯҗ, 20 бһ“бһ¶бһ‘бһё)
2. **бһҳбһ¶бһ“бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„ AZD бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бҹҒ?** вҶ’ бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ [Microservices](../../../examples/container-app/microservices) (вӯҗвӯҗвӯҗвӯҗ, 60 бһ“бһ¶бһ‘бһё)
3. **бһҖбҹҶбһ–бһ»бһ„бһҹбһ¶бһ„бһҹбһ„бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI?** вҶ’ бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷ [Microsoft Foundry Models Chat](../../../examples/azure-openai-chat) (вӯҗвӯҗ, 35 бһ“бһ¶бһ‘бһё) бһ¬бһҹбҹ’бһңбҹӮбһ„бһҡбһҖ [Retail Multi-Agent](retail-scenario.md) (вӯҗвӯҗвӯҗвӯҗ, 2+ бһҳбҹүбҹ„бһ„)
4. **бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҸбһҳбҹ’бһҡбһјбһңбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһ‘бҹҒ?** вҶ’ бһ”бҹ’бһҡбһҫбһ•бҹ’бһ“бҹӮбһҖ [бһҡбһҖбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһң](#-finding-the-right-example) бһҒбһ¶бһ„бһӣбһҫ
