## 2. Klíčové koncepty, které jste zvládli

### Azure Developer CLI (azd)
- `azd init` - Inicializovat projekty ze šablon
- `azd up` - Zajistit prostředky a nasadit jedním příkazem
- `azd env` - Spravovat proměnné prostředí
- `azd down` - Bezpečně odstranit prostředky

### AZD Template Structure
- `azure.yaml` - Konfigurace a nasazovací hooky
- `infra/` - Bicep — infrastruktura jako kód
- `src/` - Zdrojový kód aplikace
- `scripts/` - Automatizační a instalační skripty

### Microsoft Foundry Capabilities
- Služba AI Agent pro inteligentní asistenty
- Vyhledávání souborů a Azure AI Search pro získávání znalostí
- Vestavěné trasování, monitorování a vyhodnocování
- Red teaming pro ověřování bezpečnosti AI
