# AZD for AI Developers Workshop

> Gevalideerd tegen `azd 1.23.12` in maart 2026.

> [!IMPORTANT]  
> **Deze workshop is voorzien van een workshopgids die je lokaal in je browser kunt bekijken. Om te beginnen, open de repo in GitHub Codespaces, een dev container, of een lokale kloon met MkDocs geïnstalleerd—wacht vervolgens totdat je een actief VS Code-terminal ziet en typ:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Je zou een pop-updialoog moeten zien om de previewpagina in een browser te openen.**

Welkom bij de hands-on workshop om Azure Developer CLI (AZD) te leren met focus op AI-applicatie-implementatie. Deze workshop helpt je een praktische begrip van AZD-sjablonen te krijgen in 3 stappen:

1. **Discovery** - vind het sjabloon dat bij jou past.
1. **Deployment** - implementeer en valideer dat het werkt
1. **Customization** - wijzig en iteratief maak het van jou!

Gedurende deze workshop maak je ook kennis met kernontwikkelaarstools en workflows, om je end-to-end ontwikkeltraject te stroomlijnen.

## Workshop Overview

**Duur:** 3-4 hours  
**Niveau:** Beginner tot Intermediate  
**Vereisten:** Bekendheid met Azure, AI-concepten, VS Code & command-line tools.

Dit is een hands-on workshop waarin je leert door te doen. Zodra je de oefeningen hebt voltooid, raden we aan de AZD For Beginners-curriculum te bekijken om je leertraject voort te zetten richting beveiligings- en productiviteitsbest practices.

| Time| Module  | Objective |
|:---|:---|:---|
| 15 mins | Introduction | Set the stage, understand the goals |
| 30 mins | Select AI Template | Explore options and pick starter | 
| 30 mins | Validate AI Template | Deploy default solution to Azure |
| 30 mins | Deconstruct AI Template | Explore structure and configuration |
| 30 mins | Configure AI Template | Activate and try available features |
| 30 mins | Customize AI Template | Adapt the template to your needs |
| 30 mins | Teardown Infrastructure | Cleanup and release resources |
| 15 mins | Wrap-Up & Next Steps | Learning resources, Workshop challenge |
| | |

## What You'll Learn

Beschouw het AZD-sjabloon als een leeromgeving om verschillende mogelijkheden en tools te verkennen voor end-to-end ontwikkeling op Microsoft Foundry. Aan het einde van deze workshop zou je een intuïtief gevoel moeten hebben voor verschillende tools en concepten in deze context.

| Concept  | Objective |
|:---|:---|
| **Azure Developer CLI** | Begrijp toolcommando's en workflows|
| **AZD Templates**| Begrijp projectstructuur en configuratie|
| **Azure AI Agent**| Provisioning & implementatie van een Microsoft Foundry-project  |
| **Azure AI Search**| Context-engineering mogelijk maken met agents |
| **Observability**| Onderzoek tracing, monitoring en evaluaties |
| **Red Teaming**| Onderzoek adversarial testen en mitigaties |
