# AZD Voor AI-ontwikkelaars

!!! info "DOELGROEP VOOR DEZE WORKSHOP"
   
    Deze workshop is voor iedereen die AI-agentachtige applicaties op Microsoft Foundry wil bouwen door gebruik te maken van toptools en workflows. De workshop is geschikt voor zowel beginners als ervaren ontwikkelaars die enige vertrouwdheid hebben met Azure, AI-concepten en Python-code. Controleer de vereisten voordat je begint.

    **Duration:** 3-4 hours  
    **Level:** Beginner to Intermediate  

---


## Vereisten

- Een persoonlijk GitHub-account → [Maak er gratis een aan](https://github.com/signup)
- Een actief Azure-account → [Maak er gratis een aan](https://aka.ms/free)
- Vertrouwdheid met AI-concepten, Python-ontwikkeling, commandoregelhulpmiddelen
- Ervaring met VS Code en GitHub-tools en workflows.

---

## Leerdoelen

Aan het einde van deze workshop weet je hoe je:

- [X] **Werken met Azure Developer CLI** → om je Azure-resources te provisioneren, implementeren en beheren.
- [X] **AI-appsjablonen gebruiken** → om AI-agentprojecten op Microsoft Foundry een vliegende start te geven.
- [X] **Sjablonen configureren** → voor functies zoals Red Teaming, evaluaties, tracing & monitoring
- [X] **Ontleden en verkennen** → AZD-sjablonen en workflows met GitHub Copilot
- [X] **Aanpassen en opnieuw implementeren** → het AI-agent-starter-sjabloon, met jouw aangepaste vereisten

---

## Workshopoverzicht

De workshop is gestructureerd als modules, elk gefundeerd op een hands-on laboefening:

1. **Introductie** - Krijg inzicht in wat AZD-sjablonen zijn en hoe ze werken
1. **Selectie** - Ontdek het juiste "starter" sjabloon voor je volgende AI-project
1. **Validatie** - Implementeer en verken het sjabloon om te verifiëren dat het werkt zoals aangegeven
1. **Deconstructie** - Haal het sjabloon uit elkaar om de componenten & configuratie te begrijpen
1. **Configuratie** - Leer verschillende Microsoft Foundry-functies te activeren en te gebruiken
1. **Aanpassing** - Herbouw de applicatie met jouw data en vereisten in gedachten
1. **Afbouw** - Leer hoe je infrastructuur deprovisioneert en soft-verwijderde resources herstelt
