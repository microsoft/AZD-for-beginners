# အခန်း 6: တပ်ဆင်မတိုင်မီ စီမံချက်နှင့် အတည်ပြုခြင်း

**📚 သင်တန်း**: [AZD For Beginners](../../README.md) | **⏱️ ကြာချိန်**: 1 နာရီ | **⭐ ရှုပ်ထွေးမှု**: အလယ်အလတ်

---

## အကျဉ်းချုပ်

ဤအခန်းတွင် သင့်အက်ပ်လိကေးရှင်းကို တပ်ဆင်မတိုင်မီ လိုအပ်သော စီမံကိန်းများနှင့် အတည်ပြုခြင်း အဆင့်များကို ဖော်ပြပါသည်။ သင့်စွမ်းရည်ကို မှန်ကန်စွာ စီမံခြင်း၊ SKU ရွေးချယ်ခြင်းနှင့် preflight စစ်ဆေးမှုများကို ကျင့်သုံးခြင်းဖြင့် ကုန်ကျစရိတ်များစွာ ဖြစ်ပေါ်စေရန် မဖြစ်စေရန် လေ့လာပါ။

> `azd 1.23.12` အား 2026 မတ်လတွင် အတည်ပြုထားပါသည်။

## သင်ယူရမည့် ရည်မှန်းချက်များ

ဤအခန်းကို ပြီးမြောက်ပါက သင်သည်-
- တပ်ဆင်မှုပြုလုပ်မှီ preflight စစ်ဆေးမှုများကို လည်ပတ်နိုင်မည်
- စွမ်းရည် စီမံချက်ပြုလုပ်ပြီး အရင်းအမြစ် လိုအပ်ချက်များ ခန့်မှန်းနိုင်မည်
- ကုန်ကျစရိတ် လျှော့ချရေးအတွက် သင့်တော်သော SKUs ကို ရွေးချယ်နိုင်မည်
- စောင့်ကြည့်မှုအတွက် Application Insights ကို ဖွဲ့စည်းနိုင်မည်
- အဖွဲ့ညှိနှိုင်းမှု မျိုးစုံများကို နားလည်နိုင်မည်

## 🚀 စတင်လျင်မြန်နည်း

```bash
# စာရင်းအကောင့်အတွက် ကန့်သတ်ချက်များကို စစ်ဆေးပါ
az vm list-usage --location eastus --output table

# တပ်ဆင်မှုကို ကြိုကြည့်ပါ (အရင်းအမြစ်များ မဖန်တီးပါ)
azd provision --preview

# Bicep သဒ္ဒါကို စစ်ဆေးပါ
az bicep build --file infra/main.bicep

# ပတ်ဝန်းကျင် ဖွဲ့စည်းပုံကို စစ်ဆေးပါ
azd env get-values
```

## ☑️ တပ်ဆင်မှီ စစ်ဆေးစာရင်း

### မတိုင်မီ `azd provision`

- [ ] ဒေသအလိုက် Quota ကို အတည်ပြုထားသည်
- [ ] SKUs များကို သင့်တော်စွာ ရွေးချယ်ထားသည်
- [ ] ကုန်ကျစရိတ် ခန့်မှန်းချက်ကို ပြန်လည်ဆန်းစစ်ထားသည်
- [ ] နာမည်ပေးစံသတ်မှတ်ချက် များ ထပ်တူညီညွတ်နေသည်
- [ ] အချက်အလက်လုံခြုံရေး/RBAC ကို ပြင်ဆင်ထားသည်

### မတိုင်မီ `azd deploy`

- [ ] ပတ်ဝန်းကျင် လွန်ကဲဖျက်များ (Environment variables) စီစစ်ပြီးသတ်မှတ်ထားသည်
- [ ] Key Vault တွင် လျှို့ဝှက်ချက်များ ထည့်သွင်းထားသည်
- [ ] ချိတ်ဆက်မှု စာကြောင်းများကို အတည်ပြုထားသည်
- [ ] ကျန်းမာရေး စစ်ဆေးမှုများ စီစဉ်ထားသည်
