## အကျဉ်းချုပ်

ဒီအခန်းမှာ Azure Developer CLI (azd) အခြေခံများကို မိတ်ဆက်ပေးမှာဖြစ်ပါတယ်။ သင်သည် အဓိကသဘောတရားများကို သင်ယူပြီး၊ ကိရိယာများကို 설치၍ သင်၏ ပထမဆုံး အက်ပလီကေးရှင်းကို Azure သို့ ထည့်သွင်း ရပါမည်။

> `azd 1.27.1` နှင့် ၂၀၂၆ ခုနှစ် ဇူလိုင်တွင် စစ်ဆေးပြီး။

## သင်ယူရမည့် ပစ္စည်းများ

ဒီအခန်းကို ပြီးစီးစဉ်မှာ သင်သည်-
- Azure Developer CLI ဆိုတာဘာလဲ၊ Azure CLI နှင့် ဘယ်လိုကွာခြားချက်ရှိသည်ကို နားလည်နိုင်မည်
- AZD ကို သင့်ပလက်ဖောင်းပေါ်တွင် 설치လုပ်၍ ပြင်ဆင်နိုင်မည်
- `azd up` ဖြင့် သင်၏ ပထမဆုံး အက်ပလီကေးရှင်းကို Azure တွင် ထည့်သွင်းနိုင်မည်
- `azd down` ဖြင့် အရင်းအမြစ်များကို သန့်ရှင်းနိုင်မည်

## ✅ ဒီနေရာကနေ စတင်ပါ: သင့် စနစ်ကို စစ်ဆေးပါ

စတင်ရန် မတိုင်မီ၊ သင့်ဒေသတွင် Chapter 1 template အတွက် စက်ကို ပြင်ဆင်ပြီးကြောင်း အတည်ပြုပါ။

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

စာသား script မှ ကိရိယာများ မရှိကြောင်း ပြောပါက ဒါတွေကို အရင်ပြင်ဆင်ပြီးမှ ဒီအခန်းကို ဆက်လက်လုပ်ဆောင်ပါ။

---

## 🚀 အမြန်စတင်

```bash
# 설치ကိုစစ်ဆေးပါ
azd version

# AZD အတည်ပြုပါ
# ရွေးချယ်စရာ: Azure CLI အမိန့်များကိုတိုက်ရိုက်ပြေးရန် az login လုပ်ပါက
azd auth login

# သင့်ပထမဆုံး app ကိုတပ်ဆင်ပါ
azd init --template todo-nodejs-mongo
azd up

# ပြီးဆုံးလျှင် သန့်ရှင်းပါ
azd down --force --purge
```

---

## ✅ အောင်မြင်မှု အချက်ပြများ

ဒီအခန်းကို ပြီးမြောက်ပြီးနောက် သင်သည်-

```bash
azd version              # တပ်ဆင်ပြီး ဗားရှင်းကို ပြတယ်
azd init --template todo-nodejs-mongo  # စီမံကိန်းကို စတင်ထားတယ်
azd up                   # Azure သို့ ဖြန့်ချိတယ်
azd show                 # လည်ပတ်နေသော အက်ပ် URL ကို ပြတယ်
azd down --force --purge # အရင်းအမြစ်များကို သန့်ရှင်းတက်ဦး
```
