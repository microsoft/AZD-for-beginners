# အခန်း ၁: အခြေခံနှင့် အမြန် စတင်ခြင်း

**📚 သင်တန်း**: [AZD အခြေခံ သင်တန်း](../../README.md) | **⏱️ ကြာချိန်**: 30-45 မိနစ် | **⭐ အဆင့်**: အခြေခံ

---

## အနှစ်ချုပ်

ဤအခန်းသည် Azure Developer CLI (azd) ၏ အခြေခံများကို မိတ်ဆက်ပေးသည်။ သင်သည် အဓိက အယူအဆများကို လေ့လာမည်၊ ကိရိယာများကို တပ်ဆင်မည်၊ နှင့် သင့်၏ ပထမဆုံး အက်ပလီကေးရှင်းကို Azure သို့ တင်ပို့မည်။

## သင်ယူရန် ရည်ရွယ်ချက်များ

By completing this chapter, you will:
- Azure Developer CLI ဆိုတာဘာလဲ၊ Azure CLI နှင့် မည်သို့ ကွဲပြားသည်ကို နားလည်ရန်
- သင့်ပလက်ဖောင်းတွင် AZD ကို တပ်ဆင်ပြီး အပြင်အဆင်များ ပြင်ဆင်ရန်
- `azd up` ကို အသုံးပြု၍ သင့် ပထမဆုံး အက်ပလီကေးရှင်းကို Azure သို့ တင်ပို့ရန်
- `azd down` ဖြင့် ရင်းမြစ်များကို သန့်ရှင်းရန်

## 🚀 အမြန် စတင်ခြင်း

```bash
# တပ်ဆင်မှုကို စစ်ဆေးပါ
azd version

# Azure သို့ အကောင့်ဝင်ပါ
azd auth login

# သင့်ပထမဆုံး အက်ပ်ကို တပ်ဆင်ပါ
azd init --template todo-nodejs-mongo
azd up

# ပြီးဆုံးလျှင် ရှင်းလင်းပါ
azd down --force --purge
```

## ✅ အောင်မြင်မှု သတ်မှတ်ချက်များ

ဤအခန်းကို ပြီးမြောက်လျှင် သင်သည် အောက်ပါအရာများကို လုပ်နိုင်သင့်သည်။

```bash
azd version              # တပ်ဆင်ထားသော ဗားရှင်းကို ပြသည်
azd init --template todo-nodejs-mongo  # ပရောဂျက်ကို စတင်ပြင်ဆင်သည်
azd up                   # Azure သို့ တင်သည်
azd show                 # လည်ပတ်နေသော အက်ပ်၏ URL ကို ပြသည်
azd down --force --purge # ရင်းမြစ်များကို ရှင်းလင်းသည်
```
