# အခန်း ၁: အခြေခံနှင့် အမြန်စတင်ခြင်း

**📚 သင်တန်း**: [AZD စတင်သူများအတွက်](../../README.md) | **⏱️ ကာလ**: 30-45 မိနစ် | **⭐ အခက်အခဲ**: အခြေခံ

---

## အကျဉ်းချုံး

ဤအခန်းသည် Azure Developer CLI (azd) ၏ အခြေခံများကို မိတ်ဆက်ပေးသည်။ သင်သည် အဓိက အယူအဆများကို လေ့လာ၍ ကိရိယာများကို တပ်ဆင်ပြီး သင့် ပထမဆုံး အပလီကေးရှင်းကို Azure သို့ တင်ပို့ပါလိမ့်မည်။

> အတည်ပြုချက်: `azd 1.23.12` ဖြင့် မတ် 2026 တွင် စစ်ဆေးပြီးဖြစ်သည်။

## သင်ယူရည်ရွယ်ချက်များ

By completing this chapter, you will:
- Azure Developer CLI သည် 무엇이며 Azure CLI နှင့် မည်သို့ ကွဲပြားသည်ကို နားလည်နိုင်မည်
- သင့် ပလက်ဖောင်းပေါ်တွင် AZD ကို ထည့်သွင်း၍ ဖော်ပြအဆင်ပြေအောင် ဆက်တင်ပြုလုပ်နိုင်မည်
- `azd up` ဖြင့် သင့် ပထမဆုံး အပလီကေးရှင်းကို Azure သို့ တင်ပို့နိုင်မည်
- `azd down` ဖြင့် အရင်းအမြစ်များကို သန့်ရှင်းခွင့်ပေးနိုင်မည်

## ✅ ဒီနေရာမှ စပါ: သင့် စက်ပစ္စည်းကို စစ်ဆေးပါ

စတင်မီ၊ သင့် ကွန်ပျူတာသည် အခန်း ၁ တမ်းပလိတ်အတွက် အသင့်ဖြစ်ကြောင်း အတည်ပြုပါ။

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

If the script reports missing tools, fix those first and then continue with the chapter.

---

## 🚀 အမြန်စတင်ခြင်း

```bash
# တပ်ဆင်မှုကို စစ်ဆေးပါ
azd version

# AZD အတွက် အတည်ပြုပါ
# ရွေးချယ်စရာ: Azure CLI အမိန့်များကို တိုက်ရိုက် လုပ်ဆောင်လိုပါက az login ကို ပြုလုပ်ပါ
azd auth login

# ပထမဆုံး အက်ပ်ကို ဖြန့်ချိပါ
azd init --template todo-nodejs-mongo
azd up

# ပြီးဆုံးသည့်အချိန်တွင် ရှင်းလင်းပါ
azd down --force --purge
```

## ✅ အောင်မြင်မှု သတ်မှတ်ချက်များ

After completing this chapter, you should be able to:

```bash
azd version              # တပ်ဆင်ထားသော ဗားရှင်းကို ပြသည်
azd init --template todo-nodejs-mongo  # ပရောဂျက်ကို စတင်စီစဉ်သည်
azd up                   # Azure သို့ တင်ပို့သည်
azd show                 # လည်ပတ်နေသော အက်ပ်၏ URL ကို ပြသည်
azd down --force --purge # အရင်းအမြစ်များကို ရှင်းလင်းသည်
```
