# အခန်း ၁: အခြေခံနှင့် အမြန်စတင်ခြင်း

**📚 သင်တန်း**: [AZD အစပြုသူများအတွက်](../../README.md) | **⏱️ ကြာချိန်**: 30-45 မိနစ် | **⭐ အဆင့်**: အခြေခံ

---

## အကျဉ်းချုပ်

ဤအခန်းသည် Azure Developer CLI (azd) ၏ အခြေခံအကြောင်းများကို မိတ်ဆက်ပေးသည်။ သင်သည် အခြေခံ သဘောတရားများကို သင်ယူမည်၊ ကိရိယာများကို ထည့်သွင်းတပ်ဆင်မည်နှင့် သင့် ပထမဆုံး အက်ပလီကေးရှင်းကို Azure သို့ တင်ပို့မည်။

> `azd 1.25.6` အတွက် ၂၀၂၆ ဇွန်လတွင် အတည်ပြုထားသည်။

## လေ့လာရန် ရည်မှန်းချက်များ

ဤအခန်းကို ပြီးမြောက်ပါက သင်သည်:
- Azure Developer CLI ဆိုတာ ဘာလဲ၊ Azure CLI နှင့် အကွာအဝေးများကို နားလည်နိုင်မည်
- သင့်ပလက်ဖောင်းပေါ်တွင် AZD ကို ထည့်သွင်း၍ ဖွဲ့စည်းတပ်ဆင်နိုင်မည်
- `azd up` ဖြင့် သင့် ပထမဆုံး အက်ပလီကေးရှင်းကို Azure သို့ တင်ပို့နိုင်မည်
- `azd down` ဖြင့် အရင်းအမြစ်များကို ရှင်းလင်းနိုင်မည်

## ✅ စတင်ရန်: သင်၏ ဆက်တင်များကို အတည်ပြုပါ

စတင်ရန်မမီ၊ သင့် မိမိကွန်ပျူတာသည် အခန်း ၁ အတွက် အသင့်ဖြစ်ကြောင်း အတည်ပြုပါ။

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

အကယ်၍ script သည် ကိရိယာများမရှိကြောင်း ဖော်ပြလျှင်၊ အရင်ဆုံး အဆိုပါ ကိရိယာများကို ပြုပြင် ထည့်သွင်းပြီးနောက် အခန်းကို ဆက်လက်ပါ။

---

## 🚀 အမြန်စတင်ခြင်း

```bash
# တပ်ဆင်မှုကို စစ်ဆေးပါ
azd version

# AZD အတွက် အတည်ပြုပါ
# ရွေးချယ်နိုင်သည်: Azure CLI အမိန့်များကို တိုက်ရိုက် အသုံးပြုရန် ရည်ရွယ်ပါက az login လုပ်ပါ
azd auth login

# သင့် ပထမဆုံး အက်ပ်ကို တင်ပို့ပါ
azd init --template todo-nodejs-mongo
azd up

# ပြီးဆုံးပါက သန့်ရှင်းပါ
azd down --force --purge
```

## ✅ အောင်မြင်မှုသတ်မှတ်ချက်များ

ဤအခန်းကို ပြီးမြောက်ပါက သင်သည် အောက်ပါ အရာများကို ပြုလုပ်နိုင်ရမည်-

```bash
azd version              # တပ်ဆင်ထားသော ဗားရှင်းကို ပြသည်
azd init --template todo-nodejs-mongo  # ပရောဂျက်ကို စတင်တည်ဆောက်သည်
azd up                   # Azure သို့ ဖြန့်ချိသည်
azd show                 # လည်ပတ်နေသော အက်ပ်၏ URL ကို ပြသည်
azd down --force --purge # အရင်းအမြစ်များကို ရှင်းလင်းသည်
```
