# အခန်း ၅: မလ်တီ-အေဂျင့် AI ဖြေရှင်းချက်များ

---

## အနှူးအကြောင်းအရာ

ဤအခန်းတွင် မလ်တီ-အေဂျင့် အင်အားဇာတ်လမ်းပုံစံများ၊ အေဂျင့်ညှိနှိုင်းမှုနှင့် ဖက်Ym production အသုံးပြုနိုင်သော AI ဖြန့်ချိမှုများကို ကိုးကားဖော်ပြပါမည်။

## သင်ယူရမည့် ရည်မှန်းချက်များ

ဤအခန်းကို ပြီးမြောက်သည်နောက် သင်သည် -
- မလ်တီ-အေဂျင့် အင်အားဇာတ်လမ်းပုံစံများ ကိုနားလည်နိုင်မည်
- အနှိပ်ညှိနှိုင်းထားသော AI အေဂျင့် စနစ်များကို ဖြန့်ချိနိုင်မည်
- အေဂျင့်ဝင်းအချင်းချင်း ဆက်သွယ်ရေး ကို ဆောင်ရွက်နိုင်မည်
- ထုတ်လုပ်ရေး အသင့် မလ်တီ-အေဂျင့် ဖြေရှင်းချက်များ တည်ဆောက်နိုင်မည်

## 🚀 Quick Start

```bash
# လက်လီလုပ်ငန်းအတွက် မျိုးစုံအေဂျင့် ဖြေရှင်းချက်ကို တပ်ဆင်ပါ
cd examples/retail-multiagent-arm-template
./deploy.sh

# သို့မဟုတ် ပုံစံကို တိုက်ရိုက် အသုံးပြုပါ
azd init --template agent-openai-python-prompty
azd up
```

## 🎯 အထူးဖျော်ဖြေမှု: လက်လီရောင်း မလ်တီ-အေဂျင့်

The [Retail Multi-Agent Solution](../../examples/retail-scenario.md) သည် အောက်ပါအရာများကို ပြသသည်။ 

- **ဖောက်သည် အေဂျင့်**: အသုံးပြုသူ နှင့်의 ဆက်ဆံရေးများနှင့် နှစ်လိုချက်များကို ကိုင်တွယ်သည်
- **ကုန်ပစ္စည်း စာရင်း အေဂျင့်**: သိုလှောင်မှုနှင့် မှာလာမှု လုပ်ငန်းစဉ်များကို စီမံခန့်ခွဲသည်
- **ညှိနှိုင်းသူ**: အေဂျင့်များအကြား ကိုယ့်ကိုယ်ကို ညှိနှိုင်းပေးသည်
- **မျှဝေသိမ်းဆည်းရာ မှတ်ဉာဏ်**: အေဂျင့်များအကြား ကွန်တက်စ် ကို စီမံတင်ပြသည်

### အသုံးပြုသည့် ဝန်ဆောင်မှုများ

| Service | ရည်ရွယ်ချက် |
|---------|---------|
| Azure OpenAI | ဘာသာစကား နားလည်မှု |
| Azure AI Search | ထုတ်ကုန်စာရင်း |
| Cosmos DB | အေဂျင့် အခြေအနေ နှင့် မှတ်ဉာဏ် |
| Container Apps | အေဂျင့် ဟိုစတင်း |
| Application Insights | မော်နီတာင်လုပ်ငန်း |
