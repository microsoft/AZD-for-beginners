# ခေါင်းစဥ် ၇: ပြဿနာဖြေရှင်းခြင်းနှင့် မှားယွင်းချက်ရှာဖွေခြင်း

## အကျဉ်းချုပ်

ဒီခေါင်းစဉ်မှာ Azure Developer CLI ကို အသုံးပြုရာတွင် ဖြစ်တတ်တဲ့ ပုံမှန်ပြဿနာများကို အကောက်ခတ်ပြီး ဖြေရှင်းနိုင်ရန် ကူညီပေးပါသည်။ ဖြန့်ချိမှုမအောင်မြင်မှုများမှ AI-ဆိုင်ရာ ပြဿနာများအထိ။

> ၂၀၂၆ ခုနှစ် ဇူလိုင်လတွင် `azd 1.27.1` ဖြင့် အတည်ပြုထားသည်။

## သင်ယူရမည့် ရည်မှန်းချက်များ

ဒီခေါင်းစဉ်ကို အပြီးသင့်ရောက်သည်နှင့် -
- AZD ဖြန့်ချိမှုပျက်ကွက်မှုများကို ဆန်းစစ်နိုင်မည်
- မှန်ကန်မှုရှာဖွေရေးနှင့် အခွင့်အရေး ပြဿနာများကို တွေ့ရှိနိုင်မည်
- AI ဝန်ဆောင်မှု ဆက်သွယ်မှု ပြဿနာများကို ဖြေရှင်းနိုင်မည်
- ပြဿနာရှာဖွေရေးအတွက် Azure Portal နှင့် CLI ကို အသုံးပြုနိုင်မည်

## 🚨 အလိုအလျောက်ဖြေရှင်းနည်းများ

### မှန်ကန်မှု ပြဿနာများ
```bash
# AZD လုပ်ငန်းစဉ်များအတွက် လိုအပ်သည်
azd auth login

# Azure CLI ကိုတိုက်ရိုက်အသုံးပြုလျှင် ရွေးချယ်စရာ ဖြစ်သည်
az login

azd auth status
```

### စီမံအပ်နှံခြင်း မအောင်မြင်မှုများ
```bash
azd show
azd monitor --logs
az deployment sub list --query "[?properties.provisioningState!='Succeeded']"
```

### အရင်းအမြစ် တုံ့ပြန်မှု မတူညီမှုများ
```bash
azd down --force --purge
azd env new different-name
azd up
```

### ကွုတာ ကျော်လွန်ခြင်း
```bash
az vm list-usage --location eastus --output table
azd env set AZURE_LOCATION westus2
azd up
```

## 📋 အမှားကုဒ် အညွှန်း

| အမှား | အကြောင်းရင်း | ဖြေရှင်းနည်း |
|-------|-------|----------|
| `AuthenticationError` | မှတ်ပုံတင်မလုပ်ထားခြင်း | `azd auth login` |
| `ResourceNotFound` | အရင်းအမြစ် မတွေ့ရှိနိုင်ခြင်း | အရင်းအမြစ် နာမည်များစစ်ဆေးပါ |
| `QuotaExceeded` | စာရင်းသွင်းမှု အကန့်အသတ်တားဆီးမှု | ကွိုတာ တိုးမြှင့်မှု တောင်းဆိုပါ |
| `InvalidTemplate` | Bicep စာရွက်ပုံစံ မှားယွင်းမှု | `az bicep build` |
| `Conflict` | အရင်းအမြစ် ရှိနေပြီး ဖြစ်ခြင်း | နာမည်အသစ် သို့မဟုတ် ဖျက်သိမ်းပါ |
| `Forbidden` | ခွင့်ပြုချက် မလုံလောက်ခြင်း | RBAC အခန်းကဏ္ဍများ စစ်ဆေးပါ |

## 🔄 ပြန်လည်သတ်မှတ်ခြင်းနှင့် ပြန်လည်ကောင်းမွန်ခြင်း

```bash
# နူးညံ့ပြန်စတင်ခြင်း (အရင်းအမြစ်များကို ထိန်းသိမ်းထားပြီး ကုဒ်ကို ပြန်လည်တပ်ဆင်ခြင်း)
azd deploy --force

# ထိုးထွင်းပြန်စတင်ခြင်း (အရာအားလုံးကို ဖျက်ပြီး အသစ်စတင်ခြင်း)
azd down --force --purge
azd up
```
