## အကျဉ်းချုပ်

ဤအခန်းသည် Azure Developer CLI ဖြင့် အလုပ်လုပ်စဉ်တွင် တွေ့ကြုံရသည့် အထွေထွေပြဿနာများကို အပိုင်းခွဲ၍ ရှာဖွေနည်းနှင့် ဖြေရှင်းနည်းများကို အကူအညီပေးပါသည်။ ဖြန့်ချိမှု မအောင်မြင်ခြင်းမှစ၍ AI-ဆိုင်ရာ ပြဿနာများအထိ ပါဝင်သည်။

> 2026 မတ်လတွင် `azd 1.23.12` နှင့် အတည်ပြုထားပါသည်။

## သင်ယူရမည့် ရည်မှန်းချက်များ

ဤအခန်းကို ပြီးစီးမှုဖြင့် သင်သည်-
- AZD ဖြင့် ဖြစ်ပေါ်လာနိုင်သည့် ပုံမှန် ဖြန့်ချိမှု မအောင်မြင်မှုများကို ရှာဖွေနိုင်မည်
- အတည်ပြုခြင်းနှင့် ခွင့်ပြုချက်ဆိုင်ရာ ပြဿနာများကို ဒစ်ဘပ်ဂင်းလုပ်နိုင်မည်
- AI ဝန်ဆောင်မှု ဆက်သွယ်မှုပတ်သက်သည့် ပြဿနာများကို ဖြေရှင်းနိုင်မည်
- ပြဿနာရှာဖွေရေးအတွက် Azure Portal နှင့် CLI များကို အသုံးပြုနိုင်မည်

## 🚨 အမြန်ဖြေရှင်းနည်းများ

### အသိမှတ်ပြုခြင်း ပြဿနာများ
```bash
# AZD လုပ်ငန်းစဉ်များအတွက် လိုအပ်သည်
azd auth login

# သင် Azure CLI အမိန့်များကို တိုက်ရိုက်လည်း အသုံးပြုနေပါက ရွေးချယ်နိုင်သည်
az login

azd auth status
```

### Provisioning မအောင်မြင်မှုများ
```bash
azd show
azd monitor --logs
az deployment sub list --query "[?properties.provisioningState!='Succeeded']"
```

### အရင်းအမြစ် ပဋိပက္ခများ
```bash
azd down --force --purge
azd env new different-name
azd up
```

### ကွာတာ ကျော်လွန်ခြင်း
```bash
az vm list-usage --location eastus --output table
azd env set AZURE_LOCATION westus2
azd up
```

## 📋 အမှားကုဒ် ကိုးကားစာရင်း

| အမှား | အကြောင်းပြချက် | ဖြေရှင်းချက် |
|-------|---------------|--------------|
| `AuthenticationError` | လော့ဂ်အင် မဖြစ်ထားခြင်း | `azd auth login` |
| `ResourceNotFound` | အရင်းအမြစ် မတွေ့ရခြင်း | အရင်းအမြစ် အမည်များကို စစ်ဆေးပါ |
| `QuotaExceeded` | စာရင်းသွင်းမှု အကန့်အသတ်များ | ကန့်သတ်ချက် ပမာဏ တိုးမြှင့်ရန် တောင်းဆိုပါ |
| `InvalidTemplate` | Bicep စာပေဖွဲ့စည်းပုံ အမှား | `az bicep build` |
| `Conflict` | အရင်းအမြစ် ရှိပြီးသား | အမည်အသစ် သတ်မှတ်ရန် သို့မဟုတ် ဖျက်ပစ်ပါ |
| `Forbidden` | ခွင့်ပြုချက် မလုံလောက်ခြင်း | RBAC အခန်းကဏ္ဍများကို စစ်ဆေးပါ |

## 🔄 ပြန်သတ်မှတ်ခြင်းနှင့် ပြန်လည်ကယ်တင်ခြင်း

```bash
# တင်းကျပ်မှုနည်းသော ပြန်စတင်ခြင်း (ရင်းမြစ်များကို ထိန်းသိမ်းထားပြီး၊ ကုဒ်ကို ထပ်မံ ပြန်တပ်ဆင်သည်)
azd deploy --force

# ပြင်းထန်သော ပြန်စတင်ခြင်း (အားလုံးကို ဖျက်ပစ်ပြီး၊ အသစ်အဖြစ် စတင်သည်)
azd down --force --purge
azd up
```
