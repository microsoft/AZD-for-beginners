# AZD အတွက် AI Developer အလုပ်ရုံဆွေးနွေးပွဲ

> ၂၀၂၆ ခုနှစ် ဇူလိုင်လတြင် `azd 1.27.1` ဖြင့် အတည်ပြုထားခြင်း။

> [!IMPORTANT]  
> **ဤအလုပ်ရုံဆွေးနွေးပွဲသည် သင်၏ ကိုယ်ပိုင် browser တွင် ကြိုတင်ကြည့်ရှုနိုင်သည့် အလုပ်ရုံလမ်းညွှန်တစ်ခုဖြင့် ဖြန့်ချိထားပါသည်။ စတင်ရန် GitHub Codespaces, dev container သို့မဟုတ် MkDocs တပ်ဆင်ထားသော ဒေသခံ clone တွင် repo ကိုဖွင့်ပြီး VS Code terminal တက္ကသိုလ်ကြီးတက်လာသည်အထိ စောင့်ဆိုင်းပြီး အောက်ပါ အမိန့်ကို ရိုက်ထည့်ပါ:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **သင်သည် browser တွင် ကြိုတင်ကြည့်ရှုမည့် စာမျက်နှာအား ဖွင့်ရန် pop-up dialog ကို မြင်ရပါမည်။**

Azure Developer CLI (AZD) ကို AI application တပ်ဆင်ရေးအတွက် အထူးအာရုံစိုက်၍ လေ့လာသင်ယူသည့် လက်တွေ့လေ့ကျင့်ခန်းအတွက် ကြိုဆိုပါ၏။ ဤအလုပ်ရုံဆွေးနွေးပွဲသည် AZD template များကို ၃ခြေလှမ်းတွင် လေ့လာနိုင်စေမှာ ဖြစ်ပါသည်။

1. **ရှာဖွေရေး** - သင့်အတွက် သင့်တော်သော template ကို ရှာဖွေခြင်း။
1. **တပ်ဆင်ခြင်း** - တပ်ဆင်ပြီး အလုပ်လုပ်သည်ကို အတည်ပြုခြင်း။
1. **ကိုင်တွယ်ပြုပြင်ခြင်း** - ကိုယ်တိုင်လိုအပ်သလို ပြောင်းလဲပြီး တိုးတတ်ခြင်း။

ဤအလုပ်ရုံဆွေးနွေးပွဲအတွင်း သင်သည် အထွေထွေအဆင့် ဖြေရှင်းရေးကိရိယာများနှင့် လုပ်ငန်းစဉ်များကိုလည်း လေ့လာသွားတော့မည်ဖြစ်သဖြင့် သင့်၏ နောက်ဆုံးထွက် စီမံချက်လမ်းကြောင်းကို ပိုမိုအဆင်ပြေစေပါသည်။

## အလုပ်ရုံဆွေးနွေးပွဲ အနှစ်ချုပ်

**ရှည်လျားချိန်:** ၃-၄ နာရီ  
**အဆင့်:** စတင်သူမှ အလယ်အလတ်အဆင့်  
**လိုအပ်ချက်များ:** Azure, AI အကြောင်းအရာများ၊ VS Code နှင့် ကွန်မန်းလိုင်းကိရိယာများအတွက် နှစ်သက်မှု။

ဤနေရာမှာ လက်တွေ့ လေ့ကျင့်ခြင်းဖြင့် သင်ယူမည့် အလုပ်ရုံဆွေးနွေးပွဲ ဖြစ်သည်။ လေ့ကျင့်ခန်းများပြီးဆုံးသည့်နောက် AZD စတင်သူများအတွက် သင်တန်းကို ပြန်လည်သုံးသပ်၍ လုံခြုံရေးနှင့် ထုတ်လုပ်မှုအကောင်းဆုံး အကြံပြုချက်များကို ဆက်လက်လေ့လာရန် အကြံပြုပါသည်။

| အချိန်| အပိုင်း | ရည်ရွယ်ချက် |
|:---|:---|:---|
| ၁၅ မိနစ် | မိတ်ဆက်ခြင်း | နောက်ခံသဘောထား၊ ရည်မှန်းချက်များနားလည်ခြင်း |
| ၃၀ မိနစ် | AI Template ရွေးချယ်ခြင်း | ရွေးချယ်စရာများကို ရှာဖွေပြီး စတင်လုပ်ဆောင်ခြင်း | 
| ၃၀ မိနစ် | AI Template အတည်ပြုခြင်း | မူရင်းဖြေရှင်းချက်ကို Azure တွင် တပ်ဆင်ခြင်း |
| ၃၀ မိနစ် | AI Template ဖောက်ခြင်း | ဖွဲ့စည်းမှုနှင့် ဖော်ပြချက်များကို လေ့လာခြင်း |
| ၃၀ မိနစ် | AI Template သတ်မှတ်ပြင်ဆင်ခြင်း | လုပ်ဆောင်ချက်များကို ဖွင့်လှစ်ပြီး ကြိုးစားစမ်းသပ်ခြင်း |
| ၃၀ မိနစ် | AI Template ကို ကိုက်ညီအောင် ပြင်ဆင်ခြင်း | သင့်လိုအပ်ချက်ဖြင့် template ကို လိုက်ဖက်အောင် ပြင်ဆင်ခြင်း |
| ၃၀ မိနစ် | အခြေခံအဆောက်အဦး ပိတ်ပင်ခြင်း | သန့်ရှင်းရေးလုပ်ဆောင်ပြီး အရင်းအမြစ်များ ပြန်လည်လွှတ်ပေးခြင်း |
| ၁၅ မိနစ် | အလုပ်ရုံဖျော်ဖြေမှုနှင့် နောက်တစ်ဆင့်များ | သင်ယူမှုသတင်းအချက်အလက်၊ အလုပ်ရုံစိန်ခေါ်မှု |
| | |

## သင်ယူမည့်အရာများ

AZD Template ကို Microsoft Foundry တွင် အဆုံးထိ ဖြေရှင်းရေးအတွက် အရည်အချင်းများနှင့် ကိရိယာများကို စမ်းပြသည့် သင်ယူမှု Sandbox အဖြစ် စဉ်းစားပါ။ ဤအလုပ်ရုံဆွေးနွေးပွဲပြီးဆုံးသည့်အခါ၊ သင်သည် ဤ အကြောင်းအရာအတွင်း လုပ်ဆောင်ရမည့် ကိရိယာများနှင့် အယူအဆများကို နားလည်မှု အတိအကျ ရရှိမည်ဖြစ်သည်။

| အယူအဆ | ရည်ရွယ်ချက် |
|:---|:---|
| **Azure Developer CLI** | ကိရိယာ အမိန့်များနှင့် လုပ်ငန်းစဉ်များကို နားလည်ခြင်း|
| **AZD Template များ**| ပရောဂျက် ဖွဲ့စည်းမှုနှင့် ဖော်ပြချက်များကို နားလည်ခြင်း|
| **Azure AI Agent**| Microsoft Foundry ပရောဂျက်ကို တပ်ဆင်နှင့် ဖြန့်ချိခြင်း  |
| **Azure AI Search**| ကိုယ်ပိုင်အချက်အလက်အင်ဂျင်နီယာလုပ်ငန်းများအတွက် အကူအညီပြုခြင်း |
| **လေ့လာစူးစမ်းနိုင်မှု (Observability)**| ချဲ့ထွင်ခြင်း၊ စောင့်ကြည့်ခြင်းနှင့် အကဲဖြတ်မှုများကို လေ့လာခြင်း |
| **Red Teaming**| သခင်တက်အကဲဖြတ်မှုနှင့် ကာကွယ်မှုများကို လေ့လာခြင်း |
| | |
