# AZD အတွက် စတုတ္ထပိုင်းရှင်များ: သင်ကြားမှု အကြောင်းအရာနှင့် သင်ယူမှု အခြေခံလမ်းကြောင်း

## သင်တန်း အနှစ်ချုပ်

Azure Developer CLI (azd) ကို လေ့လာရေးတိုးတက်မှုအဆင့်အလိုက် ဖန်တီးထားသော အပိုင်းများဖြင့် ကျွမ်းကျင်ပါ။ **Microsoft Foundry ပေါင်းစည်းမှုဖြင့် AI လျှောက်လွှာ ထုတ်လုပ်မှုကို အထူး အာရုံစိုက်ထားသည်။**

### ဒီသင်တန်းသည် လက်ရှိသူ ဖန်တီးသူများအတွက် အရေးပါသည့်အကြောင်း

Microsoft Foundry Discord အဖွဲ့လိုက် အမြင်များကို အခြေခံပြီး၊ **ဖန်တီးသူ ၄၅% သည် AI လိုအပ်ချက်များအတွက် AZD ကို အသုံးပြုလိုသော်လည်း အခက်အခဲများရှိကြောင်း** တွေ့ရှိထားသည်။
- ပေါင်းစပ်သော အများဆာဗစ် AI ဖွဲ့စည်းမှုရှုပ်ထွေးမှု
- ထုတ်လုပ်မှု AI ထုတ်လုပ်ခြင်း ဝါကျးကွက်များ  
- Azure AI စာရင်းဆက်စပ်ခြင်းနှင့် တပ်ဆင်ခြင်း
- AI လုပ်ငန်းများအတွက် ကုန်ကျစရိတ် ထိရောက်စွာ စီမံခန့်ခွဲခြင်း
- AI သီးသန့် ထုတ်လုပ်မှု ပြဿနာများ ဖြေရှင်းခြင်း

### အခြေခံ သင်ယူရန် ရည်မှန်းချက်များ

ဒီအစီအစဉ်ကို ပြီးမြောက်စေရင် သင်သည်
- **AZD အခြေခံများ ကျွမ်းကျင်မှု**: အဓိကအကြောင်းအရာများ၊ တပ်ဆင်ခြင်းနှင့် ဖော်ပြချက်
- **AI လျှောက်လွှာ ထုတ်လုပ်ခြင်း**: AZD နှင့် Microsoft Foundry ဝန်ဆောင်မှုများအသုံးပြုခြင်း
- **Infrastructure as Code ကို ကျင့်သုံးခြင်း**: Bicep တမ်းပလိတ်များဖြင့် Azure ရင်းမြစ်များစီမံခန့်ခွဲခြင်း
- **ထုတ်လုပ်မှု ပြဿနာများ ဖြေရှင်းခြင်း**: ပုံမှန်ပြဿနာများ ဖြေရှင်းခြင်းနှင့် ဒေဘတ်အကူအညီတောင်းခြင်း
- **ထုတ်လုပ်မှုအတွက် ထိရောက်ကောင်းမွန်စေခြင်း**: လုံခြုံရေး၊ အရွယ်အစားတိုးခြင်း၊ စောင့်ကြည့်မှုနှင့် ကုန်ကျစရိတ် စီမံခန့်ခွဲမှု
- **Multi-Agent ဖြေရှင်းချက်များ ဖန်တီးခြင်း**: ရုပ်သိမ်းရှုပ်ထွေးသော AI ဖွဲ့စည်းမှုများ ထုတ်လုပ်ခြင်း

## 🎓 ပညာသင်ကြားမှုအတွေ့အကြုံ

### လေ့လာမှု ဖြန့်ဝေမှု ရွေးချယ်မှုများ
ဒီသင်တန်းကို **ကိုယ်တိုင်အမြန်ဖြေရှင်းမှု** နှင့် **လက်တွေ့ ပန်းတိုင် အားဖြင့် သင်တန်းဆရာများ လက်အောက်မှာ သင်ယူမှု** နှစ်မျိုးလုံးအတွက် ထောက်ပံ့ဖြစ်စေရန် ဒီဇိုင်းပြုလုပ်ထားသည်။

#### 🚀 ကိုယ်တိုင်အမြန်ဖြေရှင်းမှု မုဒ်
**ကိုယ်တိုင် ဖန်တီးသူများနှင့် တိုးတက်သော လေ့လာမှုအတွက် သင့်တော်သည်**

**အင်္ဂါရပ်များ**
- **ဘရောက်ဇာ အခြေခံ မျက်နှာပြင်**: MkDocs အားအသုံးပြုသည့် ဝဘ်ဘရောက်ဇာဖြင့် လွယ်ကူစွာ ဝင်ရောက်နိုင်သော သင်တန်း
- **GitHub Codespaces လက်ခံပေးမှု**: သတ်မှတ်ထားသော အတူတူ ကိရိယာများဖြင့် တစ်ချက်နှိပ်ခြင်းနဲ့ ဖန်တီးရေးခန်း
- **အပြန်အလှန် DevContainer ပတ်ဝန်းကျင်**: ဒေသိယ တပ်ဆင်ခြင်း မလိုအပ်ဘဲ ကိုယ်တိုင်အစပိုင်း ကုဒ်ရေးနိုင်ခြင်း
- **တိုးတက်မှုလမ်းကြောင်း စောင့်ကြည့်မှု**: အစိတ်အပိုင်းများ အတည်ပြုခြင်းနှင့် လေ့ကျင့်ခန်းများ
- **အသိုင်းအဝိုင်း ထောက်ခံမှု**: Azure Discord များမှ မေးမြန်းနိုင်ခြင်းနှင့် ပူးပေါင်းဆောင်ရွက်နိုင်ခြင်း

**သင်ယူမှု ဖွဲ့စည်းမှုပုံစံ**
- **ချိန်ကို သင့်လိုအပ်သလို စီမံနိုင်ခြင်း**: ရက်အတော်များ သို့မဟုတ် အပတ်များအတွင်း အပိုင်းများ ပြီးစီးနိုင်ခြင်း
- **အတည်ပြု စနစ်**: တွန်းအားမြှင့်ခွင့် ရှိမည့် အကြောင်းအရာများတိုင် အတည်ပြုခြင်း
- **အရင်းအမြစ် စုစည်းမှု**: စုံလင်သော စာတမ်းများ၊ ပုံမှန်သုံး နမူနာများနှင့် ပြဿနာဖြေရှင်းလမ်းညွှန်များ
- **အလုပ်အပ်စဥ် ဖွံ့ဖြိုးတိုးတက်မှု**: ပညာရပ်အတက် မှူးစွမ်းရည်မြှင့်တင်ရေး အတွက် ပိုင်ဆိုင်နိုင်သော ပရောဂျက်ရေတွက်ဖွဲ့ခြင်း

**စတင်ခြင်း (ကိုယ်တိုင်အမြန်ဖြေရှင်း)**  
```bash
# ရွေးချယ်စရာ ၁: GitHub Codespaces (အကြံပြုသည်)
# ကုဒ်သိုလှောင်ရာသို့သွားပြီး "Code" → "main တွင် codespace ဖန်တီးရန်" ကိုနှိပ်ပါ

# ရွေးချယ်စရာ ၂: ဒေသဆိုင်ရာ ဖွံ့ဖြိုးတိုးတက်မှု
git clone https://github.com/microsoft/azd-for-beginners.git
cd azd-for-beginners/workshop
# workshop/README.md ထဲက setup ညွှန်ကြားချက်များကိုလိုက်နာပါ
```
  
#### 🏛️ လက်တွေ့ သင်တန်း အစည်းအဝေးများ
**ကုမ္ပဏီ သင်တန်းကြီးများ၊ bootcamp များနှင့် ပညာရေးအဖွဲ့အစည်းများအတွက် သင့်တော်သည်**

**သင်တန်း ဖြန့်ဝေမှု အမျိုးအစားများ**

**📚 ပညာရေးသင်တန်း ပေါင်းစပ်မှု (၈-၁၂ လ)**
- **တက္ကသိုလ်အစီအစဉ်များ**: တစ်ရာသီပတ်လည် ၂ နာရီကာလ အတန်းများ
- **Bootcamp ပုံစံ**: နေ့စဉ် ၆-၈ နာရီ လအတွင်း ၃-၅ ရက် စောင့်ကြည့်မှု
- **ကုမ္ပဏီ သင်တန်း**: လစဉ် အဖွဲ့အစည်း အစည်းအဝေးများနှင့် လက်တွေ့ပရောဂျက်များ အသုံးပြုခြင်း
- **အကဲဖြတ်မှု စနစ်**: အဆင့်ဖွဲ့ သတ်မှတ်ချက်များ၊ မိတ်ဆွေအကဲဖြတ်ခြင်းနှင့် နောက်ဆုံးပရောဂျက်များ

**🚀 တိုတောင်းထိရောက်သော သင်တန်း (၁-၃ ရက်)**
- **နေ့ ၁**: အခြေခံ + AI ဖန်တီးမှု (အပိုင်း ၁-၂) - ၆ နာရီ
- **နေ့ ၂**: ဖော်ပြချက် + အခြေခံအင်ဖရာ (အပိုင်း ၃-၄) - ၆ နာရီ  
- **နေ့ ၃**: အဆင့်မြင့် ပုံစံများ + ထုတ်လုပ်မှု (အပိုင်း ၅-၈) - ၈ နာရီ
- **နောက်တစ်ကြိမ်**: ၂ ပတ် ကြာ မန်တာ ရရှိမှု (လိုအပ်ပါက) ကိုယ်တိုင် လုပ်ငန်း ပြီးမြောက်အောင် ထောက်ပံ့ခြင်း

**⚡ စီမံခန့်ခွဲ အကြံပေးအစည်းအဝေး (၄-၆ နာရီ)**
- **မဟာဗျူဟာ တင်ပြချက်**: AZD တန်ဖိုးနှင့် လုပ်ငန်းအကျိုးသက်ရောက်မှု (၁ နာရီ)
- **လက်တွေ့ ပြသမှု**: AI လျှောက်လွှာ မစဉ်းစားထားဘဲ ထုတ်လုပ်မှု (၂ နာရီ)
- **ဖွဲ့စည်းပုံ ပြန်လည်ဆန်းစစ်ခြင်း**: စီးပွားရေးပုံစံများနှင့် အုပ်ချုပ်မှု (၁ နာရီ)
- **အကောင်အထည်ဖော် စီမံချက်**: အဖွဲ့အစည်းလက်ခံမှု မဟာဗျူဟာ (၁-၂ နာရီ)

#### 🛠️ သင်တန်း ပညာသင်ကြားမှု နည်းဗျူဟာ
**ရှာဖွေခြင်း → ထုတ်လုပ်ခြင်း → ကိုယ်ပိုင် ပြင်ဆင်ခြင်း ဗဟုသုတ ပေါ်လစီ**

**အဆင့် ၁: ရှာဖွေခြင်း (၄၅ မိနစ်)**
- **တမ်းပလိတ် ထည့်သွင်းကြည့်ရှုခြင်း**: Microsoft Foundry တမ်းပလိတ်များနှင့် ဝန်ဆောင်မှုများ အကဲဖြတ်ခြင်း
- **ဖွဲ့စည်းပုံ ဝေစုခွဲခြင်း**: Multi-agent ပုံစံများနှင့် ထုတ်လုပ်မှု စနစ်များကို နားလည်ခြင်း
- **လိုအပ်ချက် သုံးသပ်ခြင်း**: အဖွဲ့အစည်း လိုအပ်ချက်များနှင့် ကန့်သတ်ချက်များ ခွဲခြမ်းစိတ်ဖြာခြင်း
- **ပတ်ဝန်းကျင် စီမံမှုပုံစံ**: ဖွံ့ဖြိုးရေး ပတ်ဝန်းကျင် နှင့် Azure ရင်းမြစ်များ တပ်ဆင်ပြင်ဆင်ခြင်း

**အဆင့် ၂: ထုတ်လုပ်ခြင်း (၂ နာရီ)**
- **လမ်းညွှန်ချက်နှင့်အတူ ထုတ်လုပ်ခြင်း**: AZD ဖြင့် AI လျှောက်လွှာ တစ်ခုချင်းစီ စနစ်တကျ ထုတ်လုပ်ခြင်း
- **ဝန်ဆောင်မှု ဖော်ပြချက်**: Azure AI ဝန်ဆောင်မှုများ၊ ပွင့်လင်းတံခါးများနှင့် သက်သေခံအချက်အလက်များ တပ်ဆင်ခြင်း
- **လုံခြုံရေး ပုံစံများ အသုံးပြုခြင်း**: စီးပွားရေး သဘာဝ လုံခြုံရေး ပုံစံများနှင့် အလယ်အလတ် ထိန်းချုပ်မှု
- **အတည်ပြု စမ်းသပ်မှု**: ထုတ်လုပ်မှုအောင်မြင်မှုပြုခြင်းနှင့် သာမာန်ပြဿနာများ ဖြေရှင်းခြင်း

**အဆင့် ၃: ကိုယ်ပိုင် ပြင်ဆင်ခြင်း (၄၅ မိနစ်)**
- **လျှောက်လွှာ ပြင်ဆင်မှု**: သတ်မှတ်လိုသော အသုံးပြုမှုနှင့် တောင်းဆိုချက်များအတွက် template များကို အလိုက်အသင့် ပြင်ဆင်ခြင်း
- **ထုတ်လုပ်မှုထိရောက်စေခြင်း**: စောင့်ကြည့်မှု၊ ကုန်ကျစရိတ် စီမံခန့်ခွဲမှုနှင့် အရွယ်အစား တိုးခြင်း မူဝါဒများ အကောင်အထည်ဖော်ခြင်း
- **အဆင့်မြင့် ပုံစံများ**: Multi-agent ညှိနှိုင်းမှုနှင့် ရုပ်သိမ်းရှုပ်ထွေးသော ဖွဲ့စည်းမှုများ ဆန်းစစ်ခြင်း
- **နောက်ဆက်တွဲ လမ်းကြောင်း စီမံချက်**: နည်းပညာ ကျွမ်းကျင်မှု တိုးတက်ရန် လေ့လာမှု စီမံချက် သတ်မှတ်ခြင်း

#### 🎯 သင်တန်းမှ ဆောင်ရွက်ချက်များ
**လက်တွေ့ လေ့ကျင့်မှုမှ တိုးတက်သော ကျွမ်းကျင်မှုများ**

**နည်းပညာကျွမ်းကျင်မှုများ**
- **ထုတ်လုပ်မှု AI လျှောက်လွှာများ ထုတ်လုပ်ခြင်း**: AI စွမ်းဆောင်ရည်ပြည့်စုံသော ဖြေရှင်းချက်များ အောင်မြင်စွာ ထုတ်လုပ်ခြင်းနှင့် ဖော်ပြချက်
- **Infrastructure as Code ကျွမ်းကျင်မှု**: ကိုယ်ပိုင် Bicep တမ်းပလိတ်များ ဖန်တီးခြင်းနှင့် စီမံခန့်ခွဲခြင်း
- **Multi-Agent ဖွဲ့စည်းမှု**: AI ဂျင်နီယပ်များ စုပေါင်းညှိနှိုင်းပြီး အကောင်အထည်ဖော်ခြင်း
- **ထုတ်လုပ်မှု အဆင်သင့်ဖြစ်စေရေး**: လုံခြုံရေး၊ စောင့်ကြည့်မှုနှင့် အုပ်ချုပ်မှု ပုံစံများ အသုံးချခြင်း
- **ပြဿနာ ဖြေရှင်းမှု ကျွမ်းကျင်မှု**: ထုတ်လုပ်မှုနှင့် ဖော်ပြချက် ပြဿနာများကို ကိုယ်တိုင်ဖြေရှင်းနိုင်ခြင်း

**အလုပ်အမှုဆိုင်ရာ ကျွမ်းကျင်မှုများ**
- **ပရောဂျက် အုပ်ချုပ်မှု**: ကလောင်းရုပ်သောင်း အဖွဲ့များအား ဦးဆောင်ပြီး ကုလားလုပ်ငန်းများ ဆောင်ရွက်ခြင်း
- **ဖွဲ့စည်းပုံ ဒီဇိုင်း**: တိုးတက်ပြီး ကုန်ကျစရိတ် သက်သာသော Azure ဖြေရှင်းချက်များ ဒီဇိုင်းဆွဲခြင်း
- **အသိပညာ ပေးပို့မှု**: AZD သုံးစွဲမှုအကောင်းဆုံးနည်းလမ်းများကို မိတ်ဆွေများသို့ သင်ကြားပေးခြင်းနှင့် လမ်းပြခြင်း
- **မဟာဗျူဟာ ပြုစုရေး**: အဖွဲ့အစည်း ကလောင်းလုပ်ငန်း လက်ခံမှု မဟာဗျူဟာများ ထိန်းသိမ်း ဦးဆောင်ခြင်း

#### 📋 သင်တန်း အရင်းအမြစ်များနှင့် ပစ္စည်းများ
**သင်တန်းဆရာများနှင့် သင်ယူသူများအတွက် စုံလင်သော ကိရိယာများ**

**သင်တန်းဆရာများအတွက်**
- **ဆရာညွှန်ကြားမှု လမ်းညွှန်စာအုပ်**: [သင်တန်း အနှစ်ချုပ်](workshop/README.md) - အစည်းအဝေး စီစဉ်ခြင်း နှင့် ထုတ်ဝေရေး ညွှန်ကြားချက်များ
- **တင်ဆက်မှု ပစ္စည်းများ**: စကားပြောစာရွက်များ၊ ဖွဲ့စည်းပုံ ဇယားများနှင့် ပြသမှု စာသားများ
- **အကဲဖြတ်ကိရိယာများ**: လက်တွေ့ လေ့ကျင့်ခန်းများ၊ သိမြင်မှုစစ်ဆေးမှုများနှင့် အကဲဖြတ်တမ်းများ
- **နည်းပညာ ဆက်သွယ်မှု**: ပတ်ဝန်းကျင် ဖော်ပြချက်၊ ပြဿနာဖြေရှင်း လမ်းညွှန်များနှင့် ကူညီမှုအစီအစဉ်များ

**သင်ယူသူများအတွက်**
- **အပြန်အလှန် ပြုလုပ်နိုင်သော သင်တန်းပတ်ဝန်းကျင်**: [သင်တန်းပစ္စည်းများ](workshop/README.md) - ဘရောက်ဇာမှ လေ့လာရန် စနစ်တကျအား ကူညီမှု

#### 🏢 ကုမ္ပဏီသိပ္ပံ ပညာသင်ကြားမှု တပ်ဆင်မှု
**အဖွဲ့အစည်း သုံးစွဲမှုနှင့် သင်ကြားမှု မူဝါဒများ**

**ကုမ္ပဏီ သင်တန်း အစီအစဉ်များ**
- **ဖန်တီးသူ အသစ် ဝင်ရောက်ခြင်း**: AZD အခြေခံကိုယ်ပိုင် နည်းပညာ အသိပညာ ဝင်ရောက်ခြင်း (၂-၄ ပတ်)
- **အဖွဲ့ Upskill လုပ်ခြင်း**: လက်ရှိ ဖန်တီးသူအဖွဲ့များအတွက် သင်တန်းများ ပြန်လည် ဆောင်ရွက်ခြင်း (၁-၂ ရက်)
- **ဖွဲ့စည်းမှု ပြန်လည် စစ်ဆေးမှု**: တိုင်းတာနိုင်သော အင်ဂျင်နီယာများနှင့် ဖွဲ့စည်းမှုအထူးရှာဖွေမှု (၄ နာရီ)
- **ဦးဆောင်သူ ညှိနှိုင်းမှုများ**: နည်းပညာ ဆုံးဖြတ်မှု ထိန်းချုပ်သူများ အတွက် အနည်းငယ် ရက် သင်တန်းများ

**တပ်ဆင်မှု ထောက်ပံ့မှု**
- **ကိုယ်တိုင်ပြင်ဆင် စီမံချက်**: အဖွဲ့အစည်း လိုအပ်ချက်များအတွက် သင့်တော်သော သင်တန်း အကြောင်းအရာများ
- **ပုံပြင်အစီအစဉ် စီမံခန့်ခွဲမှု**: အောင်မြင်မှု အချက်အလက်များနှင့် မျှဝေမှုနည်းလမ်းများ ထည့်သွင်းထားသော ပုံမှန် စနစ်တကျ ဖြန့်ဝေခြင်း  
- **ဆက်လက် အထောက်အပံ့မူဝါဒ**: သင်တန်းပြီးနောက် ပရောဂျက် အကောင်အထည်ဖော်မှု အတွက် ထောက်ပံ့မှု
- **အသိုင်းအဝိုင်း ဖန်တီးခြင်း**: အဖွဲ့တွင်း Azure AI ဖန်တီးသူ အသိုင်းအဝိုင်းများနှင့် အသိပညာ မျှဝေမှု

**အောင်မြင်မှု အတိုင်းအတာများ**
- **ကျွမ်းကျင်မှု ရရှိမှု**: နည်းပညာကျွမ်းကျင်မှု တိုးတက်မှု ခန့်မှန်းမှုများ
- **ထုတ်လုပ်မှုအောင်မြင်မှု**: ပါဝင်သူများမှ ထုတ်လုပ်မှု အောင်မြင်စွာ လုပ်ဆောင်နည်း ရာခိုင်နှုန်း
- **ထုတ်လုပ်မှုအချိန် လျော့ချပ်မှု**: Azure AI ပရောဂျက်အသစ်များ အတွက် စတင်လေ့လာမှု အချိန် လျော့နည်းခြင်း
- **အသိပညာ ထိန်းသိမ်းမှု**: သင်တန်းပြီးနောက် ၃-၆ လအတွင်း ပြန်လည် စစ်ဆေးမှုများ

1. **ရှာဖွေခြင်း** (၃၀ မိနစ်): template ရွေးချယ်ခြင်းနှင့် အကဲဖြတ်ခြင်း
2. **ထုတ်လုပ်ခြင်း** (၄၅ မိနစ်): AI template လုပ်ဆောင်ချက် စစ်ဆေးခြင်း  
3. **ခွဲခြမ်းစိတ်ဖြာခြင်း** (၃၀ မိနစ်): template ဖွဲ့စည်းပုံ နားလည်ခြင်း
4. **ဖော်ပြချက် ပြင်ဆင်ခြင်း** (၃၀ မိနစ်): ကိုယ်ပိုင် ရွေးချယ်မှုများ ပြင်ဆင်ခြင်း
5. **ကိုယ်ပိုင် ပြင်ဆင်ခြင်း** (၄၅ မိနစ်): ကိုယ်ပိုင် အသုံးပြုမှုအတွက် ပြင်ဆင်ပြီး ပြန်လည်စမ်းသပ်ခြင်း
6. **အလယ်အလတ်ပိတ်ဆို့ခြင်း** (၁၅ မိနစ်): ရင်းမြစ်များကို သန့်ရှင်းခြင်း နှင့် အသက်တာချွန်လျော့ခြင်း နားလည်ခြင်း
7. **သတ်မှတ်ချက် မျှဝေခြင်း** (၁၅ မိနစ်): နောက်လမ်းကြောင်းများ နှင့် ဆက်လက်လေ့လာနိုင်မှု

- Multi-environment ဖော်ပြချက် နှင့် ထုတ်လုပ်မှုများ

### အပိုင်း ၅: Multi-Agent AI ဖြေရှင်းချက်များ (၂-၃ နာရီ) 🤖🤖  
**လိုအပ်ချက်များ**: အပိုင်း ၁-၂ ပြီးမြောက်ထားမှု

- ကွန်တိနာ မိုက်ခရိုဆာဗစ် များကို agent အခြေပြု ဖြေရှင်းချက်တွင် ပေါင်းစည်းအသုံးပြုခြင်း

- **🏢 ရုံးလုပ်ငန်း**: ကဏ္ဍများအားလုံး ပြီးစီးပြီး ကဏ္ဍ ၈ (ထုတ်လုပ်မှုပုံစံများ) အာရုံပြုပါ

- **Container App လမ်းကြောင်း**: ကဏ္ဍ ၄ (Containerized deployment), ၅ (Microservices ပေါင်းစည်းခြင်း), ၈ (ထုတ်လုပ်မှုအကောင်းဆုံး လုပ်ထုံး လမ်းညွှန်များ)

- **ကုမ္ပဏီလက်ခံ သင်တန်း**: AZD နှင့် AI ထည့်သွင်းမှု ကျွမ်းကျင်မှုများ မျှဝေပေးခြင်း နှင့် သင်ကြားခြင်း
