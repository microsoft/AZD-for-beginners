# Changelog - AZD For Beginners

## အကျဉ်းချုပ်

## သင်ယူရမည့် ရည်မှန်းချက်များ

ဒီ changelog ကို ကြည့်ရှုခြင်းအားဖြင့် သင်:
- အင်္ဂါရပ်အသစ်များနှင့် အကြောင်းအရာ ထပ်မံထည့်သွင်းမှုများကို သိရှိနိုင်မည်

## Changelog အတွက် အထောက်အကူပြုမှု

### ပြောင်းလဲမှုများကို မှတ်တမ်းတင်ခြင်း
ဒီ repository ကို အထောက်အကူပြုတဲ့အခါ changelog မှတ်တမ်းတွေမှာ အောက်ပါအချက်တွေ ပါဝင်ရမည်:

1. **Version Number**: Semantic versioning (major.minor.patch) ကို လိုက်နာရမည်
2. **Date**: YYYY-MM-DD format နဲ့ ထုတ်ဝေမှုရက်စွဲ
3. **Category**: Added, Changed, Deprecated, Removed, Fixed, Security
4. **Clear Description**: ပြောင်းလဲမှုအကြောင်းကို ရှင်းလင်းစွာ ဖော်ပြရမည်
5. **Impact Assessment**: ပြောင်းလဲမှုတွေက ရှိပြီးသား အသုံးပြုသူတွေကို ဘယ်လို အကျိုးသက်ရောက်မှုရှိမလဲ
