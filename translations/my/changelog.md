## Changelog အတွက် အထောက်အကူပြုခြင်း

### Reporting Changes
ဤ repository သို့ အထောက်အကူပြုသောအခါ changelog entry များတွင် အောက်ပါအချက်များ ပါဝင်ရန် သေချာပါစေ-

1. **Version Number**: semantic versioning (major.minor.patch) ကို လိုက်နာခြင်း
2. **Date**: YYYY-MM-DD format ဖြင့် ထုတ်ဝေသည့်ရက်
3. **Category**: Added, Changed, Deprecated, Removed, Fixed, Security
4. **Clear Description**: ပြောင်းလဲမှုအကြောင်းကို ရှင်းလင်းသော ဖော်ပြချက်
5. **Impact Assessment**: ပြောင်းလဲမှုများသည် ရှိပြီးသား user များကို ဘယ်လို သက်ရောက်မှုရှိမည်ကို ဖော်ပြခြင်း
