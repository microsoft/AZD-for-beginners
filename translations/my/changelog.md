## Contributing to the Changelog

3. **Category**: Added, Changed, Deprecated, Removed, Fixed, Security

- Security vulnerabilities ဖြေရှင်းမှု

### Semantic Versioning Guidelines

#### Major Version (X.0.0)
- User action လိုအပ်သော breaking changes
- Content သို့မဟုတ် organization အရေးကြီး ပြန်လည်ဖွဲ့စည်းမှု
- Fundamental approach သို့မဟုတ် methodology ကို ပြောင်းလဲမှု

#### Minor Version (X.Y.0)
- အသစ်ထည့်သွင်းထားသော features သို့မဟုတ် content
- Backward compatibility ကို ထိန်းသိမ်းထားသော enhancements
- အသစ်ထည့်သွင်းထားသော examples, tools သို့မဟုတ် resources

#### Patch Version (X.Y.Z)
- Bug fixes နှင့် corrections
- ရှိပြီးသား content အတွက် အနည်းငယ် တိုးတက်မှု
- ရှင်းလင်းမှုများနှင့် သေးငယ်သော enhancements

### Response Commitment
- **Issue Response**: Report ပြုလုပ်ထားသော ပြဿနာများအတွက် 48 နာရီအတွင်း
