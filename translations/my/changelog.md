# ပြောင်းလဲမှုစာရင်း - AZD For Beginners

## နိဒါန်း

ဤပြောင်းလဲမှုစာရင်းသည် AZD For Beginners repository ထဲရှိ သတ်မှတ်ရန်ထင်ရှားသော ပြောင်းလဲမှုများ၊ အပ်ဒိတ်များ နှင့် တိုးတက်မှုများအားလုံးကို မှတ်တမ်းတင်ထားသည်။ ငါတို့သည် semantic versioning နည္းလမ်းများကို လိုက်နာပြီး အဆင့်များအလိုက် ဘာတွေပြောင်းလဲခဲ့ကြောင်း အသုံးပြုသူများ အလွယ်တကူ နားလည်နိုင်ရန် ဤမှတ်တမ်းကို ထိန်းသိမ်းထားသည်။

## သင်ယူရန် ရည်မှန်းချက်များ

ဤပြောင်းလဲမှုစာရင်းကို ကြည့်ရှုခြင်းအားဖြင့် သင်သည် -
- အဆိုပါ ဗားရှင်းများအကြား အသစ်ထပ်ထည့်ထားသော feature နှင့် ပစ္စည်းများအကြောင်း သတင်းရှိနေမည်
- ရှိနေသော စာတမ်းများတွင် ပြုလုပ်ထားသော တိုးတက်မှုများကို နားလည်နိုင်မည်
- စည်းကမ်းလက်နက်များနှင့် ပြင်ဆင်ချက်များကို လိုက်နာ၍ အမှားရှာဖွေမှုများကို လေ့လာနိုင်မည်
- သင်ကြားရေး ပစ္စည်းများ၏ ဖွံ့ဖြိုးတိုးတက်မှုကို အချိန်ဖြတ်၍ ချိန်းဆိုနိုင်မည်

## သင်ယူပြီးရရှိမည့် ရလဒ်များ

ပြောင်းလဲမှုစာရင်း၏ အ entries များကို ကြည့်ရှုပြီးနောက် သင်သည် -
- လေ့လာရေးအတွက် အသစ်ထည့်သွင်းထားသော အကြောင်းအရာများနှင့် အရင်းအမြစ်များကို မှတ်သားနိုင်မည်
- ဘယ်အပိုင်းများကို အပ်ဒိတ်လုပ်ခဲ့ကြောင်း သို့မဟုတ် တိုးတက်အောင် ပြုပြင်ထားကြောင်း သိနိုင်မည်
- လက်ရှိ အကြောင်းအရာများအပေါ် အခြေထားပြီး သင်ယူရေး လမ်းကြောင်းကို စီစဉ်နိုင်မည်
- အနာဂတ် အဆင်ပြေမှုများအတွက် တုံ့ပြန်ချက်များနှင့် အကြံပြုချက်များ ပံ့ပိုး ကြိုတင်ပေးနိုင်မည်

## Contributing to the Changelog

### Reporting Changes
ဤ repository သို့ ပါဝင်ရန် ဖြစ်ပါက changelog ချရေးချက်များတွင် အောက်ပါအချက်များပါဝင်ရန် အာမခံပေးပါ။

1. **Version Number**: semantic versioning အတိုင်း (major.minor.patch) ကိုလိုက်နာပါ
2. **Date**: ထုတ်ပြန်သည့် သို့မဟုတ် အပ်ဒိတ်လုပ်သည့် ရက်စွဲကို YYYY-MM-DD ပုံစံဖြင့် ဖော်ပြပါ
3. **Category**: Added, Changed, Deprecated, Removed, Fixed, Security
4. **Clear Description**: ဘာများပြောင်းလဲခဲ့သည်ကို စာတောက်လျောက် ရှင်းလင်းဖော်ပြပါ
5. **Impact Assessment**: ပြောင်းလဲမှုများသည် ရှိပြီးသား အသုံးပြုသူများကို မည်သို့ထိခိုက်စေပါသည်ကို ဖော်ပြပါ

- Accessibility: မတူကွဲပြားသော သင်ယူသူများ အတွက် တိုးတက်ကောင်းမွန်စေရန် အကြံပြုချက်များ
- AI Tool Integration: AI ဖွံ့ဖြိုးရေး လုပ်ဆောင်ချက်များ ပိုမိုကောင်းမွန်စေရန် ပစ္စည်းအတွဲ ပေါင်းစည်းမှု အကြံပြုချက်များ
- Production AI Patterns: စက်မှုအသုံးပြု AI deployment ပုံစံများအတွက် တောင်းဆိုချက်များ

### Response Commitment
- Issue Response: တင်ပြထားသည့် ပြဿနာများအတွက် ၄၈ နာရီအတွင်း တုံ့ပြန်မည်
- Feature Requests: တစ်ပတ်အတွင်း အကဲဖြတ်မည်
- Community Contributions: တစ်ပတ်အတွင်း ပြန်လည်သုံးသပ်မည်
- Security Issues: အဓိက ဦးစားပေး၍ အရေးပေါ် ပြန်လည်ဖြေရှင်းမည်

- Quarterly Updates: အဓိက အကြောင်းအရာ ထည့်သွင်းပြင်ဆင်ခြင်း
- Semi-Annual Reviews: လုံးဝ ပြန်လည်ဖွဲ့စည်းခြင်းနှင့် တိုးတက်မြှင့်တင်ခြင်း

### Monitoring and Quality Assurance
- Automated Testing: ကုဒ်နမူနာများနှင့် လင့်ခ်များကို ပုံမှန် အတည်ပြုခြင်း
- Community Feedback Integration: အသုံးပြုသူ အကြံပြုချက်များကို ပုံမှန် ထည့်သွင်းခြင်း
- Technology Updates: နောက်ဆုံး Azure ၀န်ဆောင်မှုများနှင့် azd ထုတ်မီ ယှဉ်စားညီမွေ့စေရန် ကိုက်ညီမှု
- Accessibility Audits: အားလုံးအတွက် အထောက်အကူဖြစ်စေရန် ဒီဇိုင်း မူဝါဒများကို ပုံမှန် စစ်ဆေးခြင်း

- **Compatibility Notes**: စနစ်ပျက်စီးစေသော ပြောင်းလဲမှုများဆိုင်ရာ အသေးစိတ် မှတ်စုများ
