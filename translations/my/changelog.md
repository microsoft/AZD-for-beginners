## Contributing to the Changelog

### Reporting Changes
ဤ repository သို့ အထောက်အကူပြုရန် changelog entries တွင် အောက်ပါအချက်များပါဝင်ရန် သေချာစေပါ။

1. **Version Number**: semantic versioning (major.minor.patch) ကိုလိုက်နာခြင်း
2. **Date**: YYYY-MM-DD format ဖြင့် ထုတ်ဝေသည့်ရက်
3. **Category**: Added, Changed, Deprecated, Removed, Fixed, Security
4. **Clear Description**: ပြောင်းလဲမှုအကြောင်းအရာကို တိကျသောဖော်ပြချက်

#### Added
