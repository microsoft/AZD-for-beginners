## Workshop Overview

**Time:** 3-4 hours  
**Level:** Beginner to Intermediate  
**Wetin you need sabi before:** You suppose sabi small about Azure, AI concepts, VS Code & command-line tools.

Dis na hands-on workshop wey you go learn by doing. After you finish di exercises, we go advise make you check di AZD For Beginners curriculum to continue your learning journey about Security and Productivity best practices.

| Time| Module  | Wetin you go learn |
|:---|:---|:---|
| 15 mins | Introduction | Understand di goals and wetin you go do |
| 30 mins | Select AI Template | Check di options and choose starter | 
| 30 mins | Validate AI Template | Deploy di default solution to Azure |
| 30 mins | Deconstruct AI Template | Check di structure and configuration |
| 30 mins | Configure AI Template | Activate and test di features wey dey |
| 30 mins | Customize AI Template | Change di template to fit your needs |
| 30 mins | Teardown Infrastructure | Clean up and release resources |
| 15 mins | Wrap-Up & Next Steps | Learning resources, Workshop challenge |

## Wetin You Go Learn

Think of di AZD Template as one kind learning sandbox wey you fit use explore different tools and capabilities for end-to-end development on Azure AI Foundry. By di end of dis workshop, you suppose don sabi di tools and concepts well well.

| Concept  | Wetin you go sabi |
|:---|:---|
| **Azure Developer CLI** | Sabi di tool commands and workflows|
| **AZD Templates**| Understand di project structure and config|
| **Azure AI Agent**| Set up and deploy Azure AI Foundry project  |
| **Azure AI Search**| Use agents to enable context engineering |
| **Observability**| Check tracing, monitoring and evaluations |
| **Red Teaming**| Test for adversarial issues and how to fix dem |
