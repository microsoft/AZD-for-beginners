## 2. Key Concepts wey You Master

### Azure Developer CLI (azd)
- `azd init` - Start projects from templates
- `azd up` - Provision and deploy together with one command
- `azd env` - Manage environment variables
- `azd down` - Take down resources safe

### AZD Template Structure
- `azure.yaml` - Configuration and deployment hooks
- `infra/` - Bicep infrastructure-as-code
- `src/` - Application source code
- `scripts/` - Automation and setup scripts

### Microsoft Foundry Capabilities
- AI Agent Service wey dey power intelligent assistants
- File Search and Azure AI Search for knowledge retrieval
- Built-in tracing, monitoring, and evaluation
- Red teaming to validate AI safety
