# AZD For AI Developers

!!! info "AUDIENCE FOR THIS WORKSHOP"
   
    Dis workshop na for anybody wey wan build AI Agentic applications for Azure AI Foundry by using beta tools and workflows. E fit both people wey dey start and people wey don sabi well well, as long as dem sabi small about Azure, AI concepts and Python code. Abeg check di pre-requisites before you start.

    **Duration:** 3-4 hours  
    **Level:** Beginner to Intermediate  

---


## Pre-Requisites

- You go need personal GitHub account → [Create one for free](https://github.com/signup)
- You go need active Azure account → [Create one for free](https://aka.ms/free)
- You go need sabi AI concepts, Python development, Command-line tools
- You go need experience with VS Code and GitHub tools and workflows.

---

## Learning Objectives

By di end of dis workshop, you go sabi how to:

- [X] **Work with Azure Developer CLI** → to setup, deploy, and manage your Azure resources.
- [X] **Use AI App Templates** → to quick start AI Agent projects for Azure AI Foundry.
- [X] **Configure Templates** → for features like Red Teaming, Evaluations, Tracing & Monitoring
- [X] **Deconstruct and Explore** → AZD templates and workflows using GitHub Copilot
- [X] **Customize and Redeploy** → di AI Agent starter template, make e match your own requirements

---

## Workshop Outline

Dis workshop get modules, each one get hands-on lab exercise:

1. **Introduction** - Understand wetin AZD templates be, and how dem dey work
1. **Selection** - Find di correct "starter" template for your next AI project
1. **Validation** - Deploy and check di template to make sure e dey work as dem talk
1. **Deconstruction** - Break di template down to understand di components & configuration
1. **Configuration** - Learn how to activate and use different Azure AI Foundry features
1. **Customization** - Rebuild di application with your own data and requirements
1. **Teardown** - Learn how to remove infrastructure and recover soft-deleted resources
