# AZD For AI Developers

!!! info "PEOPLE WEY DIS WORKSHOP DEY FOR"
   
    Dis workshop na for anybody wey wan build AI Agentic applications on Microsoft Foundry by using best-in-class tools and workflows. Di workshop dey good for both beginners and experienced developers wey get small sabi about Azure, AI concepts and Python code. Abeg check the pre-requisites before you start.

    **Duration:** 3-4 hours  
    **Level:** Beginner to Intermediate  

---


## Pre-Requisites

- A personal GitHub account → [Make one for free](https://github.com/signup)
- An active Azure account → [Make one for free](https://aka.ms/free)
- Small sabi of AI concepts, Python development, Command-line tools
- Experience with VS Code and GitHub tools and workflows.

---

## Learning Objectives

By the end of this workshop, you go sabi how to:

- [X] **Work with Azure Developer CLI** → make, deploy, and manage your Azure resources.
- [X] **Use AI App Templates** → to quickly start AI Agent projects on Microsoft Foundry.
- [X] **Configure Templates** → for features like Red Teaming, Evaluations, Tracing & Monitoring
- [X] **Deconstruct and Explore** → take AZD templates and workflows apart and explore dem using GitHub Copilot
- [X] **Customize and Redeploy** → make the AI Agent starter template fit your custom requirements and redeploy am

---

## Workshop Outline

The workshop dey arranged as modules, each one get a hands-on lab exercise:

1. **Introduction** - Make you understand wetin AZD templates be, and how dem dey work
1. **Selection** - Find di correct "starter" template for your next AI project
1. **Validation** - Deploy and explore the template to confirm say e work as dem talk
1. **Deconstruction** - Break the template down to understand its components & configuration
1. **Configuration** - Learn how to activate and use different Microsoft Foundry features
1. **Customization** - Rebuild the application with your own data and requirements in mind
1. **Teardown** - Learn how to deprovision infrastructure and recover soft-deleted resources
