# کارگاه AZD برای توسعه‌دهندگان هوش مصنوعی

> اعتبارسنجی شده با `azd 1.27.1` در ژوئیه ۲۰۲۶.

> [!IMPORTANT]  
> **این کارگاه مجهز به راهنمای کارگاه است که می‌توانید در مرورگر محلی خود پیش‌نمایش کنید. برای شروع، مخزن را در GitHub Codespaces، یک کانتینر توسعه یا یک کلون محلی با MkDocs نصب شده باز کنید—سپس تا زمانی که ترمینال فعال VS Code را دیدید صبر کنید و تایپ کنید:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **باید یک پنجره بازشو ببینید تا صفحه پیش‌نمایش را در مرورگر باز کنید.**

به کارگاه عملی یادگیری Azure Developer CLI (AZD) با تمرکز بر استقرار برنامه‌های هوش مصنوعی خوش آمدید. این کارگاه به شما کمک می‌کند تا در ۳ مرحله درک عملی از قالب‌های AZD کسب کنید:

1. **کشف** - قالبی را پیدا کنید که مناسب شما باشد.
1. **استقرار** - استقرار دهید و اعتبارسنجی کنید که کار می‌کند
1. **سفارشی‌سازی** - تغییر دهید و تکرار کنید تا مختص شما شود!

در طول این کارگاه، با ابزارها و روندهای اصلی توسعه‌دهندگان نیز آشنا خواهید شد تا به شما کمک کند مسیر توسعه انتها به انتهای خود را تسهیل کنید.

## نمای کلی کارگاه

**مدت زمان:** ۳-۴ ساعت  
**سطح:** مبتدی تا متوسط  
**پیش‌نیازها:** آشنایی با Azure، مفاهیم AI، VS Code و ابزارهای خط فرمان.

این یک کارگاه عملی است که شما با انجام کارها یاد می‌گیرید. پس از اتمام تمرین‌ها، توصیه می‌کنیم برنامه AZD برای مبتدیان را مرور کنید تا مسیر یادگیری خود را در بهترین روش‌های امنیت و بهره‌وری ادامه دهید.

| زمان| بخش  | هدف |
|:---|:---|:---|
| ۱۵ دقیقه | معرفی | ایجاد چارچوب، فهم اهداف |
| ۳۰ دقیقه | انتخاب قالب هوش مصنوعی | کاوش گزینه‌ها و انتخاب شروع‌کننده | 
| ۳۰ دقیقه | اعتبارسنجی قالب هوش مصنوعی | استقرار راه‌حل پیش‌فرض در Azure |
| ۳۰ دقیقه | تحلیل قالب هوش مصنوعی | بررسی ساختار و پیکربندی |
| ۳۰ دقیقه | پیکربندی قالب هوش مصنوعی | فعال‌سازی و امتحان ویژگی‌های موجود |
| ۳۰ دقیقه | سفارشی‌سازی قالب هوش مصنوعی | تطبیق قالب با نیازهای خود |
| ۳۰ دقیقه | تخریب زیرساخت | پاک‌سازی و آزادسازی منابع |
| ۱۵ دقیقه | جمع‌بندی و مراحل بعدی | منابع یادگیری، چالش کارگاه |

## آنچه خواهید آموخت

قالب AZD را به عنوان یک محیط آزمون یادگیری برای کشف توانایی‌ها و ابزارهای مختلف توسعه انتها به انتهای Microsoft Foundry در نظر بگیرید. تا پایان این کارگاه، باید حسی شهودی از ابزارها و مفاهیم مختلف در این زمینه داشته باشید.

| مفهوم  | هدف |
|:---|:---|
| **Azure Developer CLI** | درک دستورات ابزار و روندهای کاری |
| **قالب‌های AZD**| درک ساختار پروژه و پیکربندی |
| **عامل هوش مصنوعی Azure**| فراهم‌سازی و استقرار پروژه Microsoft Foundry  |
| **جستجوی هوش مصنوعی Azure**| فعال‌سازی مهندسی زمینه با عوامل |
| **نظارت‌پذیری**| کاوش ردکاوی، پایش و ارزیابی‌ها |
| **تیم قرمز**| کاوش آزمون‌های مقابله‌ای و کاهش‌دهنده‌ها |
| | |
