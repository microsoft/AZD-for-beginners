# کارگاه AZD برای توسعه‌دهندگان هوش مصنوعی

> اعتبارسنجی شده با `azd 1.25.6` در ژوئن ۲۰۲۶.

> [!IMPORTANT]  
> **این کارگاه دارای یک راهنمای کارگاه است که می‌توانید در مرورگر محلی خود پیش‌نمایش کنید. برای شروع، مخزن را در GitHub Codespaces، یک dev container، یا یک کلون محلی با MkDocs نصب‌شده باز کنید—سپس تا زمانی که یک ترمینال فعال VS Code ظاهر شود صبر کنید و تایپ کنید:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **باید یک دیالوگ پاپ‌آپ برای باز کردن صفحه پیش‌نمایش در مرورگر ببینید.**

خوش‌آمدید به کارگاه عملی برای یادگیری Azure Developer CLI (AZD) با تمرکز بر استقرار برنامه‌های هوش مصنوعی. این کارگاه به شما کمک می‌کند تا درک کاربردی از قالب‌های AZD را در 3 گام به‌دست آورید:

1. **کشف** - یافتن قالبی که برای شما مناسب است.
1. **استقرار** - استقرار و اعتبارسنجی اینکه کار می‌کند
1. **سفارشی‌سازی** - تغییر و تکرار برای شخصی‌سازی آن!

در طول این کارگاه، با ابزارها و جریان‌های کاری اصلی توسعه‌دهندگان نیز آشنا خواهید شد تا به بهینه‌سازی مسیر توسعه سرتاسری خود کمک کنید.

## مروری بر کارگاه

**مدت:** 3-4 ساعت  
**سطح:** مبتدی تا متوسط  
**پیش‌نیازها:** آشنایی با Azure، مفاهیم هوش مصنوعی، VS Code و ابزارهای خط فرمان.

این یک کارگاه عملی است که در آن با انجام دادن یاد می‌گیرید. پس از تکمیل تمرین‌ها، پیشنهاد می‌کنیم برای ادامه مسیر یادگیری خود به سراغ مجموعه آموزش AZD برای مبتدیان بروید تا بهترین شیوه‌ها در زمینه امنیت و بهره‌وری را دنبال کنید.

| زمان| ماژول  | هدف |
|:---|:---|:---|
| 15 mins | معرفی | آماده‌سازی زمینه، درک اهداف |
| 30 mins | انتخاب قالب AI | بررسی گزینه‌ها و انتخاب شروع‌کننده | 
| 30 mins | اعتبارسنجی قالب AI | استقرار راه‌حل پیش‌فرض در Azure |
| 30 mins | بررسی ساختار قالب AI | کاوش ساختار و پیکربندی |
| 30 mins | پیکربندی قالب AI | فعال‌سازی و آزمایش ویژگی‌های موجود |
| 30 mins | سفارشی‌سازی قالب AI | سازگار کردن قالب با نیازهای شما |
| 30 mins | خاتمه و پاک‌سازی زیرساخت | پاک‌سازی و آزادسازی منابع |
| 15 mins | جمع‌بندی و گام‌های بعدی | منابع یادگیری، چالش کارگاه |
| | |

## آنچه خواهید آموخت

قالب AZD را به‌عنوان یک فضای آزمایشی برای کاوش قابلیت‌ها و ابزارهای مختلف توسعه سرتاسری در Microsoft Foundry در نظر بگیرید. تا پایان این کارگاه، باید درکی شهودی از ابزارها و مفاهیم مختلف در این زمینه به‌دست آورده باشید.

| مفهوم  | هدف |
|:---|:---|
| **Azure Developer CLI** | درک دستورات ابزار و جریان‌های کاری|
| **AZD Templates**| درک ساختار پروژه و پیکربندی|
| **Azure AI Agent**| ایجاد و استقرار پروژه Microsoft Foundry  |
| **Azure AI Search**| فعال‌سازی مهندسی زمینه با عامل‌ها |
| **Observability**| بررسی ردیابی، نظارت و ارزیابی‌ها |
| **Red Teaming**| بررسی آزمون‌های خصمانه و راه‌های کاهش آن‌ها |
| | |
