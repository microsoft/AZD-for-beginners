# AZD برای توسعه‌دهندگان هوش مصنوعی

!!! info "مخاطبان این کارگاه"
   
    این کارگاه برای هر کسی است که می‌خواهد برنامه‌های عامل‌محور هوش مصنوعی را روی Microsoft Foundry با استفاده از بهترین ابزارها و گردش‌کارها بسازد. این کارگاه برای مبتدیان و توسعه‌دهندگان باتجربه که آشنایی‌ای با Azure، مفاهیم هوش مصنوعی و کدنویسی پایتون دارند مناسب است. لطفاً قبل از شروع، پیش‌نیازها را بررسی کنید.

    **مدت:** 3-4 ساعت  
    **سطح:** مبتدی تا متوسط  

---


## پیش‌نیازها

- یک حساب GitHub شخصی → [یکی را به‌صورت رایگان ایجاد کنید](https://github.com/signup)
- یک حساب Azure فعال → [یکی را به‌صورت رایگان ایجاد کنید](https://aka.ms/free)
- آشنایی با مفاهیم هوش مصنوعی، توسعه پایتون، ابزارهای خط فرمان
- تجربه کار با VS Code و ابزارها و گردش‌کارهای GitHub.

---

## اهداف یادگیری

در پایان این کارگاه، شما خواهید دانست چگونه:

- [X] **کار با Azure Developer CLI** → برای فراهم‌سازی، استقرار و مدیریت منابع Azure خود.
- [X] **استفاده از قالب‌های برنامه هوش مصنوعی** →  برای راه‌اندازی سریع پروژه‌های عامل هوش مصنوعی روی Microsoft Foundry.
- [X] **پیکربندی قالب‌ها** →  برای ویژگی‌هایی مانند Red Teaming، Evaluations، Tracing & Monitoring
- [X] **بازشکافی و بررسی** →  قالب‌ها و گردش‌های کاری AZD را با استفاده از GitHub Copilot
- [X] **سفارشی‌سازی و استقرار مجدد** →  قالب شروع‌کننده عامل هوش مصنوعی را مطابق نیازمندی‌های سفارشی خود سفارشی‌سازی و مجدداً مستقر کنید

---

## ساختار کارگاه

1. **مقدمه** - آشنایی با مفهوم قالب‌های AZD و نحوه عملکرد آن‌ها
1. **انتخاب** - قالب مناسب "شروع‌کننده" برای پروژه بعدی هوش مصنوعی خود را کشف کنید
1. **اعتبارسنجی** - استقرار و بررسی قالب برای تأیید عملکرد آن همان‌طور که اعلام شده
1. **بازشکافی** - قالب را قطعه‌قطعه کنید تا اجزا و پیکربندی آن را درک کنید
1. **پیکربندی** - یاد بگیرید چگونه ویژگی‌های مختلف Microsoft Foundry را فعال و استفاده کنید
1. **سفارشی‌سازی** - برنامه را دوباره با در نظر گرفتن داده‌ها و نیازمندی‌های خود بازسازی کنید
1. **پاک‌سازی** - یاد بگیرید چگونه زیرساخت را از حالت تخصیص خارج کرده و منابع حذف‌شده نرم را بازیابی کنید

---

## بازخورد و مشکلات

ما به‌طور فعال برنامه درسی را به‌روزرسانی می‌کنیم تا بهترین تجربه یادگیری را برای مبتدیان فراهم کنیم. از بازخورد شما استقبال می‌شود.

1. یک issue در این مخزن ارسال کنید - برای سهولت آن را با `Workshop` برچسب‌گذاری کنید.
1. به Microsoft Foundry Discord بپیوندید - با سایر یادگیرندگان ارتباط برقرار کنید!
