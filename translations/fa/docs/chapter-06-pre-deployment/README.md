## بررسی اجمالی

این فصل مراحل ضروری برنامه‌ریزی و اعتبارسنجی قبل از استقرار برنامهٔ شما را پوشش می‌دهد. بیاموزید چگونه با برنامه‌ریزی ظرفیت مناسب، انتخاب SKU و بررسی‌های پیش‌پرواز از اشتباهات پرهزینه جلوگیری کنید.

> اعتبارسنجی شده با `azd 1.23.12` در مارس ۲۰۲۶.

## اهداف یادگیری

با تکمیل این فصل، شما:
- اجرای بررسی‌های پیش‌استقرار قبل از استقرار
- برنامه‌ریزی ظرفیت و برآورد نیازمندی‌های منابع
- انتخاب SKUهای مناسب برای بهینه‌سازی هزینه
- پیکربندی Application Insights برای پایش
- درک الگوهای هماهنگی تیمی

## 🚀 شروع سریع

```bash
# سهمیه‌های اشتراک را بررسی کنید
az vm list-usage --location eastus --output table

# پیش‌نمایش استقرار (بدون ایجاد منابع)
azd provision --preview

# اعتبارسنجی نحو Bicep
az bicep build --file infra/main.bicep

# پیکربندی محیط را بررسی کنید
azd env get-values
```

---

## ☑️ چک‌لیست پیش از استقرار

### قبل از `azd provision`

- [ ] سهمیه برای منطقه تأیید شده
- [ ] SKUها به‌درستی انتخاب شده‌اند
- [ ] برآورد هزینه بررسی شده
- [ ] قواعد نام‌گذاری سازگار
- [ ] امنیت/RBAC پیکربندی شده

### قبل از `azd deploy`

- [ ] متغیرهای محیطی تنظیم شده‌اند
- [ ] مقادیر محرمانه در Key Vault
- [ ] رشته‌های اتصال بررسی شده‌اند
- [ ] بررسی‌های سلامت پیکربندی شده‌اند
