# فصل ۶: برنامه‌ریزی و اعتبارسنجی پیش از استقرار

**📚 دوره**: [AZD برای مبتدیان](../../README.md) | **⏱️ مدت زمان**: ۱ ساعت | **⭐ سطح دشواری**: متوسط

---

## مرور کلی

این فصل مراحل ضروری برنامه‌ریزی و اعتبارسنجی قبل از استقرار برنامه شما را پوشش می‌دهد. بیاموزید چگونه با برنامه‌ریزی ظرفیت مناسب، انتخاب SKU و بررسی‌های پیش‌پرواز از اشتباهات پرهزینه جلوگیری کنید.

> تایید شده با `azd 1.27.1` در ژوئیه ۲۰۲۶.

## اهداف یادگیری

با تکمیل این فصل، شما قادر خواهید بود:
- اجرای بررسی‌های پیش‌پرواز قبل از استقرار
- برنامه‌ریزی ظرفیت و تخمین نیازهای منابع
- انتخاب SKUهای مناسب برای بهینه‌سازی هزینه
- پیکربندی Application Insights برای نظارت
- درک الگوهای هماهنگی تیم

## 🚀 شروع سریع

```bash
# بررسی سهمیه‌های اشتراک
az vm list-usage --location eastus --output table

# پیش‌نمایش استقرار (هیچ منبعی ایجاد نمی‌شود)
azd provision --preview

# اعتبارسنجی نحو Bicep
az bicep build --file infra/main.bicep

# بررسی پیکربندی محیط
azd env get-values
```

---

## ☑️ چک‌لیست پیش از استقرار

### قبل از `azd provision`

- [ ] سهمیه منطقه تایید شده
- [ ] انتخاب SKUها به درستی انجام شده
- [ ] برآورد هزینه بررسی شده
- [ ] قواعد نام‌گذاری هماهنگ است
- [ ] امنیت / کنترل دسترسی تنظیم شده است

### قبل از `azd deploy`

- [ ] متغیرهای محیطی تنظیم شده‌اند
- [ ] اسرار در Key Vault قرار دارند
- [ ] رشته‌های اتصال تایید شده‌اند
- [ ] بررسی‌های سلامت پیکربندی شده
