# فصل ۶: برنامه‌ریزی و اعتبارسنجی پیش از استقرار

**📚 دوره**: [AZD For Beginners](../../README.md) | **⏱️ زمان**: ۱ ساعت | **⭐ پیچیدگی**: متوسط

---

## نمای کلی

این فصل مراحل ضروری برنامه‌ریزی و اعتبارسنجی را قبل از استقرار برنامه شما پوشش می‌دهد. یاد بگیرید چگونه با برنامه‌ریزی ظرفیت مناسب، انتخاب SKU و بررسی‌های پیش‌پرواز از اشتباهات پرهزینه جلوگیری کنید.

> اعتبارسنجی شده با `azd 1.25.6` در ژوئن ۲۰۲۶.

## اهداف یادگیری

با تکمیل این فصل، شما:
- چک‌های پیش‌پرواز را قبل از استقرار اجرا کنید
- ظرفیت را برنامه‌ریزی کرده و نیازهای منابع را برآورد کنید
- SKUها را برای بهینه‌سازی هزینه به‌درستی انتخاب کنید
- Application Insights را برای مانیتورینگ پیکربندی کنید
- الگوهای هماهنگی تیم را درک کنید

## 🚀 شروع سریع

```bash
# بررسی سهمیه‌های اشتراک
az vm list-usage --location eastus --output table

# پیش‌نمایش استقرار (هیچ منبعی ایجاد نمی‌شود)
azd provision --preview

# اعتبارسنجی نحو Bicep
az bicep build --file infra/main.bicep

# بررسی پیکربندی محیط
azd env get-values
```

---

## ☑️ چک‌لیست پیش از استقرار

### قبل از `azd provision`

- [ ] سهمیه برای منطقه تأیید شده است
- [ ] SKUها به‌صورت مناسب انتخاب شده‌اند
- [ ] برآورد هزینه بررسی شده است
- [ ] قواعد نامگذاری سازگار است
- [ ] امنیت/RBAC پیکربندی شده است

### قبل از `azd deploy`

- [ ] متغیرهای محیطی تنظیم شده‌اند
- [ ] رمزها در Key Vault قرار گرفته‌اند
- [ ] رشته‌های اتصال بررسی شده‌اند
- [ ] بررسی‌های سلامت پیکربندی شده‌اند
