# فصل 8: الگوهای تولید و سازمانی

**📚 دوره**: [AZD برای مبتدیان](../../README.md) | **⏱️ مدت**: 2-3 ساعت | **⭐ پیچیدگی**: پیشرفته

---

## نمای کلی

این فصل الگوهای استقرار آماده سازمان، سخت‌سازی امنیت، مانیتورینگ و بهینه‌سازی هزینه برای بارهای کاری هوش مصنوعی در محیط تولید را پوشش می‌دهد.

> تأیید شده در برابر `azd 1.23.12` در مارس 2026.

## اهداف یادگیری

با تکمیل این فصل، شما:
- استقرار برنامه‌های مقاوم چندمنطقه‌ای
- پیاده‌سازی الگوهای امنیتی سازمانی
- پیکربندی مانیتورینگ جامع
- بهینه‌سازی هزینه‌ها در مقیاس
- راه‌اندازی خط‌های CI/CD با AZD

## 🔐 بهترین شیوه‌های امنیتی

```bicep
// Use managed identity
identity: {
  type: 'SystemAssigned'
}

// Private endpoints for AI services
properties: {
  publicNetworkAccess: 'Disabled'
  networkAcls: {
    defaultAction: 'Deny'
  }
}
```

## 💰 بهینه‌سازی هزینه

| راهکار | صرفه‌جویی |
|----------|---------|
| مقیاس تا صفر (Container Apps) | 60-80% |
| استفاده از سطوح مصرفی برای توسعه | 50-70% |
| مقیاس‌بندی زمان‌بندی‌شده | 30-50% |
| ظرفیت رزرو شده | 20-40% |

```bash
# هشدارهای بودجه را تنظیم کنید
az consumption budget create \
  --budget-name "AI-Budget" \
  --amount 500 \
  --category Cost \
  --time-grain Monthly
```

## 📊 راه‌اندازی مانیتورینگ

```bash
# نمایش زندهٔ لاگ‌ها
azd monitor --logs

# بررسی Application Insights
azd monitor --overview

# مشاهدهٔ متریک‌ها
az monitor metrics list --resource <resource-id>
```
