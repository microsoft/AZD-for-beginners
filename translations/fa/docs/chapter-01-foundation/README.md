# فصل 1: مبانی و شروع سریع

**📚 دوره**: [AZD برای مبتدیان](../../README.md) | **⏱️ مدت**: 30-45 دقیقه | **⭐ پیچیدگی**: مبتدی

---

## نمای کلی

این فصل اصول Azure Developer CLI (azd) را معرفی می‌کند. شما با مفاهیم اصلی آشنا خواهید شد، ابزارها را نصب می‌کنید و اولین برنامه خود را به Azure مستقر می‌نمایید.

> تأیید شده با `azd 1.25.6` در ژوئن 2026.

## اهداف یادگیری

با تکمیل این فصل، شما قادر خواهید بود:
- درک کنید Azure Developer CLI چیست و چگونه با Azure CLI تفاوت دارد
- نصب و پیکربندی AZD روی پلتفرم خود
- استقرار اولین برنامه‌تان در Azure با `azd up`
- پاک‌سازی منابع با `azd down`

## 🚀 شروع سریع

```bash
# بررسی نصب
azd version

# برای AZD احراز هویت کنید
# اختیاری: az login اگر قصد دارید مستقیماً دستورات Azure CLI را اجرا کنید
azd auth login

# اولین برنامهٔ خود را مستقر کنید
azd init --template todo-nodejs-mongo
azd up

# پس از اتمام پاک‌سازی را انجام دهید
azd down --force --purge
```

## ✅ معیارهای موفقیت

بعد از تکمیل این فصل، شما باید قادر باشید:

```bash
azd version              # نسخه نصب‌شده را نشان می‌دهد
azd init --template todo-nodejs-mongo  # پروژه را مقداردهی اولیه می‌کند
azd up                   # در Azure مستقر می‌کند
azd show                 # آدرس برنامه در حال اجرا را نمایش می‌دهد
azd down --force --purge # منابع را پاک‌سازی می‌کند
```
