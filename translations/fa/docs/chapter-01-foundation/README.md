# فصل 1: مبانی و شروع سریع

**📚 دوره**: [AZD برای مبتدیان](../../README.md) | **⏱️ مدت**: 30-45 دقیقه | **⭐ درجه سختی**: مبتدی

---

## مرور کلی

این فصل مبانی Azure Developer CLI (`azd`) را معرفی می‌کند. شما مفاهیم اصلی را یاد می‌گیرید، ابزارها را نصب می‌کنید و اولین برنامه خود را در Azure مستقر خواهید کرد.

> تأیید شده با `azd 1.23.12` در مارس 2026.

## اهداف یادگیری

با تکمیل این فصل، شما قادر خواهید بود:
- درک کنید Azure Developer CLI چیست و چگونه با Azure CLI متفاوت است
- AZD را روی پلتفرم خود نصب و پیکربندی کنید
- اولین برنامه خود را با `azd up` در Azure مستقر کنید
- منابع را با `azd down` پاک کنید

## 🚀 شروع سریع

```bash
# نصب را بررسی کنید
azd version

# برای AZD احراز هویت کنید
# اختیاری: اگر قصد دارید دستورات Azure CLI را مستقیماً اجرا کنید، az login را اجرا کنید
azd auth login

# اولین برنامه خود را مستقر کنید
azd init --template todo-nodejs-mongo
azd up

# بعد از اتمام کار، پاک‌سازی کنید
azd down --force --purge
```

## ✅ معیارهای موفقیت

پس از تکمیل این فصل، شما باید بتوانید:

```bash
azd version              # نسخهٔ نصب‌شده را نشان می‌دهد
azd init --template todo-nodejs-mongo  # پروژه را مقداردهی اولیه می‌کند
azd up                   # در Azure مستقر می‌کند
azd show                 # نشانی (URL) برنامه در حال اجرا را نمایش می‌دهد
azd down --force --purge # منابع را پاک‌سازی می‌کند
```
