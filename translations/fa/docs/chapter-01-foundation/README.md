**📚 دوره**: [مقدماتی AZD](../../README.md) | **⏱️ مدت زمان**: ۳۰-۴۵ دقیقه | **⭐ درجه سختی**: مبتدی

---

## مرور کلی

این فصل اصول اولیه ابزار خط فرمان توسعه‌دهنده آژور (azd) را معرفی می‌کند. شما با مفاهیم پایه آشنا می‌شوید، ابزارها را نصب می‌کنید و اولین برنامه خود را در آژور مستقر خواهید کرد.

> اعتبارسنجی شده با `azd 1.27.1` در ژوئیه ۲۰۲۶.

## اهداف یادگیری

با تکمیل این فصل، شما قادر خواهید بود:
- درک کنید که Azure Developer CLI چیست و چگونه با Azure CLI تفاوت دارد
- نصب و پیکربندی AZD را در پلتفرم خود انجام دهید
- اولین برنامه خود را با دستور `azd up` در آژور مستقر کنید
- منابع را با دستور `azd down` پاک کنید

قبل از شروع، اطمینان حاصل کنید که دستگاه محلی شما برای قالب فصل ۱ آماده است:

**ویندوز:**
```powershell

```

**مک‌اواس / لینوکس:**
```bash

```

اگر اسکریپت ابزارهای مفقود را گزارش داد، ابتدا آن‌ها را برطرف کنید و سپس به فصل ادامه دهید.

## 🚀 شروع سریع

```bash
# بررسی نصب
azd version

# احراز هویت برای AZD
# اختیاری: az login اگر قصد دارید دستورات Azure CLI را مستقیماً اجرا کنید
azd auth login

# برنامه اول خود را مستقر کنید
azd init --template todo-nodejs-mongo
azd up

# پاکسازی پس از اتمام کار
azd down --force --purge
```

---

## ✅ معیارهای موفقیت

پس از اتمام این فصل، شما باید قادر باشید:

```bash
azd version              # نسخه نصب شده را نشان می‌دهد
azd init --template todo-nodejs-mongo  # پروژه را مقداردهی اولیه می‌کند
azd up                   # به Azure مستقر می‌کند
azd show                 # آدرس URL برنامه در حال اجرا را نشان می‌دهد
azd down --force --purge # منابع را پاک‌سازی می‌کند
```
