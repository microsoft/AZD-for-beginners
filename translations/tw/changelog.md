### [v3.6.0] - 2025-11-19

#### 重大更新：容器應用部署範例
**此版本引入了使用 Azure Developer CLI (AZD) 的全面且適用於生產環境的容器應用部署範例，並提供完整文件及整合至學習路徑。**

#### 新增
- **🚀 容器應用範例**：新增本地範例於 `examples/container-app/`：

  - [簡單 Flask API](../../examples/container-app/simple-flask-api)：適合初學者的 REST API，包含零負載縮放、健康檢查、監控及故障排除

- **測試與故障排除**：端到端測試場景、監控指令及故障排除指南

#### 更改

- **examples/README.md**：更新以突出容器應用範例，新增比較矩陣條目，並更新技術/架構參考
