# AI డెవలపర్‌ల కోసం AZD వర్క్‌షాప్  

> జూలై 2026లో `azd 1.27.1` మాపై ధృవీకరించినది.  

> [!IMPORTANT]  
> **ఈ వర్క్‌షాప్ లో మీరు స్థానిక బ్రౌజర్‌లో ముందుగా చూడగల పనివేళ గైడ్‌ను కలిగి ఉంటుంది. ప్రారంభించడానికి, GitHub Codespaces, డెవ్ కంటైనర్ లేదా MkDocs ఇన్‌స్టాల్ చేసిన స్థానిక క్లోన్‌లో రిపోను తెరచండి—ఆ తర్వాత సక్రియ VS కోడ్ టెర్మినల్ కనిపించేవరకు వేచి, టైపు చేయండి:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **మీరు బ్రౌజర్‌లో ప్రివ్యూ పేజీని తెరచేందుకు ఒక పాప్-అప్ డైలాగ్ కనబడాలి.**  

AI అప్లికేషన్ డిప్లాయ్‌మెంట్ పై దృష్టి పెట్టిన Azure Developer CLI (AZD) నేర్చుకోవడానికి సక్రియ వర్క్‌షాప్‌కు స్వాగతం. ఈ వర్క్‌షాప్ మీకు AZD టెంప్లేట్లను 3 దశల్లో అమల్లో అనుసంధానించడంలో సహాయపడుతుంది:  

1. **అన్వేషణ** - మీకు సరైన టెంప్లేట్‌ను కనుగొనండి.  
1. **డిప్లాయ్‌మెంట్** - డిప్లాయ్ చేసి అది పనిచేస్తుందో ధృవీకరించండి  
1. **అనుకూలీకరణ** - మీదుగానీ మార్చండి మరియు పునరావృతం చేయండి!  

ఈ వర్క్‌షాప్ ద్వారా మీరు కోర్ డెవలపర్ సాధనాలు మరియు వర్క్‌ఫ్లోలను కూడా పరిచయం పొందుతారు, తద్వారా మీ పూర్తి అభివృద్ధి ప్రయాణాన్ని సులభతరం చేయగలుగుతారు.

## వర్క్‌షాప్ సమీక్ష  

**కాలం:** 3-4 గంటలు  
**స్థాయి:** కొత్తవారు నుండి మధ్య స్థాయి వరకు  
**ముందస్తు అర్ధం కావలసినవి:** Azure, AI కాన్సెప్ట్‌లు, VS కోడ్ & క‌మాండ్-లైన్ సేవ‌నా అవగాహన.  

ఇది మీరు చేయడం ద్వారా నేర్చుకునే సక్రియ వర్క్‌షాప్. వ్యాయామాలు పూర్తి చేసిన తర్వాత, AZD For Beginners పాఠ్యాంశాలను సమీక్షించి, సెక్యూరిటీ మరియు ఉత్పాదకత ఉత్తమ పద్ధతులలో మీ అభ్యాసం కొనసాగించాలని మేము సిఫారసు చేస్తాము.  

| సమయం| మాడ్యూల్  | లక్ష్యం |  
|:---|:---|:---|  
| 15 నిమిషాలు | పరిచయం | వేదికను ఏర్పరచండి, లక్ష్యాలను అర్థం చేసుకోండి |  
| 30 నిమిషాలు | AI టెంప్లేట్ ఎంపిక | ఎంపికలను పరిశీలించి మొదలుపెట్టండి |  
| 30 నిమిషాలు | AI టెంప్లేట్ నిర్ధారణ | Azureకి డిఫాల్ట్ సొల్యూషన్‌ను డిప్లాయ్ చేయండి |  
| 30 నిమిషాలు | AI టెంప్లేట్ విభజన | నిర్మాణం మరియు కాన్ఫిగరేషన్‌ను పరిశీలించండి |  
| 30 నిమిషాలు | AI టెంప్లేట్ కాన్ఫిగర్ చేయండి | లభ్యమయ్యే ఫీచర్లను సక్రియం చేసి ప్రయత్నించండి |  
| 30 నిమిషాలు | AI టెంప్లేట్ అనుకూలీకరణ | మీ అవసరాలకు టెంప్లేట్‌ను అనుకూలీకరించండి |  
| 30 నిమిషాలు | ఇంఫ్రాస్ట్రక్చర్‌ను తొలగించండి | క్లీనప్ చేసి వనరులను విడుదల చేయండి |  
| 15 నిమిషాలు | సమారోపణ & తదుపరి దశలు | అధ్యయన వనరులు, వర్క్‌షాప్ సవాలు |  
| | |  

## మీరు నేర్చుకునేది  

AZD టెంప్లేట్‌ను Microsoft Foundryలో ఎండ్-టు-ఎండ్ అభివృద్ధికి వివిధ సామర్థ్యాలు మరియు సాధనాలను అన్వేషించేందుకు ఓ నేర్ప-learning sandboxగా ఆలోచించండి. ఈ వర్క్‌షాప్ చివరికి, మీరు ఈ పరిసరంలో వివిధ సాధనాలు మరియు కాన్సెప్ట్లకు సహజమైన అవగాహన కలిగి ఉండాలి.  

| కాన్సెప్ట్  | లక్ష్యం |  
|:---|:---|  
| **Azure Developer CLI** | సాధన ఆజ్ఞలు మరియు వర్క్‌ఫ్లోలను అర్థం చేసుకోండి |  
| **AZD టెంప్లేట్లు**| ప్రాజెక్ట్ నిర్మాణం మరియు కాన్ఫిగరేషన్ అర్థం చేసుకోండి |  
| **Azure AI ఏజెంట్**| Microsoft Foundry ప్రాజెక్టును ప్రావిజన్ చేసి డిప్లాయ్ చేయండి |  
| **Azure AI సెర్చ్**| ఏజెంట్లతో సందర్భ ఇంజనీరింగ్‌ను సక్రియం చేయండి |  
| **పర్యవేక్షణ**| ట్రేసింగ్, మానిటరింగ్ మరియు మూల్యాంకనాలను పరిశీలించండి |  
| **రెడ్ టీమింగ్**| ప్రత్యర్థి పరీక్షలు మరియు నివారణలను అన్వేషించండి |  
| | |
