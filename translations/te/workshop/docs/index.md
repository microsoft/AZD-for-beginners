# AI డెవలపర్ల కోసం AZD వర్క్షాప్

> మార్చ్ 2026లో `azd 1.23.12`తో ధృవీకరించబడింది.
>
> [!IMPORTANT]  
> **ఈ వర్క్షాప్ ఒక వర్క్షాప్ మార్గదర్శకంతో అమర్చబడింది, దీన్ని మీ స్థానిక బ్రౌజర్‌లో ప్రివ్యూ చేయవచ్చు. ప్రారంభించడానికి, రిపోను GitHub Codespaces, ఒక dev container, లేదా MkDocs ఇన్‌స్టాల్ చేసిన స్థానిక క్లోన్‌లో తెరవండి—తరువాత ఒక యాక్టివ్ VS Code టెర్మినల్ కనిపించే వరకు వేచి, టైప్ చేయండి:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **మీ బ్రౌజర్‌లో ప్రివ్యూ పేజీని తెరవడానికి ఒక పాప్-అప్ డైలాగ్ కనిపించాలి.**

ప్రత్యక్ష ఆచరణ ద్వారా Azure Developer CLI (AZD) నేర్చుకునే వర్క్షాప్‌కు స్వాగతం — ఇది AI అప్లికేషన్ డిప్లాయ్‌మెంట్‌పై ప్రత్యేక దృష్టి పెట్టి ఉంటుంది. ఈ వర్క్షాప్ మూడు దశల్లో AZD టెంప్లేట్లపై ఆచరణాత్మక అవగాహన పొందడంలో మీకు సహాయపడుతుంది:

1. **అన్వేషణ** - మీకు సరైన టెంప్లేటును కనుగొనండి.
1. **డిప్లాయ్‌మెంట్** - డిప్లాయ్ చేసి అది పనిచేస్తుందో ధృవీకరించండి
1. **అనుకూలీకరణ** - దాన్ని మార్చి, మీదిగా చేయడానికి పునరావృతం చేయండి!

ఈ వర్క్షాప్ నిడివిలో, ముఖ్యమైన డెవలపర్ టూల్స్ మరియు వర్క్‌ఫ్లోలతో పరిచయం కూడా పొందుతారు, తద్వారా మీ ఎండ్-టు-ఎండ్ డెవలప్‌మెంట్ ప్రయాణాన్ని సులభతరం చేసుకోవచ్చు.

## వర్క్షాప్ అవలోకనం

**అవధి:** 3-4 గంటలు  
**స్థాయి:** ప్రారంభిక నుండి మధ్యస్థాయి  
**మునుపటి పరిజ్ఞానం:** Azure, AI కాన్సెప్ట్‌లు, VS Code & కమాండ్-లైన్ టూల్స్ తో పరిచయం.

ఇది ప్రాక్టికల్ వర్క్షాప్ — మీరు చేయడం ద్వారా నేర్చుకుంటారు. మీరు వ్యాయామాలు పూర్తి చేసిన తర్వాత, Security మరియు Productivity బెస్ట్‌ప్రాక్టీస్‌లలో మీ అభ్యాసాన్ని కొనసాగించడానికి AZD For Beginners పాఠ్యక్రమాన్ని సమీక్షించమని సూచిస్తాము.

| సమయం| మాడ్యూల్  | లక్ష్యం |
|:---|:---|:---|
| 15 నిమిషాలు | పరిచయం | స్థితి సిద్ధం చేయండి, లక్ష్యాలను అర్థం చేసుకోండి |
| 30 నిమిషాలు | AI టెంప్లేట్ ఎంచుకోండి | ఎంపికలను పరిశీలించి ప్రారంభాన్ని ఎంచుకోండి | 
| 30 నిమిషాలు | AI టెంప్లేట్ ధృవీకరణ | డిఫాల్ట్ పరిష్కారాన్ని Azure పై డిప్లాయ్ చేయండి |
| 30 నిమిషాలు | AI టెంప్లేట్ విశ్లేషణ | నిర్మాణం మరియు కాన్ఫిగరేషన్‌ను అన్వేషించండి |
| 30 నిమిషాలు | AI టెంప్లేట్ కాన్ఫిగర్ చేయండి | అందుబాటులో ఉన్న ఫీచర్‌లను యాక్టివేట్ చేసి ప్రయత్నించండి |
| 30 నిమిషాలు | AI టెంప్లేట్ అనుకూలీకరణ | టెంప్లేట్‌ను మీ అవసరాలకు అనుగుణంగా మార్చండి |
| 30 నిమిషాలు | ఇన్ఫ్రాస్ట్రక్చర్ తొలగింపు | వనరులను శుభ్రపరిచి విడుదల చేయండి |
| 15 నిమిషాలు | సారాంశం & తదుపరి దశలు | శిక్షణా వనరులు, వర్క్షాప్ సవాల్ |
| | |

## మీరు ఏమి నేర్చుకుంటారు

AZD టెంప్లేట్‌ను ఒక లెర్నింగ్ శాండ్‌బాక్స్‌గా ఊహించుకోండి, ఇది Microsoft Foundry మీద ఎండ్-టు-ఎండ్ డెవలప్‌మెంట్ కోసం వివిధ సామర్థ్యాలు మరియు టూల్స్‌ను అన్వేషించడానికి సహాయపడుతుంది. ఈ వర్క్షాప్ ముగింప జగానికి, మీరు ఈ సందర్భంలో వివిధ టూల్స్ మరియు కాన్సెప్ట్‌లపై సూటిగా అవగాహన పొందాలి.

| కాన్సెప్ట్  | లక్ష్యం |
|:---|:---|
| **Azure Developer CLI** | టూల్ ఆజ్ఞలు మరియు వర్క్‌ఫ్లోలను అర్థం చేసుకోండి|
| **AZD Templates**| ప్రాజెక్ట్ నిర్మాణం మరియు కాన్ఫిగ్‌ను అర్థం చేసుకోండి|
| **Azure AI Agent**| Microsoft Foundry ప్రాజెక్టును ప్రొవిజన్ చేసి డిప్లాయ్ చేయండి  |
| **Azure AI Search**| ఏజెంట్స్‌తో కాంటెక్స్ట్ ఇంజనీరింగ్‌ను సాధ్యముచేయండి |
| **Observability**| ట్రేసింగ్, మానిటరింగ్ మరియు మూల్యాంకనాలను అన్వేషించండి |
| **Red Teaming**| ప్రత్యర్థి పరీక్షలు మరియు ఉపశమనాలపై పరిశోధన చేయండి |
| | |
