# AI డెవలపర్ల కోసం AZD వర్క్‌షాప్

> 2026 జూన్‌లో `azd 1.25.6` తో ధృవీకరించబడింది.

> [!IMPORTANT]  
> **ఈ వర్క్షాప్‌లో లోకల్ బ్రౌజర్‌లో ప్రివ్యూ చేయగల వర్క్‌షాప్ గైడ్ అమర్చబడింది. ప్రారంభించడానికి, repoని GitHub Codespacesలో, dev containerలో, లేదా MkDocs ఇన్‌స్టాల్ చేసిన స్థానిక క్లోన్‌లో తెరవండి—తరవాత ఒక సక్రియమైన VS Code టెర్మినల్ కనిపించే వరకు వేచి, టైప్ చేయండి:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **మీకు బ్రౌజర్‌లో ప్రివ్యూ పేజీని తెరవడానికి ఒక పోప్-అప్ డైలాగ్ కనిపిస్తుంది.**

Azure Developer CLI (AZD) ను AI అప్లికేషన్ డిప్లాయ్‌మెంట్ పై దృష్టి పెట్టి నేర్చుకోవడానికి జరిగే హ్యాండ్స్-ఆన్ వర్క్షాప్‌కు స్వాగతం. ఈ వర్క్షాప్ మీకు AZD టెంప్లేట్స్‌ను 3 దశల్లో ఆచరణాత్మకంగా అర్థం చేసుకోవడానికి సహాయం చేస్తుంది:

1. **Discovery** - మీకు సరిపోయే టెంప్లేట్‌ను కనుగొనండి.
1. **Deployment** - డిప్లాయ్ చేసి అది పనిచేస్తుందో ధృవీకరించండి
1. **Customization** - మీ అవసరాలకు అనుగుణంగా సవరించి పునరావృతం చేయండి!

ఈ వర్క్షాప్ సమయంలో, మీ ఎండ్-టు-ఎండ్ డెవలప్‌మెంట్ ప్రయాణాన్ని సులభతరం చేయడానికి ప్రాథమిక డెవలపర్ టూల్స్ మరియు వర్క్‌ఫ్లోలకు పరిచయమివ్వబడుతుంది.

## వర్క్షాప్ అవలోకనం

**Duration:** 3-4 గంటలు  
**Level:** ప్రారంభిక నుండి మధ్యంతర  
**Prerequisites:** Azure, AI కాన్సెప్ట్స్, VS Code & కమాండ్-లైన్ టూల్స్‌పై పరిచయం.

ఇది చర్య ద్వారా నేర్చుకునే హ్యాండ్స్-ఆన్ వర్క్షాప్. ఒకసారి మీరు వ్యాయామాలు పూర్తిచేసిన తర్వాత, భద్రత మరియు ఉత్పాదకత ఉత్తమ విధానాలపై మీ విద్యా ప్రయాణాన్ని కొనసాగించడానికి AZD For Beginners పాఠ్యక్రమాన్ని సమీక్షించాలని మేము సిఫార్సు చేస్తాము.

| సమయం| మాడ్యూల్  | లక్ష్యం |
|:---|:---|:---|
| 15 నిమిషాలు | పరిచయం | దిశని సెట్ చేయండి, లక్ష్యాలను అర్థం చేసుకోండి |
| 30 నిమిషాలు | Select AI Template | ఎంపికలను పరిశీలించి ప్రారంభకం ఎంచుకోండి | 
| 30 నిమిషాలు | Validate AI Template | డిఫాల్ట్ సొల్యూషన్‌ను Azureలో డిప్లాయ్ చేయండి |
| 30 నిమిషాలు | Deconstruct AI Template | నిర్మాణం మరియు కాన్ఫిగరేషన్‌ను పరిశీలించండి |
| 30 నిమిషాలు | Configure AI Template | ఉపలబ్ధి ఫీచర్లను యాక్టివేట్ చేసి ప్రయోగించండి |
| 30 నిమిషాలు | Customize AI Template | టెంప్లేట్‌ని మీ అవసరాలకు అనుగుణంగా మార్చండి |
| 30 నిమిషాలు | Teardown Infrastructure | రీసోర్సులను శుభ్రపరచి విడుదల చేయండి |
| 15 నిమిషాలు | Wrap-Up & Next Steps | నేర్చుకునే వనరులు, వర్క్షాప్ సవాల్ |
| | |

## మీరు ఏమి తెలుసుకుంటారు

AZD టెంప్లేట్‌ను Microsoft Foundryపై ఎండ్-టు-ఎండ్ డెవలప్‌మెంట్‌కు వివిధ సామర్థ్యాలు మరియు ఉపకరణాలను అన్వేషించేందుకు ఒక లెర్నింగ్ శాండ్‌బాక్స్‌గా భావించండి. ఈ వర్క్షాప్ ముగింపులో, మీరు ఈ సందర్భంలో వివిధ టూల్స్ మరియు కాన్సెప్ట్‌లపై ఒక భావగమనాన్ని పొందగలుగుతారు.

| కాన్సెప్ట్  | లక్ష్యం |
|:---|:---|
| **Azure Developer CLI** | టూల్ కమాండ్లు మరియు వర్క్‌ఫ్లోలను అర్థం చేసుకోండి|
| **AZD Templates**| ప్రాజెక్ట్ నిర్మాణం మరియు కాన్ఫిగ్‌ను అర్థం చేసుకోండి|
| **Azure AI Agent**| Microsoft Foundry ప్రాజెక్ట్‌ను ప్రావిజన్ & డిప్లాయ్ చేయడం  |
| **Azure AI Search**| ఏజెంట్లతో కంటెక్స్ట్ ఇంజినీరింగ్‌ను యాక్టివేట్ చేయండి |
| **Observability**| ట్రేసింగ్, మానిటరింగ్ మరియు అంచనాలను పరిశీలించండి |
| **Red Teaming**| విఘాటక పరీక్షలు మరియు ఉపశమనాలను అన్వేషించండి |
| | |
