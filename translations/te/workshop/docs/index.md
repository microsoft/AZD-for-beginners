# AI డెవలపర్ల కోసం AZD వర్క్‌షాప్

> [!IMPORTANT]  
> **ఈ వర్క్‌షాప్‌లో మీరు స్థానిక బ్రౌజర్‌లో ప్రివ్యూ చేయగల ఒక వర్క్‌షాప్ గైడ్ అమర్చబడింది. ప్రారంభించడానికి, మీరు రిపోలో GitHub Codespaces‌ను ప్రారంభించి—తరువాత సక్రియమైన VS Code టెర్మినల్ కనిపించే వరకు వేచి, టైప్ చేయండి:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **బ్రౌజర్‌లో ప్రివ్యూ పేజీని తెరవడానికి ఒక పాప్-అప్ డైలాగ్ కనిపిస్తుంది.**

AI అప్లికేషన్ డిప్లాయ్‌మెంట్‌పై దృష్టి పెట్టి Azure Developer CLI (AZD) నేర్చుకోవడానికి ప్రాక్టికల్ వర్క్‌షాప్‌కు స్వాగతం. ఈ వర్క్‌షాప్ మీకు AZD టెంప్లేట్లపై అనువర్తనాత్మక అవగాహనను 3 దశల్లో ఇస్తుంది:

1. **అన్వేషణ (Discovery)** - మీకు సరైన టెంప్లేట్‌ను కనుగొనండి.
1. **డిప్లాయ్‌మెంట్ (Deployment)** - డిప్లాయ్ చేసి అది పనిచేస్తుందో ధృవీకరించండి
1. **సర్దుబాటు (Customization)** - సవరించి, పునరావృతం చేసి మీదేనిగా మార్చుకోండి!

ఈ వర్క్‌షాప్ సమయంలో మీకు కోర్ డెవలపర్ టూల్స్ మరియు వర్క్‌ఫ్లోలు పరిచయం చేయబడతాయి, ఇవి మీ ఎండ్-టు-ఎండ్ అభివృద్ధి ప్రయాణాన్ని సులభతరం చేయడంలో సహాయపడతాయి.

## వర్క్‌షాప్ అవలోకనం

**కాలవ్యవధి:** 3-4 గంటలు  
**స్థాయి:** ప్రారంభం నుండి మధ్యస్థాయి  
**పూర్వాపరాలు:** Azure, AI సిద్ధాంతాలు, VS Code మరియు కమాండ్-లైన్ టూల్స్‌తో పరిచయం.

ఇది ఆచరణ ద్వారా నేర్చుకునే వర్క్‌షాప్. మీరు వ్యాయామాలు పూర్తి చేసిన తర్వాత, భద్రత మరియు ఉత్పాదకత ఉత్తమ పద్ధతులలో మీ అభ్యాసాన్ని కొనసాగించడానికి AZD ప్రారంభకులకు పాఠ్య కార్యక్రమాన్ని సమీక్షించమని మేము సిఫార్సు చేస్తాము.

| Time| Module  | Objective |
|:---|:---|:---|
| 15 mins | పరిచయం | స్థితిని సెట్ చేయండి, లక్ష్యాలను అర్థం చేసుకోండి |
| 30 mins | AI టెంప్లేట్ ఎంచుకోండి | ఎంపికలను అన్వేషించి స్టార్టర్‌ను ఎంచుకోండి | 
| 30 mins | AI టెంప్లేట్ నిర్ధారించండి | డిఫాల్ట్ సొల్యూషన్‌ను Azureకి డిప్లాయ్ చేయండి |
| 30 mins | AI టెంప్లేట్ విభజించండి | స్ట్రక్చర్ మరియు కాన్ఫిగరేషన్‌ను అన్వేషించండి |
| 30 mins | AI టెంప్లేట్ కాన్ఫిగర్ చేయండి | అందుబాటులో ఉన్న ఫీచర్స్‌ని శక్తి లో నిమగ్నం చేయండి మరియు ప్రయత్నించండి |
| 30 mins | AI టెంప్లేట్ సర్దుబాటు చేయండి | టెంప్లేట్‌ను మీ అవసరాలకు అనుగుణంగా మార్చండి |
| 30 mins | ఇన్ఫ్రాస్ట్రక్చర్‌ను తొలగించండి | క్లీన్అప్ చేసి వనరులను విడుదల చేయండి |
| 15 mins | Wrap-Up & Next Steps | లెర్నింగ్ రిసోర్సులు, వర్క్‌షాప్ సవాలు |
| | |

## మీరు నేర్చుకునే విషయాలు

AZD టెంప్లేట్‌ను Microsoft Foundry మీద ఎండ్-టు-ఎండ్ అభివృద్ధికి వివిధ సామర్థ్యాలు మరియు టూల్‌లు అన్వేషించడానికి ఒక లెర్నింగ్ సాండ్బాక్స్‌గా భావించండి. ఈ వర్క్‌షాప్ అమలు అయిన తర్వాత, మీరు ఈ సందర్భంలో వివిధ టూల్‌లు మరియు కాన్సెప్ట్‌లపై ఒక intuitive భావాన్ని పొందగలుగుతారు.

| Concept  | Objective |
|:---|:---|
| **Azure Developer CLI** | టూల్ కమాండ్లు మరియు వర్క్‌ఫ్లోలను అర్థం చేసుకోండి|
| **AZD టెంప్లేట్లు**| ప్రాజెక్ట్ నిర్మాణం మరియు కాన్ఫిగరేషన్‌ను అర్థం చేసుకోండి|
| **Azure AI ఏజెంట్**| Microsoft Foundry ప్రాజెక్ట్‌ను ప్రావిజన్ చేసి డిప్లాయ్ చేయండి  |
| **Azure AI Search**| ఏజెంట్లతో సందర్భంగా ఇంజనీరింగ్‌ను యాక్టీవేట్ చేయండి |
| **నిరీక్షణ (Observability)**| ట్రేసింగ్, మానిటరింగ్ మరియు మూల్యాంకనాలను అన్వేషించండి |
| **రెడ్ టీమింగ్**| ప్రత్యర్థి పరీక్షలు మరియు నివారణలు గురించి అన్వేషించండి |
| | |
