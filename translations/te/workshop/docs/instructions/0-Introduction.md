# AI డెవలపర్ల కోసం AZD

!!! info "ఈ వర్క్‌షాప్ కోసం ప్రేక్షకులు"
   
    ఈ వర్క్‌షాప్ Azure AI Foundry పై AI Agentic అప్లికేషన్లను నిర్మించడానికి ఉత్తమమైన టూల్స్ మరియు వర్క్‌ఫ్లోలను ఉపయోగించాలనుకునే వారందరికీ. Azure, AI కాన్సెప్ట్‌లు మరియు Python కోడ్‌తో కొంత పరిచయం ఉన్న మొదటిసారి మరియు అనుభవజ్ఞులైన డెవలపర్లకు ఈ వర్క్‌షాప్ అనుకూలంగా ఉంటుంది. ప్రారంభించడానికి ముందు ప్రీ-రిక్వైర్‌మెంట్స్‌ను తనిఖీ చేయండి.

    **వ్యవధి:** 3-4 గంటలు  
    **స్థాయి:** ప్రారంభం నుండి మధ్యస్థం  

---

## ప్రీ-రిక్వైర్‌మెంట్స్

- వ్యక్తిగత GitHub ఖాతా → [ఇక్కడ ఉచితంగా సృష్టించండి](https://github.com/signup)
- యాక్టివ్ Azure ఖాతా → [ఇక్కడ ఉచితంగా సృష్టించండి](https://aka.ms/free)
- AI కాన్సెప్ట్‌లు, Python డెవలప్‌మెంట్, కమాండ్-లైన్ టూల్స్‌తో పరిచయం
- VS Code మరియు GitHub టూల్స్ మరియు వర్క్‌ఫ్లోలతో అనుభవం.

---

## అభ్యాస లక్ష్యాలు

ఈ వర్క్‌షాప్ ముగిసే సమయానికి, మీరు ఈ క్రింది వాటిని నేర్చుకుంటారు:

- [X] **Azure Developer CLI తో పని చేయడం** → మీ Azure వనరులను ప్రొవిజన్, డిప్లాయ్ మరియు నిర్వహించడానికి.
- [X] **AI App Templates ఉపయోగించడం** → Azure AI Foundry పై AI Agent ప్రాజెక్టులను ప్రారంభించడానికి.
- [X] **టెంప్లేట్లను కాన్ఫిగర్ చేయడం** → Red Teaming, Evaluations, Tracing & Monitoring వంటి ఫీచర్ల కోసం.
- [X] **AZD టెంప్లేట్లు మరియు వర్క్‌ఫ్లోలను డీకన్‌స్ట్రక్ట్ మరియు ఎక్స్‌ప్లోర్ చేయడం** → GitHub Copilot ఉపయోగించి.
- [X] **AI Agent స్టార్టర్ టెంప్లేట్‌ను కస్టమైజ్ చేసి మళ్లీ డిప్లాయ్ చేయడం** → మీ కస్టమ్ అవసరాలకు అనుగుణంగా.

## వర్క్‌షాప్ అవుట్‌లైన్

ఈ వర్క్‌షాప్ మాడ్యూల్స్‌గా నిర్మించబడింది, ప్రతి మాడ్యూల్ ఒక హ్యాండ్స్-ఆన్ ల్యాబ్ వ్యాయామంతో అనుసంధానించబడింది:

1. **పరిచయం** - AZD టెంప్లేట్లు ఏమిటి మరియు అవి ఎలా పనిచేస్తాయి అనే అవగాహన పొందండి
1. **ఎంపిక** - మీ తదుపరి AI ప్రాజెక్ట్ కోసం సరైన "స్టార్టర్" టెంప్లేట్‌ను కనుగొనండి
1. **ధృవీకరణ** - టెంప్లేట్‌ను డిప్లాయ్ చేసి, అది ప్రకటన చేసిన విధంగా పనిచేస్తుందా అని అన్వేషించండి
1. **డీకన్‌స్ట్రక్షన్** - టెంప్లేట్‌ను విడదీయడం ద్వారా దాని భాగాలు మరియు కాన్ఫిగరేషన్‌ను అర్థం చేసుకోండి
1. **కాన్ఫిగరేషన్** - వివిధ Azure AI Foundry ఫీచర్లను యాక్టివేట్ చేసి ఉపయోగించడం నేర్చుకోండి
1. **కస్టమైజేషన్** - మీ డేటా మరియు అవసరాలను దృష్టిలో ఉంచుకుని అప్లికేషన్‌ను మళ్లీ నిర్మించండి
1. **టియర్‌డౌన్** - ఇన్‌ఫ్రాస్ట్రక్చర్‌ను డీప్రొవిజన్ చేయడం మరియు సాఫ్ట్-డిలీటెడ్ వనరులను రికవర్ చేయడం నేర్చుకోండి

## ఫీడ్‌బ్యాక్ & సమస్యలు

మేము ఈ కోర్సును ప్రారంభ డెవలపర్లకు ఉత్తమమైన అభ్యాస అనుభవాన్ని అందించడానికి నిరంతరం నవీకరిస్తున్నాము. మీ ఫీడ్‌బ్యాక్‌ను మేము స్వాగతిస్తున్నాము.

1. ఈ రిపోపై సమస్యను పోస్ట్ చేయండి - దానిని `Workshop` ట్యాగ్ చేయండి.
1. Azure AI Foundry Discordలో చేరండి - మీ సహ-అభ్యాసులతో కనెక్ట్ అవ్వండి!
