# AZD AI డెవలపర్ల కోసం

!!! info "ఈ వర్క్‌షాప్ కోసం ప్రేక్షకులు"
   
    ఈ వర్క్‌షాప్ ఉత్తమ తరగతి టూల్స్ మరియు వర్క్‌ఫ్లోలను ఉపయోగించి Microsoft Foundryలో AI ఏజెంటిక్ అప్లికేషన్లు నిర్మించాలనుకునే ఎవరికైనా ఉద్దేశించబడింది. ఈ వర్క్‌షాప్ Azure, AI భావనలు మరియు Python కోడ్ గురించి కొంత పరిచయం ఉన్న మొదటిదారులు మరియు అనుభవజ్ఞులైన డెవలపర్లకు సరిపోతుంది. ప్రారంభించే ముందు దయచేసి ముందస్తు అర్హతలను పరిశీలించండి.

    **కాలవ్యవధి:** 3-4 గంటలు  
    **స్థాయి:** ప్రారంభం నుండి మధ్యస్థాయి  

---


## ముందస్తు అర్హతలు

- ఒక వ్యక్తిగత GitHub ఖాతా → [ఉచితంగా ఒకటి సృష్టించండి](https://github.com/signup)
- ఒక సక్రియ Azure ఖాతా → [ఉచితంగా ఒకటి సృష్టించండి](https://aka.ms/free)
- AI భావనలు, Python అభివృద్ధి మరియు కమాండ్-లైన్ టూల్స్‌తో పరిచయం
- VS Code మరియు GitHub టూల్స్ మరియు వర్క్‌ఫ్లోస్‌లో అనుభవం.

---

## నేర్చుకునే లక్ష్యాలు

ఈ వర్క్‌షాప్ చివరిలో, మీరు క్రింది వాటి గురించి తెలుసుకుంటారు:

- [X] **Work with Azure Developer CLI** → మీ Azure వనరులను ఏర్పాటు, డిప్లాయ్ మరియు నిర్వహించడానికి
- [X] **Use AI App Templates** → Microsoft Foundryపై AI ఏజెంట్ ప్రాజెక్టులను ప్రారంభించడానికి
- [X] **Configure Templates** → Red Teaming, Evaluations, Tracing & Monitoring వంటి ఫీచర్ల కోసం కాన్ఫిగర్ చేయడం
- [X] **Deconstruct and Explore** → GitHub Copilot ఉపయోగించి AZD టెంప్లేట్లు మరియు వర్క్‌ఫ్లోలను విభజించి పరిశీలించడం
- [X] **Customize and Redeploy** → మీ ప్రత్యేక అవసరాలతో AI Agent స్టార్టర్ టెంప్లేట్‌ను అనుకూలీకరించి మళ్లీ డిప్లాయ్ చేయడం

## వర్క్‌షాప్ అవుట్‌లైన్

వర్క్‌షాప్ మాడ్యూల్స్‌గా నిర్మించబడి, ప్రతి మాడ్యూల్ ఒక ప్రాక్టికల్ ల్యాబ్ వ్యాయామంతో మద్దతుపడుతుంది:

1. **పరిచయం** - AZD టెంప్లేట్లు ఏమిటి మరియు అవి ఎలా పని చేస్తాయో అర్ధం చేసుకోండి
1. **ఎంచిక** - మీ తదుపరి AI ప్రాజెక్ట్ కోసం సరైన "స్టార్టర్" టెంప్లేట్‌ను కనుగొనండి
1. **సత్యాపన** - టెంప్లేట్‌ను డిప్లాయ్ చేసి అది ప్రకటించినట్లుగా పనిచేస్తుందో పరిశీలించండి
1. **విభజన** - టెంప్లేట్‌ను విడగొట్టి దాని భాగాలు మరియు కాన్ఫిగరేషన్‌ను అర్థం చేసుకోండి
1. **కాన్ఫిగరేషన్** - వివిధ Microsoft Foundry ఫీచర్లను సక్రియం చేయడం మరియు ఉపయోగించడం నేర్చుకోండి
1. **అనుకూలీకరణ** - మీ డేటా మరియు అవసరాలను పరిగణలోకి తీసుకుని అప్లికేషన్‌ను తిరిగి నిర్మించండి
1. **Teardown** - ఇన్‌ఫ్రాస్ట్రక్చర్‌ను డిప్రోవిజన్ చేయడం మరియు సాఫ్ట్-డిలీట్ అయిన వనరులను పునరుద్ధరించడం నేర్చుకోండి
