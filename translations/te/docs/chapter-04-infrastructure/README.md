# అధ్యాయం 4: ఇన్‌ఫ్రాస్ట్రక్చర్ ఎనిమి కోడ్ & పంపిణీ

**📚 కోర్సు**: [AZD మొదటిదరకు](../../README.md) | **⏱️ వ్యవధి**: 1-1.5 గంటలు | **⭐ కష్టతనం**: మధ్యస్థాయి

---

## అవగాహన

ఈ అధ్యాయం ఇన్‌ఫ్రాస్ట్రక్చర్ ఎనిమి కోడ్ (IaC) నమూనాలను Bicep టెంప్లేట్లు, వనరు ఆవృద్ధి, మరియు Azure Developer CLI ఉపయోగించి పంపిణీ విధానాలతో కవర్ చేస్తుంది.

> `azd 1.27.1` పై 2026 జూలైలో ధృవీకరించబడింది.

## నేర్చుకునే లక్ష్యాలు

ఈ అధ్యాయం పూర్తిచేసి మీరు:
- Bicep టెంప్లేట్ నిర్మాణం మరియు సింటాక్స్‌ను అర్థం చేసుకోవచ్చు
- `azd provision` తో Azure వనరులను సృష్టించండి
- `azd deploy` తో అప్లికేషన్‌లు పంపిణీ చేయండి
- బ్లూ-గ్రీన్ మరియు రోలింగ్ పంపిణీ విధానాలను అమలు చేయండి

## 🚀 త్వరిత ప్రారంభం

```bash
# టెంప్లేట్ నుండి ప్రారంభించండి
azd init --template azure-functions-python-v2-http

# ఏమి సృష్టించబడుతుందో ముందస్తుగా చూడండి
azd provision --preview

# కేవలం మౌలిక సదుపాయాలను ఏర్పాటుచేయండి
azd provision

# కేవలం కోడ్‌ను డిప్లాయ్ చేయండి
azd deploy

# లేదా రెండింటిని కూడా కలిసి చేయండి
azd up
```

## 🔧 అవసరమైన ఆజ్ఞలు

| ఆజ్ఞ | వివరణ |
|---------|-------------|
| `azd init` | ప్రాజెక్టును ప్రారంభించు |
| `azd provision` | Azure వనరులను సృష్టించు |
| `azd deploy` | అప్లికేషన్ కోడ్‌ను పంపిణీ చేయండి |
| `azd up` | వనరు ఏర్పాట్లు + పంపిణీ |
| `azd down` | అన్ని వనరులను తొలగించు |
