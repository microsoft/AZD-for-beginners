# అధ్యాయం 4: ఇన్ఫ్రాస్ట్రక్చర్‌ను కోడ్‌గా మరియు డిప్లాయ్‌మెంట్

## అవలోకనం

ఈ అధ్యాయం Bicep టెంప్లెట్లతో ఇన్ఫ్రాస్ట్రక్చర్‌ను కోడ్‌గా (IaC) నమూనాలు, వనరుల సంస్థాపన మరియు Azure Developer CLI ఉపయోగించి డిప్లాయ్‌మెంట్ వ్యూహాలను కలవర్ చేస్తుంది.

## అభ్యాస లక్ష్యాలు

By completing this chapter, you will:
- Bicep టెంప్లేట్ నిర్మాణం మరియు సింటాక్స్‌ను అర్థం చేసుకోవడం
- Azure వనరులను `azd provision` ఉపయోగించి సంస్థాపించడం
- అప్లికేషన్‌లను `azd deploy` ఉపయోగించి డిప్లాయ్ చేయడం
- బ్లూ-గ్రీన్ మరియు రోలింగ్ డిప్లాయ్‌మెంట్ వ్యూహాలు అమలు చేయడం

## 🚀 శీఘ్ర ప్రారంభం

```bash
# టెంప్లేట్ నుండి ప్రారంభించండి
azd init --template azure-functions-python-v2-http

# ఏం సృష్టించబడబోతుందో ముందుగా చూడండి
azd provision --preview

# కేవలం మౌలిక సదుపాయాలను ఏర్పాటు చేయండి
azd provision

# కేవలం కోడ్‌ను డిప్లాయ్ చేయండి
azd deploy

# లేదా రెండింటినీ ఒకేసారి చేయండి
azd up
```

## 🔧 అవసరమైన కమాండ్లు

| కమాండ్ | వివరణ |
|---------|-------------|
| `azd init` | ప్రాజెక్ట్‌ను ప్రారంభించండి |
| `azd provision` | Azure వనరులను సృష్టించండి |
| `azd deploy` | అప్లికేషన్ కో드를 డిప్లాయ్ చేయండి |
| `azd up` | సంస్థాపన + డిప్లాయ్ |
| `azd down` | అన్ని వనరులను తొలగించండి |
