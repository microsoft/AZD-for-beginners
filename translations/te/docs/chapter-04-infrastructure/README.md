# చాప్టర్ 4: ఇన్‌ఫ్రాస్ట్రక్చర్ ఏజ్ కోడ్ & డిప్లాయ్‌మెంట్

**📚 కోర్సు**: [AZD ప్రారంభికులకు](../../README.md) | **⏱️ కాలావధి**: 1-1.5 గంటలు | **⭐ సంకీర్ణత**: మధ్యస్థ

---

## అవలోకనం

ఈ చాప్టర్ Bicep టెంప్లేట్స్‌తో Infrastructure as Code (IaC) నమూనాలు, వనరుల ప్రావిజనింగ్, మరియు Azure Developer CLI ఉపయోగించి డిప్లాయ్‌మెంట్ వ్యూహాలను కవర్ చేస్తుంది.

> చెలామణీకి పరీక్షించబడింది `azd 1.25.6` (జూన్ 2026).

## లెర్నింగ్ లక్ష్యాలు

ఈ చాప్టర్ పూర్తి చేసిన తర్వాత మీరు:
- Bicep టెంప్లేట్ నిర్మాణం మరియు సింటాక్స్‌ను అర్థం చేసుకోవడం
- `azd provision` తో Azure వనరులను ప్రావిజన్ చేయడం
- `azd deploy` తో అప్లికేషన్లను డిప్లాయ్ చేయడం
- బ్లూ-గ్రీన్ మరియు రోలింగ్ డిప్లాయ్‌మెంట్ వ్యూహాలను అమలు చేయడం

## 🚀 త్వరిత ప్రారంభం

```bash
# టెంప్లేట్ నుండి ప్రారంభించండి
azd init --template azure-functions-python-v2-http

# ఏది సృష్టించబడబోతుందో ముందుగా చూడండి
azd provision --preview

# కేవలం మౌలిక సదుపాయాలను మాత్రమే ఏర్పాటు చేయండి
azd provision

# కేవలం కోడ్‌ను అమలు చేయండి
azd deploy

# లేదా రెండింటినీ ఒకేసారి చేయండి
azd up
```

## 🔧 అత్యావశ్యక కమాండ్లు

| Command | వివరణ |
|---------|-------------|
| `azd init` | ప్రాజెక్ట్ ప్రారంభించు |
| `azd provision` | Azure వనరులను సృష్టించు |
| `azd deploy` | అప్లికేషన్ కోడ్‌ను డిప్లాయ్ చేయు |
| `azd up` | provision + deploy |
| `azd down` | అన్ని వనరులను తొలగించు |
