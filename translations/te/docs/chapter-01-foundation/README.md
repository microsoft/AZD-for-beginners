## అవలోకనం

ఈ అధ్యాయం Azure Developer CLI (azd) యొక్క మూలాధారాలను పరిచయం చేస్తుంది. మీరు ప్రధాన సూత్రాలు తెలుసుకుంటారు, టూల్‌లను ఇన్స్టాల్ చేస్తారు, మరియు మీ మొదటి అప్లికేషన్‌ను Azure కు despleoy చేయడం నేర్చుకుంటారు.

> `azd 1.25.6` తో జూన్ 2026 లో సరిచూడబడింది.

## అభ్యాస లక్ష్యాలు

ఈ అధ్యాయాన్ని పూర్తి చేసిన తర్వాత, మీరు:
- Azure Developer CLI ఏమిటో మరియు ఇది Azure CLI తేడాలు ఏమిటో అర్థం చేసుకోవాలి
- మీ ప్లాట్‌ఫారమ్‌లో AZD ని ఇన్స్టాల్ చేసి కాన్ఫిగర్ చేయడం
- `azd up` తో మీ మొదటి అప్లికేషన్‌ను Azure కు డిప్లాయ్ చేయడం
- `azd down` తో వనరులను శుభ్రపరచడం

## ✅ ప్రారంభం ఇక్కడ: మీ సెటప్‌ను ధృవీకరించండి

నిర్వహణ మొదలుపెట్టేముందు, మీ స్థానిక యంత్రం Chapter 1 టెంప్లేట్‌కు సిద్ధంగా ఉందో లేదో నిర్ధారించుకోండి:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

స్క్రిప్ట్ లో అవసరమైన టూల్స్ లేమి చూపిస్తే, తొలుత వాటిని సరిచేసి ఆ తర్వాత అధ్యాయాన్ని కొనసాగించండి.

---

## 🚀 త్వరిత ప్రారంభం

```bash
# ఇన్‌స్టాలేషన్‌ను తనిఖీ చేయండి
azd version

# AZD కోసం ప్రామాణీకరించండి
# ఐచ్చికం: మీరు Azure CLI కమాండ్లను నేరుగా నడిపించాలనుకుంటే az login చేయండి
azd auth login

# మీ మొదటి యాప్‌ను డిప్లాయ్ చేయండి
azd init --template todo-nodejs-mongo
azd up

# పనులు పూర్తయిన తర్వాత శుభ్రపరచండి
azd down --force --purge
```

## ✅ విజయ ప్రమాణాలు

ఈ అధ్యాయాన్ని పూర్తి చేసిన తర్వాత, మీరు చేయగలగాలి:

```bash
azd version              # ఇన్‌స్టాల్ చేయబడిన సంచికను చూపిస్తుంది
azd init --template todo-nodejs-mongo  # ప్రాజెక్ట్‌ను ప్రారంభిస్తుంది
azd up                   # Azureపై డిప్లాయ్ చేస్తుంది
azd show                 # నడుస్తున్న యాప్ యొక్క URL చూపిస్తుంది
azd down --force --purge # వనరులను శుభ్రం చేస్తుంది
```
