**📚 కోర్స్**: [AZD ప్రారంభికులు కోసం](../../README.md) | **⏱️ వ్యవధి**: 30-45 నిమిషాలు | **⭐ సంక్లిష్టత**: ప్రారంభికుడు

## అవలోకనం

ఈ అధ్యాయం Azure Developer CLI (azd) పాఠాలు పరిచయం చేస్తోంది. మీరు కోర్ కాన్సెప్ట్స్ నేర్చుకుంటారు, టూల్స్ ఇన్స్టాల్ చేస్తారు, మరియు మీ మొదటి అప్లికేషన్ Azureలో డిప్లాయ్ చేస్తారు.

> జూలై 2026లో `azd 1.27.1` తో ధ్రువీకరించబడింది.

## అభ్యాస లక్ష్యాలు

ఈ అధ్యాయం పూర్తిచేసి, మీరు:
- Azure Developer CLI అంటే ఏంటి మరియు Azure CLIతో ఎలా భిన్నమో అర్థం చేసుకుంటారు
- మీ ప్లాట్‌ఫారమ్‌లో AZDని ఇన్స్టాల్ చేసి కాన్ఫిగర్ చేస్తారు
- `azd up` తో మీ మొట్టమొదటి అప్లికేషన్ Azureలో డిప్లాయ్ చేస్తారు
- `azd down` తో వనరులను శుభ్రపరిచే విధానం నేర్చుకుంటారు

## ✅ ఇక్కడ ప్రారంభించండి: మీ సెటప్‌ను ధృవీకరించండి

మీరు మొదలు పెట్టేముందు, మీరు చాప్టర్ 1 టెంప్లేట్ కోసం మీ స్థానిక యంత్రం సిద్ధంగా ఉందని నిర్ధారించుకోండి:

**విండోస్:**
```powershell
.\validate-setup.ps1
```

**మెక్OS / లినక్స్:**
```bash
bash ./validate-setup.sh
```

స్క్రిప్ట్ మిస్సింగ్ టూల్స్ నివేదిస్తే, ముందుగా వాటిని సరిచేయండి, ఆపై అధ్యాయం కొనసాగించండి.

## 🚀 శీఘ్ర ప్రారంభం

```bash
# ఇన్‌స్టాలేషన్ తనిఖీ చేయండి
azd version

# AZD కోసం ప్రమాణీకరించండి
# ఐచ్ఛికం: మీరు Azure CLI ఆజ్ఞలు నేరుగా నడపాలని యోచిస్తే az లాగిన్ చేయండి
azd auth login

# మీ మొదటి యాప్‌ను విడుదల చేయండి
azd init --template todo-nodejs-mongo
azd up

# పూర్తయిన తరువాత శుభ్రపరచండి
azd down --force --purge
```

## ✅ విజయ సూచికలు

ఈ అధ్యాయం పూర్తయ్యాక, మీరు చేయగలుగుతారు:

```bash
azd version              # సంస్థాపిత సంస్కరణను చూపిస్తుంది
azd init --template todo-nodejs-mongo  # ప్రాజెక్ట్‌ను ఆరంభిస్తుంది
azd up                   # Azureకు పంపుతుంది
azd show                 # నడుస్తున్న యాప్ URLను ప్రదర్శిస్తుంది
azd down --force --purge # వనరులను శుభ్రపరుస్తుంది
```
