# అధ్యాయం 1: ప్రాథమికాలు & త్వరిత ప్రారంభం

**📚 కోర్సు**: [AZD For Beginners](../../README.md) | **⏱️ కాలావధి**: 30-45 minutes | **⭐ సంక్లిష్టత**: ప్రాథమిక స్థాయి

---

## అవలోకనం

ఈ అధ్యాయం Azure Developer CLI (azd) ప్రాథమికాంశాలను పరిచయం చేస్తుంది. మీరు ముఖ్య సూత్రాలు నేర్చుకుంటారు, టూల్‌లు ఇన్‌స్టాల్ చేస్తారు, మరియు మీ మొదటి అప్లికేషన్‌ను Azureకు డిప్లాయ్ చేస్తారు.

## నేర్చుకోవాల్సిన లక్ష్యాలు

ఈ అధ్యాయం పూర్తి చేసిన తర్వాత, మీరు:
- Azure Developer CLI ఏమిటి మరియు ఇది Azure CLI తో ఎలా భిన్నంగా ఉంటుందో అర్థం చేసుకోవడం
- మీ ప్లాట్‌ఫార్మ్‌లో AZD ను ఇన్‌స్టాల్ చేయడం మరియు కాన్ఫిగర్ చేయడం
- `azd up` తో మీ మొదటి అప్లికేషన్‌ను Azureకి డిప్లాయ్ చేయడం
- `azd down` ఉపయోగించి రీసోర్సులను శుభ్రపరచడం

## 🚀 త్వరిత ప్రారంభం

```bash
# ఇన్‌స్టలేషన్‌ను తనిఖీ చేయండి
azd version

# Azureలో లాగిన్ చేయండి
azd auth login

# మీ మొదటి యాప్‌ను డిప్లాయ్ చేయండి
azd init --template todo-nodejs-mongo
azd up

# పనులు ముగిసిన తర్వాత శుభ్రపరచండి
azd down --force --purge
```

## ✅ విజయ ప్రమాణాలు

ఈ అధ్యాయం పూర్తి చేసిన తర్వాత, మీరు చేయగలగాలి:

```bash
azd version              # ఇన్‌స్టాల్ చేయబడిన సంస్కరణను చూపిస్తుంది
azd init --template todo-nodejs-mongo  # ప్రాజెక్ట్‌ను ప్రారంభిస్తుంది
azd up                   # Azureపై అమర్చుతుంది
azd show                 # నడుస్తున్న యాప్ యొక్క URLను చూపిస్తుంది
azd down --force --purge # వనరులను శుభ్రం చేస్తుంది
```
