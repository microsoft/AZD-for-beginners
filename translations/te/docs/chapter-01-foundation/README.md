## అవలోకనం

ఈ అధ్యాయం Azure Developer CLI (`azd`) యొక్క ప్రాథమిక అంశాలను పరిచయం చేస్తుంది. మీరు ప్రధాన కాన్సెప్ట్స్ నేర్చుకుంటారు, టూల్స్ ఇన్‌స్టాల్ చేస్తారు, మరియు మీ మొదటి అనువర్తనాన్ని Azureకి డిప్లాయ్ చేయడం గూర్చి తెలుసుకుంటారు.

> మార్చి 2026లో `azd 1.23.12` తో ధృవీకరించబడింది.

## శిక్షణ లక్ష్యాలు

ఈ అధ్యాయం పూర్తి చేసిన తర్వాత, మీరు:
- Azure Developer CLI ఏమిటి మరియు అది Azure CLI నుంచి ఎలా వేరుగా ఉంటుందో అర్థం చేసుకోగలవు
- మీ ప్లాట్‌ఫార్మ్‌పై AZD ను ఇన్‌స్టాల్ చేసి కాన్ఫిగర్ చేయండి
- `azd up` ఉపయోగించి మీ మొదటి అనువర్తనాన్ని Azureలో డిప్లాయ్ చేయండి
- `azd down` తో వనరులను తొలగించి శుభ్రపరచండి

## ✅ ఇక్కడ ప్రారంభించండి: మీ సెటప్‌ను ధృవీకరించండి

ప్రారంభించే ముందు, మీ లోకల్ మెషీన్ Chapter 1 టెంప్లేట్ కోసం సిద్ధంగా ఉందో నిర్ధారించండి:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

स्क్రిప్ట్ లోపాలు చూపిస్తే, ఆ టూల్స్ లేకపోతే ముందుగా వాటిని సరిచేసి ఆ తర్వాత అధ్యాయంతో కొనసాగండి.

---

## 🚀 త్వరిత ప్రారంభం

```bash
# ఇన్‌స్టాలేషన్‌ను తనిఖీ చేయండి
azd version

# AZD కోసం ప్రామాణీకరించండి
# ఐచ్ఛికం: మీరు Azure CLI కమాండ్లు నేరుగా నడపాలనుకుంటే az login చేయండి
azd auth login

# మీ మొదటి యాప్‌ను డిప్లాయ్ చేయండి
azd init --template todo-nodejs-mongo
azd up

# పూర్తయిన తర్వాత శుభ్రపరచండి
azd down --force --purge
```

## ✅ విజయ ప్రమాణాలు

ఈ అధ్యాయం పూర్తి చేసిన తర్వాత, మీరు దీని చేయగలగాలి:

```bash
azd version              # ఇన్‌స్టాల్ చేయబడిన సంస్కరణను చూపిస్తుంది
azd init --template todo-nodejs-mongo  # ప్రాజెక్టును ప్రారంభిస్తుంది
azd up                   # Azureకు అమర్చుతుంది
azd show                 # నడుస్తున్న యాప్ యొక్క URLను చూపిస్తుంది
azd down --force --purge # వనరులను శుభ్రపరుస్తుంది
```
