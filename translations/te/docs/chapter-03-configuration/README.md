# అధ్యాయం 3: కాన్ఫిగరేషన్ & ప్రామాణీకరణ

**📚 కోర్సు**: [AZD For Beginners](../../README.md) | **⏱️ వ్యవధి**: 45-60 నిమిషాలు | **⭐ క్లిష్టత**: మధ్యస్థ

---

## అవలోకనం

ఈ అధ్యాయం Azure Developer CLI వినియోగాల కోసం పర్యావరణ కాన్ఫిగరేషన్, ప్రామాణీకరణ నమూనాలు, మరియు భద్రతకు సంబంధించిన ఉత్తమ పద్ధతులను కవర్ చేస్తుంది।

> `azd 1.23.12` ఆధారంగా మార్చి 2026 లో నిర్ధారించబడింది.

## నేర్చుకునే లక్ష్యాలు

ఈ అధ్యాయాన్ని పూర్తి చేసిన తర్వాత, మీరు:
- AZD కాన్ఫిగరేషన్ శ్రేణుల్ని గురించి పూర్తి అవగాహన పొందగలుగుతారు
- బహుళ పర్యావరణాలను (dev, staging, prod) నిర్వహించగలుగుతారు
- Managed identities ఉపయోగించి సురక్షిత ప్రామాణీకరణను అమలు చేయగలుగుతారు
- పర్యావరణానికి ప్రత్యేక సెట్టింగులను కాన్ఫిగర్ చేయగలుగుతారు

## 🚀 త్వరిత ప్రారంభం

```bash
# అనేక ఎన్విరాన్‌మెంట్లు సృష్టించండి
azd env new dev
azd env new staging
azd env new prod

# ఎన్విరాన్‌మెంట్లను మార్చండి
azd env select prod

# ఎన్విరాన్‌మెంట్ వేరియబుల్స్‌ను సెట్ చేయండి
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# కాన్ఫిగరేషన్‌ను వీక్షించండి
azd env get-values
```

---

## 🔧 కాన్ఫిగరేషన్ శ్రేణి

AZD సెట్టింగ్‌లు ఈ క్రమంలో వర్తిస్తాయి (తరువాతి సెట్టింగ్‌లు ముందున్న వాటిని ఓవర్‌రైడ్ చేస్తాయి):

1. **డిఫాల్ట్ విలువలు** (టెంప్లేట్లలో నిర్మించబడ్డవి)
2. **azure.yaml** (ప్రాజెక్ట్ కాన్ఫిగరేషన్)
3. **పర్యావరణ వేరియబుల్స్** (`azd env set`)
4. **కమాండ్-లైన్ ఫ్లాగ్‌లు** (`--location eastus`)

## 🔐 భద్రతకు సంబంధించిన ఉత్తమ పద్ధతులు

```bash
# మేనేజ్ చేయబడిన ఐడెంటిటీ ఉపయోగించండి (సిఫార్సు చేయబడింది)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD ప్రామాణీకరణ స్థితిని తనిఖీ చేయండి
azd auth status

# ఐచ్చికం: మీరు az కమాండ్లు నడపాలని భావిస్తే Azure CLI సందర్భాన్ని నిర్ధారించుకోండి
az account show

# అవసరం అయితే మళ్లీ ప్రామాణీకరించుకోండి
azd auth login

# ఐచ్చికం: az కమాండ్ల కోసం Azure CLI ప్రామాణీకరణను నవీకరించండి
az login
```
