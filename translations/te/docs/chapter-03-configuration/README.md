# అధ్యాయం 3: కాన్ఫిగరేషన్ & ప్రామాణీకరణ

**📚 కోర్సు**: [కొత్తవారికి AZD](../../README.md) | **⏱️ వ్యవధి**: 45-60 నిమిషాలు | **⭐ క్లిష్టత**: మధ్యస్థ

---

## అవలోకనం

ఈ అధ్యాయం Azure Developer CLI డిప్లాయ్‌మెంట్‌లకు పరిసరాల కాన్ఫిగరేషన్, ప్రామాణీకరణ నమూనాలు, మరియు భద్రత ఉత్తమ పద్ధతులను కవర్ చేస్తుంది.

> `azd 1.27.1` మీద జూలై 2026న నిర్ధారించబడింది.

## అధ్యయన లక్ష్యాలు

ఈ అధ్యాయం పూర్తి చేస్తే, మీరు:
- AZD కాన్ఫిగరేషన్ పరంపర నేర్చుకుంటారు
- బహుళ పరిసరాలను (dev, staging, prod) నిర్వహిస్తారు
- నిర్వహిత ఐడెంటిటీలతో సురక్షిత ప్రామాణీకరణ అమలు చేస్తారు
- పరిసర-స్పెసిఫిక్ సెట్టింగులను కాన్ఫిగర్ చేస్తారు

## 🚀 త్వరిత ప్రారంభం

```bash
# బహుళ పర్యావరణాలను సృష్టించండి
azd env new dev
azd env new staging
azd env new prod

# పర్యావరణాలను మార్పిడి చేయండి
azd env select prod

# పర్యావరణ వేరియబుల్స్‌ను అమర్చండి
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# కాన్ఫిగరేషన్‌ను చూడండి
azd env get-values
```

---

## 🔧 కాన్ఫిగరేషన్ పరంపర

AZD ఈ క్రమంలో సెట్టింగ్స్ వర్తింపజేస్తుంది (తరువాత వుంటున్నవి ముందు వున్నదానినుండి మించి ఉంటాయి):

1. **డిఫాల్ట్ విలువలు** (టెంప్లేట్లలో నిర్మించబడినవి)
2. **azure.yaml** (ప్రాజెక్ట్ కాన్ఫిగరేషన్)
3. **పరిసర వేరియబుల్స్** (`azd env set`)
4. **కమాండ్-లైన్ ఫ్లాగ్స్** (`--location eastus`)

## 🔐 భద్రత ఉత్తమ పద్ధతులు

```bash
# నిర్వహించబడిన ఐడెంటిటీ ఉపయోగించండి (శిఫార్సు చేయబడింది)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD ప్రమాణీకరణ స్థితిని తనిఖీ చేయండి
azd auth status

# ఐచ్ఛికం: మీరు az ఆదేశాలను అమలు చేయాలనుకుంటే Azure CLI సందర్భాన్ని ధృవీకరించండి
az account show

# అవసరమైతే మళ్ళీ ప్రమాణీకరించుకోండి
azd auth login

# ఐచ్ఛికం: az ఆదేశాల కోసం Azure CLI ప్రమాణీకరణను రిఫ్రెష్ చేయండి
az login
```
