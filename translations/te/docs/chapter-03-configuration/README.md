# అధ్యాయం 3: కాన్ఫిగరేషన్ & ప్రమాణీకరణ

**📚 కోర్సు**: [AZD ప్రారంభికులకు](../../README.md) | **⏱️ వ్యవధి**: 45-60 నిమిషాలు | **⭐ సంక్లిష్టత**: మధ్యస్థ

---

## అవలోకనం

ఈ అధ్యాయం Azure Developer CLI డిప్లాయ్‌మెంటుల కోసం పర్యావరణ కాన్ఫిగరేషన్, ప్రమాణీకరణ నమూనాలు, మరియు భద్రతా ఉత్తమ పద్ధతులను చర్చిస్తుంది.

> జూన్ 2026లో `azd 1.25.6`తో ధృవీకరించబడింది.

## నేర్చుకోవాల్సిన లక్ష్యాలు

By completing this chapter, you will:
- ప్రావీణ్యం పొందండి: AZD కాన్ఫిగరేషన్ హైరార్కీని నిపుణుడిగా వినియోగించడం
- బహుళ పర్యావరణాలను నిర్వహించడం (dev, staging, prod)
- managed identities తో సురక్షిత ప్రమాణీకరణ అమలు చేయడం
- పర్యావరణ-నిర్దిష్ట సెట్టింగులను కాన్ఫిగర్ చేయడం

## 🚀 త్వరిత ప్రారంభం

```bash
# అనేక ఎన్‌విరాన్‌మెంట్‌లను సృష్టించండి
azd env new dev
azd env new staging
azd env new prod

# ఎన్‌విరాన్‌మెంట్‌లు మార్చండి
azd env select prod

# ఎన్‌విరాన్‌మెంట్ వేరియబుల్స్‌ను సెటు చేయండి
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# కాన్ఫిగరేషన్ చూడండి
azd env get-values
```

---

## 🔧 కాన్ఫిగరేషన్ హైరార్కీ

AZD సెట్టింగులను ఈ క్రమంలో వర్తింపజేస్తుంది (తరువాతివి ముందున్నవన్నింటిని ఓవర్‌రైడ్ చేస్తాయి):

1. **డిఫాల్ట్ విలువలు** (టెంప్లేట్లలో నిర్మించబడ్డవి)
2. **azure.yaml** (ప్రాజెక్ట్ కాన్ఫిగరేషన్)
3. **పర్యావరణ వేరియబుల్స్** (`azd env set`)
4. **కమాండ్-లైన్ ఫ్లాగ్‌లు** (`--location eastus`)

## 🔐 భద్రతా ఉత్తమ పద్ధతులు

```bash
# నియంత్రిత ఐడెంటిటీ ఉపయోగించండి (సిఫార్సు)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD ఆథెంటికేషన్ స్థితిని తనిఖీ చేయండి
azd auth status

# ఐచ్ఛిక: మీరు az కమాండ్లు నడపాలనుకుంటే Azure CLI సందర్భాన్ని ధృవీకరించండి
az account show

# అవసరమైతే మళ్లీ ప్రమాణీకరించండి
azd auth login

# ఐచ్ఛిక: az కమాండ్ల కోసం Azure CLI ప్రామాణీకరణను రిఫ్రెష్ చేయండి
az login
```
