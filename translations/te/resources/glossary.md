## పరిచయం

ఈ సమగ్ర గ్లోసరీ Azure Developer CLI మరియు Azure క్లౌడ్ డెవలప్‌మెంట్‌లో ఉపయోగించే పదాలు, భావనలు మరియు సంక్షిప్తరూపాల నిర్వచనాలను అందిస్తుంది. టెక్కీ డాక్యుమెంటేషన్ అర్థం చేసుకోవడం, సమస్య పరిష్కారం చేయడం మరియు azd ప్రాజెక్టుల గురించి సమర్థవంతంగా సంభాషించడానికి అవసరమైన సూచన.

## నేర్చుకునే లక్ష్యాలు

ఈ గ్లోసరీని ఉపయోగించడం ద్వారా, మీరు:
- Azure Developer CLI అసాధారణ పదజాలం మరియు భావనలను అర్థం చేసుకుంటారు
- Azure క్లౌడ్ డెవలప్‌మెంట్ హోరాహోరీ పదాలు మరియు సాంకేతిక పదజాలం నేర్చుకుంటారు
- Infrastructure as Code మరియు డిప్లాయ్‌మెంట్ టర్మినాలజీకి సమర్ధవంతంగా సూచన పొందుతారు
- Azure సేవల పేర్లు, సంక్షిప్తరూపాలు మరియు వాటి ప్రయోజనాలు అర్థం చేసుకుంటారు
- ట్రబుల్షూటింగ్ మరియు డీబగ్గింగ్ టర్మినాలజీకి నిర్వచనాలను యాక్సెస్ చేస్తారు
- అడ్వాన్స్ Azure ఆర్కిటెక్చర్ మరియు డెవలప్‌మెంట్ భావనలను నేర్చుకుంటారు

## నేర్చుకున్న ఫలితాలు

ఈ గ్లోసరీని నిత్యంగా గమనించడం ద్వారా, మీరు చేయగలుగుతారు:
- సరైన Azure Developer CLI పదజాలాన్ని ఉపయోగించి సమర్ధవంతంగా సంభాషించు
- సాంకేతిక డాక్యుమెంటేషన్ మరియు ఎర్రర్ సందేశాలను స్పష్టంగా అర్థం చేసుకోగలరు
- Azure సేవలు మరియు కాన్సెప్ట్‌లను నమ్మకంగా నావిగేట్ చేయగలరు
- సరిగ్గా టెక్నికల్ పదజాలంతో సమస్యలను ట్రబుల్‌షూట్ చేయగలరు
- సమర్థవంతమైన సాంకేతిక భాషతో టీమ్ చర్చల్లో తోడదెలుగు
- మీ Azure క్లౌడ్ డెవలప్‌మెంట్ జ్ఞానాన్ని సిస్టమాటిక్‌గా విస్తరించేరు

## A

**ARM Template**  
Azure Resource Manager టెంప్లేట్. Azure రిసోర్సులను డిక్లరేటివ్‌గా నిర్వచించడానికి మరియు డిప్లాయ్ చేయడానికి ఉపయోగించే JSON-ఆధారిత Infrastructure as Code ఫార్మాట్.

**App Service**  
విభాగం-గా-సేవ (PaaS) ఆఫర్ అయిన Azure సేవ, ఇన్ఫ్రాస్ట్రక్చర్ నిర్వహించకుండా వెబ్ అప్లికేషన్లు, REST APIs మరియు మొబైల్ బ్యాకెండ్లను హోస్ట్ చేయడానికి.

**Application Insights**  
అప్లికేషన్ పనితీరును, అందుబాటును మరియు వినియోగాన్ని లోతుగా తనిఖీ చేసే Azure యొక్క అప్లికేషన్ పనితీరు మానిటరింగ్ (APM) సేవ.

**Azure CLI**  
Azure రిసోర్సులను నిర్వహించడానికి ఉపయోగించే కమాండ్-లైన్ ఇంటర్ఫేస్. azd ద్వారా ఆథెంటికేషన్ మరియు కొన్ని ఆపరేషన్ల కోసం ఉపయోగించబడుతుంది.

**Azure Developer CLI (azd)**  
డెవలపర్-కేంద్రీకృత కమాండ్-లైన్ టూల్, టెంప్లేట్‌లు మరియు Infrastructure as Code ఉపయోగించి అప్లికేషన్లను Azureలో నిర్మించడం మరియు డిప్లాయ్ చేయడం వేగవంతం చేస్తుంది.

**azure.yaml**  
azd ప్రాజెక్ట్ కోసం ప్రధాన కాన్ఫిగరేషన్ ఫైల్, ఇది సర్వీసులు, ఇన్ఫ్రాస్ట్రక్చర్ మరియు డిప్లాయ్‌మెంట్ హుక్స్ నిర్వచిస్తుంది.

**Azure Resource Manager (ARM)**  
రీసోర్సుల సృష్టి, నవీకరణ మరియు తొలగింపు కోసం మేనేజ్‌మెంట్ లేయర్‌ను అందించే Azure యొక్క డిప్లాయ్‌మెంట్ మరియు మేనేజ్‌మెంట్ సేవ.

## B

**Bicep**  
Azure రిసోర్సులు డిప్లాయ్ చేయడానికి Microsoft అభివృద్ధి చేసిన డొమైన్-స్పెసిఫిక్ లాంగ్వేజ్ (DSL). ARM టెంప్లేట్స్‌తో పోలిస్తే సరళమైన సింటాక్స్ అందిస్తుంది మరియు ARM కి కంపైల్ అవుతుంది.

**Build**  
సోర్స్ కో드를 కంపైల్ చేయడం, డిపెండెన్సీలను ఇన్స్టాల్ చేయడం, మరియు అప్లికేషన్లను డిప్లాయ్ కోసం প্রস্তুతిచేసే ప్రక్రియ.

**Blue-Green Deployment**  
డౌన్‌టైమ్ మరియు రిస్క్ తగ్గించడానికి ఒకటే లాంటి రెండు ప్రొడక్షన్ పర్యావరణాలను (blue మరియు green) ఉపయోగించే డిప్లాయ్‌మెంట్ వ్యూహం.

## C

**Container Apps**  
కాంప్లెక్స్ ఇన్ఫ్రాస్ట్రక్చర్ నిర్వహించకుండా కంటేనరైజ్డ్ అప్లికేషన్లు నడపడానికి Azure యొక్క సర్వర్లెస్ కంటెయినర్ సేవ.

**CI/CD**  
Continuous Integration/Continuous Deployment. కోడ్ మార్పులను ఒకాకి ఒకటి సమ్మిళితం చేసి అప్లికేషన్లను ఆటోమేటెడ్‌గా డిప్లాయ్ చేసే అభ్యాసాలు.

**Cosmos DB**  
గ్లోబలిగా విస్తరించిన, మల్టీ-మోడల్ డేటాబేస్ సేవ, throughput, latency, availability మరియు consistency కొరకు సమగ్ర SLAలను అందిస్తుంది.

**Configuration**  
అప్లికేషన్ ప్రవర్తన మరియు డిప్లాయ్‌మెంట్ ఎంపికలను నియంత్రించే సెట్టింగ్స్ మరియు పారామీటర్లు.

## D

**Deployment**  
లక్ష్య ఇన్ఫ్రాస్ట్రక్చర్‌పై అప్లికేషన్లు మరియు వాటి డిపెండెన్సీలను ఇన్‌స్టాల్ చేసి కాన్ఫిగర్ చేయడం యొక్క ప్రక్రియ.

**Docker**  
కంటెయినరైజేషన్ టెక్నాలజీ ఉపయోగించి అప్లికేషన్లు అభివృద్ధి చేయడం, షిప్ చేయడం మరియు రన్ చేయడానికి ఉపయోగించే ప్లాట్‌ఫారం.

**Dockerfile**  
Docker కంటెయినర్ ఇమేజ్‌ను బిల్డ్ చేయడానికి సూచనలు కలిగిన టెక్స్ట్ ఫైల్.

## E

**Environment**  
మీ అప్లికేషన్ యొక్క ఒక నిర్దిష్ట ఇన్స్టాన్స్‌ను సూచించే డిప్లాయ్‌మెంట్ లక్ష్యము (ఉదా., development, staging, production).

**Environment Variables**  
రన్‌టైమ్‌లో అప్లికేషన్లు యాక్సెస్ చేయగల కీ-విల్యూ జంటలుగా నిల్వ చేయబడిన కాన్ఫిగరేషన్ విలువలు.

**Endpoint**  
అప్లికేషన్ లేదా సేవ యాక్సెస్ చేయదగిన URL లేదా నెట్‌వర్క్ అడ్రస్.

## F

**Function App**  
ఇన్ఫ్రాస్ట్రక్చర్ నిర్వహించకుండా ఈవెంట్-డ్రివన్ కోడ్ నడపడానికి Azure యొక్క సర్వర్లెస్ కంప్యూట్ సేవ.

## G

**GitHub Actions**  
GitHub రిపాజిటరీలతో సమకూర్చబడిన CI/CD ప్లాట్‌ఫార్మ్, వర్క్‌ఫ్లోలను ఆటోమేట్ చేయడానికి.

**Git**  
సోర్స్ కోడ్‌లో మార్చిదాలను ట్రాక్ చేయడానికి ఉపయోగించే డిస్ట్రిబ్యూటెడ్ వెర్షన్ కంట్రోల్ సిస్టమ్.

## H

**Hooks**  
డిప్లాయ్‌మెంట్ లైఫ్సైకిల్‌లో నిర్దిష్ట బిందువుల వద్ద (preprovision, postprovision, predeploy, postdeploy) నడిచే కస్టమ్ స్క్రిప్ట్స్ లేదా కమాండ్లు.

**Host**  
అప్లికేషన్ ఏ Azure సేవ టైప్‌లో డిప్లాయ్ చేయబడుతుందో సూచించే (ఉదా., appservice, containerapp, function).

## I

**Infrastructure as Code (IaC)**  
ఇన్ఫ్రాస్ట్రక్చర్‌ను మాన్యువల్ ప్రక్రియల బదులు కోడ్ ద్వారా నిర్వచించడమ మరియు నిర్వహించడం యొక్క పద్ధతి.

**Init**  
కొత్త azd ప్రాజెక్ట్‌ను సాధారణంగా టెంప్లేట్ నుండి ప్రారంభించడం యొక్క ప్రక్రియ.

## J

**JSON**  
JavaScript Object Notation. కాన్ఫిగరేషన్ ఫైల్స్ మరియు API ప్రతిస్పందనల కోసం సాధారణంగా ఉపయోగించే డేటా ఇంటర్‌చేంజ్ ఫార్మాట్.

**JWT**  
JSON Web Token. పాక్షికాల మధ్య సమాచారం సురక్షితంగా పంపులపర్చడానికి ఉపయోగించే స్టాండర్డ్ JSON ఆబ్జెక్ట్ ఫార్మాట్.

## K

**Key Vault**  
సీక్రెట్స్, కీలు మరియు సర్టిఫికెట్లను సురక్షితంగా నిల్వ చేయడానికి Azure సేవ.

**Kusto Query Language (KQL)**  
Azure Monitor, Application Insights మరియు ఇతర Azure సేవలలో డేటాను విశ్లేషించడానికి ఉపయోగించే క్వరీ భాష.

## L

**Load Balancer**  
ఇన్‌కమింగ్ నెట్‌వర్క్ ట్రాఫిక్‌ను బహుచంచెలు సర్వర్ల లేదా ఇన్స్టాన్స్‌ల మధ్య పంపిణీ చేసే సేవ.

**Log Analytics**  
క్లౌడ్ మరియు ఆన్-ప్రెమైసెస్ పర్యావరణాల నుంచి టెలిమెట్రీ డేటాను సేకరించి విశ్లేషించి చర్యలు గ్రహించే Azure సేవ.

## M

**Managed Identity**  
ఇతర Azure సేవలకు ఆథెంటికేషన్ కోసం స్వయంచాలకంగా నిర్వహించబడే ఐడెంటిటీని Azure సేవలకు అందించే ఫీచర్.

**Microservices**  
అప్లికేషన్లను చిన్న, స్వతంత్ర సర్వీసుల సమూహంగా నిర్మించే ఆర్కిటెక్చరల్ విధానం.

**Monitor**  
అప్లికేషన్ల మరియు ఇన్ఫ్రాస్ట్రక్చర్ జోరుగా ఫుల్-స్టాక్ ఆబ్జర్వబిలిటీని అందించే Azure యొక్క సమైక్య మానిటరింగ్ పరిష్కారం.

## N

**Node.js**  
చార్మ్ యొక్క V8 JavaScript ఇంజిన్‌పై నిర్మించిన JavaScript రన్‌టైమ్, సర్వర్-సైడ్ అప్లికేషన్లు నిర్మించడానికి ఉపయోగించబడుతుంది.

**npm**  
Node.js కోసం ప్యాకేజీ మేనేజర్, డిపెండెన్సీలను మరియు ప్యాకేజీలను నిర్వహిస్తుంది.

## O

**Output**  
ఇన్‌ఫ్రాస్ట్రక్చర్ డిప్లాయ్‌మెంట్ నుండి తిరిగి వచ్చే విలువలు, ఇవి అప్లికేషన్లు లేదా ఇతర రిసోర్సులు ద్వారా ఉపయోగించబడవచ్చు.

## P

**Package**  
డిప్లాయ్‌మెంట్ కోసం అప్లికేషన్ కోడ్ మరియు డిపెండెన్సీలను సిద్ధం చేసే ప్రక్రియ.

**Parameters**  
డిప్లాయ్‌మెంట్లను అనుకూలీకరించడానికి ఇన్ఫ్రాస్ట్రక్చర్ టెంప్లేట్స్‌కు ఇచ్చే ఇన్పుట్ విలువలు.

**PostgreSQL**  
ఓపెన్-సోర్స్ రిలేషనల్ డేటాబేస్ సిస్టమ్, Azureలో మ్యానేజ్ చేయబడే సేవగా సపోర్ట్ చేయబడుతుంది.

**Provisioning**  
ఇన్ఫ్రాస్ట్రక్చర్ టెంప్లేట్స్‌లో నిర్వచించిన Azure రిసోర్సులను సృష్టించడం మరియు కాన్ఫిగర్ చేయడం యొక్క ప్రక్రియ.

## Q

**Quota**  
Azure సబ్స్క్రిప్షన్ లేదా రీజియన్‌లో సృష్టించగల రిసోర్సుల పరిమితులు.

## R

**Resource Group**  
ఒక్కే లైఫ్సైకల్, అనుమతులు మరియు పాలసీలను పంచుకుంటున్న Azure రిసోర్సుల కోసం లాజికల్ కంటైనర్.

**Resource Token**  
డిప్లాయ్‌మెంట్లలో రిసోర్స్ పేర్లు ప్రత్యేకంగా ఉండేలా azd ద్వారా జనరేట్ చేయబడే ప్రత్యేక స్ట్రింగ్.

**REST API**  
HTTP మెతోడ్స్ ఉపయోగించి నెట్‌వర్క్ అప్లికేషన్లను డిజైన్ చేయడానికి ఉపయోగించే ఆర్కిటెక్చరల్ స్టైల్.

**Rollback**  
అప్లికేషన్ లేదా ఇన్ఫ్రాస్ట్రక్చర్ కాన్ఫిగరేషన్ యొక్క పూర్వవన్త స్థితికి తిరిగి వెళ్లే ప్రక్రియ.

## S

**Service**  
azure.yaml లో నిర్వచించబడిన మీ అప్లికేషన్ యొక్క ఒక భాగం (ఉదా., web frontend, API backend, database).

**SKU**  
Stock Keeping Unit. Azure రిసోర్సుల కోసం వివిధ సర్వీస్ టియర్‌లు లేదా పనితీరు స్థాయిలను సూచిస్తుంది.

**SQL Database**  
Microsoft SQL Server ఆధారంగా ఉన్న Azure యొక్క మ్యానేజ్ చేయబడే రిలేషనల్ డేటాబేస్ సేవ.

**Static Web Apps**  
సోర్స్ కోడ్ రిపాజిటరీల నుండి ఫుల్-స్టాక్ వెబ్ అప్లికేషన్లను నిర్మించడానికి మరియు డిప్లాయ్ చేయడానికి Azure సేవ.

**Storage Account**  
బ్లాబ్స్, ఫైల్స్, క్యూ మరియు టేబుల్స్ వంటి డేటా ఆబ్జెక్టుల కోసం క్లౌడ్ స్టోరేజ్‌ను అందించే Azure సేవ.

**Subscription**  
రిసోర్స్ గ్రూప్‌లు మరియు రిసోర్సులను కలిగిన Azure ఖాతా కంటైనర్, బిల్లింగ్ మరియు యాక్సెస్ మేనేజ్‌మెంట్‌తో.

## T

**Template**  
సాధారణ సన్నివేశాల కోసం అప్లికేషన్ కోడ్, ఇన్ఫ్రాస్ట్రక్చర్ నిర్వచనాలు మరియు కాన్ఫిగరేషన్‌ను కలిగిన ముందుగా సిద్ధమైన ప్రాజెక్ట్ నిర్మాణం.

**Terraform**  
బహు క్లౌడ్ ప్రొవైడర్లను సపోర్ట్ చేసే ఓపెన్-సోర్స్ Infrastructure as Code టూల్, ఇది Azureని కూడా సపోర్ట్ చేస్తుంది.

**Traffic Manager**  
గ్లోబల్ Azure రీజియన్లలో ట్రాఫిక్‌ను పంపిణీ చేయడానికి DNS-ఆధారిత ట్రాఫిక్ లోడ్ బాలాన్సర్.

## U

**URI**  
Uniform Resource Identifier. ఒక నిర్దిష్ట రిసోర్సును గుర్తించెయ్యే స్ట్రింగ్.

**URL**  
Uniform Resource Locator. ఒక రిసోర్సు ఎక్కడ ఉండదో మరియు దాన్ని ఎలా పొందాలో స్పష్టంగా సూచించే URI రకం.

## V

**Virtual Network (VNet)**  
Azureలో ప్రైవేట్ నెట్‌వర్క్‌ల బునియాది కలన koja, ఐసొలేషన్ మరియు సెగ్మెంటేషన్‌ను అందిస్తుంది.

**VS Code**  
Visual Studio Code. అద్భుతమైన Azure మరియు azd సమన్వయం ఉన్న ప్రాచీన కోడ్ ఎడిటర్.

## W

**Webhook**  
ఖచ్చితమైన ఈవెంట్స్ ద్వారా ట్రిగ్గర్ అయ్యే HTTP కాల్‌బ్యాక్, సాధారణంగా CI/CD పైప్‌లైన్స్‌లో ఉపయోగిస్తారు.

**What-if**  
డిప్లాయ్‌మెంట్ నిజంగా అమలు చేయకుండా ఏ మార్పులు చేయబడతాయో చూపించే Azure ఫీచర్.

## Y

**YAML**  
YAML Ain't Markup Language. azure.yaml వంటి కాన్ఫిగరేషన్ ఫైళ్లకు ఉపయోగించే మానవ-ఓదార్య డేటా సీరియలైజేషన్ స్టాండర్డ్.

## Z

**Zone**  
రెడండన్సీ మరియు హై అవైలబిలిటీని అందించే Azure రీజియన్‌లో శారీరకంగా వేరు చోట్లని సూచిస్తుంది.

## సందర్భానికి అనుగుణంగా టర్మ్స్

### Development Terms
- **Hot Reload**: డెవలప్‌మెంట్ సమయంలో రీస్టార్ట్ చేయకుండా అప్లికేషన్లను ఆటోమేటిగ్గా అప్‌డేట్ చేయడం
- **Build Pipeline**: కోడ్ బిల్డ్ చేయడం మరియు టెస్ట్ చేయడానికి ఆటోమేటెడ్ ప్రక్రియ
- **Deployment Slot**: App Service లోని స్టేజింగ్ వాతావరణం
- **Environment Parity**: development, staging మరియు production వాతావరణాలను సమానంగా ఉంచడం

### Security Terms
- **Managed Identity**: ఆటోమేటిక్ క్రెడెంచియల్ మేనేజ్‌మెంట్ అందించే Azure ఫీచర్
- **Key Vault**: సీక్రెట్స్, కీలు మరియు సర్టిఫికెట్ల కోసం సురక్షిత నిల్వ
- **RBAC**: Azure రిసోర్సుల కోసం పాత్ర ఆధారిత యాక్సెస్ నియంత్రణ
- **Network Security Group**: నెట్‌వర్క్ ట్రాఫిక్‌ను నియంత్రించే వర్చువల్ ఫైర్వాల్

### Monitoring Terms
- **Telemetry**: కొలతలు మరియు డేటాను ఆటోమేటిగ్గా సేకరించటం
- **Application Performance Monitoring (APM)**: సాఫ్ట్‌వేర్ పనితీరు మానిటరింగ్
- **Log Analytics**: లాగ్ డేటా సేకరించడం మరియు విశ్లేషించడం కోసం సేవ
- **Alert Rules**: మెట్రిక్స్ లేదా షరతుల ఆధారంగా ఆటోమేటెడ్ నోటిఫికేషన్లు

### Deployment Terms
- **Blue-Green Deployment**: జీరో-డౌన్‌టైమ్ డిప్లాయ్‌మెంట్ వ్యూహం
- **Canary Deployment**: వినియోగదారుల ఉపసెట్‌కు ప్రమేయిస్తూ تدريجిగా రోల్‌అవుట్
- **Rolling Update**: అప్లికేషన్ ఇన్స్టాన్సులను వరసగా మార్పిడి చేయడం
- **Rollback**: పూర్వపు అప్లికేషన్ వెర్షన్‌కు తిరిగి వెళ్లడం
