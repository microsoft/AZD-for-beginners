# AZD per sviluppatori AI - Workshop

> [!IMPORTANT]  
> **Questo workshop è dotato di una guida che puoi visualizzare in anteprima nel tuo browser locale. Per iniziare, devi avviare GitHub Codespaces sul repository—quindi attendi fino a quando non vedi un terminale VS Code attivo e digita:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Dovresti vedere una finestra pop-up per aprire la pagina di anteprima in un browser.**

Benvenuto al workshop pratico per imparare Azure Developer CLI (AZD) con un focus sul deployment di applicazioni AI. Questo workshop ti aiuta ad acquisire una comprensione applicata dei template AZD in 3 passaggi:

1. **Scoperta** - trovare il template più adatto a te.
1. **Distribuzione** - distribuire e verificare che funzioni
1. **Personalizzazione** - modificare e iterare per adattarlo a te!

Nel corso di questo workshop, ti verranno inoltre presentati strumenti e workflow fondamentali per sviluppatori, per aiutarti a snellire il tuo percorso di sviluppo end-to-end.

## Panoramica del workshop

**Durata:** 3-4 ore  
**Livello:** Principiante - Intermedio  
**Prerequisiti:** Familiarità con Azure, concetti di AI, VS Code e strumenti da riga di comando.

Questo è un workshop pratico in cui si impara facendo. Una volta completati gli esercizi, ti consigliamo di consultare il curriculum AZD per principianti per proseguire il tuo percorso di apprendimento sulle migliori pratiche di Sicurezza e Produttività.

| Tempo| Modulo  | Obiettivo |
|:---|:---|:---|
| 15 mins | Introduzione | Stabilire il contesto, comprendere gli obiettivi |
| 30 mins | Seleziona template AI | Esplora le opzioni e scegli un modello iniziale | 
| 30 mins | Valida template AI | Distribuire la soluzione predefinita su Azure |
| 30 mins | Decomponi il template AI | Esplora struttura e configurazione |
| 30 mins | Configura il template AI | Attiva e prova le funzionalità disponibili |
| 30 mins | Personalizza il template AI | Adatta il template alle tue esigenze |
| 30 mins | Smantellamento dell'infrastruttura | Pulizia e rilascio delle risorse |
| 15 mins | Conclusione e passi successivi | Risorse di apprendimento, sfida del workshop |
| | |

## Cosa imparerai

Considera il template AZD come un sandbox di apprendimento per esplorare varie capacità e strumenti per lo sviluppo end-to-end su Microsoft Foundry. Al termine del workshop, dovresti avere una comprensione intuitiva dei diversi strumenti e concetti in questo contesto.

| Concetto  | Obiettivo |
|:---|:---|
| **Azure Developer CLI** | Comprendere i comandi e i flussi di lavoro dello strumento |
| **AZD Templates**| Comprendere la struttura del progetto e la configurazione |
| **Azure AI Agent**| Provisioning e distribuzione del progetto Microsoft Foundry  |
| **Azure AI Search**| Abilitare l'ingegneria del contesto con agenti |
| **Observability**| Esplorare tracciamento, monitoraggio e valutazioni |
| **Red Teaming**| Esplorare test avversariali e mitigazioni |
