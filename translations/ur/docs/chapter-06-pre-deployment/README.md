# باب 6: قبل از تعیناتی منصوبہ بندی اور تصدیق

**📚 کورس**: [AZD For Beginners](../../README.md) | **⏱️ دورانیہ**: 1 گھنٹہ | **⭐ پیچیدگی**: درمیانہ

---

## جائزہ

یہ باب آپ کی درخواست کی تعیناتی سے پہلے لازمی منصوبہ بندی اور تصدیقی اقدامات کا احاطہ کرتا ہے۔ درست صلاحیت کی منصوبہ بندی، SKU انتخاب، اور پری فلائٹ چیکس کے ساتھ مہنگی غلطیوں سے بچنا سیکھیں۔

## سیکھنے کے مقاصد

اس باب کو مکمل کرنے کے بعد، آپ:
- تعیناتی سے پہلے پری فلائٹ چیکس چلائیں گے
- صلاحیت کی منصوبہ بندی کریں گے اور وسائل کی ضروریات کا اندازہ لگائیں گے
- لاگت کی بچت کے لیے مناسب SKU منتخب کریں گے
- مانیٹرنگ کے لیے Application Insights کو ترتیب دیں گے
- ٹیم کوآرڈینیشن کے نمونے سمجھیں گے

## 🚀 جلدی آغاز

```bash
# رکنیت کے کوٹے چیک کریں
az vm list-usage --location eastus --output table

# تعیناتی کا پیش نظارہ (کوئی وسائل تخلیق نہیں ہوئے)
azd provision --preview

# بائسک نحو کی توثیق کریں
az bicep build --file infra/main.bicep

# ماحول کی ترتیب چیک کریں
azd env get-values
```

## ☑️ قبل از تعیناتی چیک لسٹ

### `azd provision` سے پہلے

- [ ] خطے کے لیے کوٹا کی تصدیق کی گئی
- [ ] SKUs مناسب طریقے سے منتخب کیے گئے
- [ ] لاگت کا تخمینہ جائزہ لیا گیا
- [ ] نام رکھنے کا طریقہ کار یکساں ہے
- [ ] سیکیورٹی/RBAC ترتیب دی گئی

### `azd deploy` سے پہلے

- [ ] ماحول کی تبدیلیاں سیٹ کی گئیں
- [ ] راز Key Vault میں ہیں
- [ ] کنکشن سٹرنگز کی تصدیق کی گئی
- [ ] صحت کے چیکس ترتیب دیے گئے
