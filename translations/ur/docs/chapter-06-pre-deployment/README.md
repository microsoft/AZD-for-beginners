# باب 6: قبل از تعیناتی منصوبہ بندی اور توثیق

**📚 کورس**: [AZD برائے مبتدی](../../README.md) | **⏱️ دورانیہ**: 1 گھنٹہ | **⭐ پیچیدگی**: درمیانہ

---

## جائزہ

یہ باب آپ کی ایپلیکیشن کی تعیناتی سے پہلے ضروری منصوبہ بندی اور توثیق کے اقدامات کا احاطہ کرتا ہے۔ مناسب گنجائش کی منصوبہ بندی، SKU کے انتخاب، اور پری فلائٹ چیکس کے ذریعے مہنگی غلطیوں سے بچنا سیکھیں۔

## سیکھنے کے مقاصد

اس باب کو مکمل کرنے پر، آپ:
- تعیناتی سے پہلے پری فلائٹ چیکس چلائیں گے
- گنجائش کی منصوبہ بندی کریں گے اور وسائل کی ضروریات کا تخمینہ لگائیں گے
- لاگت کی بہتر تیاری کے لیے مناسب SKUs منتخب کریں گے
- مانیٹرنگ کے لیے Application Insights کو کنفیگر کریں گے
- ٹیم کوآرڈینیشن پیٹرنز کو سمجھیں گے

## 🚀 فوری آغاز

```bash
# سبسکرپشن کے کوٹے چیک کریں
az vm list-usage --location eastus --output table

# تنصیب کا پیش نظارہ (کوئی وسائل تخلیق نہیں ہوں گے)
azd provision --preview

# Bicep نحو کی توثیق کریں
az bicep build --file infra/main.bicep

# ماحول کی ترتیب چیک کریں
azd env get-values
```

## ☑️ قبل از تعیناتی چیک لسٹ

### Before `azd provision`

- [ ] علاقے کے لیے کوٹا کی تصدیق کی گئی ہو
- [ ] SKUs موزوں طریقے سے منتخب کیے گئے ہوں
- [ ] لاگت کا تخمینہ جائزہ لیا گیا ہو
- [ ] نام رکھنے کا طریقہ کار یکساں ہو
- [ ] سیکورٹی/RBAC ترتیب دی گئی ہو

### Before `azd deploy`

- [ ] ماحولیاتی متغیرات سیٹ کیے گئے ہوں
- [ ] راز Key Vault میں محفوظ ہوں
- [ ] کنکشن سٹرنگز کی تصدیق کی گئی ہو
- [ ] ہیلتھ چیکس ترتیب دیے گئے ہوں
