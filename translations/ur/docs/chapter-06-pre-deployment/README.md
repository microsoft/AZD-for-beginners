# باب 6: تعیناتی سے پہلے کی منصوبہ بندی اور تصدیق

**📚 کورس**: [نئے صارفین کے لیے AZD](../../README.md) | **⏱️ دورانیہ**: 1 گھنٹہ | **⭐ مشکل سطح**: درمیانہ

---

## جائزہ

یہ باب آپ کی درخواست کو تعینات کرنے سے پہلے ضروری منصوبہ بندی اور تصدیقی مراحل کا احاطہ کرتا ہے۔ درست صلاحیت کی منصوبہ بندی، SKU انتخاب، اور پری فلائٹ چیکس کے ذریعے مہنگی غلطیوں سے بچنا سیکھیں۔

> جولائی 2026 میں `azd 1.27.1` کے ساتھ تصدیق شدہ۔

## سیکھنے کے مقاصد

اس باب کو مکمل کرکے، آپ:
- تعیناتی سے پہلے پری فلائٹ چیکس چلائیں گے
- صلاحیت کی منصوبہ بندی کریں گے اور وسائل کی ضروریات کا اندازہ لگائیں گے
- لاگت کی بہتری کے لیے مناسب SKUs کا انتخاب کریں گے
- مانیٹرنگ کے لیے ایپلیکیشن انسائٹس کو ترتیب دیں گے
- ٹیم کوآرڈینیشن کے نمونوں کو سمجھیں گے

## 🚀 فوری آغاز

```bash
# رکنیت کوٹے چیک کریں
az vm list-usage --location eastus --output table

# تعیناتی کا پیش نظارہ (کوئی وسائل نہیں بنائے گئے)
azd provision --preview

# بائسِپ نحو کی توثیق کریں
az bicep build --file infra/main.bicep

# ماحول کی ترتیب چیک کریں
azd env get-values
```

## ☑️ پری-تعیناتی چیک لسٹ

### `azd provision` سے پہلے

- [ ] خطہ کے لیے کوٹا کی تصدیق کی گئی
- [ ] SKUs کا مناسب انتخاب کیا گیا
- [ ] لاگت کا تخمینہ جائزہ لیا گیا
- [ ] نام رکھنے کا طریقہ کار یکساں ہے
- [ ] سیکیورٹی/RBAC ترتیب دیا گیا

### `azd deploy` سے پہلے

- [ ] ماحول کے متغیرات مقرر کیے گئے
- [ ] خفیہ معلومات کیی واؤلٹ میں ہیں
- [ ] کنکشن سٹرنگز کی تصدیق کی گئی
- [ ] صحت کی جانچیں ترتیب دی گئیں
