# باب 4: انفراسٹرکچر ایز کوڈ اور تعیناتی

**📚 کورس**: [ابتدائیوں کے لیے AZD](../../README.md) | **⏱️ دورانیہ**: 1-1.5 گھنٹے | **⭐ پیچیدگی**: درمیانہ

---

## جائزہ

یہ باب Bicep ٹیمپلیٹس کے ساتھ Infrastructure as Code (IaC) پیٹرنز، وسائل کی فراہمی، اور Azure Developer CLI کے ذریعے تعیناتی کی حکمت عملیوں کا احاطہ کرتا ہے۔

> جولائی 2026 میں `azd 1.27.1` کے خلاف تصدیق شدہ۔

## سیکھنے کے مقاصد

اس باب کو مکمل کر کے، آپ:
- Bicep ٹیمپلیٹ کی ساخت اور نحو کو سمجھیں گے
- `azd provision` کے ذریعے Azure وسائل کی فراہمی کریں گے
- `azd deploy` کے ذریعے ایپلی کیشنز کی تعیناتی کریں گے
- نیلا-سبز اور رولنگ تعیناتی کی حکمت عملیوں کو نافذ کریں گے

## 🚀 فوری آغاز

```bash
# سانچوں سے ابتدائیہ کریں
azd init --template azure-functions-python-v2-http

# پیش نظارہ کریں کہ کیا بنایا جائے گا
azd provision --preview

# صرف بنیادی ڈھانچہ فراہم کریں
azd provision

# صرف کوڈ تعینات کریں
azd deploy

# یا دونوں کو ایک ساتھ
azd up
```

---

## 📁 AZD پروجیکٹ کی ساخت

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

---

## 🔧 بنیادی کمانڈز

| کمانڈ | وضاحت |
|---------|-------------|
| `azd init` | پروجیکٹ شروع کریں |
| `azd provision` | Azure وسائل بنائیں |
| `azd deploy` | ایپلیکیشن کوڈ تعینات کریں |
| `azd up` | فراہمی + تعیناتی |
| `azd down` | تمام وسائل حذف کریں |
