# باب 3: ترتیب اور توثیق

**📚 کورس**: [ابتدائیوں کے لیے AZD](../../README.md) | **⏱️ دورانیہ**: 45-60 منٹ | **⭐ پیچیدگی**: درمیانہ

---

## جائزہ

یہ باب Azure Developer CLI کی تعیناتیوں کے لیے ماحول کی ترتیب، توثیق کے پیٹرنز، اور سیکیورٹی کی بہترین مشقوں کا احاطہ کرتا ہے۔

> جولائی 2026 میں `azd 1.27.1` کے خلاف تصدیق شدہ۔

## تعلیمی مقاصد

اس باب کو مکمل کرکے، آپ:
- AZD ترتیب کے درجہ بندی کو مہارت سے سمجھیں گے
- متعدد ماحول (dev, staging, prod) کا انتظام کریں گے
- مینیجڈ شناختوں کے ساتھ محفوظ توثیق نافذ کریں گے
- ماحول کی مخصوص ترتیبات کو ترتیب دیں گے

## 🚀 جلدی آغاز

```bash
# متعدد ماحول بنائیں
azd env new dev
azd env new staging
azd env new prod

# ماحول تبدیل کریں
azd env select prod

# ماحولیاتی متغیرات سیٹ کریں
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# ترتیب دیکھیں
azd env get-values
```

---

## 🔧 ترتیب کی درجہ بندی

AZD ترتیبات کو اس ترتیب میں لاگو کرتا ہے (آخری پہلے کو اوور رائڈ کرتا ہے):

1. **ڈیفالٹ ویلیوز** (ٹیمپلیٹس میں بنائی گئی)
2. **azure.yaml** (پروجیکٹ کی ترتیب)
3. **ماحول کے متغیرات** (`azd env set`)
4. **کمانڈ لائن فلیگز** (`--location eastus`)

---

## 🔐 سیکیورٹی کی بہترین مشقیں

```bash
# مینیجڈ شناخت استعمال کریں (تجویز کردہ)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD تصدیقی حیثیت چیک کریں
azd auth status

# اختیاری: اگر آپ az کمانڈز چلانے کا ارادہ رکھتے ہیں تو Azure CLI سیاق و سباق کی تصدیق کریں
az account show

# اگر ضرورت ہو تو دوبارہ تصدیق کریں
azd auth login

# اختیاری: az کمانڈز کے لیے Azure CLI کی تصدیق کو تازہ کریں
az login
```
