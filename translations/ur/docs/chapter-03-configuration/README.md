# باب 3: ترتیب اور تصدیق

**📚 کورس**: [ابتدائیوں کے لیے AZD](../../README.md) | **⏱️ دورانیہ**: 45-60 منٹ | **⭐ پیچیدگی**: درمیانہ

---

## جائزہ

یہ باب Azure Developer CLI کی تعیناتیوں کے لیے ماحول کی ترتیب، تصدیقی نمونے، اور سیکیورٹی کی بہترین طرزعمل کا احاطہ کرتا ہے۔

> جون 2026 میں `azd 1.25.6` کے خلاف تصدیق شدہ۔

## سیکھنے کے مقاصد

اس باب کو مکمل کرکے آپ:
- AZD ترتیب کے درجہ بندی نظام پر مہارت حاصل کریں گے
- متعدد ماحول (ڈیولپمنٹ، اسٹیجنگ، پروڈکشن) کا انتظام کریں گے
- منیجڈ شناختوں کے ساتھ محفوظ تصدیق نافذ کریں گے
- ماحول کے لحاظ سے مخصوص ترتیبات کو ترتیب دیں گے

## 🚀 فوری آغاز

```bash
# متعدد ماحولیاتی نظام تخلیق کریں
azd env new dev
azd env new staging
azd env new prod

# ماحولیاتی نظام تبدیل کریں
azd env select prod

# ماحولیاتی متغیرات سیٹ کریں
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# ترتیب دیکھیں
azd env get-values
```

---

## 🔧 ترتیب کی درجہ بندی

AZD ترتیبات کو اس ترتیب میں لاگو کرتا ہے (بعد والی پہلی کو اوور رائڈ کرتی ہے):

1. **ڈیفالٹ اقدار** (سانچوں میں شامل)
2. **azure.yaml** (منصوبے کی ترتیب)
3. **ماحول کے متغیرات** (`azd env set`)
4. **کمانڈ لائن جھنڈے** (`--location eastus`)

---

## 🔐 سیکیورٹی کی بہترین طرزعمل

```bash
# منظم شناخت استعمال کریں (تجویز کردہ)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD تصدیقی صورتحال چیک کریں
azd auth status

# اختیاری: اگر آپ az کمانڈز چلانے کا ارادہ رکھتے ہیں تو Azure CLI کے سیاق و سباق کی تصدیق کریں
az account show

# ضرورت ہو تو دوبارہ تصدیق کریں
azd auth login

# اختیاری: az کمانڈز کے لیے Azure CLI تصدیق کو تازہ کریں
az login
```
