# باب 3: ترتیب اور توثیق

**📚 کورس**: [AZD مبتدیوں کے لیے](../../README.md) | **⏱️ دورانیہ**: 45-60 منٹ | **⭐ پیچیدگی**: درمیانی

---

## جائزہ

یہ باب ماحول کی ترتیب، توثیقی پیٹرنز، اور Azure Developer CLI ڈیپلائمنٹ کے لیے حفاظتی بہترین طریقوں کا احاطہ کرتا ہے۔

> مارچ 2026 میں `azd 1.23.12` کے خلاف جانچا گیا۔

## سیکھنے کے مقاصد

اس باب کو مکمل کر کے آپ:
- AZD ترتیب کی درجہ بندی میں مہارت حاصل کریں گے
- متعدد ماحول (dev, staging, prod) کا انتظام کریں گے
- منیجڈ شناختوں کے ساتھ محفوظ توثیق نافذ کریں گے
- ماحول مخصوص ترتیبات کو مرتب کریں گے

## 🚀 تیز آغاز

```bash
# متعدد ماحول بنائیں
azd env new dev
azd env new staging
azd env new prod

# ماحول تبدیل کریں
azd env select prod

# ماحول کے متغیرات مقرر کریں
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# ترتیب دیکھیں
azd env get-values
```

---

## 🔧 ترتیب کی درجہ بندی

AZD ترتیب کو اس ترتیب میں لاگو کرتا ہے (بعد والا پہلے کو اووررائڈ کرتا ہے):

1. **ڈیفالٹ اقدار** (ٹیمپلیٹس میں شامل)
2. **azure.yaml** (پروجیکٹ کی ترتیب)
3. **ماحولیاتی متغیرات** (`azd env set`)
4. **کمانڈ لائن فلیگز** (`--location eastus`)

---

## 🔐 حفاظتی بہترین طریقے

```bash
# مینجڈ شناخت کا استعمال کریں (تجویز کردہ)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD کی تصدیق کی حیثیت چیک کریں
azd auth status

# اختیاری: اگر آپ az کمانڈز چلانا چاہتے ہیں تو Azure CLI کے سیاق و سباق کی تصدیق کریں
az account show

# ضرورت ہو تو دوبارہ تصدیق کریں
azd auth login

# اختیاری: az کمانڈز کے لیے Azure CLI کی تصدیق کو ریفریش کریں
az login
```
