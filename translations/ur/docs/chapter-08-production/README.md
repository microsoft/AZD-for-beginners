# باب 8: پروڈکشن اور انٹرپرائز پیٹرنز

**📚 کورس**: [AZD برائے مبتدی](../../README.md) | **⏱️ دورانیہ**: 2-3 گھنٹے | **⭐ پیچیدگی**: اعلیٰ

---

## جائزہ

یہ باب انٹرپرائز کے لیے تیار تعیناتی کے نمونے، سیکیورٹی سختی، مانیٹرنگ، اور پروڈکشن AI ورک لوڈز کے لیے لاگت کی اصلاح کو کور کرتا ہے۔

## سیکھنے کے مقاصد

اس باب کو مکمل کرنے کے بعد آپ:
- کثیر العلاقائی مزاحم ایپلیکیشنز کی تعیناتی کریں
- انٹرپرائز سیکیورٹی پیٹرنز نافذ کریں
- جامع مانیٹرنگ ترتیب دیں
- بڑے پیمانے پر لاگت کی اصلاح کریں
- AZD کے ساتھ CI/CD پائپ لائنز قائم کریں

## 🚀 پروڈکشن چیک لسٹ

- [ ] مزاحمت کے لیے کثیر العلاقائی تعیناتی
- [ ] تصدیق کے لیے منیجڈ شناخت (کوئی کیز نہیں)
- [ ] مانیٹرنگ کے لیے Application Insights
- [ ] لاگت کے بجٹ اور الارٹس ترتیب دیے گئے
- [ ] سیکیورٹی اسکیننگ فعال
- [ ] CI/CD پائپ لائن کا انٹیگریشن
- [ ] آفات سے بحالی کا منصوبہ

## 🔐 سیکیورٹی کی بہترین مشقیں

```bicep
// Use managed identity
identity: {
  type: 'SystemAssigned'
}

// Private endpoints for AI services
properties: {
  publicNetworkAccess: 'Disabled'
  networkAcls: {
    defaultAction: 'Deny'
  }
}
```

## 💰 لاگت کی اصلاح

| حکمتِ عملی | بچت |
|----------|---------|
| زیرو تک اسکیل کریں (Container Apps) | 60-80% |
| ڈیولپمنٹ کے لیے کنزمپشن ٹائرز استعمال کریں | 50-70% |
| شیڈیولڈ اسکیلنگ | 30-50% |
| محفوظ شدہ گنجائش | 20-40% |

```bash
# بجٹ کے لیے انتباہات مقرر کریں
az consumption budget create \
  --budget-name "AI-Budget" \
  --amount 500 \
  --category Cost \
  --time-grain Monthly
```

## 📊 مانیٹرنگ سیٹ اپ

```bash
# لاگز اسٹریم کریں
azd monitor --logs

# Application Insights چیک کریں
azd monitor

# میٹرکس دیکھیں
az monitor metrics list --resource <resource-id>
```
