# باب 8: پروڈکشن اور انٹرپرائز پیٹرنز

**📚 کورس**: [AZD برائے ابتدائی](/../../README.md) | **⏱️ وقت**: 2-3 گھنٹے | **⭐ پیچیدگی**: اعلیٰ

---

## جائزہ

یہ باب انٹرپرائز کے لیے تیار ڈیپلائمنٹ پیٹرنز، سیکورٹی ہارڈننگ، مانیٹرنگ، اور پروڈکشن AI ورک لوڈز کے لیے لاگت کی بہتری پر مشتمل ہے۔

> `azd 1.23.12` کے تحت مارچ 2026 میں تصدیق شدہ۔

## سیکھنے کے اہداف

اس باب کو مکمل کرنے کے بعد، آپ:
- ملٹی ریجن مزاحم اطلاقات ڈپلائے کریں گے
- انٹرپرائز سیکورٹی پیٹرنز نافذ کریں گے
- جامع مانیٹرنگ کی ترتیب دیں گے
- بڑے پیمانے پر لاگت کو بہتر بنائیں گے
- AZD کے ساتھ CI/CD پائپ لائنز سیٹ اپ کریں گے

## 🚀 پروڈکشن چیک لسٹ

- [ ] مزاحمت کے لیے ملٹی ریجن ڈیپلائمنٹ
- [ ] تصدیق کے لیے منظم شناختی (کوئی کیز نہیں)
- [ ] مانیٹرنگ کے لیے اپلیکیشن انسائٹس
- [ ] لاگت کے بجٹ اور الرٹس کی ترتیب
- [ ] سیکورٹی اسکیننگ کو فعال کیا گیا
- [ ] CI/CD پائپ لائن انضمام
- [ ] آفات سے بحالی کا منصوبہ

## 🔐 سیکورٹی کی بہترین مشقیں

```bicep
// Use managed identity
identity: {
  type: 'SystemAssigned'
}

// Private endpoints for AI services
properties: {
  publicNetworkAccess: 'Disabled'
  networkAcls: {
    defaultAction: 'Deny'
  }
}
```

---

## 💰 لاگت کی بہتری

| حکمت عملی | بچت |
|----------|------|
| صفر کی طرف پیمائش (کنٹینر ایپس) | 60-80% |
| ترقی کے لیے کنزمپشن ٹیئرز استعمال کریں | 50-70% |
| شیڈول شدہ پیمائش | 30-50% |
| مخصوص گنجائش | 20-40% |

```bash
# بجٹ الارٹس مقرر کریں
az consumption budget create \
  --budget-name "AI-Budget" \
  --amount 500 \
  --category Cost \
  --time-grain Monthly
```

## 📊 مانیٹرنگ کی ترتیب

```bash
# لاگز کی سٹریمنگ کریں
azd monitor --logs

# ایپلیکیشن انسائٹس چیک کریں
azd monitor --overview

# میٹرکس دیکھیں
az monitor metrics list --resource <resource-id>
```
