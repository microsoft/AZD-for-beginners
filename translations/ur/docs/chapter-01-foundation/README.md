# باب 1: بنیاد اور جلد آغاز

**📚 کورس**: [آزاد برائے ابتدائی](../../README.md) | **⏱️ دورانیہ**: 30-45 منٹ | **⭐ پیچیدگی**: ابتدائی

---

## جائزہ

یہ باب Azure Developer CLI (azd) کے بنیادی اصول متعارف کرواتا ہے۔ آپ بنیادی تصورات سیکھیں گے، اوزار انسٹال کریں گے، اور اپنی پہلی ایپلیکیشن Azure پر تعینات کریں گے۔

> مارچ 2026 میں `azd 1.23.12` کے خلاف تصدیق شدہ۔

## سیکھنے کے مقاصد

اس باب کو مکمل کرکے، آپ:
- سمجھ جائیں گے کہ Azure Developer CLI کیا ہے اور یہ Azure CLI سے کیسے مختلف ہے
- اپنے پلیٹ فارم پر AZD انسٹال اور ترتیب دیں گے
- `azd up` کے ذریعے اپنی پہلی ایپلیکیشن Azure پر تعینات کریں گے
- `azd down` سے وسائل صاف کریں گے

## ✅ یہاں سے شروع کریں: اپنی ترتیب کی تصدیق کریں

شروع کرنے سے پہلے، تصدیق کریں کہ آپ کی مقامی مشین باب 1 کے سانچے کے لیے تیار ہے:

**ونڈوز:**
```powershell
.\validate-setup.ps1
```

**macOS / لینکس:**
```bash
bash ./validate-setup.sh
```

اگر اسکرپٹ میں کوئی اوزار غائب دکھائی دیں تو پہلے انہیں درست کریں اور پھر باب کے ساتھ جاری رکھیں۔

## 🚀 جلد آغاز

```bash
# تنصیب چیک کریں
azd version

# AZD کے لیے تصدیق کریں
# اختیاری: اگر آپ براہ راست Azure CLI کمانڈز چلانے کا ارادہ رکھتے ہیں تو az login کریں
azd auth login

# اپنی پہلی ایپ تعینات کریں
azd init --template todo-nodejs-mongo
azd up

# مکمل ہونے پر صفائی کریں
azd down --force --purge
```

---

## ✅ کامیابی کے معیارات

اس باب کو مکمل کرنے کے بعد، آپ کو درج ذیل کام انجام دینے کے قابل ہونا چاہیے:

```bash
azd version              # نصب شدہ ورژن دکھاتا ہے
azd init --template todo-nodejs-mongo  # پروجیکٹ کی ابتدا کرتا ہے
azd up                   # ایزور پر تعینات کرتا ہے
azd show                 # چلتی ہوئی ایپ کا یو آر ایل دکھاتا ہے
azd down --force --purge # وسائل کی صفائی کرتا ہے
```
