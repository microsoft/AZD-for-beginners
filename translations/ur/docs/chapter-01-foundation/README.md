# باب 1: بنیاد اور فوری آغاز

**📚 کورس**: [AZD برائے مبتدی](../../README.md) | **⏱️ دورانیہ**: 30-45 منٹ | **⭐ پیچیدگی**: مبتدی

---

## جائزہ

یہ باب Azure Developer CLI (azd) کی بنیادی باتوں کا تعارف کراتا ہے۔ آپ کو بنیادی تصورات سیکھنے کو ملیں گے، ٹولز انسٹال کریں گے، اور اپنا پہلا ایپلیکیشن Azure پر ڈیپلائے کریں گے۔

> `azd 1.25.6` کے خلاف جون 2026 میں تصدیق شدہ۔

## سیکھنے کے مقاصد

اس باب کو مکمل کرنے کے بعد، آپ:
- سمجھ سکیں گے کہ Azure Developer CLI کیا ہے اور یہ Azure CLI سے کیسے مختلف ہے
- اپنے پلیٹ فارم پر AZD انسٹال اور ترتیب دیں گے
- `azd up` کے ذریعے اپنا پہلا ایپلیکیشن Azure پر ڈیپلائے کریں گے
- `azd down` کے ذریعے وسائل کو صاف کریں گے

## ✅ یہاں سے شروع کریں: اپنا سیٹ اپ تصدیق کریں

شروع کرنے سے پہلے، تصدیق کریں کہ آپ کا مقامی کمپیوٹر باب 1 کے ٹیمپلیٹ کے لیے تیار ہے:

**ونڈوز:**
```powershell
.\validate-setup.ps1
```

**macOS / لینکس:**
```bash
bash ./validate-setup.sh
```

اگر اسکرپٹ آلات کی کمی کی اطلاع دیتا ہے، تو پہلے انہیں درست کریں اور پھر باب کے ساتھ جاری رکھیں۔

---

## 🚀 فوری آغاز

```bash
# تنصیب کی جانچ کریں
azd version

# AZD کے لیے تصدیق کریں
# اختیاری: اگر آپ Azure CLI کمانڈز براہ راست چلانے کا ارادہ رکھتے ہیں تو az login کریں
azd auth login

# اپنی پہلی ایپ تعینات کریں
azd init --template todo-nodejs-mongo
azd up

# ختم ہونے پر صفائی کریں
azd down --force --purge
```

---

## ✅ کامیابی کے معیار

اس باب کو مکمل کرنے کے بعد، آپ کو یہ کرنے کے قابل ہونا چاہیے:

```bash
azd version              # نصب شدہ ورژن دکھاتا ہے
azd init --template todo-nodejs-mongo  # پراجیکٹ شروع کرتا ہے
azd up                   # Azure پر تعینات کرتا ہے
azd show                 # چلنے والے ایپ کا URL دکھاتا ہے
azd down --force --purge # وسائل کو صاف کرتا ہے
```
