# باب 1: بنیاد اور فوری آغاز

**📚 کورس**: [ابتدائیوں کے لیے AZD](../../README.md) | **⏱️ دورانیہ**: 30-45 منٹ | **⭐ پیچیدگی**: ابتدائی

---

## جائزہ

یہ باب Azure Developer CLI (azd) کی بنیادی باتوں کا تعارف کراتا ہے۔ آپ بنیادی تصورات سیکھیں گے، ٹولز انسٹال کریں گے، اور اپنی پہلی ایپلیکیشن Azure پر تعینات کریں گے۔

> جولائی 2026 میں `azd 1.27.1` کے ساتھ تصدیق شدہ۔

## سیکھنے کے مقاصد

اس باب کو مکمل کرنے کے بعد، آپ:
- جان سکیں گے کہ Azure Developer CLI کیا ہے اور یہ Azure CLI سے کیسے مختلف ہے
- اپنے پلیٹ فارم پر AZD انسٹال اور ترتیب دیں گے
- `azd up` کے ذریعے اپنی پہلی ایپلیکیشن Azure پر تعینات کریں گے
- `azd down` کے ذریعے وسائل صاف کریں گے

## ✅ یہاں سے شروع کریں: اپنی ترتیب کی تصدیق کریں

شروع کرنے سے پہلے، تصدیق کریں کہ آپ کی مقامی مشین باب 1 کے ٹیمپلیٹ کے لیے تیار ہے:

**ونڈوز:**
```powershell
.\validate-setup.ps1
```

**macOS / لینکس:**
```bash
bash ./validate-setup.sh
```

اگر اسکرپٹ کوئی مفقود ٹولز بتائے، تو پہلے ان کو درست کریں اور پھر باب جاری رکھیں۔

---

## 🚀 فوری آغاز

```bash
# تنصیب کی جانچ کریں
azd version

# AZD کے لیے تصدیق کریں
# اختیاری: اگر آپ براہ راست Azure CLI کمانڈز چلانے کا ارادہ رکھتے ہیں تو az login کریں
azd auth login

# اپنی پہلی ایپ تعینات کریں
azd init --template todo-nodejs-mongo
azd up

# مکمل ہونے پر صفائی کریں
azd down --force --purge
```

---

## ✅ کامیابی کے معیارات

اس باب کو مکمل کرنے کے بعد، آپ کو درج ذیل کرنے کے قابل ہونا چاہیے:

```bash
azd version              # نصب شدہ ورژن دکھاتا ہے
azd init --template todo-nodejs-mongo  # پروجیکٹ کو شروع کرتا ہے
azd up                   # ایزور پر تعینات کرتا ہے
azd show                 # چلتے ہوئے ایپ کا یو آر ایل دکھاتا ہے
azd down --force --purge # وسائل کو صاف کرتا ہے
```
