# AZD برائے AI ڈویلپرز

!!! info "اس ورکشاپ کے لیے سامعین"
   
    یہ ورکشاپ Microsoft Foundry پر بہترین ٹولز اور ورک فلو کا استعمال کرتے ہوئے AI ایجنٹک ایپلی کیشنز بنانے والے کسی بھی شخص کے لیے ہے۔ یہ ورکشاپ ابتدائی اور تجربہ کار ڈویلپرز دونوں کے لیے موزوں ہے جنہیں Azure، AI تصورات اور Python کوڈ کی کچھ واقفیت ہے۔ براہ کرم شروع کرنے سے پہلے پری ریکوئزٹس چیک کریں۔

    **دورانیہ:** 3-4 گھنٹے  
    **سطح:** ابتدائی سے متوسط  

---


## پری ریکوئزٹس

- ایک ذاتی GitHub اکاؤنٹ → [ایک مفت بنائیں](https://github.com/signup)
- ایک فعال Azure اکاؤنٹ → [ایک مفت بنائیں](https://aka.ms/free)
- AI تصورات، Python ڈویلپمنٹ، کمانڈ لائن ٹولز سے واقفیت
- VS Code اور GitHub ٹولز اور ورک فلو کا تجربہ۔

---

## سیکھنے کے مقاصد

اس ورکشاپ کے اختتام تک، آپ جانیں گے کہ کیسے:

- [X] **Azure Developer CLI کے ساتھ کام کریں** → اپنے Azure وسائل کی پروویژننگ، تعیناتی، اور انتظام کے لیے۔
- [X] **AI ایپ ٹیمپلیٹس استعمال کریں** → Microsoft Foundry پر AI ایجنٹ پروجیکٹس کا آغاز کرنے کے لیے۔
- [X] **ٹیمپلیٹس کو کنفیگر کریں** → خصوصیات جیسے Red Teaming، Evaluations، Tracing & Monitoring کے لیے۔
- [X] **AZD ٹیمپلیٹس اور ورک فلو کو ڈی کنسٹرکٹ اور ایکسپلور کریں** → GitHub Copilot کی مدد سے۔
- [X] **AI ایجنٹ اسٹارٹر ٹیمپلیٹ کو اپنی مرضی کے مطابق بنائیں اور دوبارہ تعینات کریں** → اپنی کسٹم ضروریات کے ساتھ۔

---

## ورکشاپ کا خاکہ

ورکشاپ ماڈیولز کی شکل میں منظم ہے، ہر ایک میں ہینڈز آن لیب مشق شامل ہے:

1. **تعارف** - یہ سمجھیں کہ AZD ٹیمپلیٹس کیا ہیں اور کیسے کام کرتے ہیں
1. **انتخاب** - اپنے اگلے AI پروجیکٹ کے لیے مناسب "سٹارٹر" ٹیمپلیٹ دریافت کریں
1. **تصدیق** - ٹیمپلیٹ تعینات کریں اور اس کی جانچ کریں کہ یہ متوقع کام کرتا ہے
1. **ڈی کنسٹرکشن** - ٹیمپلیٹ کو کھول کر اس کے اجزاء اور کنفیگریشن کو سمجھیں
1. **کنفیگریشن** - Microsoft Foundry کی مختلف خصوصیات کو فعال کرنا اور استعمال کرنا سیکھیں
1. **کسٹمائزیشن** - ایپلیکیشن کو اپنے ڈیٹا اور ضروریات کے مطابق دوبارہ تعمیر کریں
1. **ٹیئر ڈاؤن** - انفراسٹرکچر کو ڈی پروویژن کرنا اور نرم حذف شدہ وسائل کو بحال کرنا سیکھیں

## تاثرات اور مسائل

ہم نصاب کو فعال طور پر اپ ڈیٹ کرتے ہیں تاکہ ابتدائیوں کے لیے بہترین سیکھنے کا تجربہ فراہم کیا جا سکے۔ آپ کی رائے کی قدر کی جاتی ہے۔

1. اس ریپو پر مسئلہ پوسٹ کریں - سہولت کے لیے اسے `Workshop` ٹیگ کریں۔
1. Microsoft Foundry Discord میں شامل ہوں - ساتھی سیکھنے والوں سے جڑیں!
