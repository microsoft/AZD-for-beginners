Azure Developer CLI (AZD) کو hands-on سیکھنے کے لیے خوش آمدید، خاص طور پر AI ایپلیکیشن کی تعیناتی کے سلسلے میں۔ یہ ورکشاپ آپ کو 3 مراحل میں AZD سانچوں کا عملی فہم حاصل کرنے میں مدد دیتی ہے:

1. **دریافت** - وہ سانچہ تلاش کریں جو آپ کے لیے موزوں ہو۔
1. **تعیناتی** - تعینات کریں اور توثیق کریں کہ یہ کام کرتا ہے۔
1. **حسب ضرورت سازی** - ترمیم کریں اور اسے اپنا بنائیں!

## ورکشاپ کا جائزہ

**دورانیہ:** 3-4 گھنٹے  
**سطح:** ابتدائی سے درمیانی  
**ضروریات:** Azure، AI تصورات، VS Code اور کمانڈ لائن ٹولز سے واقفیت۔

یہ ایک عملی ورکشاپ ہے جہاں آپ عمل کرتے ہوئے سیکھتے ہیں۔ مشقیں مکمل کرنے کے بعد، ہم تجویز کرتے ہیں کہ آپ AZD For Beginners نصاب کا جائزہ لیں تاکہ آپ اپنی سیکھنے کا سفر سیکورٹی اور پیداواری بہتری کے بہترین طریقوں میں جاری رکھ سکیں۔

| وقت| ماڈیول  | مقصد |
|:---|:---|:---|
| 15 منٹ | تعارف | پس منظر قائم کریں، مقاصد کو سمجھیں |
| 30 منٹ | AI سانچہ منتخب کریں | اختیارات دریافت کریں اور شروعاتی منتخب کریں | 
| 30 منٹ | AI سانچہ کی توثیق کریں | Azure پر ڈیفالٹ حل تعینات کریں |
| 30 منٹ | AI سانچہ کا تجزیہ کریں | ساخت اور ترتیب دریافت کریں |
| 30 منٹ | AI سانچہ ترتیب دیں | دستیاب خصوصیات کو فعال کریں اور آزمائیں |
| 30 منٹ | AI سانچہ حسبِ ضرورت بنائیں | سانچے کو اپنی ضروریات کے مطابق ڈھالیں |
| 30 منٹ | انفراسٹرکچر کو ختم کریں | صفائی کریں اور وسائل جاری کریں |
| 15 منٹ | اختتام اور اگلے مراحل | سیکھنے کے وسائل، ورکشاپ چیلنج |
| | |

## آپ کیا سیکھیں گے

AZD سانچے کو ایک لرننگ سینڈباکس کے طور پر سوچیں جہاں آپ مختلف صلاحیتوں اور ٹولز کو Microsoft Foundry پر اختتامی ترقی کے لیے دریافت کر سکتے ہیں۔ اس ورکشاپ کے اختتام تک، آپ کو اس سیاق و سباق میں مختلف ٹولز اور تصورات کا تخیلاتی فہم حاصل ہونا چاہیے۔

| تصور  | مقصد |
|:---|:---|
| **Azure Developer CLI** | ٹول کمانڈز اور ورک فلو سمجھیں |
| **AZD سانچے**| پراجیکٹ کی ساخت اور ترتیب سمجھیں |
| **Azure AI Agent**| Microsoft Foundry پروجیکٹ کی فراہمی اور تعیناتی |
| **Azure AI Search**| ایجنٹس کے ساتھ سیاق و سباق کی انجینئرنگ کو فعال کریں |
| **مشاہدہ کاری**| ٹریسنگ، مانیٹرنگ اور تجزیہ دریافت کریں |
| **ریڈ ٹیمنگ**| مخالفانہ ٹیسٹنگ اور کمیات دریافت کریں |
| | |
