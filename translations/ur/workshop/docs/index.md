# AZD برائے AI ڈیولپرز ورکشاپ

> جون 2026 میں `azd 1.25.6` کے خلاف تصدیق شدہ۔

> [!IMPORTANT]  
> **یہ ورکشاپ ایک ورکشاپ گائیڈ کے ساتھ مرتب کی گئی ہے جسے آپ اپنے مقامی براؤزر میں پیش نظارہ کر سکتے ہیں۔ شروع کرنے کے لیے، ریپو کو GitHub Codespaces، ایک ڈویلپمنٹ کنٹینر، یا MkDocs انسٹال شدہ لوکل کلون میں کھولیں—پھر اس وقت تک انتظار کریں جب تک آپ کو ایک فعال VS Code ٹرمینل نظر آ جائے اور ٹائپ کریں:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **آپ کو براؤزر میں پیش نظارہ صفحہ کھولنے کے لیے ایک پاپ اپ ڈائیلاگ نظر آنا چاہیے۔**

Azure Developer CLI (AZD) سیکھنے کے لیے ہاتھوں سے سیکھنے والی ورکشاپ میں خوش آمدید جس کی توجہ AI ایپلیکیشن کی تعیناتی پر ہے۔ یہ ورکشاپ آپ کو AZD ٹیمپلیٹس کی عملی سمجھ تین مراحل میں فراہم کرتی ہے:

1. **دریافت** - اپنے لیے موزوں ٹیمپلیٹ تلاش کریں۔
1. **تعییناتی** - تعینات کریں اور اس کی کارکردگی کی تصدیق کریں۔
1. **حسب ضرورت** - ترمیم کریں اور اسے اپنا بنائیں!

اس ورکشاپ کے دوران، آپ کو بنیادی ڈویلپر آلات اور ورک فلو سے بھی آگاہ کیا جائے گا، تاکہ آپ کی انتہا سے انتہا ترقیاتی سفر کو آسان بنایا جا سکے۔

## ورکشاپ کا جائزہ

**دورانیہ:** 3-4 گھنٹے  
**سطح:** ابتدائی سے درمیانی  
**پیشگی ضروریات:** Azure، AI تصورات، VS Code اور کمانڈ لائن ٹولز سے واقفیت۔

یہ ایک ہاتھوں سے سیکھنے والی ورکشاپ ہے جہاں آپ عمل کے ذریعے سیکھتے ہیں۔ مشقیں مکمل کرنے کے بعد، ہم آپ کو AZD برائے مبتدی نصاب کا جائزہ لینے کی سفارش کرتے ہیں تاکہ آپ اپنی سیکھنے کا سلسلہ سیکیورٹی اور پیداوری کی بہترین مشقوں کی طرف جاری رکھ سکیں۔

| وقت| ماڈیول  | مقصد |
|:---|:---|:---|
| 15 منٹ | تعارف | میدان ترتیب دیں، اہداف سمجھیں |
| 30 منٹ | AI ٹیمپلیٹ منتخب کریں | اختیارات کا جائزہ لیں اور آغاز کریں | 
| 30 منٹ | AI ٹیمپلیٹ کی تصدیق کریں | ڈیفالٹ حل Azure پر تعینات کریں |
| 30 منٹ | AI ٹیمپلیٹ کو تجزیہ کریں | ساخت اور ترتیب کو سمجھیں |
| 30 منٹ | AI ٹیمپلیٹ کی ترتیب دیں | دستیاب خصوصیات کو فعال کریں اور آزمائیں |
| 30 منٹ | AI ٹیمپلیٹ کو حسب ضرورت بنائیں | ٹیمپلیٹ کو اپنی ضروریات کے مطابق بنائیں |
| 30 منٹ | انفراسٹرکچر کو ختم کریں | صفائی کریں اور وسائل کو ریلیز کریں |
| 15 منٹ | اختتام اور اگلے مراحل | تعلیمی وسائل، ورکشاپ چیلنج |
| | |

## آپ کیا سیکھیں گے

AZD ٹیمپلیٹ کو ایک تعلیمی سینڈباکس کے طور پر سوچیں تاکہ آپ Microsoft Foundry پر انتہا سے انتہا ترقی کے مختلف صلاحیتوں اور آلات کو دریافت کر سکیں۔ اس ورکشاپ کے اختتام تک، آپ کو اس سیاق و سباق میں مختلف آلات اور تصورات کی ایک فطری سمجھ حاصل ہو جانی چاہیے۔

| تصور  | مقصد |
|:---|:---|
| **Azure Developer CLI** | ٹول کے احکامات اور ورک فلو کو سمجھیں |
| **AZD ٹیمپلیٹس**| پروجیکٹ کی ساخت اور ترتیب کو سمجھیں |
| **Azure AI Agent**| Microsoft Foundry پروجیکٹ کی فراہمی اور تعیناتی  |
| **Azure AI Search**| ایجنٹس کے ساتھ کانٹیکسٹ انجینئرنگ فعال کریں |
| **مشاہدہ پذیری**| ٹریسنگ، نگرانی اور جائزے دریافت کریں |
| **ریڈ ٹیمینگ**| حریفانہ ٹیسٹنگ اور حفاظتی تدابیر دریافت کریں |
| | |
