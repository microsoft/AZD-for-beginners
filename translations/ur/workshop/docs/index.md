# AI ڈویلپرز کے لیے AZD ورکشاپ

> جولائی 2026 میں `azd 1.27.1` کے خلاف تصدیق شدہ۔

> [!IMPORTANT]  
> **یہ ورکشاپ ایک ورکشاپ گائیڈ کے ساتھ آراستہ ہے جسے آپ اپنے مقامی براؤزر میں دیکھ سکتے ہیں۔ شروع کرنے کے لیے، رپوزٹری کو GitHub Codespaces، ایک ڈیولپمنٹ کنٹینر، یا MkDocs انسٹال شدہ مقامی کلون میں کھولیں—پھر تب تک انتظار کریں جب تک کہ آپ فعال VS کوڈ ٹرمینل نہ دیکھیں اور پھر ٹائپ کریں:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **آپ کو ایک پاپ اپ ڈائیلاگ دیکھنا چاہیے جو براؤزر میں پریویو صفحہ کھولنے کا کہے گا۔**

Azure Developer CLI (AZD) سیکھنے کے لیے ہینڈز آن ورکشاپ میں خوش آمدید، جس میں AI ایپلیکیشن تعیناتی پر خاص توجہ دی گئی ہے۔ یہ ورکشاپ آپ کو AZD ٹمپلٹس کی عملی سمجھ 3 مراحل میں حاصل کرنے میں مدد دیتی ہے:

1. **دریافت** - ایسا ٹیمپلیٹ تلاش کریں جو آپ کے لیے مناسب ہو۔
1. **تعیناتی** - تعینات کریں اور تصدیق کریں کہ یہ کام کرتا ہے
1. **حسب ضرورت** - ترمیم کریں اور دوہراتے ہوئے اسے اپنا بنائیں!

اس ورکشاپ کے دوران، آپ کو بنیادی ڈویلپر ٹولز اور ورک فلو سے بھی متعارف کرایا جائے گا، تاکہ آپ کی پورے عمل کی ترقی کے سفر کو آسان بنایا جا سکے۔

## ورکشاپ کا جائزہ

**دورانیہ:** 3-4 گھنٹے  
**سطح:** ابتدائی سے درمیانی  
**ضروریات:** Azure، AI تصورات، VS Code اور کمانڈ لائن ٹولز سے واقفیت۔

یہ ایک عملی ورکشاپ ہے جہاں آپ کرتے ہوئے سیکھتے ہیں۔ جب آپ مشقیں مکمل کر لیں، تو ہم AZD For Beginners نصاب کا جائزہ لینے کی تجویز دیتے ہیں تاکہ آپ اپنی سیکھنے کی راہ کو سیکیورٹی اور پیداواری بہترین مشقوں کی طرف بڑھائیں۔

| وقت | ماڈیول  | مقصد |
|:---|:---|:---|
| 15 منٹ | تعارف | ماحولیاتی تعین کریں، اہداف سمجھیں |
| 30 منٹ | AI ٹیمپلیٹ منتخب کریں | اختیارات دریافت کریں اور آغاز کریں | 
| 30 منٹ | AI ٹیمپلیٹ کی توثیق کریں | Azure پر ڈیفالٹ حل تعینات کریں |
| 30 منٹ | AI ٹیمپلیٹ کو ڈی کانسٹرکٹ کریں | ساخت اور ترتیب دریافت کریں |
| 30 منٹ | AI ٹیمپلیٹ کو ترتیب دیں | دستیاب خصوصیات کو فعال کریں اور آزمائیں |
| 30 منٹ | AI ٹیمپلیٹ کو حسب ضرورت بنائیں | ٹیمپلیٹ کو اپنی ضرورت کے مطابق ڈھالیں |
| 30 منٹ | انفراسٹرکچر کو ختم کریں | صفائی کریں اور وسائل جاری کریں |
| 15 منٹ | اختتامی کلمات اور اگلے اقدامات | تعلیمی وسائل، ورکشاپ چیلنج |
| | |

## آپ کیا سیکھیں گے

AZD ٹیمپلیٹ کو ایک تعلیمی سیمپل باکس کے طور پر سوچیں جہاں آپ Microsoft Foundry پر شروع سے آخر تک ترقی کے مختلف صلاحیتوں اور ٹولز کو دریافت کر سکتے ہیں۔ اس ورکشاپ کے اختتام پر، آپ کو اس سیاق و سباق میں مختلف ٹولز اور تصورات کا فطری احساس حاصل ہوگا۔

| تصور  | مقصد |
|:---|:---|
| **Azure Developer CLI** | ٹول کمانڈز اور ورک فلو سمجھیں|
| **AZD ٹیمپلیٹس**| پروجیکٹ کی ساخت اور ترتیب سمجھیں|
| **Azure AI Agent**| Microsoft Foundry پروجیکٹ کی فراہمی اور تعیناتی  |
| **Azure AI Search**| ایجنٹس کے ساتھ سیاق و سباق کی انجینئرنگ فعال کریں |
| **مشاہدہ کاری**| ٹریسنگ، مانیٹرنگ اور جانچ پڑتال کو دریافت کریں |
| **ریڈ ٹیمنگ**| مخالفانہ جانچ اور بچاؤ کے طریقے دریافت کریں |
| | |
