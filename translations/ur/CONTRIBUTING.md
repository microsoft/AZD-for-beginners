# AZD For Beginners میں تعاون

AZD For Beginners کورس میں تعاون کے لیے آپ کی دلچسپی کا شکریہ! یہ دستاویز تعاون کرنے والوں کے لیے رہنما اصول اور معلومات فراہم کرتی ہے۔

## تعاون کیسے کریں

### مسائل کی اطلاع دینا

اگر آپ کو بگز، غلطیاں یا بہتری کے لیے تجاویز ملیں تو:

1. **موجودہ مسائل چیک کریں**: ڈپلیکیٹس سے بچنے کے لیے [موجودہ مسائل](https://github.com/microsoft/azd-for-beginners/issues) تلاش کریں
2. **نیا مسئلہ بنائیں**: مناسب issue ٹیمپلیٹ استعمال کریں
3. **تفصیلات فراہم کریں**: دوبارہ پیدا کرنے کے مراحل، متوقع رویہ، اور جہاں موزوں ہو سکرین شاٹس شامل کریں

### تبدیلیاں تجویز کرنا

1. **ریپوزیٹری کو فورک کریں**: پروجیکٹ کا اپنا فورک بنائیں
2. **برانچ بنائیں**: اپنی تبدیلیوں کے لیے ایک فیچر برانچ بنائیں
3. **تبدیلیاں کریں**: ہمارے کوڈنگ اور دستاویزی معیارات پر عمل کریں
4. **اپنی تبدیلیوں کا ٹیسٹ کریں**: تصدیق کریں کہ تمام لنکس کام کرتے ہیں اور مواد درست طور پر رینڈر ہوتا ہے
5. **پل ریکویسٹ جمع کروائیں**: اپنی تبدیلیوں کی واضح وضاحت فراہم کریں

### تعاون کی اقسام

ہم درج ذیل قسم کے تعاون کا خیرمقدم کرتے ہیں:

#### مواد میں بہتری

- ٹائپوز، قواعدِ زبان، اور فارمیٹنگ کے مسائل درست کریں
- موجودہ دستاویزات کی وضاحت بہتر کریں
- قدیم ہو چکی معلومات کو اپ ڈیٹ کریں
- غائب وضاحتیں یا سیاق و سباق شامل کریں

#### نئی مثالیں

- حقیقی دنیا کے تعیناتی منظرنامے شامل کریں
- عام استعمال کے کیسز کے لیے نئے AZD ٹیمپلیٹس بنائیں
- اضافی کوڈ مثالیں اور مشقیں فراہم کریں

#### ترجمے کی معاونت

- موجودہ تراجم کی دیکھ بھال میں مدد کریں
- ترجمے کے معیار کو بہتر کریں
- نئے مواد کے لیے تراجم شامل کریں

#### خرابیوں کی رپورٹیں

- ٹوٹے ہوئے لنکس یا غلط کوڈ مثالیں رپورٹ کریں
- دستاویزات میں تضادات کی نشاندہی کریں
- رسائی سے متعلق مسائل کی نشان دہی کریں

## تعاون کے رہنما اصول

### دستاویزی معیارات

- صاف، شروعاتی سطح کے لیے دوستانہ زبان استعمال کریں
- سیکھنے کے مقاصد اور نتائج شامل کریں
- متوقع نتائج کے ساتھ مرحلہ وار ہدایات فراہم کریں
- متعلقہ مواد کے لیے نیویگیشن لنکس شامل کریں
- یکساں فارمیٹنگ اور مارک ڈاؤن انداز استعمال کریں

### کوڈ مثالیں

- یقینی بنائیں کہ تمام کوڈ مثالیں ٹیسٹ شدہ اور کام کر رہی ہوں
- اہم تصورات کی وضاحت کرنے والے تبصرے شامل کریں
- Azure اور AZD کی بہترین عملیاتی طریقہ کار کی پیروی کریں
- جہاں مناسب ہو متوقع آؤٹ پٹ فراہم کریں

### کمیٹ پیغامات

واضح اور تفصیلی کمیٹ پیغامات استعمال کریں:
- `docs: fix typo in installation guide`
- `feat: add new container app example`
- `fix: correct broken link in chapter 2`
