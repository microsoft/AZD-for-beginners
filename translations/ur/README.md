## کورس کا جائزہ

Azure Developer CLI (azd) کو منظم ابواب کے ذریعے سیکھیں جو ترقی پسند تعلیم کے لیے ڈیزائن کیے گئے ہیں۔ **Azure AI Foundry انضمام کے ساتھ AI ایپلیکیشن کی تعیناتی پر خصوصی توجہ۔**

### یہ کورس جدید ڈویلپرز کے لیے کیوں ضروری ہے؟

Azure AI Foundry Discord کمیونٹی کی بصیرت کی بنیاد پر، **45% ڈویلپرز AZD کو AI ورک لوڈز کے لیے استعمال کرنا چاہتے ہیں** لیکن درج ذیل چیلنجز کا سامنا کرتے ہیں:
- پیچیدہ کثیر سروس AI آرکیٹیکچرز  
- پروڈکشن AI تعیناتی کے بہترین طریقے  
- Azure AI سروس انضمام اور ترتیب  
- AI ورک لوڈز کے لیے لاگت کی اصلاح  
- AI مخصوص تعیناتی مسائل کا حل  

### تعلیمی مقاصد

اس منظم کورس کو مکمل کرنے کے بعد، آپ:
- **AZD بنیادی اصولوں میں مہارت حاصل کریں گے**: بنیادی تصورات، انسٹالیشن، اور ترتیب  
- **AI ایپلیکیشنز تعینات کریں گے**: AZD کو Azure AI Foundry سروسز کے ساتھ استعمال کریں  
- **انفراسٹرکچر کوڈ کے طور پر نافذ کریں گے**: Bicep ٹیمپلیٹس کے ساتھ Azure وسائل کا انتظام کریں  
- **تعیناتی کے مسائل حل کریں گے**: عام مسائل کو حل کریں اور ڈیبگ کریں  
- **پروڈکشن کے لیے اصلاح کریں گے**: سیکیورٹی، اسکیلنگ، مانیٹرنگ، اور لاگت کا انتظام  
- **کثیر ایجنٹ حل بنائیں گے**: پیچیدہ AI آرکیٹیکچرز تعینات کریں

## 📖 Azure Developer CLI کیا ہے؟

Azure Developer CLI (azd) ایک ڈویلپر مرکوز کمانڈ لائن انٹرفیس ہے جو Azure پر ایپلیکیشنز بنانے اور تعینات کرنے کے عمل کو تیز کرتا ہے۔ یہ فراہم کرتا ہے:

- **ٹیمپلیٹ پر مبنی تعیناتی** - عام ایپلیکیشن پیٹرنز کے لیے پہلے سے تیار کردہ ٹیمپلیٹس استعمال کریں
- **کوڈ کے طور پر انفراسٹرکچر** - Bicep یا Terraform کے ذریعے Azure وسائل کا انتظام کریں  
- **مربوط ورک فلو** - ایپلیکیشنز کو بغیر کسی رکاوٹ کے فراہم کریں، تعینات کریں، اور نگرانی کریں
- **ڈویلپر دوستانہ** - ڈویلپر کی پیداواریت اور تجربے کے لیے بہتر بنایا گیا

### **AZD + Azure AI Foundry: AI تعیناتی کے لیے بہترین**

**AI حل کے لیے AZD کیوں؟** AZD ان چیلنجز کو حل کرتا ہے جن کا سامنا AI ڈویلپرز کو ہوتا ہے:

- **AI کے لیے تیار ٹیمپلیٹس** - Azure OpenAI، Cognitive Services، اور ML ورک لوڈز کے لیے پہلے سے ترتیب شدہ ٹیمپلیٹس
- **محفوظ AI تعیناتی** - AI سروسز، API کیز، اور ماڈل اینڈ پوائنٹس کے لیے بلٹ ان سیکیورٹی پیٹرنز  
- **پروڈکشن AI پیٹرنز** - قابل پیمائش، کم خرچ AI ایپلیکیشن تعیناتی کے لیے بہترین طریقے
- **شروع سے آخر تک AI ورک فلو** - ماڈل کی ترقی سے لے کر پروڈکشن تعیناتی تک مناسب نگرانی کے ساتھ
- **خرچ کی اصلاح** - AI ورک لوڈز کے لیے سمارٹ وسائل کی تقسیم اور اسکیلنگ حکمت عملی
