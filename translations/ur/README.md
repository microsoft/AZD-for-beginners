## کورس کا جائزہ

Azure Developer CLI (azd) کو منظم ابواب کے ذریعے سیکھیں جو ترقی پسند تعلیم کے لیے ڈیزائن کیے گئے ہیں۔ **Azure AI Foundry انضمام کے ساتھ AI ایپلیکیشن کی تعیناتی پر خصوصی توجہ۔**

### یہ کورس جدید ڈویلپرز کے لیے کیوں ضروری ہے؟

Azure AI Foundry Discord کمیونٹی کی بصیرت کے مطابق، **45% ڈویلپرز AZD کو AI ورک لوڈز کے لیے استعمال کرنا چاہتے ہیں** لیکن درج ذیل چیلنجز کا سامنا کرتے ہیں:
- پیچیدہ کثیر سروس AI آرکیٹیکچرز
- پروڈکشن AI تعیناتی کے بہترین طریقے  
- Azure AI سروس انضمام اور ترتیب
- AI ورک لوڈز کے لیے لاگت کی اصلاح
- AI مخصوص تعیناتی کے مسائل کا حل

### تعلیمی مقاصد

اس منظم کورس کو مکمل کرنے کے بعد، آپ:
- **AZD کے بنیادی اصولوں میں مہارت حاصل کریں گے**: بنیادی تصورات، تنصیب، اور ترتیب
- **AI ایپلیکیشنز تعینات کریں گے**: AZD کو Azure AI Foundry خدمات کے ساتھ استعمال کریں
- **Infrastructure as Code نافذ کریں گے**: Bicep ٹیمپلیٹس کے ساتھ Azure وسائل کا انتظام کریں
- **تعیناتی کے مسائل حل کریں گے**: عام مسائل کو حل کریں اور ڈیبگ کریں
- **پروڈکشن کے لیے اصلاح کریں گے**: سیکیورٹی، اسکیلنگ، مانیٹرنگ، اور لاگت کا انتظام
- **کثیر ایجنٹ حل بنائیں گے**: پیچیدہ AI آرکیٹیکچرز تعینات کریں

### 🚀 باب 1: بنیاد اور فوری آغاز
**ضروریات**: Azure سبسکرپشن، بنیادی کمانڈ لائن علم  
**دورانیہ**: 30-45 منٹ  
**پیچیدگی**: ⭐

#### آپ کیا سیکھیں گے
- Azure Developer CLI کے بنیادی اصولوں کو سمجھنا
- اپنے پلیٹ فارم پر AZD انسٹال کرنا
- اپنی پہلی کامیاب تعیناتی

#### تعلیمی وسائل
- **🎯 یہاں سے شروع کریں**: [Azure Developer CLI کیا ہے؟](../..)
- **📖 نظریہ**: [AZD بنیادیات](docs/getting-started/azd-basics.md) - بنیادی تصورات اور اصطلاحات
- **⚙️ سیٹ اپ**: [تنصیب اور سیٹ اپ](docs/getting-started/installation.md) - پلیٹ فارم کے لیے مخصوص گائیڈز
- **🛠️ عملی کام**: [آپ کا پہلا پروجیکٹ](docs/getting-started/first-project.md) - مرحلہ وار ٹیوٹوریل
- **📋 فوری حوالہ**: [کمانڈ چیٹ شیٹ](resources/cheat-sheet.md)

#### عملی مشقیں
```bash
# Quick installation check
azd version

# Deploy your first application
azd init --template todo-nodejs-mongo
azd up
```

**💡 باب کا نتیجہ**: AZD کا استعمال کرتے ہوئے Azure پر ایک سادہ ویب ایپلیکیشن کامیابی سے تعینات کریں

1. **دریافت کا مرحلہ** (45 منٹ)

   - ملٹی ایجنٹ آرکیٹیکچر کے نمونوں کو سمجھیں
   - ڈیپلائمنٹ کی ضروریات اور پیشگی شرائط کا جائزہ لیں

2. **ڈیپلائمنٹ کا مرحلہ** (2 گھنٹے)
   - AZD کے ساتھ AI ایپلیکیشنز کی عملی ڈیپلائمنٹ

## 📖 Azure Developer CLI کیا ہے؟

Azure Developer CLI (azd) ایک ڈیولپر مرکوز کمانڈ لائن انٹرفیس ہے جو Azure پر ایپلیکیشنز بنانے اور ڈیپلائمنٹ کے عمل کو تیز کرتا ہے۔ یہ فراہم کرتا ہے:

- **ٹیمپلیٹ پر مبنی ڈیپلائمنٹ** - عام ایپلیکیشن نمونوں کے لیے پہلے سے تیار کردہ ٹیمپلیٹس استعمال کریں
- **انفراسٹرکچر بطور کوڈ** - Bicep یا Terraform کا استعمال کرتے ہوئے Azure وسائل کا انتظام کریں  
- **مربوط ورک فلو** - ایپلیکیشنز کو آسانی سے پروویژن، ڈیپلائمنٹ، اور مانیٹر کریں
- **ڈیولپر دوستانہ** - ڈیولپر کی پیداواریت اور تجربے کے لیے بہتر بنایا گیا

### **AZD + Azure AI Foundry: AI ڈیپلائمنٹ کے لیے بہترین**

**AI حل کے لیے AZD کیوں؟** AZD ان چیلنجز کو حل کرتا ہے جو AI ڈیولپرز کو درپیش ہیں:

- **AI کے لیے تیار ٹیمپلیٹس** - Azure OpenAI، Cognitive Services، اور ML ورک لوڈز کے لیے پہلے سے کنفیگرڈ ٹیمپلیٹس
- **محفوظ AI ڈیپلائمنٹ** - AI سروسز، API کیز، اور ماڈل اینڈ پوائنٹس کے لیے بلٹ ان سیکیورٹی نمونے  
- **پروڈکشن AI نمونے** - اسکیل ایبل، لاگت مؤثر AI ایپلیکیشن ڈیپلائمنٹ کے بہترین طریقے
- **اینڈ ٹو اینڈ AI ورک فلو** - ماڈل ڈیولپمنٹ سے پروڈکشن ڈیپلائمنٹ تک مناسب مانیٹرنگ کے ساتھ
- **لاگت کی اصلاح** - AI ورک لوڈز کے لیے سمارٹ وسائل کی تقسیم اور اسکیلنگ کی حکمت عملیاں
- **Azure AI Foundry انٹیگریشن** - AI Foundry ماڈل کیٹلاگ اور اینڈ پوائنٹس سے آسان کنکشن
