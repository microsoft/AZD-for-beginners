## تعارف

Azure Developer CLI (azd) کے لیے جامع گائیڈ میں خوش آمدید۔ یہ ریپوزٹری طلباء سے لے کر پیشہ ور ڈویلپرز تک ہر سطح کے ڈویلپرز کو Azure Developer CLI سیکھنے اور کلاؤڈ ڈپلائمنٹ کو مؤثر طریقے سے انجام دینے میں مدد فراہم کرنے کے لیے ڈیزائن کی گئی ہے۔ یہ منظم تعلیمی وسائل Azure کلاؤڈ ڈپلائمنٹ، عام مسائل کے حل، اور کامیاب AZD ٹیمپلیٹ ڈپلائمنٹ کے لیے بہترین طریقوں کو عملی طور پر سیکھنے کا موقع فراہم کرتے ہیں۔

## سیکھنے کے اہداف

اس ریپوزٹری پر کام کرتے ہوئے آپ:
- Azure Developer CLI کے بنیادی اصولوں اور اہم تصورات میں مہارت حاصل کریں گے  
- انفراسٹرکچر کوڈ کے ذریعے Azure وسائل کو ڈپلائے اور پروویژن کرنا سیکھیں گے  
- AZD ڈپلائمنٹ کے عام مسائل کے حل کے لیے مہارت حاصل کریں گے  
- پری ڈپلائمنٹ ویلیڈیشن اور صلاحیت کی منصوبہ بندی کو سمجھیں گے  
- سیکیورٹی کے بہترین طریقے اور لاگت کی اصلاح کی حکمت عملی نافذ کریں گے  
- Azure پر پروڈکشن کے لیے تیار ایپلیکیشنز کو ڈپلائے کرنے میں اعتماد حاصل کریں گے  

## سیکھنے کے نتائج

اس کورس کو مکمل کرنے کے بعد آپ:
- Azure Developer CLI کو کامیابی سے انسٹال، کنفیگر، اور استعمال کر سکیں گے  
- AZD ٹیمپلیٹس کا استعمال کرتے ہوئے ایپلیکیشنز بنا اور ڈپلائے کر سکیں گے  
- تصدیق، انفراسٹرکچر، اور ڈپلائمنٹ کے مسائل کو حل کر سکیں گے  
- پری ڈپلائمنٹ چیکس جیسے صلاحیت کی منصوبہ بندی اور SKU کا انتخاب انجام دے سکیں گے  
- مانیٹرنگ، سیکیورٹی، اور لاگت کے انتظام کے بہترین طریقے نافذ کر سکیں گے  
- AZD ورک فلو کو CI/CD پائپ لائنز میں ضم کر سکیں گے

## Azure Developer CLI کیا ہے؟

Azure Developer CLI (azd) ایک ڈویلپر مرکوز کمانڈ لائن انٹرفیس ہے جو Azure پر ایپلیکیشنز بنانے اور ڈپلائے کرنے کے عمل کو تیز کرتا ہے۔ یہ فراہم کرتا ہے:

- **ٹیمپلیٹ پر مبنی ڈپلائمنٹ** - عام ایپلیکیشن پیٹرنز کے لیے پہلے سے تیار کردہ ٹیمپلیٹس کا استعمال  
- **انفراسٹرکچر کوڈ کے طور پر** - Bicep یا Terraform کے ذریعے Azure وسائل کا انتظام  
- **مربوط ورک فلو** - ایپلیکیشنز کو پروویژن، ڈپلائے، اور مانیٹر کرنے کا آسان طریقہ  
- **ڈویلپر دوستانہ** - ڈویلپر کی پیداواریت اور تجربے کے لیے بہتر بنایا گیا  

## جلدی شروع کریں

### ضروریات
- Azure سبسکرپشن  
- Azure CLI انسٹال شدہ  
- Git (ٹیمپلیٹس کو کلون کرنے کے لیے)  

### انسٹالیشن
```bash
# Windows (PowerShell)
powershell -ex AllSigned -c "Invoke-RestMethod 'https://aka.ms/install-azd.ps1' | Invoke-Expression"

# macOS/Linux
curl -fsSL https://aka.ms/install-azd.sh | bash
```  

### آپ کا پہلا ڈپلائمنٹ
```bash
# Initialize a new project
azd init --template todo-nodejs-mongo

# Provision Azure resources and deploy
azd up
```
