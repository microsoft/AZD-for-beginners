> Validirano s `azd 1.25.6` u lipnju 2026.

> [!IMPORTANT]  
> **Ova radionica ima vodič koji možete pregledati u svom lokalnom pregledniku. Za početak, otvorite repozitorij u GitHub Codespaces, razvojnom kontejneru ili lokalnoj kopiji s instaliranim MkDocs—zatim pričekajte dok ne vidite aktivni VS Code terminal i upišite:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Trebali biste vidjeti iskačući dijalog za otvaranje pregledne stranice u pregledniku.**

Dobrodošli u praktičnu radionicu za učenje Azure Developer CLI (AZD) s fokusom na implementaciju AI aplikacija. Ova radionica pomaže vam steći primijenjeno razumijevanje AZD predložaka kroz 3 koraka:

1. **Otkriće** - pronađite predložak koji vam odgovara.  
1. **Implementacija** - implementirajte i provjerite da radi  
1. **Prilagodba** - izmijenite i iterirajte kako bi bio vaš!

Tijekom ove radionice također ćete biti upoznati s osnovnim alatima za programere i radnim tokovima, kako biste pojednostavili svoj cjelokupni razvojni proces.

## Pregled radionice

**Trajanje:** 3-4 sata  
**Razina:** Početna do Srednja  
**Preduvjeti:** Poznavanje Azure, AI koncepata, VS Code i alata naredbenog retka.

Ovo je praktična radionica u kojoj učite kroz rad. Nakon dovršetka zadataka, preporučujemo pregled kurikuluma AZD za početnike kako biste nastavili svoje učenje o sigurnosnim i produktivnim najboljim praksama.

| Vrijeme| Modul  | Cilj |
|:---|:---|:---|
| 15 min | Uvod | Postaviti okvir, razumjeti ciljeve |
| 30 min | Odabir AI predloška | Istražiti opcije i odabrati početni |
| 30 min | Validacija AI predloška | Implementirati zadano rješenje na Azure |
| 30 min | Rastavljanje AI predloška | Istražiti strukturu i konfiguraciju |
| 30 min | Konfiguracija AI predloška | Aktivirati i isprobati dostupne značajke |
| 30 min | Prilagodba AI predloška | Prilagoditi predložak vašim potrebama |
| 30 min | Rastavljanje infrastrukture | Očistiti i osloboditi resurse |
| 15 min | Zaključak i sljedeći koraci | Resursi za učenje, izazov radionice |
| | |

## Što ćete naučiti

Pomislite na AZD predložak kao na sandbox za učenje u kojem istražujete razne mogućnosti i alate za cjelokupni razvoj na Microsoft Foundry platformi. Na kraju ove radionice trebali biste intuitivno razumjeti različite alate i koncepte u ovom kontekstu.

| Koncept  | Cilj |
|:---|:---|
| **Azure Developer CLI** | Razumjeti naredbe alata i radne tokove |
| **AZD predlošci**| Razumjeti strukturu projekta i konfiguraciju |
| **Azure AI agent**| Postaviti i implementirati Microsoft Foundry projekt |
| **Azure AI Search**| Omogućiti kontekstualni inženjering s agentima |
| **Promatranje**| Istražiti praćenje, nadzor i evaluacije |
| **Red Teaming**| Istražiti testiranje suparničkih napada i zaštite |
| | |
