# AZD za AI Developere Radionica

> [!IMPORTANT]  
> **Ova radionica je opremljena vodičem za radionicu koji možete pregledati u svom lokalnom pregledniku. Za početak, morate pokrenuti GitHub Codespaces na repozitoriju — zatim pričekajte dok ne vidite aktivni VS Code terminal i upišite:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Trebali biste vidjeti skočni dijalog za otvaranje stranice za pregled u pregledniku.**

Dobrodošli u praktičnu radionicu za učenje Azure Developer CLI-a (AZD) s naglaskom na implementaciju AI aplikacija. Ova radionica pomaže vam steći praktično razumijevanje AZD predložaka u 3 koraka:

1. **Otkrivanje** - pronađite predložak koji vam odgovara.
1. **Implementacija** - implementirajte i provjerite radi li.
1. **Prilagodba** - izmijenite i iterirajte kako biste ga napravili svojim!

Tijekom ove radionice također ćete biti upoznati s osnovnim razvojnim alatima i radnim procesima, koji će vam pomoći da pojednostavite svoj razvojni put od početka do kraja.

## Pregled radionice

**Trajanje:** 3-4 sata  
**Razina:** Početnik do srednje naprednog  
**Preduvjeti:** Poznavanje Azure, AI koncepata, VS Code i alata naredbenog retka.

Ovo je praktična radionica gdje učite radeći. Nakon što završite vježbe, preporučujemo pregledavanje nastavnog plana AZD za početnike kako biste nastavili svoje učenje o sigurnosnim i produktivnim najboljim praksama.

| Vrijeme| Modul  | Cilj |
|:---|:---|:---|
| 15 min | Uvod | Postaviti temelje, razumjeti ciljeve |
| 30 min | Odabir AI predloška | Istražiti opcije i odabrati početni |
| 30 min | Validacija AI predloška | Implementirati zadano rješenje na Azure |
| 30 min | Rastavljanje AI predloška | Istražiti strukturu i konfiguraciju |
| 30 min | Konfiguracija AI predloška | Aktivirati i isprobati dostupne značajke |
| 30 min | Prilagodba AI predloška | Prilagoditi predložak svojim potrebama |
| 30 min | Uklanjanje infrastrukture | Očistiti i osloboditi resurse |
| 15 min | Završetak i sljedeći koraci | Resursi za učenje, izazov radionice |
| | |

## Što ćete naučiti

Razmislite o AZD predlošku kao o kutiji za učenje za istraživanje različitih mogućnosti i alata za razvoj od početka do kraja na Microsoft Foundry. Do kraja ove radionice trebali biste steći intuitivan osjećaj za različite alate i koncepte u ovom kontekstu.

| Koncept  | Cilj |
|:---|:---|
| **Azure Developer CLI** | Razumjeti naredbe alata i radne procese |
| **AZD predlošci**| Razumjeti strukturu projekta i konfiguraciju |
| **Azure AI Agent**| Provisionirati i implementirati Microsoft Foundry projekt |
| **Azure AI Search**| Omogućiti kontekstualno inženjerstvo s agentima |
| **Pratljivost**| Istražiti praćenje, nadzor i evaluacije |
| **Red Teaming**| Istražiti protivničko testiranje i mjere ublažavanja |
| | |
