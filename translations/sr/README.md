## Увод

Добродошли у свеобухватни водич за Azure Developer CLI (azd). Овај репозиторијум је осмишљен да помогне програмерима свих нивоа, од студената до професионалних програмера, да науче и савладају Azure Developer CLI за ефикасно коришћење облака, **са посебним фокусом на примену AI апликација користећи Azure AI Foundry**. Овај структурирани ресурс за учење пружа практично искуство са Azure облачним решењима, решавањем уобичајених проблема и применом најбољих пракси за успешну имплементацију AZD шаблона.

### **Зашто је овај водич важан за AI програмере**
На основу недавне анкете у Azure AI Foundry Discord заједници, **45% програмера је заинтересовано за коришћење AZD за AI радне задатке**, али се суочавају са изазовима као што су:  
- Комплексне AI архитектуре са више услуга  
- Најбоље праксе за примену AI у продукцији  
- Интеграција и конфигурација Azure AI услуга  
- Оптимизација трошкова за AI радне задатке  
- Решавање проблема специфичних за AI примену  

## Циљеви учења

Радећи кроз овај репозиторијум, научићете:  
- Основе и кључне концепте Azure Developer CLI  
- Како да примените и обезбедите Azure ресурсе користећи Infrastructure as Code  
- Вештине решавања уобичајених проблема са AZD применом  
- Валидацију пре примене и планирање капацитета  
- Имплементацију најбољих пракси за безбедност и оптимизацију трошкова  
- Како да стекнете самопоуздање у примени апликација спремних за продукцију на Azure  

## Резултати учења

Након завршетка овог курса, бићете у могућности да:  
- Успешно инсталирате, конфигуришете и користите Azure Developer CLI  
- Креирате и примените апликације користећи AZD шаблоне  
- Решавате проблеме са аутентификацијом, инфраструктуром и применом  
- Спроведете провере пре примене, укључујући планирање капацитета и избор SKU-а  
- Имплементирате најбоље праксе за мониторинг, безбедност и управљање трошковима  
- Интегришете AZD радне токове у CI/CD процесе

## Шта је Azure Developer CLI?

Azure Developer CLI (azd) је командна линија усмерена на програмере која убрзава процес креирања и примене апликација на Azure. Пружа:  

- **Примена заснована на шаблонима** - Користите унапред припремљене шаблоне за уобичајене обрасце апликација  
- **Infrastructure as Code** - Управљајте Azure ресурсима користећи Bicep или Terraform  
- **Интегрисани радни токови** - Лако обезбеђујте, примењујте и надгледајте апликације  
- **Прилагођено програмерима** - Оптимизовано за продуктивност и искуство програмера  

### **AZD + Azure AI Foundry: Савршено за AI примене**

**Зашто AZD за AI решења?** AZD решава главне изазове са којима се суочавају AI програмери:  

- **Шаблони спремни за AI** - Унапред конфигурисани шаблони за Azure OpenAI, Cognitive Services и ML радне задатке  
- **Безбедна AI примена** - Уграђени безбедносни обрасци за AI услуге, API кључеве и крајње тачке модела  
- **Обрасци за AI у продукцији** - Најбоље праксе за скалабилну и економичну примену AI апликација  
- **Крај-до-крај AI радни токови** - Од развоја модела до примене у продукцији уз правилно надгледање  
- **Оптимизација трошкова** - Паметне стратегије за алокацију ресурса и скалирање AI радних задатака  
- **Интеграција са Azure AI Foundry** - Лако повезивање са каталогом модела и крајњим тачкама AI Foundry  

## Брзи почетак

### Предуслови  
- Azure претплата  
- Инсталиран Azure CLI  
- Git (за клонирање шаблона)  

### Инсталација  
```bash
# Windows (PowerShell)
powershell -ex AllSigned -c "Invoke-RestMethod 'https://aka.ms/install-azd.ps1' | Invoke-Expression"

# macOS/Linux
curl -fsSL https://aka.ms/install-azd.sh | bash
```  

### Ваша прва примена  
```bash
# Initialize a new project
azd init --template todo-nodejs-mongo

# Provision Azure resources and deploy
azd up
```  

### Ваша прва AI примена  
```bash
# Initialize an AI-powered chat application with Azure OpenAI
azd init --template azure-search-openai-demo

# Configure AI services and deploy
azd up

# Or try other AI templates:
azd init --template openai-chat-app-quickstart
azd init --template ai-document-processing
azd init --template contoso-chat
```
