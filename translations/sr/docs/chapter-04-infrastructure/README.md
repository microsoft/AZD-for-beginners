# Поглавље 4: Инфраструктура као код и распоређивање

**📚 Курс**: [AZD за почетнике](../../README.md) | **⏱️ Трајање**: 1-1.5 сати | **⭐ Комплексност**: Средњи

---

## Преглед

Ово поглавље обухвата обрасце Инфраструктуре као кода (IaC) са Bicep шаблонима, провизију ресурса и стратегије распоређивања помоћу Azure Developer CLI.

## Циљеви учења

Након завршетка овог поглавља, научићете:
- Разумети структуру и синтаксу Bicep шаблона
- Обезбедити Azure ресурсе помоћу `azd provision`
- Распоредити апликације помоћу `azd deploy`
- Имплементирати blue-green и rolling стратегије распоређивања

## 🚀 Брзи почетак

```bash
# Иницијализуј из шаблона
azd init --template azure-functions-python-v2-http

# Прегледај шта ће бити креирано
azd provision --preview

# Обезбеди само инфраструктуру
azd provision

# Деплојирај само код
azd deploy

# Или оба заједно
azd up
```

---

## 📁 Структура AZD пројекта

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

---

## 🔧 Основне команде

| Команда | Опис |
|---------|-------------|
| `azd init` | Иницијализујте пројекат |
| `azd provision` | Креирајте Azure ресурсе |
| `azd deploy` | Деплоујте код апликације |
| `azd up` | Поставите инфраструктуру и деплоујте апликацију |
| `azd down` | Избришите све ресурсе |
