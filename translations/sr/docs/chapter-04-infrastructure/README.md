# Глава 4: Инфраструктура као код и распоређивање

**📚 Курс**: [AZD за почетнике](../../README.md) | **⏱️ Трајање**: 1-1.5 hours | **⭐ Сложеност**: Средњи

---

## Преглед

Ова глава покрива обрасце Infrastructure as Code (IaC) са Bicep шаблонима, провизионисање ресурса и стратегије распоређивања уз помоћ Azure Developer CLI.

> Проверено са `azd 1.23.12` у марту 2026.

## Циљеви учења

Провршивши ову главу, бићете у стању да:
- Разумете структуру и синтаксу Bicep шаблона
- Креирате Azure ресурсе помоћу `azd provision`
- Деплојујете апликације помоћу `azd deploy`
- Примените blue-green и rolling стратегије распоређивања

## 🚀 Брзи почетак

```bash
# Иницијализујте из шаблона
azd init --template azure-functions-python-v2-http

# Прегледајте шта ће бити креирано
azd provision --preview

# Обезбедите само инфраструктуру
azd provision

# Деплојирајте само код
azd deploy

# Или обоје заједно
azd up
```

## 📁 Структура AZD пројекта

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

---

## 🔧 Основне команде

| Команда | Опис |
|---------|-------------|
| `azd init` | Иницијализује пројекат |
| `azd provision` | Креира Azure ресурсе |
| `azd deploy` | Деплојира код апликације |
| `azd up` | провизионисање + деплој |
| `azd down` | Обрише све ресурсе |
