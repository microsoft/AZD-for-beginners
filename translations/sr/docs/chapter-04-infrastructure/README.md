# Поглавље 4: Инфраструктура као код и деплојмент

**📚 Курс**: [AZD За Почетнике](../../README.md) | **⏱️ Трајање**: 1-1.5 сати | **⭐ Комплексност**: Средњи ниво

---

## Преглед

Ово поглавље покрива обрасце Инфраструктуре као кода (IaC) са Бицеп шаблонима, провизију ресурса и стратегије деплојмента користећи Azure Developer CLI.

> Проверено са `azd 1.27.1` у јулу 2026.

## Циљеви учења

Завршетком овог поглавља, научићете:
- Разумевање структуре и синтаксе Бицеп шаблона
- Провизију Azure ресурса користећи `azd provision`
- Деплојмент апликација са `azd deploy`
- Имплементацију blue-green и rolling стратегија деплојмента

## 🚀 Брзи Почетак

```bash
# Иницијализуј из шаблона
azd init --template azure-functions-python-v2-http

# Прегледај шта ће бити креирано
azd provision --preview

# Овде само обезбеђуј инфраструктуру
azd provision

# Само разврстај код
azd deploy

# Или обoje заједно
azd up
```

## 📁 Структура AZD Пројекта

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

---

## 🔧 Основне Команде

| Команда | Опис |
|---------|-------------|
| `azd init` | Иницијализација пројекта |
| `azd provision` | Креирање Azure ресурса |
| `azd deploy` | Деплојмент кода апликације |
| `azd up` | провизија + деплојмент |
| `azd down` | Брисање свих ресурса |
