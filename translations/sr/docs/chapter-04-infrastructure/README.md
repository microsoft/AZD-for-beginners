# Поглавље 4: Инфраструктура као код и распоређивање

**📚 Course**: [AZD For Beginners](../../README.md) | **⏱️ Duration**: 1-1.5 сати | **⭐ Complexity**: Средњи

---

## Преглед

Ово поглавље покрива обрасце Инфраструктуре као кода (IaC) са Bicep шаблонима, провизионисање ресурса и стратегије распоређивања коришћењем Azure Developer CLI.

> Верификовано са `azd 1.25.6` у јуну 2026.

## Циљеви учења

Кроз завршетак овог поглавља ћете:
- Разумети структуру и синтаксу Bicep шаблона
- Обезбедити Azure ресурсе помоћу `azd provision`
- Распоредити апликације помоћу `azd deploy`
- Имплементирати blue-green и rolling стратегије распоређивања

## 🚀 Брзи почетак

```bash
# Иницијализуј из шаблона
azd init --template azure-functions-python-v2-http

# Прегледај шта ће бити креирано
azd provision --preview

# Само обезбеди инфраструктуру
azd provision

# Само постави код
azd deploy

# Или оба заједно
azd up
```

## 📁 Структура AZD пројекта

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

## 🔧 Основне команде

| Команда | Опис |
|---------|-------------|
| `azd init` | Иницијализовати пројекат |
| `azd provision` | Креирати Azure ресурсе |
| `azd deploy` | Распоредити код апликације |
| `azd up` | провизионисање + распоређивање |
| `azd down` | Избрисати све ресурсе |
