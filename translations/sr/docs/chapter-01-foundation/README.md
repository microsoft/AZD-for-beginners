# Поглавље 1: Основа & Брз почетак

**📚 Курс**: [AZD За почетнике](../../README.md) | **⏱️ Трајање**: 30-45 минута | **⭐ Комплексност**: Почетник

## Преглед

Ово поглавље представља основе Azure Developer CLI (azd). Научићете кључне концепте, инсталирати алате и поставити своју прву апликацију на Azure.

> Потврђено против `azd 1.27.1` у јулу 2026.

## Циљеви учења

Завршетком овог поглавља ћете:
- Разумети шта је Azure Developer CLI и како се разликује од Azure CLI
- Инсталирати и конфигурисати AZD на вашој платформи
- Поставити своју прву апликацију на Azure помоћу `azd up`
- Очистити ресурсе помоћу `azd down`

## ✅ Почните овде: Потврдите своју поставку

Пре него што почнете, потврдите да је ваш локални рачунар спреман за шаблон Поглавља 1:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

Ако скрипта пријави да недостају алати, прво решите то, па наставите са поглављем.

---

## 🚀 Брз почетак

```bash
# Проверите инсталацију
azd version

# Аутентификујте се за AZD
# Опционо: покрените az login ако планирате да директно извршавате Azure CLI команде
azd auth login

# Деплојујте своју прву апликацију
azd init --template todo-nodejs-mongo
azd up

# Очистите када завршите
azd down --force --purge
```

---

## ✅ Критеријуми успеха

Након завршетка овог поглавља, требало би да будете у могућности да:

```bash
azd version              # Проказује инсталирану верзију
azd init --template todo-nodejs-mongo  # Иницијализује пројекат
azd up                   # Деплојује на Азуре
azd show                 # Приказује УРЛ активне апликације
azd down --force --purge # Чисти ресурсе
```
