# Поглавље 1: Основа & Брзи почетак

**📚 Course**: [AZD за почетнике](../../README.md) | **⏱️ Duration**: 30–45 минута | **⭐ Complexity**: Почетнички

---

## Преглед

Ово поглавље уводи основe Azure Developer CLI (azd). Упознаћете основне концепте, инсталирати алате и поставити вашу прву апликацију на Azure.

> Проверено са `azd 1.25.6` у јуну 2026.

## Циљеви учења

Поред завршетка овог поглавља, требало би да:
- Разумете шта је Azure Developer CLI и како се разликује од Azure CLI
- Инсталирате и конфигуришете AZD на вашој платформи
- Распоредите вашу прву апликацију на Azure помоћу `azd up`
- Очистите ресурсе помоћу `azd down`

## ✅ Почните овде: Потврдите своје подешавање

Пре него што почнете, потврдите да је ваш локални рачунар спреман за шаблон Поглавља 1:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

Ако скрипта пријави недостајуће алате, најпре их исправите, а затим наставите са поглављем.

---

## 🚀 Брзи почетак

```bash
# Проверите инсталацију
azd version

# Аутентификујте се за AZD
# Опционо: az login ако планирате да директно покрећете Azure CLI команде
azd auth login

# Разместите своју прву апликацију
azd init --template todo-nodejs-mongo
azd up

# Очистите када завршите
azd down --force --purge
```

---

## ✅ Критеријуми успеха

Након завршетка овог поглавља, требало би да будете у стању да:

```bash
azd version              # Приказује инсталирану верзију
azd init --template todo-nodejs-mongo  # Иницијализује пројекат
azd up                   # Размешта на Azure
azd show                 # Приказује URL покренуте апликације
azd down --force --purge # Очишћава ресурсе
```
