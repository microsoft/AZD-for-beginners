# Поглавље 1: Основе и брзи почетак

**📚 Курс**: [AZD For Beginners](../../README.md) | **⏱️ Трајање**: 30-45 минута | **⭐ Сложеност**: Почетнички

---

## Преглед

Ово поглавље представља основе Azure Developer CLI (azd). У научићете основне појмове, како да инсталирате алате и како да разместите вашу прву апликацију на Azure.

## Циљеви учења

Након завршетка овог поглавља, бићете у стању да:
- Разумете шта је Azure Developer CLI и како се разликује од Azure CLI
- Инсталирате и конфигуришете AZD на вашој платформи
- Разместите своју прву апликацију на Azure помоћу `azd up`
- Очистите/уклоните ресурсе помоћу `azd down`

## 🚀 Брзи почетак

```bash
# Проверите инсталацију
azd version

# Пријавите се у Azure
azd auth login

# Разместите своју прву апликацију
azd init --template todo-nodejs-mongo
azd up

# Очистите када завршите
azd down --force --purge
```

---

## ✅ Критеријуми успеха

Након завршетка овог поглавља, требало би да будете у стању да:

```bash
azd version              # Приказује инсталирану верзију
azd init --template todo-nodejs-mongo  # Иницијализује пројекат
azd up                   # Распоређује на Azure
azd show                 # Приказује URL покренуте апликације
azd down --force --purge # Уклања ресурсе
```
