# Поглавље 1: Основе и брзи почетак

**📚 Course**: [AZD For Beginners](../../README.md) | **⏱️ Duration**: 30-45 minutes | **⭐ Complexity**: Beginner

## Преглед

Ово поглавље уводи основне појмове Azure Developer CLI (azd). Учићете основне концепте, инсталирати алате и разместити вашу прву апликацију на Azure.

> Проверено са `azd 1.23.12` у марту 2026.

## Циљеви учења

Завршетком овог поглавља, ви ћете:
- Разумети шта је Azure Developer CLI и како се разликује од Azure CLI
- Инсталирати и конфигурисати AZD на вашој платформи
- Разместити вашу прву апликацију на Azure помоћу `azd up`
- Очистити ресурсе помоћу `azd down`

## ✅ Почните овде: Потврдите своје подешавање

Пре него што почнете, уверите се да је ваш локални рачунар спреман за шаблон Поглавља 1:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

Ако скрипта пријави да недостају алати, прво то исправите, а затим наставите са поглављем.

---

## 🚀 Брзи почетак

```bash
# Проверите инсталацију
azd version

# Аутентификујте се за AZD
# Опционо: покрените az login ако планирате да директно покрећете Azure CLI команде
azd auth login

# Разместите вашу прву апликацију
azd init --template todo-nodejs-mongo
azd up

# Очистите ресурсе када завршите
azd down --force --purge
```

---

## ✅ Критеријуми успеха

Након завршетка овог поглавља, требало би да будете у стању да:

```bash
azd version              # Приказује инсталирану верзију
azd init --template todo-nodejs-mongo  # Иницијализује пројекат
azd up                   # Распоређује на Azure
azd show                 # Приказује URL покренуте апликације
azd down --force --purge # Чисти ресурсе
```
