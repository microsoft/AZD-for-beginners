# Поглавље 3: Конфигурација и аутентификација

**📚 Курс**: [AZD за почетнике](../../README.md) | **⏱️ Трајање**: 45-60 minutes | **⭐ Сложеност**: Средњи

## Преглед

Ово поглавље покрива конфигурацију окружења, обрасце аутентификације и најбоље безбедносне праксе за Azure Developer CLI деплоје.

> Потврђено са `azd 1.23.12` у марту 2026.

## Циљеви учења

Комплетирањем овог поглавља, ви ћете:
- Савладати хијерархију конфигурације AZD
- Управљати више окружења (dev, staging, prod)
- Имплементирати безбедну аутентификацију помоћу управљаних идентитета
- Конфигурисати поставке специфичне за окружење

## 🚀 Брзи почетак

```bash
# Креирајте више окружења
azd env new dev
azd env new staging
azd env new prod

# Пређите између окружења
azd env select prod

# Поставите променљиве окружења
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# Погледајте конфигурацију
azd env get-values
```

---

## 🔧 Хијерархија конфигурације

AZD примењује подешавања у овом редоследу (каснија преовладавају над ранијима):

1. **Подразумеване вредности** (уграђено у шаблоне)
2. **azure.yaml** (конфигурација пројекта)
3. **Променљиве окружења** (`azd env set`)
4. **Параметри командне линије** (`--location eastus`)

---

## 🔐 Најбоље праксе безбедности

```bash
# Користите управљани идентитет (препоручено)
azd env set AZURE_USE_MANAGED_IDENTITY true

# Проверите статус аутентификације AZD
azd auth status

# Опционо: проверите Azure CLI контекст ако планирате да извршавате az команде
az account show

# Поново се аутентификујте ако је потребно
azd auth login

# Опционо: освежите Azure CLI аутентификацију за az команде
az login
```
