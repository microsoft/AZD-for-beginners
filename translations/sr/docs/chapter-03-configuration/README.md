# Поглавље 3: Конфигурација и аутентикација

**📚 Курс**: [AZD за почетнике](../../README.md) | **⏱️ Трајање**: 45-60 минута | **⭐ Комплексност**: Средњи ниво

---

## Преглед

Ово поглавље покрива конфигурацију окружења, обрасце аутентикације и најбоље безбедносне праксе за Azure Developer CLI примене.

> Верификовано са `azd 1.27.1` у јулу 2026.

## Циљеви учења

Завршетком овог поглавља ћете:
- Савладати AZD хијерархију конфигурације
- Управљати са више окружења (dev, staging, prod)
- Имплементирати сигурну аутентикацију са управљаним идентитетима
- Конфигурисати подешавања специфична за окружење

## 🚀 Брзи почетак

```bash
# Креирајте више окружења
azd env new dev
azd env new staging
azd env new prod

# Пребацивање окружења
azd env select prod

# Поставите променљиве окружења
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# Прикажи конфигурацију
azd env get-values
```

---

## 🔧 Хијерархија конфигурације

AZD примењује подешавања овим редоследом (каснија преузимају раније):

1. **Подразумеване вредности** (уграђене у шаблоне)
2. **azure.yaml** (конфигурација пројекта)
3. **Променљиве окружења** (`azd env set`)
4. **Параметри командне линије** (`--location eastus`)

---

## 🔐 Најбоље безбедносне праксе

```bash
# Користите управљани идентитет (препоручено)
azd env set AZURE_USE_MANAGED_IDENTITY true

# Проверите статус аутентификације AZD-а
azd auth status

# Опционо: проверите Azure CLI контекст ако планирате да покренете az команде
az account show

# Поново се аутентификујте ако је потребно
azd auth login

# Опционо: освежите Azure CLI аутентификацију за az команде
az login
```
