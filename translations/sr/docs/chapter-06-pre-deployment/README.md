# Поглавље 6: Планирање и валидација пре деплоја

**📚 Курс**: [AZD За почетнике](../../README.md) | **⏱️ Трајање**: 1 сат | **⭐ Комплексност**: Средњи ниво

---

## Преглед

Ово поглавље покрива кључне кораке планирања и валидације пре деплоја ваше апликације. Научите како да избегнете скупе грешке уз правилно планирање капацитета, избор SKU-а и пре-деплојну проверу.

> Валидација извршена са `azd 1.27.1` у јулу 2026.

## Циљеви учења

Завршетком овог поглавља ћете:
- Извршити пре-деплојне провере
- Планирати капацитет и проценити захтеве за ресурсима
- Изабрати одговарајуће SKU-ове за оптимизацију трошкова
- Конфигурисати Application Insights за праћење
- Разумети обрасце координације тима

## 🚀 Брзи почетак

```bash
# Провери квоте претплате
az vm list-usage --location eastus --output table

# Преглед распоређивања (без креирања ресурса)
azd provision --preview

# Валидација Bicep синтаксе
az bicep build --file infra/main.bicep

# Провери конфигурацију окружења
azd env get-values
```

## ☑️ Контролна листа пре деплоја

### Пре `azd provision`

- [ ] Квота проверена за регион
- [ ] Правилан избор SKU-а
- [ ] Преглед процене трошкова
- [ ] Конзистентна конвенција именовања
- [ ] Безбедност/RBAC конфигурисано

### Пре `azd deploy`

- [ ] Постављене промењиве окружења
- [ ] Тајне у Key Vault-у
- [ ] Провера конекцијских стрингова
- [ ] Конфигурисане провере здравља

## 💰 Водич за избор SKU-а

| Радно оптерећење | Развој | Продукција |
|----------|-------------|------------|
| Container Apps | Потрошња | Посвећено D4 |
| App Service | B1/B2 | P1v3+ |
| Microsoft Foundry Models | Стандард | Стандард + PTU |
| AI Search | Основни | Стандард S2+ |
