- Имплементирати комуникацију између агената

## 🎯 Истакнуто решење: Решење за малопродају са више агената

[Решење за малопродају са више агената](../../examples/retail-scenario.md) демонстрира:

- **Агент за кориснике**: Обрађује интеракције са корисницима и преференције
- **Агент за залихе**: Управља залихама и обрадом поруџбина
- **Оркестратор**: Координише рад агената
- **Заједничка меморија**: Управљање контекстом између агената

### Коришћене услуге

| Услуга | Намена |
|---------|---------|
| Azure OpenAI | Разумевање језика |
| Azure AI Search | Каталог производа |
| Cosmos DB | Стање и меморија агента |
| Container Apps | Хостинг агената |
| Application Insights | Надгледање |
