# Поглавље 7: Решавање проблема и отклањање грешака

**📚 Курс**: [AZD За почетнике](../../README.md) | **⏱️ Трајање**: 1-1.5 сати | **⭐ Комплексност**: Средњи ниво

## Преглед

Ово поглавље вам помаже да дијагностикујете и решите уобичајене проблеме приликом рада са Azure Developer CLI-јем. Од неуспеха у развоју до проблема специфичних за AI.

> Верификовано на `azd 1.27.1` у јулу 2026.

## Циљеви учења

Завршетком овог поглавља ћете:
- Дијагностификовати уобичајене неуспехе у развоју AZD-а
- Отклањати проблеме са аутентификацијом и дозволама
- Решавати проблеме везане за повезивање AI сервиса
- Коришћити Azure Portal и CLI за решавање проблема

## 🚨 Брза решења

### Проблеми са аутентификацијом
```bash
# Захтевано за AZD токове рада
azd auth login

# Опционо ако такође директно користите Azure CLI наредбе
az login

azd auth status
```

### Неуспеси у припреми (Provisioning)
```bash
azd show
azd monitor --logs
az deployment sub list --query "[?properties.provisioningState!='Succeeded']"
```

### Конфликти ресурса
```bash
azd down --force --purge
azd env new different-name
azd up
```

### Прекорачење квоте
```bash
az vm list-usage --location eastus --output table
azd env set AZURE_LOCATION westus2
azd up
```

---

## 📋 Референца кодова грешака

| Грешка | Узрок | Решење |
|-------|-------|----------|
| `AuthenticationError` | Није пријављен/аутентификован | `azd auth login` |
| `ResourceNotFound` | Недостаје ресурс | Проверите имена ресурса |
| `QuotaExceeded` | Ограничења претплате | Затражите повећање квоте |
| `InvalidTemplate` | Грешка у Bicep синтакси | `az bicep build` |
| `Conflict` | Ресурс већ постоји | Користите ново име или обришите постојећи |
| `Forbidden` | Недовољне дозволе | Проверите RBAC улоге |

## 🔄 Ресетовање и опоравак

```bash
# Софт ресет (чувај ресурсе, поново распореди код)
azd deploy --force

# Хард ресет (обриши све, почни изнова)
azd down --force --purge
azd up
```
