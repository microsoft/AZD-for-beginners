# Поглавље 7: Решавање проблема и отклањање грешака

**📚 Курс**: [AZD за почетнике](../../README.md) | **⏱️ Трајање**: 1-1.5 сати | **⭐ Сложеност**: Средњи

---

## Преглед

Ово поглавље вам помаже да дијагностикујете и решите уобичајене проблеме при раду са Azure Developer CLI. Од неуспеха при размењавању до проблема специфичних за вештачку интелигенцију.

> Проверено са `azd 1.23.12` у марту 2026.

## Циљеви учења

Завршетком овог поглавља, моћи ћете:
- Дијагностикујете уобичајене неуспехе при размештању помоћу AZD
- Отклањате грешке у аутентикацији и проблеме са дозволама
- Решавате проблеме везе са сервисима вештачке интелигенције
- Користите Azure портал и CLI за решавање проблема

## 🚨 Брзи попravci

### Проблеми са аутентификацијом
```bash
# Потребно за AZD радне токове
azd auth login

# Опционо ако такође директно користите Azure CLI команде
az login

azd auth status
```

### Неуспеси при провизионисању
```bash
azd show
azd monitor --logs
az deployment sub list --query "[?properties.provisioningState!='Succeeded']"
```

### Конфликти ресурса
```bash
azd down --force --purge
azd env new different-name
azd up
```

### Прекорачење квоте
```bash
az vm list-usage --location eastus --output table
azd env set AZURE_LOCATION westus2
azd up
```

## 📋 Референца кодова грешака

| Грешка | Узрок | Решење |
|-------|-------|----------|
| `AuthenticationError` | Нисте пријављени | `azd auth login` |
| `ResourceNotFound` | Недостаје ресурс | Проверите имена ресурса |
| `QuotaExceeded` | Ограничења претплате | Захтевајте повећање квоте |
| `InvalidTemplate` | Синтаксна грешка у Bicep | `az bicep build` |
| `Conflict` | Ресурс већ постоји | Користите ново име или обришите |
| `Forbidden` | Недовољне дозволе | Проверите RBAC улоге |

## 🔄 Ресетовање и опоравак

```bash
# Меки ресет (задржати ресурсе, поново распоредити код)
azd deploy --force

# Хард ресет (избриши све, почни изнова)
azd down --force --purge
azd up
```
