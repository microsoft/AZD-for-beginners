# AZD за AI програмере

!!! info "НАМЕЊЕНОСТ ОВЕ РАДИОНИЦЕ"
   
    Ова радионица је намењена свима који желе да граде AI агентне апликације на Microsoft Foundry користећи најбоље алате и токове рада. Радионица је погодна и за почетнике и за искуcне програмере који имају неко познавање Azure-а, AI концепата и Python кода. Молимо вас да проверите предуслове пре него што почнете.

    **Трајање:** 3-4 часа  
    **Ниво:** Почетни до средњег  

---


## Предуслови

- Лични GitHub налог → [Create one for free](https://github.com/signup)
- Активан Azure налог → [Create one for free](https://aka.ms/free)
- Познавање AI концепата, развоја у Python-у, алата командне линије
- Искуство са VS Code и GitHub алатима и токовима рада.

---

## Циљеви учења

До краја ове радионице, знаћете како да:

- [X] **Рад са Azure Developer CLI** → да провизионишете, размештате и управљате вашим Azure ресурсима.
- [X] **Користите AI App шаблоне** → да брзо започнете AI Agent пројекте на Microsoft Foundry.
- [X] **Конфигурисање шаблона** → за функције као што су Red Teaming, Evaluations, Tracing & Monitoring
- [X] **Деконструисати и истражити** → AZD шаблоне и токове рада користећи GitHub Copilot
- [X] **Прилагодити и поново распоредити** → почетни AI Agent шаблон са вашим прилагођеним захтевима

---

## Садржај радионице

Радионица је подељена на модуле, при чему је сваки модул заснован на практичној лабораторијској вежби:

1. **Увод** - Разумевање шта су AZD шаблони, и како функционишу
1. **Избор** - Откријте прави "почетни" шаблон за ваш следећи AI пројекат
1. **Валидација** - Размештите и истражите шаблон да бисте проверили да ли ради како је описано
1. **Деконструкција** - Разложите шаблон да бисте разумели његове компоненте & конфигурацију
1. **Конфигурација** - Научите да активирате и користите различите Microsoft Foundry функције
1. **Прилагођавање** - Реконструишите апликацију имајући у виду ваше податке и захтеве
1. **Теардаун** - Научите како да де-провизионишете инфраструктуру и повратите ресурсe који су soft-deleted
