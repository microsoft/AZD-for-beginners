## 2. Ključni koncepti, ki ste jih osvojili

### Azure Developer CLI (azd)
- `azd init` - Inicializira projekte iz predlog
- `azd up` - Z enim ukazom zagotovi vire in namesti
- `azd env` - Upravlja spremenljivke okolja
- `azd down` - Varno počisti vire

### AZD Template Structure
- `azure.yaml` - Konfiguracija in hooks za namestitev
- `infra/` - Infrastruktura kot koda (Bicep)
- `src/` - Izvorna koda aplikacije
- `scripts/` - Skripti za avtomatizacijo in nastavitev

### Microsoft Foundry Capabilities
- Storitev AI agentov za inteligentne asistente
- Iskanje datotek in Azure AI Search za pridobivanje znanja
- Vgrajeno sledenje, spremljanje in ocenjevanje
- Red teaming za preverjanje varnosti AI
