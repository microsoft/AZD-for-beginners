# Розділ 2: Розробка з акцентом на ШІ

**📚 Курс**: [AZD Для Початківців](../../README.md) | **⏱️ Час**: 1-2 години | **⭐ Складність**: Середній рівень

---

## Огляд

Цей розділ зосереджений на розгортанні додатків з підтримкою ШІ за допомогою Azure Developer CLI та сервісів Microsoft Foundry. Від простих чат-додатків із ШІ до інтелектуальних агентів з інструментами.

> **Примітка щодо валідації (2026-06-15):** Потік команд та інструкції щодо розширень у цьому розділі були перевірені на відповідність `azd` `1.25.6` та поточному прев’ю розширенню AI агента `azure.ai.agents` `0.1.40-preview`. Якщо у вас стара версія AZD, оновіть її перед виконанням вправ.

## Цілі навчання

Виконавши цей розділ, ви:
- Розгорнете ШІ-додатки за допомогою готових шаблонів AZD
- Зрозумієте інтеграцію Microsoft Foundry з AZD
- Налаштуєте і кастомізуєте ШІ-агентів з інструментами
- Розгорнете додатки RAG (Retrieval-Augmented Generation)

## 🚀 Швидкий старт

```bash
# Варіант 1: Додаток для чатів RAG
azd init --template azure-search-openai-demo
azd up

# Варіант 2: AI Агенти
azd init --template get-started-with-ai-agents
azd up

# Варіант 3: Швидкий чат-додаток
azd init --template openai-chat-app-quickstart
azd up
```
