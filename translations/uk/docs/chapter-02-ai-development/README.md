# Розділ 2: Розробка з акцентом на ШІ

**📚 Курс**: [AZD Для початківців](../../README.md) | **⏱️ Тривалість**: 1-2 години | **⭐ Складність**: Середній рівень

---

## Огляд

Цей розділ зосереджений на розгортанні додатків із підтримкою ШІ за допомогою Azure Developer CLI та сервісів Microsoft Foundry. Від простих чат-додатків із ШІ до інтелектуальних агентів із інструментами.

> **Примітка щодо перевірки (2026-03-25):** Потік команд та вказівки для розширень у цьому розділі було перевірено на відповідність `azd` версії `1.23.12` та поточній прев’ю-версії розширення AI агентів `azure.ai.agents` `0.1.18-preview`. Якщо у вас стара версія AZD, оновіть її перед тим, як переходити до вправ.

## Цілі навчання

Виконавши цей розділ, ви зможете:
- Розгортати ШІ-додатки з використанням готових шаблонів AZD
- Розуміти інтеграцію Microsoft Foundry з AZD
- Налаштовувати та кастомізувати AI агентів з інструментами
- Розгортати додатки RAG (Retrieval-Augmented Generation)

## 🚀 Швидкий старт

```bash
# Варіант 1: Додаток для чатів RAG
azd init --template azure-search-openai-demo
azd up

# Варіант 2: AI агенти
azd init --template get-started-with-ai-agents
azd up

# Варіант 3: Швидкий чат-додаток
azd init --template openai-chat-app-quickstart
azd up
```
