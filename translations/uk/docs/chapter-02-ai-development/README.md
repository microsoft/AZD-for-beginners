# Розділ 2: Розробка з орієнтацією на ШІ

**📚 Курс**: [AZD для початківців](../../README.md) | **⏱️ Тривалість**: 1-2 години | **⭐ Складність**: Середній рівень

---

## Огляд

Цей розділ зосереджений на розгортанні застосунків із підтримкою ШІ за допомогою Azure Developer CLI та сервісів Microsoft Foundry. Від простих чат-застосунків ШІ до інтелектуальних агентів з інструментами.

> **Примітка про перевірку (2026-07-13):** Потік команд і рекомендації щодо розширень у цьому розділі були перевірені з `azd` `1.27.1` та поточним прев’ю розширення AI агента `azure.ai.agents` `1.0.0-beta.5`. Якщо у вас старіша версія AZD, оновіться спочатку, а потім продовжуйте вправи.

## Цілі навчання

Завершивши цей розділ, ви зможете:
- Розгортати AI-застосунки за допомогою готових шаблонів AZD
- Розуміти інтеграцію Microsoft Foundry з AZD
- Налаштовувати та персоналізувати AI-агентів з інструментами
- Розгортати застосунки RAG (Retrieval-Augmented Generation)

## 🚀 Швидкий початок

```bash
# Варіант 1: Додаток RAG Chat
azd init --template azure-search-openai-demo
azd up

# Варіант 2: AI агенти
azd init --template get-started-with-ai-agents
azd up

# Варіант 3: Швидкий чат-додаток
azd init --template openai-chat-app-quickstart
azd up
```
