# Розділ 8: Патерни для продакшену та підприємств

**📚 Курс**: [AZD для початківців](../../README.md) | **⏱️ Тривалість**: 2-3 години | **⭐ Складність**: Просунутий

---

## Огляд

Цей розділ охоплює готові для підприємств патерни розгортання, посилення безпеки, моніторинг і оптимізацію витрат для продакшен-навантажень AI.

## Навчальні цілі

Після завершення цього розділу ви:
- Розгортати стійкі додатки у кількох регіонах
- Впроваджувати корпоративні патерни безпеки
- Налаштовувати комплексний моніторинг
- Оптимізувати витрати в масштабі
- Налаштовувати CI/CD конвеєри з AZD

## 🔐 Найкращі практики безпеки

```bicep
// Use managed identity
identity: {
  type: 'SystemAssigned'
}

// Private endpoints for AI services
properties: {
  publicNetworkAccess: 'Disabled'
  networkAcls: {
    defaultAction: 'Deny'
  }
}
```

## 💰 Оптимізація витрат

| Стратегія | Економія |
|----------|---------|
| Масштабування до нуля (Container Apps) | 60-80% |
| Використовувати рівні споживання для розробки | 50-70% |
| Планове масштабування | 30-50% |
| Резервована потужність | 20-40% |

```bash
# Встановити сповіщення про бюджет
az consumption budget create \
  --budget-name "AI-Budget" \
  --amount 500 \
  --category Cost \
  --time-grain Monthly
```

## 📊 Налаштування моніторингу

```bash
# Транслювати журнали
azd monitor --logs

# Перевірити Application Insights
azd monitor

# Переглянути метрики
az monitor metrics list --resource <resource-id>
```
