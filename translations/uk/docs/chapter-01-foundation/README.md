# Розділ 1: Основи та швидкий старт

**📚 Курс**: [AZD для початківців](../../README.md) | **⏱️ Тривалість**: 30-45 хвилин | **⭐ Складність**: Початковий рівень

---

## Огляд

У цьому розділі розглядаються основи Azure Developer CLI (azd). Ви дізнаєтеся основні концепції, встановите інструменти та розгорнете свою першу програму в Azure.

> Перевірено на `azd 1.23.12` у березні 2026 року.

## Цілі навчання

Завершивши цей розділ, ви:
- Зрозумієте, що таке Azure Developer CLI і чим він відрізняється від Azure CLI
- Встановите та налаштуєте AZD на своїй платформі
- Розгорнете свою першу програму в Azure за допомогою `azd up`
- Приберете ресурси за допомогою `azd down`

## ✅ Почніть тут: перевірте свою конфігурацію

Перед початком переконайтесь, що ваш локальний комп’ютер готовий для шаблону Розділу 1:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

Якщо скрипт повідомляє про відсутні інструменти, спочатку виправте це, а потім продовжуйте з розділом.

---

## 🚀 Швидкий старт

```bash
# Перевірте встановлення
azd version

# Аутентифікуйтеся для AZD
# За бажанням: az login, якщо плануєте безпосередньо запускати команди Azure CLI
azd auth login

# Розгорніть ваш перший додаток
azd init --template todo-nodejs-mongo
azd up

# Очистіть після завершення роботи
azd down --force --purge
```

---

## ✅ Критерії успіху

Після завершення цього розділу ви зможете:

```bash
azd version              # Показує встановлену версію
azd init --template todo-nodejs-mongo  # Ініціалізує проект
azd up                   # Розгортає на Azure
azd show                 # Відображає URL запущеного додатку
azd down --force --purge # Очищає ресурси
```
