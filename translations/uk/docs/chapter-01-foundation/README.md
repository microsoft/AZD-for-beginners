## Overview

Цей розділ знайомить з основами Azure Developer CLI (azd). Ви дізнаєтеся основні поняття, встановите інструменти та розгорнете свій перший застосунок в Azure.

## Learning Objectives

Після завершення цього розділу ви:
- Зрозумієте, що таке Azure Developer CLI і чим він відрізняється від Azure CLI
- Встановите та налаштуєте AZD на вашій платформі
- Розгорнете ваш перший застосунок в Azure за допомогою `azd up`
- Очистите ресурси за допомогою `azd down`

## 🚀 Quick Start

```bash
# Перевірте встановлення
azd version

# Увійдіть у Azure
azd auth login

# Розгорніть свій перший додаток
azd init --template todo-nodejs-mongo
azd up

# Очистіть після завершення
azd down --force --purge
```

---
