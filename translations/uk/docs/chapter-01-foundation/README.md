**📚 Курс**: [AZD Для Початківців](../../README.md) | **⏱️ Тривалість**: 30-45 хвилин | **⭐ Складність**: Початковий рівень

## Огляд

Цей розділ знайомить з фундаментальними поняттями Azure Developer CLI (azd). Ви дізнаєтеся основні концепції, встановите інструменти та розгорнете ваш перший додаток у Azure.

> Перевірено на `azd 1.27.1` у липні 2026 року.

## Навчальні цілі

Виконавши цей розділ, ви:
- Зрозумієте, що таке Azure Developer CLI і чим він відрізняється від Azure CLI
- Встановите та налаштуєте AZD на вашій платформі
- Розгорнете ваш перший додаток у Azure за допомогою `azd up`
- Приберете ресурси за допомогою `azd down`

## 🚀 Швидкий Старт

```bash
# Перевірте встановлення
azd version

# Аутентифікуйтеся для AZD
# За бажанням: az login, якщо ви плануєте запускати команди Azure CLI безпосередньо
azd auth login

# Розгорніть ваш перший додаток
azd init --template todo-nodejs-mongo
azd up

# Очистіть після завершення роботи
azd down --force --purge
```

---

## ✅ Критерії Успіху

Після завершення цього розділу ви зможете:

```bash
azd version              # Показує встановлену версію
azd init --template todo-nodejs-mongo  # Ініціалізує проект
azd up                   # Розгортає в Azure
azd show                 # Відображає URL працюючого додатка
azd down --force --purge # Очищає ресурси
```
