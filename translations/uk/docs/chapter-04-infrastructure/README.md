# Розділ 4: Інфраструктура як код і розгортання

**📚 Тривалість**: 1-1.5 години | **⭐ Складність**: Середній

---

## Огляд

У цьому розділі розглядаються шаблони Infrastructure as Code (IaC) з використанням шаблонів Bicep, надання ресурсів та стратегії розгортання за допомогою Azure Developer CLI.

## Цілі навчання

Виконавши цей розділ, ви:
- Зрозумієте структуру та синтаксис шаблонів Bicep
- Створюватимете ресурси Azure за допомогою `azd provision`
- Розгортатимете додатки за допомогою `azd deploy`
- Реалізуєте стратегії розгортання blue-green та rolling

## 🚀 Швидкий старт

```bash
# Ініціалізувати з шаблону
azd init --template azure-functions-python-v2-http

# Переглянути, що буде створено
azd provision --preview

# Тільки підготувати інфраструктуру
azd provision

# Тільки розгорнути код
azd deploy

# Або обидва разом
azd up
```

---

## 📁 Структура проєкту AZD

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

---

## 🔧 Основні команди

| Command | Опис |
|---------|-------------|
| `azd init` | Ініціалізувати проєкт |
| `azd provision` | Створити ресурси Azure |
| `azd deploy` | Розгорнути код додатку |
| `azd up` | provision + deploy |
| `azd down` | Видалити всі ресурси |
