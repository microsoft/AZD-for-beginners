# Розділ 4: Інфраструктура як код і розгортання

**📚 Курс**: [AZD для початківців](../../README.md) | **⏱️ Тривалість**: 1-1.5 години | **⭐ Складність**: Середній рівень

---

## Огляд

У цьому розділі розглядаються шаблони Infrastructure as Code (IaC) за допомогою Bicep, організація ресурсів і стратегії розгортання з використанням Azure Developer CLI.

> Перевірено на `azd 1.23.12` у березні 2026 року.

## Цілі навчання

Після проходження цього розділу ви зможете:
- Розуміти структуру та синтаксис шаблонів Bicep
- Налаштовувати ресурси Azure за допомогою `azd provision`
- Розгортати додатки за допомогою `azd deploy`
- Впроваджувати стратегії розгортання blue-green та rolling

## 🚀 Швидкий старт

```bash
# Ініціалізувати з шаблону
azd init --template azure-functions-python-v2-http

# Переглянути, що буде створено
azd provision --preview

# Забезпечити лише інфраструктуру
azd provision

# Розгорнути лише код
azd deploy

# Або обидва разом
azd up
```

---

## 📁 Структура проєкту AZD

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

---

## 🔧 Основні команди

| Команда | Опис |
|---------|-------------|
| `azd init` | Ініціалізація проєкту |
| `azd provision` | Створення ресурсів Azure |
| `azd deploy` | Розгортання коду додатка |
| `azd up` | provision + deploy |
| `azd down` | Видалення всіх ресурсів |
