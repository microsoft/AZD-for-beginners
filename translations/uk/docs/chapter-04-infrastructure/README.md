# Розділ 4: Інфраструктура як код та розгортання

**📚 Курс**: [AZD для початківців](../../README.md) | **⏱️ Тривалість**: 1-1.5 години | **⭐ Складність**: Середній рівень

---

## Огляд

У цьому розділі розглядаються шаблони Infrastructure as Code (IaC) з використанням шаблонів Bicep, управління ресурсами та стратегії розгортання з використанням Azure Developer CLI.

> Перевірено на `azd 1.27.1` у липні 2026 року.

## Цілі навчання

Завершивши цей розділ, ви:
- Зрозумієте структуру та синтаксис шаблонів Bicep
- Навчитеся створювати ресурси Azure за допомогою `azd provision`
- Розгортати застосунки за допомогою `azd deploy`
- Впроваджувати стратегії розгортання blue-green та rolling

## 🚀 Швидкий старт

```bash
# Ініціалізувати з шаблону
azd init --template azure-functions-python-v2-http

# Попередній перегляд того, що буде створено
azd provision --preview

# Забезпечити інфраструктуру лише
azd provision

# Розгорнути лише код
azd deploy

# Або обидва разом
azd up
```

## 📁 Структура проекту AZD

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

---

## 🔧 Основні команди

| Команда | Опис |
|---------|-------------|
| `azd init` | Ініціалізувати проект |
| `azd provision` | Створити ресурси Azure |
| `azd deploy` | Розгорнути код застосунку |
| `azd up` | provision + deploy |
| `azd down` | Видалити всі ресурси |
