# Розділ 4: Інфраструктура як код та розгортання

**📚 Курс**: [AZD для початківців](../../README.md) | **⏱️ Тривалість**: 1-1.5 години | **⭐ Складність**: Середній рівень

---

## Огляд

Цей розділ охоплює патерни Інфраструктури як коду (IaC) з шаблонами Bicep, створення ресурсів та стратегії розгортання за допомогою Azure Developer CLI.

> Перевірено на `azd 1.25.6` у червні 2026 року.

## Навчальні цілі

Виконавши цей розділ, ви:
- Зрозумієте структуру та синтаксис шаблонів Bicep
- Створюватимете ресурси Azure за допомогою `azd provision`
- Розгортатимете застосунки за допомогою `azd deploy`
- Реалізуєте стратегії синьо-зеленого та поступового розгортання

## 🚀 Швидкий старт

```bash
# Ініціалізувати з шаблону
azd init --template azure-functions-python-v2-http

# Попередній перегляд того, що буде створено
azd provision --preview

# Постачання лише інфраструктури
azd provision

# Розгортання лише коду
azd deploy

# Або обидва разом
azd up
```

## 📁 Структура проекту AZD

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

---

## 🔧 Основні команди

| Команда | Опис |
|---------|-------------|
| `azd init` | Ініціалізація проекту |
| `azd provision` | Створення ресурсів Azure |
| `azd deploy` | Розгортання коду застосунку |
| `azd up` | створення + розгортання |
| `azd down` | Видалити всі ресурси |
