**📚 Курс**: [AZD For Beginners](../../README.md) | **⏱️ Тривалість**: 45-60 хвилин | **⭐ Складність**: Середній

## Огляд

Цей розділ охоплює конфігурацію середовища, патерни автентифікації та найкращі практики безпеки для розгортань Azure Developer CLI.

## Цілі навчання

Після завершення цього розділу ви зможете:
- Оволодіти ієрархією конфігурації AZD
- Керувати кількома середовищами (dev, staging, prod)
- Реалізувати безпечну автентифікацію з використанням керованих ідентичностей
- Налаштувати параметри, специфічні для середовища

## 🚀 Quick Start

```bash
# Створити кілька середовищ
azd env new dev
azd env new staging
azd env new prod

# Переключити середовища
azd env select prod

# Встановити змінні середовища
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# Переглянути конфігурацію
azd env get-values
```

---

## 🔧 Ієрархія конфігурації

AZD застосовує налаштування в цьому порядку (пізніші перевизначають ранні):

1. **Значення за замовчуванням** (вбудовані в шаблони)
2. **azure.yaml** (конфігурація проєкту)
3. **Змінні середовища** (`azd env set`)
4. **Прапори командного рядка** (`--location eastus`)

---

## 🔐 Найкращі практики безпеки

```bash
# Використовуйте керовану ідентичність (рекомендується)
azd env set AZURE_USE_MANAGED_IDENTITY true

# Перевірте стан автентифікації
azd auth whoami
az account show

# Повторно автентифікуйтеся при необхідності
azd auth login
az login
```
