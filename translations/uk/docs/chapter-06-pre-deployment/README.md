# Chapter 6: Планування та перевірка перед розгортанням

**📚 Курс**: [AZD For Beginners](../../README.md) | **⏱️ Тривалість**: 1 година | **⭐ Складність**: Середній рівень

---

## Огляд

Цей розділ охоплює основні кроки планування та перевірки перед розгортанням вашого додатку. Навчіться уникати дорогих помилок за допомогою правильного планування потужностей, вибору SKU та попередніх перевірок.

> Перевірено на `azd 1.23.12` у березні 2026 року.

## Мета навчання

Після проходження цього розділу ви зможете:
- Виконувати попередні перевірки перед розгортанням
- Планувати потужності та оцінювати потреби в ресурсах
- Вибирати відповідні SKU для оптимізації витрат
- Налаштовувати Application Insights для моніторингу
- Розуміти патерни координації команди

## 🚀 Швидкий старт

```bash
# Перевірити ліміти підписки
az vm list-usage --location eastus --output table

# Попередній перегляд розгортання (ресурси не створюються)
azd provision --preview

# Перевірити синтаксис Bicep
az bicep build --file infra/main.bicep

# Перевірити конфігурацію середовища
azd env get-values
```

## ☑️ Чеклист перед розгортанням

### Перед `azd provision`

- [ ] Перевірено квоту для регіону
- [ ] Вибрано відповідні SKU
- [ ] Переглянута оцінка вартості
- [ ] Узгоджена конвенція іменування
- [ ] Налаштовано безпеку/RBAC

### Перед `azd deploy`

- [ ] Встановлені змінні середовища
- [ ] Секрети в Key Vault
- [ ] Перевірено рядки підключень
- [ ] Налаштовані перевірки здоров’я
