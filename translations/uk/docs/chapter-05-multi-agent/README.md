**📚 Курс**: [AZD Для Початківців](../../README.md) | **⏱️ Тривалість**: 2-3 години | **⭐ Складність**: Просунутий

## Огляд

Цей розділ охоплює просунуті архітектурні патерни багатагентних систем, оркестрування агентів та промислові розгортання ШІ для складних сценаріїв.

## Цілі навчання

Пройшовши цей розділ, ви:
- Зрозумієте архітектурні патерни багатагентних систем
- Розгорнете скоординовані системи агентів ШІ
- Реалізуєте взаємодію між агентами
- Побудуєте готові до продакшену багатагентні рішення

## 🚀 Швидкий старт

```bash
# Розгорніть багатоагентне рішення для роздрібної торгівлі
cd examples/retail-multiagent-arm-template
./deploy.sh

# Або використайте шаблон безпосередньо
azd init --template agent-openai-python-prompty
azd up
```

         ┌───────────▼───────┐ ┌───────▼───────────┐

## 🎯 Представлене рішення: Багатагентне рішення для роздрібної торгівлі

The [Багатагентне рішення для роздрібної торгівлі](../../examples/retail-scenario.md) демонструє:

- **Агент клієнта**: Обробляє взаємодію з користувачем та уподобання
- **Агент запасів**: Керує запасами та обробкою замовлень
- **Оркестратор**: Координує роботу між агентами
- **Спільна пам'ять**: Управління контекстом між агентами

### Використані сервіси

| Сервіс | Призначення |
|---------|---------|
| Azure OpenAI | Розуміння мови |
| Azure AI Search | Каталог продуктів |
| Cosmos DB | Стан агента та пам'ять |
| Container Apps | Хостинг агентів |
| Application Insights | Моніторинг |
