# Розділ 7: Виправлення неполадок та налагодження

**📚 Курс**: [AZD для початківців](../../README.md) | **⏱️ Тривалість**: 1-1.5 hours | **⭐ Складність**: Середній

## Огляд

Цей розділ допоможе вам діагностувати та вирішувати поширені проблеми при роботі з Azure Developer CLI. Від помилок розгортання до проблем, специфічних для ШІ.

## Цілі навчання

Виконавши цей розділ, ви зможете:
- Діагностувати поширені помилки розгортання AZD
- Налагоджувати проблеми автентифікації та дозволів
- Вирішувати проблеми з підключенням сервісів ШІ
- Використовувати Azure Portal та CLI для вирішення проблем

## 🚨 Швидкі виправлення

### Authentication Issues
```bash
azd auth login
az login
azd auth whoami
```

### Provisioning Failures
```bash
azd show
azd monitor --logs
az deployment sub list --query "[?properties.provisioningState!='Succeeded']"
```

### Resource Conflicts
```bash
azd down --force --purge
azd env new different-name
azd up
```

### Quota Exceeded
```bash
az vm list-usage --location eastus --output table
azd env set AZURE_LOCATION westus2
azd up
```

---

## 📋 Довідник кодів помилок

| Помилка | Причина | Рішення |
|-------|-------|----------|
| `AuthenticationError` | Не ввійдено в систему | `azd auth login` |
| `ResourceNotFound` | Відсутній ресурс | Перевірте імена ресурсів |
| `QuotaExceeded` | Обмеження підписки | Запросіть збільшення квоти |
| `InvalidTemplate` | Помилка синтаксису Bicep | `az bicep build` |
| `Conflict` | Ресурс існує | Використайте іншу назву або видаліть |
| `Forbidden` | Недостатні повноваження | Перевірте ролі RBAC |

## 🔄 Скидання та відновлення

```bash
# М'яке скидання (зберегти ресурси, повторно розгорнути код)
azd deploy --force

# Жорстке скидання (видалити все, почати заново)
azd down --force --purge
azd up
```
