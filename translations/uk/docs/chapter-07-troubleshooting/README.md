# Розділ 7: Вирішення проблем та налагодження

**📚 Курс**: [AZD для початківців](../../README.md) | **⏱️ Тривалість**: 1-1.5 години | **⭐ Складність**: Середня

---

## Огляд

Цей розділ допоможе вам діагностувати та вирішувати типові проблеми при роботі з Azure Developer CLI. Від помилок розгортання до специфічних проблем зі штучним інтелектом.

> Перевірено на `azd 1.27.1` у липні 2026 року.

## Цілі навчання

Завершивши цей розділ, ви зможете:
- Діагностувати типові помилки розгортання AZD
- Налагоджувати проблеми автентифікації та дозволів
- Вирішувати проблеми з підключенням до AI-сервісів
- Використовувати Azure Portal і CLI для вирішення проблем

## 🚨 Швидкі виправлення

### Проблеми автентифікації
```bash
# Необхідно для робочих процесів AZD
azd auth login

# Необов’язково, якщо ви також безпосередньо використовуєте команди Azure CLI
az login

azd auth status
```

### Помилки розгортання
```bash
azd show
azd monitor --logs
az deployment sub list --query "[?properties.provisioningState!='Succeeded']"
```

### Конфлікти ресурсів
```bash
azd down --force --purge
azd env new different-name
azd up
```

### Перевищення квоти
```bash
az vm list-usage --location eastus --output table
azd env set AZURE_LOCATION westus2
azd up
```

## 📋 Відомість кодів помилок

| Помилка | Причина | Рішення |
|-------|-------|----------|
| `AuthenticationError` | Не виконано вхід | `azd auth login` |
| `ResourceNotFound` | Відсутній ресурс | Перевірте назви ресурсів |
| `QuotaExceeded` | Обмеження підписки | Запросіть збільшення квоти |
| `InvalidTemplate` | Синтаксична помилка Bicep | `az bicep build` |
| `Conflict` | Ресурс існує | Використайте іншу назву або видаліть |
| `Forbidden` | Недостатньо прав | Перевірте ролі RBAC |

## 🔄 Скидання та відновлення

```bash
# М’яке скидання (зберегти ресурси, перевпровадити код)
azd deploy --force

# Жорстке скидання (видалити все, почати заново)
azd down --force --purge
azd up
```
