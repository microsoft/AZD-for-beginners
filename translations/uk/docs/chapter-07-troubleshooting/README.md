# Розділ 7: Виявлення та усунення несправностей

**📚 Курс**: [AZD для початківців](../../README.md) | **⏱️ Тривалість**: 1-1.5 години | **⭐ Складність**: Середній

---

## Огляд

Цей розділ допоможе вам діагностувати та вирішувати поширені проблеми при роботі з Azure Developer CLI. Від помилок розгортання до специфічних проблем зі штучним інтелектом.

> Перевірено на `azd 1.25.6` у червні 2026 року.

## Цілі навчання

Після проходження цього розділу ви зможете:
- Діагностувати поширені помилки при розгортанні AZD
- Виявляти помилки автентифікації та дозволів
- Вирішувати проблеми з підключенням до AI-сервісів
- Використовувати Azure Portal і CLI для усунення несправностей

## 🚨 Швидкі виправлення

### Проблеми з автентифікацією
```bash
# Необхідно для робочих процесів AZD
azd auth login

# Необов’язково, якщо ви також безпосередньо використовуєте команди Azure CLI
az login

azd auth status
```

### Помилки при наданні ресурсів
```bash
azd show
azd monitor --logs
az deployment sub list --query "[?properties.provisioningState!='Succeeded']"
```

### Конфлікти ресурсів
```bash
azd down --force --purge
azd env new different-name
azd up
```

### Перевищення квоти
```bash
az vm list-usage --location eastus --output table
azd env set AZURE_LOCATION westus2
azd up
```

---

## 📋 Довідка по кодах помилок

| Помилка | Причина | Рішення |
|-------|-------|----------|
| `AuthenticationError` | Не виконано вхід | `azd auth login` |
| `ResourceNotFound` | Ресурс відсутній | Перевірте назви ресурсів |
| `QuotaExceeded` | Ліміти підписки перевищено | Запросіть збільшення квоти |
| `InvalidTemplate` | Помилка синтаксису Bicep | `az bicep build` |
| `Conflict` | Ресурс вже існує | Використайте нову назву або видаліть |
| `Forbidden` | Недостатньо дозволів | Перевірте ролі RBAC |

## 🔄 Скидання та відновлення

```bash
# М'яке скидання (зберегти ресурси, повторно розгорнути код)
azd deploy --force

# Жорстке скидання (видалити все, почати заново)
azd down --force --purge
azd up
```
