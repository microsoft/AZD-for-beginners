## 2. Основні Опановані Концепції

### Azure Developer CLI (azd)
- `azd init` – Ініціалізація проектів зі шаблонів
- `azd up` – Підготовка та розгортання однією командою
- `azd env` – Управління змінними середовища
- `azd down` – Безпечне очищення ресурсів

### Структура Шаблону AZD
- `azure.yaml` – Конфігурація та хуки розгортання
- `infra/` – Інфраструктура як код Bicep
- `src/` – Вихідний код додатка
- `scripts/` – Скрипти автоматизації та налаштування

### Можливості Microsoft Foundry
- Сервіс AI агентів для інтелектуальних помічників
- Пошук файлів та Azure AI Search для отримання знань
- Вбудоване трасування, моніторинг та оцінювання
- Red teaming для валідації безпеки AI
