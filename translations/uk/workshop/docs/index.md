# AZD для розробників штучного інтелекту

> [!IMPORTANT]  
> **Цей воркшоп оснащений посібником, який ви можете переглянути у своєму локальному браузері. Щоб почати, вам потрібно запустити GitHub Codespaces у репозиторії — потім зачекайте, доки з’явиться активний термінал VS Code, і введіть:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Ви маєте побачити спливаюче вікно для відкриття сторінки попереднього перегляду у браузері.**

Ласкаво просимо до практичного воркшопу з вивчення Azure Developer CLI (AZD) з акцентом на розгортанні AI-додатків. Цей воркшоп допоможе вам здобути прикладні знання про шаблони AZD у 3 кроки:

1. **Відкриття** — знайдіть шаблон, який вам підходить.  
1. **Розгортання** — розгорніть і перевірте, що він працює.  
1. **Налаштування** — змінюйте і роботайте над ним, щоб зробити свій!

Протягом воркшопу ви також ознайомитесь із основними інструментами розробника та робочими процесами, щоб допомогти спростити вашу повну розробницьку подорож.

## Огляд воркшопу

**Тривалість:** 3-4 години  
**Рівень:** від початкового до середнього  
**Передумови:** Знайомство з Azure, концепціями ШІ, VS Code та командними інструментами.

Це практичний воркшоп, де ви вчитеся, виконуючи завдання. Після завершення вправ рекомендуємо ознайомитися з навчальною програмою AZD для початківців, щоб продовжити свій шлях навчання у сферах безпеки та продуктивності.

| Час| Модуль  | Мета |
|:---|:---|:---|
| 15 хв | Вступ | Встановити контекст, зрозуміти цілі |
| 30 хв | Вибір AI шаблону | Оглянути варіанти та обрати початковий | 
| 30 хв | Перевірка AI шаблону | Розгорнути стандартне рішення в Azure |
| 30 хв | Аналіз AI шаблону | Вивчити структуру і конфігурацію |
| 30 хв | Налаштування AI шаблону | Активувати та випробувати доступні функції |
| 30 хв | Кастомізація AI шаблону | Адаптувати шаблон під свої потреби |
| 30 хв | Видалення інфраструктури | Прибрати та звільнити ресурси |
| 15 хв | Підсумок та наступні кроки | Навчальні ресурси, виклик воркшопу |
| | |

## Що ви дізнаєтесь

Розглядайте шаблон AZD як пісочницю для вивчення різних можливостей і інструментів для повномасштабної розробки у Microsoft Foundry. Наприкінці цього воркшопу ви матимете інтуїтивне розуміння різних інструментів та концепцій у цьому контексті.

| Концепція  | Мета |
|:---|:---|
| **Azure Developer CLI** | Зрозуміти команди інструменту і робочі процеси |
| **AZD шаблони**| Зрозуміти структуру проекту і конфігурацію |
| **Azure AI Agent**| Впроваджувати і розгортати проекти Microsoft Foundry  |
| **Azure AI Search**| Забезпечити інженерію контексту з агентами |
| **Спостережуваність**| Вивчити трасування, моніторинг та оцінки |
| **Red Teaming**| Дослідити атаки і методи їх запобігання |
| | |
