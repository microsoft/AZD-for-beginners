# AZD для розробників AI: майстер-клас

> Перевірено на `azd 1.23.12` у березні 2026 року.

> [!IMPORTANT]  
> **Цей майстер-клас обладнано керівництвом, яке ви можете переглянути у своєму локальному браузері. Щоб розпочати, відкрийте репозиторій у GitHub Codespaces, контейнері розробника або локальній копії з встановленим MkDocs — потім дочекайтесь активного терміналу VS Code і введіть:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Ви повинні побачити спливаюче діалогове вікно з пропозицією відкрити сторінку попереднього перегляду у браузері.**

Ласкаво просимо на практичний майстер-клас з вивчення Azure Developer CLI (AZD) з акцентом на розгортання AI-застосунків. Цей майстер-клас допоможе вам отримати практичне розуміння шаблонів AZD у 3 кроки:

1. **Відкриття** — знайти відповідний для вас шаблон.
1. **Розгортання** — розгорнути і перевірити, що він працює.
1. **Налаштування** — змінити і ітерувати, щоб зробити його своїм!

Протягом цього майстер-класу ви також познайомитеся з основними інструментами й робочими процесами для розробників, щоб оптимізувати весь ваш процес розробки.

## Огляд майстер-класу

**Тривалість:** 3-4 години  
**Рівень:** Початковий — середній  
**Передумови:** Знайомство з Azure, концепціями AI, VS Code і командним рядком.

Це практичний майстер-клас, де ви вчитеся, виконуючи завдання. Після виконання вправ радимо ознайомитися з навчальною програмою AZD для початківців, щоб продовжити навчання з найкращих практик з безпеки та продуктивності.

| Час | Модуль  | Мета |
|:---|:---|:---|
| 15 хв | Вступ | Встановити контекст, зрозуміти цілі |
| 30 хв | Вибір AI-шаблону | Розглянути варіанти та обрати стартовий | 
| 30 хв | Перевірка AI-шаблону | Розгорнути дефолтне рішення в Azure |
| 30 хв | Розбір AI-шаблону | Ознайомитись зі структурою і конфігурацією |
| 30 хв | Налаштування AI-шаблону | Активувати та спробувати доступні функції |
| 30 хв | Індивідуалізація AI-шаблону | Адаптувати шаблон під свої потреби |
| 30 хв | Припинення роботи інфраструктури | Очистити та звільнити ресурси |
| 15 хв | Підсумок і подальші кроки | Ресурси для навчання, виклик майстер-класу |
| | |

## Чого ви навчитесь

Розглядайте шаблон AZD як навчальне середовище, щоб дослідити різні можливості та інструменти для повного циклу розробки на Microsoft Foundry. Після завершення цього майстер-класу ви повинні інтуїтивно розуміти різні інструменти та концепції в цьому контексті.

| Концепція  | Мета |
|:---|:---|
| **Azure Developer CLI** | Зрозуміти команди інструменту та робочі процеси |
| **Шаблони AZD**| Зрозуміти структуру проекту та конфігурацію |
| **Azure AI Agent**| Забезпечити й розгорнути проект Microsoft Foundry  |
| **Azure AI Search**| Активувати контекстне інженерство з агентами |
| **Спостережливість**| Дослідити трасування, моніторинг і оцінки |
| **Червона команда**| Ознайомитися з тестуванням супротивника та пом’якшеннями |
| | |
