# AZD для розробників AI: майстер-клас

> Перевірено на `azd 1.27.1` у липні 2026 року.

> [!IMPORTANT]  
> **Цей майстер-клас оснащено посібником, який ви можете попередньо переглянути у вашому локальному браузері. Щоб почати, відкрийте репозиторій у GitHub Codespaces, контейнері розробника або локальному клоні з встановленим MkDocs — потім дочекайтеся активного терміналу VS Code і введіть:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Ви маєте побачити спливаюче вікно для відкриття сторінки попереднього перегляду у браузері.**

Ласкаво просимо на практичний майстер-клас з вивчення Azure Developer CLI (AZD) з фокусом на розгортанні AI-застосунків. Цей майстер-клас допоможе вам отримати практичне розуміння AZD-шаблонів у 3 кроки:

1. **Відкриття** – знайти шаблон, який вам підходить.
1. **Розгортання** – розгорнути і перевірити, що він працює
1. **Налаштування** – змінити і відтворити, щоб зробити його своїм!

Протягом майстер-класу ви також познайомитесь з основними інструментами розробника і робочими процесами, що допоможуть оптимізувати вашу повну подорож розробки.

## Огляд майстер-класу

**Тривалість:** 3-4 години  
**Рівень:** Початковий до середнього  
**Вимоги:** Знайомство з Azure, концепціями AI, VS Code та командним рядком.

Це практичний майстер-клас, де ви навчаєтесь, виконуючи. Після завершення вправ рекомендуємо ознайомитись із навчальним курсом AZD для початківців, щоб продовжити ваш шлях навчання кращим практикам безпеки та продуктивності.

| Час | Модуль  | Мета |
|:---|:---|:---|
| 15 хв | Вступ | Окреслити завдання, зрозуміти цілі |
| 30 хв | Вибір AI-шаблону | Ознайомитись з варіантами і обрати стартовий | 
| 30 хв | Перевірка AI-шаблону | Розгорнути стандартне рішення в Azure |
| 30 хв | Розбір AI-шаблону | Вивчити структуру та конфігурацію |
| 30 хв | Налаштування AI-шаблону | Активувати та випробувати доступні функції |
| 30 хв | Кастомізація AI-шаблону | Адаптувати шаблон під свої потреби |
| 30 хв | Видалення інфраструктури | Очищення і звільнення ресурсів |
| 15 хв | Підсумки та наступні кроки | Навчальні ресурси, завдання майстер-класу |
| | |

## Чому ви навчитесь

Розглядайте AZD-шаблон як навчальний майданчик для вивчення різних можливостей і інструментів для повного циклу розробки на Microsoft Foundry. Наприкінці цього майстер-класу у вас буде інтуїтивне розуміння різних інструментів і концепцій у цьому контексті.

| Концепція  | Мета |
|:---|:---|
| **Azure Developer CLI** | Розуміти команди інструменту та робочі процеси|
| **AZD-шаблони**| Розуміти структуру проекту та конфігурацію|
| **Azure AI Agent**| Надання та розгортання проекту Microsoft Foundry  |
| **Azure AI Search**| Забезпечення контекстного інжинірингу з агентами |
| **Обсервабельність**| Ознайомитись з трасуванням, моніторингом і оцінками |
| **Red Teaming**| Вивчення тестування на загрози і заходів протидії |
| | |
