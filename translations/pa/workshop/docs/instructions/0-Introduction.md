# AZD AI ਡਿਵੈਲਪਰਾਂ ਲਈ

!!! info "ਇਸ ਵਰਕਸ਼ਾਪ ਲਈ ਦਰਸ਼ਕ"
   
    ਇਹ ਵਰਕਸ਼ਾਪ ਉਹਨਾਂ ਲਈ ਹੈ ਜੋ Microsoft Foundry 'ਤੇ ਸਰੋਤ-ਸਤਹ ਤਕਨੀਕਾਂ ਅਤੇ ਵਰਕਫਲੋਜ਼ ਦਾ ਲਾਭ ਉਠਾ ਕੇ AI ਐਜੈਂਟਿਕ ਐਪਲਿਕੇਸ਼ਨਾਂ ਬਨਾਉਣ ਦੀ ਕੋਸ਼ਿਸ਼ ਕਰ ਰਹੇ ਹਨ। ਇਹ ਵਰਕਸ਼ਾਪ ਸ਼ੁਰੂਆਤੀਆਂ ਅਤੇ ਅਨੁਭਵੀ ਡਿਵੈਲਪਰਾਂ ਦੋਹਾਂ ਲਈ موزੂਨ ਹੈ, ਜਿਨ੍ਹਾਂ ਨੂੰ Azure, AI ਸੰਕਲਪਾਂ ਅਤੇ Python ਕੋਡ ਦੀ ਕੁਝ ਜਾਣੂ ਹੋਵੇ। ਕਿਰਪਾ ਕਰਕੇ ਸ਼ੁਰੂ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਪੂਰਵ-ਸ਼ਰਤਾਂ ਦੀ ਜਾਂਚ ਕਰੋ।

    **ਅਵਧੀ:** 3-4 ਘੰਟੇ  
    **ਸਤਰ:** ਸ਼ੁਰੂਆਤੀ ਤੋਂ ਦਰਮਿਆਨਾ  

---


## ਪੂਰਵ-ਸ਼ਰਤਾਂ

- ਨਿੱਜੀ GitHub ਖਾਤਾ → [ਇੱਕ ਮੁਫ਼ਤ ਖਾਤਾ ਬਣਾਓ](https://github.com/signup)
- ਸਰਗਰਮ Azure ਖਾਤਾ → [ਇੱਕ ਮੁਫ਼ਤ ਖਾਤਾ ਬਣਾਓ](https://aka.ms/free)
- AI ਸੰਕਲਪਾਂ, Python ਵਿਕਾਸ ਅਤੇ ਕਮਾਂਡ-ਲਾਈਨ ਟੂਲਾਂ ਨਾਲ ਪਰਿਚਿਤ
- VS Code ਅਤੇ GitHub ਟੂਲਾਂ ਅਤੇ ਵਰਕਫਲੋਜ਼ ਦਾ ਅਨੁਭਵ।

---

## ਸਿੱਖਣ ਦੇ ਲੱਖੜੀਆਂ

ਇਸ ਵਰਕਸ਼ਾਪ ਦੇ ਅੰਤ ਤੱਕ, ਤੁਸੀਂ ਜਾਣਦੇ ਹੋਵੋਗੇ ਕਿ:

- [X] **Azure Developer CLI ਨਾਲ ਕੰਮ ਕਰੋ** → to provision, deploy, and manage, your Azure resources.
- [X] **AI ਐਪ ਟੈਮਪਲੇਟਸ ਵਰਤੋ** →  Microsoft Foundry 'ਤੇ AI Agent ਪ੍ਰੋਜੈਕਟ ਸ਼ੁਰੂ ਕਰਨ ਲਈ।
- [X] **ਟੈਮਪਲੇਟਸ ਨੂੰ ਸੰਰਚਿਤ ਕਰੋ** →  Red Teaming, Evaluations, Tracing & Monitoring ਵਰਗੀਆਂ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ ਲਈ
- [X] **ਡਿਕੰਸਟ੍ਰਕਟ ਅਤੇ ਖੋਜ ਕਰੋ** →  GitHub Copilot ਦੀ ਵਰਤੋਂ ਕਰਕੇ AZD ਟੈਮਪਲੇਟਸ ਅਤੇ ਵਰਕਫਲੋਜ਼
- [X] **ਕਸਟਮਾਈਜ਼ ਅਤੇ ਮੁੜ-ਡਿਪਲੋਇ ਕਰੋ** →  AI Agent starter template, ਤੁਹਾਡੇ ਕਸਟਮ ਲੋੜਾਂ ਦੇ ਨਾਲ

---

## ਵਰਕਸ਼ਾਪ ਢਾਂਚਾ

ਵਰਕਸ਼ਾਪ ਮੋਡੀਊਲਾਂ ਵਜੋਂ ਬਣਿਆ ਹੋਇਆ ਹੈ, ਹਰ ਇੱਕ ਨੂੰ ਹੱਥ-ਅਭਿਆਸ ਲੈਬ ਵਰਕਆਊਟ ਨਾਲ ਜੋੜਿਆ ਗਿਆ ਹੈ:

1. **ਪਰਿਚਯ** - ਸਮਝੋ ਕਿ AZD ਟੈਮਪਲੇਟ ਕੀ ਹੁੰਦੇ ਹਨ, ਅਤੇ ਇਹ ਕਿਵੇਂ ਕੰਮ ਕਰਦੇ ਹਨ
1. **ਚੋਣ** - ਆਪਣੇ ਅਗਲੇ AI ਪ੍ਰੋਜੈਕਟ ਲਈ ਸਹੀ "starter" ਟੈਮਪਲੇਟ ਦੀ ਖੋਜ ਕਰੋ
1. **ਤਸਦੀਕ** - ਟੈਮਪਲੇਟ ਨੂੰ ਡੀਪਲੋਇ ਅਤੇ ਖੋਜੋ ਤਾਂ ਜੋ ਇਹ ਤਸਦੀਕ ਹੋ ਜਾਵੇ ਕਿ ਇਹ ਜਿਵੇਂ ਦਰਸਾਇਆ ਗਿਆ ਹੈ, ਤਿਵੇਂ ਕੰਮ ਕਰਦਾ ਹੈ
1. **ਵਿੱਖੰਡਨ** - ਟੈਮਪਲੇਟ ਨੂੰ ਵਿਭਾਜਿਤ ਕਰੋ ਤਾਂ ਜੋ ਇਸਦੇ ਹਿਸਿਆਂ ਅਤੇ ਸੰਰਚਨਾ ਨੂੰ ਸਮਝ ਸਕੋ
1. **ਸੰਰਚਨਾ** - ਵੱਖ-ਵੱਖ Microsoft Foundry ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ ਨੂੰ ਐਕਟੀਵੇਟ ਅਤੇ ਵਰਤਣ ਸਿੱਖੋ
1. **ਕਸਟਮਾਈਜੇਸ਼ਨ** - ਆਪਣਾ ਡਾਟਾ ਅਤੇ ਲੋੜਾਂ ਧਿਆਨ ਵਿੱਚ ਰੱਖ ਕੇ ਐਪਲੀਕੇਸ਼ਨ ਨੂੰ ਮੁੜ-ਤਿਆਰ ਕਰੋ
1. **ਟਿਅਰਡਾਊਨ** - ਇੰਫ੍ਰਾਸਟਰੱਕਚਰ ਦੀ ਡੀ-ਪ੍ਰੋਵਿਜ਼ਨਿੰਗ ਅਤੇ ਸੋਫਟ-ਡਿਲੀਟ ਕੀਤੇ ਗਏ ਸਰੋਤਾਂ ਦੀ ਪੁਨਰਪ੍ਰਾਪਤੀ ਸਿੱਖੋ

---

## ਫੀਡਬੈਕ ਅਤੇ ਮੁੱਦੇ

ਅਸੀਂ ਨਿਰੰਤਰ ਪਾਠ-ਕ੍ਰਮ ਨੂੰ ਅਪਡੇਟ ਕਰਦੇ ਰਹਿੰਦੇ ਹਾਂ ਤਾਂ ਜੋ ਸ਼ੁਰੂਆਤੀਆਂ ਲਈ ਸਭ ਤੋਂ ਵਧੀਆ ਸਿੱਖਣ ਅਨੁਭਵ ਪ੍ਰਦਾਨ ਕੀਤਾ ਜਾ ਸਕੇ। ਤੁਹਾਡਾ ਫੀਡਬੈਕ ਸुआਗਤ ਯੋਗ ਹੈ।

1. ਇਸ ਰਿਪੋ 'ਤੇ ਇੱਕ issue ਪੋਸਟ ਕਰੋ - ਸੁਵਿਧਾ ਲਈ ਇਸਨੂੰ `Workshop` ਟੈਗ ਕਰੋ।
1. Microsoft Foundry Discord ਵਿੱਚ ਸ਼ਾਮਲ ਹੋਵੋ - ਸਾਥੀ ਸਿੱਖਣ ਵਾਲਿਆਂ ਨਾਲ ਜੁੜੋ!
