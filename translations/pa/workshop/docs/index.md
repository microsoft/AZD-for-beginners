# AI ਡਿਵੈਲਪਰਾਂ ਲਈ AZD ਵਰਕਸ਼ਾਪ

> ਪ੍ਰਮਾਣਿਤ ਕੀਤਾ ਗਿਆ `azd 1.23.12` ਮਾਰਚ 2026 ਵਿੱਚ.
> [!IMPORTANT]  
> **ਇਸ ਵਰਕਸ਼ਾਪ ਵਿਚ ਇੱਕ ਵਰਕਸ਼ਾਪ ਗਾਈਡ ਸ਼ਾਮِل ਹੈ ਜੋ ਤੁਸੀਂ ਆਪਣੇ ਲੋਕਲ ਬ੍ਰਾਊਜ਼ਰ ਵਿੱਚ ਪ੍ਰੀਵਿਊ ਕਰ ਸਕਦੇ ਹੋ। ਸ਼ੁਰੂਆਤ ਕਰਨ ਲਈ, ਰੇਪੋ ਨੂੰ GitHub Codespaces, ਇੱਕ dev container, ਜਾਂ MkDocs ਇੰਸਟਾਲ ਕੀਤੇ ਲੋਕਲ ਕਲੋਨ ਵਿੱਚ ਖੋਲ੍ਹੋ—ਫਿਰ ਉਡੀਕ ਕਰੋ ਜਦ ਤੱਕ ਤੁਸੀਂ ਇੱਕ ਸਰਗਰਮ VS Code ਟਰਮੀਨਲ ਨਹੀਂ ਵੇਖਦੇ ਅਤੇ ਟਾਈਪ ਕਰੋ:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **ਤੁਹਾਨੂੰ ਬ੍ਰਾਊਜ਼ਰ ਵਿੱਚ ਪ੍ਰੀਵਿਊ ਪੇਜ ਖੋਲ੍ਹਣ ਲਈ ਇੱਕ ਪੌਪ-ਅੱਪ ਡਾਇਲਾਗ ਡਿਸਪਲੇ ਹੋਣਾ ਚਾਹੀਦਾ ਹੈ।**

ਸੁਆਗਤ ਹੈ ਹੱਥ-ਅਨ ਵਰਕਸ਼ਾਪ ਵਿੱਚ ਜਿੱਥੇ ਤੁਸੀਂ Azure Developer CLI (AZD) ਸਿੱਖੋਗੇ ਜਦੋਂ ਕਿ ਧਿਆਨ AI ਐਪਲੀਕੇਸ਼ਨ ਡਿਪਲੋਇਮੈਂਟ 'ਤੇ ਹੈ। ਇਹ ਵਰਕਸ਼ਾਪ ਤੁਹਾਨੂੰ 3 ਕਦਮਾਂ ਵਿੱਚ AZD ਟੈਂਪਲੇਟਸ ਦੀ ਪ੍ਰਯੋਗਾਤਮਕ ਸਮਝ ਦਿਲਾਉਂਦਾ ਹੈ:

1. **Discovery** - ਤੁਹਾਡੇ ਲਈ ਸਹੀ ਟੈਮਪਲੇਟ ਲੱਭੋ.
1. **Deployment** - ਡਿਪਲੋਇਮੈਂਟ ਕਰੋ ਅਤੇ ਜਾਂਚੋ ਕਿ ਇਹ ਕੰਮ ਕਰਦਾ ਹੈ
1. **Customization** - ਸੋਧੋ ਅਤੇ ਦੁਹਰਾਓ ਤਾਂ ਜੋ ਇਹ ਤੁਹਾਡਾ ਬਣ ਜਾਵੇ!

ਇਸ ਵਰਕਸ਼ਾਪ ਦੌਰਾਨ, ਤੁਹਾਨੂੰ ਮੁੱਖ ਡਿਵੈਲਪਰ ਟੂਲਾਂ ਅਤੇ ਵਰਕਫਲੋਜ਼ ਨਾਲ ਵੀ ਜਾਣੂ ਕਰਵਾਇਆ ਜਾਵੇਗਾ, ਜੋ ਤੁਹਾਡੇ ਐਂਡ-ਟੂ-ਐਂਡ ਵਿਕਾਸ ਯਾਤਰਾ ਨੂੰ ਸੁਗਮ ਬਣਾਉਣ ਵਿੱਚ ਮਦਦ ਕਰਨਗੇ।

## ਵਰਕਸ਼ਾਪ ਓਵਰਵਿਊ

**ਅवਧੀ:** 3-4 ਘੰਟੇ  
**ਸਤਰ:** ਸ਼ੁਰੂਆਤੀ ਤੋਂ ਦਰਮਿਆਨੀ  
**ਪੂਰਵ-ਸ਼ਰਤਾਂ:** Azure, AI ਧਾਰਨਾਵਾਂ, VS Code ਅਤੇ ਕਮਾਂਡ-ਲਾਈਨ ਟੂਲਾਂ ਨਾਲ ਪਰੀਚਿਤ ਹੋਣਾ.

ਇਹ ਇੱਕ ਹੱਥ-ਅਨ ਵਰਕਸ਼ਾਪ ਹੈ ਜਿੱਥੇ ਤੁਸੀਂ ਕਰਕੇ ਸਿੱਖਦੇ ਹੋ। ਜਦੋਂ ਤੁਸੀਂ ਕਸਰਤਾਂ ਪੂਰੀਆਂ ਕਰ ਲਿਆ, ਅਸੀਂ ਸਿਫ਼ਾਰਸ਼ ਕਰਦੇ ਹਾਂ ਕਿ ਸੁਰੱਖਿਆ ਅਤੇ ਉਤਪਾਦਕਤਾ ਵਧੀਆ ਅਭਿਆਸਾਂ ਵਿੱਚ ਆਪਣੀ ਸਿੱਖਿਆ ਨੂੰ ਜਾਰੀ ਰੱਖਣ ਲਈ [AZD ਸ਼ੁਰੂਆਤੀਆਂ ਲਈ](../README.md) ਕਰੀਕੁਲਮ ਦੀ ਸਮੀਖਿਆ ਕਰੋ।

| ਸਮਾਂ| ਮੋਡੀਊਲ  | ਉਦੇਸ਼ |
|:---|:---|:---|
| 15 ਮਿੰਟ | ਪਰਿਚਯ | ਮੰਚ ਸੈੱਟ ਕਰੋ, ਲਕੜਾਂ/ਲਕਸ਼ਾਂ ਨੂੰ ਸਮਝੋ |
| 30 ਮਿੰਟ | AI ਟੈਮਪਲੇਟ ਚੁਣੋ | ਵਿਕਲਪ ਵੇਖੋ ਅਤੇ ਸਟਾਰਟਰ ਚੁਣੋ | 
| 30 ਮਿੰਟ | AI ਟੈਮਪਲੇਟ ਦੀ ਜਾਂਚ ਕਰੋ | ਡਿਫੋਲਟ ਹੱਲ ਨੂੰ Azure 'ਤੇ ਡਿਪਲੋਇ ਕਰੋ |
| 30 ਮਿੰਟ | AI ਟੈਮਪਲੇਟ ਦੀ ਬਣਤਰ ਵਿਸ਼ਲੇਸ਼ਣ | ਸੰਰਚਨਾ ਅਤੇ ਕੰਫਿਗਰেশਨ ਖੋਜੋ |
| 30 ਮਿੰਟ | AI ਟੈਮਪਲੇਟ ਸੰਰਚਨਾ | ਉਪਲਬਧ ਫੀਚਰਾਂ ਨੂੰ ਚਾਲੂ ਅਤੇ ਟ੍ਰਾਈ ਕਰੋ |
| 30 ਮਿੰਟ | AI ਟੈਮਪਲੇਟ ਅਨੁਕੂਲਿਤ ਕਰੋ | ਆਪਣੇ ਲੋੜਾਂ ਅਨੁਸਾਰ ਟੈਮਪਲੇਟ ਨੂੰ ਅਨੁਕੂਲ ਬਣਾਓ |
| 30 ਮਿੰਟ | ਇੰਫਰਾਸਟ੍ਰਕਚਰ ਹਟਾਓ | ਸਾਫ਼-ਸਫ਼ਾਈ ਅਤੇ ਰਿਸੋਰਸ ਰਿਲੀਜ਼ ਕਰੋ |
| 15 ਮਿੰਟ | ਸੰਖੇਪ ਅਤੇ ਅਗਲੇ ਕਦਮ | ਸਿੱਖਣ ਸਰੋਤ, ਵਰਕਸ਼ਾਪ ਚੈਲੈਂਜ |

## ਤੁਸੀਂ ਕੀ ਸਿੱਖੋਗੇ

AZD ਟੈਂਪਲੇਟ ਨੂੰ ਇੱਕ ਸਿੱਖਣ ਵਾਲੇ ਸੈਂਡਬਾਕਸ ਵਜੋਂ ਸੋਚੋ ਜਿੱਥੇ ਤੁਸੀਂ Microsoft Foundry 'ਤੇ ਐਂਡ-ਟੂ-ਐਂਡ ਵਿਕਾਸ ਲਈ ਵੱਖ-ਵੱਖ ਸਮਰੱਥਾਵਾਂ ਅਤੇ ਟੂਲਾਂ ਦੀ ਜਾਂਚ ਕਰ ਸਕਦੇ ਹੋ। ਇਸ ਵਰਕਸ਼ਾਪ ਦੇ ਅੰਤ ਤੱਕ, ਤੁਹਾਨੂੰ ਇਸ ਸੰਦਰਭ ਵਿੱਚ ਵੱਖ-ਵੱਖ ਟੂਲਾਂ ਅਤੇ ਧਾਰਨਾਵਾਂ ਲਈ ਇੱਕ ਅਭਿਆਸਿਕ ਅਹਿਸਾਸ ਹੋਣਾ ਚਾਹੀਦਾ ਹੈ।

| ਧਾਰਨਾ  | ਉਦੇਸ਼ |
|:---|:---|
| **Azure Developer CLI** | ਟੂਲ ਦੇ ਕਮਾਂਡ ਅਤੇ ਵਰਕਫਲੋਜ਼ ਨੂੰ ਸਮਝੋ|
| **AZD Templates**| ਪ੍ਰੋਜੈਕਟ ਦੀ ਬਣਤਰ ਅਤੇ ਕੰਫਿਗਰ ਸਮਝੋ|
| **Azure AI Agent**| Microsoft Foundry ਪ੍ਰੋਜੈਕਟ ਦੀ ਪ੍ਰੋਵਿਜ਼ਨ ਅਤੇ ਡਿਪਲੋਇਮੈਂਟ |
| **Azure AI Search**| ਏਜੰਟਾਂ ਨਾਲ ਸੰਦਰਭ ਇੰਜੀਨੀਅਰਿੰਗ ਯੋਗ ਬਣਾਓ |
| **Observability**| ਟਰੇਸਿੰਗ, ਮਾਨੀਟਰਿੰਗ ਅਤੇ ਮੁਲਾਂਕਣ ਖੋਜੋ |
| **Red Teaming**| ਵਿਰੋਧੀ ਟੈਸਟਿੰਗ ਅਤੇ ਰਾਹਤ ਉਪਾਅ ਖੋਜੋ |
| | |
