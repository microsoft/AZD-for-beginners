# ਅਧਿਆਇ 1: ਬੁਨਿਆਦ ਅਤੇ ਜਲਦੀ ਸ਼ੁਰੂਆਤ

**📚 ਪਾਠਕ੍ਰਮ**: [AZD For Beginners](../../README.md) | **⏱️ ਅਵਧੀ**: 30-45 ਮਿੰਟ | **⭐ ਪੱਧਰ**: ਸ਼ੁਰੂਆਤੀ

---

## ਜਾਇਜ਼ਾ

ਇਸ ਅਧਿਆਇ ਵਿੱਚ Azure Developer CLI (`azd`) ਦੇ ਮੂਲ ਤੱਤਾਂ ਨੂੰ ਪੇਸ਼ ਕੀਤਾ ਗਿਆ ਹੈ। ਤੁਸੀਂ ਮੁੱਖ ਧਾਰਣਾਂ ਸਿੱਖੋਗੇ, ਟੂਲਜ਼ ਨੂੰ ਇੰਸਟਾਲ ਕਰੋਗੇ, ਅਤੇ ਆਪਣੀ ਪਹਿਲੀ ਐਪਲੀਕੇਸ਼ਨ ਨੂੰ Azure 'ਤੇ ਡਿਪਲੋਇ ਕਰਨਾ ਸਿੱਖੋਗੇ।

> ਜੂਨ 2026 ਵਿੱਚ `azd 1.25.6` ਦੇ ਨਾਲ ਪ੍ਰਮਾਣਿਤ।

## ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼

ਇਸ ਅਧਿਆਇ ਨੂੰ ਪੂਰਾ ਕਰਨ ਤੋਂ ਬਾਅਦ, ਤੁਸੀਂ:
- ਸਮਝੋਗੇ ਕਿ Azure Developer CLI ਕੀ ਹੈ ਅਤੇ ਇਹ Azure CLI ਤੋਂ ਕਿਵੇਂ ਵੱਖਰਾ ਹੈ
- ਆਪਣੇ ਪਲੇਟਫਾਰਮ 'ਤੇ AZD ਇੰਸਟਾਲ ਅਤੇ ਸੰਰਚਿਤ ਕਰੋਗੇ
- `azd up` ਨਾਲ ਆਪਣੀ ਪਹਿਲੀ ਐਪਲੀਕੇਸ਼ਨ ਨੂੰ Azure 'ਤੇ ਡਿਪਲੋਇ ਕਰੋਗੇ
- `azd down` ਨਾਲ ਰਿਸੋਰਸਾਂ ਨੂੰ ਸਾਫ਼ ਕਰੋਗੇ

## ✅ ਇੱਥੋਂ ਸ਼ੁਰੂ ਕਰੋ: ਆਪਣੀ ਸੈਟਅੱਪ ਦੀ ਜਾਂਚ ਕਰੋ

ਸ਼ੁਰੂ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ, ਪੁਸ਼ਟੀ ਕਰੋ ਕਿ ਤੁਹਾਡੀ ਲੋਕਲ ਮਸ਼ੀਨ Chapter 1 ਟੈਮਪਲੇਟ ਲਈ ਤਿਆਰ ਹੈ:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

ਜੇ ਸਕ੍ਰਿਪਟ ਘਾਟ ਟੂਲਾਂ ਦੀ ਰਿਪੋਰਟ ਕਰਦਾ ਹੈ, ਤਾਂ ਸਭ ਤੋਂ ਪਹਿਲਾਂ ਉਹ ਠੀਕ ਕਰੋ ਅਤੇ ਫਿਰ ਅਧਿਆਇ ਨਾਲ ਜਾਰੀ ਰੱਖੋ।

---

## 🚀 ਜਲਦੀ ਸ਼ੁਰੂਆਤ

```bash
# ਇੰਸਟਾਲੇਸ਼ਨ ਦੀ ਜਾਂਚ ਕਰੋ
azd version

# AZD ਲਈ ਪ੍ਰਮਾਣਿਕਤਾ ਕਰੋ
# ਆਪਸ਼ਨਲ: ਜੇ ਤੁਸੀਂ Azure CLI ਕਮਾਂਡਾਂ ਸਿੱਧੇ ਚਲਾਉਣ ਦੀ ਯੋਜਨਾ ਬਣਾਉਂਦੇ ਹੋ ਤਾਂ az login ਕਰੋ
azd auth login

# ਆਪਣੀ ਪਹਿਲੀ ਐਪ ਤੈਨਾਤ ਕਰੋ
azd init --template todo-nodejs-mongo
azd up

# ਜਦੋਂ ਮੁਕੰਮਲ ਹੋ ਜਾਵੇ ਤਾਂ ਸਾਫ਼-ਸੁੱਧਾ ਕਰੋ
azd down --force --purge
```

---

## ✅ ਸਫਲਤਾ ਦੇ ਮਾਪਦੰਡ

ਇਸ ਅਧਿਆਇ ਨੂੰ ਪੂਰਾ ਕਰਨ ਤੋਂ ਬਾਅਦ, ਤੁਸੀਂ ਇਹ ਕਰ ਸਕੋਗੇ:

```bash
azd version              # ਸਥਾਪਿਤ ਵਰਜ਼ਨ ਦਿਖਾਉਂਦਾ ਹੈ
azd init --template todo-nodejs-mongo  # ਪ੍ਰੋਜੈਕਟ ਨੂੰ ਆਰੰਭ ਕਰਦਾ ਹੈ
azd up                   # Azure ਤੇ ਤੈਨਾਤ ਕਰਦਾ ਹੈ
azd show                 # ਚੱਲ ਰਹੇ ਐਪ ਦਾ URL ਦਿਖਾਉਂਦਾ ਹੈ
azd down --force --purge # ਸੰਸਾਧਨਾਂ ਨੂੰ ਸਾਫ਼ ਕਰਦਾ ਹੈ
```
