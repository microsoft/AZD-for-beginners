# ਅਧਿਆਇ 1: ਬੁਨਿਆਦ & ਤੁਰੰਤ ਸ਼ੁਰੂ

**📚 ਕੋਰਸ**: [AZD ਸ਼ੁਰੂਆਤੀਆਂ ਲਈ](../../README.md) | **⏱️ ਅਵਧੀ**: 30-45 ਮਿੰਟ | **⭐ ਕਠਿਨਾਈ**: ਬੁਨਿਆਦੀ

---

## ਜਾਇਜ਼ਾ

ਇਸ ਅਧਿਆਇ ਵਿੱਚ Azure Developer CLI (azd) ਦੇ ਮੂਲ ਤੱਤ ਪੇਸ਼ ਕੀਤੇ ਜਾਣਗੇ। ਤੁਸੀਂ ਮੁੱਖ ਧਾਰਨਾਵਾਂ ਸਿੱਖੋਗੇ, ਟੂਲ ਇੰਸਟਾਲ ਕਰਨਾ ਸਿੱਖੋਗੇ, ਅਤੇ ਆਪਣੀ ਪਹਿਲੀ ਐਪਲੀਕੇਸ਼ਨ Azure 'ਤੇ ਡਿਪਲੋਈ ਕਰੋਗੇ।

## ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼

ਇਸ ਅਧਿਆਇ ਨੂੰ ਪੂਰਾ ਕਰਨ ਤੋਂ ਬਾਅਦ, ਤੁਸੀਂ:
- ਸਮਝ ਸਕੋਗੇ ਕਿ Azure Developer CLI ਕੀ ਹੈ ਅਤੇ ਇਹ Azure CLI ਤੋਂ ਕਿਵੇਂ ਵੱਖਰਾ ਹੈ
- ਆਪਣੇ ਪਲੇਟਫਾਰਮ 'ਤੇ AZD ਨੂੰ ਇੰਸਟਾਲ ਅਤੇ ਕਨਫਿਗਰ ਕਰ ਸਕੋਗੇ
- `azd up` ਨਾਲ ਆਪਣਾ ਪਹਿਲਾ ਐਪਲੀਕੇਸ਼ਨ Azure 'ਤੇ ਤੈਨਾਤ ਕਰ ਸਕੋਗੇ
- `azd down` ਨਾਲ ਸਰੋਤ ਸਾਫ਼ ਕਰ ਸਕੋਗੇ

## 🚀 ਤੁਰੰਤ ਸ਼ੁਰੂ

```bash
# ਸਥਾਪਨਾ ਦੀ ਜਾਂਚ ਕਰੋ
azd version

# Azure ਵਿੱਚ ਲੌਗਇਨ ਕਰੋ
azd auth login

# ਆਪਣੀ ਪਹਿਲੀ ਐਪ ਤੈਨਾਤ ਕਰੋ
azd init --template todo-nodejs-mongo
azd up

# ਮੁਕੰਮਲ ਹੋਣ ਤੇ ਸਾਫ़ ਕਰੋ
azd down --force --purge
```

## ✅ ਸਫਲਤਾ ਮਾਪਦੰਡ

ਇਸ ਅਧਿਆਇ ਨੂੰ ਪੂਰਾ ਕਰਨ ਤੋਂ ਬਾਅਦ, ਤੁਸੀਂ ਇਹ ਕਰਨ ਦੇ ਯੋਗ ਹੋਵੋਗੇ:

```bash
azd version              # ਸਥਾਪਿਤ ਵਰਜਨ ਦਿਖਾਉਂਦਾ ਹੈ
azd init --template todo-nodejs-mongo  # ਪ੍ਰੋਜੈਕਟ ਨੂੰ ਸ਼ੁਰੂ ਕਰਦਾ ਹੈ
azd up                   # Azure 'ਤੇ ਤैनਾਤ ਕਰਦਾ ਹੈ
azd show                 # ਚਲ ਰਹੀ ਐਪ ਦਾ URL ਦਿਖਾਉਂਦਾ ਹੈ
azd down --force --purge # ਸੰਸਾਧਨਾਂ ਨੂੰ ਸਾਫ਼ ਕਰਦਾ ਹੈ
```
