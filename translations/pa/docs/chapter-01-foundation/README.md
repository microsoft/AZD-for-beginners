# ਅਧਿਆਇ 1: ਬੁਨਿਆਦ ਅਤੇ ਤੇਜ਼ ਸ਼ੁਰੂਆਤ

**📚 ਕੋਰਸ**: [AZD ਬਿਲਕੁਲ ਸ਼ੁਰੂਆਤ ਕਰਨ ਵਾਲਿਆਂ ਲਈ](../../README.md) | **⏱️ ਸਮਾਂ**: 30-45 ਮਿੰਟ | **⭐ מורכבות**: ਸ਼ੁਰੂਆਤ ਕਰਨ ਵਾਲੇ ਲਈ

---

## ਝਲਕ

ਇਹ ਅਧਿਆਇ Azure Developer CLI (azd) ਦੇ ਬੁਨਿਆਦੀ ਤੱਤਾਂ ਦਾ ਪਰੀਚਯ ਕਰਵਾਉਂਦਾ ਹੈ। ਤੁਸੀਂ ਮੁੱਢਲੇ ਵਿਚਾਰਾਂ ਨੂੰ ਸਿੱਖੋਗੇ, ਟੂਲ ਇੰਸਟਾਲ ਕਰੋਗੇ, ਅਤੇ ਆਪਣੀ ਪਹਿਲੀ ਐਪਲੀਕੀਸ਼ਨ ਨੂੰ Azure 'ਤੇ ਡਿਪਲੋਇ ਕਰੋਗੇ।

> July 2026 ਵਿੱਚ `azd 1.27.1` ਨਾਲ ਸਚੈਤਾ ਕੀਤੀ ਗਈ ਹੈ।

## ਸਿੱਖਣ ਦੇ ਲਕੜੀ

ਇਸ ਅਧਿਆਇ ਨੂੰ ਪੂਰਾ ਕਰਨ ਤੋਂ ਬਾਦ ਤੁਸੀਂ:
- ਸਮਝ ਪਾਉਂਦੇ ਹੋ ਕਿ Azure Developer CLI ਕੀ ਹੈ ਅਤੇ ਇਹ Azure CLI ਤੋਂ ਕਿਵੇਂ ਵੱਖਰਾ ਹੈ
- ਆਪਣੇ ਪਲੇਟਫਾਰਮ 'ਤੇ AZD ਇੰਸਟਾਲ ਅਤੇ ਸੰਰਚਿਤ ਕਰੋ
- `azd up` ਨਾਲ ਆਪਣੀ ਪਹਿਲੀ ਐਪਲੀਕੇਸ਼ਨ ਨੂੰ Azure 'ਤੇ ਡਿਪਲੋਇ ਕਰੋ
- `azd down` ਨਾਲ ਰਿਸੋਰਸ ਸਾਫ਼ ਕਰੋ

## ✅ ਇੱਥੋਂ ਸ਼ੁਰੂ ਕਰੋ: ਆਪਣੀ ਸੈੱਟਅਪ ਦੀ ਪ੍ਰਮਾਣੀਕਰਣ ਕਰੋ

ਸ਼ੁਰੂ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ, ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਤੁਹਾਡੀ ਲੋਕਲ ਮਸ਼ੀਨ Chapter 1 ਟੈਂਪਲੇਟ ਲਈ ਤਿਆਰ ਹੈ:

**ਵਿੰਡੋਜ਼:**
```powershell
.\validate-setup.ps1
```

**macOS / ਲਿਨਕਸ:**
```bash
bash ./validate-setup.sh
```

ਜੇ ਸਕ੍ਰਿਪਟ ਟੂਲਾਂ ਦੀ ਕਮੀ ਰਿਪੋਰਟ ਕਰਦੀ ਹੈ, ਪਹਿਲਾਂ ਉਨ੍ਹਾਂ ਨੂੰ ਸਹੀ ਕਰੋ ਅਤੇ ਫਿਰ ਅਧਿਆਇ ਨਾਲ ਜਾਰੀ ਰੱਖੋ।

---

## 🚀 ਤੇਜ਼ ਸ਼ੁਰੂਆਤ

```bash
# ਇੰਸਟਾਲੇਸ਼ਨ ਦੀ ਜਾਂਚ ਕਰੋ
azd version

# AZD ਲਈ ਪ੍ਰਮਾਣਿਕਤਾ ਕਰੋ
# ਵਿਕਲਪਿਕ: ਜੇ ਤੁਸੀਂ Azure CLI ਕਮਾਂਡਾਂ ਸਿੱਧਾ ਚਲਾਉਣ ਦਾ ਯੋਜਨਾ ਬਣਾ ਰਹੇ ਹੋ ਤਾਂ az ਲੌਗਇਨ ਕਰੋ
azd auth login

# ਆਪਣਾ ਪਹਿਲਾ ਐਪ ਤੈਆਰ ਕਰੋ
azd init --template todo-nodejs-mongo
azd up

# ਖਤਮ ਹੋਣ 'ਤੇ ਸਾਫ ਕਰੋ
azd down --force --purge
```

---

## ✅ ਸੋਫਲਤਾ ਦੇ ਮਾਪਦੰਡ

ਇਹ ਅਧਿਆਇ ਮੁਕੰਮਲ ਕਰਨ ਤੋਂ ਬਾਅਦ, ਤੁਹਾਨੂੰ ਸਮਰੱਥ ਹੋਣਾ ਚਾਹੀਦਾ ਹੈ:

```bash
azd version              # ਇੰਸਟਾਲ ਕੀਤੀ ਗਈ ਵਰਜਨ ਦਿਖਾਉਂਦਾ ਹੈ
azd init --template todo-nodejs-mongo  # ਪ੍ਰੋਜੈਕਟ ਸ਼ੁਰੂ ਕਰਦਾ ਹੈ
azd up                   # ਅਜ਼ੂਰ 'ਤੇ ਡੀਪਲਾਏ ਕਰਦਾ ਹੈ
azd show                 # ਚਲ ਰਹੀ ਐਪ URL ਦਿਖਾਉਂਦਾ ਹੈ
azd down --force --purge # ਸਰੋਤ ਸਾਫ ਕਰਦਾ ਹੈ
```
