# ਅਧਿਆਇ 5: ਬਹੁ-ਏਜੰਟ ਐਆਈ ਹੱਲ

**📚 ਕੋਰਸ**: [AZD For Beginners](../../README.md) | **⏱️ ਸਮਾਂ**: 2-3 ਘੰਟੇ | **⭐ ਕਠਨਤਾ**: ਉੱਚ

---

## ਜਾਇਜ਼ਾ

ਇਹ ਅਧਿਆਇ ਉੱਨਤ ਬਹੁ-ਏਜੰਟ ਆਰਕੀਟੈਕਚਰ ਪੈਟਰਨ, ਏਜੰਟ ਸਮਨਵਯ, ਅਤੇ ਜਟਿਲ ਪਰਿਪ੍ਰੇਖਾਂ ਲਈ ਉਤਪਾਦਨ-ਤਿਆਰ AI ਡਿਪਲੋਇਮੈਂਟ ਨੂੰ ਕਵਰ ਕਰਦਾ ਹੈ।

## ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼

ਇਸ ਅਧਿਆਇ ਨੂੰ ਪੂਰਾ ਕਰਨ ਤੋਂ ਬਾਅਦ, ਤੁਸੀਂ:
- ਬਹੁ-ਏਜੰਟ ਆਰਕੀਟੈਕਚਰ ਪੈਟਰਨਸ ਨੂੰ ਸਮਝੋਗੇ
- ਸਮਨਵਿਤ AI ਏਜੰਟ ਸਿਸਟਮ ਤੈਨਾਤ ਕਰ ਸਕੋਗੇ
- ਏਜੰਟ-ਟੂ-ਏਜੰਟ ਸੰਚਾਰ ਨੂੰ ਲਾਗੂ ਕਰ ਸਕੋਗੇ
- ਉਤਪਾਦਨ-ਤਿਆਰ ਬਹੁ-ਏਜੰਟ ਹੱਲ ਤਿਆਰ ਕਰਨਗੇ

## 🚀 ਤੁਰੰਤ ਸ਼ੁਰੂਆਤ

```bash
# ਰਿਟੇਲ ਬਹੁ-ਏਜੰਟ ਸਮਾਧਾਨ ਨੂੰ ਤੈਨਾਤ ਕਰੋ
cd examples/retail-multiagent-arm-template
./deploy.sh

# ਜਾਂ ਸਿੱਧਾ ਟੈਂਪਲੇਟ ਵਰਤੋ
azd init --template agent-openai-python-prompty
azd up
```

```
┌──────────────────────────────────────────────────────────────┐
│                    Orchestrator Agent                         │
│              (Routes requests, manages workflow)              │
└────────────────────┬─────────────────┬───────────────────────┘
                     │                 │
         ┌───────────▼───────┐ ┌───────▼───────────┐
         │  Customer Agent   │ │  Inventory Agent  │

         │   preferences)    │ │   orders)         │
         └───────────────────┘ └───────────────────┘
```

### 🎯 ਪ੍ਰਮੁੱਖ ਹੱਲ: ਰਿਟੇਲ ਬਹੁ-ਏਜੰਟ

ਇਹ [ਰੀਟੇਲ ਬਹੁ-ਏਜੰਟ ਹੱਲ](../../examples/retail-scenario.md) ਦਰਸਾਉਂਦਾ ਹੈ:

- **ਕਸਟਮਰ ਏਜੰਟ**: ਉਪਭੋਗਤਾ ਇੰਟਰੈਕਸ਼ਨ ਅਤੇ ਪਸੰਦਾਂ ਨੂੰ ਸੰਭਾਲਦਾ ਹੈ
- **ਇਨਵੇਂਟਰੀ ਏਜੰਟ**: ਸਟਾਕ ਅਤੇ ਆਰਡਰ ਪ੍ਰਕਿਰਿਆ ਨੂੰ ਮੈਨੇਜ ਕਰਦਾ ਹੈ
- **ਸੰਯੋਜਕ**: ਏਜੰਟਾਂ ਦੇ ਵਿਚਕਾਰ ਸਮਨਵਯ ਕਰਦਾ ਹੈ
- **ਸਾਂਝੀ ਮੈਮੋਰੀ**: ਏਜੰਟ-ਪਾਰ ਸੰਦਰਭ ਪ੍ਰਬੰਧਨ
