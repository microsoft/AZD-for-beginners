# ਅਧਿਆਇ 4: ਇਨਫਰਾਸਟਰੱਕਚਰ ਐਜ਼ ਕੋਡ ਅਤੇ ਤैनਾਤੀ

**📚 ਕੋਰਸ**: [AZD ਸ਼ੁਰੂਆਤੀ ਲਈ](../../README.md) | **⏱️ ਸਮਾਂ**: 1-1.5 ਘੰਟੇ | **⭐ ਜਟਿਲਤਾ**: ਦਰਮਿਆਨਾ

---

## ਝਲਕ

ਇਸ ਅਧਿਆਇ ਵਿੱਚ Bicep ਟੈਮਪਲੇਟਸ ਨਾਲ ਇਨਫਰਾਸਟਰੱਕਚਰ ਐਜ਼ ਕੋਡ (IaC) ਦੇ ਨਮੂਨੇ, ਰਿਸੋਰਸ ਪ੍ਰੋਵਿਜ਼ਨਿੰਗ ਅਤੇ Azure Developer CLI ਦੀ ਵਰਤੋਂ ਨਾਲ ਤੈਨਾਤੀ ਦੀਆਂ ਰਣਨੀਤੀਆਂ ਕਵਰ ਕੀਤੀਆਂ ਗਈਆਂ ਹਨ।

> ਜੁਲਾਈ 2026 ਵਿੱਚ `azd 1.27.1` ਦੇ ਵਿਰੁੱਧ ਵੈਧਤਾ ਕੀਤੀ ਗਈ।

## ਸਿੱਖਣ ਦੇ ਲਕਸ਼ਯ

ਇਸ ਅਧਿਆਇ ਨੂੰ مکمل ਕਰਕੇ, ਤੁਸੀਂ:
- Bicep ਟੈਮਪਲੇਟ ਦੀ ਬਣਤਰ ਅਤੇ ਸਿੰਟੈਕਸ ਨੂੰ ਸਮਝੋਗੇ
- `azd provision` ਨਾਲ Azure ਰਿਸੋਰਸਾਂ ਨੂੰ ਪ੍ਰੋਵਾਇਜ਼ਨ ਕਰਵਾਉਂਦੇ ਹੋਏ
- `azd deploy` ਨਾਲ ਐਪਲੀਕੇਸ਼ਨ ਤੈਨਾਤ ਕਰਦੇ ਹੋਏ
- ਬਲੂ-ਗ੍ਰੀਨ ਅਤੇ ਰੋਲਿੰਗ ਤੈਨਾਤੀ ਰਣਨੀਤੀਆਂ ਨੂੰ ਲਾਗੂ ਕਰਨਗੇ

## 🚀 ਤੇਜ਼ ਸ਼ੁਰੂਆਤ

```bash
# ਟੈਮਪਲੇਟ ਤੋਂ ਸ਼ੁਰੂ ਕਰੋ
azd init --template azure-functions-python-v2-http

# ਜੋ ਬਣਾਇਆ ਜਾਵੇਗਾ ਉਸਦਾ ਪ੍ਰੀਵਿิว ਕਰੋ
azd provision --preview

# ਸਿਰਫ਼ ਢਾਂਚਾ ਬਣਾਉਣਾ
azd provision

# ਸਿਰਫ਼ ਕੋਡ ਨੂੰ ਡਿਪਲੋਅ ਕਰੋ
azd deploy

# ਜਾਂ ਦੋਹਾਂ ਨੂੰ ਇਕੱਠੇ
azd up
```

## 📁 AZD ਪ੍ਰੋਜੈਕਟ ਬਣਤਰ

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

## 🔧 ਜ਼ਰੂਰੀ ਕਮਾਂਡਾਂ

| ਕਮਾਂਡ | ਵਰਣਨ |
|---------|-------------|
| `azd init` | ਪ੍ਰੋਜੈਕਟ ਸ਼ੁਰੂ ਕਰੋ |
| `azd provision` | Azure ਰਿਸੋਰਸ ਬਣਾਓ |
| `azd deploy` | ਐਪਲੀਕੇਸ਼ਨ ਕੋਡ ਤੈਨਾਤ ਕਰੋ |
| `azd up` | ਪ੍ਰੋਵਿਜ਼ਨ + ਤੈਨਾਤ |
| `azd down` | ਸਾਰੇ ਰਿਸੋਰਸ ਹਟਾਓ |
