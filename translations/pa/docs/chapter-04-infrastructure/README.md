# ਅਧਿਆਇ 4: ਇੰਫਰਾਸਟਰੱਕਚਰ ਐਜ਼ ਕੋਡ ਅਤੇ ਡਿਪਲੋਇਮੈਂਟ

**📚 ਕੋਰਸ**: [AZD ਸ਼ੁਰੂਆਤੀਆਂ ਲਈ](../../README.md) | **⏱️ ਅਵਧੀ**: 1-1.5 ਘੰਟੇ | **⭐ ਜਟਿਲਤਾ**: ਦਰਮਿਆਨਾ

---

## ਓਵਰਵਿਊ

ਇਹ ਅਧਿਆਇ Bicep ਟੈਮਪਲੇਟਾਂ ਨਾਲ Infrastructure as Code (IaC) ਪੈਟਰਨ, ਰਿਸੋਰਸ ਪ੍ਰੋਵਿਜ਼ਨ ਅਤੇ Azure Developer CLI ਦੀ ਵਰਤੋਂ ਨਾਲ ਡਿਪਲੋਇਮੈਂਟ ਰਣਨੀਤੀਆਂ ਨੂੰ ਕਵਰ ਕਰਦਾ ਹੈ।

> ਮਾਰਚ 2026 ਵਿੱਚ `azd 1.23.12` ਨਾਲ ਪ੍ਰਮਾਣਿਤ।

## ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼

ਇਸ ਅਧਿਆਇ ਨੂੰ ਪੂਰਾ ਕਰਨ 'ਤੇ, ਤੁਸੀਂ:
- Bicep ਟੈਮਪਲੇਟ ਦੀ ਬਣਤਰ ਅਤੇ ਸਿੰਟੈਕਸ ਨੂੰ ਸਮਝੋਗੇ
- `azd provision` ਨਾਲ Azure ਰਿਸੋਰਸ ਪ੍ਰੋਵਿਜ਼ਨ ਕਰੋਗੇ
- `azd deploy` ਨਾਲ ਐਪਲੀਕੇਸ਼ਨ ਡਿਪਲੋਇ ਕਰੋਗੇ
- ਬਲੂ-ਗ੍ਰੀਨ ਅਤੇ ਰੋਲਿੰਗ ਡਿਪਲੋਇਮੈਂਟ ਰਣਨੀਤੀਆਂ ਲਾਗੂ ਕਰੋਗੇ

## 🚀 ਤੁਰੰਤ ਸ਼ੁਰੂਆਤ

```bash
# ਟੇਮਪਲੇਟ ਤੋਂ ਸ਼ੁਰੂ ਕਰੋ
azd init --template azure-functions-python-v2-http

# ਜੋ ਬਣਾਇਆ ਜਾਵੇਗਾ, ਉਸਦਾ ਪਹਿਲਾਂ ਨਜ਼ਾਰਾ ਵੇਖੋ
azd provision --preview

# ਸਿਰਫ਼ ਬੁਨਿਆਦੀ ਢਾਂਚਾ ਤਿਆਰ ਕਰੋ
azd provision

# ਸਿਰਫ਼ ਕੋਡ ਤੈਨਾਤ ਕਰੋ
azd deploy

# ਜਾਂ ਦੋਹਾਂ ਇਕੱਠੇ
azd up
```

## 📁 AZD ਪ੍ਰੋਜੈਕਟ ਢਾਂਚਾ

```
my-project/
├── azure.yaml            # ← Project definition
├── infra/                # ← Infrastructure as Code
│   ├── main.bicep        # Entry point
│   ├── main.parameters.json
│   └── modules/          # Reusable components
│       ├── web.bicep
│       └── database.bicep
└── src/                  # Application code
    └── api/
```

## 🔧 ਜ਼ਰੂਰੀ ਕਮਾਂਡ

| ਕਮਾਂਡ | ਵੇਰਵਾ |
|---------|-------------|
| `azd init` | ਪ੍ਰੋਜੈਕਟ ਇਨਿਸ਼ੀਅਲਾਈਜ਼ ਕਰੋ |
| `azd provision` | Azure ਰਿਸੋਰਸ ਬਣਾਓ |
| `azd deploy` | ਐਪਲੀਕੇਸ਼ਨ ਕੋਡ ਡਿਪਲੋਇ ਕਰੋ |
| `azd up` | ਪ੍ਰੋਵਿਜ਼ਨ + ਡਿਪਲੋਇ |
| `azd down` | ਸਾਰੇ ਰਿਸੋਰਸ ਮਿਟਾਓ |
