# ਅਧਿਆਇ 3: ਕੰਫਿਗਰੇਸ਼ਨ & ਪ੍ਰਮਾਣਿਕਤਾ

**📚 ਕੋਰਸ**: [AZD For Beginners](../../README.md) | **⏱️ ਸਮਾਂ**: 45-60 ਮਿੰਟ | **⭐ ਕਠਿਨਾਈ**: ਮੱਧਮ

---

## ਜਾਇਜ਼ਾ

ਇਸ ਅਧਿਆਇ ਵਿੱਚ Azure Developer CLI ਡਿਪਲੋਇਮੈਂਟਸ ਲਈ ਵਾਤਾਵਰਣ ਕਨਫਿਗਰੇਸ਼ਨ, ਪ੍ਰਮਾਣਿਕਤਾ ਪੈਟਰਨ ਅਤੇ ਸੁਰੱਖਿਆ ਲਈ ਸਰਵੋਤਮ ਅਭਿਆਸ ਕਵਰ ਕੀਤੇ ਗਏ ਹਨ।

## ਸਿੱਖਣ ਦੇ ਲਕੜ

ਇਸ ਅਧਿਆਇ ਨੂੰ ਪੂਰਾ ਕਰਕੇ, ਤੁਸੀਂ:
- AZD ਕਨਫਿਗਰੇਸ਼ਨ ਹਾਇਰਾਰਕੀ 'ਤੇ ਕਾਬੂ ਪਾਓਗੇ
- ਕਈ ਵਾਤਾਵਰਣਾਂ (dev, staging, prod) ਨੂੰ ਪ੍ਰਬੰਧਿਤ ਕਰੋਗੇ
- ਮੈਨੇਜਡ ਆਈਡੈਂਟਿਟੀ ਨਾਲ ਸੁਰੱਖਿਅਤ ਪ੍ਰਮਾਣਿਕਤਾ ਲਾਗੂ ਕਰੋਗੇ
- ਵਾਤਾਵਰਣ-ਵਿਸ਼ੇਸ਼ ਸੈਟਿੰਗਜ਼ ਕਨਫਿਗਰ ਕਰੋਗੇ

## 🚀 ਤੁਰੰਤ ਸ਼ੁਰੂਆਤ

```bash
# ਕਈ ਵਾਤਾਵਰਣ ਬਣਾਓ
azd env new dev
azd env new staging
azd env new prod

# ਵਾਤਾਵਰਣਾਂ ਬਦਲੋ
azd env select prod

# ਵਾਤਾਵਰਣ ਵੈਰੀਏਬਲ ਸੈੱਟ ਕਰੋ
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# ਸੰਰਚਨਾ ਵੇਖੋ
azd env get-values
```

---

## 🔧 ਕਨਫਿਗਰੇਸ਼ਨ ਹਾਇਰਾਰਕੀ

AZD ਸੈਟਿੰਗਜ਼ ਇਸ ਕ੍ਰਮ ਵਿੱਚ ਲਾਗੂ ਕਰਦਾ ਹੈ (ਬਾਅਦ ਵਾਲਾ ਪਹਿਲਾਂ ਵਾਲੇ ਨੂੰ ਓਵਰਰਾਈਡ ਕਰਦਾ ਹੈ):

1. **Default values** (built into templates)
2. **azure.yaml** (ਪ੍ਰੋਜੈਕਟ ਕਨਫਿਗਰੇਸ਼ਨ)
3. **Environment variables** (`azd env set`)
4. **Command-line flags** (`--location eastus`)

---

## 🔐 ਸੁਰੱਖਿਆ ਲਈ ਸਰਵੋਤਮ ਅਭਿਆਸ

```bash
# ਮੈਨੇਜਡ ਆਈਡੈਂਟੀਟੀ ਵਰਤੋ (ਸਿਫਾਰਸ਼ ਕੀਤੀ ਜਾਂਦੀ ਹੈ)
azd env set AZURE_USE_MANAGED_IDENTITY true

# ਪ੍ਰਮਾਣਿਕਤਾ ਦੀ ਸਥਿਤੀ ਦੀ ਜਾਂਚ ਕਰੋ
azd auth whoami
az account show

# ਜੇ ਲੋੜ ਹੋਵੇ ਤਾਂ ਦੁਬਾਰਾ ਪ੍ਰਮਾਣਿਕਤਾ ਕਰੋ
azd auth login
az login
```
