# ਅਧਿਆਏ 3: ਸੰਰਚਨਾ ਅਤੇ ਪ੍ਰਮਾਣਿਕਤਾ

**📚 ਕੋਰਸ**: [AZD ਸ਼ੁਰੂਆਤੀਆਂ ਲਈ](../../README.md) | **⏱️ ਅਵਧੀ**: 45-60 ਮਿੰਟ | **⭐ ਕਠਿਨਾਈ**: ਦਰਮਿਆਨਾ

---

## ਝਲਕ

ਇਸ ਅਧਿਆਏ ਵਿੱਚ Azure Developer CLI ਡਿਪਲੌਇਮੈਂਟਸ ਲਈ ਵਾਤਾਵਰਣ ਸੰਰਚਨਾ, ਪ੍ਰਮਾਣਿਕਤਾ ਪੈਟਰਨ ਅਤੇ ਸੁਰੱਖਿਆ ਦੀਆਂ ਸ਼੍ਰੇਸ਼ਠ ਪ੍ਰਥਾਵਾਂ ਬਾਰੇ ਜਾਣਕਾਰੀ ਦਿੱਤੀ ਗਈ ਹੈ।

> ਮਾਰਚ 2026 ਵਿੱਚ `azd 1.23.12` ਦੇ ਖ਼ਿਲਾਫ਼ ਜਾਂਚ ਕੀਤਾ ਗਿਆ।

## ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼

ਇਸ ਅਧਿਆਏ ਨੂੰ ਪੂਰਾ ਕਰਨ ਦੇ ਬਾਅਦ, ਤੁਸੀਂ:
- AZD ਸੰਰਚਨਾ ਹਾਇਰਾਰਕੀ 'ਤੇ ਮਾਹਰ ਹੋਵੋਗੇ
- ਕਈ ਵਾਤਾਵਰਣਾਂ (dev, staging, prod) ਦਾ ਪ੍ਰਬੰਧ ਕਰੋਗੇ
- ਪ੍ਰਬੰਧਿਤ ਆਈਡੈਂਟਿਟੀਆਂ ਨਾਲ ਸੁਰੱਖਿਅਤ ਪ੍ਰਮਾਣਿਕਤਾ ਲਾਗੂ ਕਰੋਗੇ
- ਵਾਤਾਵਰਨ-ਵਿਸ਼ੇਸ਼ ਸੈਟਿੰਗਾਂ ਕਨਫਿਗਰ ਕਰ ਸਕੋਗੇ

## 🚀 ਤੁਰੰਤ ਸ਼ੁਰੂਆਤ

```bash
# ਕਈ ਵਾਤਾਵਰਣ ਬਣਾਓ
azd env new dev
azd env new staging
azd env new prod

# ਵਾਤਾਵਰਣ ਬਦਲੋ
azd env select prod

# ਵਾਤਾਵਰਣ ਵੈਰੀਏਬਲ ਸੈੱਟ ਕਰੋ
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# ਕੰਫਿਗਰੇਸ਼ਨ ਵੇਖੋ
azd env get-values
```

---

## 🔧 ਸੰਰਚਨਾ ਹਾਇਰਾਰਕੀ

AZD ਇਹਨਾਂ ਕ੍ਰਮਾਂ ਵਿੱਚ ਸੈਟਿੰਗਾਂ ਲਗਾਉਂਦਾ ਹੈ (ਬਾਦ ਵਾਲਾ ਪਹਿਲੇ ਨੂੰ ਓਵਰਰਾਈਡ ਕਰਦਾ ਹੈ):

1. **ਡਿਫੌਲਟ ਮੁੱਲ** (ਟੈਮਪਲੇਟਾਂ ਵਿੱਚ ਬਣੇ ਹੋਏ)
2. **azure.yaml** (ਪ੍ਰਾਜੈਕਟ ਸੰਰਚਨਾ)
3. **ਵਾਤਾਵਰਣ ਵੈਰੀਏਬਲ** (`azd env set`)
4. **ਕਮਾਂਡ-ਲਾਈਨ ਫਲੈਗਸ** (`--location eastus`)

---

## 🔐 ਸੁਰੱਖਿਆ ਦੀਆਂ ਸ਼੍ਰੇਸ਼ਠ ਪ੍ਰਥਾਵਾਂ

```bash
# ਪਰਬੰਧਿਤ ਆਈਡੈਂਟਿਟੀ ਦੀ ਵਰਤੋਂ ਕਰੋ (ਸਿਫਾਰਸ਼ ਕੀਤੀ ਜਾਂਦੀ ਹੈ)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD ਪ੍ਰਮਾਣਿਕਤਾ ਦੀ ਸਥਿਤੀ ਚੈੱਕ ਕਰੋ
azd auth status

# ਵਿਕਲਪਕ: ਜੇ ਤੁਸੀਂ az ਕਮਾਂਡਾਂ ਚਲਾਉਣ ਦੀ ਯੋਜਨਾ ਬਣਾਉਂਦੇ ਹੋ ਤਾਂ Azure CLI ਸੰਦਰਭ ਦੀ ਪੁਸ਼ਟੀ ਕਰੋ
az account show

# ਲੋੜ ਹੋਵੇ ਤਾਂ ਦੁਬਾਰਾ ਪ੍ਰਮਾਣਿਕਤਾ ਕਰੋ
azd auth login

# ਵਿਕਲਪਕ: az ਕਮਾਂਡਾਂ ਲਈ Azure CLI ਪ੍ਰਮਾਣਿਕਤਾ ਨੂੰ ਤਾਜ਼ਾ ਕਰੋ
az login
```
