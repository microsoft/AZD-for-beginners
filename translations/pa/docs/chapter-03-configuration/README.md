# ਅਧਿਆਇ 3: ਸੰਰਚਨਾ ਅਤੇ ਪ੍ਰਮਾਣਿਕਤਾ

**📚 ਕੋਰਸ**: [AZD ਸ਼ੁਰੂਆਤੀਆਂ ਲਈ](../../README.md) | **⏱️ ਅਵਧੀ**: 45-60 ਮਿੰਟ | **⭐ ਜਟਿਲਤਾ**: ਮੱਧਮ

---

## ਜਾਇਜ਼ਾ

ਇਹ ਅਧਿਆਇ Azure Developer CLI ਡਿਪਲੋਇਮੈਂਟਸ ਲਈ ਵਾਤਾਵਰਣ ਸੰਰਚਨਾ, ਪ੍ਰਮਾਣਿਕਤਾ ਪੈਟਰਨ, ਅਤੇ ਸੁਰੱਖਿਆ ਦੀਆਂ ਸਰਵੋਤਮ ਪ੍ਰਥਾਵਾਂ ਨੂੰ ਕਵਰ ਕਰਦਾ ਹੈ।

> ਇਸ ਦੀ ਪੁਸ਼ਟੀ `azd 1.25.6` ਦੇ ਨਾਲ ਜੂਨ 2026 ਵਿੱਚ ਕੀਤੀ ਗਈ ਸੀ।

## ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼

ਇਸ ਅਧਿਆਇ ਨੂੰ ਪੂਰਾ ਕਰਕੇ, ਤੁਸੀਂ:
- AZD ਸੰਰਚਨਾ ਦੀ ਅਨੁਕ੍ਰਮਤਾ 'ਤੇ ਨਿਪੁੰਨ ਹੋਵੋਗੇ
- ਕਈ ਵਾਤਾਵਰਣਾਂ (dev, staging, prod) ਦਾ ਪ੍ਰਬੰਧ ਕਰਨਗੇ
- Managed identities ਨਾਲ ਸੁਰੱਖਿਅਤ ਪ੍ਰਮਾਣਿਕਤਾ ਲਾਗੂ ਕਰੋਗੇ
- ਵਾਤਾਵਰਣ-ਖਾਸ ਸੈਟਿੰਗਾਂ ਸੰਰਚਿਤ ਕਰੋਗੇ

## 🚀 ਤੁਰੰਤ ਸ਼ੁਰੂਆਤ

```bash
# ਕਈ ਵਾਤਾਵਰਣ ਬਣਾਓ
azd env new dev
azd env new staging
azd env new prod

# ਵਾਤਾਵਰਣ ਬਦਲੋ
azd env select prod

# ਵਾਤਾਵਰਣ ਵੈਰੀਏਬਲ ਸੈੱਟ ਕਰੋ
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# ਸੰਰਚਨਾ ਵੇਖੋ
azd env get-values
```

---

## 🔧 ਸੰਰਚਨਾ ਦੀ ਅਨੁਕ੍ਰਮਤਾ

AZD ਇਹ ਸੈਟਿੰਗਾਂ ਇਸ ਕ੍ਰਮ ਵਿੱਚ ਲਾਗੂ ਕਰਦਾ ਹੈ (ਬਾਅਦ ਵਾਲਾ ਪਹਿਲਾਂ ਵਾਲੇ ਨੂੰ ਓਵਰਰਾਈਡ ਕਰਦਾ ਹੈ):

1. **ਡਿਫੌਲਟ ਮੁੱਲ** (ਟੈਮਪਲੇਟਾਂ ਵਿੱਚ ਬਣੇ ਹੋਏ)
2. **azure.yaml** (ਪਰੋਜੈਕਟ ਸੰਰਚਨਾ)
3. **ਵਾਤਾਵਰਣ ਵੇਰੀਏਬਲ** (`azd env set`)
4. **ਕਮਾਂਡ-ਲਾਈਨ ਫ਼ਲੈਗ** (`--location eastus`)

---

## 🔐 ਸੁਰੱਖਿਆ ਲਈ ਸਰਵੋਤਮ ਅਭਿਆਸ

```bash
# ਮੈਨੇਜਡ ਆਈਡੈਂਟਿਟੀ ਦੀ ਵਰਤੋਂ ਕਰੋ (ਸਿਫਾਰਸ਼ ਕੀਤੀ ਜਾਂਦੀ ਹੈ)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD ਪ੍ਰਮਾਣਿਕਤਾ ਦੀ ਸਥਿਤੀ ਜਾਂਚੋ
azd auth status

# ਵਿਕਲਪਕ: ਜੇ ਤੁਸੀਂ az ਕਮਾਂਡ ਚਲਾਉਣ ਦੀ ਯੋਜਨਾ ਬਣਾਉਂਦੇ ਹੋ ਤਾਂ Azure CLI ਸੰਦਰਭ ਦੀ ਪੁਸ਼ਟੀ ਕਰੋ
az account show

# ਜੇ ਲੋੜ ਹੋਵੇ ਤਾਂ ਮੁੜ ਪ੍ਰਮਾਣਿਕਤਾ ਕਰੋ
azd auth login

# ਵਿਕਲਪਕ: az ਕਮਾਂਡਾਂ ਲਈ Azure CLI ਦੀ ਪ੍ਰਮਾਣਿਕਤਾ ਤਾਜ਼ਾ ਕਰੋ
az login
```
