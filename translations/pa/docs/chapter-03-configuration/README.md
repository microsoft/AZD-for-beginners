# ਅਧਿਆਇ 3: ਸੰਰਚਨਾ ਅਤੇ ਪ੍ਰਮਾਣਿਕਤਾ

**📚 ਕੋਰਸ**: [AZD ਸ਼ੁਰੂਆਤੀ ਲਈ](../../README.md) | **⏱️ ਸਮਾਂ**: 45-60 ਮਿੰਟ | **⭐ ਜਟਿਲਤਾ**: ਦਰਮਿਆਨਾ

---

## ਜਾਇਜ਼ਾ

ਇਹ ਅਧਿਆਇ ਐਜ਼ੂਰ ਡਿਵੈਲਪਰ CLI ਤਾਇਨਾਤੀਆਂ ਲਈ ਵਾਤਾਵਰਨ ਸੰਰਚਨਾ, ਪ੍ਰਮਾਣਿਕਤਾ ਪੈਟਰਨ ਅਤੇ ਸੁਰੱਖਿਆ ਦੀਆਂ ਸਭ ਤੋਂ ਵਧੀਆ ਰੀਤਾਂ ਨੂੰ ਢਕਦਾ ਹੈ।

> `azd 1.27.1` ਦੇ ਖਿਲਾਫ਼ ਜੁਲਾਈ 2026 ਵਿੱਚ ਪ੍ਰਮਾਣਿਤ।

## ਸਿੱਖਣ ਦੇ ਲਕੜੀ ਦੇ ਟੀਕੇ

ਇਸ ਅਧਿਆਇ ਨੂੰ ਪੂਰਾ ਕਰਕੇ, ਤੁਸੀਂ:
- AZD ਸੰਰਚਨਾ ਪੱਧਰੀ ਸ਼੍ਰੇਣੀ ਵਿੱਚ ਨਿਪੁੰਨ ਹੋਵੋਗੇ
- ਕਈ ਵਾਤਾਵਰਨਾਂ (ਡਿਵ, ਸਟੇਜਿੰਗ, ਪ੍ਰੋਡ) ਨੂੰ ਪ੍ਰਬੰਧਿਤ ਕਰੋਗੇ
- ਪ੍ਰਬੰਧਤ ਪਹਿਚਾਣਾਂ ਨਾਲ ਸੁਰੱਖਿਅਤ ਪ੍ਰਮਾਣਿਕਤਾ ਲਾਗੂ ਕਰੋਗੇ
- ਵਾਤਾਵਰਨ-ਵਿਸ਼ੇਸ਼ ਸੈਟਿੰਗਾਂ ਨੂੰ ਸੰਰਚਿਤ ਕਰੋਗੇ

## 🚀 ਤੇਜ਼ੀ ਨਾਲ ਸ਼ੁਰੂਆਤ

```bash
# ਕਈ ਵਾਤਾਵਰਣ ਬਣਾਓ
azd env new dev
azd env new staging
azd env new prod

# ਵਾਤਾਵਰਣ ਬਦਲੋ
azd env select prod

# ਵਾਤਾਵਰਣ ਵਰੀਏਬਲ ਸੈੱਟ ਕਰੋ
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# ਸੰਰਚਨਾ ਦੇਖੋ
azd env get-values
```

---

## 🔧 ਸੰਰਚਨਾ ਪੱਧਰੀ ਸ਼੍ਰੇਣੀ

AZD ਸੈਟਿੰਗਾਂ ਇਸ ਕ੍ਰਮ ਵਿੱਚ ਲਾਗੂ ਕਰਦਾ ਹੈ (ਬਾਅਦ ਵਾਲਾ ਪਹਿਲਾਂ ਵਾਲੇ ਨੂੰ ਓਵਰਰਾਈਡ ਕਰਦਾ ਹੈ):

1. **ਡਿਫੌਲਟ ਮੁੱਲ** (ਟੈਮਪਲੇਟ ਵਿੱਚ ਬਣਾਇਆ ਗਿਆ)
2. **azure.yaml** (ਪ੍ਰੋਜੈਕਟ ਸੰਰਚਨਾ)
3. **ਵਾਤਾਵਰਨ ਚਲਤਾਂ** (`azd env set`)
4. **ਕਮਾਂਡ-ਲਾਈਨ ਫਲੈਗ** (`--location eastus`)

---

## 🔐 ਸੁਰੱਖਿਆ ਦੀਆਂ ਸਭ ਤੋਂ ਵਧੀਆ ਰੀਤਾਂ

```bash
# ਪ੍ਰਬੰਧਿਤ ਪਹਚਾਨ ਦੀ ਵਰਤੋਂ ਕਰੋ (ਸਿਫਾਰਸ਼ ਕੀਤੀ ਗਈ)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD ਪ੍ਰਮਾਣਿਕਤਾ ਸਥਿਤੀ ਦੀ ਜਾਂਚ ਕਰੋ
azd auth status

# ਵਿਕਲਪਿਕ: ਜੇ ਤੁਹਾਨੂੰ az ਕਮਾਂਡਾਂ ਚਲਾਉਣੀ ਹੈ ਤਾਂ Azure CLI ਸੰਦਰਭ ਦੀ ਪੜਤਾਲ ਕਰੋ
az account show

# ਜ਼ਰੂਰਤ ਪੈਣ 'ਤੇ ਮੁੜ ਪ੍ਰਮਾਣਿਕਤਾ ਕਰੋ
azd auth login

# ਵਿਕਲਪਿਕ: az ਕਮਾਂਡਾਂ ਲਈ Azure CLI ਪ੍ਰਮਾਣਿਕਤਾ ਨੂੰ ਤਾਜ਼ਾ ਕਰੋ
az login
```
