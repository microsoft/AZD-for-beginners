# ਅਧਿਆਇ 8: ਉਤਪਾਦਨ ਅਤੇ ਐਂਟਰਪ੍ਰਾਈਜ਼ ਪੈਟਰਨ

**📚 ਕੋਰਸ**: [AZD ਸ਼ੁਰੂਆਤੀਆਂ ਲਈ](../../README.md) | **⏱️ ਅਵਧੀ**: 2-3 ਘੰਟੇ | **⭐ ਪੇਚੀਦਗੀ**: ਉੱਨਤ

---

## ਜਾਇਜ਼ਾ

ਇਹ ਅਧਿਆਇ ਐਂਟਰਪ੍ਰਾਈਜ਼-ਤਿਆਰ ਡਿਪਲੋਇਮੈਂਟ ਪੈਟਰਨ, ਸੁਰੱਖਿਆ ਕਠੋਰ ਕਰਨ, ਮਾਨੀਟਰਿੰਗ, ਅਤੇ ਉਤਪਾਦਨ AI ਵਰਕਲੋਡਸ ਲਈ ਲਾਗਤ ਅਪਟੀਮਾਈਜ਼ੇਸ਼ਨ ਨੂੰ ਕਵਰ ਕਰਦਾ ਹੈ।

## ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼

ਇਸ ਅਧਿਆਇ ਨੂੰ ਪੂਰਾ ਕਰਨ 'ਤੇ, ਤੁਸੀਂ:
- ਬਹੁ-ਰੀਜਨ ਰਿਹੈਜ਼ੀਲੈਂਟ ਐਪਲੀਕੇਸ਼ਨਾਂ ਨੂੰ ਡਿਪਲੋਇ ਕਰੋ
- ਐਂਟਰਪ੍ਰਾਈਜ਼ ਸੁਰੱਖਿਆ ਪੈਟਰਨ ਲਾਗੂ ਕਰੋ
- ਵਿਆਪਕ ਮਾਨੀਟਰਿੰਗ ਸੰਰਚਨਾ ਕਰੋ
- ਪੈਮਾਨੇ 'ਤੇ ਲਾਗਤਾਂ ਨੂੰ ਅਪਟੀਮਾਈਜ਼ ਕਰੋ
- AZD ਨਾਲ CI/CD ਪਾਈਪਲਾਈਨਾਂ ਸੈੱਟਅੱਪ ਕਰੋ

## 💰 ਲਾਗਤ ਅਪਟੀਮਾਈਜ਼ੇਸ਼ਨ

| ਰਣਨੀਤੀ | ਬਚਤ |
|----------|---------|
| ਜ਼ੀਰੋ ਤੱਕ ਸਕੇਲ ਕਰੋ (Container Apps) | 60-80% |
| ਡੈਵ ਲਈ consumption tiers ਵਰਤੋ | 50-70% |
| ਨਿਰਧਾਰਤ ਸਮੇਂ 'ਤੇ ਸਕੇਲਿੰਗ | 30-50% |
| ਰਿਜ਼ਰਵ ਕੀਤੀ ਸਮਰੱਥਾ | 20-40% |

```bash
# ਬਜਟ ਲਈ ਚੇਤਾਵਨੀਆਂ ਸੈੱਟ ਕਰੋ

  --budget-name "AI-Budget" \
  --amount 500 \
  --category Cost \
  --time-grain Monthly
```
