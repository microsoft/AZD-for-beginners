# ਅਧਿਆਏ 6: ਪ੍ਰੀ-ਡਿਪਲੋਇਮੈਂਟ ਯੋਜਨਾ ਅਤੇ ਪ੍ਰਮਾਣੀਕਰਨ

**📚 Course**: [AZD ਸ਼ੁਰੂਆਤੀਆਂ ਲਈ](../../README.md) | **⏱️ ਅਵਧੀ**: 1 ਘੰਟਾ | **⭐ ਕਠਨਾਈ**: ਦਰਮਿਆਨਾ

---

## ਜਾਇਜ਼ਾ

ਇਸ ਅਧਿਆਏ ਵਿੱਚ ਤੁਹਾਡੇ ਐਪਲੀਕੇਸ਼ਨ ਨੂੰ ਤੈਨਾਤ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਦੀਆਂ ਜਰੂਰੀ ਯੋਜਨਾ ਅਤੇ ਪ੍ਰਮਾਣੀਕਰਨ ਕਦਮ ਕਵਰ ਕੀਤੇ ਗਏ ਹਨ। ਸਹੀ ਸਮਰੱਥਾ ਯੋਜਨਾ, SKU ਚੋਣ, ਅਤੇ ਪ੍ਰੀਫਲਾਈਟ ਚੈੱਕਸ ਨਾਲ ਮਹਿੰਗੀਆਂ ਗਲਤੀਆਂ ਤੋਂ ਬਚਣਾ ਸਿੱਖੋ।

## ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼

ਇਸ ਅਧਿਆਏ ਨੂੰ مکمل ਕਰਕੇ, ਤੁਸੀਂ:
- ਤੈਨਾਤੀ ਤੋਂ ਪਹਿਲਾਂ ਪ੍ਰੀਫਲਾਈਟ ਚੈੱਕ ਚਲਾਉਣਾ
- ਸਮਰੱਥਾ ਦੀ ਯੋਜਨਾ ਬਣਾਉਣਾ ਅਤੇ ਸਰੋਤਾਂ ਦੀ ਲੋੜ ਦਾ ਅੰਦਾਜ਼ਾ ਲਗਾਉਣਾ
- ਲਾਗਤ ਅਭਿਆਸ ਲਈ ਉਚਿਤ SKUs ਚੁਣਨਾ
- ਨਿਗਰਾਨੀ ਲਈ Application Insights ਸੰਰਚਿਤ ਕਰਨਾ
- ਟੀਮ ਕੋਆਰਡੀਨੇਸ਼ਨ ਪੈਟਰਨ ਸਮਝਣਾ

## 🚀 ਤੁਰੰਤ ਸ਼ੁਰੂ

```bash
# ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਕੋਟਿਆਂ ਦੀ ਜਾਂਚ ਕਰੋ
az vm list-usage --location eastus --output table

# ਡਿਪਲੋਇਮੈਂਟ ਦਾ ਪ੍ਰੀਵਿਊ (ਕੋਈ ਸਰੋਤ ਨਹੀਂ ਬਣਾਏ ਜਾਣਗੇ)
azd provision --preview

# Bicep ਸਿੰਟੈਕਸ ਦੀ ਜਾਂਚ ਕਰੋ
az bicep build --file infra/main.bicep

# ਮਾਹੌਲ ਸੰਰਚਨਾ ਦੀ ਜਾਂਚ ਕਰੋ
azd env get-values
```

## ☑️ ਪ੍ਰੀ-ਡਿਪਲੋਇਮੈਂਟ ਚੈੱਕਲਿਸਟ

### `azd provision` ਤੋਂ ਪਹਿਲਾਂ

- [ ] ਖੇਤਰ ਲਈ ਕੋਟਾ ਦੀ ਪੁਸ਼ਟੀ ਕੀਤੀ ਗਈ
- [ ] SKUs ਉਚਿਤ ਤਰੀਕੇ ਨਾਲ ਚੁਣੇ ਗਏ
- [ ] ਲਾਗਤ ਦਾ ਅੰਦਾਜ਼ਾ ਸਮੀਖਿਆ ਕੀਤਾ ਗਿਆ
- [ ] ਨਾਮਕਰਨ ਰੀਤੀ ਇੱਕਸਾਰ ਹੈ
- [ ] ਸੁਰੱਖਿਆ/RBAC ਸੰਰਚਿਤ ਕੀਤਾ ਗਿਆ

### `azd deploy` ਤੋਂ ਪਹਿਲਾਂ

- [ ] ਇਨਵਾਇਰਨਮੈਂਟ ਵੈਰੀਏਬਲਸ ਸੈੱਟ ਕੀਤੇ ਗਏ
- [ ] Secrets ਨੂੰ Key Vault ਵਿੱਚ ਰੱਖਿਆ ਗਿਆ
- [ ] ਕਨੈਕਸ਼ਨ ਸਟ੍ਰਿੰਗਜ਼ ਦੀ ਪੁਸ਼ਟੀ ਕੀਤੀ ਗਈ
- [ ] ਹੈਲਥ ਚੈੱਕਸ ਸੰਰਚਿਤ ਕੀਤੇ ਗਏ
