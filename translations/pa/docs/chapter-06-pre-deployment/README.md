# ਅਧਿਆਇ 6: ਪ੍ਰੀ-ਡਿਪਲੋਇਮੈਂਟ ਯੋਜਨਾ ਅਤੇ ਵੈਧਤਾ

**📚 ਕੋਰਸ**: [AZD For Beginners](../../README.md) | **⏱️ ਅਵਧੀ**: 1 ਘੰਟਾ | **⭐ ਕਠਨਾਈ**: ਮੱਧਮ

---

## ਜਾਇਜ਼ਾ

ਇਸ ਅਧਿਆਇ ਵਿੱਚ ਤੁਹਾਡੇ ਐਪਲੀਕੇਸ਼ਨ ਨੂੰ ਡਿਪਲੋਇ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਆਵਸ਼ਯਕ ਯੋਜਨਾ ਅਤੇ ਵੈਧਤਾ ਕਦਮ ਸ਼ਾਮِل ਹਨ। ਠੀਕ ਕੈਪਾਸਿਟੀ ਯੋਜਨਾ, SKU ਚੋਣ ਅਤੇ ਪ੍ਰੀਫਲਾਈਟ ਚੈੱਕਸ ਨਾਲ ਮਹਿੰਗੀਆਂ ਗਲਤੀਆਂ ਤੋਂ ਬਚਣਾ ਸਿੱਖੋ।

> ਇਹ `azd 1.25.6` ਦੇ ਖਿਲਾਫ ਜੂਨ 2026 ਵਿੱਚ ਵੈਧ ਕੀਤਾ ਗਿਆ ਸੀ।

## ਸਿੱਖਣ ਦੇ ਲਕਸ਼

ਇਸ ਅਧਿਆਇ ਨੂੰ ਪੂਰਾ ਕਰਨ 'ਤੇ, ਤੁਸੀਂ:
- ਡਿਪਲੋਇਮੈਂਟ ਤੋਂ ਪਹਿਲਾਂ ਪ੍ਰੀਫਲਾਈਟ ਚੈੱਕ ਚਲਾਉਣਾ
- ਕੈਪਾਸਿਟੀ ਦੀ ਯੋਜਨਾ ਬਣਾਉਣਾ ਅਤੇ ਸਾਧਨਾਂ ਦੀ ਲੋੜ ਦਾ ਅੰਦਾਜ਼ਾ ਲਗਾਉਣਾ
- ਲਾਗਤ ਆਪਟੀਮਾਈਜ਼ੇਸ਼ਨ ਲਈ ਉਚਿਤ SKUs ਚੁਣਨਾ
- ਮਾਨੀਟਰਿੰਗ ਲਈ Application Insights ਨੂੰ ਕੰਫਿਗਰ ਕਰਨਾ
- ਟੀਮ ਕੋਆਰਡੀਨੇਸ਼ਨ ਪੈਟਰਨਸ ਨੂੰ ਸਮਝਣਾ

## 🚀 ਤੁਰੰਤ ਸ਼ੁਰੂਆਤ

```bash
# ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਕੋਟਿਆਂ ਦੀ ਜਾਂਚ ਕਰੋ
az vm list-usage --location eastus --output table

# ਤੈਨਾਤੀ ਦਾ ਪ੍ਰੀਵਿਊ (ਕੋਈ ਸੰਸਾਧਨ ਨਹੀਂ ਬਣਾਏ ਜਾਣਗੇ)
azd provision --preview

# Bicep ਵਾਕਰਚਨਾ ਦੀ ਜਾਂਚ ਕਰੋ
az bicep build --file infra/main.bicep

# ਵਾਤਾਵਰਣ ਸੰਰਚਨਾ ਦੀ ਜਾਂਚ ਕਰੋ
azd env get-values
```

## ☑️ ਪ੍ਰੀ-ਡਿਪਲੋਇਮੈਂਟ ਚੈੱਕਲਿਸਟ

### `azd provision` ਤੋਂ ਪਹਿਲਾਂ

- [ ] ਰੀਜਨ ਲਈ ਕੋਟਾ ਦੀ ਪੁਸ਼ਟੀ ਕੀਤੀ ਗਈ
- [ ] ਉਚਿਤ ਤਰੀਕੇ ਨਾਲ SKUs ਚੁਣੇ ਗਏ
- [ ] ਲਾਗਤ ਅੰਦਾਜ਼ਾ ਸਮੀਖਿਆ ਕੀਤਾ ਗਿਆ
- [ ] ਨਾਮਕਰਨ ਰੀਤੀ ਇਕਸਾਰ ਹੈ
- [ ] ਸੁਰੱਖਿਆ/RBAC ਸੰਰਚਿਤ ਹੈ

### `azd deploy` ਤੋਂ ਪਹਿਲਾਂ

- [ ] ਐਨਵਾਇਰਨਮੈਂਟ ਵੈਰੀਏਬਲ ਸੈੱਟ ਕੀਤੇ ਗਏ
- [ ] Key Vault ਵਿੱਚ Secrets
- [ ] ਕਨੈਕਸ਼ਨ ਸਟਰਿੰਗਸ ਦੀ ਜਾਂਚ ਕੀਤੀ ਗਈ
- [ ] ਹੈਲਥ ਚੈੱਕ ਸੰਰਚਿਤ ਕੀਤੇ ਗਏ
