# ਅਧਿਆਇ 6: ਪ੍ਰੀ-ਡਿਪਲੌਇਮੈਂਟ ਯੋਜਨਾ ਅਤੇ ਪ੍ਰਮਾਣਿਕਤਾ

**📚 ਕੋਰਸ**: [AZD For Beginners](../../README.md) | **⏱️ ਸਮਾਂ**: 1 ਘੰਟਾ | **⭐ ਕਠਿਨਾਈ**: ਦਰਮਿਆਨਾ

---

## ਜਾਇਜ਼ਾ

ਇਸ ਅਧਿਆਇ ਵਿੱਚ ਤੁਹਾਡੇ ਐਪਲੀਕੇਸ਼ਨ ਨੂੰ ਡਿਪਲੌਇ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਜਰੂਰੀ ਯੋਜਨਾ ਅਤੇ ਪ੍ਰਮਾਣਿਕਤਾ ਦੇ ਕਦਮ ਕਵਰ ਕੀਤੇ ਗਏ ਹਨ। ਢੁਕਵਾਂ ਕੈਪੇਸਿਟੀ ਯੋਜਨਾ, SKU ਚੋਣ, ਅਤੇ ਪ੍ਰੀਫਲਾਈਟ ਚੈੱਕਸ ਨਾਲ ਮਹਿੰਗੀਆਂ ਗਲਤੀਆਂ ਤੋਂ ਬਚੋ।

> March 2026 ਵਿੱਚ `azd 1.23.12` ਦੇ ਖ਼ਿਲਾਫ਼ ਪ੍ਰਮਾਣਿਤ ਕੀਤਾ ਗਿਆ।

## ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼

ਇਸ ਅਧਿਆਇ ਨੂੰ ਪੂਰਾ ਕਰਨ ਨਾਲ, ਤੁਸੀਂ:
- ਡਿਪਲੌਇਮੈਂਟ ਤੋਂ ਪਹਿਲਾਂ ਪ੍ਰੀਫਲਾਈਟ ਚੈੱਕ ਚਲਾਉਣਾ
- ਕੈਪੇਸਿਟੀ ਦੀ ਯੋਜਨਾ ਬਣਾਉਣਾ ਅਤੇ ਸਰੋਤਾਂ ਦੀ ਲੋੜ ਦਾ ਅੰਦਾਜ਼ਾ ਲਗਾਉਣਾ
- ਖਰਚ ਸੂਧ ਦੇ ਲਈ ਢੁਕਵੀਆਂ SKU ਚੁਣਨਾ
- ਮੋਨੀਟਰਿੰਗ ਲਈ Application Insights ਕੰਫਿਗਰ ਕਰਨਾ
- ਟੀਮ ਕੋਆਰਡੀਨੇਸ਼ਨ ਪੈਟਰਨ ਸਮਝਣਾ

## 🚀 ਤੁਰੰਤ ਸ਼ੁਰੂਆਤ

```bash
# ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਕੋਟਿਆਂ ਦੀ ਜਾਂਚ ਕਰੋ
az vm list-usage --location eastus --output table

# ਡਿਪਲੋਇਮੈਂਟ ਦਾ ਪ੍ਰੀਵਿਊ (ਕੋਈ ਰਿਸੋਰਸ ਨਹੀਂ ਬਣਾਏ ਜਾਣਗੇ)
azd provision --preview

# Bicep ਸਿੰਟੈਕਸ ਦੀ ਪੁਸ਼ਟੀ ਕਰੋ
az bicep build --file infra/main.bicep

# ਵਾਤਾਵਰਣ ਸੰਰਚਨਾ ਦੀ ਜਾਂਚ ਕਰੋ
azd env get-values
```

## ☑️ ਪ੍ਰੀ-ਡਿਪਲੌਇਮੈਂਟ ਚੈਕਲਿਸਟ

### `azd provision` ਤੋਂ ਪਹਿਲਾਂ

- [ ] ਇਲਾਕੇ ਲਈ ਕੋਟਾ ਦੀ ਪੁਸ਼ਟੀ ਕੀਤੀ ਗਈ
- [ ] SKU ਯਥੋচিত ਰੂਪ ਵਿੱਚ ਚੁਣੇ ਗਏ
- [ ] ਲਾਗਤ ਦੇ ਅੰਦਾਜ਼ੇ ਦੀ ਸਮੀਖਿਆ ਕੀਤੀ ਗਈ
- [ ] ਨਾਮਕਰਨ ਰਵਾਇਤ ਇਕਸਾਰ ਹੈ
- [ ] ਸੁਰੱਖਿਆ/RBAC ਸੰਰਚਿਤ ਹੈ

### `azd deploy` ਤੋਂ ਪਹਿਲਾਂ

- [ ] ਵਾਤਾਵਰਣ ਪਰਿਵਰਤਕ ਸੈੱਟ ਕੀਤੇ ਗਏ
- [ ] Key Vault ਵਿੱਚ ਸਿਕ੍ਰੇਟਸ
- [ ] ਕਨੈਕਸ਼ਨ ਸਟਰਿੰਗਜ਼ ਦੀ ਪੁਸ਼ਟੀ ਕੀਤੀ ਗਈ
- [ ] ਹੈਲਥ ਚੈੱਕਸ ਸੰਰਚਿਤ ਕੀਤੇ ਗਏ
