# ਅਧਿਆਇ 6: ਡਿਪਲੋਇਮੈਂਟ ਤੋਂ ਪਹਿਲਾਂ ਦੀ ਯੋਜਨਾ ਅਤੇ ਪ੍ਰਮਾਣਿਕਤਾ

**📚 ਕੋਰਸ**: [AZD ਸ਼ੁਰੂਆਤੀਆਂ ਲਈ](../../README.md) | **⏱️ ਸਮਾਂ**: 1 ਘੰਟਾ | **⭐ ਜਟਿਲਤਾ**: ਦਰਮਿਆਨਾ

---

## ਝਲਕ

ਇਹ ਅਧਿਆਇ ਤੁਹਾਡੇ ਐਪਲੀਕੇਸ਼ਨ ਨੂੰ ਡਿਪਲੋਇਤ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਜ਼ਰੂਰੀ ਯੋਜਨਾ ਅਤੇ ਪ੍ਰਮਾਣਿਕਤਾ ਦੇ ਕਦਮ ਕਵਰ ਕਰਦਾ ਹੈ। ਢੁੱਕਵੀਂ ਕੈਪੇਸਿਟੀ ਯੋਜਨਾ, SKU ਚੋਣ ਅਤੇ ਪ੍ਰੀਫਲਾਈਟ ਚੈੱਕਸ ਨਾਲ ਮਹਿੰਗੀਆਂ ਗਲਤੀਆਂ ਤੋਂ ਬਚਣਾ ਸਿੱਖੋ।

## ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼

ਇਸ ਅਧਿਆਇ ਨੂੰ ਪੂਰਾ ਕਰਨ 'ਤੇ, ਤੁਸੀਂ:
- ਡਿਪਲੋਇਮੈਂਟ ਤੋਂ ਪਹਿਲਾਂ ਪ੍ਰੀਫਲਾਈਟ ਚੈੱਕ ਚਲਾਓ
- ਕੈਪੇਸਿਟੀ ਦੀ ਯੋਜਨਾ ਬਣਾਓ ਅਤੇ ਸਰੋਤ ਲੋੜਾਂ ਦਾ ਅੰਦਾਜ਼ਾ ਲਗਾਓ
- ਲਾਗਤ ਅਨੁਕੂਲ ਕਰਨ ਲਈ ਉਚਿਤ SKU ਚੁਣੋ
- ਮਾਨੀਟਰਿੰਗ ਲਈ Application Insights ਨੂੰ ਕੰਫਿਗਰ ਕਰੋ
- ਟੀਮ ਸਮਨਵੇਅ ਪੈਟਰਨਾਂ ਨੂੰ ਸਮਝੋ

## 🚀 ਤੁਰੰਤ ਸ਼ੁਰੂਆਤ

```bash
# ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਕੁਆਟਿਆਂ ਦੀ ਜਾਂਚ ਕਰੋ
az vm list-usage --location eastus --output table

# ਡਿਪਲੋਇਮੈਂਟ ਦਾ ਪ੍ਰੀਵਿਊ (ਕੋਈ ਸੰਸਾਧਨ ਬਣਾਏ ਨਹੀਂ ਜਾਣਗੇ)
azd provision --preview

# Bicep ਸਿੰਟੈਕਸ ਦੀ ਜਾਂਚ ਕਰੋ
az bicep build --file infra/main.bicep

# ਵਾਤਾਵਰਣ ਕੰਫਿਗਰੇਸ਼ਨ ਦੀ ਜਾਂਚ ਕਰੋ
azd env get-values
```

## ☑️ ਡਿਪਲੋਇਮੈਂਟ ਤੋਂ ਪਹਿਲਾਂ ਦੀ ਜਾਂਚ-ਸੂਚੀ

### `azd provision` ਤੋਂ ਪਹਿਲਾਂ

- [ ] ਰੀਜਨ ਲਈ ਕੋਟਾ ਪੁਸ਼ਟੀ ਕੀਤੀ ਗਈ
- [ ] SKUs ਉਚਿਤ ਤਰੀਕੇ ਨਾਲ ਚੁਣੇ ਗਏ
- [ ] ਲਾਗਤ ਅੰਦਾਜ਼ੇ ਦੀ ਸਮੀਖਿਆ ਕੀਤੀ ਗਈ
- [ ] ਨਾਮਕਰਨ ਰੀਤ ਇੱਕਸਾਰ ਹੈ
- [ ] ਸੁਰੱਖਿਆ/RBAC ਸੰਰਚਿਤ ਕੀਤੀ ਗਈ

### `azd deploy` ਤੋਂ ਪਹਿਲਾਂ

- [ ] ਵਾਤਾਵਰਣ ਵੈਰੀਏਬਲ ਸੈੱਟ ਕੀਤੇ ਗਏ
- [ ] Secrets Key Vault ਵਿੱਚ ਸੁਰੱਖਿਅਤ ਕੀਤੇ ਗਏ
- [ ] ਕਨੈਕਸ਼ਨ ਸਟਰਿੰਗਸ ਦੀ ਜਾਂਚ ਕੀਤੀ ਗਈ
- [ ] ਹੈਲਥ ਚੈੱਕਸ ਸੰਰਚਿਤ ਕੀਤੇ ਗਏ
