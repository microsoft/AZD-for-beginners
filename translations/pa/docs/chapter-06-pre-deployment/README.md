# ਅਧਿਆਇ 6: ਪ੍ਰੀ-ਡਿਪਲਾਇਮੈਂਟ ਯੋਜਨਾ ਬਨਾਣਾ ਅਤੇ ਪ੍ਰਮਾਣੀਕਰਨ

**📚 ਕੋਰਸ**: [ਬਿਗਿਨਰਜ਼ ਲਈ AZD](../../README.md) | **⏱️ ਸਮਾਂ**: 1 ਘੰਟਾ | **⭐ ਜਟਿਲਤਾ**: ਦਰਮਿਆਨਾ

---

## ਝਲਕ

ਇਹ ਅਧਿਆਇ ਤੁਹਾਡੇ ਐਪਲੀਕੇਸ਼ਨ ਨੂੰ ਡਿਪਲਾਇ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਜਰੂਰੀ ਯੋਜਨਾ ਬਣਾਉਣ ਅਤੇ ਪ੍ਰਮਾਣੀਕਰਨ ਕਦਮਾਂ ਨੂੰ ਕਵਰ ਕਰਦਾ ਹੈ। ਢੁੱਕਵੀਂ ਸਮਰੱਥਾ ਯੋਜਨਾ, SKU ਚੋਣ, ਅਤੇ ਪ੍ਰੀਫਲਾਈਟ ਚੈੱਕਸ ਨਾਲ ਮਹਿੰਗੀਆਂ ਗਲਤੀਆਂ ਤੋਂ ਬਚਣਾ ਸਿੱਖੋ।

> ਜੁਲਾਈ 2026 ਵਿੱਚ `azd 1.27.1` ਨਾਲ ਪ੍ਰਮਾਣਿਤ।

## ਸਿੱਖਣ ਦੇ ਲਕਸ਼

ਇਸ ਅਧਿਆਇ ਨੂੰ ਪੂਰਾ ਕਰ ਕੇ, ਤੁਸੀਂ:
- ਡਿਪਲਾਇਮੈਂਟ ਤੋਂ ਪਹਿਲਾਂ ਪ੍ਰੀਫਲਾਈਟ ਚੈੱਕ ਚਲਾਉਣੇ
- ਸਮਰੱਥਾ ਦੀ ਯੋਜਨਾ ਬਣਾਉਣ ਅਤੇ ਸਰੋਤਾਂ ਦੀ ਲੋੜ ਦਾ ਅੰਦਾਜ਼ਾ ਲਗਾਉਣ
- ਲਾਗਤ ਬਚਾਉਣ ਲਈ ਉਚਿਤ SKU ਚੁਣਨ
- ਮਾਨੀਟਰਿੰਗ ਲਈ ਐਪਲੀਕੇਸ਼ਨ ਇਨਸਾਈਟਸ ਸੈਟਅੱਪ ਕਰਨਾ
- ਟੀਮ ਕੋਆਰਡੀਨੇਸ਼ਨ ਦੇ ਪੈਟਰਨ ਸਮਝਣ

## 🚀 ਤੇਜ਼ ਸ਼ੁਰੂਆਤ

```bash
# ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਕੋਟਾ ਚੈੱਕ ਕਰੋ
az vm list-usage --location eastus --output table

# ਡਿਪਲੋਯਮੈਂਟ ਦਾ ਪ੍ਰੀਵਿਊ (ਕੋਈ ਸਰੋਤ ਬਣਾਏ ਨਹੀਂ ਗਏ)
azd provision --preview

# ਬਾਈਸਪ ਸੰਰਚਨਾ ਨੂੰ ਸਹੀ ਥਿੱਥਾ ਜਾਚੋ
az bicep build --file infra/main.bicep

# ਵਾਤਾਵਰਣ ਸੰਰਚਨਾ ਦੀ ਜਾਂਚ ਕਰੋ
azd env get-values
```

## ☑️ ਪ੍ਰੀ-ਡਿਪਲਾਇਮੈਂਟ ਚੈੱਕਲਿਸਟ

### `azd provision` ਤੋਂ ਪਹਿਲਾਂ

- [ ] ਖੇਤਰ ਲਈ ਕੁਆਟਾ ਦੀ ਪੁਸ਼ਟੀ ਕੀਤੀ
- [ ] SKU ਯਥੋਚਿਤ ਚੁਣੇ ਗਏ
- [ ] ਲਾਗਤ ਅੰਦਾਜ਼ਾ ਸਮੀਖਿਆ ਕੀਤਾ
- [ ] ਨਾਂਕਰਨ ਸੰਸਕਾਰ ਯਥਿਰੂਪ
- [ ] ਸੁਰੱਖਿਆ/RBAC ਸੈੱਟ ਕੀਤਾ

### `azd deploy` ਤੋਂ ਪਹਿਲਾਂ

- [ ] ਵਾਤਾਵਰਣ ਵੈਰੀਏਬਲ ਸੈੱਟ ਕੀਤੇ
- [ ] ਗੁਪਤ ਚੀਜ਼ਾਂ Key Vault ਵਿੱਚ
- [ ] ਕਨੈਕਸ਼ਨ ਸਟਰਿੰਗਜ਼ ਦੀ ਜਾਂਚ ਕੀਤੀ
- [ ] ਸਿਹਤ ਚੈੱਕਸ ਕੰਫਿਗਰ ਕੀਤੇ
