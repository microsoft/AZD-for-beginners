# ਅਧਿਆਇ 7: ਸਮੱਸਿਆ ਨਿਵਾਰਣ ਅਤੇ ਡੀਬੱਗਿੰਗ

**📚 ਕੋਰਸ**: [AZD ਖ਼ਾਸ ਸ਼ੁਰੂਆਤੀ ਲਈ](../../README.md) | **⏱️ ਸਮਾਂ**: 1-1.5 ਘੰਟੇ | **⭐ ਜਟਿਲਤਾ**: ਦਰਮਿਆਨਾ

---

## ਝਲਕ

ਇਹ ਅਧਿਆਇ ਤੁਹਾਡੇ ਦੀ ਸਹਾਇਤਾ ਕਰਦਾ ਹੈ ਕਿ ਤੁਸੀਂ Azure Developer CLI ਨਾਲ ਕੰਮ ਕਰਦੇ ਸਮੇਂ ਆਮ ਸਮੱਸਿਆਵਾਂ ਦੀ ਪਹਚਾਣ ਕਰ ਸਕੋ ਅਤੇ ਹੱਲ ਕਰ ਸਕੋ। ਤੈਨਾਤੀ ਵਿਚ ਅਸਫਲਤਾਵਾਂ ਤੋਂ ਲੈ ਕੇ AI-ਵਿਸ਼ੇਸ਼ ਸਮੱਸਿਆਵਾਂ ਤੱਕ।

> `azd 1.27.1` ਨਾਲ ਜੁਲਾਈ 2026 ਵਿਚ ਪ੍ਰਮਾਣੀਕृत।

## ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼

ਇਸ ਅਧਿਆਇ ਨੂੰ ਪੂਰਾ ਕਰਕੇ, ਤੁਸੀਂ:
- ਆਮ AZD ਤੈਨਾਤੀ ਅਸਫਲਤਾਵਾਂ ਦੀ ਪਹਚਾਣ ਕਰੋਗੇ
- ਪ੍ਰਮਾਣਿਕਤਾ ਅਤੇ ਅਧਿਕਾਰ ਸਮੱਸਿਆਵਾਂ ਨੂੰ ਡੀਬੱਗ ਕਰੋਗੇ
- AI ਸੇਵਾ ਕਨੈਕਟਿਵਿਟੀ ਸਮੱਸਿਆਵਾਂ ਹੱਲ ਕਰੋਗੇ
- ਸਮੱਸਿਆ ਨਿਵਾਰਣ ਲਈ Azure ਪੋਰਟਲ ਅਤੇ CLI ਦੀ ਵਰਤੋਂ ਕਰੋਗੇ

## 🚨 ਤੇਜ਼ ਠੀਕ ਕਰਨ ਵਾਲੇ ਤਰੀਕੇ

### ਪ੍ਰਮਾਣਿਕਤਾ ਸਮੱਸਿਆਵਾਂ
```bash
# AZD ਵਰਕਫਲੋਜ਼ ਲਈ ਲਾਜ਼ਮੀ
azd auth login

# ਜੇ ਤੁਸੀਂ ਸਿੱਧਾ Azure CLI ਕਮਾਂਡਾਂ ਵੀ ਵਰਤ ਰਹੇ ਹੋ ਤਾਂ ਵਿਕਲਪੀ।
az login

azd auth status
```

### ਤੈਨਾਤੀ ਅਸਫਲਤਾਵਾਂ
```bash
azd show
azd monitor --logs
az deployment sub list --query "[?properties.provisioningState!='Succeeded']"
```

### ਸਾਧਨ ਸੰਗਰਾਮ
```bash
azd down --force --purge
azd env new different-name
azd up
```

### ਕੋਟਾ ਜ਼ਿਆਦਾ ਵਰਤਿਆ ਗਿਆ
```bash
az vm list-usage --location eastus --output table
azd env set AZURE_LOCATION westus2
azd up
```

## 📋 ਤਰੁੱਟੀ ਕੋਡ ਸੰਬੰਧੀ ਜਾਣਕਾਰੀ

| ਤਰੁੱਟੀ | ਕਾਰਨ | ਹੱਲ |
|-------|-------|----------|
| `AuthenticationError` | ਲਾਗਿਨ ਨਹੀਂ ਹੋਇਆ | `azd auth login` |
| `ResourceNotFound` | ਗੁੰਮ ਸਾਧਨ | ਸਾਧਨਾਂ ਦੇ ਨਾਂ ਜਾਂਚੋ |
| `QuotaExceeded` | ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਸੀਮਾਵਾਂ | ਕੋਟਾ ਵਧਾਉਣ ਦੀ ਬੇਨਤੀ ਕਰੋ |
| `InvalidTemplate` | ਬਾਈਸਪ ਵਿਖਿਆਤਾ ਵਿੱਚ ਗਲਤੀ | `az bicep build` |
| `Conflict` | ਸਾਧਨ ਮੌਜੂਦ ਹੈ | ਨਵਾਂ ਨਾਮ ਵਰਤੋ ਜਾਂ ਮਿਟਾਓ |
| `Forbidden` | ਪ੍ਰਮਾਣਿਕਤਾ ਅਣਪੂਰੀ | RBAC ਭੂਮਿਕਾਵਾਂ ਜਾਂਚੋ |

## 🔄 ਰੀਸੈੱਟ ਅਤੇ ਬਹਾਲੀ

```bash
# ਸੌਫਟ ਰੀਸੈਟ (ਸੰਸਾਧਨਾਂ ਨੂੰ ਰੱਖੋ, ਕੋਡ ਨੂੰ ਦੁਬਾਰਾ ਤਾਇਨਾਤ ਕਰੋ)
azd deploy --force

# ਹਾਰਡ ਰੀਸੈਟ (ਸਭ ਕੁਝ ਮਿਟਾਓ, ਨਵੀਂ ਸ਼ੁਰੂਆਤ ਕਰੋ)
azd down --force --purge
azd up
```
