# ਉਦਾਹਰਨਾਂ - ਪ੍ਰੈਕਟਿਕਲ AZD ਟੈਂਪਲੇਟ ਅਤੇ ਕਨਫਿਗਰੇਸ਼ਨ

## ਪਰਿਚਯ

ਇਹ ਡਾਇਰੈਕਟਰੀ ਪ੍ਰੈਕਟਿਕਲ ਉਦਾਹਰਨਾਂ, ਟੈਂਪਲੇਟਾਂ ਅਤੇ ਅਸਲ ਦੁਨੀਆ ਦੇ ਸਨਰੀਓਜ਼ ਨੂੰ ਸ਼ਾਮਲ ਕਰਦੀ ਹੈ, ਜੋ ਤੁਹਾਨੂੰ Azure Developer CLI ਨੂੰ ਹੱਥ-ਅਨੁਭਵ ਰਾਹੀਂ ਸਿੱਖਣ ਵਿੱਚ ਮਦਦ ਕਰਦੀ ਹੈ। ਹਰ ਉਦਾਹਰਨ ਵਿੱਚ ਪੂਰਾ ਕੰਮ ਕਰਨ ਵਾਲਾ ਕੋਡ, ਇੰਫਰਾਸਟਰਕਚਰ ਟੈਂਪਲੇਟ ਅਤੇ ਵੱਖ-ਵੱਖ ਐਪਲੀਕੇਸ਼ਨ ਆਰਕੀਟੈਕਚਰ ਅਤੇ ਡਿਪਲੋਇਮੈਂਟ ਪੈਟਰਨ ਲਈ ਵਿਸਥਾਰਿਤ ਨਿਰਦੇਸ਼ ਸ਼ਾਮਲ ਹਨ।

## ਸਿੱਖਣ ਦੇ ਲਕਸ਼

ਇਨ੍ਹਾਂ ਉਦਾਹਰਨਾਂ 'ਤੇ ਕੰਮ ਕਰਕੇ, ਤੁਸੀਂ:
- ਅਸਲ ਐਪਲੀਕੇਸ਼ਨ ਸਨਰੀਓਜ਼ ਨਾਲ Azure Developer CLI ਵਰਕਫਲੋਜ਼ ਦਾ ਅਭਿਆਸ ਕਰੋਗੇ
- ਵੱਖ-ਵੱਖ ਐਪਲੀਕੇਸ਼ਨ ਆਰਕੀਟੈਕਚਰ ਅਤੇ ਉਨ੍ਹਾਂ ਦੇ azd ਇੰਪਲੀਮੈਂਟੇਸ਼ਨ ਨੂੰ ਸਮਝੋਗੇ
- ਵੱਖ-ਵੱਖ Azure ਸੇਵਾਵਾਂ ਲਈ Infrastructure as Code ਪੈਟਰਨ ਵਿੱਚ ਮਾਹਰ ਹੋਵੋਗੇ
- ਕਨਫਿਗਰੇਸ਼ਨ ਮੈਨੇਜਮੈਂਟ ਅਤੇ ਵਾਤਾਵਰਣ-ਵਿਸ਼ੇਸ਼ ਡਿਪਲੋਇਮੈਂਟ ਰਣਨੀਤੀਆਂ ਲਾਗੂ ਕਰੋਗੇ
- ਪ੍ਰੈਕਟਿਕਲ ਸੰਦਰਭਾਂ ਵਿੱਚ ਮਾਨੀਟਰਿੰਗ, ਸੁਰੱਖਿਆ ਅਤੇ ਸਕੇਲਿੰਗ ਪੈਟਰਨ ਲਾਗੂ ਕਰੋਗੇ
- ਅਸਲ ਡਿਪਲੋਇਮੈਂਟ ਸਨਰੀਓਜ਼ ਨੂੰ ਟਰਬਲਸ਼ੂਟ ਅਤੇ ਡਿਬੱਗ ਕਰਨ ਦਾ ਅਨੁਭਵ ਪ੍ਰਾਪਤ ਕਰੋਗੇ

## ਸਿੱਖਣ ਦੇ ਨਤੀਜੇ

ਇਨ੍ਹਾਂ ਉਦਾਹਰਨਾਂ ਨੂੰ ਪੂਰਾ ਕਰਨ ਤੋਂ ਬਾਅਦ, ਤੁਸੀਂ:
- ਵੱਖ-ਵੱਖ ਐਪਲੀਕੇਸ਼ਨ ਪ੍ਰਕਾਰਾਂ ਨੂੰ Azure Developer CLI ਦੀ ਵਰਤੋਂ ਨਾਲ ਵਿਸ਼ਵਾਸਪੂਰਨ ਤਰੀਕੇ ਨਾਲ ਡਿਪਲੋਇ ਕਰ ਸਕੋਗੇ
- ਪ੍ਰਦਾਨ ਕੀਤੇ ਟੈਂਪਲੇਟਾਂ ਨੂੰ ਆਪਣੇ ਐਪਲੀਕੇਸ਼ਨ ਦੀਆਂ ਲੋੜਾਂ ਅਨੁਸਾਰ ਅਨੁਕੂਲਿਤ ਕਰ ਸਕੋਗੇ
- Bicep ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਕਸਟਮ ਇੰਫਰਾਸਟਰਕਚਰ ਪੈਟਰਨ ਡਿਜ਼ਾਈਨ ਅਤੇ ਲਾਗੂ ਕਰ ਸਕੋਗੇ
- ਸਹੀ ਡਿਪੈਂਡੈਂਸੀਜ਼ ਨਾਲ ਕੌਂਪਲੈਕਸ ਮਲਟੀ-ਸਰਵਿਸ ਐਪਲੀਕੇਸ਼ਨ ਕਨਫਿਗਰ ਕਰ ਸਕੋਗੇ
- ਅਸਲ ਸਨਰੀਓਜ਼ ਵਿੱਚ ਸੁਰੱਖਿਆ, ਮਾਨੀਟਰਿੰਗ ਅਤੇ ਪ੍ਰਦਰਸ਼ਨ ਦੇ ਸਰਵੋਤਮ ਅਭਿਆਸ ਲਾਗੂ ਕਰ ਸਕੋਗੇ
- ਪ੍ਰੈਕਟਿਕਲ ਅਨੁਭਵ ਦੇ ਆਧਾਰ 'ਤੇ ਡਿਪਲੋਇਮੈਂਟ ਨੂੰ ਟਰਬਲਸ਼ੂਟ ਅਤੇ ਅਪਟਿਮਾਈਜ਼ ਕਰ ਸਕੋਗੇ

3. **[ਕੰਟੇਨਰ ਐਪ](../../../examples/container-app)** - ਇੱਕ ਕੰਟੇਨਰਾਈਜ਼ਡ ਐਪਲੀਕੇਸ਼ਨ ਡਿਪਲੋਇ ਕਰੋ

4. **[ਡਾਟਾਬੇਸ ਐਪ](../../../examples/database-app)** - PostgreSQL ਡਾਟਾਬੇਸ ਨਾਲ ਵੈੱਬ ਐਪਲੀਕੇਸ਼ਨ  
5. **[ਸਰਵਰਲੈੱਸ ਫੰਕਸ਼ਨ](../../../examples/serverless-function)** - HTTP ਟ੍ਰਿਗਰ ਨਾਲ Azure Functions  
6. **[ਮਾਈਕ੍ਰੋਸਰਵਿਸਜ਼](../../../examples/microservices)** - API ਗੇਟਵੇ ਨਾਲ ਮਲਟੀ-ਸਰਵਿਸ ਐਪਲੀਕੇਸ਼ਨ  

## 📋 ਵਰਤੋਂ ਦੇ ਨਿਰਦੇਸ਼

### ਉਦਾਹਰਨਾਂ ਨੂੰ ਲੋਕਲ ਤੌਰ 'ਤੇ ਚਲਾਉਣਾ

1. **ਉਦਾਹਰਨ ਕਲੋਨ ਜਾਂ ਕਾਪੀ ਕਰੋ**  
   ```bash
   # Navigate to desired example
   cd examples/simple-web-app
   ```

2. **AZD ਵਾਤਾਵਰਣ ਸ਼ੁਰੂ ਕਰੋ**  
   ```bash
   # Initialize with existing template
   azd init
   
   # Or create new environment
   azd env new my-environment
   ```

3. **ਵਾਤਾਵਰਣ ਕਨਫਿਗਰ ਕਰੋ**  
   ```bash
   # Set required variables
   azd env set AZURE_LOCATION eastus
   azd env set AZURE_SUBSCRIPTION_ID your-subscription-id
   ```

4. **ਡਿਪਲੋਇ ਕਰੋ**  
   ```bash
   # Deploy infrastructure and application
   azd up
   ```

### ਉਦਾਹਰਨਾਂ ਨੂੰ ਅਨੁਕੂਲਿਤ ਕਰਨਾ

ਹਰ ਉਦਾਹਰਨ ਵਿੱਚ ਸ਼ਾਮਲ ਹੈ:
- **README.md** - ਵਿਸਥਾਰਿਤ ਸੈਟਅੱਪ ਅਤੇ ਕਸਟਮਾਈਜ਼ੇਸ਼ਨ ਨਿਰਦੇਸ਼  
- **azure.yaml** - ਟਿੱਪਣੀਆਂ ਨਾਲ AZD ਕਨਫਿਗਰੇਸ਼ਨ  
- **infra/** - ਪੈਰਾਮੀਟਰ ਵਿਆਖਿਆਵਾਂ ਨਾਲ Bicep ਟੈਂਪਲੇਟ  
- **src/** - ਨਮੂਨਾ ਐਪਲੀਕੇਸ਼ਨ ਕੋਡ  
- **scripts/** - ਆਮ ਕੰਮਾਂ ਲਈ ਸਹਾਇਕ ਸਕ੍ਰਿਪਟ

### ਸਿਫਾਰਸ਼ੀ ਤਰੱਕੀ

1. **ਸਧਾਰਨ ਵੈੱਬ ਐਪ ਨਾਲ ਸ਼ੁਰੂ ਕਰੋ**
   - ਬੇਸਿਕ AZD ਸੰਕਲਪ ਸਿੱਖੋ  
   - ਡਿਪਲੋਇਮੈਂਟ ਵਰਕਫਲੋ ਸਮਝੋ  
   - ਵਾਤਾਵਰਣ ਮੈਨੇਜਮੈਂਟ ਦਾ ਅਭਿਆਸ ਕਰੋ  

2. **ਸਟੈਟਿਕ ਵੈਬਸਾਈਟ ਦੀ ਕੋਸ਼ਿਸ਼ ਕਰੋ**
   - ਵੱਖ-ਵੱਖ ਹੋਸਟਿੰਗ ਵਿਕਲਪਾਂ ਦੀ ਪੜਚੋਲ ਕਰੋ  
   - CDN ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਸਿੱਖੋ  
   - DNS ਕਨਫਿਗਰੇਸ਼ਨ ਸਮਝੋ  

3. **ਕੰਟੇਨਰ ਐਪ ਵੱਲ ਵਧੋ**
   - ਕੰਟੇਨਰਾਈਜ਼ੇਸ਼ਨ ਦੇ ਬੇਸਿਕ ਸਿੱਖੋ  
   - ਸਕੇਲਿੰਗ ਸੰਕਲਪ ਸਮਝੋ  
   - Docker ਨਾਲ ਅਭਿਆਸ ਕਰੋ  

4. **ਡਾਟਾਬੇਸ ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਸ਼ਾਮਲ ਕਰੋ**
   - ਡਾਟਾਬੇਸ ਪ੍ਰੋਵਿਜ਼ਨਿੰਗ ਸਿੱਖੋ  
   - ਕਨੈਕਸ਼ਨ ਸਟ੍ਰਿੰਗਸ ਸਮਝੋ  
   - ਸੀਕ੍ਰੇਟ ਮੈਨੇਜਮੈਂਟ ਦਾ ਅਭਿਆਸ ਕਰੋ  

5. **ਸਰਵਰਲੈੱਸ ਦੀ ਪੜਚੋਲ ਕਰੋ**
   - ਇਵੈਂਟ-ਡ੍ਰਿਵਨ ਆਰਕੀਟੈਕਚਰ ਸਮਝੋ  
   - ਟ੍ਰਿਗਰ ਅਤੇ ਬਾਈਂਡਿੰਗ ਬਾਰੇ ਸਿੱਖੋ  
   - APIs ਨਾਲ ਅਭਿਆਸ ਕਰੋ  

6. **ਮਾਈਕ੍ਰੋਸਰਵਿਸਜ਼ ਬਣਾਓ**
   - ਸੇਵਾ ਸੰਚਾਰ ਸਿੱਖੋ  
   - ਵੰਡੇ ਹੋਏ ਸਿਸਟਮ ਸਮਝੋ  
   - ਕੌਂਪਲੈਕਸ ਡਿਪਲੋਇਮੈਂਟ ਦਾ ਅਭਿਆਸ ਕਰੋ
