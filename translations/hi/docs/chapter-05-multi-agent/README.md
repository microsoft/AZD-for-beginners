**📚 कोर्स**: [AZD For Beginners](../../README.md) | **⏱️ अवधि**: 2-3 घंटे | **⭐ जटिलता**: उन्नत

---

## अवलोकन

यह अध्याय उन्नत मल्टी-एजेंट आर्किटेक्चर पैटर्न, एजेंट ओरकेस्ट्रेशन, और जटिल परिदृश्यों के लिए प्रोडक्शन-रेडी AI डिप्लॉयमेंट को कवर करता है।

> `azd 1.27.1` के खिलाफ जुलाई 2026 में मान्य किया गया।

## सीखने के उद्देश्य

इस अध्याय को पूरा करने पर, आप:
- मल्टी-एजेंट आर्किटेक्चर पैटर्न को समझेंगे
- समन्वित AI एजेंट सिस्टम को डिप्लॉय करेंगे
- एजेंट-से-एजेंट संचार को लागू करेंगे
- प्रोडक्शन-रेडी मल्टी-एजेंट समाधान बनाएंगे

## 🚀 त्वरित शुरुआत

```bash
# विकल्प 1: एक टेम्पलेट से तैनात करें
azd init --template agent-openai-python-prompty
azd up

# विकल्प 2: एजेंट मेनिफेस्ट से तैनात करें (azure.ai.agents एक्सटेंशन की आवश्यकता होती है)
azd extension install azure.ai.agents
azd ai agent init -m agent-manifest.yaml
azd up
```

> **कौन सा तरीका?** काम करने वाले नमूने से शुरू करने के लिए `azd init --template` का उपयोग करें। जब आपके पास अपना एजेंट मैनिफेस्ट हो, तो `azd ai agent init` का उपयोग करें। पूर्ण विवरण के लिए [AZD AI CLI संदर्भ](../chapter-08-production/production-ai-practices.md#azd-ai-cli-commands-and-extensions) देखें।

## 🎯 मुख्य समाधान: रिटेल मल्टी-एजेंट

[रिटेल मल्टी-एजेंट समाधान](../../examples/retail-scenario.md) दर्शाता है:

- **ग्राहक एजेंट**: उपयोगकर्ता इंटरैक्शन और प्राथमिकताओं को संभालता है
