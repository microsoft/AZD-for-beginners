**📚 कोर्स**: [AZD शुरुआती के लिए](../../README.md) | **⏱️ अवधि**: 2-3 घंटे | **⭐ जटिलता**: उन्नत

---

## अवलोकन

यह अध्याय उन्नत मल्टी-एजेंट आर्किटेक्चर पैटर्न, एजेंट ऑर्केस्ट्रेशन, और जटिल परिदृश्यों के लिए प्रोडक्शन-तैयार AI डिप्लॉयमेंट को कवर करता है।

> मार्च 2026 में `azd 1.23.12` के खिलाफ सत्यापित।

## सीखने के उद्देश्य

इस अध्याय को पूरा करने के बाद, आप:
- मल्टी-एजेंट आर्किटेक्चर पैटर्न को समझेंगे
- समन्वित AI एजेंट सिस्टम को डिप्लॉय कर सकेंगे
- एजेंट-से-एजेंट संचार को लागू कर सकेंगे
- प्रोडक्शन-तैयार मल्टी-एजेंट समाधान बना सकेंगे

## 🚀 त्वरित शुरुआत

```bash
# विकल्प 1: एक टेम्पलेट से तैनात करें
azd init --template agent-openai-python-prompty
azd up

# विकल्प 2: एक एजेंट मेनिफेस्ट से तैनात करें (azure.ai.agents एक्सटेंशन आवश्यक)
azd extension install azure.ai.agents
azd ai agent init -m agent-manifest.yaml
azd up
```

> **कौन सा दृष्टिकोण?** कार्यशील नमूने से शुरू करने के लिए `azd init --template` का उपयोग करें। जब आपके पास अपना एजेंट मैनिफेस्ट हो, तब `azd ai agent init` का उपयोग करें। पूर्ण विवरण के लिए [AZD AI CLI संदर्भ](../chapter-08-production/production-ai-practices.md#azd-ai-cli-commands-and-extensions) देखें।

## 🎯 प्रमुख समाधान: रिटेल मल्टी-एजेंट

[रिटेल मल्टी-एजेंट समाधान](../../examples/retail-scenario.md) प्रदर्शित करता है:

- **ग्राहक एजेंट**: उपयोगकर्ता इंटरैक्शन और प्राथमिकताएँ संभालता है
- **इन्वेंट्री एजेंट**: स्टॉक और ऑर्डर प्रोसेसिंग प्रबंधित करता है
- **ऑर्केस्ट्रेटर**: एजेंटों के बीच समन्वय करता है
- **साझा मेमोरी**: एजेंटों के पार संदर्भ प्रबंधन
