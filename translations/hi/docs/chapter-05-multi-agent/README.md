# अध्याय 5: मल्टी-एजेंट एआई समाधान

**📚 पाठ्यक्रम**: [AZD For Beginners](../../README.md) | **⏱️ अवधि**: 2-3 घंटे | **⭐ जटिलता**: उन्नत

---

## अवलोकन

यह अध्याय उन्नत मल्टी-एजेंट आर्किटेक्चर पैटर्न, एजेंट समन्वय, और जटिल परिदृश्यों के लिए प्रोडक्शन-तैयार एआई तैनाती को कवर करता है।

## सीखने के उद्देश्य

इस अध्याय को पूरा करने पर, आप:
- मल्टी-एजेंट आर्किटेक्चर पैटर्न समझेंगे
- समन्वित एआई एजेंट सिस्टम तैनात करेंगे
- एजेंट-से-एजेंट संचार लागू करेंगे
- प्रोडक्शन-तैयार मल्टी-एजेंट समाधान बनाएंगे

## 🚀 त्वरित आरंभ

```bash

cd examples/retail-multiagent-arm-template
./deploy.sh

# या सीधे टेम्पलेट का उपयोग करें
azd init --template agent-openai-python-prompty
azd up
```

## 🎯 विशेष समाधान: रिटेल मल्टी-एजेंट

यह [रिटेल मल्टी-एजेंट समाधान](../../examples/retail-scenario.md) प्रदर्शित करता है:

- **ग्राहक एजेंट**: उपयोगकर्ता इंटरैक्शन और प्राथमिकताओं को संभालता है
- **इन्वेंटरी एजेंट**: स्टॉक और ऑर्डर प्रोसेसिंग का प्रबंधन करता है
- **समन्वयक**: एजेंट्स के बीच समन्वय करता है
- **साझा मेमोरी**: एजेंटों के बीच संदर्भ प्रबंधन

### प्रयुक्त सेवाएँ

| Service | Purpose |
|---------|---------|
| Azure OpenAI | भाषा की समझ |
| Azure AI Search | उत्पाद कैटलॉग |
| Cosmos DB | एजेंट स्थिति और मेमोरी |
| Container Apps | एजेंट होस्टिंग |
| Application Insights | निगरानी |
