# अध्याय 8: प्रोडक्शन और एंटरप्राइज़ पैटर्न

**📚 पाठ्यक्रम**: [AZD शुरुआती के लिए](../../README.md) | **⏱️ अवधि**: 2-3 घंटे | **⭐ जटिलता**: उन्नत

---

## अवलोकन

यह अध्याय एंटरप्राइज़-तैयार डिप्लॉयमेंट पैटर्न, सुरक्षा कड़ेकरण, मॉनिटरिंग, और प्रोडक्शन AI वर्कलोड्स के लिए लागत अनुकूलन को कवर करता है।

## सीखने के उद्देश्य

इस अध्याय को पूरा करने पर आप:
- बहु-क्षेत्र (multi-region) प्रत्यास्थ अनुप्रयोग तैनात करें
- एंटरप्राइज़ सुरक्षा पैटर्न लागू करें
- व्यापक मॉनिटरिंग कॉन्फ़िगर करें
- स्केल पर लागतों का अनुकूलन करें
- AZD के साथ CI/CD पाइपलाइनों को सेट अप करें

## 💰 लागत अनुकूलन

| रणनीति | बचत |
|----------|---------|
| शून्य तक स्केल (Container Apps) | 60-80% |
| देव के लिए उपभोग स्तरों का उपयोग करें | 50-70% |
| अनुसूचित स्केलिंग | 30-50% |
| आरक्षित क्षमता | 20-40% |

```bash
# बजट अलर्ट सेट करें
az consumption budget create \
  --budget-name "AI-Budget" \
  --amount 500 \
  --category Cost \
  --time-grain Monthly
```
