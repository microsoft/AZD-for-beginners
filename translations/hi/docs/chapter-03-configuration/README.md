# अध्याय 3: कॉन्फ़िगरेशन और प्रमाणीकरण

**📚 Course**: [AZD शुरुआती के लिए](../../README.md) | **⏱️ Duration**: 45-60 मिनट | **⭐ Complexity**: मध्यम

---

## अवलोकन

यह अध्याय Azure Developer CLI परिनियोजन के लिए वातावरण कॉन्फ़िगरेशन, प्रमाणीकरण पैटर्न, और सुरक्षा सर्वोत्तम अभ्यास कवर करता है।

> यह `azd 1.25.6` के खिलाफ जून 2026 में सत्यापित किया गया है।

## सीखने के उद्देश्य

इस अध्याय को पूरा करने के बाद, आप:
- AZD कॉन्फ़िगरेशन अनुक्रम में महारत हासिल करेंगे
- एकाधिक वातावरण (dev, staging, prod) का प्रबंधन करेंगे
- Managed identities के साथ सुरक्षित प्रमाणीकरण लागू करेंगे
- वातावरण-विशिष्ट सेटिंग्स कॉन्फ़िगर करेंगे

## 🚀 त्वरित आरंभ

```bash
# कई एनवायरनमेंट बनाएं
azd env new dev
azd env new staging
azd env new prod

# एनवायरनमेंट बदलें
azd env select prod

# एनवायरनमेंट वेरिएबल सेट करें
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# कॉन्फ़िगरेशन देखें
azd env get-values
```

---

## 🔧 कॉन्फ़िगरेशन अनुक्रम

AZD निम्न क्रम में सेटिंग्स लागू करता है (आगे वाले पिछले को ओवरराइड करते हैं):

1. **डिफ़ॉल्ट मान** (टेम्पलेट में अंतर्निहित)
2. **azure.yaml** (प्रोजेक्ट कॉन्फ़िगरेशन)
3. **पर्यावरण वेरिएबल** (`azd env set`)
4. **कमांड-लाइन फ़्लैग्स** (`--location eastus`)

---

## 🔐 सुरक्षा सर्वोत्तम अभ्यास

```bash
# प्रबंधित पहचान का उपयोग करें (अनुशंसित)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD के प्रमाणीकरण की स्थिति की जाँच करें
azd auth status

# वैकल्पिक: यदि आप az कमांड चलाने की योजना बना रहे हैं तो Azure CLI संदर्भ सत्यापित करें
az account show

# आवश्यक होने पर पुनः प्रमाणीकरण करें
azd auth login

# वैकल्पिक: az कमांड के लिए Azure CLI प्रमाणीकरण ताज़ा करें
az login
```
