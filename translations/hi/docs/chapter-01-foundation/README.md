# अध्याय 1: आधार और त्वरित आरंभ

**📚 Course**: [AZD शुरुआती के लिए](../../README.md) | **⏱️ अवधि**: 30-45 मिनट | **⭐ जटिलता**: शुरुआती

---

## अवलोकन

यह अध्याय Azure Developer CLI (azd) के मूल सिद्धांतों का परिचय कराता है। आप मुख्य अवधारणाएँ सीखेंगे, उपकरण स्थापित और कॉन्फ़िगर करना सीखेंगे, और अपना पहला एप्लिकेशन Azure पर तैनात करेंगे।

## सीखने के उद्देश्य

By completing this chapter, you will:
- समझें कि Azure Developer CLI क्या है और यह Azure CLI से किस प्रकार अलग है
- अपने प्लेटफ़ॉर्म पर AZD स्थापित और कॉन्फ़िगर करें
- `azd up` का उपयोग करके अपना पहला एप्लिकेशन Azure पर तैनात करें
- `azd down` के साथ संसाधनों को साफ़ करें

## 🚀 त्वरित आरंभ

```bash
# इंस्टॉलेशन जांचें
azd version

# Azure में लॉगिन करें
azd auth login

# अपना पहला ऐप तैनात करें
azd init --template todo-nodejs-mongo
azd up

# काम पूरा होने पर साफ़ करें
azd down --force --purge
```

---

## ✅ सफलता मानदंड

After completing this chapter, you should be able to:

```bash
azd version              # स्थापित संस्करण दिखाता है
azd init --template todo-nodejs-mongo  # परियोजना आरंभ करता है
azd up                   # Azure पर तैनात करता है
azd show                 # चल रहे ऐप का URL दिखाता है
azd down --force --purge # संसाधनों की सफाई करता है
```
