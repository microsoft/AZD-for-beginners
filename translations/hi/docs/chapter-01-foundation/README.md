# अध्याय 1: आधार और त्वरित प्रारंभ

**📚 पाठ्यक्रम**: [AZD शुरुआती के लिए](../../README.md) | **⏱️ अवधि**: 30-45 मिनट | **⭐ जटिलता**: शुरुआती

---

## अवलोकन

यह अध्याय Azure Developer CLI (azd) की मूल बातें प्रस्तुत करता है। आप मुख्य अवधारणाएँ सीखेंगे, उपकरण स्थापित करेंगे, और अपनी पहली आवेदन Azure पर तैनात करेंगे।

> जुलाई 2026 में `azd 1.27.1` के साथ सत्यापित।

## सीखने के उद्देश्य

इस अध्याय को पूरा करने के बाद, आप:
- समझेंगे कि Azure Developer CLI क्या है और यह Azure CLI से कैसे अलग है
- अपने प्लेटफ़ॉर्म पर AZD को स्थापित और कॉन्फ़िगर करेंगे
- `azd up` के साथ अपनी पहली ऐप को Azure पर तैनात करेंगे
- `azd down` के साथ संसाधनों को साफ़ करेंगे

## ✅ यहां से शुरू करें: अपनी सेटअप सत्यापित करें

शुरू करने से पहले, सुनिश्चित करें कि आपकी स्थानीय मशीन अध्याय 1 टेम्पलेट के लिए तैयार है:

**विंडोज़:**
```powershell
.\validate-setup.ps1
```

**macOS / लिनक्स:**
```bash
bash ./validate-setup.sh
```

यदि स्क्रिप्ट में टूल्स गायब हैं, तो पहले उन्हें ठीक करें और फिर अध्याय के साथ जारी रखें।

---

## 🚀 त्वरित प्रारंभ

```bash
# स्थापना जांचें
azd version

# AZD के लिए प्रमाणीकरण करें
# वैकल्पिक: यदि आप सीधे Azure CLI कमांड चलाने की योजना बना रहे हैं तो az login करें
azd auth login

# अपना पहला ऐप तैनात करें
azd init --template todo-nodejs-mongo
azd up

# समाप्त होने पर सफाई करें
azd down --force --purge
```

---

## ✅ सफलता मानदंड

इस अध्याय को पूरा करने के बाद, आप सक्षम होंगे:

```bash
azd version              # स्थापित संस्करण दिखाता है
azd init --template todo-nodejs-mongo  # परियोजना आरंभ करता है
azd up                   # Azure पर तैनात करता है
azd show                 # चल रहे ऐप URL को प्रदर्शित करता है
azd down --force --purge # संसाधनों को साफ करता है
```
