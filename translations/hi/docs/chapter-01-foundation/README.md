# अध्याय 1: आधार और त्वरित प्रारंभ

**📚 पाठ्यक्रम**: [AZD प्रारंभिक के लिए](../../README.md) | **⏱️ अवधि**: 30-45 मिनट | **⭐ जटिलता**: शुरुआती

---

## अवलोकन

यह अध्याय Azure Developer CLI (azd) के मूलभूत तत्वों से परिचित करवाता है। आप मुख्य अवधारणाएं सीखेंगे, टूल्स इंस्टॉल करेंगे, और अपना पहला आवेदन Azure पर तैनात करेंगे।

> मार्च 2026 में `azd 1.23.12` के विरुद्ध सत्यापित।

## सीखने के उद्देश्य

इस अध्याय को पूरा करने पर, आप:
- समझ पाएंगे कि Azure Developer CLI क्या है और यह Azure CLI से कैसे अलग है
- अपने प्लेटफ़ॉर्म पर AZD स्थापित और कॉन्फ़िगर करना सीखेंगे
- `azd up` का उपयोग करके अपना पहला एप्लिकेशन Azure पर तैनात करेंगे
- `azd down` के साथ संसाधनों को साफ़ करेंगे

## ✅ यहां से शुरू करें: अपनी सेटअप सत्यापित करें

आरंभ करने से पहले, पुष्टि करें कि आपकी स्थानीय मशीन अध्याय 1 टेम्पलेट के लिए तैयार है:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

यदि स्क्रिप्ट कोई टूल गायब दिखाती है, तो पहले उन्हें ठीक करें और फिर अध्याय जारी रखें।

---

## 🚀 त्वरित प्रारंभ

```bash
# इंस्टॉलेशन जांचें
azd version

# AZD के लिए प्रमाणीकरण करें
# वैकल्पिक: यदि आप सीधे Azure CLI कमांड चलाने की योजना बनाते हैं तो az login करें
azd auth login

# अपना पहला ऐप तैनात करें
azd init --template todo-nodejs-mongo
azd up

# समाप्त होने पर साफ-सफाई करें
azd down --force --purge
```

---

## ✅ सफलता के मापदंड

इस अध्याय को पूरा करने के बाद, आप सक्षम होंगे:

```bash
azd version              # इंस्टॉल की गई संस्करण दिखाता है
azd init --template todo-nodejs-mongo  # प्रोजेक्ट आरंभ करता है
azd up                   # Azure पर तैनात करता है
azd show                 # चल रहे ऐप का URL दिखाता है
azd down --force --purge # संसाधनों को साफ करता है
```
