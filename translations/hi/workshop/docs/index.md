# AI डेवलपर्स के लिए AZD कार्यशाला

> [!IMPORTANT]  
> **यह कार्यशाला एक मार्गदर्शिका के साथ सुसज्जित है जिसका आप अपने लोकल ब्राउज़र में पूर्वावलोकन कर सकते हैं। शुरू करने के लिए, आपको repo पर GitHub Codespaces लॉन्च करना होगा—फिर तब तक प्रतीक्षा करें जब तक आप एक सक्रिय VS Code टर्मिनल नहीं देखते और टाइप करें:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **आपको ब्राउज़र में पूर्वावलोकन पृष्ठ खोलने के लिए एक पॉप-अप डायलॉग दिखाई देना चाहिए।**

Azure Developer CLI (AZD) को AI एप्लिकेशन तैनाती पर ध्यान केंद्रित करके सीखने के लिए इस व्यावहारिक कार्यशाला में आपका स्वागत है। यह कार्यशाला आपको 3 चरणों में AZD टेम्पलेट का व्यावहारिक समझ प्राप्त करने में मदद करती है:

1. **अन्वेषण** - वह टेम्पलेट खोजें जो आपके लिए उपयुक्त हो।  
1. **तैनाती** - तैनात करें और सत्यापित करें कि यह काम करता है  
1. **अनुकूलन** - इसे अपनी आवश्यकताओं के अनुसार संशोधित और पुनरावृत्ति करें!

इस कार्यशाला के दौरान, आपको मुख्य डेवलपर टूल्स और वर्कफ़्लो से भी परिचित कराया जाएगा, ताकि आप अपने एंड-टू-एंड विकास सफर को सरल बना सकें।

## कार्यशाला अवलोकन

**अवधि:** 3-4 घंटे  
**स्तर:** प्रारम्भिक से मध्यवर्ती  
**पूर्वापेक्षाएँ:** Azure, AI अवधारणाओं, VS Code और कमांड-लाइन टूल्स से परिचित होना।

यह एक व्यावहारिक कार्यशाला है जहाँ आप कार्य करके सीखते हैं। अभ्यास पूरे करने के बाद, हम अनुशंसा करते हैं कि आप सुरक्षा और उत्पादकता सर्वोत्तम प्रथाओं की ओर अपनी सीख जारी रखने के लिए AZD For Beginners पाठ्यक्रम की समीक्षा करें।

| समय| मॉड्यूल  | उद्देश्य |
|:---|:---|:---|
| 15 मिनट | परिचय | परिप्रेक्ष्य तय करना, लक्ष्यों को समझना |
| 30 मिनट | AI टेम्पलेट चुनना | विकल्पों का पता लगाना और शुरुआत के लिए चुनना | 
| 30 मिनट | AI टेम्पलेट सत्यापित करना | डिफ़ॉल्ट समाधान को Azure पर तैनात करना |
| 30 मिनट | AI टेम्पलेट का विघटन | संरचना और कॉन्फ़िगरेशन का अन्वेषण करना |
| 30 मिनट | AI टेम्पलेट कॉन्फ़िगर करना | उपलब्ध फीचर्स को सक्रिय करना और आज़माना |
| 30 मिनट | AI टेम्पलेट अनुकूलित करना | टेम्पलेट को आपकी आवश्यकताओं के अनुसार ढालना |
| 30 मिनट | इन्फ्रास्ट्रक्चर हटाना | संसाधनों की सफाई और जारी करना |
| 15 मिनट | समापन और अगले कदम | शिक्षण संसाधन, कार्यशाला चुनौती |
| | |

## आप क्या सीखेंगे

AZD टेम्पलेट को Microsoft Foundry पर एंड-टू-एंड विकास के विभिन्न क्षमताओं और टूल्स का अन्वेषण करने के लिए एक लर्निंग सैंडबॉक्स के रूप में सोचें। इस कार्यशाला के अंत तक, इस संदर्भ में आपको विभिन्न टूल्स और अवधारणाओं के बारे में एक सहज बोध होना चाहिए।

| अवधारणा  | उद्देश्य |
|:---|:---|
| **Azure Developer CLI** | उपकरण कमांड और वर्कफ़्लो को समझना|
| **AZD Templates**| प्रोजेक्ट संरचना और कॉन्फ़िगरेशन को समझना|
| **Azure AI Agent**| Microsoft Foundry प्रोजेक्ट का प्रावधान और तैनाती  |
| **Azure AI Search**| एजेंट्स के साथ संदर्भ इंजीनियरिंग सक्षम करना |
| **Observability**| ट्रेसिंग, मॉनिटरिंग और आकलनों का अन्वेषण करना |
| **Red Teaming**| विरोधी परीक्षण और निवारक उपायों का अन्वेषण करना |
| | |
