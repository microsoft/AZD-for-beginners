## 2. सीखी गई प्रमुख अवधारणाएँ

### Azure Developer CLI (azd)
- `azd init` - टेम्पलेट्स से प्रोजेक्ट्स प्रारंभ करें
- `azd up` - एक कमांड में प्रोविजन और डिप्लॉय करें
- `azd env` - पर्यावरण चर प्रबंधित करें
- `azd down` - संसाधनों को सुरक्षित रूप से साफ़ करें

### AZD टेम्पलेट संरचना
- `azure.yaml` - कॉन्फ़िगरेशन और डिप्लॉयमेंट हुक्स
- `infra/` - Bicep इंफ्रास्ट्रक्चर-ऐज़-कोड
- `src/` - एप्लिकेशन स्रोत कोड
- `scripts/` - ऑटोमेशन और सेटअप स्क्रिप्ट्स

### Microsoft Foundry क्षमताएँ
- इंटेलिजेंट असिस्टेंट्स के लिए AI Agent Service
- ज्ञान पुनर्प्राप्ति के लिए File Search और Azure AI Search
- बिल्ट-इन ट्रेसिंग, मॉनिटरिंग, और मूल्यांकन
- AI सुरक्षा सत्यापन के लिए रेड टीमिंग

## 4. वर्कशॉप चुनौती

क्या आप अपनी क्षमताओं का परीक्षण करने के लिए तैयार हैं? इन चुनौतियों को आज़माएँ:

| Challenge | Description | Difficulty |
|:---|:---|:---|
| **1. मल्टी-एजेंट परिदृश्य** | किसी अन्य उद्योग के लिए तैनात और अनुकूलित करें (जैसे, स्वास्थ्य सेवा, वित्त) | ⭐⭐ |
| **2. कस्टम नॉलेज बेस** | अपने खुद के डेटा फ़ाइलें जोड़ें और एक कस्टम सर्च इंडेक्स बनाएँ | ⭐⭐⭐ |
| **3. प्रोडक्शन डिप्लॉयमेंट** | `azd pipeline config` का उपयोग करके GitHub Actions के साथ CI/CD सेट करें | ⭐⭐⭐⭐ |
