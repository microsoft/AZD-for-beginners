## सीखने के उद्देश्य

इस कार्यशाला के अंत तक, आप जानेंगे कि कैसे:

- [X] **Azure Developer CLI के साथ काम करना** → अपने Azure संसाधनों को provision, deploy, और manage करने के लिए।
- [X] **AI App Templates का उपयोग करना** → Microsoft Foundry पर AI Agent प्रोजेक्ट्स की शुरुआत करने के लिए।
- [X] **Templates को कॉन्फ़िगर करना** → Red Teaming, Evaluations, Tracing & Monitoring जैसे फीचर्स के लिए।
- [X] **डिकॉन्स्ट्रक्ट और एक्सप्लोर करना** → GitHub Copilot का उपयोग करके AZD टेम्पलेट्स और वर्कफ़्लोज़।
- [X] **कस्टमाइज़ और पुनः तैनात करना** → AI Agent starter टेम्पलेट को अपनी कस्टम आवश्यकताओं के साथ।

## कार्यशाला रूपरेखा

यह कार्यशाला मॉड्यूल में विभक्त है, प्रत्येक मॉड्यूल एक हैंड्स-ऑन लैब अभ्यास पर केंद्रित है:

1. **परिचय** - समझें कि AZD टेम्पलेट क्या हैं, और वे कैसे काम करते हैं
1. **Selection** - अपने अगले AI प्रोजेक्ट के लिए सही "starter" टेम्पलेट खोजें
1. **Validation** - टेम्पलेट को तैनात करें और अन्वेषण करें ताकि सत्यापित कर सकें कि यह बताए अनुसार काम करता है
1. **Deconstruction** - टेम्पलेट को अलग करें ताकि इसके घटकों और कॉन्फ़िगरेशन को समझा जा सके
1. **Configuration** - विभिन्न Microsoft Foundry सुविधाओं को सक्रिय और उपयोग करना सीखें
1. **Customization** - अपने डेटा और आवश्यकताओं को ध्यान में रखते हुए एप्लिकेशन को फिर से बनाएं
1. **Teardown** - इन्फ्रास्ट्रक्चर को deprovision करना और soft-deleted संसाधनों को पुनर्प्राप्त करना सीखें
