## Kursuse ülevaade

Õpi tundma Azure Developer CLI-d (azd) läbi struktureeritud peatükkide, mis on loodud järkjärguliseks õppimiseks. **Eriline rõhk on AI-rakenduste juurutamisel koos Azure AI Foundry integratsiooniga.**

### Miks see kursus on kaasaegsetele arendajatele oluline

Azure AI Foundry Discordi kogukonna andmetel soovib **45% arendajatest kasutada AZD-d AI töövoogude jaoks**, kuid nad seisavad silmitsi järgmiste väljakutsetega:
- Keerulised mitme teenusega AI arhitektuurid
- Parimad tavad AI juurutamiseks tootmises
- Azure AI teenuste integreerimine ja seadistamine
- AI töövoogude kulude optimeerimine
- AI-spetsiifiliste juurutusprobleemide tõrkeotsing

### Õpieesmärgid

Selle struktureeritud kursuse läbimisega:
- **Õpid AZD põhialuseid**: Põhikontseptsioonid, paigaldamine ja seadistamine
- **Juurutad AI-rakendusi**: Kasuta AZD-d koos Azure AI Foundry teenustega
- **Rakenda infrastruktuuri koodina**: Halda Azure'i ressursse Bicep-mallidega
- **Lahenda juurutusprobleeme**: Tuvasta ja paranda levinud vigu
- **Optimeeri tootmiseks**: Turvalisus, skaleerimine, monitooring ja kulude haldamine
- **Loo mitmeagendilisi lahendusi**: Juuruta keerulisi AI-arhitektuure

## 📖 Mis on Azure Developer CLI?

Azure Developer CLI (azd) on arendajakeskne käsurealiides, mis kiirendab rakenduste loomise ja Azure'i juurutamise protsessi. See pakub:

- **Mallipõhised juurutused** - Kasuta eelkonfigureeritud malle levinud rakendusmustrite jaoks
- **Infrastruktuur koodina** - Halda Azure'i ressursse Bicep'i või Terraformi abil  
- **Integreeritud töövood** - Rakenduste sujuv ettevalmistamine, juurutamine ja jälgimine
- **Arendajasõbralik** - Optimeeritud arendaja tootlikkuse ja kogemuse jaoks

### **AZD + Azure AI Foundry: Ideaalne tehisintellekti juurutamiseks**

**Miks valida AZD tehisintellekti lahenduste jaoks?** AZD lahendab peamised väljakutsed, millega tehisintellekti arendajad kokku puutuvad:

- **Tehisintellekti valmis mallid** - Eelkonfigureeritud mallid Azure OpenAI, Cognitive Services ja ML töökoormuste jaoks
- **Turvalised tehisintellekti juurutused** - Sisseehitatud turvalisuse mustrid AI teenuste, API võtmete ja mudeli lõpp-punktide jaoks  
- **Tootmise tehisintellekti mustrid** - Parimad tavad skaleeritavate ja kulutõhusate AI rakenduste juurutamiseks
- **Lõpuni viidud tehisintellekti töövood** - Mudeli arendamisest kuni tootmisse juurutamiseni koos nõuetekohase jälgimisega
- **Kulude optimeerimine** - Nutikad ressursside jaotamise ja skaleerimise strateegiad AI töökoormuste jaoks
- **Azure AI Foundry integratsioon** - Sujuv ühendus AI Foundry mudelikataloogi ja lõpp-punktidega
