# AZD Para Iniciantes: Estrutura do Curso e Quadro de Aprendizagem

## Visão Geral do Curso

Domine o Azure Developer CLI (azd) através de capítulos estruturados para aprendizado progressivo. **Foco especial no deployment de aplicações de IA com integração Microsoft Foundry.**

### Por Que Este Curso é Essencial para Desenvolvedores Modernos

Baseado em insights da comunidade Microsoft Foundry Discord, **45% dos desenvolvedores querem usar AZD para cargas de trabalho de IA** mas enfrentam desafios com:
- Arquiteturas complexas de IA com múltiplos serviços
- Melhores práticas para deployment de IA em produção  
- Integração e configuração dos serviços de IA do Azure
- Otimização de custos para cargas de trabalho de IA
- Solução de problemas específicos de deployment de IA

### Objetivos Principais de Aprendizagem

Ao concluir este curso estruturado, você irá:
- **Dominar os Fundamentos do AZD**: Conceitos principais, instalação e configuração
- **Deploy de Aplicações de IA**: Uso do AZD com serviços Microsoft Foundry
- **Implementar Infraestrutura como Código**: Gerenciar recursos Azure com templates Bicep
- **Resolver Problemas de Deploy**: Solução de problemas comuns e debugging
- **Otimizar para Produção**: Segurança, escalabilidade, monitoramento e gestão de custos
- **Construir Soluções Multi-Agentes**: Deploy de arquiteturas complexas de IA
