## 2. Principais Conceitos Dominados

### Azure Developer CLI (azd)
- `azd init` - Inicializar projetos a partir de templates
- `azd up` - Provisionar e implantar em um único comando
- `azd env` - Gerenciar variáveis de ambiente
- `azd down` - Limpar recursos com segurança

### Estrutura do Template AZD
- `azure.yaml` - Configuração e hooks de implantação
- `infra/` - Infraestrutura como código Bicep
- `src/` - Código-fonte da aplicação
- `scripts/` - Scripts de automação e configuração

### Capacidades do Microsoft Foundry
- AI Agent Service para assistentes inteligentes
- File Search e Azure AI Search para recuperação de conhecimento
- Rastreamento, monitoramento e avaliação integrados
- Red teaming para validação de segurança de IA
