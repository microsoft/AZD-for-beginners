# AZD Para Desenvolvedores de IA

!!! info "PÚBLICO-ALVO DESTE WORKSHOP"
   
    Este workshop é para qualquer pessoa que deseja construir aplicações agenticas de IA no Microsoft Foundry aproveitando ferramentas e fluxos de trabalho de ponta. O workshop é adequado tanto para iniciantes quanto para desenvolvedores experientes que tenham alguma familiaridade com Azure, conceitos de IA e código Python. Por favor, verifique os pré-requisitos antes de começar.

    **Duração:** 3-4 horas  
    **Nível:** Iniciante a Intermediário  

---


## Pré-Requisitos

- Uma conta pessoal no GitHub → [Create one for free](https://github.com/signup)
- Uma conta ativa do Azure → [Create one for free](https://aka.ms/free)
- Familiaridade com conceitos de IA, desenvolvimento em Python, ferramentas de linha de comando
- Experiência com VS Code e ferramentas e fluxos de trabalho do GitHub.

---

## Objetivos de Aprendizagem

Ao final deste workshop, você saberá como:

- [X] **Trabalhar com Azure Developer CLI** → para provisionar, implantar e gerenciar seus recursos do Azure.
- [X] **Usar Modelos de App de IA** → para iniciar projetos de Agente de IA no Microsoft Foundry.
- [X] **Configurar Modelos** → para recursos como Red Teaming, Avaliações, Rastreamento & Monitoramento
- [X] **Desconstruir e Explorar** → modelos e fluxos de trabalho do AZD usando o GitHub Copilot
- [X] **Personalizar e Reimplantar** → o template inicial de Agente de IA, com seus requisitos personalizados

---

## Estrutura do Workshop

The workshop is structured as modules, each anchored by a hands-on lab exercise:

1. **Introdução** - Entenda o que são os modelos AZD e como eles funcionam
1. **Seleção** - Descubra o "starter" template certo para seu próximo projeto de IA
1. **Validação** - Implemente e explore o template para verificar se ele funciona conforme anunciado
1. **Desconstrução** - Desmonte o template para entender seus componentes & configuração
1. **Configuração** - Aprenda a ativar e usar diferentes recursos do Microsoft Foundry
1. **Personalização** - Reconstrua a aplicação com seus dados e requisitos em mente
1. **Encerramento** - Aprenda a desprovisionar a infraestrutura e recuperar recursos excluídos temporariamente
