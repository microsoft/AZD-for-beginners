# AZD para Desenvolvedores de IA - Workshop

> Validado com `azd 1.25.6` em junho de 2026.

> [!IMPORTANT]  
> **Este workshop está instrumentado com um guia que você pode visualizar no seu navegador local. Para começar, abra o repo no GitHub Codespaces, em um contêiner de desenvolvimento ou em uma cópia local com o MkDocs instalado — então aguarde até ver um terminal ativo do VS Code e digite:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Você verá um diálogo pop-up para abrir a página de visualização em um navegador.**

Bem-vindo ao workshop prático para aprender o Azure Developer CLI (AZD) com foco na implantação de aplicações de IA. Este workshop ajuda você a obter uma compreensão aplicada dos modelos AZD em 3 etapas:

1. **Descoberta** - encontre o modelo mais adequado para você.
1. **Implantação** - implante e valide que funciona
1. **Personalização** - modifique e itere para torná-lo seu!

Ao longo deste workshop, você também será apresentado às principais ferramentas e fluxos de trabalho para desenvolvedores, para ajudar a otimizar sua jornada de desenvolvimento de ponta a ponta.

## Visão Geral do Workshop

**Duração:** 3-4 horas  
**Nível:** Iniciante a Intermediário  
**Pré-requisitos:** Familiaridade com Azure, conceitos de IA, VS Code e ferramentas de linha de comando.

Este é um workshop prático onde você aprende fazendo. Depois de concluir os exercícios, recomendamos revisar o currículo AZD Para Iniciantes para continuar sua jornada de aprendizado em práticas recomendadas de Segurança e Produtividade.

| Tempo| Módulo  | Objetivo |
|:---|:---|:---|
| 15 minutos | Introdução | Estabelecer o contexto, entender os objetivos |
| 30 minutos | Selecionar Modelo de IA | Explorar opções e escolher o modelo inicial | 
| 30 minutos | Validar Modelo de IA | Implantar a solução padrão no Azure |
| 30 minutos | Desconstruir Modelo de IA | Explorar estrutura e configuração |
| 30 minutos | Configurar Modelo de IA | Ativar e testar recursos disponíveis |
| 30 minutos | Personalizar Modelo de IA | Adaptar o modelo às suas necessidades |
| 30 minutos | Desmontar Infraestrutura | Limpar e liberar recursos |
| 15 minutos | Conclusão e Próximos Passos | Recursos de aprendizado, desafio do workshop |
| | |

## O que você vai aprender

Considere o Modelo AZD como um sandbox de aprendizado para explorar várias capacidades e ferramentas para desenvolvimento de ponta a ponta no Microsoft Foundry. Ao final deste workshop, você deve ter uma noção intuitiva de várias ferramentas e conceitos nesse contexto.

| Conceito  | Objetivo |
|:---|:---|
| **Azure Developer CLI** | Entender os comandos e fluxos de trabalho da ferramenta |
| **Modelos AZD**| Compreender a estrutura do projeto e a configuração |
| **Azure AI Agent**| Provisionar e implantar projeto no Microsoft Foundry  |
| **Azure AI Search**| Habilitar engenharia de contexto com agentes |
| **Observabilidade**| Explorar rastreamento, monitoramento e avaliações |
| **Red Teaming**| Explorar testes adversariais e mitigações |
