# الفصل 1: الأساس والبدء السريع

**📚 الدورة**: [AZD للمبتدئين](../../README.md) | **⏱️ المدة**: 30-45 دقيقة | **⭐ الصعوبة**: مبتدئ

---

## نظرة عامة

يقدم هذا الفصل أساسيات Azure Developer CLI (azd). ستتعلم المفاهيم الأساسية، تثبيت الأدوات، ونشر تطبيقك الأول إلى Azure.

> تم التحقق مقابل `azd 1.23.12` في مارس 2026.

## أهداف التعلم

بإكمال هذا الفصل، ستتمكن من:
- فهم ماهية Azure Developer CLI وكيف يختلف عن Azure CLI
- تثبيت وتكوين AZD على منصتك
- نشر تطبيقك الأول إلى Azure باستخدام `azd up`
- تنظيف الموارد باستخدام `azd down`

## 🚀 البدء السريع

```bash
# تحقق من التثبيت
azd version

# المصادقة لـ AZD
# اختياري: استخدم az login إذا كنت تخطط لتشغيل أوامر Azure CLI مباشرةً
azd auth login

# نشر تطبيقك الأول
azd init --template todo-nodejs-mongo
azd up

# نظّف عند الانتهاء
azd down --force --purge
```

---

## ✅ معايير النجاح

بعد إكمال هذا الفصل، يجب أن تكون قادرًا على:

```bash
azd version              # يعرض الإصدار المثبت
azd init --template todo-nodejs-mongo  # يهيئ المشروع
azd up                   # ينشر إلى أزور
azd show                 # يعرض عنوان URL للتطبيق قيد التشغيل
azd down --force --purge # ينظف الموارد
```
