# الفصل 1: الأساسيات والبدء السريع

**📚 الدورة**: [AZD للمبتدئين](../../README.md) | **⏱️ المدة**: 30-45 دقيقة | **⭐ الصعوبة**: مبتدئ

---

## نظرة عامة

يقدّم هذا الفصل أساسيات Azure Developer CLI (azd). ستتعلم المفاهيم الأساسية، وتثبت الأدوات، وتنشر تطبيقك الأول إلى Azure.

> تم التحقق باستخدام `azd 1.25.6` في يونيو 2026.

## أهداف التعلم

بإكمال هذا الفصل، ستتمكن من:
- فهم ما هو Azure Developer CLI وكيف يختلف عن Azure CLI
- تثبيت وتهيئة AZD على منصتك
- نشر تطبيقك الأول إلى Azure باستخدام `azd up`
- تنظيف الموارد باستخدام `azd down`

## ✅ ابدأ هنا: تحقق من إعدادك

قبل أن تبدأ، تأكد من أن جهازك المحلي جاهز لنموذج الفصل 1:

**Windows:**

**macOS / Linux:**

```bash
bash ./validate-setup.sh
```

## 🚀 البداية السريعة

```bash
# تحقق من التثبيت
azd version

# قم بالمصادقة لـ AZD
# اختياري: نفّذ az login إذا كنت تخطط لتشغيل أوامر Azure CLI مباشرة
azd auth login

# انشر تطبيقك الأول
azd init --template todo-nodejs-mongo
azd up

# نظّف عند الانتهاء
azd down --force --purge
```

---

## ✅ معايير النجاح

بعد إكمال هذا الفصل، يجب أن تكون قادرًا على:

```bash
azd version              # يعرض الإصدار المثبت
azd init --template todo-nodejs-mongo  # يهيئ المشروع
azd up                   # ينشر إلى أزور
azd show                 # يعرض عنوان URL للتطبيق الجاري تشغيله
azd down --force --purge # ينظف الموارد
```
