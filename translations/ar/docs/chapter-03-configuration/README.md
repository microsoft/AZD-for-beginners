# الفصل 3: التكوين والمصادقة

**📚 الدورة**: [AZD For Beginners](../../README.md) | **⏱️ المدة**: 45-60 دقائق | **⭐ مستوى الصعوبة**: متوسط

---

## نظرة عامة

يغطي هذا الفصل تكوين البيئة، أنماط المصادقة، وأفضل ممارسات الأمان لنشرات Azure Developer CLI.

> تم التحقق مع `azd 1.25.6` في يونيو 2026.

## أهداف التعلم

من خلال إكمال هذا الفصل، ستتمكن من:
- إتقان هرمية تكوين AZD
- إدارة بيئات متعددة (dev, staging, prod)
- تنفيذ مصادقة آمنة باستخدام الهويات المُدارة
- تكوين إعدادات خاصة بكل بيئة

## 🚀 البدء السريع

```bash
# إنشاء بيئات متعددة
azd env new dev
azd env new staging
azd env new prod

# التبديل بين البيئات
azd env select prod

# تعيين متغيرات البيئة
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# عرض التكوين
azd env get-values
```

---

## 🔧 هرمية التكوين

يقوم AZD بتطبيق الإعدادات بهذا الترتيب (القيم اللاحقة تتجاوز السابقة):

1. **القيم الافتراضية** (مضمّنة في القوالب)
2. **azure.yaml** (تكوين المشروع)
3. **متغيرات البيئة** (`azd env set`)
4. **وسائط سطر الأوامر** (`--location eastus`)

---

## 🔐 أفضل ممارسات الأمان

```bash
# استخدم الهوية المُدارة (موصى به)
azd env set AZURE_USE_MANAGED_IDENTITY true

# تحقق من حالة مصادقة AZD
azd auth status

# اختياري: تحقق من سياق Azure CLI إذا كنت تخطط لتشغيل أوامر az
az account show

# أعد المصادقة إذا لزم الأمر
azd auth login

# اختياري: قم بتحديث مصادقة Azure CLI لأوامر az
az login
```
