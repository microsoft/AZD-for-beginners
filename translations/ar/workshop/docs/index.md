# ورشة عمل AZD لمطوري الذكاء الاصطناعي

> تم التحقق باستخدام `azd 1.23.12` في مارس 2026.

> [!IMPORTANT]  
> **هذه الورشة مزودة بدليل ورشة يمكنك معاينته في متصفحك المحلي. للبدء، افتح المستودع في GitHub Codespaces، أو داخل حاوية تطوير، أو نسخة محلية مع تثبيت MkDocs — ثم انتظر حتى ترى طرفية VS Code نشطة واكتب:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **يجب أن ترى مربع حوار منبثق لفتح صفحة المعاينة في متصفح.**

مرحبًا بك في ورشة العمل العملية لتعلّم واجهة سطر أوامر Azure Developer CLI (AZD) مع التركيز على نشر تطبيقات الذكاء الاصطناعي. تساعدك هذه الورشة على اكتساب فهم عملي لقوالب AZD في 3 خطوات:

1. **الاكتشاف** - العثور على القالب المناسب لك.
1. **النشر** - النشر والتحقق من أنه يعمل
1. **التخصيص** - التعديل والتكرار لجعله خاصًا بك!

على مدار هذه الورشة، سيتم أيضًا تعريفك بأدوات وممارسات أساسية للمطورين، لمساعدتك على تبسيط رحلة التطوير من البداية إلى النهاية.

## نظرة عامة على الورشة

**المدة:** 3-4 ساعات  
**المستوى:** مبتدئ إلى متوسط  
**المتطلبات المسبقة:** الإلمام بـ Azure، مفاهيم الذكاء الاصطناعي، VS Code وأدوات سطر الأوامر.

هذه ورشة عملية تتعلم فيها من خلال التطبيق. بمجرد إكمال التمارين، نوصي بمراجعة منهج AZD للمبتدئين لمتابعة رحلة التعلم الخاصة بك نحو ممارسات الأمن والإنتاجية المثلى.

| الوقت| الوحدة  | الهدف |
|:---|:---|:---|
| 15 mins | Introduction | تهيئة السياق وفهم الأهداف |
| 30 mins | Select AI Template | استكشاف الخيارات واختيار قالب بداية | 
| 30 mins | Validate AI Template | نشر الحل الافتراضي في Azure |
| 30 mins | Deconstruct AI Template | استكشاف البنية والتكوين |
| 30 mins | Configure AI Template | تفعيل وتجربة الميزات المتاحة |
| 30 mins | Customize AI Template | تكييف القالب لاحتياجاتك |
| 30 mins | Teardown Infrastructure | تنظيف وإخلاء الموارد |
| 15 mins | Wrap-Up & Next Steps | موارد التعلم، تحدي الورشة |
| | |

## ما ستتعلمه

فكر في قالب AZD كصندوق رمل تعليمي لاستكشاف القدرات والأدوات المختلفة لتطوير شامل على Microsoft Foundry. بنهاية هذه الورشة، يجب أن يكون لديك إحساس حدسي بمختلف الأدوات والمفاهيم في هذا السياق.

| المفهوم  | الهدف |
|:---|:---|
| **Azure Developer CLI** | فهم أوامر الأداة وتدفقات العمل|
| **قوالب AZD**| فهم بنية المشروع والتكوين|
| **Azure AI Agent**| توفير ونشر مشروع Microsoft Foundry  |
| **Azure AI Search**| تمكين هندسة السياق باستخدام الوكلاء |
| **قابلية المراقبة**| استكشاف التتبع والمراقبة والتقييمات |
| **الهجوم الأحمر (Red Teaming)**| استكشاف الاختبارات العدائية والتخفيفات |
