# AZD For AI Developers

!!! info "AUDIENCE FOR THIS WORKSHOP"
   
    يهدف هذا الورشة لأي شخص يتطلع إلى بناء تطبيقات وكلاء ذكاء اصطناعي على Microsoft Foundry عبر الاستفادة من أفضل الأدوات وسير العمل. الورشة مناسبة للمبتدئين والمطورين ذوي الخبرة الذين لديهم بعض المعرفة بـ Azure ومفاهيم الذكاء الاصطناعي وكود Python. يرجى التحقق من المتطلبات المسبقة قبل البدء.

    **Duration:** 3-4 hours  
    **Level:** Beginner to Intermediate  

---


## Pre-Requisites

- A personal GitHub account → [Create one for free](https://github.com/signup)
- An active Azure account → [Create one for free](https://aka.ms/free)
- الإلمام بمفاهيم الذكاء الاصطناعي، وتطوير Python، وأدوات سطر الأوامر
- خبرة في VS Code وأدوات وسير عمل GitHub.

---

## Learning Objectives

بنهاية هذه الورشة، ستعرف كيف:

- [X] **Work with Azure Developer CLI** → لتوفير ونشر وإدارة موارد Azure الخاصة بك.
- [X] **Use AI App Templates** → لبدء مشاريع وكلاء الذكاء الاصطناعي على Microsoft Foundry بسرعة.
- [X] **Configure Templates** → لميزات مثل Red Teaming وEvaluations وTracing & Monitoring
- [X] **Deconstruct and Explore** → قوالب AZD وسير العمل باستخدام GitHub Copilot
- [X] **Customize and Redeploy** → قالب بدء تشغيل وكيل الذكاء الاصطناعي، مع متطلباتك المخصصة

---

## Workshop Outline

الورشة منظمة على شكل وحدات، كل منها مدعوم بتمرين مختبر عملي:

1. **Introduction** - فهم ما هي قوالب AZD، وكيف تعمل
1. **Selection** - اكتشاف قالب "البدء" المناسب لمشروع الذكاء الاصطناعي التالي الخاص بك
1. **Validation** - نشر واستكشاف القالب للتحقق من أنه يعمل كما هو معلن
1. **Deconstruction** - تفكيك القالب لفهم مكوناته وتكوينه
1. **Configuration** - تعلم تفعيل واستخدام ميزات Microsoft Foundry المختلفة
1. **Customization** - إعادة بناء التطبيق مع وضع بياناتك ومتطلباتك في الاعتبار
1. **Teardown** - تعلم إيقاف توفير البنية التحتية واسترداد الموارد المحذوفة مؤقتًا
