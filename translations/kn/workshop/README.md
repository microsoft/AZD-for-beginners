# AI ಡೆವೆಲಪರ್‌ಗಳಿಗಾಗಿ AZD ವರ್ಕ್‌ಶಾಪ್

AI ಅಪ್ಲಿಕೇಶನ್ ನಿಯೋಜನೆ ಮೇಲೆ ಗಮನಕೊಟ್ಟು Azure Developer CLI (AZD) ಕಲಿಯಲು ಕೈಗೊಳ್ಳುವ ವರ್ಕ್‌ಶಾಪ್‌ಗೆ ಸುಸ್ವಾಗತ. ಈ ವರ್ಕ್‌ಶಾಪ್ ನಿಮಗೆ AZD ಟೆಂಪ್ಲೇಟುಗಳನ್ನು 3 ಹಂತಗಳಲ್ಲಿ ಅನ್ವಯಿಕ ಅರ್ಥೈಸಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ:

1. **ಅನ್ವೇಷಣೆ** - ನಿಮ್ಮ ಕೈಗೆ ಸರಿಹೊಕುವ ಟೆಂಪ್ಲೇಟು ಹುಡುಕಿ.
1. **ನಿಯೋಜನೆ** - ನಿಯೋಜಿಸಿ ಮತ್ತು ಅದು ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ ಎಂದು ದೃಢೀಕರಿಸಿ
1. **ವ್ಯಕ್ತಿಗತೀಕರಣ** - ತಿದ್ದುಪಡಿ ಮಾಡಿ ಮತ್ತು ಸಿಂಚನ ಮಾಡಿ ನಿಮ್ಮದಾಗಿಸಿ!

ಈ ವರ್ಕ್‌ಶಾಪ್ ಅವಧಿಯಲ್ಲಿ, ಮುಖ್ಯ ಅಭಿವೃದ್ಧಿ ಸಾಧನಗಳು ಮತ್ತು ಕಾರ್ಯವಾಹಿತೆಗಳಿಗೂ ಪರಿಚಯವಾಗುತ್ತೀರಿ, ನಿಮ್ಮ ಅಂತ್ಯದಿಂದ ಅಂತ್ಯ ವಿಕಸನಯಾತ್ರೆಯನ್ನು ಸುಗಮಗೊಳಿಸಲು.

<br/>

## ಬ್ರೌಸರ್ ಆಧಾರಿತ ಮಾರ್ಗದರ್ಶನ

ವರ್ಕ್‌ಶಾಪ್ ಪಾಠಗಳು ಮಾರ್ಕ್ಡೌನ್‌ನಲ್ಲಿ ಇವೆ. ನೀವು ಅವುಗಳನ್ನು ನೇರಗೊಳಿಸಿ GitHubನಲ್ಲಿ ನವಿಗೇಟ್ ಮಾಡಬಹುದು - ಅಥವಾ ಕೆಳಗಿನ ಸ್ಕ್ರೀನ್ಶಾಟ್‌ನಲ್ಲಿ ತೋರಿಸಿದಂತೆ ಬ್ರೌಸರ್ ಆಧಾರಿತ ಪೂರ್ವವೀಕ್ಷಣೆಯನ್ನು ಪ್ರಾರಂಭಿಸಬಹುದು.

![Workshop](../../../translated_images/kn/workshop.75906f133e6f8ba0.webp)

ಈ ಆಯ್ಕೆ ಬಳಸಲು - ರೆಪೊಸಿಟರಿಯನ್ನು ನಿಮ್ಮ ಪ್ರೊಫೈಲಿಗೆ ಫೋರ್ಕ್ ಮಾಡಿ, ಮತ್ತು GitHub Codespaces ಅನ್ನು ಪ್ರಾರಂಭಿಸಿ. VS Code ಟರ್ಮಿನಲ್ ಸಕ್ರಿಯವಾದ ಬಳಿಕ ಈ ಕಮಾಂಡ್ ಟೈಪ್ ಮಾಡಿ:

ಈ ಬ್ರೌಸರ್ ಪೂರ್ವವೀಕ್ಷಣೆ GitHub Codespaces, ಡೆವ್ ಕಾಂಟೈನರ್‌ಗಳು ಅಥವಾ Python ಮತ್ತು MkDocs ಹೊಂದಿರುವ ಸ್ಥಳೀಯ ಕ್ಲೋನ್‌ನಲ್ಲಿ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ.

```bash title="" linenums="0"
mkdocs serve > /dev/null 2>&1 &
```

కొద్ది సెకన్లలో, మీరు ఒక పాప్-అప్ సంభాషణను చూడగలుగుతారు. 'Open in browser' ಆಯ್ಕೆಯನ್ನು ఎంచుకోండి. వెబ్-ఆధారిత గైడ్ ఇప్పుడు కొత్త బ్రౌజర్ ట್ಯాబ్‌లో తెరవబడుతుంది. ಈ ಪೂರ್ವವೀಕ್ಷಣೆಯ ಕೆಲವು ಲಾಭಗಳು:

1. **ಸಂರಚಿತ ಹುಡುಕಾಟ** - ಶೀಘ್ರವಾಗಿ ಕೀವರ್ಡ್‌ಗಳು ಅಥವಾ ಪಾಠಗಳನ್ನು ಹುಡುಕಿರಿ.
1. **ನಕಲು ಐಕಾನ್** - ಕೋಡ್ ಬ್ಲಾಕ್ಸ್ ಮೇಲೆ ಹೋವರ್ ಮಾಡಿದಾಗ ಈ ಆಯ್ಕೆ ಕಾಣುತ್ತದೆ
1. **ಥೀಮ್ ಟಾಗಲ್** - ಡಾರ್ಕ್ ಮತ್ತು ಲೈಟ್ ಥೀಮ್‌ಗಳ ಮಧ್ಯದಲ್ಲಿ ಬದಲಾಯಿಸಿ
1. **ಸಹಾಯ ಪಡೆಯಿರಿ** - ಫೂಟರ್‌ನಲ್ಲಿ ಡಿಸ್ಕೋರ್ಡ್ ಐಕಾನ್ ಗೆ ಕ್ಲಿಕ್ ಮಾಡಿ ಸೇರಿ!

## ವರ್ಕ್‌ಶಾಪ್ ಸವಾಲು

ನೀವು ಮತ್ತಷ್ಟು ಸವಾಲು ತೆಗೆದುಕೊಳ್ಳಲು ಇಚ್ಛಿಸುವಿರಾ? ಇಲ್ಲಿವೆ ಕೆಲವು ಪ್ರಾಜೆಕ್ಟ್ ಸಲಹೆಗಳು - ಅಥವಾ ನಿಮ್ಮ ಕಲ್ಪನೆಗಳನ್ನು ನಮ್ಮ ಜೊತೆಗೆ ಹಂಚಿಕೊಳ್ಳಿ!!

| ಪ್ರಾಜೆಕ್ಟ್ | ವಿವರಣೆ |
|:---|:---|
|1. **ಒಂದು ಸಂಕೀರ್ಣ AI ಟೆಂಪ್ಲೇಟ್ ಅನ್ನು ವಿಚ್ಛೇದಿಸಿ** | ನಾವು ಸಾರಿದ ಕಾರ್ಯವಾಹಿತ ಮತ್ತು ಸಾಧನಗಳನ್ನು ಬಳಸಿ, ಬೇರೆ AI ಪರಿಹಾರ ಟೆಂಪ್ಲೇಟ್ ಅನ್ನು ನಿಯೋಜಿಸಿ, ಮಾನ್ಯ ಮಾಡಿ ಮತ್ತು ವೈಯಕ್ತೀಕರಿಸಿ. _ನೀವು ಏನು ಕಲಿತಿರಿ?_|
|2. **ನಿಮ್ಮ ಸನ್ನಿವೇಶದೊಂದಿಗೆ ವೈಯಕ್ತೀಕರಿಸಿ**  | ಬೇರೆ ಸನ್ನಿವೇಶಕ್ಕಾಗಿ PRD (ಉತ್ಪನ್ನ ಅವಶ್ಯಕತೆಗಳು ದಸ್ತಾವೇಜು) ಬರೆದು ನೋಡಿ. ನಂತರ ನಿಮ್ಮ ಟೆಂಪ್ಲೇಟ್ ರೆಪೊದಲ್ಲಿ GitHub Copilot ಬಳಸಿ ಏಜೆಂಟ್ ಮಾದರಿಯಲ್ಲಿ ಇದನ್ನು ಕೋರಿ ಇದು ವೈಯಕ್ತೀಕರಣ ಕಾರ್ಯವಾಹಿತೆಯನ್ನು ರಚಿಸಲು. _ನೀವು ಏನು ಕಲಿತಿರಿ? ಈ ಸಲಹೆಗಳಲ್ಲಿ ನೀವು ಏನನ್ನು ಉತ್ತಮಗೊಳಿಸಬಹುದು?_|
| | |
