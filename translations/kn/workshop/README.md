# AI ಅಭಿವೃದ್ಧಿಪಡಿಸುವವರಿಗಾಗಿ AZD ಕಾರ್ಯಾಗಾರ

Azure Developer CLI (AZD) ಅನ್ನು AI ಅಪ್ಲಿಕೇಶನ್ ನಿಯೋಜನೆಗೆ ಕೇಂದ್ರೀಕರಿಸಿ ಕಲಿಯಲು ಈ ಪ್ರಾಯೋಗಿಕ ಕಾರ್ಯಾಗಾರಕ್ಕೆ ಸ್ವಾಗತ. ಈ ಕಾರ್ಯಾಗಾರವು ನಿಮಗೆ AZD ಟೆಂಪ್ಲೇಟುಗಳನ್ನು ಅನುಷ್ಠಾನಾತ್ಮಕವಾಗಿ ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು 3 ಹಂತಗಳಲ್ಲಿ ಸಹಾಯ ಮಾಡುತ್ತದೆ:

1. **ಆನ್ವೇಷಣೆ** - ನಿಮ್ಮಿಗೆ ಸೂಕ್ತವಾದ ಟೆಂಪ್ಲೇಟನ್ನು ಹುಡುಕಿ.
1. **ನಿಯೋಜನೆ** - ಅದನ್ನು ನಿಯೋಜಿಸಿ ಮತ್ತು ಅದು ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆಯೋ ಪರಿಶೀಲಿಸಿ
1. **ಕಸ್ಟಮೈಸೇಶನ್** - ಅದನ್ನು ಪರಿಷ್ಕರಿಸಿ ಮತ್ತು ನಿಮ್ಮದಾಗಿ ಮಾಡಲು ಪುನರಾವರ್ತಿಸಿ!

ಈ ಕಾರ್ಯಾಗಾರದ ಅವಧಿಯಲ್ಲಿ, ನಿಮಗೆ ಕೋರ್ ಡೆವಲಪರ್ ಸಾಧನಗಳು ಮತ್ತು ಕಾರ್ಯಪ್ರವಾಹಗಳನ್ನು ಪರಿಚಯಿಸಲಾಗುತ್ತದೆ, ಇದು ನಿಮ್ಮ ಎಂಡು-ಟ್-ಎಂಡ್ ಡೆವಲಪ್‌ಮೆಂಟ್ ಪ್ರಯಾಣವನ್ನು ಸೌಲಭ್ಯಗೊಳಿಸಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ.

<br/>

## ಬ್ರೌಸರ್ ಆಧಾರಿತ ಮಾರ್ಗದರ್ಶಿ

ಕಾರ್ಯಾಗಾರದ ಪಾಠಗಳು Markdown ನಲ್ಲಿ ಇವೆ. ನೀವು ಅವುಗಳನ್ನು ನೇರವಾಗಿ GitHub ನಲ್ಲಿ ನ್ಯಾವಿಗೇಟ್ ಮಾಡಬಹುದು - ಅಥವಾ ಕೆಳಗಿನ ಸ್ಕ್ರೀನ್‌ಶಾಟ್‌ನಲ್ಲಿ ತೋರಿಸಿದಂತೆ ಬ್ರೌಸರ್ ಆಧಾರಿತ ಪೂರ್ವದೃಶ್ಯವನ್ನು ಚಾಲನೆ ಮಾಡಿ.

![ಕಾರ್ಯಾಗಾರ](../../../translated_images/kn/workshop.75906f133e6f8ba0.webp)

ಈ ಆಯ್ಕೆಯನ್ನು ಬಳಸಲು - ರೆپೊಸಿಟರಿ ಅನ್ನು ನಿಮ್ಮ ಪ್ರೊಫೈಲಿಗೆ fork ಮಾಡಿ, ಮತ್ತು GitHub Codespaces ಪ್ರಾರಂಭಿಸಿ. VS Code ಟೆರ್ಮಿನಲ್ ಸಕ್ರಿಯವಾದ ನಂತರ, ಈ ಕಮಾಂಡ್ ಅನ್ನು ಟೈಪ್ ಮಾಡಿ:

```bash title="" linenums="0"
mkdocs serve > /dev/null 2>&1 &
```

ಕೆಲವು ಸೆಕೆಂಡುಗಳಲ್ಲಿ, ನಿಮಗೆ ಒಂದು ಪಾಪ್-ಅಪ್ ಡೈಲಾಗ್ ಕಾಣಿಸಿಕೊಳ್ಳುತ್ತದೆ. `ಬ್ರೌಸರ್‌ನಲ್ಲಿ ತೆರೆಯಿರಿ` ಆಯ್ಕೆ ಅನ್ನು ಆರಿಸಿ. ವೆಬ್ ಆಧಾರಿತ ಮಾರ್ಗದರ್ಶಿ ಈಗ ಹೊಸ ಬ್ರೌಸರ್ ಟ್ಯಾಬ್‌ನಲ್ಲಿ ತೆರೆಯುತ್ತದೆ. ಈ ಪೂರ್ವದೃಶ್ಯದ ಕೆಲವು ಲಾಭಗಳು:

1. **ಒಳನಿರ್ಮಿತ ಹುಡುಕಾಟ** - ಕೀವರ್ಡ್‌ಗಳು ಅಥವಾ ಪಾಠಗಳನ್ನು ಶೀಘ್ರವಾಗಿ ಹುಡುಕಿ.
1. **ಕಾಪಿ ಐಕಾನ್** - ಕೋಡ್‌ಬ್ಲಾಕ್ಗಳ ಮೇಲೆ ಹೋವರ್ ಮಾಡಿದರೆ ಈ ಆಯ್ಕೆಯನ್ನು ಕಾಣಬಹುದು
1. **ಥೀಮ್ ಟೋಗಲ್** - ಕತ್ತಲೆ ಮತ್ತು ಬೆಳಕಿನ ಥೀಮ್‌ಗಳ ನಡುವೆ ಬದಲಾಯಿಸಿ
1. **ಸಹಾಯ ಪಡೆದುಕೊಳ್ಳಿ** - ಸೇರಲು ಫೂಟರ್‌ನಲ್ಲಿ ಇರುವ Discord ಐಕಾನ್ ಕ್ಲಿಕ್ ಮಾಡಿ!

## ಕಾರ್ಯಾಗಾರ ಸವಾಲು

ಹೆಚ್ಚು ಮಾಡಲು ನಿಮ್ಮನ್ನು ಸವಾಲು ನೀಡಬೇಕೆಂದು ಬಯಸುತ್ತೀರಾ? ಇಲ್ಲಿ ಕೆಲವು ಯೋಜನೆ ಸಲಹೆಗಳು ಇವೆ - ಅಥವಾ ನಿಮ್ಮ ಐಡಿಯಾಗಳನ್ನು ನಮಗೆ ಹಂಚಿ!!

| ಯೋಜನೆ | ವಿವರಣೆ |

|1. **ಒಂದು ಸಂಕೀರ್ಣ AI ಟೆಂಪ್ಲೇಟನ್ನು ವಿಭಜಿಸಿ** | ನಾವು ವಿವರಿಸಿದ ಕಾರ್ಯಪ್ರವಾಹ ಮತ್ತು ಸಾಧನಗಳನ್ನು ಬಳಸಿಕೊಂಡು, ನೀವು ಬೇರೆ AI ಪರಿಹಾರ ಟೆಂಪ್ಲೇಟನ್ನು ನಿಯೋಜಿಸಿ, ಪರಿಶೀಲಿಸಿ, ಮತ್ತು ಕಸ್ಟಮೈಸ್ ಮಾಡಬಹುದೇ ಎಂದು ಪ್ರಯತ್ನಿಸಿ. _ನೀವು ಏನು ಕಲಿತಿರಿ?_|
