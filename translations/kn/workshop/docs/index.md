# AZD for AI Developers Workshop

> Validated against `azd 1.25.6` in ಜೂನ್ 2026.

> [!IMPORTANT]  
> **ಈ ಕಾರ್ಯಾಗಾರವು ನಿಮ್ಮ ಸ್ಥಳೀಯ ಬ್ರೌಸರ್‌ನಲ್ಲಿ ಪೂರ್ವವೀಕ್ಷಿಸಲು ಸಾಧ್ಯವಿರುವ ಕಾರ್ಯಾಗಾರ ಮಾರ್ಗದರ್ಶಿಯನ್ನು ಒಳಗೊಂಡಿದೆ. ಪ್ರಾರಂಭಿಸಲು, GitHub Codespaces, dev container ಅಥವಾ MkDocs ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಿರುವ ಸ್ಥಳೀಯ ಕ್ಲೋನ್‌ನಲ್ಲಿ ರೆಪೊವನ್ನು ತೆರೆಯಿರಿ—ನಂತರ ಸಕ್ರಿಯ VS Code ಟರ್ಮಿನಲ್ ಗೋಚರವಾಯಿತುವರೆಗೂ ಕಾಯಿರಿ ಮತ್ತು ಟೈಪ್ ಮಾಡಿ:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **ನೀವು ಬ್ರೌಸರ್‌ನಲ್ಲಿ ಪೂರ್ವವೀಕ್ಷಣಾ ಪುಟವನ್ನು ತೆರೆಯಲು ಪಾಪ್-ಅಪ್ ಸಂವಾದವನ್ನು ಕಾಣುವಿರಿ.**

Azure Developer CLI (AZD) ಅನ್ನು AI ಅಪ್ಲಿಕೇಶನ್ ನಿಯೋಜನೆ ಮೇಲೆ ಗಮನ ತೋರಿಸುವ ಹಸ್ತಚಾಲಿತ ಕಾರ್ಯಾಗಾರಕ್ಕೆ ಸ್ವಾಗತ. ಈ ಕಾರ್ಯಾಗಾರವು ನಿಮಗೆ AZD ಟೆಂಪ್ಲೇಟುಗಳ ಬಳಕೆಯ ಅನ್ವಯಿಕ ಅರ್ಥವನ್ನು 3 ಹಂತಗಳಲ್ಲಿ ತಿಳಿಸುತ್ತದೆ:

1. **Discovery** - ನಿಮಗೆ ಯೋಗ್ಯವಾದ ಟೆಂಪ್ಲೇಟನ್ನು ಹುಡುಕಿ.
1. **Deployment** - ನಿಯೋಜಿಸಿ ಮತ್ತು ಅದು ಕೆಲಸ ಮಾಡುತ್ತದೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ
1. **Customization** - ನಿಮ್ಮದಾಗಿಸಲು ಸಂಶೋಧಿಸಿ ಮತ್ತು ಪುನರಾವರ್ತಿಸಿ!

ಈ ಕಾರ್ಯಾಗಾರದ ಅವಧಿಯಲ್ಲಿ, ನೀವು ಕೋರ್ ಡೆವಲಪರ್ ಸಾಧನಗಳು ಮತ್ತು ವರ್ಕ್‌ಫ್ಲೋಗಳ ಪರಿಚಯವನ್ನು ಕೂಡ ಪಡೆಯುತ್ತೀರಿ, ಇದು ನಿಮ್ಮ ಎಂಡ್-ಟು-ಎಂಡ್ ಅಭಿವೃದ್ಧಿ ಪ್ರಯಾಣವನ್ನು ಸರಳಗೊಳಿಸಲು ಸಹಾಯಕವಾಗಲಿದೆ.

## Workshop Overview

**ಅವಧಿ:** 3-4 ಗಂಟೆಗಳು  
**ಮಟ್ಟ:** ಪ್ರಾರಂಭಿಕದಿಂದ ಮಧ್ಯಮ  
**ಮುಂಚಿತ ಅಗತ್ಯಗಳು:** Azure, AI ತತ್ವಗಳು, VS Code & ಕಮಾಂಡ್-ಲೈನ್ ಸಾಧನಗಳ ಪರಿಚಯ.

ಇದು ಕೈಯಲ್ಲಿ ಕಲಿಯುವ ಕಾರ್ಯಾಗಾರವಾಗಿದ್ದು, ನೀವು ಮಾಡಿ ಕಲಿಯುವ ಮೂಲಕ ಅಭ್ಯಾಸ ಮಾಡುತ್ತಾರೆ. ಅತ್ಯಂತ ವ್ಯಾಯಾಮಗಳನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ ನಂತರ, ನಾವು ನಿಮ್ಮ ಅಭ್ಯಾಸವನ್ನು Security ಮತ್ತು Productivity ಉತ್ತಮ ಕಾರ್ಯಪ್ರವಾಹಗಳ ಕಡೆಗೆ ಮುಂದುವರೆಸಲು AZD For Beginners ಪಠ್ಯಕ್ರಮವನ್ನು ಪರಾಮರ್ಶಿಸುವುದನ್ನು ಶಿಫಾರಸು ಮಾಡುತ್ತೇವೆ.

| Time| Module  | Objective |
|:---|:---|:---|
| 15 mins | ಪರಿಚಯ | ಹಂತವನ್ನು ಸೆಟ್ ಮಾಡುವುದು, ಗುರಿಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವುದು |
| 30 mins | AI ಟೆಂಪ್ಲೇಟು ಆಯ್ಕೆಮಾಡಿ | ಆಯ್ಕೆಗಳು ಅನ್ವೇಷಿಸಿ ಮತ್ತು ಪ್ರಾರಂಭಿಕವನ್ನು ಆರಿಸಿಕೊಳ್ಳಿ | 
| 30 mins | AI ಟೆಂಪ್ಲೇಟು ಪರಿಶೀಲಿಸು | ಡೀಫಾಲ್ಟ್ ಪರಿಹಾರವನ್ನು Azure ಗೆ ನಿಯೋಜಿಸಿ |
| 30 mins | AI ಟೆಂಪ್ಲೇಟು ವಿಕಾಸಿಸು | ರಚನೆ ಮತ್ತು ಸಂರಚನೆ ಅನ್ವೇಷಿಸಿ |
| 30 mins | AI ಟೆಂಪ್ಲೇಟು ಸಂರಚನೆ | ಲಭ್ಯವಿರುವ ವೈಶಿಷ್ಟ್ಯಗಳನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ ಮತ್ತು ಪ್ರಯತ್ನಿಸಿ |
| 30 mins | AI ಟೆಂಪ್ಲೇಟು ಕಸ್ಟಮೈಸ್ ಮಾಡಿ | ಟೆಂಪ್ಲೇಟನ್ನು ನಿಮ್ಮ ಅಗತ್ಯಗಳಿಗೆ ಅನುಕೂಲಗೊಳಿಸಿ |
| 30 mins | ಮೂಲಸೌಕರ್ಯರ ರದ್ದುಗೊಳಿಸು | ಕ್ಲೀನ್ ಅಪ್ ಮಾಡಿ ಮತ್ತು ಸಂಪನ್ಮೂಲ ಬಿಡುಗಡೆ ಮಾಡಿ |
| 15 mins | ಸಮಾರೋಪ & ಮುಂದಿನ ಹಂತಗಳು | ಕಲಿಕೆಯ ಸಂಪನ್ಮೂಲಗಳು, ಕಾರ್ಯಾಗಾರ ಸವಾಲು |

## What You'll Learn

AZD ಟೆಂಪ್ಲೇಟನ್ನು Microsoft Foundry ನಲ್ಲಿ ಎಂಡ್-ಟು-ಎಂಡ್ ಅಭಿವೃದ್ಧಿಗಾಗಿ ವಿವಿಧ ಸಾಮರ್ಥ್ಯಗಳು ಮತ್ತು ಸಾಧನಗಳನ್ನು ಅನ್ವೇಷಿಸಲು ಕಲಿಯುವ ಸ್ಯಾಂಡ್‌ಬಾಕ್ಸ್ ಎಂದು ಪರಿಗಣಿಸಿ. ಈ ಕಾರ್ಯಾಗಾರದ ಅಂತ್ಯಕ್ಕೆ, ನೀವು ಈ ಸ೦ದರ್ಭದಲ್ಲಿ ವಿವಿಧ ಸಾಧನಗಳು ಮತ್ತು ತತ್ವಗಳ ಬಗ್ಗೆ ಒಬ್ಬ ಅನುಭಾವಾತ್ಮಕ ಅರ್ಥವನ್ನು ಹೊಂದಿರಬೇಕು.

| Concept  | Objective |
|:---|:---|
| **Azure Developer CLI** | ಸಾಧನದ ಕಮಾಂಡ್‌ಗಳು ಮತ್ತು ವರ್ಕ್‌ಫ್ಲೋಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಿ|
| **AZD Templates**| ಪ್ರಾಜೆಕ್ಟ್ ರಚನೆ ಮತ್ತು ಕಾನ್ಫಿಗರೇಶನ್ ಅನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಿ|
| **Azure AI Agent**| Microsoft Foundry ಪ್ರಾಜೆಕ್ಟ್ ಪ್ರೊವಿಷನ್ ಮತ್ತು ನಿಯೋಜನೆ  |
| **Azure AI Search**| ಏಜೆಂಟ್‌ಗಳೊಂದಿಗೆcontext engineering (ಸಂದರ್ಭ ಎಂಜಿನಿಯರಿಂಗ್) ಸಕ್ರಿಯಗೊಳಿಸಿ |
| **Observability**| ಟ್ರೇಸಿಂಗ್, ಮಾನಿಟರಿಂಗ್ ಮತ್ತು ಮೌಲ್ಯಮಾಪನಗಳನ್ನು ಅನ್ವೇಷಿಸಿ |
| **Red Teaming**| ಪ್ರತಿಸ್ಪಂಧಾತ್ಮಕ ಪರೀಕ್ಷೆ ಮತ್ತು ಪರಿಹಾರಗಳನ್ನು ಅನ್ವೇಷಿಸಿ |
| | |
