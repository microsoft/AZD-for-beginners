# AI ಅಭಿವೃದ್ಧಿಗಾರರಿಗಾಗಿ AZD ಕಾರ್ಯಾಗಾರ

> ಜುಲೈ 2026ರಲ್ಲಿ `azd 1.27.1`ನೊಂದಿಗೆ ಪರಿಶೀಲಿಸಲಾಗಿದೆ.

> [!IMPORTANT]  
> **ಈ ಕಾರ್ಯಾಗಾರವು ಕಾರ್ಯಾಗಾರ ಮಾರ್ಗದರ್ಶಿಯೊಂದಿಗೆ ಸಂಯೋಜಿಸಲಾಗಿದೆ, ಇದನ್ನು ನೀವು ನಿಮ್ಮ ಸ್ಥಳೀಯ ಬ್ರೌಸರ್‌ನಲ್ಲಿ ಪೂರ್ವವೀಕ್ಷಣೆ ಮಾಡಬಹುದು. ಪ್ರಾರಂಭಿಸಲು, GitHub Codespaces, ಡೆವ್ ಕಂಟೈನರ್, ಅಥವಾ MkDocs ಸ್ಥಾಪಿತ ಆಗಿರುವ ಸ್ಥಳೀಯ ಕ್ಲೋನ್‌ನಲ್ಲಿ ರೆಪೊವನ್ನು ತೆರೆಯಿರಿ—ನಂತರ ಸಕ್ರಿಯ VS ಕೋಡ್ ಟರ್ಮಿನಲ್ ಕಾಣುವವರೆಗೆ ಕಾಯಿರಿ ಮತ್ತು ಟೈಪ್ ಮಾಡಿ:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **ನೀವು ಬ್ರೌಸರ್ನಲ್ಲಿ ಪೂರ್ವವೀಕ್ಷಣಾ ಪುಟವನ್ನು ತೆರೆಯಲು ಪಾಪ್-ಅಪ್ ಡೈಲಾಗ್ ಅನ್ನು ನೋಡಬೇಕು.**

AI ಅಪ್ಲಿಕೇಶನ್ ನಿಯೋಜನೆಗೆ ದೃಷ್ಠಿಪಾತ ನೀಡಿರುವ ಅಜೂರ್ ಡೆವಲಪರ್ CLI (AZD) ಪರಿಚಯಿಸಲು ಕೈಯಲ್ಲಿ ಕೈ ಸಭಾ ಕಾರ್ಯಕ್ರಮಕ್ಕೆ ಸ್ವಾಗತ. ಈ ಕಾರ್ಯಾಗಾರವು AZD ಟೆಂಪ್ಲೇಟ್ಗಳ ಅನ್ವಯಿಕ ಗ್ರಹಿಕೆಯನ್ನು 3 ಹಂತಗಳಲ್ಲಿ ಸಹಾಯ ಮಾಡುತ್ತದೆ:

1. **ಆಚಿತ್ರಣೆ** - ನಿಮ್ಮಿಗಾಗಿ ಸರಿಯಾದ ಟೆಂಪ್ಲೇಟ್ ಅನ್ನು ಹುಡುಕಿ.
1. **ನಿಯೋಜನೆ** - ನಿಯೋಜಿಸಿ ಮತ್ತು ಅದು ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ ಎಂದು ಪರಿಶೀಲಿಸಿ
1. **ಸ್ವನಿರ್ಮಾಣ** - ನೀವು ತೃಪ್ತರಾಗುವಂತೆ ಪರಿಷ್ಕರಿಸಿ ಮತ್ತು ಪುನರಾವರ್ತಿಸಿ!

ಈ ಕಾರ್ಯಾಗಾರದ ಅವಧಿಯಲ್ಲಿ, ನೀವು ಮೂಲ ಡೇವಲಪರ್ ಸಾಧನಗಳು ಮತ್ತು ಕೆಲಸಮೂಲಕಗಳನ್ನು ಪರಿಚಯಿಸಲಾಗುತ್ತದೆ, ಇದು ನಿಮ್ಮ ಪೂರ್ಣ-ಪ್ರಕ್ರಿಯೆ ಅಭಿವೃದ್ಧಿ ಯಾತ್ರೆಯನ್ನು ಸುಗಮಗೊಳಿಸಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ.

## ಕಾರ್ಯಾಗಾರದ ಅವಲೋಕನ

**ಅವಧಿ:** 3-4 ಗಂಟೆಗಳು  
** ಮಟ್ಟ:** ಪ್ರಾರಂಭಿಕರಿಂದ ಮಧ್ಯಮ ಮಟ್ಟದವರೆಗೂ  
**ಅಗತ್ಯತೆ:** ಆಜೂರ್, AI ಸಂಪ್ರದಾಯಗಳು, VS ಕೋಡ್ ಮತ್ತು ಕಮಾಂಡ್-ಲೈನ್ ಸಾಧನಗಳಲ್ಲಿ ಪರಿಚಿತತೆ.

ಇದು ಕೈಯಲ್ಲಿ ಮಾಡಬೇಕಾದ ಕಾರ್ಯಾಗಾರವೇ, ನೀವು ಆಧ್ಯಯನ ಮಾಡುವ ಮೂಲಕ ಕಲಿಯುವಿರಿ. ವ್ಯಾಯಾಮಗಳನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ ನಂತರ, ನಿಮ್ಮ ಕಲಿಕಾ ಪಯಣವನ್ನು ಭದ್ರತೆ ಮತ್ತು ಉತ್ಪಾದಕತೆ ಅತ್ಯುತ್ತಮ ಅಭ್ಯಾಸಗಳಿಗೆ ಮುಂದುವರಿಸಲು AZD For Beginners ಪಠ್ಯಕ್ರಮವನ್ನು ಪರಿಶೀಲಿಸಲು ನಾವು ಶಿಫಾರಸು ಮಾಡುತ್ತೇವೆ.

## ನೀವು ಕಲಿಯಲಿರುವುದು

AZD ಟೆಂಪ್ಲೇಟ್ ಅನ್ನು Microsoft Foundryಯಲ್ಲಿ ಪೂರ್ಣ-ಪ್ರಕ್ರಿಯೆ ಅಭಿವೃದ್ಧಿಗಾಗಿನ ವಿವಿಧ ಸಾಮರ್ಥ್ಯಗಳು ಮತ್ತು ಸಾಧನಗಳನ್ನು ಪರಿಕೀಕ್ಷಿಸಲು ಕಲಿಕೆಯ ಮರಳುಪೆಟ್ಟಿಗೆಯಾಗಿ ಮನಸ್ಸಿನಲ್ಲಿ ಇರಿಸಿ. ಈ ಕಾರ್ಯಾಗಾರದ ಅಂತ್ಯದಲ್ಲಿ, ನೀವು ಈಸಂದರ್ಭದಲ್ಲಿ ವಿವಿಧ ಸಾಧನಗಳು ಮತ್ತು ಸಂಪ್ರದಾಯಗಳ ಕುರಿತಾಗುವ ಅಂತಃಭವನೆಯನ್ನು ಹೊಂದಿರಬೇಕೆಂದು ನಿರೀಕ್ಷಿಸಲಾಗಿದೆ.

| ಸಂಪ್ರದಾಯ  | ಉದ್ದೇಶ |
|:---|:---|
| **ಅಜೂರ್ ಡೆವಲಪರ್ CLI** | ಸಾಧನ ಕಮಾಂಡ್‌ಗಳು ಮತ್ತು ಕೆಲಸಮೂಲಕಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಿ|
| **AZD ಟೆಂಪ್ಲೇಟ್ಗಳು**| ಯೋಜನೆಯ ರಚನೆ ಮತ್ತು ಸಂರಚನೆಯನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಿ|
| **ಅಜೂರ್ AI ಏಜೆಂಟ್**| Microsoft Foundry ಯೋಜನೆಯನ್ನು ವಹಿಸಿ ಮತ್ತು ನಿಯೋಜಿಸಿ  |
| **ಅಜೂರ್ AI ಹುಡುಕಾಟ**| ಏಜೆಂಟ್‌ಗಳೊಂದಿಗೆ ವಿಷಯ ವಿಜ್ಞಾನಕ್ಕೆ ಅವಕಾಶ ನೀಡಿ |
| **ಗಮನಾರ್ಹತೆ**| ಟ್ರೇಸಿಂಗ್, ಮಾನಿಟರಿಂಗ್ ಮತ್ತು ಮೌಲ್ಯಮಾಪನಗಳನ್ನು ಪರಿಶೀಲಿಸಿ |
| **ರೆಡ್ ತಂಡದ ಕಾರ್ಯ**| ವಿರುದ್ಧ ಪರೀක්ෂಣೆ ಮತ್ತು ಪ್ರತಿ ತೆರವಿಂಗಳ ಪರಿಶೀಲನೆ ಮಾಡಿ |
| | |
