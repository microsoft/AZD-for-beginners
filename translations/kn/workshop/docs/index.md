# AI ಅಭಿವೃದ್ಧಿಪರರಿಗೆ AZD ಕಾರ್ಯಾಗಾರ

> ಮಾರ್ಚ್ 2026 ರಲ್ಲಿ `azd 1.23.12` ವಿರುದ್ಧ ಪರಿಶೀಲಿಸಲಾಗಿದೆ.

> [!IMPORTANT]  
> **ಈ ಕಾರ್ಯಾಗಾರವು ನಿಮ್ಮ ಸ್ಥಳೀಯ ಬ್ರೌಸರಿನಲ್ಲಿ ಪೂರ್ವಾವಲೋಕನ ಮಾಡಲು ಸಾಧ್ಯವಾದ ಕಾರ್ಯಾಗಾರ ಗೈಡ್‌ನೊಂದಿಗೆ ಸಂಯೋಜಿಸಲಾಗಿದೆ. ಪ್ರಾರಂಭ ಮಾಡಲು, GitHub Codespaces, ಡೆವ್ ಕಂಟೇನರ್ ಅಥವಾ MkDocs ಅನ್ನು ಸ್ಥಾಪಿಸಿರುವ ಸ್ಥಳೀಯ ಕ್ಲೋನ್‌ನಲ್ಲಿ ರೆಪೊ ತೆಗೆಯಿರಿ—ನಂತರ ಕಾರ್ಯನಿಷ್ಠ VS ಕೋಡ್ ಟರ್ಮಿನಲ್ ಕಾಣಿಸಿಕೊಂಡಾಗ ಕೆಳಗಿನ ಟೈಪ್ ಮಾಡಿ:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **ನೀವು ಪಾಪ್-ಅಪ್ ಡೈಲಾಗ್ ಅನ್ನು ಬ್ರೌಸರಿನಲ್ಲಿ ಪೂರ್ವಾವಲೋಕನ ಪುಟವನ್ನು ತೆರೆಯಲು ನೋಡಬಹುದು.**

ಕೃತಕ ಬುದ್ಧಿಮತ್ತೆ ಅಪ್ಲಿಕೇಶನ್ ನಿಯೋಜನೆಯ ಮೇಲೆ ಗಮನಹರಿಸಿ Azure Developer CLI (AZD) ಕಲಿಯಿಸಲು ಕೈಗೊಳ್ಳುವ ಕಾರ್ಯಾಗಾರಕ್ಕೆ ಸುಸ್ವಾಗತ. ಈ ಕಾರ್ಯಾಗಾರವು ನಿಮ್ಮನ್ನು 3 ಹಂತಗಳಲ್ಲಿ AZD ಟೆಂಪ್ಲೇಟುಗಳ ಅನ್ವಯಾತ್ಮಕ ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ:

1. **ಹುಡುಕು** - ನಿಮ್ಮಿಗಿಂತ ಸರಿಯಾದ ಟೆಂಪ್ಲೇಟ್ ಅನ್ನು ಹುಡುಕಿ.
1. **ನಿಯೋಜನೆ** - ನಿಯೋಜಿಸಿ ಮತ್ತು ಅದು ಕೆಲಸಮಾಡುತ್ತದೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ
1. **ವೈಯಕ್ತಿಕೀಕರಣ** - ಅದನ್ನು ಮಾರ್ಪಡಿಸಿ ಮತ್ತು ನಿಮ್ಮದೇ ಮಾಡಿಕೊಳ್ಳಿ!

ಈ ಕಾರ್ಯಾಗಾರದ ಅವಧಿಯಲ್ಲಿ, ನೀವು ಕೋರ್ ಡೆವಲಪರ್ ಸಾಧನಗಳು ಮತ್ತು ಕಾರ್ಯವಿಧಾನಗಳ ಪರಿಚಯವನ್ನು ಹೊಂದಲಾಗುತ್ತದೆ, ಇದು ನಿಮ್ಮ ಅಂತರ್ಯಂತರ ಅಭಿವೃದ್ಧಿ ಪ್ರಯಾಣವನ್ನು ಸುಗಮಗೊಳಿಸಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ.

## ಕಾರ್ಯಾಗಾರದ ಅವಲೋಕನ

**ಅವಧಿ:** 3-4 ಗಂಟೆಗಳು  
**ಮಟ್ಟ:** ಆರಂಭಿಕರಿಂದ ಮಧ್ಯಮ ವರೆಗೆ  
**ಮುಂಚೂಣಿ ಜ್ಞಾನಗಳು:** Azure, AI ಕಲ್ಪನೆಗಳು, VS ಕೋಡ್ ಮತ್ತು ಕಮಾಂಡ್-ಲೈನ್ ಸಾಧನಗಳ ಪರಿಚಯ.

ಇದು ಕೈಗೊಳ್ಳುವ ಕಾರ್ಯಾಗಾರವಾಗಿದೆ, ನೀವು ಕೈಗಾರಿಕೆ ಮೂಲಕ ಕಲಿಯುವಿರಿ. ಅಭ್ಯಾಸಗಳನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ ನಂತರ, ಸುರಕ್ಷತೆ ಮತ್ತು ಉತ್ಪಾದಕತೆಯ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳ ಕುರಿತು ನಿಮ್ಮ ಕಲಿಕಾ ಪ್ರಯಾಣವನ್ನು ಮುಂದುವರಿಸಲು AZD ಆರಂಭಿಕರ ಪರಿಚಯಕ್ರಮವನ್ನು ಪರಿಶೀಲಿಸುವುದು ಸೂಕ್ತ.

| 30 ನಿಮಿ | AI ಟೆಂಪ್ಲೇಟ್ ಆಯ್ಕೆಮಾಡಿ | ಆಯ್ಕೆಗಳನ್ನು ಅನ್ವೇಷಿಸಿ ಮತ್ತು ಪ್ರಾರಂಭಿಕವನ್ನು ಆರಿಸಿ | 
| 30 ನಿಮಿ | AI ಟೆಂಪ್ಲೇಟ್ ಪರಿಶೀಲನೆ | ಡೀಫಾಲ್ಟ್ ಪರಿಹಾರವನ್ನು Azure ಗೆ ನಿಯೋಜಿಸಿ |
| 30 ನಿಮಿ | AI ಟೆಂಪ್ಲೇಟ್ನ ವಿಶ್ಲೇಷಣೆ | ರಚನೆ ಮತ್ತು ಸಂರಚನೆಯನ್ನು ಅನ್ವೇಷಿಸಿ |
| 30 ನಿಮಿ | AI ಟೆಂಪ್ಲೇಟ್ನ ಕಾನ್ಫಿಗರ್ ಮಾಡಿ | ಲಭ್ಯವಿರುವ ವೈಶಿಷ್ಟ್ಯಗಳನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ ಮತ್ತು ಪ್ರಯತ್ನಿಸಿ |
| 30 ನಿಮಿ | AI ಟೆಂಪ್ಲೇಟ್ ಕಸ್ಟಮೈಸ್ ಮಾಡಿ | ನಿಮ್ಮ ಅಗತ್ಯಗಳಿಗೆ ಹೊಂದಿಸಿ |
| 30 ನಿಮಿ | ಮೂಲಸೌಕರ್ಯ ತೊರೆ ಮಾಡಿ | ಸ್ವಚ್ಛಗೊಳಿಸಿ ಮತ್ತು ಸಂಪನ್ಮೂಲಗಳನ್ನು ಬಿಡುಗಡೆ ಮಾಡಿ |

## ನೀವು ಕಲಿಯುವ ವಿಷಯಗಳು

Microsoft Foundry ನಲ್ಲಿ ಅಂತ್ಯದಿಂದ ಆರಂಭದವರೆಗೆ ಅಭಿವೃದ್ಧಿಗೆ ವಿವಿಧ ಸಾಮರ್ಥ್ಯಗಳು ಮತ್ತು ಸಾಧನಗಳನ್ನು ಅನ್ವೇಷಿಸುವ ಕಲಿಕಾ ಸ್ಯಾಂಡ್‌ಬಾಕ್ಸ್ ಎಂದು AZD ಟೆಂಪ್ಲೇಟ್ನನ್ನು ಪರಿಗಣಿಸಿ. ಈ ಕಾರ್ಯಾಗಾರದ ಅಂತ್ಯಕ್ಕೆ, ನೀವು ಈ संदರ್ಭದಲ್ಲಿ ವಿವಿಧ ಸಾಧನಗಳು ಮತ್ತು ಕಲ್ಪನೆಗಳ ಒಂದು ಅನುಭವಾತ್ಮಕ ತಿಳಿವಳಿಕೆಯನ್ನು ಹೊಂದಿರಬೇಕು.

| ಪರಿಕಲ್ಪನೆ  | ಗುರಿ |
|:---|:---|
| **Azure Developer CLI** | ಸಾಧನ ಕಮಾಂಡ್‌ಗಳು ಮತ್ತು ಕಾರ್ಯವಿಧಾನಗಳನ್ನು ತಿಳಿದುಕೊಳ್ಳಿ |
| **AZD ಟೆಂಪ್ಲೇಟುಗಳು**| ಪ್ರಾಜೆಕ್ಟ್ ರಚನೆ ಮತ್ತು ಸಂರಚನೆಯನ್ನು ತಿಳಿದುಕೊಳ್ಳಿ |
| **Azure AI ಏಜೆಂಟ್**| Microsoft Foundry ಪ್ರಾಜೆಕ್ಟ್ ಅನ್ನು ಪೂರೈಸಿ ಮತ್ತು ನಿಯೋಜಿಸಿ  |
| **Azure AI ಶೋಧನೆ**| ಏಜೆಂಟ್‌ಗಳೊಂದಿಗೆ ಸಂಧರ್ಭ ಇಂಜಿನಿಯರಿಂಗ್ ಚಾಲನೆ ಮಾಡಿ |
| **ನಿಗಾ**| ಟ್ರೇಸಿಂಗ್, ನಿಗಾ ಮತ್ತು ಮೌಲ್ಯಾಂಕನಗಳನ್ನು ಅನ್ವೇಷಿಸಿ |
| **ರೆಡ್ ಟೀಮಿಂಗ್**| ವಿರೋಧಾತ್ಮಕ ಪರೀಕ್ಷೆ ಮತ್ತು ನಿವಾರಣಗಳನ್ನು ಅನ್ವೇಷಿಸಿ |
| | |
