# AI ಡೆವಲಪರ್‌ಗಳಿಗಾಗಿ AZD

!!! info "ಈ ಕಾರ್ಯಾಗಾರದ ಪ್ರೇಕ್ಷಕರು"
   
    ಈ ಕಾರ್ಯಾಗಾರವು Azure AI Foundry ಮೇಲೆ ಅತ್ಯುತ್ತಮ ಸಾಧನಗಳು ಮತ್ತು ಕಾರ್ಯಪ್ರವಾಹಗಳನ್ನು ಬಳಸಿಕೊಂಡು AI Agentic ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ನಿರ್ಮಿಸಲು ಆಸಕ್ತರಾಗಿರುವ ಯಾರಿಗಾದರೂ ಉದ್ದೇಶಿಸಲಾಗಿದೆ. ಈ ಕಾರ್ಯಾಗಾರವು Azure, AI ಪರಿಕಲ್ಪನೆಗಳು ಮತ್ತು Python ಕೋಡ್ ಬಗ್ಗೆ ಕೆಲವು ಪರಿಚಯವಿರುವ ಪ್ರಾರಂಭಿಕರು ಮತ್ತು ಅನುಭವೀ ಡೆವಲಪರ್‌ಗಳಿಗೆ ಸೂಕ್ತವಾಗಿದೆ. ಪ್ರಾರಂಭಿಸುವ ಮೊದಲು ಪೂರ್ವಾಪೇಕ್ಷಿತಗಳನ್ನು ಪರಿಶೀಲಿಸಿ.

    **ಅವಧಿ:** 3-4 ಗಂಟೆಗಳು  
    **ಮಟ್ಟ:** ಪ್ರಾರಂಭಿಕರಿಂದ ಮಧ್ಯಮ ಮಟ್ಟ  

---

## ಪೂರ್ವಾಪೇಕ್ಷಿತಗಳು

- ವೈಯಕ್ತಿಕ GitHub ಖಾತೆ → [ಇದನ್ನು ಉಚಿತವಾಗಿ ರಚಿಸಿ](https://github.com/signup)
- ಸಕ್ರಿಯ Azure ಖಾತೆ → [ಇದನ್ನು ಉಚಿತವಾಗಿ ರಚಿಸಿ](https://aka.ms/free)
- AI ಪರಿಕಲ್ಪನೆಗಳು, Python ಡೆವಲಪ್ಮೆಂಟ್, ಕಮಾಂಡ್-ಲೈನ್ ಸಾಧನಗಳ ಪರಿಚಯ
- VS Code ಮತ್ತು GitHub ಸಾಧನಗಳು ಮತ್ತು ಕಾರ್ಯಪ್ರವಾಹಗಳ ಅನುಭವ.

## ಕಲಿಕೆಯ ಉದ್ದೇಶಗಳು

ಈ ಕಾರ್ಯಾಗಾರದ ಅಂತ್ಯಕ್ಕೆ, ನೀವು ಈ ಕೆಳಗಿನವುಗಳನ್ನು ತಿಳಿದುಕೊಳ್ಳುತ್ತೀರಿ:

- [X] **Azure Developer CLI ಬಳಸಿ** → ನಿಮ್ಮ Azure ಸಂಪತ್ತುಗಳನ್ನು provision, deploy ಮತ್ತು ನಿರ್ವಹಿಸಲು.
- [X] **AI App Templates ಬಳಸಿ** → Azure AI Foundry ಮೇಲೆ AI Agent ಪ್ರಾಜೆಕ್ಟ್‌ಗಳನ್ನು ಪ್ರಾರಂಭಿಸಲು.
- [X] **ಟೆಂಪ್ಲೇಟುಗಳನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಿ** → Red Teaming, Evaluations, Tracing & Monitoring ಮುಂತಾದ ವೈಶಿಷ್ಟ್ಯಗಳಿಗಾಗಿ.
- [X] **AZD ಟೆಂಪ್ಲೇಟುಗಳು ಮತ್ತು ಕಾರ್ಯಪ್ರವಾಹಗಳನ್ನು ಡಿಕನ್ಸ್ಟ್ರಕ್ಟ್ ಮತ್ತು ಎಕ್ಸ್‌ಪ್ಲೋರ್ ಮಾಡಿ** → GitHub Copilot ಬಳಸಿ.
- [X] **AI Agent ಸ್ಟಾರ್ಟರ್ ಟೆಂಪ್ಲೇಟು ಕಸ್ಟಮೈಸ್ ಮಾಡಿ ಮತ್ತು ಪುನಃ ಡಿಪ್ಲಾಯ್ ಮಾಡಿ** → ನಿಮ್ಮ ಕಸ್ಟಮ್ ಅಗತ್ಯಗಳನ್ನು ಹೊಂದಿಸಿ.

## ಕಾರ್ಯಾಗಾರದ ರೂಪರೇಖೆ

ಕಾರ್ಯಾಗಾರವು ಪ್ರತಿ ಹ್ಯಾಂಡ್ಸ್-ಆನ್ ಲ್ಯಾಬ್ ವ್ಯಾಯಾಮದಿಂದ ankhor ಮಾಡಲಾದ ಮಾಸುಲ್‌ಗಳಾಗಿ ರಚಿಸಲಾಗಿದೆ:

1. **ಪರಿಚಯ** - AZD ಟೆಂಪ್ಲೇಟುಗಳು ಏನೆಂದು ಮತ್ತು ಅವು ಹೇಗೆ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತವೆ ಎಂಬುದರ ಬಗ್ಗೆ ತಿಳಿದುಕೊಳ್ಳಿ
1. **ಆಯ್ಕೆ** - ನಿಮ್ಮ ಮುಂದಿನ AI ಪ್ರಾಜೆಕ್ಟ್‌ಗಾಗಿ ಸರಿಯಾದ "ಸ್ಟಾರ್ಟರ್" ಟೆಂಪ್ಲೇಟು ಕಂಡುಹಿಡಿಯಿರಿ
1. **ಮಾನ್ಯತೆ** - ಟೆಂಪ್ಲೇಟು ಡಿಪ್ಲಾಯ್ ಮಾಡಿ ಮತ್ತು ಅದು ನಿರೀಕ್ಷಿತ ರೀತಿಯಲ್ಲಿ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತಿದೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ
1. **ಡಿಕನ್ಸ್ಟ್ರಕ್ಷನ್** - ಟೆಂಪ್ಲೇಟುವನ್ನು ಅದರ ಘಟಕಗಳು ಮತ್ತು ಕಾನ್ಫಿಗರೇಶನ್‌ಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು ಬೇರ್ಪಡಿಸಿ
1. **ಕಾನ್ಫಿಗರೇಶನ್** - ವಿವಿಧ Azure AI Foundry ವೈಶಿಷ್ಟ್ಯಗಳನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಲು ಮತ್ತು ಬಳಸಲು ಕಲಿಯಿರಿ
1. **ಕಸ್ಟಮೈಜೇಶನ್** - ನಿಮ್ಮ ಡೇಟಾ ಮತ್ತು ಅಗತ್ಯಗಳನ್ನು ಗಮನದಲ್ಲಿಟ್ಟುಕೊಂಡು ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಪುನಃ ರಚಿಸಿ
1. **ಟಿಯರ್‌ಡೌನ್** - ಮೂಲಸೌಕರ್ಯವನ್ನು ಡಿಪ್ರೊವಿಷನ್ ಮಾಡುವುದು ಮತ್ತು ಸಾಫ್ಟ್-ಡಿಲೀಟ್ ಮಾಡಿದ ಸಂಪತ್ತುಗಳನ್ನು ಪುನಃ ಪಡೆಯುವುದು ಕಲಿಯಿರಿ

## ಪ್ರತಿಕ್ರಿಯೆ ಮತ್ತು ಸಮಸ್ಯೆಗಳು

ನಾವು ಪ್ರಾರಂಭಿಕರಿಗೆ ಉತ್ತಮ ಕಲಿಕೆಯ ಅನುಭವವನ್ನು ಒದಗಿಸಲು ಪಾಠಕ್ರಮವನ್ನು ಸಕ್ರಿಯವಾಗಿ ನವೀಕರಿಸುತ್ತೇವೆ. ನಿಮ್ಮ ಪ್ರತಿಕ್ರಿಯೆಯನ್ನು ಸ್ವೀಕರಿಸಲಾಗುತ್ತದೆ.

1. ಈ ರೆಪೊದಲ್ಲಿ ಸಮಸ್ಯೆಯನ್ನು ಪೋಸ್ಟ್ ಮಾಡಿ - `Workshop` ಟ್ಯಾಗ್ ಅನ್ನು ಅನುಕೂಲಕ್ಕಾಗಿ ಬಳಸಿ.
1. Azure AI Foundry Discord ಗೆ ಸೇರಿ - ಸಹಪಾಠಿಗಳೊಂದಿಗೆ ಸಂಪರ್ಕ ಸಾಧಿಸಿ!
