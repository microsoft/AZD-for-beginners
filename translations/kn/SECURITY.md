## ಭದ್ರತೆ

ಮೈಕ್ರೋಸಾಫ್ಟ್ ನಮ್ಮ ಸಾಫ್ಟ್‌ವೇರ್ ಉತ್ಪನ್ನಗಳು ಮತ್ತು ಸೇವೆಗಳ ಭದ್ರತೆಯನ್ನು ಗಂಭೀರವಾಗಿ ಪರಿಗಣಿಸುತ್ತದೆ, ಇದರಲ್ಲಿ ನಮ್ಮ GitHub ಸಂಸ್ಥೆಗಳಲ್ಲಿನ ಎಲ್ಲಾ ಮೂಲ ಕೋಡ್ ರೆಪೊಸಿಟರಿಗಳು ಸೇರಿವೆ.

**ದಯವಿಟ್ಟು ಸಾರ್ವಜನಿಕ GitHub ಸಮಸ್ಯೆಗಳ ಮೂಲಕ ಭದ್ರತಾ ದೋಷಗಳನ್ನು ವರದಿ ಮಾಡಬೇಡಿ.**

ಭದ್ರತಾ ವರದಿ ಮಾಹಿತಿಯು, ಸ್ಥಳಗಳು, ಸಂಪರ್ಕ ಮಾಹಿತಿಯು ಮತ್ತು ನೀತಿಗಳು,
ಮೈಕ್ರೋಸಾಫ್ಟ್ ರೆಪೊಸಿಟರಿಗಳಿಗಾಗಿ ಇತ್ತೀಚಿನ ಮಾರ್ಗದರ್ಶನವನ್ನು [https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md) ನಲ್ಲಿ ಪರಿಶೀಲಿಸಿ.
