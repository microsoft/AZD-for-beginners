## ಪರಿಚಯ

ಈ ಸಮಗ್ರ ಅಧ್ಯಯನ ಮಾರ್ಗದರ್ಶಿ Azure Developer CLI (azd) ಅನ್ನು ಸಂಪೂರ್ಣವಾಗಿ ಕಲಿಯಲು ರಚಿತ ಉದ್ದೇಶಗಳು, ಪ್ರಮುಖ ತತ್ವಗಳು, ಅಭ್ಯಾಸ ವ್ಯಾಯಾಮಗಳು ಮತ್ತು ಮೌಲ್ಯಮಾಪನ ಸಾಮಗ್ರಿಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ. ಈ ಮಾರ್ಗದರ್ಶಿಯನ್ನು ನಿಮ್ಮ ಪ್ರಗತಿಯನ್ನು ಟ್ರ್ಯಾಕ್ ಮಾಡಲು ಮತ್ತು ಎಲ್ಲಾ ಅಗತ್ಯ ವಿಷಯಗಳನ್ನು ಕವರ್ ಮಾಡಿರುವುದನ್ನು ಖಚಿತಪಡಿಸಲು ಬಳಸಬಹುದು.

ಮೈಕ್ರೋಸರ್ವಿಸ್‌ಗಳ ಆಧಾರಿತ ಇ-ಕಾಮರ್ಸ್ ಪ್ಲಾಟ್‌ಫಾರ್ಮ್ ಅನ್ನು ವಿನ್ಯಾಸಗೊಳಿಸಿ ಮತ್ತು ಅನುಷ್ಠಾನಗೊಳಿಸಿ:
- ಹಲವಾರು API ಸೇವೆಗಳು (ಕ್ಯಾಟಲಾಗ್, ಆರ್ಡರ್‌ಗಳು, ಪೇಮೆಂಟ್‌ಗಳು, ಬಳಕೆದಾರರು)
- Service Bus ನೊಂದಿಗೆ ಸಂದೇಶ ಕ್ಯೂ ಏಕೀಕರಣ
- ಕಾರ್ಯಕ್ಷಮತೆಯ ಆಪ್ಟಿಮೈಸೇಶನ್‌ಗಾಗಿ Redis ಕ್ಯಾಶೆ
- ಸಮಗ್ರ ಲಾಗಿಂಗ್ ಮತ್ತು ನಿಗಾವಹಿಸುವುದು

**ಉದಾಹರಣಾ ಮಾದರಿ**: [Microservices Architecture](../../../examples/container-app/microservices) ಅನ್ನು ಉತ್ಪಾದನಾ-ತಯಾರಿನ ಟೆಂಪ್ಲೇಟು ಮತ್ತು ನಿಯೋಜನೆ ಮಾರ್ಗದರ್ಶಿಗಾಗಿ ನೋಡಿ

**ವಿತರಣಾ ವಸ್ತುಗಳು**:
- ಸಂಪೂರ್ಣ ಮೈಕ್ರೋಸರ್ವಿಸ್‌ಗಳ ಆರ್ಕಿಟೆಕ್ಚರ್
- ಇಂಟರ್-ಸರ್ವಿಸ್ ಸಂವಹನ ಮಾದರಿಗಳು
- ಕಾರ್ಯಕ್ಷಮತೆ ಪರೀಕ್ಷೆ ಮತ್ತು ಆಪ್ಟಿಮೈಸೇಶನ್
- ಉತ್ಪಾದನಾ-ತಯಾರಿನ ಭದ್ರತಾ ಅನುಷ್ಠಾನ

## ಮೌಲ್ಯಮಾಪನ ಮತ್ತು ಪ್ರಮಾಣಪತ್ರ

### ಜ್ಞಾನ ಪರಿಶೀಲನೆಗಳು

ಪ್ರತಿ ಮಾಡ್ಯೂಲ್ ನಂತರ ಈ ಮೌಲ್ಯಮಾಪನಗಳನ್ನು ಪೂರ್ಣಗೊಳಿಸಿ:

**ಮಾಡ್ಯೂಲ್ 1 ಮೌಲ್ಯಮಾಪನ**: ಮೂಲ ತತ್ವಗಳು ಮತ್ತು ಸ್ಥಾಪನೆ
- ಮೂಲ ತತ್ವಗಳ ಮೇಲೆ ಬಹು ಆಯ್ಕೆ ಪ್ರಶ್ನೆಗಳು
- ಪ್ರಾಯೋಗಿಕ ಸ್ಥಾಪನೆ ಮತ್ತು ಸಂರಚನಾ ಕಾರ್ಯಗಳು
- ಸರಳ ನಿಯೋಜನೆ ವ್ಯಾಯಾಮ

**ಮಾಡ್ಯೂಲ್ 2 ಮೌಲ್ಯಮಾಪನ**: ಸಂರಚನೆ ಮತ್ತು ಪರಿಸರಗಳು
- ಪರಿಸರ ನಿರ್ವಹಣಾ ದೃಶ್ಯಗಳು
- ಸಂರಚನಾ ತೊಂದರೆ ಪರಿಹಾರ ವ್ಯಾಯಾಮಗಳು
- ಭದ್ರತಾ ಸಂರಚನೆ ಅನುಷ್ಠಾನ

**ಮಾಡ್ಯೂಲ್ 3 ಮೌಲ್ಯಮಾಪನ**: ನಿಯೋಜನೆ ಮತ್ತು ಪ್ರೊವಿಷನಿಂಗ್
- ಮೂಲಸೌಕರ್ಯ ವಿನ್ಯಾಸ ಸವಾಲುಗಳು
- ಬಹು-ಸೇವಾ ನಿಯೋಜನೆ ದೃಶ್ಯಗಳು
- ಕಾರ್ಯಕ್ಷಮತೆ ಆಪ್ಟಿಮೈಸೇಶನ್ ವ್ಯಾಯಾಮಗಳು

**ಮಾಡ್ಯೂಲ್ 4 ಮೌಲ್ಯಮಾಪನ**: ನಿಯೋಜನೆಗೆ ಮುನ್ನದ ಮಾನ್ಯತೆ
- ಸಾಮರ್ಥ್ಯ ಯೋಜನೆ ಪ್ರಕರಣ ಅಧ್ಯಯನಗಳು
- ವೆಚ್ಚ ಆಪ್ಟಿಮೈಸೇಶನ್ ದೃಶ್ಯಗಳು
- ಮಾನ್ಯತೆ ಪೈಪ್ಲೈನ್ ಅನುಷ್ಠಾನ

**ಮಾಡ್ಯೂಲ್ 5 ಮೌಲ್ಯಮಾಪನ**: ತೊಂದರೆ ಪರಿಹಾರ ಮತ್ತು ಡಿಬಗ್ಗಿಂಗ್
- ಸಮಸ್ಯೆ ನಿರ್ಣಯ ವ್ಯಾಯಾಮಗಳು
- ನಿಗಾವಹಿಸುವುದು ಅನುಷ್ಠಾನ ಕಾರ್ಯಗಳು
- ಘಟನೆ ಪ್ರತಿಕ್ರಿಯೆ ಅನುಕರಣೆಗಳು

**ಮಾಡ್ಯೂಲ್ 6 ಮೌಲ್ಯಮಾಪನ**: ಉನ್ನತ ವಿಷಯಗಳು
- CI/CD ಪೈಪ್ಲೈನ್ ವಿನ್ಯಾಸ
- ಕಸ್ಟಮ್ ಟೆಂಪ್ಲೇಟು ಅಭಿವೃದ್ಧಿ
- ಎಂಟರ್‌ಪ್ರೈಸ್ ಆರ್ಕಿಟೆಕ್ಚರ್ ದೃಶ್ಯಗಳು

### ಅಂತಿಮ ಕ್ಯಾಪ್‌ಸ್ಟೋನ್ ಯೋಜನೆ

ಎಲ್ಲಾ ತತ್ವಗಳ ನೈಪುಣ್ಯವನ್ನು ತೋರಿಸುವ ಸಂಪೂರ್ಣ ಪರಿಹಾರವನ್ನು ವಿನ್ಯಾಸಗೊಳಿಸಿ ಮತ್ತು ಅನುಷ್ಠಾನಗೊಳಿಸಿ:

**ಅವಶ್ಯಕತೆಗಳು**:
- ಬಹು-ತಟ್ಟೆ ಅಪ್ಲಿಕೇಶನ್ ಆರ್ಕಿಟೆಕ್ಚರ್
- ಹಲವಾರು ನಿಯೋಜನೆ ಪರಿಸರಗಳು
- ಸಮಗ್ರ ನಿಗಾವಹಿಸುವುದು ಮತ್ತು ಎಚ್ಚರಿಕೆ
- ಭದ್ರತೆ ಮತ್ತು ಅನುಕೂಲತೆ ಅನುಷ್ಠಾನ
- ವೆಚ್ಚ ಆಪ್ಟಿಮೈಸೇಶನ್ ಮತ್ತು ಕಾರ್ಯಕ್ಷಮತೆ ಟ್ಯೂನಿಂಗ್
- ಸಂಪೂರ್ಣ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಮತ್ತು ರನ್‌ಬುಕ್‌ಗಳು

**ಮೌಲ್ಯಮಾಪನ ಮಾನದಂಡಗಳು**:
- ತಾಂತ್ರಿಕ ಅನುಷ್ಠಾನ ಗುಣಮಟ್ಟ
- ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಸಂಪೂರ್ಣತೆ
- ಭದ್ರತೆ ಮತ್ತು ಉತ್ತಮ ಅಭ್ಯಾಸಗಳ ಅನುಸರಣೆ
- ಕಾರ್ಯಕ್ಷಮತೆ ಮತ್ತು ವೆಚ್ಚ ಆಪ್ಟಿಮೈಸೇಶನ್
- ತೊಂದರೆ ಪರಿಹಾರ ಮತ್ತು ನಿಗಾವಹಿಸುವುದು ಪರಿಣಾಮಕಾರಿತ್ವ

- **ಸಮಸ್ಯೆ ಪರಿಹಾರ ಸಾಮರ್ಥ್ಯಗಳು**: ಸ್ವತಂತ್ರ ತೊಂದರೆ ಪರಿಹಾರ ಮತ್ತು ಆಪ್ಟಿಮೈಜೇಶನ್
- **ಸಮುದಾಯ ಮಾನ್ಯತೆ**: Azure ಡೆವಲಪರ್ ಸಮುದಾಯದಲ್ಲಿ ಸಕ್ರಿಯ ಭಾಗವಹಿಸುವಿಕೆ
- **ಕೆರಿಯರ್ ಪ್ರಗತಿ**: ಕ್ಲೌಡ್ ಮತ್ತು AI ಪಾತ್ರಗಳಿಗೆ ನೇರವಾಗಿ ಅನ್ವಯಿಸಬಹುದಾದ ಕೌಶಲ್ಯಗಳು

#### ಯಶಸ್ಸಿನ ಅಳತೆಗಳು
- **ನಿಯೋಜನೆ ಯಶಸ್ಸಿನ ಪ್ರಮಾಣ**: >95% ಯಶಸ್ವಿ ನಿಯೋಜನೆಗಳು
- **ತೊಂದರೆ ಪರಿಹಾರ ಸಮಯ**: ಸಾಮಾನ್ಯ ಸಮಸ್ಯೆಗಳಿಗೆ <30 ನಿಮಿಷ
- **ಪ್ರದರ್ಶನ ಆಪ್ಟಿಮೈಜೇಶನ್**: ವೆಚ್ಚ ಮತ್ತು ಕಾರ್ಯಕ್ಷಮತೆಯಲ್ಲಿ ತೋರಿಸಬಹುದಾದ ಸುಧಾರಣೆಗಳು
- **ಭದ್ರತಾ соответствತೆ**: ಎಲ್ಲಾ ನಿಯೋಜನೆಗಳು ಉದ್ಯಮ ಭದ್ರತಾ ಮಾನದಂಡಗಳನ್ನು ಪೂರೈಸುತ್ತವೆ
- **ಜ್ಞಾನ ಹಸ್ತಾಂತರ**: ಇತರ ಡೆವಲಪರ್‌ಗಳಿಗೆ ಮಾರ್ಗದರ್ಶನ ನೀಡುವ ಸಾಮರ್ಥ್ಯ

### ನಿರಂತರ ಕಲಿಕೆ ಮತ್ತು ಸಮುದಾಯದ ತೊಡಗಿಸಿಕೊಳ್ಳುವಿಕೆ

#### ಪ್ರಸ್ತುತವಾಗಿರಿ
- **Azure ನವೀಕರಣಗಳು**: Azure Developer CLI ಬಿಡುಗಡೆ ಟಿಪ್ಪಣಿಗಳನ್ನು ಅನುಸರಿಸಿ
- **ಸಮುದಾಯ ಕಾರ್ಯಕ್ರಮಗಳು**: Azure ಮತ್ತು AI ಡೆವಲಪರ್ ಕಾರ್ಯಕ್ರಮಗಳಲ್ಲಿ ಭಾಗವಹಿಸಿ
- **ದಸ್ತಾವೇಜುಗಳು**: ಸಮುದಾಯದ ದಸ್ತಾವೇಜುಗಳು ಮತ್ತು ಉದಾಹರಣೆಗಳಿಗೆ ಕೊಡುಗೆ ನೀಡಿ
- **ಪ್ರತಿಕ್ರಿಯಾ ಚಕ್ರ**: ಕೋರ್ಸ್ ವಿಷಯ ಮತ್ತು Azure ಸೇವೆಗಳ ಮೇಲೆ ಪ್ರತಿಕ್ರಿಯೆ ನೀಡಿ

#### ವೃತ್ತಿ ಅಭಿವೃದ್ಧಿ
- **ವೃತ್ತಿಪರ ಜಾಲತಾಣ**: Azure ಮತ್ತು AI ತಜ್ಞರೊಂದಿಗೆ ಸಂಪರ್ಕ ಸಾಧಿಸಿ
- **ಭಾಷಣದ ಅವಕಾಶಗಳು**: ಸಮ್ಮೇಳನಗಳು ಅಥವಾ ಸಭೆಗಳಲ್ಲಿ ಕಲಿಕೆಗಳನ್ನು ಪ್ರಸ್ತುತಪಡಿಸಿ
- **ಓಪನ್ ಸೋರ್ಸ್ ಕೊಡುಗೆ**: AZD ಟೆಂಪ್ಲೇಟುಗಳು ಮತ್ತು ಸಾಧನಗಳಿಗೆ ಕೊಡುಗೆ ನೀಡಿ
- **ಮಾರ್ಗದರ್ಶನ**: ಇತರ ಡೆವಲಪರ್‌ಗಳಿಗೆ ಅವರ AZD ಕಲಿಕೆಯ ಪ್ರಯಾಣದಲ್ಲಿ ಮಾರ್ಗದರ್ಶನ ನೀಡಿ
