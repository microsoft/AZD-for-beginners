## ಪರಿಚಯ

ಈ ಸಮಗ್ರ ಶಬ್ದಕೋಶವು Azure Developer CLI ಮತ್ತು Azure ಕ್ಲೌಡ್ ಅಭಿವೃದ್ಧಿಯಲ್ಲಿ ಬಳಸಲಾಗುವ ಪದಗಳು, ಕಲ್ಪನೆಗಳು ಮತ್ತು ಸಂಕೇತಪದಗಳ ವ್ಯಾಖ್ಯಾನಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ. ತಾಂತ್ರಿಕ ದಾಖಲೆಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು, ಸಮಸ್ಯೆಗಳನ್ನು ಪರಿಹರಿಸಲು ಮತ್ತು azd ಯೋಜನೆಗಳು ಮತ್ತು Azure ಸೇವೆಗಳ ಕುರಿತು ಪರಿಣಾಮಕಾರಿಯಾಗಿ ಸಂವಹನದಷ್ಟಕ್ಕೆ ಇದು ಅಗತ್ಯ ಉಲ್ಲೇಖವಾಗಿದೆ.

## ಕಲಿಕೆಯ ಗುರಿಗಳು

ಈ ಶಬ್ದಕೋಶವನ್ನು ಬಳಸಿ, ನೀವು:
- Azure Developer CLI ನ ಅಗತ್ಯ ಪದಾವಳಿ ಮತ್ತು ಕಲ್ಪನೆಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವುದು
- Azure ಕ್ಲೌಡ್ ಅಭಿವೃದ್ಧಿಯ ಪದಕೋಶ ಮತ್ತು ತಾಂತ್ರಿಕ ಪದಗಳಲ್ಲಿ ನಿಪುಣಗೊಳ್ಳುವುದು
- Infrastructure as Code ಮತ್ತು ನಿಯೋಜನೆ ಸಂಬಂಧಿ ಪದಾವಳಿಯನ್ನು ಪರಿಣಾಮಕಾರಿಯಾಗಿ ಉಲ್ಲೇಖಿಸುವುದು
- Azure ಸೇವೆಗಳ ಹೆಸರುಗಳು, ಸಂಕೇತಪದಗಳು ಮತ್ತು ಅವುಗಳ ಉದ್ದೇಶಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವುದು
- ಸಮಸ್ಯೆ ಪರಿಹಾರ ಮತ್ತು ಡೀಬಗಿಂಗ್ ಸಂಬಂಧಿ ಪದಗಳ ವ್ಯಾಖ್ಯಾನಗಳನ್ನು ಪ್ರವೇಶಿಸುವುದು
- ಉನ್ನತ Azure ವಾಸ್ತುಶಾಸ್ತ್ರ ಮತ್ತು ಅಭಿವೃದ್ಧಿ ಕಲ್ಪನೆಗಳನ್ನು ಕಲಿಯುವುದು

## ಕಲಿಕೆಯ ಫಲಿತಾಂಶಗಳು

ಈ ಶಬ್ದಕೋಶವನ್ನು ನಿಯಮಿತವಾಗಿ ಉಲ್ಲೇಖಿಸುವ ಮೂಲಕ, ನೀವು ಸಾಧಿಸಬಹುದಾದವು:
- ಸರಿಯಾದ Azure Developer CLI ಪದಾವಳಿಯನ್ನು ಬಳಸಿ ಪರಿಣಾಮಕಾರಿಯಾಗಿ ಸಂವಹನ ಮಾಡುವುದು
- ತಾಂತ್ರಿಕ ದಾಖಲೆಗಳು ಮತ್ತು ದೋಷ ಸಂದೇಶಗಳನ್ನು ಹೆಚ್ಚು ಸ್ಪಷ್ಟವಾಗಿ ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವುದು
- ವಿಶ್ವಾಸದಿಂದ Azure ಸೇವೆಗಳು ಮತ್ತು ಕಲ್ಪನೆಗಳನ್ನು ನಾವಿಗೇಟ್ ಮಾಡುವುದು
- ಸೂಕ್ತ ತಾಂತ್ರಿಕ ಶಬ್ದಕೋಶವನ್ನು ಬಳಸಿ ಸಮಸ್ಯೆಗಳನ್ನು ಟ್ರಬಲ್‌ಶೂಟ್ ಮಾಡುವುದು
- ತಂಡದ ಚರ್ಚೆಗಳಿಗೆ ನಿಖರ ತಾಂತ್ರಿಕ ಭಾಷೆಯೊಂದಿಗೆ ಕೊಡುಗೆ ನೀಡುವುದು
- ಕ್ರಮಬದ್ಧವಾಗಿ ನಿಮ್ಮ Azure ಕ್ಲೌಡ್ ಅಭಿವೃದ್ಧಿ ಜ್ಞಾನವನ್ನು ವಿಸ್ತರಿಸುವುದು

## A

**ARM Template**  
Azure Resource Manager ಟೆಂಪ್ಲೇಟ್. JSON-ಆಧಾರಿತ Infrastructure as Code ರೂಪ, ಇದನ್ನು Azure ಸಂಪನ್ಮೂಲಗಳನ್ನು ಘೋಷಣಾತ್ಮಕವಾಗಿ ನಿರ್ದಿಷ್ಟಗೊಳಿಸಲು ಮತ್ತು ನಿಯೋಜಿಸಲು ಬಳಸಲಾಗುತ್ತದೆ.

**App Service**  
Infrastructure ನಿರ್ವಹಣೆ ಮಾಡಿ ಇಲ್ಲದೆ ವೆಬ್ ಅಪ್ಲಿಕೇಶನ್‌ಗಳು, REST APIs ಮತ್ತು ಮೋಬೈಲ್ ಬ್ಯಾಕೆಂಡ್‌ಗಳನ್ನು ಹೋಸ್ಟ್ ಮಾಡಲು Azure ನ ಪ್ಲ್ಯಾಟ್‌ಫಾರ್ಮ್-ಏಸ್-ಎ-ಸರ್ವಿಸ್ (PaaS) ಸೇವೆ.

**Application Insights**  
ಅಪ್‌ಲಿಕೇಶನ್ ಕಾರ್ಯಕ್ಷಮತೆ, ಲಭ್ಯತೆ ಮತ್ತು ಬಳಕೆ ಕುರಿತು ಆಳವಾದ ಒಳನೋಟಗಳನ್ನು ಒದಗಿಸುವ Azure ನ ಅಪ್ಲಿಕೇಶನ್ ಪ್ರದರ್ಶನ ಮಾನಿಟರಿಂಗ್ (APM) ಸೇವೆ.

**Azure CLI**  
Azure ಸಂಪನ್ಮೂಲಗಳನ್ನು ನಿರ್ವಹಿಸಲು ಕಮಾಂಡ್-ಲೈನ್ ಇಂಟರ್ಫೇಸ್. azd ಮೂಲಕ ಸತ್ಯಾಪನೆ ಮತ್ತು ಕೆಲವು ಕಾರ್ಯಾಚರಣೆಗಳಿಗೆ ಬಳಸಲಾಗುತ್ತದೆ.

**Azure Developer CLI (azd)**  
ಟೆಂಪ್ಲೇಟುಗಳು ಮತ್ತು Infrastructure as Code ಬಳಸಿ ಅನ್ವಯಿಕೆಗಳನ್ನು Azure ಗೆ ನಿರ್ಮಿಸುವುದು ಮತ್ತು ನಿಯೋಜಿಸುವ ಪ್ರಕ್ರಿಯೆಯನ್ನು ವೇಗಗೊಳಿಸುವ ಡೆವಲಪರ್-ಕೇಂದ್ರಿತ ಕಮಾಂಡ್-ಲೈನ್ ಸಾಧನ.

**azure.yaml**  
ಸೇವೆಗಳು, ಮೂಲಸೌಕರ್ಯ ಮತ್ತು ನಿಯೋಜನೆ ಹೂಕ್‌ಗಳನ್ನು ವ್ಯಾಖ್ಯಾನಿಸುವ azd ಯೋಜನೆಕ್ಕಾಗಿಯೇ ಮುಖ್ಯ ಕಾನ್ಫಿಗರೇಶನ್ ಫೈಲ್.

**Azure Resource Manager (ARM)**  
ಸಂಪನ್ಮೂಲಗಳನ್ನು ರಚಿಸುವುದು, ನವೀಕರಿಸುವುದು ಮತ್ತು ಅಳಿಸಲನ್ನು ನಿರ್ವಹಿಸಲು ನಿರ್ವಹಣಾ ಪದರವನ್ನು ಒದಗಿಸುವ Azure ನ ನಿಯೋಜನೆ ಮತ್ತು ನಿರ್ವಹಣಾ ಸೇವೆ.

## B

**Bicep**  
Azure ಸಂಪನ್ಮೂಲಗಳನ್ನು ನಿಯೋಜಿಸಲು Microsoft ಅಭಿವೃದ್ಧಿಪಡಿಸಿರುವ ಡೊಮೇನ್-ನಿರ್ದಿಷ್ಟ ಭಾಷೆ (DSL). ARM ಟೆಂಪ್ಲೇಟ್ಗಳಿಗಿಂತ ಸರಳ ಸಿಂಟ್ಯಾಕ್ಸ್ ಒದಗಿಸುತ್ತದೆ ಮತ್ತು ARM ಗೆ ಕಂಪೈಲ್ ಆಗುತ್ತದೆ.

**Build**  
ಸೋರ್ಸ್ ಕೋಡ್ ಕಾಂಪೈಲ್ ಮಾಡುವ, ನಿರ್ಭರತೆಗಳನ್ನು ಸ್ಥಾಪಿಸುವ ಮತ್ತು ಅಪ್ಲಿಕೇಶನ್‍ಗಳನ್ನು ನಿಯೋಜನೆಗೆ ಸಿದ್ಧಪಡಿಸುವ ಪ್ರಕ್ರಿಯೆ.

**Blue-Green Deployment**  
ಡೌನ್‌ಟೈಮ್ ಮತ್ತು ಅಪಾಯ ಕಡಿಮೆ ಮಾಡಲು ಎರಡೂ ಸಮಾನ ಉತ್ಪಾದನಾ ಪರಿಸರಗಳನ್ನು (ಬ್ಲೂ ಮತ್ತು ಗ್ರೀನ್) ಬಳಸುವ ನಿಯೋಜನಾ ತಂತ್ರ.

## C

**Container Apps**  
ಸಂಕೀರ್ಣ ಮೂಲಸೌಕರ್ಯ ನಿರ್ವಹಣೆ ಇಲ್ಲದೆ ಕಂಟೆನರ್ ಆಧಾರಿತ ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ಚಾಲನೆ ಮಾಡಲು ಸಾಧ್ಯವಾಗಿಸುವ Azure ನ ಸರ್ವರ್‍ರ್ಲೆಸ್ ಕಂಟೆನರ್ ಸೇವೆ.

**CI/CD**  
Contínuous Integration/Contínuous Deployment. ಕೋಡ್ ಬದಲಾವಣೆಗಳನ್ನು ಒಂದುಗೂಡಿಸುವ ಹಾಗೂ ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ನಿಯೋಜಿಸುವ ಸ್ವಯಂಚಾಲಿತ ಅಭ್ಯಾಸಗಳು.

**Cosmos DB**  
ಥ್ರುಪುಟ್, ವಿಳಂಬ, ಲಭ್ಯತೆ ಮತ್ತು ಸೌಮ್ಯತೆಯ ಪರಿಪೂರ್ಣ SLAಗಳನ್ನು ಒದಗಿಸುವ ಗ್ಲೋಬಲ್-ವಿತರಿತ, ಬಹು-ಮಾದರಿ ಡೇಟಾಬೇಸ್ ಸೇವೆ.

**Configuration**  
ಅಪ್ಲಿಕೇಶನ್ ನಡೆ ಮತ್ತು ನಿಯೋಜನೆ ಆಯ್‌ಶನ್ಗಳನ್ನು നിയന്ത്രಿಸುವ ಸೆಟ್ಟಿಂಗ್‌ಗಳು ಮತ್ತು ಪರಿಮಾಣಗಳು.

## D

**Deployment**  
ಗುರಿ ಮೂಲಸೌಕರ್ಯದಲ್ಲಿ ಅಪ್ಲಿಕೇಶನ್‍ಗಳನ್ನು ಮತ್ತು ಅವುಗಳ ನಿರ್ಭರತೆಗಳನ್ನು ಸ್ಥಾಪಿಸುವ ಮತ್ತು ಸಂರಚಿಸುವ ಪ್ರಕ್ರಿಯೆ.

**Docker**  
ಕಂಟೇನರೈಜೆಶನ್ ತಂತ್ರಜ್ಞಾನದೊಂದಿಗೆ ಅಪ್ಲಿಕೇಶನ್‍ಗಳನ್ನು ಅಭಿವೃದ್ಧಿಪಡಿಸುವುದು, ಸಾಗಿಸುವುದು ಮತ್ತು ಚಲಾಯಿಸುವುದಕ್ಕಾಗಿ的平台.

**Dockerfile**  
Docker ಕಂಟೈನರ್ ಇಮೇಜ್ ನಿರ್ಮಿಸಲು ಸೂಚನೆಗಳನ್ನು ಹೊಂದಿರುವ ಪಠ್ಯ ಫೈಲ್.

## E

**Environment**  
ನಿಮ್ಮ ಅಪ್ಲಿಕೇಶನ್‌ನ ನಿರ್ದಿಷ್ಟ ಉದಾಹರಣೆಯನ್ನು ಪ್ರತಿನಿಧಿಸುವ ನಿಯೋಜನೆ ಗುರಿ (ಉದಾಹರಣೆಗೆ, development, staging, production).

**Environment Variables**  
ಅಪ್ಲಿಕೇಶನ್‌ಗಳು ರನ್‌ಟೈಮ್‌ನಲ್ಲಿ ಪ್ರವೇಶಿಸಬಹುದಾದ ಕೀ-ಮೌಲ್ಯ ಜೋಡಿಗಳಾಗಿ ಸಂಗ್ರಹಿಸಲಾದ ಕಾನ್ಫಿಗರೇಶನ್ ಮೌಲ್ಯಗಳು.

**Endpoint**  
ಅಪ್ಲಿಕೇಶನ್ ಅಥವಾ ಸೇವೆಯನ್ನು ಪ್ರವೇಶಿಸಲು URL ಅಥವಾ ನೆಟ್‌ವರ್ಕ್ ವಿಳಾಸ.

## F

**Function App**  
ಮೂಲಸೌಕರ್ಯ ನಿರ್ವಹಣೆ ಇಲ್ಲದೆ ಇವೆಂಟ್-ಚಾಲಿತ ಕೋಡ್ ಚಾಲನೆ ಮಾಡಲು ಅನುಮತಿಸುವ Azure ನ ಸರ್ವರ್‍ರ್ಲೆಸ್ ಕಂಪ್ಯೂಟ್ ಸೇವೆ.

## G

**GitHub Actions**  
GitHub ರೆಪೊಗಳೊಂದಿಗೆ ಇಂಟಿಗ್ರೇಟ್ ಆಗಿರುವ CI/CD ವೇದಿಕೆ, ವರ್ಕ್‌ಫ್ಲೋಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತಗೊಳಿಸಲು.

**Git**  
ಸೋರ್ಸ್ ಕೋಡ್‌ನಲ್ಲಿ ಬದಲಾವಣೆಗಳನ್ನು ಟ್ರ್ಯಾಕ್ ಮಾಡಲು ಬಳಸುವ ವಿತರಿತ ವರ್ಸನ್ ಕಂಟ್ರೋಲ್ ಸಿಸ್ಟಮ್.

## H

**Hooks**  
ನಿಯೋಜನೆ ಜೀವಚಕ್ರದ ನಿರ್ದಿಷ್ಟ ಬಿಂದುಗಳಲ್ಲಿ (preprovision, postprovision, predeploy, postdeploy) ಚಲಿಸುವ ಕಸ್ಟಮ್ ಸ್ಕ್ರಿಪ್ಟ್‌ಗಳು ಅಥವಾ ಕಮಾಂಡ್‌ಗಳು.

**Host**  
ಅಪ್ಲಿಕೇಶನ್ ನಿಯೋಜಿಸಲಾಗುವ Azure ಸೇವೆಯ ಪ್ರಕಾರ (ಉದಾಹರಣೆಗೆ, appservice, containerapp, function).

## I

**Infrastructure as Code (IaC)**  
ಮ್ಯಾನುಯಲ್ ಪ್ರಕ್ರಿಯೆಗಳ ಬದಲು ಕೋಡ್ ಮೂಲಕ ಮೂಲಸೌಕರ್ಯವನ್ನು ನಿರ್ದಿಷ್ಟಗೊಳಿಸುವ ಮತ್ತು ನಿರ್ವಹಿಸುವ ಅಭ್ಯಾಸ.

**Init**  
ಸಾಮಾನ್ಯವಾಗಿ ಟೆಂಪ್ಲೇಟಿನಿಂದ ಹೊಸ azd ಯೋಜನೆಯನ್ನು ಪ್ರಾರಂಭಿಸುವ ಪ್ರಕ್ರಿಯೆ.

## J

**JSON**  
JavaScript Object Notation. ಕಾನ್ಫಿಗರೇಶನ್ ಫೈಲ್‌ಗಳು ಮತ್ತು API ಪ್ರತಿಕ್ರೀಯೆಗಳಿಗೆ ಸಾಮಾನ್ಯವಾಗಿ ಬಳಸುವ ದತ್ತಾಂಶ ವಿನಿಮಯ ಸ್ವರೂಪ.

**JWT**  
JSON Web Token. JSON ಆಬ್ಜೆಕ್ಟ್‌ ಆಗಿ ಪಕ್ಷಗಳ ನಡುವೆ ಸುರಕ್ಷಿತವಾಗಿ ಮಾಹಿತಿಯನ್ನು ಪ್ರಸಾರ ಮಾಡುವ ಮಾನಕ.

## K

**Key Vault**  
ರಹಸ್ಯಗಳು, ಕೀಗಳು ಮತ್ತು ಪ್ರಮಾಣಪತ್ರಗಳನ್ನು ಸುರಕ್ಷಿತವಾಗಿ ಸಂಗ್ರಹಿಸುವ ಮತ್ತು ನಿರ್ವಹಿಸುವ Azure ಸೇವೆ.

**Kusto Query Language (KQL)**  
Azure Monitor, Application Insights ಮತ್ತು ಇತರ Azure ಸೇವೆಗಳಲ್ಲಿ ದತ್ತಾಂಶ ವಿಶ್ಲೇಷಿಸಲು ಬಳಸುವ ಪ್ರಶ್ನೆ ಭಾಷೆ.

## L

**Load Balancer**  
ಬಹು ಸರ್ವರ್‌ಗಳು ಅಥವಾ ಇನ್ಸ್‌ಟ್ಯಾನ್ಸ್‌ಗಳ ನಡುವೆ ಆಗಮಿಸು ನೇಟ್ವರ್ಕ್ ಟ್ರಾಫಿಕ್ ಅನ್ನು ಹಂಚುವ ಸೇವೆ.

**Log Analytics**  
ಕ್ಲೌಡ್ ಮತ್ತು ಆನ್-ಪ್ರೆಮೈಸಸ್ ಪರಿಸರಗಳಿಂದ ಟೆಲಿಮೆಟ್ರಿ ಡೇಟಾ সংগ্রಹಿಸಿ ವಿಶ್ಲೇಷಿಸಲು ಮತ್ತು చర్య ಕೈಗೊಳ್ಳಲು Azure ಸೇವೆ.

## M

**Managed Identity**  
ಇತರ Azure ಸೇವೆಗಳಿಗೆ ಸತ್ಯಾಪನೆಗಾಗಿ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನಿರ್ವಹಿಸಲಾದ ಐಡೆಂಟಿಟಿಯನ್ನು Azure ಕುರುಹು ಹೊರಡಿಸುವ ವೈಶಿಷ್ಟ್ಯ.

**Microservices**  
ಅನ್ವಯಿಕೆಗಳನ್ನು ಸಣ್ಣ, ಸ್ವತಂತ್ರ ಸೇವೆಗಳ ಸಂಕಲನವಾಗಿ ನಿರ್ಮಿಸುವ ವಾಸ್ತುಶಿಲ್ಪ ದೃಷ್ಠಿಕೋನ.

**Monitor**  
ಅಪ್ಲಿಕೇಶನ್‍ಗಳು ಮತ್ತು ಮೂಲಸೌಕರ್ಯದ ಪೂರ್ತಿಯಾಗಿ ವೀಕ್ಷಣಾ ಪರಿಹಾರವನ್ನು ಒದಗಿಸುವ Azure ನ ಏಕೀಕೃತ ಮಾನಿಟರಿಂಗ್ ಪರಿಹಾರ.

## N

**Node.js**  
Chrome ನ V8 ಜಾವಾಸ್ಕ್ರಿಪ್ಟ್ ಎಂಜಿನ್‌ನಲ್ಲಿ ನಿರ್ಮಿಸಲಾದ ಜಾವಾಸ್ಕ್ರಿಪ್ಟ್ ರನ್‌ಟೈಮ್, ಸರ್ವರ್-ಸೈಡ್ ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ನಿರ್ಮಿಸಲು.

**npm**  
Node.js ಗಾಗಿ ಪ್ಯಾಕೇಜ್ ಮ್ಯಾನೇಜರ್, ಇದು ನಿರ್ಭರತೆಗಳು ಮತ್ತು ಪ್ಯಾಕೇಜ್‌ಗಳನ್ನು ನಿರ್ವಹಿಸುತ್ತದೆ.

## O

**Output**  
ವೆಲ್ಯೂಗಳು ಮೂಲಸೌಕರ್ಯ ನಿಯೋಜನೆಯಿಂದ ಹಿಂತಿರುಗುತ್ತವೆ ಮತ್ತು ಅವುಗಳನ್ನು ಅಪ್ಲಿಕೇಶನ್‌ಗಳು ಅಥವಾ ಇತರ ಸಂಪನ್ಮೂಲಗಳು ಬಳಸಬಹುದು.

## P

**Package**  
ಅಪ್ಲಿಕೇಶನ್ ಕೋಡ್ ಮತ್ತು ನಿರ್ಭರತೆಗಳನ್ನು ನಿಯೋಜನೆಗೆ ಸಿದ್ಧಪಡಿಸುವ ಪ್ರಕ್ರಿಯೆ.

**Parameters**  
ನಿಯೋಜನೆಯನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡಲು ಮೂಲಸೌಕರ್ಯ ಟೆಂಪ್ಲೇಟುಗಳಿಗೆ ಪಾಸುಮಾಡಲಾದ ಇನ್‌ಪುಟ್ ಮೌಲ್ಯಗಳು.

**PostgreSQL**  
ಓಪನ್-ಸೋರ್ಸ್ ರಿಲೇಷನಲ್ ಡೇಟಾಬೇಸ್ ಸಿಸ್ಟಮ್, Azure ನಲ್ಲಿ ನಿರ್ವಹಿತ ಸೇವೆಯಾಗಿ ಬೆಂಬಲಿಸಲಾಗಿದೆ.

**Provisioning**  
ಮೂಲಸೌಕರ್ಯ ಟೆಂಪ್ಲೇಟುಗಳಲ್ಲಿ ವ್ಯಾಖ್ಯಾನಿಸಲಾದ Azure ಸಂಪನ್ಮೂಲಗಳನ್ನು ರಚಿಸುವ ಮತ್ತು ಸಂರಚಿಸುವ ಪ್ರಕ್ರಿಯೆ.

## Q

**Quota**  
Azure ಸಬ್ಸ್ಕ್ರಿಪ್ಷನ್ ಅಥವಾ ರೀಜನಿನಲ್ಲಿ ರಚಿಸಬಹುದಾದ ಸಂಪನ್ಮೂಲಗಳ ಪ್ರಮಾಣದ ಮೇಲೆ ಇರುವ ಮಿತಿ.

## R

**Resource Group**  
ಒಂದೇ ಜೀವನಚಕ್ರ, ಅನುಮತಿಗಳು ಮತ್ತು ನೀತಿಗಳನ್ನುವಾಗಿಸುತ್ತುವ Azure ಸಂಪನ್ಮೂಲಗಳ ತರ್ಕಬದ್ಧ ಕಾಂಟೇನರ್.

**Resource Token**  
ನಿಯೋಜನೆಗಳ ನಡುವೆ ಸಂಪನ್ಮೂಲ ಹೆಸರುಗಳ ಅನನ್ಯತೆಯನ್ನು ಖಚಿತಪಡಿಸಲು azd ಮೂಲಕ ತಯಾರಿಸಲಾಗುವ ವಿಭಿನ್ನ ಸ್ಟ್ರಿಂಗ್.

**REST API**  
HTTP ವಿಧಾನಗಳನ್ನು ಬಳಸಿಕೊಂಡು ನೆಟ್ವರ್ಕ್ ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ವಿನ್ಯಾಸಗೊಳಿಸುವ ಆರ್ಕಿಟೆಕ್ಚರ್ ಶೈಲಿ.

**Rollback**  
ಅಪ್ಲಿಕೇಶನ್ ಅಥವಾ ಮೂಲಸೌಕರ್ಯ ಸಂರಚನೆಯ ಹಿಂದಿನ ಆವೃತ್ತಿಗೆ ಮರುಪರಿವರ್ತಿಸುವ ಪ್ರಕ್ರಿಯೆ.

## S

**Service**  
azure.yaml ನಲ್ಲಿ ವ್ಯಾಖ್ಯಾನಿಸಲಾದ ನಿಮ್ಮ ಅಪ್ಲಿಕೇಶನ್‌ನ ಒಂದು ಘಟಕ (ಉದಾಹರಣೆಗೆ, ವೆಬ್ ಫ್ರಂಟ್‌ಎಂಡ್, API ಬ್ಯಾಕ್‌ಎಂಡ್, ಡೇಟಾಬೇಸ್).

**SKU**  
ಸ್ಟಾಕ್ ಕೀಪಿಂಗ್ ಯೂನಿಟ್. Azure ಸಂಪನ್ಮೂಲಗಳಿಗೆ ವಿಭಿನ್ನ ಸೇವಾ ತರಗತಿಗಳು ಅಥವಾ ಕಾರ್ಯಕ್ಷಮತಾ ಮಟ್ಟಗಳನ್ನು ಪ್ರತಿನಿಧಿಸುತ್ತದೆ.

**SQL Database**  
Microsoft SQL Server ಆಧಾರಿತ Azure ನ ನಿರ್ವಹಿತ ರಿಲೇಷನಲ್ ಡೇಟಾಬೇಸ್ ಸೇವೆ.

**Static Web Apps**  
ಸೋರ್ಸ್ ಕೋಡ್ ರೆಪೊಗಳಿಂದ ಫುಲ್-ಸ್ಟ್ಯಾಕ್ ವೆಬ್ ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ನಿರ್ಮಿಸಿ ನಿಯೋಜಿಸಲು Azure ನ ಸೇವೆ.

**Storage Account**  
ಬ್ಲಾಬ್‌ಗಳು, ಫೈಲ್‌ಗಳು, ಕ್ಯೂಗಳು ಮತ್ತು ಟೇಬಲ್‌ಗಳಂತಹ ಡೇಟಾ ವಸ್ತುಗಳಿಗೆ ಕ್ಲೌಡ್ ಸಂಗ್ರಹಣೆಯನ್ನು ಒದಗಿಸುವ Azure ಸೇವೆ.

**Subscription**  
ಬಿಲ್ಲಿಂಗ್ ಮತ್ತು ಪ್ರವೇಶ ನಿರ್ವಹಣೆಯೊಂದಿಗೆ ಸಂಪನ್ಮೂಲ ಗುಂಪುಗಳು ಮತ್ತು ಸಂಪನ್ಮೂಲಗಳನ್ನು ಹೊಂದಿರುವ Azure ಖಾತೆ ಕಂಟೇನರ್.

## T

**Template**  
ಸಾಮಾನ್ಯ السينಾರಿಯೊಗಳಿಗೆ ಅಪ್ಲಿಕೇಶನ್ ಕೋಡ್, ಮೂಲಸೌಕರ್ಯ ವ್ಯಾಖ್ಯಾನಗಳು ಮತ್ತು ಕಾನ್ಫಿಗರೇಶನನ್ನು ಒಳಗೊಂಡ ಪೂರ್ವ-ಬಿಲ್ಟ್ ಯೋಜನಾ ರಚನೆ.

**Terraform**  
ಬಹು-ಕ್ಲೌಡ್ ಪ್ರದಾತರನ್ನೂ ಬೆಂಬಲಿಸುವ ಓಪನ್-ಸೋರ್ಸ್ Infrastructure as Code ಸಾಧನ.

**Traffic Manager**  
ಗ್ಲೋಬಲ್ Azure ಪ್ರದೇಶಗಳ ನಡುವೆ ಟ್ರಾಫಿಕ್ ಹಂಚಿಕೊಳ್ಳಲು DNS-ಆಧಾರಿತ ಟ್ರಾಫಿಕ್ ಲೋಡ್ ಬ್ಯಾಲೆನ್ಸರ್.

## U

**URI**  
Uniform Resource Identifier. ನಿರ್ದಿಷ್ಟ ಸಂಪನ್ಮೂಲವನ್ನು ಗುರುತಿಸುವ ಸ್ಟ್ರಿಂಗ್.

**URL**  
Uniform Resource Locator. ಸಂಪನ್ಮೂಲ ಎಲ್ಲಿ ಇದೆ ಮತ್ತು ಅದನ್ನು ಹೇಗೆ ಪಡೆಯಬೇಕೆಂಬುದನ್ನು ಸೂಚಿಸುವ URI ರೀತಿ.

## V

**Virtual Network (VNet)**  
Azure ನಲ್ಲಿ ಖಾಸಗಿ ನೆಟ್‌ವರ್ಕ್‌ಗಾಗಿ ಮೂಲಭೂತ ಬಿಲ್ಡಿಂಗ್ ಬ್ಲಾಕ್, ಇದನ್ನು ಐಸೊಲೇಶನ್ ಮತ್ತು ವಿಭಾಗಕ್ಕೆ ಒದಗಿಸಲಾಗುತ್ತದೆ.

**VS Code**  
Visual Studio Code. 뛰어난 Azure ಮತ್ತು azd ಸಂಯೋಜನೆ ಹೊಂದಿರುವ ಜನಪ್ರಿಯ ಕೋಡ್ ಸಂಪಾದಕ.

## W

**Webhook**  
ನಿರ್ದಿಷ್ಟ ಘಟನೆಗಳಿಂದ ಚಾಲಿತವಾಗುವ HTTP ಕ್ಯಾಲ್ಬ್ಯಾಕ್, ಸಾಮಾನ್ಯವಾಗಿ CI/CD ಪೈಪ್‌ಲೈನ್ನಲ್ಲಿ ಬಳಸಲಾಗುತ್ತದೆ.

**What-if**  
ನಿಯೋಜನೆ ಕಾರ್ಯಗತಗೊಳ್ಳಿಸುವ ಮುನ್ನ ಯಾವ ಬದಲಾವಣೆಗಳು ಆಗಬಹುದು ಎಂಬುದನ್ನು ತೋರಿಸುವ Azure ವೈಶಿಷ್ಟ್ಯ.

## Y

**YAML**  
YAML Ain't Markup Language. azure.yaml ಮಟ್ಟದ ಕಾನ್ಫಿಗರೇಶನ್ ಫೈಲ್‌ಗಾಗಿ ಬಳಸುವ ಮಾನವ-ಆದರ್ಶದ ದತ್ತಾಂಶ ಸರಣೀಕರಣ ಸ್ಟಾಂಡರ್ಡ್.

## Z

**Zone**  
Redundancy ಮತ್ತು high availability ನೀಡಲು Azure ರೀಜನಿನ ಒಳಗಿನ ಭೌತಿಕವಾಗಿ ಪ್ರತ್ಯೇಕ ಸ್ಥಳಗಳು.

## ಸಂಧర్భ-ನಿರ್ದಿಷ್ಟ ಪದಗಳು

### ಅಭಿವೃದ್ಧಿ ಪದಗಳು
- **Hot Reload**: ಅಭಿವೃದ್ಧಿದ ಸಮಯದಲ್ಲಿ ಪುನರ್ ಪ್ರಾರಂಭವಿಲ್ಲದೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು تازهಗೊಳಿಸುವುದು
- **Build Pipeline**: ಕೋಡ್ ಅನ್ನು ನಿರ್ಮಿಸುವ ಮತ್ತು ಪರೀಕ್ಷಿಸುವ ಸ್ವಯಂಚಾಲಿತ ಪ್ರಕ್ರಿಯೆ
- **Deployment Slot**: App Service ಒಳಗಿನ ಸ್ಟೇಜಿಂಗ್ ಪರಿಸರ
- **Environment Parity**: ಅಭಿವೃದ್ಧಿ, ಸ್ಟೇಜಿಂಗ್ ಮತ್ತು ಉತ್ಪಾದನಾ ಪರಿಸರಗಳನ್ನು ಸಮಾನವಾಗಿರಿಸುವಿಕೆ

### ಭದ್ರತಾ ಪದಗಳು
- **Managed Identity**: ಸ್ವಯಂಚಾಲಿತ ಕ್ರೆಡೆನ್‌ಶಿಯಲ್ ನಿರ್ವಹಣೆಯನ್ನು ಒದಗಿಸುವ Azure ವೈಶಿಷ್ಟ್ಯ
- **Key Vault**: ರಹಸ್ಯಗಳು, ಕೀಗಳು ಮತ್ತು ಪ್ರಮಾಣಪತ್ರಗಳ ಸುರಕ್ಷಿತ ಸಂಗ್ರಹಣೆ
- **RBAC**: Azure ಸಂಪನ್ಮೂಲಗಳಿಗಾಗಿ ಪಾತ್ರಾಧಾರಿತ ಪ್ರವೇಶ ನಿಯಂತ್ರಣೆ
- **Network Security Group**: ನೆಟ್‌ವರ್ಕ್ ಹಿತಚಿಂತೆಯನ್ನು ನಿಯಂತ್ರಿಸಲು ವರ್ಚುವಲ್ ಫೈರ್‌ವಾಲ್

### ಮೇಲ್ವಿಚಾರಣಾ ಪದಗಳು
- **Telemetry**: ಅಳತೆ ಮತ್ತು ಡೇಟಾ ಗಳಿಕೆಯ ಸ್ವಯಂಚಾಲಿತ ಪ್ರಕ್ರಿಯೆ
- **Application Performance Monitoring (APM)**: ಸಾಫ್ಟ್‌ವೇರ್ ಕಾರ್ಯಕ್ಷಮತೆಗೆ ಮೇಲ್ವಿಚಾರಣೆ
- **Log Analytics**: ಲಾಗ್ ಡೇಟಾ ಸಂಗ್ರಹಣೆ ಮತ್ತು ವಿಶ್ಲೇಷಣೆಗಾಗಿ ಸೇವೆ
- **Alert Rules**: ಮೆಟ್ರಿಕ್‌ಗಳು ಅಥವಾ ಷರತ್ತುಗಳ ಆಧಾರದ ಮೇಲೆ ಸ್ವಯಂಚಾಲಿತ ಸೂಚನೆಗಳು

### ನಿಯೋಜನೆ ಪದಗಳು
- **Blue-Green Deployment**: ಶೂನ್ಯ-ಡೌನ್‌ಟೈಮ್ ನಿಯೋಜನೆಯ ತಂತ್ರ
- **Canary Deployment**: ಬಳಕೆದಾರರ ಉಪಸೆಟ್‌ಗೆ Gradual ರೋಲೌಟ್
- **Rolling Update**: ಅಪ್ಲಿಕೇಶನ್ ಇನ್ಸ್‌ಟ್ಯಾನ್ಸ್‌ಗಳ ಕ್ರಮಕ್ರಮದ ಬದಲಾವಣೆ
- **Rollback**: ಹಿಂದೆ ಇದ್ದ ಅಪ್ಲಿಕೇಶನ್ ಆವೃತ್ತಿಗೆ ಮರುಭರ್ತಿ

---

**ಬಳಕೆ ಸಲಹೆ**: ನಿರ್ದಿಷ್ಟ ಪದಗಳನ್ನು ಶೀಘ್ರವಾಗಿ ಹುಡುಕಲು `Ctrl+F` ಬಳಸಿ. ಪದಗಳು ಅಗತ್ಯವಿದ್ದಲ್ಲಿ ಪರಸ್ಪರ ಉಲ್ಲೇಖಿಸಲಾಗಿದೆ.
