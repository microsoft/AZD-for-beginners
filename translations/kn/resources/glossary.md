## ಪರಿಚಯ

ಈ ವ್ಯಾಪಕ ಶಬ್ದಕೋಶವು Azure Developer CLI ಮತ್ತು Azure ಕ್ಲೌಡ್ ಅಭಿವೃದ್ಧಿಯಲ್ಲಿ ಬಳಸಲಾಗುವ ಪದಗಳು, ಪರಿಕಲ್ಪನೆಗಳು ಮತ್ತು ಸಂಕ್ಷೇಪಣಿಗಳ (acronyms) ವ್ಯಾಖ್ಯಾನಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ. ತಾಂತ್ರಿಕ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಅನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವುದು, ಸಮಸ್ಯೆಗಳ ಪರಿಹಾರ ಮಾಡುವುದು, ಮತ್ತು azd ಯೋಜನೆಗಳು ಮತ್ತು Azure ಸೇವೆಗಳ ಬಗ್ಗೆ ಪರಿಣಾಮಕಾರಿಯಾಗಿ ಸಂವಹನ ನಡೆಸುವುದಕ್ಕಾಗಿ ಇದು ಪ್ರಮುಖ ಉಲ್ಲೇಖವಾಗಿದೆ.

## ಕಲಿಕೆಯ ಗುರಿಗಳು

By using this glossary, you will:
- Azure Developer CLIಗೆ ಸಂಬಂಧಿಸಿದ ನೋಡಬೇಕಾದ ಪದಗಳು ಮತ್ತು ಪರಿಕಲ್ಪನೆಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಿ
- Azure ಕ್ಲೌಡ್ ಅಭಿವೃದ್ಧಿ ಪದಸಂಪತ್ತಿನ ಮತ್ತು ತಾಂತ್ರಿಕ ಪದಜಾಲವನ್ನು ಪರಿಣತಿಯೊಂದಿಗೆ ಕಲಿಯಿರಿ
- Infrastructure as Code ಮತ್ತು ನಿಯೋಜನೆ ಪದರಚನೆಗಳಿಗೆ ದ್ರುತ ಉಲ್ಲೇಖವನ್ನು ಹೊಂದಿರಿ
- Azure ಸೇವೆಗಳ ಹೆಸರುಗಳು, ಸಂಕ್ಷೇಪಣಿಗಳು ಮತ್ತು ಅವುಗಳ ಉದ್ದೇಶಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಿ
- ತೊಂದರೆ ಪರಿಹಾರ ಮತ್ತು ಡಿಬಗ್ಗಿಂಗ್ ಸಂಬಂಧಿತ ಪದಗಳ ವ್ಯಾಖ್ಯಾನಗಳನ್ನು ಪ್ರವೇಶಿಸಬಹುದು
- ಉನ್ನತ Azure ಆರ್ಕಿಟೆಕ್ಚರ್ ಮತ್ತು ಅಭಿವೃದ್ಧಿ ಪರಿಕಲ್ಪನೆಗಳನ್ನು ಕಲಿಯಿರಿ

## ಕಲಿಕೆಯ ಫಲಿತಾಂಶಗಳು

With regular reference to this glossary, you will be able to:
- ಸರಿಯಾದ Azure Developer CLI ಪದಾವಳಿಯನ್ನು ಬಳಸಿ ಪರಿಣಾಮಕಾರಿಯಾಗಿ ಸಂವಹನ ನಡೆಸಲು ಸಾಧ್ಯವಾಗುತ್ತದೆ
- ತಾಂತ್ರಿಕ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಮತ್ತು ದೋಷ ಸಂದೇಶಗಳನ್ನು ಸ್ಪಷ್ಟವಾಗಿ ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು ಸಾಧ್ಯವಾಗುತ್ತದೆ
- Azure ಸೇವೆಗಳನ್ನುವು ಮತ್ತು ಪರಿಕಲ್ಪನೆಗಳನ್ನು ಆತ್ಮವಿಶ್ವಾಸದಿಂದ ನ್ಯಾವಿಗೇಟ್ ಮಾಡಿಕೊಳ್ಳಲು ಸಾಧ್ಯವಾಗುತ್ತದೆ
- ಸೂಕ್ತ ತಾಂತ್ರಿಕ ಪದಜಾಲವನ್ನು ಬಳಸಿ ಸಮಸ್ಯೆಗಳನ್ನು ಟ್ರಬಲ್‌ಶೂಟ್ ಮಾಡಬಹುದು
- ಸಮ precisão ಯುತ ತಾಂತ್ರಿಕ ಭಾಷೆಯನ್ನು ಬಳಸಿ ತಂಡದ ಚರ್ಚೆಗಳಿಗೆ ಕೊಡುಗೆ ನೀಡಬಹುದು
- ಕ್ರಮಬದ್ಧವಾಗಿ ನಿಮ್ಮ Azure ಕ್ಲೌಡ್ ಅಭಿವೃದ್ಧಿ ಜ್ಞಾನವನ್ನು ವಿಸ್ತರಿಸಬಹುದು

## A

**ARM Template**  
Azure Resource Manager ಟೆಂಪ್ಲೇಟ್. JSON ಆಧಾರಿತ Infrastructure as Code ಫಾರ್ಮ್ಯಾಟ್, ಇದನ್ನು ರೂಪದಿಂದ Azure ಸಂಪನ್ಮೂಲಗಳನ್ನು ವ್ಯಾಖ್ಯಾನಿಸಿ ನಿಯೋಜಿಸಲು ಬಳಸುತಾರೆ.

**App Service**  
ವೆಬ್ ಅಪ್ಲಿಕೇಶನ್‌ಗಳು, REST APIಗಳು ಮತ್ತು ಮೊಬೈಲ್ ಬ್ಯಾಕೆಂಡ್ಗಳನ್ನು ಇನ್ಫ್ರಾಸ್ಟ್ರಕ್ಚರ್ ನಿರ್ವಹಣೆಯಿಲ್ಲದೆ ಹೋಸ್ಟ್ ಮಾಡುವ Azure ಪ್ಲಾಟ್‌ಫಾರ್ಮ್-ಏಸ್-ಎ-ಸರ್ವಿಸ್ (PaaS) ಸೇವೆ.

**Application Insights**  
ಅಪ್ಲಿಕೇಶನ್ ಪರ್ಫಾರ್ಮನ್ಸ್‌ನ ಬಗ್ಗೆ ಆಳವಾದ ತಿಳಿವುಗಳನ್ನು ಒದಗಿಸುವ Azure ಇನ್ಫ್ರಾಸ್ಟ್ರಕ್ಚರ್ ಆಪ್ಲಿಕೇಶನ್ ಮೊನಿಟರಿಂಗ್ (APM) ಸೇವೆ — ಉದಾಹರಣೆಗೆ ಪ್ರದರ್ಶನ, ಲಭ್ಯತೆ ಮತ್ತು ಬಳಕೆ.

**Azure CLI**  
Azure ಸಂಪನ್ಮೂಲಗಳನ್ನು ನಿರ್ವಹಿಸಲು ಬಳಸುವ ಕಮಾಂಡ್-ಲೈನ್ ಇಂಟರ್ಫೇಸ್. azd ಮೂಲಕ ಪ್ರಾಮಾಣೀಕರಣ ಮತ್ತು ಕೆಲವು ಕಾರ್ಯಗಳನ್ನು ನಿರ್ವಹಿಸಲು ಇದನ್ನು ಬಳಸುತ್ತಾರೆ.

**Azure Developer CLI (azd)**  
ಡೆವಲಪರ್-ಕೇಂದ್ರಿತ ಕಮಾಂಡ್-ಲೈನ್ ಟೂಲ್ಂದು, ಟೆಂಪ್ಲೇಟ್‌ಗಳು ಮತ್ತು Infrastructure as Code ಅನ್ನು ಬಳಸಿಕೊಂಡು ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು Azure ಗೆ ನಿರ್ಮಿಸುವ ಮತ್ತು ನಿಯೋಜಿಸುವ ಪ್ರಕ್ರಿಯೆಯನ್ನು ವೇಗಗತಿಗೊಳಿಸುತ್ತದೆ.

**azure.yaml**  
azd ಪ್ರಾಜೆಕ್ಟ್‌ನ ಮುಖ್ಯ ಸಂರಚನಾ ಫೈಲ್, ಇದು ಸರ್ವೀಸ್‌ಗಳು, ಇನ್ಫ್ರಾಸ್ಟ್ರಕ್ಚರ್ ಮತ್ತು ನಿಯೋಜನೆ ಹೂಕ್ಸ್ ಅನ್ನು ವ್ಯಾಖ್ಯಾನಿಸುತ್ತದೆ.

**Azure Resource Manager (ARM)**  
ಸಂಪನ್ಮೂಲಗಳನ್ನು ಸೃಷ್ಟಿಸುವುದು, ನವೀಕರಿಸುವುದು ಮತ್ತು ಅಳಿಸುವುದು ಸೇರಿದಂತೆ ನಿರ್ವಹಣಾ ಪದರವನ್ನು ಒದಗಿಸುವ Azure ಯಿನ ನಿಯೋಜನೆ ಮತ್ತು ನಿರ್ವಹಣಾ ಸೇವೆ.

## B

**Bicep**  
Azure ಸಂಪನ್ಮೂಲಗಳನ್ನು ನಿಯೋಜಿಸಲು Microsoft ಅಭಿವೃದ್ಧಿಪಡಿಸಿದ ಡೊಮೇನ್-ವಿಶೇಷ ಭಾಷೆ (DSL). ARM ಟೆಂಪ್ಲೇಟ್ಗಳಿಗಿಂತ ಸರಳವಾದ ಸಿಂಟ್ಯಾಕ್ಸ್ ಅನ್ನು ಒದಗಿಸುತ್ತದೆ ಮತ್ತು ARM ಗೆ ಸಂಗ್ರಹಿಸುತ್ತದೆ.

**Build**  
ಮೂಲ ಕೋಡ್ ಅನ್ನು ಸಂಕಲನಗೊಳಿಸುವುದು, ಅವಲಂಬನೆಗಳನ್ನು ಇನ್ಸ್ಟಾಲ್ ಮಾಡುವುದು ಮತ್ತು ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ನಿಯೋಜನೆಗಾಗಿ ಸಿದ್ಧಪಡಿಸುವ ಪ್ರಕ್ರಿಯೆ.

**Blue-Green Deployment**  
ಡೌನ್‌ಟೈಮ್ ಮತ್ತು જોખಮ್ ಕಡಿಮೆ ಮಾಡುವ ಉದ್ದೇಶದಿಂದ ಎರಡು ಒಂದೇ ರೀತಿ ಉತ್ಪಾದನಾ ಪರಿಸರಗಳನ್ನು (ನೀಲಿ ಮತ್ತು ಹಸಿರು) ಬಳಸದಿನ ನಿಯೋಜನೆ ತಂತ್ರ.

## C

**Container Apps**  
ಸಂಕೀರ್ಣ ಇನ್ಫ್ರಾಸ್ಟ್ರಕ್ಚರ್ ಅನ್ನು ನಿರ್ವಹಿಸದೆ ಕಾಂಟೇನರೈಜ್ಡ್ ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ಚಾಲನೆ ಮಾಡಲು ಅನುಮತಿಸುವ Azure ನ ಸರ್ವರ್‍ಲೆಸ್ ಕಂಟೈನರ್ ಸೇವೆ.

**CI/CD**  
Continuous Integration/Continuous Deployment. ಕೋಡ್ ಬದಲಾವಣೆಗಳನ್ನು ಸಮಾವೇಶಗೊಳಿಸುವ ಮತ್ತು ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ನಿಯೋಜಿಸುವ үшін ಸ್ವಯಂಚಾಲಿತ ಅಭ್ಯಾಸಗಳು.

**Cosmos DB**  
ಗ್ಲೋಬಲಾಗಿ ವಿತರಿತ, ಮಲ್ಟಿ-ಮಾಡೆಲ್ ಡೇಟಾಬೇಸ್ ಸೇವೆ; Azure ಯಲ್ಲಿ throughput, latency, ಲಭ್ಯತೆ ಮತ್ತು ಕಾನ್ಸಿಸ್ಟೆನ್ಸಿಗಾಗಿ ಸಮಗ್ರ SLAಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ.

**Configuration**  
ಅಪ್ಲಿಕೇಶನ್ ವ್ಯವಹಾರ ಮತ್ತು ನಿಯೋಜನೆ વિકಲ್ಪಗಳನ್ನು ನಿಯಂತ್ರಿಸುವ ಸೆಟ್ಟಿಂಗ್ಗಳು ಮತ್ತು ಪ್ಯಾರಾಮೀಟರ್ಗಳು.

## D

**Deployment**  
ಲಕ್ಷ್ಯ ಇನ್ಫ್ರಾಸ್ಟ்ரಕ್ಚರ್ ಮೇಲೆ ಅಪ್ಲಿಕೇಶನ್‌ಗಳು ಮತ್ತು ಅವುವು ಕೇಳುವ ಅವಲಂಬನೆಗಳನ್ನು ಇನ್‌ಸ್ಟಾಲ್ ಮತ್ತು ಸಂರಚಿಸುವ ಪ್ರಕ್ರಿಯೆ.

**Docker**  
ಕಂಟೈನರೈಸೇಷನ್ ತಂತ್ರಜ್ಞಾನವನ್ನು ಬಳಸಿಕೊಂಡು ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ಅಭಿವೃದ್ಧಿಪಡಿಸುವ, ಸಾಗಿಸುವ ಮತ್ತು ಚಾಲನೆ ಮಾಡುವ ವೇದಿಕೆ.

**Dockerfile**  
ಡೊಕರ್ ಕಂಟೈನರ್ ಇಮೇಜ್ ನಿರ್ಮಿಸುವ ಸೂಚನೆಗಳನ್ನು ಒಳಗೊಂಡಿರುವ ಪಠ್ಯ ಫೈಲ್.

## E

**Environment**  
ನಿಮ್ಮ ಅಪ್ಲಿಕೇಶನ್‌ನ ನಿರ್ದಿಷ್ಟ ಉದಾಹರಣೆಯನ್ನು ಪ್ರತಿನಿಧಿಸುವ ನಿಯೋಜನೆ ಗುರಿ (ಉದಾಹರಣೆ: development, staging, production).

**Environment Variables**  
ಅಪ್ಲಿಕೇಶನ್‌ಗಳು ರನ್‌ಟೈಮ್‌ನಲ್ಲಿ ಪ್ರವೇಶಿಸಲು ಸಾಧ್ಯವಾಗುವ ಕೀ-ವ್ಯಾಲ್ಯೂ ಜೋಡಿಗಳಾಗಿ ಸಂಗ್ರಹಿಸಲಾದ ಸಂರಚನಾ ಮೌಲ್ಯಗಳು.

**Endpoint**  
ಅಪ್ಲಿಕೇಶನ್ ಅಥವಾ ಸೇವೆಯನ್ನು ಪ್ರವೇಶಿಸಲು URL ಅಥವಾ ನೆಟ್‌ವರ್ಕ್ ವಿಳಾಸ.

## F

**Function App**  
ಇನ್ಫ್ರಾಸ್ಟ್ರಕ್ಚರ್ ನಿರ್ವಹಣೆ ಇಲ್ಲದೆ ಘಟನೆಗಳ ಆಧಾರಿತ ಕೋಡ್ ಚಾಲನೆ ಮಾಡಲು ಅನುಮತಿಸುವ Azure ಯಿನ ಸರ್ವರ್‍ಲೆಸ್ ಕಂಪ್ಯೂಟ್ ಸೇವೆ.

## G

**GitHub Actions**  
GitHub ರೆಪೊಸಿಟರಿಗಳೊಂದಿಗೆ ಏಕೀಭವಿತ CI/CD ವೇದಿಕೆ, ವರ್ಕ್‌ಫ್ಲೋಗಳನ್ನು ಸ್ವಚಾಲಿತಗೊಳಿಸಲು.

**Git**  
ಮೂಲ ಕೋಡ್‌ನಲ್ಲಿನ ಬದಲಾವಣೆಗಳನ್ನು ಟ್ರ್ಯಾಕ್ ಮಾಡಲು ಬಳಸುವ ವಿತರಿಸಲಾದ ವರ್ಝನ್ ಕಂಟ್ರೋಲ್ ವ್ಯವಸ್ಥೆ.

## H

**Hooks**  
ನಿಯೋಜನೆ ಜೀವನಚಕ್ರದ ನಿರ್ದಿಷ್ಟ ಬಿಂದುಗಳಲ್ಲಿ (preprovision, postprovision, predeploy, postdeploy) ચાલುವ ಕಸ್ಟಮ್ ಸ್ಕ್ರಿಪ್ಟ್‌ಗಳು ಅಥವಾ ಕಮಾಂಡ್‌ಗಳು.

**Host**  
ಅಪ್ಲಿಕೇಶನ್ ನಿಯೋಜಿಸಲಾಗುವ Azure ಸೇವೆಯ ಪ್ರಕಾರ (ಉದಾ: appservice, containerapp, function).

## I

**Infrastructure as Code (IaC)**  
ಮಾನವೀಯ ಪ್ರಕ್ರಿಯೆಗಳ ಬದಲು ಕೋಡ್ ಮೂಲಕ ಇನ್ಫ್ರಾಸ್ಟ್ರಕ್ಚರ್ ಅನ್ನು ವ್ಯಾಖ್ಯಾನಿಸುವ ಮತ್ತು ನಿರ್ವಹಿಸುವ ಅಭ್ಯಾಸ.

**Init**  
ಸಾಮಾನ್ಯವಾಗಿ ಟೆಂಪ್ಲೇಟ್‌ನಿಂದ ಹೊಸ azd ಪ್ರಾಜೆಕ್ಟ್ ಅನ್ನು ಪ್ರಾರಂಭಿಸುವ ಪ್ರಕ್ರಿಯೆ.

## J

**JSON**  
JavaScript Object Notation. ಕಾಂಫಿಗರೇಶನ್ ಫೈಲ್‌ಗಳು ಮತ್ತು API ಪ್ರತಿಕ್ರಮೆಗಳಿಗೆ ಸಾಮಾನ್ಯವಾಗಿ ಬಳಸುವ ಡೇಟಾ ವಿನಿಮಯ ಫಾರ್ಮ್ಯಾಟ್.

**JWT**  
JSON Web Token. ಪಕ್ಷಗಳ ನಡುವೆ ಮಾಹಿತಿ ಸುರಕ್ಷಿತವಾಗಿ ಸಾರುವ ಸ್ಟ್ಯಾಂಡರ್ಡ್ (JSON ವಸ್ತುವಾಗಿ).

## K

**Key Vault**  
ಗುಪ್ತ ಕಾರ್ಯಗಳು, ಕೀಲಿಗಳು ಮತ್ತು ಪ್ರಮಾಣಪತ್ರಗಳನ್ನು ಸುರಕ್ಷಿತವಾಗಿ ಸಂಗ್ರಹಿಸಿ ನಿರ್ವಹಿಸುವ Azure ಸೇವೆ.

**Kusto Query Language (KQL)**  
Azure Monitor, Application Insights ಮತ್ತು ಇतर Azure ಸೇವೆಗಳಲ್ಲಿ ಡೇಟಾವನ್ನು ವಿಶ್ಲೇಷಿಸಲು ಬಳಸುವ ಪ್ರಶ್ನೆಭಾಷೆ.

## L

**Load Balancer**  
ಬಹುשרתಗಳು ಅಥವಾ ಪೊಲೀಸಗಳ ಮಧ್ಯೆ ಬರುತ್ತಿರುವ ನೆಟ್‌ವರ್ಕ್ ಟ್ರಾಫಿಕ್ ಅನ್ನು ಹಂಚುವ ಸೇವೆ.

**Log Analytics**  
ಕ್ಲೌಡ್ ಮತ್ತು ಆನ್-ಪ್ರೇಮೈಸಸ್ ಪರಿಸರಗಳಿಂದ ಟೆಲಿಮೆಟ್ರಿ ಡೇಟಾವನ್ನು ಸಂಗ್ರಹಿಸಲು, ವಿಶ್ಲೇಷಿಸಲು ಮತ್ತು ಕ್ರಮ ಕೈಗೊಳ್ಳಲು ಉಪಯೋಗಿಸುವ Azure ಸೇವೆ.

## M

**Managed Identity**  
ಇತರ Azure ಸೇವೆಗಳಿಗೆ ಪ್ರಾಮಾಣೀಕರಣಕ್ಕಾಗಿ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನಿರ್ವಹಿಸಲಾದ ಐಡೆಂಟಿಟಿಯನ್ನು Azure ಸೇವೆಗಳಿಗೆ ಒದಗಿಸುವ ವೈಶಿಷ್ಟ್ಯ.

**Microservices**  
ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ಸಣ್ಣ, ಸ್ವತಂತ್ರ ಸೇವೆಗಳ ಸಂಗ್ರಹವಾಗಿ ನಿರ್ಮಿಸುವ ಆರ್ಕಿಟೆಕ್ಚರ್ ಕ್ರಮ.

**Monitor**  
ಅಪ್ಲಿಕೇಶನ್‌ಗಳು ಮತ್ತು ಇನ್ಫ್ರಾಸ್ಟ್ರಕ್ಚರ್‌ಗಳನ್ನು ವ್ಯಾಪಕವಾಗಿ ಅವಲೋಕಿಸುವುದಕ್ಕಾಗಿ Azure ಯ ಏಕೀಕೃತ ಮಾಇಟರಿಂಗ್ ಪರಿಹಾರ.

## N

**Node.js**  
ಸರ್ವರ್-ಸೈಡ್ ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ನಿರ್ಮಿಸಲು Chrome ನ V8 ಜಾವಾಸ್ಕ್ರಿಪ್ಟ್ ಎಂಜಿನ್ ಮೇಲೆ ನಿರ್ಮಿತ ಜಾವಾಸ್ಕ್ರಿಪ್ಟ್ ರನ್‌ಟೈಮ್.

**npm**  
Node.js ಗೆ ಪ್ಯಾಕೇಜ್ ಮ್ಯಾನೇಜರ್; ಅವಲಂಬನೆಗಳು ಮತ್ತು ಪ್ಯಾಕೇಜ್‌ಗಳನ್ನು ನಿರ್ವಹಿಸುತ್ತದೆ.

## O

**Output**  
ಇನ್ಫ್ರಾಸ್ಟ್ರಕ್ಚರ್ ನಿಯೋಜನೆಯಿಂದ ಮರಳಿಸುವ ಮೌಲ್ಯಗಳು, ಅವುಗಳನ್ನು ಅಪ್ಲಿಕೇಶನ್‌ಗಳು ಅಥವಾ ಇತರ ಸಂಪನ್ಮೂಲಗಳು ಬಳಸಬಹುದು.

## P

**Package**  
ಅಪ್ಲಿಕೇಶನ್ ಕೋಡ್ ಮತ್ತು ಅವಲಂಬನೆಗಳನ್ನು ನಿಯೋಜನೆಗಾಗಿ ತಯಾರಿಸುವ ಪ್ರಕ್ರಿಯೆ.

**Parameters**  
ನಿಯೋಜನೆಗಳನ್ನು ಕಸ್ಟಮೈಸೇಶನ್ ಮಾಡಲು ಇನ್ಫ್ರಾಸ್ಟ್ರಕ್ಚರ್ ಟೆಂಪ್ಲೇಟ್ಗೆ ಪಾಸ್ಸಾಗುವ ಇನ್ಪುಟ್ ಮೌಲ್ಯಗಳು.

**PostgreSQL**  
ಖಾಲಿ-ಮೂಲ(Relational) ಡೇಟಾಬೇಸ್ ಸಿಸ್ಟಂ; Azure ನಲ್ಲಿ ನಿರ್ವಹಿತ ಸೇವೆಯಾಗಿ ಬೆಂಬಲಿತ.

**Provisioning**  
ಇನ್ಫ್ರಾಸ್ಟ್ರಕ್ಚರ್ ಟೆಂಪ್ಲೇಟಿನಲ್ಲಿ ವ್ಯಾಖ್ಯಾನಿಸಿದ Azure ಸಂಪನ್ಮೂಲಗಳನ್ನು ಸೃಷ್ಟಿಸುವ ಮತ್ತು ಸಂರಚಿಸುವ ಪ್ರಕ್ರಿಯೆ.

## Q

**Quota**  
Azure ಸಬ್ಸ್ಕ್ರಿಪ್ಷನ್ ಅಥವಾ ಪ್ರದೇಶದಲ್ಲಿ ಸೃಷ್ಟಿಸಬಹುದಾದ ಸಂಪನ್ಮೂಲಗಳ ಪ್ರಮಾಣದ ಮೇಲಿನ ಮಿತಿ.

## R

**Resource Group**  
ಒಂದುವೇ ಜೀವನಚಕ্ৰ, ಅನುಮತಿಗಳು ಮತ್ತು ಪಾಲಿಸಿಗಳನ್ನು ಹಂಚಿಕೊಳ್ಳುವ Azure ಸಂಪನ್ಮೂಲಗಳ ಲಾಜಿಕಲ್ ಕಂಟೈನರ್.

**Resource Token**  
ನಿಯೋಜನೆಗಳು ಜুডಿದು ಸಂಪನ್ಮೂಲ ನಾಮಗಳನ್ನು ಏಕೈಕವಾಗಿರಿಸಲು azd ರಚಿಸುವ ವಿಶಿಷ್ಟ ಸ್ಟ್ರಿಂಗ್.

**REST API**  
HTTP ವಿಧಾನಗಳನ್ನು ಬಳಸಿಕೊಂಡು ನೆಟ್‌ವರ್ಕ್ ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ವಿನ್ಯಾಸಗೊಳಿಸುವ ಆರ್ಕಿಟೆಕ್ಚರಲ್ ಶೈಲಿ.

**Rollback**  
ಅಪ್ಲಿಕೇಶನ್ ಅಥವಾ ಇನ್ಫ್ರಾಸ್ಟ್ರಕ್ಚರ್ ಸಂರಚನೆಯ ಹಿಂದಿನ ಆವೃತ್ತಿಗೆ ಮರಳಿಸುವ ಪ್ರಕ್ರಿಯೆ.

## S

**Service**  
azure.yaml ನಲ್ಲಿ ವ್ಯಾಖ್ಯಾನಿಸಲಾದ ನಿಮ್ಮ ಅಪ್ಲಿಕೇಶನ್‌ನ ಒಂದು ಘಟಕ (ಉದಾ: ವೆಬ್ ಫ್ರಂಟೆಂಡ್, API ಬ್ಯಾಕೆಂಡ್, ಡೇಟಾಬೇಸ್).

**SKU**  
Stock Keeping Unit. Azure ಸಂಪನ್ಮೂಲಗಳಿಗೆ ವಿಭಿನ್ನ ಸೇವಾ ಟಿಯರ್ ಅಥವಾ ಪ್ರದರ್ಶನ ಮಟ್ಟಗಳನ್ನು ಪ್ರತಿನಿಧಿಸುತ್ತದೆ.

**SQL Database**  
Microsoft SQL Server ಆಧಾರಿತ Azure ಯ ನಿರ್ವಹಿತ ರಿಲೇಷನಲ್ ಡೇಟಾಬೇಸ್ ಸೇವೆ.

**Static Web Apps**  
ಸೋರ್ಸ್ ಕೋಡ್ ರೆಪೊಸಿಟರಿಗಳಿಂದ ಫುಲ್-ಸ್ಟಾಕ್ ವೆಬ್ ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ನಿರ್ಮಿಸಲು ಮತ್ತು ನಿಯೋಜಿಸಲು Azure ಸೇವೆ.

**Storage Account**  
ಬ್ಲಾಬ್‍ಗಳು, ಫೈಲ್‍ಗಳು, ಕ್ಯೂಗಳು ಮತ್ತು ಟೇಬಲ್‍ಗಳು ಸೇರಿದಂತೆ ಡೇಟಾ ವಸ್ತುಗಳಿಗೆ ಕ್ಲೌಡ್ ಸಂಗ್ರಹಣೆಯನ್ನು ಒದಗಿಸುವ Azure ಸೇವೆ.

**Subscription**  
ಬಿಲಿಂಗ್ನಿ ಮತ್ತು ಪ್ರವೇಶ ನಿರ್ವಹಣೆಯೊಂದಿಗೆ ರಿಸೋರ್ಸ್ ಗ್ರೂಪ್‌ಗಳು ಮತ್ತು ಸಂಪನ್ಮೂಲಗಳನ್ನು ಹೊಂದಿರುವ Azure ಖಾತೆ ಕಂಟೈನರ್.

## T

**Template**  
ಅಪ್ಲಿಕೇಶನ್ ಕೋಡ್, ಇನ್ಫ್ರಾಸ್ಟ್ರಕ್ಚರ್ ವ್ಯಾಖ್ಯಾನಗಳು ಮತ್ತು ಸಾಮಾನ್ಯ ಸನ್ನಿವೇಶಗಳಿಗೆ ಸಂರಚನೆಯನ್ನು ಒಳಗೊಂಡ ಪೂರ್ವ-ನಿರ್ಮಿತ ಪ್ರಾಜೆಕ್ಟ್ ರಚನೆ.

**Terraform**  
ಬಹು-ಮೇಘ ಒದಗುದಾರರನ್ನು ಬೆಂಬಲಿಸುವ ಓಪನ್-ಸೋರ್ಸ್ Infrastructure as Code ಸರಂಜಾಮು, ಇದರಲ್ಲಿ Azure ಸಹ ಸೇರಿದೆ.

**Traffic Manager**  
ಗ್ಲೋಬಲ್ Azure ಪ್ರದೇಶಗಳ ಮಧ್ಯೆ ಟ್ರಾಫಿಕ್ ವಿತರಿಸಲು DNS ಆಧಾರಿತ ಟ್ರಾಫಿಕ್ ಲೋಡ್ ಬಾಲಾನ್ಸರ್.

## U

**URI**  
Uniform Resource Identifier. ಸೂಚ್ಯಂಕವಾದ ಒಂದು ನಿರ್ದಿಷ್ಟ ಸಂಪನ್ಮೂಲವನ್ನು ಗುರುತಿಸುವ ಸ್ಟ್ರಿಂಗ್.

**URL**  
Uniform Resource Locator. ಒಂದು ಸಂಪನ್ಮೂಲ ಎಲ್ಲಿ ಇದೆ ಮತ್ತು ಅದನ್ನು ಹೇಗೆ ಪಡೆಯಬೇಕು ಎಂದು ಸೂಚಿಸುವ URI ರೀತಿ.

## V

**Virtual Network (VNet)**  
Azure ನಲ್ಲಿ ಖಾಸಗಿ ನೆಟ್‌ವರ್ಕ್‌ಗಳಿಗೆ ಮೂಲಭೂತ ನಿರ್ಮಾಣ ಘಟಕ; ಐಸೊಲೇಶನ್ ಮತ್ತು ವಿಭಾಗವನ್ನು ಒದಗಿಸುತ್ತದೆ.

**VS Code**  
Visual Studio Code. ಉತ್ತಮ Azure ಮತ್ತು azd ಎಂಟಿಗ್ರೇಷನ್ ಇರುವ ಜನಪ್ರಿಯ ಕೋಡ್ ಎಡಿಟರ್.

## W

**Webhook**  
ನಿರ್ದಿಷ್ಟ ಘಟನೆಗಳಾಗಿ ಟ್ರಿಗರ್ ಆಗುವ HTTP ಕಾಲ್‌ಬ್ಯಾಕ್, ಸಾಮಾನ್ಯವಾಗಿ CI/CD ಪೈಪ್‌ಲೈನ್‌ಗಳಲ್ಲಿ ಬಳಸಲಾಗುತ್ತದೆ.

**What-if**  
ನಿಯೋಜನೆಯನ್ನು ನೈಜವಾಗಿ ಕಾರ್ಯಗತಗೊಳಿಸದೆ ಅದರಿಂದ ಯಾವ ಬದಲಾವಣೆಗಳು ಆಗುತ್ತವೆ ಎಂದು ತೋರಿಸುವ Azure ವೈಶಿಷ್ಟ್ಯ.

## Y

**YAML**  
YAML Ain't Markup Language. azure.yaml ಹಾಗು ಇತರೆ ಸಂರಚನಾ ಫೈಲ್‌ಗಳಿಗೆ ಬಳಕೆಯಲ್ಲಿರುವ ಮಾನವ-ಓದ್ಯಮ ಡೇಟಾ ಸಿರಿಯಲೈಸೇಶನ್ ಸ್ಟ್ಯಾಂಡರ್ಡ್.

## Z

**Zone**  
ಹೈ ಅವೈಲಬಿಲಿಟಿ ಮತ್ತು redundancy ಒದಗಿಸುವ Azure ಪ್ರದೇಶದ ಒಳಗಿನ ಭೌತಿಕವಾಗಿ ಪ್ರತ್ಯೇಕ ಸ್ಥಳಗಳು.

## ನೆರೆಸೂಕ್ತ ಪದಗಳು

### ಅಭಿವೃದ್ಧಿ ನಿಯೋಜನೆ ಸಂಬಂಧಿತ ಪದಗಳು
- **Hot Reload**: ಅಭಿವೃದ್ಧಿ ಸಮಯದಲ್ಲಿ ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ಮರುಪ್ರಾರಂಭಿಸದೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನವೀಕರಿಸುವುದು
- **Build Pipeline**: ಕೋಡ್ ಅನ್ನು ನಿರ್ಮಿಸಲು ಮತ್ತು ಪರೀಕ್ಷಿಸಲುಸ್ವಯಂಚಾಲಿತ ಪ್ರಕ್ರಿಯೆ
- **Deployment Slot**: App Service ಒಳಗೊಂಡ ಸ್ಟೇಜಿಂಗ್ ಪರಿಸರ
- **Environment Parity**: ಡೆವಲಪ್‌ಮೆಂಟ್, ಸ್ಟೇಜಿಂಗ್ ಮತ್ತು ಪ್ರೊಡಕ್ಷನ್ ಪರಿಸರಗಳನ್ನು ಸಮಾನವಾಗಿಡುವುದು

### ಭದ್ರತಾ ಪದಗಳು
- **Managed Identity**: ಸ್ವಯಂಚಾಲಿತ ಮಾನ್ಯತೆ ನಿರ್ವಹಣೆಯನ್ನು ಒದಗಿಸುವ Azure ವೈಶಿಷ್ಟ್ಯ
- **Key Vault**: ಗುಪ್ತಗಳು, ಕೀಲಿಗಳು ಮತ್ತು ಪ್ರಮಾಣಪತ್ರಗಳಿಗಾಗಿ ಸುರಕ್ಷಿತ ಸಂಗ್ರಹಣೆ
- **RBAC**: Azure ಸಂಪನ್ಮೂಲಗಳಿಗೆ ಯುಸರ್ ಪಾತ್ರಗಳ ಆಧಾರದ ಮೇಲೆ ಪ್ರವೇಶ ನಿಯಂತ್ರಣ
- **Network Security Group**: ನೆಟ್‌ವರ್ಕ್ ಟ್ರಾಫಿಕ್ ನಿಯಂತ್ರಿಸಲು ವರ್ಚುವಲ್ ಫೈರ್‌ವಾಲ್

### ಮಾನಿಟರಿಂಗ್ ಪದಗಳು
- **Telemetry**: ಗಣನೆ ಮತ್ತು ಡೇಟಾ ಸ್ವಯಂಚಾಲಿತ ಸಂಗ್ರಹಣೆ
- **Application Performance Monitoring (APM)**: ಸಾಫ್ಟ್‌ವೇರ್ ಪ್ರದರ್ಶನದ ಮಾನಿಟರಿಂಗ್
- **Log Analytics**: ಲಾಗ್ ಡೇಟಾವನ್ನು ಸಂಗ್ರಹಿಸಿ ವಿಶ್ಲೇಷಿಸುವ ಸೇವೆ
- **Alert Rules**: ಮ್ಯಾಟ್ರಿಕ್‌ಗಳು ಅಥವಾ ಶರತ್ತುಗಳ ಆಧಾರದ ಮೇಲೆ ಸ್ವಯಂಚಾಲಿತ ಸೂಚನೆಗಳು

### ನಿಯೋಜನೆ ಸಂಬಂಧಿತ ಪದಗಳು
- **Blue-Green Deployment**: ಜೀರೋ-ಡೌನ್‌ಟೈಮ್ ನಿಯೋಜನೆ ತಂತ್ರ
- **Canary Deployment**: ಬಳಕೆದಾರರ ಉಪಸಮೂಹಕ್ಕೆ渐渐ದಂತೆ ರೋಲೌಟ್
- **Rolling Update**: ಅಪ್ಲಿಕೇಶನ್ ಉದಾಹರಣೆಗಳ ಕ್ರಮವಾಗಿ ಪರಿವರ್ತನೆ
- **Rollback**: ಹಿಂದಿನ ಅಪ್ಲಿಕೇಶನ್ ಆವೃತ್ತಿಗೆ ಮರಳಿಸುವುದು
