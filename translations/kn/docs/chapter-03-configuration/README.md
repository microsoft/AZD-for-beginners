# ಅಧ್ಯಾಯ 3: ಸಂರಚನೆ ಮತ್ತು ಪ್ರಾಧಿಕಾರಮೂಲಕತೆ

**📚 ಕೋರ್ಸ್**: [ಮೂಲಭೂತರುಗಾಗಿ AZD](../../README.md) | **⏱️ ಅವಧಿ**: 45-60 ನಿಮಿಷಗಳು | **⭐ ಕಷ್ಟಮಟ್ಟ**: ಮಧ್ಯಮ

---

## ಅವಲೋಕನ

ಈ ಅಧ್ಯಾಯದಲ್ಲಿ ಪರಿಸರ ಸಂರಚನೆ, ಪ್ರಾಧಿಕಾರಮೂಲಕತೆ ಮಾದರಿಗಳು, ಮತ್ತು Azure Developer CLI ನ ನಿಯೋಜನೆಗಳಿಗಾಗಿ ಭದ್ರತಾ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳನ್ನು ಒಳಗೊಂಡಿದೆ.

> ಮಾರ್ಚ್ 2026 ರಲ್ಲಿ `azd 1.23.12` ಮೇಲೆ ಪರಿಶೀಲಿಸಲಾಗಿದೆ.

## ಕಲಿಕೆ ಉದ್ದೇಶಗಳು

ಈ ಅಧ್ಯಾಯವನ್ನು ಪೂರ್ಣಗೊಳಿಸುವ ಮೂಲಕ, ನೀವು:
- AZD ಸಂರಚನೆ ಹೇರಾರ್ಕಿಯನ್ನು ನಿಪುಣರಾಗಿರಿ
- ಅನೇಕ ಪರಿಸರಗಳನ್ನು (ಡೇವ್, ಸ್ಟೇಜಿಂಗ್, ಪ್ರೊಡ್) ನಿರ್ವಹಿಸಿರಿ
- ನಿರ್ವಹಿತ ಅಸ್ತಿತ್ವಗಳೊಂದಿಗೆ ಸುರಕ್ಷಿತ ಪ್ರಾಧಿಕಾರವನ್ನು ಅನುಷ್ಠಾನಗೊಳಿಸಿರಿ
- ಪರಿಸರ-ನಿರ್ದಿಷ್ಟ ಸೆಟ್ಟಿಂಗ್ಸ್ ಸಂರಚಿಸಿರಿ

## 🚀 ತ್ವರಿತ ಪ್ರಾರಂಭ

```bash
# ಅನೇಕ ಪರಿಸರಗಳನ್ನು ರಚಿಸಿ
azd env new dev
azd env new staging
azd env new prod

# ಪರಿಸರಗಳನ್ನು ಬದಲಾಯಿಸಿ
azd env select prod

# ಪರಿಸರ ಚರಗಳನ್ನು ಹೊಂದಿಸಿ
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# ಸಂರಚನೆಯನ್ನು ವೀಕ್ಷಿಸಿ
azd env get-values
```

---

## 🔧 ಸಂರಚನೆ ಹೇರಾರ್ಕಿ

AZD ಸೆಟ್ಟಿಂಗ್ಸ್ ಅನ್ನು ಈ ಕ್ರಮದಲ್ಲಿ ಅನ್ವಯಿಸುತ್ತದೆ (ಮುಂದಿನವು ಹಿಂದಿನವನ್ನು ಮೀರಿಸುತ್ತದೆ):

1. **ಪೂರ್ವನಿರ್ದಿಷ್ಟ ಮೌಲ್ಯಗಳು** (ಟೆಂಪ್ಲೇಟ್ಗಳಲ್ಲಿ ನಿರ್ಮಿತ)
2. **azure.yaml** (ಪ್ರಾಜೆಕ್ಟ್ ಸಂರಚನೆ)
3. **ಪರಿಸರ ಚರಗಳು** (`azd env set`)
4. **ಕಮಾಂಡ್-ಲೈನ್ ಫ್ಲಾಗ್‌ಗಳು** (`--location eastus`)

---

## 🔐 ಭದ್ರತಾ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು

```bash
# ನಿರ್ವಹಿತ ಐಡೆಂಟಿಟಿ ಬಳಸಿ (ಶಿಫಾರಸು ಮಾಡಲಾಗಿದೆ)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD ಪ್ರಮಾಣೀಕರಣ ಸ್ಥಿತಿಯನ್ನು ಪರಿಶೀಲಿಸಿ
azd auth status

# ಐಚ್ಛಿಕ: ನೀವು az ಆದೇಶಗಳನ್ನು ನಡಿಸಲು ಯೋಜಿಸಿರುವರೆ ಎಂದು Azure CLI ಸಾಂದರ್ಭಿಕತೆಯನ್ನು ಪರಿಶೀಲಿಸಿ
az account show

# ಅಗತ್ಯವಿರುವಲ್ಲಿ ಮರುಪ್ರಮಾಣೀಕರಿಸಿ
azd auth login

# ಐಚ್ಛಿಕ: az ಆದೇಶಗಳಿಗಾಗಿ Azure CLI ಪ್ರಮಾಣೀಕರಣವನ್ನು ನವೀಕರಿಸಿ
az login
```
