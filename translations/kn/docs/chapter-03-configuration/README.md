# ಅಧ್ಯಾಯ 3: ಸಂರಚನೆ ಮತ್ತು ಪ್ರಮಾಣೀಕರಣ

**📚 ಕೋರ್ಸ್**: [ആരಂಭಿಕರಿಗಾಗಿ AZD](../../README.md) | **⏱️ ಅವಧಿ**: 45-60 ನಿಮಿಷಗಳು | **⭐ ಸಂಕೀರ್ಣತೆ**: ಮಧ್ಯಮ

---

## ಅವಲೋಕನ

ಈ ಅಧ್ಯಾಯವು Azure Developer CLI ನಿಯೋಜನೆಗಳಿಗಾಗಿ ಪರಿಸರ ಸಂರಚನೆ, ಪ್ರಮಾಣೀಕರಣ ಮಾದರಿಗಳು ಮತ್ತು ಭದ್ರತಾ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳನ್ನು ಒಳಗೊಂಡಿದೆ.

## ಕಲಿಕೆಯ ಗುರಿಗಳು

ಈ ಅಧ್ಯಾಯವನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದರೆ, ನೀವು:
- AZD ಸಂರಚನೆ ಕ್ರಮದಲ್ಲಿ ಪರಿಣತಿಯನ್ನು ಗಳಿಸಬಹುದು
- ಬಹು-ಪರಿಸರಗಳನ್ನು ನಿರ್ವಹಿಸಬಹುದು (dev, staging, prod)
- ನಿಯಂತ್ರಿತ ಐಡೆಂಟಿಟಿಗಳೊಂದಿಗೆ ಸುರಕ್ಷಿತ ಪ್ರಮಾಣೀಕರಣವನ್ನು ಅನುಷ್ಠಾನಗೊಳಿಸಬಹುದು
- ಪರಿಸರ-ನಿರ್ದಿಷ್ಟ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಸಂರಚಿಸಬಹುದು

## 🚀 ತ್ವರಿತ ಪ್ರಾರಂಭ

```bash
# ಬಹು ಪರಿಸರಗಳನ್ನು ರಚಿಸಿ
azd env new dev
azd env new staging
azd env new prod

# ಪರಿಸರಗಳನ್ನು ಬದಲಾಯಿಸಿ
azd env select prod

# ಪರಿಸರ ಚರಗಳನ್ನು ಹೊಂದಿಸಿ
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# ರಚನೆಯನ್ನು ವೀಕ್ಷಿಸಿ
azd env get-values
```

---

## 🔧 ಸಂರಚನೆ ಕ್ರಮ

AZD ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಈ ಕ್ರಮದಲ್ಲಿ ಅನ್ವಯಿಸುತ್ತದೆ (ನಂತರದವು ಮುಂಚಿನವನ್ನ ಮೀರುತ್ತವೆ):

1. **ಡೀಫಾಲ್ಟ್ ಮೌಲ್ಯಗಳು** (ಟೆಂಪ್ಲೇಟ್ಗಳಲ್ಲಿ ನಿರ್ಮಿತ)
2. **azure.yaml** (ಪ್ರಾಜೆಕ್ಟ್ ಸಂರಚನೆ)
3. **ಪರಿಸರ ಚರಗಳು** (`azd env set`)
4. **ಕಮಾಂಡ್-ಲೈನ್ ಫ್ಲಾಗ್‌ಗಳು** (`--location eastus`)

---

## 🔐 ಭದ್ರತಾ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು

```bash
# ನಿರ್ವಹಿಸಲಾದ ಐಡೆಂಟಿಟಿಯನ್ನು ಬಳಸಿ (ಶಿಫಾರಸು ಮಾಡಲಾಗಿದೆ)
azd env set AZURE_USE_MANAGED_IDENTITY true

# ಪ್ರಮಾಣೀಕರಣ ಸ್ಥಿತಿಯನ್ನು ಪರಿಶೀಲಿಸಿ
azd auth whoami
az account show

# ಅವಶ್ಯವಿದ್ದರೆ ಪುನಃ ಪ್ರಮಾಣೀಕರಿಸಿ
azd auth login
az login
```
