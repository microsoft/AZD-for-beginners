# ಅಧ್ಯಾಯ 3: ಸಂರಚನೆ ಮತ್ತು ಪರಿಶೀಲನೆ

**📚 ಕೋರ್ಸ್**: [ಆರ್ಡಿಜಿ ಶುರುವಾತಿಗೆ](../../README.md) | **⏱️ ಅವಧಿ**: 45-60 ನಿಮಿಷಗಳು | **⭐ ಕಠಿನತೆ**: ಮಧ್ಯಮ

---

## ಅವಲೋಕನ

ಈ ಅಧ್ಯಾಯವು ಪರಿಸರ ಸಂರಚನೆ, ಪರಿಶೀಲನೆ ಮಾದರಿಗಳು ಹಾಗೂ ಅಜೂರ್ ಡೆವಲಪರ್ CLI ನಿಯೋಜನೆಗಳಿಗಾಗಿ ಭದ್ರತಾ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು ಒಳಗೊಂಡಿದೆ.

> ಜುಲೈ 2026 ರಲ್ಲಿ `azd 1.27.1` ಮೇಲೆ ಮಾನ್ಯಗೊಳಿಸಲಾಗಿದೆ.

## ಕಲಿಕೆ ಉದ್ದೇಶಗಳು

ಈ ಅಧ್ಯಾಯವನ್ನು ಸಂಪೂರ್ಣಗೊಳಿಸಿದ ಮೂಲಕ, ನೀವು:
- AZD ಸಂರಚನಾ ಹೈರಾರ್ಕಿಯನ್ನು ನಿಪುಣತೆಗೊಳಿಸುವಿರಿ
- ಹಲವಾರು ಪರಿಸರಗಳನ್ನು ನಿರ್ವಹಿಸುವಿರಿ (ಡೆವ್, ಸ್ಟೇಜಿಂಗ್, ಪ್ರೋಡ್)
- ನಿರ್ವಹಿತ ಗುರುತುಗಳೊಂದಿಗೆ ಸುರಕ್ಷಿತ ಪರಿಶೀಲನೆಯನ್ನು ಪ್ರತಿಷ್ಠಾಪಿಸುವಿರಿ
- ಪರಿಸರ-ನಿರ್ದಿಷ್ಟ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಸಂರಚಿಸುವಿರಿ

## 🚀 ವೇಗದ ಪ್ರಾರಂಭ

```bash
# ಬಹು ಪರಿಸರಗಳನ್ನು ರಚಿಸಿ
azd env new dev
azd env new staging
azd env new prod

# ಪರಿಸರಗಳನ್ನು ಬದಲಾಯಿಸಿ
azd env select prod

# ಪರಿಸರ ಚರಗಳನ್ನು ಹೊಂದಿಸಿ
azd env set AZURE_LOCATION eastus
azd env set SKU_NAME P1v3

# ಸಂರಚನೆಯನ್ನು ವೀಕ್ಷಿಸಿ
azd env get-values
```

---

## 🔧 ಸಂರಚನಾ ಹೈರಾರ್ಕಿ

AZD ಈ ಕ್ರಮದಲ್ಲಿ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಅನ್ವಯಿಸುತ್ತದೆ (ಹಿಂದಿನವು ಮೇಲಿನದು ಪೂರ್ವಗೊಳ್ಳುತ್ತದೆ):

1. **ಡಿಫಾಲ್ಟ್ ಮೌಲ್ಯಗಳು** (ಟೆಂಪ್ಲೇಟ್ಗಳಲ್ಲಿ ನಿರ್ಮಿಸಲಾಗಿದೆ)
2. **azure.yaml** (ಪ್ರಾಜೆಕ್ಟ್ ಸಂರಚನೆ)
3. **ಪರಿಸರ ಚೊಕ್ಕಟ್ಟಿನ ಮೌಲ್ಯಗಳು** (`azd env set`)
4. **ಕಮಾಂಡ್-ಲೈನ್ ಝೋಣಿಕೆಗಳು** (`--location eastus`)

---

## 🔐 ಭದ್ರತಾ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು

```bash
# ನಿರ್ವಹಿತ ಐಡેન્ટಿಟಿ ಬಳಸಿ (ಶಿಫಾರಸ್ಸು ಮಾಡಲಾಗಿದೆ)
azd env set AZURE_USE_MANAGED_IDENTITY true

# AZD ದೃಢೀಕರಣ ಸ್ಥಿತಿಯನ್ನು ಪರಿಶೀಲಿಸಿ
azd auth status

# ಐಚ್ಛಿಕ: ನೀವು az ಕಮಾಂಡ್‌ಗಳನ್ನು ರನ್ ಮಾಡಲು ಯೋಜಿಸಿರುವಲ್ಲಿ Azure CLI ಸಂಕೇತವನ್ನು ಪರಿಶೀಲಿಸಿ
az account show

# ಅಗತ್ಯವಿದ್ದರೆ ಪುನಃ ದೃಢೀಕರಿಸಿ
azd auth login

# ಐಚ್ಛಿಕ: az ಕಮಾಂಡ್‌ಗಳಿಗೆ Azure CLI ದೃಢೀಕರಣವನ್ನು ರಿಫ್ರೆಶ್ ಮಾಡಿ
az login
```
