# ಅಧ್ಯಾಯ 5: ಬಹು-ಏಜಂಟ್ AI ಪರಿಹಾರಗಳು

**📚 Course**: [AZD ಆರಂಭಿಕರಿಗೆ](../../README.md) | **⏱️ Duration**: 2-3 ಗಂಟೆಗಳು | **⭐ Complexity**: ಉನ್ನತ

---

## ಅವಲೋಕನ

ಈ ಅಧ್ಯಾಯವು ಉನ್ನತ ಬಹು-ಏಜಂಟ್ ವಾಸ್ತುಶಿಲ್ಪ ಮಾದರಿಗಳನ್ನು, ಏಜಂಟ್ ಸಂಯೋಜನೆ, ಮತ್ತು ಸಂಕೀರ್ಣ ಪರಿಸ್ಥಿತಿಗಳಿಗಾಗಿ ಉತ್ಪಾದನಾ-ಸಿದ್ಧ AI ನಿಯೋಜನೆಗಳನ್ನು ಒಳಗೊಂಡಿರುತ್ತದೆ.

## ಕಲಿಕೆ ಉದ್ದೇಶಗಳು

ಈ ಅಧ್ಯಾಯವನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದಾಗ, ನೀವು:
- ಬಹು-ಏಜಂಟ್ ವಾಸ್ತುಶಿಲ್ಪ ಮಾದರಿಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು
- ಸಮನ್ವಯಿತ AI ಏಜಂಟ್ ವ್ಯವಸ್ಥೆಗಳನ್ನು ನಿಯೋಜಿಸಲು
- ಏಜಂಟ್-ಗೂ-ಏಜಂಟ್ ಸಂವಹನವನ್ನು ಜಾರಿಗೊಳಿಸಲು
- ಉತ್ಪಾದನಾ-ಸಿದ್ಧ ಬಹು-ಏಜಂಟ್ ಪರಿಹಾರಗಳನ್ನು ನಿರ್ಮಿಸಲು

## 🎯 ಪ್ರಮುಖ ಪರಿಹಾರ: ರಿಟೇಲ್ ಬಹು-ಏಜಂಟ್

The [ರಿಟೇಲ್ ಬಹು-ಏಜಂಟ್ ಪರಿಹಾರ](../../examples/retail-scenario.md) ಪ್ರದರ್ಶಿಸುತ್ತದೆ:

- **ಗ್ರಾಹಕ ಏಜೆಂಟ್**: ಬಳಕೆದಾರ ಸಂವಹನ ಮತ್ತು ಆಸಕ್ತಿಗಳನ್ನು ನಿರ್ವಹಿಸುತ್ತದೆ
- **ಉಳಿಕೆ ಏಜೆಂಟ್**: ಸ್ಟಾಕ್ ಮತ್ತು ಆರ್ಡರ್ ಪ್ರಕ್ರಿಯೆಯನ್ನು ನಿರ್ವಹಿಸುತ್ತದೆ
- **ಸಂಯೋಜಕ**: ಏಜಂಟ್‌ಗಳ ನಡುವಿನ ಸಂಯೋಜನೆಯನ್ನು ನಡೆಸುತ್ತದೆ
- **ಹಂಚಿಕೊಂಡ ಸ್ಮೃತಿ**: ಏಜಂಟ್‌ಗಳ ನಡುವೆ ಪ್ರಸಂಗ ನಿರ್ವಹಣೆ

### ಬಳಸಿದ ಸೇವೆಗಳು

| Service | Purpose |
|---------|---------|
| Azure OpenAI | ಭಾಷಾ ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವಿಕೆ |
| Azure AI Search | ಉತ್ಪನ್ನ ಕ್ಯಾಟಾಲಾಗ್ |
| Cosmos DB | ಏಜಂಟ್ ಸ್ಥಿತಿ ಮತ್ತು ಸ್ಮೃತಿ |
| Container Apps | ಏಜಂಟ್ ಹೋಸ್ಟಿಂಗ್ |
| Application Insights | ನಿರೀಕ್ಷಣೆ |
