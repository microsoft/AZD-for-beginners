# ಅಧ್ಯಾಯ 4: ಇನ್‌ಫ್ರಾಸ್ಟ್ರಕ್ಷರ್ ಅಸ್ ಕೋಡ್ & ಡಿಪ್ಲಾಯ್‌ಮೆಂಟ್

**📚 ಕೋರ್ಸ್**: [ಆರಂಭಿಕರಿಗಾಗಿ AZD](../../README.md) | **⏱️ ಅವಧಿ**: 1-1.5 ಗಂಟೆಗಳು | **⭐ ಸಂಕೀರ್ಣತೆ**: ಮಧ್ಯಮ

## ಅವಲೋಕನ

ಈ ಅಧ್ಯಾಯವು ಬೈಸಿಪ್ ಟೆಂಪ್ಲೇಟ್ಗಳೊಂದಿಗೆ ಇನ್‌ಫ್ರಾಸ್ಟ್ರಕ್ಚರ್ ಅಸ್ ಕೋಡ್ (IaC) ಮಾದರಿಗಳನ್ನು, ಕುತೂಹಲದ ವನರಗಳನ್ನು, ಮತ್ತು ಅಜ್ಯೂರ್ ಡೆವೆಲಪರ್ CLI ಬಳಸಿ ನಿಯೋಜನೆ ನಿರ್ವಹಣೆ ತಂತ್ರಗಳನ್ನು ಒಳಗೊಂಡಿದೆ.

> ಜುಲೈ 2026ರಲ್ಲಿ `azd 1.27.1` ಬಳಸಿಕೊಂಡು ಪರಿಶೀಲಿಸಲಾಗಿದೆ.

## ಅಭ್ಯಾಸ ಗುರಿಗಳು

ಈ ಅಧ್ಯಾಯವನ್ನು ಪೂರ್ಣಗೊಳಿಸುವ ಮೂಲಕ, ನೀವು:
- ಬೈಸಿಪ್ ಟೆಂಪ್ಲೇಟ್ನ ರಚನೆ ಮತ್ತು ವಾಕ್ಯರಚನೆಯನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವುದು
- `azd provision` ಬಳಸಿ ಅಜ್ಯೂರ್ ಸಂಪನ್ಮೂಲಗಳನ್ನು ಜಾರಿಗೆ ತರಲು
- `azd deploy` ಬಳಸಿ ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ನಿಯೋಜಿಸಲು
- ಬ್ಲೂ-ಗ್ರೀನ್ ಮತ್ತು ರೋಲಿಂಗ್ ನಿಯೋಜನೆ ತಂತ್ರಗಳನ್ನು ಜಾರಿಗೆ ತರಲು

## 🚀 ವೇಗದ ಪ್ರಾರಂಭ

```bash
# ಟೆಂಪ್ಲೇಟಿನಿಂದ ಪ್ರಾರಂಭಿಸಿ
azd init --template azure-functions-python-v2-http

# ಏನು ಸೃಷ್ಟಿಯಾಗುತ್ತದೆ ಎಂಬುದನ್ನು પૂર્વದೃಶ್ಯಿಸಿ
azd provision --preview

# ಮೂಲಸೌಕರ್ಯ ಮಾತ್ರ ಒದಗಿಸಿ
azd provision

# ಕೋಡ್ ಮಾತ್ರ ನಿಯೋಜಿಸಿ
azd deploy

# ಅಥವಾ ಎರಡನ್ನೂ ಕೂಡ 함께
azd up
```

    └── api/

## 🔧 ಪ್ರಮುಖ ಕಮಾಂಡ್‌ಗಳು

| ಕಮಾಂಡ್ | ವಿವರಣೆ |
|---------|-------------|
| `azd init` | ಪ್ರಾಜೆಕ್ಟ್ ಪ್ರಾರಂಭಿಸು |
| `azd provision` | ಅಜ್ಯೂರ್ ಸಂಪನ್ಮೂಲಗಳನ್ನು ರಚಿಸು |
| `azd deploy` | ಅಪ್ಲಿಕೇಶನ್ ಕೋಡ್ ನಿಯೋಜಿಸು |
| `azd up` | provision + deploy |
| `azd down` | ಎಲ್ಲಾ ಸಂಪನ್ಮೂಲಗಳನ್ನು ಅಳಿಸು |
