# ಅಧ್ಯಾಯ 1: ನೆಲೆ ಮತ್ತು ವೇಗದ ಪ್ರಾರಂಭ

**📚 ಪಾಠ್ಯಕ್ರಮ**: [ಅಜಡ್ ಆರಂಭಿಕರಿಗೆ](../../README.md) | **⏱️ ಅವಧಿ**: 30-45 ನಿಮಿಷಗಳು | **⭐ ಸಂಕೀರ್ಣತೆ**: ಆರಂಭಿಕ

---

## ಅವಲೋಕನ

ಈ ಅಧ್ಯಾಯದಲ್ಲಿ ಅಪಾಜ್ ಡೆವಲಪರ್ CLI (azd) ನ ಮೂಲಭೂತಾಂಶಗಳನ್ನು ಪರಿಚಯಿಸಲಾಗಿದೆ. ನೀವು ಕೋರ್ ತತ್ವಗಳನ್ನು ಅಧ್ಯಯನ ಮಾಡುತ್ತೀರಿ, ಉಪಕರಣಗಳನ್ನು ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಿ, ಮತ್ತು ನಿಮ್ಮ ಮೊದಲ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಅಜೂರ್‌ಗೆ വിന್ಯಾಸಗೊಳಿಸುವುದನ್ನು ಕಲಿಯುತ್ತೀರಿ.

> `azd 1.23.12` ವಿರುದ್ಧ ಮರ್ಚ್ 2026 ರಲ್ಲಿ ಪರಿಶೀಲಿಸಲಾಗಿದೆ.

## ಕಲಿಕೆಯ ಉದ್ದೇಶಗಳು

ಈ ಅಧ್ಯಾಯವನ್ನು ಪೂರ್ಣಗೊಳಿಸುವ ಮೂಲಕ, ನೀವು:
- ಅಜರ್ ಡೆವಲಪರ್ CLI ಎಂದರೆ ಏನು ಮತ್ತು ಅದು ಅಜರ್ CLI ಯಿಂದ ಹೇಗೆ ಭಿನ್ನವಾಗಿರುತ್ತದೆ ಎಂದು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವುದು
- ನಿಮ್ಮ ವೇದಿಕೆಯ ಮೇಲೆ AZD ಅನ್ನು ಇನ್‌ಸ್ಟಾಲ್ ಮತ್ತು ಸಂರಚಿಸುವುದು
- `azd up` ಬಳಸಿಕೊಂಡು ನಿಮ್ಮ ಮೊದಲ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಅಜೂರ್‌ಗೆ ವಿನ್ಯಾಸಗೊಳಿಸುವುದು
- `azd down` ಬಳಸಿ ಸಂಪನ್ಮೂಲಗಳನ್ನು ತೊಡೆದು ಹಾಕುವುದು

## ✅ ಇಲ್ಲಿ ಪ್ರಾರಂಭಿಸಿ: ನಿಮ್ಮ ಸೆಟಪ್ ಸತ್ಯಾಪಿಸಿ

ನೀವು ಪ್ರಾರಂಭಿಸುವ ಮೊದಲು, ನಿಮ್ಮ ಲೋಕಲ್ ಮಷೀನ್ ಅಧ್ಯಾಯ 1 ಟೆಂಪ್ಲೇಟ್ಗಾಗಿ ಸಿದ್ಧವಿದೆಯೇ ಎಂದು ದೃಢೀಕರಿಸಿ:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

ಸ್ಕ್ರಿಪ್ಟ್ ಕಾಣಿಸಿಕೊಂಡ ಉಪಕರಣಗಳು ತೆಗೆದುಹಾಕದಿದ್ದರೆ, ಮೊದಲು ಅವುಗಳನ್ನು ಸರಿಪಡಿಸಿ, ನಂತರ ಅಧ್ಯಾಯವನ್ನು ಮುಂದುವರಿಸಿ.

---

## 🚀 ವೇಗದ ಪ್ರಾರಂಭ

```bash
# ಸ್ಥಾಪನೆ ಪರಿಶೀಲಿಸಿ
azd version

# AZDಗಾಗಿ ಪ್ರಾಮಾಣೀಕರಿಸಿ
# ಐಚ್ಛಿಕ: ನೀವು ನೇರವಾಗಿ Azure CLI ಕಮಾಂಡ್‌ಗಳನ್ನು ನಡೆಸಲು ಯೋಜಿಸಿರುವರೆಂದರೆ az ಲಾಗಿನ್
azd auth login

# ನಿಮ್ಮ ಪ್ರಥಮ ಆಪ್ ನಿಯೋಜಿಸಿ
azd init --template todo-nodejs-mongo
azd up

# ಮುಗಿದ ಮೇಲೆ ಸ್ವಚ್ಛಗೊಳಿಸಿ
azd down --force --purge
```

## ✅ ಯಶಸ್ಸಿನ ಮಾನದಂಡಗಳು

ಈ ಅಧ್ಯಾಯವನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ ನಂತರ, ನೀವು ಸಾಧ್ಯವಾಗಬೇಕು:

```bash
azd version              # ನಮೂದಿಸಿದ ಸಂಸ್ಕರಣೆಯನ್ನು ತೋರಿಸುತ್ತದೆ
azd init --template todo-nodejs-mongo  # ಪ್ರಾಜೆಕ್ಟ್ ಅನ್ನು ಪ್ರಾರಂಭಿಸುತ್ತದೆ
azd up                   # ಅಜೂರ್ ಗೆ ನಿಯೋಜಿಸುತ್ತದೆ
azd show                 # ಚಾಲನೆ ಆಸುಪಾಸಿನ ಅಪ್ಲಿಕೇಶನ್ URL ಅನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ
azd down --force --purge # ಸಂಪನ್ಮೂಲಗಳನ್ನು ಶುದ್ಧಗೊಳಿಸುತ್ತದೆ
```
