# ಅಧ್ಯಾಯ 1: ಮೂಲಭೂತ ಮತ್ತು ತ್ವರಿತ ಪ್ರಾರಂಭ

**📚 ಕೋರ್ಸ್**: [AZD For Beginners](../../README.md) | **⏱️ ಅವಧಿ**: 30-45 ನಿಮಿಷಗಳು | **⭐ ಸಂಕೀರ್ಣತೆ**: ಆರಂಭಿಕ

---

## ಅವಲೋಕನ

ಈ ಅಧ್ಯಾಯವು Azure Developer CLI (azd) ನ ಮೂಲತತ್ವಗಳನ್ನು ಪರಿಚಯಿಸುತ್ತದೆ. ನೀವು ಪ್ರಮುಖ ತತ್ವಗಳನ್ನು ತಿಳಿದುಕೊಳ್ಳುವಿರಿ, ಉಪಕರಣಗಳನ್ನು ಸ್ಥಾಪನೆ ಮಾಡುತ್ತೀರಿ, ಮತ್ತು ನಿಮ್ಮ ಮೊದಲ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು Azure ಗೆ ಡಿಪ್ಲಾಯ್ ಮಾಡುತ್ತೀರಿ.

> ಜೂನ್ 2026 ರಲ್ಲಿ `azd 1.25.6` ಮೂಲಕ ಪರಿಶೀಲಿಸಲಾಗಿದೆ.

## ಕಲಿಕಾ ಉದ್ದೇಶಗಳು

ಈ ಅಧ್ಯಾಯವನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದಾಗ, ನೀವು:
- Azure Developer CLI ಏನು ಮತ್ತು ಇದು Azure CLI ಗೆ ಹೇಗೆ ಭಿನ್ನವಾಗಿದೆ ಎಂಬುದನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಿ
- ನಿಮ್ಮ ವೇದಿಕೆಯ ಮೇಲೆ AZD ಅನ್ನು ಸ್ಥಾಪಿಸಿ ಮತ್ತು ಸಂರಚಿಸಿ
- `azd up` ಮೂಲಕ ನಿಮ್ಮ ಮೊದಲ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು Azure ಗೆ ಡಿಪ್ಲಾಯ್ ಮಾಡಿ
- `azd down` ಬಳಸಿ ಸಂಪನ್ಮೂಲಗಳನ್ನು ತೆರವುಗೊಳಿಸಿ

## ✅ ಇಲ್ಲಿ ಪ್ರಾರಂಭಿಸಿ: ನಿಮ್ಮ ಸಜ್ಜತೆಯನ್ನು ಪರಿಶೀಲಿಸಿ

Before you begin, confirm that your local machine is ready for the Chapter 1 template:

**Windows:**
```powershell
.\validate-setup.ps1
```

**macOS / Linux:**
```bash
bash ./validate-setup.sh
```

If the script reports missing tools, fix those first and then continue with the chapter.

---

## 🚀 ತ್ವರಿತ ಪ್ರಾರಂಭ

```bash
# ಸ್ಥಾಪನೆಯನ್ನು ಪರಿಶೀಲಿಸಿ
azd version

# AZDಗಾಗಿ ಪ್ರಾಮಾಣೀಕರಿಸಿ
# ಐಚ್ಛಿಕ: ನೀವು Azure CLI ಆಜ್ಞೆಗಳನ್ನು ನೇರವಾಗಿ ನಡೆಸಲು ಯೋಜಿಸುತ್ತಿದ್ದರೆ az login ಮಾಡಿ
azd auth login

# ನಿಮ್ಮ ಮೊದಲ ಅಪ್ಲಿಕೇಶನನ್ನು ನಿಯೋಜಿಸಿ
azd init --template todo-nodejs-mongo
azd up

# ಮುಗಿಸಿದ ನಂತರ ಸ್ವಚ್ಛಗೊಳಿಸಿ
azd down --force --purge
```

## ✅ ಯಶಸ್ಸಿನ ಮಾನದಂಡಗಳು

ಈ ಅಧ್ಯಾಯವನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ ನಂತರ, ನೀವು ಈ ಕೆಳಕಂಡ ಕಾರ್ಯಗಳನ್ನು ನಡೆಸಬಲ್ಲಿರಬೇಕು:

```bash
azd version              # ಸ್ಥಾಪಿತ ಆವೃತ್ತಿಯನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ
azd init --template todo-nodejs-mongo  # ಪ್ರಾಜೆಕ್ಟ್ ಅನ್ನು ಪ್ರಾರಂಭಿಸುತ್ತದೆ
azd up                   # Azure ಗೆ ಡಿಪ್ಲಾಯ್ ಮಾಡುತ್ತದೆ
azd show                 # ಚಾಲನೆಯಲ್ಲಿರುವ ಅಪ್ಲಿಕೇಶನ್ URL ಅನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ
azd down --force --purge # ಸಂಪನ್ಮೂಲಗಳನ್ನು ತೆರವುಗೊಳಿಸುತ್ತದೆ
```
