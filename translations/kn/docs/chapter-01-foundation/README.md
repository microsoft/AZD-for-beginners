# ಅಧ್ಯಾಯ 1: ಮೂಲಭೂತ ಮತ್ತು ವೇಗದ ಪ್ರಾರಂಭ

**📚 ಪಾಠ್ಯಕ್ರಮ**: [ಆರಂಭಿಕರಿಗಾಗಿ AZD](../../README.md) | **⏱️ ಕಾಲಾವಧಿ**: 30-45 ನಿಮಿಷ | **⭐ ಸಂಕೀರ್ಣತೆ**: ಆರಂಭಿಕ

---

## ಸಾರಾಂಶ

ಈ ಅಧ್ಯಾಯವು Azure Developer CLI (azd) ನ ಮೂಲಭೂತಗಳನ್ನು ಪರಿಚಯಿಸುತ್ತದೆ. ನೀವು ಮೂಲ ಪರಿಕಲ್ಪನೆಗಳನ್ನು ತಿಳಿದುಕೊಳ್ಳುತ್ತೀರಿ, ಸಾಧನಗಳನ್ನು ಸ್ಥಾಪಿಸಿ ಮತ್ತು ಸಂರಚಿಸಿ, ಮತ್ತು ನಿಮ್ಮ ಮೊದಲ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು Azure ಗೆ ನಿಯೋಜಿಸಲು ಕಲಿಯುತ್ತೀರಿ.

## ಕಲಿಕೆಯ ಉದ್ದೇಶಗಳು

ಈ ಅಧ್ಯಾಯವನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ ನಂತರ, ನೀವು:
- Azure Developer CLI ಎಂದರೇನು ಮತ್ತು ಅದು Azure CLI ಯಿಂದ ಹೇಗೆ ಭಿನ್ನವಾಗುತ್ತದೆ ಎಂಬನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಬಹುದು
- ನಿಮ್ಮ ವೇದಿಕೆಯಲ್ಲಿ AZD ಅನ್ನು ಸ್ಥಾಪಿಸಿ ಮತ್ತು ಹೊಂದಿಸಬಹುದು
- `azd up` ಬಳಸಿ ನಿಮ್ಮ ಮೊದಲ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು Azure ಗೆ ನಿಯೋಜಿಸಬಹುದು
- `azd down` ಬಳಸಿ ಸಂಪನ್ಮೂಲಗಳನ್ನು ಶುಚಿಗೊಳಿಸಬಹುದು

## 🚀 ವೇಗದ ಪ್ರಾರಂಭ

```bash
# ಸ್ಥಾಪನೆಯನ್ನು ಪರಿಶೀಲಿಸಿ
azd version

# ಆಜ್ಯೂರ್‌ಗೆ ಲಾಗಿನ್ ಮಾಡಿ
azd auth login

# ನಿಮ್ಮ ಮೊದಲ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ನಿಯೋಜಿಸಿ
azd init --template todo-nodejs-mongo
azd up

# ಮುಗಿದ ನಂತರ ಸ್ವಚ್ಛಗೊಳಿಸಿ
azd down --force --purge
```

## ✅ ಯಶಸ್ಸಿನ ಮಾನದಂಡ

ಈ ಅಧ್ಯಾಯವನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ ನಂತರ, ನೀವು ಈ ಸಾಮರ್ಥ್ಯಗಳನ್ನು ಹೊಂದಿರಬೇಕು:

```bash
azd version              # স্থಾಪಿಸಲಾದ ಆವೃತ್ತಿಯನ್ನು ತೋರಿಸುತ್ತದೆ
azd init --template todo-nodejs-mongo  # ಪ್ರಾಜೆಕ್ಟ್ ಅನ್ನು ಪ್ರಾರಂಭಿಸುತ್ತದೆ
azd up                   # Azure ಗೆ ನಿಯೋಜಿಸುತ್ತದೆ
azd show                 # ಚಾಲಿಸುತ್ತಿರುವ ಅಪ್ಲಿಕೇಶನ್ URL ಅನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ
azd down --force --purge # ಸಂಪನ್ಮೂಲಗಳನ್ನು ತೆರವುಗೊಳಿಸುತ್ತದೆ
```
