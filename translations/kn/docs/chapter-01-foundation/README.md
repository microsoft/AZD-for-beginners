# ಅಧ್ಯಾಯ 1: ನೋಡುಗೆಯ & ದ್ರುತ ಪ್ರಾರಂಭ

**📚 ಹಂದಗಳು**: [ಶುರುಮಾಡುವವರಿಗಾಗಿ AZD](../../README.md) | **⏱️ ಅವಧಿ**: 30-45 ನಿಮಿಷಗಳು | **⭐ ಜಟಿಲತೆ**: ಶುರುಮಾಡುವವರು

---

## ಅವಲೋಕನ

ಈ ಅಧ್ಯಾಯದಲ್ಲಿ ಅಜೂರ್ ಡೆವೆಲಪರ್ CLI (azd) ಮೂಲತತ್ವಗಳನ್ನು ಪರಿಚಯಿಸಲಾಗುತ್ತದೆ. ನೀವು ಮುಖ್ಯ ಸಂಜ್ಞೆಗಳನ್ನು ಕಲಿಯುತ್ತೀರಿ, ಸಾಧನಗಳನ್ನು ಸ್ಥಾಪಿಸುತ್ತೀರಿ ಮತ್ತು ನಿಮ್ಮ ಪ್ರಥಮ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಅಜೂರ್‌ಗೆ ನಿಯೋಜಿಸುತ್ತೀರಿ.

> ಜುಲೈ 2026 ರಲ್ಲಿ `azd 1.27.1` ಆಧಾರದ ಮೇಲೆ ಪರಿಶೀಲಿಸಲಾಗಿದೆ.

## ಕಲಿಕೆಯ ಗುರಿಗಳು

ಈ ಅಧ್ಯಾಯವನ್ನು ಪೂರ್ಣಗೊಳಿಸುವ ಮೂಲಕ ನೀವು:
- ಅಜೂರ್ ಡೆವೆಲಪರ್ CLI ಎಂದರೇನು ಮತ್ತು ಅದು ಅಜೂರ್ CLI ಯಿಂದ ಹೇಗೆ ಭಿನ್ನವಾಗುತ್ತದೆ ಎಂದು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುತ್ತೀರಿ
- ನಿಮ್ಮ ವೇದಿಕೆಯ ಮೇಲೆ AZD ಅನ್ನು ಸ್ಥಾಪಿಸಿ ಮತ್ತು ಸಂರಚಿಸುತ್ತೀರಿ
- `azd up` ಬಳಸಿ ಮೊದಲ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಅಜೂರ್‌ಗೆ ನಿಯೋಜಿಸುತ್ತೀರಿ
- `azd down` ಬಳಸಿ ಸಂಪನ್ಮೂಲಗಳನ್ನು ಶುಚಿಗೊಳಿಸುತ್ತೀರಿ

## ✅ ಇಲ್ಲಿ ಪ್ರಾರಂಭಿಸಿ: ನಿಮ್ಮ ಸಿದ್ಧತೆಯನ್ನು ಪರಿಶೀಲಿಸಿ

ಪ್ರಾರಂಭಿಸುವ ಮೊದಲು, ನಿಮ್ಮ ಸ್ಥಳೀಯ ಯಂತ್ರವು ಅಧ್ಯಾಯ 1 ಟೆಂಪ್ಲೇಟಿಗಾಗಿ ಸಿದ್ಧವಾಗಿದೆ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ:

**ವಿಂಡೋಸ್:**
```powershell
.\validate-setup.ps1
```

**ಮ್ಯಾಕ್ಓಎಸ್ / ಲಿನಕ್ಸ್:**
```bash
bash ./validate-setup.sh
```

స్క్రಿಪ్ట్ తయారుచేస్తే ಸಾಧನాలు ಕಾಣಿಸಲಿಲ್ಲని తెలిపితే, ಮೊದಲು ಅವುಗಳನ್ನು ಸರಿಪಡಿಸಿ ಮತ್ತು ನಂತರ ಅಧ್ಯಾಯವನ್ನು ಮುಂದುವರೆಸಿ.

---

## 🚀 ದ್ರುತ ಪ್ರಾರಂಭ

```bash
# ಸ್ಥಾಪನೆ ಪರಿಶೀಲಿಸಿ
azd version

# AZDಗಾಗಿ ದೃಢೀಕರಿಸಿ
# ಐಚ್ಛಿಕ: ನೀವು ನೇರವಾಗಿ Azure CLI ಆಜ್ಞೆಗಳನ್ನು 실행ಿಸಬೇಕೆಂದಿದ್ದರೆ az ಲಾಗಿನ್ ಮಾಡಿ
azd auth login

# ನಿಮ್ಮ ಮೊದಲ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ನಿಯೋಜಿಸಿ
azd init --template todo-nodejs-mongo
azd up

# ಮುಗಿಸಿದ ನಂತರ ಸ್ವಚ್ಛಗೊಳಿಸಿ
azd down --force --purge
```

## ✅ ಯಶಸ್‌ಕ್ರಿಯತೆಯ ಮಾನದಂಡಗಳು

ಈ ಅಧ್ಯಾಯವನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ ನಂತರ, ನೀವು ಸಾಧ್ಯವಾಗಿರಬೇಕು:

```bash
azd version              # ಸ್ಥಾಪಿತ ಆವೃತ್ತಿಯನ್ನು ತೋರಿಸುತ್ತದೆ
azd init --template todo-nodejs-mongo  # ಪ್ರಾಜೆಕ್ಟ್ ಪ್ರಾರಂಭಿಸುತ್ತದೆ
azd up                   # ಅಜೂರ್‌ಗೆ ನಿಯೋಜಿಸುತ್ತದೆ
azd show                 # ಚಾಲನೆಯಲ್ಲಿರುವ ಅಪ್ಲಿಕೇಶನ್ URL ಅನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ
azd down --force --purge # ಸಂಪನ್ಮೂಲಗಳನ್ನು ಸ್ವಚ್ಛಗೊಳಿಸುತ್ತದೆ
```
