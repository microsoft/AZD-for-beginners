# AZD For Beginners ಗೆ ಕೊಡುಗೆ ನೀಡುವುದು

AZD For Beginners ಪಾಠ್ಯಕ್ರಮಕ್ಕೆ ಕೊಡುಗೆ ನೀಡಲು ನೀವು ಆಸಕ್ತಿ ತೋರಿದ್ದಕ್ಕೆ ಧನ್ಯವಾದಗಳು! ಈ ದಾಖಲೆ ಕೊಡುಗುದಾರರಿಗೆ ಮಾರ್ಗದರ್ಶನ ಮತ್ತು ಮಾಹಿತಿ ಒದಗಿಸುತ್ತದೆ.

## ಕೊಡುಗೆ ನೀಡುವುದು ಹೇಗೆ

### ಸಮಸ್ಯೆಗಳನ್ನು ವರದಿ ಮಾಡುವುದು

ನೀವು ದೋಷಗಳು, ತಪ್ಪುಗಳು ಕಂಡುಹೇಳಿದರೆ ಅಥವಾ ಸುಧಾರಣೆಗಳ ಸಲಹೆಗಳು ಇದ್ದರೆ:

1. **ಇದೇ ಸಮಸ್ಯೆಗಳನ್ನು ಪರಿಶೀಲಿಸಿ**: ನಕಲಿಗಳನ್ನು ತಪ್ಪಿಸಲು [ಇದೇ ಸಮಸ್ಯೆಗಳು](https://github.com/microsoft/azd-for-beginners/issues) ಅನ್ನು ಹುಡುಕಿ
2. **ಹೊಸ ಸಮಸ್ಯೆ ಸೃಷ್ಟಿಸಿ**: ಸೂಕ್ತ ಸಮಸ್ಯೆ ಟೆಂಪ್ಲೇಟನ್ನು ಬಳಸಿರಿ
3. **ವಿವರಗಳನ್ನು ಒದಗಿಸಿ**: ಪುನರಾವರ್ತಿಸಲು ಕ್ರಮಗಳು, ನಿರೀಕ್ಷಿತ ನಡೆ ಮತ್ತು ಪ್ರಸ್ತುತ ಆಯುಕ್ತವಾಗಿದ್ದರೆ ಸ್ಕ್ರೀನ್ಶಾಟ್‌ಗಳನ್ನು ಸೇರಿಸಿ

### ಬದಲಾವಣೆಗಳನ್ನು ಪ್ರಸ್ತಾಪಿಸುವುದು

1. **ರಿಪೊಜಿಟೋರಿಯನ್ನು Fork ಮಾಡಿ**: ಯೋಜನೆಯ ನಿಮ್ಮ ಸ್ವಂತ fork ರಚಿಸಿ
2. **ಬ್ರಾಂಚ್ ರಚಿಸಿ**: ನಿಮ್ಮ ಬದಲಾವಣೆಗಳಿಗಾಗಿ ವೈಶಿಷ್ಟ್ಯ ಬ್ರಾಂಚ್ ರಚಿಸಿ
3. **ಬದಲಾವಣೆಗಳನ್ನು ಮಾಡಿ**: ನಮ್ಮ ಕೋಡಿಂಗ್ ಮತ್ತು ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಮಾನದಂಡಗಳನ್ನು ಅನುಸರಿಸಿ
4. **ನಿಮ್ಮ ಬದಲಾವಣೆಗಳನ್ನು ಪರೀಕ್ಷಿಸಿ**: ಎಲ್ಲಾ ಲಿಂಕ್‌ಗಳು ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತಿರುವರೂ ಇಲ್ಲವೆಂದು ಮತ್ತು ವಿಷಯ ಸರಿಯಾಗಿ-render ಆಗುವುದು ಎಂದು ಪರಿಶೀಲಿಸಿ
5. **Pull Request ಸಲ್ಲಿಸಿ**: ನಿಮ್ಮ ಬದಲಾವಣೆಗಳ ಸ್ಪಷ್ಟ ವಿವರಣೆಯನ್ನು ಒದಗಿಸಿ

### ಕೊಡುಗೆಗಳ ವಿಧಗಳು

ನಾವು ಕೆಳಗಿನ ವಿಧದ ಕೊಡುಗೆಗಳನ್ನು ಸುಸ್ವಾಗತ ಮಾಡುತ್ತೇವೆ:

#### ವಿಷಯ ಸುಧಾರಣೆಗಳು
- ಟೈಪ್ ತಪ್ಪುಗಳು, ವ್ಯಾಕರಣ ಮತ್ತು ಸ್ವರೂಪಣ ಸಮಸ್ಯೆಗಳನ್ನು ಸರಿಪಡಿಸಿ
- ಇತ್ತೀಚಿನಲ್ಲಿರುವ ದಾಖಲೆಗಳ ಸ್ಪಷ್ಟತೆಯನ್ನು ಸುಧಾರಿಸಿ
- ಹಳೆಯ ಮಾಹಿತಿಯನ್ನು ನವೀಕರಿಸಿ
- ಇಲ್ಲದಿರುವ ವಿವರಣೆಗಳು ಅಥವಾ ಸಂದರ್ಭಗಳನ್ನು ಸೇರಿಸಿ

#### ಹೊಸ ಉದಾಹರಣೆಗಳು
- ವಾಸ್ತವಿಕ ಜಗತ್ತಿನ ನಿಯೋಜನೆ ಪರಿಸ್ಥಿತಿಗಳನ್ನು ಸೇರಿಸಿ
- ಸಾಮಾನ್ಯ ಬಳಕೆ ಪ್ರಕರಣಗಳಿಗೆ ಹೊಸ AZD ಟೆಂಪ್ಲೇಟುಗಳನ್ನು ರಚಿಸಿ
- ಹೆಚ್ಚುವರಿ ಕೋಡ್ ಉದಾಹರಣೆಗಳು ಮತ್ತು ಅಭ್ಯಾಸಗಳನ್ನು ಒದಗಿಸಿ

#### ಅನುವಾದ ಬೆಂಬಲ
- ಇತ್ತೀಚಿನ ಅನುವಾದಗಳನ್ನು ನಿರ್ವಹಿಸಲು ಸಹಾಯ ಮಾಡಿ
- ಅನುವಾದದ ಗುಣಮಟ್ಟವನ್ನು ಸುಧಾರಿಸಿ
- ಹೊಸ ವಿಷಯಗಳಿಗಾಗಿ ಅನುವಾದಗಳನ್ನು ಸೇರಿಸಿ

#### ದೋಷ ವರದಿಗಳು
- ಮುರಿದ ಲಿಂಕ್‌ಗಳು ಅಥವಾ ತಪ್ಪು ಕೋಡ್ ಉದಾಹರಣೆಗಳನ್ನು ವರದಿ ಮಾಡಿ
- ದಾಖಲೆಗಳಲ್ಲಿ ಅಸಮঞ্জಸ್ಯತೆಯನ್ನು ಗುರುತಿಸಿ
- ಪ್ರವೇಶयोग್ಯತಾ ಸಮಸ್ಯೆಗಳನ್ನು ಗುರುತಿಸಿ

## ಕೊಡುಗೆ ಮಾರ್ಗಸೂಚிகள்

### ದಾಖಲೆ ಮಾನದಂಡಗಳು

- ಸ್ಪಷ್ಟ, ಪ್ರಾರಂಭಿಕರಿಗಾಗಿ ಸ್ನೇಹಪೂರ್ಣ ಭಾಷೆಯನ್ನು ಬಳಸಿರಿ
- ಕಲಿಕೆಯ ಉದ್ದೇಶಗಳು ಮತ್ತು ಫಲಿತಾಂಶಗಳನ್ನು ಸೇರಿಸಿ
- ನಿರೀಕ್ಷಿತ ಔಟ್ಪುಟ್‌ಗಳೊಂದಿಗೆ ಹಂತದ ಮೂಲಕ ಸೂಚನೆಗಳನ್ನು ಒದಗಿಸಿ
- ಸಂಬಂಧಿತ ವಿಷಯಗಳಿಗೆ ನ್ಯಾವಿಗೇಶನ್ ಲಿಂಕ್‌ಗಳನ್ನು ಸೇರಿಸಿ
- ನಿರಂತರ ಸ್ವರೂಪಣ ಮತ್ತು ಮಾರ್ಕ್ಡೌನ್ ಶೈಲಿಯನ್ನು ಬಳಸಿ

### ಕೋಡ್ ಉದಾಹರಣೆಗಳು

- ಎಲ್ಲಾ ಕೋಡ್ ಉದಾಹರಣೆಗಳು ಪರೀಕ್ಷಿಸಲ್ಪಟ್ಟ ಮತ್ತು ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತಿರುವುದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ
- ಪ್ರಮುಖ ಸಂಪ್ರದಾಯಗಳನ್ನು ವಿವರಿಸುವ ಕಾಮೆಂಟ್‌ಗಳನ್ನು ಸೇರಿಸಿ
- Azure ಮತ್ತು AZD ನ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳನ್ನು ಅನುಸರಿಸಿ
- ಅನ್ವಯವಾಗುವಲ್ಲಿ ನಿರೀಕ್ಷಿತ ಔಟ್ಪುಟ್ ಅನ್ನು ಒದಗಿಸಿ
