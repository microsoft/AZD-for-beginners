# AZD para Iniciantes: Plano do Curso & Estrutura de Aprendizagem

## Visão Geral do Curso

Domine o Azure Developer CLI (azd) através de capítulos estruturados concebidos para uma aprendizagem progressiva. **Foco especial na implantação de aplicações de IA com integração do Microsoft Foundry.**

### Por que Este Curso é Essencial para Desenvolvedores Modernos

Com base em insights da comunidade Microsoft Foundry no Discord, **45% dos desenvolvedores querem usar o AZD para cargas de trabalho de IA** mas enfrentam desafios com:
- Arquiteturas de IA complexas com múltiplos serviços
- Melhores práticas para implantação de IA em produção  
- Integração e configuração de serviços Azure AI
- Otimização de custos para cargas de trabalho de IA
- Resolução de problemas específicos de implantação de IA

### Objetivos Principais de Aprendizagem

Ao concluir este curso estruturado, irá:
- **Dominar os Fundamentos do AZD**: Conceitos essenciais, instalação e configuração
- **Implantar Aplicações de IA**: Usar o AZD com os serviços Microsoft Foundry
- **Implementar Infraestrutura como Código**: Gerir recursos Azure com templates Bicep
- **Resolver Problemas de Implantação**: Resolver problemas comuns e depurar erros
- **Otimizar para Produção**: Segurança, escalabilidade, monitorização e gestão de custos
- **Construir Soluções Multi-Agente**: Implantar arquiteturas de IA complexas
