# AZD Para Desenvolvedores de IA

!!! info "PÚBLICO DESTINATÁRIO DESTE WORKSHOP"
   
    Este workshop é para qualquer pessoa que pretenda construir aplicações Agentes de IA na Microsoft Foundry, aproveitando as melhores ferramentas e fluxos de trabalho da categoria. O workshop é adequado tanto para iniciantes como para desenvolvedores experientes que têm alguma familiaridade com Azure, conceitos de IA e código Python. Por favor, verifique os pré-requisitos antes de começar.

    **Duração:** 3-4 horas  
    **Nível:** Do básico ao intermédio  

---


## Pré-Requisitos

- Uma conta pessoal no GitHub → [Crie uma gratuitamente](https://github.com/signup)
- Uma conta Azure ativa → [Crie uma gratuitamente](https://aka.ms/free)
- Familiaridade com conceitos de IA, desenvolvimento em Python, ferramentas de linha de comandos
- Experiência com VS Code e ferramentas e fluxos de trabalho do GitHub.

---

## Objetivos de Aprendizagem

No final deste workshop, você saberá como:

- [X] **Trabalhar com Azure Developer CLI** → para provisionar, implementar e gerir os seus recursos Azure.
- [X] **Usar Templates de Aplicações de IA** → para iniciar projetos de Agentes de IA na Microsoft Foundry.
- [X] **Configurar Templates** → para funcionalidades como Red Teaming, Avaliações, Rastreio & Monitorização
- [X] **Desconstruir e Explorar** → templates e fluxos de trabalho AZD usando GitHub Copilot
- [X] **Personalizar e Reimplementar** → o template inicial de Agente de IA, com os seus requisitos personalizados

---

## Estrutura do Workshop

O workshop está estruturado em módulos, cada um ancorado por um exercício prático:

1. **Introdução** - Compreender o que são templates AZD e como funcionam
1. **Seleção** - Descobrir o template "inicial" certo para o seu próximo projeto de IA
1. **Validação** - Implementar e explorar o template para verificar se funciona como anunciado
1. **Desconstrução** - Desmontar o template para entender os seus componentes e configuração
1. **Configuração** - Aprender a ativar e usar diferentes funcionalidades da Microsoft Foundry
1. **Personalização** - Reconstruir a aplicação com os seus dados e requisitos em mente
1. **Desmontagem** - Aprender a desprovisionar a infraestrutura e recuperar recursos apagados suavemente
