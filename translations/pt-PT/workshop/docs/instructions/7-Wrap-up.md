## 2. Conceitos-Chave Dominados

### Azure Developer CLI (azd)
- `azd init` - Inicializar projetos a partir de modelos
- `azd up` - Provisionar e implementar num único comando
- `azd env` - Gerir variáveis de ambiente
- `azd down` - Limpar recursos com segurança

### Estrutura do Modelo AZD
- `azure.yaml` - Configuração e hooks de implementação
- `infra/` - Infraestrutura como código Bicep
- `src/` - Código-fonte da aplicação
- `scripts/` - Scripts de automação e configuração

### Capacidades do Microsoft Foundry
- Serviço de Agente de IA para assistentes inteligentes
- Pesquisa de Ficheiros e Azure AI Search para recuperação de conhecimento
- Rastreio, monitorização e avaliação incorporados
- Red teaming para validação de segurança de IA
