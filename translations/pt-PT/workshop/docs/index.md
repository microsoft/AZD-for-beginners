# AZD para Desenvolvedores de IA Workshop

> Validado com `azd 1.23.12` em março de 2026.

> [!IMPORTANT]  
> **Este workshop está instrumentado com um guia que pode visualizar no seu browser local. Para começar, abra o repositório no GitHub Codespaces, num contentor dev, ou numa cópia local com o MkDocs instalado—depois aguarde até ver um terminal VS Code ativo e escreva:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Deverá aparecer uma janela para abrir a página de pré-visualização no browser.**

Bem-vindo ao workshop prático para aprender a Azure Developer CLI (AZD) com foco em implementação de aplicações de IA. Este workshop ajuda-o a obter uma compreensão prática dos templates AZD em 3 passos:

1. **Descoberta** - encontre o template certo para si.  
1. **Implementação** - implemente e valide que funciona  
1. **Personalização** - modifique e itere para o tornar seu!

Ao longo deste workshop, será também apresentado às ferramentas e fluxos de trabalho essenciais para desenvolvedores, para o ajudar a otimizar o seu percurso de desenvolvimento completo.

## Visão Geral do Workshop

**Duração:** 3-4 horas  
**Nível:** Iniciante a Intermédio  
**Pré-requisitos:** Familiaridade com Azure, conceitos de IA, VS Code e ferramentas de linha de comando.

Este é um workshop prático onde aprende fazendo. Depois de completar os exercícios, recomendamos rever o currículo AZD Para Iniciantes para continuar a sua aprendizagem em práticas recomendadas de Segurança e Produtividade.

| Tempo | Módulo  | Objetivo |
|:---|:---|:---|
| 15 mins | Introdução | Contextualizar, entender objetivos |
| 30 mins | Selecionar Template de IA | Explorar opções e escolher início | 
| 30 mins | Validar Template de IA | Implementar solução padrão no Azure |
| 30 mins | Desconstruir Template de IA | Explorar estrutura e configuração |
| 30 mins | Configurar Template de IA | Ativar e testar funcionalidades disponíveis |
| 30 mins | Personalizar Template de IA | Adaptar o template às suas necessidades |
| 30 mins | Desmontar Infraestrutura | Limpar e libertar recursos |
| 15 mins | Conclusão & Próximos Passos | Recursos de aprendizagem, desafio do workshop |
| | |

## O que irá aprender

Pense no Template AZD como um campo de experimentação para explorar várias capacidades e ferramentas para desenvolvimento completo na Microsoft Foundry. No final deste workshop, deverá ter uma perceção intuitiva de várias ferramentas e conceitos neste contexto.

| Conceito  | Objetivo |
|:---|:---|
| **Azure Developer CLI** | Compreender comandos e fluxos da ferramenta|
| **Templates AZD**| Entender a estrutura e configuração do projeto|
| **Azure AI Agent**| Provisionar & implementar projeto Microsoft Foundry  |
| **Azure AI Search**| Capacitar engenharia de contexto com agentes |
| **Observabilidade**| Explorar tracing, monitorização e avaliações |
| **Red Teaming**| Explorar testes adversariais e mitigação |
