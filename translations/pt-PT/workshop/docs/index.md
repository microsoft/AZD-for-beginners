# Oficina AZD para Desenvolvedores de IA

> [!IMPORTANT]  
> **Esta oficina está instrumentada com um guia que pode visualizar no seu navegador local. Para começar, deve lançar o GitHub Codespaces no repositório — depois aguarde até ver um terminal VS Code ativo e escreva:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Deve aparecer uma janela pop-up para abrir a página de pré-visualização no navegador.**

Bem-vindo à oficina prática para aprender Azure Developer CLI (AZD) com foco na implementação de aplicações de IA. Esta oficina ajuda-o a obter uma compreensão aplicada dos templates AZD em 3 passos:

1. **Descoberta** - encontre o template que é adequado para si.
1. **Implementação** - implemente e valide que funciona
1. **Personalização** - modifique e itere para que seja seu!

Ao longo desta oficina, será também apresentado às ferramentas e fluxos de trabalho centrais para desenvolvedores, para ajudá-lo a simplificar a sua jornada de desenvolvimento de ponta a ponta.

## Visão Geral da Oficina

**Duração:** 3-4 horas  
**Nível:** Iniciante a Intermédio  
**Pré-requisitos:** Familiaridade com Azure, conceitos de IA, VS Code e ferramentas de linha de comandos.

Esta é uma oficina prática onde se aprende fazendo. Depois de completar os exercícios, recomendamos que reveja o currículo AZD Para Iniciantes para prosseguir a sua aprendizagem nas melhores práticas de Segurança e Produtividade.

| Tempo| Módulo  | Objetivo |
|:---|:---|:---|
| 15 minutos | Introdução | Definir o enquadramento, compreender os objetivos |
| 30 minutos | Selecionar Template de IA | Explorar opções e escolher um ponto de partida | 
| 30 minutos | Validar Template de IA | Implementar solução padrão no Azure |
| 30 minutos | Desconstruir Template de IA | Explorar estrutura e configuração |
| 30 minutos | Configurar Template de IA | Ativar e testar funcionalidades disponíveis |
| 30 minutos | Personalizar Template de IA | Adaptar o template às suas necessidades |
| 30 minutos | Desmontar Infraestrutura | Limpar e libertar recursos |
| 15 minutos | Conclusão e Próximos Passos | Recursos de aprendizagem, desafio da oficina |
| | |

## O que Vai Aprender

Pense no Template AZD como um sandbox de aprendizagem para explorar várias capacidades e ferramentas para desenvolvimento de ponta a ponta na Microsoft Foundry. No final desta oficina, deverá ter uma noção intuitiva de várias ferramentas e conceitos neste contexto.

| Conceito  | Objetivo |
|:---|:---|
| **Azure Developer CLI** | Compreender comandos da ferramenta e fluxos de trabalho |
| **Templates AZD**| Compreender estrutura do projeto e configuração |
| **Azure AI Agent**| Provisionar e implementar projeto Microsoft Foundry  |
| **Azure AI Search**| Permitir engenharia de contexto com agentes |
| **Observabilidade**| Explorar tracing, monitorização e avaliações |
| **Red Teaming**| Explorar testes adversariais e mitigações |
