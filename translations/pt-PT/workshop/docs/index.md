# AZD para Desenvolvedores de IA Workshop

> Validado contra `azd 1.27.1` em julho de 2026.

> [!IMPORTANT]  
> **Este workshop está instrumentado com um guia do workshop que pode pré-visualizar no seu navegador local. Para começar, abra o repositório no GitHub Codespaces, num contentor de dev, ou num clone local com o MkDocs instalado—depois espere até ver um terminal VS Code ativo e escreva:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Deve aparecer um diálogo para abrir a página de pré-visualização num navegador.**

Bem-vindo ao workshop prático para aprender Azure Developer CLI (AZD) com foco no deployment de aplicações AI. Este workshop ajuda-o a adquirir uma compreensão aplicada dos templates AZD em 3 passos:

1. **Descoberta** - encontrar o template adequado para si.
1. **Deployment** - implantar e validar que funciona
1. **Personalização** - modificar e iterar para o tornar seu!

Ao longo deste workshop, será também apresentado às ferramentas essenciais para desenvolvedores e fluxos de trabalho, para o ajudar a agilizar a sua jornada de desenvolvimento de ponta a ponta.

## Visão Geral do Workshop

**Duração:** 3-4 horas  
**Nível:** Principiante a Intermédio  
**Pré-requisitos:** Familiaridade com Azure, conceitos AI, VS Code e ferramentas de linha de comando.

Este é um workshop prático onde aprende fazendo. Depois de completar os exercícios, recomendamos rever o currículo AZD Para Principiantes para continuar a sua jornada de aprendizagem em práticas recomendadas de Segurança e Produtividade.

| Tempo| Módulo  | Objetivo |
|:---|:---|:---|
| 15 mins | Introdução | Definir o enquadramento, compreender os objetivos |
| 30 mins | Selecionar Template AI | Explorar opções e escolher um template inicial | 
| 30 mins | Validar Template AI | Implantar solução padrão no Azure |
| 30 mins | Desconstruir Template AI | Explorar estrutura e configurações |
| 30 mins | Configurar Template AI | Ativar e testar funcionalidades disponíveis |
| 30 mins | Personalizar Template AI | Adaptar o template às suas necessidades |
| 30 mins | Desmontar Infraestrutura | Limpar e libertar recursos |
| 15 mins | Conclusão & Próximos Passos | Recursos de aprendizagem, Desafio do Workshop |
| | |

## O Que Vai Aprender

Pense no Template AZD como um sandbox de aprendizagem para explorar várias capacidades e ferramentas para o desenvolvimento de ponta a ponta na Microsoft Foundry. No final deste workshop, deverá ter uma noção intuitiva de várias ferramentas e conceitos neste contexto.

| Conceito  | Objetivo |
|:---|:---|
| **Azure Developer CLI** | Compreender comandos de ferramentas e fluxos de trabalho |
| **Templates AZD**| Compreender estrutura do projeto e configuração |
| **Azure AI Agent**| Provisionar e implantar projeto Microsoft Foundry  |
| **Azure AI Search**| Permitir engenharia de contexto com agentes |
| **Observabilidade**| Explorar rastreamento, monitorização e avaliações |
| **Red Teaming**| Explorar testes adversariais e mitigação |
