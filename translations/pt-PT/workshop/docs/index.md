# Oficina AZD para Desenvolvedores de IA

> Validado com a versão `azd 1.25.6` em Junho de 2026.

> [!IMPORTANT]  
> **Esta oficina está equipada com um guia que pode pré-visualizar no seu navegador local. Para começar, abra o repositório no GitHub Codespaces, num contentor de desenvolvimento, ou numa cópia local com MkDocs instalado—depois aguarde até ver um terminal VS Code ativo e digite:**  
> `mkdocs serve > /dev/null 2>&1 &`  
> **Deverá aparecer uma caixa de diálogo para abrir a página de pré-visualização no navegador.**

Bem-vindo à oficina prática para aprender a Azure Developer CLI (AZD) com foco na implementação de aplicações de IA. Esta oficina ajuda-o a ganhar uma compreensão aplicada dos modelos AZD em 3 passos:

1. **Descoberta** - encontre o modelo adequado para si.  
1. **Implementação** - implemente e valide que funciona  
1. **Personalização** - modifique e itere para ser seu!

Ao longo desta oficina, será também introduzido a ferramentas e fluxos de trabalho essenciais para desenvolvedores, ajudando a otimizar a sua jornada de desenvolvimento de ponta a ponta.

## Visão Geral da Oficina

**Duração:** 3-4 horas  
**Nível:** Iniciante a Intermédio  
**Pré-requisitos:** Familiaridade com Azure, conceitos de IA, VS Code e ferramentas de linha de comando.

Esta é uma oficina prática onde aprende fazendo. Quando completar os exercícios, recomendamos rever o currículo AZD Para Iniciantes para continuar a sua aprendizagem em melhores práticas de Segurança e Produtividade.

| Tempo| Módulo  | Objetivo |
|:---|:---|:---|
| 15 mins | Introdução | Definir o cenário, compreender os objetivos |
| 30 mins | Selecionar Modelo de IA | Explorar opções e escolher um ponto de partida | 
| 30 mins | Validar Modelo de IA | Implementar a solução padrão no Azure |
| 30 mins | Desconstruir Modelo de IA | Explorar estrutura e configuração |
| 30 mins | Configurar Modelo de IA | Ativar e experimentar as funcionalidades disponíveis |
| 30 mins | Personalizar Modelo de IA | Adaptar o modelo às suas necessidades |
| 30 mins | Desmontar Infraestrutura | Limpar e libertar recursos |
| 15 mins | Conclusão & Próximos Passos | Recursos de aprendizagem, desafio da oficina |
| | |

## O Que Vai Aprender

Pense no Modelo AZD como um ambiente de aprendizagem para explorar várias capacidades e ferramentas para desenvolvimento de ponta a ponta na Microsoft Foundry. No final desta oficina, deverá ter uma noção intuitiva de várias ferramentas e conceitos neste contexto.

| Conceito  | Objetivo |
|:---|:---|
| **Azure Developer CLI** | Compreender comandos e fluxos de trabalho da ferramenta |
| **Modelos AZD** | Compreender estrutura de projeto e configuração |
| **Azure AI Agent** | Provisionar e implementar projeto Microsoft Foundry |
| **Azure AI Search** | Permitir engenharia de contexto com agentes |
| **Observabilidade** | Explorar rastreio, monitorização e avaliações |
| **Red Teaming** | Explorar testes adversariais e mitigação |
| | |
